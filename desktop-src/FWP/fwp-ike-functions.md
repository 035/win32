# IKE/AuthIP Functions

The Windows Filtering Platform (WFP) functions used to interact with keying modules—Internet Key Exchange (IKE), Internet Key Exchange v2 (IKEv2), and Authenticated Internet Protocol (AuthIP)—are as follows.

-   IkeextGetStatistics:
    -   [**IkeextGetStatistics0**](ikeextgetstatistics0.md) (Windows Vista)
    -   [**IkeextGetStatistics1**](ikeextgetstatistics1.md) (Windows 7 and later)
-   [**IkeextSaCreateEnumHandle0**](ikeextsacreateenumhandle0.md)
-   [**IkeextSaDbGetSecurityInfo0**](ikeextsadbgetsecurityinfo0.md)
-   [**IkeextSaDbSetSecurityInfo0**](ikeextsadbsetsecurityinfo0.md)
-   [**IkeextSaDeleteById0**](ikeextsadeletebyid0.md)
-   [**IkeextSaDestroyEnumHandle0**](ikeextsadestroyenumhandle0.md)
-   IkeextSaEnum:
    -   [**IkeextSaEnum0**](ikeextsaenum0.md) (Windows Vista)
    -   [**IkeextSaEnum1**](ikeextsaenum1.md) (Windows 7)
    -   [**IkeextSaEnum2**](ikeextsaenum2.md) (Windows 8)
-   IkeextSaGetById:
    -   [**IkeextSaGetById0**](ikeextsagetbyid0.md) (Windows Vista)
    -   [**IkeextSaGetById1**](ikeextsagetbyid1.md) (Windows 7)
    -   [**IkeextSaGetById2**](ikeextsagetbyid2.md) (Windows 8)
