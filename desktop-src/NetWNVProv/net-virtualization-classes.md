# NetWNV Provider

This section provides reference information for **NetWNV** provider classes defined in NetWNV.mof and implemented in NetWNV.dll.

## In this section



| Topic                                                                                                                    | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|--------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSFT\_NetSettingData**](msft-netsettingdata-netwnv.md)<br/>                                                    | Serves as a base class for classes that provide access to network settings data, such as [**MSFT\_NetVirtualizationCustomerRouteSettingData**](msft-netvirtualizationcustomerroutesettingdata.md), [**MSFT\_NetVirtualizationGlobalSettingData**](msft-netvirtualizationglobalsettingdata.md), [**MSFT\_NetVirtualizationLookupRecordSettingData**](msft-netvirtualizationlookuprecordsettingdata.md), [**MSFT\_NetVirtualizationNextHopLookup**](msft-netvirtualizationnexthoplookup.md), [**MSFT\_NetVirtualizationProviderAddressSettingData**](msft-netvirtualizationprovideraddresssettingdata.md), and [**MSFT\_NetVirtualizationProviderRouteSettingData**](msft-netvirtualizationproviderroutesettingdata.md).<br/> |
| [**MSFT\_NetVirtualizationCustomerRouteSettingData**](msft-netvirtualizationcustomerroutesettingdata.md)<br/>     | The MSFT\_NetVirtualizationCustomerRouteSettingData class represents a customer route or forwarding rule in the WNV module policy table to define virtual network topology seen by the virtual machines.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**MSFT\_NetVirtualizationGlobalSettingData**](msft-netvirtualizationglobalsettingdata.md)<br/>                   | The MSFT\_NetVirtualizationGlobalSettingData class represents the global settings and state of the WNV module for Hyper-V Network Virtualization.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**MSFT\_NetVirtualizationLookupRecordSettingData**](msft-netvirtualizationlookuprecordsettingdata.md)<br/>       | Represents a virtualization policy record, which maps virtual addressees to physical address in the WNV module to implement Hyper-V network virtualization.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**MSFT\_NetVirtualizationNextHopLookup**](msft-netvirtualizationnexthoplookup.md)<br/>                           | Represents a result of looking up a route for the customer address (CA) space address for a given destination.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**MSFT\_NetVirtualizationProviderAddressSettingData**](msft-netvirtualizationprovideraddresssettingdata.md)<br/> | Represents a physical IP address of the network interface the WNV module is configured on.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**MSFT\_NetVirtualizationProviderRouteSettingData**](msft-netvirtualizationproviderroutesettingdata.md)<br/>     | The MSFT\_NetVirtualizationProviderRouteSettingData represents a physical route configured on the network interface the WNV module is configured on.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
