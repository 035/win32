# Bluetooth Enumerations

The following enumerations support the Microsoft Bluetooth implementation.



| Section                                                                                  | Content                                                                |
|------------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**BLUETOOTH\_AUTHENTICATION\_REQUIREMENTS**](bluetooth-authentication-requirements.md) | Defines the 'Man in the Middle' protection methods for authentication. |
| [**BLUETOOTH\_AUTHENTICATION\_METHOD**](bluetooth-authentication-method.md)             | Defines the supported authentication types during device pairing.      |
| [**BLUETOOTH\_IO\_CAPABILITY**](bluetooth-io-capability.md)                             | Defines the input/output capabilities of a Bluetooth Device.           |
