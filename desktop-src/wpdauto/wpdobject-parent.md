# WPDObject.Parent property

The **Parent** property gets the immediate parent of this **WPDObject**.

This property is read-only.

## Syntax


```JScript
Parent = WPDObject.Parent
```



## Property value

A **WPDObject** that is the parent of this **WPDObject**.

## Remarks

When objects are stored in a flat hierarchy, the immediate parent is the same as a [**Service Object**](service-object.md) or a [**Storage Object**](storage-object.md).
