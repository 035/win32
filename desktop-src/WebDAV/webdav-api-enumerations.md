# WebDAV API Enumerations

The following enumeration is used in the WebDAV API.



| Enumeration                          | Description                                                                                                                                          |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AUTHNEXTSTEP**](authnextstep.md) | Specifies the next action that the WebDAV client should take after a successful call to the [*DavAuthCallback*](authcallback.md) callback function. |
