# WebDAV API Enumerations

The following enumeration is used in the WebDAV API.



| Enumeration                          | Description                                                                                                                                          |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AUTHNEXTSTEP**](/windows/win32/davclnt/ne-davclnt-authnextstep?branch=master) | Specifies the next action that the WebDAV client should take after a successful call to the [*DavAuthCallback*](/windows/win32/Davclnt/nc-davclnt-pfndavauthcallback?branch=master) callback function. |
