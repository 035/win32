# WebDAV API Structures

The following structures are used in the WebDAV API.



| Structure                                                   | Description                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DAV\_CALLBACK\_AUTH\_BLOB**](dav-callback-auth-blob.md) | This structure is used to store an authentication [*BLOB*](https://msdn.microsoft.com/library/windows/desktop/ms721569#-security-blob-gly) that was retrieved by the [*DavAuthCallback*](authcallback.md) callback function. |
| [**DAV\_CALLBACK\_AUTH\_UNP**](dav-callback-auth-unp.md)   | This structure is used to store user name and password information that was retrieved by the [*DavAuthCallback*](authcallback.md) callback function.                                               |
| [**DAV\_CALLBACK\_CRED**](dav-callback-cred.md)            | The [*DavAuthCallback*](authcallback.md) callback function uses this structure to store user credential information.                                                                               |
