# WebDAV API Structures

The following structures are used in the WebDAV API.



| Structure                                                   | Description                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DAV\_CALLBACK\_AUTH\_BLOB**](/windows/win32/Davclnt/ns-davclnt-_dav_callback_auth_blob?branch=master) | This structure is used to store an authentication [*BLOB*](https://msdn.microsoft.com/library/windows/desktop/ms721569#-security-blob-gly) that was retrieved by the [*DavAuthCallback*](/windows/win32/Davclnt/nc-davclnt-pfndavauthcallback?branch=master) callback function. |
| [**DAV\_CALLBACK\_AUTH\_UNP**](/windows/win32/Davclnt/ns-davclnt-_dav_callback_auth_unp?branch=master)   | This structure is used to store user name and password information that was retrieved by the [*DavAuthCallback*](/windows/win32/Davclnt/nc-davclnt-pfndavauthcallback?branch=master) callback function.                                               |
| [**DAV\_CALLBACK\_CRED**](/windows/win32/Davclnt/ns-davclnt-_dav_callback_cred?branch=master)            | The [*DavAuthCallback*](/windows/win32/Davclnt/nc-davclnt-pfndavauthcallback?branch=master) callback function uses this structure to store user credential information.                                                                               |
