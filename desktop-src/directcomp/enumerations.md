# Enumerations

This section describes the enumerations provided by the Microsoft DirectComposition API.

## In this section



| Topic                                                                                                  | Description                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DCOMPOSITION\_BACKFACE\_VISIBILITY**](/windows/win32/DcompTypes/ne-dcomptypes-dcomposition_backface_visibility?branch=master)<br/>              | Specifies the backface visibility to be applied to a visual. <br/>                                                                                                                            |
| [**DCOMPOSITION\_BITMAP\_INTERPOLATION\_MODE**](/windows/win32/DcompTypes/ne-dcomptypes-dcomposition_bitmap_interpolation_mode?branch=master)<br/> | Specifies the interpolation mode to be used when a bitmap is composed with any transform where the pixels in the bitmap don't line up exactly one-to-one with pixels on screen. <br/>         |
| [**DCOMPOSITION\_BORDER\_MODE**](/windows/win32/DcompTypes/ne-dcomptypes-dcomposition_border_mode?branch=master)<br/>                              | Specifies the border mode to use when composing a bitmap or applying a clip with any transform such that the edges of the bitmap or clip are not axis-aligned with integer coordinates. <br/> |
| [**DCOMPOSITION\_COMPOSITE\_MODE**](/windows/win32/DcompTypes/ne-dcomptypes-dcomposition_composite_mode?branch=master)<br/>                        | The mode to use to blend the bitmap content of a visual with the render target.<br/>                                                                                                          |
| [**DCOMPOSITION\_OPACITY\_MODE**](/windows/win32/dcomptypes/ne-dcomptypes-dcomposition_opacity_mode?branch=master)<br/>                            | Specifies how the effective opacity value of a visual is applied to that visual s content and children.<br/>                                                                                  |
| [**DCOMPOSITION\_TRANSFORM\_MODE**](/windows/win32/dcomptypes/?branch=master)<br/>                        | Specifies the transform style to be applied to a visual. <br/>                                                                                                                                |
