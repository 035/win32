# Macros

This section contains information about macros provided by the Microsoft XAudio2 API.



| Macro                          | Description                                                                                                                                               |
|--------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XAPOAlloc**](xapoalloc.md) | Memory allocation macro used by IXAPO methods that must allocate arbitrary sized structures that are subsequently returned to the application.<br/> |
| [**XAPOFree**](xapofree.md)   | Macro used to free memory allocated with the XAPOAlloc macro.<br/>                                                                                  |
