# Structures

This section contains information about structures provided by the Microsoft XAudio2 API.



| Structure                                                                                 | Description                                                                                                                                    |
|-------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XAUDIO2\_BUFFER**](xaudio2-buffer.md)                                                 | Represents an audio data buffer.<br/>                                                                                                    |
| [**XAUDIO2\_BUFFER\_WMA**](xaudio2-buffer-wma.md)                                        | Represents a WMA audio data buffer.<br/>                                                                                                 |
| [**XAUDIO2\_DEBUG\_CONFIGURATION**](xaudio2-debug-configuration.md)                      | Sets a new global debug configuration for XAudio2 when used by the SetDebugConfiguration function.                                             |
| [**XAUDIO2\_EFFECT\_CHAIN**](xaudio2-effect-chain.md)                                    | Defines an effect chain.<br/>                                                                                                            |
| [**XAUDIO2\_EFFECT\_DESCRIPTOR**](xaudio2-effect-descriptor.md)                          | Defines an effect.<br/>                                                                                                                  |
| [**XAUDIO2\_FILTER\_PARAMETERS**](xaudio2-filter-parameters.md)                          | Defines filter parameters for a source voice.<br/>                                                                                       |
| [**XAUDIO2\_PERFORMANCE\_DATA**](xaudio2-performance-data.md)                            | Retrieves performance information.<br/>                                                                                                  |
| [**XAUDIO2\_SEND\_DESCRIPTOR**](xaudio2-send-descriptor.md)                              | Describes a voice send destination.<br/>                                                                                                 |
| [**XAUDIO2\_VOICE\_DETAILS**](xaudio2-voice-details.md)                                  | Contains information about the creation flags, input channels, and sample rate of a voice.<br/>                                          |
| [**XAUDIO2\_VOICE\_SENDS**](xaudio2-voice-sends.md)                                      | Defines a set of voices to receive data from a single output voice.<br/>                                                                 |
| [**XAUDIO2\_VOICE\_STATE**](xaudio2-voice-state.md)                                      | Returns the voice's current state and cursor position data.<br/>                                                                         |
| [**XAUDIO2FX\_REVERB\_I3DL2\_PARAMETERS**](xaudio2fx-reverb-i3dl2-parameters.md)         | Describes I3DL2 (Interactive 3D Audio Rendering Guidelines Level 2.0) parameters for use in the ReverbConvertI3DL2ToNative function.           |
| [**XAUDIO2FX\_REVERB\_PARAMETERS**](xaudio2fx-reverb-parameters.md)                      | Describes parameters for use in the reverb APO.                                                                                                |
| [**XAUDIO2FX\_VOLUMEMETER\_LEVELS**](xaudio2fx-volumemeter-levels.md)                    | Describes parameters for use with the volume meter APO.                                                                                        |
| [**XAPO\_LOCKFORPROCESS\_BUFFER\_PARAMETERS**](xapo-lockforprocess-buffer-parameters.md) | Defines buffer parameters that remain constant while an XAPO is locked.<br/>                                                             |
| [**XAPO\_PROCESS\_BUFFER\_PARAMETERS**](xapo-process-buffer-parameters.md)               | Defines buffer parameters that may change from one call to the next.<br/>                                                                |
| [**XAPO\_REGISTRATION\_PROPERTIES**](xapo-registration-properties.md)                    | Describes general characteristics of an XAPO.<br/>                                                                                       |
| [**FXECHO\_INITDATA**](fxecho-initdata.md)                                               | Initialization parameters for use with the FXECHO XAPO.<br/>                                                                             |
| [**FXECHO\_PARAMETERS**](fxecho-parameters.md)                                           | Parameters for use with the FXECHO XAPO.<br/>                                                                                            |
| [**FXEQ\_PARAMETERS**](fxeq-parameters.md)                                               | Parameters for use with the FXEQ XAPO.<br/>                                                                                              |
| [**FXMASTERINGLIMITER\_PARAMETERS**](fxmasteringlimiter-parameters.md)                   | Parameters for use with the FXMasteringLimiter XAPO.<br/>                                                                                |
| [**FXREVERB\_PARAMETERS**](fxreverb-parameters.md)                                       | Parameters for use with the FXReverb XAPO.<br/>                                                                                          |
| [**X3DAUDIO\_CONE**](x3daudio-cone.md)                                                   | Specifies directionality for a single-channel non-LFE emitter by scaling DSP behavior with respect to the emitter's orientation.<br/>    |
| [**X3DAUDIO\_DISTANCE\_CURVE**](x3daudio-distance-curve.md)                              | Defines an explicit piecewise curve made up of linear segments, directly defining DSP behavior with respect to normalized distance.<br/> |
| [**X3DAUDIO\_DISTANCE\_CURVE\_POINT**](x3daudio-distance-curve-point.md)                 | Defines a DSP setting at a given normalized distance.<br/>                                                                               |
| [**X3DAUDIO\_DSP\_SETTINGS**](x3daudio-dsp-settings.md)                                  | Receives the results from a call to X3DAudioCalculate.<br/>                                                                              |
| [**X3DAUDIO\_EMITTER**](x3daudio-emitter.md)                                             | Defines a single or multi-point 3D audio source used with an arbitrary number of sound channels.<br/>                                    |
| [**X3DAUDIO\_LISTENER**](x3daudio-listener.md)                                           | Defines a point of 3D audio reception.<br/>                                                                                              |
