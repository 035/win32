# Dictionary

[**IPropertyStorage::WritePropertyNames**](ipropertystorage-writepropertynames.md) is implemented using the [Property ID 0 dictionary](https://msdn.microsoft.com/library/windows/desktop/aa380307). Property ID 0 is not accessible through [**IPropertyStorage::ReadMultiple**](ipropertystorage-readmultiple.md) or [**IPropertyStorage::WriteMultiple**](ipropertystorage-writemultiple.md).
