# Dictionary

[**IPropertyStorage::WritePropertyNames**](/windows/win32/Propidl/nf-propidl-ipropertystorage-writepropertynames?branch=master) is implemented using the [Property ID 0 dictionary](https://msdn.microsoft.com/library/windows/desktop/aa380307). Property ID 0 is not accessible through [**IPropertyStorage::ReadMultiple**](/windows/win32/Propidl/nf-propidl-ipropertystorage-readmultiple?branch=master) or [**IPropertyStorage::WriteMultiple**](/windows/win32/Propidl/nf-propidl-ipropertystorage-writemultiple?branch=master).
