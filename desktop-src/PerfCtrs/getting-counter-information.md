# Getting Counter Information

To retrieve information about a counter, such as the counter type, help text, and path, call the [**PdhGetCounterInfo**](pdhgetcounterinfo.md) function. The function returns the counter information as a [**PDH\_COUNTER\_INFO**](pdh-counter-info-str.md) structure.
