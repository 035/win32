# Getting Counter Information

To retrieve information about a counter, such as the counter type, help text, and path, call the [**PdhGetCounterInfo**](/windows/win32/Pdh/nf-pdh-pdhgetcounterinfoa?branch=master) function. The function returns the counter information as a [**PDH\_COUNTER\_INFO**](/windows/win32/Pdh/ns-pdh-_pdh_counter_info_a?branch=master) structure.
