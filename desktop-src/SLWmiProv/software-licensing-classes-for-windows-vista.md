# Software Licensing Classes for Windows Vista and Windows Server 2008

Not supported. Use the [Software Licensing Classes](https://msdn.microsoft.com/library/dn622972).

**Windows Vista and Windows Server 2008:** Software Licensing classes retrieve software licensing data for Windows Vista and Windows Server 2008. The classes are defined in SLWmi.mof, implemented in SLWmi.dll, and are located in the "root\\cimv2" namespace.



| Class                                                                         | Description                                                                                          |
|-------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**SoftwareLicensingProduct**](softwarelicensingproduct-vista.md)<br/> | Exposes the product-specific properties and methods of the Software Licensing service.<br/>    |
| [**SoftwareLicensingService**](softwarelicensingservice-vista.md)<br/> | Exposes the product-independent properties and methods of the Software Licensing service.<br/> |
