# UninstallProductKey method of the SoftwareLicensingProduct class

Not supported. Use the [**UninstallProductKey**](https://msdn.microsoft.com/library/cc534599) method.

**Windows Vista and Windows Server 2008:** Uninstalls the product key.

## Syntax


```mof
uint32 UninstallProductKey();
```



## Parameters

This method has no parameters.
