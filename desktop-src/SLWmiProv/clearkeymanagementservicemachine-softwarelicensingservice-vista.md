# ClearKeyManagementServiceMachine method of the SoftwareLicensingService class

Not supported. Use the [**ClearKeyManagementServiceMachine**](https://msdn.microsoft.com/library/cc534585) method.

**Windows Vista and Windows Server 2008:** Clears the key management service machine name.

## Syntax


```mof
uint32 ClearKeyManagementServiceMachine();
```



## Parameters

This method has no parameters.
