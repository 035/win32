# ClearProductKeyFromRegistry method of the SoftwareLicensingService class

Not supported. Use the [**ClearProductKeyFromRegistry**](https://msdn.microsoft.com/library/cc534586) method.

**Windows Vista and Windows Server 2008:** Clears the product key from the registry.

## Syntax


```mof
uint32 ClearProductKeyFromRegistry();
```



## Parameters

This method has no parameters.
