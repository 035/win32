# RefreshLicenseStatus method of the SoftwareLicensingService class

Not supported. Use the [**RefreshLicenseStatus**](https://msdn.microsoft.com/library/cc534592) method.

**Windows Vista and Windows Server 2008:** Updates the licensing status of the machine so that applications have access to current licensing information.

## Syntax


```mof
uint32 RefreshLicenseStatus();
```



## Parameters

This method has no parameters.
