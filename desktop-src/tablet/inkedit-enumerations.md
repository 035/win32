# InkEdit Enumerations

This section contains Enumerations belonging to the InkEdit Control.



| Enumeration                                            | Description                                                                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AppearanceConstants**](appearanceconstants.md)     | Specifies whether the control appears flat or 3-D.<br/>                                                                                            |
| [**BorderStyleConstants**](borderstyleconstants.md)   | Specifies whether the control has a border.<br/>                                                                                                   |
| [**InkApplicationGesture**](inkapplicationgesture.md) | Specifies the interest in a set of application-specific gestures.<br/>                                                                             |
| [**InkDisplayMode**](inkdisplaymode.md)               | Specifies whether a selection appears as ink or text.<br/>                                                                                         |
| [**InkEditStatus**](inkeditstatus.md)                 | Specifies whether the [InkEdit](inkedit-control-reference.md) control is idle, collecting ink, or recognizing ink.<br/>                           |
| [**InkInsertMode**](inkinsertmode.md)                 | Specifies how ink is inserted onto the [InkEdit](inkedit-control-reference.md) control.<br/>                                                      |
| [**InkMode**](inkmode.md)                             | Specifies the collection mode settings for drawn ink-whether ink collection is disabled, ink is collected, or ink and gestures are collected.<br/> |
| [**InkMouseButton**](inkmousebutton.md)               | Specifies which mouse button was pressed.<br/>                                                                                                     |
| [**InkMousePointer**](inkmousepointer.md)             | Specifies the type of mouse pointer that appears.<br/>                                                                                             |
| [**MouseButton**](mousebutton.md)                     | Specifies which mouse button was pressed.<br/>                                                                                                     |
| [**ScrollBarsConstants**](scrollbarsconstants.md)     | Specifies how the scroll bars of an [InkEdit](inkedit-control-reference.md) control appear on the screen.<br/>                                    |
| [**SelAlignmentConstants**](selalignmentconstants.md) | Specifies the alignment of the paragraph relative to the margins of the [InkEdit](inkedit-control-reference.md) control.<br/>                     |
