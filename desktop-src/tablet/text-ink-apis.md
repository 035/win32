# Text Ink Reference

In addition to supporting the standard OLE interfaces, the text ink object (tInk) also supports the [**IInkLineInfo**](iinklineinfo.md).

For information about specifying display properties for a text ink object, see [**INKMETRIC**](inkmetric.md) structure.
