# Core Tablet PC Enumerations

This section contains the core enumerations for Tablet PC.

## In This Section



| Enumeration                                                                                  | Description                                                                                                                                                                                                                                                                               |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InkApplicationGesture**](/windows/win32/msinkaut/ne-msinkaut-inkapplicationgesture?branch=master)                                       | Specifies the interest in a set of application-specific gesture.<br/>                                                                                                                                                                                                               |
| [**InkBoundingBoxMode Enumeration**](/windows/win32/msinkaut/ne-msinkaut-inkboundingboxmode?branch=master)                                 | Specifies which characteristics of a stroke, such as drawing attributes, are used to calculate the bounding box of the ink.<br/>                                                                                                                                                    |
| [**InkClipboardFormats**](/windows/win32/msinkaut/ne-msinkaut-inkclipboardformats?branch=master)                                           | Specifies the format of ink that is stored on the Clipboard.<br/>                                                                                                                                                                                                                   |
| [**InkClipboardModes**](/windows/win32/msinkaut/ne-msinkaut-inkclipboardmodes?branch=master)                                               | Specifies the copy options of the Clipboard.<br/>                                                                                                                                                                                                                                   |
| [**InkCollectionMode**](/windows/win32/msinkaut/ne-msinkaut-inkcollectionmode?branch=master)                                               | Determines whether ink, gestures, or ink and gestures are recognized as the user writes.<br/>                                                                                                                                                                                       |
| [**InkCollectorEventInterest**](/windows/win32/msinkaut/ne-msinkaut-inkcollectoreventinterest?branch=master)                               | Specifies whether an event occurred on an ink collector and, if so, which event fired.<br/>                                                                                                                                                                                         |
| [**InkCursorButtonState**](/windows/win32/msinkaut/ne-msinkaut-inkcursorbuttonstate?branch=master)                                         | Specifies the state of a cursor button.<br/>                                                                                                                                                                                                                                        |
| [**InkDivisionType**](inkdivisiontype.md)                                                   | Defines values for the structural types within the [**IInkDivisionResult**](/windows/win32/msinkaut15/nn-msinkaut15-iinkdivisionresult?branch=master) object.<br/>                                                                                                                                                                    |
| [**InkExtractFlags**](/windows/win32/msinkaut/ne-msinkaut-inkextractflags?branch=master)                                                   | Specifies what part of a stroke to remove from an [**InkDisp**](/windows/win32/msinkaut/?branch=master) object.<br/>                                                                                                                                                                                     |
| [**InkMouseButton**](/windows/win32/msinkaut/ne-msinkaut-inkmousebutton?branch=master)                                                     | Specifies which mouse button was pressed.<br/>                                                                                                                                                                                                                                      |
| [**InkMousePointer**](/windows/win32/msinkaut/ne-msinkaut-inkmousepointer?branch=master)                                                   | Specifies the type of mouse pointer to appear.<br/>                                                                                                                                                                                                                                 |
| [**InkOverlayAttachMode**](/windows/win32/msinkaut/ne-msinkaut-inkoverlayattachmode?branch=master)                                         | Specifies where to attach the new [**InkOverlay**](/windows/win32/msinkaut/?branch=master) object, behind or in front of the active layer.<br/>                                                                                                                                                       |
| [**InkOverlayEditingMode**](/windows/win32/msinkaut/ne-msinkaut-inkoverlayeditingmode?branch=master)                                       | Specifies the behavior mode of the [**InkOverlay**](/windows/win32/msinkaut/?branch=master) object and the [InkPicture](inkpicture-control-reference.md) control.<br/>                                                                                                                               |
| [**InkOverlayEraserMode**](/windows/win32/msinkaut/ne-msinkaut-inkoverlayerasermode?branch=master)                                         | Specifies the way in which ink is erased from the [**InkOverlay**](/windows/win32/msinkaut/?branch=master) object and the [InkPicture](inkpicture-control-reference.md) control.<br/> This mode is used when the [**InkOverlayEditingMode**](/windows/win32/msinkaut/ne-msinkaut-inkoverlayeditingmode?branch=master) is set to Delete.<br/> |
| [**InkPenTip**](/windows/win32/msinkaut/ne-msinkaut-inkpentip?branch=master)                                                               | Specifies whether the pen tip is round or rectangular.<br/>                                                                                                                                                                                                                         |
| [**InkPersistenceCompressionMode**](/windows/win32/msinkaut/ne-msinkaut-inkpersistencecompressionmode?branch=master)                       | Specifies the compression modes that are used to save the [**InkDisp**](/windows/win32/msinkaut/?branch=master) object to a serialized format.<br/>                                                                                                                                                      |
| [**InkPersistenceFormat**](/windows/win32/msinkaut/ne-msinkaut-inkpersistenceformat?branch=master)                                         | Specifies how ink is persisted.<br/>                                                                                                                                                                                                                                                |
| [**InkRasterOperation**](/windows/win32/msinkaut/ne-msinkaut-inkrasteroperation?branch=master)                                             | Defines values for performing raster operations on drawn ink.<br/>                                                                                                                                                                                                                  |
| [**InkRecognitionAlternatesSelection Enumeration**](/windows/win32/msinkaut/ne-msinkaut-inkrecognitionalternatesselection?branch=master)   | Not implemented.<br/>                                                                                                                                                                                                                                                               |
| [**InkRecognitionConfidence**](/windows/win32/msinkaut/ne-msinkaut-inkrecognitionconfidence?branch=master)                                 | Indicates the level of confidence that the recognizer has in the recognition result.<br/>                                                                                                                                                                                           |
| [**InkRecognitionModes**](/windows/win32/msinkaut/ne-msinkaut-inkrecognitionmodes?branch=master)                                           | Specifies how the recognzier interprets the ink and determines the result string.<br/>                                                                                                                                                                                              |
| [**InkRecognitionStatus**](/windows/win32/msinkaut/ne-msinkaut-inkrecognitionstatus?branch=master)                                         | Specifies whether an error occurred during recognition and, if so, which error occurred.<br/>                                                                                                                                                                                       |
| [**InkRecognizerCapabilities**](/windows/win32/msinkaut/ne-msinkaut-inkrecognizercapabilities?branch=master)                               | Specifies the attributes of a recognizer. You also use this enumeration to determine which attributes to use when you search for an installed recognizer.<br/>                                                                                                                      |
| [**InkRecognizerCharacterAutoCompletionMode**](/windows/win32/msinkaut/ne-msinkaut-inkrecognizercharacterautocompletionmode?branch=master) | Specifies types of character input modes.<br/>                                                                                                                                                                                                                                      |
| [**InkShiftKeyModifierFlags**](/windows/win32/msinkaut/ne-msinkaut-inkshiftkeymodifierflags?branch=master)                                 | Specifies which modifier key was pressed.<br/>                                                                                                                                                                                                                                      |
| [**InkSystemGesture**](/windows/win32/msinkaut/ne-msinkaut-inksystemgesture?branch=master)                                                 | Specifies the interest in a set of operating system-specific gestures.<br/>                                                                                                                                                                                                         |
| [**ItemSelectionConstants**](/windows/win32/msinkaut/ne-msinkaut-itemselectionconstants?branch=master)                                     | Specifies whether the first element or all elements within a group of points or packet values are used.<br/>                                                                                                                                                                        |
| [**PanelType**](paneltype.md)                                                               | Defines the type of input currently available in the [**PenInputPanel**](/windows/win32/msinkaut/?branch=master) object.<br/>                                                                                                                                                                      |
| [**SelectionHitResult**](/windows/win32/msinkaut/ne-msinkaut-selectionhitresult?branch=master)                                             | Specifies which part of a selection, if any, was hit during a hit test.<br/>                                                                                                                                                                                                        |
| [**TabletHardwareCapabilities**](/windows/win32/msinkaut/ne-msinkaut-tablethardwarecapabilities?branch=master)                             | Specifies the hardware capabilities of the Tablet PC.<br/>                                                                                                                                                                                                                          |
| [**TabletPropertyMetricUnit**](/windows/win32/msinkaut/ne-msinkaut-tabletpropertymetricunit?branch=master)                                 | Indicates the unit of measurement of a property.<br/>                                                                                                                                                                                                                               |
