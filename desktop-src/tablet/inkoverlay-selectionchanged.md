# InkOverlay.SelectionChanged event

Occurs when the selection of ink within the control has changed, such as through alterations to the user interface, cut-and-paste procedures, or the [**Selection**](inkoverlay-selection.md) property.

## Syntax


```C++
void SelectionChanged();
```



## Parameters

This event has no parameters.

## Return value

This event does not return a value.

## Remarks

There are no event data.

This event method is defined in the \_IInkOverlayEvents and \_IInkPictureEvents dispatch-only interfaces (dispinterfaces) with an ID of DISPID\_IOESelectionChanged.
