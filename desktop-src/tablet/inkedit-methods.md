# InkEdit Methods

This section contains Methods belonging to the InkEdit Control.



| Method                                               | Description                                                                                                           |
|------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**GetGestureStatus**](/windows/win32/inked/?branch=master) | Retrieves the interest of the [InkEdit](inkedit-control-reference.md) control in a known set of gestures.<br/> |
| [**Recognize**](/windows/win32/inked/?branch=master)               | Specifies that recognition should occur.<br/>                                                                   |
| [**Refresh**](/windows/win32/inked/?branch=master)                   | Causes the control to redraw.<br/>                                                                              |
| [**SetGestureStatus**](/windows/win32/inked/?branch=master) | Modifies the interest of the [InkEdit](inkedit-control-reference.md) control in a known set of gestures.<br/>  |
