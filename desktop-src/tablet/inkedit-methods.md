# InkEdit Methods

This section contains Methods belonging to the InkEdit Control.



| Method                                               | Description                                                                                                           |
|------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**GetGestureStatus**](inkedit-getgesturestatus.md) | Retrieves the interest of the [InkEdit](inkedit-control-reference.md) control in a known set of gestures.<br/> |
| [**Recognize**](inkedit-recognize.md)               | Specifies that recognition should occur.<br/>                                                                   |
| [**Refresh**](inkedit-refresh.md)                   | Causes the control to redraw.<br/>                                                                              |
| [**SetGestureStatus**](inkedit-setgesturestatus.md) | Modifies the interest of the [InkEdit](inkedit-control-reference.md) control in a known set of gestures.<br/>  |
