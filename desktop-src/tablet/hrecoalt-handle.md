# HRECOALT Handle

The **HRECOALT** handle is obsolete.


```C++
typedef HANDLE HRECOALT;
```



## Remarks

These functions are obsolete and need not be implemented by custom application recognizers.

-   [**DestroyAlternate**](/windows/win32/recapis/nf-recapis-destroyalternate?branch=master)
-   [**GetConfidenceLevel**](/windows/win32/recapis/?branch=master)
-   [**GetGuideIndex**](/windows/win32/recapis/?branch=master)
-   [**GetMetrics**](/windows/win32/recapis/?branch=master)
-   [**GetPropertyRanges**](/windows/win32/recapis/?branch=master)
-   [**GetRangePropertyValue**](/windows/win32/recapis/?branch=master)
-   [**GetSegmentAlternateList**](/windows/win32/recapis/?branch=master)
-   [**GetString**](/windows/win32/recapis/?branch=master)
-   [**GetStrokeRanges**](/windows/win32/recapis/?branch=master)
