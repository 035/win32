# HRECOALT Handle

The **HRECOALT** handle is obsolete.


```C++
typedef HANDLE HRECOALT;
```



## Remarks

These functions are obsolete and need not be implemented by custom application recognizers.

-   [**DestroyAlternate**](destroyalternate.md)
-   [**GetConfidenceLevel**](getconfidencelevel-function.md)
-   [**GetGuideIndex**](getguideindex-function.md)
-   [**GetMetrics**](getmetrics-function.md)
-   [**GetPropertyRanges**](getpropertyranges-function.md)
-   [**GetRangePropertyValue**](getrangepropertyvalue-function.md)
-   [**GetSegmentAlternateList**](getsegmentalternatelist-function.md)
-   [**GetString**](getstring-function.md)
-   [**GetStrokeRanges**](getstrokeranges-function.md)
