# Flicks Enumerations

This section contains the enumerations used in the [**WM\_TABLET\_FLICK Message**](wm-tablet-flick-message.md).



| Enumeration                                                 | Description                                                                                                                                                   |
|-------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FLICKDIRECTION**](flickdirection.md)                    | Represents the direction of the flick.<br/>                                                                                                             |
| [**FLICKMODE**](flickmode.md)                              | Specifies where flick gestures are visible.<br/>                                                                                                        |
| [**FLICKACTION\_COMMANDCODE**](flickaction-commandmode.md) | Specifies the command code filed in the [**FLICK\_DATA Structure**](flick-data.md) and represents what kind of flick action is to be carried out.<br/> |
| [**SCROLLDIRECTION**](scrolldirection.md)                  | Specifies what direction the scroll box should move.<br/>                                                                                               |
| [**KEYMODIFIER**](keymodifier.md)                          | Specifies which, if any, modifier keys were pressed when the flick gesture occurred.<br/>                                                               |
