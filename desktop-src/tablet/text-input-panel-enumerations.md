# Text Input Panel Enumerations

This section contains the enumerations used by the [**ITextInputPanel Interface**](itextinputpanel.md).

## In This Section



| Enumeration                                                  | Description                                                                                                                               |
|--------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**CorrectionMode Enumeration**](correctionmode.md)         | Defines values that represent the correction modes of the Tablet PC Input Panel.<br/>                                               |
| [**CorrectionPosition Enumeration**](correctionposition.md) | Defines values that represent the post insertion correction positions of the Input Panel.<br/>                                      |
| [**EventMask**](eventmask.md)                               | Specifies the events that you can set attention for.                                                                                      |
| [**InPlaceDirection Enumeration**](inplacedirection.md)     | Defines values that represent the positions where the Input Panel can be placed, relative to the current text insertion point.<br/> |
| [**InPlaceState Enumeration**](inplacestate.md)             | Defines values that represent how the Input Panel is displayed onscreen.<br/>                                                       |
| [**InteractionMode Enumeration**](interactionmode.md)       | Defines values that dictate the positioning of the Input Panel.<br/>                                                                |
| [**PanelInputArea Enumeration**](panelinputarea.md)         | Defines values that represent the input areas of the Input Panel.<br/>                                                              |
