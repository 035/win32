# Text Input Panel Enumerations

This section contains the enumerations used by the [**ITextInputPanel Interface**](/windows/win32/peninputpanel/nn-peninputpanel-itextinputpanel?branch=master).

## In This Section



| Enumeration                                                  | Description                                                                                                                               |
|--------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**CorrectionMode Enumeration**](/windows/win32/peninputpanel/ne-peninputpanel-__midl___midl_itf_peninputpanel_0000_0000_0004?branch=master)         | Defines values that represent the correction modes of the Tablet PC Input Panel.<br/>                                               |
| [**CorrectionPosition Enumeration**](/windows/win32/peninputpanel/ne-peninputpanel-__midl___midl_itf_peninputpanel_0000_0000_0005?branch=master) | Defines values that represent the post insertion correction positions of the Input Panel.<br/>                                      |
| [**EventMask**](/windows/win32/peninputpanel/ne-peninputpanel-__midl___midl_itf_peninputpanel_0000_0000_0007?branch=master)                               | Specifies the events that you can set attention for.                                                                                      |
| [**InPlaceDirection Enumeration**](/windows/win32/peninputpanel/ne-peninputpanel-__midl___midl_itf_peninputpanel_0000_0000_0006?branch=master)     | Defines values that represent the positions where the Input Panel can be placed, relative to the current text insertion point.<br/> |
| [**InPlaceState Enumeration**](/windows/win32/peninputpanel/ne-peninputpanel-__midl___midl_itf_peninputpanel_0000_0000_0002?branch=master)             | Defines values that represent how the Input Panel is displayed onscreen.<br/>                                                       |
| [**InteractionMode Enumeration**](/windows/win32/peninputpanel/ne-peninputpanel-__midl___midl_itf_peninputpanel_0000_0000_0001?branch=master)       | Defines values that dictate the positioning of the Input Panel.<br/>                                                                |
| [**PanelInputArea Enumeration**](/windows/win32/peninputpanel/ne-peninputpanel-__midl___midl_itf_peninputpanel_0000_0000_0003?branch=master)         | Defines values that represent the input areas of the Input Panel.<br/>                                                              |
