# InkWordType Complex Type

Defines the type that contains the ink data, alternates, and confidence for ink content that is an [InkWord element](inkword-element.md) (as opposed to a [Drawing element](drawing-element.md)).

``` syntax
<xs:complexType name="InkWordType" />
```

## Requirements



|                                     |                                                               |
|-------------------------------------|---------------------------------------------------------------|
| Minimum supported client<br/> | Windows XP Tablet PC Edition \[desktop apps only\]<br/> |
| Minimum supported server<br/> | None supported<br/>                                     |
