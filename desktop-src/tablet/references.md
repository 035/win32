# References

For information about enabling speech recognition in your application, refer to the Microsoft Speech SDK, which can be found at [Microsoft Speech SDK](http://go.microsoft.com/fwlink/p/?linkid=20223). This site also contains an overview of the Microsoft Speech API and application design guidelines.

For information about Microsoft Accessibility, see [Microsoft Accessibility](http://go.microsoft.com/fwlink/p/?linkid=9073).

For more information on Microsoft Active Accessibility guidelines for Windows-based applications, see the [Accessibility](nodepage.accessibility) section.
