# InkPicture.MouseEnter event

Occurs when the mouse pointer enters the [InkPicture](inkpicture-control-reference.md) control.

## Syntax


```C++
void MouseEnter();
```



## Parameters

This event has no parameters.

## Return value

This event does not return a value.

## Remarks

This event method is defined in the **\_IInkPictureEvents** interface. The **\_IInkPictureEvents** interface implements [**IDispatch**](ebbff4bc-36b2-4861-9efa-ffa45e013eb5) interface with an identifier of DISPID\_IPEMouseEnter.
