# Recognizer Structures

This section describes the recognizer structures.

## In This Section



| Structure                                                    | Description                                                                                                                                                   |
|--------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CHARACTER\_RANGE**](character-range.md)                  | Specifies a range of Unicode points (characters).<br/>                                                                                                  |
| [**LATTICE\_METRICS**](lattice-metrics.md)                  | Describes the baseline and the midline height.<br/>                                                                                                     |
| [**LINE\_SEGMENT**](line-segment.md)                        | Describes the start and end points of a line recognition segment, such as the baseline or midline.<br/>                                                 |
| [**PACKET\_DESCRIPTION**](packet-description.md)            | Describes the content of the packet for a particular tablet context.<br/>                                                                               |
| [**PACKET\_PROPERTY**](packet-property.md)                  | Describes a packet property that is reported by the tablet driver.<br/>                                                                                 |
| [**PROPERTY\_METRICS**](property-metrics.md)                | Defines the range and resolution of a packet property.<br/>                                                                                             |
| [**RECO\_ATTRS**](reco-attrs.md)                            | Retrieves the attributes of a recognizer or specifies which attributes to use when you search for an installed recognizer.<br/>                         |
| [**RECO\_GUIDE**](reco-guide.md)                            | Defines the boundaries of the ink to the recognizer.<br/>                                                                                               |
| [**RECO\_LATTICE**](reco-lattice.md)                        | Serves as the entry point into a lattice.<br/>                                                                                                          |
| [**RECO\_LATTICE\_COLUMN**](reco-lattice-column.md)         | Represents a column in the lattice.<br/>                                                                                                                |
| [**RECO\_LATTICE\_ELEMENT**](reco-lattice-element.md)       | Corresponds to one word or one East Asian character, typically; however, an element may also correspond to a gesture, a shape, or some other code.<br/> |
| [**RECO\_LATTICE\_PROPERTIES**](reco-lattice-properties.md) | Contains an array of pointers to property structures.<br/>                                                                                              |
| [**RECO\_LATTICE\_PROPERTY**](reco-lattice-property.md)     | Contains a property used in the lattice.<br/>                                                                                                           |
| [**RECO\_RANGE**](reco-range.md)                            | Identifies a range of characters in the result string.<br/>                                                                                             |
| [**STROKE\_RANGE**](stroke-range.md)                        | Specifies a range of strokes in the [**InkDisp**](inkdisp-class.md) object.<br/>                                                                       |
