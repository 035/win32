# TF\_\* Constants

TSF atom function calls use these constants to manage thread input processes.



| Constant/value                                                                                                                                                                                                                                                                       | Description                                                                                          |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="TF_PROCESS_ATOM"></span><span id="tf_process_atom"></span><dl> <dt>**TF\_PROCESS\_ATOM**</dt> <dt>TEXT(\_CTF\_PROCESS\_ATOM\_)</dt> </dl>                               | Used to notify atom functions that a thread input process has initiated.<br/>                  |
| <span id="TF_ENABLE_PROCESS_ATOM"></span><span id="tf_enable_process_atom"></span><dl> <dt>**TF\_ENABLE\_PROCESS\_ATOM**</dt> <dt>TEXT(\_CTF\_ENABLE\_PROCESS\_ATOM\_)</dt> </dl> | Used to notify TSF, through atom functions, that a thread input process has been enabled.<br/> |
