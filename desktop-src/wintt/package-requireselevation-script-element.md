# RequiresElevation (Script) Element

Determines whether the script requires elevated privileges to run.

``` syntax
<xs:element name="RequiresElevation" />
```

The **RequiresElevation** element is defined by the [**Script**](package-script-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
