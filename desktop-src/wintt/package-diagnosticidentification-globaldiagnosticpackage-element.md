# DiagnosticIdentification (GlobalDiagnosticPackage) Element

The identifier and version used to identify the pack.

``` syntax
<xs:element name="DiagnosticIdentification" />
```

The **DiagnosticIdentification** element is defined by the [**GlobalDiagnosticPackage**](package-globaldiagnosticpackage-complextype.md) complex type.

|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
