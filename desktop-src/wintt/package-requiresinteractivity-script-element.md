# RequiresInteractivity (Script) Element

Determines whether the script requires interaction with the user.

``` syntax
<xs:element name="RequiresInteractivity" />
```

The **RequiresInteractivity** element is defined by the [**Script**](package-script-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
