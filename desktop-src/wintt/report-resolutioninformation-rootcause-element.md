# ResolutionInformation (RootCause) Element

Contains the list of resolvers that can fix the root cause issue.

``` syntax
<xs:element name="ResolutionInformation"
    type="dcmRR:ResolutionInformation"
 />
```

The **ResolutionInformation** element is defined by the [**RootCause**](report-rootcause-complextype.md) complex type.

## Requirements



|                                     |                                                      |
|-------------------------------------|------------------------------------------------------|
| Minimum supported client<br/> | Windows Vista \[desktop apps only\]<br/>       |
| Minimum supported server<br/> | Windows Server 2008 \[desktop apps only\]<br/> |
