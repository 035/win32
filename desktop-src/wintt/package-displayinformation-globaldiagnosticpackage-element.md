# DisplayInformation (GlobalDiagnosticPackage) Element

The name and description of the pack.

``` syntax

```

The **DisplayInformation** element is defined by the [**GlobalDiagnosticPackage**](package-globaldiagnosticpackage-complextype.md) complex type.
