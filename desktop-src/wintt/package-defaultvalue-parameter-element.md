# DefaultValue (Parameter) Element

The default value to use for the parameter if a value is not specified.

``` syntax
<xs:element name="DefaultValue" />
```

The **DefaultValue** element is defined by the [**Parameter**](package-parameter-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
