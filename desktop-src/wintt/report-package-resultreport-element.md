# Package (ResultReport) Element

The section of the results report that contains the results from running the troubleshooting package.

``` syntax
<xs:element name="Package"
    type="dcmRR:Package"
 />
```

The **Package** element is defined by the [**ResultReport**](report-resultreport-complextype.md) complex type.

## Requirements



|                                     |                                                      |
|-------------------------------------|------------------------------------------------------|
| Minimum supported client<br/> | Windows Vista \[desktop apps only\]<br/>       |
| Minimum supported server<br/> | Windows Server 2008 \[desktop apps only\]<br/> |
