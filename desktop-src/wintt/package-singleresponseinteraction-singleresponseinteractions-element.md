# SingleResponseInteraction (SingleResponseInteractions) Element

Defines the single response interaction.

``` syntax
<xs:element name="SingleResponseInteraction" />
```

The **SingleResponseInteraction** element is defined by the [**SingleResponseInteractions**](package-interactions-complextype.md) complex type.
