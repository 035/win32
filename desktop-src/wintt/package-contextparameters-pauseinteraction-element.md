# ContextParameters (PauseInteraction) Element

Contains a list of substitution strings that get used based on the current context.

``` syntax
<xs:element name="ContextParameters" />
```

The **ContextParameters** element is defined by the [**PauseInteraction**](package-pauseinteraction-complextype.md) complex type.

|-------------------------------------|---------------------------------------------------------|
