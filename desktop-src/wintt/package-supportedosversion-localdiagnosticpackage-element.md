# SupportedOSVersion (LocalDiagnosticPackage) Element

Specifies the minimum operating system version on which the pack can run.

``` syntax
<xs:element name="SupportedOSVersion" />
```

The **SupportedOSVersion** element is defined by the [**LocalDiagnosticPackage**](package-localdiagnosticpackage-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
