# ID (LaunchUIInteraction) Element

A string that identifies the interaction.

``` syntax
<xs:element name="ID" />
```

The **ID** element is defined by the [**LaunchUIInteraction**](package-launchuiinteraction-complextype.md) complex type.

|-------------------------------------|---------------------------------------------------------|
