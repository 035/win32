# Script (Resolver) Element

Defines the resolver script to run.

``` syntax
<xs:element name="Script" />
```

The **Script** element is defined by the [**Resolver**](package-resolver-resolvers-element.md) element.

|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
