# LaunchUIInteraction (LaunchUIInteractions) Element

Defines the launch UI interaction.

``` syntax
<xs:element name="LaunchUIInteraction" />
```

The **LaunchUIInteraction** element is defined by the [**LaunchUIInteraction**](package-launchuiinteraction-complextype.md) complex type.

|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
