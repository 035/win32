# PrivacyLink (LocalDiagnosticPackage) Element

The URL to the privacy information that tells the user how the data collected by your troubleshooting pack will be used.

``` syntax
<xs:element name="PrivacyLink" />
```

The **PrivacyLink** element is defined by the [**LocalDiagnosticPackage**](package-localdiagnosticpackage-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
