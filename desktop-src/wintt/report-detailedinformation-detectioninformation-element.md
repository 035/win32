# DetailedInformation (DetectionInformation) Element

A section that contains the collection of custom details that the package author added to the report.

``` syntax
<xs:element name="DetailedInformation"
    type="dcmRR:DetailedInformation"
 />
```

The **DetailedInformation** element is defined by the [**DetectionInformation**](report-detectioninformation-complextype.md) complex type.

## Requirements



|                                     |                                                      |
|-------------------------------------|------------------------------------------------------|
| Minimum supported client<br/> | Windows Vista \[desktop apps only\]<br/>       |
| Minimum supported server<br/> | Windows Server 2008 \[desktop apps only\]<br/> |
