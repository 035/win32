# RootCauseInformation (Problem) Element

Contains the list of root causes that the troubleshooting pack can detect.

``` syntax
<xs:element name="RootCauseInformation"
    type="dcmRR:RootCauseInformation"
 />
```

The **RootCauseInformation** element is defined by the [**Problem**](report-problem-complextype.md) complex type.

|                                     |                                                      |
|-------------------------------------|------------------------------------------------------|
| Minimum supported client<br/> | Windows Vista \[desktop apps only\]<br/>       |
| Minimum supported server<br/> | Windows Server 2008 \[desktop apps only\]<br/> |
