# DisplayInformation (LocalRootcause) Element

The name and description of the root cause to use in the UI.

``` syntax
<xs:element name="DisplayInformation" />
```

The **DisplayInformation** element is defined by the [**LocalRootcause**](package-localrootcause-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
