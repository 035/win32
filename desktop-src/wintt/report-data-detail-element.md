# Data (Detail) Element

A property of the custom detail that provides a description of the custom detail that was added to the report.

``` syntax
<xs:element name="Data"
    type="dcmRR:Data"
 />
```

The **Data** element is defined by the [**Detail**](report-detail-complextype.md) complex type.

## Remarks

For details on the contents of the **Data** element, see the [**Detail**](report-detail-complextype.md) complex type.
