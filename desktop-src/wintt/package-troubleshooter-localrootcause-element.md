# Troubleshooter (LocalRootcause) Element

Defines the troubleshooting script to run.

``` syntax
<xs:element name="Troubleshooter" />
```

The **Troubleshooter** element is defined by the [**LocalRootcause**](package-localrootcause-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
