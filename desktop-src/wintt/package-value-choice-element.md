# Value (Choice) Element

Specifies the value to return if the choice is selected from the list.

``` syntax
<xs:element name="Value" />
```

The **Value** element is defined by the [**Choice**](package-choice-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
