# Detail (DetailedInformation) Element

Identifies a custom detail that the package author added to the report.

``` syntax
<xs:element name="Detail"
    type="dcmRR:Detail"
 />
```

The **Detail** element is defined by the [**DetailedInformation**](https://msdn.microsoft.com/library/windows/desktop/ee692417) complex type.

## Requirements



|                                     |                                                      |
|-------------------------------------|------------------------------------------------------|
| Minimum supported client<br/> | Windows Vista \[desktop apps only\]<br/>       |
| Minimum supported server<br/> | Windows Server 2008 \[desktop apps only\]<br/> |
