# DisplayInformation (Resolver) Element

The name and description of the resolver to use in the UI

``` syntax
<xs:element name="DisplayInformation" />
```

The **DisplayInformation** element is defined by the [**Resolver**](package-resolver-resolvers-element.md) element.

|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
