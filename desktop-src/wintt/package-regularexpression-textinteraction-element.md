# RegularExpression (TextInteraction) Element

A regular expression string that specifies the type of input that the user is allowed to enter.

``` syntax
<xs:element name="RegularExpression" />
```

The **RegularExpression** element is defined by the [**TextInteraction**](package-textinteraction-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
