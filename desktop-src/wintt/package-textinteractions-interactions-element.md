# TextInteractions (Interactions) Element

Defines a collection of text interactions.

``` syntax
<xs:element name="TextInteractions" />
```

The **TextInteractions** element is defined by the [**Interactions**](package-interactions-complextype.md) complex type.

|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
