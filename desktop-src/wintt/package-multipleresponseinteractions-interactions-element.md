# MultipleResponseInteractions (Interactions) Element

Defines a collection of multiple response interactions.

``` syntax
<xs:element name="MultipleResponseInteractions" />
```

The **MultipleResponseInteractions** element is defined by the [**Interactions**](package-interactions-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
