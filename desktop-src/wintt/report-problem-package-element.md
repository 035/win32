# Problem (Package) Element

The section of the report that contains all the problems that the troubleshooting package is trying to detect and fix.

``` syntax
<xs:element name="Problem"
    type="dcmRR:Problem"
 />
```

The **Problem** element is defined by the [**Package**](report-package-complextype.md) complex type.

## Requirements



|                                     |                                                      |
|-------------------------------------|------------------------------------------------------|
| Minimum supported client<br/> | Windows Vista \[desktop apps only\]<br/>       |
| Minimum supported server<br/> | Windows Server 2008 \[desktop apps only\]<br/> |
