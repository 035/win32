# RequiresConsent (Resolver) Element

Determines whether the user must provide consent to proceed with fixing the root cause issue.

``` syntax
<xs:element name="Re" />
```

The **Re** element is defined by the [**Resolver**](package-resolver-resolvers-element.md) element.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
