# Name (DisplayInformation) Element

The display name to use for the UI component.

``` syntax
<xs:element name="Name" />
```

The **Name** element is defined by the [**DisplayInformation**](package-displayinformation-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
