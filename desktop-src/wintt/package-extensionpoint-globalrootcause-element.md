# ExtensionPoint (GlobalRootcause) Element

Defines one or more extensions that the client uses to extend the functionality of the Rootcause node.

``` syntax
<xs:element name="ExtensionPoint" />
```

The **ExtensionPoint** element is defined by the [**GlobalRootcause**](package-globalrootcause-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
