# Data (Contents) Element

Identifies the file that contains the custom detail.

``` syntax
<xs:element name="Data"
    type="dcmRR:Data"
 />
```

The **Data** element is defined by the [**Contents**](report-contents-complextype.md) complex type.

## Remarks

For details on the contents of the **Data** element, see the [**Contents**](report-contents-complextype.md) complex type.
