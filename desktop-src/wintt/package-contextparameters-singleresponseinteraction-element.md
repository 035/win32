# ContextParameters (SingleResponseInteraction) Element

Contains a list of substitution strings that get used based on the current context.

``` syntax
<xs:element name="ContextParameters" />
```

The **ContextParameters** element is defined by the [**SingleResponseInteraction**](package-singleresponseinteraction-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
