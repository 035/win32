# Interactions (LocalDiagnosticPackage) Element

Defines a collection of interactions that the pack uses.

``` syntax
<xs:element name="Interactions" />
```

The **Interactions** element is defined by the [**LocalDiagnosticPackage**](package-localdiagnosticpackage-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
