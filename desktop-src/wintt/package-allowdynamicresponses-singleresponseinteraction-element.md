# AllowDynamicResponses (SingleResponseInteraction) Element

Determines whether the script can add choices at run time or whether the list is static.

``` syntax
<xs:element name="AllowDynamicResponses" />
```

The **AllowDynamicResponses** element is defined by the [**SingleResponseInteraction**](package-singleresponseinteraction-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
