# Choices (SingleResponseInteraction) Element

Defines a collection of choices to display to the user.

``` syntax
<xs:element name="Choices" />
```

The **Choices** element is defined by the [**SingleResponseInteraction**](package-singleresponseinteraction-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
