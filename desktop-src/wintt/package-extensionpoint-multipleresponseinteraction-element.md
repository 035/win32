# ExtensionPoint (MultipleResponseInteraction) Element

Defines one or more extension points that extends this interaction.

``` syntax
<xs:element name="ExtensionPoint" />
```

The **ExtensionPoint** element is defined by the [**MultipleResponseInteraction**](package-multipleresponseinteraction-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
