# DetectionInformation (RootCause) Element

A section that contains the custom details that the package author added to the report that are specific to the root cause.

``` syntax
<xs:element name="DetectionInformation"
    type="DetectionInformation"
 />
```

The **DetectionInformation** element is defined by the [**RootCause**](report-rootcause-complextype.md) complex type.

## Requirements



|                                     |                                                      |
|-------------------------------------|------------------------------------------------------|
| Minimum supported client<br/> | Windows Vista \[desktop apps only\]<br/>       |
| Minimum supported server<br/> | Windows Server 2008 \[desktop apps only\]<br/> |
