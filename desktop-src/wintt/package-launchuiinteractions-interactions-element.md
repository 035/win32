# LaunchUIInteractions (Interactions) Element

Defines a collection of launch UI interactions.

``` syntax
<xs:element name="LaunchUIInteractions" />
```

The **LaunchUIInteractions** element is defined by the [**Interactions**](package-interactions-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
