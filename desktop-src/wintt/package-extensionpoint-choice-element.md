# ExtensionPoint (Choice) Element

Defines one or more extension point that extends the single-response or multiple-response choice.

``` syntax
<xs:element name="ExtensionPoint" />
```

The **ExtensionPoint** element is defined by the [**Choice**](package-choice-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
