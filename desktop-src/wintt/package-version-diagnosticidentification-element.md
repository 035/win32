# Version (DiagnosticIdentification) Element

A string that specifies this version of the troubleshooting pack.

``` syntax
<xs:element name="Version" />
```

The **Version** element is defined by the [**DiagnosticIdentification**](package-diagnosticidentification-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
