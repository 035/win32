# ID (DiagnosticIdentification) Element

A string that identifies the troubleshooting pack.

``` syntax
<xs:element name="ID" />
```

The **ID** element is defined by the [**DiagnosticIdentification**](package-diagnosticidentification-complextype.md) complex type.

|-------------------------------------|---------------------------------------------------------|
