# PowerShellVersion (LocalDiagnosticPackage) Element

The minimum Windows PowerShell version required to run the pack.

``` syntax
<xs:element name="PowerShellVersion" />
```

The **PowerShellVersion** element is defined by the [**LocalDiagnosticPackage**](package-localdiagnosticpackage-complextype.md) complex type.

## Requirements



|                                     |                                                         |
|-------------------------------------|---------------------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/>              |
| Minimum supported server<br/> | Windows Server 2008 R2 \[desktop apps only\]<br/> |
