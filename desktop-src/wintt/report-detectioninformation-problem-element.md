# DetectionInformation (Problem) Element

A section that contains the custom details that the package author added to the report that are not specific to a root cause.

``` syntax
<xs:element name="DetectionInformation"
    type="DetectionInformation"
 />
```

The **DetectionInformation** element is defined by the [**Problem**](report-problem-complextype.md) complex type.

## Requirements



|                                     |                                                      |
|-------------------------------------|------------------------------------------------------|
| Minimum supported client<br/> | Windows Vista \[desktop apps only\]<br/>       |
