# Report Elements

The Report schema contains the following elements:

-   [**ResultReport**](report-resultreport-element.md)
-   [**ComputerInformation (Header)**](report-computerinformation-header-element.md)
-   [**Contents (Detail)**](report-contents-detail-element.md)
-   [**Data (ComputerInformation)**](https://msdn.microsoft.com/library/windows/desktop/ee692408)
-   [**Data (Detail)**](report-data-detail-element.md)
-   [**Data (Header)**](report-data-header-element.md)
-   [**Data (PackageInformation)**](report-data-packageinformation-element.md)
-   [**Data (Resolution)**](report-data-resolution-element.md)
-   [**Data (RootCause)**](report-data-rootcause-element.md)
-   [**Detail (DetailedInformation)**](https://msdn.microsoft.com/library/windows/desktop/ee692419)
-   [**DetailedInformation (DetectionInformation)**](report-detailedinformation-detectioninformation-element.md)
-   [**DetailedInformation (Resolution)**](report-detailedinformation-resolution-element.md)
-   [**DetectionInformation (Problem)**](report-detectioninformation-problem-element.md)
-   [**DetectionInformation (RootCause)**](https://msdn.microsoft.com/library/windows/desktop/ee692422)
-   [**Header (ResultReport)**](report-header-resultreport-element.md)
-   [**MetaPackageInformation (Header)**](report-metapackageinformation-header-element.md)
-   [**Package (ResultReport)**](report-package-resultreport-element.md)
-   [**PackageInformation (Header)**](report-packageinformation-header-element.md)
-   [**Problem (Package)**](report-problem-package-element.md)
-   [**Resolution (ResolutionInformation)**](report-resolution-resolutioninformation-element.md)
-   [**ResolutionInformation (RootCause)**](report-resolutioninformation-rootcause-element.md)
-   [**RootCause (RootCauseInformation)**](report-rootcause-rootcauseinformation-element.md)
-   [**RootCauseInformation (Problem)**](report-rootcauseinformation-problem-element.md)

The root node of a result report is the [**ResultReport**](report-resultreport-element.md) element.
