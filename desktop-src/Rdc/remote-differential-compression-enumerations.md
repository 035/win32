# Remote Differential Compression Enumerations

The following enumerations are defined in the Remote Differential Compression API.



| Enumeration                                                | Description                                                                                               |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------|
| [**GeneratorParametersType**](generatorparameterstype.md) | Defines the set of supported generator parameters.                                                        |
| [**RdcCreatedTables**](rdccreatedtables.md)               | Defines values that describe the state of the similarity traits table, similarity file ID table, or both. |
| [**RdcMappingAccessMode**](rdcmappingaccessmode.md)       | Defines the access mode values for RDC file mapping objects.                                              |
| [**RdcNeedType**](rdcneedtype.md)                         | Defines the set of data sources used to generate a remote copy.                                           |
| [**RDC\_ErrorCode**](rdc-errorcode.md)                    | Defines the set of RDC-specific error codes.                                                              |
