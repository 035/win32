# Remote Differential Compression Enumerations

The following enumerations are defined in the Remote Differential Compression API.



| Enumeration                                                | Description                                                                                               |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------|
| [**GeneratorParametersType**](/windows/previous-versions/MsRdc/ne-msrdc-__midl___midl_itf_msrdc_0000_0000_0002?branch=master) | Defines the set of supported generator parameters.                                                        |
| [**RdcCreatedTables**](/windows/previous-versions/MsRdc/ne-msrdc-__midl___midl_itf_msrdc_0000_0000_0009?branch=master)               | Defines values that describe the state of the similarity traits table, similarity file ID table, or both. |
| [**RdcMappingAccessMode**](/windows/previous-versions/MsRdc/ne-msrdc-__midl___midl_itf_msrdc_0000_0000_0010?branch=master)       | Defines the access mode values for RDC file mapping objects.                                              |
| [**RdcNeedType**](/windows/previous-versions/MsRdc/ne-msrdc-__midl___midl_itf_msrdc_0000_0000_0003?branch=master)                         | Defines the set of data sources used to generate a remote copy.                                           |
| [**RDC\_ErrorCode**](/windows/previous-versions/MsRdc/ne-msrdc-__midl___midl_itf_msrdc_0000_0000_0001?branch=master)                    | Defines the set of RDC-specific error codes.                                                              |
