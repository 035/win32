# Communications Functions

The following functions are used with communications resources.



| Function                                                   | Description                                                                                                                    |
|------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [**BuildCommDCB**](buildcommdcb.md)                       | Fills a specified [**DCB**](dcb-str.md) structure with values specified in a device-control string.                           |
| [**BuildCommDCBAndTimeouts**](buildcommdcbandtimeouts.md) | Translates a device-definition string into appropriate device-control block codes and places them into a device control block. |
| [**ClearCommBreak**](clearcommbreak.md)                   | Restores character transmission for a specified communications device and places the transmission line in a nonbreak state.    |
| [**ClearCommError**](clearcommerror.md)                   | Retrieves information about a communications error and reports the current status of a communications device.                  |
| [**CommConfigDialog**](commconfigdialog.md)               | Displays a driver-supplied configuration dialog box.                                                                           |
| [**EscapeCommFunction**](escapecommfunction.md)           | Directs a specified communications device to perform an extended function.                                                     |
| [**GetCommConfig**](getcommconfig.md)                     | Retrieves the current configuration of a communications device.                                                                |
| [**GetCommMask**](getcommmask.md)                         | Retrieves the value of the event mask for a specified communications device.                                                   |
| [**GetCommModemStatus**](getcommmodemstatus.md)           | Retrieves modem control-register values.                                                                                       |
| [**GetCommProperties**](getcommproperties.md)             | Retrieves information about the communications properties for a specified communications device.                               |
| [**GetCommState**](getcommstate.md)                       | Retrieves the current control settings for a specified communications device.                                                  |
| [**GetCommTimeouts**](getcommtimeouts.md)                 | Retrieves the time-out parameters for all read and write operations on a specified communications device.                      |
| [**GetDefaultCommConfig**](getdefaultcommconfig.md)       | Retrieves the default configuration for the specified communications device.                                                   |
| [**PurgeComm**](purgecomm.md)                             | Discards all characters from the output or input buffer of a specified communications resource.                                |
| [**SetCommBreak**](setcommbreak.md)                       | Suspends character transmission for a specified communications device and places the transmission line in a break state.       |
| [**SetCommConfig**](setcommconfig.md)                     | Sets the current configuration of a communications device.                                                                     |
| [**SetCommMask**](setcommmask.md)                         | Specifies a set of events to be monitored for a communications device.                                                         |
| [**SetCommState**](setcommstate.md)                       | Configures a communications device according to the specifications in a device-control block.                                  |
| [**SetCommTimeouts**](setcommtimeouts.md)                 | Sets the time-out parameters for all read and write operations on a specified communications device.                           |
| [**SetDefaultCommConfig**](setdefaultcommconfig.md)       | Sets the default configuration for a communications device.                                                                    |
| [**SetupComm**](setupcomm.md)                             | Initializes the communications parameters for a specified communications device.                                               |
| [**TransmitCommChar**](transmitcommchar.md)               | Transmits a specified character ahead of any pending data in the output buffer of the specified communications device.         |
| [**WaitCommEvent**](waitcommevent.md)                     | Waits for an event to occur for a specified communications device.                                                             |
