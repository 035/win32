# Communications Control Codes

The following control code is used with device input and output.



| Control code                                                        | Operation                                                                           |
|---------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**IOCTL\_SERIAL\_LSRMST\_INSERT**](/windows/win32/WinIoCtl/ni-winioctl-ioctl_serial_lsrmst_insert?branch=master) | Enables or disables placement of a line and modem status data into the data stream. |
