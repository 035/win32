# Device Management Functions

The following functions are used in device management.



| Function                                                             | Description                                                                           |
|----------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**DeviceIoControl**](deviceiocontrol.md)                           | Sends a control code directly to a specified device driver.                           |
| [**InstallNewDevice**](installnewdevice.md)                         | Installs a new device. The user is prompted to select the device.                     |
| [**RegisterDeviceNotification**](registerdevicenotification.md)     | Registers the device or type of device for which a window will receive notifications. |
| [**UnregisterDeviceNotification**](unregisterdevicenotification.md) | Closes the specified device notification handle.                                      |



 

The following functions are used with CD-ROM and DVD drives.



| Function                                                               | Description                                                                                         |
|------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**CdromDisableDigitalPlayback**](cdromdisabledigitalplayback.md)     | Disables digital playback for the specified CD-ROM or DVD drive.                                    |
| [**CdromEnableDigitalPlayback**](cdromenabledigitalplayback.md)       | Enables digital playback for the specified CD-ROM or DVD drive.                                     |
| [**CdromIsDigitalPlaybackEnabled**](cdromisdigitalplaybackenabled.md) | Determines whether digital playback is enabled for the specified CD-ROM or DVD drive.               |
| [**CdromKnownGoodDigitalPlayback**](cdromknowngooddigitalplayback.md) | Determines whether the specified CD-ROM or DVD drive has digital playback that is known to be good. |
| [**DvdLauncher**](dvdlauncher.md)                                     | Verifies that the media region in the DVD drive matches the DVD drive region.                       |
