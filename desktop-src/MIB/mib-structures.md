# MIB Structures

Use the following structures with MIB-specific functions to get and set MIB variables. These structures are defined in Iprtrmib.h and Netioapi.h.



| Structure                                                              | Description                                                                                                                                                                                                                                 |
|------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MIB\_ANYCASTIPADDRESS\_ROW**](/windows/desktop/api/Netioapi/ns-netioapi-_mib_anycastipaddress_row)         | Stores information about an anycast IP address.                                                                                                                                                                                             |
| [**MIB\_ANYCASTIPADDRESS\_TABLE**](/windows/desktop/api/Netioapi/ns-netioapi-_mib_anycastipaddress_table)     | Contains a table of anycast IP address entries.                                                                                                                                                                                             |
| [**MIB\_BEST\_IF**](/previous-versions/windows/desktop/api/Ipmib/ns-iprtrmib-_mib_best_if)                                   | Stores the index of the interface that has the best route to a particular destination address.                                                                                                                                              |
| [**MIB\_BOUNDARYROW**](/windows/desktop/api/Iprtrmib/ns-iprtrmib-mib_boundaryrow)                            | Contains the IPv4 group address value and mask for a multicast boundary.                                                                                                                                                                    |
| [**MIB\_ICMP**](/previous-versions/windows/desktop/api/Ipmib/ns-ipmib-_mib_icmp)                                          | Contains the Internet Control Message Protocol (ICMP) statistics for a particular computer.                                                                                                                                                 |
| [**MIB\_ICMP\_EX**](/previous-versions/windows/desktop/api/Ipmib/ns-ipmib-_mib_icmp_ex_xpsp1)                                   | Contains the extended ICMP statistics for a particular computer.                                                                                                                                                                            |
| [**MIB\_IFNUMBER**](/previous-versions/windows/desktop/api/Ifmib/ns-ifmib-_mib_ifnumber)                                  | Stores the number of interfaces on a particular computer.                                                                                                                                                                                   |
| [**MIB\_IFROW**](/previous-versions/windows/desktop/api/Ifmib/ns-ifmib-_mib_ifrow)                                        | Stores information about a particular interface.                                                                                                                                                                                            |
| [**MIB\_IFSTACK\_ROW**](/windows/desktop/api/Netioapi/ns-netioapi-_mib_ifstack_row)                           | Represents the relationship between two network interfaces.                                                                                                                                                                                 |
| [**MIB\_IFSTACK\_TABLE**](/windows/desktop/api/Netioapi/ns-netioapi-_mib_ifstack_table)                       | Contains a table of network interface stack row entries. This specifies the relationship of the network interfaces on an interface stack.                                                                                                   |
| [**MIB\_IFSTATUS**](/windows/desktop/api/Iprtrmib/ns-iprtrmib-_mib_ifstatus)                                  | Stores status information for a particular interface.                                                                                                                                                                                       |
| [**MIB\_IFTABLE**](/previous-versions/windows/desktop/api/Ifmib/ns-ifmib-_mib_iftable)                                    | Contains a table of interface entries.                                                                                                                                                                                                      |
| [**MIB\_IF\_ROW2**](/windows/desktop/api/Netioapi/ns-netioapi-_mib_if_row2)                                   | Stores information about a particular interface.                                                                                                                                                                                            |
| [**MIB\_IF\_TABLE2**](/windows/desktop/api/Netioapi/ns-netioapi-_mib_if_table2)                               | Contains a table of interface entries.                                                                                                                                                                                                      |
| [**MIB\_INVERTEDIFSTACK\_ROW**](/windows/desktop/api/Netioapi/ns-netioapi-_mib_invertedifstack_row)           | Represents the relationship between two network interfaces.                                                                                                                                                                                 |
| [**MIB\_INVERTEDIFSTACK\_TABLE**](/windows/desktop/api/Netioapi/ns-netioapi-_mib_invertedifstack_table)       | Contains a table of inverted network interface stack row entries. This specifies the relationship of the network interfaces on an interface stack in reverse order.                                                                         |
| [**MIB\_IPADDRROW**](/previous-versions/windows/desktop/api/Ipmib/ns-ipmib-_mib_ipaddrrow_w2k)                                | Specifies information for a particular IPv4 address in the [**MIB\_IPADDRTABLE**](/previous-versions/windows/desktop/api/Ipmib/ns-ipmib-_mib_ipaddrtable) structure.                                                                                                                           |
| [**MIB\_IPADDRTABLE**](/previous-versions/windows/desktop/api/Ipmib/ns-ipmib-_mib_ipaddrtable)                            | Contains a table of IPv4 address entries.                                                                                                                                                                                                   |
