# Backup

## In this section



| Topic                                                                           | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|---------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Backup](https://docs.microsoft.com/windows/desktop/Backup/backup)<br/>                                              | Registry keys for backup and restore allow backup applications to communicate with other applications and services about backup and restore operations. The tape backup API enables backup applications to archive data to tape. The single-instance store (SIS) API enables backup applications to use the SIS architecture for maintaining duplicate files with a minimum of overhead. The raw encryption API enables backup and restore of encrypted files.<br/> |
| [Data Deduplication API](https://docs.microsoft.com/previous-versions/windows/desktop/dedup/data-deduplication-api-portal)<br/>        | The Data Deduplication API enables backup applications to implement their own customized backup and restore methods that take advantage of the Data Deduplication feature.<br/>                                                                                                                                                                                                                                                                                     |
| [Volume Shadow Copy Service](https://docs.microsoft.com/windows/desktop/VSS/volume-shadow-copy-service-portal)<br/> | The Volume Shadow Copy Service (VSS) is a set of COM interfaces that implements a framework to allow volume backups to be performed while applications on a system continue to write to the volumes.<br/>                                                                                                                                                                                                                                                           |
| [Windows Server Backup](https://docs.microsoft.com/previous-versions/windows/desktop/wsb/windows-server-backup-portal)<br/>            | The Windows Server Backup feature automatically backs up and restores data for an application that contains a VSS writer and registers with Windows Server Backup.<br/>                                                                                                                                                                                                                                                                                             |
