# IVMRCClientControl::AboutBox method

The **AboutBox** method displays the control's copyright and version information in a dialog box.

## Syntax


```C++
HRESULT AboutBox();
```



## Parameters

This method has no parameters.

## Return value

This method can return one of these values.



| Return code                                                                          | Description                              |
|--------------------------------------------------------------------------------------|------------------------------------------|
| <dl> <dt>**S\_OK**</dt> </dl> | The operation was successful.<br/> |
