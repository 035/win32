# VMVirtualServerEvent enumeration

The **VMVirtualServerEvent** enumeration specifies events generated by an [**IVMVirtualServer**](ivmvirtualserver.md) object.

## Syntax


```C++
typedef enum  { 
  vmVirtualServerEvent_ServiceEvent        = 1,
  vmVirtualServerEvent_VMStateChange       = 2,
  vmVirtualServerEvent_EventLogged         = 3,
  vmVirtualServerEvent_VMHeartbeatStopped  = 4
} VMVirtualServerEvent;
```
