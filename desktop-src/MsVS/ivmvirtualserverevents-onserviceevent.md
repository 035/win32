# IVMVirtualServerEvents::OnServiceEvent method

The **OnServiceEvent** method is called when a Virtual Server service event occurs.

## Syntax


```C++
HRESULT OnServiceEvent(
  [in] long logMessageID
);
```



## Parameters

<dl> <dt>

*logMessageID* \[in\]
</dt> <dd>

The message identifier of the event log message that was logged.

</dd> </dl>

## Return value

A [**32-bit value**](hresult-codes-specific-to-the-virtual-server.md) that describes an error or warning.

## Remarks

This method is called when Virtual Server logs a service event message to the Windows event log. Service event log messages are a subset of the event log messages generated by Virtual Server and include events such as Virtual Server starting, stopping, running out of memory, experiencing a fatal error, and others.

The client program must implement this interface method to receive notification of the [**vmVirtualServerEvent\_ServiceEvent**](vmvirtualserverevent.md) event originating from [**IVMVirtualServer**](ivmvirtualserver.md).

To be notified when Virtual Server logs any event log message, use the [**OnEventLogged**](ivmvirtualserverevents-oneventlogged.md) method.
