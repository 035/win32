# VMRCClientEvents enumeration

The **VMRCClientEvents** enumeration specifies events generated by the [**IVMRCClientControl**](ivmrcclientcontrol.md) interface.

## Syntax


```C++
typedef enum  { 
  vmrcClientEvent_OnSwitchedDisplay  = 1,
  vmrcClientEvent_OnStateChanged     = 2
} VMRCClientEvents;
```

</dd> </dl>
