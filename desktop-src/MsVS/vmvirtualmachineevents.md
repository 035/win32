# VMVirtualMachineEvents enumeration

The **VMVirtualMachineEvents** enumeration specifies events generated by an [**IVMVirtualMachine**](ivmvirtualmachine.md) object.

## Syntax


```C++
typedef enum  { 
  vmVirtualMachineEvent_StateChanged          = 1,
  vmVirtualMachineEvent_RequestShutdown       = 2,
  vmVirtualMachineEvent_Reset                 = 3,
  vmVirtualMachineEvent_TripleFault           = 4,
  vmVirtualMachineEvent_HeartbeatStopped      = 5,
  vmVirtualMachineEvent_ConfigurationChanged  = 6
} VMVirtualMachineEvents;
```

## Remarks

The value **vmVirtualMachineEvent\_RequestShutdown** indicates that a request to shut down the virtual machine session hardware has been made. Operating system shutdown routines, such as [**IVMGuestOS:Shutdown**](ivmguestos-shutdown.md), do not fire this event directly.
