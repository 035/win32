# Virtual Server Functions

Virtual Server 2005 R2 SP1 exposes the following functions for mounting and unmounting virtual hard disk (VHD) files as a virtual disk device on the host operating system:

-   [**GetSCSIAddress**](getscsiaddress.md)
-   [**MountVHD**](mountvhd.md)
-   [**UnmountVHD**](unmountvhd.md)
