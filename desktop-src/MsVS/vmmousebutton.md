# VMMouseButton enumeration

The **VMMouseButton** enumeration specifies a mouse button.

## Syntax


```C++
typedef enum  { 
  vmMouseButton_Left    = 1,
  vmMouseButton_Right   = 2,
  vmMouseButton_Center  = 3
} VMMouseButton;
```
