# Wireless Ad Hoc Interfaces

The wireless ad hoc programming interface is composed of the following interfaces.



| Interface Name                                                                       | Description                                                                                            |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**IDot11AdHocInterface**](idot11adhocinterface.md)                                 | Represents a wireless network interface card (NIC).                                                    |
| [**IDot11AdHocInterfaceNotificationSink**](idot11adhocinterfacenotificationsink.md) | Defines the notifications supported by [**IDot11AdHocInterface**](idot11adhocinterface.md).           |
| [**IDot11AdHocManager**](idot11adhocmanager.md)                                     | Creates and manages 802.11 ad hoc networks.                                                            |
| [**IDot11AdHocManagerNotificationSink**](idot11adhocmanagernotificationsink.md)     | Defines the notifications supported by the [**IDot11AdHocManager**](idot11adhocmanager.md) interface. |
| [**IDot11AdHocNetwork**](idot11adhocnetwork.md)                                     | Represents an available ad hoc network destination within connection range.                            |
| [**IDot11AdHocNetworkNotificationSink**](idot11adhocnetworknotificationsink.md)     | Defines the notifications supported by the [**IDot11AdHocNetwork**](idot11adhocnetwork.md) interface. |
| [**IDot11AdHocSecuritySettings**](idot11adhocsecuritysettings.md)                   | Specifies the security settings for a wireless ad hoc network.                                         |
| [**IEnumDot11AdHocInterfaces**](ienumdot11adhocinterfaces.md)                       | Represents the collection of currently visible 802.11 ad hoc network interfaces.                       |
| [**IEnumDot11AdHocNetworks**](ienumdot11adhocnetworks.md)                           | Represents the collection of currently visible 802.11 ad hoc networks.                                 |
| [**IEnumDot11AdHocSecuritySettings**](ienumdot11adhocsecuritysettings.md)           | Represents the collection of security settings associated with each visible wireless ad hoc network.   |



 

> [!Note]  
> Ad hoc mode might not be available in future versions of Windows. Starting with Windows 8.1 and Windows Server 2012 R2, use [Wi-Fi Direct](about-the-wi-fi-direct-api.md) instead.
