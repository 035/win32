# IsStorageSpacesDirectCacheSupported method of the MSCluster\_ClusterUtilities class

Checks if storage spaces direct cache is supported on this node.

## Syntax


```mof
boolean IsStorageSpacesDirectCacheSupported();
```



## Parameters

This method has no parameters.

## Return value

**true** if storage spaces direct cache is supported on this node; otherwise, **false**.
