# SetNodeMajorityQuorum method of the MSCluster\_Cluster class

Sets the cluster to not have a node majority quorum.

## Syntax


```mof
void SetNodeMajorityQuorum();
```



## Parameters

This method has no parameters.

## Return value

This method does not return a value.
