# IsValidationSuccessful method of the MSCluster\_ValidationStatus class

Returns true if all the tests are run and no errors occur otherwise false.

## Syntax


```mof
boolean IsValidationSuccessful();
```



## Parameters

This method has no parameters.

## Return value

Returns **True** if all the tests are run and no errors occur otherwise **false**.
