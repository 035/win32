# MSCluster\_Property class

An abstract [*WMI class*](https://msdn.microsoft.com/library/aa373136#-wolf-wmi-class-gly) that is a superclass for properties of cluster objects.

The following syntax is simplified from MOF code.

## Syntax

``` syntax
[Abstract, UUID("{4D5365E3-A8F2-4167-8565-7A588D1F4FA5}"), AMENDMENT]
class MSCluster_Property
{
};
```

## Members

The **MSCluster\_Property** class does not define any members.
