# RenewAddress method of the MSCluster\_Resource class

Renews the IPv4 address DHCP lease. Valid only for IP address resources.

## Syntax


```mof
void RenewAddress();
```



## Parameters

This method has no parameters.

## Return value

This method does not return a value.
