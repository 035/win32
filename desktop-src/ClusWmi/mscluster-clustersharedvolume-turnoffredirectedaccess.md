# TurnOffRedirectedAccess method of the MSCluster\_ClusterSharedVolume class

Disables direct IO on the cluster shared volume.

## Syntax


```mof
void TurnOffRedirectedAccess();
```



## Parameters

This method has no parameters.

## Return value

This method does not return a value.
