# Reset method of the MSCluster\_NetworkInterface class

Requests a reset. The return value should be 0 if the request was successfully executed, 1 if the request is not supported and some other value if an error occurred.

## Syntax


```mof
uint32 Reset();
```



## Parameters

This method has no parameters.
