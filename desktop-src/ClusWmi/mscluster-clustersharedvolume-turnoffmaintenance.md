# TurnOffMaintenance method of the MSCluster\_ClusterSharedVolume class

Takes the cluster shared volume out of maintenance mode.

## Syntax


```mof
void TurnOffMaintenance();
```



## Parameters

This method has no parameters.

## Return value

This method does not return a value.
