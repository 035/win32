# Releasing Other System Resources

If a source list or log file was used, resources should be released by calling [**SetupFreeSourceList**](/windows/win32/Setupapi/nf-setupapi-setupfreesourcelista?branch=master) or [**SetupTerminateFileLog**](/windows/win32/Setupapi/nf-setupapi-setupterminatefilelog?branch=master).
