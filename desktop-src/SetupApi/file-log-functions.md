# File Log Functions

The following functions are used with file logs.



| Function                                                   | Description                                 |
|------------------------------------------------------------|---------------------------------------------|
| [**SetupInitializeFileLog**](/windows/win32/Setupapi/nf-setupapi-setupinitializefileloga?branch=master)   | Initializes a log file for use.             |
| [**SetupLogFile**](/windows/win32/Setupapi/nf-setupapi-setuplogfilea?branch=master)                       | Adds an entry to the log file.              |
| [**SetupQueryFileLog**](/windows/win32/Setupapi/nf-setupapi-setupqueryfileloga?branch=master)             | Retrieves information from a log file.      |
| [**SetupRemoveFileLogEntry**](/windows/win32/Setupapi/nf-setupapi-setupremovefilelogentrya?branch=master) | Removes an entry from a log file.           |
| [**SetupTerminateFileLog**](/windows/win32/Setupapi/nf-setupapi-setupterminatefilelog?branch=master)     | Releases resources allocated to a log file. |
