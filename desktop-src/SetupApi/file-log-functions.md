# File Log Functions

The following functions are used with file logs.



| Function                                                   | Description                                 |
|------------------------------------------------------------|---------------------------------------------|
| [**SetupInitializeFileLog**](setupinitializefilelog.md)   | Initializes a log file for use.             |
| [**SetupLogFile**](setuplogfile.md)                       | Adds an entry to the log file.              |
| [**SetupQueryFileLog**](setupqueryfilelog.md)             | Retrieves information from a log file.      |
| [**SetupRemoveFileLogEntry**](setupremovefilelogentry.md) | Removes an entry from a log file.           |
| [**SetupTerminateFileLog**](setupterminatefilelog.md)     | Releases resources allocated to a log file. |
