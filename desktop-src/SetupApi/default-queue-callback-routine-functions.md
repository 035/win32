# Default Queue Callback Routine Functions

The following functions are used with the default queue callback routine.



| Function                                                                   | Description                                                                                                                                                                            |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SetupDefaultQueueCallback**](setupdefaultqueuecallback.md)             | The default queue callback routine.                                                                                                                                                    |
| [**SetupInitDefaultQueueCallback**](setupinitdefaultqueuecallback.md)     | Initializes context information needed by the default queue callback routine.                                                                                                          |
| [**SetupInitDefaultQueueCallbackEx**](setupinitdefaultqueuecallbackex.md) | Initializes context information needed by the default queue callback routine and specifies an alternate window to display progress messages.                                           |
| [**SetupTermDefaultQueueCallback**](setuptermdefaultqueuecallback.md)     | Releases resources allocated by [**SetupInitDefaultQueueCallback**](setupinitdefaultqueuecallback.md) and [**SetupInitDefaultQueueCallbackEx**](setupinitdefaultqueuecallbackex.md). |
