# Default Queue Callback Routine Functions

The following functions are used with the default queue callback routine.



| Function                                                                   | Description                                                                                                                                                                            |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SetupDefaultQueueCallback**](/windows/win32/Setupapi/nf-setupapi-setupdefaultqueuecallbacka?branch=master)             | The default queue callback routine.                                                                                                                                                    |
| [**SetupInitDefaultQueueCallback**](/windows/win32/Setupapi/nf-setupapi-setupinitdefaultqueuecallback?branch=master)     | Initializes context information needed by the default queue callback routine.                                                                                                          |
| [**SetupInitDefaultQueueCallbackEx**](/windows/win32/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex?branch=master) | Initializes context information needed by the default queue callback routine and specifies an alternate window to display progress messages.                                           |
| [**SetupTermDefaultQueueCallback**](/windows/win32/Setupapi/nf-setupapi-setuptermdefaultqueuecallback?branch=master)     | Releases resources allocated by [**SetupInitDefaultQueueCallback**](/windows/win32/Setupapi/nf-setupapi-setupinitdefaultqueuecallback?branch=master) and [**SetupInitDefaultQueueCallbackEx**](/windows/win32/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex?branch=master). |
