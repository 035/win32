# MRU Source Functions

The following functions are used with MRU source file lists.



| Function                                                                 | Description                                                                                       |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**SetupAddToSourceList**](/windows/win32/Setupapi/nf-setupapi-setupaddtosourcelista?branch=master)                     | Adds an entry to a source list.                                                                   |
| [**SetupCancelTemporarySourceList**](/windows/win32/Setupapi/nf-setupapi-setupcanceltemporarysourcelist?branch=master) | Cancels any temporary list or no-browse behavior and reestablishes standard list behavior.        |
| [**SetupFreeSourceList**](/windows/win32/Setupapi/nf-setupapi-setupfreesourcelista?branch=master)                       | Frees resources allocated to a source list.                                                       |
| [**SetupQuerySourceList**](/windows/win32/Setupapi/nf-setupapi-setupquerysourcelista?branch=master)                     | Queries the current list of installation sources.                                                 |
| [**SetupRemoveFromSourceList**](/windows/win32/Setupapi/nf-setupapi-setupremovefromsourcelista?branch=master)           | Removes an entry from an installation source list.                                                |
| [**SetupSetSourceList**](/windows/win32/Setupapi/nf-setupapi-setupsetsourcelista?branch=master)                         | Sets the installation source list to the system MRU list, the user MRU list, or a temporary list. |
