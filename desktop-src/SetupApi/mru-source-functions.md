# MRU Source Functions

The following functions are used with MRU source file lists.



| Function                                                                 | Description                                                                                       |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**SetupAddToSourceList**](setupaddtosourcelist.md)                     | Adds an entry to a source list.                                                                   |
| [**SetupCancelTemporarySourceList**](setupcanceltemporarysourcelist.md) | Cancels any temporary list or no-browse behavior and reestablishes standard list behavior.        |
| [**SetupFreeSourceList**](setupfreesourcelist.md)                       | Frees resources allocated to a source list.                                                       |
| [**SetupQuerySourceList**](setupquerysourcelist.md)                     | Queries the current list of installation sources.                                                 |
| [**SetupRemoveFromSourceList**](setupremovefromsourcelist.md)           | Removes an entry from an installation source list.                                                |
| [**SetupSetSourceList**](setupsetsourcelist.md)                         | Sets the installation source list to the system MRU list, the user MRU list, or a temporary list. |
