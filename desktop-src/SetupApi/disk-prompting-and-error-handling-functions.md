# Disk Prompting and Error Handling Functions

The following functions provide disk prompting and an error handling user interface.



| Function                                         | Description                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------|
| [**SetupCopyError**](setupcopyerror.md)         | Generates a dialog box that informs the user of a copy error.                          |
| [**SetupDeleteError**](setupdeleteerror.md)     | Generates a dialog box that informs the user of a deletion error.                      |
| [**SetupPromptForDisk**](setuppromptfordisk.md) | Generates a dialog box that prompts the user for a media disk or source file location. |
| [**SetupRenameError**](setuprenameerror.md)     | Generates a dialog box that informs the user of a rename error.                        |
