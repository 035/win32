# Remote Desktop Services AudioEndpoint Enumeration Types

The following enumeration types are used with the AudioEndpoint API.

## In this section

<dl> <dt>

[**AE\_POSITION\_FLAGS**](ae-position-flags.md)
</dt> <dd>

Defines constants for the [**AE\_CURRENT\_POSITION**](ae-current-position.md) structure. These constants describe the degree of validity of the current position.

</dd> <dt>

[**APO\_BUFFER\_FLAGS**](apo-buffer-flags.md)
</dt> <dd>

Defines the buffer validation flags for the [**APO\_CONNECTION\_PROPERTY**](apo-connection-property.md) structure associated with each APO connection.

</dd> </dl>

## Remarks

The Remote Desktop Services AudioEndpoint API is for use in Remote Desktop scenarios; it is not for client applications.
