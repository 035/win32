# RemoteApp and Desktop Connection Management Service Structures

The RemoteApp and Desktop Connection Management Service API provides the following structures.

## In this section

<dl> <dt>

[**pluginResource**](/windows/win32/tspubplugincom/ns-tspubplugincom-__midl_itspubplugin_0001?branch=master)
</dt> <dd>

Contains information about a resource that can be assigned to users in RemoteApp and Desktop Connection.

</dd> <dt>

[**pluginResource2**](/windows/win32/tspubplugin2com/ns-tspubplugin2com-__midl_itspubplugin2_0002?branch=master)
</dt> <dd>

Contains additional information about a resource that can be assigned to users in RemoteApp and Desktop Connection.

</dd> <dt>

[**pluginResource2FileAssociation**](/windows/win32/tspubplugin2com/ns-tspubplugin2com-__midl_itspubplugin2_0001?branch=master)
</dt> <dd>

Contains information about a file association in RemoteApp and Desktop Connection.

</dd> </dl>
