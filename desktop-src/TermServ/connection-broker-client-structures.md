# Remote Desktop Connection Broker client structures

The Remote Desktop Connection Broker client API defines the following structures.

## In this section

<dl> <dt>

[**CB\_CONNECTION\_INFO**](cb-connection-info.md)
</dt> <dd>

Contains information about an incoming connection request.

</dd> <dt>

[**CB\_TARGET\_INFO**](cb-target-info.md)
</dt> <dd>

Contains information about the target computer and IP addresses where incoming connections should be redirected.

</dd> </dl>
