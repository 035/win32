# IRemoteDesktopClientSettings methods

The [**IRemoteDesktopClientSettings**](iremotedesktopclientsettings.md) interface exposes the following methods.

## In this section

<dl> <dt>

[**ApplySettings method**](iremotedesktopclientsettings-applysettings.md)
</dt> <dd>

Stores the specified contents in the RDP file.

</dd> <dt>

[**GetRdpProperty method**](iremotedesktopclientsettings-getrdpproperty.md)
</dt> <dd>

Retrieves a single named RDP property value.

</dd> <dt>

[**RetrieveSettings method**](iremotedesktopclientsettings-retrievesettings.md)
</dt> <dd>

Retrieves the entire RDP file as a string.

</dd> <dt>

[**SetRdpProperty method**](iremotedesktopclientsettings-setrdpproperty.md)
</dt> <dd>

Sets the value of a single named RDP property.

</dd> </dl>
