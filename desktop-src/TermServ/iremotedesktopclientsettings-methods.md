# IRemoteDesktopClientSettings methods

The [**IRemoteDesktopClientSettings**](/windows/win32/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientsettings?branch=master) interface exposes the following methods.

## In this section

<dl> <dt>

[**ApplySettings method**](/windows/win32/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-applysettings?branch=master)
</dt> <dd>

Stores the specified contents in the RDP file.

</dd> <dt>

[**GetRdpProperty method**](/windows/win32/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-getrdpproperty?branch=master)
</dt> <dd>

Retrieves a single named RDP property value.

</dd> <dt>

[**RetrieveSettings method**](/windows/win32/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-retrievesettings?branch=master)
</dt> <dd>

Retrieves the entire RDP file as a string.

</dd> <dt>

[**SetRdpProperty method**](/windows/win32/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-setrdpproperty?branch=master)
</dt> <dd>

Sets the value of a single named RDP property.

</dd> </dl>
