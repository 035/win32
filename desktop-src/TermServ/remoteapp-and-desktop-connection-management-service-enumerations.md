# RemoteApp and Desktop Connection Management Service Enumerations

The RemoteApp and Desktop Connection Management Service API provides the following enumerations.

## In this section

<dl> <dt>

[**TSPUB\_PLUGIN\_PD\_ASSIGNMENT\_TYPE**](tspub-plugin-pd-assignment-type.md)
</dt> <dd>

</dd> <dt>

[**TSPUB\_PLUGIN\_PD\_RESOLUTION\_TYPE**](tspub-plugin-pd-resolution-type.md)
</dt> <dd>
