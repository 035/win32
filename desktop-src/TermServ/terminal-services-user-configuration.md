# Remote Desktop Services User Configuration

The [**WTSQueryUserConfig**](/windows/win32/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga?branch=master) and [**WTSSetUserConfig**](/windows/win32/Wtsapi32/nf-wtsapi32-wtssetuserconfiga?branch=master) functions enable you to set and retrieve configuration information for a specified user account. The [**WTS\_CONFIG\_CLASS**](/windows/win32/Wtsapi32/ne-wtsapi32-_wts_config_class?branch=master) enumeration type defines the user configuration parameters.
