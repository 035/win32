# Remote Desktop Services User Configuration

The [**WTSQueryUserConfig**](wtsqueryuserconfig.md) and [**WTSSetUserConfig**](wtssetuserconfig.md) functions enable you to set and retrieve configuration information for a specified user account. The [**WTS\_CONFIG\_CLASS**](wts-config-class-str.md) enumeration type defines the user configuration parameters.
