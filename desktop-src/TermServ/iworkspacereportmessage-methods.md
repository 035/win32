# IWorkspaceReportMessage Methods

The [**IWorkspaceReportMessage**](iworkspacereportmessage.md) interface supports the following methods.

## In this section

<dl> <dt>

[**RegisterErrorLogMessage method**](iworkspacereportmessage-registererrorlogmessage.md)
</dt> <dd>

Registers the specified error message to use in the UI.

</dd> <dt>

[**IsErrorMessageRegistered method**](iworkspacereportmessage-iserrormessageregistered.md)
</dt> <dd>

Determines whether a specified error message is registered in a specified workspace.

</dd> <dt>

[**RegisterErrorEvent method**](iworkspacereportmessage-registererrorevent.md)
</dt> <dd>

Registers the specified error event message to use in the UI.

</dd> </dl>
