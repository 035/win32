# IWorkspaceReportMessage Methods

The [**IWorkspaceReportMessage**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspacereportmessage?branch=master) interface supports the following methods.

## In this section

<dl> <dt>

[**RegisterErrorLogMessage method**](/windows/win32/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorlogmessage?branch=master)
</dt> <dd>

Registers the specified error message to use in the UI.

</dd> <dt>

[**IsErrorMessageRegistered method**](/windows/win32/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-iserrormessageregistered?branch=master)
</dt> <dd>

Determines whether a specified error message is registered in a specified workspace.

</dd> <dt>

[**RegisterErrorEvent method**](/windows/win32/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorevent?branch=master)
</dt> <dd>

Registers the specified error event message to use in the UI.

</dd> </dl>
