# ITsSbTaskPluginNotifySink Methods

The [**ITsSbTaskPluginNotifySink**](itssbtaskpluginnotifysink.md) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**OnDeleteTaskTime method**](itssbtaskpluginnotifysink-ondeletetasktime.md)
</dt> <dd>

Notifies Remote Desktop Connection Broker (RD Connection Broker) that a task has been removed from the queue.

</dd> <dt>

[**OnReportTasks method**](itssbtaskpluginnotifysink-onreporttasks.md)
</dt> <dd>

Notifies RD Connection Broker of a new task report.

</dd> <dt>

[**OnSetTaskTime method**](itssbtaskpluginnotifysink-onsettasktime.md)
</dt> <dd>

Notifies RD Connection Broker that a task has been scheduled.

</dd> <dt>

[**OnUpdateTaskStatus method**](itssbtaskpluginnotifysink-onupdatetaskstatus.md)
</dt> <dd>

Notifies RD Connection Broker that the status of a task has changed.

</dd> </dl>
