# ITsSbTaskPluginNotifySink Methods

The [**ITsSbTaskPluginNotifySink**](/windows/win32/sbtsv/nn-sbtsv-itssbtaskpluginnotifysink?branch=master) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**OnDeleteTaskTime method**](/windows/win32/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-ondeletetasktime?branch=master)
</dt> <dd>

Notifies Remote Desktop Connection Broker (RD Connection Broker) that a task has been removed from the queue.

</dd> <dt>

[**OnReportTasks method**](/windows/win32/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onreporttasks?branch=master)
</dt> <dd>

Notifies RD Connection Broker of a new task report.

</dd> <dt>

[**OnSetTaskTime method**](/windows/win32/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onsettasktime?branch=master)
</dt> <dd>

Notifies RD Connection Broker that a task has been scheduled.

</dd> <dt>

[**OnUpdateTaskStatus method**](/windows/win32/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onupdatetaskstatus?branch=master)
</dt> <dd>

Notifies RD Connection Broker that the status of a task has changed.

</dd> </dl>
