# Deprecated Desktop Protocol Provider Interfaces

The following interfaces have been deprecated and should no longer be used. For new projects, use the [Remote Desktop Protocol Provider Interfaces](custom-remote-protocol-interfaces.md) interfaces instead.

## In this section

<dl> <dt>

[**IWTSProtocolConnection**](/windows/win32/wtsprotocol/nn-wtsprotocol-iwtsprotocolconnection?branch=master)
</dt> <dd>

[**IWTSProtocolConnection**](/windows/win32/wtsprotocol/nn-wtsprotocol-iwtsprotocolconnection?branch=master) is no longer available. Instead, use [**IWRdsProtocolConnection**](/windows/win32/wtsprotocol/nn-wtsprotocol-iwrdsprotocolconnection?branch=master).

</dd> <dt>

[**IWTSProtocolConnectionCallback**](/windows/win32/wtsprotocol/nn-wtsprotocol-iwtsprotocolconnectioncallback?branch=master)
</dt> <dd>

[**IWTSProtocolConnectionCallback**](/windows/win32/wtsprotocol/nn-wtsprotocol-iwtsprotocolconnectioncallback?branch=master) is no longer available. Instead, use [**IWRdsProtocolConnectionCallback**](/windows/win32/wtsprotocol/nn-wtsprotocol-iwrdsprotocolconnectioncallback?branch=master).

</dd> <dt>

[**IWTSProtocolLicenseConnection**](/windows/win32/wtsprotocol/nn-wtsprotocol-iwtsprotocollicenseconnection?branch=master)
</dt> <dd>

[**IWTSProtocolLicenseConnection**](/windows/win32/wtsprotocol/nn-wtsprotocol-iwtsprotocollicenseconnection?branch=master) is no longer available. Instead, use [**IWRdsProtocolLicenseConnection**](/windows/win32/wtsprotocol/nn-wtsprotocol-iwrdsprotocollicenseconnection?branch=master).

</dd> <dt>

</dd> <dt>
