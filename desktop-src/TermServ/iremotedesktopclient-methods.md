# IRemoteDesktopClient methods

The [**IRemoteDesktopClient**](iremotedesktopclient.md) interface exposes the following methods.

## In this section

<dl> <dt>

[**attachEvent method**](iremotedesktopclient-attachevent.md)
</dt> <dd>

Attaches an event handler to an event.

</dd> <dt>

[**Connect method**](iremotedesktopclient-connect.md)
</dt> <dd>

Initiates a connection by using the properties currently set on the Remote Desktop Protocol (RDP) app container client control.

</dd> <dt>

[**DeleteSavedCredentials method**](iremotedesktopclient-deletesavedcredentials.md)
</dt> <dd>

Deletes saved credentials for the specified remote computer.

</dd> <dt>

[**detachEvent method**](iremotedesktopclient-detachevent.md)
</dt> <dd>

Detaches an event handler from an event.

</dd> <dt>

[**Disconnect method**](iremotedesktopclient-disconnect.md)
</dt> <dd>

Disconnects the active connection.

</dd> <dt>

[**Reconnect method**](iremotedesktopclient-reconnect.md)
</dt> <dd>

Initiates an automatic reconnection of the Remote Desktop Protocol (RDP) app container client control to fit the session to the new width and height.

</dd> <dt>

[**UpdateSessionDisplaySettings method**](iremotedesktopclient-updatesessiondisplaysettings.md)
</dt> <dd>

Updates the width and height settings for the Remote Desktop Protocol (RDP) app container client control.

</dd> </dl>
