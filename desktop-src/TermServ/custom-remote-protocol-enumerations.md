# Remote Desktop Protocol Provider Enumerations

The following enumeration types are supported by the Remote Desktop Protocol Provider API.

## In this section

<dl> <dt>

[**WRDS\_CONNECTION\_SETTING\_LEVEL**](/windows/win32/Wtsdefs/ne-wtsdefs-_wrds_connection_setting_level?branch=master)
</dt> <dd>

Specifies the type of structure contained in the **WRdsConnectionSetting** member of the [**WRDS\_CONNECTION\_SETTINGS**](/windows/win32/Wtsdefs/ns-wtsdefs-_wrds_connection_settings?branch=master) structure.

</dd> <dt>

[**WRDS\_LISTENER\_SETTING\_LEVEL**](/windows/win32/Wtsdefs/ne-wtsdefs-_wrds_listener_setting_level?branch=master)
</dt> <dd>

Used to specify the type of structure that is contained in the **WRdsListenerSetting** member of the [**WRDS\_LISTENER\_SETTINGS**](/windows/win32/Wtsdefs/ns-wtsdefs-_wrds_listener_settings?branch=master) structure.

</dd> <dt>

[**WRDS\_SETTING\_LEVEL**](/windows/win32/Wtsdefs/ne-wtsdefs-_wrds_setting_level?branch=master)
</dt> <dd>

Specifies the type of structure contained in the **WRdsSetting** member of the [**WRDS\_SETTINGS**](/windows/win32/Wtsdefs/ns-wtsdefs-_wrds_settings?branch=master) structure.

</dd> <dt>

[**WRDS\_SETTING\_STATUS**](/windows/win32/Wtsdefs/ne-wtsdefs-_wrds_setting_status?branch=master)
</dt> <dd>

Specifies the status of a policy setting for various members of the [**WRDS\_SETTINGS\_1**](/windows/win32/Wtsdefs/ns-wtsdefs-_wrds_settings_1?branch=master) structure.

</dd> <dt>

[**WRDS\_SETTING\_TYPE**](/windows/win32/Wtsdefs/ne-wtsdefs-_wrds_setting_type?branch=master)
</dt> <dd>

Specifies the category of settings being stored in a [**WRDS\_SETTINGS**](/windows/win32/Wtsdefs/ns-wtsdefs-_wrds_settings?branch=master) structure.

</dd> <dt>

[**WTS\_CERT\_TYPE**](/windows/win32/Wtsdefs/ne-wtsdefs-wts_cert_type?branch=master)
</dt> <dd>

Contains values that specify the type of certificate used to obtain a license.

</dd> <dt>

[**WTS\_LOGON\_ERROR\_REDIRECTOR\_RESPONSE**](/windows/win32/Wtsdefs/ne-wtsdefs-_wts_logon_error_redirector_response?branch=master)
</dt> <dd>

Contains values that specify the preferred response of the protocol to a logon error.

</dd> <dt>

[**WTS\_RCM\_DRAIN\_STATE**](/windows/win32/Wtsdefs/ne-wtsdefs-wts_rcm_drain_state?branch=master)
</dt> <dd>

Contains information about the drain state of the Remote Desktop Session Host (RD Session Host) server.

</dd> <dt>

[**WTS\_RCM\_SERVICE\_STATE**](/windows/win32/Wtsdefs/ne-wtsdefs-wts_rcm_service_state?branch=master)
</dt> <dd>

Contains information about the state of the Remote Desktop Services service.

</dd> </dl>
