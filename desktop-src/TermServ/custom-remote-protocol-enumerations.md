# Remote Desktop Protocol Provider Enumerations

The following enumeration types are supported by the Remote Desktop Protocol Provider API.

## In this section

<dl> <dt>

[**WRDS\_CONNECTION\_SETTING\_LEVEL**](wrds-connection-setting-level.md)
</dt> <dd>

Specifies the type of structure contained in the **WRdsConnectionSetting** member of the [**WRDS\_CONNECTION\_SETTINGS**](wrds-connection-settings.md) structure.

</dd> <dt>

[**WRDS\_LISTENER\_SETTING\_LEVEL**](wrds-listener-setting-level.md)
</dt> <dd>

Used to specify the type of structure that is contained in the **WRdsListenerSetting** member of the [**WRDS\_LISTENER\_SETTINGS**](wrds-listener-settings.md) structure.

</dd> <dt>

[**WRDS\_SETTING\_LEVEL**](wrds-setting-level.md)
</dt> <dd>

Specifies the type of structure contained in the **WRdsSetting** member of the [**WRDS\_SETTINGS**](wrds-settings.md) structure.

</dd> <dt>

[**WRDS\_SETTING\_STATUS**](wrds-setting-status.md)
</dt> <dd>

Specifies the status of a policy setting for various members of the [**WRDS\_SETTINGS\_1**](wrds-settings-1.md) structure.

</dd> <dt>

[**WRDS\_SETTING\_TYPE**](wrds-setting-type.md)
</dt> <dd>

Specifies the category of settings being stored in a [**WRDS\_SETTINGS**](wrds-settings.md) structure.

</dd> <dt>

[**WTS\_CERT\_TYPE**](wts-cert-type.md)
</dt> <dd>

Contains values that specify the type of certificate used to obtain a license.

</dd> <dt>

[**WTS\_LOGON\_ERROR\_REDIRECTOR\_RESPONSE**](wts-logon-error-redirector-response.md)
</dt> <dd>

Contains values that specify the preferred response of the protocol to a logon error.

</dd> <dt>

[**WTS\_RCM\_DRAIN\_STATE**](wts-rcm-drain-state.md)
</dt> <dd>

Contains information about the drain state of the Remote Desktop Session Host (RD Session Host) server.

</dd> <dt>

[**WTS\_RCM\_SERVICE\_STATE**](wts-rcm-service-state.md)
</dt> <dd>

Contains information about the state of the Remote Desktop Services service.

</dd> </dl>
