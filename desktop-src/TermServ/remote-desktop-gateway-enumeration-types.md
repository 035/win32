# Remote Desktop Gateway Enumeration Types

The Remote Desktop Gateway (RD Gateway) API supports the following enumeration types.

## In this section

<dl> <dt>

[**AAAccountingDataType**](aaaccountingdatatype.md)
</dt> <dd>

Specifies the type of event that the [**ITSGAccountingEngine::DoAccounting**](itsgaccountingengine-doaccounting.md) method is being notified of.

</dd> <dt>

[**AAAuthSchemes**](aaauthschemes.md)
</dt> <dd>

Specifies the type of authentication used to connect to RD Gateway.

</dd> <dt>

[**PolicyAttributeType**](policyattributetype.md)
</dt> <dd>

Specifies the redirection settings associated with a connection.

</dd> </dl>
