# ITsSbServiceNotification Methods

The [**ITsSbServiceNotification**](/windows/win32/sbtsv/nn-sbtsv-itssbservicenotification?branch=master) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**NotifyServiceFailure method**](/windows/win32/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicefailure?branch=master)
</dt> <dd>

Notifies registered plug-ins that the Remote Desktop Connection Broker (RD Connection Broker) service has stopped.

</dd> <dt>

[**NotifyServiceSuccess method**](/windows/win32/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicesuccess?branch=master)
</dt> <dd>

Notifies registered plug-ins that the RD Connection Broker service has started.

</dd> </dl>
