# ITsSbServiceNotification Methods

The [**ITsSbServiceNotification**](itssbservicenotification.md) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**NotifyServiceFailure method**](itssbservicenotification-notifyservicefailure.md)
</dt> <dd>

Notifies registered plug-ins that the Remote Desktop Connection Broker (RD Connection Broker) service has stopped.

</dd> <dt>

[**NotifyServiceSuccess method**](itssbservicenotification-notifyservicesuccess.md)
</dt> <dd>

Notifies registered plug-ins that the RD Connection Broker service has started.

</dd> </dl>
