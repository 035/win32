# IWorkspaceScriptable2 Methods

The [**IWorkspaceScriptable2**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2?branch=master) interface supports the following methods.

## In this section

<dl> <dt>

[**StartWorkspaceEx method**](/windows/win32/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-startworkspaceex?branch=master)
</dt> <dd>

Associates user credentials and certificates with a connection ID; also contains additional security and UI elements.

</dd> <dt>

[**ResourceDismissed method**](/windows/win32/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-resourcedismissed?branch=master)
</dt> <dd>

Alerts the user that a resource has been disabled or otherwise dismissed.

</dd> </dl>
