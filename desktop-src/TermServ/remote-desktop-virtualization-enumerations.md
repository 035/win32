# Remote Desktop Virtualization Enumerations

The Remote Desktop Virtualization API supports the following enumeration types.

## In this section

<dl> <dt>

[**TARGET\_CHANGE\_TYPE**](/windows/win32/SessDirPublicTypes/ne-sessdirpublictypes-_target_change_type?branch=master)
</dt> <dd>

Specifies the type of change that occurred in a target.

</dd> <dt>

[**TARGET\_STATE**](/windows/win32/SessDirPublicTypes/ne-sessdirpublictypes-_target_state?branch=master)
</dt> <dd>

Indicates the state of a target.

</dd> <dt>

[**TARGET\_TYPE**](/windows/win32/SessDirPublicTypes/ne-sessdirpublictypes-_target_type?branch=master)
</dt> <dd>

Indicates whether a target belongs to a pool or farm.

</dd> <dt>

[**TSSESSION\_STATE**](/windows/win32/SessDirPublicTypes/ne-sessdirpublictypes-_tssession_state?branch=master)
</dt> <dd>

Indicates the state of a session.

</dd> <dt>

[**TSSD\_AddrV46Type**](/windows/win32/SessDirPublicTypes/ne-sessdirpublictypes-_tssd_addrv46type?branch=master)
</dt> <dd>

Indicates the IP address type.

</dd> <dt>

[**TS\_SB\_SORT\_BY**](/windows/win32/sbtsv/ne-sbtsv-_ts_sb_sort_by?branch=master)
</dt> <dd>

Specifies sort order. It is used as a parameter in the [**EnumerateTargets**](/windows/win32/sbtsv/nf-sbtsv-itssbresourcepluginstore-enumeratetargets?branch=master) method.

</dd> </dl>
