# Remote Desktop Virtualization Enumerations

The Remote Desktop Virtualization API supports the following enumeration types.

## In this section

<dl> <dt>

[**TARGET\_CHANGE\_TYPE**](target-change-type.md)
</dt> <dd>

Specifies the type of change that occurred in a target.

</dd> <dt>

[**TARGET\_STATE**](target-state.md)
</dt> <dd>

Indicates the state of a target.

</dd> <dt>

[**TARGET\_TYPE**](target-type.md)
</dt> <dd>

Indicates whether a target belongs to a pool or farm.

</dd> <dt>

[**TSSESSION\_STATE**](tssession-state.md)
</dt> <dd>

Indicates the state of a session.

</dd> <dt>

[**TSSD\_AddrV46Type**](tssd-addrv46type.md)
</dt> <dd>

Indicates the IP address type.

</dd> <dt>

[**TS\_SB\_SORT\_BY**](ts-sb-sort-by.md)
</dt> <dd>

Specifies sort order. It is used as a parameter in the [**EnumerateTargets**](itssbresourcepluginstore-enumeratetargets.md) method.

</dd> </dl>
