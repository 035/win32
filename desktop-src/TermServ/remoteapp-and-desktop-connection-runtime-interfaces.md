# RemoteApp and Desktop Connection Runtime interfaces

The RemoteApp and Desktop Connection Runtime API supports interfaces that allow the development of custom clients in RemoteApp and Desktop Connection.

## In this section

<dl> <dt>

[**IWorkspace**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspace?branch=master)
</dt> <dd>

Exposes methods that provide information about a connection in RemoteApp and Desktop Connection.

</dd> <dt>

[**IWorkspace2**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspace2?branch=master)
</dt> <dd>

Exposes additional methods that provide information about a connection in RemoteApp and Desktop Connection.

</dd> <dt>

[**IWorkspace3**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspace3?branch=master)
</dt> <dd>

Exposes methods that provide information about a connection in RemoteApp and Desktop Connection, and adds the ability to retrieve or set a claims token.

</dd> <dt>

[**IWorkspaceClientExt**](/windows/win32/Workspaceruntimeclientext/nn-workspaceruntimeclientext-iworkspaceclientext?branch=master)
</dt> <dd>

Exposes methods that allow the runtime to disconnect a custom client in RemoteApp and Desktop Connection.

</dd> <dt>

[**IWorkspaceRegistration**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspaceregistration?branch=master)
</dt> <dd>

Exposes methods that add and remove references to custom clients in RemoteApp and Desktop Connection.

</dd> <dt>

[**IWorkspaceRegistration2**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspaceregistration2?branch=master)
</dt> <dd>

Exposes methods that add and remove references to custom clients in RemoteApp and Desktop Connection.

</dd> <dt>

[**IWorkspaceReportMessage**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspacereportmessage?branch=master)
</dt> <dd>

Exposes methods that support error message handling for remote workspaces.

</dd> <dt>

[**IWorkspaceResTypeRegistry**](/windows/win32/Workspaceax/nn-workspaceax-iworkspacerestyperegistry?branch=master)
</dt> <dd>

Exposes methods that allow a plug-in to manage third-party file name extensions in RemoteApp and Desktop Connection runtime.

</dd> <dt>

[**IWorkspaceScriptable**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspacescriptable?branch=master)
</dt> <dd>

Exposes methods that manage RemoteApp and Desktop Connection credentials and connections.

</dd> <dt>

[**IWorkspaceScriptable2**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2?branch=master)
</dt> <dd>

Exposes methods that manage RemoteApp and Desktop Connection credentials and connections.

</dd> <dt>

[**IWorkspaceScriptable3**](/windows/win32/workspaceruntime/nn-workspaceruntime-iworkspacescriptable3?branch=master)
</dt> <dd>

Exposes methods that manage RemoteApp and Desktop Connection credentials and connections.

</dd> </dl>
