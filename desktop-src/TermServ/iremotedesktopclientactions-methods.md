# IRemoteDesktopClientActions methods

The [**IRemoteDesktopClientActions**](iremotedesktopclientactions.md) interface exposes the following methods.

## In this section

<dl> <dt>

[**ExecuteRemoteAction method**](iremotedesktopclientactions-executeremoteaction.md)
</dt> <dd>

Causes an action to be performed in the remote session.

</dd> <dt>

[**GetSnapshot method**](iremotedesktopclientactions-getsnapshot.md)
</dt> <dd>

Causes a snapshot of the Remote Desktop Protocol (RDP) app container client's in-session desktop to be taken.

</dd> <dt>

[**ResumeScreenUpdates method**](iremotedesktopclientactions-resumescreenupdates.md)
</dt> <dd>

Resumes screen updates being sent to the client.

</dd> <dt>

[**SuspendScreenUpdates method**](iremotedesktopclientactions-suspendscreenupdates.md)
</dt> <dd>

Suspends screen updates being sent to the client.

</dd> </dl>
