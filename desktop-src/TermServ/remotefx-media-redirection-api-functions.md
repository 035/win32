# RemoteFX Media Redirection API functions

The RemoteFX media redirection API supports the following functions.

## In this section

<dl> <dt>

[**WTSSetRenderHint**](/windows/win32/WtsHintApi/nf-wtshintapi-wtssetrenderhint?branch=master)
</dt> <dd>

Used by an application that is displaying content that can be optimized for displaying in a remote session to identify the region of a window that is the actual content.

</dd> </dl>
