# ITsSbResourceNotificationEx Methods

The [**ITsSbResourceNotificationEx**](itssbresourcenotificationex.md) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**NotifyClientConnectionStateChangeEx method**](itssbresourcenotificationex-notifyclientconnectionstatechangeex.md)
</dt> <dd>

Notifies registered plug-ins about state changes in a client connection.

</dd> <dt>

[**NotifySessionChangeEx method**](itssbresourcenotificationex-notifysessionchangeex.md)
</dt> <dd>

Notifies registered plug-ins about state changes in a session object.

</dd> <dt>

[**NotifyTargetChangeEx method**](itssbresourcenotificationex-notifytargetchangeex.md)
</dt> <dd>

Notifies registered plug-ins about state changes in a target object.

</dd> </dl>
