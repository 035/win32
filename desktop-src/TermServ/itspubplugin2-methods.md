# ItsPubPlugin2 Methods

The [**ItsPubPlugin2**](/windows/win32/tspubplugin2com/nn-tspubplugin2com-itspubplugin2?branch=master) interface exposes the following methods.

## In this section

<dl> <dt>

[**DeletePersonalDesktopAssignment method**](/windows/win32/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-deletepersonaldesktopassignment?branch=master)
</dt> <dd>

Called to delete a mapping between the specified user and a virtual machine in a personal virtual desktop collection.

</dd> <dt>

[**GetResource2 method**](/windows/win32/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2?branch=master)
</dt> <dd>

This method is reserved and should always return **E\_NOTIMPL**.

</dd> <dt>

[**GetResource2List method**](/windows/win32/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2list?branch=master)
</dt> <dd>

Retrieves a list of resources assigned to the specified user.

</dd> <dt>

[**ResolvePersonalDesktop method**](/windows/win32/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-resolvepersonaldesktop?branch=master)
</dt> <dd>

Called to resolve a mapping between the specified user and a virtual machine in a personal virtual desktop collection.

</dd> </dl>
