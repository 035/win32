# ItsPubPlugin2 Methods

The [**ItsPubPlugin2**](itspubplugin2.md) interface exposes the following methods.

## In this section

<dl> <dt>

[**DeletePersonalDesktopAssignment method**](itspubplugin2-deletepersonaldesktopassignment.md)
</dt> <dd>

Called to delete a mapping between the specified user and a virtual machine in a personal virtual desktop collection.

</dd> <dt>

[**GetResource2 method**](itspubplugin2-getresource2.md)
</dt> <dd>

This method is reserved and should always return **E\_NOTIMPL**.

</dd> <dt>

[**GetResource2List method**](itspubplugin2-getresource2list.md)
</dt> <dd>

Retrieves a list of resources assigned to the specified user.

</dd> <dt>

[**ResolvePersonalDesktop method**](itspubplugin2-resolvepersonaldesktop.md)
</dt> <dd>

Called to resolve a mapping between the specified user and a virtual machine in a personal virtual desktop collection.

</dd> </dl>
