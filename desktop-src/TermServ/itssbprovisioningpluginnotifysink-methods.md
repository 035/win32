# ITsSbProvisioningPluginNotifySink Methods

The [**ITsSbProvisioningPluginNotifySink**](itssbprovisioningpluginnotifysink.md) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**LockVirtualMachine method**](itssbprovisioningpluginnotifysink-lockvirtualmachine.md)
</dt> <dd>

Notifies Remote Desktop Connection Broker (RD Connection Broker) that the virtual machine is locked.

</dd> <dt>

[**OnJobCancelled method**](itssbprovisioningpluginnotifysink-onjobcancelled.md)
</dt> <dd>

Notifies RD Connection Broker that the job is cancelled.

</dd> <dt>

[**OnJobCompleted method**](itssbprovisioningpluginnotifysink-onjobcompleted.md)
</dt> <dd>

Notifies RD Connection Broker that the job is complete.

</dd> <dt>

[**OnJobCreated method**](itssbprovisioningpluginnotifysink-onjobcreated.md)
</dt> <dd>

Notifies RD Connection Broker that a provisioning job is created.

</dd> <dt>

[**OnVirtualMachineHostStatusChanged method**](itssbprovisioningpluginnotifysink-onvirtualmachinehoststatuschanged.md)
</dt> <dd>

Notifies RD Connection Broker that the status of the host of a virtual machine is changed.

</dd> <dt>

[**OnVirtualMachineStatusChanged method**](itssbprovisioningpluginnotifysink-onvirtualmachinestatuschanged.md)
</dt> <dd>

Notifies RD Connection Broker that the status of a virtual machine is changed.

</dd> </dl>
