# ITsSbProvisioningPluginNotifySink Methods

The [**ITsSbProvisioningPluginNotifySink**](/windows/win32/sbtsv/nn-sbtsv-itssbprovisioningpluginnotifysink?branch=master) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**LockVirtualMachine method**](/windows/win32/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-lockvirtualmachine?branch=master)
</dt> <dd>

Notifies Remote Desktop Connection Broker (RD Connection Broker) that the virtual machine is locked.

</dd> <dt>

[**OnJobCancelled method**](/windows/win32/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcancelled?branch=master)
</dt> <dd>

Notifies RD Connection Broker that the job is cancelled.

</dd> <dt>

[**OnJobCompleted method**](/windows/win32/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcompleted?branch=master)
</dt> <dd>

Notifies RD Connection Broker that the job is complete.

</dd> <dt>

[**OnJobCreated method**](/windows/win32/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcreated?branch=master)
</dt> <dd>

Notifies RD Connection Broker that a provisioning job is created.

</dd> <dt>

[**OnVirtualMachineHostStatusChanged method**](/windows/win32/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinehoststatuschanged?branch=master)
</dt> <dd>

Notifies RD Connection Broker that the status of the host of a virtual machine is changed.

</dd> <dt>

[**OnVirtualMachineStatusChanged method**](/windows/win32/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinestatuschanged?branch=master)
</dt> <dd>

Notifies RD Connection Broker that the status of a virtual machine is changed.

</dd> </dl>
