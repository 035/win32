# ITsSbProvisioning Methods

The [**ITsSbProvisioning**](/windows/win32/sbtsv/nn-sbtsv-itssbprovisioning?branch=master) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**CancelJob method**](/windows/win32/sbtsv/nf-sbtsv-itssbprovisioning-canceljob?branch=master)
</dt> <dd>

Cancels a provisioning job.

</dd> <dt>

[**CreateVirtualMachines method**](/windows/win32/sbtsv/nf-sbtsv-itssbprovisioning-createvirtualmachines?branch=master)
</dt> <dd>

Creates a virtual machine asynchronously.

</dd> <dt>

[**DeleteVirtualMachines method**](/windows/win32/sbtsv/nf-sbtsv-itssbprovisioning-deletevirtualmachines?branch=master)
</dt> <dd>

Deletes a virtual machine asynchronously.

</dd> <dt>

[**PatchVirtualMachines method**](/windows/win32/sbtsv/nf-sbtsv-itssbprovisioning-patchvirtualmachines?branch=master)
</dt> <dd>

Patches a virtual machine asynchronously.

</dd> </dl>
