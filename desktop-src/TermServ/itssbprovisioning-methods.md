# ITsSbProvisioning Methods

The [**ITsSbProvisioning**](itssbprovisioning.md) interfaces exposes the following methods.

<dl> <dt>

</dt> <dd>

Cancels a provisioning job.

</dd> <dt>

[**CreateVirtualMachines method**](itssbprovisioning-createvirtualmachines.md)

</dd> <dt>

[**DeleteVirtualMachines method**](itssbprovisioning-deletevirtualmachines.md)
</dt> <dd>

Deletes a virtual machine asynchronously.

</dd> <dt>

[**PatchVirtualMachines method**](itssbprovisioning-patchvirtualmachines.md)
</dt> <dd>

Patches a virtual machine asynchronously.

</dd> </dl>
