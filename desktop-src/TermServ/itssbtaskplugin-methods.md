# ITsSbTaskPlugin Methods

The [**ITsSbTaskPlugin**](/windows/win32/sbtsv/nn-sbtsv-itssbtaskplugin?branch=master) interfaces exposes the following methods.

## In this section

<dl> <dt>

[**InitializeTaskPlugin method**](/windows/win32/sbtsv/nf-sbtsv-itssbtaskplugin-initializetaskplugin?branch=master)
</dt> <dd>

Initializes a task that is in the queue of a Remote Desktop Connection Broker plugin.

</dd> <dt>

[**SetTaskQueue method**](/windows/win32/sbtsv/nf-sbtsv-itssbtaskplugin-settaskqueue?branch=master)
</dt> <dd>

Updates a task in the queue of a Remote Desktop Connection Broker plugin.

</dd> </dl>
