# ClearKeyManagementServiceListeningPort method of the SoftwareLicensingService class

Clears any previously specified listening port. This method applies to KMS hosts only.

## Syntax


```mof
uint32 ClearKeyManagementServiceListeningPort();
```



## Parameters

This method has no parameters.
