# Software License Provider

The Windows Management Instrumentation (WMI) Software Licensing provider retrieves software licensing data. The classes are defined in SppWmi.mof and are located in the "root\\cimv2" namespace.



| Classes                                                                                                                                                                                                                                                                             | Description                                                                                          |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| <span id="SoftwareLicensingProduct"></span><span id="softwarelicensingproduct"></span><span id="SOFTWARELICENSINGPRODUCT"></span>[**SoftwareLicensingProduct**](softwarelicensingproduct.md)<br/>                                                                            | Exposes the product-specific properties and methods of the Software Licensing service.<br/>    |
| <span id="SoftwareLicensingService"></span><span id="softwarelicensingservice"></span><span id="SOFTWARELICENSINGSERVICE"></span>[**SoftwareLicensingService**](softwarelicensingservice.md)<br/>                                                                            | Exposes the product-independent properties and methods of the Software Licensing service.<br/> |
| <span id="SoftwareLicensingTokenActivationLicense"></span><span id="softwarelicensingtokenactivationlicense"></span><span id="SOFTWARELICENSINGTOKENACTIVATIONLICENSE"></span>[**SoftwareLicensingTokenActivationLicense**](softwarelicensingtokenactivationlicense.md)<br/> | Exposes the properties of installed token-based activation licenses.<br/>                      |
