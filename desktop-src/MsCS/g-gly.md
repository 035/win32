# G

<dl> <dt>

<span id="_wolf_generic_application_resource_gly"></span><span id="_WOLF_GENERIC_APPLICATION_RESOURCE_GLY"></span>**Generic Application resource**
</dt> <dd>

An application being managed as a cluster [*resource*](https://www.bing.com/search?q=*resource*). The Cluster service defines the Generic Application [*resource type*](https://www.bing.com/search?q=*resource type*) for managing Generic Application resources.

</dd> <dt>

<span id="_wolf_generic_service_resource_gly"></span><span id="_WOLF_GENERIC_SERVICE_RESOURCE_GLY"></span>**Generic Service resource**
</dt> <dd>

A service being managed as a cluster [*resource*](https://www.bing.com/search?q=*resource*). The Cluster service defines the Generic Service [*resource type*](https://www.bing.com/search?q=*resource type*) for managing Generic Service resources.

</dd> <dt>

<span id="_wolf_global_update_manager_gly"></span><span id="_WOLF_GLOBAL_UPDATE_MANAGER_GLY"></span>**Global Update Manager**
</dt> <dd>

Software component in the [*Cluster service*](https://www.bing.com/search?q=*Cluster service*) that provides update services to the [*nodes*](https://www.bing.com/search?q=*nodes*) in a [*cluster*](https://www.bing.com/search?q=*cluster*).

</dd> <dt>

<span id="_wolf_group_gly"></span><span id="_WOLF_GROUP_GLY"></span>**group**
</dt> <dd>

A collection of [*resources*](https://www.bing.com/search?q=*resources*) managed as a single [*cluster object*](https://www.bing.com/search?q=*cluster object*). Typically a group contains all of the resources needed to run a specific application or service. [*Failover*](https://www.bing.com/search?q=*Failover*) and [*failback*](https://www.bing.com/search?q=*failback*) always act on groups.

</dd> </dl>
