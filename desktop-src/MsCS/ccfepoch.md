# CCFEpoch

Specifies the current CCF epoch of a resource group.



| Attribute            | Value                                                                   |
|----------------------|-------------------------------------------------------------------------|
| Data type<br/> | **ULARGE\_INTEGER**<br/>                                          |
| Access<br/>    | [Read/write](read-write-properties.md)                                 |
| Structure<br/> | [**CLUSPROP\_ULARGE\_INTEGER**](clusprop-ularge-integer.md)<br/> |
| Minimum<br/>   | 0<br/>                                                            |
| Maximum<br/>   | 0<br/>                                                            |
| Default<br/>   | 0<br/>                                                            |



 

## Remarks

The constant for this property is **CLUSREG\_NAME\_GRP\_CCF\_EPOCH**.

|                                     |                                |
|-------------------------------------|--------------------------------|

| Minimum supported server<br/> | Windows Server 2016<br/> |
