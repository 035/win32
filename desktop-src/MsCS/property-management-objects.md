# Property Management Objects

\[The Property Management objects are available for use in Windows Server 2008. They may be altered or unavailable in subsequent versions.\]

Property Management objects store and retrieve properties for [groups](groups.md), [networks](networks.md), [network interfaces](network-interfaces.md), [nodes](nodes.md), [resources](resources.md), and [resource types](resource-types.md).

-   [**ClusProperties**](clusproperties-collection.md) Collection
-   [**ClusProperty**](clusproperty-object.md) Object
-   [**ClusPropertyValues**](cluspropertyvalues-collection.md) Collection
-   [**ClusPropertyValue**](cluspropertyvalue-object.md) Object
-   [**ClusPropertyValueData**](cluspropertyvaluedata-collection.md) Collection
