# LogSizeMultiple

Specifies the multiple used to calculate the log size for a replication group.



| Attribute            | Value                                     |
|----------------------|-------------------------------------------|
| Data type<br/> | **DWORD**                                 |
| Access<br/>    | [Read/write](read-write-properties.md)   |
| Structure<br/> | [**CLUSPROP\_DWORD**](clusprop-dword.md) |
| Minimum<br/>   | 0<br/>                              |
| Maximum<br/>   | 0<br/>                              |
| Default<br/>   | 0<br/>                              |

|                                     |                                |
|-------------------------------------|--------------------------------|
