# W

<dl> <dt>

<span id="_wolf_windows_clustering_gly"></span><span id="_WOLF_WINDOWS_CLUSTERING_GLY"></span>**Windows Clustering**
</dt> <dd>

A feature of the operating system that supports two different types of clusters: the [*failover cluster*](https://www.bing.com/search?q=*failover cluster*) and the [*network load balancing cluster*](https://www.bing.com/search?q=*network load balancing cluster*).

</dd> <dt>

<span id="_wolf_windows_internet_name_service_gly"></span><span id="_WOLF_WINDOWS_INTERNET_NAME_SERVICE_GLY"></span>**Windows Internet Name Service (WINS)**
</dt> <dd>

An enhanced NetBIOS Name Server (NBNS) designed by Microsoft to register NetBIOS computer names and to resolve them to IP addresses for both local and remote hosts.

</dd> <dt>

<span id="_wolf_windows_management_instrumentation_gly"></span><span id="_WOLF_WINDOWS_MANAGEMENT_INSTRUMENTATION_GLY"></span>**Windows Management Instrumentation (WMI)**
</dt> <dd>

A unified architecture for describing, accessing, and instrumenting objects. Part of this architecture is a large repository of [*CIM classes*](https://www.bing.com/search?q=*CIM classes*) that WMI can use to carry out management tasks on specific objects.

</dd> <dt>

<span id="_wolf_wmi_gly"></span><span id="_WOLF_WMI_GLY"></span>**WMI**
</dt> <dd>

See [*Windows Management Instrumentation*](#-wolf-windows-management-instrumentation-gly).

</dd> <dt>

<span id="_wolf_wmi_class_gly"></span><span id="_WOLF_WMI_CLASS_GLY"></span>**WMI class**
</dt> <dd>

A template for a type of managed object. WMI classes are based on the [*Common Information Model*](https://www.bing.com/search?q=*Common Information Model*) (CIM), as implemented in [*Windows Management Instrumentation*](#-wolf-windows-management-instrumentation-gly) (WMI).

</dd> <dt>

<span id="_wolf_wolfpack_gly"></span><span id="_WOLF_WOLFPACK_GLY"></span>**Wolfpack**
</dt> <dd>

One of the former pseudonyms for the [*Failover Cluster API*](https://www.bing.com/search?q=*Failover Cluster API*).

Other former names or pseudonyms include [*Cluster Server*](https://www.bing.com/search?q=*Cluster Server*), [*Clustering Service*](https://www.bing.com/search?q=*Clustering Service*), and [*MSCS*](https://www.bing.com/search?q=*MSCS*).

</dd> </dl>
