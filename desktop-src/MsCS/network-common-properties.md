# Network Common Properties

[Common properties](common-properties.md) for [networks](networks.md) are data values stored in the [cluster database](cluster-database.md) that describe the identity and behavior of each network in a [*cluster*](https://www.bing.com/search?q=*cluster*).

## In this section

<dl> <dt>

[**Address**](networks-address.md)
</dt> <dd>

Provides the address for the entire [network](networks.md) or subnet. The following table summarizes the attributes of the [**Address**](networks-address.md) property.

</dd> <dt>

[**AutoMetric**](networks-autometric.md)
</dt> <dd>

Provides the [**AutoMetric**](networks-autometric.md) of the [network](networks.md) in the [cluster](https://www.bing.com/search?q=cluster).

</dd> <dt>

[**AddressMask**](networks-addressmask.md)
</dt> <dd>

Provides the mask that distinguishes the [network](networks.md) and host portions of an address. The following table summarizes the attributes of the [**AddressMask**](networks-addressmask.md) property.

</dd> <dt>

[**AdminExtensions**](networks-adminextensions.md)
</dt> <dd>

This property is not supported.

</dd> <dt>

[**Description**](networks-description.md)
</dt> <dd>

Provides comments about the [network](networks.md). The following table summarizes the attributes of the **Description** property.

</dd> <dt>

[**IPv4Addresses**](ipv4addresses.md)
</dt> <dd>

IPv4 addresses for the network resource.

</dd> <dt>

[**IPv4PrefixLengths**](ipv4prefixlengths.md)
</dt> <dd>

IPv4 prefix lengths for the network resource.

</dd> <dt>

[**IPv6Addresses**](ipv6addresses.md)
</dt> <dd>

IPv6 addresses for the network resource.

</dd> <dt>

[**IPv6PrefixLengths**](ipv6prefixlengths.md)
</dt> <dd>

IPv6 prefix lengths for the network resource

</dd> <dt>

[**Metric**](networks-metric.md)
</dt> <dd>

Provides the [**Metric**](networks-metric.md) of the [network](networks.md) in the [cluster](https://www.bing.com/search?q=cluster).

</dd> <dt>

[**Name**](networks-name.md)
</dt> <dd>

Provides the name of the [network](networks.md). The following table summarizes the attributes of the [**Name**](networks-name.md) property.

</dd> <dt>

[**Role**](networks-role.md)
</dt> <dd>

Provides the [**Role**](networks-role.md) of the [network](networks.md) in the [cluster](https://www.bing.com/search?q=cluster).

</dd> </dl>
