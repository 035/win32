# Local Quorum

\[The Local Quorum resource type is available for use in Windows Server 2003. It may be altered or unavailable in subsequent versions.\]

The Local Quorum resource type manages the local quorum resource in clusters that use the local quorum option.

The following table summarizes the characteristics of the Local Quorum resource type.



| Characteristic                                                | Description                                                                                           |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| Required [dependencies](resource-dependencies.md)<br/> | None<br/>                                                                                       |
| Optional private properties<br/>                        | [**Debug**](local-quorum-resources-debug.md), [**Path**](local-quorum-resources-path.md)<br/> |
