# Failover Cluster Application Types

This section describes the application types in a failover clustering environment, and how each application type interacts with the clustering environment.



| Application type                                                       | Description                                                                       |
|------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [Cluster-unaware applications](cluster-unaware-applications.md)       | Applications that do not interact with clusters.                                  |
| [Cluster-aware applications](cluster-aware-applications.md)           | Applications that implement cluster operations and react to cluster events.       |
| [Cluster Management Applications](cluster-management-applications.md) | A UI that monitors or controls various components of a failover cluster.          |
| [Custom Resource Types](custom-resource-types.md)                     | Supports a new type of cluster resource, or extends an existing cluster resource. |
