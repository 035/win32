# Virtual Machine Configuration

The Virtual Machine Configuration [resource type](resource-types.md) is used to control the state of a virtual machine configuration.

The following table summarizes the characteristics of the Virtual Machine Configuration resource type.



| Characteristic                                                   | Description                                                                                                                                                                                                                                                                                                                                                                              |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Required [dependencies](resource-dependencies.md)<br/>    | None.<br/>                                                                                                                                                                                                                                                                                                                                                                         |
| Required [private properties](private-properties.md)<br/> | [**DependsOnSharedVolumes**](virtual-machine-configurations-dependonsharedvolumes.md), [**VmId**](virtual-machine-configurations-vmid.md), [**VmStoreRootPath**](virtual-machine-configurations-vmstorerootpath.md), [**VmSwitchPorts**](virtual-machine-configurations-vmswitchports.md), and [**VmPhysicalDisks**](virtual-machine-configurations-vmphysicaldisks.md).<br/> |
| Optional private properties<br/>                           | None.<br/>                                                                                                                                                                                                                                                                                                                                                                         |
