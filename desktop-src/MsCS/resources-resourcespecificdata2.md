# ResourceSpecificData2

Contains a 64-bit value that is specific to the resource.



| Attribute | Value                                                        |
|-----------|--------------------------------------------------------------|
| Data type | Unsigned long integer                                        |
| Access    | [Read-only](read-only-properties.md)                        |
| Structure | [**CLUSPROP\_ULARGE\_INTEGER**](/previous-versions/windows/desktop/api/ClusAPI/ns-clusapi-clusprop_ularge_integer) |
| Minimum   | 0                                                            |
| Maximum   | 0                                                            |
| Default   | 0                                                            |
