# ClusPartitionEx.FreeSpace property

Gets the total disk space available to the partition in megabytes.

This property is read-only.

## Syntax


```VB
ClusPartitionEx.FreeSpace
```



## Property value

**Long** that receives the total disk space available to the partition in megabytes.
