# Storage Replica Private Properties

[Storage Replica](storage-replica.md) resources have the following private properties.

## In this section

<dl> <dt>

[**LogSizeMultiple**](logsizemultiple.md)
</dt> <dd>

Specifies the multiple used to calculate the log size for a replication group.

</dd> <dt>

[**MinimumLogSizeInBytes**](minimumlogsizeinbytes.md)
</dt> <dd>

Specifies the minimum size of an event log for storage replication, in bytes.

</dd> <dt>

[**UnitOfLogSizeChangeInBytes**](unitoflogsizechangeinbytes.md)
</dt> <dd>

Specifies the size of a change to an event log for storage replication, in bytes.

</dd> </dl>
