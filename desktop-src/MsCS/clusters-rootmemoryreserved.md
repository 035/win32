# RootMemoryReserved

Not supported.

**Windows Server 2008 R2:** Controls the amount of memory reserved for the parent partition on all cluster nodes.



| Attribute | Value                                     |
|-----------|-------------------------------------------|
| Data type | **DWORD**                                 |
| Access    | [Read/write](read-write-properties.md)   |
| Structure | [**CLUSPROP\_DWORD**](clusprop-dword.md) |
| Minimum   | 64                                        |
| Maximum   | 0xFFFFFFFF                                |
| Default   | 0xFFFFFFFF                                |
