# NodeDrainTarget

Contains the node identifier of the drain target during a targeted drain.



| Attribute | Value                                     |
|-----------|-------------------------------------------|
| Data type | **DWORD**                                 |
| Access    | [Read-only](read-only-properties.md)     |
| Structure | [**CLUSPROP\_DWORD**](/previous-versions/windows/desktop/api/ClusAPI/) |
| Minimum   | -1                                        |
| Maximum   | 0x00000000                                |
| Default   | -1                                        |
