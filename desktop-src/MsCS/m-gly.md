<dl> <dt>

<span id="_mscs_maintenance_mode_gly"></span><span id="_MSCS_MAINTENANCE_MODE_GLY"></span>**maintenance mode**
</dt> <dd>

A mechanism that places the specified disk resource in a mode that will disable health checking. After maintenance mode is enabled for a resource, Resource Monitor will ignore health check calls on the resource even though the resource is left in online mode. This will allow tools like ChkDsk to function against a resource that is in maintenance mode.

</dd> <dt>

<span id="_wolf_major_version_gly"></span><span id="_WOLF_MAJOR_VERSION_GLY"></span>**major version**
</dt> <dd>

An integer used to identify a particular release of the Cluster service. This number is combined with the [*minor version*](#-wolf-minor-version-gly) to produce a [*version number*](https://www.bing.com/search?q=*version number*).

</dd> <dt>

<span id="_wolf_managed_resource_gly"></span><span id="_WOLF_MANAGED_RESOURCE_GLY"></span>**managed resource**
</dt> <dd>

A cluster [*resource*](https://www.bing.com/search?q=*resource*) that is managed by a [*configuration object*](https://www.bing.com/search?q=*configuration object*).

</dd> <dt>

<span id="_wolf_mscs_gly"></span><span id="_WOLF_MSCS_GLY"></span>**MSCS**
</dt> <dd>

One of the former names for the [*Failover Cluster API*](https://www.bing.com/search?q=*Failover Cluster API*).

Other former names or pseudonyms include [*Cluster Server*](https://www.bing.com/search?q=*Cluster Server*), [*Clustering Service*](https://www.bing.com/search?q=*Clustering Service*), [*Wolfpack*](https://www.bing.com/search?q=*Wolfpack*).

</dd> <dt>

<span id="_wolf_minor_version_gly"></span><span id="_WOLF_MINOR_VERSION_GLY"></span>**minor version**
</dt> <dd>

The build number of a particular release of the Cluster service. This number is combined with the [*major version*](#-wolf-major-version-gly) to produce a [*version number*](https://www.bing.com/search?q=*version number*).

</dd> <dt>

<span id="_wolf_membership_manager_gly"></span><span id="_WOLF_MEMBERSHIP_MANAGER_GLY"></span>**Membership Manager**
</dt> <dd>

Software component in the [*Cluster service*](https://www.bing.com/search?q=*Cluster service*) that maintains information about the [*cluster's*](https://www.bing.com/search?q=*cluster's*) current membership.

</dd> <dt>

<span id="_wolf_mixed_mode_gly"></span><span id="_WOLF_MIXED_MODE_GLY"></span>**mixed mode**
</dt> <dd>

Describes a [*failover cluster*](https://www.bing.com/search?q=*failover cluster*) in which different [*nodes*](https://www.bing.com/search?q=*nodes*) are running different versions of the [*Cluster service*](https://www.bing.com/search?q=*Cluster service*).

</dd> <dt>

<span id="_wolf_multi_sz_value"></span><span id="_WOLF_MULTI_SZ_VALUE"></span>**MULTI-SZ value**
</dt> <dd>

A **NULL**-terminated array of **NULL**-terminated Unicode strings; see **CLUSPROP\_MULTI\_SZ**.

</dd> </dl>
