# ClusterLogSize

Controls the maximum size of the cluster log files on each of the nodes.



| Attribute | Value                                     |
|-----------|-------------------------------------------|
| Data type | **DWORD**                                 |
| Access    | [Read/write](read-write-properties.md)   |
| Structure | [**CLUSPROP\_DWORD**](/previous-versions/windows/desktop/api/ClusAPI/) |
| Minimum   | 8                                         |
| Maximum   | 1024                                      |
| Default   | 300                                       |
