<dl> <dt>

<span id="_wolf_heartbeat_gly"></span><span id="_WOLF_HEARTBEAT_GLY"></span>**heartbeat**
</dt> <dd>

Message that is sent regularly between [*cluster network drivers*](https://www.bing.com/search?q=*cluster network drivers*) on each [*node*](https://www.bing.com/search?q=*node*). Heartbeat messages are used to detect communication failure between cluster nodes.

</dd> <dt>

<span id="_wolf_high_availability_gly"></span><span id="_WOLF_HIGH_AVAILABILITY_GLY"></span>**high availability**
</dt> <dd>

Describes a resource that remains operational and usable by clients most of the time with a very low percentage of failures that interrupt service. [*Failover clusters*](https://www.bing.com/search?q=*Failover clusters*) are designed to keep applications and other cluster [*resources*](https://www.bing.com/search?q=*resources*) highly available.

See also [*fault tolerant*](https://www.bing.com/search?q=*fault tolerant*).

</dd> <dt>
