# Storage Pool Private Properties

[Storage Pool](storage-pool.md) resources have the following [private properties](private-properties.md):

## In this section

<dl> <dt>

[**Arbitrate**](storage-pools-arbitrate.md)
</dt> <dd>

Specifies whether to arbitrate on pool disks before applying a persistent reservation (PR).

</dd> <dt>

[**ConsumedCapacity**](storage-pools-consumedcapacity.md)
</dt> <dd>

Contains the consumed capacity, in bytes, of the storage pool.

</dd> <dt>

[**Description**](storage-pools-description.md)
</dt> <dd>

Contains a text description of the storage pool.

</dd> <dt>

[**Health**](storage-pools-health.md)
</dt> <dd>

Specifies the health of the storage pool.

</dd> <dt>

[**ReEvaluatePlacementTimeout**](reevaluateplacementtimeout.md)
</dt> <dd>

Specifies whether to reevaluate the placement timeout for a storage pool.

</dd> <dt>

[**State**](storage-pools-state.md)
</dt> <dd>

Specifies the state of the storage pool.

</dd> <dt>

[**TotalCapacity**](storage-pools-totalcapacity.md)
</dt> <dd>

Contains the total capacity, in bytes, of the storage pool.

</dd> </dl>
