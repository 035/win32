# ClusPropertyValue.DataCount property

\[The DataCount property is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Returns the number of [*data values*](https://www.bing.com/search?q=*data values*) currently assigned to a [*property value*](https://www.bing.com/search?q=*property value*).

## Syntax


```VB
ClusPropertyValue.DataCount As Long
```
