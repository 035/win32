# ClusSvcRegroupPruningTimeout

Controls the amount of time, in seconds, that the membership leader waits to reach full connectivity between cluster nodes.



| Attribute | Value                                     |
|-----------|-------------------------------------------|
| Data type | **DWORD**                                 |
| Access    | [Read-only](read-only-properties.md)     |
| Structure | [**CLUSPROP\_DWORD**](/previous-versions/windows/desktop/api/ClusAPI/) |
| Minimum   | 1                                         |
| Maximum   | 30                                        |
| Default   | 5                                         |
