# RestartDelay

Indicates the time delay, in milliseconds, before a failed resource is restarted.



| Attribute | Value                                     |
|-----------|-------------------------------------------|
| Data type | **DWORD**                                 |
| Access    | [Read-only](read-only-properties.md)     |
| Structure | [**CLUSPROP\_DWORD**](clusprop-dword.md) |
| Minimum   | 0                                         |
| Maximum   | ((DWORD) -1)                              |
| Default   | 500                                       |
