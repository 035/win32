# ClusPropertyValues.Count property

\[The **Count** property is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Returns the number of [*property values*](https://www.bing.com/search?q=*property values*) in the [**ClusPropertyValues**](cluspropertyvalues-collection.md) collection.

This property is read-only.

## Syntax


```VB
ClusPropertyValues.Count
```



## Property value

**Long** indicating the number of objects in the collection.
