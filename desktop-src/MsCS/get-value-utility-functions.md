# Get Value Utility Functions

The get value utility functions retrieve individual values of specific types directly from the [cluster database](cluster-database.md).



| Function                                                    | Description                                                                                                |
|-------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**ResUtilGetBinaryValue**](/windows/previous-versions/ResApi/nc-resapi-presutil_get_binary_value?branch=master)      | Retrieves a binary value from the cluster database.                                                        |
| [**ResUtilGetDwordValue**](/windows/previous-versions/ResApi/nc-resapi-presutil_get_dword_value?branch=master)        | Retrieves a **DWORD** value from the cluster database.                                                     |
| [**ResUtilGetExpandSzValue**](/windows/previous-versions/ResApi/nc-resapi-presutil_get_expand_sz_value?branch=master), | Retrieves an [*expandable string*](e-gly.md#-wolf-expandable-string-gly) value from the cluster database. |
| [**ResUtilGetMultiSzValue**](/windows/previous-versions/ResApi/nf-resapi-resutilgetmultiszvalue?branch=master)    | Retrieves a multiple string value from the cluster database.                                               |
| [**ResUtilGetSzValue**](/windows/previous-versions/ResApi/nc-resapi-presutil_get_sz_value?branch=master)              | Retrieves a string or expandable string value from the cluster database.                                   |
