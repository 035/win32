# Get Value Utility Functions

The get value utility functions retrieve individual values of specific types directly from the [cluster database](cluster-database.md).



| Function                                                    | Description                                                                                                |
|-------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**ResUtilGetBinaryValue**](/previous-versions/windows/desktop/api/ResApi/nc-resapi-presutil_get_binary_value)      | Retrieves a binary value from the cluster database.                                                        |
| [**ResUtilGetDwordValue**](/previous-versions/windows/desktop/api/ResApi/nc-resapi-presutil_get_dword_value)        | Retrieves a **DWORD** value from the cluster database.                                                     |
| [**ResUtilGetExpandSzValue**](/previous-versions/windows/desktop/api/ResApi/nc-resapi-presutil_get_expand_sz_value), | Retrieves an [*expandable string*](https://www.bing.com/search?q=*expandable string*) value from the cluster database. |
| [**ResUtilGetMultiSzValue**](/previous-versions/windows/desktop/api/ResApi/nf-resapi-resutilgetmultiszvalue)    | Retrieves a multiple string value from the cluster database.                                               |
| [**ResUtilGetSzValue**](/previous-versions/windows/desktop/api/ResApi/nc-resapi-presutil_get_sz_value)              | Retrieves a string or expandable string value from the cluster database.                                   |
