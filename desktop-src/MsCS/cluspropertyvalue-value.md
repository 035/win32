# ClusPropertyValue.Value property

\[The **Value** property is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Sets or retrieves the [*data value*](d-gly.md#-wolf-data-value-gly) of a [*property value*](p-gly.md#-wolf-property-value-gly).

This property is read-only.

## Syntax


```VB
ClusPropertyValue.Value
```
