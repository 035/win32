# ClusResGroupPreferredOwnerNodes.Count property

\[The **Count** property is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Returns the number of [nodes](nodes.md) in the [**ClusResGroupPreferredOwnerNodes**](clusresgrouppreferredownernodes-collection.md) collection.

This property is read-only.

## Syntax


```VB
ClusResGroupPreferredOwnerNodes.Count
```



## Property value

**Long** indicating the number of nodes in the collection.
