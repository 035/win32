# PrimaryToken

Contains a Shared Access Signature (SAS) that validates the credentials for an Azure Cloud Witness.



| Attribute            | Value                                                          |
|----------------------|----------------------------------------------------------------|
| Data type<br/> | Null-terminated Unicode string<br/>                      |
| Access<br/>    | [Read/write](read-write-properties.md)<br/>             |
| Structure<br/> |                                                                |
| Minimum<br/>   | **NULL**<br/>                                            |
| Maximum<br/>   | See [Maximum String Size](maximum-string-size.md).<br/> |
| Default<br/>   | **NULL**<br/>                                            |



 

## Remarks

The constant for this property is **CLUSREG\_NAME\_CLOUDWITNESS\_PRIMARY\_TOKEN**.

## Requirements



|                                     |                                |
|-------------------------------------|--------------------------------|
| Minimum supported client<br/> | None supported<br/>      |
| Minimum supported server<br/> | Windows Server 2016<br/> |
