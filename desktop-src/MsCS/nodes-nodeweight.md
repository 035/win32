# NodeWeight

Specifies a configurable weight to give to the node, which determines whether the node participates in quorum.



| Attribute | Value                                     |
|-----------|-------------------------------------------|
| Data type | **DWORD**                                 |
| Access    | [Read/write](read-write-properties.md)   |
| Structure | [**CLUSPROP\_DWORD**](clusprop-dword.md) |
| Minimum   | 0                                         |
| Maximum   | 1                                         |
| Default   | 0                                         |
