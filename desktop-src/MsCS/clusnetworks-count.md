# ClusNetworks.Count property

\[The **Count** property is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Returns the number of objects in the [**ClusNetworks**](clusnetworks-collection.md) collection.

This property is read-only.

## Syntax


```VB
ClusNetworks.Count
```



## Property value

**Long** indicating the number of objects in the [**ClusNetworks**](clusnetworks-collection.md) collection.
