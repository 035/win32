# I

<dl> <dt>

<span id="_wolf_inactive_gly"></span><span id="_WOLF_INACTIVE_GLY"></span>**inactive**
</dt> <dd>

A [*node*](https://www.bing.com/search?q=*node*) that is running but not participating in [*cluster*](https://www.bing.com/search?q=*cluster*) operations.

</dd> <dt>

<span id="_wolf_interconnect_gly"></span><span id="_WOLF_INTERCONNECT_GLY"></span>**interconnect**
</dt> <dd>

A [*network*](https://www.bing.com/search?q=*network*) that connects [*nodes*](https://www.bing.com/search?q=*nodes*) in a [*cluster*](https://www.bing.com/search?q=*cluster*).

</dd> <dt>

<span id="_wolf_internal_control_code_gly"></span><span id="_WOLF_INTERNAL_CONTROL_CODE_GLY"></span>**internal control code**
</dt> <dd>

A [*control code*](https://www.bing.com/search?q=*control code*) used by the [*Cluster service*](https://www.bing.com/search?q=*Cluster service*) to notify a [*resource DLL*](https://www.bing.com/search?q=*resource DLL*) of changes to the cluster environment. Applications cannot use internal control codes; they must use [*external control codes*](https://www.bing.com/search?q=*external control codes*).

</dd> <dt>

<span id="_wolf_ip_address_resource_gly"></span><span id="_WOLF_IP_ADDRESS_RESOURCE_GLY"></span>**IP Address resource**
</dt> <dd>

An IP address being managed as a cluster [*resource*](https://www.bing.com/search?q=*resource*) by the IP Address [*resource type*](https://www.bing.com/search?q=*resource type*). An IP Address resource can be used to create a [*failover cluster instance*](https://www.bing.com/search?q=*failover cluster instance*).

</dd> <dt>

<span id="mscs.ipv6_address_resource_gly"></span><span id="MSCS.IPV6_ADDRESS_RESOURCE_GLY"></span>**IPv6 Address resource**
</dt> <dd>

An IPv6 address being managed as a cluster [*resource*](https://www.bing.com/search?q=*resource*) by the IPv6 Address [*resource type*](https://www.bing.com/search?q=*resource type*). An IPv6 Address resource can be used to create a [*failover cluster instance*](https://www.bing.com/search?q=*failover cluster instance*).

</dd> <dt>

<span id="mscs.ipv6_tunnel_address_resource_gly"></span><span id="MSCS.IPV6_TUNNEL_ADDRESS_RESOURCE_GLY"></span>**IPv6 Tunnel Address resource**
</dt> <dd>

An IPv6 tunnel address being managed as a cluster [*resource*](https://www.bing.com/search?q=*resource*) by the IPv6 Tunnel Address [*resource type*](https://www.bing.com/search?q=*resource type*). An IPv6 Tunnel Address resource can be used to create a [*failover cluster instance*](https://www.bing.com/search?q=*failover cluster instance*).

</dd> </dl>
