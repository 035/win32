# I

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) [F](f-gly.md) [G](g-gly.md) [H](h-gly.md) I J K [L](l-gly.md) [M](m-gly.md) [N](n-gly.md) [O](o-gly.md) [P](p-gly.md) [Q](q-gly.md) [R](r-gly.md) [S](s-gly.md) [T](t-gly.md) U [V](v-gly.md) [W](w-gly.md) X Y Z

<dl> <dt>

<span id="_wolf_inactive_gly"></span><span id="_WOLF_INACTIVE_GLY"></span>**inactive**
</dt> <dd>

A [*node*](n-gly.md#-wolf-node-gly) that is running but not participating in [*cluster*](c-gly.md#-wolf-cluster-gly) operations.

</dd> <dt>

<span id="_wolf_interconnect_gly"></span><span id="_WOLF_INTERCONNECT_GLY"></span>**interconnect**
</dt> <dd>

A [*network*](n-gly.md#-wolf-network-gly) that connects [*nodes*](n-gly.md#-wolf-node-gly) in a [*cluster*](c-gly.md#-wolf-cluster-gly).

</dd> <dt>

<span id="_wolf_internal_control_code_gly"></span><span id="_WOLF_INTERNAL_CONTROL_CODE_GLY"></span>**internal control code**
</dt> <dd>

A [*control code*](c-gly.md#-wolf-control-code-gly) used by the [*Cluster service*](c-gly.md#-wolf-cluster-service-gly) to notify a [*resource DLL*](r-gly.md#-wolf-resource-dll-gly) of changes to the cluster environment. Applications cannot use internal control codes; they must use [*external control codes*](e-gly.md#-wolf-external-control-code-gly).

</dd> <dt>

<span id="_wolf_ip_address_resource_gly"></span><span id="_WOLF_IP_ADDRESS_RESOURCE_GLY"></span>**IP Address resource**
</dt> <dd>

An IP address being managed as a cluster [*resource*](r-gly.md#-wolf-resource-gly) by the IP Address [*resource type*](r-gly.md#-wolf-resource-type-gly). An IP Address resource can be used to create a [*failover cluster instance*](f-gly.md#mscs-failover-cluster-instance-gly).

</dd> <dt>

<span id="mscs.ipv6_address_resource_gly"></span><span id="MSCS.IPV6_ADDRESS_RESOURCE_GLY"></span>**IPv6 Address resource**
</dt> <dd>

An IPv6 address being managed as a cluster [*resource*](r-gly.md#-wolf-resource-gly) by the IPv6 Address [*resource type*](r-gly.md#-wolf-resource-type-gly). An IPv6 Address resource can be used to create a [*failover cluster instance*](f-gly.md#mscs-failover-cluster-instance-gly).

</dd> <dt>

<span id="mscs.ipv6_tunnel_address_resource_gly"></span><span id="MSCS.IPV6_TUNNEL_ADDRESS_RESOURCE_GLY"></span>**IPv6 Tunnel Address resource**
</dt> <dd>

An IPv6 tunnel address being managed as a cluster [*resource*](r-gly.md#-wolf-resource-gly) by the IPv6 Tunnel Address [*resource type*](r-gly.md#-wolf-resource-type-gly). An IPv6 Tunnel Address resource can be used to create a [*failover cluster instance*](f-gly.md#mscs-failover-cluster-instance-gly).

</dd> </dl>
