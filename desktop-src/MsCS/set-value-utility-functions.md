# Set Value Utility Functions

The set value utility functions write values directly to the [cluster database](cluster-database.md).



| Function                                                   | Description                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**ResUtilSetBinaryValue**](/windows/previous-versions/ResApi/nc-resapi-presutil_set_binary_value?branch=master)     | Sets a binary value in the cluster database.                                                        |
| [**ResUtilSetDwordValue**](/windows/previous-versions/ResApi/nc-resapi-presutil_set_dword_value?branch=master)       | Sets a **DWORD** value in the cluster database.                                                     |
| [**ResUtilSetExpandSzValue**](/windows/previous-versions/ResApi/nc-resapi-presutil_set_expand_sz_value?branch=master) | Sets an [*expandable string*](e-gly.md#-wolf-expandable-string-gly) value in the cluster database. |
| [**ResUtilSetMultiSzValue**](/windows/previous-versions/ResApi/nc-resapi-presutil_set_multi_sz_value?branch=master)   | Sets a multiple string value in the cluster database.                                               |
| [**ResUtilSetSzValue**](/windows/previous-versions/ResApi/nc-resapi-presutil_set_sz_value?branch=master)             | Sets a string value in the cluster database.                                                        |
