# Set Value Utility Functions

The set value utility functions write values directly to the [cluster database](cluster-database.md).



| Function                                                   | Description                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**ResUtilSetBinaryValue**](resutilsetbinaryvalue.md)     | Sets a binary value in the cluster database.                                                        |
| [**ResUtilSetDwordValue**](resutilsetdwordvalue.md)       | Sets a **DWORD** value in the cluster database.                                                     |
| [**ResUtilSetExpandSzValue**](resutilsetexpandszvalue.md) | Sets an [*expandable string*](e-gly.md#-wolf-expandable-string-gly) value in the cluster database. |
| [**ResUtilSetMultiSzValue**](resutilsetmultiszvalue.md)   | Sets a multiple string value in the cluster database.                                               |
| [**ResUtilSetSzValue**](resutilsetszvalue.md)             | Sets a string value in the cluster database.                                                        |
