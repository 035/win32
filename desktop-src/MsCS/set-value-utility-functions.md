# Set Value Utility Functions

The set value utility functions write values directly to the [cluster database](cluster-database.md).



| Function                                                   | Description                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**ResUtilSetBinaryValue**](/previous-versions/windows/desktop/api/ResApi/nc-resapi-presutil_set_binary_value)     | Sets a binary value in the cluster database.                                                        |
| [**ResUtilSetDwordValue**](/previous-versions/windows/desktop/api/ResApi/nc-resapi-presutil_set_dword_value)       | Sets a **DWORD** value in the cluster database.                                                     |
| [**ResUtilSetExpandSzValue**](/previous-versions/windows/desktop/api/ResApi/nc-resapi-presutil_set_expand_sz_value) | Sets an [*expandable string*](https://www.bing.com/search?q=*expandable string*) value in the cluster database. |
| [**ResUtilSetMultiSzValue**](/previous-versions/windows/desktop/api/ResApi/nc-resapi-presutil_set_multi_sz_value)   | Sets a multiple string value in the cluster database.                                               |
| [**ResUtilSetSzValue**](/previous-versions/windows/desktop/api/ResApi/nc-resapi-presutil_set_sz_value)             | Sets a string value in the cluster database.                                                        |
