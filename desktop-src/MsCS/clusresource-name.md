# ClusResource.Name property

\[The **Name** property is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Retrieves or sets the name of a [resource](resources.md).

This property is read/write.

## Syntax


```VB
ClusResource.Name
```



## Property value

**String** containing the new name for the resource
