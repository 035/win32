<dl> <dt>

<span id="_wolf_value_list_gly"></span><span id="_WOLF_VALUE_LIST_GLY"></span>**value list**
</dt> <dd>

A collection of data values that is used as the data portion of a property value in a [*property list*](https://www.bing.com/search?q=*property list*). Each entry except the final one in a value list is made up of a [**CLUSPROP\_VALUE**](/previous-versions/windows/desktop/api/ClusAPI/ns-clusapi-clusprop_value) structure and data. The final entry contains only the **CLUSPROP\_SYNTAX\_ENDMARK** value.

</dd> <dt>

<span id="_wolf_version_number_gly"></span><span id="_WOLF_VERSION_NUMBER_GLY"></span>**version number**
</dt> <dd>

A value that describes a version of the Cluster service as a combination of the [*major version*](https://www.bing.com/search?q=*major version*) and the [*minor version*](https://www.bing.com/search?q=*minor version*) . Version numbers are used to determine whether a [*node*](https://www.bing.com/search?q=*node*) can join a [*cluster*](https://www.bing.com/search?q=*cluster*). Examples of version numbers include the [**NodeHighestVersion**](nodes-nodehighestversion.md) and [**NodeLowestVersion**](nodes-nodelowestversion.md) properties.

</dd> <dt>

See [*failover cluster instance*](https://www.bing.com/search?q=*failover cluster instance*).

</dd> </dl>
