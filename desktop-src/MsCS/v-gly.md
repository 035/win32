<dl> <dt>

<span id="_wolf_value_list_gly"></span><span id="_WOLF_VALUE_LIST_GLY"></span>**value list**
</dt> <dd>

A collection of data values that is used as the data portion of a property value in a [*property list*](p-gly.md#-wolf-property-list-gly). Each entry except the final one in a value list is made up of a [**CLUSPROP\_VALUE**](clusprop-value.md) structure and data. The final entry contains only the **CLUSPROP\_SYNTAX\_ENDMARK** value.

</dd> <dt>

<span id="_wolf_version_number_gly"></span><span id="_WOLF_VERSION_NUMBER_GLY"></span>**version number**
</dt> <dd>

A value that describes a version of the Cluster service as a combination of the [*major version*](m-gly.md#-wolf-major-version-gly) and the [*minor version*](m-gly.md#-wolf-minor-version-gly) . Version numbers are used to determine whether a [*node*](n-gly.md#-wolf-node-gly) can join a [*cluster*](c-gly.md#-wolf-cluster-gly). Examples of version numbers include the [**NodeHighestVersion**](nodes-nodehighestversion.md) and [**NodeLowestVersion**](nodes-nodelowestversion.md) properties.

</dd> <dt>

<span id="_wolf_virtual_server_gly"></span><span id="_WOLF_VIRTUAL_SERVER_GLY"></span>**virtual server**
</dt> <dd>

See [*failover cluster instance*](f-gly.md#mscs-failover-cluster-instance-gly).

</dd> </dl>
