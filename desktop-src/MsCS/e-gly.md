<dl> <dt>

<span id="_wolf_event_log_manager_gly"></span><span id="_WOLF_EVENT_LOG_MANAGER_GLY"></span>**Event Log Manager**
</dt> <dd>

Software component in the [*Cluster service*](c-gly.md#-wolf-cluster-service-gly) that is responsible for initialization and supports event notification.

</dd> <dt>

<span id="_wolf_expandable_string_gly"></span><span id="_WOLF_EXPANDABLE_STRING_GLY"></span>**expandable string**
</dt> <dd>

A string containing unexpanded references to environment variables, such as "\\\\%SystemRoot%\\data".

</dd> <dt>

<span id="_wolf_expanded_string_gly"></span><span id="_WOLF_EXPANDED_STRING_GLY"></span>**expanded string**
</dt> <dd>

A string containing expanded references to environment variables

</dd> <dt>

<span id="_wolf_external_control_code_gly"></span><span id="_WOLF_EXTERNAL_CONTROL_CODE_GLY"></span>**external control code**
</dt> <dd>

A [*control code*](c-gly.md#-wolf-control-code-gly) that applications can use to initiate an operation on a [*cluster object*](c-gly.md#-wolf-cluster-object-gly). External control codes are passed as the *dwControlCode* parameter to [*control code functions*](c-gly.md#-wolf-control-code-functions-gly).

See also [*internal control code*](i-gly.md#-wolf-internal-control-code-gly).

</dd> </dl>
