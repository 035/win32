# MulticastLeaseServer

\[The **MulticastLeaseServer** property is no longer available for use as of Windows Server 2012.\]

This property is not supported.

**Windows Server 2003:  **

The **MulticastLeaseServer** property is a read-only property used internally by the [*cluster*](https://www.bing.com/search?q=*cluster*).
