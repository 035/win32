# IP Address Private Properties

[IP Address](ip-address.md) [resources](resources.md) have the following [private properties](private-properties.md).



| Property                                             | Status              |
|------------------------------------------------------|---------------------|
| [**Address**](ip-addresses-address.md)              | Required<br/> |
| [**DhcpAddress**](dhcpaddress.md)                   | Optional<br/> |
| [**DhcpServer**](dhcpserver.md)                     | Optional<br/> |
| [**DhcpSubnetMask**](dhcpsubnetmask.md)             | Optional<br/> |
| [**EnableDhcp**](enabledhcp.md)                     | Required<br/> |
| [**EnableNetBIOS**](ip-addresses-enablenetbios.md)  | Optional<br/> |
| [**LeaseExpiresTime**](leaseexpirestime.md)         | Optional<br/> |
| [**LeaseObtainedTime**](leaseobtainedtime.md)       | Optional<br/> |
| [**Network**](ip-addresses-network.md)              | Required<br/> |
| [**OverrideAddressMatch**](overrideaddressmatch.md) | Optional<br/> |
| [**SubnetMask**](ip-addresses-subnetmask.md)        | Required<br/> |



 

[Cluster Administrator](cluster-administrator.md) presents the IP Address private properties on the property page associated with the **Parameters** tab for IP Address resources.
