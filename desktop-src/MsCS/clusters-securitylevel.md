# SecurityLevel

Controls the level of security that applies to intracluster messages.



| Attribute | Value                                     |
|-----------|-------------------------------------------|
| Data type | **DWORD**                                 |
| Access    | [Read-only](read-only-properties.md)     |
| Structure | [**CLUSPROP\_DWORD**](clusprop-dword.md) |
| Minimum   | ClearText                                 |
| Maximum   | Encryption                                |
| Default   | Signing                                   |
