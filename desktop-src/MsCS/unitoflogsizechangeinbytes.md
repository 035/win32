# UnitOfLogSizeChangeInBytes

Specifies the size of a change to an event log for storage replication, in bytes.



| Attribute            | Value                                     |
|----------------------|-------------------------------------------|
| Data type<br/> | **DWORD**                                 |
| Access<br/>    | [Read/write](read-write-properties.md)   |
| Structure<br/> | [**CLUSPROP\_DWORD**](/windows/previous-versions/ClusAPI/?branch=master) |
| Minimum<br/>   | 0<br/>                              |
| Maximum<br/>   | 0<br/>                              |
| Default<br/>   | 0<br/>                              |

|                                     |                                |
