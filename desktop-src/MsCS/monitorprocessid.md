# MonitorProcessId

Specifies the process id of the resource monitor. The following table summarizes the attributes of the **MonitorProcessId** property.



| Attribute | Value                                                          |
|-----------|----------------------------------------------------------------|
| Data type | **DWORD**                                                      |
| Access    | [Read-only](read-only-properties.md)                          |
| Structure | [**CLUSPROP\_DWORD**](/previous-versions/windows/desktop/api/ClusAPI/)                      |
| Minimum   | **NULL**                                                       |
| Maximum   | None (but see [Maximum String Size](maximum-string-size.md)). |
| Default   | **NULL**                                                       |
