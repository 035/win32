# ClusResGroupPreferredOwnerNodes.SaveChanges method

\[The **SaveChanges** method is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Saves the list of [*preferred owner*](p-gly.md#-wolf-preferred-owner-gly) [nodes](nodes.md) to the [cluster database](cluster-database.md).

## Syntax


```VB
ClusResGroupPreferredOwnerNodes.SaveChanges()
```



## Parameters

This method has no parameters.

## Return value

This method does not return a value.
