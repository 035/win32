<td>[<strong>IDirectMusicSynth</strong>](/windows/desktop/api/dmusics/nn-dmusics-idirectmusicsynth)<br/></td>
<td>The <code>IDirectMusicSynth</code> interface is used by DirectMusic to communicate with user-mode synthesizers. If you create a custom software synthesizer that runs in user mode, it should have an <code>IDirectMusicSynth</code> interface to plug into DirectMusic. <code>IDirectMusicSynth</code> inherits from the <strong>IUnknown</strong> interface.<br/> The synthesizer is not complete without a connection to a wave sink, which is represented as an object with an [IDirectMusicSynthSink](/windows/desktop/api/dmusics/nn-dmusics-idirectmusicsynthsink) interface. <td>[<strong>IDirectMusicSynthSink</strong>](/windows/desktop/api/dmusics/nn-dmusics-idirectmusicsynthsink)<br/></td>
<td>The <code>IDirectMusicSynthSink</code> interface is now largely obsolete and is supported only by versions of DirectMusic before DirectX 8. This interface represents a streaming wave device that pulls audio from the DirectMusic software synthesizer. By separating the functionality of [IDirectMusicSynth](/windows/desktop/api/dmusics/nn-dmusics-idirectmusicsynth) from that of <code>IDirectMusicSynthSink</code>, a DirectMusic application can take the output of any synthesizer and feed it to an audio device of its choosing. <code>IDirectMusicSynthSink</code> inherits from the <strong>IUnknown</strong> interface.<br/> By default, the wave data generated by the software synth is handled by DirectMusic's internal synth sink, which feeds the wave data into DirectSound. For more information, see [IDirectMusicSynth and IDirectMusicSynthSink](https://www.bing.com/search?q=IDirectMusicSynth+and+IDirectMusicSynthSink).<br/> In addition to the methods that <code>IDirectMusicSynthSink</code> inherits from the <strong>IUnknown</strong> interface, <code>IDirectMusicSynthSink</code> supports the following methods:<br/> <dl>[<strong>IDirectMusicSynthSink::Activate</strong>](/windows/desktop/api/dmusics/nf-dmusics-idirectmusicsynthsink-activate) <br/><br />
[<strong>IDirectMusicSynthSink::GetDesiredBufferSize</strong>](/windows/desktop/api/dmusics/nf-dmusics-idirectmusicsynthsink-getdesiredbuffersize) <br/><br />
