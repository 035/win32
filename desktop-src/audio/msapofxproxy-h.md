# Msapofxproxy.h

This section contains reference topics for the Msapofxproxy.h header.

## In this section



| Topic                                                                                                             | Description                                                                                                                                                                                                                                 |
|-------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**KSP\_PINMODE**](/windows/desktop/api/Msapofxproxy/ns-msapofxproxy-tagksp_pinmode)<br/>                                                                    | The KSP\_PINMODE structure specifies the pin property and the supported audio processing modes for a pin factory.<br/>                                                                                                                |
| [**KSPROPERTY\_AUDIOEFFECTSDISCOVERY**](/windows/desktop/api/Msapofxproxy/ne-msapofxproxy-ksproperty_audioeffectsdiscovery)<br/>                          | The KSPROPERTY\_AUDIOEFFECTSDISCOVERY enumeration defines a constant that is used by the list of audio processing objects (APOs).<br/>                                                                                                |
| [**KSPROPERTY\_AUDIOEFFECTSDISCOVERY\_EFFECTSLIST**](ksproperty-audioeffectsdiscovery-effectslist.md)<br/> | The **KSPROPERTY\_AUDIOEFFECTSDISCOVERY\_EFFECTSLIST** property is a filter property whose value is a list of audio effect types that are applied to a particular KS pin factory, for a particular audio signal processing path.<br/> |
