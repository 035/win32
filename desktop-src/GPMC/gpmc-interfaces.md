# GPMC Interfaces

The following table lists the Group Policy Management Console (GPMC) interfaces.



| Interface                                                  | Description                                                                                                                                                                                                                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IGPM**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpm)                                       | Accesses GPMC interfaces to retrieve GPMC information and to create objects. Manages and queries Group Policy Objects (GPOs) and GPO links. Queries scope of management (SOM) objects.                                                                                                              |
| [**IGPMAsyncCancel**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmasynccancel)                 | Cancels an asynchronous GPMC operation, such as a backup, restore, import, copy, or report generation. The GPMC function called by the client asynchronously returns a pointer to the [**IGPMAsyncCancel**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmasynccancel) interface. **IGPMAsyncCancel** is not available through scripting. |
| [**IGPMAsyncProgress**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmasyncprogress)             | Enables client notification about the progress of an operation. Implemented by the client, and passed as a parameter to GPMC methods that can run asynchronously. [**IGPMAsyncProgress**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmasyncprogress) is not available through scripting.                                                |
| [**IGPMBackup**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmbackup)                           | Retrieves the properties of [**GPMBackup**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmbackup) objects. Deletes **GPMBackup** objects.                                                                                                                                                                                                 |
| [**IGPMBackupCollection**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmbackupcollection)       | Accesses a collection of GPO backups.                                                                                                                                                                                                                                                               |
| [**IGPMBackupDir**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmbackupdir)                     | Queries GPO backups and GPO backup collections.                                                                                                                                                                                                                                                     |
| [**IGPMClientSideExtension**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmclientsideextension) | Queries client-side extension properties such as the ID and the display name.                                                                                                                                                                                                                       |
| [**IGPMConstants**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmconstants)                     | Retrieves the values of GPMC constants.                                                                                                                                                                                                                                                             |
| [**IGPMCSECollection**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmcsecollection)             | Accesses a collection of client-side extension objects.                                                                                                                                                                                                                                             |
| [**IGPMDomain**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmdomain)                           | Queries SOM objects. Creates, restores, and queries GPOs. Creates and queries WMI filters. WMI filter queries use WMI Query Language (WQL).                                                                                                                                                         |
| [**IGPMGPO**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmgpo)                                 | Manages an individual GPO. Deletes, copies, imports, or backs up a GPO. Enables or disables user and computer configuration options. Sets security information. Retrieves multiple GPO properties for a given GPO, and generates reports of the settings in the GPO.                                |
| [**IGPMGPOCollection**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmgpocollection)             | Accesses a collection of GPOs.                                                                                                                                                                                                                                                                      |
| [**IGPMGPOLink**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmgpolink)                         | Manages a GPO link from a SOM. Sets and retrieves the properties of GPO links, such as the link order or whether a link is enabled or enforced.                                                                                                                                                     |
| [**IGPMGPOLinksCollection**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmgpolinkscollection)   | Accesses a collection of GPO links.                                                                                                                                                                                                                                                                 |
| [**IGPMPermission**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmpermission)                   | Retrieves permission properties.                                                                                                                                                                                                                                                                    |
| [**IGPMResult**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmresult)                           | Retrieves status message information while performing GPMC operations on GPOs, such as restore, import, backup, and copy operations.                                                                                                                                                                |
| [**IGPMRSOP**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmrsop)                               | Queries Resultant Set of Policy (RSoP) in logging or planning mode. Generates RSoP reports.                                                                                                                                                                                                         |
| [**IGPMSearchCriteria**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmsearchcriteria)           | Defines the criteria for search operations.                                                                                                                                                                                                                                                         |
| [**IGPMSecurityInfo**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmsecurityinfo)               | Defines the set of permissions that exist on a SOM, GPO, or WMI filter. Removes or adds permissions.                                                                                                                                                                                                |
| [**IGPMSitesContainer**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmsitescontainer)           | Queries SOM objects for particular sites in a forest.                                                                                                                                                                                                                                               |
| [**IGPMSOM**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmsom)                                 | Creates and retrieves GPO links for a SOM. Sets and retrieves security attributes and properties for a SOM.                                                                                                                                                                                         |
| [**IGPMSOMCollection**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmsomcollection)             | Accesses a collection of SOMs.                                                                                                                                                                                                                                                                      |
| [**IGPMStatusMessage**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmstatusmessage)             | Retrieves the properties of the status messages of GPO operations.                                                                                                                                                                                                                                  |
| [**IGPMStatusMsgCollection**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmstatusmsgcollection) | Accesses a collection of status messages.                                                                                                                                                                                                                                                           |
| [**IGPMTrustee**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmtrustee)                         | Retrieves information about a trustee that is a user, group, or computer in the domain.                                                                                                                                                                                                             |
| [**IGPMWMIFilter**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmwmifilter)                     | Sets or retrieves the security attributes and properties of a WMI filter.                                                                                                                                                                                                                           |
| [**IGPMWMIFilterCollection**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmwmifiltercollection) | Accesses a collection of WMI filters.                                                                                                                                                                                                                                                               |
| [**IGPMMapEntry**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmmapentry)                       | Retrieves map entries.                                                                                                                                                                                                                                                                              |
| [**IGPMMapEntryCollection**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmmapentrycollection)   | Accesses a collection of map entries.                                                                                                                                                                                                                                                               |
| [**IGPMigrationTable**](/previous-versions/windows/desktop/api/Gpmgmt/nn-gpmgmt-igpmmigrationtable)            | Accesses a migration table.                                                                                                                                                                                                                                                                         |
