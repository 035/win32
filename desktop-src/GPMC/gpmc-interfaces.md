# GPMC Interfaces

The following table lists the Group Policy Management Console (GPMC) interfaces.



| Interface                                                  | Description                                                                                                                                                                                                                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IGPM**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpm?branch=master)                                       | Accesses GPMC interfaces to retrieve GPMC information and to create objects. Manages and queries Group Policy Objects (GPOs) and GPO links. Queries scope of management (SOM) objects.                                                                                                              |
| [**IGPMAsyncCancel**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmasynccancel?branch=master)                 | Cancels an asynchronous GPMC operation, such as a backup, restore, import, copy, or report generation. The GPMC function called by the client asynchronously returns a pointer to the [**IGPMAsyncCancel**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmasynccancel?branch=master) interface. **IGPMAsyncCancel** is not available through scripting. |
| [**IGPMAsyncProgress**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmasyncprogress?branch=master)             | Enables client notification about the progress of an operation. Implemented by the client, and passed as a parameter to GPMC methods that can run asynchronously. [**IGPMAsyncProgress**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmasyncprogress?branch=master) is not available through scripting.                                                |
| [**IGPMBackup**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmbackup?branch=master)                           | Retrieves the properties of [**GPMBackup**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmbackup?branch=master) objects. Deletes **GPMBackup** objects.                                                                                                                                                                                                 |
| [**IGPMBackupCollection**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmbackupcollection?branch=master)       | Accesses a collection of GPO backups.                                                                                                                                                                                                                                                               |
| [**IGPMBackupDir**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmbackupdir?branch=master)                     | Queries GPO backups and GPO backup collections.                                                                                                                                                                                                                                                     |
| [**IGPMClientSideExtension**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmclientsideextension?branch=master) | Queries client-side extension properties such as the ID and the display name.                                                                                                                                                                                                                       |
| [**IGPMConstants**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmconstants?branch=master)                     | Retrieves the values of GPMC constants.                                                                                                                                                                                                                                                             |
| [**IGPMCSECollection**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmcsecollection?branch=master)             | Accesses a collection of client-side extension objects.                                                                                                                                                                                                                                             |
| [**IGPMDomain**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmdomain?branch=master)                           | Queries SOM objects. Creates, restores, and queries GPOs. Creates and queries WMI filters. WMI filter queries use WMI Query Language (WQL).                                                                                                                                                         |
| [**IGPMGPO**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmgpo?branch=master)                                 | Manages an individual GPO. Deletes, copies, imports, or backs up a GPO. Enables or disables user and computer configuration options. Sets security information. Retrieves multiple GPO properties for a given GPO, and generates reports of the settings in the GPO.                                |
| [**IGPMGPOCollection**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmgpocollection?branch=master)             | Accesses a collection of GPOs.                                                                                                                                                                                                                                                                      |
| [**IGPMGPOLink**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmgpolink?branch=master)                         | Manages a GPO link from a SOM. Sets and retrieves the properties of GPO links, such as the link order or whether a link is enabled or enforced.                                                                                                                                                     |
| [**IGPMGPOLinksCollection**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmgpolinkscollection?branch=master)   | Accesses a collection of GPO links.                                                                                                                                                                                                                                                                 |
| [**IGPMPermission**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmpermission?branch=master)                   | Retrieves permission properties.                                                                                                                                                                                                                                                                    |
| [**IGPMResult**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmresult?branch=master)                           | Retrieves status message information while performing GPMC operations on GPOs, such as restore, import, backup, and copy operations.                                                                                                                                                                |
| [**IGPMRSOP**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmrsop?branch=master)                               | Queries Resultant Set of Policy (RSoP) in logging or planning mode. Generates RSoP reports.                                                                                                                                                                                                         |
| [**IGPMSearchCriteria**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmsearchcriteria?branch=master)           | Defines the criteria for search operations.                                                                                                                                                                                                                                                         |
| [**IGPMSecurityInfo**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmsecurityinfo?branch=master)               | Defines the set of permissions that exist on a SOM, GPO, or WMI filter. Removes or adds permissions.                                                                                                                                                                                                |
| [**IGPMSitesContainer**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmsitescontainer?branch=master)           | Queries SOM objects for particular sites in a forest.                                                                                                                                                                                                                                               |
| [**IGPMSOM**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmsom?branch=master)                                 | Creates and retrieves GPO links for a SOM. Sets and retrieves security attributes and properties for a SOM.                                                                                                                                                                                         |
| [**IGPMSOMCollection**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmsomcollection?branch=master)             | Accesses a collection of SOMs.                                                                                                                                                                                                                                                                      |
| [**IGPMStatusMessage**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmstatusmessage?branch=master)             | Retrieves the properties of the status messages of GPO operations.                                                                                                                                                                                                                                  |
| [**IGPMStatusMsgCollection**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmstatusmsgcollection?branch=master) | Accesses a collection of status messages.                                                                                                                                                                                                                                                           |
| [**IGPMTrustee**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmtrustee?branch=master)                         | Retrieves information about a trustee that is a user, group, or computer in the domain.                                                                                                                                                                                                             |
| [**IGPMWMIFilter**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmwmifilter?branch=master)                     | Sets or retrieves the security attributes and properties of a WMI filter.                                                                                                                                                                                                                           |
| [**IGPMWMIFilterCollection**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmwmifiltercollection?branch=master) | Accesses a collection of WMI filters.                                                                                                                                                                                                                                                               |
| [**IGPMMapEntry**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmmapentry?branch=master)                       | Retrieves map entries.                                                                                                                                                                                                                                                                              |
| [**IGPMMapEntryCollection**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmmapentrycollection?branch=master)   | Accesses a collection of map entries.                                                                                                                                                                                                                                                               |
| [**IGPMigrationTable**](/windows/previous-versions/Gpmgmt/nn-gpmgmt-igpmmigrationtable?branch=master)            | Accesses a migration table.                                                                                                                                                                                                                                                                         |
