# ISchedulerJob interface

Manages the tasks and resources that are associated with a job.

To create a job, call the [**IScheduler::CreateJob**](ischeduler-createjob.md) method. To clone a job, call the [**IScheduler::CloneJob**](ischeduler-clonejob.md) method.

## Members

The **ISchedulerJob** interface inherits from the [**IDispatch**](ebbff4bc-36b2-4861-9efa-ffa45e013eb5) interface. **ISchedulerJob** also has these types of members:

-   [Methods](#methods)
-   [Properties](#properties)
