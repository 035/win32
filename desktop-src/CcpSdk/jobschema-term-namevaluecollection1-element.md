# Term (NameValueCollection1) Element

Defines an extended term or custom property for the job or task.

``` syntax
<xs:element name="Term"
    type="NameValue"
 />
```

The **Term** element is defined by the [**NameValueCollection**](jobschema-namevaluecollection-complextype.md) complex type.
