# CommandOutputType enumeration

Defines the source of the output.

## Syntax


```C++
typedef enum  { 
  CommandOutputType_Output  = 0,
  CommandOutputType_Error   = 1,
  CommandOutputType_Eof     = 2
} CommandOutputType;
```



## Constants

<dl> <dt>

<span id="CommandOutputType_Output"></span><span id="commandoutputtype_output"></span><span id="COMMANDOUTPUTTYPE_OUTPUT"></span>**CommandOutputType\_Output**
</dt> <dd>

The output was generated by the command itself (captured from standard output or standard error).

</dd> <dt>

<span id="CommandOutputType_Error"></span><span id="commandoutputtype_error"></span><span id="COMMANDOUTPUTTYPE_ERROR"></span>**CommandOutputType\_Error**
</dt> <dd>

The output is an error message string that was generated when the job that contains the command failed.

</dd> <dt>

<span id="CommandOutputType_Eof"></span><span id="commandoutputtype_eof"></span><span id="COMMANDOUTPUTTYPE_EOF"></span>**CommandOutputType\_Eof**
</dt> <dd>

There is no more output to receive. Use this in your callback to determine when there is no more output to receive.

</dd> </dl>
