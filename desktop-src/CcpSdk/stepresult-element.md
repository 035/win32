# StepResult element

Represents the result of a stage in a diagnostic test and serves as the root element of a StepResult XML file.

## Usage

``` syntax
<StepResult
  NodeName = "character_string">
  child elements
</StepResult>
```

## Attributes



| Attribute               | Type                         | Required       | Description                                                                                                                                                                                                                                         |
|-------------------------|------------------------------|----------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **NodeName**<br/> | character\_string<br/> | Yes<br/> | The name of the node that generated the result.<br/> The value can be the name of a node in the HPC cluster, or Summary, which indicates that the result is a summary of the results for all of the nodes in the test.<br/> <br/> |



## Child elements



| Element                                                          | Description                                                                                                                                                            |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BulletPoints**](bulletpoints-element.md)<br/>          | Represents a set of items in a bulleted list in a report for a diagnostic test.<br/> <br/>                                                                 |
| [**Environment**](environment-element.md)<br/>            | Represents the environment variables that you want to set for the RunStep stage of a diagnostic test.<br/> <br/>                                           |
| [**ExceptionMessage**](exceptionmessage--element.md)<br/> | Contains an exception message that is generated by the command that created the Step Result XML.<br/> <br/>                                                |
| [**FailedNodes**](failednodes-element.md)<br/>            | Indicates the nodes on which the diagnostic test failed.<br/> <br/>                                                                                        |
| [**Message**](message--element.md)<br/>                   | Contains the text of an item in a bulleted list.<br/> <br/>                                                                                                |
| [**Nodes**](nodes-element.md)<br/>                        | Indicates the nodes on which the RunStep stage of the diagnostic test should run.<br/> <br/>                                                               |
| [**Parameters**](parameters--stepresult--element.md)<br/> | Represents a set of parameters that you want to set for the RunStep stage of a diagnostic test, on one node or on all of the nodes in the test.<br/> <br/> |
| [**Result**](result-element.md)<br/>                      | Describes the result of a diagnostic test, such as whether the test completed, succeeded, or failed.<br/> <br/>                                            |
| [**Tables**](tables-element.md)<br/>                      | Represents the set of tables in the report for a diagnostic test.<br/> <br/>                                                                               |



### Child element sequence

``` syntax
ResultMessageExceptionMessageNodesFailedNodesEnvironmentParametersTablesBulletPoints
```

## Parent elements

There are no parent elements.

## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
