# Parent (ArrayOfTaskGroupDependency) Element

Defines the parent of a parent/child task group relationship.

``` syntax

    type="Parent"
 />
```

The **Parent** element is defined by the [**ArrayOfTaskGroupDependency**](jobschema-arrayoftaskgroupdependency-complextype.md) complex type.
