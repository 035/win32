# EnvironmentVariables (Task) Element

Contains the list of the environment variables for the tasks in the job.

``` syntax
<xs:element name="EnvironmentVariables"
    type="NameValueCollection1"
 />
```

The **EnvironmentVariables** element is defined by the [**Task**](jobschema-task-complextype.md) complex type.
