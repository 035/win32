# Term (NameValueCollection1) Element

Defines an extended term for the task.

``` syntax
<xs:element name="Term"
    type="NameValue"
    minOccurs="0"
    maxOccurs="unbounded"
    nillable="true"
 />
```

## Remarks

This element is a child of the [**ExtendedTerms**](schema-extendedterms-task-element.md) element for a task.
