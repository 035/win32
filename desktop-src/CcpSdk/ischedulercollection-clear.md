# ISchedulerCollection::Clear method

Removes all items from the collection.

## Syntax


```C++
HRESULT Clear();
```



## Parameters

This method has no parameters.

## Return value

If the method succeeds, the return value is S\_OK. Otherwise, the return value is an error code.
