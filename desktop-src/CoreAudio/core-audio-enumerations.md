# Core Audio Enumerations

This section describes the enumerations that are used by the Core Audio APIs in Windows Vista and later.



| Enumeration                                                                   | Description                                                                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**\_AUDCLNT\_BUFFERFLAGS**](-audclnt-bufferflags.md)                        | Status flags for an audio endpoint buffer.                                                         |
| [**AUDCLNT\_SHAREMODE**](audclnt-sharemode.md)                               | The sharing mode for a stream.                                                                     |
| [**AUDCLNT\_STREAMOPTIONS**](audclnt-streamoptions.md)                       | Defines values that describe the characteristics of an audio stream.                               |
| [**AUDIO\_STREAM\_CATEGORY**](audio-stream-category.md)                      | Specifies the category of an audio stream.                                                         |
| [**AudioSessionState**](audiosessionstate.md)                                | The state of an audio session.                                                                     |
| [**ConnectorType**](connectortype.md)                                        | The type of connector in a device topology.                                                        |
| [**DataFlow**](dataflow.md)                                                  | The data-flow direction of an audio stream.                                                        |
| [**EDataFlow**](edataflow.md)                                                | The direction in which audio data flows between an audio endpoint device and a client application. |
| [**EndpointFormFactor**](endpointformfactor.md)                              | The general physical attributes of an audio endpoint device.                                       |
| [**ERole**](erole.md)                                                        | The system role of an audio endpoint device.                                                       |
| [**KSJACK\_SINK\_CONNECTIONTYPE**](ksjack-sink-connectiontype.md)<br/> | The type of connection.<br/>                                                                 |
| [**PartType**](parttype.md)                                                  | The part type of a part (connector or subunit) in a device topology.                               |
