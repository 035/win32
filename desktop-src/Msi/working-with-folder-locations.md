# Working with Folder Locations

Use the following functions to find the source or target of a specified folder in the [Directory table](directory-table.md):

-   Obtain the path of the specified source folder by calling the [**MsiGetSourcePath**](msigetsourcepath.md) function.
-   Obtain the path of the specified target folder by calling the [**MsiGetTargetPath**](msigettargetpath.md) function.
-   Change the target location for a folder with the [**MsiSetTargetPath**](msisettargetpath.md) function.
