# ConfigurableItem.HelpLocation property

The **HelpLocation** property returns the value from the HelpLocation column of the [ModuleConfiguration table](moduleconfiguration-table.md).

This property is read-only.

## Syntax


```JScript
propVal = ConfigurableItem.HelpLocation
```



## Property value

## C++

See [**get\_HelpLocation**](/windows/win32/Mergemod/nf-mergemod-imsmconfigurableitem-get_helplocation?branch=master) function.
