# Error.Type property

The read-only **Type** property returns one of the values of msmErrorType, indicating the type of error represented by this object.

This property is read-only.

## Syntax


```JScript
propVal = Error.Type
```



## Property value

## C++

See [**get\_Type Function**](/windows/win32/Mergemod/?branch=master) function (Error Object).
