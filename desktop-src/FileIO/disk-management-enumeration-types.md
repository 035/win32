# Disk Management Enumeration Types

The following enumeration types are used with disk management.

## In this section



| Enumeration                                                                              | Description                                                                                                                                                                                                                                                                                                          |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MEDIA\_TYPE**](media-type-str.md)<br/>                                         | Represents the various forms of device media.<br/>                                                                                                                                                                                                                                                             |
| [**PARTITION\_STYLE**](partition-style-str.md)<br/>                               | Represents the format of a partition.<br/>                                                                                                                                                                                                                                                                     |
| [**STORAGE\_BUS\_TYPE**](storage-bus-type.md)<br/>                                | Provides a symbolic means of representing storage bus types.<br/>                                                                                                                                                                                                                                              |
| [**STORAGE\_COMPONENT\_HEALTH\_STATUS**](storage-component-health-status.md)<br/> | Specifies the health status of a storage component.<br/>                                                                                                                                                                                                                                                       |
| [**STORAGE\_DEVICE\_FORM\_FACTOR**](storage-device-form-factor.md)<br/>           | Specifies the form factor of a device.<br/>                                                                                                                                                                                                                                                                    |
| [**STORAGE\_DEVICE\_POWER\_CAP\_UNITS**](storage-device-power-cap-units.md)<br/>  | The units of the maximum power threshold.<br/>                                                                                                                                                                                                                                                                 |
| [**STORAGE\_PORT\_CODE\_SET**](storage-port-code-set.md)<br/>                     | Reserved for system use. <br/>                                                                                                                                                                                                                                                                                 |
| [**STORAGE\_PROPERTY\_ID**](storage-property-id.md)<br/>                          | Enumerates the possible values of the **PropertyId** member of the [**STORAGE\_PROPERTY\_QUERY**](storage-property-query.md) structure passed as input to the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) request to retrieve the properties of a storage device or adapter.<br/> |
| [**STORAGE\_PROTOCOL\_TYPE**](storage-protocol-type.md)<br/>                      | Specifies the protocol of a storage device.<br/>                                                                                                                                                                                                                                                               |
| [**STORAGE\_QUERY\_TYPE**](storage-query-type.md)<br/>                            | Used by the [**STORAGE\_PROPERTY\_QUERY**](storage-property-query.md) structure passed to the [**IOCTL\_STORAGE\_QUERY\_PROPERTY**](ioctl-storage-query-property.md) control code to indicate what information is returned about a property of a storage device or adapter.<br/>                             |
| [**WRITE\_CACHE\_CHANGE**](write-cache-change.md)<br/>                            | Indicates whether the write cache features of a device are changeable.<br/>                                                                                                                                                                                                                                    |
| [**WRITE\_CACHE\_ENABLE**](write-cache-enable.md)<br/>                            | Indicates whether the write cache is enabled or disabled.<br/>                                                                                                                                                                                                                                                 |
| [**WRITE\_CACHE\_TYPE**](write-cache-type.md)<br/>                                | Specifies the cache type.<br/>                                                                                                                                                                                                                                                                                 |
| [**WRITE\_THROUGH**](write-through.md)<br/>                                       | Specifies whether a storage device supports write-through caching.<br/>                                                                                                                                                                                                                                        |
