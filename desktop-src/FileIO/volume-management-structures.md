# Volume Management Structures

| [**FSCTL\_QUERY\_REGION\_INFO\_INPUT**](/windows/win32/WinIoctl/ns-winioctl-_fsctl_query_region_info_input?branch=master)<br/>                   | Contains the storage tier regions from the storage stack for a particular volume.<br/>                                                                                                                                                                                                                            |

| [**LOOKUP\_STREAM\_FROM\_CLUSTER\_INPUT**](/windows/win32/WinIoCtl/ns-winioctl-_lookup_stream_from_cluster_input?branch=master)<br/>             | Passed as input to the [**FSCTL\_LOOKUP\_STREAM\_FROM\_CLUSTER**](/windows/win32/WinIoCtl/?branch=master) control code.<br/>                                                                                                                                                                                        |
