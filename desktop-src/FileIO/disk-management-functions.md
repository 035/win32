# Disk Management Functions

The following functions are used in disk management.

## In this section



| Function                                                    | Description                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetDiskFreeSpace**](/windows/win32/FileAPI/nf-fileapi-getdiskfreespacea?branch=master)<br/>     | Retrieves information about the specified disk, including the amount of free space on the disk.<br/>                                                                                                                                                              |
| [**GetDiskFreeSpaceEx**](/windows/win32/FileAPI/nf-fileapi-getdiskfreespaceexa?branch=master)<br/> | Retrieves information about the amount of space that is available on a disk volume, which is the total amount of space, the total amount of free space, and the total amount of free space available to the user that is associated with the calling thread.<br/> |



 

Other functions used in disk management.



| Function                         | Description                                                                                                                                                                                                                        |
|----------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateFile**](/windows/win32/FileAPI/nf-fileapi-createfilea?branch=master) | Creates or opens a file or I/O device. The most commonly used I/O devices are as follows: file, file stream, directory, physical disk, volume, console buffer, tape drive, communications resource, mailslot, and pipe.<br/> |
| [**DeleteFile**](/windows/win32/FileAPI/nf-fileapi-deletefilea?branch=master) | Deletes an existing file.<br/>                                                                                                                                                                                               |
