| Structure                                                                                  | Description                                                                                                        |
|--------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**FILE\_NOTIFY\_EXTENDED\_INFORMATION**](file-notify-extended-information.md)<br/> | Describes the changes found by the [**ReadDirectoryChangesExW**](readdirectorychangesexw.md) function.<br/> |
| [**FILE\_NOTIFY\_INFORMATION**](file-notify-information-str.md)<br/>                | Describes the changes found by the [**ReadDirectoryChangesW**](readdirectorychangesw.md) function.<br/>     |
