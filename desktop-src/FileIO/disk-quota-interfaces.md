# Disk Quota Interfaces

The following interfaces are used with disk quotas.



| Interface                                          | Description                                                                                                    |
|----------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**IDiskQuotaControl**](idiskquotacontrol.md)     | Controls the disk quota facilities of a single NTFS file system volume.<br/>                             |
| [**IDiskQuotaEvents**](idiskquotaevents.md)       | Receives quota-related event notifications.<br/>                                                         |
| [**IDiskQuotaUser**](idiskquotauser.md)           | Represents a single user quota entry in the volume quota information file.<br/>                          |
| [**IDiskQuotaUserBatch**](idiskquotauserbatch.md) | Adds multiple quota user objects to a container that is then submitted for update in a single call.<br/> |
| [**IEnumDiskQuotaUsers**](ienumdiskquotausers.md) | Enumerates user quota entries on the volume.<br/>                                                        |
