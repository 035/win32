# TxF Functions

Transactional NTFS (TxF) provides the following functions.

## In this section



| Function                                                                      | Description                                                                                                                  |
|-------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [**TxfLogCreateFileReadContext**](txflogcreatefilereadcontext.md)<br/> | Creates a context to be used to read replication records.<br/>                                                         |
| [**TxfLogDestroyReadContext**](txflogdestroyreadcontext.md)<br/>       | Closes a read context created by the [**TxfLogCreateFileReadContext**](txflogcreatefilereadcontext.md) function.<br/> |
| [**TxfLogReadRecords**](txflogreadrecords.md)<br/>                     | Reads the redo records from the log.<br/>                                                                              |
