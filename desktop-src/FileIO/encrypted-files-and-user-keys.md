# Encrypted Files and User Keys

To create a new key for a user, use the [**SetUserFileEncryptionKey**](/windows/win32/Winefs/nf-winefs-setuserfileencryptionkey?branch=master) function. To add user keys to an encrypted file, use the [**AddUsersToEncryptedFile**](/windows/win32/Winefs/nf-winefs-adduserstoencryptedfile?branch=master) function. To query the user keys for an encrypted file, use the [**QueryUsersOnEncryptedFile**](/windows/win32/Winefs/nf-winefs-queryusersonencryptedfile?branch=master) function. To remove user keys from an encrypted file, use the [**RemoveUsersFromEncryptedFile**](/windows/win32/Winefs/nf-winefs-removeusersfromencryptedfile?branch=master) function.
