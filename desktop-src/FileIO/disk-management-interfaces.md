# Disk Management Interfaces

The following interfaces are used in disk management.

## In this section



| Interface                                                     | Description                                                                                                    |
|---------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**IDiskQuotaControl**](idiskquotacontrol.md)<br/>     | Controls the disk quota facilities of a single NTFS file system volume.<br/>                             |
| [**IDiskQuotaEvents**](idiskquotaevents.md)<br/>       | Receives quota-related event notifications.<br/>                                                         |
| [**IDiskQuotaUser**](idiskquotauser.md)<br/>           | Represents a single user quota entry in the volume quota information file.<br/>                          |
| [**IDiskQuotaUserBatch**](idiskquotauserbatch.md)<br/> | Adds multiple quota user objects to a container that is then submitted for update in a single call.<br/> |
| [**IEnumDiskQuotaUsers**](ienumdiskquotausers.md)<br/> | Enumerates user quota entries on the volume.<br/>                                                        |
