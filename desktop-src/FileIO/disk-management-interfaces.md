# Disk Management Interfaces

The following interfaces are used in disk management.

## In this section



| Interface                                                     | Description                                                                                                    |
|---------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**IDiskQuotaControl**](/windows/win32/Dskquota/?branch=master)<br/>     | Controls the disk quota facilities of a single NTFS file system volume.<br/>                             |
| [**IDiskQuotaEvents**](/windows/win32/Dskquota/?branch=master)<br/>       | Receives quota-related event notifications.<br/>                                                         |
| [**IDiskQuotaUser**](/windows/win32/Dskquota/?branch=master)<br/>           | Represents a single user quota entry in the volume quota information file.<br/>                          |
| [**IDiskQuotaUserBatch**](/windows/win32/Dskquota/?branch=master)<br/> | Adds multiple quota user objects to a container that is then submitted for update in a single call.<br/> |
| [**IEnumDiskQuotaUsers**](/windows/win32/Dskquota/?branch=master)<br/> | Enumerates user quota entries on the volume.<br/>                                                        |
