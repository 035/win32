# File Management Structures

The following structures are used in file management.

## In this section



| Structure                                                                                         | Description                                                                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BY\_HANDLE\_FILE\_INFORMATION**](/windows/win32/FileAPI/ns-fileapi-_by_handle_file_information?branch=master)<br/>                | Contains information that the [**GetFileInformationByHandle**](/windows/win32/FileAPI/nf-fileapi-getfileinformationbyhandle?branch=master) function retrieves.<br/>                                                                                                        |
| [**COPYFILE2\_EXTENDED\_PARAMETERS**](/windows/win32/WinBase/ns-winbase-copyfile2_extended_parameters?branch=master)<br/>               | Contains extended parameters for the [**CopyFile2**](/windows/win32/WinBase/nf-winbase-copyfile2?branch=master) function.<br/>                                                                                                                                             |
| [**COPYFILE2\_MESSAGE**](/windows/win32/WinBase/ns-winbase-copyfile2_message?branch=master)<br/>                                        | Passed to the [*CopyFile2ProgressRoutine*](/windows/win32/WinBase/nc-winbase-pcopyfile2_progress_routine?branch=master) callback function with information about a pending copy operation.<br/>                                                                               |
| [**CREATEFILE2\_EXTENDED\_PARAMETERS**](/windows/win32/FileAPI/ns-fileapi-_createfile2_extended_parameters?branch=master)<br/>           | Contains optional extended parameters for [**CreateFile2**](/windows/win32/FileAPI/nf-fileapi-createfile2?branch=master).<br/>                                                                                                                                             |
| [**CSV\_CONTROL\_PARAM**](/windows/win32/WinIoCtl/ns-winioctl-_csv_control_param?branch=master)<br/>                                       | Represents a type of CSV control operation.<br/>                                                                                                                                                                               |
| [**CSV\_QUERY\_FILE\_REVISION**](/windows/win32/WinIoCtl/ns-winioctl-_csv_query_file_revision?branch=master)<br/>                          | Contains information about whether files in a stream have been modified.<br/>                                                                                                                                                  |
| [**CSV\_QUERY\_MDS\_PATH**](/windows/win32/WinIoCtl/ns-winioctl-_csv_query_mds_path?branch=master)<br/>                                    | Contains the path that is used by CSV to communicate to the MDS.<br/>                                                                                                                                                          |
| [**CSV\_QUERY\_REDIRECT\_STATE**](/windows/win32/WinIoCtl/ns-winioctl-_csv_query_redirect_state?branch=master)<br/>                        | Contains information about whether files in a stream have been redirected.<br/>                                                                                                                                                |
| [**DUPLICATE\_EXTENTS\_DATA**](/windows/win32/WinIoCtl/ns-winioctl-_duplicate_extents_data?branch=master)<br/>                             | Contains parameters for the [**FSCTL\_DUPLICATE\_EXTENTS**](/windows/win32/WinIoCtl/?branch=master) control code that performs the [Block Cloning](block-cloning.md) operation.<br/>                                             |
| [**EFS\_CERTIFICATE\_BLOB**](/windows/win32/Winefs/ns-winefs-_certificate_blob?branch=master)<br/>                             | Contains a certificate.<br/>                                                                                                                                                                                                   |
| [**EFS\_HASH\_BLOB**](/windows/win32/Winefs/ns-winefs-_efs_hash_blob?branch=master)<br/>                                           | Contains a certificate hash.<br/>                                                                                                                                                                                              |
| [**ENCRYPTION\_CERTIFICATE**](/windows/win32/Winefs/ns-winefs-_encryption_certificate?branch=master)<br/>                          | Contains a certificate and the SID of its owner.<br/>                                                                                                                                                                          |
| [**ENCRYPTION\_CERTIFICATE\_HASH**](/windows/win32/WinEfs/ns-winefs-_encryption_certificate_hash?branch=master)<br/>               | Contains a certificate hash and display information for the certificate.<br/>                                                                                                                                                  |
| [**ENCRYPTION\_CERTIFICATE\_HASH\_LIST**](/windows/win32/WinEfs/ns-winefs-_encryption_certificate_hash_list?branch=master)<br/>    | Contains a list of certificate hashes.<br/>                                                                                                                                                                                    |
| [**ENCRYPTION\_CERTIFICATE\_LIST**](/windows/win32/WinEfs/ns-winefs-_encryption_certificate_list?branch=master)<br/>               | Contains a list of certificates.<br/>                                                                                                                                                                                          |
| [**EXFAT\_STATISTICS**](/windows/win32/WinIoCtl/ns-winioctl-_exfat_statistics?branch=master)<br/>                                          | Contains statistical information from the exFAT file system.<br/>                                                                                                                                                              |
| [**FAT\_STATISTICS**](/windows/win32/WinIoCtl/ns-winioctl-_fat_statistics?branch=master)<br/>                                          | Contains statistical information from the FAT file system.<br/>                                                                                                                                                                |
| [**FILE\_ALIGNMENT\_INFO**](/windows/win32/WinBase/ns-winbase-_file_alignment_info?branch=master)<br/>                                   | Contains alignment information for a file.<br/>                                                                                                                                                                                |
| [**FILE\_ALLOCATED\_RANGE\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-_file_allocated_range_buffer?branch=master)<br/>              | Indicates a range of bytes in a file.<br/>                                                                                                                                                                                     |
| [**FILE\_ALLOCATION\_INFO**](/windows/win32/WinBase/ns-winbase-_file_allocation_info?branch=master)<br/>                                 | Contains the total number of bytes that should be allocated for a file.<br/>                                                                                                                                                   |
| [**FILE\_ATTRIBUTE\_TAG\_INFO**](/windows/win32/WinBase/ns-winbase-_file_attribute_tag_info?branch=master)<br/>                          | Receives the requested file attribute information. Used for any handles.<br/>                                                                                                                                                  |
| [**FILE\_BASIC\_INFO**](/windows/win32/WinBase/ns-winbase-_file_basic_info?branch=master)<br/>                                           | Contains the basic information for a file. Used for file handles.<br/>                                                                                                                                                         |
| [**FILE\_COMPRESSION\_INFO**](/windows/win32/WinBase/ns-winbase-_file_compression_info?branch=master)<br/>                               | Receives file compression information.<br/>                                                                                                                                                                                    |
| [**FILE\_DISPOSITION\_INFO**](/windows/win32/WinBase/ns-winbase-_file_disposition_info?branch=master)<br/>                               | Indicates whether a file should be deleted. Used for any handles.<br/>                                                                                                                                                         |
| [**FILE\_END\_OF\_FILE\_INFO**](/windows/win32/WinBase/ns-winbase-_file_end_of_file_info?branch=master)<br/>                             | Contains the specified value to which the end of the file should be set. <br/>                                                                                                                                                 |
| [**FILE\_STORAGE\_INFO**](/windows/win32/WinBase/ns-winbase-_file_storage_info?branch=master)<br/>                                       | Contains directory information for a file.<br/>                                                                                                                                                                                |
| [**FILE\_FULL\_DIR\_INFO**](/windows/win32/WinBase/ns-winbase-_file_full_dir_info?branch=master)<br/>                                    | Contains directory information for a file.<br/>                                                                                                                                                                                |
| [**FILE\_ID\_128**](/windows/win32/WinBase/ns-ntifs-_csv_query_file_revision_ecp_context_file_id_128?branch=master)<br/>                                               | Defines a 128-bit file identifier.<br/>                                                                                                                                                                                        |
| [**FILE\_ID\_BOTH\_DIR\_INFO**](/windows/win32/WinBase/ns-winbase-_file_id_both_dir_info?branch=master)<br/>                             | Contains information about files in the specified directory.<br/>                                                                                                                                                              |
| [**FILE\_ID\_DESCRIPTOR**](/windows/win32/WinBase/ns-winbase-file_id_descriptor?branch=master)<br/>                                     | Specifies the type of ID that is being used.<br/>                                                                                                                                                                              |
| [**FILE\_ID\_EXTD\_DIR\_INFO**](/windows/win32/WinBase/ns-winbase-_file_id_extd_dir_info?branch=master)<br/>                             | Contains identification information for a file.<br/>                                                                                                                                                                           |
| [**FILE\_ID\_INFO**](/windows/win32/WinBase/ns-winbase-_file_id_info?branch=master)<br/>                                                 | Contains identification information for a file.<br/>                                                                                                                                                                           |
| [**FILE\_IO\_PRIORITY\_HINT\_INFO**](/windows/win32/WinBase/ns-winbase-_file_io_priority_hint_info?branch=master)<br/>                   | Specifies the priority hint for a file I/O operation.<br/>                                                                                                                                                                     |
| [**FILE\_LEVEL\_TRIM**](/windows/win32/WinIoCtl/ns-winioctl-_file_level_trim?branch=master)<br/>                                           | Used as input to the [**FSCTL\_FILE\_LEVEL\_TRIM**](/windows/win32/WinIoCtl/?branch=master) control code.<br/>                                                                                                                              |
| [**FILE\_LEVEL\_TRIM\_OUTPUT**](/windows/win32/WinIoCtl/ns-winioctl-_file_level_trim_output?branch=master)<br/>                            | Used as output to the [**FSCTL\_FILE\_LEVEL\_TRIM**](/windows/win32/WinIoCtl/?branch=master) control code.<br/>                                                                                                                             |
| [**FILE\_LEVEL\_TRIM\_RANGE**](/windows/win32/WinIoCtl/ns-winioctl-_file_level_trim_range?branch=master)<br/>                              | Specifies a range of a file that is to be trimmed.<br/>                                                                                                                                                                        |
| [**FILE\_MAKE\_COMPATIBLE\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-_file_make_compatible_buffer?branch=master)<br/>                  | Specifies the disc to close the current session for. This control code is used for UDF file systems. This structure is used for input when calling [**FSCTL\_MAKE\_MEDIA\_COMPATIBLE**](/windows/win32/WinIoCtl/?branch=master).<br/> |
| [**FILE\_NAME\_INFO**](/windows/win32/WinBase/ns-winbase-_file_name_info?branch=master)<br/>                                             | Receives the file name.<br/>                                                                                                                                                                                                   |
| [**FILE\_OBJECTID\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-_file_objectid_buffer?branch=master)<br/>                             | Contains an object identifier and user-defined metadata associated with the object identifier.<br/>                                                                                                                            |
| [**FILE\_QUERY\_ON\_DISK\_VOL\_INFO\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-_file_query_on_disk_vol_info_buffer?branch=master)<br/> | Receives the volume information from a call to [**FSCTL\_QUERY\_ON\_DISK\_VOLUME\_INFO**](/windows/win32/WinIoCtl/?branch=master).<br/>                                                                                           |
| [**FILE\_QUERY\_SPARING\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-_file_query_sparing_buffer?branch=master)<br/>                      | Contains defect management properties.<br/>                                                                                                                                                                                    |
| [**FILE\_REMOTE\_PROTOCOL\_INFO**](/windows/win32/WinBase/ns-winbase-_file_remote_protocol_info?branch=master)<br/>               | Contains file remote protocol information.<br/>                                                                                                                                                                                |
| [**FILE\_RENAME\_INFO**](/windows/win32/WinBase/ns-winbase-_file_rename_info?branch=master)<br/>                                         | Contains the name to which the file should be renamed.<br/>                                                                                                                                                                    |
| [**FILE\_SEGMENT\_ELEMENT**](/windows/win32/WinNT/ns-winnt-_file_segment_element?branch=master)<br/>                                 | Union that contains a 64-bit value that points to a page of data.<br/>                                                                                                                                                         |
| [**FILE\_SET\_DEFECT\_MGMT\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-_file_set_defect_mgmt_buffer?branch=master)<br/>                 | Specifies the defect management state to be set.<br/>                                                                                                                                                                          |
| [**FILE\_SET\_SPARSE\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-_file_set_sparse_buffer?branch=master)<br/>                            | Specifies the sparse state to be set.<br/>                                                                                                                                                                                     |
| [**FILE\_STANDARD\_INFO**](/windows/win32/WinBase/ns-winbase-_file_standard_info?branch=master)<br/>                                     | Receives extended information for the file.<br/>                                                                                                                                                                               |
| [**FILE\_STREAM\_INFO**](/windows/win32/WinBase/ns-winbase-_file_stream_info?branch=master)<br/>                                         | Receives file stream information for the specified file.<br/>                                                                                                                                                                  |
| [**FILE\_ZERO\_DATA\_INFORMATION**](/windows/win32/WinIoCtl/ns-winioctl-_file_zero_data_information?branch=master)<br/>                | Contains a range of a file to set to zeros. <br/>                                                                                                                                                                              |
| [**FILESYSTEM\_STATISTICS**](/windows/win32/WinIoCtl/ns-winioctl-_filesystem_statistics?branch=master)<br/>                                | Contains statistical information from the file system.<br/>                                                                                                                                                                    |
| [**FILESYSTEM\_STATISTICS\_EX**](/windows/win32/WinIoCtl/ns-winioctl-_filesystem_statistics_ex?branch=master)<br/>                         | Contains statistical information from the file system.<br/> Support for this structure started with Windows 10.<br/>                                                                                                     |
| [**FIND\_BY\_SID\_DATA**](/windows/win32/WinIoCtl/ns-winioctl-find_by_sid_data?branch=master)<br/>                                        | Contains data for the [**FSCTL\_FIND\_FILES\_BY\_SID**](/windows/win32/WinIoCtl/?branch=master) control code.<br/>                                                                                                                        |
| [**FIND\_BY\_SID\_OUTPUT**](/windows/win32/WinIoCtl/ns-winioctl-find_by_sid_output?branch=master)<br/>                                    | Represents a file name.<br/>                                                                                                                                                                                                   |
| [**NTFS\_FILE\_RECORD\_INPUT\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-ntfs_file_record_input_buffer?branch=master)<br/>         | Contains data for the [**FSCTL\_GET\_NTFS\_FILE\_RECORD**](/windows/win32/WinIoCtl/?branch=master) control code.<br/>                                                                                                                  |
| [**NTFS\_FILE\_RECORD\_OUTPUT\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-ntfs_file_record_output_buffer?branch=master)<br/>       | Receives output data from the [**FSCTL\_GET\_NTFS\_FILE\_RECORD**](/windows/win32/WinIoCtl/?branch=master) control code.<br/>                                                                                                          |
| [**NTFS\_STATISTICS**](/windows/win32/WinIoCtl/ns-winioctl-_ntfs_statistics?branch=master)<br/>                                        | Contains statistical information from the NTFS file system.<br/>                                                                                                                                                               |
| [**NTFS\_STATISTICS\_EX**](/windows/win32/WinIoCtl/ns-winioctl-_ntfs_statistics_ex?branch=master)<br/>                                     | Contains statistical information from the NTFS file system.<br/> Support for this structure started with Windows 10.<br/>                                                                                                |
| [**OFSTRUCT**](/windows/win32/WinBase/ns-winbase-_ofstruct?branch=master)<br/>                                                       | Contains information about a file that the [**OpenFile**](/windows/win32/WinBase/nf-winbase-openfile?branch=master) function opened or attempted to open.<br/>                                                                                                             |
| [**OVERLAPPED\_ENTRY**](/windows/win32/MinWinBase/ns-minwinbase-_overlapped_entry?branch=master)<br/>                                          | Contains the information returned by a call to the [**GetQueuedCompletionStatusEx**](getqueuedcompletionstatusex-func.md) function.<br/>                                                                                      |
| [**REQUEST\_OPLOCK\_INPUT\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-_request_oplock_input_buffer?branch=master)<br/>                  | Contains the information to request an opportunistic lock (oplock) or to acknowledge an oplock break with the [**FSCTL\_REQUEST\_OPLOCK**](/windows/win32/WinIoCtl/?branch=master) control code.<br/>                                        |
| [**REQUEST\_OPLOCK\_OUTPUT\_BUFFER**](/windows/win32/WinIoCtl/ns-winioctl-_request_oplock_output_buffer?branch=master)<br/>                | Contains the opportunistic lock (oplock) information returned by the [**FSCTL\_REQUEST\_OPLOCK**](/windows/win32/WinIoCtl/?branch=master) control code.<br/>                                                                                 |
| [**WIM\_ENTRY\_INFO**](/windows/win32/wofapi/ns-wofapi-_wim_entry_info?branch=master)<br/>                                             | Defines metadata specific to each WIM data source hosted on a volume.<br/>                                                                                                                                                     |
| [**WIM\_EXTERNAL\_FILE\_INFO**](/windows/win32/wofapi/ns-wofapi-_wim_external_file_info?branch=master)<br/>                            | Defines metadata specific to files provided by WOF\_PROVIDER\_WIM.<br/>                                                                                                                                                        |
| [**WIN32\_FILE\_ATTRIBUTE\_DATA**](/windows/win32/fileapi/ns-fileapi-_win32_file_attribute_data?branch=master)<br/>                  | Contains attribute information for a file or directory.<br/>                                                                                                                                                                   |
| [**WIN32\_FIND\_DATA**](/windows/win32/MinWinBase/ns-minwinbase-_win32_find_dataa?branch=master)<br/>                                       | Contains information about the file that is found by the [**FindFirstFile**](/windows/win32/FileAPI/nf-fileapi-findfirstfilea?branch=master), [**FindFirstFileEx**](/windows/win32/FileAPI/nf-fileapi-findfirstfileexa?branch=master), or [**FindNextFile**](/windows/win32/FileAPI/nf-fileapi-findnextfilea?branch=master) function.<br/>                            |
| [**WIN32\_FIND\_STREAM\_DATA**](/windows/win32/fileapi/ns-fileapi-_win32_find_stream_data?branch=master)<br/>                        | Contains information about the stream found by the [**FindFirstStreamW**](/windows/win32/fileapi/nf-fileapi-findfirststreamw?branch=master) or [**FindNextStreamW**](/windows/win32/fileapi/nf-fileapi-findnextstreamw?branch=master) function.<br/>                                                                   |
| [**WOF\_FILE\_COMPRESSION\_INFO\_V1**](/windows/win32/wofapi/ns-wofapi-_wof_file_compression_info_v1?branch=master)<br/>               | Defines metadata specific to files provided by WOF\_PROVIDER\_FILE.<br/>                                                                                                                                                       |
