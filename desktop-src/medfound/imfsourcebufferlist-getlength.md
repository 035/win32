# IMFSourceBufferList::GetLength method

Gets the number of [**IMFSourceBuffer**](imfsourcebuffer.md) objects in the list.

## Syntax


```C++
DWORD GetLength();
```



## Parameters

This method has no parameters.

## Return value

The number of source buffers in the list.
