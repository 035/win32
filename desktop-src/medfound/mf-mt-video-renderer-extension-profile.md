# MF\_MT\_VIDEO\_RENDERER\_EXTENSION\_PROFILE attribute

Contains a string that matches an entry in a UWP app manifest's **VideoRendererExtensionProfiles** list to select which effect to load.

## Data type

**LPCWSTR**
