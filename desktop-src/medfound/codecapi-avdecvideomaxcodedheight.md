# CODECAPI\_AVDecVideoMaxCodedHeight property

Sets the maximum picture height that a video decoder will accept as an input type.

## Data type

**LONG** (**VT\_I4**)

## Property GUID

CODECAPI\_AVDecVideoMaxCodedHeight
