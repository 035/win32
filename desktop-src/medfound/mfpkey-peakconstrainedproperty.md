# MFPKEY\_PEAKCONSTRAINED Property

Specifies whether the encoder is constrained by a peak bit rate. Read-write.

## Constant for IPropertyBag

Available only by using [**IPropertyStore**](https://msdn.microsoft.com/windows/desktop/e995aaa1-d4c9-475f-b1fa-b9123cd5b653).

## Data Type

**VT\_BOOL**

## Default Value

**VARIANT\_FALSE**
