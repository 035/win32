# MFPKEY\_PEAKCONSTRAINED Property

Specifies whether the encoder is constrained by a peak bit rate. Read-write.

## Constant for IPropertyBag

Available only by using [**IPropertyStore**](properties.IPropertyStore).

## Data Type

**VT\_BOOL**

## Default Value

**VARIANT\_FALSE**
