# Direct3D 9 Video Interfaces

Describes the Microsoft Direct3D 9 video interfaces.



| Interface                                                                | Description                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IDirect3D9ExOverlayExtension**](idirect3d9exoverlayextension.md)     | Queries the overlay hardware capabilities of a Direct3D device.                                            |
| [**IDirect3DAuthenticatedChannel9**](idirect3dauthenticatedchannel9.md) | Provides a communication channel to the graphics driver or to the Direct3D runtime.                        |
| [**IDirect3DCryptoSession9**](idirect3dcryptosession9.md)               | Represents a cryptographic session.                                                                        |
| [**IDirect3DDevice9Video**](idirect3ddevice9video.md)                   | Enables an application to use content protection and encryption services implemented by a graphics driver. |
| [**IDirect3DDXVADevice9**](idirect3ddxvadevice9.md)                     | Represents a hardware accelerator for DirectX Video Acceleration (DXVA) 1.0.                               |
| [**IDirect3DVideoDevice9**](idirect3dvideodevice9.md)                   | Enables hardware-accelerated decoding from a Direct3D device, using DXVA version 1.0.                      |
