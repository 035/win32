# OPM Structures

The following structures are used with [Output Protection Manager](output-protection-manager.md) (OPM).



| Structure                                                                                              | Description                                                                                                                                                |
|--------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GRL\_HEADER**](grl-header.md)                                                                      | Contains the global revocation list (GRL) header.                                                                                                          |
| [**MF\_SIGNATURE**](mf-signature.md)                                                                  | Contains a global revocation list (GRL) signature.                                                                                                         |
| [**OPM\_ACP\_AND\_CGMSA\_SIGNALING**](opm-acp-and-cgmsa-signaling.md)                                 | Contains the result from an [**OPM\_GET\_ACP\_AND\_CGMSA\_SIGNALING**](opm-get-acp-and-cgmsa-signaling.md) query.                                         |
| [**OPM\_ACTUAL\_OUTPUT\_FORMAT**](opm-actual-output-format.md)                                        | Contains the result of an [**OPM\_GET\_ACTUAL\_OUTPUT\_FORMAT**](opm-get-actual-output-format.md) query in Output Protection Manager (OPM).               |
| [**OPM\_CONFIGURE\_PARAMETERS**](opm-configure-parameters.md)                                         | Contains an OPM or Certified Output Protection Manager (COPP) command.                                                                                     |
| [**OPM\_CONNECTED\_HDCP\_DEVICE\_INFORMATION**](opm-connected-hdcp-device-information.md)             | Contains the result from an [**OPM\_GET\_CONNECTED\_HDCP\_DEVICE\_INFORMATION**](opm-get-connected-hdcp-device-information.md) query.                     |
| [**OPM\_COPP\_COMPATIBLE\_GET\_INFO\_PARAMETERS**](opm-copp-compatible-get-info-parameters.md)        | Contains parameters for the [**IOPMVideoOutput::COPPCompatibleGetInformation**](iopmvideooutput-iopmvideooutput--coppcompatiblegetinformation.md) method. |
| [**OPM\_ENCRYPTED\_INITIALIZATION\_PARAMETERS**](opm-encrypted-initialization-parameters.md)          | Contains initialization parameters for an OPM session.                                                                                                     |
| [**OPM\_GET\_CODEC\_INFO\_INFORMATION**](opm-get-codec-info-information.md)                           | Contains the result from an [**OPM\_GET\_CODEC\_INFO**](opm-get-codec-info.md) query.                                                                     |
| [**OPM\_GET\_CODEC\_INFO\_PARAMETERS**](opm-get-codec-info-parameters.md)                             | Contains information for the [**OPM\_GET\_CODEC\_INFO**](opm-get-codec-info.md) command.                                                                  |
| [**OPM\_GET\_INFO\_PARAMETERS**](opm-get-info-parameters.md)                                          | Contains parameters for the [**IOPMVideoOutput::GetInformation**](iopmvideooutput-iopmvideooutput--getinformation.md) method.                             |
| [**OPM\_HDCP\_KEY\_SELECTION\_VECTOR**](opm-hdcp-key-selection-vector.md)                             | Contains the key selection vector (KSV) for a High-Bandwidth Digital Content Protection (HDCP) receiver.                                                   |
| [**OPM\_OMAC**](opm-omac.md)                                                                          | Contains a Message Authentication Code (MAC) for an OPM message.                                                                                           |
| [**OPM\_OUTPUT\_ID\_DATA**](opm-output-id-data.md)                                                    | Contains the result from an [**OPM\_GET\_OUTPUT\_ID**](opm-get-output-id.md) status request.                                                              |
| [**OPM\_RANDOM\_NUMBER**](opm-random-number.md)                                                       | Contains a 128-bit random number for use with OPM.                                                                                                         |
| [**OPM\_REQUESTED\_INFORMATION**](opm-requested-information.md)                                       | Contains the result of an OPM status request.                                                                                                              |
| [**OPM\_SET\_ACP\_AND\_CGMSA\_SIGNALING\_PARAMETERS**](opm-set-acp-and-cgmsa-signaling-parameters.md) | Contains information for the [**OPM\_SET\_ACP\_AND\_CGMSA\_SIGNALING**](opm-set-acp-and-cgmsa-signaling.md) command in OPM.                               |
| [**OPM\_SET\_HDCP\_SRM\_PARAMETERS**](opm-set-hdcp-srm-parameters.md)                                 | Contains parameters for the [**OPM\_SET\_HDCP\_SRM**](opm-set-hdcp-srm.md) command.                                                                       |
| [**OPM\_SET\_PROTECTION\_LEVEL\_PARAMETERS**](opm-set-protection-level-parameters.md)                 | Contains data for the [OPM\_SET\_PROTECTION\_LEVEL](opm-set-protection-level.md) command in OPM.                                                          |
| [**OPM\_STANDARD\_INFORMATION**](opm-standard-information.md)                                         | Contains the result from an OPM status request.                                                                                                            |
