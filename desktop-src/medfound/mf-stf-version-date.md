# MF\_STF\_VERSION\_DATE attribute

This attribute stores a [FILETIME](https://msdn.microsoft.com/en-us/library/windows/desktop/ms724284.aspx) representing the date/time stamp of a sensor transform factory.

## Data type

**UINT64**
