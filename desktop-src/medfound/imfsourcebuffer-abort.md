# IMFSourceBuffer::Abort method

Aborts the processing of the current media segment.

## Syntax


```C++
HRESULT Abort();
```



## Parameters

This method has no parameters.

## Return value

If this method succeeds, it returns **S\_OK**. Otherwise, it returns an **HRESULT** error code.
