# MF\_MT\_DV\_AAUX\_CTRL\_PACK\_1 attribute

Audio auxiliary (AAUX) source control pack for the second audio block in a digital video (DV) media type.

## Data type

**UINT32**

## Remarks

This attribute corresponds to the **dwDVAAuxCtl1** member of the DirectShow [**DVINFO**](dshow.dvinfo) structure.

The GUID constant for this attribute is exported from mfuuid.lib.
