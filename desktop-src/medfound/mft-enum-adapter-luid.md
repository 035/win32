# MFT\_ENUM\_ADAPTER\_LUID attribute

Specifies the unique identifier for a video adapter. Use this attribute when calling [**MFTEnum2**](mftenum2.md) to enumerate MFTs associated with a specific adapter.

## Data type

**[**LUID**](coreaudio.luid)**
