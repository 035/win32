# MFSampleExtension\_MULTIPLEXED\_MANAGER attribute

Provides an instance of [**IMFMuxStreamSampleManager**](imfmuxstreamsamplemanager.md) which is used to access the collection of samples from the substreams of a multiplexed media source.

## Data type

**[**IUnknown**](com.iunknown)**

## Remarks

Pass this value into [**IMFAttributes::GetUnknown**](imfattributes-getunknown.md) to get an instance of [**IMFMuxStreamSampleManager**](imfmuxstreamsamplemanager.md).
