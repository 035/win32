# MF\_MT\_DV\_VAUX\_SRC\_PACK attribute

Video auxiliary (VAUX) source pack in a digital video (DV) media type.

## Data type

**UINT32**

## Remarks

This attribute corresponds to the **dwDVVAuxSrc** member of the DirectShow [**DVINFO**](dshow.dvinfo) structure.

The GUID constant for this attribute is exported from mfuuid.lib.
