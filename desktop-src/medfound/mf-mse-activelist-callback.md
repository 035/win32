# MF\_MSE\_ACTIVELIST\_CALLBACK attribute

Contains a pointer to the application's callback interface for the [**IMFBufferListNotify**](imfbufferlistnotify.md) interface for the active buffer list.

## Data type

**IMFBufferListNotify\*** stored as **IUnknown\***
