# MF\_PD\_ASF\_CONTENTENCRYPTION\_LICENSE\_URL attribute

Specifies the license acquisition URL for an encrypted Advanced Systems Format (ASF) file. This attribute corresponds to the License URL field of the Content Encryption Header, defined in the ASF specification.

## Data type

Wide-character string

## Remarks

This attribute applies to presentation descriptors for ASF content.

The [**IMFASFContentInfo::GeneratePresentationDescriptor**](/windows/win32/wmcontainer/nf-wmcontainer-imfasfcontentinfo-generatepresentationdescriptor?branch=master) method retrieves the License URL field, converts it into a wide-character string, and then populates a null-terminated array of **WCHAR**s. The size of the array equals the License URL Length field of the Content Encryption Header.
