# MFPKEY\_DYN\_STEREO\_PREPROC Property

Specifies whether the encoder performs stereo image processing. Read-write.

## Constant for IPropertyBag

Available only by using [**IPropertyStore**](properties.IPropertyStore).

## Data Type

**VT\_BOOL**

## Default Value

**VARIANT\_TRUE**
