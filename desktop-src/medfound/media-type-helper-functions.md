# Media Type Helper Functions

The following functions pertain to media type objects.



| Function                                                                     | Description                                                           |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| [**MFAverageTimePerFrameToFrameRate**](mfaveragetimeperframetoframerate.md) | Calculates the frame rate from the average duration of a video frame. |
| [**MFCalculateImageSize**](mfcalculateimagesize.md)                         | Retrieves the image size for an uncompressed video format.            |
| [**MFCompareFullToPartialMediaType**](mfcomparefulltopartialmediatype.md)   | Compares a full media type to a partial media type.                   |
| [**MFCreateMediaType**](mfcreatemediatype.md)                               | Creates an empty media type.                                          |
| [**MFFrameRateToAverageTimePerFrame**](mfframeratetoaveragetimeperframe.md) | Converts a video frame rate into a frame duration.                    |
| [**MFGetStrideForBitmapInfoHeader**](mfgetstrideforbitmapinfoheader.md)     | Retrieves the minimum surface stride for a video format.              |
| [**MFIsFormatYUV**](mfisformatyuv.md)                                       | Queries whether a FOURCC code or **D3DFORMAT** value is a YUV format. |
| [**MFValidateMediaTypeSize**](mfvalidatemediatypesize.md)                   | Validates the size of a buffer for a video format block.              |
| [**MFWrapMediaType**](mfwrapmediatype.md)                                   | Creates a media type that wraps another media type.                   |
