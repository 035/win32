# MF\_PD\_ASF\_FILEPROPERTIES\_MAX\_BITRATE attribute

Specifies the maximum instantaneous bit rate, in bits per second, for an Advanced Systems Format (ASF) file.

## Data type

**UINT32**

## Remarks

This attribute applies to presentation descriptors for ASF content.

The [**IMFASFContentInfo::GeneratePresentationDescriptor**](imfasfcontentinfo-generatepresentationdescriptor.md) method generates this attribute from the ASF metadata.
