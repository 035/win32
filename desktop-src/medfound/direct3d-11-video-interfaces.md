# Direct3D 11 Video Interfaces

This section contains reference information for the Microsoft Direct3D 11 video interfaces.

## In this section



| Topic                                                                                 | Description                                                                                              |
|---------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**ID3D11AuthenticatedChannel**](/windows/win32/d3d11/nn-d3d11-id3d11authenticatedchannel?branch=master)<br/>           | Provides a communication channel with the graphics driver or the Microsoft Direct3D runtime. <br/> |
| [**ID3D11CryptoSession**](/windows/win32/d3d11/nn-d3d11-id3d11cryptosession?branch=master)<br/>                         | Represents a cryptographic session. <br/>                                                          |
| [**ID3D11VideoContext**](/windows/win32/d3d11/nn-d3d11-id3d11videocontext?branch=master)<br/>                           | Provides the video functionality of a Direct3D 11 device. <br/>                                    |
| [**ID3D11VideoContext1**](/windows/win32/d3d11_1/nn-d3d11_1-id3d11videocontext1?branch=master)<br/>                         | Provides the video functionality of a Direct3D 11 device.<br/>                                     |
| [**ID3D11VideoContext2**](/windows/win32/d3d11/nn-d3d11_4-id3d11videocontext2?branch=master)<br/>                         | Provides the video functionality of a Direct3D 11 device. <br/>                                    |
| [**ID3D11VideoDecoder**](/windows/win32/d3d11/nn-d3d11-id3d11videodecoder?branch=master)<br/>                           | Represents a hardware-accelerated video decoder for Direct3D 11.<br/>                              |
| [**ID3D11VideoDecoderOutputView**](/windows/win32/d3d11/nn-d3d11-id3d11videodecoderoutputview?branch=master)<br/>       | Identifies the output surfaces that can be accessed during video decoding.<br/>                    |
| [**ID3D11VideoDevice**](/windows/win32/d3d11/nn-d3d11-id3d11videodevice?branch=master)<br/>                             | Provides the video decoding and video processing capabilities of a Direct3D 11 device. <br/>       |
| [**ID3D11VideoDevice1**](/windows/win32/d3d11_1/nn-d3d11_1-id3d11videodevice1?branch=master)<br/>                           | Provides the video decoding and video processing capabilities of a Direct3D 11 device. <br/>       |
| [**ID3D11VideoProcessor**](/windows/win32/d3d11/nn-d3d11-id3d11videoprocessor?branch=master)<br/>                       | Represents a video processor for Direct3D 11.<br/>                                                 |
| [**ID3D11VideoProcessorEnumerator**](/windows/win32/d3d11/nn-d3d11-id3d11videoprocessorenumerator?branch=master)<br/>   | Enumerates the video processor capabilities of a Direct3D 11 device. <br/>                         |
| [**ID3D11VideoProcessorEnumerator1**](/windows/win32/d3d11_1/nn-d3d11_1-id3d11videoprocessorenumerator1?branch=master)<br/> | Enumerates the video processor capabilities of a Direct3D 11 device.<br/>                          |
| [**ID3D11VideoProcessorInputView**](/windows/win32/d3d11/nn-d3d11-id3d11videoprocessorinputview?branch=master)<br/>     | Identifies the input surfaces that can be accessed during video processing.<br/>                   |
| [**ID3D11VideoProcessorOutputView**](/windows/win32/d3d11/nn-d3d11-id3d11videoprocessoroutputview?branch=master)<br/>   | Identifies the output surfaces that can be accessed during video processing.<br/>                  |
