# MFPKEY\_RESIZE\_PANSCANAPHEIGHT Property

Specifies the height of the pan/scan region, in pixels.

## Constant for IPropertyBag

Available only by using [**IPropertyStore**](properties.IPropertyStore).

## Data Type

VT\_I4

## Applies To
