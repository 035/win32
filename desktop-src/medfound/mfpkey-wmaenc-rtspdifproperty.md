# MFPKEY\_WMAENC\_RTSPDIF Property

Specifies whether the encoder should enable real-time S/PDIF encoding. Read-write.

## Constant for IPropertyBag

Available only by using [**IPropertyStore**](https://msdn.microsoft.com/e995aaa1-d4c9-475f-b1fa-b9123cd5b653).

## Data Type

**VT\_BOOL**

## Default Value

**VARIANT\_FALSE**
