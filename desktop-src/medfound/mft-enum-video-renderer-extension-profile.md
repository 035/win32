# MFT\_ENUM\_VIDEO\_RENDERER\_EXTENSION\_PROFILE attribute

Contains a list of all **VideoRendererExtensionProfile** entries in the **VideoRendererExtensionProfiles** tag of a UWP app manifest file. [**MFTEnumEx**](/windows/win32/mfapi/nf-mfapi-mftenumex?branch=master) stores this on the attribute store of the [**IMFActivate**](/windows/win32/mfobjects/nn-mfobjects-imfactivate?branch=master) object that **MFTEnumEx** creates for MFTs that have an associated UWP manifest containing the **VideoRendererExtensionProfiles** tag.
