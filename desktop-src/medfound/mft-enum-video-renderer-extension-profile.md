# MFT\_ENUM\_VIDEO\_RENDERER\_EXTENSION\_PROFILE attribute

Contains a list of all **VideoRendererExtensionProfile** entries in the **VideoRendererExtensionProfiles** tag of a UWP app manifest file. [**MFTEnumEx**](mftenumex.md) stores this on the attribute store of the [**IMFActivate**](imfactivate.md) object that **MFTEnumEx** creates for MFTs that have an associated UWP manifest containing the **VideoRendererExtensionProfiles** tag.
