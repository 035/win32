# MF\_MEDIATYPE\_MULTIPLEXED\_MANAGER attribute

Provides an instance of [**IMFMuxStreamMediaTypeManager**](imfmuxstreammediatypemanager.md) which can be used to get the media types of the substreams of a multiplexed media source as well as control the combination of substreams that are multiplexed by the source.

## Data type

**[**IUnknown**](com.iunknown)**

## Remarks

Pass this value into [**IMFAttributes::GetUnknown**](imfattributes-getunknown.md) to get an instance of [**IMFMuxStreamMediaTypeManager**](imfmuxstreammediatypemanager.md).
