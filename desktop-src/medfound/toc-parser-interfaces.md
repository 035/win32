# Table of Contents Parser Interfaces

The following interfaces proved access to the services of Table of Contents Parser.



| Interface                                | Description                                                                                                           |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IToc**](itoc.md)                     | Represents an individual table of contents.                                                                           |
| [**ITocCollection**](itoccollection.md) | Represents a collection of tables of contents.                                                                        |
| [**ITocEntry**](itocentry.md)           | Represents an individual entry in a table of contents.                                                                |
| [**ITocEntryList**](itocentrylist.md)   | Represents a list of entries in a table of contents.                                                                  |
| [**ITocParser**](itocparser.md)         | Provides methods for embedding a tables of contents in a video file and reading tables of contents from a video file. |
