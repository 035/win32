# Table of Contents Parser Interfaces

The following interfaces proved access to the services of Table of Contents Parser.



| Interface                                | Description                                                                                                           |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IToc**](/windows/win32/wmcodecdsp/nn-wmcodecdsp-itoc?branch=master)                     | Represents an individual table of contents.                                                                           |
| [**ITocCollection**](/windows/win32/wmcodecdsp/nn-wmcodecdsp-itoccollection?branch=master) | Represents a collection of tables of contents.                                                                        |
| [**ITocEntry**](/windows/win32/wmcodecdsp/nn-wmcodecdsp-itocentry?branch=master)           | Represents an individual entry in a table of contents.                                                                |
| [**ITocEntryList**](/windows/win32/wmcodecdsp/nn-wmcodecdsp-itocentrylist?branch=master)   | Represents a list of entries in a table of contents.                                                                  |
| [**ITocParser**](/windows/win32/wmcodecdsp/nn-wmcodecdsp-itocparser?branch=master)         | Provides methods for embedding a tables of contents in a video file and reading tables of contents from a video file. |
