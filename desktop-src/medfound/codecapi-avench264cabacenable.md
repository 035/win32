# CODECAPI\_AVEncH264CABACEnable property

Enables or disables CABAC (context-adaptive binary arithmetic coding) for H.264 entropy coding.

## Data type

**VARIANT\_BOOL** (VT\_BOOL)

## Property GUID

**CODECAPI\_AVEncH264CABACEnable**

## Property value

If the value is **VARIANT\_TRUE**, CABAC is enabled.
