# MFPKEY\_MediaProtectionSystemIdMapping property

Specifies a [**IPropertySet**](w_found_coll.ipropertyset) that defines the mapping from a property system id to a property system activation id.



Data type

PROPVARIANT type (vt)

PROPVARIANT member

**IUNKNOWN**

VT\_UNKNOWN

**punkVal**



## Remarks

This property key is used to pass an [**IPropertySet**](w_found_coll.ipropertyset) that contains mappings between protection system GUIDs and runtime class strings to a media source.
