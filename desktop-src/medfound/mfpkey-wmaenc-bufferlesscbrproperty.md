# MFPKEY\_WMAENC\_BUFFERLESSCBR Property

Specifies whether the encoder should produce 1 WMA packet per frame. Read-write.

## Constant for IPropertyBag

Available only by using [**IPropertyStore**](https://msdn.microsoft.com/e995aaa1-d4c9-475f-b1fa-b9123cd5b653).

## Data Type

**VT\_BOOL**

## Default Value

**VARIANT\_FALSE**
