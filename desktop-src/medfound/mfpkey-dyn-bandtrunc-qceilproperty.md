# MFPKEY\_DYN\_BANDTRUNC\_QCEIL Property

Specifies the quality at which minimum coded bandwidth is allowed. At quality worse than or equal to the value of this property, minimum coded bandwidth is allowed.

## Constant for IPropertyBag

Available only by using [**IPropertyStore**](properties.IPropertyStore).

## Data Type

**VT\_R4**
