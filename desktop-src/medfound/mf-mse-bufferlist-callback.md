# MF\_MSE\_BUFFERLIST\_CALLBACK attribute

Contains a pointer to the application's callback interface for the [**IMFBufferListNotify**](imfbufferlistnotify.md).

## Data type

**IMFBufferListNotify\*** stored as **IUnknown\***
