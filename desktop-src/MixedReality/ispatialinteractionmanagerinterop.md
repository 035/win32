# ISpatialInteractionManagerInterop interface

\[ Updated for UWP apps on Windows 10. For Windows 8.x articles, see the [archive](http://go.microsoft.com/fwlink/p/?linkid=619132) \]

Enables interoperability with a Universal Windows Platform (UWP) [SpatialInteractionManager](https://docs.microsoft.com/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager) object and provides access to **SpatialInteractionManager** members for accessing user input from hands, motion controllers, and system voice commands.

## Members

The **ISpatialInteractionManagerInterop** interface does not define any members.
