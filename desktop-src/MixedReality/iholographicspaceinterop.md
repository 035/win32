# IHolographicSpaceInterop interface

\[ Updated for UWP apps on Windows 10. For Windows 8.x articles, see the [archive](http://go.microsoft.com/fwlink/p/?linkid=619132) \]

Enables interoperability with a Univeral Windows Platform (UWP) [HolographicSpace](https://docs.microsoft.com/uwp/api/windows.graphics.holographic.holographicspace) object and provides access to HolographicSpace members for representing a holographic scene.

## Members

The **IHolographicSpaceInterop** interface does not define any members.
