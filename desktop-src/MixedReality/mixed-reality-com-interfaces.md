# Windows Mixed Reality COM Interfaces

\[ Updated for UWP apps on Windows 10. For Windows 8.x articles, see the [archive](http://go.microsoft.com/fwlink/p/?linkid=619132) \]

The topics contained in this section provide the reference specifications for Windows Mixed Reality interfaces.

## In this section



| Topic                                                                                              | Description                                                                                                                                                                                                                                                                                                                                            |
|----------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ISpatialInteractionManagerInterop**](https://www.bing.com/search?q=**ISpatialInteractionManagerInterop**)<br/> | Enables interoperability with a Universal Windows Platform (UWP) [SpatialInteractionManager](https://docs.microsoft.com/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager) object and provides access to **SpatialInteractionManager** members for accessing user input from hands, motion controllers, and system voice commands.<br/> |
| [**IHolographicSpaceInterop**](https://www.bing.com/search?q=**IHolographicSpaceInterop**)<br/>                   | Enables interoperability with a Univeral Windows Platform (UWP) [HolographicSpace](https://docs.microsoft.com/uwp/api/windows.graphics.holographic.holographicspace) object and provides access to HolographicSpace members for representing a holographic scene.<br/>                                                                           |
