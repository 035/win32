# Notification Activation Reference

\[Some information relates to pre-released product which may be substantially modified before it's commercially released. Microsoft makes no warranties, express or implied, with respect to the information provided here.\]

These APIs support activating an app when the user interacts with a toast in the action center in order to launch or interact with an app.

## In this section



| Topic                                                                                 | Description                                                                                                                                                                                         |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**INotificationActivationCallback**](inotificationactivationcallback.md)<br/> | Receives notification messages when an app is triggered through a toast from the action center.<br/>                                                                                          |
| [**NOTIFICATION\_USER\_INPUT\_DATA**](notification-user-input-data.md)<br/>    | Contains information about how a user interacted with a notification toast in the action center. This structure is used by [**Activate**](inotificationactivationcallback-activate.md).<br/> |
