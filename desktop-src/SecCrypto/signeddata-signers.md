# SignedData.Signers property

\[The **Signers** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**SignedCms Class**](T:System.Security.Cryptography.Pkcs.SignedCms) in the [**System.Security.Cryptography.Pkcs**](frlrfSystemSecurityCryptographyPkcs) namespace.\]

The **Signers** property retrieves the signature creators of the data.

## Syntax


```VB
SignedData.Signers As Signers
```



## Property value

The [**Signers**](signers.md) collection that represents the signature creators of the data.
