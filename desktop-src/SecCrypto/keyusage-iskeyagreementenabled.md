# KeyUsage.IsKeyAgreementEnabled property

\[The **IsKeyAgreementEnabled** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**X509EnhancedKeyUsageExtension Class**](T:System.Security.Cryptography.X509Certificates.X509EnhancedKeyUsageExtension) in the [**System.Security.Cryptography.X509Certificates**](frlrfSystemSecurityCryptographyX509Certificates) namespace.\]

The **IsKeyAgreementEnabled** property retrieves a Boolean value that indicates whether the keyAgreement bit is set.

## Syntax


```VB
KeyUsage.IsKeyAgreementEnabled As Boolean
```



## Property value

If **true**, the keyAgreement bit is set.
