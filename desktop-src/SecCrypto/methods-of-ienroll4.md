# Methods of IEnroll4

The following methods are defined by the [**IEnroll4**](ienroll4.md) interface or are methods the **IEnroll4** interface inherits from [**IEnroll**](ienroll.md) and [**IEnroll2**](ienroll2.md). The property access methods are not shown here. To see the properties for **IEnroll4**, see [Properties of IEnroll4](properties-of-ienroll4.md).



| Method                                                                                                | Description                                                                                                                                                                                                                                                                                           |
|-------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**acceptFilePKCS7WStr**](ienroll4-acceptfilepkcs7wstr.md)                                           | Accepts and processes a PKCS \#7 message containing a certificate, then stores the message to a file.                                                                                                                                                                                                 |
| [**acceptFileResponseWStr**](ienroll4-acceptfileresponsewstr.md)                                     | Accepts delivery of the credentials issued in response to an earlier call to [**createFileRequestWStr**](ienroll4-createfilerequestwstr.md) and places the credentials in the appropriate store.                                                                                                     |
| [**acceptPKCS7Blob**](ienroll4-acceptpkcs7blob.md)                                                   | Accepts and processes a PKCS \#7 message containing a certificate. The PKCS \#7 is input as a parameter.                                                                                                                                                                                              |
| [**acceptResponseBlob**](ienroll4-acceptresponseblob.md)                                             | Accepts delivery of the credentials issued in response to an earlier call to [**IEnroll4::createRequestWStr**](ienroll4-createrequestwstr.md) and places the credentials in the appropriate store.                                                                                                   |
| [**addAttributeToRequestWStr**](ienroll4-addattributetorequestwstr.md)                               | Adds an attribute to the certificate request.                                                                                                                                                                                                                                                         |
| [**AddAuthenticatedAttributesToPKCS7Request**](ienroll4-addauthenticatedattributestopkcs7request.md) | Adds authenticated attributes to a PKCS \#7 certificate request.                                                                                                                                                                                                                                      |
| [**addBlobPropertyToCertificateWStr**](ienroll4-addblobpropertytocertificatewstr.md)                 | Adds a BLOB property to a certificate.                                                                                                                                                                                                                                                                |
| [**AddCertTypeToRequestWStr**](ienroll4-addcerttypetorequestwstr.md)                                 | Adds a certificate template to a request (used to support the enterprise certification authority (CA)).                                                                                                                                                                                               |
| [**AddCertTypeToRequestWStrEx**](ienroll4-addcerttypetorequestwstrex.md)                             | Adds a certificate template (or "certificate type") to a request.                                                                                                                                                                                                                                     |
| [**AddExtensionsToRequest**](ienroll4-addextensionstorequest.md)                                     | Adds extensions to the certificate request.                                                                                                                                                                                                                                                           |
| [**addExtensionToRequestWStr**](ienroll4-addextensiontorequestwstr.md)                               | Adds an extension to the request.                                                                                                                                                                                                                                                                     |
| [**addNameValuePairToRequestWStr**](ienroll4-addnamevaluepairtorequestwstr.md)                       | Adds a name-value string pair to the request.                                                                                                                                                                                                                                                         |
| [**AddNameValuePairToSignatureWStr**](ienroll4-addnamevaluepairtosignaturewstr.md)                   | Adds the name and value pair of an attribute to the request. It is up to the CA to interpret the meaning of the name-value pair.                                                                                                                                                                      |
| [**binaryBlobToString**](ienroll4-binaryblobtostring.md)                                             | Converts a binary data BLOB to a string.                                                                                                                                                                                                                                                              |
| [**createFilePFXWStr**](ienroll4-createfilepfxwstr.md)                                               | Saves the accepted certificate chain and private key in a file in PFX format.                                                                                                                                                                                                                         |
| [**createFilePKCS10WStr**](ienroll4-createfilepkcs10wstr.md)                                         | Creates a base64-encoded PKCS \#10 certificate request and saves it in a file.                                                                                                                                                                                                                        |
| [**createFileRequestWStr**](ienroll4-createfilerequestwstr.md)                                       | Creates a PKCS \#10 certificate request, a PKCS \#7 request, or a Full CMC certificate request and stores it in a file.                                                                                                                                                                               |
| [**createPFXWStr**](ienroll4-createpfxwstr.md)                                                       | Saves the accepted certificate chain and private key in a Personal Information Exchange (PFX) format string. The PFX format is also known as PKCS \#12.                                                                                                                                               |
| [**createPKCS10WStr**](ienroll4-createpkcs10wstr.md)                                                 | Creates a base64-encoded PKCS \#10 certificate request.                                                                                                                                                                                                                                               |
| [**CreatePKCS7RequestFromRequest**](ienroll4-createpkcs7requestfromrequest.md)                       | Creates a PKCS \#7 request from an existing certificate request.                                                                                                                                                                                                                                      |
| [**createRequestWStr**](ienroll4-createrequestwstr.md)                                               | Creates a PKCS \#10, PKCS \#7, or Full CMC format certificate request and stores it in a string.                                                                                                                                                                                                      |
| [**EnumAlgs**](ienroll4-enumalgs.md)                                                                 | Retrieves the IDs of cryptographic algorithms in a given algorithm class that are supported by the current CSP.                                                                                                                                                                                       |
| [**enumContainersWStr**](ienroll4-enumcontainerswstr.md)                                             | Retrieves the names of the containers for the CSP specified by the [**ProviderNameWStr**](ienroll4-providernamewstr.md) property.                                                                                                                                                                    |
| [**enumPendingRequestWStr**](ienroll4-enumpendingrequestwstr.md)                                     | Enumerates pending certificate requests and retrieves a specified property from each.                                                                                                                                                                                                                 |
| [**enumProvidersWStr**](ienroll4-enumproviderswstr.md)                                               | Retrieves the names of the available cryptographic service providers (CSPs) specified by the [**ProviderType**](ienroll4-providertype.md) property.                                                                                                                                                  |
| [**freeRequestInfoBlob**](ienroll4-freerequestinfoblob.md)                                           | Deletes a certificate context.                                                                                                                                                                                                                                                                        |
| [**GetAlgNameWStr**](ienroll4-getalgnamewstr.md)                                                     | Retrieves the name of a cryptographic algorithm given its ID. The values retrieved by this method depend on the current CSP.                                                                                                                                                                          |
| [**getCAStore**](ienroll4-getcastore.md)                                                             | Reserved for future use.                                                                                                                                                                                                                                                                              |
| [**getCertContextFromFileResponseWStr**](ienroll4-getcertcontextfromfileresponsewstr.md)             | Retrieves the certificate from a file containing a response from a CA.                                                                                                                                                                                                                                |
| [**getCertContextFromPKCS7**](ienroll4-getcertcontextfrompkcs7.md)                                   | Retrieves the certificate, contained in a PKCS \#7 message, that was issued in response to a PKCS \#10 certificate request.                                                                                                                                                                           |
| [**getCertContextFromResponseBlob**](ienroll4-getcertcontextfromresponseblob.md)                     | Retrieves the certificate from a CA's response.                                                                                                                                                                                                                                                       |
| [**GetKeyLen**](ienroll4-getkeylen.md)                                                               | Retrieves the minimum and maximum key lengths for the signature and exchange keys.                                                                                                                                                                                                                    |
| [**GetKeyLenEx**](ienroll4-getkeylenex.md)                                                           | Retrieves size information for the signature and exchange keys.                                                                                                                                                                                                                                       |
| [**getMyStore**](ienroll4-getmystore.md)                                                             | Reserved for future use.                                                                                                                                                                                                                                                                              |
| [**GetPrivateKeyArchiveCertificate**](ienroll4-getprivatekeyarchivecertificate.md)                   | Retrieves the certificate used to archive the private key.                                                                                                                                                                                                                                            |
| [**getProviderTypeWStr**](ienroll4-getprovidertypewstr.md)                                           | Retrieves the type of the specified CSP.                                                                                                                                                                                                                                                              |
| [**getROOTHStore**](ienroll4-getroothstore.md)                                                       | Reserved for future use.                                                                                                                                                                                                                                                                              |
| [**GetSupportedKeySpec**](ienroll4-getsupportedkeyspec.md)                                           | Retrieves information regarding the CSP's support for signature or exchange keys.                                                                                                                                                                                                                     |
| [**InstallPKCS7Blob**](ienroll4-installpkcs7blob.md)                                                 | Processes a certificate or chain of certificates, placing them into the appropriate certificate stores. This method differs from the [**acceptPKCS7Blob**](ienroll4-acceptpkcs7blob.md) method in that [**InstallPKCS7Blob**](ienroll4-installpkcs7blob.md) does not receive a request certificate. |
| [**InstallPKCS7BlobEx**](ienroll4-installpkcs7blobex.md)                                             | Extends [**InstallPKCS7Blob**](ienroll4-installpkcs7blob.md) by returning the number of certificates actually installed in local stores.                                                                                                                                                             |
| [**removePendingRequestWStr**](ienroll4-removependingrequestwstr.md)                                 | Removes a pending request from the client's request store.                                                                                                                                                                                                                                            |
| [**Reset**](ienroll4-reset.md)                                                                       | Returns the certificate enrollment control object to its initial state.                                                                                                                                                                                                                               |
| [**resetAttributes**](ienroll4-resetattributes.md)                                                   | Removes all attributes from the request.                                                                                                                                                                                                                                                              |
| [**resetExtensions**](ienroll4-resetextensions.md)                                                   | Removes all extensions from the request.                                                                                                                                                                                                                                                              |
| [**SetHStoreCA**](ienroll4-sethstoreca.md)                                                           | Specifies the handle to use for the CA store.                                                                                                                                                                                                                                                         |
| [**SetHStoreMy**](ienroll4-sethstoremy.md)                                                           | Specifies the handle to use for the MY store.                                                                                                                                                                                                                                                         |
| [**SetHStoreRequest**](ienroll4-sethstorerequest.md)                                                 | Specifies the handle to use for the request store.                                                                                                                                                                                                                                                    |
| [**SetHStoreROOT**](ienroll4-sethstoreroot.md)                                                       | Specifies the handle to use for the ROOT store.                                                                                                                                                                                                                                                       |
| [**setPendingRequestInfoWStr**](ienroll4-setpendingrequestinfowstr.md)                               | Sets properties for a pending request.                                                                                                                                                                                                                                                                |
| [**SetPrivateKeyArchiveCertificate**](ienroll4-setprivatekeyarchivecertificate.md)                   | Specifies the certificate used to archive the private key.                                                                                                                                                                                                                                            |
| [**SetSignerCertificate**](ienroll4-setsignercertificate.md)                                         | Specifies the signer's certificate.                                                                                                                                                                                                                                                                   |
| [**stringToBinaryBlob**](ienroll4-stringtobinaryblob.md)                                             | Converts a string to a binary data BLOB.                                                                                                                                                                                                                                                              |
