# Template.OID property

\[The **OID** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**X509Extension Class**](T:System.Security.Cryptography.X509Certificates.X509Extension) in the [**System.Security.Cryptography.X509Certificates**](frlrfSystemSecurityCryptographyX509Certificates) namespace by calling the constructor that takes an OID as a parameter, and then use the OID for Certificate Template to retrieve the certificate extension template.\]

The **OID** property retrieves an [**OID**](oid.md) object that identifies the [**Template**](template.md) object.

This property is read-only.

## Syntax


```VB
Template.OID As OID
```



## Property value

An [**OID**](oid.md) object that identifies the [**Template**](template.md) object.
