# PROV\_DH\_SCHANNEL

The PROV\_DH\_SCHANNEL provider type supports both Diffie-Hellman and [*Schannel*](https://msdn.microsoft.com/3e9d7672-2314-45c8-8178-5a0afcfd0c50) protocols.

## Algorithms Supported

For descriptions of each of these algorithms, see the glossary.



| Purpose      | Supported algorithms                                                                                                                                   |
|--------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| Key Exchange | DH (ephemeral)                                                                                                                                         |
| Signature    | [*DSS*](https://msdn.microsoft.com/d007cbb9-b547-4dc7-bc22-b526f650f7c2)                                                                                            |
| Encryption   | [*DES*](https://msdn.microsoft.com/d007cbb9-b547-4dc7-bc22-b526f650f7c2)[*Triple DES*](https://msdn.microsoft.com/11f2e098-1d1e-473b-90ff-7b86eb923e9f)<br/> |
| Hashing      | [*MD5*](https://msdn.microsoft.com/4c4402e9-7455-4868-978f-3899a8fd86c1)[*SHA*](https://msdn.microsoft.com/3e9d7672-2314-45c8-8178-5a0afcfd0c50)<br/>                      |
