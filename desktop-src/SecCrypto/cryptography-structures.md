| [**CRYPT\_XML\_OBJECT**](crypt-xml-object.md)                                    | Describes an **Object** element in the signature.                                                                                                                                                                                                                                                                                                                         |
| [**CRYPT\_XML\_PROPERTY**](crypt-xml-property.md)                                | Contains information about a CryptXML property.                                                                                                                                                                                                                                                                                                                           |
| [**CRYPT\_XML\_REFERENCE**](crypt-xml-reference.md)                              | Contains information used to populate the **Reference** element.                                                                                                                                                                                                                                                                                                          |
| [**CRYPT\_XML\_REFERENCES**](crypt-xml-references.md)                            | Defines an array of [**CRYPT\_XML\_REFERENCE**](crypt-xml-reference.md) structures.                                                                                                                                                                                                                                                                                      |
| [**CRYPT\_XML\_SIGNATURE**](crypt-xml-signature.md)                              | Contains information used to populate the **Signature** element.                                                                                                                                                                                                                                                                                                          |
| [**CRYPT\_XML\_SIGNED\_INFO**](crypt-xml-signed-info.md)                         | Returns information about the signature validation status, summary status information about a **SignedInfo** element, or summary status information about an array of **Reference** elements.                                                                                                                                                                             |

| [**CRYPT\_XML\_TRANSFORM\_INFO**](crypt-xml-transform-info.md)                   | Contains information that is used when applying the data transform.                                                                                                                                                                                                                                                                                                       |
| [**CRYPT\_XML\_X509DATA**](crypt-xml-x509data.md)                                | Represents the sequence of choices in the **X509Data** element.                                                                                                                                                                                                                                                                                                           |
| [**CRYPT\_XML\_X509DATA\_ITEM**](crypt-xml-x509data-item.md)                     | Represents [*X.509*](security.x_gly#-security-x-509-gly) data that is to be encoded in an X509Data named element.                                                                                                                                                                                                                                     |
