# Signer.AuthenticatedAttributes property

\[The **AuthenticatedAttributes** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**CmsSigner Class**](https://www.bing.com/search?q=**CmsSigner Class**) in the [**System.Security.Cryptography.Pkcs**](https://www.bing.com/search?q=**System.Security.Cryptography.Pkcs**) namespace.\]

The **AuthenticatedAttributes** property retrieves the collection of authenticated attributes.

## Syntax


```VB
Signer.AuthenticatedAttributes As Attributes
```



## Property value

An [**Attributes**](attributes.md) collection that represents the signer's authenticated attributes.
