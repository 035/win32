# Methods of IOCSPCAConfigurationCollection

The following methods are defined by the [**IOCSPCAConfigurationCollection**](iocspcaconfigurationcollection.md) interface. The property access methods are not shown here. To see the properties for **IOCSPCAConfigurationCollection**, see [Properties of IOCSPCAConfigurationCollection](properties-of-iocspcaconfigurationcollection.md)



| Method                                                                                       | Description                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateCAConfiguration**](iocspcaconfigurationcollection-createcaconfiguration-method.md) | Creates a new [*certification authority*](security.c_gly#-security-certification-authority-gly) (CA) configuration and adds it to the configuration set. |
| [**DeleteCAConfiguration**](iocspcaconfigurationcollection-deletecaconfiguration-method.md) | Removes a named CA configuration from the configuration set.                                                                                                                                   |
