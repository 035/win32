# Properties of IOCSPPropertyCollection

The following properties are defined by the [**IOCSPPropertyCollection**](iocsppropertycollection.md) interface.



| Property                                                        | Description                                              |
|-----------------------------------------------------------------|----------------------------------------------------------|
| [**\_NewEnum**](iocsppropertycollection--newenum-method.md)    | Gets an enumerator for a property set.                   |
| [**Count**](iocsppropertycollection-count-method.md)           | Gets the number of properties in a property set.         |
| [**Item**](iocsppropertycollection-item-method.md)             | Gets the property identified by index in a property set. |
| [**ItemByName**](iocsppropertycollection-itembyname-method.md) | Gets the property identified by name in a property set.  |
