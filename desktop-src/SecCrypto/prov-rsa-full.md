# PROV\_RSA\_FULL

The PROV\_RSA\_FULL provider type supports both [*digital signatures*](https://www.bing.com/search?q=*digital signatures*) and data encryption. It is considered a general purpose CSP. The RSA public key algorithm is used for all public key operations.

## Algorithms Supported

For descriptions of each of these algorithms, see the glossary.



| Purpose      | Supported algorithms                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Key Exchange | [*RSA*](https://www.bing.com/search?q=*RSA*)                                                                       |
| Signature    | RSA                                                                                                                               |
| Encryption   | [*RC2*](https://www.bing.com/search?q=*RC2*)[*RC4*](https://www.bing.com/search?q=*RC4*)<br/> |
| Hashing      | [*MD5*](https://www.bing.com/search?q=*MD5*)[*SHA*](https://www.bing.com/search?q=*SHA*)<br/> |



 

## Related Documentation

This provider type is defined by Microsoft and RSA Data Security. It is described in the following documents:

-   RSA Laboratories, Public Key Cryptography Standards, RSA Data Security, November 1993.
