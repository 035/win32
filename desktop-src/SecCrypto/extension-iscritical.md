# Extension.IsCritical property

\[CAPICOM is a 32-bit only component that is available for use in the following operating systems: Windows Server 2008, Windows Vista, and Windows XP. Instead, use the [**X509Extension Class**](T:System.Security.Cryptography.X509Certificates.X509Extension) in the [**System.Security.Cryptography.X509Certificates**](frlrfSystemSecurityCryptographyX509Certificates) namespace.\]

The **IsCritical** property retrieves a Boolean value that indicates whether the extension is marked critical.

## Syntax


```VB
Extension.IsCritical As Boolean
```



## Property value

If **true**, the extension is marked critical.
