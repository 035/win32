# Properties of IOCSPCAConfigurationCollection

The following properties are defined by the [**IOCSPCAConfigurationCollection**](iocspcaconfigurationcollection.md) interface.



| Property                                                               | Description                                                                                                                                                                              |
|------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](iocspcaconfigurationcollection--newenum-method.md)    | Gets an enumerator for the configuration set.                                                                                                                                            |
| [**Count**](iocspcaconfigurationcollection-count-method.md)           | Gets the number of [*certification authority*](security.c_gly#-security-certification-authority-gly) (CA) configurations in the configuration set. |
| [**Item**](iocspcaconfigurationcollection-item-method.md)             | Gets a CA configuration identified by index in the configuration set.                                                                                                                    |
| [**ItemByName**](iocspcaconfigurationcollection-itembyname-method.md) | Gets a CA configuration identified by name in the configuration set.                                                                                                                     |
