# Properties of IOCSPCAConfigurationCollection

The following properties are defined by the [**IOCSPCAConfigurationCollection**](/windows/win32/Certadm/nn-certadm-iocspcaconfigurationcollection?branch=master) interface.



| Property                                                               | Description                                                                                                                                                                              |
|------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/win32/Certadm/nf-certadm-iocspcaconfigurationcollection-get__newenum?branch=master)    | Gets an enumerator for the configuration set.                                                                                                                                            |
| [**Count**](/windows/win32/Certadm/nf-certadm-iocspcaconfigurationcollection-get_count?branch=master)           | Gets the number of [*certification authority*](security.c_gly#-security-certification-authority-gly) (CA) configurations in the configuration set. |
| [**Item**](/windows/win32/Certadm/nf-certadm-iocspcaconfigurationcollection-get_item?branch=master)             | Gets a CA configuration identified by index in the configuration set.                                                                                                                    |
| [**ItemByName**](/windows/win32/Certadm/nf-certadm-iocspcaconfigurationcollection-get_itembyname?branch=master) | Gets a CA configuration identified by name in the configuration set.                                                                                                                     |
