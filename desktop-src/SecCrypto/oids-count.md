# OIDs.Count property

\[The **Count** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**OidCollection Class**](T:System.Security.Cryptography.OidCollection) in the [**System.Security.Cryptography**](frlrfSystemSecurityCryptography) namespace.\]

The **Count** property retrieves the number of OIDs in the collection.

## Syntax


```VB
OIDs.Count As Long
```



## Property value

Number of [**OID**](oid.md) objects in the collection.
