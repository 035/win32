# Properties of ICertSrvSetup

The following properties are defined by the [**ICertSrvSetup**](/windows/win32/Casetup/nn-casetup-icertsrvsetup?branch=master) interface.



| Property                                             | Description                                                                                                                                                                                                  |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAErrorId**](/windows/win32/Casetup/nf-casetup-icertsrvsetup-get_caerrorid?branch=master)         | Gets the ID for additional error information related to a failed [*certification authority*](security.c_gly#-security-certification-authority-gly) (CA) specification. |
| [**CAErrorString**](/windows/win32/Casetup/nf-casetup-icertsrvsetup-get_caerrorstring?branch=master) | Gets the string data for additional error information related to a failed CA specification.                                                                                                                  |
