# OIDs.Item property

\[The **Item** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**OidCollection Class**](T:System.Security.Cryptography.OidCollection) in the [**System.Security.Cryptography**](frlrfSystemSecurityCryptography) namespace.\]

The **Item** property retrieves an [**OID**](oid.md) object from the collection. This is the default property.

## Syntax


```VB
OIDs.Item( _
  ByVal Index _
) As Variant
```



## Property value

The [**OID**](oid.md) object at the specified index, or the **OID** object with the specified dotted value.
