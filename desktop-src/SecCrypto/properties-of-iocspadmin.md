# Properties of IOCSPAdmin

The following properties are defined by the [**IOCSPAdmin**](/windows/win32/certadm/nn-certadm-iocspadmin?branch=master) interface.



| Property                                                                                 | Description                                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OCSPCAConfigurationCollection**](/windows/win32/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection?branch=master) | Gets an instance of an [**OCSPCAConfigurationCollection**](/windows/win32/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection?branch=master) object. This object represents the set of certificates for which a responder service can handle status requests. |
| [**OCSPServiceProperties**](/windows/win32/Certadm/nf-certadm-iocspadmin-get_ocspserviceproperties?branch=master)                 | Gets an instance of a [**OCSPPropertyCollection**](/windows/win32/Certadm/nn-certadm-iocsppropertycollection?branch=master) object. This object represents the attributes of a responder service.                                                                            |
