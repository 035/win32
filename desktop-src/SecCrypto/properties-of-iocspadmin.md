# Properties of IOCSPAdmin

The following properties are defined by the [**IOCSPAdmin**](iocspadmin.md) interface.



| Property                                                                                 | Description                                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OCSPCAConfigurationCollection**](iocspadmin-ocspcaconfigurationcollection-method.md) | Gets an instance of an [**OCSPCAConfigurationCollection**](iocspadmin-ocspcaconfigurationcollection-method.md) object. This object represents the set of certificates for which a responder service can handle status requests. |
| [**OCSPServiceProperties**](iocspadmin-ocspserviceproperties-method.md)                 | Gets an instance of a [**OCSPPropertyCollection**](iocsppropertycollection.md) object. This object represents the attributes of a responder service.                                                                            |
