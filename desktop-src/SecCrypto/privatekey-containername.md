# PrivateKey.ContainerName property

\[The **ContainerName** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**X509Certificate2.PrivateKey Property**](https://www.bing.com/search?q=**X509Certificate2.PrivateKey+Property**) in the [**System.Security.Cryptography.X509Certificates**](https://www.bing.com/search?q=**System.Security.Cryptography.X509Certificates**) namespace.\]

The **ContainerName** property retrieves the private key container name. This is the default property.

## Syntax


```VB
PrivateKey.ContainerName As String
```



## Property value

A string that contains the private key container name.
