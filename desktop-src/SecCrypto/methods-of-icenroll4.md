# Methods of ICEnroll4

The following methods are defined by the [**ICEnroll4**](icenroll4.md) interface or are methods that the **ICEnroll4** interface inherits from [**ICEnroll**](icenroll.md), [**ICEnroll2**](icenroll2.md), and [**ICEnroll3**](icenroll3.md). The property access methods are not shown here. To see the properties for **ICEnroll4**, see [Properties of ICEnroll4](properties-of-icenroll4.md).



| Method                                                                         | Description                                                                                                                                                                                                                                                                                                                                                                            |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**acceptFilePKCS7**](icenroll4-acceptfilepkcs7.md)                           | Accepts and processes a file that contains a PKCS \#7 message containing a certificate.<br/>                                                                                                                                                                                                                                                                                     |
| [**acceptFileResponse**](icenroll4-acceptfileresponse.md)                     | Accepts delivery of the credentials issued in response to an earlier call to [**createFileRequest**](icenroll4-createfilerequest.md), and it places the credentials in the appropriate store.<br/>                                                                                                                                                                              |
| [**acceptPKCS7**](icenroll4-acceptpkcs7.md)                                   | Accepts and processes a PKCS \#7 message containing a certificate. The PKCS \#7 is input as a parameter.<br/>                                                                                                                                                                                                                                                                    |
| [**acceptResponse**](icenroll4-acceptresponse.md)                             | Accepts delivery of the credentials issued in response to an earlier call to [**createRequest**](icenroll4-createrequest.md) and places the credentials in the appropriate store.<br/>                                                                                                                                                                                          |
| [**addAttributeToRequest**](icenroll4-addattributetorequest.md)               | Adds an [*attribute*](security.a_gly#-security-attribute-gly) to the [*certificate request*](security.c_gly#-security-certificate-request-gly).<br/>                                                                                                                                                                   |
| [**addBlobPropertyToCertificate**](icenroll4-addblobpropertytocertificate.md) | Adds a [*BLOB*](security.b_gly#-security-blob-gly) property to a certificate.<br/>                                                                                                                                                                                                                                                                            |
| [**addCertTypeToRequest**](icenroll4-addcerttypetorequest.md)                 | Adds a certificate template to a request (used to support the enterprise [*certification authority*](security.c_gly#-security-certification-authority-gly) (CA)).<br/>                                                                                                                                                                     |
| [**addCertTypeToRequestEx**](icenroll4-addcerttypetorequestex.md)             | Adds a certificate template (or "certificate type") to a request.<br/>                                                                                                                                                                                                                                                                                                           |
| [**addExtensionToRequest**](icenroll4-addextensiontorequest.md)               | Adds an extension to the request.<br/>                                                                                                                                                                                                                                                                                                                                           |
| [**addNameValuePairToRequest**](icenroll4-addnamevaluepairtorequest.md)       | Adds a name-value string pair to the request.<br/>                                                                                                                                                                                                                                                                                                                               |
| [**addNameValuePairToSignature**](icenroll4-addnamevaluepairtosignature.md)   | Adds the name and value pair of an [*attribute*](security.a_gly#-security-attribute-gly) to the request. It is up to the CA to interpret the meaning of the name-value pair.<br/>                                                                                                                                                                        |
| [**binaryToString**](icenroll4-binarytostring.md)                             | Converts a binary data [*BLOB*](security.b_gly#-security-blob-gly) to a string.<br/>                                                                                                                                                                                                                                                                          |
| [**createFilePFX**](icenroll4-createfilepfx.md)                               | Saves the accepted certificate chain and [*private key*](security.p_gly#-security-private-key-gly) in a file in Personal Information Exchange (PFX) format.<br/>                                                                                                                                                                                       |
| [**createFilePKCS10**](icenroll4-createfilepkcs10.md)                         | Creates a base64-encoded PKCS \#10 [*certificate request*](security.c_gly) and saves it in a file.<br/>                                                                                                                                                                                                      |
| [**createFileRequest**](icenroll4-createfilerequest.md)                       | Creates a PKCS \#10 certificate request, a PKCS \) (CMC) request and stores it in a file.<br/>                                                                                                               |
| [**createPFX**](icenroll4-createpfx.md)                                       | Saves the accepted certificate chain and private key in a Personal Information Exchange (PFX) format string. The PFX format is also known as PKCS \#12.<br/>                                                                                                                                                                                                                     |
| [**createPKCS10**](icenroll4-createpkcs10.md)                                 | Creates a base64-encoded PKCS \#10 certificate request.<br/>                                                                                                                                                                                                                                                                                                                     |
| [**createRequest**](icenroll4-createrequest.md)                               | Creates a PKCS \#10, PKCS \) (CMC) format [*certificate request*](security.c_gly#-security-certificate-request-gly) and stores it in a string.<br/>                                  |
| [**EnumAlgs**](icenroll4-enumalgs.md)                                         | Retrieves the IDs of cryptographic algorithms in a given algorithm class that are supported by the current CSP.<br/>                                                                                                                                                                                                                                                             |
| [**enumContainers**](icenroll4-enumcontainers.md)                             | Retrieves the names of the containers for the [*cryptographic service provider*](security.c_gly#-security-cryptographic-service-provider-gly) (CSP) specified by the [**ProviderName**](icenroll4-providername.md) property.<br/>                                                                                                  |
| [**enumPendingRequest**](icenroll4-enumpendingrequest.md)                     | Enumerates pending [*certificate requests*](security.c_gly#-security-certificate-request-gly) and retrieves a specified property from each.<br/>                                                                                                                                                                                               |
| [**enumProviders**](icenroll4-enumproviders.md)                               | Retrieves the names of the available CSPs specified by the [**ProviderType**](icenroll4-providertype.md) property.<br/>                                                                                                                                                                                                                                                         |
| [**freeRequestInfo**](icenroll4-freerequestinfo.md)                           | Cleans up the stores if an error occurs. Currently not implemented.<br/>                                                                                                                                                                                                                                                                                                         |
| [**GetAlgName**](icenroll4-getalgname.md)                                     | Retrieves the name of a cryptographic algorithm given its ID. The values retrieved by this method depend on the current CSP.<br/>                                                                                                                                                                                                                                                |
| [**getCertFromFileResponse**](icenroll4-getcertfromfileresponse.md)           | Retrieves the certificate from a file containing a response from a [*certification authority*](security.c_gly#-security-certification-authority-gly).<br/>                                                                                                                                                                                 |
| [**getCertFromPKCS7**](icenroll4-getcertfrompkcs7.md)                         | Retrieves the certificate, contained in a PKCS \#7 message, that was issued in response to a PKCS \#10 certificate request.<br/>                                                                                                                                                                                                                                                 |
| [**getCertFromResponse**](icenroll4-getcertfromresponse.md)                   | Retrieves the certificate from a [*certification authority's*](security.c_gly#-security-certification-authority-gly) response.<br/>                                                                                                                                                                                                        |
| [**GetKeyLen**](icenroll4-getkeylen.md)                                       | Retrieves the minimum and maximum key lengths for the [*signature*](security.s_gly#-security-signature-key-pair-gly) and [*exchange keys*](security.e_gly#-security-exchange-key-pair-gly).<br/>                                                                                                                |
| [**GetKeyLenEx**](icenroll4-getkeylenex.md)                                   | Retrieves size information for the [*signature*](security.s_gly#-security-signature-key-pair-gly) and [*exchange keys*](security.e_gly#-security-exchange-key-pair-gly).<br/>                                                                                                                                   |
| [**GetSupportedKeySpec**](icenroll4-getsupportedkeyspec.md)                   | Retrieves information regarding the CSP's support for signature or exchange keys.<br/>                                                                                                                                                                                                                                                                                           |
| [**InstallPKCS7**](icenroll4-installpkcs7.md)                                 | Processes a certificate or chain of certificates, placing them into the appropriate [*certificate stores*](security.c_gly#-security-certificate-store-gly). This method differs from the [**acceptPKCS7**](icenroll4-acceptpkcs7.md) method in that [**InstallPKCS7**](icenroll4-installpkcs7.md) does not receive a request certificate.<br/> |
| [**InstallPKCS7Ex**](icenroll4-installpkcs7ex.md)                             | The same as [**InstallPKCS7**](icenroll4-installpkcs7.md) except that it returns the number of certificates actually installed in local stores.<br/>                                                                                                                                                                                                                            |
| [**removePendingRequest**](icenroll4-removependingrequest.md)                 | Removes a pending request from the client's request store.<br/>                                                                                                                                                                                                                                                                                                                  |
| [**Reset**](icenroll4-reset.md)                                               | Returns the certificate enrollment control object to its initial [*state*](security.s_gly#-security-state-gly).<br/>                                                                                                                                                                                                                                         |
| [**resetAttributes**](icenroll4-resetattributes.md)                           | Removes all attributes from the request.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**resetBlobProperties**](icenroll4-resetblobproperties.md)                   | Resets the properties of a [*BLOB*](security.b_gly#-security-blob-gly).<br/>                                                                                                                                                                                                                                                                                  |
| [**resetExtensions**](icenroll4-resetextensions.md)                           | Removes all extensions from the request.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**setPendingRequestInfo**](icenroll4-setpendingrequestinfo.md)               | Sets properties for a pending request.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**stringToBinary**](icenroll4-stringtobinary.md)                             | Converts an encoded string to a binary data [*BLOB*](security.b_gly#-security-blob-gly).<br/>                                                                                                                                                                                                                                                                 |
