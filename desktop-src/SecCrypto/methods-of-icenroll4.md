# Methods of ICEnroll4

The following methods are defined by the [**ICEnroll4**](/windows/win32/Xenroll/nn-xenroll-icenroll4?branch=master) interface or are methods that the **ICEnroll4** interface inherits from [**ICEnroll**](/windows/win32/Xenroll/nn-xenroll-icenroll?branch=master), [**ICEnroll2**](/windows/win32/Xenroll/nn-xenroll-icenroll2?branch=master), and [**ICEnroll3**](/windows/win32/Xenroll/nn-xenroll-icenroll3?branch=master). The property access methods are not shown here. To see the properties for **ICEnroll4**, see [Properties of ICEnroll4](properties-of-icenroll4.md).



| Method                                                                         | Description                                                                                                                                                                                                                                                                                                                                                                            |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**acceptFilePKCS7**](/windows/win32/Xenroll/nf-xenroll-icenroll-acceptfilepkcs7?branch=master)                           | Accepts and processes a file that contains a PKCS \#7 message containing a certificate.<br/>                                                                                                                                                                                                                                                                                     |
| [**acceptFileResponse**](/windows/win32/Xenroll/nf-xenroll-icenroll4-acceptfileresponse?branch=master)                     | Accepts delivery of the credentials issued in response to an earlier call to [**createFileRequest**](/windows/win32/Xenroll/nf-xenroll-icenroll4-createfilerequest?branch=master), and it places the credentials in the appropriate store.<br/>                                                                                                                                                                              |
| [**acceptPKCS7**](/windows/win32/Xenroll/nf-xenroll-icenroll-acceptpkcs7?branch=master)                                   | Accepts and processes a PKCS \#7 message containing a certificate. The PKCS \#7 is input as a parameter.<br/>                                                                                                                                                                                                                                                                    |
| [**acceptResponse**](/windows/win32/Xenroll/nf-xenroll-icenroll4-acceptresponse?branch=master)                             | Accepts delivery of the credentials issued in response to an earlier call to [**createRequest**](/windows/win32/Xenroll/nf-xenroll-icenroll4-createrequest?branch=master) and places the credentials in the appropriate store.<br/>                                                                                                                                                                                          |
| [**addAttributeToRequest**](/windows/win32/Xenroll/nf-xenroll-icenroll4-addattributetorequest?branch=master)               | Adds an [*attribute*](security.a_gly#-security-attribute-gly) to the [*certificate request*](security.c_gly#-security-certificate-request-gly).<br/>                                                                                                                                                                   |
| [**addBlobPropertyToCertificate**](/windows/win32/Xenroll/nf-xenroll-icenroll4-addblobpropertytocertificate?branch=master) | Adds a [*BLOB*](security.b_gly#-security-blob-gly) property to a certificate.<br/>                                                                                                                                                                                                                                                                            |
| [**addCertTypeToRequest**](/windows/win32/Xenroll/nf-xenroll-icenroll2-addcerttypetorequest?branch=master)                 | Adds a certificate template to a request (used to support the enterprise [*certification authority*](security.c_gly#-security-certification-authority-gly) (CA)).<br/>                                                                                                                                                                     |
| [**addCertTypeToRequestEx**](/windows/win32/Xenroll/nf-xenroll-icenroll4-addcerttypetorequestex?branch=master)             | Adds a certificate template (or "certificate type") to a request.<br/>                                                                                                                                                                                                                                                                                                           |
| [**addExtensionToRequest**](/windows/win32/Xenroll/nf-xenroll-icenroll4-addextensiontorequest?branch=master)               | Adds an extension to the request.<br/>                                                                                                                                                                                                                                                                                                                                           |
| [**addNameValuePairToRequest**](/windows/win32/Xenroll/nf-xenroll-icenroll4-addnamevaluepairtorequest?branch=master)       | Adds a name-value string pair to the request.<br/>                                                                                                                                                                                                                                                                                                                               |
| [**addNameValuePairToSignature**](/windows/win32/Xenroll/nf-xenroll-icenroll2-addnamevaluepairtosignature?branch=master)   | Adds the name and value pair of an [*attribute*](security.a_gly#-security-attribute-gly) to the request. It is up to the CA to interpret the meaning of the name-value pair.<br/>                                                                                                                                                                        |
| [**binaryToString**](/windows/win32/Xenroll/nf-xenroll-icenroll4-binarytostring?branch=master)                             | Converts a binary data [*BLOB*](security.b_gly#-security-blob-gly) to a string.<br/>                                                                                                                                                                                                                                                                          |
| [**createFilePFX**](/windows/win32/Xenroll/nf-xenroll-icenroll4-createfilepfx?branch=master)                               | Saves the accepted certificate chain and [*private key*](security.p_gly#-security-private-key-gly) in a file in Personal Information Exchange (PFX) format.<br/>                                                                                                                                                                                       |
| [**createFilePKCS10**](/windows/win32/Xenroll/nf-xenroll-icenroll-createfilepkcs10?branch=master)                         | Creates a base64-encoded PKCS \#10 [*certificate request*](security.c_gly) and saves it in a file.<br/>                                                                                                                                                                                                      |
| [**createFileRequest**](/windows/win32/Xenroll/nf-xenroll-icenroll4-createfilerequest?branch=master)                       | Creates a PKCS \#10 certificate request, a PKCS \) (CMC) request and stores it in a file.<br/>                                                                                                               |
| [**createPFX**](/windows/win32/Xenroll/nf-xenroll-icenroll4-createpfx?branch=master)                                       | Saves the accepted certificate chain and private key in a Personal Information Exchange (PFX) format string. The PFX format is also known as PKCS \#12.<br/>                                                                                                                                                                                                                     |
| [**createPKCS10**](/windows/win32/Xenroll/nf-xenroll-icenroll-createpkcs10?branch=master)                                 | Creates a base64-encoded PKCS \#10 certificate request.<br/>                                                                                                                                                                                                                                                                                                                     |
| [**createRequest**](/windows/win32/Xenroll/nf-xenroll-icenroll4-createrequest?branch=master)                               | Creates a PKCS \#10, PKCS \) (CMC) format [*certificate request*](security.c_gly#-security-certificate-request-gly) and stores it in a string.<br/>                                  |
| [**EnumAlgs**](/windows/win32/Xenroll/nf-xenroll-icenroll3-enumalgs?branch=master)                                         | Retrieves the IDs of cryptographic algorithms in a given algorithm class that are supported by the current CSP.<br/>                                                                                                                                                                                                                                                             |
| [**enumContainers**](/windows/win32/Xenroll/nf-xenroll-icenroll-enumcontainers?branch=master)                             | Retrieves the names of the containers for the [*cryptographic service provider*](security.c_gly#-security-cryptographic-service-provider-gly) (CSP) specified by the [**ProviderName**](/windows/win32/Xenroll/?branch=master) property.<br/>                                                                                                  |
| [**enumPendingRequest**](/windows/win32/Xenroll/nf-xenroll-icenroll4-enumpendingrequest?branch=master)                     | Enumerates pending [*certificate requests*](security.c_gly#-security-certificate-request-gly) and retrieves a specified property from each.<br/>                                                                                                                                                                                               |
| [**enumProviders**](/windows/win32/Xenroll/nf-xenroll-icenroll-enumproviders?branch=master)                               | Retrieves the names of the available CSPs specified by the [**ProviderType**](/windows/win32/Xenroll/?branch=master) property.<br/>                                                                                                                                                                                                                                                         |
| [**freeRequestInfo**](/windows/win32/Xenroll/nf-xenroll-icenroll-freerequestinfo?branch=master)                           | Cleans up the stores if an error occurs. Currently not implemented.<br/>                                                                                                                                                                                                                                                                                                         |
| [**GetAlgName**](/windows/win32/Xenroll/nf-xenroll-icenroll3-getalgname?branch=master)                                     | Retrieves the name of a cryptographic algorithm given its ID. The values retrieved by this method depend on the current CSP.<br/>                                                                                                                                                                                                                                                |
| [**getCertFromFileResponse**](/windows/win32/Xenroll/nf-xenroll-icenroll4-getcertfromfileresponse?branch=master)           | Retrieves the certificate from a file containing a response from a [*certification authority*](security.c_gly#-security-certification-authority-gly).<br/>                                                                                                                                                                                 |
| [**getCertFromPKCS7**](/windows/win32/Xenroll/nf-xenroll-icenroll-getcertfrompkcs7?branch=master)                         | Retrieves the certificate, contained in a PKCS \#7 message, that was issued in response to a PKCS \#10 certificate request.<br/>                                                                                                                                                                                                                                                 |
| [**getCertFromResponse**](/windows/win32/Xenroll/nf-xenroll-icenroll4-getcertfromresponse?branch=master)                   | Retrieves the certificate from a [*certification authority's*](security.c_gly#-security-certification-authority-gly) response.<br/>                                                                                                                                                                                                        |
| [**GetKeyLen**](/windows/win32/Xenroll/nf-xenroll-icenroll3-getkeylen?branch=master)                                       | Retrieves the minimum and maximum key lengths for the [*signature*](security.s_gly#-security-signature-key-pair-gly) and [*exchange keys*](security.e_gly#-security-exchange-key-pair-gly).<br/>                                                                                                                |
| [**GetKeyLenEx**](/windows/win32/Xenroll/nf-xenroll-icenroll4-getkeylenex?branch=master)                                   | Retrieves size information for the [*signature*](security.s_gly#-security-signature-key-pair-gly) and [*exchange keys*](security.e_gly#-security-exchange-key-pair-gly).<br/>                                                                                                                                   |
| [**GetSupportedKeySpec**](/windows/win32/Xenroll/nf-xenroll-icenroll3-getsupportedkeyspec?branch=master)                   | Retrieves information regarding the CSP's support for signature or exchange keys.<br/>                                                                                                                                                                                                                                                                                           |
| [**InstallPKCS7**](/windows/win32/Xenroll/nf-xenroll-icenroll3-installpkcs7?branch=master)                                 | Processes a certificate or chain of certificates, placing them into the appropriate [*certificate stores*](security.c_gly#-security-certificate-store-gly). This method differs from the [**acceptPKCS7**](/windows/win32/Xenroll/nf-xenroll-icenroll-acceptpkcs7?branch=master) method in that [**InstallPKCS7**](/windows/win32/Xenroll/nf-xenroll-icenroll3-installpkcs7?branch=master) does not receive a request certificate.<br/> |
| [**InstallPKCS7Ex**](/windows/win32/Xenroll/nf-xenroll-icenroll4-installpkcs7ex?branch=master)                             | The same as [**InstallPKCS7**](/windows/win32/Xenroll/nf-xenroll-icenroll3-installpkcs7?branch=master) except that it returns the number of certificates actually installed in local stores.<br/>                                                                                                                                                                                                                            |
| [**removePendingRequest**](/windows/win32/Xenroll/nf-xenroll-icenroll4-removependingrequest?branch=master)                 | Removes a pending request from the client's request store.<br/>                                                                                                                                                                                                                                                                                                                  |
| [**Reset**](/windows/win32/Xenroll/nf-xenroll-icenroll3-reset?branch=master)                                               | Returns the certificate enrollment control object to its initial [*state*](security.s_gly#-security-state-gly).<br/>                                                                                                                                                                                                                                         |
| [**resetAttributes**](/windows/win32/Xenroll/nf-xenroll-icenroll4-resetattributes?branch=master)                           | Removes all attributes from the request.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**resetBlobProperties**](/windows/win32/Xenroll/nf-xenroll-icenroll4-resetblobproperties?branch=master)                   | Resets the properties of a [*BLOB*](security.b_gly#-security-blob-gly).<br/>                                                                                                                                                                                                                                                                                  |
| [**resetExtensions**](/windows/win32/Xenroll/nf-xenroll-icenroll4-resetextensions?branch=master)                           | Removes all extensions from the request.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**setPendingRequestInfo**](/windows/win32/Xenroll/nf-xenroll-icenroll4-setpendingrequestinfo?branch=master)               | Sets properties for a pending request.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**stringToBinary**](/windows/win32/Xenroll/nf-xenroll-icenroll4-stringtobinary?branch=master)                             | Converts an encoded string to a binary data [*BLOB*](security.b_gly#-security-blob-gly).<br/>                                                                                                                                                                                                                                                                 |
