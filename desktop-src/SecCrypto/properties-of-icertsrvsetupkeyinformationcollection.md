# Properties of ICertSrvSetupKeyInformationCollection

The following properties are defined by the [**ICertSrvSetupKeyInformationCollection**](/windows/win32/Casetup/nn-casetup-icertsrvsetupkeyinformationcollection?branch=master) interface.



| Property                                                            | Description                                                                                                              |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/win32/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get__newenum?branch=master) | Gets an enumerator for the collection.                                                                                   |
| [**Count**](/windows/win32/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_count?branch=master)        | Gets the number of [**ICertSrvSetupKeyInformation**](/windows/win32/Casetup/nn-casetup-icertsrvsetupkeyinformation?branch=master) objects in the collection.         |
| [**Item**](/windows/win32/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_item?branch=master)          | Gets an [**ICertSrvSetupKeyInformation**](/windows/win32/Casetup/nn-casetup-icertsrvsetupkeyinformation?branch=master) object identified by index in the collection. |
