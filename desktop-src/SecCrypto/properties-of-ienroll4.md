# Properties of IEnroll4

The following properties are defined by the [**IEnroll4**](ienroll4.md) interface or are properties the **IEnroll4** interface inherits from [**IEnroll**](ienroll.md) and [**IEnroll2**](ienroll2.md).



| Property                                                                              | Description                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAStoreFlags**](ienroll4-castoreflags.md)                                         | Sets or retrieves a flag that controls the certificate store when it is opened.                                                                                                                                                                                                                                         |
| [**CAStoreNameWStr**](ienroll4-castorenamewstr.md)                                   | Sets or retrieves the name of the store where all non-"ROOT" and non-"MY" certificates are kept.                                                                                                                                                                                                                        |
| [**CAStoreTypeWStr**](ienroll4-castoretypewstr.md)                                   | Sets or retrieves the type of store to use for the store specified by the [**CAStoreNameWStr**](ienroll4-castorenamewstr.md) property.                                                                                                                                                                                 |
| [**ClientId**](ienroll4-clientid.md)                                                 | Sets or retrieves the client ID request attribute.                                                                                                                                                                                                                                                                      |
| [**ContainerNameWStr**](ienroll4-containernamewstr.md)                               | Sets or retrieves the name of the [*key container*](security.k_gly#-security-key-container-gly) to use.                                                                                                                                                                                     |
| [**DeleteRequestCert**](ienroll4-deleterequestcert.md)                               | Sets or retrieves a Boolean indicator that controls whether dummy certificates in the request store are deleted.                                                                                                                                                                                                        |
| [**EnableSMIMECapabilities**](ienroll4-enablesmimecapabilities.md)                   | Sets or retrieves a Boolean value that controls whether the PKCS \#10 will contain a signed attribute for [*Secure/Multipurpose Internet Mail Extensions*](security.s_gly) (S/MIME) capabilities. |
| [**EnableT61DNEncoding**](ienroll4-enablet61dnencoding.md)                           | Sets or retrieves a Boolean value that controls whether the distinguished name in the request is encoded as a T61 string instead of as a [*UNICODE*](security.u_gly#-security-unicode-gly) string.                                                                                                |
| [**GenKeyFlags**](ienroll4-genkeyflags.md)                                           | Sets or retrieves the values passed to [**CryptGenKey**](cryptgenkey.md) when the certificate request is generated.                                                                                                                                                                                                    |
| [**HashAlgID**](ienroll4-hashalgid.md)                                               | Sets or retrieves the hash algorithm used when signing a PKCS \#10 certificate request.                                                                                                                                                                                                                                 |
| [**HashAlgorithmWStr**](ienroll4-hashalgorithmwstr.md)                               | Sets or retrieves only the signature hash algorithm used to sign the PKCS \#10.                                                                                                                                                                                                                                         |
| [**IncludeSubjectKeyID**](ienroll4-includesubjectkeyid.md)                           | Specifies whether the subject key ID extension is added to the certificate request.                                                                                                                                                                                                                                     |
| [**KeySpec**](ienroll4-keyspec.md)                                                   | Sets or retrieves the type of key generated.                                                                                                                                                                                                                                                                            |
| [**LimitExchangeKeyToEncipherment**](ienroll4-limitexchangekeytoencipherment.md)     | Sets or retrieves a Boolean value that controls whether an AT\_KEYEXCHANGE request contains [*digital signature*](security.d_gly#-security-digital-signature-gly) and non-repudiation key usages.                                                                                       |
| [**MyStoreFlags**](ienroll4-mystoreflags.md)                                         | Sets or retrieves the registry location used for the MY store.                                                                                                                                                                                                                                                          |
| [**MyStoreNameWStr**](ienroll4-mystorenamewstr.md)                                   | Sets or retrieves the name of the store where certificates with linked private keys are kept.                                                                                                                                                                                                                           |
| [**MyStoreTypeWStr**](ienroll4-mystoretypewstr.md)                                   | Sets or retrieves the type of store specified by the [**MyStoreNameWStr**](ienroll4-mystorenamewstr.md) property.                                                                                                                                                                                                      |
| [**ProviderFlags**](ienroll4-providerflags.md)                                       | Sets or retrieves the CSP type.                                                                                                                                                                                                                                                                                         |
| [**ProviderNameWStr**](ienroll4-providernamewstr.md)                                 | Sets or retrieves the name of the CSP to use.                                                                                                                                                                                                                                                                           |
| [**ProviderType**](ienroll4-providertype.md)                                         | Sets or retrieves the type of provider.                                                                                                                                                                                                                                                                                 |
| [**PVKFileNameWStr**](ienroll4-pvkfilenamewstr.md)                                   | Sets or retrieves the name of the file that will contain exported keys.                                                                                                                                                                                                                                                 |
| [**RenewalCertificate**](ienroll4-renewalcertificate.md)                             | Specifies the certificate context for the renewal certificate.                                                                                                                                                                                                                                                          |
| [**RequestStoreFlags**](ienroll4-requeststoreflags.md)                               | Sets or retrieves the registry location used for the REQUEST store.                                                                                                                                                                                                                                                     |
| [**RequestStoreNameWStr**](ienroll4-requeststorenamewstr.md)                         | Sets or retrieves the name of the store that contains the dummy certificate. This dummy certificate, along with the added private keys, remains in the request store until a certification authority processes the request and responds with a PKCS \#7.                                                                |
| [**RequestStoreTypeWStr**](ienroll4-requeststoretypewstr.md)                         | Sets or retrieves the type of store to use for the store specified by the [**RequestStoreNameWStr**](ienroll4-requeststorenamewstr.md) property.                                                                                                                                                                       |
| [**ReuseHardwareKeyIfUnableToGenNew**](ienroll4-reusehardwarekeyifunabletogennew.md) | Sets or retrieves a Boolean value that determines the action taken by the certificate enrollment control object if an error is encountered when generating a new key.                                                                                                                                                   |
| [**RootStoreFlags**](ienroll4-rootstoreflags.md)                                     | Sets or retrieves the registry location used for the ROOT store.                                                                                                                                                                                                                                                        |
| [**RootStoreNameWStr**](ienroll4-rootstorenamewstr.md)                               | Sets or retrieves the name of the root store where all intrinsically trusted self-signed ROOT certificates are kept.                                                                                                                                                                                                    |
| [**RootStoreTypeWStr**](ienroll4-rootstoretypewstr.md)                               | Sets or retrieves the type of store to use for the store specified by the [**RootStoreNameWStr**](ienroll4-rootstorenamewstr.md) property.                                                                                                                                                                             |
| [**SPCFileNameWStr**](ienroll4-spcfilenamewstr.md)                                   | Sets or retrieves the name of the file to write the resulting base64-encoded PKCS \#7 as returned from the certification authority.                                                                                                                                                                                     |
| [**ThumbPrintWStr**](ienroll4-thumbprintwstr.md)                                     | Sets or retrieves a [*hash*](security.h_gly#-security-hash-gly) of the certificate data.                                                                                                                                                                                                             |
| [**UseExistingKeySet**](ienroll4-useexistingkeyset.md)                               | Sets or retrieves a Boolean value that indicates whether the existing keys should be used.                                                                                                                                                                                                                              |
| [**WriteCertToCSP**](ienroll4-writecerttocsp.md)                                     | Sets or retrieves a Boolean value that indicates whether a certificate should be written to the CSP.                                                                                                                                                                                                                    |
| [**WriteCertToUserDS**](ienroll4-writecerttouserds.md)                               | Sets or retrieves a Boolean value that controls whether the certificate is written to the user's Active Directory store.                                                                                                                                                                                                |
