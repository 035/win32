# Properties of IEnroll4

The following properties are defined by the [**IEnroll4**](/windows/win32/Xenroll/nn-xenroll-ienroll4?branch=master) interface or are properties the **IEnroll4** interface inherits from [**IEnroll**](/windows/win32/Xenroll/nn-xenroll-ienroll?branch=master) and [**IEnroll2**](/windows/win32/Xenroll/nn-xenroll-ienroll2?branch=master).



| Property                                                                              | Description                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAStoreFlags**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_castoreflags?branch=master)                                         | Sets or retrieves a flag that controls the certificate store when it is opened.                                                                                                                                                                                                                                         |
| [**CAStoreNameWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_castorenamewstr?branch=master)                                   | Sets or retrieves the name of the store where all non-"ROOT" and non-"MY" certificates are kept.                                                                                                                                                                                                                        |
| [**CAStoreTypeWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_castoretypewstr?branch=master)                                   | Sets or retrieves the type of store to use for the store specified by the [**CAStoreNameWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_castorenamewstr?branch=master) property.                                                                                                                                                                                 |
| [**ClientId**](/windows/win32/Xenroll/nf-xenroll-ienroll4-get_clientid?branch=master)                                                 | Sets or retrieves the client ID request attribute.                                                                                                                                                                                                                                                                      |
| [**ContainerNameWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_containernamewstr?branch=master)                               | Sets or retrieves the name of the [*key container*](security.k_gly#-security-key-container-gly) to use.                                                                                                                                                                                     |
| [**DeleteRequestCert**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_deleterequestcert?branch=master)                               | Sets or retrieves a Boolean indicator that controls whether dummy certificates in the request store are deleted.                                                                                                                                                                                                        |
| [**EnableSMIMECapabilities**](/windows/win32/Xenroll/nf-xenroll-ienroll2-get_enablesmimecapabilities?branch=master)                   | Sets or retrieves a Boolean value that controls whether the PKCS \#10 will contain a signed attribute for [*Secure/Multipurpose Internet Mail Extensions*](security.s_gly) (S/MIME) capabilities. |
| [**EnableT61DNEncoding**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_enablet61dnencoding?branch=master)                           | Sets or retrieves a Boolean value that controls whether the distinguished name in the request is encoded as a T61 string instead of as a [*UNICODE*](security.u_gly#-security-unicode-gly) string.                                                                                                |
| [**GenKeyFlags**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_genkeyflags?branch=master)                                           | Sets or retrieves the values passed to [**CryptGenKey**](/windows/win32/Wincrypt/nf-wincrypt-cryptgenkey?branch=master) when the certificate request is generated.                                                                                                                                                                                                    |
| [**HashAlgID**](/windows/win32/Xenroll/?branch=master)                                               | Sets or retrieves the hash algorithm used when signing a PKCS \#10 certificate request.                                                                                                                                                                                                                                 |
| [**HashAlgorithmWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_hashalgorithmwstr?branch=master)                               | Sets or retrieves only the signature hash algorithm used to sign the PKCS \#10.                                                                                                                                                                                                                                         |
| [**IncludeSubjectKeyID**](/windows/win32/Xenroll/nf-xenroll-ienroll4-get_includesubjectkeyid?branch=master)                           | Specifies whether the subject key ID extension is added to the certificate request.                                                                                                                                                                                                                                     |
| [**KeySpec**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_keyspec?branch=master)                                                   | Sets or retrieves the type of key generated.                                                                                                                                                                                                                                                                            |
| [**LimitExchangeKeyToEncipherment**](/windows/win32/Xenroll/?branch=master)     | Sets or retrieves a Boolean value that controls whether an AT\_KEYEXCHANGE request contains [*digital signature*](security.d_gly#-security-digital-signature-gly) and non-repudiation key usages.                                                                                       |
| [**MyStoreFlags**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_mystoreflags?branch=master)                                         | Sets or retrieves the registry location used for the MY store.                                                                                                                                                                                                                                                          |
| [**MyStoreNameWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_mystorenamewstr?branch=master)                                   | Sets or retrieves the name of the store where certificates with linked private keys are kept.                                                                                                                                                                                                                           |
| [**MyStoreTypeWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_mystoretypewstr?branch=master)                                   | Sets or retrieves the type of store specified by the [**MyStoreNameWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_mystorenamewstr?branch=master) property.                                                                                                                                                                                                      |
| [**ProviderFlags**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_providerflags?branch=master)                                       | Sets or retrieves the CSP type.                                                                                                                                                                                                                                                                                         |
| [**ProviderNameWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_providernamewstr?branch=master)                                 | Sets or retrieves the name of the CSP to use.                                                                                                                                                                                                                                                                           |
| [**ProviderType**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_providertype?branch=master)                                         | Sets or retrieves the type of provider.                                                                                                                                                                                                                                                                                 |
| [**PVKFileNameWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_pvkfilenamewstr?branch=master)                                   | Sets or retrieves the name of the file that will contain exported keys.                                                                                                                                                                                                                                                 |
| [**RenewalCertificate**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_renewalcertificate?branch=master)                             | Specifies the certificate context for the renewal certificate.                                                                                                                                                                                                                                                          |
| [**RequestStoreFlags**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_requeststoreflags?branch=master)                               | Sets or retrieves the registry location used for the REQUEST store.                                                                                                                                                                                                                                                     |
| [**RequestStoreNameWStr**](/windows/win32/Xenroll/?branch=master)                         | Sets or retrieves the name of the store that contains the dummy certificate. This dummy certificate, along with the added private keys, remains in the request store until a certification authority processes the request and responds with a PKCS \#7.                                                                |
| [**RequestStoreTypeWStr**](/windows/win32/Xenroll/?branch=master)                         | Sets or retrieves the type of store to use for the store specified by the [**RequestStoreNameWStr**](/windows/win32/Xenroll/?branch=master) property.                                                                                                                                                                       |
| [**ReuseHardwareKeyIfUnableToGenNew**](/windows/win32/Xenroll/?branch=master) | Sets or retrieves a Boolean value that determines the action taken by the certificate enrollment control object if an error is encountered when generating a new key.                                                                                                                                                   |
| [**RootStoreFlags**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_rootstoreflags?branch=master)                                     | Sets or retrieves the registry location used for the ROOT store.                                                                                                                                                                                                                                                        |
| [**RootStoreNameWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_rootstorenamewstr?branch=master)                               | Sets or retrieves the name of the root store where all intrinsically trusted self-signed ROOT certificates are kept.                                                                                                                                                                                                    |
| [**RootStoreTypeWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_rootstoretypewstr?branch=master)                               | Sets or retrieves the type of store to use for the store specified by the [**RootStoreNameWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_rootstorenamewstr?branch=master) property.                                                                                                                                                                             |
| [**SPCFileNameWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_spcfilenamewstr?branch=master)                                   | Sets or retrieves the name of the file to write the resulting base64-encoded PKCS \#7 as returned from the certification authority.                                                                                                                                                                                     |
| [**ThumbPrintWStr**](/windows/win32/Xenroll/nf-xenroll-ienroll4-get_thumbprintwstr?branch=master)                                     | Sets or retrieves a [*hash*](security.h_gly#-security-hash-gly) of the certificate data.                                                                                                                                                                                                             |
| [**UseExistingKeySet**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_useexistingkeyset?branch=master)                               | Sets or retrieves a Boolean value that indicates whether the existing keys should be used.                                                                                                                                                                                                                              |
| [**WriteCertToCSP**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_writecerttocsp?branch=master)                                     | Sets or retrieves a Boolean value that indicates whether a certificate should be written to the CSP.                                                                                                                                                                                                                    |
| [**WriteCertToUserDS**](/windows/win32/Xenroll/nf-xenroll-ienroll-get_writecerttouserds?branch=master)                               | Sets or retrieves a Boolean value that controls whether the certificate is written to the user's Active Directory store.                                                                                                                                                                                                |
