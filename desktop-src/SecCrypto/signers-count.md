# Signers.Count property

\[The **Count** property is available for use in the operating systems specified in the Requirements section. Instead, use a collection of CmsSigner objects. For more information, see the [**CmsSigner Class**](T:System.Security.Cryptography.Pkcs.CmsSigner) in the [**System.Security.Cryptography.Pkcs**](frlrfSystemSecurityCryptographyPkcs) namespace.\]

The **Count** property retrieves the number of [**Signer**](signer.md) objects in the collection.

## Syntax


```VB
Signers.Count As Long
```



## Property value

Number of [**Signer**](signer.md) objects in the collection.
