# Properties of ICEnroll4

The following properties are defined by the [**ICEnroll4**](icenroll4.md) interface or are properties the **ICEnroll4** interface inherits from [**ICEnroll**](icenroll.md), [**ICEnroll2**](icenroll2.md), and [**ICEnroll3**](icenroll3.md).



| Property                                                                               | Description                                                                                                                                                                                                                                                                                                                                                         |
|----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAStoreFlags**](icenroll4-castoreflags.md)                                         | Sets or retrieves a flag that controls the [*certification authority*](security.c_gly#-security-certification-authority-gly) (CA) store when the store is opened.<br/>                                                                                                                                                  |
| [**CAStoreName**](icenroll4-castorename.md)                                           | Sets or retrieves the name of the store where all non-"ROOT" and non-"MY" certificates are kept.<br/>                                                                                                                                                                                                                                                         |
| [**CAStoreType**](icenroll4-castoretype.md)                                           | Sets or retrieves the type of store to use for the store specified by the [**CAStoreName**](icenroll4-castorename.md) property.<br/>                                                                                                                                                                                                                         |
| [**ClientId**](icenroll4-clientid.md)                                                 | Sets or retrieves the client ID request attribute.<br/>                                                                                                                                                                                                                                                                                                       |
| [**ContainerName**](icenroll4-containername.md)                                       | Sets or retrieves the name of the [*key container*](security.k_gly#-security-key-container-gly) to use.<br/>                                                                                                                                                                                                                      |
| [**DeleteRequestCert**](icenroll4-deleterequestcert.md)                               | Sets or retrieves a Boolean value that determines whether dummy certificates in the request store are deleted.<br/>                                                                                                                                                                                                                                           |
| [**EnableSMIMECapabilities**](icenroll4-enablesmimecapabilities.md)                   | Controls whether the PKCS10 will contain a signed attribute for [*Secure/Multipurpose Internet Mail Extensions*](security.s_gly#-security-secure-multipurpose-internet-mail-extensions-gly) (S/MIME) capabilities.<br/>                                                                            |
| [**EnableT61DNEncoding**](icenroll4-enablet61dnencoding.md)                           | Sets or retrieves a Boolean value that determines whether the distinguished name in the request is encoded as a T61 string instead of as a [*UNICODE*](security.u_gly#-security-unicode-gly) string.<br/>                                                                                                                               |
| [**GenKeyFlags**](icenroll4-genkeyflags.md)                                           | Sets or retrieves the values passed to [**CryptGenKey**](cryptgenkey.md) when the certificate request is generated.<br/>                                                                                                                                                                                                                                     |
| [**HashAlgID**](icenroll4-hashalgid.md)                                               | Sets or retrieves the [*hash algorithm*](security.h_gly#-security-hashing-algorithm-gly) used when signing a PKCS \#10 [*certificate request*](security.c_gly#-security-certificate-request-gly).<br/>                                                                                      |
| [**HashAlgorithm**](icenroll4-hashalgorithm.md)                                       | Sets or retrieves only the signature [*hash algorithm*](security.h_gly#-security-hashing-algorithm-gly) used to sign the PKCS \#10.<br/>                                                                                                                                                                                      |
| [**IncludeSubjectKeyID**](icenroll4-includesubjectkeyid.md)                           | Sets or retrieves a Boolean value that controls whether a subject key identifier extension is included in the certificate request.<br/>                                                                                                                                                                                                                       |
| [**KeySpec**](icenroll4-keyspec.md)                                                   | Sets or retrieves the type of key generated.<br/>                                                                                                                                                                                                                                                                                                             |
| [**LimitExchangeKeyToEncipherment**](icenroll4-limitexchangekeytoencipherment.md)     | Sets or retrieves a Boolean value that determines whether an AT\_KEYEXCHANGE request contains [*digital signature*](security.d_gly#-security-digital-signature-gly) and nonrepudiation key usages.<br/>                                                                                                                       |
| [**MyStoreFlags**](icenroll4-mystoreflags.md)                                         | Sets or retrieves the registry location used for MY store.<br/>                                                                                                                                                                                                                                                                                               |
| [**MyStoreName**](icenroll4-mystorename.md)                                           | Sets or retrieves the name of the store where certificates with linked [*private keys*](security.p_gly#-security-private-key-gly) are kept.<br/>                                                                                                                                                                                    |
| [**MyStoreType**](icenroll4-mystoretype.md)                                           | Sets or retrieves the type of store specified by the [**MyStoreName**](icenroll4-mystorename.md) property.<br/>                                                                                                                                                                                                                                              |
| [**PrivateKeyArchiveCertificate**](icenroll4-privatekeyarchivecertificate.md)         | Sets or retrieves the certificate that is used to archive a private key with a PKCS \#7 or [*Certificate Management over CMS*](security.c_gly) (CMC) request.<br/>                                                                                                |
| [**ProviderFlags**](icenroll4-providerflags.md)                                       | Sets or retrieves the provider type.<br/>                                                                                                                                                                                                                                                                                                                     |
| [**ProviderName**](icenroll4-providername.md)                                         | Sets or retrieves the name of the [*cryptographic service provider*](security.c_gly#-security-cryptographic-service-provider-gly) (CSP) to use.<br/>                                                                                                                                                             |
| [**ProviderType**](icenroll4-providertype.md)                                         | Sets or retrieves the type of provider.<br/>                                                                                                                                                                                                                                                                                                                  |
| [**PVKFileName**](icenroll4-pvkfilename.md)                                           | Sets or retrieves the name of the file that will contain exported keys.<br/>                                                                                                                                                                                                                                                                                  |
| [**RequestStoreFlags**](icenroll4-requeststoreflags.md)                               | Sets or retrieves the registry location used for the request store.<br/>                                                                                                                                                                                                                                                                                      |
| [**RequestStoreName**](icenroll4-requeststorename.md)                                 | Sets or retrieves the name of the store that contains the dummy certificate. This dummy certificate, along with the added private keys, remains in the Request store until a [*certification authority*](security.c_gly#-security-certification-authority-gly) processes the request and responds with a PKCS \#7.<br/> |
| [**RequestStoreType**](icenroll4-requeststoretype.md)                                 | Sets or retrieves the type of store to use for the store specified by the [**RequestStoreName**](icenroll4-requeststorename.md) property.<br/>                                                                                                                                                                                                               |
| [**ReuseHardwareKeyIfUnableToGenNew**](icenroll4-reusehardwarekeyifunabletogennew.md) | Sets or retrieves a Boolean value that determines the action taken by the certificate enrollment control object if an error is encountered when generating a new key.<br/>                                                                                                                                                                                    |
| [**RootStoreFlags**](icenroll4-rootstoreflags.md)                                     | Sets or retrieves the registry location used for the ROOT store.<br/>                                                                                                                                                                                                                                                                                         |
| [**RootStoreName**](icenroll4-rootstorename.md)                                       | Sets or retrieves the name of the root store where all intrinsically trusted self-signed [*ROOT certificates*](security.r_gly#-security-root-certificate-gly) are kept.<br/>                                                                                                                                                   |
| [**RootStoreType**](icenroll4-rootstoretype.md)                                       | Sets or retrieves the type of store to use for the store specified by the [**RootStoreName**](icenroll4-rootstorename.md) property.<br/>                                                                                                                                                                                                                     |
| [**SignerCertificate**](icenroll4-signercertificate.md)                               | Sets the signing certificate.<br/>                                                                                                                                                                                                                                                                                                                            |
| [**SPCFileName**](icenroll4-spcfilename.md)                                           | Sets or retrieves the name of the file in which to write the resulting base64-encoded PKCS \#7 (in **BSTR** form) as returned from the [*certification authority*](security.c_gly#-security-certification-authority-gly).<br/>                                                                                          |
| [**ThumbPrint**](icenroll4-thumbprint.md)                                             | Sets or retrieves a [*hash*](security.h_gly#-security-hash-gly) of the certificate data.<br/>                                                                                                                                                                                                                                              |
| [**UseExistingKeySet**](icenroll4-useexistingkeyset.md)                               | Sets or retrieves a Boolean value that determines whether the existing keys should be used.<br/>                                                                                                                                                                                                                                                              |
| [**WriteCertToCSP**](icenroll4-writecerttocsp.md)                                     | Sets or retrieves a Boolean value that determines whether a certificate should be written to the [*cryptographic service provider*](security.c_gly#-security-cryptographic-service-provider-gly) (CSP).<br/>                                                                                                     |
| [**WriteCertToUserDS**](icenroll4-writecerttouserds.md)                               | Sets or retrieves a Boolean value that determines whether the certificate is written to the user's Active Directory store.<br/>                                                                                                                                                                                                                               |
