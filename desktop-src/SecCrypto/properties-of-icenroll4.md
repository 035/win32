# Properties of ICEnroll4

The following properties are defined by the [**ICEnroll4**](/windows/win32/Xenroll/nn-xenroll-icenroll4?branch=master) interface or are properties the **ICEnroll4** interface inherits from [**ICEnroll**](/windows/win32/Xenroll/nn-xenroll-icenroll?branch=master), [**ICEnroll2**](/windows/win32/Xenroll/nn-xenroll-icenroll2?branch=master), and [**ICEnroll3**](/windows/win32/Xenroll/nn-xenroll-icenroll3?branch=master).



| Property                                                                               | Description                                                                                                                                                                                                                                                                                                                                                         |
|----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAStoreFlags**](/windows/win32/Xenroll/?branch=master)                                         | Sets or retrieves a flag that controls the [*certification authority*](security.c_gly#-security-certification-authority-gly) (CA) store when the store is opened.<br/>                                                                                                                                                  |
| [**CAStoreName**](/windows/win32/Xenroll/?branch=master)                                           | Sets or retrieves the name of the store where all non-"ROOT" and non-"MY" certificates are kept.<br/>                                                                                                                                                                                                                                                         |
| [**CAStoreType**](/windows/win32/Xenroll/?branch=master)                                           | Sets or retrieves the type of store to use for the store specified by the [**CAStoreName**](/windows/win32/Xenroll/?branch=master) property.<br/>                                                                                                                                                                                                                         |
| [**ClientId**](/windows/win32/Xenroll/nf-xenroll-icenroll4-get_clientid?branch=master)                                                 | Sets or retrieves the client ID request attribute.<br/>                                                                                                                                                                                                                                                                                                       |
| [**ContainerName**](/windows/win32/Xenroll/?branch=master)                                       | Sets or retrieves the name of the [*key container*](security.k_gly#-security-key-container-gly) to use.<br/>                                                                                                                                                                                                                      |
| [**DeleteRequestCert**](/windows/win32/Xenroll/?branch=master)                               | Sets or retrieves a Boolean value that determines whether dummy certificates in the request store are deleted.<br/>                                                                                                                                                                                                                                           |
| [**EnableSMIMECapabilities**](/windows/win32/Xenroll/nf-xenroll-icenroll3-get_enablesmimecapabilities?branch=master)                   | Controls whether the PKCS10 will contain a signed attribute for [*Secure/Multipurpose Internet Mail Extensions*](security.s_gly#-security-secure-multipurpose-internet-mail-extensions-gly) (S/MIME) capabilities.<br/>                                                                            |
| [**EnableT61DNEncoding**](/windows/win32/Xenroll/nf-xenroll-icenroll2-get_enablet61dnencoding?branch=master)                           | Sets or retrieves a Boolean value that determines whether the distinguished name in the request is encoded as a T61 string instead of as a [*UNICODE*](security.u_gly#-security-unicode-gly) string.<br/>                                                                                                                               |
| [**GenKeyFlags**](/windows/win32/Xenroll/?branch=master)                                           | Sets or retrieves the values passed to [**CryptGenKey**](/windows/win32/Wincrypt/nf-wincrypt-cryptgenkey?branch=master) when the certificate request is generated.<br/>                                                                                                                                                                                                                                     |
| [**HashAlgID**](/windows/win32/Xenroll/nf-xenroll-icenroll3-get_hashalgid?branch=master)                                               | Sets or retrieves the [*hash algorithm*](security.h_gly#-security-hashing-algorithm-gly) used when signing a PKCS \#10 [*certificate request*](security.c_gly#-security-certificate-request-gly).<br/>                                                                                      |
| [**HashAlgorithm**](/windows/win32/Xenroll/?branch=master)                                       | Sets or retrieves only the signature [*hash algorithm*](security.h_gly#-security-hashing-algorithm-gly) used to sign the PKCS \#10.<br/>                                                                                                                                                                                      |
| [**IncludeSubjectKeyID**](/windows/win32/Xenroll/nf-xenroll-icenroll4-get_includesubjectkeyid?branch=master)                           | Sets or retrieves a Boolean value that controls whether a subject key identifier extension is included in the certificate request.<br/>                                                                                                                                                                                                                       |
| [**KeySpec**](/windows/win32/Xenroll/?branch=master)                                                   | Sets or retrieves the type of key generated.<br/>                                                                                                                                                                                                                                                                                                             |
| [**LimitExchangeKeyToEncipherment**](/windows/win32/Xenroll/nf-xenroll-icenroll3-get_limitexchangekeytoencipherment?branch=master)     | Sets or retrieves a Boolean value that determines whether an AT\_KEYEXCHANGE request contains [*digital signature*](security.d_gly#-security-digital-signature-gly) and nonrepudiation key usages.<br/>                                                                                                                       |
| [**MyStoreFlags**](/windows/win32/Xenroll/?branch=master)                                         | Sets or retrieves the registry location used for MY store.<br/>                                                                                                                                                                                                                                                                                               |
| [**MyStoreName**](/windows/win32/Xenroll/?branch=master)                                           | Sets or retrieves the name of the store where certificates with linked [*private keys*](security.p_gly#-security-private-key-gly) are kept.<br/>                                                                                                                                                                                    |
| [**MyStoreType**](/windows/win32/Xenroll/?branch=master)                                           | Sets or retrieves the type of store specified by the [**MyStoreName**](/windows/win32/Xenroll/?branch=master) property.<br/>                                                                                                                                                                                                                                              |
| [**PrivateKeyArchiveCertificate**](/windows/win32/Xenroll/nf-xenroll-icenroll4-get_privatekeyarchivecertificate?branch=master)         | Sets or retrieves the certificate that is used to archive a private key with a PKCS \#7 or [*Certificate Management over CMS*](security.c_gly) (CMC) request.<br/>                                                                                                |
| [**ProviderFlags**](/windows/win32/Xenroll/?branch=master)                                       | Sets or retrieves the provider type.<br/>                                                                                                                                                                                                                                                                                                                     |
| [**ProviderName**](/windows/win32/Xenroll/?branch=master)                                         | Sets or retrieves the name of the [*cryptographic service provider*](security.c_gly#-security-cryptographic-service-provider-gly) (CSP) to use.<br/>                                                                                                                                                             |
| [**ProviderType**](/windows/win32/Xenroll/?branch=master)                                         | Sets or retrieves the type of provider.<br/>                                                                                                                                                                                                                                                                                                                  |
| [**PVKFileName**](/windows/win32/Xenroll/?branch=master)                                           | Sets or retrieves the name of the file that will contain exported keys.<br/>                                                                                                                                                                                                                                                                                  |
| [**RequestStoreFlags**](/windows/win32/Xenroll/?branch=master)                               | Sets or retrieves the registry location used for the request store.<br/>                                                                                                                                                                                                                                                                                      |
| [**RequestStoreName**](/windows/win32/Xenroll/?branch=master)                                 | Sets or retrieves the name of the store that contains the dummy certificate. This dummy certificate, along with the added private keys, remains in the Request store until a [*certification authority*](security.c_gly#-security-certification-authority-gly) processes the request and responds with a PKCS \#7.<br/> |
| [**RequestStoreType**](/windows/win32/Xenroll/?branch=master)                                 | Sets or retrieves the type of store to use for the store specified by the [**RequestStoreName**](/windows/win32/Xenroll/?branch=master) property.<br/>                                                                                                                                                                                                               |
| [**ReuseHardwareKeyIfUnableToGenNew**](/windows/win32/Xenroll/nf-xenroll-icenroll3-get_reusehardwarekeyifunabletogennew?branch=master) | Sets or retrieves a Boolean value that determines the action taken by the certificate enrollment control object if an error is encountered when generating a new key.<br/>                                                                                                                                                                                    |
| [**RootStoreFlags**](/windows/win32/Xenroll/?branch=master)                                     | Sets or retrieves the registry location used for the ROOT store.<br/>                                                                                                                                                                                                                                                                                         |
| [**RootStoreName**](/windows/win32/Xenroll/?branch=master)                                       | Sets or retrieves the name of the root store where all intrinsically trusted self-signed [*ROOT certificates*](security.r_gly#-security-root-certificate-gly) are kept.<br/>                                                                                                                                                   |
| [**RootStoreType**](/windows/win32/Xenroll/?branch=master)                                       | Sets or retrieves the type of store to use for the store specified by the [**RootStoreName**](/windows/win32/Xenroll/?branch=master) property.<br/>                                                                                                                                                                                                                     |
| [**SignerCertificate**](/windows/win32/Xenroll/nf-xenroll-icenroll4-put_signercertificate?branch=master)                               | Sets the signing certificate.<br/>                                                                                                                                                                                                                                                                                                                            |
| [**SPCFileName**](/windows/win32/Xenroll/?branch=master)                                           | Sets or retrieves the name of the file in which to write the resulting base64-encoded PKCS \#7 (in **BSTR** form) as returned from the [*certification authority*](security.c_gly#-security-certification-authority-gly).<br/>                                                                                          |
| [**ThumbPrint**](/windows/win32/Xenroll/nf-xenroll-icenroll4-get_thumbprint?branch=master)                                             | Sets or retrieves a [*hash*](security.h_gly#-security-hash-gly) of the certificate data.<br/>                                                                                                                                                                                                                                              |
| [**UseExistingKeySet**](/windows/win32/Xenroll/?branch=master)                               | Sets or retrieves a Boolean value that determines whether the existing keys should be used.<br/>                                                                                                                                                                                                                                                              |
| [**WriteCertToCSP**](/windows/win32/Xenroll/nf-xenroll-icenroll-get_writecerttocsp?branch=master)                                     | Sets or retrieves a Boolean value that determines whether a certificate should be written to the [*cryptographic service provider*](security.c_gly#-security-cryptographic-service-provider-gly) (CSP).<br/>                                                                                                     |
| [**WriteCertToUserDS**](/windows/win32/Xenroll/nf-xenroll-icenroll2-get_writecerttouserds?branch=master)                               | Sets or retrieves a Boolean value that determines whether the certificate is written to the user's Active Directory store.<br/>                                                                                                                                                                                                                               |
