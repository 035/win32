# Signer.Chain property

\[The **Chain** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**CmsSigner Class**](T:System.Security.Cryptography.Pkcs.CmsSigner) in the [**System.Security.Cryptography.Pkcs**](frlrfSystemSecurityCryptographyPkcs) namespace.\]

The **Chain** property retrieves the signer's chain.

This property is read-only.

## Syntax


```VB
Signer.Chain As Chain
```



## Property value

The signer's chain.
