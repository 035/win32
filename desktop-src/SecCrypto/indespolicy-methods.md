# INDESPolicy Methods

The [**INDESPolicy**](/windows/win32/certpol/nn-certpol-indespolicy?branch=master) interface supports the following methods.

## In this section



| Topic                                                                        | Description                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**GenerateChallenge method**](/windows/win32/certpol/nf-certpol-indespolicy-generatechallenge?branch=master)<br/> | Performs the policy decision whether to issue a challenge password to the SCEP client.<br/> |
| [**Initialize method**](/windows/win32/certpol/nf-certpol-indespolicy-initialize?branch=master)<br/>               | Initializes the NDES policy module.<br/>                                                    |
| [**Notify method**](/windows/win32/certpol/nf-certpol-indespolicy-notify?branch=master)<br/>                       | Notifies the plug-in of the transaction status of the SCEP certificate request.<br/>        |
| [**Uninitialize method**](/windows/win32/certpol/nf-certpol-indespolicy-uninitialize?branch=master)<br/>           | Uninitializes the NDES policy module.<br/>                                                  |
| [**VerifyRequest method**](/windows/win32/certpol/nf-certpol-indespolicy-verifyrequest?branch=master)<br/>         | Verifies the NDES certificate request for submission to the CA.<br/>                        |
