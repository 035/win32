# INDESPolicy Methods

The [**INDESPolicy**](indespolicy.md) interface supports the following methods.

## In this section



| Topic                                                                        | Description                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**GenerateChallenge method**](indespolicy-generatechallenge.md)<br/> | Performs the policy decision whether to issue a challenge password to the SCEP client.<br/> |
| [**Initialize method**](indespolicy-initialize.md)<br/>               | Initializes the NDES policy module.<br/>                                                    |
| [**Notify method**](indespolicy-notify.md)<br/>                       | Notifies the plug-in of the transaction status of the SCEP certificate request.<br/>        |
| [**Uninitialize method**](indespolicy-uninitialize.md)<br/>           | Uninitializes the NDES policy module.<br/>                                                  |
| [**VerifyRequest method**](indespolicy-verifyrequest.md)<br/>         | Verifies the NDES certificate request for submission to the CA.<br/>                        |
