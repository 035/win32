# Store.Certificates property

\[The **Certificates** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**X509Store Class**](T:System.Security.Cryptography.X509Certificates.X509Store) in the [**System.Security.Cryptography.X509Certificates**](frlrfSystemSecurityCryptographyX509Certificates) namespace.\]

The **Certificates** property retrieves the collection that includes all of the certificates in the store. This is the default property.

## Syntax


```VB
Store.Certificates As Certificates
```



## Property value

The collection of certificates in the store.
