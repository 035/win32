# Methods of ICertSrvSetupKeyInformationCollection

The following methods are defined by the [**ICertSrvSetupKeyInformationCollection**](/windows/win32/Casetup/nn-casetup-icertsrvsetupkeyinformationcollection?branch=master) interface. The property access methods are not shown here. To see the properties for **ICertSrvSetupKeyInformationCollection**, see [Properties of ICertSrvSetupKeyInformationCollection](properties-of-icertsrvsetupkeyinformationcollection.md).



| Method                                                   | Description                                                                                          |
|----------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**Add**](/windows/win32/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-add?branch=master) | Adds an [**ICertSrvSetupKeyInformation**](/windows/win32/Casetup/nn-casetup-icertsrvsetupkeyinformation?branch=master) object to the collection. |
