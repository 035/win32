# PROV\_MS\_EXCHANGE

The PROV\_MS\_EXCHANGE provider type is designed for the cryptographic needs of the Microsoft Exchange mail application and other applications compatible with Microsoft Mail.

## Algorithms Supported

For descriptions of each of these algorithms, see the glossary.



| Purpose      | Supported algorithms                                           |
|--------------|----------------------------------------------------------------|
| Key Exchange | [*RSA*](https://msdn.microsoft.com/ce589e18-02ac-42c2-b76b-776deb686bbd)    |
| Signature    | RSA                                                            |
| Encryption   | [*CAST*](https://msdn.microsoft.com/db46def4-bfdc-4801-a57d-d568e94a2dbb) |
| Hashing      | [*MD5*](https://msdn.microsoft.com/4c4402e9-7455-4868-978f-3899a8fd86c1)    |
