# Template.MinorVersion property

\[The **MinorVersion** property is available for use in the operating systems specified in the Requirements section. Instead, use the [**X509Extension Class**](T:System.Security.Cryptography.X509Certificates.X509Extension) in the [**System.Security.Cryptography.X509Certificates**](frlrfSystemSecurityCryptographyX509Certificates) namespace by calling the constructor that takes an OID as a parameter, and then use the OID for Certificate Template to retrieve the certificate extension template.\]

The **MinorVersion** property retrieves the minor version number of the template.

## Syntax


```VB
Template.MinorVersion As Long
```



## Property value

The minor version number of the template.
