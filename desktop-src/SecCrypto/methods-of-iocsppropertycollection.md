# Methods of IOCSPPropertyCollection

The following methods are defined by the [**IOCSPPropertyCollection**](/windows/win32/Certadm/nn-certadm-iocsppropertycollection?branch=master) interface. The property access methods are not shown here. To see the properties for **IOCSPPropertyCollection**, see [Properties of IOCSPPropertyCollection](properties-of-iocsppropertycollection.md)



| Method                                                                                      | Description                                                                               |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**CreateProperty**](/windows/win32/Certadm/nf-certadm-iocsppropertycollection-createproperty?branch=master)                     | Creates a new property and adds it to a property set..                                    |
| [**DeleteProperty**](/windows/win32/Certadm/nf-certadm-iocsppropertycollection-deleteproperty?branch=master)                     | Removes a named property from a property set.                                             |
| [**GetAllProperties**](/windows/win32/Certadm/nf-certadm-iocsppropertycollection-getallproperties?branch=master)                 | Gets all properties in a property set.                                                    |
| [**InitializeFromProperties**](/windows/win32/Certadm/nf-certadm-iocsppropertycollection-initializefromproperties?branch=master) | Creates a property set from the properties contained in an existing server configuration. |
