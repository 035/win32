# OIDs.Clear method

\[The **Clear** method is available for use in the operating systems specified in the Requirements section. Instead, use the [**OidCollection Class**](T:System.Security.Cryptography.OidCollection) in the [**System.Security.Cryptography**](frlrfSystemSecurityCryptography) namespace.\]

The **Clear** method clears all [**OID**](oid.md) objects from the collection.

## Syntax


```VB
OIDs.Clear()
```



## Parameters

This method has no parameters.

## Return value

This method does not return a value.
