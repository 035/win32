# Create a Windows Runtime component for device access

You can create a Windows Runtime component to wrap the component that accesses your driver. You can then write a Windows Store device app for your device in C# or JavaScript.

## Instructions

### 

See the [Custom Driver Access Sample]( http://go.microsoft.com/fwlink/p/?linkid=242014).This sample implements a Windows Runtime component for providingthe functionality of a specialized device, and it uses the functionality in a Windows Store device app.
