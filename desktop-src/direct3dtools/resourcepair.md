# <span id="vspixengine.resourcepair"></span>ResourcePair structure

Represents a shared resource which is encoded as a COM string.

## Syntax


```C++
} ResourcePair;
```

## Members

**id**  
An ID that identifies the resource. For more information, see the Resource\_Id enum.

**value**  
A COM string contianing the value of the shared resource.
