# Client Security During an Asynchronous Call

The proxy manager created by MIDL for objects that use standard marshaling implements the [**IClientSecurity**](iclientsecurity.md) interface. Clients can manage the security of marshaled calls by querying for **IClientSecurity** on the call object and obtaining or changing security settings.
