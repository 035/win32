# CLSID

Associates a ProgID with a CLSID.

## Registry Entry

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes
   {ProgID}
      CLSID = CLSID
```

## Remarks

This is a **REG\_SZ** value that specifies the object's CLSID.
