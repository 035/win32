# What Support for an Interface Means

Besides the [**IUnknown**](iunknown.md) interface, an ActiveX Control — or COM Object for that matter — expresses whatever optional functionality it supports through additional interfaces. This is to say that no other interfaces are required above **IUnknown**. To that end, the following table lists the interfaces that an ActiveX Control might support, and what it means to support that interface.

| [**IOleObject**](ioleobject.md)<br/>                                                                                                                                                                                                                                                                                                                                     | If the control requires communication with its client site for anything other than events (see [**IConnectionPointContainer**](iconnectionpointcontainer.md)), then [**IOleObject**](ioleobject.md) is a necessity. When implementing this interface, the control must also support the semantics of the following methods: [**SetHostNames**](ioleobject-sethostnames.md), [**Close**](ioleobject-close.md), [**EnumVerbs**](ioleobject-enumverbs.md), [**Update**](ioleobject-update.md), [**IsUpToDate**](ioleobject-isuptodate.md), [**GetUserClassID**](ioleobject-getuserclassid.md), [**GetUserType**](ioleobject-getusertype.md), [**GetMiscStatus**](ioleobject-getmiscstatus.md), and the [**Advise**](ioleobject-advise.md), [**Unadvise**](ioleobject-unadvise.md), and [**EnumAdvise**](ioleobject-enumadvise.md) methods that work in conjunction with a container's [**IAdviseSink**](iadvisesink.md) implementation. A control implementing **IOleObject** must be able to handle **IAdviseSink** if the container provides one; a container may not, in which case a control ensures, of course, that it does not attempt to call a non-existent sink.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IOleInPlaceObject**](ioleinplaceobject.md)<br/>                                                                                                                                                                                                                                                                                                                       | Expresses the control's ability to be in-place activated and possibly UI activated. This interface means that the control has a user interface of some kind that can be activated, and [**IOleInPlaceActiveObject**](ioleinplaceactiveobject.md) is supported as well. Required methods are [**GetWindow**](iolewindow-getwindow.md), [**InPlaceDeactivate**](ioleinplaceobject-inplacedeactivate.md), [**UIDeactivate**](ioleinplaceobject-uideactivate.md), [**SetObjectRects**](ioleinplaceobject-setobjectrects.md), and [**ReactivateAndUndo**](ioleinplaceobject-reactivateandundo.md). Support for this interface requires support for [**IOleObject**](ioleobject.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |

supports [**IOleInPlaceObject**](ioleinplaceobject.md) must also provide this interface as well, though the control itself doesn't necessarily implement the interface directly.<br/> | [**IOleControl**](iolecontrol.md)<br/>                                                                                                                                                                                                                                                                                                                                   | Expresses the control's ability and desire to deal with (a) mnemonics ([**GetControlInfo**](iolecontrol-getcontrolinfo.md), [**OnMnemonic**](iolecontrol-onmnemonic.md) methods), (b) ambient properties ([**OnAmbientPropertyChange**](iolecontrol-onambientpropertychange.md)), and/or (c) events that the control requires the container to handle ([**FreezeEvents**](iolecontrol-freezeevents.md)). Note that mnemonics are different than accelerators that are handled through [**IOleInPlaceActiveObject**](ioleinplaceactiveobject.md): mnemonics have associated UI and are active even when the control is not UI active. A control's support for mnemonics means that the control also knows how to use the container's [**IOleControlSite::OnControlInfoChanged**](iolecontrolsite-oncontrolinfochanged.md) method. Because this requires the control to know the container's site, support for mnemonics also means support for [**IOleObject**](ioleobject.md). In addition, knowledge of mnemonics requires in-place support and thus [**IOleInPlaceObject**](ioleinplaceobject.md).<br/> If a control uses any container-ambient properties, then it must also implement this interface to receive change notifications, as following the semantics of changes is required. Because ambient properties are only available through the container site's **IDispatch**, ambient property support means that the control supports [**IOleObject**](ioleobject.md) (to get the site) as well as being able to generate **IDispatch::Invoke** calls.<br/> The [**FreezeEvents**](iolecontrol-freezeevents.md) method is necessary for controls that must know when a container is not going to handle an event—this is the only way for control to know this condition. If **FreezeEvents** is only necessary in isolation, such that other [**IOleControl**](iolecontrol.md) methods are not implemented, then **IOleControl** can stand alone without [**IOleObject**](ioleobject.md) or [**IOleInPlaceObject**](ioleinplaceobject.md).<br/> |
| [**IDataObject**](idataobject.md)<br/>                                                                                                                                                                                                                                                                                                                                   | Indicates that the control can supply at least (a) graphical renderings of the control (CF\_METAFILEPICT is the minimum if the control has any visuals at all) and/or (b) property sets, if the control has any properties to provide. The methods [**GetData**](idataobject-getdata.md), [**QueryGetData**](idataobject-querygetdata.md), [**EnumFormatEtc**](idataobject-enumformatetc.md), [**DAdvise**](idataobject-dadvise.md), [**DUnadvise**](idataobject-dunadvise.md), and [**EnumDAdvise**](idataobject-enumdadvise.md) are required. Support for graphical formats other than CF\_METAFILEPICT is optional.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**IViewObject2**](iviewobject2.md)<br/>                                                                                                                                                                                                                                                                                                                                 | Indicates that the control has some interesting visuals when it is not in-place active. If implemented, a control must support the methods [**Draw**](iviewobject-draw.md), [**GetAdvise**](iviewobject-getadvise.md), [**SetAdvise**](iviewobject-setadvise.md), and [**GetExtent**](iviewobject2-getextent.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **IDispatch**<br/>                                                                                                                                                                                                                                                                                                                                                        | Indicates that the control has either (a) custom methods, or (b) custom properties that are both available via late-binding through **IDispatch::Invoke**. This also requires that the control provides type information through other **IDispatch** methods. A control may support multiple **IDispatch** implementations where only one is associated with IID\_IDispatch—the others must have their own unique dispinterface identifiers.<br/> A control is encouraged to supply dual interfaces for custom method and property access, but this is optional if methods and properties exist.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**IConnectionPointContainer**](iconnectionpointcontainer.md)<br/>                                                                                                                                                                                                                                                                                                       | Indicates that a control supports at least one outgoing interface, such as events or property change notifications. All methods of this interface must be implemented if this interface is available at all, including [**EnumConnectionPoints**](iconnectionpointcontainer-enumconnectionpoints.md) which requires a separate object with [**IEnumConnectionPoints**](ienumconnectionpoints.md).<br/> Support for [**IConnectionPointContainer**](iconnectionpointcontainer.md) means that the object also supports one or more related objects with [**IConnectionPoint**](iconnectionpoint.md) that are available through **IConnectionPointContainer** methods. Each connection point object itself must implement the full **IConnectionPoint** interface, including [**EnumConnections**](iconnectionpoint-enumconnections.md), which requires another enumerator object with the [**IEnumConnections**](ienumconnections.md) interface.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IProvideClassInfo**](iprovideclassinfo.md)<br/> [**IProvideClassInfo2**](iprovideclassinfo2.md)<br/>                                                                                                                                                                                                                                                           | Indicates that the object can provide its own coclass type information directly through [**IProvideClassInfo::GetClassInfo**](iprovideclassinfo-getclassinfo.md). If the control supports the later variation [**IProvideClassInfo2**](iprovideclassinfo2.md), then it also indicates its ability to provide its primary source IID through [**IProvideClassInfo2::GetGUID**](iprovideclassinfo2-getguid.md). All methods of this interface must be implemented.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |

If (c) is supported, then it also means that the object's property pages mapped through [**IPerPropertyBrowsing::MapPropertyToPage**](iperpropertybrowsing-mappropertytopage.md) themselves implement [**IPropertyPage2**](ipropertypage2.md) as opposed to the basic [**IPropertyPage**](ipropertypage.md) interface.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
