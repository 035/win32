# Using the Container's Functionality

The previous sections have described some of the necessary caller-side support that an ActiveX control must have in order to access certain features of its container. The following table describes a control's usage of container-side interfaces and when such usage would occur.



| Interface                                                     | Container Object    | Usage                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------|---------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IOleClientSite**](ioleclientsite.md)<br/>           | Site<br/>     | Controls that implement [**IOleObject**](ioleobject.md) call [**IOleClientSite**](ioleclientsite.md) methods as part of the standard OLE embedding protocol, specifically the methods [**SaveObject**](ioleclientsite-saveobject.md), [**ShowObject**](ioleclientsite-showobject.md), [**OnShowWindow**](ioleclientsite-onshowwindow.md) (only if a separate-window activation state is supported), [**RequestNewObjectLayout**](ioleclientsite-requestnewobjectlayout.md), and GetContainer (if communication with other controls is desired). The [**GetMoniker**](ioleclientsite-getmoniker.md) method is only used when the control can be linked to externally, that is, the control is not marked with OLEMISC\_CANTLINKINSIDE.<br/> |
| [**IOleInPlaceSite**](ioleinplacesite.md)<br/>         | Site<br/>     | Controls that have an in-place activate and possibly a UI active state will call [**IOleInPlaceSite**](ioleinplacesite.md) methods (generally all of them with the exception of [**ContextSensitiveHelp**](iolewindow-contextsensitivehelp.md)) as part of the standard OLE in-place activation protocol.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**IAdviseSink**](iadvisesink.md)<br/>                 | Site<br/>     | Control calls [**OnDataChange**](iadvisesink-ondatachange.md) if the control supports [**IDataObject**](idataobject.md), [**OnViewChange**](iadvisesink-onviewchange.md) if the control supports [**IViewObject2**](iviewobject2.md), and [**OnClose**](iadvisesink-onclose.md), [**OnSave**](iadvisesink-onsave.md), and [**OnRename**](iadvisesink-onrename.md) if the control supports [**IOleObject**](ioleobject.md).<br/>                                                                                                                                                                                                                                                                                                           |
| [**IOleControlSite**](iolecontrolsite.md)<br/>         | Site<br/>     | If supported, control calls [**OnControlInfoChanged**](iolecontrolsite-oncontrolinfochanged.md) when mnemonics change, [**LockInPlaceActive**](iolecontrolsite-lockinplaceactive.md) and [**TransformCoords**](iolecontrolsite-transformcoords.md) if events are fired (the latter method is only used if coordinates are passed as event arguments), [**OnFocus**](iolecontrolsite-onfocus.md) and [**TranslateAccelerator**](iolecontrolsite-translateaccelerator.md) if the control has a UI active state, and [**GetExtendedControl**](iolecontrolsite-getextendedcontrol.md) if the control wants to look at extended-control (container-owned) properties.<br/>                                                                       |
| **IDispatch** (ambient properties)<br/>                 | Site<br/>     | Used to access ambient properties.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IPropertyNotifySink**](ipropertynotifysink.md)<br/> | Varies<br/>   | A control must generate [**OnChanged**](ipropertynotifysink-onchanged.md) and [**OnRequestEdit**](ipropertynotifysink-onrequestedit.md) for any control properties that are marked as \[[**bindable**](https://msdn.microsoft.com/library/windows/desktop/aa366738)\] and \[[**request**](https://msdn.microsoft.com/library/windows/desktop/aa367155)\], respectively.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Other event sink interfaces<br/>                        | Varies<br/>   | A control that has outgoing interfaces other than [**IPropertyNotifySink**](ipropertynotifysink.md) will be handed other interface pointers of the correct IID to the control's [**IConnectionPoint::Advise**](iconnectionpoint-advise.md) implementations (which are usually found in sub-objects of the control). A control always knows how to call its own event interfaces because the control defines those interfaces.<br/>                                                                                                                                                                                                                                                                                                              |
| [**IOleInPlaceFrame**](ioleinplaceframe.md)<br/>       | Frame<br/>    | Used when a control has an in-place UI active state that requires frame-level tools or menu items.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IOleInPlaceUIWindow**](ioleinplaceuiwindow.md)<br/> | Document<br/> | Used only when a control has an in-place UI active state that requires document-level or pane-level UI tools. This is rare.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
