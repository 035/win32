# Using the Container's Functionality

The previous sections have described some of the necessary caller-side support that an ActiveX control must have in order to access certain features of its container. The following table describes a control's usage of container-side interfaces and when such usage would occur.



| Interface                                                     | Container Object    | Usage                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------|---------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IOleClientSite**](/windows/win32/OleIdl/nn-oleidl-ioleclientsite?branch=master)<br/>           | Site<br/>     | Controls that implement [**IOleObject**](/windows/win32/OleIdl/nn-oleidl-ioleobject?branch=master) call [**IOleClientSite**](/windows/win32/OleIdl/nn-oleidl-ioleclientsite?branch=master) methods as part of the standard OLE embedding protocol, specifically the methods [**SaveObject**](/windows/win32/OleIdl/nf-oleidl-ioleclientsite-saveobject?branch=master), [**ShowObject**](/windows/win32/OleIdl/nf-oleidl-ioleclientsite-showobject?branch=master), [**OnShowWindow**](/windows/win32/OleIdl/nf-oleidl-ioleclientsite-onshowwindow?branch=master) (only if a separate-window activation state is supported), [**RequestNewObjectLayout**](/windows/win32/OleIdl/nf-oleidl-ioleclientsite-requestnewobjectlayout?branch=master), and GetContainer (if communication with other controls is desired). The [**GetMoniker**](/windows/win32/OleIdl/nf-oleidl-ioleclientsite-getmoniker?branch=master) method is only used when the control can be linked to externally, that is, the control is not marked with OLEMISC\_CANTLINKINSIDE.<br/> |
| [**IOleInPlaceSite**](/windows/win32/OleIdl/nn-oleidl-ioleinplacesite?branch=master)<br/>         | Site<br/>     | Controls that have an in-place activate and possibly a UI active state will call [**IOleInPlaceSite**](/windows/win32/OleIdl/nn-oleidl-ioleinplacesite?branch=master) methods (generally all of them with the exception of [**ContextSensitiveHelp**](/windows/win32/OleIdl/nf-oleidl-iolewindow-contextsensitivehelp?branch=master)) as part of the standard OLE in-place activation protocol.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**IAdviseSink**](/windows/win32/ObjIdl/nn-objidl-iadvisesink?branch=master)<br/>                 | Site<br/>     | Control calls [**OnDataChange**](/windows/win32/ObjIdl/nf-objidl-iadvisesink-ondatachange?branch=master) if the control supports [**IDataObject**](/windows/win32/ObjIdl/nn-objidl-idataobject?branch=master), [**OnViewChange**](/windows/win32/ObjIdl/nf-objidl-iadvisesink-onviewchange?branch=master) if the control supports [**IViewObject2**](/windows/win32/OleIdl/nn-oleidl-iviewobject2?branch=master), and [**OnClose**](/windows/win32/ObjIdl/nf-objidl-iadvisesink-onclose?branch=master), [**OnSave**](/windows/win32/ObjIdl/nf-objidl-iadvisesink-onsave?branch=master), and [**OnRename**](/windows/win32/ObjIdl/nf-objidl-iadvisesink-onrename?branch=master) if the control supports [**IOleObject**](/windows/win32/OleIdl/nn-oleidl-ioleobject?branch=master).<br/>                                                                                                                                                                                                                                                                                                           |
| [**IOleControlSite**](/windows/win32/OCIdl/nn-ocidl-iolecontrolsite?branch=master)<br/>         | Site<br/>     | If supported, control calls [**OnControlInfoChanged**](/windows/win32/OCIdl/nf-ocidl-iolecontrolsite-oncontrolinfochanged?branch=master) when mnemonics change, [**LockInPlaceActive**](/windows/win32/OCIdl/nf-ocidl-iolecontrolsite-lockinplaceactive?branch=master) and [**TransformCoords**](/windows/win32/OCIdl/nf-ocidl-iolecontrolsite-transformcoords?branch=master) if events are fired (the latter method is only used if coordinates are passed as event arguments), [**OnFocus**](/windows/win32/OCIdl/nf-ocidl-iolecontrolsite-onfocus?branch=master) and [**TranslateAccelerator**](/windows/win32/OCIdl/nf-ocidl-iolecontrolsite-translateaccelerator?branch=master) if the control has a UI active state, and [**GetExtendedControl**](/windows/win32/OCIdl/nf-ocidl-iolecontrolsite-getextendedcontrol?branch=master) if the control wants to look at extended-control (container-owned) properties.<br/>                                                                       |
| **IDispatch** (ambient properties)<br/>                 | Site<br/>     | Used to access ambient properties.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IPropertyNotifySink**](/windows/win32/OCIdl/nn-ocidl-ipropertynotifysink?branch=master)<br/> | Varies<br/>   | A control must generate [**OnChanged**](/windows/win32/OCIdl/nf-ocidl-ipropertynotifysink-onchanged?branch=master) and [**OnRequestEdit**](/windows/win32/OCIdl/nf-ocidl-ipropertynotifysink-onrequestedit?branch=master) for any control properties that are marked as \[[**bindable**](https://msdn.microsoft.com/library/windows/desktop/aa366738)\] and \[[**request**](https://msdn.microsoft.com/library/windows/desktop/aa367155)\], respectively.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Other event sink interfaces<br/>                        | Varies<br/>   | A control that has outgoing interfaces other than [**IPropertyNotifySink**](/windows/win32/OCIdl/nn-ocidl-ipropertynotifysink?branch=master) will be handed other interface pointers of the correct IID to the control's [**IConnectionPoint::Advise**](/windows/win32/OCIdl/nf-ocidl-iconnectionpoint-advise?branch=master) implementations (which are usually found in sub-objects of the control). A control always knows how to call its own event interfaces because the control defines those interfaces.<br/>                                                                                                                                                                                                                                                                                                              |
| [**IOleInPlaceFrame**](/windows/win32/OleIdl/nn-oleidl-ioleinplaceframe?branch=master)<br/>       | Frame<br/>    | Used when a control has an in-place UI active state that requires frame-level tools or menu items.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IOleInPlaceUIWindow**](/windows/win32/OleIdl/nn-oleidl-ioleinplaceuiwindow?branch=master)<br/> | Document<br/> | Used only when a control has an in-place UI active state that requires document-level or pane-level UI tools. This is rare.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
