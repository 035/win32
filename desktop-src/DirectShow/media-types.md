# Media Types

Microsoft® DirectShow® uses the [**AM\_MEDIA\_TYPE**](am-media-type.md) structure to describe media samples. This structure includes **GUID** members for major type, subtype, and format type. This section summarizes the major type and subtype GUIDs that are defined for DirectShow. Third-party filters may define their own media type GUIDs.
