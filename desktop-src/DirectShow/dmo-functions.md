# DMO Functions

This section describes the Microsoft DirectX Media Object (DMO) functions.



| Function                                             | Description                                                   |
|------------------------------------------------------|---------------------------------------------------------------|
| [**DMOEnum**](/windows/win32/Dmoreg/nf-dmoreg-dmoenum?branch=master)                           | Enumerates DMOs listed in the registry.                       |
| [**DMOGetName**](/windows/win32/Dmoreg/nf-dmoreg-dmogetname?branch=master)                     | Retrieves the name of a DMO from the registry.                |
| [**DMOGetTypes**](/windows/win32/Dmoreg/nf-dmoreg-dmogettypes?branch=master)                   | Retrieves the media types registered for a DMO.               |
| [**DMORegister**](/windows/win32/Dmoreg/nf-dmoreg-dmoregister?branch=master)                   | Registers a DMO.                                              |
| [**DMOUnregister**](/windows/win32/Dmoreg/nf-dmoreg-dmounregister?branch=master)               | Unregisters a DMO.                                            |
| [**MoCopyMediaType**](/windows/win32/Dmort/nf-dmort-mocopymediatype?branch=master)           | Copies one media type structure to another.                   |
| [**MoCreateMediaType**](/windows/win32/Dmort/nf-dmort-mocreatemediatype?branch=master)       | Allocates a new media type structure.                         |
| [**MoDeleteMediaType**](/windows/win32/Dmort/nf-dmort-modeletemediatype?branch=master)       | Deletes a media type structure that was previously allocated. |
| [**MoDuplicateMediaType**](/windows/win32/Dmort/nf-dmort-moduplicatemediatype?branch=master) | Duplicates a media type structure.                            |
| [**MoFreeMediaType**](/windows/win32/Dmort/nf-dmort-mofreemediatype?branch=master)           | Frees the allocated members of a media type structure.        |
| [**MoInitMediaType**](/windows/win32/Dmort/nf-dmort-moinitmediatype?branch=master)           | Initializes a media type structure.                           |
