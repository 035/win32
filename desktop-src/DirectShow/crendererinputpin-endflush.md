# CRendererInputPin.EndFlush method

The `EndFlush` method ends a flush operation. This method overrides the [**CBaseInputPin::EndFlush**](cbaseinputpin-endflush.md) method.

## Syntax


```C++
HRESULT EndFlush();
```



## Parameters

This method has no parameters.

## Return value

Returns an **HRESULT** value.
