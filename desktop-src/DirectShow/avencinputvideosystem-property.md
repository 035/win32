# AVEncInputVideoSystem property

Specifies the video system of the source content.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncInputVideoSystem**

## Property value

The value of this property is a member of the [**eAVEncInputVideoSystem**](eavencinputvideosystem.md) enumeration.
