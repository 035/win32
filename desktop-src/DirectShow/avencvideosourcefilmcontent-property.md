# AVEncVideoSourceFilmContent property

Specifies whether the original source of the input video was film or video.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncVideoSourceFilmContent**

## Property value

The value of this property is a member of the [**eAVEncVideoFilmContent**](/windows/win32/codecapi/?branch=master) enumeration.
