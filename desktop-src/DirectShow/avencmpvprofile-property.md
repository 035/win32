# AVEncMPVProfile property

Specifies the MPEG-2 profile. This property applies to MPEG video encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncMPVProfile**

## Property value

The value of this property is a member of the [**eAVEncMPVProfile**](/windows/win32/codecapi/?branch=master) enumeration.
