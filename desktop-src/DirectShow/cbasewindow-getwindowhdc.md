# CBaseWindow.GetWindowHDC method

The `GetWindowHDC` method retrieves a handle to the window's device context (DC).

## Syntax


```C++
HDC GetWindowHDC();
```



## Parameters

This method has no parameters.

## Return value

Returns a handle to the DC.
