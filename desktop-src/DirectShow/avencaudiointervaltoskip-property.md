# AVEncAudioIntervalToSkip property

Specifies the number of audio samples for the encoder to skip.

This property is read/write.

## Data type

**UINT64** (**VT\_UI8**)

## Property GUID

**CODECAPI\_AVEncAudioIntervalToSkip**
