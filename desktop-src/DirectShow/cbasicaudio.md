# CBasicAudio class

![cbasicaudio class hierarchy](images/cutil07.png)

The `CBasicAudio` class implements the **IDispatch** methods of the [**IBasicAudio**](/windows/win32/Control/nn-control-ibasicaudio?branch=master) interface. The remaining **IBasicAudio** methods are left as pure virtual methods, and must be implemented by a derived class.

The **IDispatch** methods in this class are standard implementations, and are not described in detail here.
