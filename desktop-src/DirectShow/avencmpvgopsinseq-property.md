# AVEncMPVGOPSInSeq property

Specifies the number of groups of pictures (GOPs) between sequence headers. This property applies to MPEG video encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncMPVGOPSInSeq**

## Remarks

This property is read/write.
