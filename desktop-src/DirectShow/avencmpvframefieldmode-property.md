# AVEncMPVFrameFieldMode property

Specifies whether the encoder produces encoded fields or encoded frames. This property applies to MPEG video encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncMPVFrameFieldMode**

## Property value

The value of this property is a member of the [**eAVEncMPVFrameFieldMode**](/windows/win32/codecapi/ne-codecapi-eavencmpvframefieldmode?branch=master) enumeration.
