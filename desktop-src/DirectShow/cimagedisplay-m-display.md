# CImageDisplay::m\_Display member

The `m_Display` member is a [**VIDEOINFO**](/windows/win32/amvideo/ns-amvideo-tagvideoinfo?branch=master) structure that describes the current display format.

## Syntax


```C++
VIDEOINFO  m_Display;
```
