# AVEncVideoOutputColorPrimaries property

Specifies the color primaries for the encoded video.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncVideoOutputColorPrimaries**

## Property value

The value of this property is a member of the [**eAVEncVideoColorPrimaries**](/windows/win32/codecapi/ne-codecapi-eavencvideocolorprimaries?branch=master) enumeration.
