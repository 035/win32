# Shortcuts to C++ DVD Reference Pages

The following interfaces are those typically used when writing a DVD application in C++.



| Interface                                    | Purpose                                                                                          |
|----------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**IAMLine21Decoder**](iamline21decoder.md) | Controls closed captioning display.                                                              |
| [**IDvdCmd**](idvdcmd.md)                   | Used in advanced command synchronization techniques to control a command object.                 |
| [**IDvdControl2**](idvdcontrol2.md)         | Issues commands to the [DVD Navigator](dvd-navigator-filter.md) filter (the "set" methods).     |
| [**IDvdGraphBuilder**](idvdgraphbuilder.md) | Creates the DVD filter graph.                                                                    |
| [**IDvdInfo2**](idvdinfo2.md)               | Queries the DVD Navigator for state information and information on the disc (the "get" methods). |
| [**IDvdState**](idvdstate.md)               | Saves information, such as bookmarks, to disc.                                                   |
| [**IVideoFrameStep**](ivideoframestep.md)   | Controls "frame stepping" if the decoder supports it.                                            |
