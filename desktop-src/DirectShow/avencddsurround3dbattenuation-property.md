# AVEncDDSurround3dBAttenuation property

Specifies whether the Surround channels in a Dolby Digital audio stream are attenuated 3 decibels (dB) before encoding. This property applies to Dolby Digital audio encoders.

This property is read/write.

## Data type

**VARIANT\_BOOL** (**VT\_BOOL**)

## Property GUID

**CODECAPI\_AVEncDDSurround3dBAttenuation**
