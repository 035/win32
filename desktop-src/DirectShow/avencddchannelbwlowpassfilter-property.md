# AVEncDDChannelBWLowPassFilter property

Specifies whether a low pass filter is applied to the main input channels in a Dolby Digital audio stream. This property applies to Dolby Digital audio encoders.

This property is read/write.

## Data type

**VARIANT\_BOOL** (**VT\_BOOL**)

## Property GUID

**CODECAPI\_AVEncDDChannelBWLowPassFilter**
