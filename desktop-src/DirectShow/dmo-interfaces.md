# DMO Interfaces

This section contains reference entries for all the Microsoft DirectX Media Object (DMO) COM interfaces and their methods.



| Interface                                                            | Description                                                              |
|----------------------------------------------------------------------|--------------------------------------------------------------------------|
| [**IDMOQualityControl**](/windows/win32/Mediaobj/nn-mediaobj-idmoqualitycontrol?branch=master)                     | Supports quality control on a DMO.                                       |
| [**IDMOVideoOutputOptimizations**](/windows/win32/Mediaobj/nn-mediaobj-idmovideooutputoptimizations?branch=master) | Supports video optimizations on a DMO.                                   |
| [**IEnumDMO**](/windows/win32/Mediaobj/nn-mediaobj-ienumdmo?branch=master)                                         | Provides methods for enumerating DMOs.                                   |
| [**IMediaBuffer**](/windows/win32/Mediaobj/nn-mediaobj-imediabuffer?branch=master)                                 | Provides methods for manipulating a data buffer.                         |
| [**IMediaObject**](/windows/win32/Mediaobj/nn-mediaobj-imediaobject?branch=master)                                 | Provides methods for manipulating a DMO.                                 |
| [**IMediaObjectInPlace**](/windows/win32/Mediaobj/nn-mediaobj-imediaobjectinplace?branch=master)                   | Provides methods for processing data in place.                           |
| [**IMediaParamInfo**](/windows/win32/Medparam/nn-medparam-imediaparaminfo?branch=master)                           | Retrieves information about the media parameters that an object supports |
| [**IMediaParams**](/windows/win32/Medparam/nn-medparam-imediaparams?branch=master)                                 | Sets and retrieves media parameters on an object.                        |
