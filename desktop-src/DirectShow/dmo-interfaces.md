# DMO Interfaces

This section contains reference entries for all the Microsoft DirectX Media Object (DMO) COM interfaces and their methods.



| Interface                                                            | Description                                                              |
|----------------------------------------------------------------------|--------------------------------------------------------------------------|
| [**IDMOQualityControl**](idmoqualitycontrol.md)                     | Supports quality control on a DMO.                                       |
| [**IDMOVideoOutputOptimizations**](idmovideooutputoptimizations.md) | Supports video optimizations on a DMO.                                   |
| [**IEnumDMO**](ienumdmo.md)                                         | Provides methods for enumerating DMOs.                                   |
| [**IMediaBuffer**](imediabuffer.md)                                 | Provides methods for manipulating a data buffer.                         |
| [**IMediaObject**](imediaobject.md)                                 | Provides methods for manipulating a DMO.                                 |
| [**IMediaObjectInPlace**](imediaobjectinplace.md)                   | Provides methods for processing data in place.                           |
| [**IMediaParamInfo**](imediaparaminfo.md)                           | Retrieves information about the media parameters that an object supports |
| [**IMediaParams**](imediaparams.md)                                 | Sets and retrieves media parameters on an object.                        |
