# CMediaSample::m\_End member

Sample end time. This value is valid only if the [**CMediaSample::m\_dwFlags**](cmediasample-m-dwflags.md) member variable contains the AM\_SAMPLE\_STOPVALID flag.

## Syntax


```C++
REFERENCE_TIME m_End;
```
