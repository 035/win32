# AVEncVideoInputColorPrimaries property

Specifies the color primaries for the input video.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncVideoInputColorPrimaries**

## Property value

The value of this property is a member of the [**eAVEncVideoColorPrimaries**](/windows/win32/codecapi/ne-codecapi-eavencvideocolorprimaries?branch=master) enumeration.
