# AVEncVideoOutputScanType property

Specifies how the encoder interlaces the output video.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncVideoOutputScanType**

## Property value

The value of this property is a member of the [**eAVEncVideoOutputScanType**](eavencvideooutputscantype.md) enumeration.
