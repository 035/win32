# AVEncVideoInputColorTransferFunction property

Specifies the conversion function from RGB to R'G'B' for input video.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncVideoInputColorTransferFunction**

## Property value

The value of this property is a member of the [**eAVEncVideoColorTransferFunction**](/windows/win32/codecapi/ne-codecapi-eavencvideocolortransferfunction?branch=master) enumeration.
