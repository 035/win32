# CMediaSample::m\_MediaEnd member

Media stop time. This value is calculated as an offset from the media start time, specified by the [**CMediaSample::m\_MediaStart**](cmediasample-m-mediastart.md) member variable.

## Syntax


```C++
LONG m_MediaEnd;
```
