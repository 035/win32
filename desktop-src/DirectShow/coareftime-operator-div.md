# COARefTime.operator/ method

This operator divides a reference time by a value.

## Syntax


```C++
COARefTime operator/(
   LONG l
);
```



## Parameters

<dl> <dt>

*l* 
</dt> <dd>

Divisor.

</dd> </dl>

## Return value

Returns a new **COARefTime** object equal to the quotient of this object and **l**.
