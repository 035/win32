# CBaseWindow::m\_RealizePalette member

Private message that realizes the palette.

## Syntax


```C++
UINT m_RealizePalette;
```



## Remarks

The [**CBaseWindow::SetPalette**](cbasewindow-setpalette.md) method sends this message.
