# AVEncMuxOutputStreamType property

Specifies the type of output stream produced by a multiplexer.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncMuxOutputStreamType**

## Property value

The value of this property is a member of the [**eAVEncMuxOutput**](/windows/win32/codecapi/?branch=master) enumeration.
