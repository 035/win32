# AVEncMPVQScaleType property

Specifies whether the quantizer scale is linear or non-linear. This property applies to MPEG video encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncMPVQScaleType**

## Property value

The value of this property is a member of the [**eAVEncMPVQScaleType**](/windows/win32/codecapi/ne-codecapi-eavencmpvqscaletype?branch=master) enumeration.
