# AVEncVideoOutputColorNominalRange property

Specifies the nominal range for the encoded video.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncVideoOutputColorNominalRange**

## Property value

The value of this property is a member of the [**eAVEncVideoColorNominalRange**](eavencvideocolornominalrange.md) enumeration.
