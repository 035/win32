# AVEncMPVLevel property

Specifies the MPEG-2 level. This property applies to MPEG video encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncMPVLevel**

## Property value

The value of this property is a member of the [**eAVEncMPVLevel**](/windows/win32/codecapi/ne-codecapi-eavencmpvlevel?branch=master) enumeration.
