# DMO Enumerated Types

This section describes the Microsoft DirectX Media Object (DMO) enumerated types.



| Enumerated type                                                            | Description                                                                  |
|----------------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**DMO\_ENUM\_FLAGS**](dmo-enum-flags.md)                                 | Defines flags that specify search criteria when enumerating DMOs.            |
| [**DMO\_INPUT\_DATA\_BUFFER\_FLAGS**](dmo-input-data-buffer-flags.md)     | Defines flags that describe an input buffer.                                 |
| [**DMO\_INPUT\_STREAM\_INFO\_FLAGS**](dmo-input-stream-info-flags.md)     | Defines flags that describe an input stream.                                 |
| [**DMO\_OUTPUT\_DATA\_BUFFER\_FLAGS**](dmo-output-data-buffer-flags.md)   | Defines flags that describe an output buffer.                                |
| [**DMO\_OUTPUT\_STREAM\_INFO\_FLAGS**](dmo-output-stream-info-flags.md)   | Defines flags that describe an output stream.                                |
| [**DMO\_PROCESS\_OUTPUT\_FLAGS**](dmo-process-output-flags.md)            | Defines flags that specify output processing requests.                       |
| [**DMO\_REGISTER\_FLAGS**](dmo-register-flags.md)                         | Defines flags that specify registry information for a DMO.                   |
| [**DMO\_SET\_TYPE\_FLAGS**](dmo-set-type-flags.md)                        | Defines flags for setting the media type on a stream.                        |
| [**DMO\_VIDEO\_OUTPUT\_STREAM\_FLAGS**](dmo-video-output-stream-flags.md) | Defines flags that describe requested features for video optimizations.      |
| [**MP\_CURVE\_TYPE**](mp-curve-type.md)                                   | Defines the curve that a media parameter follows within an envelope segment. |
| [**MP\_TYPE**](mp-type.md)                                                | Specifies the data type for a parameter.                                     |
