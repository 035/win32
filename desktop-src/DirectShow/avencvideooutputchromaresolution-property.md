# AVEncVideoOutputChromaResolution property

Specifies the chroma resolution of the encoded video.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncVideoOutputChromaResolution**

## Property value

The value of this property is a member of the [**eAVEncVideoChromaResolution**](/windows/win32/codecapi/?branch=master) enumeration.
