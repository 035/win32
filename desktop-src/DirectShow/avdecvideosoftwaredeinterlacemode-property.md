# AVDecVideoSoftwareDeinterlaceMode property

Specifies the decoder's software deinterlace mode.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVDecVideoSoftwareDeinterlaceMode**

## Property value

The value of this property is a member of the [**eAVDecVideoSoftwareDeinterlaceMode**](eavdecvideosoftwaredeinterlacemode.md) enumeration.
