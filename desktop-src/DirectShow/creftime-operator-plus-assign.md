# CRefTime.operator+= method

The += operator adds two reference times.

## Syntax


```C++
CRefTime&amp; operator+=(
  [ref] const CRefTime &amp;rt
);
```



## Parameters

<dl> <dt>

*rt* \[ref\]
</dt> <dd>

Reference to a **CRefTime** object.

</dd> </dl>

## Return value

Returns a reference to the object.
