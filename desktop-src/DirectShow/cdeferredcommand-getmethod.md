# CDeferredCommand.GetMethod method

The `GetMethod` method retrieves the dispatch identifier (DISPID) of the method to be run.

## Syntax


```C++
long GetMethod();
```



## Parameters

This method has no parameters.

## Return value

Returns the DISPID of the method to run.
