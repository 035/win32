# CVideoTransformFilter::m\_bQualityChanged member

Boolean value that indicates whether the filter has dropped frames. This member variable controls when the filter sends [**EC\_QUALITY\_CHANGE**](ec-quality-change.md) events.

## Syntax


```C++
BOOL m_bQualityChanged;
```
