# AVEncDDAtoDConverterType property

Specifies the type of analog-to-digital (A/D) conversion for a Dolby Digital audio stream. This property applies to Dolby Digital audio encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncDDAtoDConverterType**

## Property value

The value of this property is a member of the [**eAVEncDDAtoDConverterType**](eavencddatodconvertertype.md) enumeration.
