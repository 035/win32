# External Device Control Interfaces

These interfaces support application control over external devices, such as DV camcorders.



| Interface                                            | Description                                                                             |
|------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**IAMCameraControl**](iamcameracontrol.md)         | Control a camera.                                                                       |
| [**IAMExtDevice**](iamextdevice.md)                 | Control an external device such as a VCR or camcorder.                                  |
| [**IAMExtTransport**](iamexttransport.md)           | Control VCR transport functions, such as play, pause, record, fast-forward, and rewind. |
| [**IAMTimecodeDisplay**](iamtimecodedisplay.md)     | Set properties on a device that displays SMPTE timecode.                                |
| [**IAMTimecodeGenerator**](iamtimecodegenerator.md) | Set properties on a device that generates SMPTE timecode.                               |
| [**IAMTimecodeReader**](iamtimecodereader.md)       | Read SMPTE timecode from an external device.                                            |
