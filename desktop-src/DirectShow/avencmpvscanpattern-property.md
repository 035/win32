# AVEncMPVScanPattern property

Specifies the macroblock scan pattern. This property applies to MPEG video encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncMPVScanPattern**

## Property value

The value of this property is a member of the [**eAVEncMPVScanPattern**](/windows/win32/codecapi/?branch=master) enumeration.
