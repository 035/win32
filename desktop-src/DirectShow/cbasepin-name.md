# CBasePin.Name method

The `Name` method retrieves the pin identifier.

## Syntax


```C++
LPWSTR Name();
```



## Parameters

This method has no parameters.

## Return value

Returns the value of the [**CBasePin::m\_pName**](cbasepin-m-pname.md) member variable.
