# COARefTime.operator REFERENCE\_TIME method

This operator retrieves the reference time as a **REFERENCE\_TIME** value.

## Syntax


```C++
REFERENCE_TIME operator REFERENCE_TIME();
```



## Parameters

This method has no parameters.

## Return value

Returns the reference time in 100-nanosecond units.
