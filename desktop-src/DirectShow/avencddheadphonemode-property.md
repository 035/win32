# AVEncDDHeadphoneMode property

Specifies the headphone mode for a Dolby Digital audio stream. This property applies to Dolby Digital audio encoders.

This property is read/write.

## Data type

**UINT32** (**VT\_UI4**)

## Property GUID

**CODECAPI\_AVEncDDHeadphoneMode**

## Property value

The value of this property is a member of the [**eAVEncDDHeadphoneMode**](/windows/win32/codecapi/ne-codecapi-eavencddheadphonemode?branch=master) enumeration.
