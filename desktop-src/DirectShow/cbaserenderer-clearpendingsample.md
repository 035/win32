# CBaseRenderer.ClearPendingSample method

The `ClearPendingSample` method releases the current sample.

## Syntax


```C++
virtual HRESULT ClearPendingSample();
```



## Parameters

This method has no parameters.

## Return value

Returns S\_OK.
