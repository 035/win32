# DMO Structures

This section describes the DirectX Media Object (DMO) structures.



| Structure                                                   | Description                                                 |
|-------------------------------------------------------------|-------------------------------------------------------------|
| [**DMO\_MEDIA\_TYPE**](dmo-media-type.md)                  | Describes the format of the data used by a stream in a DMO. |
| [**DMO\_OUTPUT\_DATA\_BUFFER**](dmo-output-data-buffer.md) | Describes an output buffer used by a DMO.                   |
| [**DMO\_PARTIAL\_MEDIATYPE**](dmo-partial-mediatype.md)    | Partially describes a media type used by a DMO.             |
| [**MP\_ENVELOPE\_SEGMENT**](mp-envelope-segment.md)        | Defines an envelope segment used by a media parameter.      |
| [**MP\_PARAMINFO**](mp-paraminfo.md)                       | Contains information about a media parameter.               |
