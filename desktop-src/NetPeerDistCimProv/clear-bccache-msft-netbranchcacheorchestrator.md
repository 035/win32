# Clear\_BCCache method of the MSFT\_NetBranchCacheOrchestrator class

Deletes all data in all cache files.

## Syntax


```mof
uint32 Clear_BCCache(
  [in] boolean Force
);
```



## Parameters

<dl> <dt>

*Force* \[in\]
</dt> <dd>

Indicates the operation should not prompt for confirmation.

</dd> </dl>
