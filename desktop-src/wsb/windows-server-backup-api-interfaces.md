# Windows Server Backup API Interfaces

The following interfaces are used with the Windows Server Backup API.



| Interface                                                              | Description                                                                                 |
|------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| [**IWsbApplicationAsync**](/windows/previous-versions/WsbApp/nn-wsbapp-iwsbapplicationasync?branch=master)                   | Defines methods to monitor and control the progress of an asynchronous operation.           |
| [**IWsbApplicationBackupSupport**](/windows/previous-versions/WsbApp/nn-wsbapp-iwsbapplicationbackupsupport?branch=master)   | Defines a method for checking the consistency of the application's VSS writer's components. |
| [**IWsbApplicationRestoreSupport**](/windows/previous-versions/WsbApp/nn-wsbapp-iwsbapplicationrestoresupport?branch=master) | Defines methods for performing application-specific restore tasks.                          |
