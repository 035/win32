# Windows Server Backup API Interfaces

The following interfaces are used with the Windows Server Backup API.



| Interface                                                              | Description                                                                                 |
|------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| [**IWsbApplicationAsync**](iwsbapplicationasync.md)                   | Defines methods to monitor and control the progress of an asynchronous operation.           |
| [**IWsbApplicationBackupSupport**](iwsbapplicationbackupsupport.md)   | Defines a method for checking the consistency of the application's VSS writer's components. |
| [**IWsbApplicationRestoreSupport**](iwsbapplicationrestoresupport.md) | Defines methods for performing application-specific restore tasks.                          |
