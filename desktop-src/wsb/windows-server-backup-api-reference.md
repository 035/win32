# Windows Server Backup API Reference

The following elements are used with the Windows Server Backup API.

## In this section

<dl> <dt>

[Windows Server Backup API Interfaces](windows-server-backup-api-interfaces.md)
</dt> <dd>

The following interfaces are used with the Windows Server Backup API.

</dd> <dt>

[Windows Server Backup API Registry Keys](windows-server-backup-api-registry-keys.md)
</dt> <dd>

To register with the Windows Server Backup feature, an application must contain a Volume Shadow Copy Service (VSS) writer and must set the **Application Support** and **Application Identifier** registry keys. An application that implements an add-in using the [Windows Server Backup API interfaces](windows-server-backup-api-interfaces.md) must also set the **CLSID** key.

</dd> <dt>

[Managed Reference for Windows Server Backup PowerShell Command Classes](managed-reference-for-windows-server-backup-powershell-command-classes.md)
</dt> <dd>

[Windows Server Backup](http://go.microsoft.com/fwlink/p/?linkid=111182) can use Windows PowerShell cmdlets to create and manage backups in Windows Server 2008 R2.

</dd> </dl>
