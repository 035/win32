# Cloud Backup Provider API Structures

The following structures are used with the Cloud Backup Provider API.

## In this section

<dl> <dt>

[**WSB\_OB\_REGISTRATION\_INFO**](/previous-versions/windows/desktop/api/WsbOnline/ns-wsbonline-_wsb_ob_registration_info)
</dt> <dd>

Contains information to register a cloud backup provider with Windows Server Backup.

</dd> <dt>

[**WSB\_OB\_STATUS\_ENTRY**](/previous-versions/windows/desktop/api/WsbOnline/ns-wsbonline-_wsb_ob_status_entry)
</dt> <dd>

Contains status information for one entry to be shown in the Windows Server Backup MMC snap-in.

</dd> <dt>

[**WSB\_OB\_STATUS\_ENTRY\_VALUE\_TYPE\_PAIR**](/previous-versions/windows/desktop/api/WsbOnline/ns-wsbonline-_wsb_ob_status_entry_value_type_pair)
</dt> <dd>

Contains the value and value type for a parameter used to expand the value resource string.

</dd> <dt>

[**WSB\_OB\_STATUS\_INFO**](/previous-versions/windows/desktop/api/WsbOnline/ns-wsbonline-_wsb_ob_status_info)
</dt> <dd>

Contains information to update the cloud backup provider status in the Windows Server Backup MMC snap-in.

</dd> </dl>
