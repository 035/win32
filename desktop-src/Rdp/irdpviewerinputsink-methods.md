# IRDPViewerInputSink Methods

\[The [**IRDPViewerInputSink**](/windows/win32/RdpEncomAPI/nn-rdpencomapi-irdpviewerinputsink?branch=master) interface is no longer available for use for UWP applications as of Windows 10, version 1709. It is still supported for Desktop apps.\]

The [**IRDPViewerInputSink**](/windows/win32/RdpEncomAPI/nn-rdpencomapi-irdpviewerinputsink?branch=master) interface exposes the following methods.

## In this section

<dl> <dt>

[**AddTouchInput method**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpviewerinputsink-addtouchinput?branch=master)
</dt> <dd>

Accepts a description of a touch input.

</dd> <dt>

[**BeginTouchFrame method**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpviewerinputsink-begintouchframe?branch=master)
</dt> <dd>

Begins to accept a series of touch inputs.

</dd> <dt>

[**EndTouchFrame method**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpviewerinputsink-endtouchframe?branch=master)
</dt> <dd>

Stops to accept a series of touch inputs.

</dd> <dt>

[**SendKeyboardEvent method**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpviewerinputsink-sendkeyboardevent?branch=master)
</dt> <dd>

Sends a keyboard event message.

</dd> <dt>

[**SendMouseButtonEvent method**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpviewerinputsink-sendmousebuttonevent?branch=master)
</dt> <dd>

Sends a mouse button event message.

</dd> <dt>

[**SendMouseMoveEvent method**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpviewerinputsink-sendmousemoveevent?branch=master)
</dt> <dd>

Sends a mouse move event message.

</dd> <dt>

[**SendMouseWheelEvent method**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpviewerinputsink-sendmousewheelevent?branch=master)
</dt> <dd>

Sends a mouse wheel event message.

</dd> <dt>

[**SendSyncEvent method**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpviewerinputsink-sendsyncevent?branch=master)
</dt> <dd>

Sends an event message to indicate a change in the state of the keyboard, such as when the Caps Lock key is pressed.

</dd> </dl>
