# IRDPViewerInputSink Methods

\[The [**IRDPViewerInputSink**](irdpviewerinputsink.md) interface is no longer available for use for UWP applications as of Windows 10, version 1709. It is still supported for Desktop apps.\]

The [**IRDPViewerInputSink**](irdpviewerinputsink.md) interface exposes the following methods.

## In this section

<dl> <dt>

[**AddTouchInput method**](irdpviewerinputsink-addtouchinput.md)
</dt> <dd>

Accepts a description of a touch input.

</dd> <dt>

[**BeginTouchFrame method**](irdpviewerinputsink-begintouchframe.md)
</dt> <dd>

Begins to accept a series of touch inputs.

</dd> <dt>

[**EndTouchFrame method**](irdpviewerinputsink-endtouchframe.md)
</dt> <dd>

Stops to accept a series of touch inputs.

</dd> <dt>

[**SendKeyboardEvent method**](irdpviewerinputsink-sendkeyboardevent.md)
</dt> <dd>

Sends a keyboard event message.

</dd> <dt>

[**SendMouseButtonEvent method**](irdpviewerinputsink-sendmousebuttonevent.md)
</dt> <dd>

Sends a mouse button event message.

</dd> <dt>

[**SendMouseMoveEvent method**](irdpviewerinputsink-sendmousemoveevent.md)
</dt> <dd>

Sends a mouse move event message.

</dd> <dt>

[**SendMouseWheelEvent method**](irdpviewerinputsink-sendmousewheelevent.md)
</dt> <dd>

Sends a mouse wheel event message.

</dd> <dt>

[**SendSyncEvent method**](irdpviewerinputsink-sendsyncevent.md)
</dt> <dd>

Sends an event message to indicate a change in the state of the keyboard, such as when the Caps Lock key is pressed.

</dd> </dl>
