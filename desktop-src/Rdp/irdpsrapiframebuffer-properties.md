# IRDPSRAPIFrameBuffer Properties

The [**IRDPSRAPIFrameBuffer**](irdpsrapiframebuffer.md) interface exposes the following properties.

## In this section

<dl> <dt>

[**Bpp property**](irdpsrapiframebuffer-bpp.md)
</dt> <dd>

The bits per pixel for the frame buffer.

</dd> <dt>

[**Height property**](irdpsrapiframebuffer-height.md)
</dt> <dd>

Height, in pixels, of the frame buffer.

</dd> <dt>

[**Width property**](irdpsrapiframebuffer-width.md)
</dt> <dd>

Width, in pixels, of the frame buffer.

</dd> </dl>
