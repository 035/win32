# IRDPSRAPIFrameBuffer Properties

The [**IRDPSRAPIFrameBuffer**](/windows/win32/RdpEncomAPI/nn-rdpencomapi-irdpsrapiframebuffer?branch=master) interface exposes the following properties.

## In this section

<dl> <dt>

[**Bpp property**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpsrapiframebuffer-get_bpp?branch=master)
</dt> <dd>

The bits per pixel for the frame buffer.

</dd> <dt>

[**Height property**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpsrapiframebuffer-get_height?branch=master)
</dt> <dd>

Height, in pixels, of the frame buffer.

</dd> <dt>

[**Width property**](/windows/win32/RdpEncomAPI/nf-rdpencomapi-irdpsrapiframebuffer-get_width?branch=master)
</dt> <dd>

Width, in pixels, of the frame buffer.

</dd> </dl>
