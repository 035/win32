# Windows Desktop Sharing Classes

Windows Desktop Sharing defines the following classes.

## In this section

<dl> <dt>

[**RDPSession**](/windows/desktop/api/RdpEncomAPI/)
</dt> <dd>

The [**RDPSession**](/windows/desktop/api/RdpEncomAPI/) class includes the [**IRDPSRAPISharingSession**](/windows/desktop/api/RdpEncomAPI/nn-rdpencomapi-irdpsrapisharingsession), [**IRDPSRAPISharingSession2**](/windows/desktop/api/RdpEncomAPI/nn-rdpencomapi-irdpsrapisharingsession2), and the [**\_IRDPSessionEvents**](/windows/desktop/api/RdpEncomAPI/) interfaces.

</dd> <dt>

[**RDPSRAPIFrameBuffer**](/windows/desktop/api/RdpEncomAPI/)
</dt> <dd>

The [**RDPSRAPIFrameBuffer**](/windows/desktop/api/RdpEncomAPI/) class includes the [**IRDPSRAPIFrameBuffer**](/windows/desktop/api/RdpEncomAPI/nn-rdpencomapi-irdpsrapiframebuffer) interface.

</dd> <dt>

[**RDPTransportStreamBuffer**](/windows/desktop/api/RdpEncomAPI/)
</dt> <dd>

The [**RDPTransportStreamBuffer**](/windows/desktop/api/RdpEncomAPI/) class includes the [**IRDPSRAPITransportStreamBuffer**](/windows/desktop/api/RdpEncomAPI/nn-rdpencomapi-irdpsrapitransportstreambuffer) interface.

</dd> <dt>

[**RDPTransportStreamEvents**](/windows/desktop/api/RdpEncomAPI/)
</dt> <dd>

The [**RDPTransportStreamEvents**](/windows/desktop/api/RdpEncomAPI/) class includes the [**IRDPSRAPITransportStreamEvents**](/windows/desktop/api/RdpEncomAPI/nn-rdpencomapi-irdpsrapitransportstreamevents) interface.

</dd> </dl>
