# Windows Desktop Sharing Classes

Windows Desktop Sharing defines the following classes.

## In this section

<dl> <dt>

[**RDPSession**](/windows/win32/RdpEncomAPI/?branch=master)
</dt> <dd>

The [**RDPSession**](/windows/win32/RdpEncomAPI/?branch=master) class includes the [**IRDPSRAPISharingSession**](/windows/win32/RdpEncomAPI/nn-rdpencomapi-irdpsrapisharingsession?branch=master), [**IRDPSRAPISharingSession2**](/windows/win32/RdpEncomAPI/nn-rdpencomapi-irdpsrapisharingsession2?branch=master), and the [**\_IRDPSessionEvents**](/windows/win32/RdpEncomAPI/?branch=master) interfaces.

</dd> <dt>

[**RDPSRAPIFrameBuffer**](/windows/win32/RdpEncomAPI/?branch=master)
</dt> <dd>

The [**RDPSRAPIFrameBuffer**](/windows/win32/RdpEncomAPI/?branch=master) class includes the [**IRDPSRAPIFrameBuffer**](/windows/win32/RdpEncomAPI/nn-rdpencomapi-irdpsrapiframebuffer?branch=master) interface.

</dd> <dt>

[**RDPTransportStreamBuffer**](/windows/win32/RdpEncomAPI/?branch=master)
</dt> <dd>

The [**RDPTransportStreamBuffer**](/windows/win32/RdpEncomAPI/?branch=master) class includes the [**IRDPSRAPITransportStreamBuffer**](/windows/win32/RdpEncomAPI/nn-rdpencomapi-irdpsrapitransportstreambuffer?branch=master) interface.

</dd> <dt>

[**RDPTransportStreamEvents**](/windows/win32/RdpEncomAPI/?branch=master)
</dt> <dd>

The [**RDPTransportStreamEvents**](/windows/win32/RdpEncomAPI/?branch=master) class includes the [**IRDPSRAPITransportStreamEvents**](/windows/win32/RdpEncomAPI/nn-rdpencomapi-irdpsrapitransportstreamevents?branch=master) interface.

</dd> </dl>
