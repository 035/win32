# IRDPSRAPIAudioStream Methods

The [**IRDPSRAPIAudioStream**](/windows/desktop/api/RdpEncomAPI/nn-rdpencomapi-irdpsrapiaudiostream) interface exposes the following methods.

## In this section

<dl> <dt>

[**FreeBuffer method**](/windows/desktop/api/RdpEncomAPI/nf-rdpencomapi-irdpsrapiaudiostream-freebuffer)
</dt> <dd>

Releases the hold on the buffer after the [**GetBuffer**](/windows/desktop/api/RdpEncomAPI/nf-rdpencomapi-irdpsrapiaudiostream-getbuffer) method is called.

</dd> <dt>

[**GetBuffer method**](/windows/desktop/api/RdpEncomAPI/nf-rdpencomapi-irdpsrapiaudiostream-getbuffer)
</dt> <dd>

Gets audio data from the buffer.

</dd> <dt>

[**Initialize method**](/windows/desktop/api/RdpEncomAPI/nf-rdpencomapi-irdpsrapiaudiostream-initialize)
</dt> <dd>

Initializes the audio stream.

</dd> <dt>

[**Start method**](/windows/desktop/api/RdpEncomAPI/nf-rdpencomapi-irdpsrapiaudiostream-start)
</dt> <dd>

Starts the audio stream.

</dd> <dt>

[**Stop method**](/windows/desktop/api/RdpEncomAPI/nf-rdpencomapi-irdpsrapiaudiostream-stop)
</dt> <dd>

Stops the audio stream.

</dd> </dl>
