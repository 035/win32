# Windows Desktop Sharing Enumerations

Windows Desktop Sharing defines the following enumerations.

## In this section

<dl> <dt>

[**ATTENDEE\_DISCONNECT\_REASON**](/windows/win32/Rdpencomapi/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0002?branch=master)
</dt> <dd>

Defines values for the reasons why an attendee was disconnected from the session.

</dd> <dt>

[**CHANNEL\_ACCESS\_ENUM**](/windows/win32/Rdpencomapi/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0005?branch=master)
</dt> <dd>

Defines values for the type of access granted to the attendee for the channel.

</dd> <dt>

[**CHANNEL\_FLAGS**](/windows/win32/Rdpencomapi/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0004?branch=master)
</dt> <dd>

Defines values for how data is sent on the channel.

</dd> <dt>

[**CHANNEL\_PRIORITY**](/windows/win32/Rdpencomapi/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0003?branch=master)
</dt> <dd>

Defines values for the priority used to send packets on the channel.

</dd> <dt>

[**CTRL\_LEVEL**](/windows/win32/Rdpencomapi/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0001?branch=master)
</dt> <dd>

Defines the level of control that an attendee has on a session.

</dd> <dt>

[**RDPENCOMAPI\_ATTENDEE\_FLAGS**](/windows/win32/Rdpencomapi/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0006?branch=master)
</dt> <dd>

Defines values for the type of attendee.

</dd> <dt>

[**RDPENCOMAPI\_CONSTANTS**](/windows/win32/Rdpencomapi/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0028_0001?branch=master)
</dt> <dd>

Defines values for some of the constants used in this API.

</dd> <dt>

[**RDPSRAPI\_APP\_FLAGS**](/windows/win32/Rdpencomapi/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0008?branch=master)
</dt> <dd>

Defines values for the type of application.

</dd> <dt>

[**RDPSRAPI\_KBD\_CODE\_TYPE**](/windows/win32/RdpEncomAPI/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0010?branch=master)
</dt> <dd>

Defines values for the type of encoding of a keyboard.

</dd> <dt>

[**RDPSRAPI\_KBD\_SYNC\_FLAG**](/windows/win32/RdpEncomAPI/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0011?branch=master)
</dt> <dd>

Defines values for the type of keys that control the state of a keyboard, such as the Caps Lock key.

</dd> <dt>

[**RDPSRAPI\_MOUSE\_BUTTON\_TYPE**](/windows/win32/RdpEncomAPI/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0009?branch=master)
</dt> <dd>

Defines values for the type of mouse buttons.

</dd> <dt>

[**RDPSRAPI\_WND\_FLAGS**](/windows/win32/Rdpencomapi/ne-rdpencomapi-__midl___midl_itf_rdpencomapi_0000_0000_0007?branch=master)
</dt> <dd>

Defines values for the type of window.

</dd> </dl>
