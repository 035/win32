# Update Assessor Interfaces

The following interfaces are used by the Update Accessor provider.



| Function                               | Description                                                                                                                 |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [**IWaaSAssessor**](/windows/win32/waasapi/nn-waasapi-iwaasassessor?branch=master) | Gets the OS update assessment by comparing the latest build from Microsoft against the build running on the current device. |
