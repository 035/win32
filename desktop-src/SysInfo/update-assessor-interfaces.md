# Update Assessor Interfaces

The following interfaces are used by the Update Accessor provider.



| Function                               | Description                                                                                                                 |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [**IWaaSAssessor**](iwaasassessor.md) | Gets the OS update assessment by comparing the latest build from Microsoft against the build running on the current device. |
