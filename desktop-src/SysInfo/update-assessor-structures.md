# Update Assessor Structures

The following functions are used by the Update Accessor provider.



| Function                                         | Description                                                           |
|--------------------------------------------------|-----------------------------------------------------------------------|
| [**OSUpdateAssessment**](osupdateassessment.md) | Defines how up-to-date the OS on a targeted device is.                |
| [**UpdateAssessment**](osupdateassessment.md)   | Contains information that assesses how up-to-date an installed OS is. |
