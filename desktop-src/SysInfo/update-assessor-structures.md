# Update Assessor Structures

The following functions are used by the Update Accessor provider.



| Function                                         | Description                                                           |
|--------------------------------------------------|-----------------------------------------------------------------------|
| [**OSUpdateAssessment**](/windows/win32/waasapitypes/ns-waasapitypes-tagosupdateassessment?branch=master) | Defines how up-to-date the OS on a targeted device is.                |
| [**UpdateAssessment**](/windows/win32/waasapitypes/ns-waasapitypes-tagosupdateassessment?branch=master)   | Contains information that assesses how up-to-date an installed OS is. |
