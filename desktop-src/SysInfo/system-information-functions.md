# System Information Functions

The following functions are used to retrieve or set system information.



| Function                                                                     | Description                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CeipIsOptedIn**](/windows/win32/Windowsceip/nf-windowsceip-ceipisoptedin?branch=master)                                       | Checks whether the user has opted in for SQM data collection as part of the Customer Experience Improvement Program (CEIP).                                                                                                       |
| [**DnsHostnameToComputerName**](/windows/win32/Winbase/nf-winbase-dnshostnametocomputernamea?branch=master)               | Converts a DNS name to a NetBIOS name.                                                                                                                                                                                            |
| [**EnumSystemFirmwareTables**](/windows/win32/Winbase/?branch=master)                 | Enumerates all system firmware tables of the specified type.                                                                                                                                                                      |
| [**ExpandEnvironmentStrings**](/windows/win32/Winbase/?branch=master)                 | Replaces environment-variable strings with their defined values.                                                                                                                                                                  |
| [**GetComputerName**](/windows/win32/Winbase/nf-winbase-getcomputernamea?branch=master)                                   | Retrieves the NetBIOS name of the local computer.                                                                                                                                                                                 |
| [**GetComputerNameEx**](/windows/win32/Winbase/?branch=master)                               | Retrieves the NetBIOS or DNS name of the local computer.                                                                                                                                                                          |
| [**GetComputerObjectName**](/windows/win32/Secext/nf-secext-getcomputerobjectnamea?branch=master)                       | Retrieves the local computer's name in a specified format.                                                                                                                                                                        |
| [**GetCurrentHwProfile**](/windows/win32/Winbase/nf-winbase-getcurrenthwprofilea?branch=master)                           | Retrieves the current hardware profile for the local computer.                                                                                                                                                                    |
| [**GetFirmwareEnvironmentVariable**](/windows/win32/Winbase/nf-winbase-getfirmwareenvironmentvariablea?branch=master)     | Retrieves the value of the specified firmware environment variable from NVRAM.                                                                                                                                                    |
| [**GetFirmwareEnvironmentVariableEx**](/windows/win32/Winbase/nf-winbase-getfirmwareenvironmentvariableexa?branch=master) | Retrieves the value of the specified firmware environment variable and its attributes.                                                                                                                                            |
| [**GetFirmwareType**](/windows/win32/Winbase/nf-winbase-getfirmwaretype?branch=master)                                   | Retrieves the firmware type of the local computer.                                                                                                                                                                                |
| [**GetIntegratedDisplaySize**](/windows/win32/winbase/?branch=master)                 | Retrieves the best estimate of the diagonal size of the built-in screen, in inches.                                                                                                                                               |
| [**GetNativeSystemInfo**](/windows/win32/Winbase/?branch=master)                           | Retrieves information about the current system for an application running under WOW64.                                                                                                                                            |
| [**GetProductInfo**](/windows/win32/Winbase/?branch=master)                                     | Retrieves the product type for the operating system on the local computer, and maps the type to the product types supported by the specified operating system.                                                                    |
| [**GetSystemDirectory**](/windows/win32/Winbase/?branch=master)                             | Retrieves the path of the system directory.                                                                                                                                                                                       |
| [**GetSystemFirmwareTable**](/windows/win32/Winbase/?branch=master)                     | Retrieves the specified firmware table from the firmware table provider.                                                                                                                                                          |
| [**GetSystemInfo**](/windows/win32/Winbase/?branch=master)                                       | Retrieves information about the current system.                                                                                                                                                                                   |
| [**GetSystemRegistryQuota**](/windows/win32/Winbase/nf-winbase-getsystemregistryquota?branch=master)                     | Retrieves the current size of the registry and the maximum size that the registry is allowed to attain on the system.                                                                                                             |
| [**GetSystemWindowsDirectory**](/windows/win32/Winbase/?branch=master)               | Retrieves the path of the shared Windows directory on a multi-user system.                                                                                                                                                        |
| [**GetSystemWow64Directory**](getsystemwow64directory.md)                   | Retrieves the path of the system directory used by WOW64.                                                                                                                                                                         |
| [**GetSystemWow64Directory2**](/windows/win32/wow64apiset/nf-wow64apiset-getsystemwow64directory2a?branch=master)                 | Retrieves the path of the system directory used by WOW64, using the specified image file machine type.                                                                                                                            |
| [**GetUserName**](/windows/win32/Winbase/nf-winbase-getusernamea?branch=master)                                           | Retrieves the user name of the current thread.                                                                                                                                                                                    |
| [**GetUserNameEx**](/windows/win32/Secext/nf-secext-getusernameexa?branch=master)                                       | Retrieves the name of the user or other security principal associated with the calling thread. You can specify the format of the returned name.                                                                                   |
| [**GetVersion**](/windows/win32/Winbase/?branch=master)                                             | Retrieves the version number of the current operating system.                                                                                                                                                                     |
| [**GetVersionEx**](/windows/win32/Winbase/?branch=master)                                         | Retrieves information about the current operating system.                                                                                                                                                                         |
| [**GetWindowsDirectory**](/windows/win32/Winbase/?branch=master)                           | Retrieves the path of the Windows directory.                                                                                                                                                                                      |
| [**IsNativeVhdBoot**](/windows/win32/Winbase/nf-winbase-isnativevhdboot?branch=master)                                   | Indicates if the OS was booted from a VHD container.                                                                                                                                                                              |
| [**IsWow64GuestMachineSupported**](/windows/win32/wow64apiset/nf-wow64apiset-iswow64guestmachinesupported?branch=master)         |                                                                                                                                                                                                                                   |
| [**IsProcessorFeaturePresent**](isprocessorfeaturepresent.md)               | Determines whether a processor feature is supported by the current computer.                                                                                                                                                      |
| [**IsWow64Message**](https://msdn.microsoft.com/library/windows/desktop/ms684136)                                    | Determines whether the last message read from the queue of the current process originated from a WOW64 process.                                                                                                                   |
| [**IsWow64Process**](https://msdn.microsoft.com/library/windows/desktop/ms684139)                                    | Determines whether a process is running under WOW64.                                                                                                                                                                              |
| [**QueryPerformanceCounter**](/windows/win32/Winbase/?branch=master)                   | Retrieves the current value of the high-resolution performance counter.                                                                                                                                                           |
| [**QueryPerformanceFrequency**](/windows/win32/Winbase/?branch=master)               | Retrieves the frequency of the high-resolution performance counter.                                                                                                                                                               |
| [**RtlGetSuiteMask**](rtlgetsuitemask.md)                                   | Retrieves a bit mask that identifies the product suites available on the system. If this function is called in an application that runs in the context of a server silo, the suite mask for the server silo is retrieved instead. |
| [**SetComputerName**](/windows/win32/Winbase/?branch=master)                                   | Sets a new NetBIOS name for the local computer.                                                                                                                                                                                   |
| [**SetComputerNameEx**](/windows/win32/Winbase/?branch=master)                               | Sets a new NetBIOS or DNS name for the local computer.                                                                                                                                                                            |
| [**SetFirmwareEnvironmentVariable**](/windows/win32/Winbase/nf-winbase-setfirmwareenvironmentvariablea?branch=master)     | Sets the value of the specified firmware environment variable.                                                                                                                                                                    |
| [**SetFirmwareEnvironmentVariableEx**](/windows/win32/Winbase/nf-winbase-setfirmwareenvironmentvariableexa?branch=master) | Sets the value of the specified firmware environment variable and its attributes.                                                                                                                                                 |
| [**TranslateName**](/windows/win32/Secext/nf-secext-translatenamea?branch=master)                                       | Converts a directory service object name from one format to another.                                                                                                                                                              |
| [**VerifyVersionInfo**](/windows/win32/Winbase/nf-winbase-verifyversioninfoa?branch=master)                               | Compares a set of version requirements to the values for the current operating system.                                                                                                                                            |
| [**VerSetConditionMask**](/windows/win32/Winnt/nf-winnt-versetconditionmask?branch=master)                           | Builds the condition mask for the [**VerifyVersionInfo**](/windows/win32/Winbase/nf-winbase-verifyversioninfoa?branch=master) function.                                                                                                                                        |
| [**Wow64DisableWow64FsRedirection**](https://msdn.microsoft.com/library/windows/desktop/aa365743)    | Disables file system redirection for the calling thread.                                                                                                                                                                          |
| [**Wow64EnableWow64FsRedirection**](https://msdn.microsoft.com/library/windows/desktop/aa365744)      | Enables or disables file system redirection for the calling thread.                                                                                                                                                               |
| [**Wow64RevertWow64FsRedirection**](https://msdn.microsoft.com/library/windows/desktop/aa365745)      | Restores file system redirection for the calling thread.                                                                                                                                                                          |
