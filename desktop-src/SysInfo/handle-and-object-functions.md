# Handle and Object Functions

The following functions are used with handles and objects.



| Function                                             | Description                                                                                  |
|------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**CloseHandle**](/windows/win32/Winbase/nf-lkmdtel-lkmdtelclosehandle?branch=master)                   | Closes an open object handle.                                                                |
| [**CompareObjectHandles**](/windows/win32/handleapi/nf-handleapi-compareobjecthandles?branch=master) | Compares two object handles to determine if they refer to the same underlying kernel object. |
| [**DuplicateHandle**](duplicatehandle.md)           | Duplicates an object handle.                                                                 |
| [**GetHandleInformation**](gethandleinformation.md) | Retrieves certain properties of an object handle.                                            |
| [**SetHandleInformation**](sethandleinformation.md) | Sets certain properties of an object handle.                                                 |
