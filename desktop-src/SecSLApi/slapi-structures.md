# SLAPI Structures

The Software Licensing API (SLAPI) uses the following structures.

## In this section



| Topic                                                                        | Description                                                                                                                                                                  |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SL\_ACTIVATION\_INFO\_HEADER**](sl-activation-info-header.md)<br/> | Specifies the product activation information.<br/>                                                                                                                     |
| [**SL\_AD\_ACTIVATION\_INFO**](sl-ad-activation-info.md)<br/>         | Specifies information used for the retail or Active Directory phone activation of a license.<br/>                                                                      |
| [**SL\_LICENSING\_STATUS**](sl-licensing-status.md)<br/>              | Represents the licensing status.<br/>                                                                                                                                  |
| [**SL\_NONGENUINE\_UI\_OPTIONS**](sl-nongenuine-ui-options.md)<br/>   | Specifies an application that displays a dialog box when the [**SLIsGenuineLocal**](slisgenuinelocal.md) function indicates that an installation is not genuine.<br/> |
