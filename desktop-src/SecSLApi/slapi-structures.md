# SLAPI Structures

The Software Licensing API (SLAPI) uses the following structures.

## In this section



| Topic                                                                        | Description                                                                                                                                                                  |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SL\_ACTIVATION\_INFO\_HEADER**](/windows/win32/slpublic/ns-slpublic-_tagsl_activation_info_header?branch=master)<br/> | Specifies the product activation information.<br/>                                                                                                                     |
| [**SL\_AD\_ACTIVATION\_INFO**](/windows/win32/slpublic/ns-slpublic-_tagsl_ad_activation_info?branch=master)<br/>         | Specifies information used for the retail or Active Directory phone activation of a license.<br/>                                                                      |
| [**SL\_LICENSING\_STATUS**](/windows/win32/slpublic/ns-slpublic-_tagsl_licensing_status?branch=master)<br/>              | Represents the licensing status.<br/>                                                                                                                                  |
| [**SL\_NONGENUINE\_UI\_OPTIONS**](/windows/win32/Slpublic/ns-slpublic-_tagsl_nongenuine_ui_options?branch=master)<br/>   | Specifies an application that displays a dialog box when the [**SLIsGenuineLocal**](/windows/win32/Slpublic/nf-slpublic-slisgenuinelocal?branch=master) function indicates that an installation is not genuine.<br/> |
