# SLAPI Functions

The Software Licensing API (SLAPI) uses the following functions.

## In this section



| Topic                                                                                           | Description                                                                                                                                                                                                                          |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SLActivateProduct**](/windows/desktop/api/slpublic/nf-slpublic-slactivateproduct)<br/>                                       | Acquires a use license from the Software License Server (SLS).<br/>                                                                                                                                                            |
| [**SLAcquireGenuineTicket**](/windows/desktop/api/Slpublic/nf-slpublic-slacquiregenuineticket)<br/>                             | Gets a XrML genuine ticket acquired from the Software Licensing Server (SLS).<br/>                                                                                                                                             |
| [**SLClose**](/windows/desktop/api/slpublic/nf-slpublic-slclose)<br/>                                                           | Closes the Software Licensing Client (SLC) context handle.<br/>                                                                                                                                                                |
| [**SLConsumeRight**](/windows/desktop/api/slpublic/nf-slpublic-slconsumeright)<br/>                                             | Let an application to exercise rights on a locally-stored licenses.<br/>                                                                                                                                                       |
| [**SLDepositMigrationBlob**](/windows/desktop/api/slpublic/nf-slpublic-sldepositmigrationblob)<br/>                             | Deposits licensing information previously collected and gathered using the [**SLGatherMigrationBlob**](/windows/desktop/api/slpublic/nf-slpublic-slgathermigrationblob) function.<br/>                                                                               |
| [**SLDepositOfflineConfirmationId**](/windows/desktop/api/slpublic/nf-slpublic-sldepositofflineconfirmationid)<br/>             | Deposits Installation ID (IID) and Confirmation ID (CID) for offline activation.<br/>                                                                                                                                          |
| [**SLDepositOfflineConfirmationIdEx**](/windows/desktop/api/slpublic/nf-slpublic-sldepositofflineconfirmationidex)<br/>         | Deposits Installation ID (IID) and Confirmation ID (CID) for offline activation.<br/>                                                                                                                                          |
| [**SLFireEvent**](/windows/desktop/api/slpublic/nf-slpublic-slfireevent)<br/>                                                   | Sends a specified event to a registered listener.<br/>                                                                                                                                                                         |
| [**SLGatherMigrationBlob**](/windows/desktop/api/slpublic/nf-slpublic-slgathermigrationblob)<br/>                               | Gathers licensing information for the provided file handle. This licensing information can later be applied or deposited using the [**SLDepositMigrationBlob**](/windows/desktop/api/slpublic/nf-slpublic-sldepositmigrationblob) function.<br/>                     |
| [**SLGenerateOfflineInstallationId**](/windows/desktop/api/slpublic/nf-slpublic-slgenerateofflineinstallationid)<br/>           | Generates the Installation ID (IID). <br/>                                                                                                                                                                                     |
| [**SLGenerateOfflineInstallationIdEx**](/windows/desktop/api/slpublic/nf-slpublic-slgenerateofflineinstallationidex)<br/>       | Generates Installation ID (IID).<br/>                                                                                                                                                                                          |
| [**SLGetApplicationInformation**](/windows/desktop/api/slpublic/nf-slpublic-slgetapplicationinformation)<br/>                   | Gets information about the specified application.<br/>                                                                                                                                                                         |
| [**SLGetApplicationPolicy**](/windows/desktop/api/slpublic/nf-slpublic-slgetapplicationpolicy)<br/>                             | Queries a policy from the set stored with the [**SLPersistApplicationPolicies**](/windows/desktop/api/slpublic/nf-slpublic-slpersistapplicationpolicies) function and loaded using the [**SLLoadApplicationPolicies**](/windows/desktop/api/slpublic/nf-slpublic-slloadapplicationpolicies) function.<br/> |
| [**SLGetAuthenticationResult**](/windows/desktop/api/slpublic/nf-slpublic-slgetauthenticationresult)<br/>                       | Gets the authentication results.<br/>                                                                                                                                                                                          |
| [**SLGetGenuineInformation**](/windows/desktop/api/Slpublic/nf-slpublic-slgetgenuineinformation)<br/>                           | Gets information about the genuine state of a Windows computer.<br/>                                                                                                                                                           |
| [**SLSetGenuineInformation**](/windows/desktop/api/Slpublic/nf-slpublic-slsetgenuineinformation)<br/>                           | Specifies information about the genuine status of a Windows computer.<br/>                                                                                                                                                     |
| [**SLGetGenuineInformationEx**](/windows/desktop/api/slpublic/nf-slpublic-slgetgenuineinformationex)<br/>                       | Specifies information about the genuine status of a Windows computer.<br/>                                                                                                                                                     |
| [**SLGetInstalledProductKeyIds**](/windows/desktop/api/slpublic/nf-slpublic-slgetinstalledproductkeyids)<br/>                   | This function returns a list of product key IDs associated with the specified Product SKU ID.<br/>                                                                                                                             |
| [**SLGetInstalledSAMLicenseApplications**](/windows/desktop/api/Slpublic/)<br/> | Retrieves a list of applications that have Software Asset Management (SAM) licenses that were installed by using the [**SLInstallSAMLicense**](/windows/desktop/api/Slpublic/) function.<br/>                                         |
| [**SLGetLicense**](/windows/desktop/api/slpublic/nf-slpublic-slgetlicense)<br/>                                                 | Returns the license file BLOB.<br/>                                                                                                                                                                                            |
| [**SLGetLicenseFileId**](/windows/desktop/api/slpublic/nf-slpublic-slgetlicensefileid)<br/>                                     | Checks if the license BLOB has been installed already.<br/>                                                                                                                                                                    |
| [**SLGetLicenseInformation**](/windows/desktop/api/slpublic/nf-slpublic-slgetlicenseinformation)<br/>                           | Gets the specified license information.<br/>                                                                                                                                                                                   |
| [**SLGetLicensingStatusInformation**](/windows/desktop/api/slpublic/nf-slpublic-slgetlicensingstatusinformation)<br/>           | Gets the licensing status of the specified application or SKU.<br/>                                                                                                                                                            |
| [**SLGetPKeyId**](/windows/desktop/api/slpublic/nf-slpublic-slgetpkeyid)<br/>                                                   | Gets the registered product key ID associated with the product.<br/>                                                                                                                                                           |
| [**SLGetPKeyInformation**](/windows/desktop/api/slpublic/nf-slpublic-slgetpkeyinformation)<br/>                                 | Gets the information of the specified product key.<br/>                                                                                                                                                                        |
| [**SLGetPolicyInformation**](/windows/desktop/api/slpublic/nf-slpublic-slgetpolicyinformation)<br/>                             | Gets the policy information after right has been consumed successfully.<br/>                                                                                                                                                   |
| [**SLGetPolicyInformationDWORD**](/windows/desktop/api/slpublic/nf-slpublic-slgetpolicyinformationdword)<br/>                   | Gets the policy information after right has been consumed successfully.<br/>                                                                                                                                                   |
| [**SLGetProductSkuInformation**](/windows/desktop/api/slpublic/nf-slpublic-slgetproductskuinformation)<br/>                     | Gets information about the specified product SKU.<br/>                                                                                                                                                                         |
| [**SLGetReferralInformation**](/windows/desktop/api/slpublic/nf-slpublic-slgetreferralinformation)<br/>                         | Gets referral information for the specified product.<br/>                                                                                                                                                                      |
| [**SLGetSAMLicense**](/windows/desktop/api/Slpublic/)<br/>                                           | Retrieves information about an installed Software Asset Management (SAM) license.<br/>                                                                                                                                         |
| [**SLGetServerStatus**](/windows/desktop/api/slpublic/nf-slpublic-slgetserverstatus)<br/>                                       | Checks the server status according to the specified URL and RequestType.<br/>                                                                                                                                                  |
| [**SLGetServiceInformation**](/windows/desktop/api/slpublic/nf-slpublic-slgetserviceinformation)<br/>                           | Gets global data information.<br/>                                                                                                                                                                                             |
| [**SLGetSLIDList**](/windows/desktop/api/slpublic/nf-slpublic-slgetslidlist)<br/>                                               | Gets a list of **SLID**s according to the input query ID type and the ID value.<br/>                                                                                                                                           |
| [**SLGetWindowsInformation**](/windows/desktop/api/Slpublic/nf-slpublic-slgetwindowsinformation)<br/>                           | Retrieves the value portion of a name-value pair from the licensing policy of a software component.<br/>                                                                                                                       |
| [**SLGetWindowsInformationDWORD**](/windows/desktop/api/Slpublic/nf-slpublic-slgetwindowsinformationdword)<br/>                 | Retrieves the **DWORD** value portion of a name-value pair from the licensing policy of a software component.<br/>                                                                                                             |
| [**SLInstallLicense**](/windows/desktop/api/slpublic/nf-slpublic-slinstalllicense)<br/>                                         | Stores the specified license and returns a license file ID.<br/>                                                                                                                                                               |
| [**SLInstallProofOfPurchase**](/windows/desktop/api/slpublic/nf-slpublic-slinstallproofofpurchase)<br/>                         | Registers the product key with SL.<br/>                                                                                                                                                                                        |
| [**SLInstallProofOfPurchaseEx**](/windows/desktop/api/slpublic/nf-slpublic-slinstallproofofpurchaseex)<br/>                     | Register the product key with SL.<br/>                                                                                                                                                                                         |
| [**SLInstallSAMLicense**](/windows/desktop/api/Slpublic/)<br/>                                   | Installs a Software Asset Management (SAM) license for the specified application.<br/>                                                                                                                                         |
| [**SLIsGenuineLocal**](/windows/desktop/api/Slpublic/nf-slpublic-slisgenuinelocal)<br/>                                         | Checks whether the specified application is a genuine Windows installation.<br/>                                                                                                                                               |
| [**SLIsGenuineLocalEx**](/windows/desktop/api/Slpublic/nf-slpublic-slisgenuinelocalex)<br/>                                     | Checks whether the specified application installation is genuine.<br/>                                                                                                                                                         |
| [**SLLoadApplicationPolicies**](/windows/desktop/api/slpublic/nf-slpublic-slloadapplicationpolicies)<br/>                       | Loads the application policies set with the [**SLPersistApplicationPolicies**](/windows/desktop/api/slpublic/nf-slpublic-slpersistapplicationpolicies) function for use by the [**SLGetApplicationPolicy**](/windows/desktop/api/slpublic/nf-slpublic-slgetapplicationpolicy) function.<br/>               |
| [**SLOpen**](/windows/desktop/api/slpublic/nf-slpublic-slopen)<br/>                                                             | Initializes the Software Licensing Client (SLC) and connects SLC to the Software Licensing Service (SLS).<br/>                                                                                                                 |
| [**SLPersistApplicationPolicies**](/windows/desktop/api/slpublic/nf-slpublic-slpersistapplicationpolicies)<br/>                 | Stores the current consumed policies to disk for fast policy access.<br/>                                                                                                                                                      |
| [**SLPersistRTSPayloadOverride**](/windows/desktop/api/slpublic/nf-slpublic-slpersistrtspayloadoverride)<br/>                   | Associates information with the specified product for both online and phone activation.<br/>                                                                                                                                   |
| [**SLQueryLicenseValueFromApp**](/windows/desktop/api/slpublic/nf-slpublic-slquerylicensevaluefromapp)<br/>                     | Gets the value for the specified component policy.<br/>                                                                                                                                                                        |
| [**SLReArm**](/windows/desktop/api/slpublic/nf-slpublic-slrearm)<br/>                                                           | This function is rearm application activation.<br/>                                                                                                                                                                            |
| [**SLRegisterEvent**](/windows/desktop/api/slpublic/nf-slpublic-slregisterevent)<br/>                                           | Registers an event in the SL service.<br/>                                                                                                                                                                                     |
| [**SLSetAuthenticationData**](/windows/desktop/api/slpublic/nf-slpublic-slsetauthenticationdata)<br/>                           | Sets authentication data.<br/>                                                                                                                                                                                                 |
| [**SLSetCurrentProductKey**](/windows/desktop/api/slpublic/nf-slpublic-slsetcurrentproductkey)<br/>                             | Sets the current product key to the previously installed product key.<br/>                                                                                                                                                     |
| [**SLUninstallLicense**](/windows/desktop/api/slpublic/nf-slpublic-sluninstalllicense)<br/>                                     | Uninstalls the license specified by the license file ID and target user option.<br/>                                                                                                                                           |
| [**SLUninstallProofOfPurchase**](/windows/desktop/api/slpublic/nf-slpublic-sluninstallproofofpurchase)<br/>                     | Unregisters the product key information.<br/>                                                                                                                                                                                  |
| [**SLUninstallSAMLicense**](/windows/desktop/api/Slpublic/)<br/>                               | Uninstalls a Software Asset Management (SAM) license for the specified application.<br/>                                                                                                                                       |
| [**SLUnloadApplicationPolicies**](/windows/desktop/api/slpublic/nf-slpublic-slunloadapplicationpolicies)<br/>                   | Releases the policy context handle returned by the [**SLLoadApplicationPolicies**](/windows/desktop/api/slpublic/nf-slpublic-slloadapplicationpolicies) function.<br/>                                                                                               |
| [**SLUnregisterEvent**](/windows/desktop/api/slpublic/nf-slpublic-slunregisterevent)<br/>                                       | Unregisters a registered event in the SL service.<br/>                                                                                                                                                                         |
