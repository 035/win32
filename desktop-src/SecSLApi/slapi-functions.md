# SLAPI Functions

The Software Licensing API (SLAPI) uses the following functions.

## In this section



| Topic                                                                                           | Description                                                                                                                                                                                                                          |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SLActivateProduct**](slactivateproduct.md)<br/>                                       | Acquires a use license from the Software License Server (SLS).<br/>                                                                                                                                                            |
| [**SLAcquireGenuineTicket**](slacquiregenuineticket.md)<br/>                             | Gets a XrML genuine ticket acquired from the Software Licensing Server (SLS).<br/>                                                                                                                                             |
| [**SLClose**](slclose.md)<br/>                                                           | Closes the Software Licensing Client (SLC) context handle.<br/>                                                                                                                                                                |
| [**SLConsumeRight**](slconsumeright.md)<br/>                                             | Let an application to exercise rights on a locally-stored licenses.<br/>                                                                                                                                                       |
| [**SLDepositMigrationBlob**](sldepositmigrationblob.md)<br/>                             | Deposits licensing information previously collected and gathered using the [**SLGatherMigrationBlob**](slgathermigrationblob.md) function.<br/>                                                                               |
| [**SLDepositOfflineConfirmationId**](sldepositofflineconfirmationid.md)<br/>             | Deposits Installation ID (IID) and Confirmation ID (CID) for offline activation.<br/>                                                                                                                                          |
| [**SLDepositOfflineConfirmationIdEx**](sldepositofflineconfirmationidex.md)<br/>         | Deposits Installation ID (IID) and Confirmation ID (CID) for offline activation.<br/>                                                                                                                                          |
| [**SLFireEvent**](slfireevent.md)<br/>                                                   | Sends a specified event to a registered listener.<br/>                                                                                                                                                                         |
| [**SLGatherMigrationBlob**](slgathermigrationblob.md)<br/>                               | Gathers licensing information for the provided file handle. This licensing information can later be applied or deposited using the [**SLDepositMigrationBlob**](sldepositmigrationblob.md) function.<br/>                     |
| [**SLGenerateOfflineInstallationId**](slgenerateofflineinstallationid.md)<br/>           | Generates the Installation ID (IID). <br/>                                                                                                                                                                                     |
| [**SLGenerateOfflineInstallationIdEx**](slgenerateofflineinstallationidex.md)<br/>       | Generates Installation ID (IID).<br/>                                                                                                                                                                                          |
| [**SLGetApplicationInformation**](slgetapplicationinformation.md)<br/>                   | Gets information about the specified application.<br/>                                                                                                                                                                         |
| [**SLGetApplicationPolicy**](slgetapplicationpolicy.md)<br/>                             | Queries a policy from the set stored with the [**SLPersistApplicationPolicies**](slpersistapplicationpolicies.md) function and loaded using the [**SLLoadApplicationPolicies**](slloadapplicationpolicies.md) function.<br/> |
| [**SLGetAuthenticationResult**](slgetauthenticationresult.md)<br/>                       | Gets the authentication results.<br/>                                                                                                                                                                                          |
| [**SLGetGenuineInformation**](slgetgenuineinformation.md)<br/>                           | Gets information about the genuine state of a Windows computer.<br/>                                                                                                                                                           |
| [**SLSetGenuineInformation**](slsetgenuineinformation.md)<br/>                           | Specifies information about the genuine status of a Windows computer.<br/>                                                                                                                                                     |
| [**SLGetGenuineInformationEx**](slgetgenuineinformationex.md)<br/>                       | Specifies information about the genuine status of a Windows computer.<br/>                                                                                                                                                     |
| [**SLGetInstalledProductKeyIds**](slgetinstalledproductkeyids.md)<br/>                   | This function returns a list of product key IDs associated with the specified Product SKU ID.<br/>                                                                                                                             |
| [**SLGetInstalledSAMLicenseApplications**](slgetinstalledsamlicenseapplications.md)<br/> | Retrieves a list of applications that have Software Asset Management (SAM) licenses that were installed by using the [**SLInstallSAMLicense**](slinstallsamlicense.md) function.<br/>                                         |
| [**SLGetLicense**](slgetlicense.md)<br/>                                                 | Returns the license file BLOB.<br/>                                                                                                                                                                                            |
| [**SLGetLicenseFileId**](slgetlicensefileid.md)<br/>                                     | Checks if the license BLOB has been installed already.<br/>                                                                                                                                                                    |
| [**SLGetLicenseInformation**](slgetlicenseinformation.md)<br/>                           | Gets the specified license information.<br/>                                                                                                                                                                                   |
| [**SLGetLicensingStatusInformation**](slgetlicensingstatusinformation.md)<br/>           | Gets the licensing status of the specified application or SKU.<br/>                                                                                                                                                            |
| [**SLGetPKeyId**](slgetpkeyid.md)<br/>                                                   | Gets the registered product key ID associated with the product.<br/>                                                                                                                                                           |
| [**SLGetPKeyInformation**](slgetpkeyinformation.md)<br/>                                 | Gets the information of the specified product key.<br/>                                                                                                                                                                        |
| [**SLGetPolicyInformation**](slgetpolicyinformation.md)<br/>                             | Gets the policy information after right has been consumed successfully.<br/>                                                                                                                                                   |
| [**SLGetPolicyInformationDWORD**](slgetpolicyinformationdword.md)<br/>                   | Gets the policy information after right has been consumed successfully.<br/>                                                                                                                                                   |
| [**SLGetProductSkuInformation**](slgetproductskuinformation.md)<br/>                     | Gets information about the specified product SKU.<br/>                                                                                                                                                                         |
| [**SLGetReferralInformation**](slgetreferralinformation.md)<br/>                         | Gets referral information for the specified product.<br/>                                                                                                                                                                      |
| [**SLGetSAMLicense**](slgetsamlicense.md)<br/>                                           | Retrieves information about an installed Software Asset Management (SAM) license.<br/>                                                                                                                                         |
| [**SLGetServerStatus**](slgetserverstatus.md)<br/>                                       | Checks the server status according to the specified URL and RequestType.<br/>                                                                                                                                                  |
| [**SLGetServiceInformation**](slgetserviceinformation.md)<br/>                           | Gets global data information.<br/>                                                                                                                                                                                             |
| [**SLGetSLIDList**](slgetslidlist.md)<br/>                                               | Gets a list of **SLID**s according to the input query ID type and the ID value.<br/>                                                                                                                                           |
| [**SLGetWindowsInformation**](slgetwindowsinformation.md)<br/>                           | Retrieves the value portion of a name-value pair from the licensing policy of a software component.<br/>                                                                                                                       |
| [**SLGetWindowsInformationDWORD**](slgetwindowsinformationdword.md)<br/>                 | Retrieves the **DWORD** value portion of a name-value pair from the licensing policy of a software component.<br/>                                                                                                             |
| [**SLInstallLicense**](slinstalllicense.md)<br/>                                         | Stores the specified license and returns a license file ID.<br/>                                                                                                                                                               |
| [**SLInstallProofOfPurchase**](slinstallproofofpurchase.md)<br/>                         | Registers the product key with SL.<br/>                                                                                                                                                                                        |
| [**SLInstallProofOfPurchaseEx**](slinstallproofofpurchaseex.md)<br/>                     | Register the product key with SL.<br/>                                                                                                                                                                                         |
| [**SLInstallSAMLicense**](slinstallsamlicense.md)<br/>                                   | Installs a Software Asset Management (SAM) license for the specified application.<br/>                                                                                                                                         |
| [**SLIsGenuineLocal**](slisgenuinelocal.md)<br/>                                         | Checks whether the specified application is a genuine Windows installation.<br/>                                                                                                                                               |
| [**SLIsGenuineLocalEx**](slisgenuinelocalex.md)<br/>                                     | Checks whether the specified application installation is genuine.<br/>                                                                                                                                                         |
| [**SLLoadApplicationPolicies**](slloadapplicationpolicies.md)<br/>                       | Loads the application policies set with the [**SLPersistApplicationPolicies**](slpersistapplicationpolicies.md) function for use by the [**SLGetApplicationPolicy**](slgetapplicationpolicy.md) function.<br/>               |
| [**SLOpen**](slopen.md)<br/>                                                             | Initializes the Software Licensing Client (SLC) and connects SLC to the Software Licensing Service (SLS).<br/>                                                                                                                 |
| [**SLPersistApplicationPolicies**](slpersistapplicationpolicies.md)<br/>                 | Stores the current consumed policies to disk for fast policy access.<br/>                                                                                                                                                      |
| [**SLPersistRTSPayloadOverride**](slpersistrtspayloadoverride.md)<br/>                   | Associates information with the specified product for both online and phone activation.<br/>                                                                                                                                   |
| [**SLQueryLicenseValueFromApp**](slquerylicensevaluefromapp.md)<br/>                     | Gets the value for the specified component policy.<br/>                                                                                                                                                                        |
| [**SLReArm**](slrearm.md)<br/>                                                           | This function is rearm application activation.<br/>                                                                                                                                                                            |
| [**SLRegisterEvent**](slregisterevent.md)<br/>                                           | Registers an event in the SL service.<br/>                                                                                                                                                                                     |
| [**SLSetAuthenticationData**](slsetauthenticationdata.md)<br/>                           | Sets authentication data.<br/>                                                                                                                                                                                                 |
| [**SLSetCurrentProductKey**](slsetcurrentproductkey.md)<br/>                             | Sets the current product key to the previously installed product key.<br/>                                                                                                                                                     |
| [**SLUninstallLicense**](sluninstalllicense.md)<br/>                                     | Uninstalls the license specified by the license file ID and target user option.<br/>                                                                                                                                           |
| [**SLUninstallProofOfPurchase**](sluninstallproofofpurchase.md)<br/>                     | Unregisters the product key information.<br/>                                                                                                                                                                                  |
| [**SLUninstallSAMLicense**](sluninstallsamlicense.md)<br/>                               | Uninstalls a Software Asset Management (SAM) license for the specified application.<br/>                                                                                                                                       |
| [**SLUnloadApplicationPolicies**](slunloadapplicationpolicies.md)<br/>                   | Releases the policy context handle returned by the [**SLLoadApplicationPolicies**](slloadapplicationpolicies.md) function.<br/>                                                                                               |
| [**SLUnregisterEvent**](slunregisterevent.md)<br/>                                       | Unregisters a registered event in the SL service.<br/>                                                                                                                                                                         |
