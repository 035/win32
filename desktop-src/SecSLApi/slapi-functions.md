# SLAPI Functions

The Software Licensing API (SLAPI) uses the following functions.

## In this section



| Topic                                                                                           | Description                                                                                                                                                                                                                          |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SLActivateProduct**](/windows/win32/slpublic/nf-slpublic-slactivateproduct?branch=master)<br/>                                       | Acquires a use license from the Software License Server (SLS).<br/>                                                                                                                                                            |
| [**SLAcquireGenuineTicket**](/windows/win32/Slpublic/nf-slpublic-slacquiregenuineticket?branch=master)<br/>                             | Gets a XrML genuine ticket acquired from the Software Licensing Server (SLS).<br/>                                                                                                                                             |
| [**SLClose**](/windows/win32/slpublic/nf-slpublic-slclose?branch=master)<br/>                                                           | Closes the Software Licensing Client (SLC) context handle.<br/>                                                                                                                                                                |
| [**SLConsumeRight**](/windows/win32/slpublic/nf-slpublic-slconsumeright?branch=master)<br/>                                             | Let an application to exercise rights on a locally-stored licenses.<br/>                                                                                                                                                       |
| [**SLDepositMigrationBlob**](/windows/win32/slpublic/nf-slpublic-sldepositmigrationblob?branch=master)<br/>                             | Deposits licensing information previously collected and gathered using the [**SLGatherMigrationBlob**](/windows/win32/slpublic/nf-slpublic-slgathermigrationblob?branch=master) function.<br/>                                                                               |
| [**SLDepositOfflineConfirmationId**](/windows/win32/slpublic/nf-slpublic-sldepositofflineconfirmationid?branch=master)<br/>             | Deposits Installation ID (IID) and Confirmation ID (CID) for offline activation.<br/>                                                                                                                                          |
| [**SLDepositOfflineConfirmationIdEx**](/windows/win32/slpublic/nf-slpublic-sldepositofflineconfirmationidex?branch=master)<br/>         | Deposits Installation ID (IID) and Confirmation ID (CID) for offline activation.<br/>                                                                                                                                          |
| [**SLFireEvent**](/windows/win32/slpublic/nf-slpublic-slfireevent?branch=master)<br/>                                                   | Sends a specified event to a registered listener.<br/>                                                                                                                                                                         |
| [**SLGatherMigrationBlob**](/windows/win32/slpublic/nf-slpublic-slgathermigrationblob?branch=master)<br/>                               | Gathers licensing information for the provided file handle. This licensing information can later be applied or deposited using the [**SLDepositMigrationBlob**](/windows/win32/slpublic/nf-slpublic-sldepositmigrationblob?branch=master) function.<br/>                     |
| [**SLGenerateOfflineInstallationId**](/windows/win32/slpublic/nf-slpublic-slgenerateofflineinstallationid?branch=master)<br/>           | Generates the Installation ID (IID). <br/>                                                                                                                                                                                     |
| [**SLGenerateOfflineInstallationIdEx**](/windows/win32/slpublic/nf-slpublic-slgenerateofflineinstallationidex?branch=master)<br/>       | Generates Installation ID (IID).<br/>                                                                                                                                                                                          |
| [**SLGetApplicationInformation**](/windows/win32/slpublic/nf-slpublic-slgetapplicationinformation?branch=master)<br/>                   | Gets information about the specified application.<br/>                                                                                                                                                                         |
| [**SLGetApplicationPolicy**](/windows/win32/slpublic/nf-slpublic-slgetapplicationpolicy?branch=master)<br/>                             | Queries a policy from the set stored with the [**SLPersistApplicationPolicies**](/windows/win32/slpublic/nf-slpublic-slpersistapplicationpolicies?branch=master) function and loaded using the [**SLLoadApplicationPolicies**](/windows/win32/slpublic/nf-slpublic-slloadapplicationpolicies?branch=master) function.<br/> |
| [**SLGetAuthenticationResult**](/windows/win32/slpublic/nf-slpublic-slgetauthenticationresult?branch=master)<br/>                       | Gets the authentication results.<br/>                                                                                                                                                                                          |
| [**SLGetGenuineInformation**](/windows/win32/Slpublic/nf-slpublic-slgetgenuineinformation?branch=master)<br/>                           | Gets information about the genuine state of a Windows computer.<br/>                                                                                                                                                           |
| [**SLSetGenuineInformation**](/windows/win32/Slpublic/nf-slpublic-slsetgenuineinformation?branch=master)<br/>                           | Specifies information about the genuine status of a Windows computer.<br/>                                                                                                                                                     |
| [**SLGetGenuineInformationEx**](/windows/win32/slpublic/nf-slpublic-slgetgenuineinformationex?branch=master)<br/>                       | Specifies information about the genuine status of a Windows computer.<br/>                                                                                                                                                     |
| [**SLGetInstalledProductKeyIds**](/windows/win32/slpublic/nf-slpublic-slgetinstalledproductkeyids?branch=master)<br/>                   | This function returns a list of product key IDs associated with the specified Product SKU ID.<br/>                                                                                                                             |
| [**SLGetInstalledSAMLicenseApplications**](/windows/win32/Slpublic/?branch=master)<br/> | Retrieves a list of applications that have Software Asset Management (SAM) licenses that were installed by using the [**SLInstallSAMLicense**](/windows/win32/Slpublic/?branch=master) function.<br/>                                         |
| [**SLGetLicense**](/windows/win32/slpublic/nf-slpublic-slgetlicense?branch=master)<br/>                                                 | Returns the license file BLOB.<br/>                                                                                                                                                                                            |
| [**SLGetLicenseFileId**](/windows/win32/slpublic/nf-slpublic-slgetlicensefileid?branch=master)<br/>                                     | Checks if the license BLOB has been installed already.<br/>                                                                                                                                                                    |
| [**SLGetLicenseInformation**](/windows/win32/slpublic/nf-slpublic-slgetlicenseinformation?branch=master)<br/>                           | Gets the specified license information.<br/>                                                                                                                                                                                   |
| [**SLGetLicensingStatusInformation**](/windows/win32/slpublic/nf-slpublic-slgetlicensingstatusinformation?branch=master)<br/>           | Gets the licensing status of the specified application or SKU.<br/>                                                                                                                                                            |
| [**SLGetPKeyId**](/windows/win32/slpublic/nf-slpublic-slgetpkeyid?branch=master)<br/>                                                   | Gets the registered product key ID associated with the product.<br/>                                                                                                                                                           |
| [**SLGetPKeyInformation**](/windows/win32/slpublic/nf-slpublic-slgetpkeyinformation?branch=master)<br/>                                 | Gets the information of the specified product key.<br/>                                                                                                                                                                        |
| [**SLGetPolicyInformation**](/windows/win32/slpublic/nf-slpublic-slgetpolicyinformation?branch=master)<br/>                             | Gets the policy information after right has been consumed successfully.<br/>                                                                                                                                                   |
| [**SLGetPolicyInformationDWORD**](/windows/win32/slpublic/nf-slpublic-slgetpolicyinformationdword?branch=master)<br/>                   | Gets the policy information after right has been consumed successfully.<br/>                                                                                                                                                   |
| [**SLGetProductSkuInformation**](/windows/win32/slpublic/nf-slpublic-slgetproductskuinformation?branch=master)<br/>                     | Gets information about the specified product SKU.<br/>                                                                                                                                                                         |
| [**SLGetReferralInformation**](/windows/win32/slpublic/nf-slpublic-slgetreferralinformation?branch=master)<br/>                         | Gets referral information for the specified product.<br/>                                                                                                                                                                      |
| [**SLGetSAMLicense**](/windows/win32/Slpublic/?branch=master)<br/>                                           | Retrieves information about an installed Software Asset Management (SAM) license.<br/>                                                                                                                                         |
| [**SLGetServerStatus**](/windows/win32/slpublic/nf-slpublic-slgetserverstatus?branch=master)<br/>                                       | Checks the server status according to the specified URL and RequestType.<br/>                                                                                                                                                  |
| [**SLGetServiceInformation**](/windows/win32/slpublic/nf-slpublic-slgetserviceinformation?branch=master)<br/>                           | Gets global data information.<br/>                                                                                                                                                                                             |
| [**SLGetSLIDList**](/windows/win32/slpublic/nf-slpublic-slgetslidlist?branch=master)<br/>                                               | Gets a list of **SLID**s according to the input query ID type and the ID value.<br/>                                                                                                                                           |
| [**SLGetWindowsInformation**](/windows/win32/Slpublic/nf-slpublic-slgetwindowsinformation?branch=master)<br/>                           | Retrieves the value portion of a name-value pair from the licensing policy of a software component.<br/>                                                                                                                       |
| [**SLGetWindowsInformationDWORD**](/windows/win32/Slpublic/nf-slpublic-slgetwindowsinformationdword?branch=master)<br/>                 | Retrieves the **DWORD** value portion of a name-value pair from the licensing policy of a software component.<br/>                                                                                                             |
| [**SLInstallLicense**](/windows/win32/slpublic/nf-slpublic-slinstalllicense?branch=master)<br/>                                         | Stores the specified license and returns a license file ID.<br/>                                                                                                                                                               |
| [**SLInstallProofOfPurchase**](/windows/win32/slpublic/nf-slpublic-slinstallproofofpurchase?branch=master)<br/>                         | Registers the product key with SL.<br/>                                                                                                                                                                                        |
| [**SLInstallProofOfPurchaseEx**](/windows/win32/slpublic/nf-slpublic-slinstallproofofpurchaseex?branch=master)<br/>                     | Register the product key with SL.<br/>                                                                                                                                                                                         |
| [**SLInstallSAMLicense**](/windows/win32/Slpublic/?branch=master)<br/>                                   | Installs a Software Asset Management (SAM) license for the specified application.<br/>                                                                                                                                         |
| [**SLIsGenuineLocal**](/windows/win32/Slpublic/nf-slpublic-slisgenuinelocal?branch=master)<br/>                                         | Checks whether the specified application is a genuine Windows installation.<br/>                                                                                                                                               |
| [**SLIsGenuineLocalEx**](/windows/win32/Slpublic/nf-slpublic-slisgenuinelocalex?branch=master)<br/>                                     | Checks whether the specified application installation is genuine.<br/>                                                                                                                                                         |
| [**SLLoadApplicationPolicies**](/windows/win32/slpublic/nf-slpublic-slloadapplicationpolicies?branch=master)<br/>                       | Loads the application policies set with the [**SLPersistApplicationPolicies**](/windows/win32/slpublic/nf-slpublic-slpersistapplicationpolicies?branch=master) function for use by the [**SLGetApplicationPolicy**](/windows/win32/slpublic/nf-slpublic-slgetapplicationpolicy?branch=master) function.<br/>               |
| [**SLOpen**](/windows/win32/slpublic/nf-slpublic-slopen?branch=master)<br/>                                                             | Initializes the Software Licensing Client (SLC) and connects SLC to the Software Licensing Service (SLS).<br/>                                                                                                                 |
| [**SLPersistApplicationPolicies**](/windows/win32/slpublic/nf-slpublic-slpersistapplicationpolicies?branch=master)<br/>                 | Stores the current consumed policies to disk for fast policy access.<br/>                                                                                                                                                      |
| [**SLPersistRTSPayloadOverride**](/windows/win32/slpublic/nf-slpublic-slpersistrtspayloadoverride?branch=master)<br/>                   | Associates information with the specified product for both online and phone activation.<br/>                                                                                                                                   |
| [**SLQueryLicenseValueFromApp**](/windows/win32/slpublic/nf-slpublic-slquerylicensevaluefromapp?branch=master)<br/>                     | Gets the value for the specified component policy.<br/>                                                                                                                                                                        |
| [**SLReArm**](/windows/win32/slpublic/nf-slpublic-slrearm?branch=master)<br/>                                                           | This function is rearm application activation.<br/>                                                                                                                                                                            |
| [**SLRegisterEvent**](/windows/win32/slpublic/nf-slpublic-slregisterevent?branch=master)<br/>                                           | Registers an event in the SL service.<br/>                                                                                                                                                                                     |
| [**SLSetAuthenticationData**](/windows/win32/slpublic/nf-slpublic-slsetauthenticationdata?branch=master)<br/>                           | Sets authentication data.<br/>                                                                                                                                                                                                 |
| [**SLSetCurrentProductKey**](/windows/win32/slpublic/nf-slpublic-slsetcurrentproductkey?branch=master)<br/>                             | Sets the current product key to the previously installed product key.<br/>                                                                                                                                                     |
| [**SLUninstallLicense**](/windows/win32/slpublic/nf-slpublic-sluninstalllicense?branch=master)<br/>                                     | Uninstalls the license specified by the license file ID and target user option.<br/>                                                                                                                                           |
| [**SLUninstallProofOfPurchase**](/windows/win32/slpublic/nf-slpublic-sluninstallproofofpurchase?branch=master)<br/>                     | Unregisters the product key information.<br/>                                                                                                                                                                                  |
| [**SLUninstallSAMLicense**](/windows/win32/Slpublic/?branch=master)<br/>                               | Uninstalls a Software Asset Management (SAM) license for the specified application.<br/>                                                                                                                                       |
| [**SLUnloadApplicationPolicies**](/windows/win32/slpublic/nf-slpublic-slunloadapplicationpolicies?branch=master)<br/>                   | Releases the policy context handle returned by the [**SLLoadApplicationPolicies**](/windows/win32/slpublic/nf-slpublic-slloadapplicationpolicies?branch=master) function.<br/>                                                                                               |
| [**SLUnregisterEvent**](/windows/win32/slpublic/nf-slpublic-slunregisterevent?branch=master)<br/>                                       | Unregisters a registered event in the SL service.<br/>                                                                                                                                                                         |
