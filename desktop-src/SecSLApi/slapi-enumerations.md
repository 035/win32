# SLAPI Enumerations

The Software Licensing API (SLAPI) uses the following enumerations.

## In this section



| Topic                                                         | Description                                                                                                                                        |
|---------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SL\_ACTIVATION\_TYPE**](/windows/desktop/api/slpublic/ne-slpublic-_tagsl_activation_type)<br/> | Represents the type of offline activation for a license.<br/>                                                                                |
| [**SL\_GENUINE\_STATE**](/windows/desktop/api/Slpublic/ne-slpublic-_sl_genuine_state)<br/>     | Specifies the state of an application installation.<br/>                                                                                     |
| [**SLDATATYPE**](/windows/desktop/api/Slpublic/ne-slpublic-_tagsldatatype)<br/>                   | Specifies the data type of the buffer returned by the [**SLGetWindowsInformation**](/windows/desktop/api/Slpublic/nf-slpublic-slgetwindowsinformation) function.<br/>              |
| [**SLIDTYPE**](/windows/desktop/api/slpublic/ne-slpublic-_tagslidtype)<br/>                       | Represents the type of Software Licensing ID.<br/>                                                                                           |
| [**SLLICENSINGSTATUS**](/windows/desktop/api/slpublic/ne-slpublic-_tagsllicensingstatus)<br/>     | Represents the licensing status.<br/>                                                                                                        |
| [**SLREFERRALTYPE**](/windows/desktop/api/slpublic/ne-slpublic-slreferraltype)<br/>           | Represents the types of information that can be queried with the [**SLGetReferralInformation**](/windows/desktop/api/slpublic/nf-slpublic-slgetreferralinformation) function.<br/> |
