# SLAPI Enumerations

The Software Licensing API (SLAPI) uses the following enumerations.

## In this section



| Topic                                                         | Description                                                                                                                                        |
|---------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SL\_ACTIVATION\_TYPE**](sl-activation-type.md)<br/> | Represents the type of offline activation for a license.<br/>                                                                                |
| [**SL\_GENUINE\_STATE**](sl-genuine-state.md)<br/>     | Specifies the state of an application installation.<br/>                                                                                     |
| [**SLDATATYPE**](sldatatype.md)<br/>                   | Specifies the data type of the buffer returned by the [**SLGetWindowsInformation**](slgetwindowsinformation.md) function.<br/>              |
| [**SLIDTYPE**](slidtype.md)<br/>                       | Represents the type of Software Licensing ID.<br/>                                                                                           |
| [**SLLICENSINGSTATUS**](sllicensingstatus.md)<br/>     | Represents the licensing status.<br/>                                                                                                        |
| [**SLREFERRALTYPE**](slreferraltype.md)<br/>           | Represents the types of information that can be queried with the [**SLGetReferralInformation**](slgetreferralinformation.md) function.<br/> |
