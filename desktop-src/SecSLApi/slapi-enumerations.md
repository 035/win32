# SLAPI Enumerations

The Software Licensing API (SLAPI) uses the following enumerations.

## In this section



| Topic                                                         | Description                                                                                                                                        |
|---------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SL\_ACTIVATION\_TYPE**](/windows/win32/slpublic/ne-slpublic-_tagsl_activation_type?branch=master)<br/> | Represents the type of offline activation for a license.<br/>                                                                                |
| [**SL\_GENUINE\_STATE**](/windows/win32/Slpublic/ne-slpublic-_sl_genuine_state?branch=master)<br/>     | Specifies the state of an application installation.<br/>                                                                                     |
| [**SLDATATYPE**](/windows/win32/Slpublic/ne-slpublic-_tagsldatatype?branch=master)<br/>                   | Specifies the data type of the buffer returned by the [**SLGetWindowsInformation**](/windows/win32/Slpublic/nf-slpublic-slgetwindowsinformation?branch=master) function.<br/>              |
| [**SLIDTYPE**](/windows/win32/slpublic/ne-slpublic-_tagslidtype?branch=master)<br/>                       | Represents the type of Software Licensing ID.<br/>                                                                                           |
| [**SLLICENSINGSTATUS**](/windows/win32/slpublic/ne-slpublic-_tagsllicensingstatus?branch=master)<br/>     | Represents the licensing status.<br/>                                                                                                        |
| [**SLREFERRALTYPE**](/windows/win32/slpublic/ne-slpublic-slreferraltype?branch=master)<br/>           | Represents the types of information that can be queried with the [**SLGetReferralInformation**](/windows/win32/slpublic/nf-slpublic-slgetreferralinformation?branch=master) function.<br/> |
