# User Access Logging functions

User Access Logging provides the following functions.

## In this section

<dl> <dt>

[**UalInstrument**](/previous-versions/windows/desktop/api/Ual/nf-ual-ualinstrument)
</dt> <dd>

Records the specified data to the User Access Logging (UAL) framework by using information from a [**UAL\_DATA\_BLOB**](/previous-versions/windows/desktop/api/Ual/ns-ual-tagual_data_blob) structure.

</dd> <dt>

[**UalRegisterProduct**](/previous-versions/windows/desktop/api/Ual/nf-ual-ualregisterproduct)
</dt> <dd>

Registers a product with User Access Logging (UAL).

</dd> <dt>

[**UalStart**](/previous-versions/windows/desktop/api/Ual/nf-ual-ualstart)
</dt> <dd>

Starts a User Access Logging (UAL) session.

</dd> <dt>

[**UalStop**](/previous-versions/windows/desktop/api/Ual/nf-ual-ualstop)
</dt> <dd>

Stops a User Access Logging (UAL) session.

</dd> </dl>
