# About the Network Load Balancing Provider

The Network Load Balancing Clusters dynamically distribute the flow of incoming TCP and UDP traffic among the cluster [*nodes*](https://msdn.microsoft.com/library/aa371764#mscs-n---z-5-gly) according to a set of traffic-handling rules. The [*Network Load Balancing Provider*](https://msdn.microsoft.com/library/aa371764#mscs-n---z-9-gly) allows applications to interact with Network Load Balancing clusters through the unified management framework of Windows Management Instrumentation (WMI).
