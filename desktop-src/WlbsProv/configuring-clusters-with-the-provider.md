# Configuring Clusters with the Provider

The Network Load Balancing [*Provider*](https://msdn.microsoft.com/library/aa371764#mscs-n---z-9-gly) allows you to create customized administration tools for configuring Network Load Balancing [*clusters*](https://msdn.microsoft.com/library/aa367183#mscs-a---e-5-gly). The following sections describe the Provider's role in various aspects of cluster configuration.



| Topic                                                                                                               | Information                                                                                                                                                         |
|---------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Standard Usage Model](standard-usage-model.md)<br/>                                                         | Describes the recommended procedures for using the Network Load Balancing provider to configure clusters.<br/>                                                |
| [Remote Setup of a Network Load Balancing Cluster](remote-setup-of-a-network-load-balancing-cluster.md)<br/> | Describes the process of performing a remote installation of Network Load Balancing.<br/>                                                                     |
| [Configuring User Interface Settings](configuring-user-interface-settings.md)<br/>                           | Maps the settings shown on the **Network Load Balancing Properties** dialog to the Provider classes, properties, and methods that affect those settings.<br/> |
