# Network Load Balancing Provider Reference

The Network Load Balancing [*Provider*](https://msdn.microsoft.com/library/aa371764#mscs-n---z-9-gly) is defined in WlbsProv.mof. By convention, this documentation describes only the classes, methods, and associations specific to Network Load Balancing. Top-level [*WMI classes*](https://msdn.microsoft.com/library/aa371764#mscs-n---z-22-gly) (those with the CIM\_ prefix) that are defined in the MOF file are not specific to Network Load Balancing and are described in the Windows SDK documentation for [Windows Management Instrumentation](https://msdn.microsoft.com/library/aa394582) (WMI).
