# Refresh method of the MSIscsiInitiator\_SendTargetPortalClass class

The [**Refresh**](refreshtargetlist-msiscsiinitiator-methodclass.md) method refreshes the list of discovered targets from the Send Target Portal.

## Syntax


```mof
uint32 Refresh();
```



## Parameters

This method has no parameters.
