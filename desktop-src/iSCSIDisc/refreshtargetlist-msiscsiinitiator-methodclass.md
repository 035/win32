# RefreshTargetList method of the MSIscsiInitiator\_MethodClass class

The **RefreshTargetList** method refreshes the entire list of targets using all available discovery methods.

## Syntax


```mof
uint32 RefreshTargetList();
```



## Parameters

This method has no parameters.
