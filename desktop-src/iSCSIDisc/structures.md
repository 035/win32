# iSCSI Discovery Library Structures

This section details the following iSCSI Discovery Library Structures.



| Structure                                                                       | Description                                                                                                                           |
|---------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [**IKE\_AUTHENTICATION\_INFORMATION**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-ike_authentication_information?branch=master)      | Contains Internet Key Exchange (IKE) authentication information that establishes a secure channel between two key management daemons. |
| [**IKE\_AUTHENTICATION\_PRESHARED\_KEY**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-ike_authentication_preshared_key?branch=master) | Contains information about the preshared key used in the Internet Key Exchange (IKE) protocol.                                        |
| [**ISCSI\_CONNECTION\_INFO**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-iscsi_connection_infoa?branch=master)                        | Contains information about a connection.                                                                                              |
| [**ISCSI\_DEVICE\_ON\_SESSION**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-iscsi_device_on_sessiona?branch=master)                   | Specifies multiple methods for identifying a device associated with an iSCSI login session.                                           |
| [**ISCSI\_LOGIN\_OPTIONS**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-iscsi_login_options?branch=master)                            | Utilized by initiators to specify the characteristics of a login session.                                                             |
| [**ISCSI\_SESSION\_INFO**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-iscsi_session_infoa?branch=master)                              | Contains session information in either ASCII or wide character format.                                                                |
| [**ISCSI\_TARGET\_MAPPING**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-iscsi_target_mappinga?branch=master)                          | Contains information about a target and the HBAs and buses through which the target can be reached.                                   |
| [**ISCSI\_TARGET\_PORTAL\_GROUP**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-iscsi_target_portal_groupa?branch=master)               | Contains information about the portals associated with a portal group.                                                                |
| [**ISCSI\_TARGET\_PORTAL**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-iscsi_target_portala?branch=master)                            | Contains information about a portal.                                                                                                  |
| [**ISCSI\_TARGET\_PORTAL\_INFO**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-iscsi_target_portal_infoa?branch=master)                 | Contains information about a target portal.                                                                                           |
| [**ISCSI\_TARGET\_PORTAL\_INFO\_EX**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-iscsi_target_portal_info_exa?branch=master)          | Contains information about login credentials to a target portal.                                                                      |
| [**ISCSI\_UNIQUE\_CONNECTION\_ID**](iscsidisc.iscsi_unique_connection_id)       | Contains data that uniquely identifies a connection.                                                                                  |
| [**ISCSI\_UNIQUE\_SESSION\_ID**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-_iscsi_unique_session_id?branch=master)                   | Contains data that uniquely identifies a session.                                                                                     |
| [**ISCSI\_VERSION\_INFO**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-iscsi_version_info?branch=master)                              | Contains the version and build numbers of the iSCSI software initiator and initiator service.                                         |
| [**PERSISTENT\_ISCSI\_LOGIN\_INFO**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-persistent_iscsi_login_infoa?branch=master)           | Describes a login session that the Microsoft iSCSI initiator service reestablishes automatically after the computer boots up.         |
| [**SCSI\_LUN\_LIST**](/windows/previous-versions/Iscsidsc/ns-iscsidsc-scsi_lun_list?branch=master)                                        | Utilized to construct a list of logical unit numbers (LUNs) associated with target devices.                                           |
