# iSCSI Discovery Library Structures

This section details the following iSCSI Discovery Library Structures.



| Structure                                                                       | Description                                                                                                                           |
|---------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [**IKE\_AUTHENTICATION\_INFORMATION**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-ike_authentication_information)      | Contains Internet Key Exchange (IKE) authentication information that establishes a secure channel between two key management daemons. |
| [**IKE\_AUTHENTICATION\_PRESHARED\_KEY**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-ike_authentication_preshared_key) | Contains information about the preshared key used in the Internet Key Exchange (IKE) protocol.                                        |
| [**ISCSI\_CONNECTION\_INFO**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-iscsi_connection_infoa)                        | Contains information about a connection.                                                                                              |
| [**ISCSI\_DEVICE\_ON\_SESSION**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-iscsi_device_on_sessiona)                   | Specifies multiple methods for identifying a device associated with an iSCSI login session.                                           |
| [**ISCSI\_LOGIN\_OPTIONS**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-iscsi_login_options)                            | Utilized by initiators to specify the characteristics of a login session.                                                             |
| [**ISCSI\_SESSION\_INFO**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-iscsi_session_infoa)                              | Contains session information in either ASCII or wide character format.                                                                |
| [**ISCSI\_TARGET\_MAPPING**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-iscsi_target_mappinga)                          | Contains information about a target and the HBAs and buses through which the target can be reached.                                   |
| [**ISCSI\_TARGET\_PORTAL\_GROUP**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-iscsi_target_portal_groupa)               | Contains information about the portals associated with a portal group.                                                                |
| [**ISCSI\_TARGET\_PORTAL**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-iscsi_target_portala)                            | Contains information about a portal.                                                                                                  |
| [**ISCSI\_TARGET\_PORTAL\_INFO**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-iscsi_target_portal_infoa)                 | Contains information about a target portal.                                                                                           |
| [**ISCSI\_TARGET\_PORTAL\_INFO\_EX**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-iscsi_target_portal_info_exa)          | Contains information about login credentials to a target portal.                                                                      |
| [**ISCSI\_UNIQUE\_CONNECTION\_ID**](https://www.bing.com/search?q=**ISCSI\_UNIQUE\_CONNECTION\_ID**)       | Contains data that uniquely identifies a connection.                                                                                  |
| [**ISCSI\_UNIQUE\_SESSION\_ID**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-_iscsi_unique_session_id)                   | Contains data that uniquely identifies a session.                                                                                     |
| [**ISCSI\_VERSION\_INFO**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-iscsi_version_info)                              | Contains the version and build numbers of the iSCSI software initiator and initiator service.                                         |
| [**PERSISTENT\_ISCSI\_LOGIN\_INFO**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-persistent_iscsi_login_infoa)           | Describes a login session that the Microsoft iSCSI initiator service reestablishes automatically after the computer boots up.         |
| [**SCSI\_LUN\_LIST**](/previous-versions/windows/desktop/api/Iscsidsc/ns-iscsidsc-scsi_lun_list)                                        | Utilized to construct a list of logical unit numbers (LUNs) associated with target devices.                                           |
