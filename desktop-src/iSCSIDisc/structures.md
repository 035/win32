# iSCSI Discovery Library Structures

This section details the following iSCSI Discovery Library Structures.



| Structure                                                                       | Description                                                                                                                           |
|---------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [**IKE\_AUTHENTICATION\_INFORMATION**](ike-authentication-information.md)      | Contains Internet Key Exchange (IKE) authentication information that establishes a secure channel between two key management daemons. |
| [**IKE\_AUTHENTICATION\_PRESHARED\_KEY**](ike-authentication-preshared-key.md) | Contains information about the preshared key used in the Internet Key Exchange (IKE) protocol.                                        |
| [**ISCSI\_CONNECTION\_INFO**](iscsi-connection-info.md)                        | Contains information about a connection.                                                                                              |
| [**ISCSI\_DEVICE\_ON\_SESSION**](iscsi-device-on-session.md)                   | Specifies multiple methods for identifying a device associated with an iSCSI login session.                                           |
| [**ISCSI\_LOGIN\_OPTIONS**](iscsi-login-options.md)                            | Utilized by initiators to specify the characteristics of a login session.                                                             |
| [**ISCSI\_SESSION\_INFO**](iscsi-session-info.md)                              | Contains session information in either ASCII or wide character format.                                                                |
| [**ISCSI\_TARGET\_MAPPING**](iscsi-target-mapping.md)                          | Contains information about a target and the HBAs and buses through which the target can be reached.                                   |
| [**ISCSI\_TARGET\_PORTAL\_GROUP**](iscsi-target-portal-group.md)               | Contains information about the portals associated with a portal group.                                                                |
| [**ISCSI\_TARGET\_PORTAL**](iscsi-target-portal.md)                            | Contains information about a portal.                                                                                                  |
| [**ISCSI\_TARGET\_PORTAL\_INFO**](iscsi-target-portal-info.md)                 | Contains information about a target portal.                                                                                           |
| [**ISCSI\_TARGET\_PORTAL\_INFO\_EX**](iscsi-target-portal-info-ex.md)          | Contains information about login credentials to a target portal.                                                                      |
| [**ISCSI\_UNIQUE\_CONNECTION\_ID**](iscsidisc.iscsi_unique_connection_id)       | Contains data that uniquely identifies a connection.                                                                                  |
| [**ISCSI\_UNIQUE\_SESSION\_ID**](iscsi-unique-session-id.md)                   | Contains data that uniquely identifies a session.                                                                                     |
| [**ISCSI\_VERSION\_INFO**](iscsi-version-info.md)                              | Contains the version and build numbers of the iSCSI software initiator and initiator service.                                         |
| [**PERSISTENT\_ISCSI\_LOGIN\_INFO**](persistent-iscsi-login-info.md)           | Describes a login session that the Microsoft iSCSI initiator service reestablishes automatically after the computer boots up.         |
| [**SCSI\_LUN\_LIST**](scsi-lun-list.md)                                        | Utilized to construct a list of logical unit numbers (LUNs) associated with target devices.                                           |
