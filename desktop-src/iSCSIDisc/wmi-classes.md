# iSCSI Discovery Library WMI Classes

The iSCSI Windows Management Instrumentation (WMI) Initiator classes provide various components of the iSCSI Discovery Library API that facilitate discovery and connection to Targets within a Session.

This section details the following iSCSI Discovery Library WMI Initiator classes.



| MSiSCSIInitiator Class                                                                          | Description                                                                                                                                                                       |
|-------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ConnectionInformation**](msiscsiinitiator-connectioninformation.md)                         | Describes the characteristics of an iSCSI connection.<br/>                                                                                                                  |
| [**DeviceOnSession**](msiscsiinitiator-deviceonsession.md)                                     | Describes the characteristics of a device currently associated with a session.<br/>                                                                                         |
| [**IKEPresharedKeyAuthenticationInfo**](msiscsiinitiator-ikepresharedkeyauthenticationinfo.md) | Describes the characteristics of shared keys utilized by the Initiator and the Target.<br/>                                                                                 |
| [**InitiatorClass**](msiscsiinitiator-initiatorclass.md)                                       | Contains the name of the Initiator.<br/>                                                                                                                                    |
| [**IsnsServerClass**](msiscsiinitiator-isnsserverclass.md)                                     | Retrieves and contains information about the iSNS server, which is used to discover targets on a network.<br/>                                                              |
| [**LUNList**](msiscsiinitiator-lunlist.md)                                                     | Defines a mapping between the LUN number that is used by the operating system and the LUN number that is configured in the iSCSI target.<br/>                               |
| [**MethodClass**](msiscsiinitiator-methodclass.md)                                             | Contains all miscellaneous operation methods associated with the iSCSI Initiator.<br/>                                                                                      |
| [**PersistentDevices**](msiscsiinitiator-persistentdevices.md)                                 | Contains path information associated with a persistent device.<br/>                                                                                                         |
| [**PersistentLoginClass**](msiscsiinitiator-persistentloginclass.md)                           | Defines a persistent logon that the operating system initiates automatically when the computer boots up.<br/>                                                               |
| [**Portal**](msiscsiinitiator-portal.md)                                                       | Describes the characteristics of an iSCSI initiator portal. <br/>                                                                                                           |
| [**PortalGroup**](msiscsiinitiator-portalgroup.md)                                             | Describes the characteristics of the initiator portal group.<br/>                                                                                                           |
| [**SendTargetPortalClass**](msiscsiinitiator-sendtargetportalclass.md)                         | Describes the characteristics of a discovered target portal.<br/>                                                                                                           |
| [**SessionClass**](msiscsiinitiator-sessionclass.md)                                           | Describes the characteristics of a session and provides methods that allow the creation and management of connections within the session established by the Initiator.<br/> |
| [**TargetAddress**](msiscsiinitiator-targetaddress.md)                                         | Describes the characteristics of the SCSI address mapped to a target by the operation system.<br/>                                                                          |
| [**TargetClass**](msiscsiinitiator-targetclass.md)                                             | Provides information about connected iSCSI targets as well the ability to login to a specified target.<br/>                                                                 |
| [**TargetLoginOptions**](msiscsiinitiator-targetloginoptions.md)                               | Contains the target login options.<br/>                                                                                                                                     |
| [**TargetMappings**](msiscsiinitiator-targetmappings.md)                                       | Describes the characteristics of the logical unit number (LUN) mapping.<br/>                                                                                                |
