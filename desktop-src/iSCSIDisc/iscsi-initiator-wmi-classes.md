# iSCSI Initiator WMI Classes

The iSCSI Initiator WMI API is included in Windows 8, Windows Server 2012, and later.

## In this section



| Class                                                                                                  | Description                                                                                                                             |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**MSFT\_InitiatorPortToiSCSIConnection**](msft-initiatorporttoiscsiconnection.md)<br/>         | Association between [**InitiatorPort**](https://msdn.microsoft.com/library/windows/desktop/hh830507) and [**iSCSIConnection**](msft-iscsiconnection.md).<br/>    |

| [**MSFT\_iSCSIConnection**](msft-iscsiconnection.md)<br/>                                       | Represents a connection from an iSCSI initiator to an iSCSI target.<br/>                                                          |
| [**MSFT\_iSCSIConnectionToDisk**](msft-iscsiconnectiontodisk.md)<br/>                           | Association between [**iSCSIConnection**](msft-iscsiconnection.md) and [**MSFT\_Disk**](https://msdn.microsoft.com/library/windows/desktop/hh830493).<br/>                |
| [**MSFT\_iSCSIConnectionToiSCSITargetPortal**](msft-iscsiconnectiontoiscsitargetportal.md)<br/> | Association between [**iSCSIConnection**](msft-iscsiconnection.md) and [**iSCSITargetPortal**](msft-iscsitargetportal.md).<br/> |
| [**MSFT\_iSCSISession**](msft-iscsisession.md)<br/>                                             | Represents an iSCSI session.<br/>                                                                                                 |
| [**MSFT\_iSCSISessionToDisk**](msft-iscsisessiontodisk.md)<br/>                                 | Association between [**iSCSISession**](msft-iscsisession.md) and [**MSFT\_Disk**](https://msdn.microsoft.com/library/windows/desktop/hh830493).<br/>                      |
| [**MSFT\_iSCSISessionToiSCSIConnection**](msft-iscsisessiontoiscsiconnection.md)<br/>           | Association between [**iSCSISession**](msft-iscsisession.md) and [**iSCSIConnection**](msft-iscsiconnection.md).<br/>           |
| [**MSFT\_iSCSISessionToiSCSITargetPortal**](msft-iscsisessiontoiscsitargetportal.md)<br/>       | Association between [**iSCSISession**](msft-iscsisession.md) and [**iSCSITargetPortal**](msft-iscsitargetportal.md).<br/>       |
| [**MSFT\_iSCSITarget**](msft-iscsitarget.md)<br/>                                               | Represents an iSCSI target.<br/>                                                                                                  |
| [**MSFT\_iSCSITargetPortal**](msft-iscsitargetportal.md)<br/>                                   | Represents an iSCSI target portal.<br/>                                                                                           |
| [**MSFT\_iSCSITargetToiSCSIConnection**](msft-iscsitargettoiscsiconnection.md)<br/>             | Association between [**iSCSITarget**](msft-iscsitarget.md) and [**iSCSIConnection**](msft-iscsiconnection.md).<br/>             |
| [**MSFT\_iSCSITargetToiSCSISession**](msft-iscsitargettoiscsisession.md)<br/>                   | Association between [**iSCSITarget**](msft-iscsitarget.md) and [**iSCSISession**](msft-iscsisession.md).<br/>                   |
| [**MSFT\_iSCSITargetToiSCSITargetPortal**](msft-iscsitargettoiscsitargetportal.md)<br/>         | Association between [**iSCSITarget**](msft-iscsitarget.md) and [**iSCSITargetPortal**](msft-iscsitargetportal.md).<br/>         |
