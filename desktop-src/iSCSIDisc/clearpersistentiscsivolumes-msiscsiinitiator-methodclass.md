# ClearPersistentIscsiVolumes method of the MSIscsiInitiator\_MethodClass class

The **ClearPersistentIscsiVolumes** method clears all persistent iSCSI volumes.

## Syntax


```mof
uint32 ClearPersistentIscsiVolumes();
```



## Parameters

This method has no parameters.
