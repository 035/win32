# iSCSI Discovery Library Enumerations

This section details the following iSCSI Discovery Library Enumerations:



| Enumerations                                                     | Description                                                                                                                                         |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IKE\_AUTHENTICATION\_METHOD**](ike-authentication-method.md) | Specifies the type of Internet Key Exchange (IKE) authentication method.                                                                            |
| [**ISCSI\_AUTH\_TYPES**](iscsi-auth-types.md)                   | Specifies the type of authentication method utilized.                                                                                               |
| [**ISCSI\_DIGEST\_TYPES**](iscsi-digest-types.md)               | Specifies the digest type.                                                                                                                          |
| [**TARGET\_INFORMATION\_CLASS**](target-information-class.md)   | Specifies information about the indicated target device that the [**GetIScsiTargetInformation**](getiscsitargetinformation.md) function retrieves. |
| [**TARGETPROTOCOLTYPE**](targetprotocoltype.md)                 | Specifies the type of protocol that the initiator must use to communicate with the target.                                                          |
