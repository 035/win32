# iSCSI Discovery Library Enumerations

This section details the following iSCSI Discovery Library Enumerations:



| Enumerations                                                     | Description                                                                                                                                         |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IKE\_AUTHENTICATION\_METHOD**](/previous-versions/windows/desktop/api/Iscsidsc/ne-iscsidsc-ike_authentication_method) | Specifies the type of Internet Key Exchange (IKE) authentication method.                                                                            |
| [**ISCSI\_AUTH\_TYPES**](/previous-versions/windows/desktop/api/Iscsidsc/ne-iscsidsc-iscsi_auth_types)                   | Specifies the type of authentication method utilized.                                                                                               |
| [**ISCSI\_DIGEST\_TYPES**](/previous-versions/windows/desktop/api/Iscsidsc/ne-iscsidsc-iscsi_digest_types)               | Specifies the digest type.                                                                                                                          |
| [**TARGET\_INFORMATION\_CLASS**](/previous-versions/windows/desktop/api/Iscsidsc/ne-iscsidsc-target_information_class)   | Specifies information about the indicated target device that the [**GetIScsiTargetInformation**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-getiscsitargetinformationa) function retrieves. |
| [**TARGETPROTOCOLTYPE**](/previous-versions/windows/desktop/api/Iscsidsc/ne-iscsidsc-targetprotocoltype)                 | Specifies the type of protocol that the initiator must use to communicate with the target.                                                          |
