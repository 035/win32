# iSCSI Discovery Library Enumerations

This section details the following iSCSI Discovery Library Enumerations:



| Enumerations                                                     | Description                                                                                                                                         |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IKE\_AUTHENTICATION\_METHOD**](/windows/previous-versions/Iscsidsc/ne-iscsidsc-ike_authentication_method?branch=master) | Specifies the type of Internet Key Exchange (IKE) authentication method.                                                                            |
| [**ISCSI\_AUTH\_TYPES**](/windows/previous-versions/Iscsidsc/ne-iscsidsc-iscsi_auth_types?branch=master)                   | Specifies the type of authentication method utilized.                                                                                               |
| [**ISCSI\_DIGEST\_TYPES**](/windows/previous-versions/Iscsidsc/ne-iscsidsc-iscsi_digest_types?branch=master)               | Specifies the digest type.                                                                                                                          |
| [**TARGET\_INFORMATION\_CLASS**](/windows/previous-versions/Iscsidsc/ne-iscsidsc-target_information_class?branch=master)   | Specifies information about the indicated target device that the [**GetIScsiTargetInformation**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-getiscsitargetinformationa?branch=master) function retrieves. |
| [**TARGETPROTOCOLTYPE**](/windows/previous-versions/Iscsidsc/ne-iscsidsc-targetprotocoltype?branch=master)                 | Specifies the type of protocol that the initiator must use to communicate with the target.                                                          |
