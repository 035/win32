# iSCSI Discovery Library Functions

This section details the following iSCSI Discovery Library functions.



| Function                                                                           | Description                                                                                                                                                                                                                             |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddIsnsServer**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-addisnsservera)                                             | Adds a new server to the list of iSNS servers that the iSCSI initiator service uses to discover targets.                                                                                                                                |
| [**AddIscsiConnection**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-addiscsiconnectiona)                                   | Adds a new connection to an existing session.                                                                                                                                                                                           |
| [**AddIscsiSendTargetPortal**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-addiscsisendtargetportala)                       | Adds a static target portal to the list of target portals to which the iSCSI initiator service transmits **SendTargets** requests.                                                                                                      |
| [**AddIscsiStaticTarget**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-addiscsistatictargeta)                               | Adds a target to the list of static targets available to the iSCSI initiator.                                                                                                                                                           |
| [**AddPersistentIscsiDevice**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-addpersistentiscsidevicea)                       | Adds a device name, drive letter, or mount point symbolic link for a volume to the list of iSCSI persistently bound volumes and devices.                                                                                                |
| [**AddRadiusServer**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-addradiusservera)                                         | Adds a new RADIUS server to the list referenced by the iSCSI initiator service during authentication.                                                                                                                                   |
| [**ClearPersistentIscsiDevices**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-clearpersistentiscsidevices)                 | Removes all volumes and devices from the list of persistently bound iSCSI volumes.                                                                                                                                                      |
| [**GetDevicesForIscsiSession**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-getdevicesforiscsisessiona)                     | Retrieves information about the devices associated with the session.                                                                                                                                                                    |
| [**GetIscsiIKEInfo**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-getiscsiikeinfoa)                                         | Retrieves the IPsec policy and any established pre-shared key values associated with an initiator Host-Bus Adapter (HBA).                                                                                                               |
| [**GetIscsiInitiatorNodeName**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-getiscsiinitiatornodenamea)                     | Retrieves the common initiator node name that is used when establishing sessions from this computer.                                                                                                                                    |
| [**GetIscsiSessionList**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-getiscsisessionlista)                                 | Retrieves the list of active iSCSI sessions.                                                                                                                                                                                            |
| [**GetIscsiTargetInformation**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-getiscsitargetinformationa)                     | Retrieves information about the specified target.                                                                                                                                                                                       |
| [**GetIscsiVersionInformation**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-getiscsiversioninformation)                   | Retrieves information about the initiator version.                                                                                                                                                                                      |
| [**LoginIscsiTarget**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-loginiscsitargeta)                                       | Establishes a full featured login session with the indicated target.                                                                                                                                                                    |
| [**LogoutIscsiTarget**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-logoutiscsitarget)                                     | Closes the indicated login session.                                                                                                                                                                                                     |
| [**RefreshIsnsServer**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-refreshisnsservera)                                     | Instructs the iSCSI initiator service to query the indicated iSNS server to refresh the list over discovered targets for the iSCSI initiator service.                                                                                   |
| [**RefreshIscsiSendTargetPortal**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-refreshiscsisendtargetportala)               | Instructs the iSCSI initiator service to establish a discovery session with the indicated target portal and transmit a **SendTargets** request to the portal to refresh the list of discovered targets for the iSCSI initiator service. |
| [**RemoveIsnsServer**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-removeisnsservera)                                       | Removes a server from the list of iSNS servers that the iSCSI initiator service uses to discover targets.                                                                                                                               |
| [**RemoveIscsiConnection**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-removeiscsiconnection)                             | Removes a connection from an active session.                                                                                                                                                                                            |
| [**RemoveIscsiPersistentTarget**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-removeiscsipersistenttargeta)                 | Removes a persistent login for the specified hardware initiator (HBA), iniator port and target portal.                                                                                                                                  |
| [**RemoveIscsiSendTargetPortal**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-removeiscsisendtargetportala)                 | removes a portal from the list of portals to which the iSCSI initiator service sends **SendTargets** requests to discover targets.                                                                                                      |
| [**RemoveIscsiStaticTarget**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-removeiscsistatictargeta)                         | Removes a target from the list of static targets available to the computer.                                                                                                                                                             |
| [**RemovePersistentIscsiDevice**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-removepersistentiscsidevicea)                 | Removes a device or volume from the list of persistently bound iSCSI volumes.                                                                                                                                                           |
| [**RemoveRadiusServer**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-removeradiusservera)                                   | Removes a RADIUS server entry from the RADIUS server list that an iSCSI initiator is configured with.                                                                                                                                   |
| [**ReportActiveIscsiTargetMappings**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-reportactiveiscsitargetmappingsa)         | Retrieves the target mappings that are currently active for all initiators on the computer.                                                                                                                                             |
| [**ReportIsnsServerList**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-reportisnsserverlista)                               | Retrieves the list of iSNS servers that the iSCSI initiator service queries for discovered targets.                                                                                                                                     |
| [**ReportIscsiInitiatorList**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-reportiscsiinitiatorlista)                       | Retrieves the list of HBA initiators that are running on the computer.                                                                                                                                                                  |
| [**ReportIscsiPersistentLogins**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-reportiscsipersistentloginsa)                 | Retrieves the list of persistent login targets.                                                                                                                                                                                         |
| [**ReportIscsiSendTargetPortals**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-reportiscsisendtargetportalsa)               | Retrieves a list of target portals that the iSCSI initiator service uses to perform automatic discovery with **SendTarget** requests.                                                                                                   |
| [**ReportIscsiSendTargetPortalsEx**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-reportiscsisendtargetportalsexa)           | Retrieves a list of static target portals that the iSCSI initiator service uses to perform automatic discovery with **SendTarget** requests.                                                                                            |
| [**ReportIscsiTargets**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-reportiscsitargetsa)                                   | Retrieves the list of targets that the iSCSI initiator service has discovered, and optionally instructs the iSCSI initiator service to refresh the list.                                                                                |
| [**ReportIscsiTargetPortals**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-reportiscsitargetportalsa)                       | Retrieves target portal information discovered by the iSCSI initiator service.                                                                                                                                                          |
| [**ReportPersistentIscsiDevices**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-reportpersistentiscsidevicesa)               | Retrieves the list of persistently bound volumes and devices.                                                                                                                                                                           |
| [**ReportRadiusServerList**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-reportradiusserverlista)                           | Retrieves the list of persistently bound volumes and devices.                                                                                                                                                                           |
| [**SendScsiInquiry**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-sendscsiinquiry)                                         | Sends a SCSI INQUIRY command to the specified target.                                                                                                                                                                                   |
| [**SendScsiReadCapacity**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-sendscsireadcapacity)                               | Sends a SCSI READ CAPACITY command to the indicated target.                                                                                                                                                                             |
| [**SendScsiReportLuns**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-sendscsireportluns)                                   | Sends a SCSI REPORT LUNS command to the indicated target.                                                                                                                                                                               |
| [**SetIscsiGroupPresharedKey**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-setiscsigrouppresharedkey)                     | Establishes the default group preshared key for all initiators on the computer.                                                                                                                                                         |
| [**SetIscsiIKEInfo**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-setiscsiikeinfoa)                                         | Establishes the IPsec policy and preshared key for the indicated initiator to use when performing iSCSI connections.                                                                                                                    |
| [**SetIscsiInitiatorCHAPSharedSecret**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-setiscsiinitiatorchapsharedsecret)     | Establishes the default CHAP shared secret for all initiators on the computer.                                                                                                                                                          |
| [**SetIscsiInitiatorRADIUSSharedSecret**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-setiscsiinitiatorradiussharedsecret) | Establishes the RADIUS shared secret.                                                                                                                                                                                                   |
| [**SetIscsiInitiatorNodeName**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-setiscsiinitiatornodenamea)                     | Establishes an initiator node name for the computer that all initiators on the computer use when communicating with other nodes.                                                                                                        |
| [**SetIscsiTunnelModeOuterAddress**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-setiscsitunnelmodeouteraddressa)           | Establishes the tunnel-mode outer address that the indicated HBA initiator uses when communicating in IPsec tunnel mode through the specified port.                                                                                     |
| [**SetupPersistentIscsiDevices**](/previous-versions/windows/desktop/api/Iscsidsc/nf-iscsidsc-setuppersistentiscsidevices)                 | Builds the list of devices and volumes that belong to iSCSI targets that are connected to the computer, and saves the list in non-volatile cache of the iSCSI initiator service.                                                        |
