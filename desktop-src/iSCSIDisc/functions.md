# iSCSI Discovery Library Functions

This section details the following iSCSI Discovery Library functions.



| Function                                                                           | Description                                                                                                                                                                                                                             |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddIsnsServer**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-addisnsservera?branch=master)                                             | Adds a new server to the list of iSNS servers that the iSCSI initiator service uses to discover targets.                                                                                                                                |
| [**AddIscsiConnection**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-addiscsiconnectiona?branch=master)                                   | Adds a new connection to an existing session.                                                                                                                                                                                           |
| [**AddIscsiSendTargetPortal**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-addiscsisendtargetportala?branch=master)                       | Adds a static target portal to the list of target portals to which the iSCSI initiator service transmits **SendTargets** requests.                                                                                                      |
| [**AddIscsiStaticTarget**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-addiscsistatictargeta?branch=master)                               | Adds a target to the list of static targets available to the iSCSI initiator.                                                                                                                                                           |
| [**AddPersistentIscsiDevice**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-addpersistentiscsidevicea?branch=master)                       | Adds a device name, drive letter, or mount point symbolic link for a volume to the list of iSCSI persistently bound volumes and devices.                                                                                                |
| [**AddRadiusServer**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-addradiusservera?branch=master)                                         | Adds a new RADIUS server to the list referenced by the iSCSI initiator service during authentication.                                                                                                                                   |
| [**ClearPersistentIscsiDevices**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-clearpersistentiscsidevices?branch=master)                 | Removes all volumes and devices from the list of persistently bound iSCSI volumes.                                                                                                                                                      |
| [**GetDevicesForIscsiSession**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-getdevicesforiscsisessiona?branch=master)                     | Retrieves information about the devices associated with the session.                                                                                                                                                                    |
| [**GetIscsiIKEInfo**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-getiscsiikeinfoa?branch=master)                                         | Retrieves the IPsec policy and any established pre-shared key values associated with an initiator Host-Bus Adapter (HBA).                                                                                                               |
| [**GetIscsiInitiatorNodeName**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-getiscsiinitiatornodenamea?branch=master)                     | Retrieves the common initiator node name that is used when establishing sessions from this computer.                                                                                                                                    |
| [**GetIscsiSessionList**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-getiscsisessionlista?branch=master)                                 | Retrieves the list of active iSCSI sessions.                                                                                                                                                                                            |
| [**GetIscsiTargetInformation**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-getiscsitargetinformationa?branch=master)                     | Retrieves information about the specified target.                                                                                                                                                                                       |
| [**GetIscsiVersionInformation**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-getiscsiversioninformation?branch=master)                   | Retrieves information about the initiator version.                                                                                                                                                                                      |
| [**LoginIscsiTarget**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-loginiscsitargeta?branch=master)                                       | Establishes a full featured login session with the indicated target.                                                                                                                                                                    |
| [**LogoutIscsiTarget**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-logoutiscsitarget?branch=master)                                     | Closes the indicated login session.                                                                                                                                                                                                     |
| [**RefreshIsnsServer**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-refreshisnsservera?branch=master)                                     | Instructs the iSCSI initiator service to query the indicated iSNS server to refresh the list over discovered targets for the iSCSI initiator service.                                                                                   |
| [**RefreshIscsiSendTargetPortal**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-refreshiscsisendtargetportala?branch=master)               | Instructs the iSCSI initiator service to establish a discovery session with the indicated target portal and transmit a **SendTargets** request to the portal to refresh the list of discovered targets for the iSCSI initiator service. |
| [**RemoveIsnsServer**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-removeisnsservera?branch=master)                                       | Removes a server from the list of iSNS servers that the iSCSI initiator service uses to discover targets.                                                                                                                               |
| [**RemoveIscsiConnection**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-removeiscsiconnection?branch=master)                             | Removes a connection from an active session.                                                                                                                                                                                            |
| [**RemoveIscsiPersistentTarget**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-removeiscsipersistenttargeta?branch=master)                 | Removes a persistent login for the specified hardware initiator (HBA), iniator port and target portal.                                                                                                                                  |
| [**RemoveIscsiSendTargetPortal**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-removeiscsisendtargetportala?branch=master)                 | removes a portal from the list of portals to which the iSCSI initiator service sends **SendTargets** requests to discover targets.                                                                                                      |
| [**RemoveIscsiStaticTarget**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-removeiscsistatictargeta?branch=master)                         | Removes a target from the list of static targets available to the computer.                                                                                                                                                             |
| [**RemovePersistentIscsiDevice**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-removepersistentiscsidevicea?branch=master)                 | Removes a device or volume from the list of persistently bound iSCSI volumes.                                                                                                                                                           |
| [**RemoveRadiusServer**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-removeradiusservera?branch=master)                                   | Removes a RADIUS server entry from the RADIUS server list that an iSCSI initiator is configured with.                                                                                                                                   |
| [**ReportActiveIscsiTargetMappings**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-reportactiveiscsitargetmappingsa?branch=master)         | Retrieves the target mappings that are currently active for all initiators on the computer.                                                                                                                                             |
| [**ReportIsnsServerList**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-reportisnsserverlista?branch=master)                               | Retrieves the list of iSNS servers that the iSCSI initiator service queries for discovered targets.                                                                                                                                     |
| [**ReportIscsiInitiatorList**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-reportiscsiinitiatorlista?branch=master)                       | Retrieves the list of HBA initiators that are running on the computer.                                                                                                                                                                  |
| [**ReportIscsiPersistentLogins**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-reportiscsipersistentloginsa?branch=master)                 | Retrieves the list of persistent login targets.                                                                                                                                                                                         |
| [**ReportIscsiSendTargetPortals**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-reportiscsisendtargetportalsa?branch=master)               | Retrieves a list of target portals that the iSCSI initiator service uses to perform automatic discovery with **SendTarget** requests.                                                                                                   |
| [**ReportIscsiSendTargetPortalsEx**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-reportiscsisendtargetportalsexa?branch=master)           | Retrieves a list of static target portals that the iSCSI initiator service uses to perform automatic discovery with **SendTarget** requests.                                                                                            |
| [**ReportIscsiTargets**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-reportiscsitargetsa?branch=master)                                   | Retrieves the list of targets that the iSCSI initiator service has discovered, and optionally instructs the iSCSI initiator service to refresh the list.                                                                                |
| [**ReportIscsiTargetPortals**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-reportiscsitargetportalsa?branch=master)                       | Retrieves target portal information discovered by the iSCSI initiator service.                                                                                                                                                          |
| [**ReportPersistentIscsiDevices**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-reportpersistentiscsidevicesa?branch=master)               | Retrieves the list of persistently bound volumes and devices.                                                                                                                                                                           |
| [**ReportRadiusServerList**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-reportradiusserverlista?branch=master)                           | Retrieves the list of persistently bound volumes and devices.                                                                                                                                                                           |
| [**SendScsiInquiry**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-sendscsiinquiry?branch=master)                                         | Sends a SCSI INQUIRY command to the specified target.                                                                                                                                                                                   |
| [**SendScsiReadCapacity**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-sendscsireadcapacity?branch=master)                               | Sends a SCSI READ CAPACITY command to the indicated target.                                                                                                                                                                             |
| [**SendScsiReportLuns**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-sendscsireportluns?branch=master)                                   | Sends a SCSI REPORT LUNS command to the indicated target.                                                                                                                                                                               |
| [**SetIscsiGroupPresharedKey**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-setiscsigrouppresharedkey?branch=master)                     | Establishes the default group preshared key for all initiators on the computer.                                                                                                                                                         |
| [**SetIscsiIKEInfo**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-setiscsiikeinfoa?branch=master)                                         | Establishes the IPsec policy and preshared key for the indicated initiator to use when performing iSCSI connections.                                                                                                                    |
| [**SetIscsiInitiatorCHAPSharedSecret**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-setiscsiinitiatorchapsharedsecret?branch=master)     | Establishes the default CHAP shared secret for all initiators on the computer.                                                                                                                                                          |
| [**SetIscsiInitiatorRADIUSSharedSecret**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-setiscsiinitiatorradiussharedsecret?branch=master) | Establishes the RADIUS shared secret.                                                                                                                                                                                                   |
| [**SetIscsiInitiatorNodeName**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-setiscsiinitiatornodenamea?branch=master)                     | Establishes an initiator node name for the computer that all initiators on the computer use when communicating with other nodes.                                                                                                        |
| [**SetIscsiTunnelModeOuterAddress**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-setiscsitunnelmodeouteraddressa?branch=master)           | Establishes the tunnel-mode outer address that the indicated HBA initiator uses when communicating in IPsec tunnel mode through the specified port.                                                                                     |
| [**SetupPersistentIscsiDevices**](/windows/previous-versions/Iscsidsc/nf-iscsidsc-setuppersistentiscsidevices?branch=master)                 | Builds the list of devices and volumes that belong to iSCSI targets that are connected to the computer, and saves the list in non-volatile cache of the iSCSI initiator service.                                                        |
