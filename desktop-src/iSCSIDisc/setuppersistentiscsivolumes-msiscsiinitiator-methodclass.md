# SetupPersistentIScsiVolumes method of the MSIscsiInitiator\_MethodClass class

The **SetupPersistentIscsiVolumes** method sets iSCSI volumes to be persistent.

## Syntax


```mof
uint32 SetupPersistentIScsiVolumes();
```



## Parameters

This method has no parameters.
