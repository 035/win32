# WebSocket Protocol Component API Enumerations

The WebSocket Protocol Component API defines these enumerations.

## In this section



| Topic                                                                      | Description                                                                                            |
|----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**WEB\_SOCKET\_ACTION**](/windows/win32/Websocket/ne-websocket-_web_socket_action?branch=master)<br/>                | specifies actions to be taken by WebSocket applications<br/>                                     |
| [**WEB\_SOCKET\_ACTION\_QUEUE**](/windows/win32/Websocket/ne-websocket-_web_socket_action_queue?branch=master)<br/>   | specifies the action types returned by [**WebSocketGetAction**](https://msdn.microsoft.com/library/windows/desktop/hh449336)<br/> |
| [**WEB\_SOCKET\_BUFFER\_TYPE**](/windows/win32/Websocket/ne-websocket-_web_socket_buffer_type?branch=master)<br/>     | specifies the bit values used to construct the WebSocket frame header<br/>                       |
| [**WEB\_SOCKET\_CLOSE\_STATUS**](/windows/win32/Websocket/ne-websocket-_web_socket_close_status?branch=master)<br/>   | specifies the WebSocket close status<br/>                                                        |
| [**WEB\_SOCKET\_PROPERTY\_TYPE**](/windows/win32/Websocket/ne-websocket-_web_socket_property_type?branch=master)<br/> | specifies a WebSocket property type<br/>                                                         |
