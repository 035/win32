# WebSocket Protocol Component API Functions

The WebSocket Protocol Component API defines these functions.

## In this section



| Topic                                                                             | Description                                                                                                                                                                                                 |
|-----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WebSocketAbortHandle**](websocketaborthandle.md)<br/>                   | aborts a WebSocket session handle created by [**WebSocketCreateClientHandle**](https://msdn.microsoft.com/library/windows/desktop/hh449325) or [**WebSocketCreateServerHandle**](https://msdn.microsoft.com/library/windows/desktop/hh449327).<br/>   |
| [**WebSocketBeginClientHandshake**](websocketbeginclienthandshake.md)<br/> | begins the client-side handshake.<br/>                                                                                                                                                                |
| [**WebSocketBeginServerHandshake**](websocketbeginserverhandshake.md)<br/> | begins the server-side handshake.<br/>                                                                                                                                                                |
| [**WebSocketCompleteAction**](websocketcompleteaction.md)<br/>             | completes an action started by [**WebSocketGetAction**](https://msdn.microsoft.com/library/windows/desktop/hh449336).<br/>                                                                                                             |
| [**WebSocketCreateClientHandle**](websocketcreateclienthandle.md)<br/>     | creates a client-side WebSocket session handle.<br/>                                                                                                                                                  |
| [**WebSocketCreateServerHandle**](websocketcreateserverhandle.md)<br/>     | creates a server-side WebSocket session handle.<br/>                                                                                                                                                  |
| [**WebSocketDeleteHandle**](websocketdeletehandle.md)<br/>                 | deletes a WebSocket session handle created by [**WebSocketCreateClientHandle**](https://msdn.microsoft.com/library/windows/desktop/hh449325) or [**WebSocketCreateServerHandle**](https://msdn.microsoft.com/library/windows/desktop/hh449327).<br/>  |
| [**WebSocketEndClientHandshake**](websocketendclienthandshake.md)<br/>     | completes the client-side handshake.<br/>                                                                                                                                                             |
| [**WebSocketEndServerHandshake**](websocketendserverhandshake.md)<br/>     | completes the server-side handshake.<br/>                                                                                                                                                             |
| [**WebSocketGetAction**](websocketgetaction.md)<br/>                       | returns an action from a call to [**WebSocketSend**](https://msdn.microsoft.com/library/windows/desktop/hh449340), [**WebSocketReceive**](https://msdn.microsoft.com/library/windows/desktop/hh449338) or [**WebSocketCompleteAction**](https://msdn.microsoft.com/library/windows/desktop/hh449323).<br/> |
| [**WebSocketGetGlobalProperty**](websocketgetglobalproperty.md)<br/>       | gets a single WebSocket property.<br/>                                                                                                                                                                |
| [**WebSocketReceive**](websocketreceive.md)<br/>                           | adds a receive operation to the protocol component operation queue.<br/>                                                                                                                              |
| [**WebSocketSend**](websocketsend.md)<br/>                                 | adds a send operation to the protocol component operation queue.<br/>                                                                                                                                 |
