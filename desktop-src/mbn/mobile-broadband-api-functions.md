# Wireless Wide Area Network Functions

The wireless wide area network (WWAN, also known as Mobile Broadband) API defines and uses the following functions.

## In this section



| Topic                                                                 | Description                                                                                                                                                                                                                   |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WwanCloseHandle**](wwanclosehandle.md)<br/>                 | The [**WwanCloseHandle**](wwanclosehandle.md) function closes a handle to the mobile broadband service.<br/>                                                                                                           |
| [**WwanEnumerateInterfaces**](wwanenumerateinterfaces.md)<br/> | The [**WwanEnumerateInterfaces**](wwanenumerateinterfaces.md) function enumerates all of the [mobile broadband interfaces](mobile-broadband-networks-api-interfaces.md) currently enabled on the local computer.<br/> |
| [**WwanFreeMemory**](wwanfreememory.md)<br/>                   | The [**WwanFreeMemory**](wwanfreememory.md) function frees a block of memory.<br/>                                                                                                                                     |
| [**WwanOpenHandle**](wwanopenhandle.md)<br/>                   | The [**WwanOpenHandle**](wwanopenhandle.md) function opens a connection to the server.<br/>                                                                                                                            |
| [**WwanQueryInterface**](wwanqueryinterface.md)<br/>           | The [**WwanQueryInterface**](wwanqueryinterface.md) function queries the mobile broadband service for info about a [mobile broadband interface](mobile-broadband-networks-api-interfaces.md).<br/>                    |



 

> [!Note]  
> These functions may be altered or unavailable for releases after Windows 8.1. Instead, use the [mobile broadband interfaces](mobile-broadband-networks-api-interfaces.md).
