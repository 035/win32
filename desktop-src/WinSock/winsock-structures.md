# Winsock Structures

The following list provides concise descriptions of each Winsock structure and typedef. For additional information on any structure or typedef, click the structure or typedef name.



| Structure                                                                                 | Description                                                                                                                                                                                                                                                                       |
|-------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**addrinfo**](addrinfo-2.md)                                                            | Used by the [**getaddrinfo**](getaddrinfo-2.md) function to hold host address information.                                                                                                                                                                                       |
| [**addrinfoW**](addrinfow.md)                                                            | Used by the [**GetAddrInfoW**](getaddrinfow.md) function to hold host address information.                                                                                                                                                                                       |
| [**addrinfoex**](addrinfoex.md)                                                          | Used by the [**GetAddrInfoEx**](getaddrinfoex.md) function to hold host address information.                                                                                                                                                                                     |
| [**addrinfoex2**](addrinfoex2.md)                                                        | Used by the [**GetAddrInfoEx**](getaddrinfoex.md) function to hold host address information when both a canonical name and a fully qualified domain name have been requested.                                                                                                    |
| [**addrinfoex3**](addrinfoex3.md)                                                        | Used by the [**GetAddrInfoEx**](getaddrinfoex.md) function to hold host address information when a specific network interface has been requested.                                                                                                                                |
| [**addrinfoex4**](addrinfoex4.md)                                                        | Used by the [**GetAddrInfoEx**](getaddrinfoex.md) function to hold host address information when a handle to the fully qualified domain name has been requested.                                                                                                                 |
| [**AFPROTOCOLS**](afprotocols-2.md)                                                      | Supplies a list of protocols to which application programmers can constrain queries. Used for query purposes only.                                                                                                                                                                |
| [**BLOB**](blob-2.md)                                                                    | Contains information about a block of data. Derived from Binary Large Object.                                                                                                                                                                                                     |
| [**CSADDR\_INFO**](csaddr-info-2.md)                                                     | Contains Winsock address information for a network service or namespace provider.                                                                                                                                                                                                 |
| [**fd\_set**](fd-set-2.md)                                                               | Used by various Winsock functions and service providers, such as [**select**](select-2.md), to place sockets into a "set" for various purposes, such as testing a given socket for readability.                                                                                  |
| [**GROUP\_FILTER**](group-filter.md)                                                     | Provides multicast filtering parameters for multicast IPv6 or IPv4 addresses.                                                                                                                                                                                                     |
| [**GROUP\_REQ**](group-req.md)                                                           | Provides multicast group information for IPv6 or IPv4 addresses.                                                                                                                                                                                                                  |
| [**GROUP\_SOURCE\_REQ**](group-source-req.md)                                            | Provides multicast group information for IPv6 or IPv4 addresses that includes the source IP address.                                                                                                                                                                              |
| [**hostent**](hostent-2.md)                                                              | Stores information about a given host, such as host name, IP address, and so forth.                                                                                                                                                                                               |
| [**in\_addr**](in-addr-2.md)                                                             | Represents an IPv4 Internet address.                                                                                                                                                                                                                                              |
| [**in\_pktinfo**](in-pktinfo-2.md)                                                       | Stores received packet address information, and is used by Windows to return information about received packets.                                                                                                                                                                  |
| [**in6\_addr**](in6-addr.md)                                                             | Represents an IPv6 Internet address.                                                                                                                                                                                                                                              |
| [**in6\_pktinfo**](in6-pktinfo-2.md)                                                     | Stores received IPv6 packet address information, and is used by Windows to return information about received packets.                                                                                                                                                             |
| [**INET\_PORT\_RANGE**](inet-port-range.md)                                              | Provides input data used by the the **SIO\_ACQUIRE\_PORT\_RESERVATION** Ioctl to acquire a runtime reservation for a block of TCP or UDP ports.                                                                                                                                   |
| [**INET\_PORT\_RESERVATION\_INSTANCE**](inet-port-reservation-instance.md)               | Contains a port reservation and a token for a block of TCP or UDP ports.                                                                                                                                                                                                          |
| [**INET\_PORT\_RESERVATION\_TOKEN**](inet-port-reservation-token.md)                     | Contains a port reservation token for a block of TCP or UDP ports.                                                                                                                                                                                                                |
| [**ip\_mreq**](ip-mreq.md)                                                               | Provides multicast group information for IPv4 addresses.                                                                                                                                                                                                                          |
| [**ip\_mreq\_source**](ip-mreq-source.md)                                                | Provides multicast group and source information for IPv4 addresses.                                                                                                                                                                                                               |
| [**ip\_msfilter**](ip-msfilter.md)                                                       | Provides multicast filtering parameters for IPv4 addresses.                                                                                                                                                                                                                       |
| [**ipv6\_mreq**](ipv6-mreq.md)                                                           | Provides multicast group information for IPv6 addresses.                                                                                                                                                                                                                          |
| [**linger**](linger-2.md)                                                                | Maintains information about a specific socket that specifies how that socket should behave when data is queued to be sent and the [**closesocket**](closesocket-2.md) function is called on the socket.                                                                          |
| [**NAPI\_DOMAIN\_DESCRIPTION\_BLOB**](napi-domain-description-blob.md)                   | Describes a domain handled by a namespace provider for the NS\_EMAIL namespace.                                                                                                                                                                                                   |
| [**NAPI\_PROVIDER\_INSTALLATION\_BLOB**](napi-provider-installation-blob.md)             | Contains the information required to install a namespace provider for the NS\_EMAIL namespace.                                                                                                                                                                                    |
| [**NS\_SERVICE\_INFO**](ns-service-info-2.md)                                            | Contains information about a network service or a network service type in the context of a specified namespace, or a set of default namespaces.                                                                                                                                   |
| [**PROTOCOL\_INFO**](protocol-info-2.md)                                                 | Contains information about a protocol.                                                                                                                                                                                                                                            |
| [**protoent**](protoent-2.md)                                                            | Contains the name and protocol numbers that correspond to a given protocol name.                                                                                                                                                                                                  |
| [**REAL\_TIME\_NOTIFICATION\_SETTING\_INPUT**](real-time-notification-setting-input.md)  | Contains input settings to apply for the **REAL\_TIME\_NOTIFICATION\_CAPABILITY** transport setting for a TCP socket that is used with [**ControlChannelTrigger**](w_net_sock.controlchanneltrigger) to receive background network notifications in a Windows Store app           |
| [**REAL\_TIME\_NOTIFICATION\_SETTING\_OUTPUT**](real-time-notification-setting-input.md) | Contains the output settings from a query for the **REAL\_TIME\_NOTIFICATION\_CAPABILITY** transport setting for a TCP socket that is used with [**ControlChannelTrigger**](w_net_sock.controlchanneltrigger) to receive background network notifications in a Windows Store app. |
| [**RIO\_EXTENSION\_FUNCTION\_TABLE**](rio-extension-function-table.md)                   | Contains information on the functions that implement the Winsock registered I/O extensions.                                                                                                                                                                                       |
| [**RIO\_BUF**](rio-buf.md)                                                               | Specifies a portion of a registered buffer used for sending or receiving network data with the Winsock registered I/O extensions.                                                                                                                                                 |
| [**RIO\_BUFFERID**](rio-bufferid.md)                                                     | Specifies a registered buffer descriptor used with the Winsock registered I/O extensions.                                                                                                                                                                                         |
| [**RIO\_CQ**](riocqueue.md)                                                              | Specifies a completion queue descriptor used for I/O completion notification by send and receive requests with the Winsock registered I/O extensions.                                                                                                                             |
| [**RIO\_NOTIFICATION\_COMPLETION**](rio-notification-completion.md)                      | Specifies the method for I/O completion to be used with a [**RIONotify**](rionotify.md) function for sending or receiving network data with the Winsock registered I/O extensions.                                                                                               |
| [**RIO\_RQ**](riorqueue.md)                                                              | Specifies a socket descriptor used by send and receive requests with the Winsock registered I/O extensions.                                                                                                                                                                       |
| [**RIORESULT**](rioresult.md)                                                            | Contains data used to indicate request completion results used with the Winsock registered I/O extensions.                                                                                                                                                                        |
| [**RM\_FEC\_INFO**](rm-fec-info.md)                                                      | Specifies settings for using forward error correction (FEC) with Reliable Multicast.                                                                                                                                                                                              |
| [**RM\_RECEIVER\_STATS**](rm-receiver-stats.md)                                          | Provides statistical information for a Reliable Multicast receiver session.                                                                                                                                                                                                       |
| [**RM\_SEND\_WINDOW**](rm-send-window.md)                                                | Specifies the Reliable Multicast send window.                                                                                                                                                                                                                                     |
| [**RM\_SENDER\_STATS**](rm-sender-stats.md)                                              | Provides statistical information for a Reliable Multicast sender session.                                                                                                                                                                                                         |
| [**servent**](servent-2.md)                                                              | Stores or returns the name and service number for a given service name.                                                                                                                                                                                                           |
| [**SERVICE\_ADDRESS**](service-address-2.md)                                             | Contains address information for a service.                                                                                                                                                                                                                                       |
| [**SERVICE\_ADDRESSES**](service-addresses-2.md)                                         | Contains an array of [**SERVICE\_ADDRESS**](service-address-2.md) data structures.                                                                                                                                                                                               |
| [**SERVICE\_INFO**](service-info-2.md)                                                   | Contains information about a network service or a network service type.                                                                                                                                                                                                           |
| [**SERVICE\_TYPE\_INFO\_ABS**](service-type-info-abs-2.md)                               | Contains information about a network service type.                                                                                                                                                                                                                                |
| [**SERVICE\_TYPE\_VALUE\_ABS**](service-type-value-abs-2.md)                             | Contains information about a network-service type value. The information may be specific to a namespace.                                                                                                                                                                          |
| [**sockaddr**](sockaddr-2.md)                                                            | Contains socket address information. The [**sockaddr**](sockaddr-2.md) structure varies depending on the protocol selected. Limited to IPv4; use [**SOCKADDR\_STORAGE**](sockaddr-storage-2.md) instead.                                                                        |
| [**SOCKADDR\_IRDA**](sockaddr-irda-2.md)                                                 | Used with IrDA socket operations, defined by address family *AF\_IRDA*.                                                                                                                                                                                                           |
| [**SOCKADDR\_STORAGE**](sockaddr-storage-2.md)                                           | Stores socket address information, and is sufficiently large to store IPv4 or IPv6 address information, promoting protocol-family and protocol-version independence. Use this structure in place of the [**sockaddr**](sockaddr-2.md) structure.                                 |
| [**SOCKET\_ADDRESS**](socket-address-2.md)                                               | Stores protocol-specific address information.                                                                                                                                                                                                                                     |
| [**SOCKET\_ADDRESS\_LIST**](socket-address-list.md)                                      | Stores an array of [**SOCKET\_ADDRESS**](socket-address-2.md) structures that contain protocol-specific address information.                                                                                                                                                     |
| [**SOCKET\_PEER\_TARGET\_NAME**](socket-peer-target-name.md)                             | Contains the IP address and name for a peer target and the type of security protocol to be used on a socket.                                                                                                                                                                      |
| [**SOCKET\_SECURITY\_QUERY\_INFO**](socket-security-query-info.md)                       | Contains security information returned by the [**WSAQuerySocketSecurity**](wsaquerysocketsecurity.md) function.                                                                                                                                                                  |
| [**SOCKET\_SECURITY\_QUERY\_TEMPLATE**](socket-security-query-template.md)               | Contains the security template used by the [**WSAQuerySocketSecurity**](wsaquerysocketsecurity.md) function.                                                                                                                                                                     |
| [**SOCKET\_SECURITY\_SETTINGS**](socket-security-settings.md)                            | Specifies generic security requirements for a socket.                                                                                                                                                                                                                             |
| [**SOCKET\_SECURITY\_SETTINGS\_IPSEC**](socket-security-settings-ipsec.md)               | Specifies various security requirements and settings that are specific to IPsec.                                                                                                                                                                                                  |
| [**timeval**](timeval-2.md)                                                              | Used to specify time values. Associated with the Berkeley Software Distribution (BSD) file Time.h.                                                                                                                                                                                |
| [**TRANSMIT\_FILE\_BUFFERS**](transmit-file-buffers-2.md)                                | Specifies data to be transmitted before and after file data during a [**TransmitFile**](transmitfile-2.md) transfer operation.                                                                                                                                                   |
| [**TRANSMIT\_PACKETS\_ELEMENT**](transmit-packets-element-2.md)                          | Specifies a single data element to be transmitted by the [**TransmitPackets**](transmitpackets-2.md) function.                                                                                                                                                                   |
| [**TRANSPORT\_SETTING\_ID**](transport-setting-id.md)                                    | Specifies the transport setting ID used by the [**SIO\_APPLY\_TRANSPORT\_SETTING**](sio-apply-transport-setting.md) and [**SIO\_QUERY\_TRANSPORT\_SETTING**](sio-query-transport-setting.md) IOCTLs to apply or query the transport setting for a socket.                       |
| [**WSABUF**](wsabuf-2.md)                                                                | Enables the creation or manipulation of a data buffer.                                                                                                                                                                                                                            |
| [**WSACOMPLETION**](wsacompletion.md)                                                    | Specifies completion notification settings for I/O control calls made to a registered namespace.                                                                                                                                                                                  |
| [**WSADATA**](wsadata-2.md)                                                              | Contains information about the Windows Sockets implementation.                                                                                                                                                                                                                    |
| [**WSAMSG**](wsamsg-2.md)                                                                | Stores address and optional control information about connected and unconnected sockets. Used with the [**WSARecvMsg**](wsarecvmsg-2.md) function.                                                                                                                               |
| [**WSANAMESPACE\_INFO**](wsanamespace-info-2.md)                                         | Contains registration information for a namespace provider.                                                                                                                                                                                                                       |
| [**WSANAMESPACE\_INFOEX**](wsanamespace-infoex.md)                                       | Contains enhanced registration information for a namespace provider.                                                                                                                                                                                                              |
| [**WSANETWORKEVENTS**](wsanetworkevents-2.md)                                            | Stores a socket's internal information about network events.                                                                                                                                                                                                                      |
| [**WSANSCLASSINFO**](wsansclassinfo.md)                                                  | Provides individual parameter information for a specific Winsock namespace.                                                                                                                                                                                                       |
| [**WSAOVERLAPPED**](wsaoverlapped-2.md)                                                  | Provides a communication medium between the initiation of an overlapped I/O operation and its subsequent completion.                                                                                                                                                              |
| [**WSAPOLLFD**](pollfd.md)                                                               | Stores socket information used by the [**WSAPoll**](wsapoll.md) function.                                                                                                                                                                                                        |
| [**WSAPROTOCOL\_INFO**](wsaprotocol-info-2.md)                                           | Stores or retrieves complete information for a given protocol.                                                                                                                                                                                                                    |
| [**WSAPROTOCOL\_INFOW**](wsaprotocol-info-2.md)                                          | Store or retrieves complete information for a given protocol. The protocol name is represented as an array of Unicode characters.                                                                                                                                                 |
| [**WSAPROTOCOLCHAIN**](wsaprotocolchain-2.md)                                            | Contains a counted list of Catalog Entry identifiers that comprise a protocol chain.                                                                                                                                                                                              |
| [**WSAQUERYSET**](wsaqueryset-2.md)                                                      | Provides relevant information about a given service.                                                                                                                                                                                                                              |
| [**WSAQUERYSET2**](wsaqueryset2.md)                                                      | Provides relevant information about a given service.                                                                                                                                                                                                                              |
| [**WSASERVICECLASSINFO**](wsaserviceclassinfo-2.md)                                      | Contains information about a specified service class.                                                                                                                                                                                                                             |
| [**WSAVERSION**](wsaversion-2.md)                                                        | Provides version comparison in Winsock.                                                                                                                                                                                                                                           |
