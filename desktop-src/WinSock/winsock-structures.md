# Winsock Structures

The following list provides concise descriptions of each Winsock structure and typedef. For additional information on any structure or typedef, click the structure or typedef name.



| Structure                                                                                 | Description                                                                                                                                                                                                                                                                       |
|-------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**addrinfo**](/windows/win32/Ws2tcpip/nf-ws2tcpip-freeaddrinfo?branch=master)                                                            | Used by the [**getaddrinfo**](/windows/win32/Ws2tcpip/nf-ws2tcpip-getaddrinfo?branch=master) function to hold host address information.                                                                                                                                                                                       |
| [**addrinfoW**](/windows/win32/Ws2def/ns-ws2def-addrinfow?branch=master)                                                            | Used by the [**GetAddrInfoW**](/windows/win32/Ws2tcpip/nf-ws2tcpip-getaddrinfow?branch=master) function to hold host address information.                                                                                                                                                                                       |
| [**addrinfoex**](/windows/win32/Ws2def/ns-ws2def-addrinfoexw?branch=master)                                                          | Used by the [**GetAddrInfoEx**](/windows/win32/Ws2tcpip/nf-ws2tcpip-getaddrinfoexa?branch=master) function to hold host address information.                                                                                                                                                                                     |
| [**addrinfoex2**](/windows/win32/Ws2def/ns-ws2def-addrinfoex2w?branch=master)                                                        | Used by the [**GetAddrInfoEx**](/windows/win32/Ws2tcpip/nf-ws2tcpip-getaddrinfoexa?branch=master) function to hold host address information when both a canonical name and a fully qualified domain name have been requested.                                                                                                    |
| [**addrinfoex3**](/windows/win32/Ws2def/ns-ws2def-addrinfoex3?branch=master)                                                        | Used by the [**GetAddrInfoEx**](/windows/win32/Ws2tcpip/nf-ws2tcpip-getaddrinfoexa?branch=master) function to hold host address information when a specific network interface has been requested.                                                                                                                                |
| [**addrinfoex4**](/windows/win32/Ws2def/ns-ws2def-addrinfoex4?branch=master)                                                        | Used by the [**GetAddrInfoEx**](/windows/win32/Ws2tcpip/nf-ws2tcpip-getaddrinfoexa?branch=master) function to hold host address information when a handle to the fully qualified domain name has been requested.                                                                                                                 |
| [**AFPROTOCOLS**](/windows/win32/Winsock2/ns-winsock2-_afprotocols?branch=master)                                                      | Supplies a list of protocols to which application programmers can constrain queries. Used for query purposes only.                                                                                                                                                                |

Derived from Binary Large Object.                                                                                                                                                                                                     |

| [**fd\_set**](/windows/win32/winsock/nf-winsock-fd_set?branch=master)                                                               | Used by various Winsock functions and service providers, such as [**select**](/windows/win32/Winsock2/nf-winsock2-select?branch=master), to place sockets into a "set" for various purposes, such as testing a given socket for readability.                                                                                  |
| [**GROUP\_FILTER**](/windows/win32/Ws2ipdef/ns-ws2ipdef-group_filter?branch=master)                                                     | Provides multicast filtering parameters for multicast IPv6 or IPv4 addresses.                                                                                                                                                                                                     |
| [**GROUP\_REQ**](/windows/win32/Ws2ipdef/ns-ws2ipdef-group_req?branch=master)                                                           | Provides multicast group information for IPv6 or IPv4 addresses.                                                                                                                                                                                                                  |
| [**GROUP\_SOURCE\_REQ**](/windows/win32/Ws2ipdef/ns-ws2ipdef-group_source_req?branch=master)                                            | Provides multicast group information for IPv6 or IPv4 addresses that includes the source IP address.                                                                                                                                                                              |
| [**hostent**](/windows/win32/winsock/ns-winsock-hostent?branch=master)                                                              | Stores information about a given host, such as host name, IP address, and so forth.                                                                                                                                                                                               |
| [**in\_addr**](/windows/win32/Inaddr/ns-winsock2-in_addr?branch=master)                                                             | Represents an IPv4 Internet address.                                                                                                                                                                                                                                              |
| [**in\_pktinfo**](/windows/win32/Ws2ipdef/ns-ws2ipdef-in_pktinfo?branch=master)                                                       | Stores received packet address information, and is used by Windows to return information about received packets.                                                                                                                                                                  |
| [**in6\_addr**](/windows/win32/In6addr/?branch=master)                                                             | Represents an IPv6 Internet address.                                                                                                                                                                                                                                              |
| [**in6\_pktinfo**](/windows/win32/Ws2ipdef/ns-ws2ipdef-in6_pktinfo?branch=master)                                                     | Stores received IPv6 packet address information, and is used by Windows to return information about received packets.                                                                                                                                                             |
| [**INET\_PORT\_RANGE**](/windows/win32/Mstcpip/ns-mstcpip-_inet_port_range?branch=master)                                              | Provides input data used by the the **SIO\_ACQUIRE\_PORT\_RESERVATION** Ioctl to acquire a runtime reservation for a block of TCP or UDP ports.                                                                                                                                   |
| [**INET\_PORT\_RESERVATION\_INSTANCE**](/windows/win32/Mstcpip/ns-mstcpip-inet_port_reservation_instance?branch=master)               | Contains a port reservation and a token for a block of TCP or UDP ports.                                                                                                                                                                                                          |
| [**INET\_PORT\_RESERVATION\_TOKEN**](/windows/win32/Mstcpip/ns-mstcpip-inet_port_reservation_token?branch=master)                     | Contains a port reservation token for a block of TCP or UDP ports.                                                                                                                                                                                                                |
| [**ip\_mreq**](/windows/win32/Ws2ipdef/ns-ws2ipdef-ip_mreq?branch=master)                                                               | Provides multicast group information for IPv4 addresses.                                                                                                                                                                                                                          |
| [**ip\_mreq\_source**](/windows/win32/Ws2ipdef/ns-ws2ipdef-ip_mreq_source?branch=master)                                                | Provides multicast group and source information for IPv4 addresses.                                                                                                                                                                                                               |
| [**ip\_msfilter**](/windows/win32/Ws2ipdef/ns-ws2ipdef-ip_msfilter?branch=master)                                                       | Provides multicast filtering parameters for IPv4 addresses.                                                                                                                                                                                                                       |
| [**ipv6\_mreq**](/windows/win32/Ws2ipdef/ns-ws2ipdef-ipv6_mreq?branch=master)                                                           | Provides multicast group information for IPv6 addresses.                                                                                                                                                                                                                          |
| [**linger**](/windows/win32/winsock/ns-winsock-linger?branch=master)                                                                | Maintains information about a specific socket that specifies how that socket should behave when data is queued to be sent and the [**closesocket**](/windows/win32/winsock/nf-winsock-closesocket?branch=master) function is called on the socket.                                                                          |
| [**NAPI\_DOMAIN\_DESCRIPTION\_BLOB**](/windows/win32/Nsemail/ns-nsemail-napi_domain_description_blob_tag?branch=master)                   | Describes a domain handled by a namespace provider for the NS\_EMAIL namespace.                                                                                                                                                                                                   |
| [**NAPI\_PROVIDER\_INSTALLATION\_BLOB**](/windows/win32/Nsemail/ns-nsemail-napi_provider_installation_blob_tag?branch=master)             | Contains the information required to install a namespace provider for the NS\_EMAIL namespace.                                                                                                                                                                                    |
| [**NS\_SERVICE\_INFO**](/windows/win32/Nspapi/ns-nspapi-_ns_service_infoa?branch=master)                                            | Contains information about a network service or a network service type in the context of a specified namespace, or a set of default namespaces.                                                                                                                                   |
| [**PROTOCOL\_INFO**](/windows/win32/Nspapi/ns-nspapi-_protocol_infoa?branch=master)                                                 | Contains information about a protocol.                                                                                                                                                                                                                                            |
| [**protoent**](/windows/win32/winsock/ns-winsock-protoent?branch=master)                                                            | Contains the name and protocol numbers that correspond to a given protocol name.                                                                                                                                                                                                  |

| [**REAL\_TIME\_NOTIFICATION\_SETTING\_OUTPUT**](/windows/win32/Mstcpip/ns-mstcpip-_real_time_notification_setting_input?branch=master) | Contains the output settings from a query for the **REAL\_TIME\_NOTIFICATION\_CAPABILITY** transport setting for a TCP socket that is used with [**ControlChannelTrigger**](w_net_sock.controlchanneltrigger) to receive background network notifications in a Windows Store app. |
| [**RIO\_EXTENSION\_FUNCTION\_TABLE**](/windows/win32/Mswsockdef/ns-mswsock-_rio_extension_function_table?branch=master)                   | Contains information on the functions that implement the Winsock registered I/O extensions.                                                                                                                                                                                       |
| [**RIO\_BUF**](/windows/win32/Mswsockdef/ns-mswsockdef-_rio_buf?branch=master)                                                               | Specifies a portion of a registered buffer used for sending or receiving network data with the Winsock registered I/O extensions.                                                                                                                                                 |
