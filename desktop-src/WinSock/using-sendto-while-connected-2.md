# Using Sendto While Connected

[**WSPSendTo**](/windows/win32/Ws2spi/?branch=master) will always deliver the data to the specified address, even though a designated peer for the sending socket has been established in [**WSPConnect**](/windows/win32/Ws2spi/?branch=master).
