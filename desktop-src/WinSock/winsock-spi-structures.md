# Winsock SPI Structures

The following list provides concise descriptions of each Winsock SPI structure. For additional information on any Winsock SPI structure, click the structure name.



| Winsock SPI structure                                         | Description                                                                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**NSP\_ROUTINE**](nsp-routine.md)                           | Contains information on the functions implemented by a namespace service provider version 1 (NSPv1) provider. |
| [**NSPV2\_ROUTINE**](nspv2-routine.md)                       | Contains information on the functions implemented by a namespace service provider version 2 (NSPv2) provider. |
| [**WSATHREADID**](wsathreadid-2.md)                          | Enables a provider to identify a thread on which asynchronous procedure calls (APCs) can be queued.           |
| [**WSC\_PROVIDER\_AUDIT\_INFO**](wsc-provider-audit-info.md) | Contains audit information for a layered service provider (LSP) entry.                                        |
| [**WSPDATA**](wspdata-2.md)                                  | Contains service provider information.                                                                        |
| [**WSPPROC\_TABLE**](wspproc-table.md)                       | Contains a table of pointers to service provider functions.                                                   |
| [**WSPUPCALLTABLE**](wspupcalltable.md)                      | Contains a table of pointers to service provider upcall functions.                                            |
