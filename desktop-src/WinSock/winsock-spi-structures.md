# Winsock SPI Structures

The following list provides concise descriptions of each Winsock SPI structure. For additional information on any Winsock SPI structure, click the structure name.



| Winsock SPI structure                                         | Description                                                                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**NSP\_ROUTINE**](/windows/win32/Ws2spi/ns-ws2spi-_nsp_routine?branch=master)                           | Contains information on the functions implemented by a namespace service provider version 1 (NSPv1) provider. |
| [**NSPV2\_ROUTINE**](/windows/win32/Ws2spi/ns-ws2spi-_nspv2_routine?branch=master)                       | Contains information on the functions implemented by a namespace service provider version 2 (NSPv2) provider. |
| [**WSATHREADID**](/windows/win32/Ws2spi/ns-ws2spi-_wsathreadid?branch=master)                          | Enables a provider to identify a thread on which asynchronous procedure calls (APCs) can be queued.           |
| [**WSC\_PROVIDER\_AUDIT\_INFO**](/windows/win32/Ws2spi/ns-ws2spi-_wsc_provider_audit_info?branch=master) | Contains audit information for a layered service provider (LSP) entry.                                        |
| [**WSPDATA**](/windows/win32/Ws2spi/ns-ws2spi-wspdata?branch=master)                                  | Contains service provider information.                                                                        |
| [**WSPPROC\_TABLE**](/windows/win32/Ws2spi/ns-ws2spi-_wspproc_table?branch=master)                       | Contains a table of pointers to service provider functions.                                                   |
| [**WSPUPCALLTABLE**](/windows/win32/Ws2spi/ns-ws2spi-_wspupcalltable?branch=master)                      | Contains a table of pointers to service provider upcall functions.                                            |
