# QoS Templates in the SPI

The Quality of Service (QoS) implementation details are provided in a separate, dedicated QoS section in the Microsoft Windows Software Development Kit (SDK). Please see the [Quality of Service](qos.qos_start_page) section of the Windows SDK for information on QoS usage guidelines, including available QoS templates and their associated [**FLOWSPEC**](qos.flowspec) values.
