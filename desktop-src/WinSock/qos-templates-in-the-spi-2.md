# QoS Templates in the SPI

The Quality of Service (QoS) implementation details are provided in a separate, dedicated QoS section in the Microsoft Windows Software Development Kit (SDK). Please see the [Quality of Service](https://msdn.microsoft.com/2a017166-eba4-4dd9-b623-f0d696e11de2) section of the Windows SDK for information on QoS usage guidelines, including available QoS templates and their associated [**FLOWSPEC**](https://msdn.microsoft.com/268e0d3a-2b04-40fd-91eb-f1780236b3e4) values.
