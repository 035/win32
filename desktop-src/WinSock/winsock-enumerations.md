# Winsock Enumerations

The following list provides concise descriptions of each Winsock enumeration. For additional information on any enumeration, click the enumeration name.



| Enumeration                                                                   | Description                                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CONTROL\_CHANNEL\_TRIGGER\_STATUS**](rio-notification-completion-type.md) | Specifies the status from a query for the **REAL\_TIME\_NOTIFICATION\_CAPABILITY** transport setting for a TCP socket that is used with [**ControlChannelTrigger**](w_net_sock.controlchanneltrigger) to receive background network notifications in a Windows Store app. |
| [**eWINDOW\_ADVANCE\_METHOD**](ewindow-advance-method.md)                    | Specifies the window advance mode used for Reliable Multicast.                                                                                                                                                                                                            |
| [**GUARANTEE**](guarantee-2.md)                                              | No longer used.                                                                                                                                                                                                                                                           |
| [**MULTICAST\_MODE\_TYPE**](multicast-mode-type.md)                          | Specifies the filter mode for multicast group addresses.                                                                                                                                                                                                                  |
| [**NAPI\_PROVIDER\_LEVEL**](napi-provider-level.md)                          | Specifies the provider authority level of a NS\_EMAIL namespace provider for a given domain.                                                                                                                                                                              |
| [**NAPI\_PROVIDER\_TYPE**](napi-provider-type.md)                            | Specifies the type of hosting expected for a namespace provider.                                                                                                                                                                                                          |
| [**SOCKET\_SECURITY\_PROTOCOL**](socket-security-protocol.md)                | Indicates the type of security protocol to be used on a socket.                                                                                                                                                                                                           |
| [**SOCKET\_USAGE\_TYPE**](socket-usage-type.md)                              | Used to specified the usage type for the socket.                                                                                                                                                                                                                          |
| [**WSAECOMPARATOR**](wsaecomparator-2.md)                                    | Used for version-comparison semantics.                                                                                                                                                                                                                                    |
| [**WSC\_PROVIDER\_INFO\_TYPE**](wsc-provider-info-type.md)                   | Specifies the information class of a layered service protocol (LSP).                                                                                                                                                                                                      |
