# DBasicIMEvents::OnLogoff event

\[**OnLogoff** is no longer available for use as of Windows Vista. See [Windows Messenger](im-messenger-entry.md) for more information.\]

Indicates that the logoff is complete.

> [!Note]  
> The **OnLogoff** event is available for use in Windows Messenger 4.7. It might be altered or unavailable in subsequent versions of Windows Messenger. You should use [**OnSignout**](im-dmessengerevents-onsignout.md) instead.

 

## Syntax


```C++
void OnLogoff();
```



## Parameters

This event has no parameters.

## Return value

This event does not return a value.
