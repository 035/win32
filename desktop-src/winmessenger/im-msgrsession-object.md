# MsgrSession object

\[**MsgrSession** is no longer available for use as of Windows Vista. See [Windows Messenger](im-messenger-entry.md) for more information.\]

The Messenger client session object.

-   [Interfaces](#interfaces)

### Interfaces

The **MsgrSession** object defines these interfaces.



| Interface                                           | Description                                                                                                                   |
|:----------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------|
| [**DMsgrSessionEvents**](im-dmsgrsessionevents.md) | Fires events in the Messenger client when notifications from [**IMsgrSession**](im-imsgrsession.md) are received.<br/> |
| [**IMsgrLock**](im-imsgrlock.md)                   | Enables an application to unlock Licensed Messenger objects.<br/>                                                       |
| [**IMsgrSession**](im-imsgrsession.md)             |                                                                                                                               |
