# Parental Controls Minimum Compliance API

The minimum compliance API implements simple interface methods for applications to obtain basic state information on a given parentally controlled user account, its web restrictions settings, and games restrictions settings.

-   [**IWindowsParentalControls**](/windows/win32/Wpcapi/nn-wpcapi-iwindowsparentalcontrols?branch=master)
-   [**IWPCGamesSettings**](/windows/win32/Wpcapi/nn-wpcapi-iwpcgamessettings?branch=master)
-   [**IWPCSettings**](/windows/win32/Wpcapi/nn-wpcapi-iwpcsettings?branch=master)
-   [**IWPCWebSettings**](/windows/win32/Wpcapi/nn-wpcapi-iwpcwebsettings?branch=master)
