# Parental Controls Logging Enumerations

Parental Controls provides the following logging enumerations.

Flag enumerations:

-   [**WPCFLAG\_IM\_FEATURE**](/windows/win32/Wpcevent/ne-wpcevent-tagwpcflag_im_feature?branch=master)
-   [**WPCFLAG\_IM\_LEAVE\_FLAG**](/windows/win32/Wpcevent/ne-wpcevent-tagwpcflag_im_leave_flag?branch=master)
-   [**WPCFLAG\_ISBLOCKED**](/windows/win32/Wpcevent/ne-wpcevent-tagwpcflag_isblocked?branch=master)
-   [**WPCFLAG\_LOGOFF\_TYPE**](/windows/win32/Wpcevent/ne-wpcevent-tagwpcflag_logoff_type?branch=master)

Enumerations that are defined in Wpcevent.h for parameters to be passed to the Windows Event Log publishing API for each event type:

-   [**WPC\_ARGS\_CONVERSATIONINITEVENT**](/windows/win32/Wpcevent/ne-wpcevent-tagwpc_args_conversationinitevent?branch=master)
