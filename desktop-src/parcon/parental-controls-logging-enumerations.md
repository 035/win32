# Parental Controls Logging Enumerations

Parental Controls provides the following logging enumerations.

Flag enumerations:

-   [**WPCFLAG\_IM\_FEATURE**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpcflag_im_feature)
-   [**WPCFLAG\_IM\_LEAVE\_FLAG**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpcflag_im_leave_flag)
-   [**WPCFLAG\_ISBLOCKED**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpcflag_isblocked)
-   [**WPCFLAG\_LOGOFF\_TYPE**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpcflag_logoff_type)

Enumerations that are defined in Wpcevent.h for parameters to be passed to the Windows Event Log publishing API for each event type:

-   [**WPC\_ARGS\_CONVERSATIONINITEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_conversationinitevent)
-   [**WPC\_ARGS\_CONVERSATIONJOINEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_conversationjoinevent)

-   [**WPC\_ARGS\_CUSTOMEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_customevent)
-   [**WPC\_ARGS\_EMAILCONTACTEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_emailcontactevent)
-   [**WPC\_ARGS\_EMAILRECEIVEDEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_emailreceievedevent)
-   [**WPC\_ARGS\_EMAILSENTEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_emailsentevent)
-   [**WPC\_ARGS\_FILEDOWNLOADEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_filedownloadevent)
-   [**WPC\_ARGS\_GAMESTARTEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_gamestartevent)
-   [**WPC\_ARGS\_IMCONTACTEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_imcontactevent)
-   [**WPC\_ARGS\_IMFEATUREEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_imfeatureevent)
-   [**WPC\_ARGS\_MEDIADOWNLOADEVENT**](/windows/desktop/api/Wpcevent/ne-wpcevent-tagwpc_args_mediadownloadevent)
