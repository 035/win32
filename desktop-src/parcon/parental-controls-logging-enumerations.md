# Parental Controls Logging Enumerations

Parental Controls provides the following logging enumerations.

Flag enumerations:

-   [**WPCFLAG\_IM\_FEATURE**](wpcflag-im-feature.md)
-   [**WPCFLAG\_IM\_LEAVE\_FLAG**](wpcflag-im-leave-flag.md)
-   [**WPCFLAG\_ISBLOCKED**](wpcflag-isblocked.md)
-   [**WPCFLAG\_LOGOFF\_TYPE**](wpcflag-logoff-type.md)

Enumerations that are defined in Wpcevent.h for parameters to be passed to the Windows Event Log publishing API for each event type:

-   [**WPC\_ARGS\_CONVERSATIONINITEVENT**](wpc-args-conversationinitevent.md)
-   [**WPC\_ARGS\_CONVERSATIONJOINEVENT**](wpc-args-conversationjoinevent.md)
-   [**WPC\_ARGS\_CONVERSATIONLEAVEEVENT**](wpc-args-conversationleaveevent.md)
-   [**WPC\_ARGS\_CUSTOMEVENT**](wpc-args-customevent.md)
-   [**WPC\_ARGS\_EMAILCONTACTEVENT**](wpc-args-emailcontactevent.md)
-   [**WPC\_ARGS\_EMAILRECEIVEDEVENT**](wpc-args-emailreceivedevent.md)
-   [**WPC\_ARGS\_EMAILSENTEVENT**](wpc-args-emailsentevent.md)
-   [**WPC\_ARGS\_FILEDOWNLOADEVENT**](wpc-args-filedownloadevent.md)
-   [**WPC\_ARGS\_GAMESTARTEVENT**](wpc-args-gamestartevent.md)
-   [**WPC\_ARGS\_IMCONTACTEVENT**](wpc-args-imcontactevent.md)
-   [**WPC\_ARGS\_IMFEATUREEVENT**](wpc-args-imfeatureevent.md)
-   [**WPC\_ARGS\_MEDIADOWNLOADEVENT**](wpc-args-mediadownloadevent.md)
-   [**WPC\_ARGS\_MEDIAPLAYBACKEVENT**](wpc-args-mediaplaybackevent.md)
-   [**WPC\_ARGS\_SAFERAPPBLOCKED**](wpc-args-saferappblocked.md)
-   [**WPC\_ARGS\_SETTINGSCHANGEEVENT**](wpc-args-settingschangeevent.md)
-   [**WPC\_ARGS\_URLVISITEVENT**](wpc-args-urlvisitevent.md)
