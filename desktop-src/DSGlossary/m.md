# M

<dl> <dt>

<span id="_ds_microsoft_management_console"></span><span id="_DS_MICROSOFT_MANAGEMENT_CONSOLE"></span>**Microsoft Management Console**
</dt> <dd>

The Microsoft Management Console is the Windows 2000 user interface for managing networks.

</dd> <dt>

<span id="_ds_mixed_mode"></span><span id="_DS_MIXED_MODE"></span>**mixed mode**
</dt> <dd>

In addition to Windows 2000 or Windows Server 2003 domain controllers, the domain may have one or more Windows NT 4.0 backup domain controllers present. Nested groups are not supported in mixed mode. See also [*native mode*](n.md#-ds-native-mode).

</dd> <dt>

<span id="_ds_mmc"></span><span id="_DS_MMC"></span>**MMC**
</dt> <dd>

See *Microsoft Management Console*.

</dd> </dl>
