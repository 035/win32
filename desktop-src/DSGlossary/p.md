# P

<dl> <dt>

<span id="_ds_partial_update"></span><span id="_DS_PARTIAL_UPDATE"></span>**partial update**
</dt> <dd>

A *partial update* can occur while replication is in progress if an application reads an object during a brief interval when some, but not all, of an update has been replicated to the domain controller from which the application is reading.

</dd> <dt>

<span id="_ds_path"></span><span id="_DS_PATH"></span>**path**
</dt> <dd>

Specifies the location of an object in a hierarchical system.

</dd> <dt>

<span id="_ds_pdc"></span><span id="_DS_PDC"></span>**PDC**
</dt> <dd>

See *primary domain controller*.

</dd> <dt>

<span id="_ds_primary_domain_controller"></span><span id="_DS_PRIMARY_DOMAIN_CONTROLLER"></span>**primary domain controller**
</dt> <dd>

A computer running Windows NT Server 4.0 that authenticates logons and maintains the directory database for a Windows NT 4.0 domain. A primary domain controller is not used in a Windows 2000 domain. Instead, one domain controller is designated as the primary domain controller emulator for backward compatibility.

</dd> </dl>
