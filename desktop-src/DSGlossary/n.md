# N

<dl> <dt>

<span id="_ds_name_resolution"></span><span id="_DS_NAME_RESOLUTION"></span>**name resolution**
</dt> <dd>

The process of translating a name into some object or information that the name represents.

</dd> <dt>

<span id="_ds_namespace"></span><span id="_DS_NAMESPACE"></span>**namespace**
</dt> <dd>

A logical, hierarchical naming scheme for grouping related types. For example, DNS, NetBIOS, and LDAP.

</dd> <dt>

<span id="_ds_naming_context"></span><span id="_DS_NAMING_CONTEXT"></span>**naming context**
</dt> <dd>

A contiguous Active Directory® subtree that is replicated on one or more domain controllers in a forest. Also known as a [*directory partition*](https://www.bing.com/search?q=*directory partition*).

</dd> <dt>

<span id="_ds_native_mode"></span><span id="_DS_NATIVE_MODE"></span>**native mode**
</dt> <dd>

A domain where native mode has been enabled using the domain property page in the Active Directory Users and Computers MMC snap-in. Also see [*mixed mode*](https://www.bing.com/search?q=*mixed mode*).

Domains must be operating in native mode for nested groups to be supported.

</dd> <dt>

<span id="_ds_nc"></span><span id="_DS_NC"></span>**NC**
</dt> <dd>

See *naming context*.

</dd> <dt>

<span id="_ds_nds"></span><span id="_DS_NDS"></span>**NDS**
</dt> <dd>

NetWare Directory Services.

</dd> </dl>
