# U

<dl> <dt>

<span id="_ds_universal_group"></span><span id="_DS_UNIVERSAL_GROUP"></span>**universal group**
</dt> <dd>

A [*group*](g.md#-ds-group) that can contain members from any domain and can be granted permissions to resources in any domain in the current domain forest.

</dd> <dt>

<span id="_ds_update_sequence_number"></span><span id="_DS_UPDATE_SEQUENCE_NUMBER"></span>**update sequence number**
</dt> <dd>

A type of identifier that is used to control replication between domain controllers in Windows 2000 and Windows Server 2003.

</dd> <dt>

<span id="_ds_upn"></span><span id="_DS_UPN"></span>**UPN**
</dt> <dd>

See *user principal name*.

</dd> <dt>

<span id="_ds_user_principal_name"></span><span id="_DS_USER_PRINCIPAL_NAME"></span>**user principal name**
</dt> <dd>

A logon name type of a user on a Windows 2000 or Windows Server 2003 network. The user principal name consists of the user object name used in Active Directory, followed by the at symbol (@) and then, typically, the Domain Name System parent domain. For example, JeffSmith in the Fabrikam.com domain tree might have the user principal name jeffsmith@fabrikam.com.

</dd> <dt>

<span id="_ds_usn"></span><span id="_DS_USN"></span>**USN**
</dt> <dd>

See *update sequence number*.

</dd> </dl>
