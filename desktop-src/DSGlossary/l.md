# L

<dl> <dt>

<span id="_ds_latency"></span><span id="_DS_LATENCY"></span>**latency**
</dt> <dd>

The time delay that occurs when a data packet or signal is transmitted over a communications system.

</dd> <dt>

<span id="_ds_ldap"></span><span id="_DS_LDAP"></span>**LDAP**
</dt> <dd>

See *Lightweight Directory Access Protocol*.

</dd> <dt>

<span id="_ds_lightweight_directory_access_protocol"></span><span id="_DS_LIGHTWEIGHT_DIRECTORY_ACCESS_PROTOCOL"></span>**Lightweight Directory Access Protocol**
</dt> <dd>

A standard protocol for accessing data in a directory.

</dd> <dt>

<span id="_ds_local_group"></span><span id="_DS_LOCAL_GROUP"></span>**local group**
</dt> <dd>

A group that belongs to a single computer and does not appear in a directory.

</dd> </dl>
