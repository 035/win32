# A

<dl> <dt>

<span id="_ds_access_control_entry"></span><span id="_DS_ACCESS_CONTROL_ENTRY"></span>**access-control entry**
</dt> <dd>

An entry, in an access-control list, that contains a set of access rights and a [*security identifier*](s.md#-ds-security-identifier), that identifies a trustee for whom rights are granted, denied, or audited.

</dd> <dt>

<span id="_ds_access_control_list"></span><span id="_DS_ACCESS_CONTROL_LIST"></span>**access-control list**
</dt> <dd>

A list of access-control entries that define permissions for an object.

</dd> <dt>

<span id="_ds_ace"></span><span id="_DS_ACE"></span>**ACE**
</dt> <dd>

See *access-control entry*.

</dd> <dt>

<span id="_ds_acl"></span><span id="_DS_ACL"></span>**ACL**
</dt> <dd>

See *access-control list*.

</dd> <dt>

<span id="_ds_active_directory_service_interfaces"></span><span id="_DS_ACTIVE_DIRECTORY_SERVICE_INTERFACES"></span>**Active Directory Service Interfaces**
</dt> <dd>

A set of specifications for [*COM*](c.md#-ds-component-object-model) objects and interfaces used to perform common administrative operations, such as adding a new user or managing a print queue.

</dd> <dt>

<span id="_ds_adsi"></span><span id="_DS_ADSI"></span>**ADSI**
</dt> <dd>

See *Active Directory Service Interfaces*.

</dd> <dt>

<span id="_ds_adspath"></span><span id="_DS_ADSPATH"></span>**ADsPath**
</dt> <dd>

A string that uniquely identifies an object in a directory service, including the directory service provider.

</dd> <dt>

<span id="_ds_automation"></span><span id="_DS_AUTOMATION"></span>**Automation**
</dt> <dd>

A set of methods, accessed through the [**IDispatch**](ebbff4bc-36b2-4861-9efa-ffa45e013eb5) interface, used to manipulate objects within a scripting environment.

</dd> </dl>
