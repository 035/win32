# Structures

Available XInput Structures

## In this section



| Topic                                                                         | Description                                                                                                                                                                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XINPUT\_BATTERY\_INFORMATION**](xinput-battery-information.md)<br/> | Contains information on battery type and charge state.<br/>                                                                                                                                  |
| [**XINPUT\_CAPABILITIES**](xinput-capabilities.md)<br/>                | Describes the capabilities of a connected controller. The [**XInputGetCapabilities**](xinputgetcapabilities.md) function returns [**XINPUT\_CAPABILITIES**](xinput-capabilities.md). <br/> |
| [**XINPUT\_GAMEPAD**](xinput-gamepad.md)<br/>                          | Describes the current state of the Xbox 360 Controller.<br/>                                                                                                                                 |
| [**XINPUT\_KEYSTROKE**](xinput-keystroke.md)<br/>                      | Specifies keystroke data returned by [**XInputGetKeystroke**](xinputgetkeystroke.md).<br/>                                                                                                  |
| [**XINPUT\_STATE**](xinput-state.md)<br/>                              | Represents the state of a controller.<br/>                                                                                                                                                   |
| [**XINPUT\_VIBRATION**](xinput-vibration.md)<br/>                      | Specifies motor speed levels for the vibration function of a controller.<br/>                                                                                                                |
