# Structures

Available XInput Structures

## In this section



| Topic                                                                         | Description                                                                                                                                                                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XINPUT\_BATTERY\_INFORMATION**](/windows/win32/XInput/ns-xinput-_xinput_battery_information?branch=master)<br/> | Contains information on battery type and charge state.<br/>                                                                                                                                  |
| [**XINPUT\_CAPABILITIES**](/windows/win32/XInput/ns-xinput-_xinput_capabilities?branch=master)<br/>                | Describes the capabilities of a connected controller. The [**XInputGetCapabilities**](/windows/win32/XInput/nf-xinput-xinputgetcapabilities?branch=master) function returns [**XINPUT\_CAPABILITIES**](/windows/win32/XInput/ns-xinput-_xinput_capabilities?branch=master). <br/> |
| [**XINPUT\_GAMEPAD**](/windows/win32/XInput/ns-xinput-_xinput_gamepad?branch=master)<br/>                          | Describes the current state of the Xbox 360 Controller.<br/>                                                                                                                                 |
| [**XINPUT\_KEYSTROKE**](/windows/win32/XInput/ns-xinput-_xinput_keystroke?branch=master)<br/>                      | Specifies keystroke data returned by [**XInputGetKeystroke**](/windows/win32/XInput/nf-xinput-xinputgetkeystroke?branch=master).<br/>                                                                                                  |
| [**XINPUT\_STATE**](/windows/win32/XInput/ns-xinput-_xinput_state?branch=master)<br/>                              | Represents the state of a controller.<br/>                                                                                                                                                   |
| [**XINPUT\_VIBRATION**](/windows/win32/XInput/ns-xinput-_xinput_vibration?branch=master)<br/>                      | Specifies motor speed levels for the vibration function of a controller.<br/>                                                                                                                |
