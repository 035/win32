# Functions

Available XInput functions.

## In this section



| Topic                                                                                 | Description                                                                                                                                             |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XInputEnable**](/windows/win32/XInput/nf-xinput-xinputenable?branch=master)<br/>                                       | Sets the reporting state of XInput.<br/>                                                                                                          |
| [**XInputGetAudioDeviceIds**](/windows/win32/XInput/nf-xinput-xinputgetaudiodeviceids?branch=master)<br/>                 | Retrieves the sound rendering and sound capture audio device IDs that are associated with the headset connected to the specified controller.<br/> |
| [**XInputGetBatteryInformation**](/windows/win32/XInput/nf-xinput-xinputgetbatteryinformation?branch=master)<br/>         | Retrieves the battery type and charge status of a wireless controller.<br/>                                                                       |
| [**XInputGetCapabilities**](/windows/win32/XInput/nf-xinput-xinputgetcapabilities?branch=master)<br/>                     | Retrieves the capabilities and features of a connected controller.<br/>                                                                           |
| [**XInputGetDSoundAudioDeviceGuids**](/windows/win32/XInput/nf-xinput-xinputgetdsoundaudiodeviceguids?branch=master)<br/> | Gets the sound rendering and sound capture device GUIDs that are associated with the headset connected to the specified controller.<br/>          |
| [**XInputGetKeystroke**](/windows/win32/XInput/nf-xinput-xinputgetkeystroke?branch=master)<br/>                           | Retrieves a gamepad input event.<br/>                                                                                                             |
| [**XInputGetState**](/windows/win32/XInput/nf-xinput-xinputgetstate?branch=master)<br/>                                   | Retrieves the current state of the specified controller.<br/>                                                                                     |
| [**XInputSetState**](/windows/win32/XInput/nf-xinput-xinputsetstate?branch=master)<br/>                                   | Sends data to a connected controller. This function is used to activate the vibration function of a controller.<br/>                              |
