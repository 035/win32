# MIDI Output Data Types

Windows defines the following data types for MIDI output functions.



| Value                              | Meaning                                                                              |
|------------------------------------|--------------------------------------------------------------------------------------|
| **HMIDIOUT**                       | Handle of a MIDI output device.                                                      |
| [**MIDIHDR**](midihdr.md)         | Header for a block of MIDI system-exclusive or stream data.                          |
| [**MIDIOUTCAPS**](midioutcaps.md) | Structure used to inquire about the capabilities of a particular MIDI output device. |
