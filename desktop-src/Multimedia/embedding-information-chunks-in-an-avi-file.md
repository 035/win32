# Embedding Information Chunks in an AVI File

You can insert information chunks, such as text or custom data, in an AVI file by using the [**WM\_CAP\_FILE\_SET\_INFOCHUNK**](wm-cap-file-set-infochunk.md) message (or the [**capFileSetInfoChunk**](/windows/win32/Vfw/nf-vfw-capfilesetinfochunk?branch=master) macro). You can also use this message to clear information chunks from an AVI file.
