# Basic Capture Options

By modifying one or more of the capture parameters defined in the [**CAPTUREPARMS**](captureparms.md) structure, you can perform the following tasks:

-   Change the frame capture rate.
-   Specify keyboard or mouse control for ending a capture session.
-   Specify a duration for a capture session.
