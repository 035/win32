# Video Format

You can retrieve the structure that specifies the video format or the size of that structure by sending the [**WM\_CAP\_GET\_VIDEOFORMAT**](wm-cap-get-videoformat.md) message (or the [**capGetVideoFormat**](capgetvideoformat.md) and [**capGetVideoFormatSize**](capgetvideoformatsize.md) macros) to a capture window. You can set the format of captured video data by sending the [**WM\_CAP\_SET\_VIDEOFORMAT**](wm-cap-set-videoformat.md) message (or the [**capSetVideoFormat**](capsetvideoformat.md) macro) to a capture window.
