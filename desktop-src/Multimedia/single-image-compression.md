# Single-Image Compression

You can use the [**ICImageCompress**](/windows/win32/Vfw/nf-vfw-icimagecompress?branch=master) function to compress a single image. This function returns a handle of the compressed device-independent bitmap (DIB). The compressed DIB is packed using the CF\_DIB format.
