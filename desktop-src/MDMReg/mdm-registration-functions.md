# MDM Registration Functions

The following functions are used by MDM Registration.

## In this section

<dl> <dt>

[**DiscoverManagementService**](discovermanagementservice.md)
</dt> <dd>

Discovers the MDM service.

</dd> <dt>

[**DiscoverManagementServiceEx**](discovermanagementserviceex.md)
</dt> <dd>

Discovers the MDM service using a candidate server.

</dd> <dt>

[**GetDeviceRegistrationInfo**](getdeviceregistrationinfo.md)
</dt> <dd>

Retrieves the device registration information.

</dd> <dt>

[**GetManagementAppHyperlink**](getmanagementapphyperlink.md)
</dt> <dd>

Retrieves the management app hyperlink associated with the MDM service.

</dd> <dt>

[**IsDeviceRegisteredWithManagement**](isdeviceregisteredwithmanagement.md)
</dt> <dd>

Checks whether the device is registered with an MDM service.

</dd> <dt>

[**IsManagementRegistrationAllowed**](ismanagementregistrationallowed.md)
</dt> <dd>

Checks whether MDM registration is allowed by local policy.

</dd> <dt>

[**RegisterDeviceWithManagement**](registerdevicewithmanagement.md)
</dt> <dd>

Registers a device with a MDM service, using the [\[MS-MDE\]: Mobile Device Enrollment Protocol](5c841535-042e-489e-913c-9d783d741267).

</dd> <dt>

[**RegisterDeviceWithManagementUsingAADCredentials**](registerdevicewithmanagementusingaadcredentials.md)
</dt> <dd>

Registers a device with a MDM service, using Azure Active Directory (AAD) credentials.

</dd> <dt>

[**SetManagedExternally**](setmanagedexternally.md)
</dt> <dd>

Indicates to the MDM agent that the device is managed externally and is not to be registered with an MDM service.

</dd> <dt>

[**UnregisterDeviceWithManagement**](unregisterdevicewithmanagement.md)
</dt> <dd>

Unregisters a device with the MDM service

</dd> </dl>
