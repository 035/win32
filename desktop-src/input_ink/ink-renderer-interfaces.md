# Ink renderer interfaces

The topics contained in this section provide the reference specifications for [Ink renderer](ink-renderer.md) interfaces.

## In this section



| Topic                                                 | Description                                                                                                                                                                                                                                            |
|-------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IInkD2DRenderer**](/windows/desktop/api/inkrenderer/nn-inkrenderer-iinkd2drenderer)<br/> | An [**IInkD2DRenderer**](/windows/desktop/api/inkrenderer/nn-inkrenderer-iinkd2drenderer) object enables the rendering of ink strokes onto the designated Direct2D device context of a Universal Windows app, instead of the default [**InkCanvas**](https://www.bing.com/search?q=**InkCanvas**) control.<br/> |
