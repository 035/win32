# Ink renderer

The Ink renderer APIs enable the rendering of ink strokes onto the designated Direct2D device context of a Universal Windows app, instead of the default [**InkCanvas**](https://www.bing.com/search?q=**InkCanvas**) control.

Ink renderer is designed for use by Universal Windows app developers interested in customizing how ink strokes are rendered "dry", and of identical quality to the [**InkCanvas**](https://www.bing.com/search?q=**InkCanvas**) control.
