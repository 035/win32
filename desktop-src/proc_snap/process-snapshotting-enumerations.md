# Process Snapshotting Enumerations

The Process Snapshotting API defines the following enumerations.



| Structure                                                             | Description                                                                                                      |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**PSS\_CAPTURE\_FLAGS**](/previous-versions/windows/desktop/api/processsnapshot/ne-processsnapshot-pss_capture_flags)                      | Flags that specify what [**PssCaptureSnapshot**](/previous-versions/windows/desktop/api/processsnapshot/nf-processsnapshot-psscapturesnapshot) captures.<br/>                    |
| [**PSS\_DUPLICATE\_FLAGS**](/previous-versions/windows/desktop/api/processsnapshot/ne-processsnapshot-pss_duplicate_flags)                  | Duplication flags for use by [**PssDuplicateSnapshot**](/previous-versions/windows/desktop/api/processsnapshot/nf-processsnapshot-pssduplicatesnapshot).<br/>                    |
| [**PSS\_HANDLE\_FLAGS**](/previous-versions/windows/desktop/api/processsnapshot/ne-processsnapshot-pss_handle_flags)                        | Flags to specify what parts of a [**PSS\_HANDLE\_ENTRY**](/previous-versions/windows/desktop/api/processsnapshot/ns-processsnapshot-pss_handle_entry) structure are valid. <br/> |
| [**PSS\_OBJECT\_TYPE**](/previous-versions/windows/desktop/api/processsnapshot/ne-processsnapshot-pss_object_type)                          | Specifies the object type in a [**PSS\_HANDLE\_ENTRY**](/previous-versions/windows/desktop/api/processsnapshot/ns-processsnapshot-pss_handle_entry) structure.<br/>              |
| [**PSS\_PROCESS\_FLAGS**](/previous-versions/windows/desktop/api/processsnapshot/ne-processsnapshot-pss_process_flags)                      | Flags that describe a process.<br/>                                                                        |
| [**PSS\_QUERY\_INFORMATION\_CLASS**](/previous-versions/windows/desktop/api/processsnapshot/ne-processsnapshot-pss_query_information_class) | Specifies what information [**PssQuerySnapshot**](/previous-versions/windows/desktop/api/processsnapshot/nf-processsnapshot-pssquerysnapshot) function returns.<br/>             |
| [**PSS\_THREAD\_FLAGS**](/previous-versions/windows/desktop/api/processsnapshot/ne-processsnapshot-pss_thread_flags)                        | Flags that describe a thread.<br/>                                                                         |
| [**PSS\_WALK\_INFORMATION\_CLASS**](/previous-versions/windows/desktop/api/processsnapshot/ne-processsnapshot-pss_walk_information_class)   | Specifies what information the [**PssWalkSnapshot**](/previous-versions/windows/desktop/api/processsnapshot/nf-processsnapshot-psswalksnapshot) function returns.<br/>           |
