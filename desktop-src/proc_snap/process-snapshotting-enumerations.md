# Process Snapshotting Enumerations

The Process Snapshotting API defines the following enumerations.



| Structure                                                             | Description                                                                                                      |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**PSS\_CAPTURE\_FLAGS**](/windows/previous-versions/processsnapshot/ne-processsnapshot-pss_capture_flags?branch=master)                      | Flags that specify what [**PssCaptureSnapshot**](/windows/previous-versions/processsnapshot/nf-processsnapshot-psscapturesnapshot?branch=master) captures.<br/>                    |
| [**PSS\_DUPLICATE\_FLAGS**](/windows/previous-versions/processsnapshot/ne-processsnapshot-pss_duplicate_flags?branch=master)                  | Duplication flags for use by [**PssDuplicateSnapshot**](/windows/previous-versions/processsnapshot/nf-processsnapshot-pssduplicatesnapshot?branch=master).<br/>                    |
| [**PSS\_HANDLE\_FLAGS**](/windows/previous-versions/processsnapshot/ne-processsnapshot-pss_handle_flags?branch=master)                        | Flags to specify what parts of a [**PSS\_HANDLE\_ENTRY**](/windows/previous-versions/processsnapshot/ns-processsnapshot-pss_handle_entry?branch=master) structure are valid. <br/> |
| [**PSS\_OBJECT\_TYPE**](/windows/previous-versions/processsnapshot/ne-processsnapshot-pss_object_type?branch=master)                          | Specifies the object type in a [**PSS\_HANDLE\_ENTRY**](/windows/previous-versions/processsnapshot/ns-processsnapshot-pss_handle_entry?branch=master) structure.<br/>              |
| [**PSS\_PROCESS\_FLAGS**](/windows/previous-versions/processsnapshot/ne-processsnapshot-pss_process_flags?branch=master)                      | Flags that describe a process.<br/>                                                                        |
| [**PSS\_QUERY\_INFORMATION\_CLASS**](/windows/previous-versions/processsnapshot/ne-processsnapshot-pss_query_information_class?branch=master) | Specifies what information [**PssQuerySnapshot**](/windows/previous-versions/processsnapshot/nf-processsnapshot-pssquerysnapshot?branch=master) function returns.<br/>             |
| [**PSS\_THREAD\_FLAGS**](/windows/previous-versions/processsnapshot/ne-processsnapshot-pss_thread_flags?branch=master)                        | Flags that describe a thread.<br/>                                                                         |
| [**PSS\_WALK\_INFORMATION\_CLASS**](/windows/previous-versions/processsnapshot/ne-processsnapshot-pss_walk_information_class?branch=master)   | Specifies what information the [**PssWalkSnapshot**](/windows/previous-versions/processsnapshot/nf-processsnapshot-psswalksnapshot?branch=master) function returns.<br/>           |
