# Process Snapshotting Enumerations

The Process Snapshotting API defines the following enumerations.



| Structure                                                             | Description                                                                                                      |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**PSS\_CAPTURE\_FLAGS**](pss-capture-flags.md)                      | Flags that specify what [**PssCaptureSnapshot**](psscapturesnapshot.md) captures.<br/>                    |
| [**PSS\_DUPLICATE\_FLAGS**](pss-duplicate-flags.md)                  | Duplication flags for use by [**PssDuplicateSnapshot**](pssduplicatesnapshot.md).<br/>                    |
| [**PSS\_HANDLE\_FLAGS**](pss-handle-flags.md)                        | Flags to specify what parts of a [**PSS\_HANDLE\_ENTRY**](pss-handle-entry.md) structure are valid. <br/> |
| [**PSS\_OBJECT\_TYPE**](pss-object-type.md)                          | Specifies the object type in a [**PSS\_HANDLE\_ENTRY**](pss-handle-entry.md) structure.<br/>              |
| [**PSS\_PROCESS\_FLAGS**](pss-process-flags.md)                      | Flags that describe a process.<br/>                                                                        |
| [**PSS\_QUERY\_INFORMATION\_CLASS**](pss-query-information-class.md) | Specifies what information [**PssQuerySnapshot**](pssquerysnapshot.md) function returns.<br/>             |
| [**PSS\_THREAD\_FLAGS**](pss-thread-flags.md)                        | Flags that describe a thread.<br/>                                                                         |
| [**PSS\_WALK\_INFORMATION\_CLASS**](pss-walk-information-class.md)   | Specifies what information the [**PssWalkSnapshot**](psswalksnapshot.md) function returns.<br/>           |
