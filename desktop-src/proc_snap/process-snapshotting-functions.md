# Process Snapshotting Functions

The Process Snapshotting API defines the following functions.



| Function                                                             | Description                                                                                             |
|----------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**PssCaptureSnapshot**](psscapturesnapshot.md)                     | Captures a snapshot of a target process.<br/>                                                     |
| [**PssDuplicateSnapshot**](pssduplicatesnapshot.md)                 | Duplicates a snapshot handle from one process to another.<br/>                                    |
| [**PssFreeSnapshot**](pssfreesnapshot.md)                           | Frees a snapshot.<br/>                                                                            |
| [**PssQuerySnapshot**](pssquerysnapshot.md)                         | Queries the snapshot.<br/>                                                                        |
| [**PssWalkMarkerCreate**](psswalkmarkercreate.md)                   | Creates a walk marker.<br/>                                                                       |
| [**PssWalkMarkerFree**](psswalkmarkerfree.md)                       | Frees a walk marker created by [**PssWalkMarkerCreate**](psswalkmarkercreate.md). <br/>          |
| [**PssWalkMarkerGetPosition**](psswalkmarkergetposition.md)         | Returns the current position of a walk marker.<br/>                                               |
| [**PssWalkMarkerSeekToBeginning**](psswalkmarkerseektobeginning.md) | Rewinds a walk marker back to the beginning.<br/>                                                 |
| [**PssWalkMarkerSetPosition**](psswalkmarkersetposition.md)         | Sets the position of a walk marker.<br/>                                                          |
| [**PssWalkSnapshot**](psswalksnapshot.md)                           | Returns information from the current snapshot position, then advances to the next position. <br/> |
