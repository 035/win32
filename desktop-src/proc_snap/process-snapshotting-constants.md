# Process Snapshotting Constants

There is a constant, **PSS\_PERF\_RESOLUTION**, that sets the resolution of the performance counters.



| Constant/value                                                                                                                                                                                                                                | Description                                                                                                                              |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------|
| <span id="PSS_PERF_RESOLUTION"></span><span id="pss_perf_resolution"></span><dl> <dt>**PSS\_PERF\_RESOLUTION**</dt> <dt>1000000</dt> </dl> | The resolution in microseconds of the performance counters in [**PSS\_PERFORMANCE\_COUNTERS**](/windows/previous-versions/processsnapshot/ns-processsnapshot-pss_performance_counters?branch=master).<br/> |
