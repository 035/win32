# Interaction Context Structures

The topics in this section provide the reference specifications for [Interaction Context](interaction-context-portal.md) structures.

## In this section



| Topic                                                                                         | Description                                                                                                                                                                 |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CROSS\_SLIDE\_PARAMETER**](cross-slide-parameter.md)<br/>                           | Defines the cross-slide threshold and its distance threshold.<br/>                                                                                                    |
| [**INTERACTION\_ARGUMENTS\_CROSS\_SLIDE**](interaction-arguments-cross-slide.md)<br/>  | Defines the state of the cross-slide interaction.<br/>                                                                                                                |
| [**INTERACTION\_ARGUMENTS\_MANIPULATION**](interaction-arguments-manipulation.md)<br/> | Defines the state of a manipulation.<br/>                                                                                                                             |
| [**INTERACTION\_ARGUMENTS\_TAP**](interaction-arguments-tap.md)<br/>                   | Defines the state of the tap interaction.<br/>                                                                                                                        |
| [**INTERACTION\_CONTEXT\_CONFIGURATION**](interaction-context-configuration.md)<br/>   | Defines the configuration of an [Interaction Context](interaction-context-portal.md) object that enables, disables, or modifies the behavior of an interaction.<br/> |
| [**INTERACTION\_CONTEXT\_OUTPUT**](interaction-context-output.md)<br/>                 | Defines the output of the [Interaction Context](interaction-context-portal.md) object.<br/>                                                                          |
| [**MANIPULATION\_TRANSFORM**](manipulation-transform.md)<br/>                          | Defines the transformation data for a manipulation.<br/>                                                                                                              |
| [**MANIPULATION\_VELOCITY**](manipulation-velocity.md)<br/>                            | Defines the velocity data of a manipulation.<br/>                                                                                                                     |
