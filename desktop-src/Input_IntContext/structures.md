# Interaction Context Structures

The topics in this section provide the reference specifications for [Interaction Context](interaction-context-portal.md) structures.

## In this section



| Topic                                                                                         | Description                                                                                                                                                                 |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CROSS\_SLIDE\_PARAMETER**](/windows/previous-versions/interactioncontext/ns-interactioncontext-cross_slide_parameter?branch=master)<br/>                           | Defines the cross-slide threshold and its distance threshold.<br/>                                                                                                    |
| [**INTERACTION\_ARGUMENTS\_CROSS\_SLIDE**](/windows/previous-versions/interactioncontext/ns-interactioncontext-interaction_arguments_cross_slide?branch=master)<br/>  | Defines the state of the cross-slide interaction.<br/>                                                                                                                |
| [**INTERACTION\_ARGUMENTS\_MANIPULATION**](/windows/previous-versions/interactioncontext/ns-interactioncontext-interaction_arguments_manipulation?branch=master)<br/> | Defines the state of a manipulation.<br/>                                                                                                                             |
| [**INTERACTION\_ARGUMENTS\_TAP**](/windows/previous-versions/interactioncontext/ns-interactioncontext-interaction_arguments_tap?branch=master)<br/>                   | Defines the state of the tap interaction.<br/>                                                                                                                        |
| [**INTERACTION\_CONTEXT\_CONFIGURATION**](/windows/previous-versions/interactioncontext/ns-interactioncontext-interaction_context_configuration?branch=master)<br/>   | Defines the configuration of an [Interaction Context](interaction-context-portal.md) object that enables, disables, or modifies the behavior of an interaction.<br/> |
| [**INTERACTION\_CONTEXT\_OUTPUT**](/windows/previous-versions/interactioncontext/ns-interactioncontext-interaction_context_output?branch=master)<br/>                 | Defines the output of the [Interaction Context](interaction-context-portal.md) object.<br/>                                                                          |
| [**MANIPULATION\_TRANSFORM**](/windows/previous-versions/interactioncontext/ns-interactioncontext-manipulation_transform?branch=master)<br/>                          | Defines the transformation data for a manipulation.<br/>                                                                                                              |
| [**MANIPULATION\_VELOCITY**](/windows/previous-versions/interactioncontext/ns-interactioncontext-manipulation_velocity?branch=master)<br/>                            | Defines the velocity data of a manipulation.<br/>                                                                                                                     |
