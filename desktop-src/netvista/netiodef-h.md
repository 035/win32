# Netiodef.h

This section contains kernel mode network driver reference topics for the Netiodef.h header. This header is included in the Windows SDK as it is also shared with user mode networking applications.

The Netiodef.h header contains definitions and structures for the network I/O subsystem.

## In this section



| Topic                                            | Description                                                                               |
|--------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**NPI\_MODULEID**](npi-moduleid.md)<br/> | The NPI\_MODULEID structure defines the data for identifying a network module.<br/> |
