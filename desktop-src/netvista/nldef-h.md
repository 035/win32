# Nldef.h

This section contains kernel mode network driver reference topics for the Nldef.h header. This header is included in the Windows SDK as it is also shared with user mode networking applications.

The Nldef.h header contains definitions for the basic network layer.

## In this section



| Topic                                                                                   | Description                                                                                                                                            |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NL\_ADDRESS\_TYPE**](/windows/win32/nldef/ne-nldef-nl_address_type?branch=master)<br/>                                 | The NL\_ADDRESS\_TYPE enumeration type specifies the IP address type of the network layer.<br/>                                                  |
| [**NL\_DAD\_STATE**](/windows/win32/nldef/?branch=master)<br/>                                       | The NL\_DAD\_STATE enumeration type defines the duplicate address detection (DAD) state.<br/>                                                    |
| [**NL\_LINK\_LOCAL\_ADDRESS\_BEHAVIOR**](/windows/win32/nldef/ne-nldef-_nl_link_local_address_behavior?branch=master)<br/> | The NL\_LINK\_LOCAL\_ADDRESS\_BEHAVIOR enumeration type defines the link local address behavior.<br/>                                            |
| [**NL\_NEIGHBOR\_STATE**](/windows/win32/nldef/ne-nldef-_nl_neighbor_state?branch=master)<br/>                             | The NL\_NEIGHBOR\_STATE enumeration type defines the state of a network layer neighbor IP address, as described in RFC 2461, section 7.3.2.<br/> |
| [**NL\_PREFIX\_ORIGIN**](/windows/win32/nldef/?branch=master)<br/>                               | The NL\_PREFIX\_ORIGIN enumeration type defines the origin of the prefix or network part of the IP address.<br/>                                 |
| [**NL\_ROUTE\_ORIGIN**](/windows/win32/nldef/ne-nldef-_nl_route_origin?branch=master)<br/>                                 | The NL\_ROUTE\_ORIGIN enumeration type defines the origin of the IP route.<br/>                                                                  |
| [**NL\_ROUTE\_PROTOCOL**](/windows/win32/nldef/ne-nldef-nl_route_protocol?branch=master)<br/>                             | The NL\_ROUTE\_PROTOCOL enumeration type defines the routing mechanism that an IP route was added with, as described in RFC 4292.<br/>           |
| [**NL\_ROUTER\_DISCOVERY\_BEHAVIOR**](/windows/win32/nldef/ne-nldef-_nl_router_discovery_behavior?branch=master)<br/>      | The NL\_ROUTER\_DISCOVERY\_BEHAVIOR enumeration type defines the router discovery behavior, as described in RFC 2461.<br/>                       |
| [**NL\_SUFFIX\_ORIGIN**](/windows/win32/nldef/?branch=master)<br/>                               | The NL\_SUFFIX\_ORIGIN enumeration type defines the origin of the suffix or host part of the IP address.<br/>                                    |
