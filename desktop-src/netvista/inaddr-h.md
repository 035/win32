# Inaddr.h

This section contains kernel mode network driver reference topics for the Inaddr.h header. This header is included in the Windows SDK as it is also shared with user mode networking applications.

The Inaddr.h header contains the definition for an IPv4 Internet address structure.

## In this section



| Topic                                  | Description                                                            |
|----------------------------------------|------------------------------------------------------------------------|
| [**IN\_ADDR**](in-addr.md)<br/> | The IN\_ADDR structure specifies an IPv4 transport address.<br/> |
