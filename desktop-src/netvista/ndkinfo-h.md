# Ndkinfo.h

This section contains kernel mode network driver reference topics for the Ndkinfo.h header. This header is included in the Windows SDK as it is also shared with user mode networking applications.

The Ndkinfo.h header contains definitions for Network Direct adapter information.

## In this section



| Topic                                                     | Description                                                                                                                                    |
|-----------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NDK\_ADAPTER\_INFO**](ndk-adapter-info.md)<br/> | The [**NDK\_ADAPTER\_INFO**](ndk-adapter-info.md) structure specifies information about limits and capabilities of an NDK adapter.<br/> |
| [**NDK\_VERSION**](ndk-version.md)<br/>            | The [**NDK\_VERSION**](ndk-version.md) structure specifies major and minor versions in the NDK interface.<br/>                          |
