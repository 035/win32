# MSFT\_SMiSCSIPortalEndpoint class

Represents an endpoint of an iSCSI portal.

The following syntax is simplified from Managed Object Format (MOF) code and includes all of the inherited properties.

**Windows Server 2012 R2 and Windows Server 2012:** This class does not inherit from [**MSFT\_SMStorageObject**](msft-smstorageobject.md) which is new for Windows Server 2016.

## Syntax

``` syntax
[dynamic, provider("WMIStorage"), AMENDMENT]
class MSFT_SMiSCSIPortalEndpoint : MSFT_SMStorageObject
{
  String ObjectId;
  String Identifier;
  string IPv4Address;
  string IPv6Address;
  string SubnetMask;
  uint16 IPEndpointOpStatus[];
  uint32 PortNumber;
  uint16 TCPEndpointOpStatus[];
};
```
