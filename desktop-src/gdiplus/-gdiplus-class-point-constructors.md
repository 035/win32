# Point.Point constructors

This topic lists the constructors of the [**Point**](-gdiplus-class-point-class.md) class. For a complete class listing, see **Point Class**.

### Overload list



| Constructor                                                | Description                                                                                                                                                                      |
|:-----------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Point(Size&)**](-gdiplus-class-point-point-size-.md)   | Creates a [**Point**](-gdiplus-class-point-class.md) object using a [**Size**](-gdiplus-class-size-class.md) object to initialize the **X** and **Y** data members.<br/> |
| [**Point(Point&)**](-gdiplus-class-point-point-point-.md) | Creates a new [**Point**](-gdiplus-class-point-class.md) object and copies the data members from another **Point** object.<br/>                                           |
| [**Point(INT,INT)**](-gdiplus-class-point-point-x-y-.md)  | Creates a [**Point**](-gdiplus-class-point-class.md) object using two integers to initialize the **X** and **Y** data members.<br/>                                       |
| [**Point()**](-gdiplus-class-point-point-.md)             | Creates a [**Point**](-gdiplus-class-point-class.md) object and initializes the **X** and **Y** data members to zero. This is the default constructor.<br/>               |
