# Point.Point constructors

This topic lists the constructors of the [**Point**](/windows/win32/gdiplustypes/nl-gdiplustypes-point?branch=master) class. For a complete class listing, see **Point Class**.

### Overload list



| Constructor                                                | Description                                                                                                                                                                      |
|:-----------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Point(Size&)**](/windows/win32/Gdiplustypes/?branch=master)   | Creates a [**Point**](/windows/win32/gdiplustypes/nl-gdiplustypes-point?branch=master) object using a [**Size**](/windows/win32/gdiplustypes/nl-gdiplustypes-size?branch=master) object to initialize the **X** and **Y** data members.<br/> |
| [**Point(Point&)**](/windows/win32/Gdiplustypes/?branch=master) | Creates a new [**Point**](/windows/win32/gdiplustypes/nl-gdiplustypes-point?branch=master) object and copies the data members from another **Point** object.<br/>                                           |
| [**Point(INT,INT)**](/windows/win32/Gdiplustypes/?branch=master)  | Creates a [**Point**](/windows/win32/gdiplustypes/nl-gdiplustypes-point?branch=master) object using two integers to initialize the **X** and **Y** data members.<br/>                                       |
| [**Point()**](/windows/win32/Gdiplustypes/nf-gdiplustypes-point-point(in const point &)?branch=master)             | Creates a [**Point**](/windows/win32/gdiplustypes/nl-gdiplustypes-point?branch=master) object and initializes the **X** and **Y** data members to zero. This is the default constructor.<br/>               |
