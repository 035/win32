# Point.Point constructors

This topic lists the constructors of the [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) class. For a complete class listing, see **Point Class**.

### Overload list



| Constructor                                                | Description                                                                                                                                                                      |
|:-----------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Point(Size&)**](/windows/desktop/api/Gdiplustypes/)   | Creates a [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) object using a [**Size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) object to initialize the **X** and **Y** data members.<br/> |
| [**Point(Point&)**](/windows/desktop/api/Gdiplustypes/) | Creates a new [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) object and copies the data members from another **Point** object.<br/>                                           |
| [**Point(INT,INT)**](/windows/desktop/api/Gdiplustypes/)  | Creates a [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) object using two integers to initialize the **X** and **Y** data members.<br/>                                       |
| [**Point()**](/windows/desktop/api/Gdiplustypes/nf-gdiplustypes-point-point(in const point &))             | Creates a [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) object and initializes the **X** and **Y** data members to zero. This is the default constructor.<br/>               |
