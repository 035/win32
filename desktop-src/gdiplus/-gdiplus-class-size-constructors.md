# Size.Size constructors

This topic lists the constructors of the [**Size**](-gdiplus-class-size-class.md) class. For a complete class listing, see **Size Class**.

### Overload list



| Constructor                                                     | Description                                                                                                                                          |
|:----------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Size(Size&)**](-gdiplus-class-size-size-size-.md)           | Creates a [**Size**](-gdiplus-class-size-class.md) object and initializes its members by copying the members of another **Size** object.<br/> |
| [**Size(INT,INT)**](-gdiplus-class-size-size-width-height-.md) | Creates a [**Size**](-gdiplus-class-size-class.md) object and initializes its **Width** and **Height** data members.<br/>                     |
| [**Size()**](-gdiplus-class-size-size-.md)                     | Creates a new [**Size**](-gdiplus-class-size-class.md) object and initializes the members to zero. This is the default constructor.<br/>      |
