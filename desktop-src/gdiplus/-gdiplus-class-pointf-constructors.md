# PointF.PointF constructors

This topic lists the constructors of the [**PointF**](/windows/win32/gdiplustypes/nl-gdiplustypes-pointf?branch=master) class. For a complete class listing, see **PointF Class**.

### Overload list



| Constructor                                                    | Description                                                                                                                                                                       |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PointF(SizeF&)**](/windows/win32/Gdiplustypes/?branch=master)   | Creates a [**PointF**](/windows/win32/gdiplustypes/nl-gdiplustypes-pointf?branch=master) object using a [**SizeF**](/windows/win32/gdiplustypes/nl-gdiplustypes-sizef?branch=master) object to specify the **X** and **Y** data members.<br/> |
| [**PointF(PointF&)**](/windows/win32/Gdiplustypes/?branch=master) | Creates a new [**PointF**](/windows/win32/gdiplustypes/nl-gdiplustypes-pointf?branch=master) object and copies the data from another **PointF** object.<br/>                                                 |
| [**PointF(REAL,REAL)**](/windows/win32/Gdiplustypes/?branch=master) | Creates a [**PointF**](/windows/win32/gdiplustypes/nl-gdiplustypes-pointf?branch=master) object using two real numbers to specify the **X** and **Y** data members.<br/>                                     |
| [**PointF()**](/windows/win32/Gdiplustypes/nf-gdiplustypes-pointf-pointf(in const pointf &)?branch=master)              | Creates a [**PointF**](/windows/win32/gdiplustypes/nl-gdiplustypes-pointf?branch=master) object and initializes the **X** and **Y** data members to zero. This is the default constructor.<br/>              |
