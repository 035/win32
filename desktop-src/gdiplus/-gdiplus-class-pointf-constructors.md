# PointF.PointF constructors

This topic lists the constructors of the [**PointF**](-gdiplus-class-pointf-class.md) class. For a complete class listing, see **PointF Class**.

### Overload list



| Constructor                                                    | Description                                                                                                                                                                       |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PointF(SizeF&)**](-gdiplus-class-pointf-pointf-size-.md)   | Creates a [**PointF**](-gdiplus-class-pointf-class.md) object using a [**SizeF**](-gdiplus-class-sizef-class.md) object to specify the **X** and **Y** data members.<br/> |
| [**PointF(PointF&)**](-gdiplus-class-pointf-pointf-point-.md) | Creates a new [**PointF**](-gdiplus-class-pointf-class.md) object and copies the data from another **PointF** object.<br/>                                                 |
| [**PointF(REAL,REAL)**](-gdiplus-class-pointf-pointf-x-y-.md) | Creates a [**PointF**](-gdiplus-class-pointf-class.md) object using two real numbers to specify the **X** and **Y** data members.<br/>                                     |
| [**PointF()**](-gdiplus-class-pointf-pointf-.md)              | Creates a [**PointF**](-gdiplus-class-pointf-class.md) object and initializes the **X** and **Y** data members to zero. This is the default constructor.<br/>              |
