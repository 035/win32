# Region.Region constructors

This topic lists the constructors of the [**Region**](-gdiplus-class-region-class.md) class. For a complete class listing, see **Region Class**.

### Overload list



| Constructor                                                                 | Description                                                                                                                                                                                      |
|:----------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Region(HRGN)**](-gdiplus-class-region-region-hrgn-.md)                  | Creates a region that is identical to the region that is specified by a handle to a GDI region.<br/>                                                                                       |
| [**Region(Rect&)**](-gdiplus-class-region-region-rect-rect-.md)            | Creates a region that is defined by a rectangle.<br/>                                                                                                                                      |
| [**Region(RectF&)**](-gdiplus-class-region-region-rectf-rect-.md)          | Creates a region that is defined by a rectangle.<br/>                                                                                                                                      |
| [**Region(BYTE\*,INT)**](-gdiplus-class-region-region-regiondata-size-.md) | Creates a region that is defined by data obtained from another region. <br/>                                                                                                               |
| [**Region(GraphicsPath\*)**](-gdiplus-class-region-region-path-.md)        | Creates a region that is defined by a path (a [**GraphicsPath**](-gdiplus-class-graphicspath-class.md) object) and has a fill mode that is contained in the **GraphicsPath** object.<br/> |
| [**Region()**](-gdiplus-class-region-region-.md)                           | Creates a region that is infinite. This is the default constructor. <br/>                                                                                                                  |
