# Image Encoder Constants

The [Image::Save Methods](-gdiplus-class-image-save-methods.md) and [Image::SaveAdd Methods](-gdiplus-class-image-saveadd-methods.md) methods of the [**Image**](-gdiplus-class-image-class.md) class receive an [**EncoderParameters**](-gdiplus-class-encoderparameters-class.md) object that contains an array of [**EncoderParameter**](-gdiplus-class-encoderparameter-class.md) objects. Each **EncoderParameter** object has a GUID data member that specifies the parameter category. The following constants, defined in Gdiplusimaging.h, represent GUIDs that specify the various parameter categories.

-   EncoderChrominanceTable
-   EncoderColorDepth
-   EncoderColorSpace
-   EncoderCompression
-   EncoderLuminanceTable
-   EncoderQuality
-   EncoderRenderMethod
-   EncoderSaveFlag
-   EncoderScanMethod
-   EncoderTransformation
-   EncoderVersion
-   EncoderImageItems
-   EncoderSaveAsCMYK
