# Rect.Rect constructors

This topic lists the constructors of the [**Rect**](/windows/win32/gdiplustypes/nl-gdiplustypes-rect?branch=master) class. For a complete class listing, see **Rect Class**.

### Overload list



| Constructor                                                                 | Description                                                                                                                                                                                                                                                                                        |
|:----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Rect(Point&,Size&)**](/windows/win32/Gdiplustypes/?branch=master)       | Creates a [**Rect**](/windows/win32/gdiplustypes/nl-gdiplustypes-rect?branch=master) object by using a [**Point**](/windows/win32/gdiplustypes/nl-gdiplustypes-point?branch=master) object to initialize the **X** and **Y** data members and a [**Size**](/windows/win32/gdiplustypes/nl-gdiplustypes-size?branch=master) object to initialize the **Width** and **Height** data members.<br/> |
| [**Rect(INT,INT,INT,INT)**](/windows/win32/Gdiplustypes/?branch=master) | Creates a [**Rect**](/windows/win32/gdiplustypes/nl-gdiplustypes-rect?branch=master) object by using four integers to initialize the **X**, **Y**, **Width**, and **Height** data members.<br/>                                                                                                                               |
| [**Rect()**](/windows/win32/Gdiplustypes/nf-gdiplustypes-rect-rect(in const point &,in const size &)?branch=master)                                 | Creates a [**Rect**](/windows/win32/gdiplustypes/nl-gdiplustypes-rect?branch=master) object whose x-coordinate, y-coordinate, width, and height are all zero. This is the default constructor. <br/>                                                                                                                          |
