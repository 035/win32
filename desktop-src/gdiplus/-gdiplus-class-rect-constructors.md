# Rect.Rect constructors

This topic lists the constructors of the [**Rect**](-gdiplus-class-rect-class.md) class. For a complete class listing, see **Rect Class**.

### Overload list



| Constructor                                                                 | Description                                                                                                                                                                                                                                                                                        |
|:----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Rect(Point&,Size&)**](-gdiplus-class-rect-rect-location-size-.md)       | Creates a [**Rect**](-gdiplus-class-rect-class.md) object by using a [**Point**](-gdiplus-class-point-class.md) object to initialize the **X** and **Y** data members and a [**Size**](-gdiplus-class-size-class.md) object to initialize the **Width** and **Height** data members.<br/> |
| [**Rect(INT,INT,INT,INT)**](-gdiplus-class-rect-rect-x-y-width-height-.md) | Creates a [**Rect**](-gdiplus-class-rect-class.md) object by using four integers to initialize the **X**, **Y**, **Width**, and **Height** data members.<br/>                                                                                                                               |
| [**Rect()**](-gdiplus-class-rect-rect-.md)                                 | Creates a [**Rect**](-gdiplus-class-rect-class.md) object whose x-coordinate, y-coordinate, width, and height are all zero. This is the default constructor. <br/>                                                                                                                          |
