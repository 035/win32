# Matrix.Matrix constructors

This topic lists the constructors of the [**Matrix**](/windows/desktop/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) class. For a complete class listing, see **Matrix Class**.

### Overload list



| Constructor                                                                                          | Description                                                                                                                                                                          |
|:-----------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix(Rect&,Point\*)**](/windows/desktop/api/Gdiplusmatrix/)                | Creates a [**Matrix::Matrix**](/windows/desktop/api/Gdiplusmatrix/) object based on a rectangle and a point.<br/>                                         |
| [**Matrix(RectF&,PointF\*)**](/windows/desktop/api/Gdiplusmatrix/)            | Creates a [**Matrix::Matrix**](/windows/desktop/api/Gdiplusmatrix/) object based on a rectangle and a point.<br/>                                       |
| [**Matrix(REAL,REAL,REAL,REAL,REAL,REAL)**](/windows/desktop/api/Gdiplusmatrix/) | Creates and initializes a [**Matrix::Matrix**](/windows/desktop/api/Gdiplusmatrix/) object based on six numbers that define an affine transformation.<br/> |
| [**Matrix()**](/windows/desktop/api/Gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(const matrix &))                                                    | Creates and initializes a [**Matrix::Matrix**](/windows/desktop/api/Gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(const matrix &)) object that represents the identity matrix.<br/>                                             |
