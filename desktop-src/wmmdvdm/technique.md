# Technique element

Specifies the effect technique that the specified button will use.

## Usage

``` syntax
<Technique/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                   |
|-------------------------------------------|
| [**DVDButton**](dvdbutton.md)<br/> |



## Remarks

The available techniques are defined by an .fx file such as the one included with this SDK, in [Samples](samples.md).

This element is required within each **DVDButton** element.

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
