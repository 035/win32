# StaticThumbnail element

Specifies an image that represents the button or menu style (depending on the context) in the Windows DVD Maker user interface (UI).

## Usage

``` syntax
<StaticThumbnail/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                         |
|-------------------------------------------------|
| [**DVDButton**](dvdbutton.md)<br/>       |
| [**DVDMenuStyle**](dvdmenustyle.md)<br/> |



## Remarks

This element is required within each **DVDButton** element.

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
