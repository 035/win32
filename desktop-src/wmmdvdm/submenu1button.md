# Submenu1Button element

Specifies the location and size of the "Submenu1" button. This is usually the "Scenes" button.

## Usage

``` syntax
<Submenu1Button/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                       |
|---------------------------------------------------------------|
| [**ButtonLocations**](ButtonLocationsElement_ACRC)<br/> |



## Remarks

This button can be used only on the main menu.

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
