# Enumeration Types

Transforms can use the following enumerations.



| Enumeration                                              | Description                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [**EVALUATION\_CURVE\_TYPE**](evaluation-curve-type.md) | Describes how a property should change over time.              |
| [**FORMAT\_TYPE**](format-type.md)                      | Describes the format of a buffer and how it should be treated. |
| [MEDIA\_FORMAT\_TYPE](media-format-type.md)             | Describes the media type handled by a transform.               |
