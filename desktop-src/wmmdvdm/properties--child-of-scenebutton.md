# Properties (Child of SceneButton) element

Specifies the properties of the scene button.

## Usage

``` syntax
<Properties/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                       |
|-----------------------------------------------|
| [**SceneButton**](scenebutton.md)<br/> |



## Remarks

The [**Alpha**](alpha-element.md), [Rotate](rotate.md), [Offset](offset.md), and [Scale](scale.md) child elements are required for the **Properties** element when it is a child of the **SceneButton** element. The [**ButtonSelection**](buttonselection.md) element is optional in this case.

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
