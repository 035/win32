# DVDMenuStyleDLL element

The boundary tag for a single DVD menu style object.

## Usage

``` syntax
<DVDMenuStyleDLL/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                           |
|---------------------------------------------------|
| [**DVDMenuStyles**](dvdmenustyles.md)<br/> |



## Remarks

This object can hold one or more DVD menu styles.
