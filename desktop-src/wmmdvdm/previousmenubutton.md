# PreviousMenuButton element

Specifies the location and size of the "Previous Menu" button.

## Usage

``` syntax
<PreviousMenuButton/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                       |
|---------------------------------------------------------------|
| [**ButtonLocations**](https://www.bing.com/search?q=**ButtonLocations**)<br/> |



## Remarks

When available, the "Previous Menu" button sends users from the current scenes menu to the previous scenes menu in the current menu level. This button can be used only in the [ScenesMenu2](scenesmenu2.md), [ScenesMenu4](scenesmenu4.md), and [ScenesMenu6](scenesmenu6.md) menus.

|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
