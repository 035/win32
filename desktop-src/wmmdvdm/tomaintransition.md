# ToMainTransition element

Specifies the transition into the main menu (the "opening" transition) that is displayed when the main menu is displayed for the first time.

## Usage

``` syntax
<ToMainTransition/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
