# ISurface interface

The **ISurface** interface wraps an **IDirect3DSurface9** interface that represents a video surface.

## Members

The **ISurface** interface inherits from [**IBuffer**](ibuffer.md). **ISurface** also has these types of members:

-   [Methods](#methods)

### Methods

The **ISurface** interface has these methods.



| Method                                           | Description                                                                               |
|:-------------------------------------------------|:------------------------------------------------------------------------------------------|
| [**get\_Surface**](isurface-get-surface.md)     | Retrieves an **IDirect3DSurface9** interface that points to the video surface.<br/> |
| [**get\_VideoSize**](isurface-get-videosize.md) | Retrieves the size of the video surface.<br/>                                       |
