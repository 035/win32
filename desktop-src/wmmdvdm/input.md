# Input element

Creates a source (media node) in the pipeline graph.

## Usage

``` syntax
<Input/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                              |
|--------------------------------------|
| [**Graph**](GraphElement)<br/> |
| [**Token**](TokenElement)<br/> |



## Remarks

User-imported content is available for use in menu styles by using the "Background" and "InsetVideo" sources. These sources consist of five seconds of user content, followed by a half-second crossfade into another five seconds of user content, and so on. The user can override these sources from within the Windows DVD Maker user interface (UI).
