# Windows Movie Maker APIs

The Microsoft Windows Movie Maker 6.0 SDK supports a range of interfaces, functions, classes, enumeration types, and structures. You must include the files provided in [Samples](samples.md) to implement these APIs. The following sections document these APIs in detail.



| Section                                                                          | Description                                                                                                   |
|----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [Transform Interfaces](transform-interfaces.md)                                 | The main Windows Movie Maker interfaces.                                                                      |
| [Transform Helper Functions](transform-helper-functions.md)                     | Helper functions that are used to get Direct3D surfaces and textures for transforms to read from or write to. |
| [Property Management Interfaces](property-management-interfaces.md)             | Interfaces for managing properties.                                                                           |
| [Property Management Helper Functions](property-management-helper-functions.md) | Helper functions that are used to create new property objects and collections.                                |
| [Classes](classes.md)                                                           | Classes defined by Windows Movie Maker for use by transforms.                                                 |
| [Enumeration Types](enumeration-types.md)                                       | Enumerations defined by Windows Movie Maker for use by transforms.                                            |
| [Structures](structures.md)                                                     | Structures defined by Windows Movie Maker for use by transforms.                                              |
