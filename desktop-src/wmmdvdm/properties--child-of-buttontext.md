# Properties (Child of ButtonText) element

Specifies the properties of the button text media node.

## Usage

``` syntax
<Properties/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                     |
|---------------------------------------------|
| [**ButtonText**](buttontext.md)<br/> |



## Remarks

The [**ExitDuration**](exitduration.md), [**ExitEffect**](exiteffect.md), and [**ExitPosition**](exitposition.md) elements are optional.

|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
