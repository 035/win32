# Windows DVD Maker XML Extensibility

This topic provides one sample transition, one button, and one menu style for Windows DVD Maker, along with reference documentation. You can use the XML code that is provided in this topic as a template to help you create your own transitions, buttons, and menu styles for Windows DVD Maker.

**Important** XML file parsing in Windows DVD Maker is case sensitive.

This topic contains the following sections.



| Topic                                                                      | Description                                                                                                                         |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [Sample Windows DVD Maker XML File](sample-windows-dvd-maker-xml-file.md) | A valid XML file (with all elements linked to reference topics) that defines one sample transition, one button, and one menu style. |
| [Elements](elements.md)                                                   | A complete list of the XML elements used to create custom transitions, buttons, and menu styles with Windows DVD Maker.             |
