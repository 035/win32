# Programming Guide

This section of the Windows Movie Maker and Windows DVD Maker SDK provides "how-to" information for extending both Windows Movie Maker 6.0 and Windows DVD Maker 1.0 in Windows Vista.

This section includes the following topics.



| Topic                                                                                                                                                                                | Description                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Creating Custom Transforms Using XML](creating-custom-transforms-using-xml.md)                                                                                                     | Explains how to modify existing Windows Movie Maker and Windows DVD Maker effects and transitions by using XML.                                                      |
| [Creating Custom Effects with XML and DirectX Pixel Shaders](creating-custom-effects-with-xml-and-directx-pixel-shaders.md)                                                         | Explains how to create new effects using both XML and DirectX pixel shaders.                                                                                         |
| [Creating Custom Transforms Using COM and DirectX](creating-custom-transforms-using-com-and-directx.md)                                                                             | Explains how to create new transforms by using COM and DirectX.                                                                                                      |
| [Adding Custom Icons to Windows Movie Maker](adding-custom-icons-to-windows-movie-maker.md)                                                                                         | Explains how to create your own custom icons to represent your transforms in the Windows Movie Maker user interface.                                                 |
| [Getting Information from Windows Vista About Windows Movie Maker and Windows DVD Maker](getting-information-from-windows-vista-about-windows-movie-maker-and-windows-dvd-maker.md) | Explains how to use specific licensing APIs, system APIs and registry entries to get information from Windows Vista about Windows Movie Maker and Windows DVD Maker. |
