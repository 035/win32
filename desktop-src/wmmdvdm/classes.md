Transforms use the following classes that are defined by Windows Movie Maker.



| Class                               | Description                                                                           |
|-------------------------------------|---------------------------------------------------------------------------------------|
| [PIPELINE\_TIME](pipeline-time.md) | Describes either a frame presentation time or the duration of a transition or effect. |
