# Graph (Child of MainToScenesXTransition) element

Defines the pipeline graph for the DVD transition to scenes menus.

## Usage

``` syntax
<Graph/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                               |
|-----------------------------------------------------------------------|
| [**MainToScenes2Transition**](maintoscenes2transition.md)<br/> |
| [**MainToScenes4Transition**](maintoscenes4transition.md)<br/> |
| [**MainToScenes6Transition**](maintoscenes6transition.md)<br/> |



## Remarks

The pipeline graph defines the flow of the transition and contains inputs (that is, media sources imported by the user) and media nodes (that is, transitions and effects).

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
