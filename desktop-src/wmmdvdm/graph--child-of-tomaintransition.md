# Graph (Child of ToMainTransition) element

Defines the pipeline graph for the DVD transition into the main menu (the "opening" transition).

## Usage

``` syntax
<Graph/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                 |
|---------------------------------------------------------|
| [**ToMainTransition**](tomaintransition.md)<br/> |



## Remarks

The pipeline graph defines the flow of the transition and contains inputs (that is, media sources imported by the user) and media nodes (that is, transitions and effects).

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
