# ThumbnailTime element

Specifies the time offset into the menu in the Windows DVD Maker user interface (UI) that renders a static preview of the menu style.

## Usage

``` syntax
<ThumbnailTime/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
