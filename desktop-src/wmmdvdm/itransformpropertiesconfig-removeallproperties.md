# ITransformPropertiesConfig::RemoveAllProperties method

The **RemoveAllChildProperties** method clears all properties from the current collection.

## Syntax


```C++
HRESULT RemoveAllProperties();
```



## Parameters

This method has no parameters.

## Return value

The method returns an **HRESULT** of S\_OK for success, or a standard COM error code for failure.
