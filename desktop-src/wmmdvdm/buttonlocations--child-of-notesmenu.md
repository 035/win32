# ButtonLocations (Child of NotesMenu) element

Specifies the normalized (with respect to a screen with a 16:9 aspect ratio) position and dimensions of the button on the notes menu. This is the rectangular region that defines each button's location for the DVD player.

## Usage

``` syntax
<ButtonLocations/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

|-------------------------------------------|
| [**NotesMenu**](notesmenu.md)<br/> |
