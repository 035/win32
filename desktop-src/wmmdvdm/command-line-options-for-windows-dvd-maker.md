# Command-line Options for Windows DVD Maker

Windows® DVD Maker 1.0 for Windows Vista™ can be started using several command-line options that cause DVD Maker to take additional actions when it starts.

These command-line options are documented in the following topics.



| Topic                                                                                                                                              | Description                                                                                                        |
|----------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [To Specify a DVD Disc Burner](to-specify-a-dvd-disc-burner.md)                                                                                   | Describes the -drive option that specifies which drive to use when burning DVDs.                                   |
| [To Import Several Media Files into Windows DVD Maker at the Same Time](to-import-several-media-files-into-windows-dvd-maker-at-the-same-time.md) | Describes how to create an XML file for importing media files into Windows DVD Maker.                              |
| [To Restore Windows DVD Maker to Factory Default Settings](to-restore-windows-dvd-maker-to-factory-default-settings.md)                           | Describes the \_clear\_ option that specifies that Windows DVD Maker will start with its factory default settings. |
| [Special-Purpose Command-line Options for Windows DVD Maker](special-purpose-command-line-options-for-windows-dvd-maker.md)                       | Describes the -clickonce and /AppRestart options.                                                                  |
