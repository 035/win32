# NavigationButton element

Causes a media node to be created that renders a scene button.

## Usage

``` syntax
<NavigationButton/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                     |
|---------------------------------------------|
| [**Graph**](https://www.bing.com/search?q=**Graph**)<br/>        |
| [**Subpicture**](subpicture.md)<br/> |



## Remarks

The media node created is based on the button style defined by the [NavigationButtonTFXToken element](navigationbuttontfxtoken.md) for this menu style.

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
