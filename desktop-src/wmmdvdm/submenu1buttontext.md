# Submenu1ButtonText element

Specifies the text used for this menu style's "Submenu1" button. This is usually the "Scenes" button.

## Usage

``` syntax
<Submenu1ButtonText/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                         |
|-------------------------------------------------|
| [**DVDMenuStyle**](dvdmenustyle.md)<br/> |



## Remarks

This button can be used only on the main menu.

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
