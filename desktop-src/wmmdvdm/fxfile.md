# FXFile element

Points to an .fx file that contains the pixel and vertex shaders used in the button style.

## Usage

``` syntax
<FXFile/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                   |
|-------------------------------------------|
| [**DVDButton**](dvdbutton.md)<br/> |



## Remarks

A sample .fx file is included with this SDK, in [Samples](samples.md).

This element is required within each **DVDButton** element.

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
