# To Open a Project File

This option specifies a Windows Movie Maker project file to open.

**Syntax**


```C++
moviemk 
projectfile.mswmm
```



*Parameters*

*projectfile.mswmm*

Specifies the path of a Windows Movie Maker project file to open.

**Examples**


```C++
moviemk C:\Users\johnevans\dadsbirthday.mswmm
```
