# InsetVideoZoom element

Indicates whether to zoom the inset video image to fill the screen or to letterbox or pillarbox it.

## Usage

``` syntax
<InsetVideoZoom/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
