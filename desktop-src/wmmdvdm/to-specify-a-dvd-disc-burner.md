# To Specify a DVD Disc Burner

The -drive option specifies which drive to use when burning DVDs.

**Syntax**


```C++
dvdmaker -drive:
letter
```



**Parameters**

*letter*

Specifies the drive letter of the disc burner to use when burning DVDs.

**Examples**


```C++
dvdmaker -drive:F
```
