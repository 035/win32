# Transitions

This section provides the XML for the Windows Movie Maker built-in transitions and describes the customizable parameters within them. This section contains the following topics.



| Category                                       | Description                                                                                      |
|------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [Wipe Transitions](wipe-transitions.md)       | Gradual transitions from one scene to another using a distinct edge to "wipe" across the screen. |
| [Shatter Transitions](shatter-transitions.md) | Scene transitions that use a "shattering glass" effect.                                          |



 

For more information on how to modify the XML to create your own custom transitions, see [Making Your Own XML File](making-your-own-xml-file.md).
