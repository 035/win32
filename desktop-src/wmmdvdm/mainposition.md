# MainPosition element

Defines the normalized position of the text (within the text's [**BoundingRect**](boundingrect.md)) when it is not "entering" or "exiting" the viewport.

## Usage

``` syntax
<MainPosition/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
