# Scene1Button element

Specifies the location and size of the "Scene 1" button.

## Usage

``` syntax
<Scene1Button/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                       |
|---------------------------------------------------------------|
| [**ButtonLocations**](ButtonLocationsElement_ACRC)<br/> |



## Remarks

When available, this button plays the clip that the user assigned to the "Scene 1" button.

|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
