# TextTFXToken element

Specifies the globally unique identifier (GUID) of a collection of text attributes to be applied to text in this menu style.

## Usage

``` syntax
<TextTFXToken/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
