# ParentMenuButton element

Specifies the location and size of the "Parent Menu" button.

## Usage

``` syntax
<ParentMenuButton/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                       |
|---------------------------------------------------------------|
| [**ButtonLocations**](ButtonLocationsElement_ACRC)<br/> |



## Remarks

The "Parent Menu" button returns users to the next menu up (that is, the current menu's "parent"). This button cannot be used on the main menu.

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
