# To Restore Windows DVD Maker to Factory Default Settings

The \_clear\_ option specifies that Windows DVD Maker will start with its factory default settings.

**Syntax**


```C++
dvdmaker _clear_
```



## Remarks

This command-line option indicates that Windows DVD Maker will start with factory default settings for screen position, window size, menu style, and so on. This option is for debugging and diagnostic purposes only.
