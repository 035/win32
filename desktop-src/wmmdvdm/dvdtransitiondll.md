# DVDTransitionDLL element

The boundary tag for a single DVD transition object.

## Usage

``` syntax
<DVDTransitionDLL/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                             |
|-----------------------------------------------------|

## Remarks

This object can hold one or more DVD transitions.
