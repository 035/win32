# ScenesMenu6 element

Defines a menu that has six or fewer scene buttons.

## Usage

``` syntax
<ScenesMenu6/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Remarks

This element is required in every menu style.
