# Windows DVD Maker Programming Reference

This section of the Windows Movie Maker and Windows DVD Maker SDK contains reference information for extending Windows DVD Maker 1.0 in Windows Vista. It contains the following sections.



| Section                                                                                      | Description                                                                                                                                                                                                                           |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Windows DVD Maker XML Extensibility](windows-dvd-maker-xml-extensibility.md)               | Provides one sample transition, button, and menu style for Windows DVD Maker, and describes each XML object in detail.                                                                                                                |
| [Command-line Options for Windows DVD Maker](command-line-options-for-windows-dvd-maker.md) | Describes the command-line options supported by Windows DVD Maker.                                                                                                                                                                    |
| [Windows DVD Maker Project File Format](windows-dvd-maker-project-file-format.md)           | Documents the Windows DVD Maker Project File Format. A simple XML-based file consisting of elements that describe the media content, title, menu style, button names, and other information contained in a Windows DVD Maker project. |
