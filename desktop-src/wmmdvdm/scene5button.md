# Scene5Button element

Specifies the location and size of the "Scene 5" button.

## Usage

``` syntax
<Scene5Button/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                       |
|---------------------------------------------------------------|
| [**ButtonLocations**](https://www.bing.com/search?q=**ButtonLocations**)<br/> |



## Remarks

When available, this button plays the clip that the user assigned to the "Scene 5" button.

|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
