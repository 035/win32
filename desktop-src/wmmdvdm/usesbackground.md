# UsesBackground element

Specifies whether the user will be able to override the specified background source.

## Usage

``` syntax
<UsesBackground/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                         |
|-------------------------------------------------|
| [**DVDMenuStyle**](dvdmenustyle.md)<br/> |



## Remarks

A background source is a sequence of portions of user-imported clips with crossfades between clips. Five seconds of user content is followed by a half-second crossfade into another five seconds of user content, and so on.
