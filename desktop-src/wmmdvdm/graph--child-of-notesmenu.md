# Graph (Child of NotesMenu) element

Defines the pipeline graph for the notes menu.

## Usage

``` syntax
<Graph/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                   |
|-------------------------------------------|
| [**NotesMenu**](notesmenu.md)<br/> |



## Remarks

The pipeline graph defines the flow of the menu and contains inputs (that is, media sources imported by the user) and media nodes (that is, transitions and effects).

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
