# Structures

Transforms in Windows Movie Maker 6.0 can use the following structures.



| Structure                            | Description                                |
|--------------------------------------|--------------------------------------------|
| [**BufferFormat**](bufferformat.md) | Describes the format of a buffer.          |
| [TIME\_INFO](time-info.md)          | Describes the current sample time.         |
| [VIDEO\_SIZE](video-size.md)        | Describes the dimensions of a video frame. |
