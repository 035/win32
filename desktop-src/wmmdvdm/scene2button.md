# Scene2Button element

Specifies the location and size of the "Scene 2" button.

## Usage

``` syntax
<Scene2Button/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                       |
|---------------------------------------------------------------|
| [**ButtonLocations**](https://www.bing.com/search?q=**ButtonLocations**)<br/> |



## Remarks

When available, this button plays the clip that the user assigned to the "Scene 2" button.

|                                     |               |
|-------------------------------------|---------------|
