# SubpictureTextures element

Specifies the source files that will be created when rendering the subpicture.

## Usage

``` syntax
<SubpictureTextures/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                   |
|-------------------------------------------|
| [**DVDButton**](dvdbutton.md)<br/> |



## Remarks

The "subpicture" refers to the "highlighting" of menu buttons you see during menu navigation.

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
