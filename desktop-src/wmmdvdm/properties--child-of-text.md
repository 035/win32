# Properties (Child of Text) element

Specifies the properties of the text media node.

## Usage

``` syntax
<Properties/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                         |
|---------------------------------|
| [**Text**](text.md)<br/> |



## Remarks

The [**ExitDuration**](exitduration.md), [**ExitEffect**](exiteffect.md), and [**ExitPosition**](exitposition.md) elements are optional.

|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
