# Command-line Options for Windows Movie Maker

Microsoft® Windows® Movie Maker 6.0 for Windows Vista™ can be started using several command-line options that cause Movie Maker to take additional actions when it starts.

These command-line options are documented in the following topics.



| Topic                                                                                                                                                  | Description                                                                             |
|--------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [To Open a Project File](to-open-a-project-file.md)                                                                                                   | Specifies a Windows Movie Maker project file to open.                                   |
| [To Import a Video or Image or Audio File](to-import-a-video-or-image-or-audio-file.md)                                                               | Specifies the path of a media file to import.                                           |
| [To Import Several Media Files into Windows Movie Maker at the Same Time](to-import-several-media-files-into-windows-movie-maker-at-the-same-time.md) | Describes how to create an XML file for importing media files into Windows Movie Maker. |
