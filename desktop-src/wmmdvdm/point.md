# Point element

Specifies the values of the parameter at specific times.

## Usage

``` syntax
<Point/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements

|--------------------------------------|

| [**Rotate**](rotate.md)<br/>  |



## Remarks

When an element has **Point** child elements, it should not also have a **value** attribute.
