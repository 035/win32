# Subpicture element

Describes the pipeline graph that Windows DVD Maker will use when rendering the subpicture.

## Usage

``` syntax
<Subpicture/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                       |
|-----------------------------------------------|
| [**MainMenu**](mainmenu.md)<br/>       |
| [**NotesMenu**](notesmenu.md)<br/>     |
| [**ScenesMenu2**](scenesmenu2.md)<br/> |
| [**ScenesMenu4**](scenesmenu4.md)<br/> |
| [**ScenesMenu6**](scenesmenu6.md)<br/> |



## Remarks

The "subpicture" refers to the "highlighting" of menu buttons you see during menu navigation. Only a single frame (time zero) is used for the subpicture.

|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
