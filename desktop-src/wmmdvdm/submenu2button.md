# Submenu2Button element

Specifies the location and size of the "Submenu2" button. This is usually the "Notes" button.

## Usage

``` syntax
<Submenu2Button/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                       |
|---------------------------------------------------------------|
| [**ButtonLocations**](ButtonLocationsElement_ACRC)<br/> |



## Remarks

This button can be used only on the main menu, but is optional and will appear only if the user has added notes via the Windows DVD Maker UI.

## Element information



|                                     |               |
|-------------------------------------|---------------|
| Minimum supported system<br/> | Windows Vista |
| Can be empty                        | Yes           |
