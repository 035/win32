# To add the most common HTML character formatting tags to your file

1.  Select the text you want to format.
2.  On the **Tags** menu, point to **Character Formatting**, and then click the formatting you want.

> [!Note]  
> To add a &lt;HEADING&gt;, &lt;COMMENT&gt;, or &lt;TITLE&gt; tag, click the **Tags** menu, and then click **Heading**, **Comment**, or **Title**.
