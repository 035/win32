# To enable full-text search

1.  Open a project (.hhp) file, and then click **Change Project Options**.

    

    |                        |                                                                            |
    |------------------------|----------------------------------------------------------------------------|
    | ![](images/sb-cpo.gif) | Specifies files, compiler, window, and other project settings. <br/> |

    

     

2.  Click the **Compiler** tab, and select the **Compile full-text search information** check box.

## Notes

-   You must also [add a Search tab](to-add-a-search-tab-to-the-navigation-pane.md) to any window definition where you want one to appear.
-   You can decrease the time it takes for a full-text search by including a [full-text search stop list](to-add-a-full-text-search-stop-list-to-a-help-project.md) in your project file.
