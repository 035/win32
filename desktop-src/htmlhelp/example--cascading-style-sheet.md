# Example: Cascading Style Sheet

If your users have Microsoft Internet Explorer version 3.02 as their browser, you will need to ship two separate cascading style sheets. You need to create one style sheet works with Microsoft Internet Explorer version 3.02 and another that works with Internet Explorer version 4.0.

Both can be used as external style sheets and attached to each file using the **&lt;LINK&gt;** tag:

-   [Sample of an Internet Explorer 3.02 cascading style sheet](example--internet-explorer-3-02-cascading-style-sheet.md).
-   [Sample of an Internet Explorer 4.0 cascading style sheet](example--internet-explorer-4-0-cascading-style-sheet.md).
