# To add or remove buttons in a help window

1.  Open a project (.hhp) file, and then click **Add/Modify Window Definitions**. 

    |                        |                                                                                                  |
    |------------------------|--------------------------------------------------------------------------------------------------|
    | ![](images/sb-adm.gif) | Adds or edits window definitions and the general appearance of the HTML Help Viewer. <br/> |

    

     

2.  Click the **Buttons** tab, and then, in the **Window type** box, select the window for which you want to add or remove buttons.
3.  Select the options you want.

## Notes

-   You must create a new [secondary window](creating-secondary-windows.md) or customize the [Help Viewer](using-the-help-viewer-for-topics.md) before you can add or remove buttons.
-   See the [list of button functions](adding-buttons-to-a-help-window.md) for the functionality included with each button type.
-   If you [create a Jump button](to-add-a-jump-button-to-a-help-window.md), you will be guided through a wizard to specify the jump location.
