# To view the complete frameset in a browser

Open the frameset in Microsoft Internet Explorer. For this example, double-click frames.htm.

## Note

You have now created the basic template for a frameset. You can make any modifications you want to the HTML files that appear. You can also [edit the HTML Help ActiveX control](to-edit-the-html-help-activex-control-in-a-file.md).
