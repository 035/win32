# To include a header file in a project file

1.  Open a project (.hhp) file, and then click **HTML Help API Information**.

    

    |                        |                                                                                                                          |
    |------------------------|--------------------------------------------------------------------------------------------------------------------------|
    | ![](images/sb-api.gif) | Adds or removes map, alias and text pop-up files that are used by the program calling into the HtmlHelp API. <br/> |

    

     

2.  On the **Map** tab, click **Header File**.
3.  Locate the header (.h) file you want to include.
4.  Save the project.
