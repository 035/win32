# To Create a New Help Project File

1.  On the **File** menu, click **New**, and then click **Project**.
2.  Follow the instructions in the New Project Wizard.

> [!Note]  
> You can [convert an existing help project into an HTML Help project](converting-existing-help-projects.md).
