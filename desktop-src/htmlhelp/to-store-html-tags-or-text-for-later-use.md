# To store HTML tags or text for later use

1.  Copy the tag or text you want to reuse.
2.  On the **Tags** menu, point to **Insert Text Block**, and then click an **Empty** command.
3.  Click **Yes**.
4.  In the **Name of text block** box, type a name for the text block.
5.  In the **Text block** box, enter the tag or text.

> [!Note]  
> To edit a stored tag, click the **Tags** menu, point to **Edit Text Block**, and then click the tag you want to edit.
