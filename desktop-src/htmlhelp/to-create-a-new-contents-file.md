# To create a new contents file

1.  On the **File** menu, click **New**, and then click **Table of Contents**.
2.  Click **Insert a Heading** or **Insert a Page** to add the titles and subtitles that you want to appear in the table of contents. 

    |                          |                                                                                                                          |
    |--------------------------|--------------------------------------------------------------------------------------------------------------------------|
    | ![](images/sb-inshd.gif) | Adds a heading level icon to a table of contents with its associated name, file, URL, and style information. <br/> |

    

     

    

    |                          |                                                                                                                       |
    |--------------------------|-----------------------------------------------------------------------------------------------------------------------|
    | ![](images/sb-inspg.gif) | Adds a page level icon to a table of contents with its associated name, file, URL, and style information. <br/> |

    

     

## Notes

-   You can move an entry up or down, left or right, by clicking the Move Arrows. When you move a heading entry, all the entries that follow it are moved too. 

    |                         |                                                           |
    |-------------------------|-----------------------------------------------------------|
    | ![](images/sb-move.gif) | Moves the selection up, down, left, or right. <br/> |

    

     

-   To change an entry, click **Edit Selection**. 

    |                          |                                                                                                   |
    |--------------------------|---------------------------------------------------------------------------------------------------|
    | ![](images/sb-edsel.gif) | Edits the associated name, file, URL, keyword or style information for the selection. <br/> |

    

     

-   Before you can display a contents file in the Help Viewer, you need to add it to the help project file, and then compile the project.
