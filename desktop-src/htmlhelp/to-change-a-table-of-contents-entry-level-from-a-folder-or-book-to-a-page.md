# To change a table of contents entry level from a folder or book to a page

1.  Open the contents file, and then select the entry you want to change to a folder or book.
2.  Click **Edit Selection**, and then click the **Advanced** tab. 

    |                          |                                                                                                   |
    |--------------------------|---------------------------------------------------------------------------------------------------|
    | ![](images/sb-edsel.gif) | Edits the associated name, file, URL, keyword or style information for the selection. <br/> |

    

     

3.  Select the **Change entry to a heading** check box.

## Note

You can also use this procedure to change a page to a folder or book.
