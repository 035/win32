# To specify the location of index files

1.  Open a project (.hhp) file, and then click **Change Project Options**.

    

    |                        |                                                                           |
    |------------------------|---------------------------------------------------------------------------|
    | ![](images/sb-cpo.gif) | Specifies files, compiler, window, and other project settings.<br/> |

    

     

2.  Click the **Files** tab, and then in the **Index file** box, enter the location and name of your index (.hhk) file.

> [!Note]  
> You can also specify [how the index entries you create will respond](to-set-actions-to-take-when-single--or-double-clicking-an-index-entry.md) when you single-click or double-click them in HTML Help Workshop.
