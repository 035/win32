# To apply an icon to a contents entry

1.  Open a project (.hhp) file, and then click the **Contents** tab.
2.  Click a contents entry, and then click the **Edit Selection** button. 

    |                          |                                                                                                   |
    |--------------------------|---------------------------------------------------------------------------------------------------|
    | ![](images/sb-edsel.gif) | Edits the associated name, file, URL, keyword or style information for the selection. <br/> |

    

     

3.  Click the **Advanced** tab.
4.  In the **Image** index box, select the image you want to apply to the contents entry.

## Notes

-   You can [create new icon images](about-creating-a-new-icon-image-strip-for-a-contents-file.md) for any type of contents entry.
-   Some icons are not supported in a [binary table of contents](about-binary-contents-files.md).
