# Select the button options

1.  Choose an option for what you would like the button to display (text, bitmap, or icon).
2.  Enter the text to display on the button, or a path to the .bmp or .ico file you would like to include.
3.  Click **Next**.

## Notes

-   If you chose the "hidden" option in [step 2](specify-how-you-want-the-command-to-appear-to-the-user.md), you will not see this Control Wizard screen.
-   HTML Help Workshop comes with a built-in "shortcut" icon. To use this icon on a shortcut button, select **Display a bitmap on the button**, and type in the **Bitmap** field.
