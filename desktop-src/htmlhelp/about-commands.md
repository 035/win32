# About Commands

The HTML Help API supports a set of commands (*uCommand* values) that enable you to access functionality programmatically.

The most basic of the commands is [HH\_DISPLAY\_TOPIC](hh-display-topic-command.md), which enables you to open a compiled help (.chm) file in a specific help window and display a specific topic within the help file.
