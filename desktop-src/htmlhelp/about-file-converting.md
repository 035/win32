# About File Converting

After you have opened an image in the HTML Help Image Editor, you can convert the image file to any other format that the program supports.

In the HTML Help Image Editor, input capabilities have been expanded to include: .gif, .jpg, .tga, .pcd, .png, and .wmf files. New output formats include: .gif, .jpeg, and .tga files.
