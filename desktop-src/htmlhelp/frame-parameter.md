# Frame Parameter

Use the **Frame** parameter to specify the frame in which you want target topics to appear.

## Applies to

-   [ALink](alink.md)
-   [KLink](klink.md)
-   [Related Topics](related-topics.md)

## Valid Values



| Syntax                                          | Description                                                                           | Notes                                |
|-------------------------------------------------|---------------------------------------------------------------------------------------|--------------------------------------|
| &lt;PARAM name="Frame" value="*frame name*"&gt; | If you are using a frameset, specifies the frame for the selected topic to appear in. | The **Frame** parameter is optional. |
