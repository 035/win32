# Creating a Contents File

You make a table of contents by creating a contents (.hhc) file.

## Notes

-   If you [design a help system using the Help Viewer](using-the-help-viewer-for-topics.md), adding a contents file to your help project creates the Contents tab in the Navigation pane of the Help Viewer once the project is compiled.
-   If you design for a Web site or your own frameset, make sure you [add the HTML Help ActiveX control](inserting-the-html-help-activex-control.md) to the appropriate files and [specify a default frame](to-specify-the-default-frame-in-a-contents-or-index-file.md) to display your contents file.
