# To set an index keyword to be inactive

1.  Open an index file, click the keyword you want to turn into an inactive link, and then click **Edit Selection**.

    

    |                          |                                                                                                  |
    |--------------------------|--------------------------------------------------------------------------------------------------|
    | ![](images/sb-edsel.gif) | Edits the associated name, file, URL, keyword or style information for the selection.<br/> |

    

     

2.  Click the **Advanced** tab, and then select the **Target is another keyword** check box.
3.  Click the **General** tab, and then click **Edit**.
4.  In the **File or URL** box, replace the file name or URL with the keyword exactly as it appears in the **Keyword** box, and then click **OK**.

> [!Note]  
> This sets an index entry to jump to itself, creating the appearance of an inactive link.
