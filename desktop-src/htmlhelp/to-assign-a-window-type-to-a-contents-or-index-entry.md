# To assign a window type to a contents or index entry

1.  Open a contents (.hhc) or index (.hhk) file, select the entry you want to appear in a specific window type, and then click **Edit Selection**. 

    |                          |                                                                                                   |
    |--------------------------|---------------------------------------------------------------------------------------------------|
    | ![](images/sb-edsel.gif) | Edits the associated name, file, URL, keyword or style information for the selection. <br/> |

    

     

2.  Click the **Advanced** tab, and in the **Window** box, type the name of the window type.

## Notes

-   Topics will automatically appear in the Help Viewer unless you assign them to another window type.
-   You must [create a new window type](to-add-a-secondary-window.md) in your project file or the topic will continue to appear in the default window.
