# To create a window without a Navigation pane

1.  Open a help project file, and then click **Add/Modify Window Definitions**. 

    |                        |                                                                                                  |
    |------------------------|--------------------------------------------------------------------------------------------------|
    | ![](images/sb-adm.gif) | Adds or edits window definitions and the general appearance of the HTML Help Viewer. <br/> |

    

     

2.  In the **Create a new window type named** box, type a window name for the new window.
3.  Click the **Navigation Pane** tab, and then clear the Window with navigation pane, topic pane, and button bar check box.

> [!Note]  
> To customize the Help Viewer to appear as a window without a Navigation pane, you must create a new window. Any changes to your new window are then noted in the \[WINDOWS\] section of the project (.hhp) file.
