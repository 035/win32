# Start the control wizard and specify the shortcut command

1.  In HTML Help Workshop, open an HTML file, and then position your cursor at the location where you want to create a shortcut.
2.  Click **HTML Help ActiveX Control**.

    

    |                        |                                                                                                                          |
    |------------------------|--------------------------------------------------------------------------------------------------------------------------|
    | ![](images/sb-wiz.gif) | Opens the HTML Help ActiveX Control Wizard, which enables you to insert or edit the control in an HTML file. <br/> |

    

     

3.  In the **Specify the command** box, click **ShortCut**, and then enter an ID name for this instance of the control.
4.  Click **Next**.
