# To set actions to take when single- or double-clicking an index entry

1.  On the **File** menu, click **Preferences**.
2.  Click the **Index** tab, and then select the options you want.

> [!Note]  
> These preferences apply to how the index editor functions when a help author is using HTML Help workshop. They do not apply to how the index functions in a compiled help file.
