# To customize the appearance of a contents file

1.  Open a project (.hhp) file, click the **Contents** tab, and then click **Contents Properties**.

    

    |                          |                                                                      |
    |--------------------------|----------------------------------------------------------------------|
    | ![](images/sb-conpr.gif) | Defines the general appearance of the table of contents. <br/> |

    

     

2.  Click the **Styles** tab, and then click **Add** or **Remove** to specify the styles you want.
