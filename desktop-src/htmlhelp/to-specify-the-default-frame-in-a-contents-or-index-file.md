# To specify the default frame in a contents or index file

1.  Open a help project file.
2.  Click the **Contents** or **Index** tab, and then click **Contents Properties** or **Index Properties**. 

    |                          |                                                                      |
    |--------------------------|----------------------------------------------------------------------|
    | ![](images/sb-conpr.gif) | Defines the general appearance of the table of contents. <br/> |

    

     

    

    |                          |                                                          |
    |--------------------------|----------------------------------------------------------|
    | ![](images/sb-idxpr.gif) | Defines the general appearance of the index. <br/> |

    

     

3.  In the **Default frame** box, type the HTML frame name where your pages will jump.

## Notes

-   The frame you specify must be defined in an HTML document.
-   You do not need to create your own frameset if you plan to use the [Help Viewer](using-the-help-viewer-for-topics.md) to display your content.
