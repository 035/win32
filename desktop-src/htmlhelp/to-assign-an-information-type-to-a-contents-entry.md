# To assign an information type to a contents entry

1.  Open a contents (.hhc) file, select the entry you want to add an information type to, and then click **Edit Selection**. 

    |                          |                                                                                                  |
    |--------------------------|--------------------------------------------------------------------------------------------------|
    | ![](images/sb-edsel.gif) | Edits the associated name, file, URL, keyword or style information for the selection.<br/> |

    

     

2.  In the **Available information types** box, select the name of the information type you want to associate with the table of contents entry.

## Notes

-   You must either [define information types that use categories](to-define-categories-and-information-types.md), or a list of [information types](to-define-information-types-when-not-using-categories.md) before you can assign them to a topic.
-   If used with a [large collection](manage-large-document-sets.md) of merged compiled help (.chm) files, information types should be standardized in the table of contents for all the files in the collection.
-   Information types are not supported when using a [binary table of contents](about-binary-contents-files.md).
