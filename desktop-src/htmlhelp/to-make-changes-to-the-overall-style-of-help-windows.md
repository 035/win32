# To make changes to the overall style of help windows

1.  Open a project (.hhp) file, and then click **Add/Modify Window Definitions**.

    

    |                        |                                                                                                  |
    |------------------------|--------------------------------------------------------------------------------------------------|
    | ![](images/sb-adm.gif) | Adds or edits window definitions and the general appearance of the HTML Help Viewer. <br/> |

    

     

2.  Click the **Styles** or **Extended Styles** tab, and in the **Window type** box, select the window for which you want to specify styles.
3.  Select the options you want.

## Notes

-   Use this procedure, for example, to enable context-sensitive help for a window type.
-   You can click the question mark in the upper-right corner of the screen, and then click a style option to see a definition of the option.
