# Design Help

These topics contain information of special interest when planning a help system that will ship with a program.

-   You use the [Help Viewer](using-the-help-viewer-for-topics.md) to display your topics.
-   You can create unique [help system window designs](designing-help-windows.md).
-   You can design [context-sensitive help](designing-context-sensitive-help.md) for your system.
-   You can create [training cards](designing-training-card-help.md) for your help system.
-   You can add [shortcuts](example--creating-a-shortcut.md) to your help topics.
