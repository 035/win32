# To add a default frame location to the table of contents file

1.  Open the contents file that will appear in your frameset and click **Contents Properties**. 

    |                          |                                                                      |
    |--------------------------|----------------------------------------------------------------------|
    | ![](images/sb-conpr.gif) | Defines the general appearance of the table of contents. <br/> |

    

     

2.  In the **Default frame** box, type: **right**.
3.  Save the contents file.
