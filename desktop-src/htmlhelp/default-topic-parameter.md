# Default Topic Parameter

Use the **Default Topic** parameter to specify a topic to navigate to if an [ALink](alink.md) or [KLink](klink.md) cannot locate the target topic.

## Applies to

-   [ALink](alink.md)
-   [KLink](klink.md)

## Valid Values



| Syntax                                                             | Description                                                                  | Notes                                        |
|--------------------------------------------------------------------|------------------------------------------------------------------------------|----------------------------------------------|
| &lt;PARAM name="Default Topic" value="*File name or HTTP URL*"&gt; | Specifies an HTML file name or an HTTP URL to navigate to if the link fails. | The **Default Topic** parameter is optional. |
