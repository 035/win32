# To hide compiler messages

1.  Open a project (.hhp) file, click **Change Project Options**, and then click the **Compiler** tab. 

    |                        |                                                                           |
    |------------------------|---------------------------------------------------------------------------|
    | ![](images/sb-cpo.gif) | Specifies files, compiler, window, and other project settings.<br/> |

    

     

2.  If you want to hide notes, clear the **Notes** check box.
3.  If you want to hide the progress report, clear the **Progress** check box. The progress report includes the name of each HTML file as it is processed.
