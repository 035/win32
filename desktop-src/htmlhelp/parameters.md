# Parameters

A parameter is a set of [&lt;PARAM&gt;](html-help-activex-control-syntax.md) tags with NAME and VALUE attributes that specify a [command](about-commands.md) and information required by the command.

For example, when you specify the [Contents](contents.md) command, you must also specify a contents (.hhc) file using the **Item1** parameter.

Some parameters are used with more than one command. For information about a specific instance of a parameter, see the command for which it is used.
