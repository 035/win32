# To flag a new table of contents entry

1.  Open the contents (.hhc) file, and then select the entry you want to flag as a new entry.
2.  Click **Edit Selection**, and then click the **Advanced** tab.

    

    |                          |                                                                                                   |
    |--------------------------|---------------------------------------------------------------------------------------------------|
    | ![](images/sb-edsel.gif) | Edits the associated name, file, URL, keyword or style information for the selection. <br/> |

    

     

3.  Select the **Mark as a new entry** check box.

## Note

New icons are not supported in a [binary table of contents](about-binary-contents-files.md).
