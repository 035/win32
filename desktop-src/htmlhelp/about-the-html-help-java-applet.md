# About the HTML Help Java Applet

The HTML Help Java Applet is a control containing navigation features that you can [insert into an HTML file](inserting-the-html-help-java-applet.md). Use the HTML Help Java Applet with any Web browser that supports Java.

The HTML Help Java Applet enables you to add the following navigation features to your HTML files:

-   Table of contents
-   Index
-   Related Topics
