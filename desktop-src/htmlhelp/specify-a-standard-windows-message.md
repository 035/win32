# Specify a standard Windows message

1.  Select a standard Windows message from the **Message** pulldown, or enter the numeric value for a Windows message.
2.  Enter values in the **WPARAM** and **LPARAM** boxes.
3.  In the **File/URL** box, enter the name of an HTML file to open if the program cannot be found. You can leave this box blank if you want to.
