# tabpos Member

The *tabpos* member is used to specify where the tabs are to be placed on the Navigation pane of the HTML Help Viewer.

This member can be one of the following values:



| Value                 | Description                    |
|-----------------------|--------------------------------|
| HHWIN\_NAVTAB\_BOTTOM | Places the tabs at the bottom. |
| HHWIN\_NAVTAB\_LEFT   | Places the tabs at the left.   |
| HHWIN\_NAVTAB\_TOP    | Places the tabs at the top.    |
