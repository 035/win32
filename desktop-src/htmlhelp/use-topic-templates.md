# Use Topic Templates

An HTML topic template is a file that contains all the font, style, heading, and design elements you use most frequently. You can [create HTML topic templates](creating-a-topic-template.md) to format all the content in your help system or Web site. You can view some [example topic templates](example--topic-templates.md) to get an idea of how they might be constructed.
