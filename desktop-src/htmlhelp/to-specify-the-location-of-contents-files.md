# To specify the location of contents files

1.  Open a project (.hhp) file, and then click **Change Project Options**.

    

    |                        |                                                                           |
    |------------------------|---------------------------------------------------------------------------|
    | ![](images/sb-cpo.gif) | Specifies files, compiler, window, and other project settings.<br/> |

    

     

2.  Click the **Files** tab, and then in the **Contents file** box, enter the name and location of your contents (.hhc) file.

## Notes

-   If you want a contents file created for you, select the **Automatically create contents file (.hhc) when compiling** check box, and then in the **Maximum heading level** box, select the heading level at which you want contents entries to be created.
-   You can also specify [how the contents entries you create will respond](to-set-actions-to-take-when-single--or-double-clicking-a-contents-entry.md) when you single-click or double-click them in HTML Help Workshop.
