# InfoTech API Reference

The InfoTech API reference provides information on the InfoTech interfaces that are used by HTML Help.

To enable your program to use HTML Help, you do not need to use this API.

The header file infotech.h is required to use this API. It is installed in the include directory of the Windows SDK.

The following interfaces are covered:

-   [**IITDatabase Interface**](/windows/previous-versions/Infotech/nn-infotech-iitdatabase?branch=master)
-   [**IITPropList Interface**](/windows/previous-versions/Infotech/nn-infotech-iitproplist?branch=master)
-   [**IITResultSet Interface**](/windows/previous-versions/Infotech/nn-infotech-iitresultset?branch=master)
-   [**IITWordWheel Interface**](/windows/previous-versions/Infotech/nn-infotech-iitwordwheel?branch=master)
-   [**IStemmerConfig Interface**](/windows/previous-versions/Infotech/nn-infotech-istemmerconfig?branch=master)
