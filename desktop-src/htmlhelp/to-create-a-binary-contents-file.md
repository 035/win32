# To create a binary contents file

1.  Open a project file, and then click **Change Project Options**.

    

    |                        |                                                                            |
    |------------------------|----------------------------------------------------------------------------|
    | ![](images/sb-cpo.gif) | Specifies files, compiler, window, and other project settings. <br/> |

    

     

2.  Click the **Compiler** tab, and select the **Create a binary TOC** checkbox.

## Notes

-   Only book and page icons are supported. Alternate icons, such as the ones for a new entry or Web page entry, are not supported.
-   Site map customizations are not supported.
-   Information types are not supported.
