# About the Reference Section

The reference section provides comprehensive information about how to use the HTML Help API commands, data structures, and notification messages. There are three sections:

-   The [commands reference](about-commands.md) provides syntax descriptions and code examples for all commands (*uCommand* parameter values) that can be passed to **HtmlHelp()**.
-   The [structures reference](about-structures.md) provides syntax descriptions for all data structures used by **HtmlHelp()**.
-   The [notification messages reference](about-notification-messages.md) provides syntax descriptions for all notification messages sent by **HtmlHelp()**.
