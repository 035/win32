# HTML Help API Reference

The HTML Help API Reference provides information on the HTML Help application programming interface (API), which enables a program to display a help window. It includes comprehensive information about how to use the HTML Help API commands, data structures, and notification messages.

The reference provides the following sections:

-   The [overview](html-help-api-overview.md) section describes the HTML Help API, and explains the syntax required to access it.
-   The [reference](about-the-reference-section.md) topics provide syntax descriptions and code examples for all commands that can be passed to HtmlHelp(), and syntax descriptions for all structures and notification messages used by HtmlHelp().
