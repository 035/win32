# curNavType Member

The *curNavType* member is used to specify which tab in the Navigation pane of the HTML Help Viewer should have focus.

This member can be one of the following values:



| Value                     | Description                 |
|---------------------------|-----------------------------|
| HHWIN\_NAVTYPE\_FAVORITES | Displays the Favorites tab. |
| HHWIN\_NAVTYPE\_INDEX     | Displays the Index tab.     |
| HHWIN\_NAVTYPE\_SEARCH    | Displays the Search tab.    |
| HHWIN\_NAVTYPE\_TOC       | Displays the Contents tab.  |
