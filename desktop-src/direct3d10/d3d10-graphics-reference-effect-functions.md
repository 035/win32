# Effect Functions (Direct3D 10)

This section contains information about the following effect functions:



| Functions                                                                      | Description                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**D3D10CompileEffectFromMemory**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10compileeffectfrommemory?branch=master)           | Compile an effect.                                                  |
| [**D3D10CreateEffectFromMemory**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10createeffectfrommemory?branch=master)             | Create an effect.                                                   |
| [**D3D10CreateEffectPoolFromMemory**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10createeffectpoolfrommemory?branch=master)     | Create an effect pool.                                              |
| [**D3D10CreateStateBlock**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10createstateblock?branch=master)                         | Create a state block.                                               |
| [**D3D10DisassembleEffect**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10disassembleeffect?branch=master)                       | Disassemble a compiled effect into shader assembly instructions.    |
| [**D3D10StateBlockMaskDifference**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdifference?branch=master)         | Combine two state-block masks with a bitwise XOR.                   |
| [**D3D10StateBlockMaskDisableAll**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisableall?branch=master)         | Disable state capturing.                                            |
| [**D3D10StateBlockMaskDisableCapture**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisablecapture?branch=master) | Disable state capturing with a state-block mask.                    |
| [**D3D10StateBlockMaskEnableAll**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenableall?branch=master)           | Enable a state-block mask to capture and apply all state variables. |
| [**D3D10StateBlockMaskEnableCapture**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenablecapture?branch=master)   | Enable a range of state values in a state block mask.               |
| [**D3D10StateBlockMaskGetSetting**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskgetsetting?branch=master)         | Get an element in a state-block mask.                               |
| [**D3D10StateBlockMaskIntersect**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskintersect?branch=master)           | Combine two state-block masks with a bitwise AND.                   |
| [**D3D10StateBlockMaskUnion**](/windows/win32/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskunion?branch=master)                   | Combine two state-block masks with a bitwise OR.                    |
