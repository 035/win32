# Effect Functions (Direct3D 10)

This section contains information about the following effect functions:



| Functions                                                                      | Description                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**D3D10CompileEffectFromMemory**](d3d10compileeffectfrommemory.md)           | Compile an effect.                                                  |
| [**D3D10CreateEffectFromMemory**](d3d10createeffectfrommemory.md)             | Create an effect.                                                   |
| [**D3D10CreateEffectPoolFromMemory**](d3d10createeffectpoolfrommemory.md)     | Create an effect pool.                                              |
| [**D3D10CreateStateBlock**](d3d10createstateblock.md)                         | Create a state block.                                               |
| [**D3D10DisassembleEffect**](d3d10disassembleeffect.md)                       | Disassemble a compiled effect into shader assembly instructions.    |
| [**D3D10StateBlockMaskDifference**](d3d10stateblockmaskdifference.md)         | Combine two state-block masks with a bitwise XOR.                   |
| [**D3D10StateBlockMaskDisableAll**](d3d10stateblockmaskdisableall.md)         | Disable state capturing.                                            |
| [**D3D10StateBlockMaskDisableCapture**](d3d10stateblockmaskdisablecapture.md) | Disable state capturing with a state-block mask.                    |
| [**D3D10StateBlockMaskEnableAll**](d3d10stateblockmaskenableall.md)           | Enable a state-block mask to capture and apply all state variables. |
| [**D3D10StateBlockMaskEnableCapture**](d3d10stateblockmaskenablecapture.md)   | Enable a range of state values in a state block mask.               |
| [**D3D10StateBlockMaskGetSetting**](d3d10stateblockmaskgetsetting.md)         | Get an element in a state-block mask.                               |
| [**D3D10StateBlockMaskIntersect**](d3d10stateblockmaskintersect.md)           | Combine two state-block masks with a bitwise AND.                   |
| [**D3D10StateBlockMaskUnion**](d3d10stateblockmaskunion.md)                   | Combine two state-block masks with a bitwise OR.                    |
