# Shader Structures

This section contains information about the following shader structures:



| Structures                                                                         | Description                                                                           |
|------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**D3D10\_SHADER\_BUFFER\_DESC**](/windows/win32/D3D10Shader/ns-d3d10shader-_d3d10_shader_buffer_desc?branch=master)                    | Describes a shader-constant buffer or a shader-texture buffer.                        |
| [**D3D10\_SHADER\_DESC**](/windows/win32/D3D10Shader/ns-d3d10shader-_d3d10_shader_desc?branch=master)                                   | Describes a shader.                                                                   |
| [**D3D10\_SHADER\_DEBUG\_FILE\_INFO**](/windows/win32/d3d10_1shader/ns-d3d10_1shader-_d3d10_shader_debug_file_info?branch=master)           | Describes files included by a shader.                                                 |
| [**D3D10\_SHADER\_DEBUG\_INFO**](/windows/win32/d3d10_1shader/ns-d3d10_1shader-_d3d10_shader_debug_info?branch=master)                      | Describes the format of the ID3D10Blob Interface returned by D3D10GetShaderDebugInfo. |
| [**D3D10\_SHADER\_DEBUG\_INPUT\_INFO**](/windows/win32/d3d10_1shader/ns-d3d10_1shader-_d3d10_shader_debug_input_info?branch=master)         | Describes a shader input.                                                             |
| [**D3D10\_SHADER\_DEBUG\_INST\_INFO**](/windows/win32/d3d10_1shader/ns-d3d10_1shader-_d3d10_shader_debug_inst_info?branch=master)           | Contains instruction data.                                                            |
| [**D3D10\_SHADER\_DEBUG\_OUTPUTREG\_INFO**](/windows/win32/d3d10_1shader/ns-d3d10_1shader-_d3d10_shader_debug_outputreg_info?branch=master) | Describes a shader output register.                                                   |
| [**D3D10\_SHADER\_DEBUG\_OUTPUTVAR**](/windows/win32/d3d10_1shader/ns-d3d10_1shader-_d3d10_shader_debug_outputvar?branch=master)            | Describes a shader output variable.                                                   |
| [**D3D10\_SHADER\_DEBUG\_SCOPE\_INFO**](/windows/win32/d3d10_1shader/ns-d3d10_1shader-_d3d10_shader_debug_scope_info?branch=master)         | Contains scope data that maps variable names to debug variables.                      |
| [**D3D10\_SHADER\_DEBUG\_SCOPEVAR\_INFO**](/windows/win32/d3d10_1shader/ns-d3d10_1shader-_d3d10_shader_debug_scopevar_info?branch=master)   | Describes a scope variable.                                                           |
| [**D3D10\_SHADER\_DEBUG\_TOKEN\_INFO**](/windows/win32/d3d10_1shader/ns-d3d10_1shader-_d3d10_shader_debug_token_info?branch=master)         | Gives the source location for a shader element.                                       |
| [**D3D10\_SHADER\_DEBUG\_VARTYPE**](/windows/win32/d3d10_1shader/ne-d3d10_1shader-_d3d10_shader_debug_vartype?branch=master)                | Distinguishes variables from functions in a scope.                                    |
| [**D3D10\_SHADER\_DEBUG\_VAR\_INFO**](/windows/win32/d3d10_1shader/ns-d3d10_1shader-_d3d10_shader_debug_var_info?branch=master)             | Contains information on a variable.                                                   |
| [**D3D10\_SHADER\_INPUT\_BIND\_DESC**](/windows/win32/D3D10Shader/ns-d3d10shader-_d3d10_shader_input_bind_desc?branch=master)           | Describes how a shader resource is bound to a shader input.                           |
| [**D3D10\_SHADER\_MACRO**](/windows/win32/D3D10Shader/?branch=master)                                 | Defines a shader macro.                                                               |
| [**D3D10\_SHADER\_TYPE\_DESC**](/windows/win32/D3D10Shader/ns-d3d10shader-_d3d10_shader_type_desc?branch=master)                        | Describes a shader-variable type.                                                     |
| [**D3D10\_SHADER\_VARIABLE\_DESC**](/windows/win32/D3D10Shader/ns-d3d10shader-_d3d10_shader_variable_desc?branch=master)                | Describes a shader variable.                                                          |
| [**D3D10\_SIGNATURE\_PARAMETER\_DESC**](/windows/win32/D3D10Shader/ns-d3d10shader-_d3d10_signature_parameter_desc?branch=master)        | Describes a shader signature.                                                         |
