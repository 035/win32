# Shader Structures

This section contains information about the following shader structures:



| Structures                                                                         | Description                                                                           |
|------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**D3D10\_SHADER\_BUFFER\_DESC**](d3d10-shader-buffer-desc.md)                    | Describes a shader-constant buffer or a shader-texture buffer.                        |
| [**D3D10\_SHADER\_DESC**](d3d10-shader-desc.md)                                   | Describes a shader.                                                                   |
| [**D3D10\_SHADER\_DEBUG\_FILE\_INFO**](d3d10-shader-debug-file-info.md)           | Describes files included by a shader.                                                 |
| [**D3D10\_SHADER\_DEBUG\_INFO**](d3d10-shader-debug-info.md)                      | Describes the format of the ID3D10Blob Interface returned by D3D10GetShaderDebugInfo. |
| [**D3D10\_SHADER\_DEBUG\_INPUT\_INFO**](d3d10-shader-debug-input-info.md)         | Describes a shader input.                                                             |
| [**D3D10\_SHADER\_DEBUG\_INST\_INFO**](d3d10-shader-debug-inst-info.md)           | Contains instruction data.                                                            |
| [**D3D10\_SHADER\_DEBUG\_OUTPUTREG\_INFO**](d3d10-shader-debug-outputreg-info.md) | Describes a shader output register.                                                   |
| [**D3D10\_SHADER\_DEBUG\_OUTPUTVAR**](d3d10-shader-debug-outputvar.md)            | Describes a shader output variable.                                                   |
| [**D3D10\_SHADER\_DEBUG\_SCOPE\_INFO**](d3d10-shader-debug-scope-info.md)         | Contains scope data that maps variable names to debug variables.                      |
| [**D3D10\_SHADER\_DEBUG\_SCOPEVAR\_INFO**](d3d10-shader-debug-scopevar-info.md)   | Describes a scope variable.                                                           |
| [**D3D10\_SHADER\_DEBUG\_TOKEN\_INFO**](d3d10-shader-debug-token-info.md)         | Gives the source location for a shader element.                                       |
| [**D3D10\_SHADER\_DEBUG\_VARTYPE**](d3d10-shader-debug-vartype.md)                | Distinguishes variables from functions in a scope.                                    |
| [**D3D10\_SHADER\_DEBUG\_VAR\_INFO**](d3d10-shader-debug-var-info.md)             | Contains information on a variable.                                                   |
| [**D3D10\_SHADER\_INPUT\_BIND\_DESC**](d3d10-shader-input-bind-desc.md)           | Describes how a shader resource is bound to a shader input.                           |
| [**D3D10\_SHADER\_MACRO**](d3d10-shader-macro.md)                                 | Defines a shader macro.                                                               |
| [**D3D10\_SHADER\_TYPE\_DESC**](d3d10-shader-type-desc.md)                        | Describes a shader-variable type.                                                     |
| [**D3D10\_SHADER\_VARIABLE\_DESC**](d3d10-shader-variable-desc.md)                | Describes a shader variable.                                                          |
| [**D3D10\_SIGNATURE\_PARAMETER\_DESC**](d3d10-signature-parameter-desc.md)        | Describes a shader signature.                                                         |
