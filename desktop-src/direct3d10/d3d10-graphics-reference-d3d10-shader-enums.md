# Shader Enumerations

This section contains information about the following shader enumerations:



| Enumeration                                                             | Description                                                                               |
|-------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**D3D10\_CBUFFER\_TYPE**](/windows/desktop/api/D3D10Shader/)                      | These flags identify constant-buffer-data types.                                          |
| [**D3D10\_INCLUDE\_TYPE**](/windows/desktop/api/D3D10Shader/)                      | These flags identify the location of an include file.                                     |
| [**D3D10\_PRIMITIVE**](/windows/desktop/api/D3D10/)                             | These flags identify primitive types for a Direct3D 10.1 device.                          |
| [**D3D10\_SHADER\_DEBUG\_REGTYPE**](/windows/desktop/api/D3D10_1Shader/ne-d3d10_1shader-_d3d10_shader_debug_regtype)     | Shader register types.                                                                    |
| [**D3D10\_SHADER\_DEBUG\_SCOPETYPE**](/windows/desktop/api/d3d10_1shader/ne-d3d10_1shader-_d3d10_shader_debug_scopetype) | Scope types.                                                                              |
| [**D3D10\_NAME**](/windows/desktop/api/D3D10Shader/)                                       | These flags identify shader parameters that use system-value semantics.                   |
| [**D3D10\_SHADER\_CBUFFER\_FLAGS**](/windows/desktop/api/D3D10Shader/)     | These flags identify constant-buffer properties.                                          |
| [**D3D10\_SHADER\_INPUT\_FLAGS**](/windows/desktop/api/D3D10Shader/)         | These flags identify shader-input properties.                                             |
| [**D3D10\_SHADER\_INPUT\_TYPE**](/windows/desktop/api/D3D10Shader/)           | These flags identify a shader-resource type.                                              |
| [**D3D10\_SHADER\_VARIABLE\_CLASS**](/windows/desktop/api/D3D10Shader/)   | These flags identify the shader-variable class.                                           |
| [**D3D10\_SHADER\_VARIABLE\_FLAGS**](/windows/desktop/api/D3D10Shader/)   | These flags identify a shader-variable property.                                          |
| [**D3D10\_SHADER\_VARIABLE\_TYPE**](/windows/desktop/api/D3D10Shader/)     | These flags identify the shader-variable type.                                            |
| [**D3D10\_SRV\_DIMENSION**](/windows/desktop/api/D3D10/)                    | These flags identify the type of resource that will be viewed for a Direct3D 10.0 device. |
| [**D3D10\_SRV\_DIMENSION1**](/windows/desktop/api/D3D10_1/)                  | These flags identify the type of resource that will be viewed for a Direct3D 10.1 device. |
