# Shader Enumerations

This section contains information about the following shader enumerations:



| Enumeration                                                             | Description                                                                               |
|-------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**D3D10\_CBUFFER\_TYPE**](d3d10-cbuffer-type.md)                      | These flags identify constant-buffer-data types.                                          |
| [**D3D10\_INCLUDE\_TYPE**](d3d10-include-type.md)                      | These flags identify the location of an include file.                                     |
| [**D3D10\_PRIMITIVE**](d3d10-primitive.md)                             | These flags identify primitive types for a Direct3D 10.1 device.                          |
| [**D3D10\_SHADER\_DEBUG\_REGTYPE**](d3d10-shader-debug-regtype.md)     | Shader register types.                                                                    |
| [**D3D10\_SHADER\_DEBUG\_SCOPETYPE**](d3d10-shader-debug-scopetype.md) | Scope types.                                                                              |
| [**D3D10\_NAME**](d3d10-name.md)                                       | These flags identify shader parameters that use system-value semantics.                   |
| [**D3D10\_SHADER\_CBUFFER\_FLAGS**](d3d10-shader-cbuffer-flags.md)     | These flags identify constant-buffer properties.                                          |
| [**D3D10\_SHADER\_INPUT\_FLAGS**](d3d10-shader-input-flags.md)         | These flags identify shader-input properties.                                             |
| [**D3D10\_SHADER\_INPUT\_TYPE**](d3d10-shader-input-type.md)           | These flags identify a shader-resource type.                                              |
| [**D3D10\_SHADER\_VARIABLE\_CLASS**](d3d10-shader-variable-class.md)   | These flags identify the shader-variable class.                                           |
| [**D3D10\_SHADER\_VARIABLE\_FLAGS**](d3d10-shader-variable-flags.md)   | These flags identify a shader-variable property.                                          |
| [**D3D10\_SHADER\_VARIABLE\_TYPE**](d3d10-shader-variable-type.md)     | These flags identify the shader-variable type.                                            |
| [**D3D10\_SRV\_DIMENSION**](d3d10-srv-dimension.md)                    | These flags identify the type of resource that will be viewed for a Direct3D 10.0 device. |
| [**D3D10\_SRV\_DIMENSION1**](d3d10-srv-dimension1.md)                  | These flags identify the type of resource that will be viewed for a Direct3D 10.1 device. |
