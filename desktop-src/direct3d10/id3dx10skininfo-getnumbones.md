# ID3DX10SkinInfo::GetNumBones method

Get the number of bones in ID3DX10SkinInfo.

## Syntax


```C++

```



## Parameters

This method has no parameters.

## Return value

Type: **[**UINT**](https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46)**
