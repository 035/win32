# Core Enumerations

This section contains information about the following core enumerations:



| Enumeration                                                               | Description                                                                                                                                         |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**D3D10\_ASYNC\_GETDATA\_FLAG**](d3d10-async-getdata-flag.md)           | Optional flags that control what happens when data is retrieved by calling [**ID3D10Asynchronous::GetData**](id3d10asynchronous-getdata.md).       |
| [**D3D10\_BLEND**](d3d10-blend.md)                                       | Blend options.                                                                                                                                      |
| [**D3D10\_BLEND\_OP**](d3d10-blend-op.md)                                | Blending operations between source and destination pixels.                                                                                          |
| [**D3D10\_CLEAR\_FLAG**](d3d10-clear-flag.md)                            | Specifies which parts of the depth stencil to clear.                                                                                                |
| [**D3D10\_COLOR\_WRITE\_ENABLE**](d3d10-color-write-enable.md)           | Color writing masks.                                                                                                                                |
| [**D3D10\_COMPARISON\_FUNC**](d3d10-comparison-func.md)                  | Comparison functions.                                                                                                                               |
| [**D3D10\_COUNTER**](d3d10-counter.md)                                   | Types of performance counters.                                                                                                                      |
| [**D3D10\_COUNTER\_TYPE**](d3d10-counter-type.md)                        | Data type of a counter.                                                                                                                             |
| [**D3D10\_CREATE\_DEVICE\_FLAG**](d3d10-create-device-flag.md)           | Device creation flags.                                                                                                                              |
| [**D3D10\_CULL\_MODE**](d3d10-cull-mode.md)                              | Rasterizer cull modes.                                                                                                                              |
| [**D3D10\_DEPTH\_WRITE\_MASK**](d3d10-depth-write-mask.md)               | Determines which portion of the depth stencil is writable.                                                                                          |
| [**D3D10\_DEVICE\_STATE\_TYPES**](d3d10-device-state-types.md)           | Used in [**ID3D10StateBlock Interface**](id3d10stateblock.md) function calls.                                                                      |
| [**D3D10\_DRIVER\_TYPE**](d3d10-driver-type.md)                          | Type of device driver.                                                                                                                              |
| [**D3D10\_FEATURE\_LEVEL1**](d3d10-feature-level1.md)                    | The version of hardware acceleration.                                                                                                               |
| [**D3D10\_FILL\_MODE**](d3d10-fill-mode.md)                              | Rasterizer fill modes.                                                                                                                              |
| [**D3D10\_FILTER**](d3d10-filter.md)                                     | Types of sampler filters.                                                                                                                           |
| [**D3D10\_FILTER\_TYPE**](d3d10-filter-type.md)                          | Types of texture-sampling filters.                                                                                                                  |
| [**D3D10\_FORMAT\_SUPPORT**](d3d10-format-support.md)                    | Which resources are supported for a given format and given device. See [**ID3D10Device::CheckFormatSupport**](id3d10device-checkformatsupport.md). |
| [**D3D10\_INPUT\_CLASSIFICATION**](d3d10-input-classification.md)        | Types of input data.                                                                                                                                |
| [**D3D10\_MESSAGE\_CATEGORY**](d3d10-message-category.md)                | Categories of debug messages.                                                                                                                       |
| [**D3D10\_MESSAGE\_ID**](d3d10-message-id.md)                            | A debug message's unique ID.                                                                                                                        |
| [**D3D10\_MESSAGE\_SEVERITY**](d3d10-message-severity.md)                | Severity of a debug message.                                                                                                                        |
| [**D3D10\_PRIMITIVE\_TOPOLOGY**](d3d10-primitive-topology.md)            | Types of primitive topology or the way a primitive's data is arranged.                                                                              |
| [**D3D10\_QUERY**](d3d10-query.md)                                       | Types of queries.                                                                                                                                   |
| [**D3D10\_QUERY\_MISC\_FLAG**](d3d10-query-misc-flag.md)                 | Flags that describe miscellaneous query behavior.                                                                                                   |
| [**D3D10\_RAISE\_FLAG**](d3d10-raise-flag.md)                            | Flags that indicate how to handle internal driver errors.                                                                                           |
| [**D3D10\_REGISTER\_COMPONENT\_TYPE**](d3d10-register-component-type.md) | The register component types, usually used in [**D3D10\_SIGNATURE\_PARAMETER\_DESC**](d3d10-signature-parameter-desc.md).                          |
| [**D3D10\_RESOURCE\_RETURN\_TYPE**](d3d10-resource-return-type.md)       | The return type of a resource. See [**D3D10\_SHADER\_INPUT\_BIND\_DESC**](d3d10-shader-input-bind-desc.md).                                        |
| [**D3D10\_STENCIL\_OP**](d3d10-stencil-op.md)                            | Stencil operations.                                                                                                                                 |
| [**D3D10\_TEXTURE\_ADDRESS\_MODE**](d3d10-texture-address-mode.md)       | Actions to perform when a texture coordinate is outside of the boundaries of a texture.                                                             |
| [**D3D10\_TEXTURECUBE\_FACE**](d3d10-texturecube-face.md)                | The different faces of a cube texture.                                                                                                              |
