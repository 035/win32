# Core Structures

This section contains information about the following core structures:



| Structures                                                                               | Description                                                          |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------|
| [**D3D10\_BLEND\_DESC**](/windows/win32/D3D10/ns-d3d10-d3d10_blend_desc?branch=master)                                           | Blending options for a Direct3D 10.0 device.                         |
| [**D3D10\_BLEND\_DESC1**](/windows/win32/D3D10_1/ns-d3d10_1-d3d10_blend_desc1?branch=master)                                         | Blending options for a Direct3D 10.1 device.                         |
| [**D3D10\_BOX**](/windows/win32/D3D10/ns-d3d10-d3d10_box?branch=master)                                                          | A rectangular region, often in a texture.                            |
| [**D3D10\_COUNTER\_DESC**](/windows/win32/D3D10/ns-d3d10-d3d10_counter_desc?branch=master)                                       | A performance-counter description.                                   |
| [**D3D10\_COUNTER\_INFO**](/windows/win32/D3D10/ns-d3d10-d3d10_counter_info?branch=master)                                       | Information about the video card's performance counter capabilities. |
| [**D3D10\_DEPTH\_STENCIL\_DESC**](/windows/win32/D3D10/ns-d3d10-d3d10_depth_stencil_desc?branch=master)                          | A depth-stencil description.                                         |
| [**D3D10\_DEPTH\_STENCILOP\_DESC**](/windows/win32/D3D10/ns-d3d10-d3d10_depth_stencilop_desc?branch=master)                      | A depth-stencil operation.                                           |
| [**D3D10\_INFO\_QUEUE\_FILTER**](/windows/win32/d3d10sdklayers/ns-d3d10sdklayers-d3d10_info_queue_filter?branch=master)                            | A debug-message filter.                                              |
| [**D3D10\_INFO\_QUEUE\_FILTER\_DESC**](/windows/win32/d3d10sdklayers/ns-d3d10sdklayers-d3d10_info_queue_filter_desc?branch=master)                 | A debug-message-filter description.                                  |
| [**D3D10\_INPUT\_ELEMENT\_DESC**](/windows/win32/D3D10/ns-d3d10-d3d10_input_element_desc?branch=master)                          | A vertex-buffer element description for an input slot.               |
| [**D3D10\_QUERY\_DATA\_PIPELINE\_STATISTICS**](/windows/win32/D3D10/ns-d3d10-d3d10_query_data_pipeline_statistics?branch=master) | Pipeline query statistics.                                           |
| [**D3D10\_QUERY\_DATA\_SO\_STATISTICS**](/windows/win32/D3D10/ns-d3d10-d3d10_query_data_so_statistics?branch=master)             | Stream-output stage query statistics.                                |
| [**D3D10\_QUERY\_DATA\_TIMESTAMP\_DISJOINT**](/windows/win32/D3D10/ns-d3d10-d3d10_query_data_timestamp_disjoint?branch=master)   | Timestamp query statistics.                                          |
| [**D3D10\_QUERY\_DESC**](/windows/win32/D3D10/ns-d3d10-d3d10_query_desc?branch=master)                                           | A query description.                                                 |
| [**D3D10\_MESSAGE**](/windows/win32/d3d10sdklayers/ns-d3d10sdklayers-d3d10_message?branch=master)                                                  | A debug message for the information queue.                           |
| [**D3D10\_RASTERIZER\_DESC**](/windows/win32/D3D10/ns-d3d10-d3d10_rasterizer_desc?branch=master)                                 | Rasterizer-stage description.                                        |
| [**D3D10\_RECT**](d3d10-rect.md)                                                        | A 2D rectangle.                                                      |
| [**D3D10\_RENDER\_TARGET\_BLEND\_DESC1**](/windows/win32/d3d10_1/ns-d3d10_1-d3d10_render_target_blend_desc1?branch=master)           | Blending options for a Direct3D 10.1 render target.                  |
| [**D3D10\_SAMPLER\_DESC**](/windows/win32/D3D10/ns-d3d10-d3d10_sampler_desc?branch=master)                                       | A sampler description.                                               |
| [**D3D10\_SHADER\_RESOURCE\_VIEW\_DESC**](/windows/win32/d3d10/ns-d3d10-d3d10_shader_resource_view_desc?branch=master)           | Describes a shader-resource view for Direct3D 10.0.                  |
| [**D3D10\_SHADER\_RESOURCE\_VIEW\_DESC1**](/windows/win32/d3d10_1/ns-d3d10_1-d3d10_shader_resource_view_desc1?branch=master)         | Describes a shader-resource view for Direct3D 10.1.                  |
| [**D3D10\_SIGNATURE\_PARAMETER\_DESC**](/windows/win32/D3D10Shader/ns-d3d10shader-_d3d10_signature_parameter_desc?branch=master)              | A shader-parameter description.                                      |
| [**D3D10\_STATE\_BLOCK\_MASK**](/windows/win32/d3d10effect/ns-d3d10effect-_d3d10_state_block_mask?branch=master)                              | A state-block mask.                                                  |
| [**D3D10\_SO\_DECLARATION\_ENTRY**](/windows/win32/D3D10/ns-d3d10-d3d10_so_declaration_entry?branch=master)                      | A vertex-buffer element description for an output slot.              |
| [**D3D10\_VIEWPORT**](/windows/win32/D3D10/ns-d3d10-d3d10_viewport?branch=master)                                                | Viewport dimensions.                                                 |



 

In addition, there is a 2D rectangle structure defined in d3d10.h.


```
typedef RECT D3D10_RECT;
```



For documentation, see [RECT](http://msdn2.microsoft.com/en-us/library/ms536136.aspx).
