# Core Structures

This section contains information about the following core structures:



| Structures                                                                               | Description                                                          |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------|
| [**D3D10\_BLEND\_DESC**](d3d10-blend-desc.md)                                           | Blending options for a Direct3D 10.0 device.                         |
| [**D3D10\_BLEND\_DESC1**](d3d10-blend-desc1.md)                                         | Blending options for a Direct3D 10.1 device.                         |
| [**D3D10\_BOX**](d3d10-box.md)                                                          | A rectangular region, often in a texture.                            |
| [**D3D10\_COUNTER\_DESC**](d3d10-counter-desc.md)                                       | A performance-counter description.                                   |
| [**D3D10\_COUNTER\_INFO**](d3d10-counter-info.md)                                       | Information about the video card's performance counter capabilities. |
| [**D3D10\_DEPTH\_STENCIL\_DESC**](d3d10-depth-stencil-desc.md)                          | A depth-stencil description.                                         |
| [**D3D10\_DEPTH\_STENCILOP\_DESC**](d3d10-depth-stencilop-desc.md)                      | A depth-stencil operation.                                           |
| [**D3D10\_INFO\_QUEUE\_FILTER**](d3d10-info-queue-filter.md)                            | A debug-message filter.                                              |
| [**D3D10\_INFO\_QUEUE\_FILTER\_DESC**](d3d10-info-queue-filter-desc.md)                 | A debug-message-filter description.                                  |
| [**D3D10\_INPUT\_ELEMENT\_DESC**](d3d10-input-element-desc.md)                          | A vertex-buffer element description for an input slot.               |
| [**D3D10\_QUERY\_DATA\_PIPELINE\_STATISTICS**](d3d10-query-data-pipeline-statistics.md) | Pipeline query statistics.                                           |
| [**D3D10\_QUERY\_DATA\_SO\_STATISTICS**](d3d10-query-data-so-statistics.md)             | Stream-output stage query statistics.                                |
| [**D3D10\_QUERY\_DATA\_TIMESTAMP\_DISJOINT**](d3d10-query-data-timestamp-disjoint.md)   | Timestamp query statistics.                                          |
| [**D3D10\_QUERY\_DESC**](d3d10-query-desc.md)                                           | A query description.                                                 |
| [**D3D10\_MESSAGE**](d3d10-message.md)                                                  | A debug message for the information queue.                           |
| [**D3D10\_RASTERIZER\_DESC**](d3d10-rasterizer-desc.md)                                 | Rasterizer-stage description.                                        |
| [**D3D10\_RECT**](d3d10-rect.md)                                                        | A 2D rectangle.                                                      |
| [**D3D10\_RENDER\_TARGET\_BLEND\_DESC1**](d3d10-render-target-blend-desc1.md)           | Blending options for a Direct3D 10.1 render target.                  |
| [**D3D10\_SAMPLER\_DESC**](d3d10-sampler-desc.md)                                       | A sampler description.                                               |
| [**D3D10\_SHADER\_RESOURCE\_VIEW\_DESC**](d3d10-shader-resource-view-desc.md)           | Describes a shader-resource view for Direct3D 10.0.                  |
| [**D3D10\_SHADER\_RESOURCE\_VIEW\_DESC1**](d3d10-shader-resource-view-desc1.md)         | Describes a shader-resource view for Direct3D 10.1.                  |
| [**D3D10\_SIGNATURE\_PARAMETER\_DESC**](d3d10-signature-parameter-desc.md)              | A shader-parameter description.                                      |
| [**D3D10\_STATE\_BLOCK\_MASK**](d3d10-state-block-mask.md)                              | A state-block mask.                                                  |
| [**D3D10\_SO\_DECLARATION\_ENTRY**](d3d10-so-declaration-entry.md)                      | A vertex-buffer element description for an output slot.              |
| [**D3D10\_VIEWPORT**](d3d10-viewport.md)                                                | Viewport dimensions.                                                 |



 

In addition, there is a 2D rectangle structure defined in d3d10.h.


```
typedef RECT D3D10_RECT;
```



For documentation, see [RECT](http://msdn2.microsoft.com/en-us/library/ms536136.aspx).
