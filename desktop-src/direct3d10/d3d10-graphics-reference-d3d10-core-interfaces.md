# Core Interfaces

This section contains information about the following core interfaces:



| Interfaces                                                           | Description                                         |
|----------------------------------------------------------------------|-----------------------------------------------------|
| [**ID3D10BlendState Interface**](/windows/win32/D3D10/nn-d3d10-id3d10blendstate?branch=master)               | Accesses blending state for a Direct3D 10.0 device. |
| [**ID3D10BlendState1 Interface**](/windows/win32/D3D10_1/nn-d3d10_1-id3d10blendstate1?branch=master)             | Accesses blending state for a Direct3D 10.1 device. |
| [**ID3D10DepthStencilState Interface**](/windows/win32/D3D10/nn-d3d10-id3d10depthstencilstate?branch=master) | Accesses depth-stencil state.                       |
| [**ID3D10InputLayout Interface**](/windows/win32/D3D10/?branch=master)             | Accesses pipeline-input data from memory.           |
| [**ID3D10RasterizerState Interface**](/windows/win32/D3D10/nn-d3d10-id3d10rasterizerstate?branch=master)     | Accesses rasterizer state.                          |
| [**ID3D10SamplerState Interface**](/windows/win32/D3D10/nn-d3d10-id3d10samplerstate?branch=master)           | Accesses sampler state.                             |



 



| Interfaces                                                 | Description                                                                          |
|------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**ID3D10Asynchronous Interface**](/windows/win32/D3D10/nn-d3d10-id3d10asynchronous?branch=master) | Retrieves data from the GPU asynchronously.                                          |
| [**ID3D10Blob Interface**](/windows/win32/D3DCommon/nn-d3dcommon-id3d10blob?branch=master)                 | Return data from memory.                                                             |
| [**ID3D10Counter Interface**](/windows/win32/D3D10/nn-d3d10-id3d10counter?branch=master)           | Measures GPU performance.                                                            |
| [**ID3D10Debug Interface**](/windows/win32/D3D10SDKLayers/nn-d3d10sdklayers-id3d10debug?branch=master)               | Enables/disables the debug layer.                                                    |
| [**ID3D10Device Interface**](/windows/win32/D3D10/nn-d3d10-id3d10device?branch=master)             | Represents a virtual adapter for Direct3D 10.0.                                      |
| [**ID3D10Device1 Interface**](/windows/win32/D3D10_1/nn-d3d10_1-id3d10device1?branch=master)           | Represents a virtual adapter for Direct3D 10.1.                                      |
| [**ID3D10DeviceChild Interface**](/windows/win32/D3D10/nn-d3d10-id3d10devicechild?branch=master)   | Accesses data used by a device.                                                      |
| [**ID3D10Include Interface**](/windows/win32/D3D10Shader/?branch=master)           | Provides user-overridable methods for handling include files when loading an effect. |
| [**ID3D10InfoQueue Interface**](/windows/win32/D3D10SDKLayers/nn-d3d10sdklayers-id3d10infoqueue?branch=master)       | Stores, retrieves, and filters debug messages.                                       |
| [**ID3D10Multithread Interface**](/windows/win32/D3D10/nn-d3d10-id3d10multithread?branch=master)   | Accesses multithread settings.                                                       |
| [**ID3D10Predicate Interface**](/windows/win32/D3D10/?branch=master)       | Determines whether geometry should be processed.                                     |
| [**ID3D10Query Interface**](/windows/win32/D3D10/nn-d3d10-id3d10query?branch=master)               | Queries information from the GPU.                                                    |
| [**ID3D10StateBlock Interface**](/windows/win32/d3d10effect/nn-d3d10effect-id3d10stateblock?branch=master)     | Encapsulates render states.                                                          |
| [**ID3D10SwitchToRef Interface**](/windows/win32/D3D10SDKLayers/nn-d3d10sdklayers-id3d10switchtoref?branch=master)   | Switches between a hardware and software device.                                     |
