# Core Interfaces

This section contains information about the following core interfaces:



| Interfaces                                                           | Description                                         |
|----------------------------------------------------------------------|-----------------------------------------------------|
| [**ID3D10BlendState Interface**](id3d10blendstate.md)               | Accesses blending state for a Direct3D 10.0 device. |
| [**ID3D10BlendState1 Interface**](id3d10blendstate1.md)             | Accesses blending state for a Direct3D 10.1 device. |
| [**ID3D10DepthStencilState Interface**](id3d10depthstencilstate.md) | Accesses depth-stencil state.                       |
| [**ID3D10InputLayout Interface**](id3d10inputlayout.md)             | Accesses pipeline-input data from memory.           |
| [**ID3D10RasterizerState Interface**](id3d10rasterizerstate.md)     | Accesses rasterizer state.                          |
| [**ID3D10SamplerState Interface**](id3d10samplerstate.md)           | Accesses sampler state.                             |



 



| Interfaces                                                 | Description                                                                          |
|------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**ID3D10Asynchronous Interface**](id3d10asynchronous.md) | Retrieves data from the GPU asynchronously.                                          |
| [**ID3D10Blob Interface**](id3d10blob.md)                 | Return data from memory.                                                             |
| [**ID3D10Counter Interface**](id3d10counter.md)           | Measures GPU performance.                                                            |
| [**ID3D10Debug Interface**](id3d10debug.md)               | Enables/disables the debug layer.                                                    |
| [**ID3D10Device Interface**](id3d10device.md)             | Represents a virtual adapter for Direct3D 10.0.                                      |
| [**ID3D10Device1 Interface**](id3d10device1.md)           | Represents a virtual adapter for Direct3D 10.1.                                      |
| [**ID3D10DeviceChild Interface**](id3d10devicechild.md)   | Accesses data used by a device.                                                      |
| [**ID3D10Include Interface**](id3d10include.md)           | Provides user-overridable methods for handling include files when loading an effect. |
| [**ID3D10InfoQueue Interface**](id3d10infoqueue.md)       | Stores, retrieves, and filters debug messages.                                       |
| [**ID3D10Multithread Interface**](id3d10multithread.md)   | Accesses multithread settings.                                                       |
| [**ID3D10Predicate Interface**](id3d10predicate.md)       | Determines whether geometry should be processed.                                     |
| [**ID3D10Query Interface**](id3d10query.md)               | Queries information from the GPU.                                                    |
| [**ID3D10StateBlock Interface**](id3d10stateblock.md)     | Encapsulates render states.                                                          |
| [**ID3D10SwitchToRef Interface**](id3d10switchtoref.md)   | Switches between a hardware and software device.                                     |

-   [Shaders](d3d10-graphics-reference-d3d10-shader-interfaces.md)
-   [Effects](d3d10-graphics-reference-effect-interfaces.md)
