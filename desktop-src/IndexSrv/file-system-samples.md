# File System Samples

> [!Note]  
> Indexing Service is no longer supported as of Windows XP and is unavailable for use as of Windows 8. Instead, use [Windows Search](https://msdn.microsoft.com/windows/desktop/6da601c6-3742-40ad-99f2-8817f7f642b3) for client side search and [Microsoft Search Server Express]( http://go.microsoft.com/fwlink/p/?linkid=258445) for server side search.

 

The file system samples illustrate application programming and script writing using the OLE DB Provider for Indexing Service API, OLE DB Helper API, Microsoft ActiveX Data Objects (ADO) API, Admin Helper API, and Query Helper API. The following samples are included in the Platform SDK.

-   [AdvQuery Sample](advquery-sample.md). A complex example of querying using the OLE DB Provider API.
-   [ChgState Sample](chgstate-sample.md). An example of managing the status of Indexing Service using the OLE DB Helper API.
-   [Simple Sample](simple-sample.md). A simple example of querying using the OLE DB Helper API.
-   [SQuery Sample](squery-sample.md). A set of three scripts that are examples of managing and querying using the Helper APIs.
-   [VBAdmin Sample](vbadmin-sample.md). An example of managing Indexing Service using the Admin Helper API.
-   [VBQuery Sample](vbquery-sample.md). An example of querying using the ADO and Query Helper APIs and the Indexing Service query languages.
-   [VJQuery Sample](vjquery-sample.md). An example of querying using the ADO API and SQL.

The section on [Web server samples](web-server-samples.md) contains additional samples of scripting that are web server-oriented.
