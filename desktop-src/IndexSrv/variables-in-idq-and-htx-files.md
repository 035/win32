# Variables in .Idq and .Htx Files

> [!Note]  
> Indexing Service is no longer supported as of Windows XP and is unavailable for use as of Windows 8. Instead, use [Windows Search](https://msdn.microsoft.com/library/windows/desktop/aa965362) for client side search and [Microsoft Search Server Express]( http://go.microsoft.com/fwlink/p/?linkid=258445) for server side search.

 

This topic contains a description of variables that are used in Indexing Service's ISAPI implementation.

-   [Read-Only Variables Available in .Htx Files](read-only-variables-available-in-htx-files.md): Lists and describes read-only variables for .htx files.
-   [Variables in Forms and in .Idq Files](variables-in-forms-and-in-idq-files.md): Lists and explains variables that can be set in forms, in .idq files, or as CGI variables.
-   [String Variables Useful for Formatting Displayed Columns](string-variables-useful-for-formatting-displayed-columns.md): Lists and gives the meanings for variables that you can set in the .idq file but cannot use to refer to other parameters.
-   [Parameter for Use with Bookmarks](parameter-for-use-with-bookmarks.md): Explains how the *CiBookmarkSkipCount* variable, in conjunction with *CiBookmark*, sets an offset from the previous page of the query.
-   [Standard Common Gateway Interface Variables Available](standard-common-gateway-interface-variables-available.md): Lists the variables that are normally set by CGI program execution and that can be referred to in the .idq or .htx files.
-   [Valid Locale Identifiers:](valid-locale-identifiers.md) Shows valid locale identifiers (LCIDs) for use in *CiLocale*.
-   [Recognized Character Set Tags](recognized-character-set-tags.md): Lists the ISO codes currently recognized by Indexing Service and the HTML filter, for use in *CiCodepage*.
