# Language Resources Sample

> [!Note]  
> Indexing Service is no longer supported as of Windows XP and is unavailable for use as of Windows 8. Instead, use [Windows Search](https://msdn.microsoft.com/library/windows/desktop/aa965362) for client side search and [Microsoft Search Server Express]( http://go.microsoft.com/fwlink/p/?linkid=258445) for server side search.

 

The following sample provides an example of custom word breaker and stemmer implementations that extend language resources for Indexing Service. The table gives the name of the sample and the tool, language, and API that the sample employs.



| Name                                      | Tool          | Language   | API                                                                       |
|-------------------------------------------|---------------|------------|---------------------------------------------------------------------------|
| [Lrsample](language-resource-samples.md) | Visual Studio | Visual C++ | [IWordBreaker](iwordbreaker.md) and [IStemmer](istemmer.md) interfaces. |
