# Webhits Parameters

> [!Note]  
> Indexing Service is no longer supported as of Windows XP and is unavailable for use as of Windows 8. Instead, use [Windows Search](https://msdn.microsoft.com/library/windows/desktop/aa965362) for client side search and [Microsoft Search Server Express]( http://go.microsoft.com/fwlink/p/?linkid=258445) for server side search.

 

This section lists and describes the parameters that can help you fine-tune a query with the hit-highlighter (Webhits.dll).

-   [CiBold](cibold.md)
-   [CiCodepage](cicodepage.md)
-   [CiHiliteColor](cihilitecolor.md)
-   [CiHiliteType](cihilitetype.md)
-   [CiItalic](ciitalic.md)
-   [CiLocale](cilocale.md)
-   [CiMaxLineLength](cimaxlinelength.md)
-   [CiQueryFile](ciqueryfile.md)
-   [CiRestriction](cirestriction.md)
-   [CiUserParamNumber](ciuserparamnumber.md)
-   [CiWebhitsFile](ciwebhitsfile.md)

> [!Note]  
> CiBeginHilite and CiEndHilite will no longer be supported in versions of Index Server later than 2.0, due to the potential security risk posed by allowing HTML tags to be passed as the arguments for these parameters.
