# Structures

> [!Note]  
> Indexing Service is no longer supported as of Windows XP and is unavailable for use as of Windows 8. Instead, use [Windows Search](https://msdn.microsoft.com/library/windows/desktop/aa965362) for client side search and [Microsoft Search Server Express]( http://go.microsoft.com/fwlink/p/?linkid=258445) for server side search.

 

Indexing Service defines several structures for use with interfaces and functions. The structures include:

-   [**CI\_STATE**](/windows/desktop/api/Ntquery/ns-ntquery-_ci_state)
-   [**CIPROPERTYDEF**](/windows/desktop/api/Ntquery/ns-ntquery-tagcipropertydef)
-   [**FILTERREGION**](/windows/desktop/api/Filter/ns-filter-tagfilterregion)
-   [**FULLPROPSPEC**](/windows/desktop/api/Filter/ns-filter-tagfullpropspec)
-   [**STAT\_CHUNK**](/windows/desktop/api/Filter/ns-filter-tagstat_chunk)
-   [**TEXT\_SOURCE**](text-source.md)
