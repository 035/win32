# Return Values

> [!Note]  
> Indexing Service is no longer supported as of Windows XP and is unavailable for use as of Windows 8. Instead, use [Windows Search](https://msdn.microsoft.com/windows/desktop/6da601c6-3742-40ad-99f2-8817f7f642b3) for client side search and [Microsoft Search Server Express]( http://go.microsoft.com/fwlink/p/?linkid=258445) for server side search.

 

Indexing Service produces specific return values and error codes that relate to its processing of documents and catalogs, including filtering, word breaking, and querying. The range of return values for Indexing Service is 0x*HHHH*1600 to 0x*HHHH*1850, where *HHHH* is a hexadecimal value that depends on the severity code and facility code of each individual return value.

The following topics enumerate and describe the return values.

-   [Query-Execution Values](query-execution-values.md)
-   [Query-Parsing Values](query-parsing-values.md)
-   [Content-Indexing Values](content-indexing-values.md)
-   [Hit-Searching Values](hit-searching-values.md)
-   [Filter-Interface Values](filter-interface-values.md)
-   [Word-Breaking Values](word-breaking-values.md)
-   [Helper-Function Values](helper-function-values.md)
