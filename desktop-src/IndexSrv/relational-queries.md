# Relational Queries

> [!Note]  
> Indexing Service is no longer supported as of Windows XP and is unavailable for use as of Windows 8. Instead, use [Windows Search](https://msdn.microsoft.com/library/windows/desktop/aa965362) for client side search and [Microsoft Search Server Express]( http://go.microsoft.com/fwlink/p/?linkid=258445) for server side search.

 

Relational queries are property-value queries that use relational and bitwise-comparison operators to compare the values of value-type properties to specified values. Under the following topics, this section describes the use of relational queries.

-   [Relational Operators](relational-operators.md)
-   [Relational Queries with Vector Properties](relational-queries-with-vector-properties.md)
-   [Bitwise-Comparison Operators](bitwise-comparison-operators.md)
-   [Examples of Relational Queries](examples-of-relational-queries.md)
