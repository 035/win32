# NoiseFile

> [!Note]  
> Indexing Service is no longer supported as of Windows XP and is unavailable for use as of Windows 8. Instead, use [Windows Search](https://msdn.microsoft.com/library/windows/desktop/aa965362) for client side search and [Microsoft Search Server Express]( http://go.microsoft.com/fwlink/p/?linkid=258445) for server side search.

 

The **NoiseFile** entry contains the file name of the noise-word list for the locale corresponding to the language specified by the language subkey.

### Summary



|       |             |
|-------|-------------|
| Type: | **REG\_SZ** |



 

### Remarks

The value of the **NoiseFile** entry must be either an absolute path name, or the file must be located in the %SystemRoot% \\System32 directory.
