# Property Sets

> [!Note]  
> Indexing Service is no longer supported as of Windows XP and is unavailable for use as of Windows 8. Instead, use [Windows Search](https://msdn.microsoft.com/library/windows/desktop/aa965362) for client side search and [Microsoft Search Server Express]( http://go.microsoft.com/fwlink/p/?linkid=258445) for server side search.

 

Indexing Service provides several property sets to use with [OLE DB Provider for Indexing Service](ole-db-provider-for-indexing-service.md). In addition, Indexing Service uses several COM property sets.

For examples of how to use these property sets, see the [Simple sample](simple-sample.md) and the [AdvQuery sample](advquery-sample.md) in the Platform SDK directories mssdk\\samples\\winbase\\indexing\\simple and mssdk\\samples\\winbase\\indexing\\advquery.

The following table lists the globally unique identifier (GUID) and a short description of each property set:



| GUID identifier                                                       | Description                                                                                                      |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [DBPROPSET\_CIFRMWRKCORE\_EXT](dbpropset-cifrmwrkcore-ext.md)        | Extended core properties including the machine name and class identifier (CLSID) of clients of indexing service. |
| [DBPROPSET\_FSCIFRMWRK\_EXT](dbpropset-fscifrmwrk-ext.md)            | Extended file system properties including catalog and scope properties.                                          |
| [DBPROPSET\_MSIDXS\_ROWSETEXT](dbpropset-msidxs-rowsetext.md)        | Rowset properties including the query string and its locale.                                                     |
| [DBPROPSET\_QUERYEXT](dbpropset-queryext.md)                         | Extended query properties including query control properties.                                                    |
| [PSGUID\_AUDIO](psguid-audio.md)                                     | Audio property set.                                                                                              |
| [PSGUID\_DRM](psguid-drm.md)                                         | Digital rights property set.                                                                                     |
| [PSGUID\_IMAGESUMMARYINFORMATION](psguid-imagesummaryinformation.md) | Image information property set.                                                                                  |
| [PSGUID\_MUSIC](psguid-music.md)                                     | Music property set.                                                                                              |
| [PSGUID\_QUERY](psguid-query.md)                                     | Query properties including query result properties.                                                              |
| [PSGUID\_STORAGE](psguid-storage.md)                                 | Storage properties including file properties.                                                                    |
| [PSGUID\_VIDEO](psguid-video.md)                                     | Video property set.                                                                                              |
