# Function Discovery Interfaces

\[Function Discovery is available for use in the following versions of Windows: Windows Server 2012, Windows 8, Windows Server 2008 R2, Windows 7, Windows Server 2008, and Windows Vista. It may be altered or unavailable in subsequent versions.\]

The Function Discovery client programming interface defines and uses the following interfaces:

-   [**IFunctionDiscovery**](/windows/desktop/api/FunctionDiscoveryAPI/nn-functiondiscoveryapi-ifunctiondiscovery)
-   [**IFunctionDiscoveryNotification**](/windows/desktop/api/FunctionDiscoveryNotification/nn-functiondiscoveryapi-ifunctiondiscoverynotification)
-   [**IFunctionInstance**](/windows/desktop/api/FunctionDiscoveryAPI/nn-functiondiscoveryapi-ifunctioninstance)
-   [**IFunctionInstanceCollection**](/windows/desktop/api/FunctionDiscoveryAPI/nn-functiondiscoveryapi-ifunctioninstancecollection)
-   [**IFunctionInstanceCollectionQuery**](/windows/desktop/api/FunctionDiscoveryAPI/nn-functiondiscoveryapi-ifunctioninstancecollectionquery)
-   [**IFunctionInstanceQuery**](/windows/desktop/api/FunctionDiscoveryAPI/nn-functiondiscoveryapi-ifunctioninstancequery)
