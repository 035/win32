# Function Discovery Providers

\[Function Discovery is available for use in the following versions of Windows: Windows Server 2012, Windows 8, Windows Server 2008 R2, Windows 7, Windows Server 2008, and Windows Vista. It may be altered or unavailable in subsequent versions.\]

## Purpose

The Function Discovery providers programming interface allows a component's discovery provider to be integrated into the Function Discovery system.

## Developer audience

Independent Hardware Vendors (IHV) should implement a discovery provider that will enable their devices to be integrated into the Function Discovery system.

## Run-time requirements

Function Discovery is a component of the Windows Connect Now technologies built into Windows Server 2008 and Windows Vista.
