# Function discovery provider Interfaces

\[Function Discovery is available for use in the following versions of Windows: Windows Server 2012, Windows 8, Windows Server 2008 R2, Windows 7, Windows Server 2008, and Windows Vista. It may be altered or unavailable in subsequent versions.\]

The Function discovery provider programming interface defines and uses the following interfaces:

-   [**IFunctionDiscoveryProvider**](/windows/desktop/api/FunctionDiscoveryProvider/nn-functiondiscoveryprovider-ifunctiondiscoveryprovider)
-   [**IFunctionDiscoveryProviderFactory**](/windows/desktop/api/FunctionDiscoveryProvider/nn-functiondiscoveryprovider-ifunctiondiscoveryproviderfactory)
-   [**IFunctionDiscoveryProviderQuery**](/windows/desktop/api/FunctionDiscoveryProvider/nn-functiondiscoveryprovider-ifunctiondiscoveryproviderquery)
-   [**IFunctionDiscoveryServiceProvider**](/windows/desktop/api/FunctionDiscoveryProvider/nn-functiondiscoveryprovider-ifunctiondiscoveryserviceprovider)
-   [**IProviderProperties**](/windows/desktop/api/FunctionDiscoveryProvider/nn-functiondiscoveryprovider-iproviderproperties)
-   [**IProviderPropertyConstraintCollection**](/windows/desktop/api/FunctionDiscoveryProvider/nn-functiondiscoveryprovider-iproviderpropertyconstraintcollection)
-   [**IProviderPublishing**](/windows/desktop/api/FunctionDiscoveryProvider/nn-functiondiscoveryprovider-iproviderpublishing)
-   [**IProviderQueryConstraintCollection**](/windows/desktop/api/FunctionDiscoveryProvider/nn-functiondiscoveryprovider-iproviderqueryconstraintcollection)
