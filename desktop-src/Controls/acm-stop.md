# ACM\_STOP message

Stops playing an AVI clip in an animation control. You can send this message explicitly or by using the [**Animate\_Stop**](animate-stop.md) macro.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns nonzero if successful, or zero otherwise.
