# RB\_GETBARHEIGHT message

Retrieves the height of the rebar control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns a **UINT** value that represents the height, in pixels, of the control.
