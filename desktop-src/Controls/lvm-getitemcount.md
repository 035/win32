# LVM\_GETITEMCOUNT message

Retrieves the number of items in a list-view control. You can send this message explicitly or by using the [**ListView\_GetItemCount**](listview-getitemcount.md) macro.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns the number of items.
