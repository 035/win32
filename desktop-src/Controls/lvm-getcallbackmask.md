# LVM\_GETCALLBACKMASK message

Gets the callback mask for a list-view control. You can send this message explicitly or by using the [**ListView\_GetCallbackMask**](/windows/win32/Commctrl/nf-commctrl-listview_getcallbackmask?branch=master) macro.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns the callback mask.
