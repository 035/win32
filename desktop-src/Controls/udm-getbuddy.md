# UDM\_GETBUDDY message

Retrieves the handle to the current buddy window.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

The return value is the handle to the current buddy window.
