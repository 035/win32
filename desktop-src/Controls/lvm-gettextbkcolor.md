# LVM\_GETTEXTBKCOLOR message

Retrieves the text background color of a list-view control. You can send this message explicitly or by using the [**ListView\_GetTextBkColor**](/windows/win32/Commctrl/nf-commctrl-listview_gettextbkcolor?branch=master) macro.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns the background color of the text.
