# EM\_GETHYPHENATEINFO message

Retrieves information about hyphenation for a Microsoft Rich Edit control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

The [**HYPHENATEINFO**](hyphenateinfo.md) structure.

</dd> <dt>

*lParam* 
</dt> <dd>

Not used; must be zero.

</dd> </dl>
