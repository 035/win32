# TCM\_DELETEALLITEMS message

Removes all items from a tab control. You can send this message explicitly or by using the [**TabCtrl\_DeleteAllItems**](/windows/win32/Commctrl/nf-commctrl-tabctrl_deleteallitems?branch=master) macro.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns **TRUE** if successful, or **FALSE** otherwise.
