# Texture1D::SampleLevel methods

Samples a [**Texture1D**](sm5-object-texture1d.md) on the specified mipmap level.

### Overload list



| Method                                                                                 | Description                                                                                        |
|:---------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**SampleLevel(S,float,float,int)**](dx-graphics-hlsl-to-samplelevel.md)              | Samples a texture on the specified mipmap level.<br/>                                        |
| [**SampleLevel(S,float,float,int,uint)**](t1d-samplelevel-s-float-float-int-uint-.md) | Samples a texture on the specified mipmap level and returns status about the operation.<br/> |
