# nop - vs

No operation is performed.

## Syntax



| nop |
|-----|



 

## Remarks



| Vertex shader versions | 1\_1 | 2\_0 | 2\_x | 2\_sw | 3\_0 | 3\_sw |
|------------------------|------|------|------|-------|------|-------|
| nop                    | x    | x    | x    | x     | x    | x     |



 

This instruction performs a no-op, or no operation. The syntax for calling it is as follows:


```
nop
```
