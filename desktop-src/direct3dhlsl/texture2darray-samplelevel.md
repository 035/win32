# Texture2DArray::SampleLevel methods

Samples a [**Texture2DArray**](sm5-object-texture2darray.md) on the specified mipmap level.

### Overload list



| Method                                                                                      | Description                                                                                        |
|:--------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**SampleLevel(S,float,float,int)**](dx-graphics-hlsl-to-samplelevel.md)                   | Samples a texture on the specified mipmap level.<br/>                                        |
| [**SampleLevel(S,float,float,int,uint)**](t2darray-samplelevel-s-float-float-int-uint-.md) | Samples a texture on the specified mipmap level and returns status about the operation.<br/> |
