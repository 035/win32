# break - vs

Break out of the current loop at the nearest [endloop - vs](endloop---vs.md) or [endrep - vs](endrep---vs.md).

## Syntax



| break |
|-------|



 

## Remarks

This instruction is supported in the following versions.



| Vertex shader versions | 1\_1 | 2\_0 | 2\_x | 2\_sw | 3\_0 | 3\_sw |
|------------------------|------|------|------|-------|------|-------|
| break                  |      |      | x    | x     | x    | x     |
