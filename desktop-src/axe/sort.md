# Sort element

The value of the Sort node gives the programmatic name of the metric the test case table should initially be sorted by.

## Usage

``` syntax
<Sort/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                           | Description                                                                              |
|---------------------------------------------------|------------------------------------------------------------------------------------------|
| [**TestCaseTable**](testcasetable.md)<br/> | The test case table node defines a table of test cases on a page.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
