# Normal (JobKind) element

The job is a normal job; that is, it is neither a diagnostic module nor a job that tests battery life.

## Usage

``` syntax
<Normal (JobKind)/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements

There are no parent elements.
