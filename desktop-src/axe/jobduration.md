# JobDuration element

The duration of the job in milliseconds.

## Usage

``` syntax
<JobDuration>
  text
</JobDuration>
```

## Attributes

There are no attributes.

## Text value

The duration of the job in milliseconds.

## Child elements

There are no child elements.
