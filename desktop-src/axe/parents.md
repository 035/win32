# Parents element

A collection of parents of the [**TestCase**](testcase.md) to support nested report structures.

## Usage

``` syntax
<Parents>
  child elements
</Parents>
```

## Attributes

There are no attributes.

## Child elements



| Element                             | Description                                                                 |
|-------------------------------------|-----------------------------------------------------------------------------|
| [**Parent**](parent.md)<br/> | A single parent of the [**TestCase**](testcase.md).<br/> <br/> |



### Child element sequence

``` syntax
Parent
```

## Parent elements



| Element                                 | Description                                |
|-----------------------------------------|--------------------------------------------|
| [**TestCase**](testcase.md)<br/> | An individual test.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
