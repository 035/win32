# Execution Assessment Interfaces

The interfaces of the [AXE Execution Assessment API](axe-execution-assessment-api.md) are implemented with the **struct** keyword and are documented in the [Execution Assessment Structures](execution-assessment-structures.md) section. To use the AXE Execution Assessment API initialize AXE using the [**AxeInitAssessment**](axeinitassessment.md) function.
