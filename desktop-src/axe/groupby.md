# GroupBy element

The name of one of the columns in the table that specifies how the test cases are initially grouped.

## Usage

``` syntax
<GroupBy>
  text
</GroupBy>
```

## Attributes

There are no attributes.

## Text value

The name of one of the columns in the table that specifies how the test cases are initially grouped.

## Child elements

There are no child elements.

## Parent elements



| Element                                           | Description                                                                              |
|---------------------------------------------------|------------------------------------------------------------------------------------------|
| [**TestCaseTable**](testcasetable.md)<br/> | The test case table node defines a table of test cases on a page.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
