# AuthorUrl element

This URL refers to the main web site or page for the Assessment s author.

## Usage

``` syntax
<AuthorUrl>
  text
</AuthorUrl>
```

## Attributes

There are no attributes.

## Text value

A URL to the web site for the assessment s author.

## Child elements

There are no child elements.
