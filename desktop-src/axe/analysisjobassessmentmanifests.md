# AnalysisJobAssessmentManifests element

The assessment manifests for the analysis jobs that were run against the assessment results. See [**AxeJobResults**](axejobresults.md) for a description of how AXE aggregates assessment result files into the job results file.

## Usage

``` syntax
<AnalysisJobAssessmentManifests/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements

There are no parent elements.

## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
