# ParameterDefinitions element

A collection of [**ParameterDefinition**](parameterdefinition.md) elements.

## Usage

``` syntax
<ParameterDefinitions>
  child elements
</ParameterDefinitions>
```

## Attributes

There are no attributes.

## Child elements



| Element                                                       | Description                                                                       |
|---------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [**ParameterDefinition**](parameterdefinition.md)<br/> | Information that fully describes an assessment parameter. <br/> <br/> |



### Child element sequence

``` syntax
ParameterDefinition
```

## Parent elements



| Element                                                     | Description                                             |
|-------------------------------------------------------------|---------------------------------------------------------|
| [**AssessmentManifest**](https://msdn.microsoft.com/library/windows/desktop/hh449288)<br/> | The root node of the assessment.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
