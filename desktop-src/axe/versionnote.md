# VersionNote element

This element contains a note that is associated with the version of this assessment.

## Usage

``` syntax
<VersionNote>
  text
</VersionNote>
```

## Attributes

There are no attributes.

## Text value

A note that is associated with the version of this assessment. The maximum length is 200 characters. This information can be used by the UX of the solution to differentiate between multiple assessment versions.

## Child elements

There are no child elements.
