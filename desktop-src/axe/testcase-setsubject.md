# TestCase::SetSubject methods

Sets the [**Subject**](subject.md) of the **TestCase**.

### Overload list



| Method                                                                      | Description                                                         |
|:----------------------------------------------------------------------------|:--------------------------------------------------------------------|
| [**SetSubject(className,subject)**](testcase-setsubject-1.md)              | Sets the [**Subject**](subject.md) of the **TestCase**.<br/> |
| [**SetSubject(className,instanceName,subject)**](testcase-setsubject-2.md) | Sets the [**Subject**](subject.md) of the **TestCase**.<br/> |



## Remarks

The **TestCase** objects hold information from the **Iteration/TestCases/TestCase** elements.

The **Subject** holds information from element **TestCase/Subject**.
