# ZeroIsSuccess element

If present, AXE should interpret the process exit value of zero to mean the assessment was successful.

## Usage

``` syntax
<ZeroIsSuccess/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                 | Description                                                                                |
|---------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**ExitValueMeaning**](exitvaluemeaning.md)<br/> | Indicates whether the exit value means success or is a **HRESULT**.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
