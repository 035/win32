# MetricValueCollection::AddItem methods

Creates a [**MetricValue**](metricvalue-struct.md) and adds it to the **MetricValueCollection**

### Overload list



| Method                                                                                 | Description                                                                   |
|:---------------------------------------------------------------------------------------|:------------------------------------------------------------------------------|
| [**Additem(programmaticName,value,metricValue)**](metricvaluecollection-additem-1.md) | Creates and adds a **MetricValue** whose value is a **LPCWSTR**.<br/>   |
| [**Additem(programmaticName,value,metricValue)**](metricvaluecollection-additem-2.md) | Creates and adds a **MetricValue** whose value is an **INT**.<br/>      |
| [**Additem(programmaticName,value,metricValue)**](metricvaluecollection-additem-3.md) | Creates and adds a **MetricValue** whose value is a **UINT**.<br/>      |
| [**Additem(programmaticName,value,metricValue)**](metricvaluecollection-additem-4.md) | Creates and adds a **MetricValue** whose value is a **LONGLONG**.<br/>  |
| [**Additem(programmaticName,value,metricValue)**](metricvaluecollection-additem-5.md) | Creates and adds a **MetricValue** whose value is a **ULONGLONG**.<br/> |
| [**Additem(programmaticName,value,metricValue)**](metricvaluecollection-additem-6.md) | Creates and adds a **MetricValue** whose value is a **DOUBLE**.<br/>    |



## Remarks

A **MetricValueCollection** holds data from an **Issue/MetricValues**, **Iteration/MetricValues**, or **TestCase/MetricValues** element.

A **MetricValue** object holds data from a **MetricValues/MetricValue** element.
