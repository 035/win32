# FilePath (Properties) element

Describes the path to the file.

## Usage

``` syntax
<FilePath (Properties)>
  text
</FilePath (Properties)>
```

## Attributes

There are no attributes.

## Text value

This element describes the path to the file. Source for this element is the assessment.

## Child elements

There are no child elements.

## Parent elements



| Element                         | Description                                                     |
|---------------------------------|-----------------------------------------------------------------|
| [**File**](file.md)<br/> | Holds information that describes a file.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
