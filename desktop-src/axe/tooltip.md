# ToolTip element

This is a short description that can be used in the UX.

## Usage

``` syntax
<ToolTip>
  text
</ToolTip>
```

## Attributes

There are no attributes.

## Text value

A short (200 characters) description that can be used in the UX.

## Child elements

There are no child elements.

| Element                                       | Description                                                   |
|-----------------------------------------------|---------------------------------------------------------------|
