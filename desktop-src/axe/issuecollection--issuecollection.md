# IssueCollection::~IssueCollection destructor

Destructor object.

## Syntax


```C++
virtual ~IssueCollection();
```



## Remarks

The **IssueCollection** object holds information from the **Iteration/Issues** element.

The **Issue** objects hold data from **Issues/Issue** elements.
