# Block element

If this file is running as a process, AXE blocks the assessment from running.

## Usage

``` syntax
<Block>
  text
</Block>
```

## Attributes

There are no attributes.

## Text value

Filename for the executable file for the process that blocks the assessment.

## Child elements

There are no child elements.

## Parent elements

There are no parent elements.

## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
