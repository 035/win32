# Url element

This is a URL that refers to a web page about the assessment.

## Usage

``` syntax
<Url>
  text
</Url>
```

## Attributes

There are no attributes.

## Text value

A URL that refers to a web page about the assessment.

## Child elements

There are no child elements.

## Parent elements



| Element                                     | Description                                               |
|---------------------------------------------|-----------------------------------------------------------|
| [**Properties**](properties.md)<br/> | This element describes properties.<br/> <br/> |



## Remarks

AXE does not use this URL itself. This can be used by a solution to provide a link to more information to a human.

## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
