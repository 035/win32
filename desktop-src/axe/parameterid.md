# ParameterId element

Each **ParameterId** element contains the programmatic name of a parameter and a value for that parameter.

## Usage

``` syntax
<ParameterId/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                          | Description                                                               |
|--------------------------------------------------|---------------------------------------------------------------------------|
| [**ParameterIds**](https://msdn.microsoft.com/library/windows/desktop/hh449479)<br/> | Contains a collection of **ParameterId** elements.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
