# AnalysisLinks element

A container for [**Link**](link.md) elements used to launch analysis tools.

## Usage

``` syntax
<AnalysisLinks>
  child elements
</AnalysisLinks>
```

## Attributes

There are no attributes.

## Child elements



| Element                         | Description                                                                 |
|---------------------------------|-----------------------------------------------------------------------------|
| [**Link**](link.md)<br/> | A hyperlink to more information about this solution.<br/> <br/> |



### Child element sequence

``` syntax
Link
```

## Parent elements



| Element                             | Description                                                         |
|-------------------------------------|---------------------------------------------------------------------|
| [**Issues**](issues.md)<br/> | A container element for one or more issues. <br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
