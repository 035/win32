# RequiresUIAccess element

The presence of this tag indicates that the assessment requires UI access in order to manipulate UI elements.

## Usage

``` syntax
<RequiresUIAccess/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                           | Description                                                                                                                          |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateProcess**](createprocess.md)<br/> | AXE uses the fundamental create process API to specify an executable and a command line for that executable. <br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
