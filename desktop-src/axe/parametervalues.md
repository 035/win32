# ParameterValues element

Contains zero or more [**ParameterValue**](parametervalue.md) elements.

## Usage

``` syntax
<ParameterValues>
  child elements
</ParameterValues>
```

## Attributes

There are no attributes.

## Child elements



| Element                                             | Description                                                     |
|-----------------------------------------------------|-----------------------------------------------------------------|
| [**ParameterValue**](parametervalue.md)<br/> | Describes a single assessment parameter.<br/> <br/> |



### Child element sequence

``` syntax
ParameterValue
```

## Parent elements



| Element                                           | Description                                                                 |
|---------------------------------------------------|-----------------------------------------------------------------------------|
| [**AssessmentRun**](assessmentrun.md)<br/> | Defines the instance of an assessment for execution.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
