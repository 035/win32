# RequiresACPower element

If present then this assessment requires AC power and cannot run on DC power. The AXE engine will block execution if the system is on DC power.

## Usage

``` syntax
<RequiresACPower/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
