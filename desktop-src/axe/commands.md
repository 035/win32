# Commands element

Parent of elements that provide commands to execute the assessment (element [**ExecuteWorkload**](executeworkload.md)), analyze the assessment results (element [**AnalyzeFolder**](analyzefolder.md)), or analyze the assessment trace element [**AnalyzeTrace**](analyzetrace.md).

## Usage

``` syntax
<Commands/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements

There are no parent elements.
