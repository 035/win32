# ProblemDevicesWarn element

If present then if any devices in the system have an issue, a warning will be generated. Problem devices are shown in Device Manager with a yellow exclamation point.

## Usage

``` syntax
<ProblemDevicesWarn/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
