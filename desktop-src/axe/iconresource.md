# IconResource element

Name of the icon that represents the assessment.

## Usage

``` syntax
<IconResource>
  text
</IconResource>
```

## Attributes

There are no attributes.

## Text value

The value can be one of the following names for icons used by the "Windows Assessment Console".

<dl> CameraIcon  
CheckedBoxIcon  
CheckerBoardIcon  
DriverIcon  
FilesIcon  
FirstStartIcon  
IEIcon  
MemoryIcon  
MovieClipboardIcon  
PerfGraphIcon  
PowerButtonIcon  
ProjectorIcon  
SleepIcon  
SleepHiberIcon  
WinEIIcon  
</dl>

## Child elements

There are no child elements.
