# ShouldRunSilent element

If this node is present, then the assessment should run without any progress UX or other visible harness level UX. This includes the light weight launcher. This will be used for some performance and power assessments.

## Usage

``` syntax
<ShouldRunSilent/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
