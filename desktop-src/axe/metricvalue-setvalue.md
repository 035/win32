# MetricValue::SetValue methods

Sets the value of the **MetricValue**.

### Overload list



| Method                                            | Description                                                           |
|:--------------------------------------------------|:----------------------------------------------------------------------|
| [**SetValue(value)**](metricvalue-setvalue-1.md) | Sets a value of type **LPCWSTR** to the **MetricValue**.<br/>   |
| [**SetValue(value)**](metricvalue-setvalue-2.md) | Sets a value of type **INT** to the **MetricValue**.<br/>       |
| [**SetValue(value)**](metricvalue-setvalue-3.md) | Sets a value of type **UINT** to the **MetricValue**.<br/>      |
| [**SetValue(value)**](metricvalue-setvalue-4.md) | Sets a value of type **LONGLONG** to the **MetricValue**.<br/>  |
| [**SetValue(value)**](metricvalue-setvalue-5.md) | Sets a value of type **ULONGLONG** to the **MetricValue**.<br/> |
| [**SetValue(value)**](metricvalue-setvalue-6.md) | Sets a value of type **DOUBLE** to the **MetricValue**.<br/>    |
| [**SetValue(value)**](metricvalue-setvalue-7.md) | Sets a value of type **bool** to the **MetricValue**.<br/>      |



## Remarks

A **MetricValue** object holds data from an **Issue/MetricValues/MetricValue**, **Iteration/MetricValues/MetricValue**, or **TestCase/MetricValues/MetricValue** element.

The value is the value of element **MetricValue/Value**.
