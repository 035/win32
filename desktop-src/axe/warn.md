# Warn element

If this file is running as a process, AXE warns the users but the assessment can still run.

## Usage

``` syntax
<Warn>
  text
</Warn>
```

## Attributes

There are no attributes.

## Text value

Filename for the executable file for the process which causes a warning.

## Child elements

There are no child elements.

## Parent elements

There are no parent elements.

## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
