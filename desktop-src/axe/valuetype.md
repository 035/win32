# ValueType element

This node contains the type of the metric threshold value.

## Usage

``` syntax
<ValueType>
  text
</ValueType>
```

## Attributes

There are no attributes.

## Text value

The type identifiers are case insensitive and leading and trailing spaces are ignored.

The allowed types are:

<dl>  
String  
int  
Int16  
Int32  
Int64  
UInt16  
UInt32  
UInt64  
Byte  
SByte  
single  
double  
float  
bool  
Boolean  
char  
</dl>

## Child elements

There are no child elements.
