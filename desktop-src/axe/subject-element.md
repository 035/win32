# Subject element

The subject of the test case.

## Usage

``` syntax
<Subject>
  child elements
</Subject>
```

## Attributes

There are no attributes.

## Child elements



| Element                                     | Description                                                   |
|---------------------------------------------|---------------------------------------------------------------|
| [**Class**](class--subject-.md)<br/> | The class of the test case subject.<br/> <br/>    |
| [**Instance**](instance.md)<br/>     | The instance of the test case subject.<br/> <br/> |



### Child element sequence

``` syntax
(
  Class, 
  Instance
)
```

## Parent elements

There are no parent elements.

## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
