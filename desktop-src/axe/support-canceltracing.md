# Support::CancelTracing method

Cancels tracing.

## Syntax


```C++
virtual HRESULT CancelTracing() = 0;
```



## Parameters

This method has no parameters.

## Return value

If the function succeeds, it returns **S\_OK**. If it fails, it returns an error value.
