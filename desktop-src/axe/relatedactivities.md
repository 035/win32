# RelatedActivities element

A container element for one or more [**ActivityReference**](activityreference.md) elements.

## Usage

``` syntax
<RelatedActivities>
  child elements
</RelatedActivities>
```

## Attributes

There are no attributes.

## Child elements



| Element                                                   | Description                                                                         |
|-----------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**ActivityReference**](activityreference.md)<br/> | This element holds a reference to another related activity. <br/> <br/> |



### Child element sequence

``` syntax
ActivityReference
```

## Parent elements



| Element                             | Description                                               |
|-------------------------------------|-----------------------------------------------------------|
| [**Impact**](impact.md)<br/> | Describes the impact of the issue.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
