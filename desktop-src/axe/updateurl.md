# UpdateUrl element

This is URL that refers to a location that solution can use to check for an update to the assessment.

## Usage

``` syntax
<UpdateUrl>
  text
</UpdateUrl>
```

## Attributes

There are no attributes.

## Text value

A URL that refers to a location that solution can use to check for an update to the assessment.
