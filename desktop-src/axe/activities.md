# Activities element

A container element for [**Activity**](activity.md) elements.

## Usage

``` syntax
<Activities>
  child elements
</Activities>
```

## Attributes

There are no attributes.

## Child elements



| Element                                 | Description                                                                               |
|-----------------------------------------|-------------------------------------------------------------------------------------------|
| [**Activity**](activity.md)<br/> | This element holds information about a category of related issues.<br/> <br/> |



### Child element sequence

``` syntax
Activity
```

## Parent elements



| Element                                   | Description                                                                      |
|-------------------------------------------|----------------------------------------------------------------------------------|
| [**Iteration**](iteration.md)<br/> | Receives metric values for a single assessment iteration.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
