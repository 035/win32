# StopOnAssessmentError element

AXE stops the job if this element is present and an assessment returns a non-zero value. The default is to continue after assessment errors.

## Usage

``` syntax
<StopOnAssessmentError/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                           | Description                                                          |
|---------------------------------------------------|----------------------------------------------------------------------|
| [**JobParameters**](jobparameters.md)<br/> | This node describes the parameters for a job.<br/> <br/> |

|              |     |
|--------------|-----|
| Can be empty | Yes |
