# MetricValue::GetValue methods

Returns the value of the **MetricValue**.

### Overload list



| Method                                            | Description                                                                         |
|:--------------------------------------------------|:------------------------------------------------------------------------------------|
| [**GetValue(value)**](metricvalue-getvalue-1.md) | Returns the value of the **MetricValue** for a type **LPCWSTR** value.<br/>   |
| [**GetValue(value)**](metricvalue-getvalue-2.md) | Returns the value of the **MetricValue** for a type **INT** value.<br/>       |
| [**GetValue(value)**](metricvalue-getvalue-3.md) | Returns the value of the **MetricValue** for a type **UINT** value.<br/>      |
| [**GetValue(value)**](metricvalue-getvalue-4.md) | Returns the value of the **MetricValue** for a type **LONGLONG** value.<br/>  |
| [**GetValue(value)**](metricvalue-getvalue-5.md) | Returns the value of the **MetricValue** for a type **ULONGLONG** value.<br/> |
| [**GetValue(value)**](metricvalue-getvalue-6.md) | Returns the value of the **MetricValue** for a type **DOUBLE** value.<br/>    |
| [**GetValue(value)**](metricvalue-getvalue-7.md) | Returns the value of the **MetricValue** for a type **bool** value.<br/>      |



## Remarks

A **MetricValue** object holds data from an **Issue/MetricValues/MetricValue**, **Iteration/MetricValues/MetricValue**, or **TestCase/MetricValues/MetricValue** element.

The value is the value of element **MetricValue/Value**.
