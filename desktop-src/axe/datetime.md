# DateTime element

The data type is an instant in time, typically expressed as a date and time of day, for example "6/15/2009 1:45:30 PM".

## Usage

``` syntax
<DateTime/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements

There are no parent elements.
