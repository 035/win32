# BasicDisplayBlock element

If present then the operating system must not be using the basic display device driver. A device driver specific to the display hardware must be in use by the operating system.

## Usage

``` syntax
<BasicDisplayBlock/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
