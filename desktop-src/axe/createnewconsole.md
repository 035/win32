# CreateNewConsole element

The presence of this tag causes AXE to set the CREATE\_NEW\_CONSOLE flag in the call to CreateProcess.

## Usage

``` syntax
<CreateNewConsole/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                           | Description                                                                                                                          |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateProcess**](createprocess.md)<br/> | AXE uses the fundamental create process API to specify an executable and a command line for that executable. <br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
