# LinkToolTip element

A tool tip for the hyperlink.

## Usage

``` syntax
<LinkToolTip>
  text
</LinkToolTip>
```

## Attributes

There are no attributes.

## Text value

A tool tip for the hyperlink. The tool tip is serves as both UI hover text and for screen readers.

## Child elements

There are no child elements.

## Parent elements



| Element                         | Description                                                                  |
|---------------------------------|------------------------------------------------------------------------------|
| [**Link**](link.md)<br/> | A hyper-link to more information about this solution.<br/> <br/> |

|--------------|-----|
