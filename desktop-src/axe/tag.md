# Tag element

A tag that enables the UX to use filtering, grouping or sorting.

## Usage

``` syntax
<Tag>
  text
</Tag>
```

## Attributes

There are no attributes.

## Text value

The tag value.

## Child elements

There are no child elements.

## Parent elements



| Element                         | Description                                                   |
|---------------------------------|---------------------------------------------------------------|
| [**Tags**](tags.md)<br/> | Contains one or more **Tag** elements.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
