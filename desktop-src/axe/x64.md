# X64 element

Indicates the assessment should run on x64 platforms.

## Usage

``` syntax
<X64/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements

There are no parent elements.

## Remarks

See the [**OnlyForProcessor**](onlyforprocessor.md) element for more information about how to indicate the platforms for the assessment.

|              |     |
|--------------|-----|
