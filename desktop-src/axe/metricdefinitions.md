# MetricDefinitions element

This element contains a collection of [**MetricDefinition**](metricdefinition.md) elements.

## Usage

``` syntax
<MetricDefinitions>
  child elements
</MetricDefinitions>
```

## Attributes

There are no attributes.

## Child elements



| Element                                                 | Description                                                                          |
|---------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**MetricDefinition**](metricdefinition.md)<br/> | This element describes the metrics produced by an assessment.<br/> <br/> |



### Child element sequence

``` syntax
MetricDefinition
```

## Parent elements



| Element                                                     | Description                                             |
|-------------------------------------------------------------|---------------------------------------------------------|
| [**AssessmentManifest**](https://msdn.microsoft.com/library/windows/desktop/hh449288)<br/> | The root node of the assessment.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
