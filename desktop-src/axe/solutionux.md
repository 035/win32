# SolutionUX element

The mode of interacting with the user when the job runs.

## Usage

``` syntax
<SolutionUX>
  child elements
</SolutionUX>
```

## Attributes

There are no attributes.

## Child elements



| Element                                          | Description                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------|
| [**Console**](console.md)<br/>            | Indicates a console will be used for interaction with the user.<br/> <br/> |
| [**Normal**](normal--solutionux-.md)<br/> | Indicates normal interaction with the user.<br/> <br/>                     |
| [**Silent**](silent.md)<br/>              | Indicates no interaction with the user.<br/> <br/>                         |



### Child element sequence

``` syntax
ConsoleNormalSilent
```

## Parent elements

There are no parent elements.

## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
