# SortAscending element

Specifies whether the initial sort is ascending or descending.

## Usage

``` syntax
<SortAscending>
  text
</SortAscending>
```

## Attributes

There are no attributes.

## Text value

The value is either True or False, describing whether the initial sort specified by the [**Sort**](sort.md) node is ascending or descending.

## Child elements

There are no child elements.
