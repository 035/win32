# LastSavedTimeAndDate element

Last date and time a solution saved the manifest to a file.

## Usage

``` syntax
<LastSavedTimeAndDate>
  text
</LastSavedTimeAndDate>
```

## Attributes

There are no attributes.

## Text value

Last date and time a solution saved the manifest to a file. This is in Coordinated Universal Time (UTC). This should be written by the manifest authoring tools. Not required.

## Child elements

There are no child elements.
