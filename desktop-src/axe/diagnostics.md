# Diagnostics element

Holds diagnostic information for the assessment, including [**DiagnosticModules**](diagnosticmodules.md) and [**TracingProfiles**](tracingprofiles.md) elements.

## Usage

``` syntax
<Diagnostics/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements

There are no parent elements.
