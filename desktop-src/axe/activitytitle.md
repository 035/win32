# ActivityTitle element

Description of steps to resolve the issue.

## Usage

``` syntax
<ActivityTitle>
  text
</ActivityTitle>
```

## Attributes

There are no attributes.

## Text value

It is a simple text element that may contain steps to resolve the issue. Source for this element is the assessment.

## Child elements

There are no child elements.

## Parent elements



| Element                                 | Description                                                                               |
|-----------------------------------------|-------------------------------------------------------------------------------------------|
| [**Activity**](activity.md)<br/> | This element holds information about a category of related issues.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
