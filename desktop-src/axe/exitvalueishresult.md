# ExitValueIsHresult element

If present, AXE should interpret the process exit value as an **HRESULT** to determine success or failure.

## Usage

``` syntax
<ExitValueIsHresult/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.

## Parent elements



| Element                                                 | Description                                                                                |
|---------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**ExitValueMeaning**](exitvaluemeaning.md)<br/> | Indicates whether the exit value means success or is a **HRESULT**.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
