# IssueToolTip element

A tooltip for the UI to display.

## Usage

``` syntax
<IssueToolTip>
  text
</IssueToolTip>
```

## Attributes

There are no attributes.

## Text value

A tooltip for the UI to display. This may be displayed if the [**IssueTitle**](issuetitle.md).

## Child elements

There are no child elements.
