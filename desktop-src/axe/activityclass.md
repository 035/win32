# ActivityClass element

The class of this activity.

## Usage

``` syntax
<ActivityClass>
  text
</ActivityClass>
```

## Attributes

There are no attributes.

## Text value

The class of this activity. Examples: Device Start, Service Start. Source for this element is the assessment.

## Child elements

There are no child elements.

## Parent elements



| Element                                 | Description                                                                               |
|-----------------------------------------|-------------------------------------------------------------------------------------------|
| [**Activity**](activity.md)<br/> | This element holds information about a category of related issues.<br/> <br/> |

|              |     |
|--------------|-----|
