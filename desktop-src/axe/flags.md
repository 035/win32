# Flags element

Contains one or more [**Flag**](flag.md) elements.

## Usage

``` syntax
<Flags>
  child elements
</Flags>
```

## Attributes

There are no attributes.

## Child elements



| Element                         | Description                              |
|---------------------------------|------------------------------------------|
| [**Flag**](flag.md)<br/> | Describes a flag.<br/> <br/> |



### Child element sequence

``` syntax
Flag
```

## Parent elements



| Element                                                       | Description                                                                       |
|---------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [**ParameterDefinition**](parameterdefinition.md)<br/> | Information that fully describes an assessment parameter. <br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
