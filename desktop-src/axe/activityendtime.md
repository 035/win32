# ActivityEndTime element

End time for this activity in nanoseconds.

## Usage

``` syntax
<ActivityEndTime>
  text
</ActivityEndTime>
```

## Attributes

There are no attributes.

## Text value

End time for this activity in nanoseconds. Source for this element is the assessment.

## Child elements

There are no child elements.

## Parent elements



| Element                                 | Description                                                                               |
|-----------------------------------------|-------------------------------------------------------------------------------------------|
| [**Activity**](activity.md)<br/> | This element holds information about a category of related issues.<br/> <br/> |

|              |     |
|--------------|-----|
