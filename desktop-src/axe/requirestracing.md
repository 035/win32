# RequiresTracing element

If present then no system tracing can be running prior to the assessment running. The assessment will be doing its own tracing, and there can only be one tracing session open at a time.

## Usage

``` syntax
<RequiresTracing/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
