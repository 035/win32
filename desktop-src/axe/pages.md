# Pages element

A collection of Page nodes describing all pages in the report.

## Usage

``` syntax
<Pages>
  child elements
</Pages>
```

## Attributes

There are no attributes.

## Child elements



| Element                         | Description                                     |
|---------------------------------|-------------------------------------------------|
| [**Page**](page.md)<br/> | A description of a page.<br/> <br/> |



### Child element sequence

``` syntax
Page
```

## Parent elements



| Element                                                   | Description                                                             |
|-----------------------------------------------------------|-------------------------------------------------------------------------|
| [**PresentationHints**](presentationhints.md)<br/> | The root node of the presentation hints section.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
