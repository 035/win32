# Tags element

Contains one or more [**Tag**](tag.md) elements. These are defined by the assessment author.

## Usage

``` syntax
<Tags>
  child elements
</Tags>
```

## Attributes

There are no attributes.

## Child elements



| Element                       | Description                                                                             |
|-------------------------------|-----------------------------------------------------------------------------------------|
| [**Tag**](tag.md)<br/> | A tag that enables the UX to use filtering, grouping or sorting.<br/> <br/> |



### Child element sequence

``` syntax
Tag
```

## Parent elements



| Element                                       | Description                                                   |
|-----------------------------------------------|---------------------------------------------------------------|
| [**Description**](description.md)<br/> | Provides a human-readable description.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
