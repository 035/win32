# PageLink element

This node is the main page of the report.

## Usage

``` syntax
<PageLink>
  text
</PageLink>
```

## Attributes

There are no attributes.

## Text value

The value of this node is the main page of the report. If this report does not have a main page, this node can be omitted.

## Child elements

There are no child elements.

## Parent elements



| Element                                                   | Description                                                             |
|-----------------------------------------------------------|-------------------------------------------------------------------------|
| [**PresentationHints**](presentationhints.md)<br/> | The root node of the presentation hints section.<br/> <br/> |

|--------------|-----|
