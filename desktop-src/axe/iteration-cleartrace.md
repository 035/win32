# Iteration::ClearTrace method

Clear the trace for the **Iteration**.

## Syntax


```C++
virtual HRESULT ClearTrace() = 0;
```



## Parameters

This method has no parameters.

## Return value

If the function succeeds, it returns **S\_OK**. If it fails, it returns an error value.
