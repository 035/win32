# LogFiles element

A container to hold [**LogFile**](logfile.md) elements.

## Usage

``` syntax
<LogFiles>
  child elements
</LogFiles>
```

## Attributes

There are no attributes.

## Child elements



| Element                               | Description                                                                    |
|---------------------------------------|--------------------------------------------------------------------------------|
| [**LogFile**](logfile.md)<br/> | Path to any log files associated with this assessment. <br/> <br/> |



### Child element sequence

``` syntax
LogFile
```

## Parent elements



| Element                                                 | Description                                                                |
|---------------------------------------------------------|----------------------------------------------------------------------------|
| [**AssessmentResult**](assessmentresult.md)<br/> | Describes the results for an individual assessment.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
| Can be empty | No  |
