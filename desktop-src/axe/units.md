# Units element

This element describes the units of the metric value.

## Usage

``` syntax
<Units>
  text
</Units>
```

## Attributes

There are no attributes.

## Text value

This element describes the metric s units in engineering units, such as  MB/s  or  bytes .

## Child elements

There are no child elements.

## Parent elements



| Element                                                 | Description                                                                          |
|---------------------------------------------------------|--------------------------------------------------------------------------------------|
