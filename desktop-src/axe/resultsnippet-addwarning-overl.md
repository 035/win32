# ResultSnippet::AddWarning methods

Creates an [**ErrorWarning**](errorwarning.md) object for use by the [**ResultSnippet**](resultsnippet.md) object.

### Overload list



| Method                                                                   | Description                                                                                                                             |
|:-------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------|
| [**AddWarning(message)**](resultsnippet-addwarning-1.md)                | Creates an [**ErrorWarning**](errorwarning.md) warning object for use by the [**ResultSnippet**](resultsnippet.md) object.<br/> |
| [**AddWarning(result,message)**](resultsnippet-addwarning-3.md)         | Creates an [**ErrorWarning**](errorwarning.md) warning object for use by the [**ResultSnippet**](resultsnippet.md) object.<br/> |
| [**AddWarning(message,warning)**](resultsnippet-addwarning-2.md)        | Creates an [**ErrorWarning**](errorwarning.md) warning object for use by the [**ResultSnippet**](resultsnippet.md) object.<br/> |
| [**AddWarning(result,message,warning)**](resultsnippet-addwarning-4.md) | Creates an [**ErrorWarning**](errorwarning.md) warning object for use by the [**ResultSnippet**](resultsnippet.md) object.<br/> |
