# RequiresAutoLogOnBlock element

If present then the operating system must be configured to log on automatically. This is useful if the assessment will reboot the system.

## Usage

``` syntax
<RequiresAutoLogOnBlock/>
```

## Attributes

There are no attributes.

## Child elements

There are no child elements.
