# Groups element

This node is a container for one or more groups.

## Usage

``` syntax
<Groups>
  child elements
</Groups>
```

## Attributes

There are no attributes.

## Child elements



| Element                           | Description                                                 |
|-----------------------------------|-------------------------------------------------------------|
| [**Group**](group.md)<br/> | A named arrangements of metric data.<br/> <br/> |



### Child element sequence

``` syntax
Group
```

## Parent elements



| Element                         | Description                                     |
|---------------------------------|-------------------------------------------------|
| [**Page**](page.md)<br/> | A description of a page.<br/> <br/> |
