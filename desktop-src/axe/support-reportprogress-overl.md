# Support::ReportProgress methods

Reports the progress of the assessment to the AXE engine and can inform the assessment if it should stop running.

| Method                                                                                                          | Description                                                                                                                  |
|:----------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------|
| [**ReportProgress(progressType,progressValue,progressMessage)**](support-reportprogress-1.md)                  | Reports the progress of the assessment to the AXE engine and can inform the assessment if it should stop running.<br/> |
| [**ReportProgress(progressType,progressValue,progressMessage,cancelAssessment)**](support-reportprogress-2.md) | Reports the progress of the assessment to the AXE engine and can inform the assessment if it should stop running.<br/> |
