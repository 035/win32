# LengthConstraint element

The maximum length of the string parameter.

## Usage

``` syntax
<LengthConstraint>
  text
</LengthConstraint>
```

## Attributes

There are no attributes.

## Text value

The maximum length of the string parameter. This element must contain an integer value greater than zero.

## Child elements

There are no child elements.

## Parent elements



| Element                                       | Description                                                           |
|-----------------------------------------------|-----------------------------------------------------------------------|
| [**Constraints**](constraints.md)<br/> | Constraints specified for a parameter's value.<br/> <br/> |
