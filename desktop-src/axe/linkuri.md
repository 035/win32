# LinkURI element

The path to an executable or web page.

## Usage

``` syntax
<LinkURI>
  text
</LinkURI>
```

## Attributes

There are no attributes.

## Text value

The path to an executable, web page or a UI that can be invoked using the **ShellExecute** function.

## Child elements

There are no child elements.

|---------------------------------|------------------------------------------------------------------------------|
