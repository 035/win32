# DisplayName element

A human-readable name for the item.

## Usage

``` syntax
<DisplayName>
  text
</DisplayName>
```

## Attributes

There are no attributes.

## Text value

A human-readable name.

## Child elements

There are no child elements.

## Parent elements



| Element                                       | Description                                                   |
|-----------------------------------------------|---------------------------------------------------------------|
| [**Description**](description.md)<br/> | Provides a human-readable description.<br/> <br/> |



## Remarks

The DisplayName cannot be used as the assessments identity because the value can be localized.

## Element information



|              |     |
|--------------|-----|
| Can be empty | Yes |
