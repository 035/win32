# ActivityStartTime element

Start time for this activity in nanoseconds.

## Usage

``` syntax
<ActivityStartTime>
  text
</ActivityStartTime>
```

## Attributes

There are no attributes.

## Text value

Start time for this activity in nanoseconds. Source for this element is the assessment.

## Child elements

There are no child elements.

## Parent elements



| Element                                 | Description                                                                               |
|-----------------------------------------|-------------------------------------------------------------------------------------------|
| [**Activity**](activity.md)<br/> | This element holds information about a category of related issues.<br/> <br/> |



## Element information



|              |     |
|--------------|-----|
