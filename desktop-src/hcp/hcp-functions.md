# HCP Functions

The following are the Hardware Counter Profiling (HCP) functions.



| Function                                                   | Description                                                                         |
|------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**DisableThreadProfiling**](/windows/desktop/api/Winbase/nf-winbase-disablethreadprofiling)   | Disables thread profiling.<br/>                                               |
| [**EnableThreadProfiling**](/windows/desktop/api/Winbase/nf-winbase-enablethreadprofiling)     | Enables thread profiling on the specified thread.<br/>                        |
| [**QueryThreadProfiling**](/windows/desktop/api/Winbase/nf-winbase-querythreadprofiling)       | Determines whether thread profiling is enabled for the specified thread.<br/> |
| [**ReadThreadProfilingData**](/windows/desktop/api/Winbase/nf-winbase-readthreadprofilingdata) | Reads the specified profiling data associated with the thread.<br/>           |
