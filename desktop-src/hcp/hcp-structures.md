# HCP Structures

The following are the Hardware Counter Profiling (HCP) structures.



| Structure                                                | Description                                                                            |
|----------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**HARDWARE\_COUNTER\_DATA**](hardware-counter-data.md) | Contains the hardware counter value.<br/>                                        |
| [**PERFORMANCE\_DATA**](performance-data.md)            | Contains the thread profiling and hardware counter data that you requested.<br/> |
