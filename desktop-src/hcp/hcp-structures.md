# HCP Structures

The following are the Hardware Counter Profiling (HCP) structures.



| Structure                                                | Description                                                                            |
|----------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**HARDWARE\_COUNTER\_DATA**](/windows/desktop/api/Winnt/ns-winnt-_hardware_counter_data) | Contains the hardware counter value.<br/>                                        |
| [**PERFORMANCE\_DATA**](/windows/desktop/api/Winnt/ns-winnt-_performance_data)            | Contains the thread profiling and hardware counter data that you requested.<br/> |
