# MONITOR\_STATUS Return Codes

The following error codes can be returned by the information WMI classes for DFSR.

<dl> <dt>

<span id="MONITOR_STATUS_SUCCESS"></span><span id="monitor_status_success"></span>**MONITOR\_STATUS\_SUCCESS**
</dt> <dd>

0

Success.

</dd> <dt>

<span id="MONITOR_STATUS_GENERIC_DB_ERROR"></span><span id="monitor_status_generic_db_error"></span>**MONITOR\_STATUS\_GENERIC\_DB\_ERROR**
</dt> <dd>

1

Generic database error.

</dd> <dt>

<span id="MONITOR_STATUS_IDRECORD_NOT_FOUND"></span><span id="monitor_status_idrecord_not_found"></span>**MONITOR\_STATUS\_IDRECORD\_NOT\_FOUND**
</dt> <dd>

2

ID record not found.

</dd> <dt>

<span id="MONITOR_STATUS_VOLUME_NOT_FOUND"></span><span id="monitor_status_volume_not_found"></span>**MONITOR\_STATUS\_VOLUME\_NOT\_FOUND**
</dt> <dd>

3

Volume not found.

</dd> <dt>

<span id="MONITOR_STATUS_ACCESS_DENIED"></span><span id="monitor_status_access_denied"></span>**MONITOR\_STATUS\_ACCESS\_DENIED**
</dt> <dd>

4

Access denied.

</dd> <dt>

<span id="MONITOR_STATUS_GENERIC_ERROR"></span><span id="monitor_status_generic_error"></span>**MONITOR\_STATUS\_GENERIC\_ERROR**
</dt> <dd>

5

Generic error.

</dd> </dl>
