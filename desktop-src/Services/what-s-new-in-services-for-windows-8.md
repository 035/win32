# What's New in Services for Windows 8

Windows 8 and Windows Server 2012 include the following new capabilities for services.

## New Service Functions



| Function                                                                            | Description                                                                    |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**QueryServiceDynamicInformation**](queryservicedynamicinformation.md)<br/> | Retrieves dynamic information related to the current service start.<br/> |



 

## Updated API Elements



| API Element                                                                                     | Description                                                                                                                                                   |
|-------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**HandlerEx**](handlerex.md)<br/>                                                       | Added **SERVICE\_CONTROL\_USERMODEREBOOT**.<br/>                                                                                                        |
| [**SERVICE\_STATUS**](service-status-str.md)<br/>                                        | Added **SERVICE\_ACCEPT\_USERMODEREBOOT**.<br/>                                                                                                         |
| [**SERVICE\_TRIGGER\_SPECIFIC\_DATA\_ITEM**](service-trigger-specific-data-item.md)<br/> | Added **SERVICE\_TRIGGER\_DATA\_TYPE\_LEVEL**, **SERVICE\_TRIGGER\_DATA\_TYPE\_KEYWORD\_ANY**, and **SERVICE\_TRIGGER\_DATA\_TYPE\_KEYWORD\_ALL**.<br/> |
