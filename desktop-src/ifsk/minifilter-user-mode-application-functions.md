# Minifilter User-Mode Application Functions

This section describes the system-supplied **Filter***Xxx* functions that user-mode applications can use to communicate with kernel-mode file system minifilter drivers. The functions are in alphabetical order.

## 

## In this section



| Topic                                                                               | Description                                                                                                                                                                                                                                                                                                                                                 |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FilterAttach**](/windows/desktop/api/fltuser/nf-fltuser-filterattach)<br/>                                     | The **FilterAttach** function attaches a new minifilter instance to the given volume. <br/>                                                                                                                                                                                                                                                           |
| [**FilterAttachAtAltitude**](/windows/desktop/api/fltuser/nf-fltuser-filterattachataltitude)<br/>                 | The **FilterAttachAtAltitude** function is a debugging support function that attaches a new minifilter instance to a volume at a specified altitude, overriding any settings in the minifilter's setup information (INF) file. <br/>                                                                                                                  |
| [**FilterClose**](/windows/desktop/api/fltuser/nf-fltuser-filterclose)<br/>                                       | The **FilterClose** function closes an open minifilter handle. <br/>                                                                                                                                                                                                                                                                                  |
| [**FilterConnectCommunicationPort**](/windows/desktop/api/fltuser/nf-fltuser-filterconnectcommunicationport)<br/> | **FilterConnectCommunicationPort** opens a new connection to a communication server port that is created by a file system minifilter. <br/>                                                                                                                                                                                                           |
| [**FilterCreate**](/windows/desktop/api/fltuser/nf-fltuser-filtercreate)<br/>                                     | The **FilterCreate** function creates a handle for the given minifilter. <br/>                                                                                                                                                                                                                                                                        |
| [**FilterDetach**](/windows/desktop/api/fltuser/nf-fltuser-filterdetach)<br/>                                     | The **FilterDetach** function detaches the given minifilter instance from the given volume. <br/>                                                                                                                                                                                                                                                     |
| [**FilterFindClose**](/windows/desktop/api/fltuser/nf-fltuser-filterfindclose)<br/>                               | The **FilterFindClose** function closes the specified minifilter search handle. The [**FilterFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filterfindfirst) and [**FilterFindNext**](/windows/desktop/api/fltuser/nf-fltuser-filterfindnext) functions use this search handle to locate minifilters. <br/>                                                                                                             |
| [**FilterFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filterfindfirst)<br/>                               | The **FilterFindFirst** function returns information about a filter driver (minifilter driver instance or legacy filter driver) and is used to begin scanning the filters in the global list of registered filters.<br/>                                                                                                                              |
| [**FilterFindNext**](/windows/desktop/api/fltuser/nf-fltuser-filterfindnext)<br/>                                 | The **FilterFindNext** function continues a filter search started by a call to [**FilterFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filterfindfirst). <br/>                                                                                                                                                                                                                      |
| [**FilterGetDosName**](/windows/desktop/api/fltuser/nf-fltuser-filtergetdosname)<br/>                             | The **FilterGetDosName** function returns the MS-DOS device name that corresponds to the given volume name. <br/>                                                                                                                                                                                                                                     |
| [**FilterGetInformation**](/windows/desktop/api/fltuser/nf-fltuser-filtergetinformation)<br/>                     | The **FilterGetInformation** function returns various kinds of information about a minifilter.<br/>                                                                                                                                                                                                                                                   |
| [**FilterGetMessage**](/windows/desktop/api/fltuser/nf-fltuser-filtergetmessage)<br/>                             | The **FilterGetMessage** function gets a message from a kernel-mode minifilter. <br/>                                                                                                                                                                                                                                                                 |
| [**FilterInstanceClose**](/windows/desktop/api/fltuser/nf-fltuser-filterinstanceclose)<br/>                       | The **FilterInstanceClose** function closes a minifilter instance handle opened by **FilterInstanceCreate**. <br/>                                                                                                                                                                                                                                    |
| [**FilterInstanceCreate**](/windows/desktop/api/fltuser/nf-fltuser-filterinstancecreate)<br/>                     | The **FilterInstanceCreate** function creates a handle that can be used to communicate with the given minifilter instance. <br/>                                                                                                                                                                                                                      |
| [**FilterInstanceFindClose**](/windows/desktop/api/fltuser/nf-fltuser-filterinstancefindclose)<br/>               | The **FilterInstanceFindClose** function closes the specified minifilter instance search handle. The [**FilterInstanceFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filterinstancefindfirst) and [**FilterInstanceFindNext**](/windows/desktop/api/fltuser/nf-fltuser-filterinstancefindnext) functions use this search handle to locate instances of a minifilter. <br/>                                              |
| [**FilterInstanceFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filterinstancefindfirst)<br/>               | The **FilterInstanceFindFirst** function returns information about a minifilter driver instance and is used as a starting point for scanning the instances of a minifilter. <br/>                                                                                                                                                                     |
| [**FilterInstanceFindNext**](/windows/desktop/api/fltuser/nf-fltuser-filterinstancefindnext)<br/>                 | The **FilterInstanceFindNext** function continues a minifilter driver instance search started by a call to [**FilterInstanceFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filterinstancefindfirst). <br/>                                                                                                                                                                          |
| [**FilterInstanceGetInformation**](/windows/desktop/api/fltuser/nf-fltuser-filterinstancegetinformation)<br/>     | The **FilterInstanceGetInformation** function returns various kinds of information about a minifilter instance. <br/>                                                                                                                                                                                                                                 |
| [**FilterLoad**](/windows/desktop/api/fltuser/nf-fltuser-filterload)<br/>                                         | The **FilterLoad** function dynamically loads a minifilter driver into the system. <br/>                                                                                                                                                                                                                                                              |
| [**FilterReplyMessage**](/windows/desktop/api/fltuser/nf-fltuser-filterreplymessage)<br/>                         | The **FilterReplyMessage** function replies to a message from a kernel-mode minifilter. <br/>                                                                                                                                                                                                                                                         |
| [**FilterSendMessage**](/windows/desktop/api/fltuser/nf-fltuser-filtersendmessage)<br/>                           | The **FilterSendMessage** function sends a message to a kernel-mode minifilter. <br/>                                                                                                                                                                                                                                                                 |
| [**FilterUnload**](/windows/desktop/api/fltuser/nf-fltuser-filterunload)<br/>                                     | An application that has loaded a supporting minifilter by calling [**FilterLoad**](/windows/desktop/api/fltuser/nf-fltuser-filterload) can unload the minifilter by calling the **FilterUnload** function. <br/>                                                                                                                                                                          |
| [**FilterVolumeFindClose**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumefindclose)<br/>                   | The **FilterVolumeFindClose** function closes the specified volume search handle. [**FilterVolumeFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumefindfirst) and [**FilterVolumeFindNext**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumefindnext) use this search handle to locate volumes. <br/>                                                                                                     |
| [**FilterVolumeFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumefindfirst)<br/>                   | The **FilterVolumeFindFirst** function returns information about a volume.<br/>                                                                                                                                                                                                                                                                       |
| [**FilterVolumeFindNext**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumefindnext)<br/>                     | The **FilterVolumeFindNext** function continues a volume search started by a call to [**FilterVolumeFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumefindfirst).<br/>                                                                                                                                                                                                     |
| [**FilterVolumeInstanceFindClose**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumeinstancefindclose)<br/>   | The [**FilterVolumeInstanceFindClose**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumeinstancefindclose) function closes the specified volume instance search handle. [**FilterVolumeInstanceFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumeinstancefindfirst) and [**FilterVolumeInstanceFindNext**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumeinstancefindnext) use this search handle to locate instances on a volume. <br/> |
| [**FilterVolumeInstanceFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumeinstancefindfirst)<br/>   | The **FilterVolumeInstanceFindFirst** function returns information about a minifilter driver instance or legacy filter driver and is used to begin scanning the filter drivers that are attached to a volume.<br/>                                                                                                                                    |
| [**FilterVolumeInstanceFindNext**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumeinstancefindnext)<br/>     | The **FilterVolumeInstanceFindNext** function continues a minifilter driver instance or legacy filter driver search started by a call to [**FilterVolumeInstanceFindFirst**](/windows/desktop/api/fltuser/nf-fltuser-filtervolumeinstancefindfirst). <br/>                                                                                                                                |
