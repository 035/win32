# StartService method of the Msvm\_TerminalService class

Starts the service.

## Syntax


```mof
uint32 StartService();
```



## Parameters

This method has no parameters.

## Return value

This method returns one of the following values:

<dl> <dt>

**Completed with No Error** (0)
</dt> <dt>

**Not supported** (1)
</dt> </dl>
