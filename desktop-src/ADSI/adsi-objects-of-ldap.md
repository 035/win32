# ADSI Objects of LDAP

| ADSI object            | Description                                                                                                                           | Supported interfaces                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

| **Group**              | An ADSI object that represents a group.                                                                                               | [**IADsGroup**](iadsgroup.md)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |

| **Property**           | An ADSI object that represents the attribute definition in the schema.                                                                | <dl> <dt>[**IADs**](iads.md)</dt> <dt>[**IADsProperty**](iadsproperty.md)</dt> </dl>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |

| **UserCollection**     | An ADSI object that implements [**IADsMembers**](iadsmembers.md) for maintaining and managing a collection of user accounts.         | [**IADsMembers**](iadsmembers.md)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
