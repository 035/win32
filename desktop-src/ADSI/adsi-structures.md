# ADSI Structures

Active Directory Service Interfaces (ADSI) define and use the structures listed in the following table.



| Structure                                                                      | Description                                                                                                    |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**ADS\_ATTR\_DEF**](ads-attr-def.md)<br/>                              | Describes a set of attribute definitions of an **attributeSchema** object.<br/>                          |
| [**ADS\_ATTR\_INFO**](ads-attr-info.md)<br/>                            | Holds the value of a named attribute and specifies operations to modify the attribute.<br/>              |
| [**ADS\_BACKLINK**](ads-backlink.md)<br/>                               | Represents an ADSI representation of the **Back Link** attribute.<br/>                                   |
| [**ADS\_CASEIGNORE\_LIST**](ads-caseignore-list.md)<br/>                | Represents an ADSI representation of the **Case Ignore List** attribute.<br/>                            |
| [**ADS\_CLASS\_DEF**](ads-class-def.md)<br/>                            | Describes schema data for an object.<br/>                                                                |
| [**ADS\_DN\_WITH\_BINARY**](ads-dn-with-binary.md)<br/>                 | An ADSI-defined structure that maps a distinguished name to an object **GUID**.<br/>                     |
| [**ADS\_DN\_WITH\_STRING**](ads-dn-with-string.md)<br/>                 | An ADSI-defined structure that maps a distinguished name of an object to a nonvarying string value.<br/> |
| [**ADS\_EMAIL**](ads-email.md)<br/>                                     | Represents an ADSI representation of the **EMail** attribute.<br/>                                       |
| [**ADS\_FAXNUMBER**](ads-faxnumber.md)<br/>                             | Represents an ADSI representation of the **Facsimile Telephone Number** attribute.<br/>                  |
| [**ADS\_HOLD**](ads-hold.md)<br/>                                       | Represents an ADSI representation of the **Hold** attribute.<br/>                                        |
| [**ADS\_NETADDRESS**](ads-netaddress.md)<br/>                           | Represents an ADSI representation of the **Net Address** attribute.<br/>                                 |
| [**ADS\_NT\_SECURITY\_DESCRIPTOR**](ads-nt-security-descriptor.md)<br/> | Describes a security descriptor.<br/>                                                                    |
| [**ADS\_OBJECT\_INFO**](ads-object-info.md)<br/>                        | Describes directory service object data.<br/>                                                            |
| [**ADS\_OCTET\_LIST**](ads-octet-list.md)<br/>                          | Represents an ADSI representation of the **Octet List** attribute.<br/>                                  |
| [**ADS\_OCTET\_STRING**](ads-octet-string.md)<br/>                      | Represents an ADSI representation of the **Octet String** attribute.<br/>                                |
| [**ADS\_PATH**](ads-path.md)<br/>                                       | Represents an ADSI representation of the **Path** attribute.<br/>                                        |
| [**ADS\_POSTALADDRESS**](ads-postaladdress.md)<br/>                     | Represents an ADSI representation of the **Postal Address** attribute.<br/>                              |
| [**ADS\_PROV\_SPECIFIC**](ads-prov-specific.md)<br/>                    | Describes a provider-specific data type.<br/>                                                            |
| [**ADS\_REPLICAPOINTER**](ads-replicapointer.md)<br/>                   | Represents an ADSI representation of the Replica Pointer attribute.<br/>                                 |
| [**ADS\_SEARCH\_COLUMN**](ads-search-column.md)<br/>                    | Represents a column resulting from a directory query.<br/>                                               |
| [**ADS\_SEARCHPREF\_INFO**](ads-searchpref-info.md)<br/>                | Describes query preferences.<br/>                                                                        |
| [**ADS\_SORTKEY**](ads-sortkey.md)<br/>                                 | Describes the sort key used for sorting a query.<br/>                                                    |
| [**ADS\_TIMESTAMP**](ads-timestamp.md)<br/>                             | Represents an ADSI representation of the **Timestamp** attribute.<br/>                                   |
| [**ADS\_TYPEDNAME**](ads-typedname.md)<br/>                             | Represents an ADSI representation of the **Typed Name** attribute.<br/>                                  |
| [**ADSVALUE**](adsvalue.md)<br/>                                        | Describes the value of an ADSI data type.<br/>                                                           |
| [**ADS\_VLV**](ads-vlv.md)<br/>                                         | Specifies that the virtual list view should be used when returning search results from the server.<br/>  |
