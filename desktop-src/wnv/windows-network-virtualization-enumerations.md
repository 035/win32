# WNV notification enumerations

The Windows Network Virtualization (WNV) notification API defines the following enumerations.

## In this section



| Topic                                                               | Description                                                                                                                                                  |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WNV\_NOTIFICATION\_TYPE**](/windows/previous-versions/wnvapi/ne-wnvapi-_wnv_notification_type?branch=master)<br/> | Specifies the type of a given Windows Network Virtualization (WNV) notification.<br/>                                                                  |
| [**WNV\_OBJECT\_TYPE**](/windows/previous-versions/wnvapi/ne-wnvapi-_wnv_object_type?branch=master)<br/>             | Specifies the object type of a given Windows Network Virtualization (WNV) notification when the WNV notification type is **WnvObjectChangeType**.<br/> |
