# WNV notification enumerations

The Windows Network Virtualization (WNV) notification API defines the following enumerations.

## In this section



| Topic                                                               | Description                                                                                                                                                  |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WNV\_NOTIFICATION\_TYPE**](wnv-notification-type.md)<br/> | Specifies the type of a given Windows Network Virtualization (WNV) notification.<br/>                                                                  |
| [**WNV\_OBJECT\_TYPE**](wnv-object-type.md)<br/>             | Specifies the object type of a given Windows Network Virtualization (WNV) notification when the WNV notification type is **WnvObjectChangeType**.<br/> |
