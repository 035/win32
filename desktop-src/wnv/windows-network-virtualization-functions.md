# WNV notification functions

The Windows Network Virtualization (WNV) notification API defines the following functions.

## In this section



| Topic                                                               | Description                                                                                                                                 |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**WnvOpen**](wnvopen.md)<br/>                               | Provides a handle to the Windows Network Virtualization (WNV) driver object to be used to request and receive WNV notifications.<br/> |
| [**WnvRequestNotification**](wnvrequestnotification.md)<br/> | Requests notification from the Windows Network Virtualization (WNV) driver whenever a certain type of event occurs.<br/>              |
