# Enumerations

The Gaming Device Information API contains these enumerations.

## In this section



| Topic                                                                    | Description                                                                 |
|--------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**GAMING\_DEVICE\_DEVICE\_ID**](/previous-versions/windows/desktop/api/gamingdeviceinformation/ne-gamingdeviceinformation-gaming_device_device_id)<br/> | Indicates the type of device that the game is running on.<br/>        |
| [**GAMING\_DEVICE\_VENDOR\_ID**](/previous-versions/windows/desktop/api/gamingdeviceinformation/ne-gamingdeviceinformation-gaming_device_vendor_id)<br/> | Indicates the vendor of the console that the game is running on.<br/> |
