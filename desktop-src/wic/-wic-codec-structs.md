# Structures

This section contains information about the Windows Imaging Component (WIC) structures.

## In this section



| Topic                                                                          | Description                                                                                                                   |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WICBitmapPattern**](-wic-codec-wicbitmappattern.md)<br/>             | Contains members that identify a pattern within an image file which can be used to identify a particular format.<br/>   |
| [**WICBitmapPlane**](wicbitmapplane.md)<br/>                            | Specifies the pixel format, buffer, stride and size of a component plane for a planar pixel format.<br/>                |
| [**WICBitmapPlaneDescription**](wicbitmapplanedescription.md)<br/>      | Specifies the pixel format and size of a component plane.<br/>                                                          |
| [**WICDdsFormatInfo**](wicddsformatinfo.md)<br/>                        | Specifies the [**DXGI\_FORMAT**](https://msdn.microsoft.com/library/windows/desktop/bb173059) and block information of a DDS format.<br/>                  |
| [**WICDdsParameters**](wicddsparameters.md)<br/>                        | Specifies the DDS image dimension, [**DXGI\_FORMAT**](https://msdn.microsoft.com/library/windows/desktop/bb173059) and alpha mode of contained data.<br/>  |
| [**WICImageParameters**](wicimageparameters.md)<br/>                    | This defines parameters that you can use to override the default parameters normally used when encoding an image. <br/> |
| [**WICJpegFrameHeader**](wicjpegframeheader.md)<br/>                    | Represents a JPEG frame header.<br/>                                                                                    |
| [**WICJpegScanHeader**](wicjpegscanheader.md)<br/>                      | Represents a JPEG frame header.<br/>                                                                                    |
| [**WICMetadataHeader**](-wic-codec-wicmetadataheader.md)<br/>           | Represents metadata header.<br/>                                                                                        |
| [**WICMetadataPattern**](-wic-codec-wicmetadatapattern.md)<br/>         | Represents a metadata pattern.<br/>                                                                                     |
| [**WICRawCapabilitiesInfo**](-wic-codec-wicrawcapabilitiesinfo.md)<br/> | Defines raw codec capabilites.<br/>                                                                                     |
| [**WICRawToneCurve**](-wic-codec-wicrawtonecurve.md)<br/>               | Represents a raw image tone curve.<br/>                                                                                 |
| [**WICRawToneCurvePoint**](-wic-codec-wicrawtonecurvepoint.md)<br/>     | Represents a raw image tone curve point.<br/>                                                                           |
| [**WICRect**](-wic-codec-wicrect.md)<br/>                               | Represents a rectangle for WIC API.<br/>                                                                                |
