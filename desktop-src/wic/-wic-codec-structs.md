# Structures

This section contains information about the Windows Imaging Component (WIC) structures.

## In this section



| Topic                                                                          | Description                                                                                                                   |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WICBitmapPattern**](/windows/win32/Wincodec/ns-wincodec-wicbitmappattern?branch=master)<br/>             | Contains members that identify a pattern within an image file which can be used to identify a particular format.<br/>   |
| [**WICBitmapPlane**](/windows/win32/Wincodec/ns-wincodec-wicbitmapplane?branch=master)<br/>                            | Specifies the pixel format, buffer, stride and size of a component plane for a planar pixel format.<br/>                |
| [**WICBitmapPlaneDescription**](/windows/win32/Wincodec/ns-wincodec-wicbitmapplanedescription?branch=master)<br/>      | Specifies the pixel format and size of a component plane.<br/>                                                          |
| [**WICDdsFormatInfo**](/windows/win32/Wincodec/ns-wincodec-wicddsformatinfo?branch=master)<br/>                        | Specifies the [**DXGI\_FORMAT**](https://msdn.microsoft.com/library/windows/desktop/bb173059) and block information of a DDS format.<br/>                  |
| [**WICDdsParameters**](/windows/win32/Wincodec/ns-wincodec-wicddsparameters?branch=master)<br/>                        | Specifies the DDS image dimension, [**DXGI\_FORMAT**](https://msdn.microsoft.com/library/windows/desktop/bb173059) and alpha mode of contained data.<br/>  |
| [**WICImageParameters**](/windows/win32/Wincodec/ns-wincodec-wicimageparameters?branch=master)<br/>                    | This defines parameters that you can use to override the default parameters normally used when encoding an image. <br/> |
| [**WICJpegFrameHeader**](/windows/win32/wincodec/ns-wincodec-wicjpegframeheader?branch=master)<br/>                    | Represents a JPEG frame header.<br/>                                                                                    |
| [**WICJpegScanHeader**](/windows/win32/wincodec/ns-wincodec-wicjpegscanheader?branch=master)<br/>                      | Represents a JPEG frame header.<br/>                                                                                    |
| [**WICMetadataHeader**](/windows/win32/Wincodecsdk/ns-wincodecsdk-wicmetadataheader?branch=master)<br/>           | Represents metadata header.<br/>                                                                                        |
| [**WICMetadataPattern**](/windows/win32/Wincodecsdk/ns-wincodecsdk-wicmetadatapattern?branch=master)<br/>         | Represents a metadata pattern.<br/>                                                                                     |
| [**WICRawCapabilitiesInfo**](/windows/win32/Wincodec/ns-wincodec-wicrawcapabilitiesinfo?branch=master)<br/> | Defines raw codec capabilites.<br/>                                                                                     |
| [**WICRawToneCurve**](/windows/win32/Wincodec/ns-wincodec-wicrawtonecurve?branch=master)<br/>               | Represents a raw image tone curve.<br/>                                                                                 |
| [**WICRawToneCurvePoint**](/windows/win32/Wincodec/ns-wincodec-wicrawtonecurvepoint?branch=master)<br/>     | Represents a raw image tone curve point.<br/>                                                                           |
| [**WICRect**](/windows/win32/Wincodec/ns-wincodec-wicrect?branch=master)<br/>                               | Represents a rectangle for WIC API.<br/>                                                                                |
