# Functions

This section contains information about the Windows Imaging Component (WIC) functions.

## In this section



| Topic                                                                                      | Description                                                                                                                                                                                                           |
|--------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [*ProgressNotificationCallback*](-wic-codec-progressnotificationcallback.md)<br/>   | Application defined callback function called when codec component progress is made.<br/>                                                                                                                        |
| [**WICConvertBitmapSource**](-wic-codec-wicconvertbitmapsource.md)<br/>             | Obtains a [**IWICBitmapSource**](-wic-codec-iwicbitmapsource.md) in the desired pixel format from a given **IWICBitmapSource**.<br/>                                                                           |
| [**WICCreateBitmapFromSection**](-wic-codec-wiccreatebitmapfromsection.md)<br/>     | Returns a [**IWICBitmapSource**](-wic-codec-iwicbitmapsource.md) that is backed by the pixels of a Windows Graphics Device Interface (GDI) section handle.<br/>                                                |
| [**WICCreateBitmapFromSectionEx**](-wic-codec-wiccreatebitmapfromsectionex.md)<br/> | Returns a [**IWICBitmapSource**](-wic-codec-iwicbitmapsource.md) that is backed by the pixels of a GDI section handle.<br/>                                                                                    |
| [**WICGetMetadataContentSize**](-wic-codec-wicgetmetadatacontentsize.md)<br/>       | Returns the size of the metadata content contained by the specified [**IWICMetadataWriter**](-wic-codec-iwicmetadatawriter.md). The returned size accounts for the header and the length of the metadata.<br/> |
| [**WICMapGuidToShortName**](-wic-codec-wicmapguidtoshortname.md)<br/>               | Obtains the short name associated with a given GUID.<br/>                                                                                                                                                       |
| [**WICMapSchemaToName**](-wic-codec-wicmapschematoname.md)<br/>                     | Obtains the name associated with a given schema.<br/>                                                                                                                                                           |
| [**WICMapShortNameToGuid**](-wic-codec-wicmapshortnametoguid.md)<br/>               | Obtains the GUID associated with the given short name.<br/>                                                                                                                                                     |
| [**WICMatchMetadataContent**](-wic-codec-wicmatchmetadatacontent.md)<br/>           | Obtains a metadata format GUID for a specified container format and vendor that best matches the content within a given stream.<br/>                                                                            |
| [**WICSerializeMetadataContent**](-wic-codec-wicserializemetadatacontent.md)<br/>   | Writes metadata into a given stream.<br/>                                                                                                                                                                       |
| [WIC Proxy Functions](wic-proxy-functions.md)<br/>                                  | This section contains the WIC proxy functions.<br/>                                                                                                                                                             |
