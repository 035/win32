# Trustmon Provider

The Trustmon provider is an instance provider that creates classes with information about the trust relationships between domains.

The Trustmon provider defines the classes in the following table. These classes reside in the "\\root\\microsoftactivedirectory" namespace.



| Class                                                               | Description                      |
|---------------------------------------------------------------------|----------------------------------|
| [**Microsoft\_DomainTrustStatus**](microsoft-domaintruststatus.md) | Trust enumerator class.          |
| [**Microsoft\_LocalDomainInfo**](microsoft-localdomaininfo.md)     | Local domain information class.  |
| [**Microsoft\_TrustProvider**](microsoft-trustprovider.md)         | Provider parameterization class. |
