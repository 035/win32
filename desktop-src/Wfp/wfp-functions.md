# WRP Functions

The following functions allow you to enumerate protected resources.



| Function                                         | Description                                                 |
|--------------------------------------------------|-------------------------------------------------------------|
| [**SfcIsFileProtected**](/windows/win32/Sfc/nf-sfc-sfcisfileprotected?branch=master) | Determines whether the specified file is protected.         |
| [**SfcIsKeyProtected**](/windows/win32/Sfc/nf-sfc-sfciskeyprotected?branch=master)   | Determines whether the specified registry key is protected. |



 

If available for the operating system, the functions in this table should be used rather than the deprecated functions: [**SfcGetNextProtectedFile**](/windows/win32/Sfc/nf-sfc-sfcgetnextprotectedfile?branch=master) and [**SfcGetFiles**](sfcgetfiles.md).
