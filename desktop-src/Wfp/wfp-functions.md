# WRP Functions

The following functions allow you to enumerate protected resources.



| Function                                         | Description                                                 |
|--------------------------------------------------|-------------------------------------------------------------|
| [**SfcIsFileProtected**](sfcisfileprotected.md) | Determines whether the specified file is protected.         |
| [**SfcIsKeyProtected**](sfciskeyprotected.md)   | Determines whether the specified registry key is protected. |



 

If available for the operating system, the functions in this table should be used rather than the deprecated functions: [**SfcGetNextProtectedFile**](sfcgetnextprotectedfile.md) and [**SfcGetFiles**](sfcgetfiles.md).
