# Memory Classes

The memory classes model the memory for a virtual system.

The following are virtualization WMI classes related to memory.

## In this section



| Class                                                                | Description                                                                    |
|----------------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**Msvm\_Memory**](msvm-memory.md)<br/>                       | Represents the memory currently allocated to a virtual system.<br/>      |
| [**Msvm\_MemorySettingData**](msvm-memorysettingdata.md)<br/> | Represents the configured state of the memory for a virtual system.<br/> |
