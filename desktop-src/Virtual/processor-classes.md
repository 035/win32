# Processor Classes

The processor classes model a processor for a virtual system.

The following are virtualization WMI classes related to the processor.

## In this section



| Class                                                                      | Description                                                                              |
|----------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**Msvm\_Processor**](msvm-processor.md)<br/>                       | Represents the virtual processor in a virtual computer system.<br/>                |
| [**Msvm\_ProcessorPool**](msvm-processorpool.md)<br/>               | Aggregates the processor resources that may be allocated to a virtual system.<br/> |
| [**Msvm\_ProcessorSettingData**](msvm-processorsettingdata.md)<br/> | Represents the virtual processor settings for a virtual machine (VM).<br/>         |
