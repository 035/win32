<dl> <dt>

<span id="wmi_v2.gloss_reference_property"></span><span id="WMI_V2.GLOSS_REFERENCE_PROPERTY"></span>**reference property**
</dt> <dd>

A [*property*](https://www.bing.com/search?q=*property*) that uses one of the user-defined classes as its type.

</dd> <dt>

<span id="wmi_v2.gloss_refresher"></span><span id="WMI_V2.GLOSS_REFRESHER"></span>**refresher**
</dt> <dd>

An object that contains instances of [*MOF*](https://www.bing.com/search?q=*MOF*) classes so that the data can be updated. [*Performance*](https://www.bing.com/search?q=*Performance*) classes that derive from [**Win32\_Perf**](https://msdn.microsoft.com/library/aa394252) require the use of a refresher object to get performance data.

</dd> <dt>

<span id="wmi_v2.gloss_required_property"></span><span id="WMI_V2.GLOSS_REQUIRED_PROPERTY"></span>**required property**
</dt> <dd>

A [*property*](https://www.bing.com/search?q=*property*) that requires a value.

</dd> <dt>

<span id="wmi_v2.gloss_reverse_adapter"></span><span id="WMI_V2.GLOSS_REVERSE_ADAPTER"></span>**Reverse Adapter**
</dt> <dd>

A process that moves WMI performance classes to performance libraries.

</dd> </dl>
