<span id="wmi_v2.gloss_cim"></span><span id="WMI_V2.GLOSS_CIM"></span>**CIM**
</dt> <dd>

See *Common Information Model (CIM)*.

</dd> <dt>

<span id="wmi_v2.gloss_cim_datetime"></span><span id="WMI_V2.GLOSS_CIM_DATETIME"></span>**CIM datetime**
</dt> <dd>

The combined date and time format defined by the Distributed Management Task Force (DMTF).

</dd> <dt>

<span id="wmi_v2.gloss_cim_object_manager"></span><span id="WMI_V2.GLOSS_CIM_OBJECT_MANAGER"></span>**CIM Object Manager**
</dt> <dd>

A component in the WMI infrastructure that handles the interaction between management applications and [*providers*](gloss-p.md#wmi-v2-gloss-provider).

</dd> <dt>

<span id="wmi_v2.gloss_cim_repository"></span><span id="WMI_V2.GLOSS_CIM_REPOSITORY"></span>**CIM repository**
</dt> <dd>

See [*WMI repository*](gloss-w.md#wmi-v2-gloss-wmi-repository).

</dd> <dt>

<span id="wmi_v2.gloss_cim_schema"></span><span id="WMI_V2.GLOSS_CIM_SCHEMA"></span>**CIM schema**
</dt> <dd>

A collection of class definitions used to represent [*managed objects*](gloss-m.md#wmi-v2-gloss-managed-object) that occur in every managed environment. Also see *common model*, and [*extension schema*](gloss-e.md#wmi-v2-gloss-extension-schema).

</dd> <dt>

<span id="wmi_v2.gloss_common_information_model"></span><span id="WMI_V2.GLOSS_COMMON_INFORMATION_MODEL"></span>**Common Information Model (CIM)**
</dt> <dd>

The [*DMTF*](gloss-d.md#wmi-v2-gloss-distributed-management-task-force) standard \[DSP0004\] for describing the structure and behavior of the managed resources such as storage, network, or software components. For more information, visit <Http://Go.Microsoft.Com/FWLink/p/?LinkID=269835>. Windows Management Infrastructure (MI) is a CIM server that implements the CIM standard on Windows.

</dd> <dt>

<span id="wmi_v2.gloss_common_model"></span><span id="WMI_V2.GLOSS_COMMON_MODEL"></span>**common model**
</dt> <dd>

The second layer of the *CIM schema*, which includes a series of domain-specific, but platform-independent classes. The domains are systems, networks, applications, and other management-related data. The common model is derived from the core model. Also see [*extension schema*](gloss-e.md#wmi-v2-gloss-extension-schema).

</dd> <dt>

<span id="wmi_v2.gloss_consumer_provider"></span><span id="WMI_V2.GLOSS_CONSUMER_PROVIDER"></span>**consumer provider**
</dt> <dd>

See [*event consumer provider*](gloss-e.md#wmi-v2-gloss-event-consumer-provider).

</dd> </dl>
