# W

[A](gloss-a.md) [C](gloss-c.md) [D](gloss-d.md) [E](gloss-e.md) [F](gloss-f.md) [H](gloss-h.md) [I](gloss-i.md) [K](gloss-k.md) [L](gloss-l.md) [M](gloss-m.md) [N](gloss-n.md) [O](gloss-o.md) [P](gloss-p.md) [Q](gloss-q.md) [R](gloss-r.md) [S](gloss-s.md) [T](gloss-t.md)

<dl> <dt>

<span id="wmi_v2.gloss_win32_schema"></span><span id="WMI_V2.GLOSS_WIN32_SCHEMA"></span>**Win32 schema**
</dt> <dd>

A Microsoft extension to the [*CIM schema*](gloss-c.md#wmi-v2-gloss-cim-schema) that contains definitions of classes and instances to represent [*managed objects*](gloss-m.md#wmi-v2-gloss-managed-object) that exist in a typical Windows environment.

</dd> <dt>

<span id="wmi_v2.gloss_windows_management_infrastructure"></span><span id="WMI_V2.GLOSS_WINDOWS_MANAGEMENT_INFRASTRUCTURE"></span>**Windows Management Infrastructure (MI)**
</dt> <dd>

Windows Management Infrastructure (MI) is a CIM server that implements the CIM standard on Windows.

</dd> <dt>

<span id="wmi_v2.gloss_wmi_control"></span><span id="WMI_V2.GLOSS_WMI_CONTROL"></span>**WMI Control**
</dt> <dd>

A Microsoft Management Console (MMC) snap-in that enables a user to configure WMI settings on a local or remote computer.

</dd> <dt>

<span id="wmi_v2.gloss_wmi_for_windows_driver_model"></span><span id="WMI_V2.GLOSS_WMI_FOR_WINDOWS_DRIVER_MODEL"></span>**WMI for Windows Driver Model (WDM)**
</dt> <dd>

A low-level component that provides access between WMI and hardware devices.

</dd> <dt>

<span id="wmi_v2.gloss_wmi_odbc_adapter"></span><span id="WMI_V2.GLOSS_WMI_ODBC_ADAPTER"></span>**WMI ODBC Adapter**
</dt> <dd>

An interface that enables an application to get information from Windows Management using *WMI Query Language*. This provides a standard API that allows ODBC-based applications to use the data in the [*CIM repository*](gloss-c.md#wmi-v2-gloss-cim-repository) as if it were a database.

</dd> <dt>

<span id="wmi_v2.gloss_wmi_query_language"></span><span id="WMI_V2.GLOSS_WMI_QUERY_LANGUAGE"></span>**WMI Query Language (WQL)**
</dt> <dd>

A subset of structured query language (SQL) with extensions to support WMI event notification and other WMI-specific features.

</dd> <dt>

<span id="wmi_v2.gloss_wmi_method"></span><span id="WMI_V2.GLOSS_WMI_METHOD"></span>**WMI method**
</dt> <dd>

A method implemented by the WMI COM API or the Scripting API. Two examples are the COM method IWbemServices::ExecMethod and the similar scripting API method SWbemServices.ExecMethod.

</dd> <dt>

<span id="wmi_v2.gloss_wmi_repository"></span><span id="WMI_V2.GLOSS_WMI_REPOSITORY"></span>**WMI repository**
</dt> <dd>

A central storage area managed by the [*CIM Object Manager*](gloss-c.md#wmi-v2-gloss-cim-object-manager).

</dd> <dt>

<span id="wmi_v2.gloss_wql"></span><span id="WMI_V2.GLOSS_WQL"></span>**WQL**
</dt> <dd>

See *WMI Query Language (WQL)*.

</dd> </dl>
