<span id="wmi_v2.gloss_flavor"></span><span id="WMI_V2.GLOSS_FLAVOR"></span>**flavor**

A WMI-defined flag that describes the behavior of a [*qualifier*](gloss-q.md#wmi-v2-gloss-qualifier). WMI flavors describe rules that specify whether a qualifier can be propagated to derived classes and instances and whether a derived class or instance can override the qualifier's original value.
