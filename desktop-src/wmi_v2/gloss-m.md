# M

[A](gloss-a.md) [C](gloss-c.md) [D](gloss-d.md) [E](gloss-e.md) [F](gloss-f.md) [H](gloss-h.md) [I](gloss-i.md) [K](gloss-k.md) [L](gloss-l.md) [N](gloss-n.md) [O](gloss-o.md) [P](gloss-p.md) [Q](gloss-q.md) [R](gloss-r.md) [S](gloss-s.md) [T](gloss-t.md) [W](gloss-w.md)

<dl> <dt>

<span id="wmi_v2.gloss_managed_object"></span><span id="WMI_V2.GLOSS_MANAGED_OBJECT"></span>**managed object**
</dt> <dd>

A hardware or software system component that is represented as an instance of a WMI class.

</dd> <dt>

<span id="wmi_v2.gloss_managed_object_format"></span><span id="WMI_V2.GLOSS_MANAGED_OBJECT_FORMAT"></span>**Managed Object Format (MOF)**
</dt> <dd>

A language based on Interface Definition Language (IDL) that describes management information.

</dd> <dt>

<span id="wmi_v2.gloss_management_information_base"></span><span id="WMI_V2.GLOSS_MANAGEMENT_INFORMATION_BASE"></span>**management information base (MIB)**
</dt> <dd>

A database that stores a collection of representations of network objects.

</dd> <dt>

<span id="wmi_v2.gloss_metamodel"></span><span id="WMI_V2.GLOSS_METAMODEL"></span>**metamodel**
</dt> <dd>

A CIM component that describes the entities and relationships that represent *managed objects*. For example, classes, instances, and associations are included in the metamodel.

</dd> <dt>

<span id="wmi_v2.gloss_metaschema"></span><span id="WMI_V2.GLOSS_METASCHEMA"></span>**metaschema**
</dt> <dd>

The [*schema*](gloss-s.md#wmi-v2-gloss-schema) of the *metamodel*.

</dd> <dt>

<span id="wmi_v2.gloss_method_provider"></span><span id="WMI_V2.GLOSS_METHOD_PROVIDER"></span>**method provider**
</dt> <dd>

A provider that enables WMI access to the methods of a class.

</dd> <dt>

<span id="wmi_v2.gloss_wmi_class"></span><span id="WMI_V2.GLOSS_WMI_CLASS"></span>**MI Class**
</dt> <dd>

A template for a type of *managed object*. WMI classes define the basic units of management.

</dd> <dt>

<span id="wmi_v2.gloss_wmi_class_provider"></span><span id="WMI_V2.GLOSS_WMI_CLASS_PROVIDER"></span>**MI Class Provider**
</dt> <dd>

A COM server that supplies class definitions. Class providers implement methods in the [**IWbemServices**](https://msdn.microsoft.com/library/aa392093) interface to support data retrieval, modification, deletion, enumeration, and query processing.

</dd> <dt>

<span id="wmi_v2.gloss_mib"></span><span id="WMI_V2.GLOSS_MIB"></span>**MIB**
</dt> <dd>

See *management information base (MIB)*.

</dd> <dt>

<span id="wmi_v2.gloss_mof"></span><span id="WMI_V2.GLOSS_MOF"></span>**MOF**
</dt> <dd>

See *Managed Object Format (MOF)*.

</dd> <dt>

<span id="wmi_v2.gloss_mof_file"></span><span id="WMI_V2.GLOSS_MOF_FILE"></span>**MOF file**
</dt> <dd>

A text file that contains definitions of classes and instances specified using the *Managed Object Format (MOF)* language.

</dd> </dl>
