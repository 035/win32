# Structures

This section presents information about the following enumerations used by Bitmap Effects:



| Topic                                           | Contents                                                                    |
|-------------------------------------------------|-----------------------------------------------------------------------------|
| [**MIL\_2DPOINTD**](/windows/previous-versions/Mileffects/ns-mileffects-milpoint2d?branch=master)     | Represents an x- and y-coordinate pair in two-dimensional space.<br/> |
| [**MIL\_MATRIX3X2D**](/windows/previous-versions/Mileffects/ns-mileffects-milmatrix3x2d?branch=master) | Represents a 3x3 matrix.<br/>                                         |
| [**MIL\_RECTD**](/windows/previous-versions/Mileffects/ns-mileffects-milrectd?branch=master)           | Describes the width, height, and location of a rectangle.<br/>        |
| [**MILMatrixF**](/windows/previous-versions/Mileffects/ns-mileffects-milmatrixf?branch=master)          | Represents a 4x4 affine transformation matrix.<br/>                   |
