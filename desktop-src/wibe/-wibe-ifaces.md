# Interfaces

This section presents information about the following enumerations used by Bitmap Effects:



| Topic                                                                                            | Contents                                                                                                                                           |
|--------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IMILBitmapEffect**](-wibe-imilbitmapeffect.md)                                               | Exposes methods that define a Windows Presentation Foundation (WPF) bitmap effect.<br/>                                                      |
| [**IMILBitmapEffectConnections**](-wibe-imilbitmapeffectconnections.md)                         | Exposes methods used to retrieve input and output connectors exposed by the bitmap effect.<br/>                                              |
| [**IMILBitmapEffectConnectionsInfo**](-wibe-imilbitmapeffectconnectionsinfo.md)                 | Exposes methods that retrieves information about what input and output pins are exposed by the bitmap effect. <br/>                          |
| [**IMILBitmapEffectConnector**](-wibe-imilbitmapeffectconnector.md)                             | Exposes methods that define an effect output pin.<br/>                                                                                       |
| [**IMILBitmapEffectConnectorInfo**](-wibe-imilbitmapeffectconnectorinfo.md)                     | Exposes methods that retrieve information about a specific input or output connector pin.<br/>                                               |
| [**IMILBitmapEffectEvents**](-wibe-imilbitmapeffectevents.md)                                   | Exposes methods that define an effect event.<br/>                                                                                            |
| [**IMILBitmapEffectFactory**](-wibe-imilbitmapeffectfactory.md)                                 | Exposes methods used to create WPF Microsoft Win32 bitmap effect objects.<br/>                                                               |
| [**IMILBitmapEffectGroup**](-wibe-imilbitmapeffectgroup.md)                                     | Exposes methods used to access a group of effects.<br/>                                                                                      |
| [**IMILBitmapEffectGroupImpl**](-wibe-imilbitmapeffectgroupimpl.md)                             | Exposes methods that define an effect group.<br/>                                                                                            |
| [**IMILBitmapEffectImpl**](-wibe-imilbitmapeffectimpl.md)                                       | Exposes methods that define an an out [**IMILBitmapEffect**](-wibe-imilbitmapeffect.md) object.<br/>                                        |
| [**IMILBitmapEffectInputConnector**](-wibe-imilbitmapeffectinputconnector.md)                   | Exposes methods that define an input connect.<br/>                                                                                           |
| [**IMILBitmapEffectInteriorInputConnector**](-wibe-imilbitmapeffectinteriorinputconnector.md)   | Exposes methods that define an interior input connector.<br/>                                                                                |
| [**IMILBitmapEffectInteriorOutputConnector**](-wibe-imilbitmapeffectinterioroutputconnector.md) | Exposes methods that define an interior output connector.<br/>                                                                               |
| [**IMILBitmapEffectOutputConnector**](-wibe-imilbitmapeffectoutputconnector.md)                 | Exposes methods that define an output connector.<br/>                                                                                        |
| [**IMILBitmapEffectOutputConnectorImpl**](-wibe-imilbitmapeffectoutputconnectorimpl.md)         | Exposes methods that define an output connector.<br/>                                                                                        |
| [**IMILBitmapEffectPrimitive**](-wibe-imilbitmapeffectprimitive.md)                             | Exposes methods that create a bitmap effect's output. This interface must be implemented to create third party WPF bitmap effects.<br/>      |
| [**IMILBitmapEffectPrimitiveImpl**](-wibe-imilbitmapeffectprimitiveimpl.md)                     | Exposes methods that report the state of a bitmap effect. This interface must be implemented to create third party WPF bitmap effects. <br/> |
| [**IMILBitmapEffectRenderContext**](-wibe-imilbitmapeffectrendercontext.md)                     | Exposes methods that define a [**IMILBitmapEffectRenderContext**](-wibe-imilbitmapeffectrendercontext.md) object.<br/>                      |
| [**IMILBitmapEffectRenderContextImpl**](-wibe-imilbitmapeffectrendercontextimpl.md)             | Exposes methods that define a [**IMILBitmapEffectRenderContext**](-wibe-imilbitmapeffectrendercontext.md).<br/>                             |
| [**IMILBitmapEffects**](-wibe-imilbitmapeffects.md)                                             | Exposes methods that define an enumeration of effects.<br/>                                                                                  |
