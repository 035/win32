# Interfaces

This section presents information about the following enumerations used by Bitmap Effects:



| Topic                                                                                            | Contents                                                                                                                                           |
|--------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IMILBitmapEffect**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffect?branch=master)                                               | Exposes methods that define a Windows Presentation Foundation (WPF) bitmap effect.<br/>                                                      |
| [**IMILBitmapEffectConnections**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectconnections?branch=master)                         | Exposes methods used to retrieve input and output connectors exposed by the bitmap effect.<br/>                                              |
| [**IMILBitmapEffectConnectionsInfo**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectconnectionsinfo?branch=master)                 | Exposes methods that retrieves information about what input and output pins are exposed by the bitmap effect. <br/>                          |
| [**IMILBitmapEffectConnector**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectconnector?branch=master)                             | Exposes methods that define an effect output pin.<br/>                                                                                       |
| [**IMILBitmapEffectConnectorInfo**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectconnectorinfo?branch=master)                     | Exposes methods that retrieve information about a specific input or output connector pin.<br/>                                               |
| [**IMILBitmapEffectEvents**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectevents?branch=master)                                   | Exposes methods that define an effect event.<br/>                                                                                            |
| [**IMILBitmapEffectFactory**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectfactory?branch=master)                                 | Exposes methods used to create WPF Microsoft Win32 bitmap effect objects.<br/>                                                               |
| [**IMILBitmapEffectGroup**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectgroup?branch=master)                                     | Exposes methods used to access a group of effects.<br/>                                                                                      |
| [**IMILBitmapEffectGroupImpl**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectgroupimpl?branch=master)                             | Exposes methods that define an effect group.<br/>                                                                                            |
| [**IMILBitmapEffectImpl**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectimpl?branch=master)                                       | Exposes methods that define an an out [**IMILBitmapEffect**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffect?branch=master) object.<br/>                                        |
| [**IMILBitmapEffectInputConnector**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectinputconnector?branch=master)                   | Exposes methods that define an input connect.<br/>                                                                                           |
| [**IMILBitmapEffectInteriorInputConnector**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectinteriorinputconnector?branch=master)   | Exposes methods that define an interior input connector.<br/>                                                                                |
| [**IMILBitmapEffectInteriorOutputConnector**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectinterioroutputconnector?branch=master) | Exposes methods that define an interior output connector.<br/>                                                                               |
| [**IMILBitmapEffectOutputConnector**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectoutputconnector?branch=master)                 | Exposes methods that define an output connector.<br/>                                                                                        |
| [**IMILBitmapEffectOutputConnectorImpl**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectoutputconnectorimpl?branch=master)         | Exposes methods that define an output connector.<br/>                                                                                        |
| [**IMILBitmapEffectPrimitive**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectprimitive?branch=master)                             | Exposes methods that create a bitmap effect's output. This interface must be implemented to create third party WPF bitmap effects.<br/>      |
| [**IMILBitmapEffectPrimitiveImpl**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectprimitiveimpl?branch=master)                     | Exposes methods that report the state of a bitmap effect. This interface must be implemented to create third party WPF bitmap effects. <br/> |
| [**IMILBitmapEffectRenderContext**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectrendercontext?branch=master)                     | Exposes methods that define a [**IMILBitmapEffectRenderContext**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectrendercontext?branch=master) object.<br/>                      |
| [**IMILBitmapEffectRenderContextImpl**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectrendercontextimpl?branch=master)             | Exposes methods that define a [**IMILBitmapEffectRenderContext**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffectrendercontext?branch=master).<br/>                             |
| [**IMILBitmapEffects**](/windows/previous-versions/Mileffects/nn-mileffects-imilbitmapeffects?branch=master)                                             | Exposes methods that define an enumeration of effects.<br/>                                                                                  |
