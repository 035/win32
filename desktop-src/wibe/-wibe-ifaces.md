# Interfaces

This section presents information about the following enumerations used by Bitmap Effects:



| Topic                                                                                            | Contents                                                                                                                                           |
|--------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IMILBitmapEffect**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffect)                                               | Exposes methods that define a Windows Presentation Foundation (WPF) bitmap effect.<br/>                                                      |
| [**IMILBitmapEffectConnections**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectconnections)                         | Exposes methods used to retrieve input and output connectors exposed by the bitmap effect.<br/>                                              |
| [**IMILBitmapEffectConnectionsInfo**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectconnectionsinfo)                 | Exposes methods that retrieves information about what input and output pins are exposed by the bitmap effect. <br/>                          |
| [**IMILBitmapEffectConnector**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectconnector)                             | Exposes methods that define an effect output pin.<br/>                                                                                       |
| [**IMILBitmapEffectConnectorInfo**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectconnectorinfo)                     | Exposes methods that retrieve information about a specific input or output connector pin.<br/>                                               |
| [**IMILBitmapEffectEvents**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectevents)                                   | Exposes methods that define an effect event.<br/>                                                                                            |
| [**IMILBitmapEffectFactory**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectfactory)                                 | Exposes methods used to create WPF Microsoft Win32 bitmap effect objects.<br/>                                                               |
| [**IMILBitmapEffectGroup**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectgroup)                                     | Exposes methods used to access a group of effects.<br/>                                                                                      |
| [**IMILBitmapEffectGroupImpl**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectgroupimpl)                             | Exposes methods that define an effect group.<br/>                                                                                            |
| [**IMILBitmapEffectImpl**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectimpl)                                       | Exposes methods that define an an out [**IMILBitmapEffect**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffect) object.<br/>                                        |
| [**IMILBitmapEffectInputConnector**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectinputconnector)                   | Exposes methods that define an input connect.<br/>                                                                                           |
| [**IMILBitmapEffectInteriorInputConnector**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectinteriorinputconnector)   | Exposes methods that define an interior input connector.<br/>                                                                                |
| [**IMILBitmapEffectInteriorOutputConnector**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectinterioroutputconnector) | Exposes methods that define an interior output connector.<br/>                                                                               |
| [**IMILBitmapEffectOutputConnector**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectoutputconnector)                 | Exposes methods that define an output connector.<br/>                                                                                        |
| [**IMILBitmapEffectOutputConnectorImpl**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectoutputconnectorimpl)         | Exposes methods that define an output connector.<br/>                                                                                        |
| [**IMILBitmapEffectPrimitive**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectprimitive)                             | Exposes methods that create a bitmap effect's output. This interface must be implemented to create third party WPF bitmap effects.<br/>      |
| [**IMILBitmapEffectPrimitiveImpl**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectprimitiveimpl)                     | Exposes methods that report the state of a bitmap effect. This interface must be implemented to create third party WPF bitmap effects. <br/> |
| [**IMILBitmapEffectRenderContext**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectrendercontext)                     | Exposes methods that define a [**IMILBitmapEffectRenderContext**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectrendercontext) object.<br/>                      |
| [**IMILBitmapEffectRenderContextImpl**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectrendercontextimpl)             | Exposes methods that define a [**IMILBitmapEffectRenderContext**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffectrendercontext).<br/>                             |
| [**IMILBitmapEffects**](/previous-versions/windows/desktop/api/Mileffects/nn-mileffects-imilbitmapeffects)                                             | Exposes methods that define an enumeration of effects.<br/>                                                                                  |
