# IMusicDisplayProperties interface

Provides properties for music information that is displayed by [**ISystemMediaTransportControlsDisplayUpdater**](isystemmediatransportcontrolsdisplayupdater.md).

## Members

The **IMusicDisplayProperties** interface inherits from [**IInspectable**](https://msdn.microsoft.com/library/windows/desktop/br205821). **IMusicDisplayProperties** also has these types of members:

-   [Methods](#methods)

### Methods

The **IMusicDisplayProperties** interface has these methods.



| Method                                                              | Description                                          |
|:--------------------------------------------------------------------|:-----------------------------------------------------|
| [**get\_AlbumArtist**](imusicdisplayproperties-get-albumartist.md) | Gets the album artist of the music item. <br/> |
| [**get\_Artist**](imusicdisplayproperties-get-artist.md)           | Gets the artist of the music item.<br/>        |
| [**get\_Title**](imusicdisplayproperties-get-title.md)             | Gets the title of the music item.<br/>         |
| [**put\_AlbumArtist**](imusicdisplayproperties-put-albumartist.md) | Sets the album artist of the music item.<br/>  |
| [**put\_Artist**](imusicdisplayproperties-put-artist.md)           | Sets the artist of the music item.<br/>        |
| [**put\_Title**](imusicdisplayproperties-put-title.md)             | Sets the title of the music item.<br/>         |
