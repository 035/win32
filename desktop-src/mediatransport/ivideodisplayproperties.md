# IVideoDisplayProperties interface

Provides properties for video information that is displayed by [**ISystemMediaTransportControlsDisplayUpdater**](isystemmediatransportcontrolsdisplayupdater.md).

## Members

The **IVideoDisplayProperties** interface inherits from [**IInspectable**](https://msdn.microsoft.com/library/windows/desktop/br205821). **IVideoDisplayProperties** also has these types of members:

-   [Methods](#methods)

### Methods

The **IVideoDisplayProperties** interface has these methods.



| Method                                                        | Description                                |
|:--------------------------------------------------------------|:-------------------------------------------|
| [**get\_Subtitle**](ivideodisplayproperties-get-subtitle.md) | Gets the subtitle of the video.<br/> |
| [**get\_Title**](ivideodisplayproperties-get-title.md)       | Gets the title of the video.<br/>    |
| [**put\_Subtitle**](ivideodisplayproperties-put-subtitle.md) | Sets the subtitle of the video.<br/> |
| [**put\_Title**](ivideodisplayproperties-put-title.md)       | Sets the title of the video.<br/>    |
