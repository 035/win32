# ISystemMediaTransportControlsButtonPressedEventArgs interface

Provides data for the [**ButtonPressed**](https://msdn.microsoft.com/library/windows/apps/dn278706) event.

## Members

The **ISystemMediaTransportControlsButtonPressedEventArgs** interface inherits from [**IInspectable**](https://msdn.microsoft.com/library/windows/desktop/br205821). **ISystemMediaTransportControlsButtonPressedEventArgs** also has these types of members:

-   [Methods](#methods)

### Methods

The **ISystemMediaTransportControlsButtonPressedEventArgs** interface has these methods.



| Method                                                                                           | Description                                                                          |
|:-------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------|
| [**get\_Button**](https://msdn.microsoft.com/library/windows/desktop/dn892301) | Gets a value indicating the button that was pressed to trigger the event.<br/> |
