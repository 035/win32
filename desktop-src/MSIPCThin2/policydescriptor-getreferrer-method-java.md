# PolicyDescriptor.getReferrer method

Gets the optional referrer. This contains a URI which should be contacted when requesting more access than is currently granted.

## Signature

``` syntax
public Uri getReferrer()
```

## Returns

A **Uri** containing the URI of the referrer.

## Defined in

PolicyDescriptor.java
