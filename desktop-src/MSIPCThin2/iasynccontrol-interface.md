# IAsyncControl interface

Provides a cancel method for some asynchronous operations.

## Signature

``` syntax
public interface IAsyncControl
```

## Methods



| Name                                                     | Description                                      |
|----------------------------------------------------------|--------------------------------------------------|
| [**cancel**](iasynccontrol-cancel-method.md)<br/> | Cancel the current asynchronous task.<br/> |
