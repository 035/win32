# MSRole viewer property

User will only be able to view the document.

## Signature

``` syntax
+ (NSString *)viewer;
```

## Property Values



| Name                | Datatype                   | Note                                         |
|---------------------|----------------------------|----------------------------------------------|
| *viewer*<br/> | **NSString** \*<br/> | User cannot edit, copy, or print.<br/> |



 

## Defined in

MSRole.h
