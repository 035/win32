# LicenseMetadata getContentPath method

Gets the path of the content to be tracked. This property does not include the file name. For more information see [**getContentName**](licensemetadata-getcontentname-method-java.md), [**setContentName**](licensemetadata-setcontentname-method-java.md) and [**setContentPath**](licensemetadata-setcontentpath-method-java.md).

