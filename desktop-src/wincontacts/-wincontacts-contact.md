# Contact object

Do not use. The **Contact** object implements the [**IContact**](/previous-versions/windows/desktop/api/icontact/nn-icontact-icontact) interface. It represents reading and writing properties for a single contact.

## Remarks

### Interfaces Implemented



[**IContact**](/previous-versions/windows/desktop/api/icontact/nn-icontact-icontact)
