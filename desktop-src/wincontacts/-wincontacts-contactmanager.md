# ContactManager object

Do not use. The **ContactManager** object implements the [**IContactManager**](/previous-versions/windows/desktop/api/icontact/nn-icontact-icontactmanager) interface. It represents interactions with contacts that have contact ID strings.

## Remarks

### Interfaces Implemented



[**IContactManager**](/previous-versions/windows/desktop/api/icontact/nn-icontact-icontactmanager)
