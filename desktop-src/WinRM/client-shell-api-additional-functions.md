# Client Shell API Additional Functions

The following table provides an overview of the non-core functions for the Windows Remote Management (WinRM) Client Shell application programming interface (API).



| Additional functions                                                   | Description                       |
|------------------------------------------------------------------------|-----------------------------------|
| [**WSManGetErrorMessage**](/windows/win32/Wsman/nf-wsman-wsmangeterrormessage?branch=master)                   | Gets error messages.              |
| [**WSManGetSessionOptionAsDword**](/windows/win32/Wsman/nf-wsman-wsmangetsessionoptionasdword?branch=master)   | Gets session options as a DWORD.  |
| [**WSManGetSessionOptionAsString**](/windows/win32/Wsman/nf-wsman-wsmangetsessionoptionasstring?branch=master) | Gets session options as a string. |
| [**WSManSetSessionOption**](/windows/win32/Wsman/nf-wsman-wsmansetsessionoption?branch=master)                 | Sets session options.             |
