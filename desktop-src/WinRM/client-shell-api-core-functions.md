# Client Shell API Core Functions

The following table provides an overview of the core functions for the Windows Remote Management (WinRM) Client Shell API.



| Core functions                                                   | Description                                                                                                                                                                     |
|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WSManCloseCommand**](wsmanclosecommand.md)                   | Closes a command.                                                                                                                                                               |
| [**WSManCloseOperation**](wsmancloseoperation.md)               | Closes an operation.                                                                                                                                                            |
| [**WSManCloseSession**](wsmanclosesession.md)                   | Closes a WinRM session.                                                                                                                                                         |
| [**WSManCloseShell**](wsmancloseshell.md)                       | Closes a shell object.                                                                                                                                                          |
| [**WSManConnectShell**](wsmanconnectshell.md)                   | Connects to an existing server session.                                                                                                                                         |
| [**WSManConnectShellCommand**](wsmanconnectshellcommand.md)     | Connects to an existing command running in a shell.                                                                                                                             |
| [**WSManCreateSession**](wsmancreatesession.md)                 | Creates a WinRM session.                                                                                                                                                        |
| [**WSManCreateShell**](wsmancreateshell.md)                     | Creates a shell object.                                                                                                                                                         |
| [**WSManCreateShellEx**](wsmancreateshellex.md)                 | Creates a shell object by using the same functionality as the [**WSManCreateShell**](wsmancreateshell.md) function, with the addition of a client-specified shell ID.          |
| [**WSManDeinitialize**](wsmandeinitialize.md)                   | Deinitializes the WinRM client stack.                                                                                                                                           |
| [**WSManDisconnectShell**](wsmandisconnectshell.md)             | Disconnects the network connection of an active shell and its associated commands.                                                                                              |
| [**WSManInitialize**](wsmaninitialize.md)                       | Initializes WinRM.                                                                                                                                                              |
| [**WSManReceiveShellOutput**](wsmanreceiveshelloutput.md)       | Receives shell output.                                                                                                                                                          |
| [**WSManReconnectShell**](wsmanreconnectshell.md)               | Reconnects a previously disconnected shell session. To reconnect the shell session's associated commands, use [**WSManReconnectShellCommand**](wsmanreconnectshellcommand.md). |
| [**WSManReconnectShellCommand**](wsmanreconnectshellcommand.md) | Reconnects a previously disconnected command.                                                                                                                                   |
| [**WSManRunShellCommand**](wsmanrunshellcommand.md)             | Runs a shell command.                                                                                                                                                           |
| [**WSManRunShellCommandEx**](wsmanrunshellcommandex.md)         | Provides the same functionality as the [**WSManRunShellCommand**](wsmanrunshellcommand.md) function, with the addition of a command ID option.                                 |
| [**WSManSendShellInput**](wsmansendshellinput.md)               | Sends input to a shell.                                                                                                                                                         |
| [**WSManSignalShell**](wsmansignalshell.md)                     | Signals a shell.                                                                                                                                                                |
