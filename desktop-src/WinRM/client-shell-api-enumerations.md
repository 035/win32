# Client Shell API Enumerations

The following table provides an overview of the enumerations in the Windows Remote Management (WinRM) Client Shell application programming interface (API).



| Enumerations                                                 | Description                                                                                      |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**WSManAuthenticationFlags**](wsmanauthenticationflags.md) | Defines the authentication scheme to use for both server and proxy authentication.               |
| [**WSManCallbackFlags**](wsmancallbackflags.md)             | Defines a set of flags used by all callback functions.                                           |
| [**WSManDataType**](wsmandatatype.md)                       | Specifies the current data type of the union in the [**WSMAN\_DATA**](wsman-data.md) structure. |
| [**WSManProxyAccessType**](wsmanproxyaccesstype.md)         | Defines the proxy access type.                                                                   |
| [**WSManSessionOption**](wsmansessionoption.md)             | Defines a set of extended options for the session.                                               |
