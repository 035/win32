# Client Shell API Enumerations

The following table provides an overview of the enumerations in the Windows Remote Management (WinRM) Client Shell application programming interface (API).



| Enumerations                                                 | Description                                                                                      |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**WSManAuthenticationFlags**](/windows/win32/Wsman/ne-wsman-wsmanauthenticationflags?branch=master) | Defines the authentication scheme to use for both server and proxy authentication.               |
| [**WSManCallbackFlags**](/windows/win32/Wsman/ne-wsman-wsmancallbackflags?branch=master)             | Defines a set of flags used by all callback functions.                                           |
| [**WSManDataType**](/windows/win32/Wsman/ne-wsman-wsmandatatype?branch=master)                       | Specifies the current data type of the union in the [**WSMAN\_DATA**](/windows/win32/Wsman/ns-wsman-_wsman_data?branch=master) structure. |
| [**WSManProxyAccessType**](/windows/win32/Wsman/ne-wsman-wsmanproxyaccesstype?branch=master)         | Defines the proxy access type.                                                                   |
| [**WSManSessionOption**](/windows/win32/Wsman/ne-wsman-wsmansessionoption?branch=master)             | Defines a set of extended options for the session.                                               |
