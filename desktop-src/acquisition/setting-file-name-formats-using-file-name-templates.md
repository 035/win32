# Setting File Name Formats Using File Name Templates

A common task when acquiring photos is setting the format of the file names. [**IPhotoAcquireSettings::SetOutputFilenameTemplate**](/windows/win32/PhotoAcquire/nf-photoacquire-iphotoacquiresettings-setoutputfilenametemplate?branch=master) enables you to specify a format string (template) that specifies the format of file names. For more information about using file name templates, see **IPhotoAcquireSettings::SetOutputFilenameTemplate**.
