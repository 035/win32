# About the Picture Acquisition API

The Picture Acquisition application programming interface (API) is designed to allow applications to do the following:

-   Enumerate available image acquisition devices
-   Enumerate and filter image items for acquisition
-   Acquire images from a device using standard and high-performance transfer mechanisms
-   Provide standard and customizable dialog boxes for device selection, acquisition options, and image transfer progress
-   Be notified of events that may occur during image acquisition
