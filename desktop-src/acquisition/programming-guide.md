# Programming Guide

The Programming Guide is intended to help you learn how to use the Picture Acquisition Software application programming interface (API) to build applications that manage the photo acquisition process. It is divided into the following sections.



| Section                                      | Description                                                                                                                 |
|----------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [Acquiring Images](acquiring-images.md)     | Describes how to use the Picture Acquisition API to acquire images from a device from within an application.                |
| [Creating a Plug-in](creating-a-plug-in.md) | Describes how to create a plug-in that extends the Windows experience and runs each time images are acquired from a device. |
