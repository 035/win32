# DRM\_LicenseState\_PlaylistBurn

The **DRM\_LicenseState\_PlaylistBurn** property contains a [**WM\_LICENSE\_STATE\_DATA**](wm-license-state-data.md) structure that contains details about how this right has been applied to the content.

## Global Constant

g\_wszWMDRM\_LicenseState\_PlaylistBurn

## Data Type

**WMT\_TYPE\_BINARY**

## Remarks

This is a read-only property that is retrieved using [**IWMDRMReader::GetDRMProperty**](iwmdrmreader-getdrmproperty.md).
