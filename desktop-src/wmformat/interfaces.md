# Interfaces

The following two tables describe the interfaces supported by this SDK and show their relationship to objects and other interfaces.



| Interface                                                          | Description                                                                                                                                                                                                                                                       |
|--------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**INSNetSourceCreator**](/windows/win32/wmnetsourcecreator/nn-wmnetsourcecreator-insnetsourcecreator?branch=master)                 | Creates an administrative network source [*plug-in*](wmformat-glossary.md#wmformat-plug-in), which can be used to cache passwords and locate proxy servers.                                                                                                      |
| [**INSSBuffer**](/windows/win32/wmsbuffer/nn-wmsbuffer-inssbuffer?branch=master)                                   | Controls a buffer stored in a buffer object. This interface is used for passing samples to and from methods in this SDK.                                                                                                                                          |
| [**INSSBuffer2**](/windows/win32/wmsbuffer/nn-wmsbuffer-inssbuffer2?branch=master)                                 | Inherits from **INSSBuffer**. The new methods of this interface are undocumented because they are not implemented in this release.                                                                                                                                |
| [**INSSBuffer3**](/windows/win32/wmsbuffer/nn-wmsbuffer-inssbuffer3?branch=master)                                 | Sets and retrieves buffer properties. Buffer properties are used to convey information with a buffer when passed to or from methods of this SDK.                                                                                                                  |
| [**INSSBuffer4**](/windows/win32/wmsbuffer/nn-wmsbuffer-inssbuffer4?branch=master)                                 | Enumerates buffer properties.                                                                                                                                                                                                                                     |
| [**IReferenceClock**](ireferenceclock.md)                         | Provides access to a standard reference clock.                                                                                                                                                                                                                    |
| [**IRMGetLicense**](irmgetlicense.md)                             | Performs client-side license acquisition.                                                                                                                                                                                                                         |
| [**IWMAddressAccess**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmaddressaccess?branch=master)                       | Controls IP access lists on the writer network sink object.                                                                                                                                                                                                       |
| [**IWMAddressAccess2**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmaddressaccess2?branch=master)                     | Adds IPv6 support to the **IWMAddressAccess** interface.                                                                                                                                                                                                          |
| [**IWMAuthorizer**](/windows/win32/wmsecure/?branch=master)                             | Provides access to certificates.                                                                                                                                                                                                                                  |
| [**IWMBackupRestoreProps**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmbackuprestoreprops?branch=master)             | Sets, retrieves, and removes the properties required by the **IWMLicenseBackup** and **IWMLicenseRestore** interfaces.                                                                                                                                            |
| [**IWMBandwidthSharing**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmbandwidthsharing?branch=master)                 | Configures bandwidth sharing objects. Bandwidth sharing objects are created by profile objects to specify streams that, regardless of their individual bit rates, will never use more than a certain amount of bandwidth between them.                            |
| [**IWMClientConnections**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmclientconnections?branch=master)               | Retrieves information about clients connected to a writer network sink object.                                                                                                                                                                                    |
| [**IWMClientConnections2**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmclientconnections2?branch=master)             | Retrieves advanced information about clients connected to a writer network sink object.                                                                                                                                                                           |
| [**IWMCodecAMVideoAccelerator**](/windows/win32/wmdxva/nn-wmdxva-iwmcodecamvideoaccelerator?branch=master)   | Exposed by the Windows Media Decoder [*DMO*](wmformat-glossary.md#wmformat-directx-media-object--dmo-) and called by a media player's source filter to set up the various connections required to enable DirectX VA for decoding of Windows Media Video content. |
| [**IWMCodecInfo**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmcodecinfo?branch=master)                               | Retrieves the available formats for the supported audio codecs loaded on the system. These formats are used when creating audio streams in profiles.                                                                                                              |
| [**IWMCodecInfo2**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmcodecinfo2?branch=master)                             | Provides access to the names of codecs and descriptions of the formats supported by each.                                                                                                                                                                         |
| [**IWMCodecInfo3**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmcodecinfo3?branch=master)                             | Provides access to codec properties and audio format properties. Also enables you to query codecs for feature support.                                                                                                                                            |
| **IWMCodecVideoAccelerator**                                       | This interface is not supported. Use [**IWMCodecAMVideoAccelerator**](/windows/win32/wmdxva/nn-wmdxva-iwmcodecamvideoaccelerator?branch=master) instead.                                                                                                                                                    |
| [**IWMCredentialCallback**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmcredentialcallback?branch=master)             | Acquires the credentials of users and checks that they have permission to access a remote site.                                                                                                                                                                   |
| [**IWMDeviceRegistration**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmdeviceregistration?branch=master)             | Manages the device registration database, which contains entries for digital media playback devices connected to local networks.                                                                                                                                  |
| [**IWMDRMEditor**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmdrmeditor?branch=master)                               | Enables applications to examine [*DRM*](wmformat-glossary.md#wmformat-digital-rights-management--drm-) header properties without having a license for the protected content.                                                                                     |
| [**IWMDRMMessageParser**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmdrmmessageparser?branch=master)                 | Parses messages received from digital media playback devices.                                                                                                                                                                                                     |
| [**IWMDRMReader**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmdrmreader?branch=master)                               | Acquires licenses, configures [*DRM*](wmformat-glossary.md#wmformat-digital-rights-management--drm-) properties, and individualizes clients.                                                                                                                     |
| [**IWMDRMReader2**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmdrmreader2?branch=master)                             | Reads licenses created by using the Windows Media Rights Manager 10 SDK.                                                                                                                                                                                          |
| [**IWMDRMReader3**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmdrmreader3?branch=master)                             | Enables content transcription by providing a method to get protection systems approved by a license.                                                                                                                                                              |
| [**IWMDRMTranscryptor**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmdrmtranscryptor?branch=master)                   | Converts DRM-protected content into a data stream that can be sent using the Windows Media DRM 10 for Network Devices secure transfer protocol.                                                                                                                   |
| [**IWMDRMWriter**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmdrmwriter?branch=master)                               | Provides license creation capabilities from the writer object.                                                                                                                                                                                                    |
| [**IWMDRMWriter2**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmdrmwriter2?branch=master)                             | Configures the writer to accept stream samples containing data encrypted for streaming to devices that support Windows Media DRM 10 for Network Devices.                                                                                                          |
| [**IWMDRMWriter3**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmdrmwriter3?branch=master)                             | Enables writing of encrypted stream samples for importing protected content.                                                                                                                                                                                      |
| [**IWMGetSecureChannel**](/windows/win32/wmsecure/nn-wmsecure-iwmgetsecurechannel?branch=master)                 | This interface is used by one communication party to get the other party's [**IWMSecureChannel**](/windows/win32/Wmsecure/?branch=master) interface.                                                                                                                                |
| [**IWMHeaderInfo**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmheaderinfo?branch=master)                             | Manages header information, such as metadata, [*markers*](wmformat-glossary.md#wmformat-marker), and so on.                                                                                                                                                      |
| [**IWMHeaderInfo2**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmheaderinfo2?branch=master)                           | Provides access to information about the codecs used to create the file.                                                                                                                                                                                          |
| [**IWMHeaderInfo3**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmheaderinfo3?branch=master)                           | Provides attribute language support and other advanced metadata features.                                                                                                                                                                                         |
| [**IWMImageInfo**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmimageinfo?branch=master)                               | Retrieves ID3v2 "APIC" frames from a file. Use of this interface should be avoided in favor of the [**WM/Picture**](wmpicture.md) attribute.                                                                                                                     |
| [**IWMIndexer**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmindexer?branch=master)                                   | Provides methods to manually index files.                                                                                                                                                                                                                         |
| [**IWMIndexer2**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmindexer2?branch=master)                                 | Provides indexing configuration options.                                                                                                                                                                                                                          |
| [**IWMInputMediaProps**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwminputmediaprops?branch=master)                   | Manages the properties of an input media stream.                                                                                                                                                                                                                  |
| [**IWMIStreamProps**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmistreamprops?branch=master)                         | Reads the properties of an **IStream** object.                                                                                                                                                                                                                    |
| [**IWMLanguageList**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmlanguagelist?branch=master)                         | Manages the language list for an ASF file.                                                                                                                                                                                                                        |
| [**IWMLicenseBackup**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmlicensebackup?branch=master)                       | Backs up licenses, typically so that they can be restored onto another computer.                                                                                                                                                                                  |

| [**IWMMediaProps**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmmediaprops?branch=master)                             | Provides access to the media type structure for a stream.                                                                                                                                                                                                         |
| [**IWMMetadataEditor**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmmetadataeditor?branch=master)                     | Provides file management for metadata editing.                                                                                                                                                                                                                    |
| [**IWMMetadataEditor2**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmmetadataeditor2?branch=master)                   | Provides an improved method for opening files for use by the metadata editor.                                                                                                                                                                                     |
| [**IWMMutualExclusion**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmmutualexclusion?branch=master)                   | Configures mutual exclusion objects. | [**IWMPlayerHook**](/windows/win32/wmsdkidl/nn-wmsdkidl-iwmplayerhook?branch=master)                             | Implemented by a player application that uses DirectX Video Acceleration (DirectX VA). ## Remarks

The interface identifier for each of the interfaces in this SDK is the name of the interface preceded by "IID\_". For example, the identifier for the **IWMReaderAdvanced** interface is IID\_IWMReaderAdvanced. To obtain a pointer to the **IWMReaderAdvanced** interface of a reader object, use the following code, where *pReader* is a pointer to an **IWMReader** interface:


```C++
IWMReaderAdvanced *pReaderAdvanced;
hr = pReader->QueryInterface(IID_IWMReaderAdvanced,
                             (LPVOID*) &amp;pReaderAdvanced);

```
