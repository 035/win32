# Enumeration Types

The Windows Media Format SDK implements the following enumeration types.



| Enumeration type                                                               | Description                                                                                                                      |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**DRM\_HTTP\_STATUS**](drm-http-status.md)                                   | Defines a range of states for a DRM request.                                                                                     |
| [**DRM\_LICENSE\_STATE\_CATEGORY**](drm-license-state-category.md)            | Defines the categories for DRM license strings.                                                                                  |
| [**DRM\_INDIVIDUALIZATION\_STATUS**](drm-individualization-status.md)         | Defines the valid states for DRM individualization.                                                                              |
| [**NETSOURCE\_URLCREDPOLICY\_SETTINGS**](netsource-urlcredpolicy-settings.md) | Defines possible security policy settings that can exist on a client computer.                                                   |
| [**WM\_AETYPE**](wm-aetype.md)                                                | Specifies the permissions for an entry in an IP address access list.                                                             |
| [**WMT\_ATTR\_DATATYPE**](wmt-attr-datatype.md)                               | Defines a range of basic data types. These values are used to identify the type of data returned by various methods in this SDK. |
| [**WMT\_ATTR\_IMAGETYPE**](wmt-attr-imagetype.md)                             | Defines a range of image types that can be stored in an ASF file header.                                                         |
| [**WMT\_CODEC\_INFO\_TYPE**](wmt-codec-info-type.md)                          | Defines a range of data sizes used by a codec.                                                                                   |
| [**WMT\_CREDENTIAL\_FLAGS**](wmt-credential-flags.md)                         | Defines the flags that can be used when acquiring credentials.                                                                   |
| [**WMT\_DRMLA\_TRUST**](wmt-drmla-trust.md)                                   | Defines the trust state of a DRM license acquisition URL.                                                                        |
| [**WMT\_FILESINK\_MODE**](wmt-filesink-mode.md)                               | Defines the types of input accepted by the file sink.                                                                            |
| [**WMT\_IMAGE\_TYPE**](wmt-image-type.md)                                     | Defines the possible image types used for banner ads.                                                                            |
| [**WMT\_INDEX\_TYPE**](wmt-index-type.md)                                     | Defines the object with which a time-based index can be associated.                                                              |
| [**WMT\_INDEXER\_TYPE**](wmt-indexer-type.md)                                 | Defines the types of indexing supported by the indexer.                                                                          |
| [**WMT\_NET\_PROTOCOL**](wmt-net-protocol.md)                                 | Defines the types of protocols that the network sink supports.                                                                   |
| [**WMT\_MUSICSPEECH\_CLASS\_MODE**](wmt-musicspeech-class-mode.md)            | Defines the compression modes available to the Windows Media Audio 9 Voice codec.                                                |
| [**WMT\_OFFSET\_FORMAT**](wmt-offset-format.md)                               | Defines the types of offsets used in this SDK.                                                                                   |
| [**WMT\_PLAY\_MODE**](wmt-play-mode.md)                                       | Defines the playback options of the reader.                                                                                      |
| [**WMT\_PROXY\_SETTINGS**](wmt-proxy-settings.md)                             | Defines the network proxy settings for use with a reader object.                                                                 |
| [**WMT\_RIGHTS**](wmt-rights.md)                                              | Defines the rights that may be specified in a DRM license.                                                                       |
| [**WMT\_STATUS**](wmt-status.md)                                              | Defines a range of status flags.                                                                                                 |
| [**WMT\_STORAGE\_FORMAT**](wmt-storage-format.md)                             | Defines the file types that can be manipulated with this SDK.                                                                    |
| [**WMT\_STREAM\_SELECTION**](wmt-stream-selection.md)                         | Defines the status of a stream.                                                                                                  |
| [**WMT\_TRANSPORT\_TYPE**](wmt-transport-type.md)                             | Defines the transport types supported by this SDK.                                                                               |
| [**WMT\_VERSION**](wmt-version.md)                                            | Defines the version numbers of the Windows Media Format SDK.                                                                     |
| [**WMT\_WATERMARK\_ENTRY\_TYPE**](wmt-watermark-entry-type.md)                | Defines the types of supported watermarks.                                                                                       |
