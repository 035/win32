# DRM\_IsDRM

The **DRM\_IsDRM** property indicates whether a file is a DRM-protected file.

## Global Constant

g\_wszWMDRM\_IsDRM

## Data Type

**WMT\_TYPE\_BOOL**

This is a read-only property that is retrieved using [**IWMDRMReader::GetDRMProperty**](iwmdrmreader-getdrmproperty.md).
