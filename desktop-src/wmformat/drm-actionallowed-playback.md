# DRM\_ActionAllowed\_Playback

The **DRM\_ActionAllowed\_Playback** attribute indicates whether playback of the content is allowed.

## Global Constant

g\_wszWMDRM\_ActionAllowed\_Playback

## Data Type

**WMT\_TYPE\_BOOL**

## Remarks

This is a read-only property that is retrieved using [**IWMDRMReader::GetDRMProperty**](/windows/win32/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty?branch=master).
