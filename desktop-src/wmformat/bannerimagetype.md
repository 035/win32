# BannerImageType

The **BannerImageType** attribute specifies the type of image stored in the [**BannerImageData**](bannerimagedata.md) attribute. This attribute is a value from the [**WMT\_IMAGE\_TYPE**](/windows/win32/Wmsdkidl/ne-wmsdkidl-wmt_image_type?branch=master) enumeration type.

## Global Constant

g\_wszWMBannerImageType

## Data Type

**WMT\_TYPE\_DWORD**
