# DRM\_LASignatureCert

The **DRM\_LASignatureCert** attribute contains the certificate that authenticates the signature in the DRM header.

## Global Constant

g\_wszWMDRM\_LASignatureCert

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

This property can be set with the [**IWMDRMWriter::SetDRMAttribute**](/windows/win32/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute?branch=master) method. It is not accessible to the reader object.
