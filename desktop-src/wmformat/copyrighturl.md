# CopyrightURL

The **CopyrightURL** attribute contains the Web address of a copyright page associated with branding.

## Global Constant

g\_wszWMCopyrightURL

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

This attribute applies to the entire file.
