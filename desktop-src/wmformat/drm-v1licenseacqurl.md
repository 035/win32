# DRM\_V1LicenseAcqURL

The **DRM\_V1LicenseAcqURL** attribute contains the address of a Web site where a user can obtain a DRM version 1 license.

## Global Constant

g\_wszWMDRM\_V1LicenseAcqURL

## Data Type

**WMT\_TYPE\_STRING**

## Remarks

This attribute can be set using [**IWMDRMWriter::SetDRMAttribute**](/windows/win32/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute?branch=master) and it can be retrieved with [**IWMDRMReader::GetDRMProperty**](/windows/win32/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty?branch=master).
