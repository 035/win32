# Battery Class Driver Routines

## 

The battery class driver exports the following routines for use by miniclass drivers:

<dl>

[**BatteryClassInitializeDevice**](batteryclassinitializedevice.md)  
[**BatteryClassIoctl**](batteryclassioctl.md)  
[**BatteryClassQueryWmiDataBlock**](batteryclassquerywmidatablock.md)  
[**BatteryClassStatusNotify**](batteryclassstatusnotify.md)  
[**BatteryClassSystemControl**](batteryclasssystemcontrol.md)  
[**BatteryClassUnload**](batteryclassunload.md)  
</dl>

These routines are declared in Batclass.h.
