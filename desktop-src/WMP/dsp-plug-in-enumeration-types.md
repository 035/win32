# DSP Plug-in Enumeration Types

The Windows Media Player SDK implements the following enumeration type for creating DSP plug-ins.



| Enumeration Type                                        | Description                                                                                    |
|---------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [WMPPlugin\_Caps](wmpplugin-caps.md)                   | Used with **IWMPPlugin::GetCaps** to indicate whether the plug-in can convert between formats. |
| [WMPServices\_StreamState](wmpservices-streamstate.md) | Indicates the whether the stream is currently stopped, paused, or playing.                     |
