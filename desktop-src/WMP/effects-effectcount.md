# EFFECTS.effectCount

The **effectCount** attribute retrieves the number of visualizations available.

``` syntax
        elementID.effectCount
```

## Possible Values

This attribute is a read-only **Number** (**long**).

## Requirements



|                    |                                                         |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player for Windows XP or later<br/> |
