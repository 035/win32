# LISTBOX.itemCount

The **itemCount** attribute retrieves the number of items in the list box control.

``` syntax
        elementID.itemCount
```

## Possible Values

This attribute is a read-only **Number** (**long**).

## Requirements



|                    |                                                         |
|--------------------|---------------------------------------------------------|
| Version<br/> | Windows Media Player for Windows XP or later<br/> |
