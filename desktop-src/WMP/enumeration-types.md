# Enumeration Types

The Windows Media Player SDK supports the following enumeration types.



| Enumeration type                                                             | Description                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [WMPBurnFormat](wmpburnformat.md)                                           | Defines the possible types of CDs for burning.                                                                         |
| [WMPBurnState](wmpburnstate.md)                                             | Defines the possible operational states of Windows Media Player as it burns a CD.                                      |
| [WMPDeviceStatus](wmpdevicestatus.md)                                       | Defines the possible values for the current status of a device.                                                        |
| [WMPFolderScanState](wmpfolderscanstate.md)                                 | Defines the possible operational states of Windows Media Player as it monitors file folders for digital media content. |
| [WMPLibraryType](wmplibrarytype.md)                                         | Defines the possible library types to which Windows Media Player can connect.                                          |
| [WMPOpenState](wmpopenstate.md)                                             | Defines the possible operational states of Windows Media Player as it opens a digital media file.                      |
| [WMPPlaylistChangeEventType](wmpplaylistchangeeventtype.md)                 | Defines the types of changes that can be made to a playlist.                                                           |
| [WMPPlayState](wmpplaystate.md)                                             | Defines the possible operational states of Windows Media Player as it plays a digital media file.                      |
| [WMPRipState](wmpripstate.md)                                               | Defines the possible operational states of Windows Media Player as it rips a CD.                                       |
| [WMPStringCollectionChangeEventType](wmpstringcollectionchangeeventtype.md) | Defines the types of changes that can occur in a string collection.                                                    |
| [WMPSyncState](wmpsyncstate.md)                                             | Defines the possible operational states of Windows Media Player as it synchronizes digital media to a device.          |
