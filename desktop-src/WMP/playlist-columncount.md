# PLAYLIST.columnCount

The **columnCount** attribute retrieves the number of columns displayed.

``` syntax
        elementID.columnCount
```

## Possible Values

This attribute is a read-only **Number** (**long**).

## Requirements



|                    |                                                      |
|--------------------|------------------------------------------------------|
| Version<br/> | Windows Media Player version 7.0 or later<br/> |
