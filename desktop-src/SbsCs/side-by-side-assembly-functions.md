# Side-by-Side Assembly Functions

The side-by-side assembly API uses the following functions to access the Side-by-Side Assembly interfaces.



| Function                                                     | Description                                                                    |
|--------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**CreateAssemblyCache**](/windows/win32/Winsxs/nf-winsxs-createassemblycache?branch=master)           | Obtains an instance of the [**IAssemblyCache**](/windows/win32/winsxs/nn-winsxs-iassemblycache?branch=master) interface. |
| [**CreateAssemblyNameObject**](/windows/win32/Winsxs/nf-winsxs-createassemblynameobject?branch=master) | Obtains an instance of the [**IAssemblyName**](/windows/win32/winsxs/nn-winsxs-iassemblyname?branch=master) interface.   |
