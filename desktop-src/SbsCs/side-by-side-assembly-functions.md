# Side-by-Side Assembly Functions

The side-by-side assembly API uses the following functions to access the Side-by-Side Assembly interfaces.



| Function                                                     | Description                                                                    |
|--------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**CreateAssemblyCache**](createassemblycache.md)           | Obtains an instance of the [**IAssemblyCache**](iassemblycache.md) interface. |
| [**CreateAssemblyNameObject**](createassemblynameobject.md) | Obtains an instance of the [**IAssemblyName**](iassemblyname.md) interface.   |
