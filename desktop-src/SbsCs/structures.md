# Side-by-Side Assembly Structures

The side-by-side assembly API uses the following enumerations.



| Enumeration                                                     | Description                                                                            |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**ASSEMBLY\_INFO**](assembly-info-.md)                        | Contains information about an assembly in the system's side-by-side assembly store.    |
| [**FUSION\_INSTALL\_REFERENCE**](fusion-install-reference-.md) | Contains information about the application which references the side-by-side assembly. |
