# Side-by-Side Assembly Structures

The side-by-side assembly API uses the following enumerations.



| Enumeration                                                     | Description                                                                            |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**ASSEMBLY\_INFO**](/windows/win32/Winsxs/ns-winsxs-_assembly_info?branch=master)                        | Contains information about an assembly in the system's side-by-side assembly store.    |
| [**FUSION\_INSTALL\_REFERENCE**](/windows/win32/Winsxs/ns-winsxs-_fusion_install_reference_?branch=master) | Contains information about the application which references the side-by-side assembly. |
