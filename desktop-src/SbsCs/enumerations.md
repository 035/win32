# Side-by-Side Assembly Enumerations

The side-by-side assembly API uses the following enumerations.



| Enumeration                                                         | Description                                                                                                                                                                                |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ASM\_CMP\_FLAGS**](/windows/win32/Winsxs/ne-winsxs-__midl_iassemblyname_0004?branch=master)                           | Used by the [**IsEqual**](/windows/win32/winsxs/nf-winsxs-iassemblyname-isequal?branch=master) method to specify which parts of two assembly names to compare.                                                                       |
| [**ASM\_DISPLAY\_FLAGS**](/windows/win32/Winsxs/ne-winsxs-__midl_iassemblyname_0003?branch=master)                   | Used by the [**GetDisplayName**](/windows/win32/winsxs/nf-winsxs-iassemblyname-getdisplayname?branch=master) method to specify which portions of the side-by-side assembly name to include in the string representation of the name. |
| [**ASM\_NAME**](/windows/win32/Winsxs/ne-winsxs-__midl_iassemblyname_0001?branch=master)                                      | Property IDs for the name-value pairs in an side-by-side assembly name.                                                                                                                    |
| [**CREATE\_ASM\_NAME\_OBJ\_FLAGS**](/windows/win32/Winsxs/ne-winsxs-__midl___midl_itf_winsxs_0000_0003_0001?branch=master) | Used by the [**CreateAssemblyNameObject**](/windows/win32/Winsxs/nf-winsxs-createassemblynameobject?branch=master) function to specify the side-by-side assembly name.                                                               |
