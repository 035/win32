# Side-by-Side Assembly Enumerations

The side-by-side assembly API uses the following enumerations.



| Enumeration                                                         | Description                                                                                                                                                                                |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ASM\_CMP\_FLAGS**](asm-cmp-flags-.md)                           | Used by the [**IsEqual**](iassemblyname-isequal.md) method to specify which parts of two assembly names to compare.                                                                       |
| [**ASM\_DISPLAY\_FLAGS**](asm-display-flags-.md)                   | Used by the [**GetDisplayName**](iassemblyname-getdisplayname.md) method to specify which portions of the side-by-side assembly name to include in the string representation of the name. |
| [**ASM\_NAME**](asm-name-.md)                                      | Property IDs for the name-value pairs in an side-by-side assembly name.                                                                                                                    |
| [**CREATE\_ASM\_NAME\_OBJ\_FLAGS**](create-asm-name-obj-flags-.md) | Used by the [**CreateAssemblyNameObject**](createassemblynameobject.md) function to specify the side-by-side assembly name.                                                               |
