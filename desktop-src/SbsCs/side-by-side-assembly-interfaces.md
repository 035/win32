# Side-by-Side Assembly Interfaces

The side-by-side assembly API uses the following interfaces.



| Interface                                        | Description                                                                                                                                            |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAssemblyCache**](/windows/win32/winsxs/nn-winsxs-iassemblycache?branch=master)         | Used to install, uninstall, or query a side-by-side assembly.                                                                                          |
| [**IAssemblyCacheItem**](/windows/win32/winsxs/nn-winsxs-iassemblycacheitem?branch=master) | Used to install side-by-side assemblies into the side-by-side store using a stream-based installation.                                                 |
| [**IAssemblyName**](/windows/win32/winsxs/nn-winsxs-iassemblyname?branch=master)           | Represents a side-by-side assembly name. The side-by-side assembly name consists of a set of name-value pairs that describe the side-by-side assembly. |
