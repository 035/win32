# FaxAccountSet.GetAccounts method

Returns an [**IFaxAccounts**](/previous-versions/windows/desktop/api/FaxComex/nn-faxcomex-ifaxaccounts) object that represents all the fax accounts on the fax server.

## Syntax


```VB
FaxAccountSet.GetAccounts() As IFaxAccounts
```



## Parameters

This method has no parameters.

## Return value

Type: **[**IFaxAccounts**](/previous-versions/windows/desktop/api/FaxComex/nn-faxcomex-ifaxaccounts)\*\***

The address of a pointer to an [**IFaxAccounts**](/previous-versions/windows/desktop/api/FaxComex/nn-faxcomex-ifaxaccounts) object.
