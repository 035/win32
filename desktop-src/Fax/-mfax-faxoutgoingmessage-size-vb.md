# FaxOutgoingMessage.Size property

The Size property is a value that indicates the size of the Tagged Image File Format Class F (TIFF Class F) file associated with the outbound fax message.

This property is read-only.

## Syntax


```VB
Property Size As Long
```



## Property value

A **Long** that receives the size, in bytes, of the TIFF Class F file associated with the outbound fax message.
