# FaxServer.ReceiptOptions property

The **ReceiptOptions** property creates a [**FaxReceiptOptions**](-mfax-faxreceiptoptions.md) configuration object. The object permits a fax client application to set and retrieve the receipt configuration that the fax service uses to send fax receipts.

This property is read-only.

## Syntax


```VB
Property ReceiptOptions As FaxReceiptOptions
```



## Property value

A [**FaxReceiptOptions**](-mfax-faxreceiptoptions.md) object.
