# FaxSender.OfficePhone property

The **OfficePhone** property is a null-terminated string that contains the office telephone number associated with the sender.

This property is read/write.

## Syntax


```VB
Property OfficePhone As String
```



## Property value

A **String** that specifies or receives the office telephone number associated with the sender.
