# FaxConfiguration.RetryDelay property

Sets or retrieves a value that indicates the length of time the fax service should wait before retrying a failed fax transmission.

This property is read/write.

## Syntax


```VB
Property RetryDelay As Integer
```



## Property value

A value of type **Integer** that indicates the length of time the fax service should wait before retrying a failed fax transmission.
