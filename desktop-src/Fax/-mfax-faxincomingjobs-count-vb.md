# FaxIncomingJobs.Count property

The **Count** property represents the number of objects in the [**FaxIncomingJobs**](-mfax-faxincomingjobs.md) collection. This is the total number of incoming jobs for the fax server.

This property is read-only.

## Syntax


```VB
Property Count As Long
```



## Property value

A **Long** that receives the number of objects in the [**FaxIncomingJobs**](-mfax-faxincomingjobs.md) collection.
