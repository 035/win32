# FaxConfiguration.Retries property

Sets or retrieves a value that indicates the number of redial attempts for a given fax job.

This property is read/write.

## Syntax


```VB
Property Retries As Integer
```



## Property value

A value of type **Integer** that indicates the number of redial attempts for a given fax job.
