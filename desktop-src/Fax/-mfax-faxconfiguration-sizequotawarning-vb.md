# FaxConfiguration.SizeQuotaWarning property

Sets or retrieves a value that indicates whether the size quota warning is turned on.

This property is read/write.

## Syntax


```VB
Property SizeQuotaWarning As Boolean
```



## Property value

A value of type **Boolean** that indicates whether the size quota warning is turned on.
