# FaxDeviceIds.Item property

The **Item** property represents a device ID from the [**FaxDeviceIds**](-mfax-faxdeviceids.md) collection.

This property is read/write.

## Syntax


```VB
Property Item( _
  ByVal lIndex As Long _
) As Long
```



## Property value

A **Long** value that receives the item requested.
