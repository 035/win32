# FaxInboundRoutingMethod.ExtensionFriendlyName property

The **ExtensionFriendlyName** property is the user-friendly name for the fax routing extension that exports the inbound fax routing method.

This property is read-only.

## Syntax


```VB
Property ExtensionFriendlyName As String
```



## Property value

A **String** that receives the user-friendly name for the fax routing extension that exports the inbound fax routing method. The string is suitable for display to users.
