# FaxConfiguration.DiscountRateEnd property

Sets or retrieves a value that indicates the time at which the discount rate period ends.

This property is read/write.

## Syntax


```VB
Property DiscountRateEnd As Date
```



## Property value

A value of type **Date** that indicates the time at which the discount rate period ends.
