# FaxOutgoingJobs.Item property

The **Item** property returns a [**FaxOutgoingJob**](-mfax-faxoutgoingjob.md) object from the [**FaxOutgoingJobs**](-mfax-faxoutgoingjobs.md) collection.

This property is read-only.

## Syntax


```VB
Property Item( _
  ByVal vIndex As Variant _
) As FaxOutgoingJob
```



## Property value

A [**FaxOutgoingJob**](-mfax-faxoutgoingjob.md) object.
