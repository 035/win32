# FaxAccountFolders.OutgoingArchive property

Represents the archive of outgoing faxes for a particular fax account that have been sent.

This property is read-only.

## Syntax


```VB
Property OutgoingArchive As IFaxAccountOutgoingArchive
```



## Property value

A variable of type [**IFaxAccountOutgoingArchive**](/previous-versions/windows/desktop/api/FaxComex/nn-faxcomex-ifaxaccountoutgoingarchive) that receives a [**FaxOutgoingArchive**](-mfax-faxoutgoingarchive.md) object.
