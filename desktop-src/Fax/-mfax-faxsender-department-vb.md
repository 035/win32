# FaxSender.Department property

The **Department** property is a null-terminated string that contains the department associated with the sender.

This property is read/write.

## Syntax


```VB
Property Department As String
```



## Property value

A **String** that specifies or receives the name or identifier of the department that is associated with the sender.
