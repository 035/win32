# FaxOutgoingMessage.Sender property

The **Sender** property retrieves an object containing information about the sender of the fax message.

This property is read-only.

## Syntax


```VB
Property Sender As FaxSender
```



## Property value

A [**FaxSender**](-mfax-faxsender.md) object.
