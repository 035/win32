# FaxDeviceProvider.TapiProviderName property

The **TapiProviderName** property is a null-terminated string that contains the name of the telephony service provider (TSP) associated with the fax service provider (FSP) fax devices.

This property is read-only.

## Syntax


```VB
Property TapiProviderName As String
```



## Property value

A **String** that receives the name of the TSP associated with the FSP fax devices.
