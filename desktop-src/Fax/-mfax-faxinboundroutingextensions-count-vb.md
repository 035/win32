# FaxInboundRoutingExtensions.Count property

The **Count** property represents the number of objects in the [**FaxInboundRoutingExtensions**](-mfax-faxinboundroutingextensions.md) collection. This is the total number of inbound routing extensions associated with the fax server.

This property is read-only.

## Syntax


```VB
Property Count As Long
```



## Property value

A **Long** that receives the number of objects in the [**FaxInboundRoutingExtensions**](-mfax-faxinboundroutingextensions.md) collection.
