# FaxConfiguration.UseDeviceTSID property

Sets or retrieves a value that indicates whether the transmitting station identifier (TSID) is used.

This property is read/write.

## Syntax


```VB
Property UseDeviceTSID As Boolean
```



## Property value

A value of type **Boolean** that indicates whether the TSID is used.
