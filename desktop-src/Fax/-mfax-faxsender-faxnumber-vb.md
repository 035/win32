# FaxSender.FaxNumber property

The **FaxNumber** property is a null-terminated string that contains the fax number associated with the sender.

This property is read/write.

## Syntax


```VB
Property FaxNumber As String
```



## Property value

A **String** that specifies or receives the fax number associated with the sender.
