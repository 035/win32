# FaxSender.BillingCode property

The **BillingCode** property is a null-terminated string that contains the billing code associated with the sender.

This property is read/write.

## Syntax


```VB
Property BillingCode As String
```



## Property value

A **String** that specifies or receives the billing code associated with the sender.
