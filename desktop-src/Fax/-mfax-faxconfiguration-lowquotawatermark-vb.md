# FaxConfiguration.LowQuotaWaterMark property

Sets or retrieves a value that indicates the minimum size of a watermark.

This property is read/write.

## Syntax


```VB
Property LowQuotaWaterMark As Integer
```



## Property value

A value of type **Integer** that indicates the minimum size of a watermark.
