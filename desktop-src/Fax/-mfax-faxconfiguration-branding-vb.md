# FaxConfiguration.Branding property

Sets or retrieves a value that indicates whether the fax server generates a branding mark on outgoing faxes.

This property is read/write.

## Syntax


```VB
Property Branding As Boolean
```



## Property value

A value of type **Boolean** that indicates whether the fax server generates a branding mark.
