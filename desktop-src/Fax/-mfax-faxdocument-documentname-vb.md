# FaxDocument.DocumentName property

The **DocumentName** property is a null-terminated string that contains the user-friendly name to display for the fax document. The value is for display purposes only.

This property is read/write.

## Syntax


```VB
Property DocumentName As String
```



## Property value

A **String** that specifies or receives the user-friendly name to associate with the fax document.
