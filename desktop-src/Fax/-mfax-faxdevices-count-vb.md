# FaxDevices.Count property

The **Count** property represents the number of objects in the [**FaxDevices**](-mfax-faxdevices.md) collection. This is the total number of devices used by the fax server.

This property is read-only.

## Syntax


```VB
Property Count As Long
```



## Property value

A **Long** that receives the number of fax devices in the collection.
