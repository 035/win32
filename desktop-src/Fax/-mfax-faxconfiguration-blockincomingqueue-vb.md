# FaxConfiguration.IncomingQueueBlocked property

Sets or retrieves a value that indicates whether the fax server queue for incoming faxes has been blocked.

This property is read/write.

## Syntax


```VB
Property IncomingQueueBlocked As Boolean
```



## Property value

A value of type **Boolean** that indicates whether the fax server queue for incoming faxes has been blocked.
