# FaxJobStatus.DeviceId property

The **DeviceId** property indicates the device ID of the device associated with the fax job.

This property is read-only.

## Syntax


```VB
Property DeviceId As Long
```



## Property value

A **Long** that receives the device ID for the fax device associated with the fax job.
