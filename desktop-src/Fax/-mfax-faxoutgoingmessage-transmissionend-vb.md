# FaxOutgoingMessage.TransmissionEnd property

The **TransmissionEnd** property indicates the time that the fax outbound message completed transmission.

This property is read-only.

## Syntax


```VB
Property TransmissionEnd As Date
```



## Property value

**Date** value. An 8-byte floating-point number that contains the time, expressed in local system time, that the fax message completed transmission.

For more information about the **Date** data type, see the *Microsoft Visual C++ Programmer's Guide*.
