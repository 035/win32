# FaxDevice.Description property

The **Description** property is a null-terminated string that contains a user-friendly description for the fax device. This string is suitable for display to users.

This property is read/write.

## Syntax


```VB
Property Description As String
```



## Property value

A **String** that specifies or receives the friendly description for the device.
