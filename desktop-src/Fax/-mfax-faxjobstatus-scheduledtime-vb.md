# FaxJobStatus.ScheduledTime property

The **ScheduledTime** property indicates the time that the fax job is scheduled for transmission.

This property is read-only.

## Syntax


```VB
Property ScheduledTime As Date
```



## Property value

A **Date** that receives the hour and minute, expressed in local system time, that the fax is scheduled for transmission. **Date** is an 8-byte floating-point number.
