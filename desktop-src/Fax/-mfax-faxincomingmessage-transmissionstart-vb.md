# FaxIncomingMessage.TransmissionStart property

The **TransmissionStart** property indicates the time that the inbound fax message began transmitting.

This property is read-only.

## Syntax


```VB
Property TransmissionStart As Date
```



## Property value

A **Date** that receives the hour and minute, expressed in local system time, that the fax began transmitting. **Date** is an 8-byte floating-point number.
