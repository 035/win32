# FaxIncomingMessage.Pages property

The **Pages** property is a value that indicates the total number of pages in the inbound fax message.

This property is read-only.

## Syntax


```VB
Property Pages As Long
```



## Property value

A **Long** that receives the total number of pages in the fax message.
