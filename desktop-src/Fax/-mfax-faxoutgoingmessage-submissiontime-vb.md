# FaxOutgoingMessage.SubmissionTime property

The **SubmissionTime** property indicates the time that the outbound fax message was submitted for processing.

This property is read-only.

## Syntax


```VB
Property SubmissionTime As Date
```



## Property value

**Date** value. An 8-byte floating-point number that contains the time, expressed in local system time, when the fax message was submitted for processing.

For more information about the **Date** data type, see the *Microsoft Visual C++ Programmer's Guide*.
