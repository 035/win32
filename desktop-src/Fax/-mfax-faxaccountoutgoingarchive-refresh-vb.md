# FaxAccountOutgoingArchive.Refresh method

Refreshes [**FaxAccountOutgoingArchive**](-mfax-faxaccountoutgoingarchive.md) object information for a particular fax account from the fax server.

## Syntax


```VB
FaxAccountOutgoingArchive.Refresh() As Long
```



## Parameters

This method has no parameters.

## Remarks

To use this method, a user must have the [****far2QUERY\_CONFIG****](/windows/previous-versions/FaxComex/ne-faxcomex-fax_access_rights_enum_2?branch=master) access right.
