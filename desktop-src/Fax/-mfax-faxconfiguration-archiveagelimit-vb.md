# FaxConfiguration.ArchiveAgeLimit property

Sets or retrieves a value that indicates how long a fax message is kept on the server.

This property is read/write.

## Syntax


```VB
Property ArchiveAgeLimit As Integer
```



## Property value

A value of type **Integer** that indicates how long a fax message is kept on the server.
