# FaxOutgoingQueue.Save method

The **Save** method saves the [**FaxOutgoingQueue**](-mfax-faxoutgoingqueue.md) object data.

## Syntax


```VB
FaxOutgoingQueue.Save() As Long
```



## Parameters

This method has no parameters.

## Remarks

To use this method, a user must have the [****farMANAGE\_CONFIG****](/previous-versions/windows/desktop/api/FaxComex/ne-faxcomex-fax_access_rights_enum) or [****farQUERY\_CONFIG****](/previous-versions/windows/desktop/api/FaxComex/ne-faxcomex-fax_access_rights_enum) access right.
