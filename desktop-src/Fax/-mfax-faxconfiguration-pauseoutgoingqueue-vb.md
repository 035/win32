# FaxConfiguration.OutgoingQueuePaused property

Sets or retrieves a value that indicates whether the outgoing queue has been paused.

This property is read/write.

## Syntax


```VB
Property OutgoingQueuePaused As Boolean
```



## Property value

A value of type **Boolean** that indicates whether the outgoing queue has been paused.
