# FaxServer.FaxAccountSet property

The **FaxAccountSet** property returns a [**IFaxAccountSet**](-mfax-faxaccountset-cpp.md) object used to manage the fax accounts on the fax server.

This property is read-only.

## Syntax


```VB
Property FaxAccountSet As IFaxAccountSet
```



## Property value

The address of a pointer to an [**IFaxAccountSet**](-mfax-faxaccountset-cpp.md) object.
