# FaxDeviceIds.Count property

The **Count** property represents the number of objects in the [**FaxDeviceIds**](-mfax-faxdeviceids.md) collection. This is the total number of device IDs associated with the fax server.

This property is read-only.

## Syntax


```VB
Property Count As Long
```



## Property value

A **Long** that receives the number of fax devices in the collection.
