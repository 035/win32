# FaxSender.OfficeLocation property

The **OfficeLocation** property is a null-terminated string that contains the office location of the sender.

This property is read/write.

## Syntax


```VB
Property OfficeLocation As String
```



## Property value

A **String** that specifies or receives the office location of the sender.
