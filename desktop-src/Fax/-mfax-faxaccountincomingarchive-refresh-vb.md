# FaxAccountIncomingArchive.Refresh method

Refreshes [**FaxAccountIncomingArchive**](-mfax-faxaccountincomingarchive.md) object information for a particular fax account from the fax server.

## Syntax


```VB
FaxAccountIncomingArchive.Refresh() As Long
```



## Parameters

This method has no parameters.

## Remarks

To use this method, a user must have the [****far2QUERY\_CONFIG****](-mfax-fax-access-rights-enum-2.md) access right.
