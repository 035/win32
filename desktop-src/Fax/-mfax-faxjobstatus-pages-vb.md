# FaxJobStatus.Pages property

The **Pages** property is a number that indicates the total number of pages received so far in the fax transmission.

This property is read-only.

## Syntax


```VB
Property Pages As Long
```



## Property value

A **Long** that receives the total number of pages received so far in the fax transmission.
