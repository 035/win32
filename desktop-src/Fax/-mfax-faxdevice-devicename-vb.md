# FaxDevice.DeviceName property

The **DeviceName** property is a null-terminated string that contains the name of the fax device.

This property is read-only.

## Syntax


```VB
Property DeviceName As String
```



## Property value

A **String** that receives the name of the fax device. The **String** is null-terminated.
