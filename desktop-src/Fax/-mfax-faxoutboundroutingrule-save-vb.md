# FaxOutboundRoutingRule.Save method

The **Save** method saves the [**FaxOutboundRoutingRule**](-mfax-faxoutboundroutingrule.md) object data.

## Syntax


```VB
FaxOutboundRoutingRule.Save() As Long
```



## Parameters

This method has no parameters.

## Remarks

To use this method, a user must have the [**farMANAGE\_CONFIG**](/previous-versions/windows/desktop/api/FaxComex/ne-faxcomex-fax_access_rights_enum) and **farQUERY\_CONFIG** access rights.
