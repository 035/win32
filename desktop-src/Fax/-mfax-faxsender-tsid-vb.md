# FaxSender.TSID property

The **TSID** property is a null-terminated string that contains the transmitting station identifier (TSID) for the sender's device.

This property is read/write.

## Syntax


```VB
Property TSID As String
```



## Property value

A **String** that specifies or receives the transmitting station identifier (TSID) for the sender's device.
