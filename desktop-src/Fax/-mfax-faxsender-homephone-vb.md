# FaxSender.HomePhone property

The **HomePhone** property is a null-terminated string that contains the home telephone number associated with the sender.

This property is read/write.

## Syntax


```VB
Property HomePhone As String
```



## Property value

A **String** that specifies or receives the home telephone number associated with the sender.
