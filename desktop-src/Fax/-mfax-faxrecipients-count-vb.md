# FaxRecipients.Count property

The **Count** property represents the number of objects in the [**FaxRecipients**](-mfax-faxrecipients.md) collection. This is the total number of recipients associated with the fax server or fax document.

This property is read-only.

## Syntax


```VB
Property Count As Long
```



## Property value

A **Long** that receives the number of recipients in the collection.
