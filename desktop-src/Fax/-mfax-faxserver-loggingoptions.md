# FaxServer.LoggingOptions property

The **LoggingOptions** property creates a [**FaxLoggingOptions**](-mfax-faxloggingoptions.md) configuration object. The object permits configuration of both the activity logging options and the event logging categories that the fax service uses.

This property is read-only.

## Syntax


```VB
Property LoggingOptions As FaxLoggingOptions
```



## Property value

A [**FaxLoggingOptions**](-mfax-faxloggingoptions.md) object.
