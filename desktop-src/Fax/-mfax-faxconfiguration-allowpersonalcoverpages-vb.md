# FaxConfiguration.AllowPersonalCoverPages property

Sets or retrieves a value that indicates whether personal cover pages are allowed.

This property is read/write.

## Syntax


```VB
Property AllowPersonalCoverPages As Boolean
```



## Property value

A value of type **Boolean** that indicates whether personal cover pages are allowed.
