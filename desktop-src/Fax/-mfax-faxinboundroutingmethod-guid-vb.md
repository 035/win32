# FaxInboundRoutingMethod.GUID property

The **GUID** property is a null-terminated string that specifies the GUID that uniquely identifies the fax routing method.

This property is read-only.

## Syntax


```VB
Property GUID As String
```



## Property value

A **String** that receives the GUID that uniquely identifies the fax routing method.
