# FaxConfiguration.UseArchive property

Sets or retrieves a value that indicates whether faxes should be archived.

This property is read/write.

## Syntax


```VB
Property UseArchive As Boolean
```



## Property value

A value of type **Boolean** that indicates whether faxes should be archived.
