# FaxAccounts.Count property

Holds the number of items in the [**IFaxAccounts**](/previous-versions/windows/desktop/api/FaxComex/nn-faxcomex-ifaxaccounts) collection.

This property is read-only.

## Syntax


```VB
Property Count As Long
```



## Property value

Pointer to the total number of accounts in the collection.
