# FaxConfiguration.IncomingFaxesArePublic property

Indicates whether incoming faxes are either viewable by everyone or private.

This property is read/write.

## Syntax


```VB
Property IncomingFaxesArePublic As Boolean
```



## Property value

A value of type **Boolean** that indicates whether incoming faxes are viewable by everyone.
