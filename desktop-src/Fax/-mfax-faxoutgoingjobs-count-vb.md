# FaxOutgoingJobs.Count property

The **Count** property represents the number of objects in the [**FaxOutgoingJobs**](-mfax-faxoutgoingjobs.md) collection. This is the total number of outgoing jobs for the fax server.

This property is read-only.

## Syntax


```VB
Property Count As Long
```



## Property value

A **Long** that receives the number of outbound jobs in the fax queue.
