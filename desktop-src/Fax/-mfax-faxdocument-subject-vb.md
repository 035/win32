# FaxDocument.Subject property

The **Subject** property is a null-terminated string that contains the contents of the subject field on the cover page of the fax.

This property is read/write.

## Syntax


```VB
Property Subject As String
```



## Property value

A **String** that specifies or receives text that describes the subject of the fax transmission.
