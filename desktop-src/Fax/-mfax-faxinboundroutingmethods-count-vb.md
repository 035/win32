# FaxInboundRoutingMethods.Count property

The **Count** property represents the number of objects in the [**FaxInboundRoutingMethods**](-mfax-faxinboundroutingmethods.md) collection. This is the total number of inbound routing methods associated with the fax server.

This property is read-only.

## Syntax


```VB
Property Count As Long
```



## Property value

A **Long** that receives the number of inbound fax routing methods in the collection.
