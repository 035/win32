# FaxOutgoingMessage.DocumentName property

The **DocumentName** property is a null-terminated string that contains the user-friendly name to display for the fax message.

This property is read-only.

## Syntax


```VB
Property DocumentName As String
```



## Property value

A **String** that receives the user-friendly name to associate with the fax message. The value is for display purposes only.
