# FaxConfiguration.DiscountRateStart property

Sets or retrieves a value that indicates the time at which the discount rate period begins.

This property is read/write.

## Syntax


```VB
Property DiscountRateStart As Date
```



## Property value

A value of type **Date** that indicates the time at which the discount rate period begins.
