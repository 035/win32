# FaxOutboundRoutingGroups.Count property

The **Count** property represents the number of objects in the [**FaxOutboundRoutingGroups**](-mfax-faxoutboundroutinggroups.md) collection. This is the total number of outbound routing groups associated with the fax server.

This property is read-only.

## Syntax


```VB
Property Count As Long
```



## Property value

A **Long** that receives the number of fax routing groups in the collection.
