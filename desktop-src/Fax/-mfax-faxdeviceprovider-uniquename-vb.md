# FaxDeviceProvider.UniqueName property

The **UniqueName** property is a null-terminated string that contains the unique name that identifies the fax service provider (FSP).

This property is read-only.

## Syntax


```VB
Property UniqueName As String
```



## Property value

A **String** that receives the unique name that identifies the FSP.
