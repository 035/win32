# IOCTL\_DISK\_GET\_MEDIA\_TYPES control code

In Microsoft Windows 2000 and later operating systems, this IOCTL is replaced by [**IOCTL\_STORAGE\_GET\_MEDIA\_TYPES**](ioctl-storage-get-media-types.md). The only difference between the two IOCTLs is the base value.
