# Initialization Routines

## 

This section lists the initialization routines supplied by the Storport driver.

## In this section



| Topic                                                                                                   | Description                                                                                                                                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**StorPortEnablePassiveInitialization**](storportenablepassiveinitialization.md)<br/>           | The **StorPortEnablePassiveInitialization** routine enables the miniport's [**HwStorPassiveInitializeRoutine**](hwstorpassiveinitializeroutine.md) callback routine to execute at PASSIVE\_LEVEL during miniport initialization.<br/>                                                                                                                                                                          |
| [**StorPortGetActiveGroupCount**](storportgetactivegroupcount.md)<br/>                           | The **StorPortGetActiveGroupCount** routine returns the number of processor groups that are present in the system.<br/>                                                                                                                                                                                                                                                                                         |
| [**StorPortGetActiveNodeCount**](storportgetactivenodecount.md)<br/>                             | The **StorPortGetActiveNodeCount** routine returns the number of nodes that are present in the system.<br/>                                                                                                                                                                                                                                                                                                     |
| [**StorPortGetBusData**](storportgetbusdata.md)<br/>                                             | The **StorPortGetBusData** routine retrieves the bus-specific configuration information necessary to initialize the HBA.<br/>                                                                                                                                                                                                                                                                                   |
| [**StorPortGetCurrentProcessorNumber**](storportgetcurrentprocessornumber.md)<br/>               | The **StorPortGetCurrentProcessorNumber** routine retrieves the current processor number from the kernel.<br/>                                                                                                                                                                                                                                                                                                  |
| [**StorPortGetGroupAffinity**](storportgetgroupaffinity.md)<br/>                                 | The **StorPortGetGroupAffinity** routine constructs a mask of the active processors in a requested group.<br/>                                                                                                                                                                                                                                                                                                  |
| [**StorPortGetHighestNodeNumber**](storportgethighestnodenumber.md)<br/>                         | The **StorPortGetHighestNodeNumber** routine returns the largest possible node number on the system.<br/>                                                                                                                                                                                                                                                                                                       |
| [**StorPortGetLogicalProcessorRelationship**](storportgetlogicalprocessorrelationship.md)<br/>   | The **StorPortGetLogicalProcessorRelationship** routine returns relationship information for one or more specified types. These types include groups, physical packages, and nodes in the host system. The information that is returned includes processor affinity masks that are composed of the logical processors in the host system. These logical processors share the specified relationship types.<br/> |
| [**StorPortGetLogicalUnit**](storportgetlogicalunit.md)<br/>                                     | The **StorPortGetLogicalUnit** routine returns a pointer to the miniport driver's per-logical-unit storage area.<br/>                                                                                                                                                                                                                                                                                           |
| [**StorPortGetNodeAffinity**](storportgetnodeaffinity.md)<br/>                                   | The **StorPortGetNodeAffinity** routine constructs a mask of the active processors in a requested non-uniform memory access (NUMA) node.<br/>                                                                                                                                                                                                                                                                   |
| [**StorPortGetStartIoPerfParams**](storportgetstartioperfparams.md)<br/>                         | The [**StorPortGetStartIoPerfParams**](storportgetstartioperfparams.md) routine places the performance parameters for a given I/O request in a [**STARTIO\_PERFORMANCE\_PARAMETERS**](startio-performance-parameters.md) structure.<br/>                                                                                                                                                                      |
| [**StorPortInitialize**](storportinitialize.md)<br/>                                             | The **StorPortInitilize** routine initializes the port driver parameters and extension data. **StorPortInitilize** also saves the adapter information provided from the miniport driver.<br/>                                                                                                                                                                                                                   |
| [**StorPortInitializePerfOpts**](storportinitializeperfopts.md)<br/>                             | The [**StorPortInitializePerfOpts**](storportinitializeperfopts.md) function initializes the performance optimizations that both the miniport driver and the Storport driver support using a [**PERF\_CONFIGURATION\_DATA**](perf-configuration-data.md) structure.<br/>                                                                                                                                      |
| [**StorPortSetAdapterBusType**](storportsetadapterbustype.md)<br/>                               | Used to adjust the BusType of the adapter depending on its current configuration. Setting the BusType with this routine will allow you to override the global property set in the miniport INF without having to re-install the driver. This is useful for scenarios such as RAID support or support for multiple adapters with a different bus type.<br/>                                                      |
| [**StorPortSetBusDataByOffset**](storportsetbusdatabyoffset.md)<br/>                             | The **StorPortSetBusDataByOffset** routine writes bus-specific configuration information.<br/>                                                                                                                                                                                                                                                                                                                  |
| [**StorPortSetDeviceQueueDepth**](storportsetdevicequeuedepth.md)<br/>                           | The **StorPortSetDeviceQueueDepth** routine sets the maximum depth of the device queue for the indicated device. <br/>                                                                                                                                                                                                                                                                                          |
| [**StorPortSetPowerSettingNotificationGuids**](storportsetpowersettingnotificationguids.md)<br/> | The **StorPortSetPowerSettingNotificationGuids** routine enables a miniport to receive power setting notifications. The miniport registers an array of GUIDs which identify the power settings to receive power change notifications for.<br/>                                                                                                                                                                  |
| [**StorPortSetUnitAttributes**](storportsetunitattributes.md)<br/>                               | The **StorPortSetUnitAttributes** routine registers the power attributes of a storage unit device with the Storport driver.<br/>                                                                                                                                                                                                                                                                                |
