# iSCSI WMI Structures That Manage Initiators, Ports, Sessions and Connections

## 

The iSCSI WMI structures that this section describes are used in communications between the iSCSI Discovery Library and iSCSI Initiators. The WMI tool suite generates declarations for these structures in the *Iscsimgt.h* file when it compiles *Iscsimgt.mof*.

This section describes the following structures:

<dl>

[**ISCSI\_ConnectionStaticInfo**](iscsi-connectionstaticinfo.md)  
[**ISCSI\_Path**](iscsi-path.md)  
[**ISCSI\_PortalInfo**](iscsi-portalinfo.md)  
[**ISCSI\_RedirectPortalInfo**](iscsi-redirectportalinfo.md)  
[**ISCSI\_RedirectSessionInfo**](iscsi-redirectsessioninfo.md)  
[**ISCSI\_SessionStaticInfo**](iscsi-sessionstaticinfo.md)  
[**ISCSI\_Supported\_LB\_Policies**](iscsi-supported-lb-policies.md)  
[**MSiSCSI\_EventLog**](msiscsi-eventlog.md)  
[**MSiSCSI\_HBAInformation**](msiscsi-hbainformation.md)  
[**MSiSCSI\_HBASessionConfig**](msiscsi-hbasessionconfig.md)  
[**MSiSCSI\_InitiatorInstanceFailureEvent**](msiscsi-initiatorinstancefailureevent.md)  
[**MSiSCSI\_InitiatorNodeFailureEvent**](msiscsi-initiatornodefailureevent.md)  
[**MSiSCSI\_InitiatorSessionInfo**](msiscsi-initiatorsessioninfo.md)  
[MSiSCSI\_LB\_Operations](msiscsi-lb-operations.md)  
[MSiSCSI\_ManagementOperations](msiscsi-managementoperations.md)  
[**MSiSCSI\_PortalInfoClass**](msiscsi-portalinfoclass.md)  
[**MSiSCSI\_QueryLBPolicy**](msiscsi-querylbpolicy.md)
