# iSCSI WMI Class Structures

## 

This section discusses structures that are related to [iSCSI WMI Classes](https://msdn.microsoft.com/library/windows/hardware/ff561578). All the classes described in this section are part of the iSCSI headers and MOF files that are part of the platform SDK and WDK. The principal categories of iSCSI WMI structures are as follows:

<dl>

[iSCSI WMI Common Structures](iscsi-wmi-common-structures.md)  
[iSCSI WMI Structures That Manage Initiators, Ports, Sessions, and Connections](iscsi-wmi-structures-that-manage-initiators--ports--sessions-and-conne.md)  
[iSCSI WMI Structures That Report Performance Statistics](iscsi-wmi-structures-that-report-performance-statistics.md)  
[iSCSI WMI Operations Class Structures](iscsi-wmi-operations-class-structures.md)  
[iSCSI WMI Discovery Class Structures](iscsi-wmi-discovery-class-structures.md)  
[iSCSI WMI Network Class Structures](iscsi-wmi-network-class-structures.md)  
</dl>
