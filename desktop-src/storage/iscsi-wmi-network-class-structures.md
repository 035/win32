# iSCSI WMI Network Class Structures

## 

The iSCSI WMI network class structures and enumerations that this section describes are used in network configuration. The WMI tool suite generates declarations for these structures in the *Iscsicfg.h* file when it compiles *Config.mof*.

This section describes the following structures:

<dl>

[**ISCSI\_ENCRYPTION\_TYPES**](iscsi-encryption-types.md)  
[**ISCSI\_NIC\_LINKSTATE**](iscsi-nic-linkstate.md)  
[**MSiSCSI\_BootConfiguration**](msiscsi-bootconfiguration.md)  
[**MSiSCSI\_DiscoveryConfig**](msiscsi-discoveryconfig.md)  
[**MSiSCSI\_NICConfig**](msiscsi-nicconfig.md)  
[**MSiSCSI\_RADIUSConfig**](msiscsi-radiusconfig.md)  
[**MSiSCSI\_SecurityCapabilities**](msiscsi-securitycapabilities.md)  
[**MSiSCSI\_TCPIPConfig**](msiscsi-tcpipconfig.md)  
</dl>
