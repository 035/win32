# Timer Routines

## 

This section lists the timer routines supplied by the Storport driver.

## In this section



| Topic                                                                 | Description                                                                                                                                    |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**StorPortFreeTimer**](storportfreetimer.md)<br/>             | Frees a Storport timer context object previously created by the [**StorPortInitializeTimer**](storportinitializetimer.md) routine.<br/> |
| [**StorPortInitializeTimer**](storportinitializetimer.md)<br/> | Creates a Storport timer context object.<br/>                                                                                            |
| [**StorPortRequestTimer**](storportrequesttimer.md)<br/>       | Schedules a callback event for a Storport timer context object.<br/>                                                                     |
