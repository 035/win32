# EAPHost Supplicant Enumerations

The EAPHost Supplicant API enumerations are as follows.



| Enumeration                                                                  | Description                                                                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**EAP\_INTERACTIVE\_UI\_DATA\_TYPE**](eap-interactive-ui-data-type.md)     | Specifies the set of types of interactive user interface context data supplied to certain supplicant API calls.    |
| [**EAP\_METHOD\_PROPERTY\_TYPE**](eap-method-property-type.md)              | Windows 7 or later: Specifies the method property type.                                                            |
| [**EAP\_METHOD\_PROPERTY\_VALUE\_TYPE**](eap-method-property-value-type.md) | Windows 7 or later: Specifies the data type of a method property value.                                            |
| [**EAPHOST\_AUTH\_STATUS**](eaphost-auth-status.md)                         | Defines the set of possible EAP authentication session status values.                                              |
| [**EapHostPeerAuthParams**](eaphostpeerauthparams.md)                       | Defines the set of possible authentication parameter values.                                                       |
| [**EapHostPeerMethodResultReason**](eaphostpeermethodresultreason.md)       | Defines the set of possible reasons that describe the results returned by an EAP method to a supplicant.           |
| [**EapHostPeerResponseAction**](eaphostpeerresponseaction.md)               | Defines the set of actions an EAP authenticator or peer method can indicate to a supplicant during authentication. |
| [**ISOLATION\_STATE**](isolation-state.md)                                  | Defines the set of possible isolation status values of a machine.                                                  |
