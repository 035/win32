# DVBCLocator Object

The **DVBCLocator** object contains tuning information for a cable DVB (DVB-C) network.



|                           |                                      |
|---------------------------|--------------------------------------|
| Interfaces                | [**IDVBCLocator**](idvbclocator.md) |
| Outgoing Event Interfaces | None                                 |
| CLSID                     | CLSID\_DVBCLocator                   |
