# BDA Structures

The following table lists the structures defined for the Broadcast Driver Architecture (BDA) in DirectShow.



| Structure                                                       | Description                                                                                                                    |
|-----------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [**ATSC\_FILTER\_OPTIONS**](atsc-filter-options.md)            | Specifies additional criteria for matching ATSC section headers.                                                               |
| [**BDANODE\_DESCRIPTOR**](bdanode-descriptor.md)               | Describes a control node in BDA device filter.                                                                                 |
| [**DSMCC\_ELEMENT**](dsmcc-element.md)                         | Contains information from a DSM-CC element.                                                                                    |
| [**DSMCC\_FILTER\_OPTIONS**](dsmcc-filter-options.md)          | Specifies additional filtering criteria for the DSM-CC portions of the section header.                                         |
| [**DSMCC\_SECTION**](dsmcc-section.md)                         | Represents a DSM-CC section header.                                                                                            |
| [**DVB\_EIT\_FILTER\_OPTIONS**](dvb-eit-filter-options.md)     | Specifies a section within a Digital Video Broadcast (DVB) Event Information Table (EIT) section header. (Windows 7 and later) |
| [**DVR\_STREAM\_DESC**](dvr-stream-desc.md)                    | Describes a stream captured by a tuner. (Windows 7 and later)                                                                  |
| [**EALocationCodeType**](ealocationcodetype.md)                | Defines an Emergency Alert (EA) location code.                                                                                 |
| [**LONG\_SECTION**](long-section.md)                           | Represents a long MPEG-2 section header.                                                                                       |
| [**MPE\_ELEMENT**](mpe-element.md)                             | Contains information from a multi-protocol encapsulation (MPE) element.                                                        |
| [**MPEG2\_FILTER**](mpeg2-filter.md)                           | Specifies criteria for matching MPEG-2 section headers.                                                                        |
| [**MPEG2\_FILTER2**](mpeg2-filter2.md)                         | Specifies criteria for matching MPEG-2 section headers (Windows 7 and later).                                                  |
| [**MPEG\_BCS\_DEMUX**](mpeg-bcs-demux.md)                      | Identifies the filter graph that is providing the MPEG-2 data stream.                                                          |
| [**MPEG\_CONTEXT**](mpeg-context.md)                           | Identifies the source of an MPEG-2 data stream.                                                                                |
| [**MPEG\_DATE**](mpeg-date.md)                                 | Specifies a date.                                                                                                              |
| [**MPEG\_DATE\_AND\_TIME**](mpeg-date-and-time.md)             | Specifies a date and time.                                                                                                     |
| [**MPEG\_DURATION**](mpeg-duration.md)                         | Identical to the [**MPEG\_TIME**](mpeg-time.md) structure.                                                                    |
| [**MPEG\_HEADER\_BITS**](mpeg-header-bits.md)                  | Contains the first 16 bits that follow the table\_id in a generic MPEG-2 section header.                                       |
| [**MPEG\_HEADER\_VERSION\_BITS**](mpeg-header-version-bits.md) | Contains the first 8 bits following the TSID in an MPEG-2 PSI section.                                                         |
| [**MPEG\_PACKET\_LIST**](mpeg-packet-list.md)                  | Contains a list of MPEG-2 sections.                                                                                            |
| [**MPEG\_RQST\_PACKET**](mpeg-rqst-packet.md)                  | Defines a buffer to receive MPEG-2 section data.                                                                               |
| [**MPEG\_STREAM\_BUFFER**](mpeg-stream-buffer.md)              | Defines a buffer that receives MPEG-2 data.                                                                                    |
| [**MPEG\_TIME**](mpeg-time.md)                                 | Represents a time of day.                                                                                                      |
| [**MPEG\_WINSOCK**](mpeg-winsock.md)                           | Not supported.                                                                                                                 |
| [**SECTION**](section.md)                                      | Represents a short header from an MPEG-2 section.                                                                              |
| [**SmartCardApplication**](smartcardapplication.md)            | Identifies a smart card application.                                                                                           |
