# TV Ratings Enumerations

This topic applies to Windows XP Service Pack 1 or later.

The following enumerations are used by the TV Ratings objects.



| Enumeration                                                           | Description                                                                    |
|-----------------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**BfEnTvRat\_Attributes\_CAE\_TV**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-bfentvrat_attributes_cae_tv) | Specifies content rating attributes in the English-Canadian television system. |
| [**BfEnTvRat\_Attributes\_CAF\_TV**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-bfentvrat_attributes_caf_tv) | Specifies content rating attributes in the French-Canadian television system.  |
| [**BfEnTvRat\_Attributes\_MPAA**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-bfentvrat_attributes_mpaa)      | Specifies content rating attributes in the MPAA rating system.                 |
| [**BfEnTvRat\_Attributes\_US\_TV**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-bfentvrat_attributes_us_tv)   | Specifies content rating attributes in the US television system.               |
| [**BfEnTvRat\_GenericAttributes**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-bfentvrat_genericattributes)   | Specifies content rating attributes that can be used for any rating system.    |
| [**EnTvRat\_CAE\_TV**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-entvrat_cae_tv)                            | Specifies rating levels for English-Canadian television.                       |
| [**EnTvRat\_CAF\_TV**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-entvrat_caf_tv)                            | Specifies rating levels for French-Canadian television.                        |
| [**EnTvRat\_GenericLevel**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-entvrat_genericlevel)                 | Specifies a generic range of rating levels.                                    |
| [**EnTvRat\_MPAA**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-entvrat_mpaa)                                 | Specifies rating levels in the MPAA rating system.                             |
| [**EnTvRat\_System**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-entvrat_system)                             | Specifies a rating system.                                                     |
| [**EnTvRat\_US\_TV**](/previous-versions/windows/desktop/api/Tvratings/ne-tvratings-entvrat_us_tv)                              | Specifies rating levels for US television.                                     |
| [**ProtType**](/previous-versions/windows/desktop/api/encdec/ne-encdec-prottype)                                          | Specifies various types of content protection.                                 |
