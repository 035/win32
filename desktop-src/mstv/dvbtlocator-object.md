# DVBTLocator Object

The **DVBTLocator** object is used by the Network Provider to obtain information about tuning within a terrestrial DVB (DVB-T) tuning space.



|                           |                                      |
|---------------------------|--------------------------------------|
| Interfaces                | [**IDVBTLocator**](/windows/previous-versions/tuner/nn-tuner-idvbtlocator?branch=master) |
| Outgoing Event Interfaces | None                                 |
| CLSID                     | CLSID\_DVBTLocator                   |
