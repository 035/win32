# XDSToRat Object

This topic applies to Windows XP Service Pack 1 or later.

The **XDSToRat** object parses rating information from line 21 data and converts it into program ratings.

This object is created by third parties. It must support the **IXDSToRat** interface. This object is used by the XDS Codec filter. Applications do not use this object directly.



|            |                                |
|------------|--------------------------------|
| CLSID      | CLSID\_XDSToRat                |
| Interfaces | [**IXDSToRat**](ixdstorat.md) |
