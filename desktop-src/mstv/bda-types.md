# BDA Types

The following table lists the enumeration types defined for the Broadcast Driver Architecture (BDA) in DirectShow.



| Enumeration                                                  | Description                                                                      |
|--------------------------------------------------------------|----------------------------------------------------------------------------------|
| [**ApplicationTypeType**](applicationtypetype.md)           | Specifies the smart card application type.                                       |
| [**BDA\_Comp\_Flags**](bda-comp-flags.md)                   | Defines flags for the [**IBDAComparable**](/previous-versions/windows/desktop/api/tuner/nn-tuner-ibdacomparable) interface.        |
| [**BDA\_DrmPairingError**](/windows/desktop/api/Bdaiface/ne-bdaiface-bda_drmpairingerror)          | Specifies the status of a DRM handshake between a tuner and the user's computer. |
| [**MPEG\_CONTEXT\_TYPE**](/previous-versions/windows/desktop/api/Mpeg2Structs/ne-mpeg2structs-__midl___midl_itf_mpeg2structs_0000_0000_0027)             | Identifies the source of an MPEG-2 data stream.                                  |
| [**MPEG\_CURRENT\_NEXT\_BIT**](/previous-versions/windows/desktop/api/Mpeg2Structs/ne-mpeg2structs-__midl___midl_itf_mpeg2structs_0000_0000_0004)    | Specifies the value of the current\_next\_indicator bit.                         |
| [**MPEG\_REQUEST\_TYPE**](/previous-versions/windows/desktop/api/Mpeg2Structs/ne-mpeg2structs-__midl___midl_itf_mpeg2structs_0000_0000_0032)             | Specifies a request for MPEG-2 data.                                             |
| [**ScanModulationTypes**](scanmodulationtypes.md)           | Specifies the modulation type.                                                   |
| [**SmartCardAssociationType**](/windows/desktop/api/Bdaiface/ne-bdaiface-smartcardassociationtype) | Specifies a smart card association.                                              |
| [**SmartCardStatusType**](/windows/desktop/api/Bdaiface/ne-bdaiface-smartcardstatustype)           | Specifies the status of a smart card.                                            |
| [**TunerLockType**](tunerlocktype.md)                       | Specifies how well a television tuner has locked onto a signal.                  |
| [**UICloseReasonType**](/windows/desktop/api/Bdaiface/ne-bdaiface-uiclosereasontype)               | Specifies the reason that a user-interface dialog for a smart card was closed.   |
