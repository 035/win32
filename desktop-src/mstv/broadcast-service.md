# Broadcast Service

The Broadcast Event Service object handles broadcast events for the Filter Graph Manager. For more information, see [**IBroadcastEvent**](/previous-versions/windows/desktop/api/tuner/nn-tuner-ibroadcastevent).



|                           |                                                                  |
|---------------------------|------------------------------------------------------------------|
| Interfaces                | [**IBroadcastEvent**](/previous-versions/windows/desktop/api/tuner/nn-tuner-ibroadcastevent), **IConnectionPoint** |
| Outgoing Event Interfaces | None.                                                            |
