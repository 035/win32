# Stream Buffer Engine Attributes

The following attributes are supported for Stream Buffer Engine (SBE) files. To retrieve these attributes, use the [**IStreamBufferRecordingAttribute**](/previous-versions/windows/desktop/api/Sbe/nn-sbe-istreambufferrecordingattribute) interface.



| Attribute                                                                | Description                                                            |
|--------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**WM/WMRVContentProtected**](wm-wmrvcontentprotected.md)               | Specifies whether a significant portion of a DVR-MS file is protected. |
| [**WM/WMRVContentProtectedPercent**](wm-wmrvcontentprotectedpercent.md) | Specifies the percentage of a DVR-MS file that is protected.           |
| [**WM/WMRVExpirationDate**](wm-wmrvexpirationdate.md)                   | Specifies the date when the license expires for a DVR-MS file.         |
