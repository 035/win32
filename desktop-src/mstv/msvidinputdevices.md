# MSVidInputDevices

This topic applies to Windows XP or later.

The **MSVidInputDevices** object represents a collection of input devices.



|                           |                                                  |
|---------------------------|--------------------------------------------------|
| Interfaces                | [**IMSVidInputDevices**](imsvidinputdevices.md) |
| Outgoing Event Interfaces | None.                                            |



 

## Remarks

To obtain this collection, call the [**IMSVidCtl::get\_InputsAvailable**](imsvidctl-get-inputsavailable.md) method.
