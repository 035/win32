# MSVidInputDevices

This topic applies to Windows XP or later.

The **MSVidInputDevices** object represents a collection of input devices.



|                           |                                                  |
|---------------------------|--------------------------------------------------|
| Interfaces                | [**IMSVidInputDevices**](/windows/win32/segment/nn-segment-imsvidinputdevices?branch=master) |
| Outgoing Event Interfaces | None.                                            |



 

## Remarks

To obtain this collection, call the [**IMSVidCtl::get\_InputsAvailable**](/windows/previous-versions/msvidctl/nf-msvidctl-imsvidctl-get_inputsavailable?branch=master) method.
