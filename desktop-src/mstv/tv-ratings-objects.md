# TV Ratings Objects

This topic applies to Windows XP Service Pack 1 or later.

The following table lists the TV Ratings objects.



| Object                          | Description                                                                                                                |
|---------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [EvalRat](evalrat-object.md)   | Evaluates program ratings. This object is created by third parties.                                                        |
| [XDSToRat](xdstorat-object.md) | Parses rating information from line 21 data and converts it into program ratings. This object is created by third parties. |
