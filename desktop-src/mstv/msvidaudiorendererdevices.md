# MSVidAudioRendererDevices

This topic applies to Windows XP or later.

The **MSVidAudioRendererDevices** object represents a collection of audio renderers.



|                           |                                                                  |
|---------------------------|------------------------------------------------------------------|
| Interfaces                | [**IMSVidAudioRendererDevices**](/windows/desktop/api/segment/nn-segment-imsvidaudiorendererdevices) |
| Outgoing Event Interfaces | None.                                                            |



 

## Remarks

You can obtain this object by calling the [**IMSVidCtl::get\_AudioRenderersAvailable**](imsvidctl-get-audiorenderersavailable.md) method.
