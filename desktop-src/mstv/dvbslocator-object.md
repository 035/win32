# DVBSLocator Object

The **DVBSLocator** object is a locator for a satellite DVB (DVB-S) network.

|---------------------------|--------------------------------------|
| Interfaces                | [**IDVBSLocator**](/previous-versions/windows/desktop/api/tuner/nn-tuner-idvbslocator) |
| Outgoing Event Interfaces | None                                 |
| CLSID                     | CLSID\_DVBSLocator                   |
