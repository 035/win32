# Unified Tuning Model Reference

This section contains the reference documentation for the [Unified Tuning Model](unified-tuning-model.md) application programming interfaces (APIs). The objects and interfaces documented here provide the tuning abstraction layer that enables applications to remain network-independent.

The process of obtaining, storing and submitting tuning information using the Tuning Model objects differs depending on whether the platform is Windows XP or later, or an earlier version of Windows.
