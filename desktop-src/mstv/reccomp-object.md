# RecComp Object

This topic applies only to Windows XP Service Pack 1 or later.

The **RecComp** object creates new content recordings by concatenating existing recordings.

|-----------|------------------------------------------------------|
| CLSID     | CLSID\_StreamBufferComposeRecording                  |
