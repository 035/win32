# Tuning Model Enumerations

The following table lists all the enumeration types used by the Microsoft Unified Tuning Model interface methods.



| Enumeration Type                                               | Description                                                                           |
|----------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**ATSCComponentTypeFlags**](atsccomponenttypeflags.md)       | Specifies an AC3 audio stream.                                                        |
| [**BinaryConvolutionCodeRate**](binaryconvolutioncoderate.md) | Specifies the binary convolution code rate.                                           |
| [**ComponentCategory**](componentcategory.md)                 | Specifies the component (substream) category.                                         |
| [**ComponentStatus**](componentstatus.md)                     | Specifies the status of a component (substream).                                      |
| [**DVBSystemType**](dvbsystemtype.md)                         | Specifies the type of DVB system supported in a Tuning Space.                         |
| [**FECMethod**](fecmethod.md)                                 | Specifies the type of forward error correction being used at the transmission source. |
| [**GuardInterval**](guardinterval.md)                         | Specifies the guard interval.                                                         |
| [**HierarchyAlpha**](hierarchyalpha.md)                       | Specifies the hierarchy alpha.                                                        |
| [**KS\_CC\_SUBSTREAM**](ks-cc-substream.md)                   | Specifies line 21 services.                                                           |
| [**ModulationType**](modulationtype.md)                       | Specifies the modulation type.                                                        |
| [**MPEG2StreamType**](mpeg2streamtype.md)                     | Specifies the MPEG2 stream type.                                                      |
| [**Polarisation**](polarisation.md)                           | Specifies the polarisation type.                                                      |
| [**SpectralInversion**](spectralinversion.md)                 | Specifies the spectral inversion.                                                     |
| [**TransmissionMode**](transmissionmode.md)                   | Specifies the transmission mode.                                                      |
