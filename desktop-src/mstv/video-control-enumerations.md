# Video Control Enumerations

This topic applies to Windows XP or later.

The following table lists all the enumerations used by the Video Control interface methods.



| Enumeration                                       | Description                                                                       |
|---------------------------------------------------|-----------------------------------------------------------------------------------|
| [**DisplaySizeList**](/previous-versions/windows/desktop/api/msvidctl/ne-msvidctl-displaysizelist)        | Defines the sizes at which the video rectangle may be rendered.                   |
| [**DVDSPExt**](/windows/desktop/api/segment/ne-segment-dvdspext)                      | Indicates the default DVD subpicture language extension.                          |
| [**DVDTextStringType**](/windows/desktop/api/segment/ne-segment-dvdtextstringtype)    | Indicates the type of information contained in a DVD text string.                 |
| [**DVDMenuIDConstants**](/windows/desktop/api/Segment/ne-segment-dvdmenuidconstants) | Describes menu types.                                                             |
| [**MSVidCCService**](/windows/desktop/api/segment/ne-segment-msvidccservice)          | Indicates closed captioning services.                                             |
| [**MSVidCtlStateList**](/previous-versions/windows/desktop/api/msvidctl/ne-msvidctl-msvidctlstatelist)    | Defines the possible states of the video control and the underlying filter graph. |
| [**MSVidSinkStreams**](/windows/desktop/api/segment/ne-segment-msvidsinkstreams)      | Defines the stream types.                                                         |
| [**SourceSizeList**](/windows/desktop/api/segment/ne-segment-sourcesizelist)          | Defines the ways in which the source video rectangle may be clipped.              |
| [**MSVidCCService**](/windows/desktop/api/segment/ne-segment-msvidccservice)          | Indicates closed captioning services.                                             |
