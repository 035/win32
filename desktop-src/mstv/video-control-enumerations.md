# Video Control Enumerations

This topic applies to Windows XP or later.

The following table lists all the enumerations used by the Video Control interface methods.



| Enumeration                                       | Description                                                                       |
|---------------------------------------------------|-----------------------------------------------------------------------------------|
| [**DisplaySizeList**](/windows/previous-versions/msvidctl/ne-msvidctl-displaysizelist?branch=master)        | Defines the sizes at which the video rectangle may be rendered.                   |
| [**DVDSPExt**](/windows/win32/segment/ne-segment-dvdspext?branch=master)                      | Indicates the default DVD subpicture language extension.                          |
| [**DVDTextStringType**](/windows/win32/segment/ne-segment-dvdtextstringtype?branch=master)    | Indicates the type of information contained in a DVD text string.                 |
| [**DVDMenuIDConstants**](/windows/win32/Segment/ne-segment-dvdmenuidconstants?branch=master) | Describes menu types.                                                             |
| [**MSVidCCService**](/windows/win32/segment/ne-segment-msvidccservice?branch=master)          | Indicates closed captioning services.                                             |
| [**MSVidCtlStateList**](/windows/previous-versions/msvidctl/ne-msvidctl-msvidctlstatelist?branch=master)    | Defines the possible states of the video control and the underlying filter graph. |
| [**MSVidSinkStreams**](/windows/win32/segment/ne-segment-msvidsinkstreams?branch=master)      | Defines the stream types.                                                         |
| [**SourceSizeList**](/windows/win32/segment/ne-segment-sourcesizelist?branch=master)          | Defines the ways in which the source video rectangle may be clipped.              |
| [**MSVidCCService**](/windows/win32/segment/ne-segment-msvidccservice?branch=master)          | Indicates closed captioning services.                                             |
