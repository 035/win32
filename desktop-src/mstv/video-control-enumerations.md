# Video Control Enumerations

This topic applies to Windows XP or later.

The following table lists all the enumerations used by the Video Control interface methods.



| Enumeration                                       | Description                                                                       |
|---------------------------------------------------|-----------------------------------------------------------------------------------|
| [**DisplaySizeList**](displaysizelist.md)        | Defines the sizes at which the video rectangle may be rendered.                   |
| [**DVDSPExt**](dvdspext.md)                      | Indicates the default DVD subpicture language extension.                          |
| [**DVDTextStringType**](dvdtextstringtype.md)    | Indicates the type of information contained in a DVD text string.                 |
| [**DVDMenuIDConstants**](dvdmenuid-constants.md) | Describes menu types.                                                             |
| [**MSVidCCService**](msvidccservice.md)          | Indicates closed captioning services.                                             |
| [**MSVidCtlStateList**](msvidctlstatelist.md)    | Defines the possible states of the video control and the underlying filter graph. |
| [**MSVidSinkStreams**](msvidsinkstreams.md)      | Defines the stream types.                                                         |
| [**SourceSizeList**](sourcesizelist.md)          | Defines the ways in which the source video rectangle may be clipped.              |
| [**MSVidCCService**](msvidccservice.md)          | Indicates closed captioning services.                                             |
