# Video Control Interfaces

This topic applies to Windows XP or later.

The following table lists the interfaces implemented by the Video Control and its related objects.



| Interface                                                                        | Description                                                                                                   |
|----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**IBroadcastEventEx**](ibroadcasteventex.md)                                   | Extended version of **IBroadcastEvent**.                                                                      |
| [**IBroadcastEvent**](ibroadcastevent.md)                                       | Enables an object to receive events from another object without setting up a direct connection point.         |
| [**IMSVidAnalogTuner**](imsvidanalogtuner.md)                                   | Represents an analog-only tuner card that does not support the Broadcast Driver Architecture (BDA).           |
| [**IMSVidAnalogTuner2**](imsvidanalogtuner2.md)                                 | Represents an analog-only tuner card that does not support the Broadcast Driver Architecture (BDA).           |
| [**IMSVidAudioRendererDevices**](imsvidaudiorendererdevices.md)                 | Represents a collection of audio renderers.                                                                   |
| [**IMSVidAudioRenderer**](imsvidaudiorenderer.md)                               | Represents an audio renderer device.                                                                          |
| [**IMSVidClosedCaptioning**](imsvidclosedcaptioning.md)                         | Enables or disables closed captioning.                                                                        |
| [**IMSVidClosedCaptioning2**](imsvidclosedcaptioning2.md)                       | Sets the closed captioning service.                                                                           |
| [**IMSVidClosedCaptioning3**](imsvidclosedcaptioning3.md)                       | Retrieves the teletext filter.                                                                                |
| [**IMSVidCtl**](imsvidctl.md)                                                   | Provides the primary interface for the Video Control.                                                         |
| [**IMSVidDataServices**](imsviddataservices.md)                                 | Represents the data services feature.                                                                         |
| [**IMSVidDevice**](imsviddevice.md)                                             | Provides the base interface for all the devices and features that the Video Control supports.                 |
| [**IMSVidDevice2**](imsviddevice2.md)                                           | Extends the [**IMSVidDevice**](imsviddevice.md) interface.                                                   |
| [**IMSVidEncoder**](imsvidencoder.md)                                           | Represents the [MSVidEncoder](msvidencoder.md) feature.                                                      |
| [**IMSVidEVR**](imsvidevr.md)                                                   | Represents the Enhanced Video Renderer (EVR) filter within the Video Control filter graph.                    |
| [**IMSVidFeature**](imsvidfeature.md)                                           | Represents a feature that is available through the Video Control, such as data services or closed captioning. |
| [**IMSVidFeatures**](imsvidfeatures.md)                                         | Represents a collection of Video Control features.                                                            |
| [**IMSVidFilePlayback**](imsvidfileplayback.md)                                 | Enables the client to specify a local file for playback.                                                      |
| [**IMSVidFilePlayback2**](imsvidfileplayback2.md)                               | Extends the [**IMSVidFilePlayback**](imsvidfileplayback.md) interface.                                       |
| [**IMSVidGenericSink**](imsvidgenericsink.md)                                   | Represents a generic output device.                                                                           |
| [**IMSVidGenericSink2**](imsvidgenericsink2.md)                                 | Extends the [**IMSVidGenericSink**](imsvidgenericsink.md) interface.                                         |
| [**IMSVidGraphSegmentContainer**](imsvidgraphsegmentcontainer.md)               | Obtains a pointer to the Filter Graph Manager.                                                                |
| [**IMSVidInputDevice**](imsvidinputdevice.md)                                   | Represents any input device that is recognized by the Video Control, such as a television tuner card.         |
| [**IMSVidInputDevices**](imsvidinputdevices.md)                                 | Represents a collection of input devices.                                                                     |
| [**IMSVidOutputDevice**](imsvidoutputdevice.md)                                 | Represents an output device.                                                                                  |
| [**IMSVidOutputDevices**](imsvidoutputdevices.md)                               | Represents a collection of output devices.                                                                    |
| [**IMSVidPlayback**](imsvidplayback.md)                                         | Controls a playback device.                                                                                   |
| [**IMSVidRect**](imsvidrect.md)                                                 | Represents a rectangle with an associated window handle.                                                      |
| [**IMSVidStreamBufferRecordingControl**](imsvidstreambufferrecordingcontrol.md) | Enables an application to manage a stream buffer recording object through the Video Control.                  |
| [**IMSVidStreamBufferSink**](imsvidstreambuffersink.md)                         | Represents the Stream Buffer Sink filter within the Video Control.                                            |
| [**IMSVidStreamBufferSink2**](imsvidstreambuffersink2.md)                       | Extends the [**IMSVidStreamBufferSink**](imsvidstreambuffersink.md) interface.                               |
| [**IMSVidStreamBufferSink3**](imsvidstreambuffersink3.md)                       | Extends the [**IMSVidStreamBufferSink2**](imsvidstreambuffersink2.md) interface.                             |
| [**IMSVidStreamBufferSource**](imsvidstreambuffersource.md)                     | Represents the Stream Buffer Source filter within the Video Control.                                          |
| [**IMSVidStreamBufferSource2**](imsvidstreambuffersource2.md)                   | Extends the [**IMSVidStreamBufferSource**](imsvidstreambuffersource.md) interface.                           |
| [**IMSVidTuner**](imsvidtuner.md)                                               | Manages tuning devices.                                                                                       |
| [**IMSVidVideoInputDevice**](imsvidvideoinputdevice.md)                         | Represents a video input device.                                                                              |
| [**IMSVidVideoRenderer**](imsvidvideorenderer.md)                               | Represents a video renderer device.                                                                           |
| [**IMSVidVideoRenderer2**](imsvidvideorenderer2.md)                             | Represents a video renderer device.                                                                           |
| [**IMSVidVideoRendererDevices**](imsvidvideorendererdevices.md)                 | Represents a collection of video renderers.                                                                   |
| [**IMSVidVMR9**](imsvidvmr9.md)                                                 | Represents the Video Mixing Renderer Filter 9 (VMR-9) within the Video Control filter graph.                  |
| [**IMSVidXDS**](imsvidxds.md)                                                   | Represents the extended data services feature.                                                                |
