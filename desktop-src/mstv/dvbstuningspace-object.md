# DVBSTuningSpace Object

The **DVBSTuningSpace** object is a tuning space for satellite DVB (DVB-S) networks.



|                           |                                                                                                            |
|---------------------------|------------------------------------------------------------------------------------------------------------|
| Interfaces                | [**IDVBSTuningSpace**](idvbstuningspace.md), [**IMPEG2TuneRequestSupport**](impeg2tunerequestsupport.md) |
| Outgoing Event Interfaces | None                                                                                                       |
| CLSID                     | CLSID\_DVBSTuningSpace                                                                                     |



 

## Remarks

Tune request objects created by this tuning space support the [**ITuneRequest**](itunerequest.md) and [**IDVBTuneRequest**](idvbtunerequest.md) interfaces.
