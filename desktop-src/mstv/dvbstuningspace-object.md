# DVBSTuningSpace Object

The **DVBSTuningSpace** object is a tuning space for satellite DVB (DVB-S) networks.



|                           |                                                                                                            |
|---------------------------|------------------------------------------------------------------------------------------------------------|
| Interfaces                | [**IDVBSTuningSpace**](/previous-versions/windows/desktop/api/tuner/nn-tuner-idvbstuningspace), [**IMPEG2TuneRequestSupport**](/previous-versions/windows/desktop/api/tuner/) |
| Outgoing Event Interfaces | None                                                                                                       |
| CLSID                     | CLSID\_DVBSTuningSpace                                                                                     |



 

## Remarks

Tune request objects created by this tuning space support the [**ITuneRequest**](/previous-versions/windows/desktop/api/tuner/nn-tuner-itunerequest) and [**IDVBTuneRequest**](/previous-versions/windows/desktop/api/tuner/nn-tuner-idvbtunerequest) interfaces.
