# MSVidInputDevice

This topic applies to Windows XP or later.

The **MSVidInputDevice** object represents an input device, such as a tuner.



|                           |                                                |
|---------------------------|------------------------------------------------|
| Interfaces                | [**IMSVidInputDevice**](/windows/win32/segment/nn-segment-imsvidinputdevice?branch=master) |
| Outgoing Event Interfaces | None.                                          |



 

## Remarks

This object represents an abstract class. It is never directly instantiated, and exists only to support polymorphism.
