# TV Ratings Filters

This topic applies to Windows XP Service Pack 1 or later.

The following table lists the TV Ratings filters.



| Filter                                              | Description                                                        |
|-----------------------------------------------------|--------------------------------------------------------------------|
| [Decrypter/Detagger](decrypter-detagger-filter.md) | Decrypts samples.                                                  |
| [Encrypter/Tagger](encrypter-tagger-filter.md)     | Encrypts samples and tags them with ratings information.           |
| [XDS Codec](xds-codec-filter.md)                   | Supports third-party software that decodes TV ratings information. |
