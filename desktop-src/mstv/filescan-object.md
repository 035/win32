# FileScan Object object

The **FileScan** object repairs a corrupted .WTV file used in a Stream Buffer Engine (SBE) Version 2 (SBE2) media graph.



|           |                                        |
|-----------|----------------------------------------|
| CLSID     | **CLSID\_SBE2FileScan**                |
| Interface | [**ISBE2FileScan**](isbe2filescan.md) |



 

## Requirements



|                                     |                                            |
|-------------------------------------|--------------------------------------------|
| Minimum supported client<br/> | Windows 7 \[desktop apps only\]<br/> |
| Minimum supported server<br/> | None supported<br/>                  |
