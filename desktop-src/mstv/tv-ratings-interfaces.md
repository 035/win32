# TV Ratings Interfaces

This topic applies to Windows XP Service Pack 1 or later.

The following table lists the interfaces that are defined for the TV Ratings objects.



| Interface                                  | Description                                                                         |
|--------------------------------------------|-------------------------------------------------------------------------------------|
| [**IDTFilter**](idtfilter.md)             | Exposed by the Decrypter/Detagger filter.                                           |
| [**IDTFilter2**](idtfilter2.md)           | Extends the **IDTFilter** interface.                                                |
| [**IDTFilter3**](idtfilter3.md)           | Extends the **IDTFilter2** interface.                                               |
| [**IDTFilterConfig**](idtfilterconfig.md) | Configures the Decrypter/Detagger filter.                                           |
| [**IETFilter**](ietfilter.md)             | Exposed by the Encrypter/Tagger filter.                                             |
| [**IETFilterConfig**](ietfilterconfig.md) | Configures the Encrypter/Tagger filter.                                             |
| [**IEvalRat**](ievalrat.md)               | Evaluates content ratings carried by a broadcast stream.                            |
| [**IXDSCodec**](ixdscodec.md)             | Exposed by the XDS Codec filter.                                                    |
| [**IXDSCodecConfig**](ixdscodecconfig.md) | Configures the XDS Codec filter.                                                    |
| [**IXDSToRat**](ixdstorat.md)             | Parses rating information from extended data services (XDS) information in line 21. |
