# TV Ratings Interfaces

This topic applies to Windows XP Service Pack 1 or later.

The following table lists the interfaces that are defined for the TV Ratings objects.



| Interface                                  | Description                                                                         |
|--------------------------------------------|-------------------------------------------------------------------------------------|
| [**IDTFilter**](/previous-versions/windows/desktop/api/EncDec/nn-encdec-idtfilter)             | Exposed by the Decrypter/Detagger filter.                                           |
| [**IDTFilter2**](/previous-versions/windows/desktop/api/EncDec/nn-encdec-idtfilter2)           | Extends the **IDTFilter** interface.                                                |
| [**IDTFilter3**](/previous-versions/windows/desktop/api/encdec/nn-encdec-idtfilter3)           | Extends the **IDTFilter2** interface.                                               |
| [**IDTFilterConfig**](/previous-versions/windows/desktop/api/EncDec/nn-encdec-idtfilterconfig) | Configures the Decrypter/Detagger filter.                                           |
| [**IETFilter**](/previous-versions/windows/desktop/api/EncDec/nn-encdec-ietfilter)             | Exposed by the Encrypter/Tagger filter.                                             |
| [**IETFilterConfig**](/previous-versions/windows/desktop/api/EncDec/nn-encdec-ietfilterconfig) | Configures the Encrypter/Tagger filter.                                             |
| [**IEvalRat**](/previous-versions/windows/desktop/api/Tvratings/nn-tvratings-ievalrat)               | Evaluates content ratings carried by a broadcast stream.                            |
| [**IXDSCodec**](/previous-versions/windows/desktop/api/EncDec/nn-encdec-ixdscodec)             | Exposed by the XDS Codec filter.                                                    |
| [**IXDSCodecConfig**](/previous-versions/windows/desktop/api/encdec/nn-encdec-ixdscodecconfig) | Configures the XDS Codec filter.                                                    |
| [**IXDSToRat**](/previous-versions/windows/desktop/api/Tvratings/nn-tvratings-ixdstorat)             | Parses rating information from extended data services (XDS) information in line 21. |
