# TV Ratings Interfaces

This topic applies to Windows XP Service Pack 1 or later.

The following table lists the interfaces that are defined for the TV Ratings objects.



| Interface                                  | Description                                                                         |
|--------------------------------------------|-------------------------------------------------------------------------------------|
| [**IDTFilter**](/windows/previous-versions/EncDec/nn-encdec-idtfilter?branch=master)             | Exposed by the Decrypter/Detagger filter.                                           |
| [**IDTFilter2**](/windows/previous-versions/EncDec/nn-encdec-idtfilter2?branch=master)           | Extends the **IDTFilter** interface.                                                |
| [**IDTFilter3**](/windows/previous-versions/encdec/nn-encdec-idtfilter3?branch=master)           | Extends the **IDTFilter2** interface.                                               |
| [**IDTFilterConfig**](/windows/previous-versions/EncDec/nn-encdec-idtfilterconfig?branch=master) | Configures the Decrypter/Detagger filter.                                           |
| [**IETFilter**](/windows/previous-versions/EncDec/nn-encdec-ietfilter?branch=master)             | Exposed by the Encrypter/Tagger filter.                                             |
| [**IETFilterConfig**](/windows/previous-versions/EncDec/nn-encdec-ietfilterconfig?branch=master) | Configures the Encrypter/Tagger filter.                                             |
| [**IEvalRat**](/windows/previous-versions/Tvratings/nn-tvratings-ievalrat?branch=master)               | Evaluates content ratings carried by a broadcast stream.                            |
| [**IXDSCodec**](/windows/previous-versions/EncDec/nn-encdec-ixdscodec?branch=master)             | Exposed by the XDS Codec filter.                                                    |
| [**IXDSCodecConfig**](/windows/previous-versions/encdec/nn-encdec-ixdscodecconfig?branch=master) | Configures the XDS Codec filter.                                                    |
| [**IXDSToRat**](/windows/previous-versions/Tvratings/nn-tvratings-ixdstorat?branch=master)             | Parses rating information from extended data services (XDS) information in line 21. |
