# Video Control

The Video Control is an ActiveX control that handles television tuning, DVD and file playback, and stream buffering. It provides a higher-level wrapper over the base [DirectShow](https://msdn.microsoft.com/library/windows/desktop/dd375454) and [Microsoft TV Technologies](microsoft-tv-technologies-portal.md) APIs.
