# Stream Buffer Engine Structures

This section contains reference information for the structures used by the Stream Buffer Engine.



| Structure                                                 | Description                                              |
|-----------------------------------------------------------|----------------------------------------------------------|
| [**DVR\_STREAM\_DESC**](dvr-stream-desc.md)              | Describes a stream captured by a tuner.                  |
| [**SBE\_PIN\_DATA**](sbe-pin-data.md)                    | Contains performance data.                               |
| [**SBE2\_STREAM\_DESC**](sbe2-stream-desc.md)            | Describes a stream produced by the stream buffer engine. |
| [**STREAMBUFFER\_ATTRIBUTE**](streambuffer-attribute.md) | Describes an attribute of a stream buffer file.          |
