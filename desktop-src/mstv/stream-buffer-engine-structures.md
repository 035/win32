# Stream Buffer Engine Structures

This section contains reference information for the structures used by the Stream Buffer Engine.



| Structure                                                 | Description                                              |
|-----------------------------------------------------------|----------------------------------------------------------|
| [**DVR\_STREAM\_DESC**](/previous-versions/windows/desktop/api/sbe/ns-sbe-__midl___midl_itf_sbe_0000_0015_0002)              | Describes a stream captured by a tuner.                  |
| [**SBE\_PIN\_DATA**](/previous-versions/windows/desktop/api/Sbe/ns-sbe-sbe_pin_data)                    | Contains performance data.                               |
| [**SBE2\_STREAM\_DESC**](/previous-versions/windows/desktop/api/sbe/ns-sbe-__midl___midl_itf_sbe_0000_0015_0001)            | Describes a stream produced by the stream buffer engine. |
| [**STREAMBUFFER\_ATTRIBUTE**](/previous-versions/windows/desktop/api/Sbe/ns-sbe-streambuffer_attribute) | Describes an attribute of a stream buffer file.          |
