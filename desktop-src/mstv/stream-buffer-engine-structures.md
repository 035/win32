# Stream Buffer Engine Structures

This section contains reference information for the structures used by the Stream Buffer Engine.



| Structure                                                 | Description                                              |
|-----------------------------------------------------------|----------------------------------------------------------|
| [**DVR\_STREAM\_DESC**](/windows/previous-versions/sbe/ns-sbe-__midl___midl_itf_sbe_0000_0015_0002?branch=master)              | Describes a stream captured by a tuner.                  |
| [**SBE\_PIN\_DATA**](/windows/previous-versions/Sbe/ns-sbe-sbe_pin_data?branch=master)                    | Contains performance data.                               |
| [**SBE2\_STREAM\_DESC**](/windows/previous-versions/sbe/ns-sbe-__midl___midl_itf_sbe_0000_0015_0001?branch=master)            | Describes a stream produced by the stream buffer engine. |
| [**STREAMBUFFER\_ATTRIBUTE**](/windows/previous-versions/Sbe/ns-sbe-streambuffer_attribute?branch=master) | Describes an attribute of a stream buffer file.          |
