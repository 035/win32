# TV Ratings Components

> [!Note]  
> This topic applies to Windows XP Service Pack 1 or later.

 

The Microsoft TV Ratings components allow an application to parse out rating information about a program from the broadcast stream, set a maximum allowable rating, and halt playback or perform another suitable action if the rating exceeds this maximum value. Rating information is provided by the broadcaster, and describes the appropriate age group and content attributes.

The Microsoft TV Ratings components are supported on Microsoft Windows XP Service Pack 1, and they are only used for analog television broadcasts. This section contains the following topics:

-   [Overview of the TV Ratings System](overview-of-the-tv-ratings-system.md)
-   [TV Ratings Objects](tv-ratings-objects.md)
-   [TV Ratings Filters](tv-ratings-filters.md)
-   [TV Ratings Interfaces](tv-ratings-interfaces.md)
-   [TV Ratings Enumerations](tv-ratings-enumerations.md)
