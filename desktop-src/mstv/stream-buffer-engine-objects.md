# Stream Buffer Engine Objects

The Stream Buffer Engine consists of the following objects.



| Object                                                | Description                                                                                                    |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [FileScan](filescan-object.md)                       | Repairs a corrupted .WTV file used in an SBE2 media graph.                                                     |
| [RecComp](reccomp-object.md)                         | Creates new recordings by concatenating existing recordings.                                                   |
| [RecordingAttributes](recordingattributes-object.md) | Retrieves and updates metadata in the header of a Stream Buffer Engine recording file.                         |
| [Recording](recording-object.md)                     | Creates permanent recordings.                                                                                  |
| [StreamBufferConfig](streambufferconfig-object.md)   | Configures the Stream Buffer Engine, by setting the location, number, and size of the temporary backing files. |
