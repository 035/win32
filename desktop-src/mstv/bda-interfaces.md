# BDA Interfaces

The following table lists the DirectShow interfaces that are used in the Broadcast Driver Architecture (BDA).



| Interface                                                                                    | Description                                                                                                                                                                                                                                                           |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IATSC\_EIT**](/previous-versions/windows/desktop/api/atscpsipparser/nn-atscpsipparser-iatsc_eit)                                                              | Retrieves data from an ATSC event information table (EIT).                                                                                                                                                                                                            |
| [**IATSC\_ETT**](/previous-versions/windows/desktop/api/atscpsipparser/nn-atscpsipparser-iatsc_ett)                                                              | Retrieves information from an extended text table (ETT).                                                                                                                                                                                                              |
| [**IATSC\_MGT**](/previous-versions/windows/desktop/api/atscpsipparser/nn-atscpsipparser-iatsc_mgt)                                                              | Retrieves data from a master guide table (MGT).                                                                                                                                                                                                                       |
| [**IATSC\_STT**](/previous-versions/windows/desktop/api/atscpsipparser/nn-atscpsipparser-iatsc_stt)                                                              | Retrieves data from a system time table (STT).                                                                                                                                                                                                                        |
| [**IATSC\_VCT**](/previous-versions/windows/desktop/api/atscpsipparser/nn-atscpsipparser-iatsc_vct)                                                              | Retrieves data from a virtual channel table (VCT).                                                                                                                                                                                                                    |
| [**IAtscContentAdvisoryDescriptor**](/previous-versions/windows/desktop/api/atscpsipparser/nn-atscpsipparser-iatsccontentadvisorydescriptor)                     | Gets content advisory descriptors from a Program and System Information Protocol (PSIP) table in an ATSC stream.                                                                                                                                                      |
| [**IAtscPsipParser**](/previous-versions/windows/desktop/api/atscpsipparser/nn-atscpsipparser-iatscpsipparser)                                                   | Retrieves ATSC Program and System Information Protocol (PSIP) tables.                                                                                                                                                                                                 |
| [**IAttributeGet**](/previous-versions/windows/desktop/api/dsattrib/nn-dsattrib-iattributeget)                                                       | Gets key/value pairs from an object, where the key is a GUID and the value is any binary data.                                                                                                                                                                        |
| [**IAttributeSet**](/previous-versions/windows/desktop/api/dsattrib/nn-dsattrib-iattributeset)                                                       | Sets key/value pairs on an object, where the key is a GUID and the value is any binary data.                                                                                                                                                                          |
| [**IBDA\_AutoDemodulate**](/windows/desktop/api/bdaiface/nn-bdaiface-ibda_autodemodulate)                                          | If a BDA device filter, specifically a demodulator, exposes this interface, it indicates that the filter can automatically detect signal characteristics.                                                                                                             |

| [**IBDA\_ConditionalAccessEx**](/windows/desktop/api/bdaiface/nn-bdaiface-ibda_conditionalaccessex)                                | Provides extended access to a device's Conditional Access Service (CAS) (Windows 7 and later).                                                                                                                                                                        |
| [**IBDA\_DeviceControl**](/windows/desktop/api/bdaiface/nn-bdaiface-ibda_devicecontrol)                                            | Implemented on all BDA device filters. The methods provided by this interface are called by a Network Provider to control a BDA device.                                                                                                                               |
| [**IBDA\_DiagnosticProperties**](/windows/desktop/api/bdaiface/)                              | Provides clients with read-only access to the properties of a BDA device filter for diagnostic purposes.                                                                                                                                                              |
