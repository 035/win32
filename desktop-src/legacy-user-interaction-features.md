# Legacy User Interaction Features

User interaction features for Windows 7 and earlier.

## In this section



| Topic                                                               | Description                                                                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Touch Input](https://msdn.microsoft.com/e4fa4615-6d12-4f21-a899-cfa2282baa6d)<br/>             | Describes how to use the Windows Touch API to build touch-enabled Windows-based applications.<br/>                                                                                                                                                                                                                                                                |
| [Keyboard and Mouse Input](https://msdn.microsoft.com/VS|winui|~\winui\windowsuserinterface\userinput.htm)<br/>          | Describes how to capture user input from the keyboard and the mouse.<br/>                                                                                                                                                                                                                                                                                         |
| [Multipoint Mouse](https://msdn.microsoft.com/windows/desktop/0d3f1d0f-9b25-4823-80eb-ae83189e3e62)<br/> | Describes the framework that enables you to build applications that support multiple mouse devices concurrently on one computer.<br/>                                                                                                                                                                                                                             |
| [Tablet PC](https://msdn.microsoft.com/f0fc7ab9-7653-45c5-be06-a628c6c3e526)<br/>          | Describes how to build applications for the Tablet PC, which is a fully functional personal computer geared for pen-enabled, handwriting-enabled, and speech-enabled applications. <br/>                                                                                                                                                                          |
| [DirectInput](https://msdn.microsoft.com/windows/desktop/c5d0c734-9f01-30f5-651c-82ac9c21310e)<br/>      | Describes how to use the [DirectInput](https://msdn.microsoft.com/windows/desktop/c5d0c734-9f01-30f5-651c-82ac9c21310e) API to process data from a joystick, or other game controller. DirectInput is legacy and not available for Windows Store apps. Use [XInput](https://msdn.microsoft.com/windows/desktop/fbbc651b-9264-9b5d-01c6-efc58f50b19d) instead.<br/>                                                                                      |
| [Text Services Framework](https://msdn.microsoft.com/ecc34b2e-89e8-48a8-8a8e-442d2145fe24)<br/>   | Describes how to use the Microsoft Windows Text Services Framework (TSF), which is an API that enables text services to supply text to applications in a way that does not require any detailed knowledge of the applications that produce or receive the text. For example, a text service can provide text input from speech or handwriting input devices.<br/> |
