# Supported platforms

This topic identifies the Rights Management Services SDK 2.1 supported client and server platforms.

## Supported platforms

-   Windows 10
-   Windows Server 2012 R2
-   Windows Server 2012
-   Windows 8
-   Windows Server 2008 R2 SP1
-   Windows 7 + SP1

> [!Note]  
> Use of the templates feature is supported beginning with Windows Server 2008.
