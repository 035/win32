# ID3DXAnimationSet::GetName method

Gets the animation set name.

## Syntax


```C++
LPCSTR GetName();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**LPCSTR**](winprog.windows_data_types)**

Name of the animation set.
