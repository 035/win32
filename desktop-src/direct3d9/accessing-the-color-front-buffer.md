# Accessing the Color Front Buffer (Direct3D 9)

Accessing the front buffer is allowed through the [**IDirect3DDevice9::GetFrontBufferData**](idirect3ddevice9--getfrontbufferdata.md) method. This method is the only way to get a screen shot of an anti-aliased scene.
