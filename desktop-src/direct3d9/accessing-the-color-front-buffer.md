# Accessing the Color Front Buffer (Direct3D 9)

Accessing the front buffer is allowed through the [**IDirect3DDevice9::GetFrontBufferData**](/windows/win32/d3d9helper/nf-d3d9-idirect3ddevice9-getfrontbufferdata?branch=master) method. This method is the only way to get a screen shot of an anti-aliased scene.
