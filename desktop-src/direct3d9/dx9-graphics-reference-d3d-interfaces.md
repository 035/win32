# Direct3D Interfaces

This section contains reference information for the Component Object Model (COM) interfaces provided by Microsoft Direct3D 9.

> [!Note]  
> Direct3D Windows Store apps can't use the Direct3D 9 API. Instead, use the Microsoft Direct3D 11 or later API with [feature level](direct3d11.overviews_direct3d_11_devices_downlevel_intro#overview) 9\_x.

 

The following interfaces are used with Direct3D 9.

-   [**ID3DXFile**](id3dxfile.md)
-   [**ID3DXFileData**](id3dxfiledata.md)
-   [**ID3DXFileEnumObject**](id3dxfileenumobject.md)
-   [**ID3DXFileSaveData**](id3dxfilesavedata.md)
-   [**ID3DXFileSaveObject**](id3dxfilesaveobject.md)
-   [**IDirect3D9**](/windows/win32/d3d9helper/nn-d3d9-idirect3d9?branch=master)
-   [**IDirect3DBaseTexture9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dbasetexture9?branch=master)
-   [**IDirect3DCubeTexture9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dcubetexture9?branch=master)
-   [**IDirect3DDevice9**](/windows/win32/d3d9helper/nn-d3d9-idirect3ddevice9?branch=master)
-   [**IDirect3DIndexBuffer9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dindexbuffer9?branch=master)
-   [**IDirect3DPixelShader9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dpixelshader9?branch=master)
-   [**IDirect3DQuery9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dquery9?branch=master)
-   [**IDirect3DResource9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dresource9?branch=master)
-   [**IDirect3DStateBlock9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dstateblock9?branch=master)
-   [**IDirect3DSurface9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dsurface9?branch=master)
-   [**IDirect3DSwapChain9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dswapchain9?branch=master)
-   [**IDirect3DTexture9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dtexture9?branch=master)
-   [**IDirect3DVertexBuffer9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dvertexbuffer9?branch=master)
-   [**IDirect3DVertexDeclaration9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dvertexdeclaration9?branch=master)
-   [**IDirect3DVertexShader9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dvertexshader9?branch=master)
-   [**IDirect3DVolume9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dvolume9?branch=master)
-   [**IDirect3DVolumeTexture9**](/windows/win32/d3d9helper/nn-d3d9-idirect3dvolumetexture9?branch=master)

-   [**IDirect3D9Ex**](/windows/win32/d3d9/nn-d3d9-idirect3d9ex?branch=master)
-   [**IDirect3D9ExOverlayExtension**](mf.idirect3d9exoverlayextension)

-   [**IDirect3DCryptoSession9**](mf.idirect3dcryptosession9)
