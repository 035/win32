# Direct3D Interfaces

This section contains reference information for the Component Object Model (COM) interfaces provided by Microsoft Direct3D 9.

> [!Note]  
> Direct3D Windows Store apps can't use the Direct3D 9 API. Instead, use the Microsoft Direct3D 11 or later API with [feature level](direct3d11.overviews_direct3d_11_devices_downlevel_intro#overview) 9\_x.

 

The following interfaces are used with Direct3D 9.

-   [**ID3DXFile**](id3dxfile.md)
-   [**ID3DXFileData**](id3dxfiledata.md)
-   [**ID3DXFileEnumObject**](id3dxfileenumobject.md)
-   [**ID3DXFileSaveData**](id3dxfilesavedata.md)
-   [**ID3DXFileSaveObject**](id3dxfilesaveobject.md)
-   [**IDirect3D9**](idirect3d9.md)
-   [**IDirect3DBaseTexture9**](idirect3dbasetexture9.md)
-   [**IDirect3DCubeTexture9**](idirect3dcubetexture9.md)
-   [**IDirect3DDevice9**](idirect3ddevice9.md)
-   [**IDirect3DIndexBuffer9**](idirect3dindexbuffer9.md)
-   [**IDirect3DPixelShader9**](idirect3dpixelshader9.md)
-   [**IDirect3DQuery9**](idirect3dquery9.md)
-   [**IDirect3DResource9**](idirect3dresource9.md)
-   [**IDirect3DStateBlock9**](idirect3dstateblock9.md)
-   [**IDirect3DSurface9**](idirect3dsurface9.md)
-   [**IDirect3DSwapChain9**](idirect3dswapchain9.md)
-   [**IDirect3DTexture9**](idirect3dtexture9.md)
-   [**IDirect3DVertexBuffer9**](idirect3dvertexbuffer9.md)
-   [**IDirect3DVertexDeclaration9**](idirect3dvertexdeclaration9.md)
-   [**IDirect3DVertexShader9**](idirect3dvertexshader9.md)
-   [**IDirect3DVolume9**](idirect3dvolume9.md)
-   [**IDirect3DVolumeTexture9**](idirect3dvolumetexture9.md)

The following interfaces are only available in Direct3D 9Ex.

-   [**IDirect3D9Ex**](idirect3d9ex.md)
-   [**IDirect3D9ExOverlayExtension**](mf.idirect3d9exoverlayextension)
-   [**IDirect3DAuthenticatedChannel9**](mf.idirect3dauthenticatedchannel9)
-   [**IDirect3DCryptoSession9**](mf.idirect3dcryptosession9)
-   [**IDirect3DDevice9Ex**](idirect3ddevice9ex.md)
-   [**IDirect3DDevice9Video**](mf.idirect3ddevice9video)
-   [**IDirect3DSwapChain9Ex**](d3d9l-idirect3dswapchain9.md)
