# D3DGETDATA\_FLUSH

This macro creates the value passed to [**GetData**](/windows/desktop/api) to flush query data.

``` syntax
#define D3DGETDATA_FLUSH (1 << 0)
```

## Remarks

For more information about using queries, see [Queries (Direct3D 9)](queries.md).
