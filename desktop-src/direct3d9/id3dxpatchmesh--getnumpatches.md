# ID3DXPatchMesh::GetNumPatches method

Gets the number of patches in the mesh.

## Syntax


```C++
DWORD GetNumPatches();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**DWORD**](https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46)**

The number of patches.
