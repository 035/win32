# ID3DXPatchMesh::GetOptions method

Gets the type of patch.

## Syntax


```C++
DWORD GetOptions();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**DWORD**](winprog.windows_data_types)**

The patch type.

## Remarks

For more information about patch types, see [**D3DXPATCHMESHTYPE**](direct3d9.d3dxpatchmeshtype).
