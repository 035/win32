# D3DCOMPOSERECTS\_MAXNUMRECTS

Specifies the maximum number of rectangles used in composing glyphs in [**ComposeRects**](/windows/win32/d3d9/nf-d3d9-idirect3ddevice9ex-composerects?branch=master).



|                              |                                                                                                                                      |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| \#define                     | Description                                                                                                                          |
| D3DCOMPOSERECTS\_MAXNUMRECTS | Maximum number of rectangle glyphs to compose together in a text string. See [**ComposeRects**](/windows/win32/d3d9/nf-d3d9-idirect3ddevice9ex-composerects?branch=master). |
