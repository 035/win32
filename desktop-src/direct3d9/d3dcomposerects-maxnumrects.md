# D3DCOMPOSERECTS\_MAXNUMRECTS

Specifies the maximum number of rectangles used in composing glyphs in [**ComposeRects**](idirect3ddevice9ex-composerect.md).



|                              |                                                                                                                                      |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| \#define                     | Description                                                                                                                          |
| D3DCOMPOSERECTS\_MAXNUMRECTS | Maximum number of rectangle glyphs to compose together in a text string. See [**ComposeRects**](idirect3ddevice9ex-composerect.md). |
