# D3DISSUE\_BEGIN

This macro creates a value used by [**Issue**](idirect3dquery9--issue.md) to issue a query begin.

``` syntax
#define D3DISSUE_BEGIN (1 << 1)
```

## Remarks

D3DISSUE\_BEGIN is valid for the following query type.

-   D3DQUERYTYPE\_OCCLUSION
