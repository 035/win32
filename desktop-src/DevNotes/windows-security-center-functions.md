# Windows Security Center Functions

The following functions are used with Windows Security Center.

## In this section



| Topic                                                                           | Description                                                                                                                                                                              |
|---------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WscGetSecurityProviderHealth**](wscgetsecurityproviderhealth.md)<br/> | Gets the aggregate health state of the security provider categories represented by the specified [**WSC\_SECURITY\_PROVIDER**](wsc-security-provider.md) enumeration values.<br/> |
| [**WscRegisterForChanges**](wscregisterforchanges.md)<br/>               | Registers a callback function to be run when Windows Security Center (WSC) detects a change that could affect the health of one of the security providers.<br/>                    |
| [**WscUnRegisterChanges**](wscunregisterchanges.md)<br/>                 | Cancels a callback registration that was made by a call to the [**WscRegisterForChanges**](wscregisterforchanges.md) function.<br/>                                               |
