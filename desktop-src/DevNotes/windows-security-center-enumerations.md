# Windows Security Center Enumerations

The following enumerations are used with Windows Security Center.

## In this section



| Topic                                                                                | Description                                                                                                      |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**WSC\_SECURITY\_PRODUCT\_STATE**](/windows/win32/Iwscapi/ne-iwscapi-wsc_security_product_state?branch=master)<br/>       | Defines the current state of the security product that is made available to Windows Security Center. <br/> |
| [**WSC\_SECURITY\_PROVIDER**](/windows/win32/Wscapi/ne-wscapi-_wsc_security_provider?branch=master)<br/>                  | Defines all the services that are monitored by Windows Security Center (WSC).<br/>                         |
| [**WSC\_SECURITY\_PROVIDER\_HEALTH**](/windows/win32/Wscapi/ne-wscapi-_wsc_security_provider_health?branch=master)<br/>   | Defines the possible states for any service monitored by Windows Security Center (WSC).<br/>               |
| [**WSC\_SECURITY\_SIGNATURE\_STATUS**](/windows/win32/Iwscapi/ne-iwscapi-_wsc_security_signature_status?branch=master)<br/> | Reports the current version status of the security product to Windows Security Center. <br/>               |
