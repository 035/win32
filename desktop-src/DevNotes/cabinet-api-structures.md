# Cabinet API Structures

This section details the following Cabinet API Structures:



| Structure                                              | Description                                                                                     |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**CABINETDLLVERSIONINFO**](cabinetdllversioninfo.md) | Contains the version information for Cabinet.dll.<br/>                                    |
| [**CCAB**](ccab.md)                                   | Contains cabinet information.<br/>                                                        |
| [**ERF**](erf.md)                                     | Returns error information from FCI/FDI. The caller should not modify this structure.<br/> |
| [**FDICABINETINFO**](fdicabinetinfo.md)               | Contains details about a particular cabinet file.<br/>                                    |
| [**FDINOTIFICATION**](fdinotification.md)             | Provide information to [**FNFDINOTIFY**](fnfdinotify.md).<br/>                           |
| [**SESSION**](session.md)                             | Contains information about the current session.<br/>                                      |
