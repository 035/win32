# Cabinet API Structures

This section details the following Cabinet API Structures:



| Structure                                              | Description                                                                                     |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**CABINETDLLVERSIONINFO**](cabinetdllversioninfo.md) | Contains the version information for Cabinet.dll.<br/>                                    |
| [**CCAB**](/windows/win32/Fci/ns-fci-ccab?branch=master)                                   | Contains cabinet information.<br/>                                                        |
| [**ERF**](/windows/win32/Fdi/ns-acpitabl-_firmware_boot_performance_table?branch=master)                                     | Returns error information from FCI/FDI. The caller should not modify this structure.<br/> |
| [**FDICABINETINFO**](/windows/win32/Fdi/ns-fdi-fdicabinetinfo?branch=master)               | Contains details about a particular cabinet file.<br/>                                    |
| [**FDINOTIFICATION**](/windows/win32/Fdi/ns-fdi-fdinotification?branch=master)             | Provide information to [**FNFDINOTIFY**](/windows/win32/fdi/nf-fdi-fnfdinotify?branch=master).<br/>                           |
| [**SESSION**](session.md)                             | Contains information about the current session.<br/>                                      |
