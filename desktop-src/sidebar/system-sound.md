# System.Sound object

\[The Windows Gadget Platform/Sidebar is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. \]

Defines the methods available for playing sounds.

## Members

The **System.Sound** object has these types of members:

-   [Methods](#methods)

### Methods

The **System.Sound** object has these methods.



| Method                                      | Description                                |
|:--------------------------------------------|:-------------------------------------------|
| [**beep**](system-sound-beep.md)           | Plays a simple waveform sound. <br/> |
| [**playSound**](system-sound-playsound.md) | Plays a sound file. <br/>            |
