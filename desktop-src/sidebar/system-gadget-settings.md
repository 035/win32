# System.Gadget.Settings object

\[The Windows Gadget Platform/Sidebar is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. \]

Defines the methods that provide gadget **Settings** functionality.

## Members

The **System.Gadget.Settings** object has these types of members:

-   [Methods](#methods)

### Methods

The **System.Gadget.Settings** object has these methods.



| Method                                                    | Description                                                                                         |
|:----------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| [**read**](system-gadget-settings-read.md)               | Retrieves a stored value of unspecified type associated with a gadget **Settings** key. <br/> |
| [**readString**](system-gadget-settings-readstring.md)   | Retrieves a stored **String** value associated with a gadget **Settings** key. <br/>          |
| [**write**](system-gadget-settings-write.md)             | Stores a value of unspecified type with an associated **Settings** key. <br/>                 |
| [**writeString**](system-gadget-settings-writestring.md) | Stores a **String** value with an associated **Settings** key. <br/>                          |
