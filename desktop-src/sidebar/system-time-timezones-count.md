# timeZones.count property

\[ The Windows Gadget Platform/Sidebar is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. \]

Gets the number of [**System.Time.timeZone**](system-time-timezone.md) items in the [**timeZones**](system-time-timezones.md) collection.

This property is read-only.

## Syntax


```JScript
icount = timeZones.count
```



## Property value

An **Integer** that receives the number of items.
