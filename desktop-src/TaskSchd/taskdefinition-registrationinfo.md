# TaskDefinition.RegistrationInfo property

For scripting, gets or sets the registration information that is used to describe a task, such as the description of the task, the author of the task, and the date the task is registered.

## Syntax


```VB
TaskDefinition.RegistrationInfo As RegistrationInfo
```



## Property value

The registration information that is used to describe a task, such as the description of the task, the author of the task, and the date the task is registered.
