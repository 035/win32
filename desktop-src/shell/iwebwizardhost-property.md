# WebWizardHost.Property property

Sets or retrieves a property's current value.

This property is read/write.

## Syntax


```JScript
Property = WebWizardHost.Property(bstrPropertyName)
WebWizardHost.Property = Property
```



## Property value

A variable of type [**BSTR**](https://msdn.microsoft.com/windows/desktop/1b2d7d2c-47af-4389-a6b6-b01b7e915228) that specifies or receives the property.
