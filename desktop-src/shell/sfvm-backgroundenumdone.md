# SFVM\_BACKGROUNDENUMDONE message

Notifies the callback object that background enumeration is complete. Used by [**IShellFolderViewCB::MessageSFVCB**](/windows/win32/Shlobj/?branch=master).


```C++

                SFVM_BACKGROUNDENUMDONE

            
```



## Parameters

This message has no parameters.
