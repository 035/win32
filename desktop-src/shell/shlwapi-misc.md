# Shell Miscellaneous Utility Functions

This section describes those Windows Shell functions exported by Shlwapi.dll and defined in Shlwapi.h and Shlwapi.lib that do not fall under one of the other utility function categories.

## In this section



| Topic                                                                   | Description                                                                                                                                                                                                                                                                      |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DllInstall**](dllinstall.md)<br/>                             | Handles installation and setup for a DLL.<br/>                                                                                                                                                                                                                             |
| [**GetProcessReference**](getprocessreference.md)<br/>           | Retrieves the process-specific object supplied by [**SetProcessReference**](setprocessreference.md), incrementing the reference count to keep the process alive.<br/>                                                                                                     |
| [**HashData**](hashdata.md)<br/>                                 | Hashes an array of data.<br/>                                                                                                                                                                                                                                              |
| [**IsInternetESCEnabled**](isinternetescenabled.md)<br/>         | Determines whether Windows Internet Explorer is in the Enhanced Security Configuration.<br/>                                                                                                                                                                               |
| [**IsOS**](isos.md)<br/>                                         | Checks for specified operating systems and operating system features.<br/>                                                                                                                                                                                                 |
| [**IStream\_Copy**](istream-copy.md)<br/>                        | Copies a stream to another stream.<br/>                                                                                                                                                                                                                                    |
| [**IStream\_Read**](istream-read.md)<br/>                        | Reads bytes from a specified stream and returns a value that indicates whether all bytes were successfully read.<br/>                                                                                                                                                      |
| [**IStream\_ReadPidl**](istream-readpidl.md)<br/>                | Reads a pointer to an item identifier list (PIDL) from an [**IStream**](stg.istream) object into a PIDLIST\_RELATIVE object.<br/>                                                                                                                                          |
| [**IStream\_ReadStr**](istream-readstr.md)<br/>                  | Reads from a stream and writes into a string.<br/>                                                                                                                                                                                                                         |
| [**IStream\_Reset**](istream-reset.md)<br/>                      | Moves the seek position in a specified stream to the beginning of the stream.<br/>                                                                                                                                                                                         |
| [**IStream\_Size**](istream-size.md)<br/>                        | Retrieves the size, in bytes, of a specified stream.<br/>                                                                                                                                                                                                                  |
| [**IStream\_Write**](istream-write.md)<br/>                      | Writes data of unknown format from a buffer to a specified stream.<br/>                                                                                                                                                                                                    |
| [**IStream\_WritePidl**](istream-writepidl.md)<br/>              | Writes a PIDL from a PCUIDLIST\_RELATIVE object into an [**IStream**](stg.istream) object.<br/>                                                                                                                                                                            |
| [**IStream\_WriteStr**](istream-writestr.md)<br/>                | Reads from a string and writes into a stream.<br/>                                                                                                                                                                                                                         |
| [**IUnknown\_AtomicRelease**](iunknown-atomicrelease.md)<br/>    | Releases a Component Object Model (COM) pointer and sets it to **NULL**.<br/>                                                                                                                                                                                              |
| [**IUnknown\_GetSite**](iunknown-getsite.md)<br/>                | Calls the specified object's [**IObjectWithSite::GetSite**](com.iobjectwithsite_getsite) method.<br/>                                                                                                                                                                      |
| [**IUnknown\_GetWindow**](iunknown-getwindow.md)<br/>            | Attempts to retrieve a window handle from a COM object by querying for various interfaces that have a **GetWindow** method.<br/>                                                                                                                                           |
| [**IUnknown\_QueryService**](iunknown-queryservice.md)<br/>      | Retrieves an interface for a service from a specified object.<br/>                                                                                                                                                                                                         |
| [**IUnknown\_Set**](iunknown-set.md)<br/>                        | Changes the value of a COM interface pointer and releases the previous interface.<br/>                                                                                                                                                                                     |
| [**IUnknown\_SetSite**](iunknown-setsite.md)<br/>                | Sets the specified object's site by calling its [**IObjectWithSite::SetSite**](com.iobjectwithsite_setsite) method.<br/>                                                                                                                                                   |
| [**ParseURL**](parseurl.md)<br/>                                 | Performs rudimentary parsing of a URL.<br/>                                                                                                                                                                                                                                |
| [**QISearch**](qisearch.md)<br/>                                 | A table-driven implementation of the [**IUnknown::QueryInterface**](com.iunknown_queryinterface) method.<br/>                                                                                                                                                              |
| [**SetProcessReference**](setprocessreference.md)<br/>           | Provides a COM object that allows hosted Shell extensions and other components to prevent their host process from closing prematurely. The host process is typically Windows Explorer or Internet Explorer, but this function can also be used by other applications.<br/> |
| [**SHAutoComplete**](shell.ShAutoComplete)<br/>                   | Instructs system edit controls to use AutoComplete to help complete URLs or file system paths.<br/>                                                                                                                                                                        |
| [**SHCreateMemStream**](shcreatememstream.md)<br/>               | Creates a memory stream using a similar process to [**CreateStreamOnHGlobal**](stg.createstreamonhglobal).<br/>                                                                                                                                                            |
| [**SHCreateStreamOnFileEx**](shcreatestreamonfileex.md)<br/>     | Opens or creates a file and retrieves a stream to read or write to that file.<br/>                                                                                                                                                                                         |
| [**SHCreateThread**](shcreatethread.md)<br/>                     | Creates a thread.<br/>                                                                                                                                                                                                                                                     |
| [**SHCreateThreadRef**](shcreatethreadref.md)<br/>               | Creates a per-thread reference to a COM object.<br/>                                                                                                                                                                                                                       |
| [**SHCreateThreadWithHandle**](shcreatethreadwithhandle.md)<br/> | Creates a new thread and retrieves its handle.<br/>                                                                                                                                                                                                                        |
| [**SHGetThreadRef**](shgetthreadref.md)<br/>                     | Retrieves the per-thread object reference set by [**SHSetThreadRef**](shsetthreadref.md).<br/>                                                                                                                                                                            |
| [**SHGlobalCounterDecrement**](shglobalcounterdecrement.md)<br/> | Decrements a global counter.<br/>                                                                                                                                                                                                                                          |
| [**SHGlobalCounterGetValue**](shglobalcountergetvalue.md)<br/>   | Gets the current value of a global counter.<br/>                                                                                                                                                                                                                           |
| [**SHGlobalCounterIncrement**](shglobalcounterincrement.md)<br/> | Increments a global counter.<br/>                                                                                                                                                                                                                                          |
| [**SHIsLowMemoryMachine**](shislowmemorymachine.md)<br/>         |                                                                                                                                                                                                                                                                                  |
| [**SHReleaseThreadRef**](shreleasethreadref.md)<br/>             | Releases a thread reference before the thread procedure returns.<br/>                                                                                                                                                                                                      |
| [**SHSetThreadRef**](shsetthreadref.md)<br/>                     | Stores a per-thread reference to a COM object. This allows the caller to control the thread's lifetime so that it can ensure that Windows won't shut down the thread before the caller is ready.<br/>                                                                      |
