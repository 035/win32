# Shell Macros

This section describes the Windows Shell utility macros.

## In this section



| Topic                                                                  | Description                                                                                                                                                                                                                                                     |
|------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DEFINE\_PROPERTYKEY**](define-propertykey.md)<br/>           | Used to pack a format identifier (FMTID) and property identifier (PID) into a [**PROPERTYKEY**](properties.PROPERTYKEY) structure that represents a property key.<br/>                                                                                    |
| [**IID\_PPV\_ARGS**](iid-ppv-args.md)<br/>                      | Used to retrieve an interface pointer, supplying the IID value of the requested interface automatically based on the type of the interface pointer used. This avoids a common coding error by checking the type of the value passed at compile time.<br/> |
| [**IsEqualPropertyKey**](isequalpropertykey.md)<br/>            | Compares the members of two [**PROPERTYKEY**](properties.PROPERTYKEY) structures and returns whether they are equal.<br/>                                                                                                                                 |
| [**MAKEDLLVERULL**](makedllverull.md)<br/>                      | Used to pack DLL version information into a ULONGLONG value.<br/>                                                                                                                                                                                         |
| [**NetAddr\_DisplayErrorTip**](netaddr-displayerrortip.md)<br/> | Displays an error message in the balloon tip associated with the network address control.<br/>                                                                                                                                                            |
| [**NetAddr\_GetAddress**](netaddr-getaddress.md)<br/>           | Indicates whether a network address conforms to a specified type and format.<br/>                                                                                                                                                                         |
| [**NetAddr\_GetAllowType**](netaddr-getallowtype.md)<br/>       | Retrieves the network address types that a specified network address control accepts.<br/>                                                                                                                                                                |
| [**NetAddr\_SetAllowType**](netaddr-setallowtype.md)<br/>       | Sets the network address types that a specified network address control accepts.<br/>                                                                                                                                                                     |
