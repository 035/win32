# Shell Macros

This section describes the Windows Shell utility macros.

## In this section



| Topic                                                                  | Description                                                                                                                                                                                                                                                     |
|------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DEFINE\_PROPERTYKEY**](/windows/win32/Propkeydef/nf-propkeydef-define_propertykey?branch=master)<br/>           | Used to pack a format identifier (FMTID) and property identifier (PID) into a [**PROPERTYKEY**](properties.PROPERTYKEY) structure that represents a property key.<br/>                                                                                    |
| [**IID\_PPV\_ARGS**](/windows/win32/Objbase/?branch=master)<br/>                      | Used to retrieve an interface pointer, supplying the IID value of the requested interface automatically based on the type of the interface pointer used. This avoids a common coding error by checking the type of the value passed at compile time.<br/> |
| [**IsEqualPropertyKey**](/windows/win32/Propkeydef/nf-propkeydef-isequalpropertykey?branch=master)<br/>            | Compares the members of two [**PROPERTYKEY**](properties.PROPERTYKEY) structures and returns whether they are equal.<br/>                                                                                                                                 |
| [**MAKEDLLVERULL**](/windows/win32/Shlwapi/nf-shlwapi-makedllverull?branch=master)<br/>                      | Used to pack DLL version information into a ULONGLONG value.<br/>                                                                                                                                                                                         |
| [**NetAddr\_DisplayErrorTip**](/windows/win32/Shellapi/nf-shellapi-netaddr_displayerrortip?branch=master)<br/> | Displays an error message in the balloon tip associated with the network address control.<br/>                                                                                                                                                            |
| [**NetAddr\_GetAddress**](/windows/win32/Shellapi/nf-shellapi-netaddr_getaddress?branch=master)<br/>           | Indicates whether a network address conforms to a specified type and format.<br/>                                                                                                                                                                         |
| [**NetAddr\_GetAllowType**](/windows/win32/Shellapi/nf-shellapi-netaddr_getallowtype?branch=master)<br/>       | Retrieves the network address types that a specified network address control accepts.<br/>                                                                                                                                                                |
| [**NetAddr\_SetAllowType**](/windows/win32/Shellapi/nf-shellapi-netaddr_setallowtype?branch=master)<br/>       | Sets the network address types that a specified network address control accepts.<br/>                                                                                                                                                                     |
