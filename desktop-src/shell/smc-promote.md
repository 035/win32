# SMC\_PROMOTE message

Promote the item specified by the accompanying [**SMDATA**](smdata.md) structure.


```C++
SMC_PROMOTE
```



## Parameters

This message has no parameters.

## Return value

Returns S\_OK.

## Remarks

This notification is received by the [**IShellMenuCallback::CallbackSM**](ishellmenucallback-callbacksm.md) method.
