# Shell Callback Functions

This section describes the Windows Shell callback functions.

## In this section



| Topic                                                                     | Description                                                                                                                                                                                                                   |
|---------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BFFCALLBACK**](browsecallbackproc.md)<br/>                      | Specifies an application-defined callback function used to send messages to, and process messages from, a **Browse** dialog box displayed in response to a call to [**SHBrowseForFolder**](shbrowseforfolder.md).<br/> |
| [*FMExtensionProc*](fmextensionproc.md)<br/>                       | Specifies an application-defined callback function called by File Manager to communicate with a File Manager extension.<br/>                                                                                            |
| [*MRUCMPPROC*](mrucmpproc.md)<br/>                                 | Used to determine whether an item is present in a most recently used (MRU) list.<br/>                                                                                                                                   |
| [**PAPPSTATE\_CHANGE\_ROUTINE**](pappstate-change-routine.md)<br/> | Specifies an app-defined callback function that notifies the app when the app is entering or leaving a suspended state.<br/>                                                                                            |
| [**SUBCLASSPROC**](subclassproc-function.md)<br/>                  | Defines the prototype for the callback function used by [**RemoveWindowSubclass**](removewindowsubclass.md) and [**SetWindowSubclass**](setwindowsubclass.md).<br/>                                                   |
| [**FM\_UNDELETE\_PROC**](undeletefile.md)<br/>                     | Specifies an application-defined callback function called by File Manager when the user chooses the **Undelete** command from the **File** menu.<br/>                                                                   |
