# Shell Callback Functions

This section describes the Windows Shell callback functions.

## In this section



| Topic                                                                     | Description                                                                                                                                                                                                                   |
|---------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BFFCALLBACK**](/windows/win32/Shlobj/nc-shlobj_core-bffcallback?branch=master)<br/>                      | Specifies an application-defined callback function used to send messages to, and process messages from, a **Browse** dialog box displayed in response to a call to [**SHBrowseForFolder**](/windows/win32/shlobj_core/nf-shlobj_core-shbrowseforfoldera?branch=master).<br/> |
| [*FMExtensionProc*](fmextensionproc.md)<br/>                       | Specifies an application-defined callback function called by File Manager to communicate with a File Manager extension.<br/>                                                                                            |
| [*MRUCMPPROC*](mrucmpproc.md)<br/>                                 | Used to determine whether an item is present in a most recently used (MRU) list.<br/>                                                                                                                                   |
| [**PAPPSTATE\_CHANGE\_ROUTINE**](/windows/win32/appnotify/nc-appnotify-pappstate_change_routine?branch=master)<br/> | Specifies an app-defined callback function that notifies the app when the app is entering or leaving a suspended state.<br/>                                                                                            |
| [**SUBCLASSPROC**](/windows/win32/Commctrl/nc-commctrl-subclassproc?branch=master)<br/>                  | Defines the prototype for the callback function used by [**RemoveWindowSubclass**](/windows/win32/Commctrl/nf-commctrl-removewindowsubclass?branch=master) and [**SetWindowSubclass**](/windows/win32/Commctrl/nf-commctrl-setwindowsubclass?branch=master).<br/>                                                   |
| [**FM\_UNDELETE\_PROC**](undeletefile.md)<br/>                     | Specifies an application-defined callback function called by File Manager when the user chooses the **Undelete** command from the **File** menu.<br/>                                                                   |
