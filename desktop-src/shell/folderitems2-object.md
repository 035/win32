# FolderItems2 object

Extends the [**FolderItems**](folderitems.md) object. It supports one additional method.

## Members

The **FolderItems2** object has these types of members:

-   [Methods](#methods)

### Methods

The **FolderItems2** object has these methods.



| Method                                            | Description                                                                                                                                                                                                                                         |
|:--------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InvokeVerbEx**](folderitems2-invokeverbex.md) | Executes a verb on a collection of [**FolderItem**](folderitem.md) objects. This method is an extension of the [**InvokeVerb**](folderitem-invokeverb.md) method, allowing additional control of the operation through a set of flags.<br/> |
