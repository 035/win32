# CFolderItems class

**CFolderItems** is a collection of [**FolderItems**](folderitems.md). It implements the following interfaces: [**IPersistFolder**](ipersistfolder.md), [**IObjectWithSite**](com.iobjectwithsite), [**FolderItems3**](folderitems3-object.md), [**IObjectSafety**](_inet_IObjectSafety_Interface_cpp), [**IDispatch**](ebbff4bc-36b2-4861-9efa-ffa45e013eb5).

## Remarks

The **CFolderItems** class only enumerates items from a specific folder.
