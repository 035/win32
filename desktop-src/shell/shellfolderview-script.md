# ShellFolderView.Script property

\[This property is not supported in Windows XP or later.\]

Contains the scripting object for the view.

This property is read-only.

## Syntax


```JScript
objScript = ShellFolderView.Script
```



## Property value

A variable of type [**IDispatch**](https://msdn.microsoft.com/windows/desktop/ebbff4bc-36b2-4861-9efa-ffa45e013eb5) that receives the scripting object.
