# Properties

This section contains the properties for the [**IManipulationProcessor**](/windows/win32/manipulations/nn-manipulations-imanipulationprocessor?branch=master) interface.

The following properties are exposed by the IManipulationProcessor interface.



| Property                                                                            | Description                                                                                                |
|-------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**MinimumScaleRotateRadius**](imanipulationprocessor-minimumscalerotateradius.md) | Specifies how large the distance contacts on a scale or rotate gesture need to be to trigger manipulation. |
| [**PivotPointX**](/windows/win32/manipulations/nf-manipulations-imanipulationprocessor-get_pivotpointx?branch=master)                           | This property is the horizontal center of the object.                                                      |
| [**PivotPointY**](/windows/win32/manipulations/nf-manipulations-imanipulationprocessor-get_pivotpointy?branch=master)                           | This property is the vertical center of the object.                                                        |
| [**PivotRadius**](/windows/win32/manipulations/nf-manipulations-imanipulationprocessor-get_pivotradius?branch=master)                           | This property is used to determine how much rotation is used in single finger manipulation.                |
| [**SupportedManipulations**](/windows/win32/manipulations/nf-manipulations-imanipulationprocessor-get_supportedmanipulations?branch=master)     | This property is used to indicate which manipulations are supported by an object.                          |
