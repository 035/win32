# Properties

This section contains the properties for the [**IManipulationProcessor**](imanipulationprocessor.md) interface.

The following properties are exposed by the IManipulationProcessor interface.



| Property                                                                            | Description                                                                                                |
|-------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**MinimumScaleRotateRadius**](imanipulationprocessor-minimumscalerotateradius.md) | Specifies how large the distance contacts on a scale or rotate gesture need to be to trigger manipulation. |
| [**PivotPointX**](imanipulationprocessor-pivotpointx.md)                           | This property is the horizontal center of the object.                                                      |
| [**PivotPointY**](imanipulationprocessor-pivotpointy.md)                           | This property is the vertical center of the object.                                                        |
| [**PivotRadius**](imanipulationprocessor-pivotradius.md)                           | This property is used to determine how much rotation is used in single finger manipulation.                |
| [**SupportedManipulations**](imanipulationprocessor-supportedmanipulations.md)     | This property is used to indicate which manipulations are supported by an object.                          |
