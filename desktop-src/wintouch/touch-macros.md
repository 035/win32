# Macros

The following macros are used for Windows Touch input.



| Macro                                                   | Description                                      |
|---------------------------------------------------------|--------------------------------------------------|
| [**TOUCH\_COORD\_TO\_PIXEL**](/windows/win32/winuser/nf-winuser-touch_coord_to_pixel?branch=master) | Converts touch coordinates to pixel coordinates. |
