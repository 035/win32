# Methods

This section contains the methods for the [**IManipulationProcessor**](imanipulationprocessor.md) interface.

The following methods are exposed by the [**IManipulationProcessor**](imanipulationprocessor.md) interface.



| Method                                                                      | Description                                                                              |
|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**CompleteManipulation**](imanipulationprocessor-completemanipulation.md) | This method is called when the developer chooses to end the manipulation.                |
| [**GetAngularVelocity**](imanipulationprocessor-getangularvelocity.md)     | Calculates the rotational velocity that the target object is moving at.                  |
| [**GetExpansionVelocity**](imanipulationprocessor-getexpansionvelocity.md) | Calculates the rate that the target object is expanding at.                              |
| [**GetVelocityX**](imanipulationprocessor-getvelocityx.md)                 | Calculates and returns the horizontal velocity for the target object.                    |
| [**GetVelocityY**](imanipulationprocessor-getvelocityy.md)                 | Calculates and returns the vertical velocity.                                            |
| [**ProcessDown**](imanipulationprocessor-processdown.md)                   | Feeds data to the manipulation processor associated with a target.                       |
| [**ProcessMove**](imanipulationprocessor-processmove.md)                   | Feeds data to the manipulation processor associated with a target.                       |
| [**ProcessUp**](imanipulationprocessor-processup.md)                       | Feeds data to the manipulation processor associated with a target.                       |
| [**ProcessDownWithTime**](imanpiulationprocessor-processdownwithtime.md)   | Feeds data including a timestamp to the manipulation processor associated with a target. |
| [**ProcessMoveWithTime**](imanpiulationprocessor-processmovewithtime.md)   | Feeds data including a timestamp to the manipulation processor associated with a target. |
| [**ProcessUpWithTime**](imanpiulationprocessor-processupwithtime.md)       | Feeds data including a timestamp to the manipulation processor associated with a target. |
