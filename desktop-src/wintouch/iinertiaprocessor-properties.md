# Properties

This section contains the properties for the [**IInertiaProcessor**](iinertiaprocessor.md) interface.



| Property                                                                               | Desription                                                                                               |
|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**InitialOriginX**](iinertiaprocessor-initialoriginx.md)                             | Specifies the starting horizontal location for a target with intertia.                                   |
| [**InitialOriginY**](iinertiaprocessor-initialoriginy.md)                             | Specifies the starting vertical location for a target with intertia.                                     |
| [**InitialVelocityX**](iinertiaprocessor-initialvelocityx.md)                         | Specifies the initial movement of the target object on the horizontal axis.                              |
| [**InitialVelocityY**](iinertiaprocessor-initialvelocityy.md)                         | Specifies the initial movement of the target object on the vertical axis.                                |
| [**InitialAngularVelocity**](iinertiaprocessor-initialangularvelocity.md)             | Specifies the rotational velocity of the target when movement begins.                                    |
| [**InitialExpansionVelocity**](iinertiaprocessor-initialexpansionvelocity.md)         | Specifies the rate of radius expansion of the target when movement begins.                               |
| [**InitialRadius**](iinertiaprocessor-initialradius.md)                               | Specifies the distance from the edge of the target to its center before the object was changed.          |
| [**BoundaryLeft**](iinertiaprocessor-leftboundary.md)                                 | Limits how far toward the left of the screen the target object can move.                                 |
| [**BoundaryTop**](iinertiaprocessor-topboundary.md)                                   | Limits how far toward the top of the screen the target object can move.                                  |
| [**BoundaryRight**](iinertiaprocessor-rightboundary.md)                               | Limits how far toward the right side of the screen the target object can move.                           |
| [**BoundaryBottom**](iinertiaprocessor-bottomboundary.md)                             | Limits how far toward the bottom of the screen the target object can move.                               |
| [**ElasticMarginLeft**](iinertiaprocessor-elasticmarginleft.md)                       | Specifies the leftmost region for bouncing the target object.                                            |
| [**ElasticMarginTop**](iinertiaprocessor-elasticmargintop.md)                         | Specifies the topmost region for bouncing the target object.                                             |
| [**ElasticMarginRight**](iinertiaprocessor-elasticmarginright.md)                     | Specifies the rightmost region for bouncing the target object.                                           |
| [**ElasticMarginBottom**](iinertiaprocessor-elasticmarginbottom.md)                   | Specifies the bottom region for bouncing the target object.                                              |
| [**DesiredAngularDeceleration**](iinertiaprocessor-desiredangulardeceleration.md)     | Specifies the desired rate that the target object will stop spinning in radians per msec.                |
| [**DesiredRotation**](iinertiaprocessor-desiredrotation.md)                           | Specifies the desired distance (in radians) that an object will be manipulated by the inertia processor. |
| [**DesiredExpansion**](iinertiaprocessor-desiredexpansion.md)                         | Specifies the desired change in the average radius of the object.                                        |
| [**DesiredExpansionDeceleration**](iinertiaprocessor-desiredexpansiondeceleration.md) | Specifies the rate at which the object will stop expanding.                                              |
| [**DesiredDeceleration**](iinertiaprocessor-desireddeceleration.md)                   | Specifies the desired rate at which translation operations will decelerate.                              |
| [**DesiredDisplacement**](iinertiaprocessor-desireddisplacement.md)                   | Specifies the desired distance that the object will travel.                                              |
| [**InitialTimestamp**](iinertiaprocessor-initialtimestamp.md)                         | Specifies the starting time stamp for a target object that has inertia.                                  |
