# Properties

This section contains the properties for the [**IInertiaProcessor**](/windows/win32/manipulations/nn-manipulations-iinertiaprocessor?branch=master) interface.



| Property                                                                               | Desription                                                                                               |
|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**InitialOriginX**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_initialoriginx?branch=master)                             | Specifies the starting horizontal location for a target with intertia.                                   |
| [**InitialOriginY**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_initialoriginy?branch=master)                             | Specifies the starting vertical location for a target with intertia.                                     |
| [**InitialVelocityX**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_initialvelocityx?branch=master)                         | Specifies the initial movement of the target object on the horizontal axis.                              |
| [**InitialVelocityY**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_initialvelocityy?branch=master)                         | Specifies the initial movement of the target object on the vertical axis.                                |
| [**InitialAngularVelocity**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_initialangularvelocity?branch=master)             | Specifies the rotational velocity of the target when movement begins.                                    |
| [**InitialExpansionVelocity**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_initialexpansionvelocity?branch=master)         | Specifies the rate of radius expansion of the target when movement begins.                               |
| [**InitialRadius**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_initialradius?branch=master)                               | Specifies the distance from the edge of the target to its center before the object was changed.          |
| [**BoundaryLeft**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_boundaryleft?branch=master)                                 | Limits how far toward the left of the screen the target object can move.                                 |
| [**BoundaryTop**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_boundarytop?branch=master)                                   | Limits how far toward the top of the screen the target object can move.                                  |
| [**BoundaryRight**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_boundaryright?branch=master)                               | Limits how far toward the right side of the screen the target object can move.                           |
| [**BoundaryBottom**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_boundarybottom?branch=master)                             | Limits how far toward the bottom of the screen the target object can move.                               |
| [**ElasticMarginLeft**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_elasticmarginleft?branch=master)                       | Specifies the leftmost region for bouncing the target object.                                            |
| [**ElasticMarginTop**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_elasticmargintop?branch=master)                         | Specifies the topmost region for bouncing the target object.                                             |
| [**ElasticMarginRight**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_elasticmarginright?branch=master)                     | Specifies the rightmost region for bouncing the target object.                                           |
| [**ElasticMarginBottom**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_elasticmarginbottom?branch=master)                   | Specifies the bottom region for bouncing the target object.                                              |
| [**DesiredAngularDeceleration**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_desiredangulardeceleration?branch=master)     | Specifies the desired rate that the target object will stop spinning in radians per msec.                |
| [**DesiredRotation**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_desiredrotation?branch=master)                           | Specifies the desired distance (in radians) that an object will be manipulated by the inertia processor. |
| [**DesiredExpansion**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_desiredexpansion?branch=master)                         | Specifies the desired change in the average radius of the object.                                        |
| [**DesiredExpansionDeceleration**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_desiredexpansiondeceleration?branch=master) | Specifies the rate at which the object will stop expanding.                                              |
| [**DesiredDeceleration**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_desireddeceleration?branch=master)                   | Specifies the desired rate at which translation operations will decelerate.                              |
| [**DesiredDisplacement**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_desireddisplacement?branch=master)                   | Specifies the desired distance that the object will travel.                                              |
| [**InitialTimestamp**](/windows/win32/manipulations/nf-manipulations-iinertiaprocessor-get_initialtimestamp?branch=master)                         | Specifies the starting time stamp for a target object that has inertia.                                  |
