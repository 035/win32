# Methods

This section contains the methods for the [**\_IManipulationEvents**](/windows/win32/manipulations/nn-manipulations-_imanipulationevents?branch=master) interface.



| Method                                                                      | Description                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------|
| [**ManipulationStarted**](/windows/win32/manipulations/nf-manipulations-_imanipulationevents-manipulationstarted?branch=master)     | Handles the event for when manipulation or inertia begins.    |
| [**ManipulationDelta**](/windows/win32/manipulations/nf-manipulations-_imanipulationevents-manipulationdelta?branch=master)         | Handles events that happen when a manipulated object changes. |
| [**ManipulationCompleted**](/windows/win32/manipulations/nf-manipulations-_imanipulationevents-manipulationcompleted?branch=master) | Handles the event when manipulation or inertia finishes.      |
