# Methods

This section contains the methods for the [**\_IManipulationEvents**](-imanipulationevents.md) interface.



| Method                                                                      | Description                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------|
| [**ManipulationStarted**](-imanipulationevents-manipulationstarted.md)     | Handles the event for when manipulation or inertia begins.    |
| [**ManipulationDelta**](-imanipulationevents-manipulationdelta.md)         | Handles events that happen when a manipulated object changes. |
| [**ManipulationCompleted**](-imanipulationevents-manipulationcompleted.md) | Handles the event when manipulation or inertia finishes.      |
