# Functions

This section describes the boundary feedback functions.



| Function                                               | Description                                                                                        |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**BeginPanningFeedback**](/windows/win32/uxtheme/?branch=master)     | This function is invoked by the user before using boundary feedback.                               |
| [**EndPanningFeedback**](/windows/win32/uxtheme/?branch=master)     | This function is called upon completion of boundary feedback and optionally triggers an animation. |
| [**UpdatePanningFeedback**](/windows/win32/uxtheme/?branch=master) | This function is called during boundary feedback and optionally triggers an animation.             |
