# Structures

This section contains structures for Windows Touch gestures.

The following structures are specified in this section.



| Structure                                          | Description                                                                              |
|----------------------------------------------------|------------------------------------------------------------------------------------------|
| [**GESTURECONFIG**](gestureconfig.md)             | Is used to set the configuration of which gesture messages are enabled or disabled.      |
| [**GESTUREINFO**](gestureinfo.md)                 | Contains information about a gesture.                                                    |
| [**GESTURENOTIFYSTRUCT**](gesturenotifystruct.md) | Contains information about a [**WM\_GESTURENOTIFY**](wm-gesturenotify.md) notification. |
