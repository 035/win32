# Classes and Interfaces

This section contains class and interface references for inertia and manipulations.

The following classes and interfaces are used for inertia and manipulations.



| Class/Interface                                          | Description                                                                  |
|----------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_IManipulationEvents**](/windows/win32/manipulations/nn-manipulations-_imanipulationevents?branch=master)    | Handles manipulation events.                                                 |
| [**IInertiaProcessor**](/windows/win32/manipulations/nn-manipulations-iinertiaprocessor?branch=master)           | Handles calculations regarding object motion for Windows Touch.              |
| [**IManipulationProcessor**](/windows/win32/manipulations/nn-manipulations-imanipulationprocessor?branch=master) | Provides functionality for monitoring and responding to Windows Touch input. |
