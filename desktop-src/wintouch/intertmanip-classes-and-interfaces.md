# Classes and Interfaces

This section contains class and interface references for inertia and manipulations.

The following classes and interfaces are used for inertia and manipulations.



| Class/Interface                                          | Description                                                                  |
|----------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_IManipulationEvents**](-imanipulationevents.md)    | Handles manipulation events.                                                 |
| [**IInertiaProcessor**](iinertiaprocessor.md)           | Handles calculations regarding object motion for Windows Touch.              |
| [**IManipulationProcessor**](imanipulationprocessor.md) | Provides functionality for monitoring and responding to Windows Touch input. |
