# Update method of the MSFT\_FSRMClassificationPropertyDefinition class

Forces an update of the global classification property definitions from Active Directory if any. This method attempts to synchronize the server s property definitions with the Resource property definitions in Active Directory.

## Syntax


```mof
uint64 Update();
```



## Parameters

This method has no parameters.
