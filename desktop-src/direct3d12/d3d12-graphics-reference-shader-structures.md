# Shader Structures

d3d12shader.h declares the following structures, which are used to create and use shaders.

## In this section



| Topic                                                                                  | Description                                                             |
|----------------------------------------------------------------------------------------|-------------------------------------------------------------------------|
| [**D3D12\_FUNCTION\_DESC**](d3d12-function-desc.md)<br/>                        | Describes a function. <br/>                                       |
| [**D3D12\_LIBRARY\_DESC**](d3d12-library-desc.md)<br/>                          | Describes a library. <br/>                                        |
| [**D3D12\_PARAMETER\_DESC**](d3d12-parameter-desc.md)<br/>                      | Describes a function parameter. <br/>                             |
| [**D3D12\_SHADER\_BUFFER\_DESC**](d3d12-shader-buffer-desc.md)<br/>             | Describes a shader constant-buffer. <br/>                         |
| [**D3D12\_SHADER\_DESC**](d3d12-shader-desc.md)<br/>                            | Describes a shader. <br/>                                         |
| [**D3D12\_SHADER\_INPUT\_BIND\_DESC**](d3d12-shader-input-bind-desc.md)<br/>    | Describes how a shader resource is bound to a shader input. <br/> |
| [**D3D12\_SHADER\_TYPE\_DESC**](d3d12-shader-type-desc.md)<br/>                 | Describes a shader-variable type. <br/>                           |
| [**D3D12\_SHADER\_VARIABLE\_DESC**](d3d12-shader-variable-desc.md)<br/>         | Describes a shader variable. <br/>                                |
| [**D3D12\_SIGNATURE\_PARAMETER\_DESC**](d3d12-signature-parameter-desc.md)<br/> | Describes a shader signature. <br/>                               |
