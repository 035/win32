# D3D12_GPU_VIRTUAL_ADDRESS

A D3D12_GPU_VIRTUAL_ADDRESS represents a virtual address in GPU memory. The data type is defined in Direct3d12.h as follows:


```
typedef UINT64 D3D12_GPU_VIRTUAL_ADDRESS;
```
