# Core Functions

The following functions are declared in d3d12.h.

## In this section



| Topic                                                                                                             | Description                                                                                                                                                                                                 |
|-------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**D3D12CreateDevice**](d3d12createdevice.md)<br/>                                                         | Creates a device that represents the display adapter. <br/>                                                                                                                                           |
| [**D3D12CreateRootSignatureDeserializer**](d3d12createrootsignaturedeserializer.md)<br/>                   | Deserializes a root signature so you can determine the layout definition ([**D3D12\_ROOT\_SIGNATURE\_DESC**](d3d12-root-signature-desc.md)). <br/>                                                   |
| [**D3D12CreateVersionedRootSignatureDeserializer**](d3d12createversionedrootsignaturedeserializer.md)<br/> | Generates an interface that can return the deserialized data structure, via [**GetUnconvertedRootSignatureDesc**](id3d12versionedrootsignaturedeserializer-getunconvertedrootsignaturedesc.md).<br/> |
| [**D3D12EnableExperimentalFeatures**](d3d12enableexperimentalfeatures.md)<br/>                             | Enables a list of experimental features.<br/>                                                                                                                                                         |
| [**D3D12GetDebugInterface**](d3d12getdebuginterface.md)<br/>                                               | Gets a debug interface. <br/>                                                                                                                                                                         |
| [**D3D12SerializeRootSignature**](d3d12serializerootsignature.md)<br/>                                     | Serializes a root signature version 1.0 that can be passed to [**ID3D12Device::CreateRootSignature**](id3d12device-createrootsignature.md). <br/>                                                    |
| [**D3D12SerializeVersionedRootSignature**](d3d12serializeversionedrootsignature.md)<br/>                   | Serializes a root signature of any version that can be passed to [**ID3D12Device::CreateRootSignature**](id3d12device-createrootsignature.md). <br/>                                                 |
