# Interface Hierarchy

The diagram shows the interface inheritance hierarchy.

Most apps will use only the interfaces at the top of the hierarchy, and not directly use the inherited interfaces; [**ID3D12Pageable**](id3d12pageable.md), [**ID3D12DeviceChild**](id3d12devicechild.md), [**ID3D12Object**](id3d12object.md) and [**ID3D12CommandList**](id3d12commandlist.md).

New interfaces are shown in a green box.
