# ID3D12Heap::GetDesc method

Gets the heap description.

## Syntax


```C++
D3D12_HEAP_DESC GetDesc();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**D3D12\_HEAP\_DESC**](d3d12-heap-desc.md)**

Returns the [**D3D12\_HEAP\_DESC**](d3d12-heap-desc.md) structure that describes the heap.
