# CQPM\_RELEASE message

The **CQPM\_RELEASE** message is sent to the [**CQPageProc**](cqpageproc.md) callback function of a query form extension page when the page is about to be unloaded.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Not used.

</dd> <dt>

*lParam* 
</dt> <dd>

Not used.

</dd> </dl>

## Return value

The return value for this message is ignored.
