# CFSTR\_DS\_PARENTHWND

<dl> <dt>

<span id="CFSTR_DS_PARENTHWND"></span><span id="cfstr_ds_parenthwnd"></span>**CFSTR\_DS\_PARENTHWND**
</dt> <dd> <dl> <dt>

"DsAdminParentHwndClipFormat"
</dt> <dt>



The **CFSTR\_DS\_PARENTHWND** clipboard format provides a global memory handle (**HGLOBAL**) that contains the window handle (**HWND**) of the main MMC window.

> [!Note]  
> This clipboard format value is not defined in a published header file. To use this clipboard format value, define it in the exact format shown.
