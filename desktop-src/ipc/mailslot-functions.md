# Mailslot Functions

The following functions can be used with mailslots.



| Function                                   | Description                                                                  |
|--------------------------------------------|------------------------------------------------------------------------------|
| [**CreateMailslot**](createmailslot.md)   | Creates a mailslot with the specified name.                                  |
| [**GetMailslotInfo**](getmailslotinfo.md) | Retrieves information about the specified mailslot.                          |
| [**SetMailslotInfo**](setmailslotinfo.md) | Sets the time-out value used by the specified mailslot for a read operation. |
