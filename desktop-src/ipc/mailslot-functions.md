# Mailslot Functions

The following functions can be used with mailslots.



| Function                                   | Description                                                                  |
|--------------------------------------------|------------------------------------------------------------------------------|
| [**CreateMailslot**](/windows/win32/Winbase/nf-winbase-createmailslota?branch=master)   | Creates a mailslot with the specified name.                                  |
| [**GetMailslotInfo**](/windows/win32/Winbase/nf-winbase-getmailslotinfo?branch=master) | Retrieves information about the specified mailslot.                          |
| [**SetMailslotInfo**](/windows/win32/Winbase/nf-winbase-setmailslotinfo?branch=master) | Sets the time-out value used by the specified mailslot for a read operation. |
