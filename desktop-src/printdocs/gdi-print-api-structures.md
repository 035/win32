# GDI Print API Structures

## In this section



| Structure                                                      | Description                                                                                                                                                                                                                                                            |
|----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DEVMODE**](/windows/desktop/api/Wingdi/ns-wingdi-_devicemodea)<br/>                          | The [**DEVMODE**](https://msdn.microsoft.com/library/windows/desktop/dd183565) data structure contains information about the initialization and environment of a printer or a display device. <br/>                                                                                                              |
| [**DOCINFO**](/windows/desktop/api/Wingdi/ns-wingdi-_docinfoa)<br/>                          | The [**DOCINFO**](https://msdn.microsoft.com/library/windows/desktop/dd183574) structure contains the input and output file names and other information used by the [**StartDoc**](/windows/desktop/api/Wingdi/nf-wingdi-startdoca) function.<br/>                                                                                                  |
| [**DRAWPATRECT**](/windows/desktop/api/Wingdi/ns-wingdi-_drawpatrect)<br/>                  | The [**DRAWPATRECT**](https://msdn.microsoft.com/library/windows/desktop/dd162494) structure defines a rectangle to be created.<br/>                                                                                                                                                                         |
| [**PSFEATURE\_CUSTPAPER**](/windows/desktop/api/Wingdi/ns-wingdi-_psfeature_custpaper)<br/> | The [**PSFEATURE\_CUSTPAPER**](https://msdn.microsoft.com/library/windows/desktop/dd162875) structure contains information about a custom paper size for a PostScript driver. This structure is used with the [**GET\_PS\_FEATURESETTING**](https://msdn.microsoft.com/en-us/library/Dd144954(v=VS.85).aspx) printer escape function.<br/> |
| [**PSFEATURE\_OUTPUT**](/windows/desktop/api/Wingdi/ns-wingdi-_psfeature_output)<br/>       | The [**PSFEATURE\_OUTPUT**](https://msdn.microsoft.com/library/windows/desktop/dd162876) structure contains information about PostScript driver output options. This structure is used with the [**GET\_PS\_FEATURESETTING**](https://msdn.microsoft.com/en-us/library/Dd144954(v=VS.85).aspx) printer escape function.<br/>                  |
| [**PSINJECTDATA**](/windows/desktop/api/Wingdi/ns-wingdi-_psinjectdata)<br/>                | The [**PSINJECTDATA**](https://msdn.microsoft.com/library/windows/desktop/dd162877) structure is a header for the input buffer used with the [**POSTSCRIPT\_INJECTION**](https://msdn.microsoft.com/en-us/library/Dd162830(v=VS.85).aspx) printer escape function.<br/>                                                                            |
