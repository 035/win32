# GDI Printer Escape Functions

The following printer escape functions are used with the [**Escape**](/windows/desktop/api/Wingdi/nf-wingdi-escape) or [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) functions.

## In this section



| Function                                                                            | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|-------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CHECKJPEGFORMAT Printer Escape**](https://www.bing.com/search?q=**CHECKJPEGFORMAT Printer Escape**)<br/>                | The [**CHECKJPEGFORMAT**](https://msdn.microsoft.com/library/windows/desktop/dd183421) printer escape function determines whether a printer supports printing a JPEG image.<br/> To perform this check, call the [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) function with the following parameters.<br/>                                                                                                                                                                                                                                                                                                                  |
| [**CHECKPNGFORMAT Printer Escape**](https://www.bing.com/search?q=**CHECKPNGFORMAT Printer Escape**)<br/>                  | The [**CHECKPNGFORMAT**](https://msdn.microsoft.com/library/windows/desktop/dd183424) printer escape function determines whether a printer supports printing a PNG image.<br/> To perform this check, call the [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) function with the following parameters.<br/>                                                                                                                                                                                                                                                                                                                     |
| [**DRAWPATTERNRECT Printer Escape**](https://www.bing.com/search?q=**DRAWPATTERNRECT Printer Escape**)<br/>                | The [**DRAWPATTERNRECT**](https://msdn.microsoft.com/library/windows/desktop/dd162495) printer escape creates a white, gray scale, or solid black rectangle by using the pattern and rule capabilities of Page Control Language (PCL) on Hewlett-Packard LaserJet or LaserJet-compatible printers. A gray scale is a gray pattern that contains a specific mixture of black and white pixels.<br/> To perform this operation, call the [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) function with the following parameters.<br/>                                                                                            |
| [**GET\_PS\_FEATURESETTING Printer Escape**](https://www.bing.com/search?q=**GET\_PS\_FEATURESETTING Printer Escape**)<br/>  | The [**GET\_PS\_FEATURESETTING**](https://msdn.microsoft.com/library/windows/desktop/dd144954) printer escape function retrieves information about a specified feature setting for a PostScript driver.<br/> This escape function is supported only if the PostScript driver is in PostScript-centric mode or in GDI-centric mode. To set the PostScript driver mode, call the [**POSTSCRIPT\_IDENTIFY**](https://www.bing.com/search?q=**POSTSCRIPT\_IDENTIFY**) escape function.<br/> To perform this operation, call the [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) function with the following parameters.<br/>                          |
| [**GETTECHNOLOGY Printer Escape**](https://www.bing.com/search?q=**GETTECHNOLOGY Printer Escape**)<br/>                    | The [**GETTECHNOLOGY**](https://www.bing.com/search?q=**GETTECHNOLOGY**) printer escape function identifies the type of printer driver.<br/> To perform this operation, call the [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) function with the following parameters.<br/>                                                                                                                                                                                                                                                                                                                                        |
| [**PASSTHROUGH Printer Escape**](https://www.bing.com/search?q=**PASSTHROUGH Printer Escape**)<br/>                        | The [**PASSTHROUGH**](https://msdn.microsoft.com/library/windows/desktop/dd162776) printer escape function sends data directly to a printer driver. To perform this operation, call the [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) function with the following parameters.<br/>                                                                                                                                                                                                                                                                                                                                                     |
| [**POSTSCRIPT\_DATA Printer Escape**](https://www.bing.com/search?q=**POSTSCRIPT\_DATA Printer Escape**)<br/>               | The [**POSTSCRIPT\_DATA**](https://msdn.microsoft.com/library/windows/desktop/dd162828) printer escape function sends data directly to a printer driver. To perform this operation, call the [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) function with the following parameters.<br/>                                                                                                                                                                                                                                                                                                                                            |
| [**POSTSCRIPT\_IDENTIFY Printer Escape**](https://www.bing.com/search?q=**POSTSCRIPT\_IDENTIFY Printer Escape**)<br/>       | The [**POSTSCRIPT\_IDENTIFY**](https://msdn.microsoft.com/library/windows/desktop/dd162829) printer escape function sets a PostScript driver to GDI-centric mode or PostScript-centric mode.<br/> To perform this operation, call the [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) function with the following parameters.<br/>                                                                                                                                                                                                                                                                                         |
| [**POSTSCRIPT\_INJECTION Printer Escape**](https://www.bing.com/search?q=**POSTSCRIPT\_INJECTION Printer Escape**)<br/>     | The [**POSTSCRIPT\_INJECTION**](https://msdn.microsoft.com/library/windows/desktop/dd162830) printer escape function inserts a block of raw data at a specified point in a PostScript job stream.<br/> A PostScript driver supports this escape function in GDI-centric mode or PostScript-centric mode support, but not in compatibility mode.<br/> To set the PostScript driver's mode, call the [**POSTSCRIPT\_IDENTIFY**](https://www.bing.com/search?q=**POSTSCRIPT\_IDENTIFY**) escape function.<br/> To perform this operation, call the [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) function with the following parameters.<br/> |
| [**POSTSCRIPT\_PASSTHROUGH Printer Escape**](https://www.bing.com/search?q=**POSTSCRIPT\_PASSTHROUGH Printer Escape**)<br/> | The [**POSTSCRIPT\_PASSTHROUGH**](https://msdn.microsoft.com/library/windows/desktop/dd162831) printer escape function sends data directly to a PostScript printer driver.<br/> A PostScript driver supports this escape function when in PostScript-centric mode or in compatibility mode, but not in GDI-centric mode.<br/> To set the PostScript driver's mode, call the [**POSTSCRIPT\_IDENTIFY**](https://www.bing.com/search?q=**POSTSCRIPT\_IDENTIFY**) escape function.<br/> To perform this operation, call the [**ExtEscape**](/windows/desktop/api/Wingdi/nf-wingdi-extescape) function with the following parameters.<br/>                      |
| [**QUERYESCSUPPORT Printer Escape**](https://www.bing.com/search?q=**QUERYESCSUPPORT Printer Escape**)<br/>                | The [**QUERYESCSUPPORT**](https://www.bing.com/search?q=**QUERYESCSUPPORT**) printer escape function checks the implementation of a printer escape function.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**SPCLPASSTHROUGH2 Printer Escape**](https://www.bing.com/search?q=**SPCLPASSTHROUGH2 Printer Escape**)<br/>              | The [**SPCLPASSTHROUGH2**](https://msdn.microsoft.com/library/windows/desktop/dd145110) printer escape function allows applications that print to PostScript devices using EPSPRINTING to include private PostScript procedures and other resources at the document-level save context.<br/> This escape is supported only for backward compatibility with Adobe Acrobat.
