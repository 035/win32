# Router Manager (Transport) Functions

Use the following functions to administer the router managers. These functions also allow a developer to read and write the global information for the router managers, and the global information for router clients (such as routing protocols).



| Administration function                                      | Configuration function                                             |
|--------------------------------------------------------------|--------------------------------------------------------------------|
| [**MprAdminTransportCreate**](/windows/win32/Mprapi/nf-mprapi-mpradmintransportcreate?branch=master)   | [**MprConfigTransportCreate**](/windows/win32/Mprapi/nf-mprapi-mprconfigtransportcreate?branch=master)       |
| No administration function                                   | [**MprConfigTransportDelete**](/windows/win32/Mprapi/nf-mprapi-mprconfigtransportdelete?branch=master)       |
| [**MprAdminTransportGetInfo**](/windows/win32/Mprapi/nf-mprapi-mpradmintransportgetinfo?branch=master) | [**MprConfigTransportGetInfo**](/windows/win32/Mprapi/nf-mprapi-mprconfigtransportgetinfo?branch=master)     |
| [**MprAdminTransportSetInfo**](/windows/win32/Mprapi/nf-mprapi-mpradmintransportsetinfo?branch=master) | [**MprConfigTransportSetInfo**](/windows/win32/Mprapi/nf-mprapi-mprconfigtransportsetinfo?branch=master)     |
| No administration function                                   | [**MprConfigTransportEnum**](/windows/win32/Mprapi/nf-mprapi-mprconfigtransportenum?branch=master)           |
| No administration function                                   | [**MprConfigTransportGetHandle**](/windows/win32/Mprapi/nf-mprapi-mprconfigtransportgethandle?branch=master) |
