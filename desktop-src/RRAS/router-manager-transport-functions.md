# Router Manager (Transport) Functions

Use the following functions to administer the router managers. These functions also allow a developer to read and write the global information for the router managers, and the global information for router clients (such as routing protocols).



| Administration function                                      | Configuration function                                             |
|--------------------------------------------------------------|--------------------------------------------------------------------|
| [**MprAdminTransportCreate**](mpradmintransportcreate.md)   | [**MprConfigTransportCreate**](mprconfigtransportcreate.md)       |
| No administration function                                   | [**MprConfigTransportDelete**](mprconfigtransportdelete.md)       |
| [**MprAdminTransportGetInfo**](mpradmintransportgetinfo.md) | [**MprConfigTransportGetInfo**](mprconfigtransportgetinfo.md)     |
| [**MprAdminTransportSetInfo**](mpradmintransportsetinfo.md) | [**MprConfigTransportSetInfo**](mprconfigtransportsetinfo.md)     |
| No administration function                                   | [**MprConfigTransportEnum**](mprconfigtransportenum.md)           |
| No administration function                                   | [**MprConfigTransportGetHandle**](mprconfigtransportgethandle.md) |
