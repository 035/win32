# RAS User Administration Functions

Use the following functions to manage dial-up users:

-   [**MprAdminGetPDCServer**](/windows/win32/Mprapi/nf-mprapi-mpradmingetpdcserver?branch=master)
-   [**MprAdminSendUserMessage**](/windows/win32/Mprapi/nf-mprapi-mpradminsendusermessage?branch=master)
-   [**MprAdminUserGetInfo**](/windows/win32/Mprapi/nf-mprapi-mpradminusergetinfo?branch=master)
-   [**MprAdminUserSetInfo**](/windows/win32/Mprapi/nf-mprapi-mpradminusersetinfo?branch=master)

To obtain a list of current users on a particular domain, use the [**NetQueryDisplayInformation**](_win32_netquerydisplayinformation) function. The prototype for this function is in the lmaccess.h header file.
