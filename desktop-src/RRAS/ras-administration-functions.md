# RAS Administration Functions

This documentation describes RRAS functions that are used to develop software to administer RAS dial-up connections. These functions include:

-   [**MprAdminConnectionClearStats**](mpradminconnectionclearstats.md)
-   [**MprAdminConnectionEnum**](mpradminconnectionenum.md)
-   [**MprAdminConnectionEnumEx**](mpradminconnectionenumex.md)
-   [**MprAdminConnectionGetInfo**](mpradminconnectiongetinfo.md)
-   [**MprAdminConnectionGetInfoEx**](mpradminconnectiongetinfoex.md)
-   [**MprAdminConnectionRemoveQuarantine**](mpradminconnectionremovequarantine.md)
-   [**MprAdminPortClearStats**](mpradminportclearstats.md)
-   [**MprAdminPortDisconnect**](mpradminportdisconnect.md)
-   [**MprAdminPortEnum**](mpradminportenum.md)
-   [**MprAdminPortGetInfo**](mpradminportgetinfo.md)
-   [**MprAdminPortReset**](mpradminportreset.md)

Additional functions are used for both RAS administration and router administration. The following functions documented in the [Router Administration Functions](router-administration-functions.md) reference:

-   [**MprAdminBufferFree**](mpradminbufferfree.md)
-   [**MprAdminGetErrorString**](mpradmingeterrorstring.md)
-   [**MprAdminIsServiceRunning**](mpradminisservicerunning.md)
-   [**MprAdminServerConnect**](mpradminserverconnect.md)
-   [**MprAdminServerDisconnect**](mpradminserverdisconnect.md)

To implement a RAS administration DLL, use the functions described in the following topic:

-   [RAS Admin DLL Functions](ras-admin-dll-functions.md)

To manage users of a RAS server, use the functions described in the following topic:

-   [RAS User Administration Functions](ras-user-administration-functions.md)
