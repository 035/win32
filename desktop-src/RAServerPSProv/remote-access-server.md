# RAServerPSProvider Provider

The RAServerPSProvider provider defines classes to manage a remote access server in a Remote Access Services (RAS) environment. This section contains the classes defined by RAServerPSProvider.

## In this section

<dl> <dt>

[**CIM\_Error**](cim-error.md)
</dt> <dd>

CIM\_Error is a specialized class that contains information about the severity, cause, recommended actions and other data related to the failure of a CIM Operation. Instances of this type MAY be included as part of the response to a CIM Operation.

</dd> <dt>

[**DACertificateContext**](dacertificatecontext.md)
</dt> <dd>

Defines the DirectAccess certificate context object.

</dd> <dt>

[**DACorpPrefixConfiguration**](dacorpprefixconfiguration.md)
</dt> <dd>

Defines an object that contains the Corp Prefix, base Prefix, and a Boolean value that indicates whether native IPv6 is deployed.

</dd> <dt>

[**DASiteAddresses**](dasiteaddresses.md)
</dt> <dd>

Defines the list of IPAddresses corresponding to all the codes of a DirectAccess site.

</dd> <dt>

[**MSFT\_WmiError**](msft-wmierror.md)
</dt> <dd>

A Windows error object that encapsulates what is in CIM\_Error as well as includes error code, errorCategory, errorMessage and errorType.

</dd> <dt>

[**RemoteAccessActiveConnectionSummaryLocal**](remoteaccessactiveconnectionsummarylocal.md)
</dt> <dd>

Remote Access Connection Statistics Summary for Active Connections

</dd> <dt>

[**RemoteAccessError**](remoteaccesserror.md)
</dt> <dd>

An extended error object that contains the error details of the RemoteAccess cmdlets. It inherits from the MSFT\_WmiErrorand extends it by adding the following fields to the error object

</dd> <dt>

[**PS\_RemoteAccessAccountingStatisticsSummary**](ps-remoteaccessaccountingstatisticssummary.md)
</dt> <dd>

Refers to the connection statistics summary based on accounting data

</dd> <dt>

[**PS\_RemoteAccessConnectionStatisticsLocal**](ps-remoteaccessconnectionstatisticslocal.md)
</dt> <dd>

Refers to the real-time connection stats and stats based on accounting data.

</dd> <dt>

[**PS\_RemoteAccessHealthLocal**](ps-remoteaccesshealthlocal.md)
</dt> <dd>

Server Provider for getting health of Remote Access cluster, server (DA and VPN) and its components.

</dd> <dt>

[**PS\_RemoteAccessInboxAccountingLocal**](ps-remoteaccessinboxaccountinglocal.md)
</dt> <dd>

Represents the various types of accounting supported for Remote Access.

</dd> <dt>

[**PS\_RemoteAccessLocal**](ps-remoteaccesslocal.md)
</dt> <dd>

Represents the server provider functions.

</dd> <dt>

[**PS\_RemoteAccessUserActivityLocal**](ps-remoteaccessuseractivitylocal.md)
</dt> <dd>

Refers to the resources in corpnet accessed by a user over a remote access connection.

</dd> <dt>

[**PS\_VpnUserLocal**](ps-vpnuserlocal.md)
</dt> <dd>

Refers to the connection stats summary based on accounting data.

</dd> <dt>

[**RemoteAccessAccountingConnectionLocal**](remoteaccessaccountingconnectionlocal.md)
</dt> <dd>

Remote Access Accounting Connection Record.

</dd> <dt>

[**RemoteAccessAccountingConnectionSummaryLocal**](remoteaccessaccountingconnectionsummarylocal.md)
</dt> <dd>

Remote Access Connection Statistics Summary Record From Accounting Data.

</dd> <dt>

[**RemoteAccessConfigurationVersion**](remoteaccessconfigurationversion.md)
</dt> <dd>

Defines the object containing the server GPO versioning information.

</dd> <dt>

[**RemoteAccessConnectionLocal**](remoteaccessconnectionlocal.md)
</dt> <dd>

Remote Access Common Connection Info Record.

</dd> <dt>

[**RemoteAccessConnectionSummaryLocal**](remoteaccessconnectionsummarylocal.md)
</dt> <dd>

Contains the remote access connection statistics summary record.

</dd> <dt>

[**RemoteAccessError**](remoteaccesserror-raserverpsprovider.md)
</dt> <dd>

An extended error object that contains the error details of the RemoteAccess cmdlets. It inherits from the **MSFT\_WmiError** and extends it by adding the following fields to the error object

</dd> <dt>

[**RemoteAccessInboxAccountingLocal**](remoteaccessinboxaccountinglocal.md)
</dt> <dd>

Remote Access Inbox Accounting Configuration Record.

</dd> <dt>

[**RemoteAccessMonitoringConnectionLocal**](remoteaccessmonitoringconnectionlocal.md)
</dt> <dd>

Remote Access Monitoring Connection Record.

</dd> <dt>

[**RemoteAccessServerHealthLocal**](remoteaccessserverhealthlocal.md)
</dt> <dd>

Remote Access Server Health object.

</dd> <dt>

[**RemoteAccessUserActivityLocal**](remoteaccessuseractivitylocal.md)
</dt> <dd>

Remote Access User Activity object.

</dd> </dl>
