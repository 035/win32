# COM

## In this section



| Topic                                                                              | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Component Object Model (COM)](https://msdn.microsoft.com/library/windows/desktop/ms680573)<br/> | COM is a platform-independent, distributed, object-oriented system for creating binary software components that can interact. COM is the foundation technology for Microsoft's OLE (compound documents) and ActiveX (Internet-enabled components) technologies.<br/>                                                                                                                                                                                                              |
| [Automation](75622ac1-a458-4e74-9e7f-371f8c7dbe19)<br/>                      | Automation enables software packages to expose their unique features to scripting tools and other applications. Automation uses the Component Object Model (COM), but may be implemented independently from other OLE features, such as in-place activation.<br/>                                                                                                                                                                                                                 |
| [Microsoft Interface Definition Language (MIDL)](https://msdn.microsoft.com/library/windows/desktop/aa367091)<br/>  | The Microsoft Interface Definition Language (MIDL) defines interfaces between client and server programs. Microsoft includes the MIDL compiler with the Platform Software Development Kit (SDK) to enable developers to create the interface definition language (IDL) files and application configuration files (ACF) required for remote procedure call (RPC) interfaces and COM/DCOM interfaces. MIDL also supports the generation of type libraries for OLE Automation. <br/> |
| [Structured Storage](https://msdn.microsoft.com/library/windows/desktop/aa380369)<br/>                 | Structured Storage provides file and data persistence in COM by handling a single file as a structured collection of objects known as storages and streams.<br/>                                                                                                                                                                                                                                                                                                                  |
