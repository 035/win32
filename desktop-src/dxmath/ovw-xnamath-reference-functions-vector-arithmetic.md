# Vector Arithmetic Functions

Lists the vector arithmetic functions.

## In this section



| Topic                                                                                   | Description                                                                                                               |
|-----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorAbs**](xmvectorabs.md)<br/>                                           | Computes the absolute value of each component of an [**XMVECTOR**](xmvector-data-type.md).<br/>                    |
| [**XMVectorAdd**](xmvectoradd.md)<br/>                                           | Computes the sum of two vectors.<br/>                                                                               |
| [**XMVectorAddAngles**](xmvectoraddangles.md)<br/>                               | Adds two vectors representing angles.<br/>                                                                          |
| [**XMVectorCeiling**](xmvectorceiling.md)<br/>                                   | Computes the ceiling of each component of an [**XMVECTOR**](xmvector-data-type.md).<br/>                           |
| [**XMVectorClamp**](xmvectorclamp.md)<br/>                                       | Clamps the components of a vector to a specified minimum and maximum range.<br/>                                    |
| [**XMVectorDivide**](xmvectordivide.md)<br/>                                     | Divides one instance of `XMVECTOR` by a second instance, returning the result in a third instance.<br/>             |
| [**XMVectorFloor**](xmvectorfloor.md)<br/>                                       | Computes the floor of each component of an [**XMVECTOR**](xmvector-data-type.md).<br/>                             |
| [**XMVectorIsInfinite**](xmvectorisinfinite.md)<br/>                             | Performs a per-component test for +/- infinity on a vector.<br/>                                                    |
| [**XMVectorIsNaN**](xmvectorisnan.md)<br/>                                       | Performs a per-component NaN test on a vector.<br/>                                                                 |
| [**XMVectorMax**](xmvectormax.md)<br/>                                           | Makes a per-component comparison between two vectors, and returns a vector containing the largest components.<br/>  |
| [**XMVectorMin**](xmvectormin.md)<br/>                                           | Makes a per-component comparison between two vectors, and returns a vector containing the smallest components.<br/> |
| [**XMVectorMod**](xmvectormod.md)<br/>                                           | Computes the per-component floating-point remainder of the quotient of two vectors.<br/>                            |
| [**XMVectorModAngles**](xmvectormodangles.md)<br/>                               | Computes the per-component angle modulo 2PI.<br/>                                                                   |
| [**XMVectorMultiply**](xmvectormultiply.md)<br/>                                 | Computes the per-component product of two vectors.<br/>                                                             |
| [**XMVectorMultiplyAdd**](xmvectormultiplyadd.md)<br/>                           | Computes the product of the first two vectors added to the third vector.<br/>                                       |
| [**XMVectorNegate**](xmvectornegate.md)<br/>                                     | Computes the negation of a vector.<br/>                                                                             |
| [**XMVectorNegativeMultiplySubtract**](xmvectornegativemultiplysubtract.md)<br/> | Computes the difference of a third vector and the product of the first two vectors.<br/>                            |
| [**XMVectorPow**](xmvectorpow.md)<br/>                                           | Computes *V1* raised to the power of *V2*.<br/>                                                                     |
| [**XMVectorReciprocal**](xmvectorreciprocal.md)<br/>                             | Computes the per-component reciprocal of a vector.<br/>                                                             |
| [**XMVectorReciprocalEst**](xmvectorreciprocalest.md)<br/>                       | Estimates the per-component reciprocal of a vector.<br/>                                                            |
| [**XMVectorReciprocalSqrt**](xmvectorreciprocalsqrt.md)<br/>                     | Computes the per-component reciprocal square root of a vector.<br/>                                                 |
| [**XMVectorReciprocalSqrtEst**](xmvectorreciprocalsqrtest.md)<br/>               | Estimates the per-component reciprocal square root of a vector.<br/>                                                |
| [**XMVectorRound**](xmvectorround.md)<br/>                                       | Rounds each component of a vector to the nearest integer.<br/>                                                      |
| [**XMVectorSaturate**](xmvectorsaturate.md)<br/>                                 | Saturates each component of a vector to the range 0.0f to 1.0f.<br/>                                                |
| [**XMVectorScale**](xmvectorscale.md)<br/>                                       | Scalar multiplies a vector by a floating-point value.<br/>                                                          |
| [**XMVectorSqrt**](xmvectorsqrt.md)<br/>                                         | Computes the per-component square root of a vector.<br/>                                                            |
| [**XMVectorSqrtEst**](xmvectorsqrtest.md)<br/>                                   | Estimates the per-component square root of a vector.<br/>                                                           |
| [**XMVectorSubtract**](xmvectorsubtract.md)<br/>                                 | Computes the difference of two vectors.<br/>                                                                        |
| [**XMVectorSubtractAngles**](xmvectorsubtractangles.md)<br/>                     | Subtracts two vectors representing angles.<br/>                                                                     |
| [**XMVectorSum**](xmvectorsum.md)<br/>                                           | Computes the horizontal sum of the components of an [**XMVECTOR**](xmvector-data-type.md).<br/>                    |
| [**XMVectorTruncate**](xmvectortruncate.md)<br/>                                 | Rounds each component of a vector to the nearest integer value in the direction of zero.<br/>                       |
