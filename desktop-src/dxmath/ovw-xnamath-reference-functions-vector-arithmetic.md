# Vector Arithmetic Functions

Lists the vector arithmetic functions.

## In this section



| Topic                                                                                   | Description                                                                                                               |
|-----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorAbs**](https://www.bing.com/search?q=**XMVectorAbs**)<br/>                                           | Computes the absolute value of each component of an [**XMVECTOR**](xmvector-data-type.md).<br/>                    |
| [**XMVectorAdd**](https://www.bing.com/search?q=**XMVectorAdd**)<br/>                                           | Computes the sum of two vectors.<br/>                                                                               |
| [**XMVectorAddAngles**](https://www.bing.com/search?q=**XMVectorAddAngles**)<br/>                               | Adds two vectors representing angles.<br/>                                                                          |
| [**XMVectorCeiling**](https://www.bing.com/search?q=**XMVectorCeiling**)<br/>                                   | Computes the ceiling of each component of an [**XMVECTOR**](xmvector-data-type.md).<br/>                           |
| [**XMVectorClamp**](https://www.bing.com/search?q=**XMVectorClamp**)<br/>                                       | Clamps the components of a vector to a specified minimum and maximum range.<br/>                                    |
| [**XMVectorDivide**](https://www.bing.com/search?q=**XMVectorDivide**)<br/>                                     | Divides one instance of `XMVECTOR` by a second instance, returning the result in a third instance.<br/>             |
| [**XMVectorFloor**](https://www.bing.com/search?q=**XMVectorFloor**)<br/>                                       | Computes the floor of each component of an [**XMVECTOR**](xmvector-data-type.md).<br/>                             |
| [**XMVectorIsInfinite**](https://www.bing.com/search?q=**XMVectorIsInfinite**)<br/>                             | Performs a per-component test for +/- infinity on a vector.<br/>                                                    |
| [**XMVectorIsNaN**](https://www.bing.com/search?q=**XMVectorIsNaN**)<br/>                                       | Performs a per-component NaN test on a vector.<br/>                                                                 |
| [**XMVectorMax**](https://www.bing.com/search?q=**XMVectorMax**)<br/>                                           | Makes a per-component comparison between two vectors, and returns a vector containing the largest components.<br/>  |
| [**XMVectorMin**](https://www.bing.com/search?q=**XMVectorMin**)<br/>                                           | Makes a per-component comparison between two vectors, and returns a vector containing the smallest components.<br/> |
| [**XMVectorMod**](https://www.bing.com/search?q=**XMVectorMod**)<br/>                                           | Computes the per-component floating-point remainder of the quotient of two vectors.<br/>                            |
| [**XMVectorModAngles**](https://www.bing.com/search?q=**XMVectorModAngles**)<br/>                               | Computes the per-component angle modulo 2PI.<br/>                                                                   |
| [**XMVectorMultiply**](https://www.bing.com/search?q=**XMVectorMultiply**)<br/>                                 | Computes the per-component product of two vectors.<br/>                                                             |
| [**XMVectorMultiplyAdd**](https://www.bing.com/search?q=**XMVectorMultiplyAdd**)<br/>                           | Computes the product of the first two vectors added to the third vector.<br/>                                       |
| [**XMVectorNegate**](https://www.bing.com/search?q=**XMVectorNegate**)<br/>                                     | Computes the negation of a vector.<br/>                                                                             |
| [**XMVectorNegativeMultiplySubtract**](https://www.bing.com/search?q=**XMVectorNegativeMultiplySubtract**)<br/> | Computes the difference of a third vector and the product of the first two vectors.<br/>                            |
| [**XMVectorPow**](https://www.bing.com/search?q=**XMVectorPow**)<br/>                                           | Computes *V1* raised to the power of *V2*.<br/>                                                                     |
| [**XMVectorReciprocal**](https://www.bing.com/search?q=**XMVectorReciprocal**)<br/>                             | Computes the per-component reciprocal of a vector.<br/>                                                             |
| [**XMVectorReciprocalEst**](https://www.bing.com/search?q=**XMVectorReciprocalEst**)<br/>                       | Estimates the per-component reciprocal of a vector.<br/>                                                            |
| [**XMVectorReciprocalSqrt**](https://www.bing.com/search?q=**XMVectorReciprocalSqrt**)<br/>                     | Computes the per-component reciprocal square root of a vector.<br/>                                                 |
| [**XMVectorReciprocalSqrtEst**](https://www.bing.com/search?q=**XMVectorReciprocalSqrtEst**)<br/>               | Estimates the per-component reciprocal square root of a vector.<br/>                                                |
| [**XMVectorRound**](https://www.bing.com/search?q=**XMVectorRound**)<br/>                                       | Rounds each component of a vector to the nearest integer.<br/>                                                      |
| [**XMVectorSaturate**](https://www.bing.com/search?q=**XMVectorSaturate**)<br/>                                 | Saturates each component of a vector to the range 0.0f to 1.0f.<br/>                                                |
| [**XMVectorScale**](https://www.bing.com/search?q=**XMVectorScale**)<br/>                                       | Scalar multiplies a vector by a floating-point value.<br/>                                                          |
| [**XMVectorSqrt**](https://www.bing.com/search?q=**XMVectorSqrt**)<br/>                                         | Computes the per-component square root of a vector.<br/>                                                            |
| [**XMVectorSqrtEst**](https://www.bing.com/search?q=**XMVectorSqrtEst**)<br/>                                   | Estimates the per-component square root of a vector.<br/>                                                           |
| [**XMVectorSubtract**](https://www.bing.com/search?q=**XMVectorSubtract**)<br/>                                 | Computes the difference of two vectors.<br/>                                                                        |
| [**XMVectorSubtractAngles**](https://www.bing.com/search?q=**XMVectorSubtractAngles**)<br/>                     | Subtracts two vectors representing angles.<br/>                                                                     |
| [**XMVectorSum**](https://www.bing.com/search?q=**XMVectorSum**)<br/>                                           | Computes the horizontal sum of the components of an [**XMVECTOR**](xmvector-data-type.md).<br/>                    |
| [**XMVectorTruncate**](https://www.bing.com/search?q=**XMVectorTruncate**)<br/>                                 | Rounds each component of a vector to the nearest integer value in the direction of zero.<br/>                       |
