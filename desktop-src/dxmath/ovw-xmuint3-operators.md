# XMUINT3 Operators

The [**XMUINT3**](/windows/win32/DirectXMath/?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                              | Description                                                                                                        |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmuint3-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMUINT3` to the current instance of `XMUINT3`. <br/> |
