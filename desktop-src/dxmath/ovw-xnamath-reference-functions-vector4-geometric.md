# DirectXMath Library 4D Vector Geometric Functions

Lists the 4D vector geometric functions.

## In this section



| Topic                                                                                 | Description                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**XMVector4AngleBetweenNormals**](xmvector4anglebetweennormals.md)<br/>       | Compute the radian angle between two normalized 4D vectors.<br/>        |
| [**XMVector4AngleBetweenNormalsEst**](xmvector4anglebetweennormalsest.md)<br/> | Estimates the radian angle between two normalized 4D vectors.<br/>      |
| [**XMVector4AngleBetweenVectors**](xmvector4anglebetweenvectors.md)<br/>       | Compute the radian angle between two 4D vectors.<br/>                   |
| [**XMVector4ClampLength**](xmvector4clamplength.md)<br/>                       | Clamps the length of a 4D vector to a given range.<br/>                 |
| [**XMVector4ClampLengthV**](xmvector4clamplengthv.md)<br/>                     | Clamps the length of a 4D vector to a given range.<br/>                 |
| [**XMVector4Cross**](xmvector4cross.md)<br/>                                   | Computes the 4D cross product.<br/>                                     |
| [**XMVector4Dot**](xmvector4dot.md)<br/>                                       | Computes the dot product between 4D vectors.<br/>                       |
| [**XMVector4InBounds**](xmvector4inbounds.md)<br/>                             | Tests whether the components of a 4D vector are within set bounds.<br/> |
| [**XMVector4Length**](xmvector4length.md)<br/>                                 | Computes the length of a 4D vector.<br/>                                |
| [**XMVector4LengthEst**](xmvector4lengthest.md)<br/>                           | Estimates the length of a 4D vector.<br/>                               |
| [**XMVector4LengthSq**](xmvector4lengthsq.md)<br/>                             | Computes the square of the length of a 4D vector.<br/>                  |
| [**XMVector4Normalize**](xmvector4normalize.md)<br/>                           | Returns the normalized version of a 4D vector.<br/>                     |
| [**XMVector4NormalizeEst**](xmvector4normalizeest.md)<br/>                     | Estimates the normalized version of a 4D vector.<br/>                   |
| [**XMVector4Orthogonal**](xmvector4orthogonal.md)<br/>                         | Computes a vector perpendicular to a 4D vector.<br/>                    |
| [**XMVector4ReciprocalLength**](xmvector4reciprocallength.md)<br/>             | Computes the reciprocal of the length of a 4D vector.<br/>              |
| [**XMVector4ReciprocalLengthEst**](xmvector4reciprocallengthest.md)<br/>       | Estimates the reciprocal of the length of a 4D vector.<br/>             |
| [**XMVector4Reflect**](xmvector4reflect.md)<br/>                               | Reflects an incident 4D vector across a 4D normal vector.<br/>          |
| [**XMVector4Refract**](xmvector4refract.md)<br/>                               | Refracts an incident 4D vector across a 4D normal vector.<br/>          |
| [**XMVector4RefractV**](xmvector4refractv.md)<br/>                             | Refracts an incident 4D vector across a 4D normal vector.<br/>          |
