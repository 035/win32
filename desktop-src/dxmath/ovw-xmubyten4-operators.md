# XMUBYTEN4 Operators

The [**XMUBYTEN4**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmubyten4?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                | Description                                                                                                            |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmubyten4-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMUBYTEN4` to the current instance of `XMUBYTEN4`. <br/> |
