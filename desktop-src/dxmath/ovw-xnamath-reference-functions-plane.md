# DirectXMath Library Plane Functions

Lists the plane functions provided by DirectXMath.

These functions use an XMVECTOR 4-vector to represent the coefficients of the plane equation, Ax+By+Cz+D = 0, where the X-component is A, the Y-component is B, the Z-component is C, and the W-component is D.

## In this section



| Topic                                                               | Description                                                                                                      |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**XMPlaneDot**](xmplanedot.md)<br/>                         | Calculates the dot product between an input plane and a 4D vector.<br/>                                    |
| [**XMPlaneDotCoord**](xmplanedotcoord.md)<br/>               | Calculates the dot product between an input plane and a 3D vector.<br/>                                    |
| [**XMPlaneDotNormal**](xmplanedotnormal.md)<br/>             | Calculates the dot product between the normal vector of a plane and a 3D vector.<br/>                      |
| [**XMPlaneEqual**](xmplaneequal.md)<br/>                     | Determines if two planes are equal.<br/>                                                                   |
| [**XMPlaneFromPointNormal**](xmplanefrompointnormal.md)<br/> | Computes the equation of a plane constructed from a point in the plane and a normal vector.<br/>           |
| [**XMPlaneFromPoints**](xmplanefrompoints.md)<br/>           | Computes the equation of a plane constructed from three points in the plane.<br/>                          |
| [**XMPlaneIntersectLine**](xmplaneintersectline.md)<br/>     | Finds the intersection between a plane and a line.<br/>                                                    |
| [**XMPlaneIntersectPlane**](xmplaneintersectplane.md)<br/>   | Finds the intersection of two planes.<br/>                                                                 |
| [**XMPlaneIsInfinite**](xmplaneisinfinite.md)<br/>           | Tests whether any of the coefficients of a plane is positive or negative infinity.<br/>                    |
| [**XMPlaneIsNaN**](xmplaneisnan.md)<br/>                     | Tests whether any of the coefficients of a plane is a NaN.<br/>                                            |
| [**XMPlaneNearEqual**](xmplanenearequal.md)<br/>             | Determines whether two planes are nearly equal.<br/>                                                       |
| [**XMPlaneNormalize**](xmplanenormalize.md)<br/>             | Normalizes the coefficients of a plane so that coefficients of x, y, and z form a unit normal vector.<br/> |
| [**XMPlaneNormalizeEst**](xmplanenormalizeest.md)<br/>       | Estimates the coefficients of a plane so that coefficients of x, y, and z form a unit normal vector.<br/>  |
| [**XMPlaneNotEqual**](xmplanenotequal.md)<br/>               | Determines if two planes are unequal.<br/>                                                                 |
| [**XMPlaneTransform**](xmplanetransform.md)<br/>             | Transforms a plane by a given matrix.<br/>                                                                 |
| [**XMPlaneTransformStream**](xmplanetransformstream.md)<br/> | Transforms a stream of planes by a given matrix.<br/>                                                      |
