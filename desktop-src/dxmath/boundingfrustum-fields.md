# BoundingFrustum Fields



| Field                                                            | Description                                                                                                  |
|------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**BottomSlope**](boundingfrustum-bottomslope.md)<br/>    | The slope of the bottom of the [**BoundingFrustum**](/windows/win32/DirectXCollision/ns-directxcollision-boundingfrustum?branch=master).<br/>                        |
| [**CORNER\_COUNT**](boundingfrustum-corner-count.md)<br/> | The number of corners defining the [**BoundingFrustum**](/windows/win32/DirectXCollision/ns-directxcollision-boundingfrustum?branch=master).<br/>                    |
| [**Far**](boundingfrustum-far.md)<br/>                    | The distance of the far plane from the origin of the [**BoundingFrustum**](/windows/win32/DirectXCollision/ns-directxcollision-boundingfrustum?branch=master).<br/>  |
| [**LeftSlope**](boundingfrustum-leftslope.md)<br/>        | The slope of the left side of the [**BoundingFrustum**](/windows/win32/DirectXCollision/ns-directxcollision-boundingfrustum?branch=master).<br/>                     |
| [**Near**](boundingfrustum-near.md)<br/>                  | The distance of the near plane of the [**BoundingFrustum**](/windows/win32/DirectXCollision/ns-directxcollision-boundingfrustum?branch=master) from its origin.<br/> |
| [**Orientation**](boundingfrustum-orientation.md)<br/>    | The orientation of the [**BoundingFrustum**](/windows/win32/DirectXCollision/ns-directxcollision-boundingfrustum?branch=master).<br/>                                |
| [**Origin**](boundingfrustum-origin.md)<br/>              | The origin of the [**BoundingFrustum**](/windows/win32/DirectXCollision/ns-directxcollision-boundingfrustum?branch=master).<br/>                                     |
| [**RightSlope**](boundingfrustum-rightslope.md)<br/>      | The slope of the right side of the [**BoundingFrustum**](/windows/win32/DirectXCollision/ns-directxcollision-boundingfrustum?branch=master).<br/>                    |
| [**TopSlope**](boundingfrustum-topslope.md)<br/>          | The slope of the top of the [**BoundingFrustum**](/windows/win32/DirectXCollision/ns-directxcollision-boundingfrustum?branch=master).<br/>                           |
