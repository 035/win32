# BoundingSphere Methods



| Method                                                                           | Description                                                                                                  |
|----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**Contains**](boundingsphere-contains.md)<br/>                           | Tests whether the BoundingSphere contains a specified object.<br/>                                     |
| [**Intersects**](boundingsphere-intersects.md)<br/>                       | Tests the BoundingSphere for intersection with an object.<br/>                                         |
| [**Transform**](boundingsphere-transform.md)<br/>                         | Transforms the BoundingSphere.<br/>                                                                    |
| [**ContainedBy**](boundingsphere-containedby.md)<br/>                     | Tests whether the [**BoundingSphere**](boundingsphere.md) is contained by the specified frustum.<br/> |
| [**CreateFromBoundingBox**](boundingsphere-createfromboundingbox.md)<br/> | Creates a BoundingSphere containing the specified BoundingBox.<br/>                                    |
| [**CreateFromPoints**](boundingsphere-createfrompoints.md)<br/>           | Creates a new BoundingSphere from a list of points.<br/>                                               |
| [**CreateMerged**](boundingsphere-createmerged.md)<br/>                   | Creates a BoundingSphere that contains the two specified BoundingSphere objects.<br/>                  |
| [**op\_Assignment**](boundingsphere-op-assignment.md)<br/>                | Initializes the BoundingSphere with values from a specified BoundingSphere.<br/>                       |
