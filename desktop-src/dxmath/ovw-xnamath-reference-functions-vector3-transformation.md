# DirectXMath Library 3D Vector Transformation Functions

Lists the 3D vector transformation functions.

## In this section



| Topic                                                                               | Description                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3InverseRotate**](https://www.bing.com/search?q=**XMVector3InverseRotate**)<br/>                 | Rotates a 3D vector using the inverse of a quaternion.<br/>                                                                                |
| [**XMVector3Project**](https://www.bing.com/search?q=**XMVector3Project**)<br/>                             | Project a 3D vector from object space into screen space.<br/>                                                                              |
| [**XMVector3ProjectStream**](https://www.bing.com/search?q=**XMVector3ProjectStream**)<br/>                 | Projects a stream of 3D vectors from object space into screen space.<br/>                                                                  |
| [**XMVector3Rotate**](https://www.bing.com/search?q=**XMVector3Rotate**)<br/>                               | Rotates a 3D vector using a quaternion.<br/>                                                                                               |
| [**XMVector3Transform**](https://www.bing.com/search?q=**XMVector3Transform**)<br/>                         | Transforms a 3D vector by a matrix.<br/>                                                                                                   |
| [**XMVector3TransformCoord**](https://www.bing.com/search?q=**XMVector3TransformCoord**)<br/>               | Transforms a 3D vector by a given matrix, projecting the result back into w = 1.<br/>                                                      |
| [**XMVector3TransformCoordStream**](https://www.bing.com/search?q=**XMVector3TransformCoordStream**)<br/>   | Transforms a stream of 3D vectors by a given matrix, projecting the resulting vectors such that their w coordinates are equal to 1.0.<br/> |
| [**XMVector3TransformNormal**](https://www.bing.com/search?q=**XMVector3TransformNormal**)<br/>             | Transforms the 3D vector normal by the given matrix.<br/>                                                                                  |
| [**XMVector3TransformNormalStream**](https://www.bing.com/search?q=**XMVector3TransformNormalStream**)<br/> | Transforms a stream of 3D normal vectors by a given matrix.<br/>                                                                           |
| [**XMVector3TransformStream**](https://www.bing.com/search?q=**XMVector3TransformStream**)<br/>             | Transforms a stream of 3D vectors by a given matrix.<br/>                                                                                  |
| [**XMVector3Unproject**](https://www.bing.com/search?q=**XMVector3Unproject**)<br/>                         | Projects a 3D vector from screen space into object space.<br/>                                                                             |
| [**XMVector3UnprojectStream**](https://www.bing.com/search?q=**XMVector3UnprojectStream**)<br/>             | Transforms a stream of 3D vectors from screen space to object space.<br/>                                                                  |
