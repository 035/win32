# DirectXMath Library 3D Vector Transformation Functions

Lists the 3D vector transformation functions.

## In this section



| Topic                                                                               | Description                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3InverseRotate**](xmvector3inverserotate.md)<br/>                 | Rotates a 3D vector using the inverse of a quaternion.<br/>                                                                                |
| [**XMVector3Project**](xmvector3project.md)<br/>                             | Project a 3D vector from object space into screen space.<br/>                                                                              |
| [**XMVector3ProjectStream**](xmvector3projectstream.md)<br/>                 | Projects a stream of 3D vectors from object space into screen space.<br/>                                                                  |
| [**XMVector3Rotate**](xmvector3rotate.md)<br/>                               | Rotates a 3D vector using a quaternion.<br/>                                                                                               |
| [**XMVector3Transform**](xmvector3transform.md)<br/>                         | Transforms a 3D vector by a matrix.<br/>                                                                                                   |
| [**XMVector3TransformCoord**](xmvector3transformcoord.md)<br/>               | Transforms a 3D vector by a given matrix, projecting the result back into w = 1.<br/>                                                      |
| [**XMVector3TransformCoordStream**](xmvector3transformcoordstream.md)<br/>   | Transforms a stream of 3D vectors by a given matrix, projecting the resulting vectors such that their w coordinates are equal to 1.0.<br/> |
| [**XMVector3TransformNormal**](xmvector3transformnormal.md)<br/>             | Transforms the 3D vector normal by the given matrix.<br/>                                                                                  |
| [**XMVector3TransformNormalStream**](xmvector3transformnormalstream.md)<br/> | Transforms a stream of 3D normal vectors by a given matrix.<br/>                                                                           |
| [**XMVector3TransformStream**](xmvector3transformstream.md)<br/>             | Transforms a stream of 3D vectors by a given matrix.<br/>                                                                                  |
| [**XMVector3Unproject**](xmvector3unproject.md)<br/>                         | Projects a 3D vector from screen space into object space.<br/>                                                                             |
| [**XMVector3UnprojectStream**](xmvector3unprojectstream.md)<br/>             | Transforms a stream of 3D vectors from screen space to object space.<br/>                                                                  |
