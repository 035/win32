# Geometric Vector Functions

Lists the geometric vector functions.

## In this section



| Topic                                                           | Description                                                                                                                                                                                                      |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorBaryCentric**](https://www.bing.com/search?q=**XMVectorBaryCentric**)<br/>   | Returns a point in Barycentric coordinates, using the specified position vectors.<br/>                                                                                                                     |
| [**XMVectorBaryCentricV**](https://www.bing.com/search?q=**XMVectorBaryCentricV**)<br/> | Returns a point in Barycentric coordinates, using the specified position vectors.<br/>                                                                                                                     |
| [**XMVectorCatmullRom**](https://www.bing.com/search?q=**XMVectorCatmullRom**)<br/>     | Performs a Catmull-Rom interpolation, using the specified position vectors.<br/>                                                                                                                           |
| [**XMVectorCatmullRomV**](https://www.bing.com/search?q=**XMVectorCatmullRomV**)<br/>   | Performs a Catmull-Rom interpolation, using the specified position vectors.<br/>                                                                                                                           |
| [**XMVectorHermite**](https://www.bing.com/search?q=**XMVectorHermite**)<br/>           | Performs a Hermite spline interpolation, using the specified vectors.<br/>                                                                                                                                 |
| [**XMVectorHermiteV**](https://www.bing.com/search?q=**XMVectorHermiteV**)<br/>         | Performs a Hermite spline interpolation, using the specified vectors.<br/>                                                                                                                                 |
| [**XMVectorInBounds**](https://www.bing.com/search?q=**XMVectorInBounds**)<br/>         | Tests whether the components of a given vector are within set bounds.<br/>                                                                                                                                 |
| [**XMVectorInBoundsR**](https://www.bing.com/search?q=**XMVectorInBoundsR**)<br/>       | Tests whether the components of a given vector are within certain bounds and sets a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVectorLerp**](https://www.bing.com/search?q=**XMVectorLerp**)<br/>                 | Performs a linear interpolation between two vectors.<br/>                                                                                                                                                  |
| [**XMVectorLerpV**](https://www.bing.com/search?q=**XMVectorLerpV**)<br/>               | Performs a linear interpolation between two vectors.<br/>                                                                                                                                                  |
