# Geometric Vector Functions

Lists the geometric vector functions.

## In this section



| Topic                                                           | Description                                                                                                                                                                                                      |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorBaryCentric**](xmvectorbarycentric.md)<br/>   | Returns a point in Barycentric coordinates, using the specified position vectors.<br/>                                                                                                                     |
| [**XMVectorBaryCentricV**](xmvectorbarycentricv.md)<br/> | Returns a point in Barycentric coordinates, using the specified position vectors.<br/>                                                                                                                     |
| [**XMVectorCatmullRom**](xmvectorcatmullrom.md)<br/>     | Performs a Catmull-Rom interpolation, using the specified position vectors.<br/>                                                                                                                           |
| [**XMVectorCatmullRomV**](xmvectorcatmullromv.md)<br/>   | Performs a Catmull-Rom interpolation, using the specified position vectors.<br/>                                                                                                                           |
| [**XMVectorHermite**](xmvectorhermite.md)<br/>           | Performs a Hermite spline interpolation, using the specified vectors.<br/>                                                                                                                                 |
| [**XMVectorHermiteV**](xmvectorhermitev.md)<br/>         | Performs a Hermite spline interpolation, using the specified vectors.<br/>                                                                                                                                 |
| [**XMVectorInBounds**](xmvectorinbounds.md)<br/>         | Tests whether the components of a given vector are within set bounds.<br/>                                                                                                                                 |
| [**XMVectorInBoundsR**](xmvectorinboundsr.md)<br/>       | Tests whether the components of a given vector are within certain bounds and sets a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/win32/DirectXMath/nf-directxmath-xmcomparisonalltrue?branch=master).<br/> |
| [**XMVectorLerp**](xmvectorlerp.md)<br/>                 | Performs a linear interpolation between two vectors.<br/>                                                                                                                                                  |
| [**XMVectorLerpV**](xmvectorlerpv.md)<br/>               | Performs a linear interpolation between two vectors.<br/>                                                                                                                                                  |
