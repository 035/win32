# Component-Wise Vector Functions

Lists the component-wise vector functions.

## In this section



| Topic                                                             | Description                                                                                                                              |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorInsert**](xmvectorinsert.md)<br/>               | Rotates a vector left by a given number of 32-bit components and insert selected elements of that result into another vector.<br/> |
| [**XMVectorMergeXY**](xmvectormergexy.md)<br/>             | Creates a new vector by combining the x and y-components of two vectors.<br/>                                                      |
| [**XMVectorMergeZW**](xmvectormergezw.md)<br/>             | Creates a new vector by combining the z and w-components of two vectors.<br/>                                                      |
| [**XMVectorPermute**](xmvectorpermute.md)<br/>             | Permutes the components of two vectors to create a new vector.<br/>                                                                |
| [**XMVectorRotateLeft**](xmvectorrotateleft.md)<br/>       | Rotates the vector left by a given number of 32-bit elements.<br/>                                                                 |
| [**XMVectorRotateRight**](xmvectorrotateright.md)<br/>     | Rotates the vector right by a given number of 32-bit elements.<br/>                                                                |
| [**XMVectorSelect**](xmvectorselect.md)<br/>               | Performs a per-component selection between two input vectors and returns the resulting vector.<br/>                                |
| [**XMVectorSelectControl**](xmvectorselectcontrol.md)<br/> | Defines a control vector for use in [**XMVectorSelect**](xmvectorselect.md).<br/>                                                 |
| [**XMVectorShiftLeft**](xmvectorshiftleft.md)<br/>         | Shifts a vector left by a given number of 32-bit elements, filling the vacated elements with elements from a second vector.<br/>   |
| [**XMVectorSplatW**](xmvectorsplatw.md)<br/>               | Replicates the w component of a vector to all of the components.<br/>                                                              |
| [**XMVectorSplatX**](xmvectorsplatx.md)<br/>               | Replicates the x component of a vector to all of the components.<br/>                                                              |
| [**XMVectorSplatY**](xmvectorsplaty.md)<br/>               | Replicates the y component of a vector to all of the components.<br/>                                                              |
| [**XMVectorSplatZ**](xmvectorsplatz.md)<br/>               | Replicates the z component of a vector to all of the components.<br/>                                                              |
| [**XMVectorSwizzle**](xmvectorswizzle.md)<br/>             | Swizzles a vector.<br/>                                                                                                            |
