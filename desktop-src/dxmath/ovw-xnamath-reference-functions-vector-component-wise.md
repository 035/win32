# Component-Wise Vector Functions

Lists the component-wise vector functions.

## In this section



| Topic                                                             | Description                                                                                                                              |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorInsert**](https://www.bing.com/search?q=**XMVectorInsert**)<br/>               | Rotates a vector left by a given number of 32-bit components and insert selected elements of that result into another vector.<br/> |
| [**XMVectorMergeXY**](https://www.bing.com/search?q=**XMVectorMergeXY**)<br/>             | Creates a new vector by combining the x and y-components of two vectors.<br/>                                                      |
| [**XMVectorMergeZW**](https://www.bing.com/search?q=**XMVectorMergeZW**)<br/>             | Creates a new vector by combining the z and w-components of two vectors.<br/>                                                      |
| [**XMVectorPermute**](https://www.bing.com/search?q=**XMVectorPermute**)<br/>             | Permutes the components of two vectors to create a new vector.<br/>                                                                |
| [**XMVectorRotateLeft**](https://www.bing.com/search?q=**XMVectorRotateLeft**)<br/>       | Rotates the vector left by a given number of 32-bit elements.<br/>                                                                 |
| [**XMVectorRotateRight**](https://www.bing.com/search?q=**XMVectorRotateRight**)<br/>     | Rotates the vector right by a given number of 32-bit elements.<br/>                                                                |
| [**XMVectorSelect**](https://www.bing.com/search?q=**XMVectorSelect**)<br/>               | Performs a per-component selection between two input vectors and returns the resulting vector.<br/>                                |
| [**XMVectorSelectControl**](https://www.bing.com/search?q=**XMVectorSelectControl**)<br/> | Defines a control vector for use in [**XMVectorSelect**](https://www.bing.com/search?q=**XMVectorSelect**).<br/>                                                 |
| [**XMVectorShiftLeft**](https://www.bing.com/search?q=**XMVectorShiftLeft**)<br/>         | Shifts a vector left by a given number of 32-bit elements, filling the vacated elements with elements from a second vector.<br/>   |
| [**XMVectorSplatW**](https://www.bing.com/search?q=**XMVectorSplatW**)<br/>               | Replicates the w component of a vector to all of the components.<br/>                                                              |
| [**XMVectorSplatX**](https://www.bing.com/search?q=**XMVectorSplatX**)<br/>               | Replicates the x component of a vector to all of the components.<br/>                                                              |
| [**XMVectorSplatY**](https://www.bing.com/search?q=**XMVectorSplatY**)<br/>               | Replicates the y component of a vector to all of the components.<br/>                                                              |
| [**XMVectorSplatZ**](https://www.bing.com/search?q=**XMVectorSplatZ**)<br/>               | Replicates the z component of a vector to all of the components.<br/>                                                              |
| [**XMVectorSwizzle**](https://www.bing.com/search?q=**XMVectorSwizzle**)<br/>             | Swizzles a vector.<br/>                                                                                                            |
