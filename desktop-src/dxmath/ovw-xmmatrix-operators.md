# XMMATRIX Operators

The [**XMMATRIX**](xmmatrix.md) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                    | Description                                                                                                                                                                                   |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**operator \***](xmmatrix-operator-mul.md)<br/>    | Performs a matrix multiplication of the current instance of `XMMATRIX` by another instance of `XMMATRIX`.<br/>                                                                          |
| [**operator \*=**](xmmatrix-operator-muleq.md)<br/> | Performs a matrix multiplication of the current instance of `XMMATRIX` by another instance of `XMMATRIX` and returns a reference to the current instance, which has been updated. <br/> |
| [**operator =**](xmmatrix-operator-eq.md)<br/>      | Assigns the matrix data of one instance of `XMMATRIX` to the current instance of `XMMATRIX` and returns a `reference` to the current instance. <br/>                                    |
| [**operator ()**](xmmatrix-operator-parens.md)<br/> | Accesses specific matrix elements referenced by row and column from the current instance of `XMMATRIX`. <br/>                                                                           |
