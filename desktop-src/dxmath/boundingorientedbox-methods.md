# BoundingOrientedBox Methods



| Method                                                                                | Description                                                                                                        |
|---------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**Contains**](boundingorientedbox-contains.md)<br/>                           | Tests whether the [**BoundingOrientedBox**](boundingorientedbox.md) contains another object.<br/>           |
| [**Intersects**](boundingorientedbox-intersects.md)<br/>                       | Tests the [**BoundingOrientedBox**](boundingorientedbox.md) for intersection with another object.<br/>      |
| [**Transform**](boundingorientedbox-transform.md)<br/>                         | Transforms the [**BoundingOrientedBox**](boundingorientedbox.md).<br/>                                      |
| [**ContainedBy**](boundingorientedbox-containedby.md)<br/>                     | Tests whether the [**BoundingOrientedBox**](boundingorientedbox.md) is contained by a frustum.<br/>         |
| [**CreateFromBoundingBox**](boundingorientedbox-createfromboundingbox.md)<br/> | Creates a [**BoundingOrientedBox**](boundingorientedbox.md) from a [**BoundingBox**](boundingbox.md).<br/> |
| [**CreateFromPoints**](boundingorientedbox-createfrompoints.md)<br/>           | Creates a [**BoundingOrientedBox**](boundingorientedbox.md) from a collection of points.<br/>               |
| [**GetCorners**](boundingorientedbox-getcorners.md)<br/>                       | Retrieves the corners of the [**BoundingOrientedBox**](boundingorientedbox.md).<br/>                        |
| [**op\_Assignment**](boundingorientedbox-op-assignment.md)<br/>                | Copies values from another [**BoundingOrientedBox**](boundingorientedbox.md).<br/>                          |
