# DirectXMath Library 2D Vector Comparison Functions

Lists the 2D vector comparison functions.

## In this section



| Topic                                                                   | Description                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Equal**](https://www.bing.com/search?q=**XMVector2Equal**)<br/>                     | Tests whether two 2D vectors are equal.<br/>                                                                                                                                                                                                        |
| [**XMVector2EqualInt**](https://www.bing.com/search?q=**XMVector2EqualInt**)<br/>               | Tests whether two 2D vectors are equal, treating each component as an unsigned integer.<br/>                                                                                                                                                        |
| [**XMVector2EqualIntR**](https://www.bing.com/search?q=**XMVector2EqualIntR**)<br/>             | Tests whether two 2D vectors are equal, treating each component as an unsigned integer. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector2EqualR**](https://www.bing.com/search?q=**XMVector2EqualR**)<br/>                   | Tests whether two 2D vectors are equal. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector2Greater**](https://www.bing.com/search?q=**XMVector2Greater**)<br/>                 | Tests whether one 2D vector is greater than another 2D vector.<br/>                                                                                                                                                                                 |
| [**XMVector2GreaterOrEqual**](https://www.bing.com/search?q=**XMVector2GreaterOrEqual**)<br/>   | Tests whether one 2D vector is greater-than-or-equal-to another 2D vector.<br/>                                                                                                                                                                     |
| [**XMVector2GreaterOrEqualR**](https://www.bing.com/search?q=**XMVector2GreaterOrEqualR**)<br/> | Tests whether one 2D vector is greater-than-or-equal-to another 2D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector2GreaterR**](https://www.bing.com/search?q=**XMVector2GreaterR**)<br/>               | Tests whether one 2D vector is greater than another 2D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector2IsInfinite**](https://www.bing.com/search?q=**XMVector2IsInfinite**)<br/>           | Tests whether any component of a 2D vector is positive or negative infinity.<br/>                                                                                                                                                                   |
| [**XMVector2IsNaN**](https://www.bing.com/search?q=**XMVector2IsNaN**)<br/>                     | Tests whether any component of a 2D vector is a NaN.<br/>                                                                                                                                                                                           |
| [**XMVector2Less**](https://www.bing.com/search?q=**XMVector2Less**)<br/>                       | Tests whether one 2D vector is less than another 2D vector.<br/>                                                                                                                                                                                    |
| [**XMVector2LessOrEqual**](https://www.bing.com/search?q=**XMVector2LessOrEqual**)<br/>         | Tests whether one 2D vector is less-than-or-equal-to another 2D vector.<br/>                                                                                                                                                                        |
| [**XMVector2NearEqual**](https://www.bing.com/search?q=**XMVector2NearEqual**)<br/>             | Tests whether one 2D vector is near another 2D vector.<br/>                                                                                                                                                                                         |
| [**XMVector2NotEqual**](https://www.bing.com/search?q=**XMVector2NotEqual**)<br/>               | Tests whether two 2D vectors are not equal.<br/>                                                                                                                                                                                                    |
| [**XMVector2NotEqualInt**](https://www.bing.com/search?q=**XMVector2NotEqualInt**)<br/>         | Test whether two vectors are not equal, treating each component as an unsigned integer.<br/>                                                                                                                                                        |
