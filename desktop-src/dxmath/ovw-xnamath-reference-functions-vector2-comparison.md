# DirectXMath Library 2D Vector Comparison Functions

Lists the 2D vector comparison functions.

## In this section



| Topic                                                                   | Description                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Equal**](xmvector2equal.md)<br/>                     | Tests whether two 2D vectors are equal.<br/>                                                                                                                                                                                                        |
| [**XMVector2EqualInt**](xmvector2equalint.md)<br/>               | Tests whether two 2D vectors are equal, treating each component as an unsigned integer.<br/>                                                                                                                                                        |
| [**XMVector2EqualIntR**](xmvector2equalintr.md)<br/>             | Tests whether two 2D vectors are equal, treating each component as an unsigned integer. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](xmcomparisonalltrue.md).<br/> |
| [**XMVector2EqualR**](xmvector2equalr.md)<br/>                   | Tests whether two 2D vectors are equal. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](xmcomparisonalltrue.md).<br/>                                                 |
| [**XMVector2Greater**](xmvector2greater.md)<br/>                 | Tests whether one 2D vector is greater than another 2D vector.<br/>                                                                                                                                                                                 |
| [**XMVector2GreaterOrEqual**](xmvector2greaterorequal.md)<br/>   | Tests whether one 2D vector is greater-than-or-equal-to another 2D vector.<br/>                                                                                                                                                                     |
| [**XMVector2GreaterOrEqualR**](xmvector2greaterorequalr.md)<br/> | Tests whether one 2D vector is greater-than-or-equal-to another 2D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](xmcomparisonalltrue.md).<br/>                                      |
| [**XMVector2GreaterR**](xmvector2greaterr.md)<br/>               | Tests whether one 2D vector is greater than another 2D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](xmcomparisonalltrue.md).<br/>                                                  |
| [**XMVector2IsInfinite**](xmvector2isinfinite.md)<br/>           | Tests whether any component of a 2D vector is positive or negative infinity.<br/>                                                                                                                                                                   |
| [**XMVector2IsNaN**](xmvector2isnan.md)<br/>                     | Tests whether any component of a 2D vector is a NaN.<br/>                                                                                                                                                                                           |
| [**XMVector2Less**](xmvector2less.md)<br/>                       | Tests whether one 2D vector is less than another 2D vector.<br/>                                                                                                                                                                                    |
| [**XMVector2LessOrEqual**](xmvector2lessorequal.md)<br/>         | Tests whether one 2D vector is less-than-or-equal-to another 2D vector.<br/>                                                                                                                                                                        |
| [**XMVector2NearEqual**](xmvector2nearequal.md)<br/>             | Tests whether one 2D vector is near another 2D vector.<br/>                                                                                                                                                                                         |
| [**XMVector2NotEqual**](xmvector2notequal.md)<br/>               | Tests whether two 2D vectors are not equal.<br/>                                                                                                                                                                                                    |
| [**XMVector2NotEqualInt**](xmvector2notequalint.md)<br/>         | Test whether two vectors are not equal, treating each component as an unsigned integer.<br/>                                                                                                                                                        |
