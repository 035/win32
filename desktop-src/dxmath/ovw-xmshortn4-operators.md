# XMSHORTN4 Operators

The [**XMSHORTN4**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmshortn4?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                | Description                                                                                                            |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmshortn4-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMSHORTN4` to the current instance of `XMSHORTN4`. <br/> |
