# DirectXMath Library Utility Functions

Lists the utility functions provided by DirectXMath.

## In this section



| Topic                                                                       | Description                                                                                                                |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**XMComparisonAllFalse**](xmcomparisonallfalse.md)<br/>             | Tests the comparison value to determine if all of the compared components are false.<br/>                            |
| [**XMComparisonAllInBounds**](xmcomparisonallinbounds.md)<br/>       | Tests the comparison value to determine if all of the compared components are within set bounds.<br/>                |
| [**XMComparisonAllTrue**](xmcomparisonalltrue.md)<br/>               | Tests the comparison value to determine if all of the compared components are true.<br/>                             |
| [**XMComparisonAnyFalse**](xmcomparisonanyfalse.md)<br/>             | Tests the comparison value to determine if any of the compared components are false.<br/>                            |
| [**XMComparisonAnyOutOfBounds**](xmcomparisonanyoutofbounds.md)<br/> | Tests the comparison value to determine if any of the compared components are outside the set bounds.<br/>           |
| [**XMComparisonAnyTrue**](xmcomparisonanytrue.md)<br/>               | Tests the comparison value to determine if any of the compared components are true.<br/>                             |
| [**XMComparisonMixed**](xmcomparisonmixed.md)<br/>                   | Tests the comparison value to determine if the compared components had mixed results--some true and some false.<br/> |
| [**XMFresnelTerm**](xmfresnelterm.md)<br/>                           | Calculates the Fresnel term for unpolarized light.<br/>                                                              |
| [**XMVerifyCPUSupport**](xmverifycpusupport.md)<br/>                 | Indicates if the DirectXMath Library supports the current platform.<br/>                                             |
