# XMHALF4 Operators

The [**XMHALF4**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmhalf4?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                              | Description                                                                                                        |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmhalf4-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMHALF4` to the current instance of `XMHALF4`. <br/> |
