|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmubyte4-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMUBYTE4` to the current instance of `XMUBYTE4`. <br/> |
