# XMVECTORU8 Operators

The following operators are exposed by the [**XMVECTORU8**](xmvectoru8-data-type.md) structure.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                              | Description                                                                                                                                                                                                                                                                                             |
|----------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**operator \_\_m128i**](/windows/win32/DirectXMath/?branch=master)<br/> | Returns the object implementing the current instance of [**XMVECTORU8**](xmvectoru8-data-type.md) cast to a [\_\_m128i](http://msdn.microsoft.com/en-us/library/26232t5c.aspx) instance of \[Windows systems supporting [SSE](http://msdn.microsoft.com/en-us/library/t467de55.aspx) Only\]<br/> |
| [**operator XMVECTOR**](/windows/win32/DirectXMath/?branch=master)<br/> | Returns the instance of `XMVECTOR` implementing the current instance of `XMVECTORU8`.<br/>                                                                                                                                                                                                        |
