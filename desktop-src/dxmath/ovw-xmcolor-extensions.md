# XMCOLOR Extensions

[**XMCOLOR**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmcolor?branch=master) describes a 32-bit Alpha Red Green Blue (ARGB) color vector, where each color channel is specified by 8 bits.

> [!Note]  
> The features listed here are only available when developing with C++.
