# XMUBYTE2 Operators

The [**XMUBYTE2**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmubyte2?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available with C++.

 

## In this section



| Methods                                               | Description                                                                                                         |
|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmubyte2-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMUBYTE2` to the current instance of `XMUBYTE2`.<br/> |
