# DirectXMath Library 3D Vector Comparison Functions

Lists the 3D vector comparison functions.

## In this section



| Topic                                                                   | Description                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3Equal**](https://www.bing.com/search?q=**XMVector3Equal**)<br/>                     | Tests whether two 3D vectors are equal.<br/>                                                                                                                                                                                                        |
| [**XMVector3EqualInt**](https://www.bing.com/search?q=**XMVector3EqualInt**)<br/>               | Tests whether two 3D vectors are equal, treating each component as an unsigned integer.<br/>                                                                                                                                                        |
| [**XMVector3EqualIntR**](https://www.bing.com/search?q=**XMVector3EqualIntR**)<br/>             | Tests whether two 3D vectors are equal, treating each component as an unsigned integer. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector3EqualR**](https://www.bing.com/search?q=**XMVector3EqualR**)<br/>                   | Tests whether two 3D vectors are equal. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector3Greater**](https://www.bing.com/search?q=**XMVector3Greater**)<br/>                 | Tests whether one 3D vector is greater than another 3D vector.<br/>                                                                                                                                                                                 |
| [**XMVector3GreaterOrEqual**](https://www.bing.com/search?q=**XMVector3GreaterOrEqual**)<br/>   | Tests whether one 3D vector is greater-than-or-equal-to another 3D vector.<br/>                                                                                                                                                                     |
| [**XMVector3GreaterOrEqualR**](https://www.bing.com/search?q=**XMVector3GreaterOrEqualR**)<br/> | Tests whether one 3D vector is greater-than-or-equal-to another 3D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector3GreaterR**](https://www.bing.com/search?q=**XMVector3GreaterR**)<br/>               | Tests whether one 3D vector is greater than another 3D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector3IsInfinite**](https://www.bing.com/search?q=**XMVector3IsInfinite**)<br/>           | Tests whether any component of a 3D vector is positive or negative infinity.<br/>                                                                                                                                                                   |
| [**XMVector3IsNaN**](https://www.bing.com/search?q=**XMVector3IsNaN**)<br/>                     | Tests whether any component of a 3D vector is a NaN.<br/>                                                                                                                                                                                           |
| [**XMVector3Less**](https://www.bing.com/search?q=**XMVector3Less**)<br/>                       | Tests whether one 3D vector is less than another 3D vector.<br/>                                                                                                                                                                                    |
| [**XMVector3LessOrEqual**](https://www.bing.com/search?q=**XMVector3LessOrEqual**)<br/>         | Tests whether one 3D vector is less-than-or-equal-to another 3D vector.<br/>                                                                                                                                                                        |
| [**XMVector3NearEqual**](https://www.bing.com/search?q=**XMVector3NearEqual**)<br/>             | Tests whether one 3D vector is near another 3D vector.<br/>                                                                                                                                                                                         |
| [**XMVector3NotEqual**](https://www.bing.com/search?q=**XMVector3NotEqual**)<br/>               | Tests whether two 3D vectors are not equal.<br/>                                                                                                                                                                                                    |
| [**XMVector3NotEqualInt**](https://www.bing.com/search?q=**XMVector3NotEqualInt**)<br/>         | Test whether two 3D vectors are not equal, treating each component as an unsigned integer.<br/>                                                                                                                                                     |
