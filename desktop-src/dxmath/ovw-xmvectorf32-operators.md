# XMVECTORF32 Operators

The following operators are exposed by the [**XMVECTORF32**](xmvectorf32-data-type.md) structure.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                               | Description                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**operator \_\_m128d**](xmvectorf32-operator---m128d.md)<br/> | Returns the object implementing the current instance of [**XMVECTORF32**](xmvectorf32-data-type.md) cast to a [\_\_m128d](http://msdn.microsoft.com/en-us/library/25537780.aspx) instance \[Windows systems supporting [SSE](http://msdn.microsoft.com/en-us/library/t467de55.aspx) Only\]<br/>    |
| [**operator \_\_m128i**](xmvectorf32-operator---m128i.md)<br/> | Returns the object implementing the current instance of [**XMVECTORF32**](xmvectorf32-data-type.md) cast to a [\_\_m128i](http://msdn.microsoft.com/en-us/library/26232t5c.aspx) instance of \[Windows systems supporting [SSE](http://msdn.microsoft.com/en-us/library/t467de55.aspx) Only\]<br/> |
| [**operator XMVECTOR**](xmvectorf32-operator-xmvector.md)<br/> | Returns the instance of `XMVECTOR` implementing the current instance of `XMVECTORF32`.<br/>                                                                                                                                                                                                         |
