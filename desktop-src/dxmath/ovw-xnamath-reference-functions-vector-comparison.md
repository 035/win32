# Vector Comparison Functions

Lists the vector comparison functions.

## In this section



| Topic                                                                 | Description                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorEqual**](https://www.bing.com/search?q=**XMVectorEqual**)<br/>                     | Performs a per-component test for equality of two vectors.<br/>                                                                                                                                                                                                     |
| [**XMVectorEqualInt**](https://www.bing.com/search?q=**XMVectorEqualInt**)<br/>               | Performs a per-component test for the equality of two vectors, treating each component as an unsigned integer.<br/>                                                                                                                                                 |
| [**XMVectorEqualIntR**](https://www.bing.com/search?q=**XMVectorEqualIntR**)<br/>             | Performs a per-component test for equality of two vectors, treating each component as an unsigned integer. In addition, this function sets a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVectorEqualR**](https://www.bing.com/search?q=**XMVectorEqualR**)<br/>                   | Performs a per-component test for equality of two vectors and sets a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                                         |
| [**XMVectorGreater**](https://www.bing.com/search?q=**XMVectorGreater**)<br/>                 | Performs a per-component test for greater-than between two vectors.<br/>                                                                                                                                                                                            |
| [**XMVectorGreaterOrEqual**](https://www.bing.com/search?q=**XMVectorGreaterOrEqual**)<br/>   | Performs a per-component test for greater-than-or-equal between two vectors.<br/>                                                                                                                                                                                   |
| [**XMVectorGreaterOrEqualR**](https://www.bing.com/search?q=**XMVectorGreaterOrEqualR**)<br/> | Performs a per-component test for greater-than-or-equal between two vectors and sets a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                       |
| [**XMVectorGreaterR**](https://www.bing.com/search?q=**XMVectorGreaterR**)<br/>               | Performs a per-component test for greater-than between two vectors and sets a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                                |
| [**XMVectorLess**](https://www.bing.com/search?q=**XMVectorLess**)<br/>                       | Performs a per-component test for less-than between two vectors.<br/>                                                                                                                                                                                               |
| [**XMVectorLessOrEqual**](https://www.bing.com/search?q=**XMVectorLessOrEqual**)<br/>         | Performs a per-component test for less-than-or-equal between two vectors.<br/>                                                                                                                                                                                      |
| [**XMVectorNearEqual**](https://www.bing.com/search?q=**XMVectorNearEqual**)<br/>             | Performs a per-component test for equality of two vectors within a given threshold. <br/>                                                                                                                                                                           |
