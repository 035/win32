# DirectXMath Library Color Functions

Lists the color functions provided by DirectXMath.

## In this section



| Topic                                                                 | Description                                                                                                                                   |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMColorSRGBToRGB**](-xmcolorsrgbtorgb.md)<br/>              | Converts an sRGB color vector to RGB.<br/>                                                                                              |
| [**XMColorRGBToSRGB**](-xmcolorrgbtosrgb.md)<br/>              | Converts an RGB color vector to sRGB.<br/>                                                                                              |
| [**XMColorAdjustContrast**](xmcoloradjustcontrast.md)<br/>     | Adjusts the contrast value of a color.<br/>                                                                                             |
| [**XMColorAdjustSaturation**](xmcoloradjustsaturation.md)<br/> | Adjusts the saturation value of a color.<br/>                                                                                           |
| [**XMColorEqual**](xmcolorequal.md)<br/>                       | Tests for the equality of two colors.<br/>                                                                                              |
| [**XMColorGreater**](xmcolorgreater.md)<br/>                   | Tests whether all the components of the first color are greater than the corresponding components in the second color.<br/>             |
| [**XMColorGreaterOrEqual**](xmcolorgreaterorequal.md)<br/>     | Tests whether all the components of the first color are greater than or equal to the corresponding components of the second color.<br/> |
| [**XMColorHSLToRGB**](xmcolorhsltorgb.md)<br/>                 | Converts HSL color values to RGB color values.<br/>                                                                                     |
| [**XMColorHSVToRGB**](xmcolorhsvtorgb.md)<br/>                 | Converts HSV color values to RGB color values.<br/>                                                                                     |
| [**XMColorIsInfinite**](xmcolorisinfinite.md)<br/>             | Tests to see whether any of the components of a color are either positive or negative infinity.<br/>                                    |
| [**XMColorIsNaN**](xmcolorisnan.md)<br/>                       | Tests to see whether any component of a color is not a number (NaN).<br/>                                                               |
| [**XMColorLess**](xmcolorless.md)<br/>                         | Tests whether all the components of the first color are less than the corresponding components of the second color.<br/>                |
| [**XMColorLessOrEqual**](xmcolorlessorequal.md)<br/>           | Tests whether all the components of the first color are less than or equal to the corresponding components of the second color.<br/>    |
| [**XMColorModulate**](xmcolormodulate.md)<br/>                 | Blends two colors by multiplying corresponding components together.<br/>                                                                |
| [**XMColorNegative**](xmcolornegative.md)<br/>                 | Determines the negative RGB color value of a color.<br/>                                                                                |
| [**XMColorNotEqual**](xmcolornotequal.md)<br/>                 | Tests to see whether two colors are unequal.<br/>                                                                                       |
| [**XMColorRGBToHSL**](xmcolorrgbtohsl.md)<br/>                 | Converts RGB color values to HSL color values.<br/>                                                                                     |
| [**XMColorRGBToHSV**](xmcolorrgbtohsv.md)<br/>                 | Converts RGB color values to HSV color values.<br/>                                                                                     |
| [**XMColorRGBToXYZ**](xmcolorrgbtoxyz.md)<br/>                 | Converts RGB color values to XYZ color values.<br/>                                                                                     |
| [**XMColorRGBToYUV**](xmcolorrgbtoyuv.md)<br/>                 | Converts RGB color values to YUV color values.<br/>                                                                                     |
| [**XMColorRGBToYUV\_HD**](xmcolorrgbtoyuv-hd.md)<br/>          | Converts RGB color values to YUV HD color values.<br/>                                                                                  |
| [**XMColorSRGBToXYZ**](xmcolorsrgbtoxyz.md)<br/>               | Converts SRGB color values to XYZ color values.<br/>                                                                                    |
| [**XMColorXYZToRGB**](xmcolorxyztorgb.md)<br/>                 | Converts XYZ color values to RGB color values.<br/>                                                                                     |
| [**XMColorXYZToSRGB**](xmcolorxyztosrgb.md)<br/>               | Converts XYZ color values to SRGB color values.<br/>                                                                                    |
| [**XMColorYUVToRGB**](xmcoloryuvtorgb.md)<br/>                 | Converts YUV color values to RGB color values.<br/>                                                                                     |
| [**XMColorYUVToRGB\_HD**](xmcoloryuvtorgb-hd.md)<br/>          | Converts YUV color values to RGB HD color values.<br/>                                                                                  |
