# XMSHORTN2 Operators

The [**XMSHORTN2**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmshortn2?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                | Description                                                                                                           |
|--------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmshortn2-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMSHORTN2` to the current instance of `XMSHORTN2`.<br/> |
