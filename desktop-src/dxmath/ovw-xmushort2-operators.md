# XMUSHORT2 Operators

The [**XMUSHORT2**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmushort2?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                | Description                                                                                                            |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmushort2-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMUSHORT2` to the current instance of `XMUSHORT2`. <br/> |
