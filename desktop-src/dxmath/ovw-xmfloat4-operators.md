# XMFLOAT4 Operators

The [**XMFLOAT4**](/windows/win32/DirectXMath/?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                               | Description                                                                                                         |
|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmfloat4-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMFLOAT4` to the current instance of `XMFLOAT4`.<br/> |
