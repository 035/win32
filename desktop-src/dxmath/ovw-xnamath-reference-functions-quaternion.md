# DirectXMath Library Quaternion Functions

Lists the quaternion functions provided by DirectXMath.

| Topic                                                                                                       | Description                                                                                                     |
|-------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**XMQuaternionBaryCentric**](xmquaternionbarycentric.md)<br/>                                       | Returns a point in barycentric coordinates, using the specified quaternions.<br/>                         |

| [**XMQuaternionConjugate**](xmquaternionconjugate.md)<br/>                                           | Computes the conjugate of a quaternion.<br/>                                                              |
| [**XMQuaternionDot**](xmquaterniondot.md)<br/>                                                       | Computes the dot product of two quaternions.<br/>                                                         |
| [**XMQuaternionEqual**](xmquaternionequal.md)<br/>                                                   | Tests whether two quaternions are equal.<br/>                                                             |
| [**XMQuaternionExp**](xmquaternionexp.md)<br/>                                                       | Computes the exponential of a given pure quaternion.<br/>                                                 |
| [**XMQuaternionIdentity**](xmquaternionidentity.md)<br/>                                             | Returns the identity quaternion.<br/>                                                                     |
| [**XMQuaternionInverse**](xmquaternioninverse.md)<br/>                                               | Computes the inverse of a quaternion.<br/>                                                                |
| [**XMQuaternionIsIdentity**](xmquaternionisidentity.md)<br/>                                         | Tests whether a specific quaternion is the identity quaternion.<br/>                                      |
| [**XMQuaternionIsInfinite**](xmquaternionisinfinite.md)<br/>                                         | Test whether any component of a quaternion is either positive or negative infinity.<br/>                  |
| [**XMQuaternionIsNaN**](xmquaternionisnan.md)<br/>                                                   | Test whether any component of a quaternion is a NaN.<br/>                                                 |
| [**XMQuaternionLength**](xmquaternionlength.md)<br/>                                                 | Computes the magnitude of a quaternion.<br/>                                                              |
| [**XMQuaternionLengthSq**](xmquaternionlengthsq.md)<br/>                                             | Computes the square of the magnitude of a quaternion.<br/>                                                |
| [**XMQuaternionLn**](xmquaternionln.md)<br/>                                                         | Computes the natural logarithm of a given unit quaternion.<br/>                                           |
| [**XMQuaternionMultiply**](xmquaternionmultiply.md)<br/>                                             | Computes the product of two quaternions.<br/>                                                             |
| [**XMQuaternionNormalize**](xmquaternionnormalize.md)<br/>                                           | Normalizes a quaternion.<br/>                                                                             |
| [**XMQuaternionNormalizeEst**](xmquaternionnormalizeest.md)<br/>                                     | Estimates the normalized version of a quaternion.<br/>                                                    |
| [**XMQuaternionNotEqual**](xmquaternionnotequal.md)<br/>                                             | Tests whether two quaternions are not equal.<br/>                                                         |
| [**XMQuaternionReciprocalLength**](xmquaternionreciprocallength.md)<br/>                             | Computes the reciprocal of the magnitude of a quaternion.<br/>                                            |
| [**XMQuaternionRotationAxis**](xmquaternionrotationaxis.md)<br/>                                     | Computes a rotation quaternion about an axis.<br/>                                                        |
| [**XMQuaternionRotationMatrix**](xmquaternionrotationmatrix.md)<br/>                                 | Computes a rotation quaternion from a rotation matrix.<br/>                                               |
| [**XMQuaternionRotationNormal**](xmquaternionrotationnormal.md)<br/>                                 | Computes the rotation quaternion about a normal vector.<br/>                                              |

| [**XMQuaternionSlerpV**](xmquaternionslerpv.md)<br/>                                                 | Interpolates between two unit quaternions, using spherical linear interpolation.<br/>                     |
| [**XMQuaternionSquad**](xmquaternionsquad.md)<br/>                                                   | Interpolates between four unit quaternions, using spherical quadrangle interpolation.<br/>                |
| [**XMQuaternionSquadSetup**](xmquaternionsquadsetup.md)<br/>                                         | Provides addresses of setup control points for spherical quadrangle interpolation.<br/>                   |
| [**XMQuaternionSquadV**](xmquaternionsquadv.md)<br/>                                                 | Interpolates between four unit quaternions, using spherical quadrangle interpolation.<br/>                |
