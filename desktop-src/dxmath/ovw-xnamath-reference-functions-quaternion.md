# DirectXMath Library Quaternion Functions

Lists the quaternion functions provided by DirectXMath.

## In this section



| Topic                                                                                                       | Description                                                                                                     |
|-------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**XMQuaternionBaryCentric**](/windows/desktop/api/DirectXMath/)<br/>                                       | Returns a point in barycentric coordinates, using the specified quaternions.<br/>                         |
| [**XMQuaternionBaryCentricV**](/windows/desktop/api/DirectXMath/)<br/>                                     | Returns a point in barycentric coordinates, using the specified quaternions.<br/>                         |
| [**XMQuaternionConjugate**](/windows/desktop/api/DirectXMath/)<br/>                                           | Computes the conjugate of a quaternion.<br/>                                                              |
| [**XMQuaternionDot**](/windows/desktop/api/DirectXMath/)<br/>                                                       | Computes the dot product of two quaternions.<br/>                                                         |
| [**XMQuaternionEqual**](/windows/desktop/api/DirectXMath/)<br/>                                                   | Tests whether two quaternions are equal.<br/>                                                             |
| [**XMQuaternionExp**](/windows/desktop/api/DirectXMath/)<br/>                                                       | Computes the exponential of a given pure quaternion.<br/>                                                 |
| [**XMQuaternionIdentity**](/windows/desktop/api/DirectXMath/)<br/>                                             | Returns the identity quaternion.<br/>                                                                     |
| [**XMQuaternionInverse**](/windows/desktop/api/DirectXMath/)<br/>                                               | Computes the inverse of a quaternion.<br/>                                                                |
| [**XMQuaternionIsIdentity**](/windows/desktop/api/DirectXMath/)<br/>                                         | Tests whether a specific quaternion is the identity quaternion.<br/>                                      |
| [**XMQuaternionIsInfinite**](/windows/desktop/api/DirectXMath/)<br/>                                         | Test whether any component of a quaternion is either positive or negative infinity.<br/>                  |
| [**XMQuaternionIsNaN**](/windows/desktop/api/DirectXMath/)<br/>                                                   | Test whether any component of a quaternion is a NaN.<br/>                                                 |
| [**XMQuaternionLength**](/windows/desktop/api/DirectXMath/)<br/>                                                 | Computes the magnitude of a quaternion.<br/>                                                              |
| [**XMQuaternionLengthSq**](/windows/desktop/api/DirectXMath/)<br/>                                             | Computes the square of the magnitude of a quaternion.<br/>                                                |
| [**XMQuaternionLn**](/windows/desktop/api/DirectXMath/)<br/>                                                         | Computes the natural logarithm of a given unit quaternion.<br/>                                           |
| [**XMQuaternionMultiply**](/windows/desktop/api/DirectXMath/)<br/>                                             | Computes the product of two quaternions.<br/>                                                             |
| [**XMQuaternionNormalize**](/windows/desktop/api/DirectXMath/)<br/>                                           | Normalizes a quaternion.<br/>                                                                             |
| [**XMQuaternionNormalizeEst**](/windows/desktop/api/DirectXMath/)<br/>                                     | Estimates the normalized version of a quaternion.<br/>                                                    |
| [**XMQuaternionNotEqual**](/windows/desktop/api/DirectXMath/)<br/>                                             | Tests whether two quaternions are not equal.<br/>                                                         |
| [**XMQuaternionReciprocalLength**](/windows/desktop/api/DirectXMath/)<br/>                             | Computes the reciprocal of the magnitude of a quaternion.<br/>                                            |
| [**XMQuaternionRotationAxis**](/windows/desktop/api/DirectXMath/)<br/>                                     | Computes a rotation quaternion about an axis.<br/>                                                        |
| [**XMQuaternionRotationMatrix**](/windows/desktop/api/DirectXMath/)<br/>                                 | Computes a rotation quaternion from a rotation matrix.<br/>                                               |
| [**XMQuaternionRotationNormal**](/windows/desktop/api/DirectXMath/)<br/>                                 | Computes the rotation quaternion about a normal vector.<br/>                                              |
| [**XMQuaternionRotationRollPitchYaw**](/windows/desktop/api/DirectXMath/)<br/>                     | Computes a rotation quaternion based on the pitch, yaw, and roll (Euler angles).<br/>                     |
| [**XMQuaternionRotationRollPitchYawFromVector**](/windows/desktop/api/DirectXMath/)<br/> | Computes a rotation quaternion based on a vector containing the Euler angles (pitch, yaw, and roll).<br/> |
| [**XMQuaternionSlerp**](/windows/desktop/api/DirectXMath/)<br/>                                                   | Interpolates between two unit quaternions, using spherical linear interpolation.<br/>                     |
| [**XMQuaternionSlerpV**](/windows/desktop/api/DirectXMath/)<br/>                                                 | Interpolates between two unit quaternions, using spherical linear interpolation.<br/>                     |
| [**XMQuaternionSquad**](/windows/desktop/api/DirectXMath/)<br/>                                                   | Interpolates between four unit quaternions, using spherical quadrangle interpolation.<br/>                |
| [**XMQuaternionSquadSetup**](/windows/desktop/api/DirectXMath/)<br/>                                         | Provides addresses of setup control points for spherical quadrangle interpolation.<br/>                   |
| [**XMQuaternionSquadV**](/windows/desktop/api/DirectXMath/)<br/>                                                 | Interpolates between four unit quaternions, using spherical quadrangle interpolation.<br/>                |
| [**XMQuaternionToAxisAngle**](/windows/desktop/api/DirectXMath/)<br/>                                       | Computes an axis and angle of rotation about that axis for a given quaternion.<br/>                       |
