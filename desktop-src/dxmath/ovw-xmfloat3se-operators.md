# XMFLOAT3SE Operators

The [**XMFLOAT3SE**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmfloat3se) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                               | Description                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**operator uint32\_t**](/windows/desktop/api/DirectXPackedVector/nf-directxpackedvector-xmcolor-operator uint32_t)<br/> | Returns an instance of `uint32_t` containing the components of the `XMFLOAT3SE` instance in a packed format.<br/> |
| [**operator =**](xmfloat3se-operator-eq.md)<br/>               | Assigns vector component data to the current instance of `XMFLOAT3SE`.<br/>                                       |
