# XMFLOAT2 Operators

The [**XMFLOAT2**](/windows/win32/DirectXMath/?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                               | Description                                                                                                         |
|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmfloat2-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMFLOAT2` to the current instance of `XMFLOAT2`.<br/> |
