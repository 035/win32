# DirectXMath Library Scalar Functions

Lists the scalar functions provided by DirectXMath.

## In this section



| Topic                                                     | Description                                                          |
|-----------------------------------------------------------|----------------------------------------------------------------------|
| [**XMScalarACos**](xmscalaracos.md)<br/>           | Computes the arccosine of a floating-point number.<br/>        |
| [**XMScalarACosEst**](xmscalaracosest.md)<br/>     | Estimates the arccosine of a floating-point number.<br/>       |
| [**XMScalarASin**](xmscalarasin.md)<br/>           | Computes the arcsine of a floating-point number.<br/>          |
| [**XMScalarASinEst**](xmscalarasinest.md)<br/>     | Estimates the arcsine of a floating-point number.<br/>         |
| [**XMScalarCos**](xmscalarcos.md)<br/>             | Computes the cosine of a radian angle.<br/>                    |
| [**XMScalarCosEst**](xmscalarcosest.md)<br/>       | Estimates the cosine of a radian angle.<br/>                   |
| [**XMScalarModAngle**](xmscalarmodangle.md)<br/>   | Computes an angle between -XM\_PI and XM\_PI.<br/>             |
| [**XMScalarNearEqual**](xmscalarnearequal.md)<br/> | Determines if two floating-point values are nearly equal.<br/> |
| [**XMScalarSin**](xmscalarsin.md)<br/>             | Computes the sine of a radian angle.<br/>                      |
| [**XMScalarSinCos**](xmscalarsincos.md)<br/>       | Computes both the sine and cosine of a radian angle.<br/>      |
| [**XMScalarSinCosEst**](xmscalarsincosest.md)<br/> | Estimates both the sine and cosine of a radian angle.<br/>     |
| [**XMScalarSinEst**](xmscalarsinest.md)<br/>       | Estimates the sine of a radian angle.<br/>                     |
