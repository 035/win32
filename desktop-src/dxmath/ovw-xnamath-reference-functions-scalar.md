# DirectXMath Library Scalar Functions

Lists the scalar functions provided by DirectXMath.

## In this section



| Topic                                                     | Description                                                          |
|-----------------------------------------------------------|----------------------------------------------------------------------|
| [**XMScalarACos**](/windows/win32/DirectXMath/?branch=master)<br/>           | Computes the arccosine of a floating-point number.<br/>        |
| [**XMScalarACosEst**](/windows/win32/DirectXMath/?branch=master)<br/>     | Estimates the arccosine of a floating-point number.<br/>       |
| [**XMScalarASin**](/windows/win32/DirectXMath/?branch=master)<br/>           | Computes the arcsine of a floating-point number.<br/>          |
| [**XMScalarASinEst**](/windows/win32/DirectXMath/?branch=master)<br/>     | Estimates the arcsine of a floating-point number.<br/>         |
| [**XMScalarCos**](/windows/win32/DirectXMath/?branch=master)<br/>             | Computes the cosine of a radian angle.<br/>                    |
| [**XMScalarCosEst**](/windows/win32/DirectXMath/?branch=master)<br/>       | Estimates the cosine of a radian angle.<br/>                   |
| [**XMScalarModAngle**](/windows/win32/DirectXMath/?branch=master)<br/>   | Computes an angle between -XM\_PI and XM\_PI.<br/>             |
| [**XMScalarNearEqual**](/windows/win32/DirectXMath/?branch=master)<br/> | Determines if two floating-point values are nearly equal.<br/> |
| [**XMScalarSin**](/windows/win32/DirectXMath/?branch=master)<br/>             | Computes the sine of a radian angle.<br/>                      |
| [**XMScalarSinCos**](/windows/win32/DirectXMath/?branch=master)<br/>       | Computes both the sine and cosine of a radian angle.<br/>      |
| [**XMScalarSinCosEst**](/windows/win32/DirectXMath/?branch=master)<br/> | Estimates both the sine and cosine of a radian angle.<br/>     |
| [**XMScalarSinEst**](/windows/win32/DirectXMath/?branch=master)<br/>       | Estimates the sine of a radian angle.<br/>                     |
