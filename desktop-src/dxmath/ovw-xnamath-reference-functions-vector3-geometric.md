# DirectXMath Library 3D Vector Geometric Functions

Lists the 3D vector geometric functions.

## In this section



| Topic                                                                                 | Description                                                                                                                       |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3AngleBetweenNormals**](xmvector3anglebetweennormals.md)<br/>       | Computes the radian angle between two normalized 3D vectors.<br/>                                                           |
| [**XMVector3AngleBetweenNormalsEst**](xmvector3anglebetweennormalsest.md)<br/> | Estimates the radian angle between two normalized 3D vectors.<br/>                                                          |
| [**XMVector3AngleBetweenVectors**](xmvector3anglebetweenvectors.md)<br/>       | Computes the radian angle between two 3D vectors.<br/>                                                                      |
| [**XMVector3ClampLength**](xmvector3clamplength.md)<br/>                       | Clamps the length of a 3D vector to a given range.<br/>                                                                     |

| [**XMVector3ComponentsFromNormal**](xmvector3componentsfromnormal.md)<br/>     | Using a reference normal vector, splits a 3D vector into components that are parallel and perpendicular to the normal.<br/> |
| [**XMVector3Cross**](xmvector3cross.md)<br/>                                   | Computes the cross product between two 3D vectors.<br/>                                                                     |
| [**XMVector3Dot**](xmvector3dot.md)<br/>                                       | Computes the dot product between 3D vectors.<br/>                                                                           |
| [**XMVector3InBounds**](xmvector3inbounds.md)<br/>                             | Tests whether the components of a 3D vector are within set bounds.<br/>                                                     |

| [**XMVector3NormalizeEst**](xmvector3normalizeest.md)<br/>                     | Estimates the normalized version of a 3D vector.<br/>                                                                       |
| [**XMVector3Orthogonal**](xmvector3orthogonal.md)<br/>                         | Computes a vector perpendicular to a 3D vector.<br/>                                                                        |
| [**XMVector3ReciprocalLength**](xmvector3reciprocallength.md)<br/>             | Computes the reciprocal of the length of a 3D vector.<br/>                                                                  |
| [**XMVector3ReciprocalLengthEst**](xmvector3reciprocallengthest.md)<br/>       | Estimates the reciprocal of the length of a 3D vector.<br/>                                                                 |
| [**XMVector3Reflect**](xmvector3reflect.md)<br/>                               | Reflects an incident 3D vector across a 3D normal vector.<br/>                                                              |
| [**XMVector3Refract**](xmvector3refract.md)<br/>                               | Refracts an incident 3D vector across a 3D normal vector.<br/>                                                              |
| [**XMVector3RefractV**](xmvector3refractv.md)<br/>                             | Refracts an incident 3D vector across a 3D normal vector.<br/>                                                              |
