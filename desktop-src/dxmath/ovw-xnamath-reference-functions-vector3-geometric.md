# DirectXMath Library 3D Vector Geometric Functions

Lists the 3D vector geometric functions.

## In this section



| Topic                                                                                 | Description                                                                                                                       |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3AngleBetweenNormals**](https://www.bing.com/search?q=**XMVector3AngleBetweenNormals**)<br/>       | Computes the radian angle between two normalized 3D vectors.<br/>                                                           |
| [**XMVector3AngleBetweenNormalsEst**](https://www.bing.com/search?q=**XMVector3AngleBetweenNormalsEst**)<br/> | Estimates the radian angle between two normalized 3D vectors.<br/>                                                          |
| [**XMVector3AngleBetweenVectors**](https://www.bing.com/search?q=**XMVector3AngleBetweenVectors**)<br/>       | Computes the radian angle between two 3D vectors.<br/>                                                                      |
| [**XMVector3ClampLength**](https://www.bing.com/search?q=**XMVector3ClampLength**)<br/>                       | Clamps the length of a 3D vector to a given range.<br/>                                                                     |
| [**XMVector3ClampLengthV**](https://www.bing.com/search?q=**XMVector3ClampLengthV**)<br/>                     | Clamps the length of a 3D vector to a given range.<br/>                                                                     |
| [**XMVector3ComponentsFromNormal**](https://www.bing.com/search?q=**XMVector3ComponentsFromNormal**)<br/>     | Using a reference normal vector, splits a 3D vector into components that are parallel and perpendicular to the normal.<br/> |
| [**XMVector3Cross**](https://www.bing.com/search?q=**XMVector3Cross**)<br/>                                   | Computes the cross product between two 3D vectors.<br/>                                                                     |
| [**XMVector3Dot**](https://www.bing.com/search?q=**XMVector3Dot**)<br/>                                       | Computes the dot product between 3D vectors.<br/>                                                                           |
| [**XMVector3InBounds**](https://www.bing.com/search?q=**XMVector3InBounds**)<br/>                             | Tests whether the components of a 3D vector are within set bounds.<br/>                                                     |

| [**XMVector3LengthEst**](https://www.bing.com/search?q=**XMVector3LengthEst**)<br/>                           | Estimates the length of a 3D vector.<br/>                                                                                   |
| [**XMVector3LengthSq**](https://www.bing.com/search?q=**XMVector3LengthSq**)<br/>                             | Computes the square of the length of a 3D vector.<br/>                                                                      |
| [**XMVector3LinePointDistance**](https://www.bing.com/search?q=**XMVector3LinePointDistance**)<br/>           | Computes the minimum distance between a line and a point.<br/>                                                              |
| [**XMVector3Normalize**](https://www.bing.com/search?q=**XMVector3Normalize**)<br/>                           | Returns the normalized version of a 3D vector.<br/>                                                                         |
| [**XMVector3NormalizeEst**](https://www.bing.com/search?q=**XMVector3NormalizeEst**)<br/>                     | Estimates the normalized version of a 3D vector.<br/>                                                                       |

| [**XMVector3ReciprocalLengthEst**](https://www.bing.com/search?q=**XMVector3ReciprocalLengthEst**)<br/>       | Estimates the reciprocal of the length of a 3D vector.<br/>                                                                 |
| [**XMVector3Reflect**](https://www.bing.com/search?q=**XMVector3Reflect**)<br/>                               | Reflects an incident 3D vector across a 3D normal vector.<br/>                                                              |
