# XMFLOAT4X4 Operators

The [**XMFLOAT4X4**](/windows/win32/DirectXMath/?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                      | Description                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmfloat4x4-operator-eq.md)<br/>      | Assigns the vector component data from one instance of `XMFLOAT4X4` to the current instance of `XMFLOAT4X4`. <br/> |
| [**operator ()**](xmfloat4x4-operator-parens.md)<br/> | Accesses specific matrix elements referenced by row and column from the current instance of `XMFLOAT4X4`. <br/>    |
