# DirectXMath Library 2D Vector Geometric Functions

Lists the 2D vector geometric functions.

## In this section



| Topic                                                                                 | Description                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**XMVector2AngleBetweenNormals**](xmvector2anglebetweennormals.md)<br/>       | Computes the radian angle between two normalized 2D vectors.<br/>       |
| [**XMVector2AngleBetweenNormalsEst**](xmvector2anglebetweennormalsest.md)<br/> | Estimates the radian angle between two normalized 2D vectors.<br/>      |
| [**XMVector2AngleBetweenVectors**](xmvector2anglebetweenvectors.md)<br/>       | Computes the radian angle between two 2D vectors.<br/>                  |
| [**XMVector2ClampLength**](xmvector2clamplength.md)<br/>                       | Clamps the length of a 2D vector to a given range.<br/>                 |
| [**XMVector2ClampLengthV**](xmvector2clamplengthv.md)<br/>                     | Clamps the length of a 2D vector to a given range.<br/>                 |
| [**XMVector2Cross**](xmvector2cross.md)<br/>                                   | Computes the 2D cross product.<br/>                                     |
| [**XMVector2Dot**](xmvector2dot.md)<br/>                                       | Computes the dot product between 2D vectors.<br/>                       |
| [**XMVector2InBounds**](xmvector2inbounds.md)<br/>                             | Tests whether the components of a 2D vector are within set bounds.<br/> |
| [**XMVector2IntersectLine**](xmvector2intersectline.md)<br/>                   | Finds the intersection of two lines.<br/>                               |
| [**XMVector2Length**](xmvector2length.md)<br/>                                 | Computes the length of a 2D vector.<br/>                                |
| [**XMVector2LengthEst**](xmvector2lengthest.md)<br/>                           | Estimates the length of a 2D vector.<br/>                               |

| [**XMVector2Normalize**](xmvector2normalize.md)<br/>                           | Returns the normalized version of a 2D vector.<br/>                     |
| [**XMVector2NormalizeEst**](xmvector2normalizeest.md)<br/>                     | Estimates the normalized version of a 2D vector.<br/>                   |
| [**XMVector2Orthogonal**](xmvector2orthogonal.md)<br/>                         | Computes a vector perpendicular to a 2D vector.<br/>                    |
| [**XMVector2ReciprocalLength**](xmvector2reciprocallength.md)<br/>             | Computes the reciprocal of the length of a 2D vector.<br/>              |
| [**XMVector2ReciprocalLengthEst**](xmvector2reciprocallengthest.md)<br/>       | Estimates the reciprocal of the length of a 2D vector.<br/>             |
| [**XMVector2Reflect**](xmvector2reflect.md)<br/>                               | Reflects an incident 2D vector across a 2D normal vector.<br/>          |
| [**XMVector2Refract**](xmvector2refract.md)<br/>                               | Refracts an incident 2D vector across a 2D normal vector.<br/>          |
| [**XMVector2RefractV**](xmvector2refractv.md)<br/>                             | Refracts an incident 2D vector across a 2D normal vector.<br/>          |
