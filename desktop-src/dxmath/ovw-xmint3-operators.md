# XMINT3 Operators

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                             | Description                                                                                                      |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmint3-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMINT3` to the current instance of `XMINT3`. <br/> |
