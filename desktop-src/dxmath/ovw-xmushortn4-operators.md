# XMUSHORTN4 Operators

The [**XMUSHORTN4**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmushortn4?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                 | Description                                                                                                              |
|---------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmushortn4-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMUSHORTN4` to the current instance of `XMUSHORTN4`. <br/> |
