# XMINT2 Operators

The [**XMINT2**](/windows/win32/DirectXMath/?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                             | Description                                                                                                     |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmint2-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMINT2` to the current instance of `XMINT2`.<br/> |
