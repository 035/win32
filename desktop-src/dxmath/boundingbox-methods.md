# BoundingBox Methods



| Method                                                              | Description                                                                                            |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Contains**](boundingbox-contains.md)<br/>                 | Tests whether the BoundingBox contains a specified object.<br/>                                  |
| [**CreateFromPoints**](boundingbox-createfrompoints.md)<br/> | Creates a BoundingBox from points.<br/>                                                          |
| [**Intersects**](boundingbox-intersects.md)<br/>             | Tests the BoundingBox for intersection with another object.<br/>                                 |
| [**Transform**](boundingbox-transform.md)<br/>               | Transforms the BoundingBox.<br/>                                                                 |
| [**ContainedBy**](boundingbox-containedby.md)<br/>           | Tests whether the [**BoundingBox**](boundingbox.md) is contained by the specified frustum.<br/> |
| [**CreateFromSphere**](boundingbox-createfromsphere.md)<br/> | Creates a BoundingBox large enough to contain the a specified BoundingSphere.<br/>               |
| [**CreateMerged**](boundingbox-createmerged.md)<br/>         | Creates a BoundingBox large enough to contains two specified BoundBox intances.<br/>             |
| [**GetCorners**](boundingbox-getcorners.md)<br/>             | Retrieves the corners of the BoundingBox.<br/>                                                   |
| [**op\_Assignment**](boundingbox-op-assignment.md)<br/>      | Copies values from another BoundingBox.<br/>                                                     |
