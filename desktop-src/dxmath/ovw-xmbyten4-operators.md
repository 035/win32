# XMBYTEN4 Operators

The [**XMBYTEN4**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmbyten4?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                               | Description                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmbyten4-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMBYTEN4` to the current instance of `XMBYTEN4`. <br/> |
