# XMBYTEN4 Operators

The [**XMBYTEN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmbyten4) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                               | Description                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operator =**](https://www.bing.com/search?q=**operator+=**)<br/> | Assigns the vector component data from one instance of `XMBYTEN4` to the current instance of `XMBYTEN4`. <br/> |
