# Bit-Wise Vector Functions

Lists the bit-wise vector functions.

## In this section



| Topic                                                         | Description                                                                                                                             |
|---------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorAndCInt**](xmvectorandcint.md)<br/>         | Computes the logical AND of one vector with the negation of a second vector, treating each component as an unsigned integer.<br/> |
| [**XMVectorAndInt**](xmvectorandint.md)<br/>           | Computes the logical AND of two vectors, treating each component as an unsigned integer.<br/>                                     |
| [**XMVectorNorInt**](xmvectornorint.md)<br/>           | Computes the logical NOR of two vectors, treating each component as an unsigned integer.<br/>                                     |
| [**XMVectorNotEqual**](xmvectornotequal.md)<br/>       | Performs a per-component test for the inequality of two vectors.<br/>                                                             |
| [**XMVectorNotEqualInt**](xmvectornotequalint.md)<br/> | Performs a per-component test for the inequality of two vectors, treating each component as an unsigned integer.<br/>             |
| [**XMVectorOrInt**](xmvectororint.md)<br/>             | Computes the logical OR of two vectors, treating each component as an unsigned integer.<br/>                                      |
| [**XMVectorXorInt**](xmvectorxorint.md)<br/>           | Computes the logical XOR of two vectors, treating each component as an unsigned integer.<br/>                                     |
