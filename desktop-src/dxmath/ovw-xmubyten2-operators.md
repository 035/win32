# XMUBYTEN2 Operators

The [**XMUBYTEN2**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmubyten2?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available with C++.

 

## In this section



| Methods                                                | Description                                                                                                           |
|--------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmubyten2-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMUBYTEN2` to the current instance of `XMUBYTEN2`.<br/> |
