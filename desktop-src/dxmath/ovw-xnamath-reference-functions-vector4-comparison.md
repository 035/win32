# DirectXMath Library 4D Vector Comparison Functions

Lists the 4D vector comparison functions.

## In this section



| Topic                                                                   | Description                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector4Equal**](xmvector4equal.md)<br/>                     | Tests whether two 4D vectors are equal.<br/>                                                                                                                                                                                                        |
| [**XMVector4EqualInt**](xmvector4equalint.md)<br/>               | Tests whether two 4D vectors are equal, treating each component as an unsigned integer.<br/>                                                                                                                                                        |
| [**XMVector4EqualIntR**](xmvector4equalintr.md)<br/>             | Tests whether two 4D vectors are equal, treating each component as an unsigned integer. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/win32/DirectXMath/nf-directxmath-xmcomparisonalltrue?branch=master).<br/> |
| [**XMVector4EqualR**](xmvector4equalr.md)<br/>                   | Tests whether two 4D vectors are equal. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/win32/DirectXMath/nf-directxmath-xmcomparisonalltrue?branch=master).<br/>                                                 |
| [**XMVector4Greater**](xmvector4greater.md)<br/>                 | Tests whether one 4D vector is greater than another 4D vector.<br/>                                                                                                                                                                                 |
| [**XMVector4GreaterOrEqual**](xmvector4greaterorequal.md)<br/>   | Tests whether one 4D vector is greater-than-or-equal-to another 4D vector.<br/>                                                                                                                                                                     |
| [**XMVector4GreaterOrEqualR**](xmvector4greaterorequalr.md)<br/> | Tests whether one 4D vector is greater-than-or-equal-to another 4D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/win32/DirectXMath/nf-directxmath-xmcomparisonalltrue?branch=master).<br/>                                      |
| [**XMVector4GreaterR**](xmvector4greaterr.md)<br/>               | Tests whether one 4D vector is greater than another 4D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/win32/DirectXMath/nf-directxmath-xmcomparisonalltrue?branch=master).<br/>                                                  |
| [**XMVector4IsInfinite**](xmvector4isinfinite.md)<br/>           | Tests whether any component of a 4D vector is positive or negative infinity.<br/>                                                                                                                                                                   |
| [**XMVector4IsNaN**](xmvector4isnan.md)<br/>                     | Tests whether any component of a 4D vector is a NaN.<br/>                                                                                                                                                                                           |
| [**XMVector4Less**](xmvector4less.md)<br/>                       | Tests whether one 4D vector is less than another 4D vector.<br/>                                                                                                                                                                                    |
| [**XMVector4LessOrEqual**](xmvector4lessorequal.md)<br/>         | Tests whether one 4D vector is less-than-or-equal-to another 4D vector.<br/>                                                                                                                                                                        |
| [**XMVector4NearEqual**](xmvector4nearequal.md)<br/>             | Tests whether one 4D vector is near another 4D vector.<br/>                                                                                                                                                                                         |
| [**XMVector4NotEqual**](xmvector4notequal.md)<br/>               | Tests whether two 4D vectors are not equal.<br/>                                                                                                                                                                                                    |
| [**XMVector4NotEqualInt**](xmvector4notequalint.md)<br/>         | Test whether two 4D vectors are not equal, treating each component as an unsigned integer.<br/>                                                                                                                                                     |
