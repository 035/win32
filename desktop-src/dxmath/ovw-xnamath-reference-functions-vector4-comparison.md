# DirectXMath Library 4D Vector Comparison Functions

Lists the 4D vector comparison functions.

## In this section



| Topic                                                                   | Description                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector4Equal**](https://www.bing.com/search?q=**XMVector4Equal**)<br/>                     | Tests whether two 4D vectors are equal.<br/>                                                                                                                                                                                                        |
| [**XMVector4EqualInt**](https://www.bing.com/search?q=**XMVector4EqualInt**)<br/>               | Tests whether two 4D vectors are equal, treating each component as an unsigned integer.<br/>                                                                                                                                                        |
| [**XMVector4EqualIntR**](https://www.bing.com/search?q=**XMVector4EqualIntR**)<br/>             | Tests whether two 4D vectors are equal, treating each component as an unsigned integer. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector4EqualR**](https://www.bing.com/search?q=**XMVector4EqualR**)<br/>                   | Tests whether two 4D vectors are equal. In addition, this function returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector4Greater**](https://www.bing.com/search?q=**XMVector4Greater**)<br/>                 | Tests whether one 4D vector is greater than another 4D vector.<br/>                                                                                                                                                                                 |
| [**XMVector4GreaterOrEqual**](https://www.bing.com/search?q=**XMVector4GreaterOrEqual**)<br/>   | Tests whether one 4D vector is greater-than-or-equal-to another 4D vector.<br/>                                                                                                                                                                     |
| [**XMVector4GreaterOrEqualR**](https://www.bing.com/search?q=**XMVector4GreaterOrEqualR**)<br/> | Tests whether one 4D vector is greater-than-or-equal-to another 4D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector4GreaterR**](https://www.bing.com/search?q=**XMVector4GreaterR**)<br/>               | Tests whether one 4D vector is greater than another 4D vector and returns a comparison value that can be examined using functions such as [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector4IsInfinite**](https://www.bing.com/search?q=**XMVector4IsInfinite**)<br/>           | Tests whether any component of a 4D vector is positive or negative infinity.<br/>                                                                                                                                                                   |
| [**XMVector4IsNaN**](https://www.bing.com/search?q=**XMVector4IsNaN**)<br/>                     | Tests whether any component of a 4D vector is a NaN.<br/>                                                                                                                                                                                           |
| [**XMVector4Less**](https://www.bing.com/search?q=**XMVector4Less**)<br/>                       | Tests whether one 4D vector is less than another 4D vector.<br/>                                                                                                                                                                                    |
| [**XMVector4LessOrEqual**](https://www.bing.com/search?q=**XMVector4LessOrEqual**)<br/>         | Tests whether one 4D vector is less-than-or-equal-to another 4D vector.<br/>                                                                                                                                                                        |
| [**XMVector4NearEqual**](https://www.bing.com/search?q=**XMVector4NearEqual**)<br/>             | Tests whether one 4D vector is near another 4D vector.<br/>                                                                                                                                                                                         |
| [**XMVector4NotEqual**](https://www.bing.com/search?q=**XMVector4NotEqual**)<br/>               | Tests whether two 4D vectors are not equal.<br/>                                                                                                                                                                                                    |
| [**XMVector4NotEqualInt**](https://www.bing.com/search?q=**XMVector4NotEqualInt**)<br/>         | Test whether two 4D vectors are not equal, treating each component as an unsigned integer.<br/>                                                                                                                                                     |
