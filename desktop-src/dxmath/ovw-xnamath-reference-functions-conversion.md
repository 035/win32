# DirectXMath Library Conversion Functions

Lists the conversion functions provided by DirectXMath.

## In this section



| Topic                                                                       | Description                                                                                                                                                          |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMConvertFloatToHalf**](xmconvertfloattohalf.md)<br/>             | Converts a single-precision floating-point value to a half-precision floating-point value.<br/>                                                                |
| [**XMConvertFloatToHalfStream**](xmconvertfloattohalfstream.md)<br/> | Converts a stream of single-precision floating-point values to a stream of half-precision floating-point values.<br/>                                          |
| [**XMConvertHalfToFloat**](xmconverthalftofloat.md)<br/>             | Converts a half-precision floating-point value to a single-precision floating-point value.<br/>                                                                |
| [**XMConvertHalfToFloatStream**](xmconverthalftofloatstream.md)<br/> | Converts a stream of half-precision floating-point values to a stream of single-precision floating-point values.<br/>                                          |
| [**XMConvertToDegrees**](xmconverttodegrees.md)<br/>                 | Converts an angle measured in radians into one measured in degrees.<br/>                                                                                       |
| [**XMConvertToRadians**](xmconverttoradians.md)<br/>                 | Converts an angle measured in degrees into one measured in radians.<br/>                                                                                       |
| [**XMConvertVectorFloatToInt**](xmconvertvectorfloattoint.md)<br/>   | Converts an [**XMVECTOR**](xmvector-data-type.md) with **float** components to an **XMVECTOR** with **int32\_t** components and applies a uniform bias.<br/>  |
| [**XMConvertVectorFloatToUInt**](xmconvertvectorfloattouint.md)<br/> | Converts an [**XMVECTOR**](xmvector-data-type.md) with **float** components to an **XMVECTOR** with **uint32\_t** components and applies a uniform bias.<br/> |
| [**XMConvertVectorIntToFloat**](xmconvertvectorinttofloat.md)<br/>   | Converts an [**XMVECTOR**](xmvector-data-type.md) with **int32\_t** components to an **XMVECTOR** with **float** components and applies a uniform bias.<br/>  |
| [**XMConvertVectorUIntToFloat**](xmconvertvectoruinttofloat.md)<br/> | Converts an [**XMVECTOR**](xmvector-data-type.md) with **uint32\_t** components to an **XMVECTOR** with **float** components and applies a uniform bias.<br/> |
