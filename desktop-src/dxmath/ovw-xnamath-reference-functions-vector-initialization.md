# Vector Initialization Functions

Lists the vector initialization functions.

## In this section



| Topic                                                                     | Description                                                                                                                                      |
|---------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorFalseInt**](xmvectorfalseint.md)<br/>                   | Returns the zero (false) vector.<br/>                                                                                                      |
| [**XMVectorReplicate**](xmvectorreplicate.md)<br/>                 | Replicates a floating-point value into all four components of a vector.<br/>                                                               |
| [**XMVectorReplicateInt**](xmvectorreplicateint.md)<br/>           | Replicates an integer value into all four components of a vector.<br/>                                                                     |
| [**XMVectorReplicateIntPtr**](xmvectorreplicateintptr.md)<br/>     | Replicates an integer value referenced by a pointer, into all four components of a vector.<br/>                                            |
| [**XMVectorReplicatePtr**](xmvectorreplicateptr.md)<br/>           | Replicates a floating-point value referenced by pointer into all four components of a vector.<br/>                                         |
| [**XMVectorSet**](xmvectorset.md)<br/>                             | Creates a vector using four floating-point values.<br/>                                                                                    |
| [**XMVectorSetBinaryConstant**](/windows/win32/DirectXMath/?branch=master)<br/> | Creates a vector, each of whose components is either 0.0f or 1.0f.<br/>                                                                    |
| [**XMVectorSetInt**](xmvectorsetint.md)<br/>                       | Creates a vector with unsigned integer components.<br/>                                                                                    |
| [**XMVectorSplatConstant**](/windows/win32/DirectXMath/?branch=master)<br/>         | Creates a vector with identical floating-point components. Each component is a constant divided by two raised to an integer exponent.<br/> |
| [**XMVectorSplatConstantInt**](/windows/win32/DirectXMath/?branch=master)<br/>   | Creates a vector with identical integer components.<br/>                                                                                   |
| [**XMVectorSplatEpsilon**](xmvectorsplatepsilon.md)<br/>           | Returns a vector, each of whose components are epsilon (1.192092896e-7).<br/>                                                              |
| [**XMVectorSplatInfinity**](xmvectorsplatinfinity.md)<br/>         | Returns a vector, each of whose components are infinity (0x7F800000).<br/>                                                                 |
| [**XMVectorSplatOne**](xmvectorsplatone.md)<br/>                   | Returns a vector, each of whose components are one.<br/>                                                                                   |
| [**XMVectorSplatQNaN**](xmvectorsplatqnan.md)<br/>                 | Returns a vector, each of whose components are QNaN (0x7CF00000).<br/>                                                                     |
| [**XMVectorSplatSignMask**](xmvectorsplatsignmask.md)<br/>         | Returns a vector, each of whose components are the sign mask (0x80000000).<br/>                                                            |
| [**XMVectorTrueInt**](xmvectortrueint.md)<br/>                     | Returns a vector, each of whose components represents true (0xFFFFFFFF).<br/>                                                              |
| [**XMVectorZero**](xmvectorzero.md)<br/>                           | Creates the zero vector.<br/>                                                                                                              |
