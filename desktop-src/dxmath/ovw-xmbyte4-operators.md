# XMBYTE4 Operators

The [**XMBYTE4**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmbyte4?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                              | Description                                                                                                        |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmbyte4-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMBYTE4` to the current instance of `XMBYTE4`. <br/> |
