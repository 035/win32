# XMHALF2 Operators

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                              | Description                                                                                                        |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**operator =**](xmhalf2-operator-eq.md)<br/> | Assigns the vector component data from one instance of `XMHALF2` to the current instance of `XMHALF2`. <br/> |
