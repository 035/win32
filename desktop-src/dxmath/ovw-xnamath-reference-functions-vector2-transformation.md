# DirectXMath Library 2D Vector Transformation Functions

Lists the 2D vector transformation functions.

## In this section



| Topic                                                                               | Description                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Transform**](xmvector2transform.md)<br/>                         | Transforms a 2D vector by a matrix.<br/>                                                                                                   |
| [**XMVector2TransformCoord**](xmvector2transformcoord.md)<br/>               | Transforms a 2D vector by a given matrix, projecting the result back into w = 1.<br/>                                                      |
| [**XMVector2TransformCoordStream**](xmvector2transformcoordstream.md)<br/>   | Transforms a stream of 2D vectors by a given matrix, projecting the resulting vectors such that their w coordinates are equal to 1.0.<br/> |
| [**XMVector2TransformNormal**](xmvector2transformnormal.md)<br/>             | Transforms the 2D vector normal by the given matrix.<br/>                                                                                  |
| [**XMVector2TransformNormalStream**](xmvector2transformnormalstream.md)<br/> | Transforms a stream of 2D normal vectors by a given matrix.<br/>                                                                           |
| [**XMVector2TransformStream**](xmvector2transformstream.md)<br/>             | Transforms a stream of 2D vectors by a given matrix.<br/>                                                                                  |
