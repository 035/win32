# XMUNIBBLE4 Operators

The [**XMUNIBBLE4**](/windows/win32/DirectXPackedVector/ns-directxpackedvector-xmunibble4?branch=master) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                               | Description                                                                                                              |
|-----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operator uint16\_t**](/windows/win32/DirectXPackedVector/?branch=master)<br/> | Returns an instance of `uint16_t` containing the components of the `XMUNIBBLE4` instance in a packed format. <br/> |
| [**operator =**](xmunibble4-operator-eq.md)<br/>               | Assigns vector component data to the current instance of `XMUNIBBLE4`. <br/>                                       |
