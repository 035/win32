# XMUNIBBLE4 Operators

The [**XMUNIBBLE4**](xmunibble4.md) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                               | Description                                                                                                              |
|-----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operator uint16\_t**](xmunibble4-operator-uint16-t.md)<br/> | Returns an instance of `uint16_t` containing the components of the `XMUNIBBLE4` instance in a packed format. <br/> |
| [**operator =**](xmunibble4-operator-eq.md)<br/>               | Assigns vector component data to the current instance of `XMUNIBBLE4`. <br/>                                       |
