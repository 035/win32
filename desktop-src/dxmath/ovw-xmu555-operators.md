# XMU555 Operators

The [**XMU555**](xmu555.md) structure exposes the following operators.

> [!Note]  
> The operators listed here are only available under C++.

 

## In this section



| Methods                                                           | Description                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operator uint16\_t**](xmu555-operator-uint16-t.md)<br/> | Returns an instance of `uint16_t` containing the components of the `XMU555` instance in a packed format. <br/> |
| [**operator =**](xmu555-operator-eq.md)<br/>               | Assigns vector component data to the current instance of `XMU555`. <br/>                                       |
