# Grouping API Enumerations

The Grouping API uses the following enumerations:



| Enumeration                                                                        | Description                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PEER\_GROUP\_AUTHENTICATION\_SCHEME**](/windows/win32/P2P/ne-p2p-peer_group_authentication_scheme_tag?branch=master)    | Defines the set of possible authentication schemes that can be used to authenticate peers joining a peer group.                                                                                                                                           |
| [**PEER\_GROUP\_EVENT\_TYPE**](/windows/win32/P2P/ne-p2p-peer_group_event_type_tag?branch=master)                          | Contains the specific peer event types that can occur within a peer group.                                                                                                                                                                                |
| [**PEER\_GROUP\_ISSUE\_CREDENTIAL\_FLAGS**](/windows/win32/P2P/ne-p2p-peer_issue_credential_flags_tag?branch=master) | Specifies if user credentials are stored within a group.                                                                                                                                                                                                  |
| [**PEER\_GROUP\_PROPERTY\_FLAGS**](/windows/win32/P2P/ne-p2p-peer_group_property_flags_tag?branch=master)                  | Specifies peer group membership settings.                                                                                                                                                                                                                 |
| [**PEER\_GROUP\_STATUS**](/windows/win32/P2P/ne-p2p-peer_group_status_tag?branch=master)                                   | Indicates whether or not the peer group has connections present.                                                                                                                                                                                          |
| [**PEER\_MEMBER\_CHANGE\_TYPE**](/windows/win32/P2P/ne-p2p-peer_member_change_type_tag?branch=master)                      | Defines the set of possible peer group membership and presence states for a peer.                                                                                                                                                                         |
| [**PEER\_MEMBER\_FLAGS**](/windows/win32/P2P/ne-p2p-peer_member_flags_tag?branch=master)                                   | Allows an application to specify whether all members or only present ones should be enumerated when the [**PeerGroupEnumMembers**](/windows/win32/P2P/nf-p2p-peergroupenummembers?branch=master) function is called, or to indicate whether or not a member is present within the peer group. |
