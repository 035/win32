# Grouping API Enumerations

The Grouping API uses the following enumerations:



| Enumeration                                                                        | Description                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PEER\_GROUP\_AUTHENTICATION\_SCHEME**](peer-group-authentication-scheme.md)    | Defines the set of possible authentication schemes that can be used to authenticate peers joining a peer group.                                                                                                                                           |
| [**PEER\_GROUP\_EVENT\_TYPE**](peer-group-event-type.md)                          | Contains the specific peer event types that can occur within a peer group.                                                                                                                                                                                |
| [**PEER\_GROUP\_ISSUE\_CREDENTIAL\_FLAGS**](peer-group-issue-credential-flags.md) | Specifies if user credentials are stored within a group.                                                                                                                                                                                                  |
| [**PEER\_GROUP\_PROPERTY\_FLAGS**](peer-group-property-flags.md)                  | Specifies peer group membership settings.                                                                                                                                                                                                                 |
| [**PEER\_GROUP\_STATUS**](peer-group-status.md)                                   | Indicates whether or not the peer group has connections present.                                                                                                                                                                                          |
| [**PEER\_MEMBER\_CHANGE\_TYPE**](peer-member-change-type.md)                      | Defines the set of possible peer group membership and presence states for a peer.                                                                                                                                                                         |
| [**PEER\_MEMBER\_FLAGS**](peer-member-flags.md)                                   | Allows an application to specify whether all members or only present ones should be enumerated when the [**PeerGroupEnumMembers**](peergroupenummembers.md) function is called, or to indicate whether or not a member is present within the peer group. |
