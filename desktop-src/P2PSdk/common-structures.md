# Common Structures

The Peer Infrastructure uses the following common structures.



| Structure                                                                          | Description                                                                                                                         |
|------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**PEER\_ADDRESS**](/windows/win32/P2P/ns-p2p-peer_address_tag?branch=master)                                              | Specifies the information about the IP address.                                                                                     |
| [**PEER\_CONNECTION\_INFO**](/windows/win32/P2P/ns-p2p-peer_connection_info_tag?branch=master)                             | Contains information about a connection. This structure is returned when you are enumerating peer graphing or grouping connections. |
| [**PEER\_DATA**](/windows/win32/P2P/ns-p2p-peer_data_tag?branch=master)                                                    | Contains binary data.                                                                                                               |
| [**PEER\_EVENT\_CONNECTION\_CHANGE\_DATA**](/windows/win32/P2P/ns-p2p-peer_event_connection_change_data_tag?branch=master) | Contains updated information that includes changes to a neighbor or direct connection.                                              |
| [**PEER\_EVENT\_INCOMING\_DATA**](/windows/win32/P2P/ns-p2p-peer_event_incoming_data_tag?branch=master)                    | Contains updated information that includes data changes a node receives from a neighbor or direct connection.                       |
| [**PEER\_EVENT\_RECORD\_CHANGE\_DATA**](/windows/win32/P2P/ns-p2p-peer_event_record_change_data_tag?branch=master)         | Contains updated information that an application requests for data changes to a record or record type.                              |
| [**PEER\_RECORD**](/windows/win32/P2P/ns-p2p-peer_record_tag?branch=master)                                                | Contains the record object that an application uses.                                                                                |
| [**PEER\_VERSION\_DATA**](/windows/win32/P2P/ns-p2p-peer_version_data_tag?branch=master)                                   | Contains the version information about the Peer Graphing and Grouping APIs.                                                         |
