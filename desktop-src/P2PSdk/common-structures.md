# Common Structures

The Peer Infrastructure uses the following common structures.



| Structure                                                                          | Description                                                                                                                         |
|------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**PEER\_ADDRESS**](peer-address.md)                                              | Specifies the information about the IP address.                                                                                     |
| [**PEER\_CONNECTION\_INFO**](peer-connection-info.md)                             | Contains information about a connection. This structure is returned when you are enumerating peer graphing or grouping connections. |
| [**PEER\_DATA**](peer-data.md)                                                    | Contains binary data.                                                                                                               |
| [**PEER\_EVENT\_CONNECTION\_CHANGE\_DATA**](peer-event-connection-change-data.md) | Contains updated information that includes changes to a neighbor or direct connection.                                              |
| [**PEER\_EVENT\_INCOMING\_DATA**](peer-event-incoming-data.md)                    | Contains updated information that includes data changes a node receives from a neighbor or direct connection.                       |
| [**PEER\_EVENT\_RECORD\_CHANGE\_DATA**](peer-event-record-change-data.md)         | Contains updated information that an application requests for data changes to a record or record type.                              |
| [**PEER\_RECORD**](peer-record.md)                                                | Contains the record object that an application uses.                                                                                |
| [**PEER\_VERSION\_DATA**](peer-version-data.md)                                   | Contains the version information about the Peer Graphing and Grouping APIs.                                                         |
