# Graphing API Constants

The Graphing API uses the following constants:



| Constant                                                          | Description                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------|
| [**PEER\_GRAPH\_EVENT\_TYPE**](/windows/win32/P2P/ne-p2p-peer_graph_event_type_tag?branch=master)         | Specifies peer event types the application is to be notified for. |
| [**PEER\_GRAPH\_PROPERTY\_FLAGS**](/windows/win32/P2P/ne-p2p-peer_graph_property_flags_tag?branch=master) | Specifies properties of a peer graph.                             |
| [**PEER\_GRAPH\_SCOPE**](/windows/win32/P2P/ne-p2p-peer_graph_scope_tag?branch=master)                    | Specifies the network scope of a peer graph.                      |
| [**PEER\_GRAPH\_STATUS\_FLAGS**](/windows/win32/P2P/ne-p2p-peer_graph_status_flags_tag?branch=master)     | Specifies the current status of a node within the peer graph.     |
| [**PEER\_NODE\_CHANGE\_TYPE**](/windows/win32/P2P/ne-p2p-peer_node_change_type_tag?branch=master)         | Specifies the types of peer node graph status changes.            |
