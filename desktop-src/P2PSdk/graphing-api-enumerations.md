# Graphing API Constants

The Graphing API uses the following constants:



| Constant                                                          | Description                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------|
| [**PEER\_GRAPH\_EVENT\_TYPE**](peer-graph-event-type.md)         | Specifies peer event types the application is to be notified for. |
| [**PEER\_GRAPH\_PROPERTY\_FLAGS**](peer-graph-property-flags.md) | Specifies properties of a peer graph.                             |
| [**PEER\_GRAPH\_SCOPE**](peer-graph-scope.md)                    | Specifies the network scope of a peer graph.                      |
| [**PEER\_GRAPH\_STATUS\_FLAGS**](peer-graph-status-flags.md)     | Specifies the current status of a node within the peer graph.     |
| [**PEER\_NODE\_CHANGE\_TYPE**](peer-node-change-type.md)         | Specifies the types of peer node graph status changes.            |
