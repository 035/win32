# Graphing API Callbacks

The Graphing API uses the following callbacks:



| Callback                                                          | Description                                                                                                                                                                                                                  |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [*PFNPEER\_FREE\_SECURITY\_DATA*](pfnpeer-free-security-data.md) | Specifies the function that the Peer Graphing Infrastructure calls to free data returned by [*PFNPEER\_SECURE\_RECORD*](pfnpeer-secure-record.md) and [*PFNPEER\_VALIDATE\_RECORD*](pfnpeer-validate-record.md) callbacks. |
| [*PFNPEER\_SECURE\_RECORD*](pfnpeer-secure-record.md)            | Specifies the function that the Peer Graphing Infrastructure calls to secure records.                                                                                                                                        |
| [*PFNPEER\_VALIDATE\_RECORD*](pfnpeer-validate-record.md)        | Specifies the function that the Peer Graphing Infrastructure calls to validate records.                                                                                                                                      |
