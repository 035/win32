# Collaboration API Functions

The Peer Collaboration Infrastructure supports the following functions.



| Function                                                                                       | Description                                                                                                                                                                                     |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PeerCollabAddContact**](/windows/win32/P2P/nf-p2p-peercollabaddcontact?branch=master)                                           | Adds a contact to the contact list of a peer.                                                                                                                                                   |
| [**PeerCollabAsyncInviteContact**](/windows/win32/P2P/nf-p2p-peercollabasyncinvitecontact?branch=master)                           | Sends an invitation to a trusted peer contact to join the sender's Peer Collaboration activity over a secured connection.                                                                       |
| [**PeerCollabAsyncInviteEndpoint**](/windows/win32/P2P/nf-p2p-peercollabasyncinviteendpoint?branch=master)                         | Sends an invitation to a specified peer endpoint to join the sender's Peer Collaboration activity. The availability of the response to the invitation is updated through an asynchronous event. |
| [**PeerCollabCancelInvitation**](/windows/win32/P2P/nf-p2p-peercollabcancelinvitation?branch=master)                               | Cancels an invitation previously sent by the caller to a contact.                                                                                                                               |
| [**PeerCollabCloseHandle**](/windows/win32/P2P/nf-p2p-peercollabclosehandle?branch=master)                                         | Closes the handle to a Peer Collaboration activity invitation.                                                                                                                                  |
| [**PeerCollabDeleteContact**](/windows/win32/P2P/nf-p2p-peercollabdeletecontact?branch=master)                                     | Deletes a contact from the current peer.                                                                                                                                                        |
| [**PeerCollabDeleteEndpointData**](/windows/win32/P2P/nf-p2p-peercollabdeleteendpointdata?branch=master)                           | Deletes the peer endpoint data on the calling peer node that matches the supplied endpoint data.                                                                                                |
| [**PeerCollabDeleteObject**](/windows/win32/P2P/nf-p2p-peercollabdeleteobject?branch=master)                                       | Deletes a peer object from the calling endpoint.                                                                                                                                                |
| [**PeerCollabEnumApplications**](/windows/win32/P2P/nf-p2p-peercollabenumapplications?branch=master)                               | Returns the handle to an enumeration that contains the capabilities registered to a specific peer's endpoint(s).                                                                                |
| [**PeerCollabEnumApplicationRegistrationInfo**](/windows/win32/P2P/nf-p2p-peercollabenumapplicationregistrationinfo?branch=master) | Obtains the enumeration handle used to retrieve peer application information.                                                                                                                   |
| [**PeerCollabEnumContacts**](/windows/win32/P2P/nf-p2p-peercollabenumcontacts?branch=master)                                       | Returns a handle to an enumerated set that contains all of the peer collaboration network contacts currently available on the calling peer.                                                     |
| [**PeerCollabEnumEndpoints**](/windows/win32/P2P/nf-p2p-peercollabenumendpoints?branch=master)                                     | Returns the handle to an enumeration that contains the endpoints associated with a specific peer contact.                                                                                       |
| [**PeerCollabEnumObjects**](/windows/win32/P2P/nf-p2p-peercollabenumobjects?branch=master)                                         | Returns the handle to an enumeration that contains the peer objects associated with a specific peer's endpoint.                                                                                 |
| [**PeerCollabEnumPeopleNearMe**](/windows/win32/P2P/nf-p2p-peercollabenumpeoplenearme?branch=master)                               | Returns a handle to an enumerated set that contains all of the peer collaboration network "people near me" endpoints currently available on the subnet of the calling peer.                     |
| [**PeerCollabExportContact**](/windows/win32/P2P/nf-p2p-peercollabexportcontact?branch=master)                                     | Exports the contact data associated with a peer name to a contact XML data string buffer.                                                                                                       |
| [**PeerCollabGetAppLaunchInfo**](/windows/win32/P2P/nf-p2p-peercollabgetapplaunchinfo?branch=master)                               | Obtains the peer application launch information, including the contact name, the peer endpoint, and the invitation request.                                                                     |
| [**PeerCollabGetApplicationRegistrationInfo**](/windows/win32/P2P/nf-p2p-peercollabgetapplicationregistrationinfo?branch=master)   | Obtains specific application registration information.                                                                                                                                          |
| [**PeerCollabGetContact**](/windows/win32/P2P/nf-p2p-peercollabgetcontact?branch=master)                                           | Obtains the information for a specific peer contact given the peer name of the contact.                                                                                                         |
| [**PeerCollabGetEndpointName**](/windows/win32/P2P/nf-p2p-peercollabgetendpointname?branch=master)                                 | Retrieves the name of the current endpoint of the calling peer previously set by a call to PeerCollabSetEndpointName.                                                                           |
| [**PeerCollabGetEventData**](/windows/win32/P2P/nf-p2p-peercollabgeteventdata?branch=master)                                       | Obtains the data associated with a Peer Collaboration event raised on the peer.                                                                                                                 |
| [**PeerCollabGetInvitationResponse**](/windows/win32/P2P/nf-p2p-peercollabgetinvitationresponse?branch=master)                     | Obtains the response from peer previously invited to join a Peer Collaboration activity.                                                                                                        |
| [**PeerCollabGetPresenceInfo**](/windows/win32/P2P/nf-p2p-peercollabgetpresenceinfo?branch=master)                                 | Retrieves the presence information for the endpoint associated with a specific contact.                                                                                                         |
| [**PeerCollabGetSigninOptions**](/windows/win32/P2P/nf-p2p-peercollabgetsigninoptions?branch=master)                               | Obtains the peer's current signed-in peer collaboration network presence options.                                                                                                               |
| [**PeerCollabInviteContact**](/windows/win32/P2P/nf-p2p-peercollabinvitecontact?branch=master)                                     | Sends an invitation to join a Peer Collaboration activity to a trusted contact. This call is synchronous and, if successful, obtains a response from the contact.                               |
| [**PeerCollabInviteEndpoint**](/windows/win32/P2P/nf-p2p-peercollabinviteendpoint?branch=master)                                   | Sends an invitation to a specified peer endpoint to join the sender's Peer Collaboration activity. This call is synchronous and, if successful, obtains a response from the peer endpoint.      |
| [**PeerCollabParseContact**](/windows/win32/P2P/nf-p2p-peercollabparsecontact?branch=master)                                       | Parses a Unicode string buffer containing contact XML data into a [**PEER\_CONTACT**](/windows/win32/P2P/ns-p2p-peer_contact_tag?branch=master) data structure.                                                                         |
| [**PeerCollabQueryContactData**](/windows/win32/P2P/nf-p2p-peercollabquerycontactdata?branch=master)                               | Retrieves the contact information for the supplied peer endpoint.                                                                                                                               |
| [**PeerCollabRefreshEndpointData**](/windows/win32/P2P/nf-p2p-peercollabrefreshendpointdata?branch=master)                         | Updates the calling peer node with new endpoint data.                                                                                                                                           |
| [**PeerCollabRegisterApplication**](/windows/win32/P2P/nf-p2p-peercollabregisterapplication?branch=master)                         | Registers an application with the local computer so that it can be launched in a peer collaboration activity.                                                                                   |
| [**PeerCollabRegisterEvent**](/windows/win32/P2P/nf-p2p-peercollabregisterevent?branch=master)                                     | Registers an application with the Peer Collaboration infrastructure to receive callbacks for specific Peer Collaboration events.                                                                |
| [**PeerCollabSetEndpointName**](/windows/win32/P2P/nf-p2p-peercollabsetendpointname?branch=master)                                 | Sets the name of the current endpoint used by the peer application.                                                                                                                             |
| [**PeerCollabSetObject**](/windows/win32/P2P/nf-p2p-peercollabsetobject?branch=master)                                             | Creates or updates a peer data object used in a peer collaboration network.                                                                                                                     |
| [**PeerCollabSetPresenceInfo**](/windows/win32/P2P/nf-p2p-peercollabsetpresenceinfo?branch=master)                                 | Updates the caller's presence information to any contacts watching it.                                                                                                                          |
| [**PeerCollabSignIn**](/windows/win32/P2P/nf-p2p-peercollabsignin?branch=master)                                                   | Signs the peer into a hosted Internet (serverless presence) or subnet ("People Near Me") peer collaboration network presence provider.                                                          |
| [**PeerCollabSignOut**](/windows/win32/P2P/nf-p2p-peercollabsignout?branch=master)                                                 | Signs a peer out of a specific type of peer collaboration network presence provider.                                                                                                            |
| [**PeerCollabShutdown**](/windows/win32/P2P/nf-p2p-peercollabshutdown?branch=master)                                               | Shuts down the Peer Collaboration infrastructure and releases any resources associated with it.                                                                                                 |
| [**PeerCollabStartup**](/windows/win32/P2P/nf-p2p-peercollabstartup?branch=master)                                                 | Initializes the Peer Collaboration infrastructure.                                                                                                                                              |
| [**PeerCollabSubscribeEndpointData**](/windows/win32/P2P/nf-p2p-peercollabsubscribeendpointdata?branch=master)                     | Creates a subscription to an available endpoint.                                                                                                                                                |
| [**PeerCollabUnregisterApplication**](/windows/win32/P2P/nf-p2p-peercollabunregisterapplication?branch=master)                     | Unregisters the specific applications of a peer from the local computer.                                                                                                                        |
| [**PeerCollabUnregisterEvent**](/windows/win32/P2P/nf-p2p-peercollabunregisterevent?branch=master)                                 | Deregisters an application from notification about specific Peer Collaboration events.                                                                                                          |
| [**PeerCollabUnsubscribeEndpointData**](/windows/win32/P2P/nf-p2p-peercollabunsubscribeendpointdata?branch=master)                 | Removes a subscription to an endpoint created with PeerCollabSubscribeEndpointData.                                                                                                             |
| [**PeerCollabUpdateContact**](/windows/win32/P2P/nf-p2p-peercollabupdatecontact?branch=master)                                     | Updates a peer participating in a peer collaboration network with new information on a peer contact.                                                                                            |
