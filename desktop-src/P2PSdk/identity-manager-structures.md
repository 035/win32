# Identity Manager Structures

The Peer Identity Manager API uses the following structure:



| Structure                                  | Description                                                                   |
|--------------------------------------------|-------------------------------------------------------------------------------|
| [**PEER\_NAME\_PAIR**](/windows/win32/P2P/ns-p2p-peer_name_pair_tag?branch=master) | Contains the results of a call to [**PeerGetNextItem**](/windows/win32/P2P/nf-p2p-peergetnextitem?branch=master). |
