# Collaboration API Enumerations

The following enumerations support the Peer Collaboration Infrastructure functions.



| Enumeration                                                                         | Description                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PEER\_APPLICATION\_REGISTRATION\_TYPE**](peer-application-registration-type.md) | Defines the set of peer application registration flags.                                                                                                                                                                                                                       |
| [**PEER\_CHANGE\_TYPE**](peer-change-type.md)                                      | Defines the set of changes that can be performed on a peer object, endpoint, or application.                                                                                                                                                                                  |
| [**PEER\_SIGNIN\_FLAGS**](peer-signin-flags.md)                                    | Defines the set of peer presence publication behaviors available when the peer signs in to a peer collaboration network.                                                                                                                                                      |
| [**PEER\_WATCH\_PERMISSION**](peer-watch-permission.md)                            | Defines the set of permissions that represent a contact's presence to a watching peer.                                                                                                                                                                                        |
| [**PEER\_COLLAB\_EVENT\_TYPE**](peer-collab-event-type.md)                         | Defines the set of events that can be raised on a peer by the peer collaboration network event infrastructure.                                                                                                                                                                |
| [**PEER\_PRESENCE\_STATUS**](peer-presence-status.md)                              | Defines the set of possible presence status settings available to a peer that participates in a peer collaboration network. These settings can be set by a peer collaboration network endpoint to indicate the peer contact's current level of participation to its watchers. |
| [**PEER\_INVITATION\_RESPONSE\_TYPE**](peer-invitation-response-type.md)           | Defines the set of responses received to an invitation to join a Peer Collaboration activity.                                                                                                                                                                                 |
| [**PEER\_PUBLICATION\_SCOPE**](peer-publication-scope.md)                          | Defines the set of scopes for the publication of peer objects or data.                                                                                                                                                                                                        |
