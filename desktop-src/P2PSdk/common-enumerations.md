# Common Constants

The Peer Infrastructure uses the following common constants.



| Constant                                                      | Description                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [**PEER\_CONNECTION\_FLAGS**](peer-connection-flags.md)      | Specifies the types of connections that a peer can have.      |
| [**PEER\_CONNECTION\_STATUS**](peer-connection-status.md)    | Specifies the status of a peer direct or neighbor connection. |
| [**PEER\_RECORD\_CHANGE\_TYPE**](peer-record-change-type.md) | Specifies the changes that can occur to a record.             |
| [**PEER\_RECORD\_FLAGS**](peer-record-flags.md)              | Specifies a set of flags for peer record behaviors.           |
