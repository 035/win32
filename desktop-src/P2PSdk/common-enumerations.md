# Common Constants

The Peer Infrastructure uses the following common constants.



| Constant                                                      | Description                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [**PEER\_CONNECTION\_FLAGS**](/windows/win32/P2P/ne-p2p-peer_connection_flags_tag?branch=master)      | Specifies the types of connections that a peer can have.      |
| [**PEER\_CONNECTION\_STATUS**](/windows/win32/P2P/ne-p2p-peer_connection_status_tag?branch=master)    | Specifies the status of a peer direct or neighbor connection. |
| [**PEER\_RECORD\_CHANGE\_TYPE**](/windows/win32/P2P/ne-p2p-peer_record_change_type_tag?branch=master) | Specifies the changes that can occur to a record.             |
| [**PEER\_RECORD\_FLAGS**](/windows/win32/P2P/ne-p2p-peer_record_flags_tag?branch=master)              | Specifies a set of flags for peer record behaviors.           |
