# Graphing API Structures

The Graphing API uses the following structures:



| Structure                                                                 | Description                                                                                                   |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**PEER\_EVENT\_NODE\_CHANGE\_DATA**](/windows/win32/P2P/ns-p2p-peer_event_node_change_data_tag?branch=master)    | Contains a pointer to the data if a **PEER\_GRAPH\_EVENT\_NODE\_CHANGE** event is triggered.                  |
| [**PEER\_EVENT\_SYNCHRONIZED\_DATA**](/windows/win32/P2P/ns-p2p-peer_event_synchronized_data_tag?branch=master)   | Indicates the type of record that has been synchronized.                                                      |
| [**PEER\_GRAPH\_EVENT\_DATA**](/windows/win32/P2P/ns-p2p-peer_graph_event_data_tag?branch=master)                 | Contains data associated with a peer event.                                                                   |
| [**PEER\_GRAPH\_EVENT\_REGISTRATION**](/windows/win32/P2P/ns-p2p-peer_graph_event_registration_tag?branch=master) | Specifies which peer events an application requires notifications for.                                        |
| [**PEER\_GRAPH\_PROPERTIES**](/windows/win32/P2P/ns-p2p-peer_graph_properties_tag?branch=master)                  | Contains data about the policy of a peer graph, ID, scope, and other information.                             |
| [**PEER\_NODE\_INFO**](/windows/win32/P2P/ns-p2p-peer_node_info_tag?branch=master)                                | Contains information that is specific to a particular node in a peer graph.                                   |
| [**PEER\_SECURITY\_INTERFACE**](/windows/win32/P2P/ns-p2p-peer_security_interface_tag?branch=master)              | Specifies the security interfaces that calls to Peer Graphing APIs use to validate, secure, and free records. |
