# Grouping API Structures

The Grouping API uses the following structures.



| Structure                                                                  | Description                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**PEER\_CREDENTIAL\_INFO**](/windows/win32/P2P/ns-p2p-peer_credential_info_tag?branch=master)                     | Defines information used to obtain and issue a peer's security credentials.                                   |
| [**PEER\_EVENT\_MEMBER\_CHANGE\_DATA**](/windows/win32/P2P/ns-p2p-peer_event_member_change_data_tag?branch=master) | Contains data that describes a change in the status of a peer group member.                                   |
| [**PEER\_GROUP\_EVENT\_DATA**](/windows/win32/P2P/ns-p2p-peer_group_event_data_tag?branch=master)                  | Contains information about a specific peer group event that has occurred.                                     |
| [**PEER\_GROUP\_EVENT\_REGISTRATION**](/windows/win32/P2P/ns-p2p-peer_group_event_registration_tag?branch=master)  | Defines the particular peer group event a member can register for.                                            |
| [**PEER\_GROUP\_PROPERTIES**](/windows/win32/P2P/ns-p2p-peer_group_properties_tag?branch=master)                   | Contains data about the membership policy of a peer group.                                                    |
| [**PEER\_INVITATION\_INFO**](/windows/win32/P2P/ns-p2p-peer_invitation_info_tag?branch=master)                     | Defines information about an invitation to join a peer group. Invitations are represented as Unicode strings. |
| [**PEER\_MEMBER**](/windows/win32/P2P/ns-p2p-peer_member_tag?branch=master)                                        | Contains information that describes a member of a peer group.                                                 |
