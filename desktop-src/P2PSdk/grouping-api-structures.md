# Grouping API Structures

The Grouping API uses the following structures.



| Structure                                                                  | Description                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**PEER\_CREDENTIAL\_INFO**](peer-credential-info.md)                     | Defines information used to obtain and issue a peer's security credentials.                                   |
| [**PEER\_EVENT\_MEMBER\_CHANGE\_DATA**](peer-event-member-change-data.md) | Contains data that describes a change in the status of a peer group member.                                   |
| [**PEER\_GROUP\_EVENT\_DATA**](peer-group-event-data.md)                  | Contains information about a specific peer group event that has occurred.                                     |
| [**PEER\_GROUP\_EVENT\_REGISTRATION**](peer-group-event-registration.md)  | Defines the particular peer group event a member can register for.                                            |
| [**PEER\_GROUP\_PROPERTIES**](peer-group-properties.md)                   | Contains data about the membership policy of a peer group.                                                    |
| [**PEER\_INVITATION\_INFO**](peer-invitation-info.md)                     | Defines information about an invitation to join a peer group. Invitations are represented as Unicode strings. |
| [**PEER\_MEMBER**](peer-member.md)                                        | Contains information that describes a member of a peer group.                                                 |
