# PNRP Constants

The PNRP Namespace Provider API uses the following constants.



| Constant                                                 | Description                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**PNRP\_CLOUD\_FLAGS**](pnrp-cloud-flags.md)           | Specifies the validity of a cloud name.                    |
| [**PNRP\_CLOUD\_STATE**](pnrp-cloud-state.md)           | Specifies the different states a PNRP cloud can be in.     |
| [**PNRP\_RESOLVE\_CRITERIA**](pnrp-resolve-criteria.md) | Specifies the criteria that PNRP uses to resolve searches. |
