# PNRP Constants

The PNRP Namespace Provider API uses the following constants.



| Constant                                                 | Description                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**PNRP\_CLOUD\_FLAGS**](/windows/win32/Pnrpdef/ne-pnrpdef-_pnrp_cloud_flags?branch=master)           | Specifies the validity of a cloud name.                    |
| [**PNRP\_CLOUD\_STATE**](/windows/win32/Pnrpdef/ne-pnrpdef-_pnrp_cloud_state?branch=master)           | Specifies the different states a PNRP cloud can be in.     |
| [**PNRP\_RESOLVE\_CRITERIA**](/windows/win32/Pnrpdef/ne-pnrpdef-_pnrp_resolve_criteria?branch=master) | Specifies the criteria that PNRP uses to resolve searches. |
