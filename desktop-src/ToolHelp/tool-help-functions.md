# Tool Help Functions

The following functions are part of the tool help library.



| Function                                                           | Description                                                                                                      |
|--------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**CreateToolhelp32Snapshot**](createtoolhelp32snapshot.md)       | Takes a snapshot of the specified processes, as well as the heaps, modules, and threads used by these processes. |
| [**Heap32First**](heap32first.md)                                 | Retrieves information about the first block of a heap that has been allocated by a process.                      |
| [**Heap32ListFirst**](heap32listfirst.md)                         | Retrieves information about the first heap that has been allocated by a specified process.                       |
| [**Heap32ListNext**](heap32listnext.md)                           | Retrieves information about the next heap that has been allocated by a process.                                  |
| [**Heap32Next**](heap32next.md)                                   | Retrieves information about the next block of a heap that has been allocated by a process.                       |
| [**Module32First**](module32first.md)                             | Retrieves information about the first module associated with a process.                                          |
| [**Module32Next**](module32next.md)                               | Retrieves information about the next module associated with a process or thread.                                 |
| [**Process32First**](process32first.md)                           | Retrieves information about the first process encountered in a system snapshot.                                  |
| [**Process32Next**](process32next.md)                             | Retrieves information about the next process recorded in a system snapshot.                                      |
| [**Thread32First**](thread32first.md)                             | Retrieves information about the first thread of any process encountered in a system snapshot.                    |
| [**Thread32Next**](thread32next.md)                               | Retrieves information about the next thread of any process encountered in the system memory snapshot.            |
| [**Toolhelp32ReadProcessMemory**](toolhelp32readprocessmemory.md) | Copies memory allocated to another process into an application-supplied buffer.                                  |
