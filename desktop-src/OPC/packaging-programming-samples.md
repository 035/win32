# Packaging API Samples

The topics in this section briefly describe the sample programs that accompany the Microsoft Windows Software Development Kit (SDK) for Windows 7.

The following table lists the topics that describe the Packaging API samples that are provided in the Windows SDK.

## Contents



|                                                                    |                                                                                                                                                                |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Set Author Sample](set-author-sample.md)                         | A sample application that uses the Packaging API to read and modify an existing package.<br/>                                                            |
| [Music Bundle Sample](music-bundle-sample.md)                     | A sample application that uses the Packaging API to produce and consume a package of the custom, music bundle package format.<br/>                       |
| [Music Bundle Signature Sample](music-bundle-signature-sample.md) | A sample application that uses the Packaging API to sign a package of the custom, music bundle package format and validate the generated signature.<br/> |
