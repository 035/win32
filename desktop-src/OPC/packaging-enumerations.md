# Packaging Enumerations

A listing of Packaging API enumerations.

## Contents



|                                                                                            |                                                                                                                                                                                                                                                                      |
|--------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OPC\_CANONICALIZATION\_METHOD**](opc-canonicalization-method.md)<br/>            | Describes the canonicalization method to be applied to XML markup. <br/>                                                                                                                                                                                       |
| [**OPC\_CERTIFICATE\_EMBEDDING\_OPTION**](opc-certificate-embedding-option.md)<br/> | Describes the storage location of a certificate that is used in signing.<br/>                                                                                                                                                                                  |
| [**OPC\_COMPRESSION\_OPTIONS**](opc-compression-options.md)<br/>                    | Describes ways to compress part content. <br/>                                                                                                                                                                                                                 |
| [**OPC\_READ\_FLAGS**](opc-read-flags.md)<br/>                                      | Describes the read settings for caching package components and validating them against *ECMA-376 OpenXML, 1st Edition, Part 2: Open Packaging Conventions (OPC)* conformance requirements. <br/>                                                               |
| [**OPC\_RELATIONSHIP\_SELECTOR**](opc-relationship-selector.md)<br/>                | Describes how to interpret the *selectionCriterion* parameter of the [**IOpcRelationshipSelector::GetSelectionCriterion**](iopcrelationshipselector-getselectioncriterion.md) method. <br/>                                                                   |
| [**OPC\_RELATIONSHIPS\_SIGNING\_OPTION**](opc-relationships-signing-option.md)<br/> | Describes whether a reference represented by the [**IOpcSignatureRelationshipReference**](iopcsignaturerelationshipreference.md) interface refers to all or a subset of relationships represented as relationship objects in a relationship set object. <br/> |
| [**OPC\_SIGNATURE\_TIME\_FORMAT**](opc-signature-time-format.md)<br/>               | Describes how to interpret the *signingTime* parameter, which is a record of when a signature was created, of the [**IOpcDigitalSignature::GetSigningTime**](iopcdigitalsignature-getsigningtime.md) method. <br/>                                            |
| [**OPC\_SIGNATURE\_VALIDATION\_RESULT**](opc-signature-validation-result.md)<br/>   | Indicates the status of the signature.<br/>                                                                                                                                                                                                                    |
| [**OPC\_STREAM\_IO\_MODE**](opc-stream-io-mode.md)<br/>                             | Describes the read/write status of a stream.<br/>                                                                                                                                                                                                              |
| [**OPC\_URI\_TARGET\_MODE**](opc-uri-target-mode.md)<br/>                           | Indicates the target mode of a relationship. <br/>                                                                                                                                                                                                             |
| [**OPC\_WRITE\_FLAGS**](opc-write-flags.md)<br/>                                    | Describes the encoding method that is used by the serialization object to produce the package. <br/>                                                                                                                                                           |
