# Monitor Interfaces

The following COM interfaces are used to develop monitors.



| Interface                              | Description                                                                                                                                             |
|----------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [IMonitor](imonitor.md)               | Provides methods that control monitor operation. This interface must be implemented by the monitor.                                                     |
| [IMonitorEventer](imonitoreventer.md) | Provides methods used to send events. This interface is provided by the [*Monitor Control Service*](m.md#-netmon-monitor-control-service-gly) (MCSVC). |
