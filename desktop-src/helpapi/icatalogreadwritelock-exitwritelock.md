# ICatalogReadWriteLock::ExitWriteLock method

method ExitWriteLock - exits the write lock

## Syntax


```C++
HRESULT ExitWriteLock();
```



## Parameters

This method has no parameters.

## Return value

If this method succeeds, it returns **S\_OK**. Otherwise, it returns an **HRESULT** error code.
