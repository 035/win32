# IndexException interface

An interface that inherits from the **IDispatch** interface.

## Members

The **IndexException** interface inherits from the [**IDispatch**](https://msdn.microsoft.com/windows/desktop/ebbff4bc-36b2-4861-9efa-ffa45e013eb5) interface but does not have additional members.
