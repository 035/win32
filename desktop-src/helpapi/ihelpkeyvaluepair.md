# IHelpKeyValuePair interface

Key/value pair for IHelpFilter objects

## Members

The **IHelpKeyValuePair** interface inherits from the [**IDispatch**](https://msdn.microsoft.com/windows/desktop/ebbff4bc-36b2-4861-9efa-ffa45e013eb5) interface. **IHelpKeyValuePair** also has these types of members:

-   [Methods](#methods)
-   [Properties](#properties)

### Methods

The **IHelpKeyValuePair** interface has these methods.



| Method                                   | Description                                                 |
|:-----------------------------------------|:------------------------------------------------------------|
| [**value**](ihelpkeyvaluepair-value.md) | method Value - get the value of a key/value pair<br/> |



 

### Properties

The **IHelpKeyValuePair** interface has these properties.



| Property                                        | Access type          | Description                                               |
|:------------------------------------------------|:---------------------|:----------------------------------------------------------|
| [**key**](ihelpkeyvaluepair-key.md)<br/> | Read-only<br/> | property Key - get the key of a key/value pair<br/> |
