# ICatalogReadWriteLock::ExitNonBlockingWriteOperation method

method ExitNonBlockingWriteOperation - exits a non-blocking write lock

## Syntax


```C++
HRESULT ExitNonBlockingWriteOperation();
```



## Parameters

This method has no parameters.

## Return value

If this method succeeds, it returns **S\_OK**. Otherwise, it returns an **HRESULT** error code.
