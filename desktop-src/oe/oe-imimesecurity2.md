# IMimeSecurity2 interface

Do not use. Encodes and decodes message bodies using Secure/Multipurpose Internet Mail Extensions (S/MIME) version 3.

## Members

The **IMimeSecurity2** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
