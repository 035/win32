# ISMTPTransport interface

\[**ISMTPTransport** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Transport for Simple Mail Transport Protocol (SMTP) messages.

## Members

The **ISMTPTransport** interface inherits from [**IInternetTransport**](oe-iinternettransport.md) but does not have additional members.
