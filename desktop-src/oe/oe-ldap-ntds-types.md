# LDAP NTDS Types

Do not use. Lightweight Directory Access Protocol (LDAP) NTDS type values that can be set.



| Constant/value                                                                                                                                                                                                                         | Description                                 |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------|
| <span id="LDAP_NTDS_UNKNOWN"></span><span id="ldap_ntds_unknown"></span><dl> <dt>**LDAP\_NTDS\_UNKNOWN**</dt> <dt>0x0000</dt> </dl> | Indicates NTDS state is unknown.<br/> |
| <span id="LDAP_NTDS_IS"></span><span id="ldap_ntds_is"></span><dl> <dt>**LDAP\_NTDS\_IS**</dt> <dt>0x0001</dt> </dl>                | Indicates is NTDS.<br/>               |
| <span id="LDAP_NTDS_ISNOT"></span><span id="ldap_ntds_isnot"></span><dl> <dt>**LDAP\_NTDS\_ISNOT**</dt> <dt>0x0002</dt> </dl>       | Indicates is not NTDS.<br/>           |
| <span id="LDAP_NTDS_MAX"></span><span id="ldap_ntds_max"></span><dl> <dt>**LDAP\_NTDS\_MAX**</dt> <dt>0x0002</dt> </dl>             | Indicates maximum valid value.<br/>   |
