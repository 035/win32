# IHTTPMailTransport2 interface

\[**IHTTPMailTransport2** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Extends the [**IHTTPMailTransport**](oe-ihttpmailtransport.md) interface.

## Members

The **IHTTPMailTransport2** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
