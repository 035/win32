# IOEExecRules interface

\[**IOEExecRules** is no longer available for use as of Windows Vista.\]

The methods of the **IOEExecRules** interface execute rules when certain criteria are met.

## Members

The **IOEExecRules** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
