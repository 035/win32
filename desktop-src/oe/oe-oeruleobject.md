# OERule object

\[**OERule** is no longer available for use as of Windows Vista.\]

The **OERule** object implements the [**IOERule**](oe-ioerule.md) interface. It represents a message rule.

## Remarks

### Interfaces Implemented



[**IOERule**](oe-ioerule.md)
