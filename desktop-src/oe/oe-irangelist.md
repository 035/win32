# IRangeList interface

\[**IRangeList** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Represents a range or set of ranges of messages.

## Members

The **IRangeList** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
