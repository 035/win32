# IFontCacheNotify interface

\[**IFontCacheNotify** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Supplies methods used by [**OnOptionChange**](oe-ifontcache-onoptionchange.md).

## Members

The **IFontCacheNotify** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
