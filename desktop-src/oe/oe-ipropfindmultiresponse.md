# IPropFindMultiResponse interface

\[**IPropFindMultiResponse** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

## Members

The **IPropFindMultiResponse** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
