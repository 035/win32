# IOEActions interface

\[**IOEActions** is no longer available for use as of Windows Vista.\]

If the criteria are met, the methods of the **IOEActions** interface handle the actions.

## Members

The **IOEActions** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
