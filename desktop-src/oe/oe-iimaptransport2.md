# IIMAPTransport2 interface

\[**IIMAPTransport2** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Extends the [**IIMAPTransport**](oe-iimaptransport.md) interface.

## Members

The **IIMAPTransport2** interface inherits from [**IIMAPTransport**](oe-iimaptransport.md) but does not have additional members.
