# IMimeSecurity interface

Do not use. Encodes and decodes messages using Secure/Multipurpose Internet Mail Extensions (S/MIME).

## Members

The **IMimeSecurity** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
