# INNTPTransport interface

\[**INNTPTransport** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Wraps the details of Network News Transport Protocol (NNTP) in a Component Object Model (COM) interface.

## Members

The **INNTPTransport** interface inherits from [**IInternetTransport**](oe-iinternettransport.md) but does not have additional members.
