# IHTTPMailTransport interface

\[**IHTTPMailTransport** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Transport for HTTPMail clients.

## Members

The **IHTTPMailTransport** interface inherits from [**IInternetTransport**](oe-iinternettransport.md) but does not have additional members.
