# Account Flags

Do not use. Account type flag values that can be set and their meaning.



| Constant/value                                                                                                                                                                                                                                                                      | Description                                        |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| <span id="ACCT_FLAG_NEWS"></span><span id="acct_flag_news"></span><dl> <dt>**ACCT\_FLAG\_NEWS**</dt> <dt>0x0001</dt> </dl>                                                       | Indicates a news account.<br/>               |
| <span id="ACCT_FLAG_MAIL"></span><span id="acct_flag_mail"></span><dl> <dt>**ACCT\_FLAG\_MAIL**</dt> <dt>0x0002</dt> </dl>                                                       | Indicates a mail account.<br/>               |
| <span id="ACCT_FLAG_DIR_SERV"></span><span id="acct_flag_dir_serv"></span><dl> <dt>**ACCT\_FLAG\_DIR\_SERV**</dt> <dt>0x0004</dt> </dl>                                          | Indicates a directory services account.<br/> |
| <span id="ACCT_FLAG_ALL"></span><span id="acct_flag_all"></span><dl> <dt>**ACCT\_FLAG\_ALL**</dt> <dt>(ACCT\_FLAG\_NEWS \| ACCT\_FLAG\_MAIL \| ACCT\_FLAG\_DIR\_SERV)</dt> </dl> | Indicates all account types.<br/>            |
