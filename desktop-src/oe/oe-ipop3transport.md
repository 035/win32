# IPOP3Transport interface

\[**IPOP3Transport** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Transport for Post Office Protocol version 3 (POP3) clients.

## Members

The **IPOP3Transport** interface inherits from [**IInternetTransport**](oe-iinternettransport.md) but does not have additional members.
