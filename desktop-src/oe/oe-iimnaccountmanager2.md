# IImnAccountManager2 interface

\[**IImnAccountManager2** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. Instead, use [New Handling for Account Data](oe-account-entry.md).\]

Extends [**IImnAccountManager**](oe-iimnaccountmanager.md).

## Members

The **IImnAccountManager2** interface inherits from [**IImnAccountManager**](oe-iimnaccountmanager.md) but does not have additional members.
