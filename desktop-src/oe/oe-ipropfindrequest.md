# IPropFindRequest interface

\[**IPropFindRequest** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

## Members

The **IPropFindRequest** interface inherits from [**IDAVNamespaceArbiter**](oe-idavnamespacearbiter.md) but does not have additional members.
