# ISMTPTransport2 interface

\[**ISMTPTransport2** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Extends the [**ISMTPTransport**](oe-ismtptransport.md) interface.

## Members

The **ISMTPTransport2** interface inherits from [**ISMTPTransport**](oe-ismtptransport.md) but does not have additional members.
