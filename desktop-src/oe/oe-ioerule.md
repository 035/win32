# IOERule interface

\[**IOERule** is no longer available for use as of Windows Vista.\]

The **IOERule** interface represents a message rule.

## Members

The **IOERule** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
