# IRASTransport interface

\[**IRASTransport** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Transport interface for Remote Access Service (RAS).

## Members

The **IRASTransport** interface inherits from [**IInternetTransport**](oe-iinternettransport.md) but does not have additional members.
