# IOutlookExpress interface

\[**IOutlookExpress** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Represents the application object.

## Members

The **IOutlookExpress** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
