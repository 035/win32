# IImnAccount interface

\[**IImnAccount** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. Instead, use [New Handling for Account Data](oe-account-entry.md).\]

Represents the base Internet Mail and News (IMN) account object.

## Members

The **IImnAccount** interface inherits from [**IPropertyContainer**](oe-ipropertycontainer.md) but does not have additional members.
