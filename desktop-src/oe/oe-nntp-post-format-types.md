# NNTP Post Format Types

Do not use. Network News Transport Protocol (NNTP) posting format type values that can be set, and their meaning.



| Constant/value                                                                                                                                                                                                                                | Description                                                      |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------|
| <span id="POST_USE_DEFAULT"></span><span id="post_use_default"></span><dl> <dt>**POST\_USE\_DEFAULT**</dt> <dt>0x0000</dt> </dl>           | Indicates to use the default format for sending news.<br/> |
| <span id="POST_USE_PLAIN_TEXT"></span><span id="post_use_plain_text"></span><dl> <dt>**POST\_USE\_PLAIN\_TEXT**</dt> <dt>0x0001</dt> </dl> | Indicates to use plain text format for sending news.<br/>  |
| <span id="POST_USE_HTML"></span><span id="post_use_html"></span><dl> <dt>**POST\_USE\_HTML**</dt> <dt>0x0002</dt> </dl>                    | Indicates to use HTML format for sending news.<br/>        |
