# IPersistMime interface

\[**IPersistMime** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Provides methods for loading and saving messages.

## Members

The **IPersistMime** interface inherits from [**IPersist**](https://msdn.microsoft.com/library/windows/desktop/ms688695) but does not have additional members.
