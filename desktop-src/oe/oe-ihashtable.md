# IHashTable interface

\[**IHashTable** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Provides a table of unique identifying names. Also provides the methods to operate on those table entries.

## Members

The **IHashTable** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
