# IIMAPCallback interface

\[**IIMAPCallback** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Callback interface for the Internet Message Access Protocol (IMAP) transport.

## Members

The **IIMAPCallback** interface inherits from [**ITransportCallback**](oe-itransportcallback.md) but does not have additional members.
