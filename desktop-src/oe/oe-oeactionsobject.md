# OEActions object

\[**OEActions** is no longer available for use as of Windows Vista.\]

The **OEActions** object implements the [**IOEActions**](oe-ioeactions.md) and [IPersistStream](http://msdn.microsoft.com/library/com/htm/cmi_n2p_02b1.asp) interfaces.

## Remarks
