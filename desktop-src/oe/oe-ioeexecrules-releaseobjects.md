# IOEExecRules::ReleaseObjects method

\[**IOEExecRules::ReleaseObjects** is no longer available for use as of Windows Vista.\]

Releases rule folders, rule files, and sound files.

## Syntax


```C++
HRESULT ReleaseObjects();
```



## Parameters

This method has no parameters.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S\_OK**. Otherwise, it returns an **HRESULT** error code.
