# IDatabaseSession interface

Do not use. Enables legacy Windows Mail database session operations.

## Members

The **IDatabaseSession** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
