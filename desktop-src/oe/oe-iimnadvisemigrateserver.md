# IImnAdviseMigrateServer interface

\[**IImnAdviseMigrateServer** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. Instead, use [New Handling for Account Data](oe-account-entry.md). \]

Used to receive notification when an account is migrated.

## Members

The **IImnAdviseMigrateServer** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
