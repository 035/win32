# IOECriteria interface

\[**IOECriteria** is no longer available for use as of Windows Vista.\]

The **IOECriteria** interface represents criteria used for rules and actions.

## Members

The **IOECriteria** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.
