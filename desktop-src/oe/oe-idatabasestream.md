# IDatabaseStream interface

Do not use. Enables streaming operations for legacy Windows Mail database.

## Members

The **IDatabaseStream** interface inherits from [**IStream**](https://msdn.microsoft.com/library/windows/desktop/aa380034) but does not have additional members.
