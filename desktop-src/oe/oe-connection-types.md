# Connection Types

Do not use. Connection type values that can be set, and their meaning.



| Constant/value                                                                                                                                                                                                                                                          | Description                                                          |

| <span id="CONNECTION_TYPE_LAN"></span><span id="connection_type_lan"></span><dl> <dt>**CONNECTION\_TYPE\_LAN**</dt> <dt>0x0000</dt> </dl>                            | Specifies a LAN connection.<br/>                               |
| <span id="CONNECTION_TYPE_MANUAL"></span><span id="connection_type_manual"></span><dl> <dt>**CONNECTION\_TYPE\_MANUAL**</dt> <dt>0x0001</dt> </dl>                   | Specifies a manual connection.<br/>                            |
| <span id="CONNECTION_TYPE_RAS"></span><span id="connection_type_ras"></span><dl> <dt>**CONNECTION\_TYPE\_RAS**</dt> <dt>0x0002</dt> </dl>                            | Specifies a RAS dialup connection.<br/>                        |
