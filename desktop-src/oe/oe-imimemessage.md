# IMimeMessage interface

Do not use. Creates and manipulates messages.

## Members

The **IMimeMessage** interface inherits from [**IMimeMessageTree**](oe-imimemessagetree.md) but does not have additional members.

## Remarks

The methods of [**IMimeMessageW**](oe-imimemessagew.md) support Unicode.
