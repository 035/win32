# ISMTPCallback interface

\[**ISMTPCallback** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions.\]

Callback interface for the Simple Mail Transport Protocol (SMTP) transport.

## Members

The **ISMTPCallback** interface inherits from [**ITransportCallback**](oe-itransportcallback.md) but does not have additional members.
