# CODEPAGEID

Do not use. DWORD that identifies a code page. MimeOLE defines a few constants for commonly-used code page IDs.



| Constant/value                                                                                                                                                                                                                 | Description                                                              |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------|
| <span id="CP_USASCII"></span><span id="cp_usascii"></span><dl> <dt>**CP\_USASCII**</dt> <dt>1252</dt> </dl>                 | Indicates the USASCII character set, Windows code page 1252. <br/> |
| <span id="CP_UNICODE"></span><span id="cp_unicode"></span><dl> <dt>**CP\_UNICODE**</dt> <dt>1200</dt> </dl>                 | Indicates the Unicode character set, Windows code page 1200. <br/> |
| <span id="CP_JAUTODETECT"></span><span id="cp_jautodetect"></span><dl> <dt>**CP\_JAUTODETECT**</dt> <dt>50932</dt> </dl>    | Indicates Japanese auto-detect (50932). <br/>                      |
| <span id="CP_KAUTODETECT"></span><span id="cp_kautodetect"></span><dl> <dt>**CP\_KAUTODETECT**</dt> <dt>50949</dt> </dl>    | Indicates Korean auto-detect (50949). <br/>                        |
| <span id="CP_ISO2022JPESC"></span><span id="cp_iso2022jpesc"></span><dl> <dt>**CP\_ISO2022JPESC**</dt> <dt>50221</dt> </dl> | Indicates the Internet character set ISO-2022-JP-ESC. <br/>        |
| <span id="CP_ISO2022JPSIO"></span><span id="cp_iso2022jpsio"></span><dl> <dt>**CP\_ISO2022JPSIO**</dt> <dt>50222</dt> </dl> | Indicates the Internet character set ISO-2022-JP-SIO. <br/>        |
