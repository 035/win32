# IImnEnumAccounts interface

\[**IImnEnumAccounts** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. Instead, use [New Handling for Account Data](oe-account-entry.md).\]

Used to enumerate Internet Mail and News (IMN) accounts.

## Members

The **IImnEnumAccounts** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/library/windows/desktop/ms680509) interface but does not have additional members.

## Remarks

Use [**Enumerate**](oe-iimnaccountmanager-enumerate.md) to obtain a pointer to this interface.
