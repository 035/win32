# MultiPoint Services Interfaces

MultiPoint Services includes the following interfaces.

## In this section

<dl> <dt>

[**IWmsCustomPresenterPlugin**](iwmscustompresenterplugin.md)
</dt> <dd>

Retrieves a custom input device that is connected to a MultiPoint station.

</dd> <dt>

[**IWmsPresentationDevice**](iwmspresentationdevice.md)
</dt> <dd>

Displays screen data on a monitor that is connected to the MultiPoint station.

</dd> <dt>

[**IWmsStationMousePresenter**](iwmsstationmousepresenter.md)
</dt> <dd>

Retrieves mouse cursor updates from a MultiPoint station.

</dd> <dt>

[**IWmsStationPresenter**](iwmsstationpresenter.md)
</dt> <dd>

Displays frame buffers from an input device that is connected to the MultiPoint station.

</dd> <dt>

[**IWmsStationPresenter2**](iwmsstationpresenter2.md)
</dt> <dd>

Displays frame buffers from an input device that is connected to the MultiPoint station, and manages the idle notifications for the input device.

</dd> </dl>
