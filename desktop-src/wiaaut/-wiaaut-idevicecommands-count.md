# DeviceCommands.Count property

Retrieves the number of members in the [**DeviceCommands**](-wiaaut-devicecommands.md) collection.

This property is read-only.

## Syntax


```VB
Property Count( _
)
```



## Property value

**Long** value.

## Remarks

For example code, see [Enumerate the Supported Commands](https://www.bing.com/search?q=Enumerate the Supported Commands) in Shared Samples.
