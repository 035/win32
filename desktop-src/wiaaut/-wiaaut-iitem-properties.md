# Item.Properties property

Retrieves a collection of all properties for this [**Item**](-wiaaut-item.md).

This property is read-only.

## Syntax


```VB
Property Properties( _
)
```



## Property value

[**Properties**](-wiaaut-properties.md) object.

## Remarks

For example code, see [Count the Number of Child Items Available for Transfer](https://www.bing.com/search?q=Count the Number of Child Items Available for Transfer) in Shared Samples.
