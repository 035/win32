# Formats.Count property

Retrieves the number of members in the [**Formats**](-wiaaut-formats.md) collection.

This property is read-only.

## Syntax


```VB
Property Count( _
)
```



## Property value

**Long** value.

## Remarks

For example code, see [List the Supported Transfer Formats](-wiaaut-shared-samples.md#list-the-supported-transfer-formats) in Shared Samples.
