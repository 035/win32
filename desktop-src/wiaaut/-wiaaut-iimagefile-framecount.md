# ImageFile.FrameCount property

Retrieves the number of frames in the image.

This property is read-only.

## Syntax


```VB
Property FrameCount( _
)
```



## Property value

**Long** value.

## Remarks

For example code, see [Frame Filter: Create a Multi-Page TIFF from Three Pictures](https://www.bing.com/search?q=Frame Filter: Create a Multi-Page TIFF from Three Pictures).
