# DeviceEvent.Description property

Retrieves the [**DeviceEvent**](-wiaaut-deviceevent.md) description.

This property is read-only.

## Syntax


```VB
Property Description( _
)
```



## Property value

**String** value.

For example code, see [Enumerate all the Supported Events for the Selected Device](https://www.bing.com/search?q=Enumerate all the Supported Events for the Selected Device) in Shared Samples.
