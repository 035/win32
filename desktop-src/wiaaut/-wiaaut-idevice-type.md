# Device.Type property

Retrieves the type of [**Device**](-wiaaut-device.md).

This property is read-only.

## Syntax


```VB
Property Type( _
)
```



## Property value

[**WiaDeviceType**](-wiaaut-wiadevicetype.md) value.

## Remarks

For example code, see [Determine if the Selected Device is a Camera](-wiaaut-shared-samples.md#determine-whether-the-selected-device-is-a-camera) in Shared Samples.
