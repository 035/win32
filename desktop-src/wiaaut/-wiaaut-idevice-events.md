# Device.Events property

Retrieves a collection of all events for this imaging device.

This property is read-only.

## Syntax


```VB
Property Events( _
)
```



## Property value

[**DeviceEvents**](-wiaaut-deviceevents.md) collection.

## Remarks

For example code, see [Determine the Event Type](https://www.bing.com/search?q=Determine the Event Type) in Shared Samples.
