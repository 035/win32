# ImageFile.Properties property

Retrieves an [**ImageFile**](-wiaaut-imagefile.md) collection of all properties for this image.

This property is read-only.

## Syntax


```VB
Property Properties( _
)
```



## Property value

[**Properties**](-wiaaut-properties.md) object.

## Remarks

For example code, see [Display all ImageFile Properties](-wiaaut-shared-samples.md#display-all-imagefile-properties) in Shared Samples.
