# DeviceCommand.Description property

Retrieves the [**DeviceCommand**](-wiaaut-devicecommand.md) description.

This property is read-only.

## Syntax


```VB
Property Description( _
)
```



## Property value

**String** value.

## Remarks

For example code, see [Enumerate the Supported Commands](-wiaaut-shared-samples.md#enumerate-the-supported-commands) in Shared Samples.
