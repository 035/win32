# Device.Commands property

Retrieves a collection of all commands for this imaging device.

This property is read-only.

## Syntax


```VB
Property Commands( _
)
```



## Property value

[**DeviceCommands**](-wiaaut-devicecommands.md) collection.

## Remarks

For example code, see [Enumerate Supported Commands in Commands Collection](-wiaaut-shared-samples.md#enumerate-supported-commands-in-commands-collection) in Shared Samples.
