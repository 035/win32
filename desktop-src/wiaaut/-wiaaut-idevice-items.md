# Device.Items property

Retrieves a collection of all items for this imaging device.

This property is read-only.

## Syntax


```VB
Property Items( _
)
```



## Property value

[**Items**](-wiaaut-items.md) collection.

## Remarks

For example code, see [Count Root Level Images for Transfer](https://www.bing.com/search?q=Count Root Level Images for Transfer) in Shared Samples.
