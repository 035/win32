# DeviceEvent.Type property

Retrieves the type of this [**DeviceEvent**](-wiaaut-deviceevent.md).

This property is read-only.

## Syntax


```VB
Property Type( _
)
```



## Property value

[**WiaEventFlag**](-wiaaut-wiaeventflag.md) value.

## Remarks

For example code, see [Determine the Event Type](https://www.bing.com/search?q=Determine the Event Type) in Shared Samples.
