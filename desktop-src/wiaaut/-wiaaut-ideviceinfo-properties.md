# DeviceInfo.Properties property

Retrieves a collection of all properties for this imaging device that are applicable when the device is not connected.

This property is read-only.

## Syntax


```VB
Property Properties( _
)
```



## Property value

[**Properties**](-wiaaut-properties.md) object.

## Remarks

For example code, see [List all Available Devices by Name and DeviceID](https://www.bing.com/search?q=List all Available Devices by Name and DeviceID) in Shared Samples.
