# ImageFile.HorizontalResolution property

Retrieves the horizontal pixels per inch of the image.

This property is read-only.

## Syntax


```VB
Property HorizontalResolution( _
)
```



## Property value

Double value.

## Remarks

For example code, see [Display Detailed Image Information](https://www.bing.com/search?q=Display Detailed Image Information) in Shared Samples.
