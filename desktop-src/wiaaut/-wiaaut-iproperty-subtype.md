# Property.SubType property

Retrieves the SubType of the [**Property**](-wiaaut-property.md), if any exist.

This property is read-only.

## Syntax


```VB
Property SubType( _
)
```



## Property value

[**WiaSubType**](-wiaaut-wiasubtype.md) value.

## Remarks

For example code, see [Display Detailed Property Information](-wiaaut-shared-samples.md#display-detailed-property-information) in Shared Samples.
