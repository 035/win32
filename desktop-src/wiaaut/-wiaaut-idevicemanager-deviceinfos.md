# DeviceManager.DeviceInfos property

Retrieves a collection of all imaging devices connected to this computer.

This property is read-only.

## Syntax


```VB
Property DeviceInfos( _
)
```



## Property value

[**DeviceInfos**](-wiaaut-deviceinfos.md) value.

## Remarks

For example code, see [List all Available Devices by Name and DeviceID](-wiaaut-shared-samples.md#list-all-available-devices-by-name-and-deviceid) in Shared Samples.
