# ImageProcess.FilterInfos property

Retrieves a collection of all available filters.

This property is read-only.

## Syntax


```VB
Property FilterInfos( _
)
```



## Property value

[**FilterInfos**](-wiaaut-filterinfos.md) collection.

## Remarks

For example code, see [Convert Filter: Create a Compressed JPEG File from Another File](https://www.bing.com/search?q=Convert Filter: Create a Compressed JPEG File from Another File).
