# VideoPreview.Pause property

Sets or retrieves a value indicating whether video playback is paused.

This property is read/write.

## Syntax


```VB
Property Pause( _
)
```



## Property value

**Boolean** value.

## Remarks

For example code, see [Use VideoPreview Control in HTML](-wiaaut-shared-samples.md#use-videopreview-control-in-html) in Shared Samples.
