# Vector.Item property

Sets or retrieves the specified item in the [**Vector**](-wiaaut-vector.md) by position.

This property is read/write.

## Syntax


```VB
Property Item( _
  ByVal Index As LONG _
)
```



## Property value

Required. **Long** value.

## Remarks

For example code, see [Create and Initialize a Vector Object](https://www.bing.com/search?q=Create and Initialize a Vector Object) in Shared Samples.
