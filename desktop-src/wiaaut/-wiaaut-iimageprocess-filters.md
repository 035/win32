# ImageProcess.Filters property

Retrieves a collection of the filters to be applied in this process.

This property is read-only.

## Syntax


```VB
Property Filters( _
)
```



## Property value

[**Filters**](-wiaaut-filters.md) collection.

## Remarks

For example code, see [Convert Filter: Create a Compressed JPEG File from Another File](https://www.bing.com/search?q=Convert Filter: Create a Compressed JPEG File from Another File).
