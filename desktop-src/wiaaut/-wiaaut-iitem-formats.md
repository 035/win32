# Item.Formats property

Retrieves a collection of all supported format types for this [**Item**](-wiaaut-item.md).

This property is read-only.

## Syntax


```VB
Property Formats( _
)
```



## Property value

[**Formats**](-wiaaut-formats.md) collection.

## Remarks

For example code, see [List the Supported Transfer Formats](-wiaaut-shared-samples.md#list-the-supported-transfer-formats) in Shared Samples.
