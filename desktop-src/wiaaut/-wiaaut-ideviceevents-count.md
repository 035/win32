# DeviceEvents.Count property

Retrieves the number of members in the [**DeviceEvents**](-wiaaut-deviceevents.md) collection.

This property is read-only.

## Syntax


```VB
Property Count( _
)
```



## Property value

**Long** value.

## Remarks

For example code, see [Enumerate all the Supported Events for the Selected Device](https://www.bing.com/search?q=Enumerate all the Supported Events for the Selected Device) in Shared Samples.
