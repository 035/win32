# DeviceInfo.Type property

Retrieves the type of [**DeviceInfo**](-wiaaut-deviceinfo.md) object.

This property is read-only.

## Syntax


```VB
Property Type( _
)
```



## Property value

[**WiaDeviceType**](-wiaaut-wiadevicetype.md) value.

## Remarks

For example code, see [Implement a Web Camera ASP Page](-wiaaut-shared-samples.md#implement-a-web-camera-asp-page) in Shared Samples.
