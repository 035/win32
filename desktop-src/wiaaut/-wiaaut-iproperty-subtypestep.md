# Property.SubTypeStep property

Retrieves the step increment of [**Property**](-wiaaut-property.md) values.

This property is read-only.

## Syntax


```VB
Property SubTypeStep( _
)
```



## Property value

Returns the step increment of [**Property**](-wiaaut-property.md) values if the SubType is [**RangeSubType**](-wiaaut-wiasubtype.md), otherwise generates an error.
