# ImageFile.IsIndexedPixelFormat property

Specifies whether the pixel data is an index into a palette or the actual color data.

This property is read-only.

## Syntax


```VB
Property IsIndexedPixelFormat( _
)
```



## Property value

**Boolean** value.

## Remarks

For example code, see [Display Detailed Image Information](https://www.bing.com/search?q=Display Detailed Image Information) in Shared Samples.
