# DeviceInfo.DeviceID property

Retrieves the **DeviceID (DeviceInfo)** for this [**DeviceInfo**](-wiaaut-deviceinfo.md) object.

This property is read-only.

## Syntax


```VB
Property DeviceID( _
)
```



## Property value

**String** value.

## Remarks

For example code, see [List all Available Devices by Name and DeviceID](-wiaaut-shared-samples.md#list-all-available-devices-by-name-and-deviceid) in Shared Samples.
