# DeviceEvent.EventID property

Retrieves the [**EventID Constants**](-wiaaut-consts-eventid.md) for this [**DeviceEvent**](-wiaaut-deviceevent.md).

This property is read-only.

## Syntax


```VB
Property EventID( _
)
```



## Property value

**String** value.

## Remarks

For example code, see [Enumerate all the Supported Events for the Selected Device](-wiaaut-shared-samples.md#enumerate-all-the-supported-events-for-the-selected-device) in Shared Samples.
