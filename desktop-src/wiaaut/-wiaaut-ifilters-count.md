# Filters.Count property

Retrieves the number of members in the [**Filters**](-wiaaut-filters.md) collection.

This property is read-only.

## Syntax


```VB
Property Count( _
)
```



## Property value

**Long** value.

## Remarks

For example code, see [Frame Filter: Create a Multi-Page TIFF from Three Pictures](-wiaaut-howto-use-filters.md#frame-filter-create-a-multipage-tiff-from-three-pictures).
