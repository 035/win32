# Rational.Denominator property

Sets or retrieves the [**Rational**](-wiaaut-rational.md) value **Denominator**.

This property is read/write.

## Syntax


```VB
Property Denominator( _
)
```



## Property value

**Long** value to retrieve.

## Remarks

For example code, see [Set Rational Numerator and Denominator](-wiaaut-shared-samples.md#set-rational-numerator-and-denominator) in Shared Samples.
