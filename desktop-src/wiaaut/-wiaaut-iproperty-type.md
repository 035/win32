# Property.Type property

Retrieves either a [**WiaPropertyType**](-wiaaut-wiapropertytype.md) or a [**WiaImagePropertyType**](-wiaaut-wiaimagepropertytype.md) value.

This property is read-only.

## Syntax


```VB
Property Type( _
)
```



## Property value

**Long** value.

## Remarks

For example code, see [Display all ImageFile Properties](https://www.bing.com/search?q=Display all ImageFile Properties) in Shared Samples.
