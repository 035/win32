# Device.Properties property

Retrieves a collection of all properties for this imaging device.

This property is read-only.

## Syntax


```VB
Property Properties( _
)
```



## Property value

[**Properties**](-wiaaut-properties.md) collection.

## Remarks

For example code, see [Display Detailed Property Information](https://www.bing.com/search?q=Display Detailed Property Information) in Shared Samples.
