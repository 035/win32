# Items.Count property

Retrieves the number of members in the [**Items**](-wiaaut-items.md) collection.

This property is read-only.

## Syntax


```VB
Property Count( _
)
```



## Property value

**Long** value.

## Remarks

For example code, see [Determine the Number of Items Returned by ShowSelectItems](https://www.bing.com/search?q=Determine the Number of Items Returned by ShowSelectItems) in Shared Samples.
