# ImageFile.ActiveFrame property

Sets or retrieves the current frame in the image.

This property is read/write.

## Syntax


```VB
Property ActiveFrame( _
)
```



## Property value

**Long** value.

## Remarks

For example code, see [Frame Filter: Create a Multi-Page TIFF from Three Pictures](-wiaaut-howto-use-filters.md#frame-filter-create-a-multipage-tiff-from-three-pictures).
