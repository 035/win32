# DeviceCommand.CommandID property

Retrieves the [**CommandID Constants**](-wiaaut-consts-commandid.md) for this [**DeviceCommand**](-wiaaut-devicecommand.md).

This property is read-only.

## Syntax


```VB
Property CommandID( _
)
```



## Property value

**String** value that receives the Command ID.

## Remarks

For example code, see [Enumerate Supported Commands in Commands Collection](-wiaaut-shared-samples.md#enumerate-supported-commands-in-commands-collection) in Shared Samples.
