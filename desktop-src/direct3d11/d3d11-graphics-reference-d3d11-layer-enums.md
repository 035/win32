# Layer Enumerations

This section contains information about the layer enumerations.

## 

## In this section



| Topic                                                                                             | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**D3D11\_MESSAGE\_CATEGORY**](d3d11-message-category.md)<br/>                             | Categories of debug messages. This will identify the category of a message when retrieving a message with [**ID3D11InfoQueue::GetMessage**](id3d11infoqueue-getmessage.md) and when adding a message with [**ID3D11InfoQueue::AddMessage**](id3d11infoqueue-addmessage.md). When creating an [**info queue filter**](d3d11-info-queue-filter.md), these values can be used to allow or deny any categories of messages to pass through the storage and retrieval filters.<br/> |
| [**D3D11\_MESSAGE\_ID**](d3d11-message-id.md)<br/>                                         | Debug messages for setting up an info-queue filter (see [**D3D11\_INFO\_QUEUE\_FILTER**](d3d11-info-queue-filter.md)); use these messages to allow or deny message categories to pass through the storage and retrieval filters. These IDs are used by methods such as [**ID3D11InfoQueue::GetMessage**](id3d11infoqueue-getmessage.md) or [**ID3D11InfoQueue::AddMessage**](id3d11infoqueue-addmessage.md). <br/>                                                             |
| [**D3D11\_MESSAGE\_SEVERITY**](d3d11-message-severity.md)<br/>                             | Debug message severity levels for an information queue.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**D3D11\_RLDO\_FLAGS**](d3d11-rldo-flags.md)<br/>                                         | Options for the amount of information to report about a device object's lifetime.<br/>                                                                                                                                                                                                                                                                                                                                                                                            |
| [**D3D11\_SHADER\_TRACKING\_OPTIONS**](d3d11-shader-tracking-options.md)<br/>              | Options that specify how to perform shader debug tracking.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**D3D11\_SHADER\_TRACKING\_RESOURCE\_TYPE**](d3d11-shader-tracking-resource-type.md)<br/> | Indicates which resource types to track.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                     |
