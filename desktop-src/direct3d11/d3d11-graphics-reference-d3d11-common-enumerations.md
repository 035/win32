# Common Version Enumerations

This section contains information about the common version enumerations.

## 

## In this section



| Topic                                                                                      | Description                                                                                                                                                                                                                                                                                                                                |
|--------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**D3D\_CBUFFER\_TYPE**](/windows/win32/D3DCommon/ne-d3dcommon-_d3d_cbuffer_type?branch=master)<br/>                                  | Values that identify the intended use of constant-buffer data. <br/>                                                                                                                                                                                                                                                                 |
| [**D3D\_DRIVER\_TYPE**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_driver_type?branch=master)<br/>                                    | Driver type options.<br/>                                                                                                                                                                                                                                                                                                            |
| [**D3D\_FEATURE\_LEVEL**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_feature_level?branch=master)<br/>                                | Describes the set of features targeted by a Direct3D device. <br/>                                                                                                                                                                                                                                                                   |
| [**D3D\_INCLUDE\_TYPE**](/windows/win32/D3DCommon/ne-d3dcommon-_d3d_include_type?branch=master)<br/>                                  | Values that indicate the location of a shader \#include file. <br/>                                                                                                                                                                                                                                                                  |
| [**D3D\_INTERPOLATION\_MODE**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_interpolation_mode?branch=master)<br/>                      | Specifies interpolation mode, which affects how values are calculated during rasterization.<br/>                                                                                                                                                                                                                                     |
| [**D3D\_MIN\_PRECISION**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_min_precision?branch=master)<br/>                                | Values that indicate the minimum desired interpolation precision.<br/>                                                                                                                                                                                                                                                               |
| [**D3D\_NAME**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_name?branch=master)<br/>                                                   | Values that identify shader parameters that use system-value semantics.<br/>                                                                                                                                                                                                                                                         |
| [**D3D\_PRIMITIVE**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_primitive?branch=master)<br/>                                         | Values that indicate how the pipeline interprets geometry or hull shader input primitives.<br/>                                                                                                                                                                                                                                      |
| [**D3D\_PRIMITIVE\_TOPOLOGY**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_primitive_topology?branch=master)<br/>                      | Values that indicate how the pipeline interprets vertex data that is bound to the [input-assembler stage](d3d10-graphics-programming-guide-input-assembler-stage.md). These [primitive topology values](https://msdn.microsoft.com/library/windows/desktop/bb205124) determine how the vertex data is rendered on screen.<br/> |
| [**D3D\_REGISTER\_COMPONENT\_TYPE**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_register_component_type?branch=master)<br/>           | Values that identify the data types that can be stored in a register.<br/>                                                                                                                                                                                                                                                           |
| [**D3D\_RESOURCE\_RETURN\_TYPE**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_resource_return_type?branch=master)<br/>                 | Values that identify the return type of a resource.<br/>                                                                                                                                                                                                                                                                             |
| [**D3D\_SHADER\_CBUFFER\_FLAGS**](/windows/win32/D3DCommon/ne-d3dcommon-_d3d_shader_cbuffer_flags?branch=master)<br/>                 | Values that identify the indended use of a constant-data buffer.<br/>                                                                                                                                                                                                                                                                |
| [**D3D\_SHADER\_INPUT\_FLAGS**](/windows/win32/D3DCommon/ne-d3dcommon-_d3d_shader_input_flags?branch=master)<br/>                     | Values that identify shader-input options.<br/>                                                                                                                                                                                                                                                                                      |
| [**D3D\_SHADER\_INPUT\_TYPE**](/windows/win32/D3DCommon/ne-d3dcommon-_d3d_shader_input_type?branch=master)<br/>                       | Values that identify resource types that can be bound to a shader and that are reflected as part of the resource description for the shader.<br/>                                                                                                                                                                                    |
| [**D3D\_SHADER\_VARIABLE\_CLASS**](/windows/win32/D3DCommon/ne-d3dcommon-_d3d_shader_variable_class?branch=master)<br/>               | Values that identify the class of a shader variable.<br/>                                                                                                                                                                                                                                                                            |
| [**D3D\_SHADER\_VARIABLE\_FLAGS**](/windows/win32/D3DCommon/ne-d3dcommon-_d3d_shader_variable_flags?branch=master)<br/>               | Values that identify information about a shader variable.<br/>                                                                                                                                                                                                                                                                       |
| [**D3D\_SHADER\_VARIABLE\_TYPE**](/windows/win32/D3DCommon/ne-d3dcommon-_d3d_shader_variable_type?branch=master)<br/>                 | Values that identify various data, texture, and buffer types that can be assigned to a shader variable.<br/>                                                                                                                                                                                                                         |
| [**D3D\_SRV\_DIMENSION**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_srv_dimension?branch=master)<br/>                                | Values that identify the type of resource to be viewed as a shader resource.<br/>                                                                                                                                                                                                                                                    |
| [**D3D\_TESSELLATOR\_DOMAIN**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_tessellator_domain?branch=master)<br/>                      | Values that identify domain options for tessellator data.<br/>                                                                                                                                                                                                                                                                       |
| [**D3D\_TESSELLATOR\_OUTPUT\_PRIMITIVE**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_tessellator_output_primitive?branch=master)<br/> | Values that identify output primitive types.<br/>                                                                                                                                                                                                                                                                                    |
| [**D3D\_TESSELLATOR\_PARTITIONING**](/windows/win32/D3DCommon/ne-d3dcommon-d3d_tessellator_partitioning?branch=master)<br/>          | Values that identify partitioning options.<br/>                                                                                                                                                                                                                                                                                      |
