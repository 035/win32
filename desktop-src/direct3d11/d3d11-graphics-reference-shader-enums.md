# Shader Enumerations

Enumerations are used to specify information about shaders.

## 

## In this section



| Topic                                                                                          | Description                                                        |
|------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| [**D3D11\_CBUFFER\_TYPE**](d3d11-cbuffer-type.md)<br/>                                  | Indicates a constant buffer's type.<br/>                     |
| [**D3D\_PARAMETER\_FLAGS**](d3d-parameter-flags.md)<br/>                                | Indicates semantic flags for function parameters.<br/>       |
| [**D3D11\_RESOURCE\_RETURN\_TYPE**](d3d11-resource-return-type.md)<br/>                 | Indicates return value type.<br/>                            |
| [**D3D11\_SHADER\_TYPE**](d3d11-shader-type.md)<br/>                                    | Identifies a shader type for tracing.<br/>                   |
| [**D3D11\_SHADER\_VERSION\_TYPE**](d3d11-shader-version-type.md)<br/>                   | Indicates shader type.<br/>                                  |
| [**D3D11\_TESSELLATOR\_DOMAIN**](d3d11-tessellator-domain.md)<br/>                      | Domain options for tessellator data.<br/>                    |
| [**D3D11\_TESSELLATOR\_PARTITIONING**](d3d11-tessellator-partitioning.md)<br/>          | Partitioning options.<br/>                                   |
| [**D3D11\_TESSELLATOR\_OUTPUT\_PRIMITIVE**](d3d11-tessellator-output-primitive.md)<br/> | Output primitive types.<br/>                                 |
| [**D3D11\_TRACE\_GS\_INPUT\_PRIMITIVE**](d3d11-trace-gs-input-primitive.md)<br/>        | Identifies the type of geometry shader input primitive.<br/> |
| [**D3D11\_TRACE\_REGISTER\_TYPE**](d3d11-trace-register-type.md)<br/>                   | Identifies a type of trace register.<br/>                    |
