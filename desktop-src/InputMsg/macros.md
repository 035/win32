# Macros

The topics in this section provide the reference specifications for [Pointer Input Messages and Notifications](messages-and-notifications.md) macros for retrieving information from pointer input message parameters.

## In this section



| Topic                                                                                  | Description                                                                                                                         |
|----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**GET\_POINTERID\_WPARAM**](get-pointerid-wparam.md)<br/>                      | Retrieves the pointer ID using the specified value. <br/>                                                                     |
| [**HAS\_POINTER\_CONFIDENCE\_WPARAM**](has-pointer-confidence-wparam.md)<br/>   | Checks whether the specified pointer message is considered intentional rather than accidental.<br/>                           |
| [**IS\_POINTER\_CANCELED\_WPARAM**](is-pointer-canceled-wparam.md)<br/>         | Checks whether the specified pointer input ended abruptly, or was invalid, indicating the interaction was not completed.<br/> |
| [**IS\_POINTER\_FIFTHBUTTON\_WPARAM**](is-pointer-fifthbutton-wparam.md)<br/>   | Checks whether the specified pointer took fifth action. <br/>                                                                 |
| [**IS\_POINTER\_FIRSTBUTTON\_WPARAM**](is-pointer-firstbutton-wparam.md)<br/>   | Checks whether the specified pointer took first action.<br/>                                                                  |
| [**IS\_POINTER\_FLAG\_SET\_WPARAM**](is-pointer-flag-set-wparam.md)<br/>        | Checks whether a pointer macro sets the specified flag. <br/>                                                                 |
| [**IS\_POINTER\_FOURTHBUTTON\_WPARAM**](is-pointer-fourthbutton-wparam.md)<br/> | Checks whether the specified pointer took fourth action. <br/>                                                                |
| [**IS\_POINTER\_INCONTACT\_WPARAM**](is-pointer-incontact-wparam.md)<br/>       | Checks whether the specified pointer is in contact. <br/>                                                                     |
| [**IS\_POINTER\_INRANGE\_WPARAM**](is-pointer-inrange-wparam.md)<br/>           | Checks whether the specified pointer is in range. <br/>                                                                       |
| [**IS\_POINTER\_NEW\_WPARAM**](is-pointer-new-wparam.md)<br/>                   | Checks whether the specified pointer is a new pointer. <br/>                                                                  |
| [**IS\_POINTER\_PRIMARY\_WPARAM**](is-pointer-primary-wparam.md)<br/>           | Checks whether the specified pointer is the primary pointer. <br/>                                                            |
| [**IS\_POINTER\_SECONDBUTTON\_WPARAM**](is-pointer-secondbutton-wparam.md)<br/> | Checks whether the specified pointer took second action. <br/>                                                                |
| [**IS\_POINTER\_THIRDBUTTON\_WPARAM**](is-pointer-thirdbutton-wparam.md)<br/>   | Checks whether the specified pointer took third action. <br/>                                                                 |
