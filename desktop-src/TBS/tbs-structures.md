# TBS Structures

TBS supports the following structures.

## In this section



| Topic                                                           | Description                                                                                                                                          |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TBS\_CONTEXT\_PARAMS**](/windows/win32/Tbs/ns-tbs-tdtbs_context_params?branch=master)<br/>   | Specifies the version of the TBS context implementation.<br/>                                                                                  |
| [**TBS\_CONTEXT\_PARAMS2**](/windows/win32/Tbs/ns-tbs-tdtbs_context_params2?branch=master)<br/> | Specifies the version of the TBS context implementation. You must use this structure if your application works with both versions of TPM.<br/> |
| [**TPM\_DEVICE\_INFO**](/windows/win32/Tbs/ns-tbs-_tpm_device_info?branch=master)<br/>         | Provides information about the version of the TPM.<br/>                                                                                        |
