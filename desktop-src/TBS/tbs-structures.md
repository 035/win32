# TBS Structures

TBS supports the following structures.

## In this section



| Topic                                                           | Description                                                                                                                                          |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TBS\_CONTEXT\_PARAMS**](tbs-context-params.md)<br/>   | Specifies the version of the TBS context implementation.<br/>                                                                                  |
| [**TBS\_CONTEXT\_PARAMS2**](tbs-context-params2.md)<br/> | Specifies the version of the TBS context implementation. You must use this structure if your application works with both versions of TPM.<br/> |
| [**TPM\_DEVICE\_INFO**](tpm-device-info.md)<br/>         | Provides information about the version of the TPM.<br/>                                                                                        |
