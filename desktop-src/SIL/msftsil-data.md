# MsftSil\_Data class

The **MsftSil\_Data** WMI class is an abstract base class that retrieves Software Inventory Logging data from a server.

The following syntax is simplified from Managed Object Format (MOF) code and includes all of the inherited properties.

## Syntax

``` syntax
[Abstract, ClassVersion("1.0.0"), AMENDMENT]
class MsftSil_Data
{
};
```

## Members

The **MsftSil\_Data** class does not define any members.
