# MS-CHAP Password Management Functions

Use the following functions to change user passwords using the Microsoft Challenge-Handshake Authentication Protocol (MS-CHAP).



| Function                                                     | Description                                  |
|--------------------------------------------------------------|----------------------------------------------|
| [**MSChapSrvChangePassword**](mschapsrvchangepassword.md)   | Change user password.                        |
| [**MSChapSrvChangePassword2**](mschapsrvchangepassword2.md) | Change user password with mutual encryption. |
