# MS-CHAP Password Management Functions

Use the following functions to change user passwords using the Microsoft Challenge-Handshake Authentication Protocol (MS-CHAP).



| Function                                                     | Description                                  |
|--------------------------------------------------------------|----------------------------------------------|
| [**MSChapSrvChangePassword**](/previous-versions/windows/desktop/api/MsChapp/nf-mschapp-mschapsrvchangepassword)   | Change user password.                        |
| [**MSChapSrvChangePassword2**](/previous-versions/windows/desktop/api/MsChapp/nf-mschapp-mschapsrvchangepassword2) | Change user password with mutual encryption. |
