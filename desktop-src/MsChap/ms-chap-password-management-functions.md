# MS-CHAP Password Management Functions

Use the following functions to change user passwords using the Microsoft Challenge-Handshake Authentication Protocol (MS-CHAP).



| Function                                                     | Description                                  |
|--------------------------------------------------------------|----------------------------------------------|
| [**MSChapSrvChangePassword**](/windows/previous-versions/MsChapp/nf-mschapp-mschapsrvchangepassword?branch=master)   | Change user password.                        |
| [**MSChapSrvChangePassword2**](/windows/previous-versions/MsChapp/nf-mschapp-mschapsrvchangepassword2?branch=master) | Change user password with mutual encryption. |
