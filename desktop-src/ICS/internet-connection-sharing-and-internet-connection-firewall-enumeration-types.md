# Internet Connection Sharing and Internet Connection Firewall Enumerated Types

-   [**NETCON\_STATUS**](/previous-versions/windows/desktop/api/NetCon/ne-netcon-tagnetcon_status)
-   [**NETCON\_TYPE**](/previous-versions/windows/desktop/api/NetCon/ne-netcon-tagnetcon_type)
