# IPv6 Internet Connection Firewall Interfaces

\[The IPv6 Internet Connection Firewall is available for use in the operating systems specified in the Requirements section. It is unavailable in subsequent versions. Instead, use the [Windows Firewall API](windows-firewall-start-page.md).\]

The following table contains links and descriptions for the interfaces of this API.



| Interface                                                                      | Purpose                                                                                                                                                                           |
|--------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IEnumNetFwV6Connections**](ienumnetfwv6connections.md)                     | Standard COM enumerator for the [**INetFwV6Connection**](inetfwv6connection.md) interface.                                                                                       |
| [**IEnumNetFwV6ConnectionsAsVariants**](ienumnetfwv6connectionsasvariants.md) | Standard COM enumerator for the [**INetFwV6Connection**](inetfwv6connection.md) interface which yields items of type VARIANT, as is required by the Automation collection model. |
| [**IEnumNetFwV6OpenPorts**](ienumnetfwv6openports.md)                         | Standard COM enumerator for the [**INetFWV6OpenPort**](inetfwv6openport.md) interface.                                                                                           |
| [**IEnumNetFwV6OpenPortsAsVariants**](ienumnetfwv6openportsasvariants.md)     | Standard COM enumerator for the [**INetFWV6OpenPort**](inetfwv6openport.md) interface which yields items of type VARIANT, as is required by the Automation collection model.     |
| [**INetFwV6Connection**](inetfwv6connection.md)                               | Main interface to perform configuration tasks.                                                                                                                                    |
| [**INetFwV6ConnectionCollection**](inetfwv6connectioncollection.md)           | Standard Automation collection for the [**INetFwV6Connection**](inetfwv6connection.md) interface.                                                                                |
| [**INetFwV6Mgr**](inetfwv6mgr.md)                                             | Entry point interface for IPv6 Firewall Configuration API.                                                                                                                        |
| [**INetFWV6OpenPort**](inetfwv6openport.md)                                   | Intended only for use by the open port enumerations and collections.                                                                                                              |
| [**INetFwV6OpenPortCollection**](inetfwv6openportscollection.md)              | Standard Automation collection for the [**INetFWV6OpenPort**](inetfwv6openport.md) interface.                                                                                    |
