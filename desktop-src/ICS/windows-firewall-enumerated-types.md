# Windows Firewall for Windows XP with SP2 Enumerated Types

\[The Windows Firewall API is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. For Windows Vista and later, use of the [Windows Firewall with Advanced Security](windows-firewall-advanced-security-start-page.md) API is recommended.\]

The Windows Firewall API uses the following enumerated types.

-   [**NET\_FW\_POLICY\_TYPE**](/previous-versions/windows/desktop/api/Icftypes/ne-icftypes-net_fw_policy_type_)
-   [**NET\_FW\_PROFILE\_TYPE**](/previous-versions/windows/desktop/api/Icftypes/ne-icftypes-net_fw_profile_type_)
-   [**NET\_FW\_PROFILE\_TYPE2**](/previous-versions/windows/desktop/api/Icftypes/ne-icftypes-net_fw_profile_type2_)
-   [**NET\_FW\_IP\_VERSION**](/previous-versions/windows/desktop/api/Icftypes/ne-icftypes-net_fw_ip_version_)
-   [**NET\_FW\_SCOPE**](/previous-versions/windows/desktop/api/Icftypes/ne-icftypes-net_fw_scope_)
-   [**NET\_FW\_IP\_PROTOCOL**](/previous-versions/windows/desktop/api/Icftypes/ne-icftypes-net_fw_ip_protocol_)
-   [**NET\_FW\_SERVICE\_TYPE**](/previous-versions/windows/desktop/api/Icftypes/ne-icftypes-net_fw_service_type_)
