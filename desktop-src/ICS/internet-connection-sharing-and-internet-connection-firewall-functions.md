# Internet Connection Sharing and Internet Connection Firewall Functions

\[Internet Connection Firewall may be altered or unavailable in subsequent versions. Instead, use the [Windows Firewall API](windows-firewall-start-page.md).\]

The ICS/ICF API exposes the following functions.



| Function                                                   | Purpose                                                           |
|------------------------------------------------------------|-------------------------------------------------------------------|
| [**NcFreeNetconProperties**](/previous-versions/windows/desktop/api/NetCon/nf-netcon-ncfreenetconproperties)   | Frees [**NETCON\_PROPERTIES**](/previous-versions/windows/desktop/api/NetCon/ns-netcon-tagnetcon_properties) structures. |
| [**NcIsValidConnectionName**](/previous-versions/windows/desktop/api/NetCon/nf-netcon-ncisvalidconnectionname) | Checks if connection name is valid.                               |
