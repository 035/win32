# About IPv6 Internet Connection Firewall

\[The IPv6 Internet Connection Firewall is available for use in the operating systems specified in the Requirements section. It is unavailable in subsequent versions. Instead, use the [Windows Firewall API](windows-firewall-start-page.md).\]

The IPv6 Internet Connection Firewall (IPv6 ICF) API allows applications to enumerate connections, enable or disable the IPv6 ICF on any connection, and configure port mappings.

The IPv6 ICF API makes changes only on the local computer.
