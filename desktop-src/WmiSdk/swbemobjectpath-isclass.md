# SWbemObjectPath.IsClass property

The **IsClass** property of the [**SWbemObjectPath**](swbemobjectpath.md) object is a Boolean value that indicates if this path represents a class. This property is read-only.

For an explanation of this syntax, see [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

This property is read-only.

## Syntax


```VB
SWbemObjectPath.IsClass As Boolean
```
