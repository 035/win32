# Macros

DirectWrite defines the following macros.

## In this section



| Topic                                                                      | Description                                                                                                                                                                                                                                                                                                                   |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DWRITE\_MAKE\_OPENTYPE\_TAG**](dwrite-make-opentype-tag.md)<br/> | Creates an OpenType tag as a 32-bit integer, such that the first character in the tag is the lowest byte (least significant on little endian architectures), which can be used to compare with tags in the font file. This macro is compatible with [**DWRITE\_FONT\_FEATURE\_TAG**](dwrite-font-feature-tag.md).<br/> |
