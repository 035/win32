# DWRITE\_RENDERING\_MODE enumerations

Starting in Windows 8, the **DWRITE\_RENDERING\_MODE** enumeration added new enumeration values and deprecated others.

Starting in Windows 8, the [**DWRITE\_TEXT\_ANTIALIAS\_MODE**](dwrite-text-antialias-mode.md) enumeration determines whether text is rendered using ClearType. As a result, all the ClearType rendering modes in the **DWRITE\_RENDERING\_MODE** enumeration are deprecated. These enumeration values now map to new rendering modes.

The table here shows the old enumeration values and the new values they are mapped to. For descriptions of the new values, see [**DWRITE\_RENDERING\_MODE**](dwrite-rendering-mode.md).



| Old mode                                                                                | New mode                                                                                |
|-----------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**DWRITE\_RENDERING\_MODE\_CLEARTYPE\_GDI\_CLASSIC**](dwrite-rendering-mode.md)       | [**DWRITE\_RENDERING\_MODE\_GDI\_CLASSIC**](dwrite-rendering-mode.md)                  |
| [**DWRITE\_RENDERING\_MODE\_CLEARTYPE\_GDI\_NATURAL**](dwrite-rendering-mode.md)       | [**DWRITE\_RENDERING\_MODE\_GDI\_NATURAL**](dwrite-rendering-mode.md)                  |
| [**DWRITE\_RENDERING\_MODE\_CLEARTYPE\_NATURAL**](dwrite-rendering-mode.md)            | [**DWRITE\_RENDERING\_MODE\_CLEARTYPE\_NATURAL**](dwrite-rendering-mode.md)            |
| [**DWRITE\_RENDERING\_MODE\_CLEARTYPE\_NATURAL\_SYMMETRIC**](dwrite-rendering-mode.md) | [**DWRITE\_RENDERING\_MODE\_CLEARTYPE\_NATURAL\_SYMMETRIC**](dwrite-rendering-mode.md) |
