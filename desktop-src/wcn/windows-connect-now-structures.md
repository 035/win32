# Windows Connect Now Structures

The following structures support the Windows Connect Now API functions.



| Structure                                                                             | Description                                                                                                                                            |
|---------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WCN\_VALUE\_TYPE\_PRIMARY\_DEVICE\_TYPE**](/windows/win32/wcntypes/ns-wcntypes-tagwcn_value_type_primary_device_type?branch=master) | Contains enumerated data that identifies the type of a device by category, sub-category, and a manufacturer specific OUI (Organization ID).<br/> |
| [**WCN\_VENDOR\_EXTENSION\_SPEC**](/windows/win32/Wcndevice/ns-wcndevice-tagwcn_vendor_extension_spec?branch=master)                     | Contains data that defines a vendor extension. <br/>                                                                                             |
