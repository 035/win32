# Windows Connect Now Structures

The following structures support the Windows Connect Now API functions.



| Structure                                                                             | Description                                                                                                                                            |
|---------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WCN\_VALUE\_TYPE\_PRIMARY\_DEVICE\_TYPE**](wcn-value-type-primary-device-type.md) | Contains enumerated data that identifies the type of a device by category, sub-category, and a manufacturer specific OUI (Organization ID).<br/> |
| [**WCN\_VENDOR\_EXTENSION\_SPEC**](wcn-vendor-extension-spec.md)                     | Contains data that defines a vendor extension. <br/>                                                                                             |
