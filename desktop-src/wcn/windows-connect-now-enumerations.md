# Windows Connect Now Enumerations

Windows Connect Now defines the following enumeration types.



| Enumeration                                                                                             | Description                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WCN\_ATTRIBUTE\_TYPE**](wcn-attribute-type.md)                                                      | Defines the various attribute buffer types defined for Wi-Fi Protected Setup.<br/>                                      |
| [**WCN\_PASSWORD\_TYPE**](wcn-password-type.md)                                                        | Defines the authentication that will be used in a WPS session. <br/>                                                    |
| [**WCN\_SESSION\_STATUS**](wcn-session-status.md)                                                      | Defines the status of a WPS session.<br/>                                                                               |
| [**WCN\_VALUE\_TYPE\_ASSOCIATION\_STATE**](wcn-value-type-association-state.md)                        | Defines the possible association states of a wireless station during a Discovery request.<br/>                          |
| [**WCN\_VALUE\_TYPE\_BOOLEAN**](wcn-value-type-boolean.md)                                             | Defines values used to represent true/false conditions encountered during device setup and association.<br/>            |
| [**WCN\_VALUE\_TYPE\_AUTHENTICATION\_TYPE**](wcn-value-type-authentication-type.md)                    | Defines the authentication methods supported by the Enrollee (access point or station).<br/>                            |
| [**WCN\_VALUE\_TYPE\_CONFIG\_METHODS**](wcn-value-type-config-methods.md)                              | Defines the configuration methods supported by the Enrollee or Registrar.<br/>                                          |
| [**WCN\_VALUE\_TYPE\_CONFIGURATION\_ERROR**](wcn-value-type-configuration-error.md)                    | Defines possible error values returned to a device while attempting to configure to, and associate with, the WLAN.<br/> |
| [**WCN\_VALUE\_TYPE\_DEVICE\_PASSWORD\_ID**](wcn-value-type-device-password-id.md)                     | Defines the possible password values.<br/>                                                                              |
| [**WCN\_VALUE\_TYPE\_ENCRYPTION\_TYPE**](wcn-value-type-encryption-type.md)                            | Defines the supported WLAN encryption types.<br/>                                                                       |
| [**WCN\_VALUE\_TYPE\_RF\_BANDS**](wcn-value-type-rf-bands.md)                                          | Defines the radio frequency band on which an enrollee sends Discovery requests.<br/>                                    |
| [**WCN\_VALUE\_TYPE\_VERSION**](wcn-value-type-version.md)                                             | Defines the supported version of Wi-Fi Protected Setup (WPS).<br/>                                                      |
| [**WCN\_VALUE\_TYPE\_WI\_FI\_PROTECTED\_SETUP\_STATE**](wcn-value-type-wi-fi-protected-setup-state.md) | Defines values that indicate if a device is configured.<br/>                                                            |
