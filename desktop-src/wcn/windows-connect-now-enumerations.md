# Windows Connect Now Enumerations

Windows Connect Now defines the following enumeration types.



| Enumeration                                                                                             | Description                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WCN\_ATTRIBUTE\_TYPE**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_attribute_type)                                                      | Defines the various attribute buffer types defined for Wi-Fi Protected Setup.<br/>                                      |
| [**WCN\_PASSWORD\_TYPE**](/windows/desktop/api/wcndevice/ne-wcndevice-tagwcn_password_type)                                                        | Defines the authentication that will be used in a WPS session. <br/>                                                    |
| [**WCN\_SESSION\_STATUS**](/windows/desktop/api/wcndevice/ne-wcndevice-tagwcn_session_status)                                                      | Defines the status of a WPS session.<br/>                                                                               |
| [**WCN\_VALUE\_TYPE\_ASSOCIATION\_STATE**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_value_type_association_state)                        | Defines the possible association states of a wireless station during a Discovery request.<br/>                          |
| [**WCN\_VALUE\_TYPE\_BOOLEAN**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_value_type_boolean)                                             | Defines values used to represent true/false conditions encountered during device setup and association.<br/>            |
| [**WCN\_VALUE\_TYPE\_AUTHENTICATION\_TYPE**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_value_type_authentication_type)                    | Defines the authentication methods supported by the Enrollee (access point or station).<br/>                            |
| [**WCN\_VALUE\_TYPE\_CONFIG\_METHODS**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_value_type_config_methods)                              | Defines the configuration methods supported by the Enrollee or Registrar.<br/>                                          |
| [**WCN\_VALUE\_TYPE\_CONFIGURATION\_ERROR**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_value_type_configuration_error)                    | Defines possible error values returned to a device while attempting to configure to, and associate with, the WLAN.<br/> |
| [**WCN\_VALUE\_TYPE\_DEVICE\_PASSWORD\_ID**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_value_type_device_password_id)                     | Defines the possible password values.<br/>                                                                              |
| [**WCN\_VALUE\_TYPE\_ENCRYPTION\_TYPE**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_value_type_encryption_type)                            | Defines the supported WLAN encryption types.<br/>                                                                       |
| [**WCN\_VALUE\_TYPE\_RF\_BANDS**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_value_type_rf_bands)                                          | Defines the radio frequency band on which an enrollee sends Discovery requests.<br/>                                    |
| [**WCN\_VALUE\_TYPE\_VERSION**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_value_type_version)                                             | Defines the supported version of Wi-Fi Protected Setup (WPS).<br/>                                                      |
| [**WCN\_VALUE\_TYPE\_WI\_FI\_PROTECTED\_SETUP\_STATE**](/windows/desktop/api/wcntypes/ne-wcntypes-tagwcn_value_type_wi_fi_protected_setup_state) | Defines values that indicate if a device is configured.<br/>                                                            |
