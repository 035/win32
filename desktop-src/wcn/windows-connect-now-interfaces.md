# Windows Connect Now Interfaces

The [Windows Connect Now API](portal.md) (WCNAPI) defines and utilizes the following interfaces.



| Interface                                      | Description                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](/windows/win32/WcnDevice/nn-wcndevice-iwcnconnectnotify?branch=master) | Receive a success or failure notification when a Windows Connect Now connect session completes.<br/> |
| [**IWCNDevice**](/windows/win32/WcnDevice/nn-wcndevice-iwcndevice?branch=master)               | Configures the device and initiates the session.<br/>                                                |
