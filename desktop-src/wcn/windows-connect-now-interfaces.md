# Windows Connect Now Interfaces

The [Windows Connect Now API](portal.md) (WCNAPI) defines and utilizes the following interfaces.



| Interface                                      | Description                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](iwcnconnectnotify.md) | Receive a success or failure notification when a Windows Connect Now connect session completes.<br/> |
| [**IWCNDevice**](iwcndevice.md)               | Configures the device and initiates the session.<br/>                                                |
