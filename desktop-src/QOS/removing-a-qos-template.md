# Removing a QOS Template

The process of removing a QOS template is contained within the [**WSCRemoveQOSTemplate**](wscremoveqostemplate.md) function. Note that the removal of a QOS template requires administrative privilege. For more details about the removal of a QOS template, see the **WSCRemoveQOSTemplate** reference page.
