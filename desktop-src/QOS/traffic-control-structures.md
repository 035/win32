# Traffic Control Structures

This section describes the following traffic control structures:

-   [**ADDRESS\_LIST\_DESCRIPTOR**](/windows/previous-versions/Traffic/ns-traffic-_address_list_descriptor?branch=master)
-   [**ENUMERATION\_BUFFER**](/windows/previous-versions/Traffic/ns-traffic-_enumeration_buffer?branch=master)
-   [**IP\_PATTERN**](/windows/previous-versions/Traffic/ns-traffic-_ip_pattern?branch=master)
-   [**IPX\_PATTERN**](/windows/previous-versions/Traffic/ns-traffic-_ipx_pattern?branch=master)
-   [**QOS\_DIFFSERV\_RULE**](/windows/previous-versions/QosObjs/ns-qosobjs-_qos_diffserv_rule?branch=master)
-   [**TC\_GEN\_FILTER**](/windows/previous-versions/Traffic/ns-traffic-_tc_gen_filter?branch=master)
-   [**TC\_GEN\_FLOW**](/windows/previous-versions/Traffic/ns-traffic-_tc_gen_flow?branch=master)
-   [**TC\_IFC\_DESCRIPTOR**](/windows/previous-versions/Traffic/ns-traffic-_tc_ifc_descriptor?branch=master)
-   [**TCI\_CLIENT\_FUNC\_LIST**](/windows/previous-versions/Traffic/ns-traffic-_tci_client_func_list?branch=master)

There is also a collection of structures associated with traffic control that enable application developers to query or gather statistical information regarding the packet scheduler and its traffic control faculties. That collection of statistic-enabling structures is the following:

-   [**GUID**](guid.md)
-   [**PS\_COMPONENT\_STATS**](/windows/previous-versions/Ntddpsch/ns-ntddpsch-_ps_component_stats?branch=master)
-   [**PS\_ADAPTER\_STATS**](/windows/previous-versions/Ntddpsch/ns-ntddpsch-_ps_adapter_stats?branch=master)
-   [**PS\_FLOW\_STATS**](/windows/previous-versions/Ntddpsch/ns-ntddpsch-_ps_flow_stats?branch=master)
-   [**PS\_CONFORMER\_STATS**](/windows/previous-versions/Ntddpsch/ns-ntddpsch-_ps_conformer_stats?branch=master)
-   [**PS\_SHAPER\_STATS**](/windows/previous-versions/Ntddpsch/ns-ntddpsch-_ps_shaper_stats?branch=master)
-   [**PS\_DRRSEQ\_STATS**](/windows/previous-versions/Ntddpsch/ns-ntddpsch-_ps_drrseq_stats?branch=master)

Many structures explained in this section work closely with the [**FLOWSPEC**](/windows/previous-versions/Qos/ns-qos-_flowspec?branch=master) structure.
