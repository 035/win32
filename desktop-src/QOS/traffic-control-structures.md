# Traffic Control Structures

This section describes the following traffic control structures:

-   [**ADDRESS\_LIST\_DESCRIPTOR**](address-list-descriptor.md)
-   [**ENUMERATION\_BUFFER**](enumeration-buffer.md)
-   [**IP\_PATTERN**](ip-pattern.md)
-   [**IPX\_PATTERN**](ipx-pattern.md)
-   [**QOS\_DIFFSERV\_RULE**](qos-diffserv-rule.md)
-   [**TC\_GEN\_FILTER**](tc-gen-filter.md)
-   [**TC\_GEN\_FLOW**](tc-gen-flow.md)
-   [**TC\_IFC\_DESCRIPTOR**](tc-ifc-descriptor.md)
-   [**TCI\_CLIENT\_FUNC\_LIST**](tci-client-func-list.md)

There is also a collection of structures associated with traffic control that enable application developers to query or gather statistical information regarding the packet scheduler and its traffic control faculties. That collection of statistic-enabling structures is the following:

-   [**GUID**](guid.md)
-   [**PS\_COMPONENT\_STATS**](ps-component-stats.md)
-   [**PS\_ADAPTER\_STATS**](ps-adapter-stats.md)
-   [**PS\_FLOW\_STATS**](ps-flow-stats.md)
-   [**PS\_CONFORMER\_STATS**](ps-conformer-stats.md)
-   [**PS\_SHAPER\_STATS**](ps-shaper-stats.md)
-   [**PS\_DRRSEQ\_STATS**](ps-drrseq-stats.md)

Many structures explained in this section work closely with the [**FLOWSPEC**](flowspec.md) structure.
