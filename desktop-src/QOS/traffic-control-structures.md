# Traffic Control Structures

This section describes the following traffic control structures:

-   [**ADDRESS\_LIST\_DESCRIPTOR**](/previous-versions/windows/desktop/api/Traffic/ns-traffic-_address_list_descriptor)
-   [**ENUMERATION\_BUFFER**](/previous-versions/windows/desktop/api/Traffic/ns-traffic-_enumeration_buffer)
-   [**IP\_PATTERN**](/previous-versions/windows/desktop/api/Traffic/ns-traffic-_ip_pattern)
-   [**IPX\_PATTERN**](/previous-versions/windows/desktop/api/Traffic/ns-traffic-_ipx_pattern)
-   [**QOS\_DIFFSERV\_RULE**](/previous-versions/windows/desktop/api/QosObjs/ns-qosobjs-_qos_diffserv_rule)
-   [**TC\_GEN\_FILTER**](/previous-versions/windows/desktop/api/Traffic/ns-traffic-_tc_gen_filter)
-   [**TC\_GEN\_FLOW**](/previous-versions/windows/desktop/api/Traffic/ns-traffic-_tc_gen_flow)
-   [**TC\_IFC\_DESCRIPTOR**](/previous-versions/windows/desktop/api/Traffic/ns-traffic-_tc_ifc_descriptor)
-   [**TCI\_CLIENT\_FUNC\_LIST**](/previous-versions/windows/desktop/api/Traffic/ns-traffic-_tci_client_func_list)

There is also a collection of structures associated with traffic control that enable application developers to query or gather statistical information regarding the packet scheduler and its traffic control faculties. That collection of statistic-enabling structures is the following:

-   [**GUID**](guid.md)
-   [**PS\_COMPONENT\_STATS**](/previous-versions/windows/desktop/api/Ntddpsch/ns-ntddpsch-_ps_component_stats)
-   [**PS\_ADAPTER\_STATS**](/previous-versions/windows/desktop/api/Ntddpsch/ns-ntddpsch-_ps_adapter_stats)
-   [**PS\_FLOW\_STATS**](/previous-versions/windows/desktop/api/Ntddpsch/ns-ntddpsch-_ps_flow_stats)
-   [**PS\_CONFORMER\_STATS**](/previous-versions/windows/desktop/api/Ntddpsch/ns-ntddpsch-_ps_conformer_stats)
-   [**PS\_SHAPER\_STATS**](/previous-versions/windows/desktop/api/Ntddpsch/ns-ntddpsch-_ps_shaper_stats)
-   [**PS\_DRRSEQ\_STATS**](/previous-versions/windows/desktop/api/Ntddpsch/ns-ntddpsch-_ps_drrseq_stats)

Many structures explained in this section work closely with the [**FLOWSPEC**](/previous-versions/windows/desktop/api/Qos/ns-qos-_flowspec) structure.
