# Built-in QOS Templates

The RSVP SP provides a set of built-in QOS templates that enable applications to apply well-known codecs to QOS-connection requests. The QOS templates provided are as follows:

-   G711
-   G723.1
-   G729
-   H263QCIF
-   H263CIF
-   H261QCIF
-   H261CIF
-   GSM6.10
