# System.Image.CompressedBitsPerPixel

Indicates the image compression level. . Calculated from PKEY\_Image\_CompressedBitsPerPixelNumerator and PKEY\_Image\_CompressedBitsPerPixelDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.CompressedBitsPerPixel
   shellPKey = PKEY_Image_CompressedBitsPerPixel
   formatID = 364B6FA9-37AB-482A-BE2B-AE02F60D4318
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
