# System.Devices.Dnssd.TextAttributes

Text data associated with the service instance. Each string is typically a key-value pair, separated by "=".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Dnssd.TextAttributes
   shellPKey = PKEY_Devices_Dnssd_TextAttributes
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
