# System.PropList.InfoTip

The list of properties to show in the infotip. Properties with empty values will not be displayed. Register under the regvalue of "InfoTip".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.InfoTip
   shellPKey = PKEY_PropList_InfoTip
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
