# System.Link.TargetSFGAOFlagsStrings

Expresses the SFGAO flags of a link as string values, and is used as a query optimization. See PKEY\_Shell\_SFGAOFlagsStrings for possible values of this property.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Link.TargetSFGAOFlagsStrings
   shellPKey = PKEY_Link_TargetSFGAOFlagsStrings
   formatID = D6942081-D53B-443D-AD47-5E059D9CD27A
   propID = 3
   SearchInfo
      InInvertedIndex = true
      IsColumn = false
   typeInfo
      type = Multivalue String
```

## Remarks

PKEY values are defined in Propkey.h.
