# System.GPS.DestDistanceRef

Indicates the unit used to express the distance to the destination (for example, kilometers, miles, knots).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestDistanceRef
   shellPKey = PKEY_GPS_DestDistanceRef
   formatID = ED4DF2D3-8695-450B-856F-F5C1C53ACB66
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
