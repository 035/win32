# System.Media.DateEncoded

Represents the date and time the file was encoded. The DateTime is in UTC (in the doc, not file system).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Media.DateEncoded
   shellPKey = PKEY_Media_DateEncoded
   formatID = 2E4B640D-5019-46D8-8881-55414CC5CAA0
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
```

## Windows Vista

```
propertyDescription
   name = System.Media.DateEncoded
   shellPKey = PKEY_Media_DateEncoded
   formatID = 2E4B640D-5019-46D8-8881-55414CC5CAA0
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
```

## Remarks

PKEY values are defined in Propkey.h.
