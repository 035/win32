# System.GPS.DestLatitudeDenominator

The denominator of PKEY\_GPS\_DestLatitude.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestLatitudeDenominator
   shellPKey = PKEY_GPS_DestLatitudeDenominator
   formatID = 3A372292-7FCA-49A7-99D5-E47BB2D4E7AB
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue UInt32
```

PKEY values are defined in Propkey.h.
