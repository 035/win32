# System.GPS.DestLongitudeDenominator

The denominator of PKEY\_GPS\_DestLongitude.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestLongitudeDenominator
   shellPKey = PKEY_GPS_DestLongitudeDenominator
   formatID = 425D69E5-48AD-4900-8D80-6EB6B8D0AC86
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue UInt32
```

PKEY values are defined in Propkey.h.
