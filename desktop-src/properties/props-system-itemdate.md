# System.ItemDate

The primary date of interest for an item. In the case of photos, for example, this property maps to [System.Photo.DateTaken](https://www.bing.com/search?q=System.Photo.DateTaken).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.ItemDate
   shellPKey = PKEY_ItemDate
   formatID = F7DB74B4-4287-4103-AFBA-F1B13DCD75CF
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.ItemDate
   shellPKey = PKEY_ItemDate
   formatID = F7DB74B4-4287-4103-AFBA-F1B13DCD75CF
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
