# System.Devices.Dnssd.PortNumber

Port number on which the service is listening.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Dnssd.PortNumber
   shellPKey = PKEY_Devices_Dnssd_PortNumber
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
