# System.Devices.AepService.Bluetooth.GattService.Device

Sets the Bluetooth device address to query.

## Windows 10, version 1511

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.GattService.Device
   shellPKey = PKEY_Devices_AepService_Bluetooth_GattService_Device
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
