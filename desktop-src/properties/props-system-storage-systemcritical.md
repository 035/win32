# System.Storage.SystemCritical

Indicates if the storage media is system critical.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Storage.SystemCritical
   shellPKey = PKEY_Storage_SystemCritical
   formatID = 4D1EBEE8-0803-4774-9842-B77DB50265E9
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## Remarks

PKEY values are defined in Propkey.h.
