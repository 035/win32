# Audio

## In this section



| Topic                                                                                        | Description                                                                                                 |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [System.Audio.ChannelCount](properties.props_system_audio_channelcount)<br/>           | Indicates the channel count for the audio file. Possible values are 1 for mono and 2 for stereo.<br/> |
| [System.Audio.Compression](properties.props_system_audio_compression)<br/>             | Indicates the audio compression used on the audio file.<br/>                                          |
| [System.Audio.EncodingBitrate](properties.props_system_audio_encodingbitrate)<br/>     | Indicates the average data rate in Hertz (Hz) for the audio file in bits per second.<br/>             |
| [System.Audio.Format](properties.props_system_audio_format)<br/>                       | Indicates the format of the audio file.<br/>                                                          |
| [System.Audio.IsVariableBitRate](properties.props_system_audio_isvariablebitrate)<br/> | Indicates whether the audio file had a variable or constant bit rate.<br/>                            |
| [System.Audio.PeakValue](properties.props_system_audio_peakvalue)<br/>                 |                                                                                                             |
| [System.Audio.SampleRate](properties.props_system_audio_samplerate)<br/>               | Indicates the sample rate for the audio file in samples per second.<br/>                              |
| [System.Audio.SampleSize](properties.props_system_audio_samplesize)<br/>               | Indicates the sample size for the audio file in bits per sample.<br/>                                 |
| [System.Audio.StreamName](properties.props_system_audio_streamname)<br/>               | Identifies the name of the stream for the audio file.<br/>                                            |
| [System.Audio.StreamNumber](properties.props_system_audio_streamnumber)<br/>           | Identifies the stream number of the audio file.<br/>                                                  |
