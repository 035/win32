# System.AppUserModel.ToastActivatorCLSID

Used to CoCreate an INotificationActivationCallback interface to notify about toast activations.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.AppUserModel.ToastActivatorCLSID
   shellPKey = PKEY_AppUserModel_ToastActivatorCLSID
   formatID = 9F4C2855-9F79-4B39-A8D0-E1D42DE1D5F3
   propID = 26
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
