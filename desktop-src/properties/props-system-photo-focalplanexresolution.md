# System.Photo.FocalPlaneXResolution

Indicates the number of pixels in the image width (X direction) per FocalPlaneResolutionUnit on the camera focal plane. Calculated from PKEY\_Photo\_FocalPlaneXResolutionNumerator and PKEY\_Photo\_FocalPlaneXResolutionDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FocalPlaneXResolution
   shellPKey = PKEY_Photo_FocalPlaneXResolution
   formatID = CFC08D97-C6F7-4484-89DD-EBEF4356FE76
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
