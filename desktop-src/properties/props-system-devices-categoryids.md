# System.Devices.CategoryIds

Indicates the actual raw category

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.CategoryIds
   shellPKey = PKEY_Devices_CategoryIds
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 90
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
