# System.Communication.FollowupIconIndex

The icon index used on messages marked for followup.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Communication.FollowupIconIndex
   shellPKey = PKEY_Communication_FollowupIconIndex
   formatID = 83A6347E-6FE4-4F40-BA9C-C4865240D1F4
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
```

## Remarks

PKEY values are defined in Propkey.h.
