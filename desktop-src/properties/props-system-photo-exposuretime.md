# System.Photo.ExposureTime

The exposure time for the photo, in seconds, as read from the Exchangeable Image File (EXIF) information. This property is calculated from [System.Photo.ExposureTimeNumerator](https://www.bing.com/search?q=System.Photo.ExposureTimeNumerator) and [System.Photo.ExposureTimeDenominator](https://www.bing.com/search?q=System.Photo.ExposureTimeDenominator).

The following is a list of possible values taken from the EXIF 2.2 specification.

-   30
-   15
-   8
-   4
-   2
-   1
-   1/2
-   1/4
-   1/8
-   1/15
-   1/30
-   1/60
-   1/125
-   1/250
-   1/500
-   1/1000
-   1/2000

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ExposureTime
   shellPKey = PKEY_Photo_ExposureTime
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 33434
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
