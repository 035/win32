# System.GPS.ImgDirectionNumerator

The numerator of PKEY\_GPS\_Img Direction.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.ImgDirectionNumerator
   shellPKey = PKEY_GPS_ImgDirectionNumerator
   formatID = DC5877C7-225F-45F7-BAC7-E81334B6130A
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
```
