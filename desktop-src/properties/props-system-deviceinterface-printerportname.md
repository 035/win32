# System.DeviceInterface.PrinterPortName

The port where the printer is located.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DeviceInterface.PrinterPortName
   shellPKey = PKEY_DeviceInterface_PrinterPortName
   formatID = EEC7B761-6F94-41B1-949F-C729720DD13C
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
