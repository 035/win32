# System.PropList.PreviewTitle

The one or two properties to display in the preview pane title section. The optional second property is displayed as a subtitle. Register under the regvalue of "PreviewTitle".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.PreviewTitle
   shellPKey = PKEY_PropList_PreviewTitle
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
