# System.GPS.SpeedRef

Indicates the unit used to express the speed of the GPS receiver movement, (for example,kilometers per hour, miles per hour, knots).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.SpeedRef
   shellPKey = PKEY_GPS_SpeedRef
   formatID = ECF7F4C9-544F-4D6D-9D98-8AD79ADAF453
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
