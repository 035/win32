# System.Video.TotalBitrate

Indicates the total data rate in "bits per second" for all video and audio streams.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Video.TotalBitrate
   shellPKey = PKEY_Video_TotalBitrate
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 43
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
