# System.Search.QueryPropertyHits

Contains the list of matched properties from a query.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.Search.QueryPropertyHits
   shellPKey = PKEY_Search_QueryPropertyHits
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 21
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
