# System.Photo.LensManufacturer

String indicating the manufacturer of the lens used to take the picture. Can be blank or not present.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.LensManufacturer
   shellPKey = PKEY_Photo_LensManufacturer
   formatID = E6DDCAF7-29C5-4F0A-9A68-D19412EC7090
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
