# System.Devices.LaunchDeviceStageFromExplorer

Indicates whether to launch Device Stage.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.LaunchDeviceStageFromExplorer
   shellPKey = PKEY_Devices_LaunchDeviceStageFromExplorer
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 77
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
