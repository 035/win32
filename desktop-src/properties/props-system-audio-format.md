# System.Audio.Format

Indicates the format of the audio file.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Audio.Format
   shellPKey = PKEY_Audio_Format
   formatID = 64440490-4C8B-11D1-8B70-080036B11A03
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Windows 7, Windows Vista

```
propertyDescription
   name = System.Audio.Format
   shellPKey = PKEY_Audio_Format
   formatID = 64440490-4C8B-11D1-8B70-080036B11A03
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
