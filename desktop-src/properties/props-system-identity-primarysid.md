# System.Identity.PrimarySid

The primary SID of an identity.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.PrimarySid
   shellPKey = PKEY_Identity_PrimarySid
   formatID = 2B1B801E-C0C1-4987-9EC5-72FA89814787
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
