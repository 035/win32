# System.Photo.FlashModel

String indicating the model of the flash used to take the picture. Can be blank or not present.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FlashModel
   shellPKey = PKEY_Photo_FlashModel
   formatID = FE83BB35-4D1A-42E2-916B-06F3E1AF719E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
