# System.Devices.WiFiDirect.InformationElements

Indicates full set of IEs provided by the Wi-Fi Direct Device

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.InformationElements
   shellPKey = PKEY_Devices_WiFiDirect_InformationElements
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
