# System.DRM.IsProtected

Indicates whether the file is protected under Digital Rights Management (DRM).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DRM.IsProtected
   shellPKey = PKEY_DRM_IsProtected
   formatID = AEAC19E4-89AE-4508-B9B7-BB867ABEE2ED
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.DRM.IsProtected
   shellPKey = PKEY_DRM_IsProtected
   formatID = AEAC19E4-89AE-4508-B9B7-BB867ABEE2ED
   propID = 2
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
