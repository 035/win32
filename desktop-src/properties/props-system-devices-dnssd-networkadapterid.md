# System.Devices.Dnssd.NetworkAdapterId

GUID for the network adapter on which to search for a service.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Dnssd.NetworkAdapterId
   shellPKey = PKEY_Devices_Dnssd_NetworkAdapterId
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
