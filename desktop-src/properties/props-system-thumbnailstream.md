# System.ThumbnailStream

Data that represents the thumbnail in VT\_STREAM format, supported by Windows GDI+ and Windows codecs such as .jpg and .png.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.ThumbnailStream
   shellPKey = PKEY_ThumbnailStream
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 27
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Stream
      IsInnate = false
```

## Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ThumbnailStream
   shellPKey = PKEY_ThumbnailStream
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 27
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Stream
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
