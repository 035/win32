# System.Photo.TranscodedForSync

A VT\_BOOL that indicates whether the image has been transcoded for synchronizing with an external device.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.TranscodedForSync
   shellPKey = PKEY_Photo_TranscodedForSync
   formatID = 9A8EBB75-6458-4E82-BACB-35C0095B03BB
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## Remarks

PKEY values are defined in Propkey.h.
