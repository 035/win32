# System.Devices.AepContainer.DialProtocol.InstalledApplications

List of applications supporting DIAL protocol on the Device Association End Point Container

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.AepContainer.DialProtocol.InstalledApplications
   shellPKey = PKEY_Devices_AepContainer_DialProtocol_InstalledApplications
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
```
