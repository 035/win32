# System.GPS.DestDistance

Indicates the distance to the destination point. Calculated from PKEY\_GPS\_DestDistanceNumerator and PKEY\_GPS\_DestDistanceDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestDistance
   shellPKey = PKEY_GPS_DestDistance
   formatID = A93EAE04-6804-4F24-AC81-09B266452118
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
