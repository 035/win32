# System.Devices.LocalMachine

If true, the device in question is the computer.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.LocalMachine
   shellPKey = PKEY_Devices_LocalMachine
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 70
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Windows 7

```
propertyDescription
   name = System.Devices.LocalMachine
   shellPKey = PKEY_Devices_IsLocalMachine
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 70
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
