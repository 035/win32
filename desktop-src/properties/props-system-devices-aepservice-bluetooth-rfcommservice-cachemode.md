# System.Devices.AepService.Bluetooth.RfcommService.CacheMode

Sets the Bluetooth RFCOMM cache mode for the query.

## Windows 10, version 1511

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.RfcommService.CacheMode
   shellPKey = PKEY_Devices_AepService_Bluetooth_RfcommService_CacheMode
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
