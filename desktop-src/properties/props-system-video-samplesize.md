# System.Video.SampleSize

Indicates the sample size in bits for the video stream. "SampleSize".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Video.SampleSize
   shellPKey = PKEY_Video_SampleSize
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.Video.SampleSize
   shellPKey = PKEY_Video_SampleSize
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 9
   SearchInfo
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
