# System.Media.DlnaProfileID

The DLNA profile ID for media content, defined by DLNA standards.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Media.DlnaProfileID
   shellPKey = PKEY_Media_DlnaProfileID
   formatID = CFA31B45-525D-4998-BB44-3F7D81542FA4
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
```
