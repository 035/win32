# System.Identity.Blob

Blob used to import and export identities.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Identity.Blob
   shellPKey = PKEY_Identity_Blob
   formatID = 8C3B93A4-BAED-1A83-9A32-102EE313F6EB
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Blob
```

## Remarks

PKEY values are defined in Propkey.h.
