# System.Devices.AepService.ProtocolId

Identity of the protocol this Device Association Endpoint Service was discovered over

```
propertyDescription
   name = System.Devices.AepService.ProtocolId
   shellPKey = PKEY_Devices_AepService_ProtocolId
   formatID = C9C141A9-1B4C-4F17-A9D1-F298538CADB8
   propID = 5
   SearchInfo
      InInvertedIndex = false
```
