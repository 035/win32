# System.Video.Orientation

This is the video orientation in degrees.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Video.Orientation
   shellPKey = PKEY_Video_Orientation
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 99
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

PKEY values are defined in Propkey.h.
