# System.ItemUrl

Represents a well-formed URL that points to the item.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ItemUrl
   shellPKey = PKEY_ItemUrl
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 9
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.

To reference Shell namespace items through Shell APIs, use [System.ParsingPath](shell.props_System_ParsingPath).

Example values:



| Item type | Example                        |
|-----------|--------------------------------|
| File      | file:///c:/mydir/bar/hello.txt |
| File      | csc://{GUID}/...               |
| Message   | mapi://...                     |
