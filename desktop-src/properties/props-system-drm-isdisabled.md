# System.DRM.IsDisabled

Indicates whether the media file has been disabled by DRM.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.DRM.IsDisabled
   shellPKey = PKEY_DRM_IsDisabled
   formatID = AEAC19E4-89AE-4508-B9B7-BB867ABEE2ED
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## Remarks

PKEY values are defined in Propkey.h.
