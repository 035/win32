# System.Contact.HomeAddressCountry

Home address country name of the contact.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.HomeAddressCountry
   shellPKey = PKEY_Contact_HomeAddressCountry
   formatID = 08A65AA1-F4C9-43DD-9DDF-A33D8E7EAD85
   propID = 100

      IsColumn = true
```
