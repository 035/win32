# System.Media.SeriesName

A name that represents a specific series, such as a podcast or recorded television series.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Media.SeriesName
   shellPKey = PKEY_Media_SeriesName
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 42
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```

PKEY values are defined in Propkey.h.
