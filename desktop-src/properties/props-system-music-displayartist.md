# System.Music.DisplayArtist

This property returns the best representation of the album artist for a specific music file based upon [System.Music.AlbumArtist](shell.props_System_Music_AlbumArtist), [System.Music.Artist](shell.props_System_Music_Artist), and [System.Music.IsCompilation](shell.props_System_Music_IsCompilation) information.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Music.DisplayArtist
   shellPKey = PKEY_Music_DisplayArtist
   formatID = FD122953-FA93-4EF7-92C3-04C946B2F7C8
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
