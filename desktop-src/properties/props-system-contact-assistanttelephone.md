# System.Contact.AssistantTelephone

Telephone number of the contact's assistant.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.AssistantTelephone
   shellPKey = PKEY_Contact_AssistantTelephone
   formatID = 9A93244D-A7AD-4FF8-9B99-45EE4CC09AF6
   propID = 100
```
