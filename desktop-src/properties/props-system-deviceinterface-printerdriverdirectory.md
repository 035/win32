# System.DeviceInterface.PrinterDriverDirectory

The directory location for the printer driver.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DeviceInterface.PrinterDriverDirectory
   shellPKey = PKEY_DeviceInterface_PrinterDriverDirectory
   formatID = 847C66DE-B8D6-4AF9-ABC3-6F4F926BC039
   propID = 14
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
