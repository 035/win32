# System.Devices.SafeRemovalRequired

Indicates whether a device requires safe removal.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.SafeRemovalRequired
   shellPKey = PKEY_Devices_SafeRemovalRequired
   formatID = AFD97640-86A3-4210-B67C-289C41AABE55
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
