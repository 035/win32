# System.Devices.Aep.Bluetooth.Le.IsConnectable

Whether the Bluetooth LE device is currently advertising a connectable advertisement.

## Windows 10, version 1703

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Le.IsConnectable
   shellPKey = PKEY_Devices_Aep_Bluetooth_Le_IsConnectable
   formatID = 995EF0B0-7EB3-4A8B-B9CE-068BB3F4AF69
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
