# System.IsLocationSupported

Identifies whether a location was indexed (locally or remotely) at the time it was added to the library.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.IsLocationSupported
   shellPKey = PKEY_IsLocationSupported
   formatID = 5D76B67F-9B3D-44BB-B6AE-25DA4F638A67
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## Remarks

PKEY values are defined in Propkey.h.
