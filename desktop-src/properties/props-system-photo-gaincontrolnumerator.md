# System.Photo.GainControlNumerator

The numerator of PKEY\_Photo\_GainControl.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.GainControlNumerator
   shellPKey = PKEY_Photo_GainControlNumerator
   formatID = 8E8ECF7C-B7B8-4EB8-A63F-0EE715C96F9E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
```

PKEY values are defined in Propkey.h.
