# System.Computer.DecoratedFreeSpace

The computer's free space and total space stated as "%s free of %s".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Computer.DecoratedFreeSpace
   shellPKey = PKEY_Computer_DecoratedFreeSpace
   formatID = 9B174B35-40FF-11D2-A27E-00C04FC30871
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue UInt64
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
