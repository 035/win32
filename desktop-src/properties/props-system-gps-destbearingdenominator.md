# System.GPS.DestBearingDenominator

The denominator of PKEY\_GPS\_DestBearing.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestBearingDenominator
   shellPKey = PKEY_GPS_DestBearingDenominator
   formatID = 7ABCF4F8-7C3F-4988-AC91-8D2C2E97ECA5
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
```
