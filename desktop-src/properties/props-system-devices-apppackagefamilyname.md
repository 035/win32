# System.Devices.AppPackageFamilyName

The package family name registered as the app for this device.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.AppPackageFamilyName
   shellPKey = PKEY_Devices_AppPackageFamilyName
   formatID = 51236583-0C4A-4FE8-B81F-166AEC13F510
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
