# System.Devices.Aep.ModelName

Device Association Endpoint's Model Name

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Aep.ModelName
   shellPKey = PKEY_Devices_Aep_ModelName
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 3
   SearchInfo
      InInvertedIndex = false
```
