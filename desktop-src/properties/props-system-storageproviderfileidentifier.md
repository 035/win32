# System.StorageProviderFileIdentifier

The storage provider identifier for this file.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.StorageProviderFileIdentifier
   shellPKey = PKEY_StorageProviderFileIdentifier
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
