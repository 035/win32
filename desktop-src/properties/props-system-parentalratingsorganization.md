# System.ParentalRatingsOrganization

The name of the organization whose rating system is used for [System.ParentalRating](https://www.bing.com/search?q=System.ParentalRating).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRatingsOrganization
   shellPKey = PKEY_ParentalRatingsOrganization
   formatID = A7FE0840-1344-46F0-8D37-52ED712A4BF9
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
