# System.Devices.Dnssd.InstanceName

Instance portion of DNS-SD service instance name.(e.g. "myservice" in "myservice.\_http.\_tcp.local")

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Dnssd.InstanceName
   shellPKey = PKEY_Devices_Dnssd_InstanceName
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
