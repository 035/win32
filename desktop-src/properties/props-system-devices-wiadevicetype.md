# System.Devices.WiaDeviceType

Windows Image Acquisition (WIA) device type.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.Devices.WiaDeviceType
   shellPKey = PKEY_Devices_WiaDeviceType
   formatID = 6BDD1FC6-810F-11D0-BEC7-08002BE2092F
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
