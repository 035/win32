# System.Contact.PrimaryAddressCountry

Primary address country name of the contact.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.PrimaryAddressCountry
   shellPKey = PKEY_Contact_PrimaryAddressCountry
   formatID = E53D799D-0F3F-466E-B2FF-74634A3CB7A4
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
