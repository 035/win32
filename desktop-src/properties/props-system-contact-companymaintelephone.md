# System.Contact.CompanyMainTelephone

Indicates the main telephone number of the contact's company.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.CompanyMainTelephone
   shellPKey = PKEY_Contact_CompanyMainTelephone
   formatID = 8589E481-6040-473D-B171-7FA89C2708ED
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
