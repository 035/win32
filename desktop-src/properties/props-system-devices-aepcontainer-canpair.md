# System.Devices.AepContainer.CanPair

Whether one of the child Device Association Endpoints can be paired with the system or not.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription

   shellPKey = PKEY_Devices_AepContainer_CanPair

   propID = 3

      IsColumn = false
```
