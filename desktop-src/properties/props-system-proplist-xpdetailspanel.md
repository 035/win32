# System.PropList.XPDetailsPanel

Obsolete. The list of properties to display in the XP webview details panel.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.XPDetailsPanel
   shellPKey = PKEY_PropList_XPDetailsPanel
   formatID = F2275480-F782-4291-BD94-F13693513AEC
   propID = 0
```
