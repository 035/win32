# System.IdentityProvider.Picture

Picture for the identity provider.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.IdentityProvider.Picture
   shellPKey = PKEY_IdentityProvider_Picture
   formatID = 2425166F-5642-4864-992F-98FD98F294C3
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
```
