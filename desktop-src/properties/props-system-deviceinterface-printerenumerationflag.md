# System.DeviceInterface.PrinterEnumerationFlag

Printer information Printer Enumeration Flag.

```
propertyDescription
   name = System.DeviceInterface.PrinterEnumerationFlag
   shellPKey = PKEY_DeviceInterface_PrinterEnumerationFlag
```
