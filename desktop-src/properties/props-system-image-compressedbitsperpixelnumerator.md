# System.Image.CompressedBitsPerPixelNumerator

The numerator of PKEY\_Image\_CompressedBitsPerPixel.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.CompressedBitsPerPixelNumerator
   shellPKey = PKEY_Image_CompressedBitsPerPixelNumerator
   formatID = D21A7148-D32C-4624-8900-277210F79C0F
   propID = 100
```
