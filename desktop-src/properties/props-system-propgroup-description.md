# System.PropGroup.Description

The property group separator used in property lists to separate descriptions from other types. Do not use this property for getting or setting values. This property is intended only as a marker.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Description
   shellPKey = PKEY_PropGroup_Description
   formatID = 8969B275-9475-4E00-A887-FF93B8B41E44
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## Remarks

PKEY values are defined in Propkey.h.
