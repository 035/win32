# Property Bag Functions

This section describes a set of Windows helper functions used with [**IPropertyBag**](_inet_IPropertyBag_Interface_cpp) objects.



| Topic                                                                       | Contents                                                                                                                     |
|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [**PSPropertyBag\_Delete**](shell.PSPropertyBag_Delete)                     | Deletes a property from a property bag.<br/>                                                                           |
| [**PSPropertyBag\_ReadBOOL**](shell.PSPropertyBag_ReadBOOL)                 | Reads the **BOOL** data value of a property in a property bag.<br/>                                                    |
| [**PSPropertyBag\_ReadBSTR**](shell.PSPropertyBag_ReadBSTR)                 | Reads a **BSTR** data value from a property in a property bag.<br/>                                                    |
| [**PSPropertyBag\_ReadDWORD**](shell.PSPropertyBag_ReadDWORD)               | Reads a **DWORD** data value from property in a property bag.<br/>                                                     |
| [**PSPropertyBag\_ReadGUID**](shell.PSPropertyBag_ReadGUID)                 | Reads the GUID data value from a property in a property bag.<br/>                                                      |
| [**PSPropertyBag\_ReadInt**](shell.PSPropertyBag_ReadInt)                   | Reads an **int** data value from a property in a property bag.<br/>                                                    |
| [**PSPropertyBag\_ReadLONG**](shell.PSPropertyBag_ReadLONG)                 | Reads a **LONG** data value from a property in a property bag.<br/>                                                    |
| [**PSPropertyBag\_ReadPOINTL**](shell.PSPropertyBag_ReadPOINTL)             | Retrieves the property coordinates stored in a [**POINTL**](gdi.pointl) structure of a specified property bag.<br/>    |
| [**PSPropertyBag\_ReadPOINTS**](shell.PSPropertyBag_ReadPOINTS)             | Retrieves the property coordinates stored in a [**POINTS**](gdi.points) structure of a specified property bag.<br/>    |
| [**PSPropertyBag\_ReadPropertyKey**](shell.PSPropertyBag_ReadPropertyKey)   | Reads the property key of a property in a specified property bag.<br/>                                                 |
| [**PSPropertyBag\_ReadRECTL**](shell.PSPropertyBag_ReadRECTL)               | Retrieves the coordinates of a rectangle stored in a property contained in a specified property bag.<br/>              |
| [**PSPropertyBag\_ReadSHORT**](shell.PSPropertyBag_ReadSHORT)               | Reads the **SHORT** data value of a property in a property bag.<br/>                                                   |
| [**PSPropertyBag\_ReadStr**](shell.PSPropertyBag_ReadStr)                   | Reads the **string** data value of a property in a property bag.<br/>                                                  |
| [**PSPropertyBag\_ReadStrAlloc**](shell.PSPropertyBag_ReadStrAlloc)         | Reads a **string** data value from a property in a property bag and allocates memory for the string that is read.<br/> |
| [**PSPropertyBag\_ReadStream**](shell.PSPropertyBag_ReadStream)             | Reads the data stream stored in a given property contained in a specified property bag.<br/>                           |
| [**PSPropertyBag\_ReadType**](shell.PSPropertyBag_ReadType)                 | Reads the type of data value of a property that is stored in a property bag.<br/>                                      |
| [**PSPropertyBag\_ReadULONGLONG**](shell.PSPropertyBag_ReadULONGLONG)       | Reads a **ULONGLONG** data value from a property in a property bag.<br/>                                               |
| [**PSPropertyBag\_ReadUnknown**](shell.PSPropertyBag_ReadUnknown)           | Reads a given property of an unknown data value in a property bag.<br/>                                                |
| [**PSPropertyBag\_WriteBOOL**](shell.PSPropertyBag_WriteBOOL)               | Sets the **BOOL** data value of a property in a property bag.<br/>                                                     |
| [**PSPropertyBag\_WriteBSTR**](shell.PSPropertyBag_WriteBSTR)               | Sets a **BSTR** data value from a property in a property bag.<br/>                                                     |
| [**PSPropertyBag\_WriteDWORD**](shell.PSPropertyBag_WriteDWORD)             | Sets a **DWORD** data value from property in a property bag.<br/>                                                      |
| [**PSPropertyBag\_WriteGUID**](shell.PSPropertyBag_WriteGUID)               | Sets the GUID data value from a property in a property bag.<br/>                                                       |
| [**PSPropertyBag\_WriteInt**](shell.PSPropertyBag_WriteInt)                 | Sets an **int** data value from a property in a property bag.<br/>                                                     |
| [**PSPropertyBag\_WriteLONG**](shell.PSPropertyBag_WriteLONG)               | Sets a **LONG** data value from a property in a property bag.<br/>                                                     |
| [**PSPropertyBag\_WritePOINTL**](shell.PSPropertyBag_WritePOINTL)           | Stores the property coordinates stored in a [**POINTL**](gdi.pointl) structure of a specified property bag.<br/>       |
| [**PSPropertyBag\_WritePOINTS**](shell.PSPropertyBag_WritePOINTS)           | Stores the property coordinates stored in a [**POINTS**](gdi.points) structure of a specified property bag.<br/>       |
| [**PSPropertyBag\_WritePropertyKey**](shell.PSPropertyBag_WritePropertyKey) | Sets the property key of a property in a specified property bag.<br/>                                                  |
| [**PSPropertyBag\_WriteRECTL**](shell.PSPropertyBag_WriteRECTL)             | Stores the coordinates of a rectangle stored in a property contained in a specified property bag.<br/>                 |
| [**PSPropertyBag\_WriteSHORT**](shell.PSPropertyBag_WriteSHORT)             | Sets the **SHORT** data value of a property in a property bag.<br/>                                                    |
| [**PSPropertyBag\_WriteStr**](shell.PSPropertyBag_WriteStr)                 | Sets the **string** data value of a property in a property bag.<br/>                                                   |
| [**PSPropertyBag\_WriteStream**](shell.PSPropertyBag_WriteStream)           | Sets the data stream stored in a given property contained in a specified property bag.<br/>                            |
| [**PSPropertyBag\_WriteULONGLONG**](shell.PSPropertyBag_WriteULONGLONG)     | Sets a **ULONGLONG** data value from a property in a property bag.<br/>                                                |
| [**PSPropertyBag\_WriteUnknown**](shell.PSPropertyBag_WriteUnknown)         | Sets a given property of an unknown data value in a property bag.<br/>                                                 |
