# System.StorageProviderFileVersion

The storage provider file version for this file.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.StorageProviderFileVersion
   shellPKey = PKEY_StorageProviderFileVersion
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
