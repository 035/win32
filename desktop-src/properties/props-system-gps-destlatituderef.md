# System.GPS.DestLatitudeRef

Indicates whether the latitude destination point is north or south latitude.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestLatitudeRef
   shellPKey = PKEY_GPS_DestLatitudeRef
   formatID = CEA820B9-CE61-4885-A128-005D9087C192
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
