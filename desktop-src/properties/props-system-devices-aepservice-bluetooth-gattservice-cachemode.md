# System.Devices.AepService.Bluetooth.GattService.CacheMode

Sets the Bluetooth Gatt cache mode for the query.

## Windows 10, version 1511

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.GattService.CacheMode
   shellPKey = PKEY_Devices_AepService_Bluetooth_GattService_CacheMode
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
