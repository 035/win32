# System.Image.Dimensions

The image dimensions in string format as `horizontal pixels x vertical pixels`. For example, `3080x2100`.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.Dimensions
   shellPKey = PKEY_Image_Dimensions
   formatID = 6444048F-4C8B-11D1-8B70-080036B11A03
   propID = 13
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
