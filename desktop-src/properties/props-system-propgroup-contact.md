# System.PropGroup.Contact

The property group separator used in property lists to separate contacts from other types. Do not use this property for getting or setting values. This property is intended only as a marker.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Contact
   shellPKey = PKEY_PropGroup_Contact
   formatID = DF975FD3-250A-4004-858F-34E29A3E37AA
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## Remarks

PKEY values are defined in Propkey.h.
