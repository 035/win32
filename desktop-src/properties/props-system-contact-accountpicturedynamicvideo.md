# System.Contact.AccountPictureDynamicVideo

This is a stream containing the data necessary to render a contact's dynamic video account picture.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Contact.AccountPictureDynamicVideo
   shellPKey = PKEY_Contact_AccountPictureDynamicVideo
   formatID = 0B8BB018-2725-4B44-92BA-7933AEB2DDE7
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Stream
```

## Remarks

PKEY values are defined in Propkey.h.
