# System.GPS.SpeedDenominator

The denominator of PKEY\_GPS\_Speed.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.SpeedDenominator
   shellPKey = PKEY_GPS_SpeedDenominator
   formatID = 7D122D5A-AE5E-4335-8841-D71E7CE72F53
   propID = 100
   SearchInfo
```
