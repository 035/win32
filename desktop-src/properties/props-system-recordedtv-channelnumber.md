# System.RecordedTV.ChannelNumber

The recorded TV channels. For example, 42, 5, 53.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.RecordedTV.ChannelNumber
   shellPKey = PKEY_RecordedTV_ChannelNumber
   formatID = 6D748DE2-8D38-4CC3-AC60-F009B057C557
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## Windows Vista

```
propertyDescription
   name = System.RecordedTV.ChannelNumber
   shellPKey = PKEY_RecordedTV_ChannelNumber
   formatID = 6D748DE2-8D38-4CC3-AC60-F009B057C557
```

PKEY values are defined in Propkey.h.
