# System.GPS.DOP

Indicates the GPS DOP (data degree of precision). Calculated from PKEY\_GPS\_DOPNumerator and PKEY\_GPS\_DOPDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DOP
   shellPKey = PKEY_GPS_DOP
   formatID = 0CF8FB02-1837-42F1-A697-A7017AA289B9
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
