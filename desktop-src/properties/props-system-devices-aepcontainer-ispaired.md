# System.Devices.AepContainer.IsPaired

Whether one of the child Device Association Endpoints is paired with the system or not.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.AepContainer.IsPaired
   shellPKey = PKEY_Devices_AepContainer_IsPaired
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
