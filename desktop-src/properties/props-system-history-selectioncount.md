# System.History.SelectionCount

The count of instances the user has selected the item.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.History.SelectionCount
   shellPKey = PKEY_History_SelectionCount
   formatID = 1CE0D6BC-536C-4600-B0DD-7E0C66B350D5
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## Remarks

PKEY values are defined in Propkey.h.
