# System.Identity.InternetSid

The internet SID of an identity.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.InternetSid
   shellPKey = PKEY_Identity_InternetSid
   formatID = 6D6D5D49-265D-4688-9F4E-1FDD33E7CC83
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
