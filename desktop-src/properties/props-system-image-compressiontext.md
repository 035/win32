# System.Image.CompressionText

The user-friendly form of System.Image.Compression. Not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.CompressionText
   shellPKey = PKEY_Image_CompressionText
   formatID = 3F08E66F-2F44-4BB9-A682-AC35D2562322
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
