# System.Photo.SaturationText

The user-friendly form of [System.Photo.Saturation](https://www.bing.com/search?q=System.Photo.Saturation). Not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.SaturationText
   shellPKey = PKEY_Photo_SaturationText
   formatID = 61478C08-B600-4A84-BBE4-E99C45F0A072
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
