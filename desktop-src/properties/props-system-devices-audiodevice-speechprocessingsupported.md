# System.Devices.AudioDevice.SpeechProcessingSupported

Speech mode support for the audio device. If VARIANT\_TRUE the device supports speech mode.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.AudioDevice.SpeechProcessingSupported
   shellPKey = PKEY_Devices_AudioDevice_SpeechProcessingSupported
   formatID = FB1DE864-E06D-47F4-82A6-8A0AEF44493C
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
