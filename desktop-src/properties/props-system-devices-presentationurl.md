# System.Devices.PresentationUrl

URL of a human readable webpage on the device.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.PresentationUrl
   shellPKey = PKEY_Devices_PresentationUrl
   formatID = 656A3BB3-ECC0-43FD-8477-4AE0404A96CD
   propID = 8198
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```
