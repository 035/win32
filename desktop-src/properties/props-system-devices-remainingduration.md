# System.Devices.RemainingDuration

The remaining playing time on the device, in 100ns units.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.RemainingDuration
   shellPKey = PKEY_Devices_RemainingDuration
   formatID = 3633DE59-6825-4381-A49B-9F6BA13A1471
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
