# System.Devices.AepContainer.SupportsImages

Indicates if the Device Association Endpoint Container Supports Image Casting

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.AepContainer.SupportsImages
   shellPKey = PKEY_Devices_AepContainer_SupportsImages
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
