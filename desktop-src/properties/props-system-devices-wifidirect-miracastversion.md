# System.Devices.WiFiDirect.MiracastVersion

Indicates version of Miracast protocol if Wi-Fi Direct Device is Miracast capable

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.MiracastVersion
   shellPKey = PKEY_Devices_WiFiDirect_MiracastVersion
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
