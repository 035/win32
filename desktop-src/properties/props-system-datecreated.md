# System.DateCreated

The date and time the item was created on the file system where it is currently located. This property is automatically promoted by the file system. The Indexing Service friendly name is "create".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DateCreated
   shellPKey = PKEY_DateCreated
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 15
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.DateCreated
   shellPKey = PKEY_DateCreated
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 15
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
