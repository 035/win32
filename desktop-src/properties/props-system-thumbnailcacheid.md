# System.ThumbnailCacheId

A unique value used as a key to cache thumbnails. The value changes when the name, volume, or data modified of an item changes.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ThumbnailCacheId
   shellPKey = PKEY_ThumbnailCacheId
   formatID = 446D16B1-8DAD-4870-A748-402EA43D788C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt64
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
