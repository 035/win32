# System.Devices.InterfaceEnabled

Indicates if the interface is enabled or not.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.InterfaceEnabled
   shellPKey = PKEY_Devices_InterfaceEnabled
   formatID = 026E516E-B814-414B-83CD-856D6FEF4822
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
