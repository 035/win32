# System.PropList.PreviewDetails

The list of properties to display in the preview pane. Register under the regvalue of "PreviewDetails".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.PreviewDetails
   shellPKey = PKEY_PropList_PreviewDetails
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
