# System.Photo.FocalLengthInFilm

The focal length of the lens when the photo was taken, as converted to a 35mm film measurement.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FocalLengthInFilm
   shellPKey = PKEY_Photo_FocalLengthInFilm
   formatID = A0E74609-B84D-4F49-B860-462BD9971F98
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt16
```

## Remarks

PKEY values are defined in Propkey.h.
