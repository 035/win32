# System.Devices.Dnssd.HostName

DNS name of device is hosting the service.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Dnssd.HostName
   shellPKey = PKEY_Devices_Dnssd_HostName
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
