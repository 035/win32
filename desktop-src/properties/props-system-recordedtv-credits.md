# System.RecordedTV.Credits

Indicates the credits for the program, in the following format. "Actor1/Actor2/Actor3...;Director1/Director2/Director3...;Host1/Host2/Host3...;GuestStar".

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.RecordedTV.Credits
   shellPKey = PKEY_RecordedTV_Credits
   formatID = 6D748DE2-8D38-4CC3-AC60-F009B057C557
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
