# System.Photo.ExposureIndexNumerator

The numerator of PKEY\_Photo\_ExposureIndex.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ExposureIndexNumerator
   shellPKey = PKEY_Photo_ExposureIndexNumerator
   formatID = CDEDCF30-8919-44DF-8F4C-4EB2FFDB8D89
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
```
