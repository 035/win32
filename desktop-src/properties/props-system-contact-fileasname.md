# System.Contact.FileAsName

Indicates the FileAs name of the contact.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.FileAsName
   shellPKey = PKEY_Contact_FileAsName
   formatID = F1A24AA7-9CA7-40F6-89EC-97DEF9FFE8DB
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
