# System.GPS.LongitudeDecimal

Indicates the longitude based on the reference in PKEY\_GPS\_LongitudeRef. Calculated from PKEY\_GPS\_LongitudeNumerator and PKEY\_GPS\_LongitudeDenominator

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.GPS.LongitudeDecimal
   shellPKey = PKEY_GPS_LongitudeDecimal
   formatID = 4679C1B5-844D-4590-BAF5-F322231F1B81
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
