# System.Contact.OtherAddressCity

Other address city name of the contact.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.OtherAddressCity
   shellPKey = PKEY_Contact_OtherAddressCity
   formatID = 6E682923-7F7B-4F0C-A337-CFCA296687BF
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
