# System.StatusBarSelectedItemCount

The count of selected items in the view and the estimated total size.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.StatusBarSelectedItemCount
   shellPKey = PKEY_StatusBarSelectedItemCount
   formatID = 26DC287C-6E3D-4BD3-B2B0-6A26BA2E346D
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```
