# System.DeviceInterface.WinUsb.UsbProductId

Product ID from the USB device's USB Device Descriptor.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.WinUsb.UsbProductId
   shellPKey = PKEY_DeviceInterface_WinUsb_UsbProductId
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
```

## Remarks

PKEY values are defined in Propkey.h.
