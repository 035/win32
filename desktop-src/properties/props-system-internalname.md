# System.InternalName

The name of a .exe or .dll file as stored in a resource section within that file.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.InternalName
   shellPKey = PKEY_InternalName
   formatID = 0CEF7D53-FA64-11D1-A203-0000F81FEDEE
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.

Historically, this information was often displayed in the Windows XPÂ **Version** property sheet page. Under Windows Vista, the information was moved to the **Details** tab. However, this particular property was not made public in Windows Vista until Service Pack 1 (SP1).
