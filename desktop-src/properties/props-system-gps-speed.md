# System.GPS.Speed

Indicates the speed of the GPS receiver movement. Calculated from PKEY\_GPS\_SpeedNumerator and PKEY\_GPS\_SpeedDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.Speed
   shellPKey = PKEY_GPS_Speed
   formatID = DA5D0862-6E76-4E1B-BABD-70021BD25494
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
