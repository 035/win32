# System.Video.FrameWidth

Indicates the frame width for the video stream.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Video.FrameWidth
   shellPKey = PKEY_Video_FrameWidth
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.Video.FrameWidth
   shellPKey = PKEY_Video_FrameWidth
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 3
   SearchInfo
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
