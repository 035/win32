# System.GPS.Status

Indicates the status of the GPS receiver when the image was recorded (for example, measurement in progress, measurement interoperability).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.Status
   shellPKey = PKEY_GPS_Status
   formatID = 125491F4-818F-46B2-91B5-D537753617B2
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
