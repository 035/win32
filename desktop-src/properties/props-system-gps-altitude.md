# System.GPS.Altitude

Indicates altitude based on the reference in PKEY\_GPS\_AltitudeRef. Calculated from PKEY\_GPS\_AltitudeNumerator and PKEY\_GPS\_AltitudeDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.Altitude
   shellPKey = PKEY_GPS_Altitude
   formatID = 827EDB4F-5B73-44A7-891D-FDFFABEA35CA
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.

The requirement of a specific indirect string reference for the `label` attribute of **labelInfo** was added for Windows Vista with Service Pack 1 (SP1).
