# System.Photo.FlashManufacturer

A string indicating the manufacturer of the flash used to take the picture. Can be blank or not present.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FlashManufacturer
   shellPKey = PKEY_Photo_FlashManufacturer
   formatID = AABAF6C9-E0C5-4719-8585-57B103E584FE
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
