# System.Calendar.RequiredAttendeeAddresses

Addresses of the all the required attendees.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Calendar.RequiredAttendeeAddresses
   shellPKey = PKEY_Calendar_RequiredAttendeeAddresses
   formatID = 0BA7D6C3-568D-4159-AB91-781A91FB71E5
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```
