# System.ItemNameDisplayWithoutExtension

This is similar to System.ItemNameDisplay except that it never includes a file extension.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.ItemNameDisplayWithoutExtension
   shellPKey = PKEY_ItemNameDisplayWithoutExtension
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 24
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
