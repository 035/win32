# System.Devices.Notifications.StorageFullLinkText

Link text for the device storage-full notification.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Notifications.StorageFullLinkText
   shellPKey = PKEY_Devices_Notifications_StorageFullLinkText
   formatID = A0E00EE1-F0C7-4D41-B8E7-26A7BD8D38B0
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enumRange
            name = StorageFullLinkText
            minValue = 0
            setValue = 0
            text = Browse device storage...
```
