# System.Devices.Notifications.StorageFull

Device storage-full notification.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Notifications.StorageFull
   shellPKey = PKEY_Devices_Notifications_StorageFull
   formatID = A0E00EE1-F0C7-4D41-B8E7-26A7BD8D38B0
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enumRange
            name = StorageFullNotification
            minValue = 0
            setValue = 0
            text = Your device is almost full. Delete files to create free space.
```
