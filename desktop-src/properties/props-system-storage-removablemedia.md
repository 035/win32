# System.Storage.RemovableMedia

Indicates if the storage media is removable.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Storage.RemovableMedia
   shellPKey = PKEY_Storage_RemovableMedia
   formatID = 4D1EBEE8-0803-4774-9842-B77DB50265E9
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## Remarks

PKEY values are defined in Propkey.h.
