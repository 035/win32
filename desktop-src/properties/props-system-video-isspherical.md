# System.Video.IsSpherical

Indicates whether the media file has a spherical video stream.

## Windows 10, version 1703

```
propertyDescription
   name = System.Video.IsSpherical
   shellPKey = PKEY_Video_IsSpherical
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
