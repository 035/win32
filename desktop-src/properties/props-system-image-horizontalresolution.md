# System.Image.HorizontalResolution

Indicates the number of pixels per resolution unit in the image width.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Image.HorizontalResolution
   shellPKey = PKEY_Image_HorizontalResolution
   formatID = 6444048F-4C8B-11D1-8B70-080036B11A03
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.Image.HorizontalResolution
   shellPKey = PKEY_Image_HorizontalResolution
   formatID = 6444048F-4C8B-11D1-8B70-080036B11A03
   propID = 5
   SearchInfo
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
