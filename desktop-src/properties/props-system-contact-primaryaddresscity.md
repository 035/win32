# System.Contact.PrimaryAddressCity

Primary address city name of the contact.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.PrimaryAddressCity
   shellPKey = PKEY_Contact_PrimaryAddressCity
   formatID = C8EA94F0-A9E3-4969-A94B-9C62A95324E0
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
