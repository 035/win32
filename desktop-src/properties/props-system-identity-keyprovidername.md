# System.Identity.KeyProviderName

Identity key provider name

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.KeyProviderName
   shellPKey = PKEY_Identity_KeyProviderName
   formatID = A26F4AFC-7346-4299-BE47-EB1AE613139F
   propID = 16
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.
