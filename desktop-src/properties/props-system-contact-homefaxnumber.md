# System.Contact.HomeFaxNumber

Home fax number of the contact.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.HomeFaxNumber
   shellPKey = PKEY_Contact_HomeFaxNumber
   formatID = 660E04D6-81AB-4977-A09F-82313113AB26
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
