# System.Photo.DigitalZoom

The digital zoom ratio when the image was shot. Read from the camera in the file's Exchangeable Image File (EXIF) information. This property is calculated from [System.Photo.DigitalZoomNumerator](https://www.bing.com/search?q=System.Photo.DigitalZoomNumerator) and [System.Photo.DigitalZoomDenominator](https://www.bing.com/search?q=System.Photo.DigitalZoomDenominator). If the numerator of the recorded value is 0, a digital zoom was not used.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.DigitalZoom
   shellPKey = PKEY_Photo_DigitalZoom
   formatID = F85BF840-A925-4BC2-B0C4-8E36B598679E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
