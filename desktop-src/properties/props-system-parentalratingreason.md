# System.ParentalRatingReason

Explains file ratings. For example, "Violence, Foul Language" for a rating of R under the MPAA rating system.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRatingReason
   shellPKey = PKEY_ParentalRatingReason
   formatID = 10984E0A-F9F2-4321-B7EF-BAF195AF4319
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
