# System.Storage.Portable

Indicates if the drive for the storage is portable.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Storage.Portable
   shellPKey = PKEY_Storage_Portable
   formatID = 4D1EBEE8-0803-4774-9842-B77DB50265E9
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## Remarks

PKEY values are defined in Propkey.h.
