# System.IsAttachment

Identifies whether the item is an attachment.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.IsAttachment
   shellPKey = PKEY_IsAttachment
   formatID = F23F425C-71A1-4FA8-922F-678EA4A60408
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## Windows Vista

```
propertyDescription
   name = System.IsAttachment
   shellPKey = PKEY_IsAttachment
   formatID = F23F425C-71A1-4FA8-922F-678EA4A60408
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
