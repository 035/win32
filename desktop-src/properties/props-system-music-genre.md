# System.Music.Genre

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Music.Genre
   shellPKey = PKEY_Music_Genre
   formatID = 56A3372E-CE9C-11D2-9F0E-006097C686F6
   propID = 11
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## Remarks

PKEY values are defined in Propkey.h.

Note that some file formats, such as a .mp3 file, only support a single genre. Others, such as a .wma files, support multiple genres. If you specify multiple genres for a single-genre file format, only the first genre in the list will be recognized.
