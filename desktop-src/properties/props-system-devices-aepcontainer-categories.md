# System.Devices.AepContainer.Categories

Categories the device is part of. e.g. Printer, Camera, etc.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.AepContainer.Categories
   shellPKey = PKEY_Devices_AepContainer_Categories
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
