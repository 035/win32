# System.Communication.HeaderItem

This property is true if the item is a header item, which means that the item has not been fully downloaded.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Communication.HeaderItem
   shellPKey = PKEY_Communication_HeaderItem
   formatID = C9C34F84-2241-4401-B607-BD20ED75AE7F
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## Remarks

PKEY values are defined in Propkey.h.
