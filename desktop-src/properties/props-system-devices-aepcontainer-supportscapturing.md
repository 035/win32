# System.Devices.AepContainer.SupportsCapturing

Indicates if the Device Association Endpoint Container Supports Capturing

## Windows 10, version 1703, Windows 10, version 1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsCapturing
   shellPKey = PKEY_Devices_AepContainer_SupportsCapturing
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
