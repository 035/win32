# System.Devices.Aep.SignalStrength

Signal strength of the device. Only applicable for some protocols.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Aep.SignalStrength
   shellPKey = PKEY_Devices_Aep_SignalStrength
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Int32
      IsInnate = true
```

PKEY values are defined in Propkey.h.
