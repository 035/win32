# System.SensitivityText

The user-friendly form of [System.Sensitivity](shell.props_System_Sensitivity). This value is not intended to be parsed programmatically.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.SensitivityText
   shellPKey = PKEY_SensitivityText
   formatID = D0C7F054-3F72-4725-8527-129A577CB269
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
