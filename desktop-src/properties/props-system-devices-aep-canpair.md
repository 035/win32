# System.Devices.Aep.CanPair

Whether the Device Association Endpoint can be paired with the system or not

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Aep.CanPair
   shellPKey = PKEY_Devices_Aep_CanPair
   formatID = E7C3FB29-CAA7-4F47-8C8B-BE59B330D4C5
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
