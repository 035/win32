# System.Message.Flags

Flags associated with e-mail messages (identifying that a read receipt is pending, for example). The values stored here by Outlook are defined for PR\_MESSAGE\_FLAGS.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Message.Flags
   shellPKey = PKEY_Message_Flags
   formatID = A82D9EE7-CA67-4312-965E-226BCEA85023
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = false
```

## Remarks

PKEY values are defined in Propkey.h.
