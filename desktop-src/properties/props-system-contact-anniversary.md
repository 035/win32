# System.Contact.Anniversary

Anniversary of the contact.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Contact.Anniversary
   shellPKey = PKEY_Contact_Anniversary
   formatID = 9AD5BADB-CEA7-4470-A03D-B84E51B9949E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
```
