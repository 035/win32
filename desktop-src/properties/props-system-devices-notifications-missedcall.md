# System.Devices.Notifications.MissedCall

Device missed call notification.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Notifications.MissedCall
   shellPKey = PKEY_Devices_Notifications_MissedCall
   formatID = 6614EF48-4EFE-4424-9EDA-C79F404EDF3E
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Byte
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enumRange
            name = MissedCallNotification
            minValue = 0
            setValue = 0
            text = You missed a call.
```
