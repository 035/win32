# System.Photo.GainControlText

The user-friendly form of [System.Photo.GainControl](https://www.bing.com/search?q=System.Photo.GainControl). Not intended to be parsed programmatically.

```
propertyDescription
   name = System.Photo.GainControlText
   shellPKey = PKEY_Photo_GainControlText

   propID = 100
```
