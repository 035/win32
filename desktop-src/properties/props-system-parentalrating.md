# System.ParentalRating

The parental rating stored in a format typically determined by the organization named in [System.ParentalRatingsOrganization](https://www.bing.com/search?q=System.ParentalRatingsOrganization).

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRating
   shellPKey = PKEY_ParentalRating
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 21
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## Remarks

PKEY values are defined in Propkey.h.

This string is displayed directly to the user, so user-friendly wordings should be used. For example, "R-Restricted" when basing ratings on the Motion Picture Association of America (MPAA) rating system.
