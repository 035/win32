# System.Devices.WiFiDirect.IsVisible

Indicates Wi-Fi Direct Device's Current Visibility

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.IsVisible
   shellPKey = PKEY_Devices_WiFiDirect_IsVisible
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
