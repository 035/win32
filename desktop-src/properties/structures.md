# Structures

This section describes the Windows Property System structures.



| Topic                                | Contents                                                                                                                                    |
|--------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**PROPERTYKEY**](shell.PROPERTYKEY) | Specifies the FMTID/PID identifier that programmatically identifies a property. Replaces [**SHCOLUMNID**](shell.SHCOLUMNID_str).<br/> |
| [**PROPPRG**](propprg.md)           | This structure contains information from a .pif file. It is used by [**PifMgr\_GetProperties**](pifmgr-getproperties.md).<br/>       |
