# Structures

This section describes the Windows Property System structures.



| Topic                                | Contents                                                                                                                                    |
|--------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**PROPERTYKEY**](shell.PROPERTYKEY) | Specifies the FMTID/PID identifier that programmatically identifies a property. Replaces [**SHCOLUMNID**](shell.SHCOLUMNID_str).<br/> |
| [**PROPPRG**](/windows/win32/Shlobj_core/ns-shlobj_core-propprg?branch=master)           | This structure contains information from a .pif file. It is used by [**PifMgr\_GetProperties**](/windows/win32/Shlobj_core/nf-shlobj_core-pifmgr_getproperties?branch=master).<br/>       |
