# System.GPS.Track

Indicates the direction of the GPS receiver movement. Calculated from PKEY\_GPS\_TrackNumerator and PKEY\_GPS\_TrackDenominator.

## Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.Track
   shellPKey = PKEY_GPS_Track
   formatID = 76C09943-7C33-49E3-9E7E-CDBA872CFADA
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## Remarks

PKEY values are defined in Propkey.h.
