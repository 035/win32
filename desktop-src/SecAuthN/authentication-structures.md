## SSPI Structures

The following structures, defined in Sspi.h, are used in SSPI functions.



| Structure                                                                   | Description                                                                                                                                                                                                                                            |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CREDSSP\_CRED**](/windows/win32/Credssp/ns-credssp-_credssp_cred?branch=master)                                       | Specifies authentication data for both Schannel and Negotiate [*security packages*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-package-gly)                                                                                      |
| [**SEC\_WINNT\_AUTH\_IDENTITY**](/windows/win32/Rpcdce/ns-sspi-_sec_winnt_auth_identity_a?branch=master)               | Used to pass a particular user name and password to the run-time library for the purpose of authentication.                                                                                                                                            |
| [**SEC\_WINNT\_AUTH\_IDENTITY\_EX**](/windows/win32/Sspi/ns-sspi-_sec_winnt_auth_identity_exa?branch=master)        | Contains information about a user. Both an ANSI and [*Unicode*](https://msdn.microsoft.com/library/windows/desktop/ms721629#-security-unicode-gly) form of this structure are provided.                                                                                       |
| [**SecBuffer**](/windows/win32/Sspi/ns-sspi-_secbuffer?branch=master)                                              | Buffer allocated by a transport application to pass to a [*security package*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-package-gly).                                                                                           |
| [**SecBufferDesc**](/windows/win32/Sspi/ns-sspi-_secbufferdesc?branch=master)                                      | Array of [**SecBuffer**](/windows/win32/Sspi/ns-sspi-_secbuffer?branch=master) structures to pass from a transport application to a security package.                                                                                                                                         |
| [**SecPkgContext\_AccessToken**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_accesstoken?branch=master)             | Contains a handle to the access token of the [*security context*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-context-gly).                                                                                                       |
| [**SecPkgContext\_ClientCreds**](/windows/win32/Credssp/ns-credssp-_secpkgcontext_clientcreds?branch=master)             | Specifies client credentials when calling the [**QueryContextAttributes (CredSSP)**](/windows/win32/Sspi/nf-sspi-querycontextattributesa?branch=master) function.                                                                                                                   |
| [**SecPkgContext\_ConnectionInfo**](/windows/win32/Schannel/ns-schannel-_secpkgcontext_connectioninfo?branch=master)       | Contains protocol and cipher information. This structure is used by the [**QueryContextAttributes (General)**](/windows/win32/Sspi/?branch=master) function.                                                                                         |
| [**SecPkgContext\_CredentialName**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_credentialnamea?branch=master)       | Specifies the credential name.                                                                                                                                                                                                                         |
| [**SecPkgContext\_DceInfo**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_dceinfo?branch=master)                     | Contains authorization data used by DCE services.                                                                                                                                                                                                      |
| [**SecPkgContext\_EapKeyBlock**](/windows/win32/Schannel/ns-schannel-_secpkgcontext_eapkeyblock?branch=master)             | Contains key data used by the EAP TLS Authentication Protocol.                                                                                                                                                                                         |
| [**SecPkgContext\_Flags**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_flags?branch=master)                         | Contains information about the flags in the security context.                                                                                                                                                                                          |
| [**SecPkgContext\_IssuerListInfoEx**](/windows/win32/Schannel/ns-schannel-_secpkgcontext_issuerlistinfoex?branch=master)   | Contains a list of trusted [*certification authorities*](https://msdn.microsoft.com/library/windows/desktop/ms721572#-security-certification-authority-gly) (CAs).                                                                                            |
| [**SecPkgContext\_Lifespan**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_lifespan?branch=master)                   | Indicates the life span of a security context.                                                                                                                                                                                                         |
| [**SecPkgContext\_Names**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_namesa?branch=master)                         | Contains the name of the user associated with a security context.                                                                                                                                                                                      |
| [**SecPkgContext\_NativeNames**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_nativenamesa?branch=master)             | Contains the client and server principal names from the outbound ticket.                                                                                                                                                                               |
| [**SecPkgContext\_NegotiationInfo**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_negotiationinfoa?branch=master)     | Contains information about the [*security package*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-package-gly) that is being set up or has been set up. It also gives the status on the negotiation to set up the security package. |
| [**SecPkgContext\_PackageInfo**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_packageinfoa?branch=master)             | Contains the name of a [*security support provider*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-support-provider-gly) (SSP).                                                                                            |
| [**SecPkgContext\_PasswordExpiry**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_passwordexpiry?branch=master)       | Contains information about the expiration of a password or other credential.                                                                                                                                                                           |
| [**SecPkgContext\_SessionKey**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_sessionkey?branch=master)               | Contains information about the session key.                                                                                                                                                                                                            |
| [**SecPkgContext\_Sizes**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_sizes?branch=master)                         | Contains the sizes of important structures used in the message support functions.                                                                                                                                                                      |
| [**SecPkgContext\_StreamSizes**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_streamsizes?branch=master)             | Contains the sizes of the various stream attributes for use with the message support functions.                                                                                                                                                        |
| [**SecPkgContext\_TargetInformation**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_targetinformation?branch=master) | Contains information about the credential used for the security context.                                                                                                                                                                               |
| [**SecPkgCredentials\_Names**](/windows/win32/Sspi/ns-sspi-_secpkgcredentials_namesa?branch=master)                 | Holds the name of the user associated with a [*context*](https://msdn.microsoft.com/library/windows/desktop/ms721572#-security-context-gly).                                                                                                                                  |
| [**SecPkgInfo**](/windows/win32/Sspi/ns-sspi-_secpkginfoa?branch=master)                                            | Provides general information about a [*security package*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-package-gly), such as its name and capabilities.                                                                            |
| [**SECURITY\_INTEGER**](/windows/win32/Sspi/ns-sspi-_security_integer?branch=master)                               | Structure to hold a numeric value. It is used in defining other types.                                                                                                                                                                                 |
| [**SecurityFunctionTable**](/windows/win32/Sspi/ns-sspi-_security_function_table_a?branch=master)                      | Dispatch table that contains pointers to the functions defined in SSPI.                                                                                                                                                                                |



 

## Schannel Structures

The following structures are defined for use with Schannel.



| Structure                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                        |
|---------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCH\_CRED\_PUBLIC\_CERTCHAIN**](/windows/win32/Schannel/ns-schannel-_sch_cred_public_certchain?branch=master)         | Contains a single certificate. A certification chain can be built from this certificate.                                                                                                                                                                                                                                                                                           |
| [**SCH\_CRED\_SECRET\_PRIVKEY**](/windows/win32/Schannel/ns-schannel-_sch_cred_secret_privkey?branch=master)             | Contains [*private key*](https://msdn.microsoft.com/library/windows/desktop/ms721603#-security-private-key-gly) information needed to authenticate a client or server.                                                                                                                                                                                                                                |
| [**SCHANNEL\_CERT\_HASH**](/windows/win32/Schannel/ns-schannel-_schannel_cert_hash?branch=master)                        | Contains the hash store data for the certificate that Schannel uses.                                                                                                                                                                                                                                                                                                               |
| [**SCHANNEL\_CERT\_HASH\_STORE**](/windows/win32/Schannel/ns-schannel-_schannel_cert_hash_store?branch=master)           | Contains the hash store data for the certificate that Schannel uses in kernel-mode.                                                                                                                                                                                                                                                                                                |
| [**SCHANNEL\_ALERT\_TOKEN**](/windows/win32/Schannel/ns-schannel-_schannel_alert_token?branch=master)                    | Generates a [Secure Sockets Layer Protocol](secure-sockets-layer-protocol.md) (SSL) or Transport Layer Security Protocol (TSL) alert to be sent to the target of a call to either the [**InitializeSecurityContext (Schannel)**](/windows/win32/Sspi/?branch=master) function or the [**AcceptSecurityContext (Schannel)**](/windows/win32/Sspi/?branch=master) function. |
| [**SCHANNEL\_CLIENT\_SIGNATURE**](/windows/win32/Schannel/ns-schannel-_schannel_client_signature?branch=master)          | Specifies a client signature when a call to the [**InitializeSecurityContext (Schannel)**](/windows/win32/Sspi/?branch=master) function cannot access the private key for a client certificate (in this case, the function returns **SEC\_I\_SIGNATURE\_NEEDED**).                                                                                                           |
| [**SCHANNEL\_CRED**](/windows/win32/Schannel/ns-schannel-_schannel_cred?branch=master)                                   | Contains the data for an Schannel credential.                                                                                                                                                                                                                                                                                                                                      |
| [**SCHANNEL\_SESSION\_TOKEN**](/windows/win32/Schannel/ns-schannel-_schannel_session_token?branch=master)                | Specifies whether reconnections are enabled for an authentication session created by calling either the [**InitializeSecurityContext (Schannel)**](/windows/win32/Sspi/?branch=master) function or the [**AcceptSecurityContext (Schannel)**](/windows/win32/Sspi/?branch=master) function.                                                                                |
| [**SecPkgContext\_Authority**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_authoritya?branch=master)               | Contains the name of the authenticating authority if one is available. It can be a [*certification authority*](https://msdn.microsoft.com/library/windows/desktop/ms721572#-security-certification-authority-gly) (CA) or the name of a server or domain that authenticated the connection.                                                                                               |
| [**SecPkgContext\_ConnectionInfo**](/windows/win32/Schannel/ns-schannel-_secpkgcontext_connectioninfo?branch=master)     | Contains protocol and cipher information. This structure is used by the [**QueryContextAttributes (General)**](/windows/win32/Sspi/?branch=master) function.                                                                                                                                                                                                                     |
| [**SecPkgContext\_IssuerListInfoEx**](/windows/win32/Schannel/ns-schannel-_secpkgcontext_issuerlistinfoex?branch=master) | Contains a list of trusted [*certification authorities*](https://msdn.microsoft.com/library/windows/desktop/ms721572#-security-certification-authority-gly).                                                                                                                                                                                                                              |
| [**SecPkgContext\_KeyInfo**](/windows/win32/Sspi/ns-sspi-_secpkgcontext_keyinfoa?branch=master)                   | Contains information about the [*session keys*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-session-key-gly) used in a [*security context*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-context-gly). This structure has been superseded by the [**SecPkgContext\_ConnectionInfo**](/windows/win32/Schannel/ns-schannel-_secpkgcontext_connectioninfo?branch=master) structure.                       |
| [**SecPkgContext\_ProtoInfo**](/windows/win32/Schannel/ns-ntifs-_secpkgcontext_protoinfow?branch=master)               | Holds information about the protocol in use.                                                                                                                                                                                                                                                                                                                                       |
| [**SecPkgContext\_SessionAppData**](/windows/win32/Schannel/ns-schannel-_secpkgcontext_sessionappdata?branch=master)     | Stores application data for a session context.                                                                                                                                                                                                                                                                                                                                     |
| [**SecPkgCred\_CipherStrengths**](/windows/win32/Schannel/?branch=master)         | Holds the minimum and maximum strength permitted for the cipher used by the specified Schannel credential.                                                                                                                                                                                                                                                                         |
| [**SecPkgCred\_SupportedAlgs**](/windows/win32/Schannel/?branch=master)             | Contains identifiers for algorithms permitted with a specified Schannel credential.                                                                                                                                                                                                                                                                                                |
| [**SecPkgCred\_SupportedProtocols**](/windows/win32/Schannel/?branch=master)   | Indicates the protocols permitted with a specified Schannel credential.                                                                                                                                                                                                                                                                                                            |
| [**X509Certificate**](/windows/win32/Schannel/ns-schannel-_x509certificate?branch=master)                                | Represents an [*X.509*](https://msdn.microsoft.com/library/windows/desktop/ms721636#-security-x-509-gly) certificate.                                                                                                                                                                                                                                                                                       |



 

## Custom Security Package Structures

Custom [*security package*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-package-gly) use the following structures.



| Structure                                                                   | Description                                                                                                                                                                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**LSA\_SECPKG\_FUNCTION\_TABLE**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_lsa_secpkg_function_table?branch=master)           | A table of pointers to the [*Local Security Authority*](https://msdn.microsoft.com/library/windows/desktop/ms721592#-security-local-security-authority-gly) (LSA) functions that custom security packages can call. |
| [**SECPKG\_CALL\_INFO**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_call_info?branch=master)                              | Contains information about an executing function call.                                                                                                                                                        |
| [**SECPKG\_CLIENT\_INFO**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_client_info?branch=master)                          | Contains information about the user of a security package.                                                                                                                                                    |
| [**SECPK\_CONTEXT\_THUNKS**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_context_thunks?branch=master)                     | Contains information about calls to the security package that will be executed in-process with the LSA.                                                                                                       |
| [**SECPKG\_DLL\_FUNCTIONS**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_dll_functions?branch=master)                      | Contains the functions available to custom security packages executing in-process with a client/server application.                                                                                           |
| [**SECPKG\_EVENT\_DOMAIN\_CHANGE**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_parameters?branch=master)                  | Contains [*session*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-session-gly) and computer information. This structure name is an alias for the [**SECPKG\_PARAMETERS**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_parameters?branch=master) structure. |
| [**SECPKG\_EVENT\_NOTIFY**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_event_notify?branch=master)                        | Contains information about a security-related event.                                                                                                                                                          |
| [**SECPKG\_EVENT\_PACKAGE\_CHANGE**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_event_package_change?branch=master)       | Contains information about security package availability and use.                                                                                                                                             |
| [**SECPKG\_EXTENDED\_INFORMATION**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_extended_information?branch=master)        | Contains extended information about the security package.                                                                                                                                                     |
| [**SECPKG\_FUNCTION\_TABLE**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_function_table?branch=master)                    | Contains pointers to the functions implemented by security packages.                                                                                                                                          |
| [**SECPKG\_GSS\_INFO**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_gss_info?branch=master)                                | Contains information on the GSS OID used to identify a security package.                                                                                                                                      |
| [**SECPKG\_MUTUAL\_AUTH\_LEVEL**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_mutual_auth_level?branch=master)             | Contains information about the mutual authentication level used by a security package.                                                                                                                        |
| [**SECPKG\_PARAMETERS**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_parameters?branch=master)                             | Contains [*session*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-session-gly) and machine information.                                                                                                     |
| [**SECPKG\_PRIMARY\_CRED**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_primary_cred?branch=master)                        | Contains [*primary credentials*](https://msdn.microsoft.com/library/windows/desktop/ms721603#-security-primary-credentials-gly) information.                                                                             |
| [**SECPKG\_SUPPLEMENTAL\_CRED**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_supplemental_cred?branch=master)              | Contains [*supplemental credentials*](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-supplemental-credentials-gly) information.                                                              |
| [**SECPKG\_SUPPLEMENTAL\_CRED\_ARRAY**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_supplemental_cred_array?branch=master) | Contains supplemental credentials information.                                                                                                                                                                |
| [**SECPKG\_USER\_FUNCTION\_TABLE**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_secpkg_user_function_table?branch=master)         | Contains the functions implemented by a security package loaded in-process with client/server applications.                                                                                                   |
| [**SecurityUserData**](/windows/win32/Ntsecpkg/ns-ntsecpkg-_security_user_data?branch=master)                                | Contains information about the logged on user.                                                                                                                                                                |

| [**KERB\_INTERACTIVE\_PROFILE**](/windows/win32/Ntsecapi/ns-ntsecapi-_kerb_interactive_profile?branch=master)                               | Contains information about an interactive logon profile.                                                                                                                                                                                                                                                                            |
