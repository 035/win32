# Winlogon

[*Winlogon*](security.w_gly#-security-winlogon-gly) handles interface functions that are independent of authentication policy. It creates the desktops for the window station, implements time-out operations, and provides a set of support functions for the [*GINA*](security.g_gly#-security-gina-gly).
