# MSV1\_0\_GETUSERINFO\_RESPONSE

The MSV1\_0\_GETUSERINFO\_RESPONSE structure is obsolete.

For retrieving information about the workstation user, see the [**NetWkstaUserGetInfo**](https://msdn.microsoft.com/windows/desktop/25ec7a49-fd26-4105-823b-a257a57f724e) function.
