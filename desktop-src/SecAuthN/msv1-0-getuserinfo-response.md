# MSV1\_0\_GETUSERINFO\_RESPONSE

The MSV1\_0\_GETUSERINFO\_RESPONSE structure is obsolete.

For retrieving information about the workstation user, see the [**NetWkstaUserGetInfo**](netmgmt.netwkstausergetinfo) function.
