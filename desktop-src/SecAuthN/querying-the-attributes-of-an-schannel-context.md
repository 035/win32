# Querying the Attributes of an Schannel Context

The [**QueryContextAttributes (Schannel)**](querycontextattributes--schannel-.md) function provides Schannel-specific information about a [*security context*](security.s_gly#-security-security-context-gly). Most of the information is originally specified when the context is created by using the client [**InitializeSecurityContext (Schannel)**](initializesecuritycontext--schannel-.md) or server [**AcceptSecurityContext (Schannel)**](acceptsecuritycontext--schannel-.md) function. Some information is specified when obtaining a handle to the credential used to create the context. For more information, see [Obtaining Schannel Credentials](obtaining-schannel-credentials.md).
