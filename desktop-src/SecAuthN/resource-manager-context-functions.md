# Resource Manager Context Functions

The following functions establish and release the [*resource manager context*](security.r_gly#-security-resource-manager-context-gly) that is used by the database query and database management functions.



| Topic                                                  | Description                                                                                                                                          |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardEstablishContext**](scardestablishcontext.md) | Establishes a context for accessing the [*smart card database*](security.s_gly#-security-smart-card-database-gly). |
| [**SCardReleaseContext**](scardreleasecontext.md)     | Closes an established context.                                                                                                                       |
