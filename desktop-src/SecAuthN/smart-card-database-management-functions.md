# Smart Card Database Management Functions

The following functions manage the [*smart card database*](security.s_gly#-security-smart-card-database-gly), updating the database by using a specified [*resource manager context*](security.r_gly#-security-resource-manager-context-gly).

> [!Note]  
> Database security is maintained by placing access restrictions on the database, rather than by adding security mechanisms to the [*smart card subsystem*](security.s_gly#-security-smart-card-subsystem-gly).

 



| Topic                                                            | Description                                                                                                                                                             |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardAddReaderToGroup**](scardaddreadertogroup.md)           | Add a [*reader*](security.r_gly#-security-reader-gly) to a [*reader group*](security.r_gly#-security-reader-group-gly). |
| [**SCardForgetCardType**](scardforgetcardtype.md)               | Remove a smart card from the system.                                                                                                                                    |
| [**SCardForgetReader**](scardforgetreader.md)                   | Remove a reader from the system.                                                                                                                                        |
| [**SCardForgetReaderGroup**](scardforgetreadergroup.md)         | Remove a reader group from the system.                                                                                                                                  |
| [**SCardIntroduceCardType**](scardintroducecardtype.md)         | Introduce a new card to the system.                                                                                                                                     |
| [**SCardIntroduceReader**](scardintroducereader.md)             | Introduce a new reader to the system.                                                                                                                                   |
| [**SCardIntroduceReaderGroup**](scardintroducereadergroup.md)   | Introduce a new reader group to the system.                                                                                                                             |
| [**SCardRemoveReaderFromGroup**](scardremovereaderfromgroup.md) | Remove a reader from a reader group.                                                                                                                                    |
