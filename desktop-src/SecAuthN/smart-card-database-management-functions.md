# Smart Card Database Management Functions

The following functions manage the [*smart card database*](security.s_gly#-security-smart-card-database-gly), updating the database by using a specified [*resource manager context*](security.r_gly#-security-resource-manager-context-gly).

> [!Note]  
> Database security is maintained by placing access restrictions on the database, rather than by adding security mechanisms to the [*smart card subsystem*](security.s_gly#-security-smart-card-subsystem-gly).

 



| Topic                                                            | Description                                                                                                                                                             |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardAddReaderToGroup**](/windows/win32/Winscard/nf-winscard-scardaddreadertogroupa?branch=master)           | Add a [*reader*](security.r_gly#-security-reader-gly) to a [*reader group*](security.r_gly#-security-reader-group-gly). |
| [**SCardForgetCardType**](/windows/win32/Winscard/nf-winscard-scardforgetcardtypea?branch=master)               | Remove a smart card from the system.                                                                                                                                    |
| [**SCardForgetReader**](/windows/win32/Winscard/nf-winscard-scardforgetreadera?branch=master)                   | Remove a reader from the system.                                                                                                                                        |
| [**SCardForgetReaderGroup**](/windows/win32/Winscard/nf-winscard-scardforgetreadergroupa?branch=master)         | Remove a reader group from the system.                                                                                                                                  |
| [**SCardIntroduceCardType**](/windows/win32/Winscard/nf-winscard-scardintroducecardtypea?branch=master)         | Introduce a new card to the system.                                                                                                                                     |
| [**SCardIntroduceReader**](/windows/win32/Winscard/nf-winscard-scardintroducereadera?branch=master)             | Introduce a new reader to the system.                                                                                                                                   |
| [**SCardIntroduceReaderGroup**](/windows/win32/Winscard/nf-winscard-scardintroducereadergroupa?branch=master)   | Introduce a new reader group to the system.                                                                                                                             |
| [**SCardRemoveReaderFromGroup**](/windows/win32/Winscard/nf-winscard-scardremovereaderfromgroupa?branch=master) | Remove a reader from a reader group.                                                                                                                                    |
