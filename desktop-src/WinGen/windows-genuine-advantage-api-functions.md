# Windows Genuine Advantage API Functions

\[The Windows Genuine Advantage API Functions are no longer available for use as of Windows 7.\]

The Windows Genuine Advantage (WGA) API provides the following functions.

The WGA functions are available only on Windows Vista and Windows XP installations that have been validated by clicking **Validate Windows** on [http://www.microsoft.com/genuine](http://go.microsoft.com/fwlink/p/?linkid=157937).



| Function                                                | Description                                                                                                     |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**GetLastErrors**](getlasterrors.md)<br/>       | Retrieves the trace output of the most recent validation session.<br/>                                    |
| [**IsAlreadyGenuine**](isalreadygenuine.md)<br/> | Specifies whether the copy of Windows running on the current system is genuine.<br/>                      |
| [**LegitCheck**](legitcheck.md)<br/>             | Returns a value that indicates whether the copy of Windows running on the current system is genuine.<br/> |
| [**SetPartnerID**](setpartnerid.md)<br/>         | Sets the value of the partner ID used for reporting WGA status.<br/>                                      |
