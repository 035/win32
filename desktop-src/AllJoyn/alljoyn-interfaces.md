# AllJoyn Interfaces

These interfaces provide common methods for AllJoyn APIs.

## In this section



| Topic                                                                                                          | Description                                                                                                     |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**IWindowsDevicesAllJoynBusAttachmentInterop**](/previous-versions/windows/desktop/api/windows.devices.alljoyn.interop/nn-windows-devices-alljoyn-interop-iwindowsdevicesalljoynbusattachmentinterop)               | This interface allows for retrieval of the underlying **alljoyn\_busattachment** object.                        |
| [**IWindowsDevicesAllJoynBusAttachmentFactoryInterop**](/previous-versions/windows/desktop/api/windows.devices.alljoyn.interop/nn-windows-devices-alljoyn-interop-iwindowsdevicesalljoynbusattachmentfactoryinterop) | This interface allows for the creation of **alljoyn\_busattachment** without taking ownership of the reference. |
