<span id="_irda_discovery_polling_loop_gly"></span><span id="_IRDA_DISCOVERY_POLLING_LOOP_GLY"></span>**discovery polling loop**
</dt> <dd>

A means of initiating an IrDA connection that constantly polls the IrDA interface for connectable devices.

</dd> <dt>

<span id="_irda_discovery_protocol_gly"></span><span id="_IRDA_DISCOVERY_PROTOCOL_GLY"></span>**discovery protocol**
</dt> <dd>

A protocol used directly by an application to learn about currently visible devices, their nicknames, and device (MAC) addresses.

</dd> <dt>

<span id="_irda_dongle_gly"></span><span id="_IRDA_DONGLE_GLY"></span>**dongle**

A hardware device attached to computers to enable IrDA connections.
