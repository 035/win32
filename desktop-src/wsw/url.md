# Url

The WWSAPI URL API elements provide mechanisms to split and unescape URLs into component parts, and to escape and combine component parts back into a URL.

-   To split and unescape URLs into component parts, use the [**WsDecodeUrl**](wsdecodeurl.md) function.
-   To escape and combine component parts back into a URL, use the [**WsEncodeUrl**](wsencodeurl.md) function.
-   To combine relative URLs with an absolute base URL forming an absolute URL, use the [**WsCombineUrl**](wscombineurl.md) function.

The following enumerations are part of the URL:

-   [**WS\_URL\_FLAGS**](ws-url-flags.md)
-   [**WS\_URL\_SCHEME\_TYPE**](ws-url-scheme-type.md)

The following functions are part of the URL:

-   [**WsCombineUrl**](wscombineurl.md)
-   [**WsDecodeUrl**](wsdecodeurl.md)
-   [**WsEncodeUrl**](wsencodeurl.md)

The following structures are part of the URL:

-   [**WS\_HTTPS\_URL**](ws-https-url.md)
-   [**WS\_HTTP\_URL**](ws-http-url.md)
-   [**WS\_NETPIPE\_URL**](ws-namedpipe-sspi-transport-security-binding.md)
-   [**WS\_NETTCP\_URL**](ws-nettcp-url.md)
-   [**WS\_SOAPUDP\_URL**](ws-soapudp-url.md)
-   [**WS\_URL**](ws-url.md)
