# DO Enumerations

The [Delivery Optimization (DO) interfaces](do-interfaces.md) use the following enumerations.

## In this section



| Topic                                                                       | Description                                                                                                                                                                                                                                                         |
|-----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BG_ERROR_CONTEXT**](bg-error-context.md)<br/>                   | The **BG_ERROR_CONTEXT** enumeration defines the constant values that specify the context in which the error occurred.<br/>                                                                                                                                 |
| [**BG_JOB_PRIORITY**](bg-job-priority-.md)<br/>                    | The **BG_JOB_PRIORITY** enumeration defines the constant values that specify the priority level of a job. <br/>                                                                                                                                             |
| [**BG_JOB_STATE**](bg-job-state-.md)<br/>                          | The **BG_JOB_STATE** enumeration defines constant values for the different states of a job.<br/>                                                                                                                                                            |
| [**BG_JOB_TYPE**](bg-job-type.md)<br/>                             | The **BG_JOB_TYPE** enumeration defines constant values that specify the type of transfer job, such as download.<br/>                                                                                                                                       |
| [**BITS_FILE_PROPERTY_ID**](bits-file-property-id-.md)<br/>       | The **BITS_FILE_PROPERTY_ID** enumeration specifies values that define ID values corresponding to **BackgroundCopyFile** properties.<br/>                                                                                                                  |
| [**BITS_JOB_PROPERTY_ID**](bits-job-property-id.md)<br/>          | The **BITS_JOB_PROPERTY_ID** enumeration specifies the ID of the property for the DO job. This enumeration is used in the [**BITS_JOB_PROPERTY_VALUE**](bits-job-property-value-.md) union to determine the type of value contained in the union.<br/> |
| [**BITS_JOB_TRANSFER_POLICY**](bits-job-transfer-policy-.md)<br/> | The **BITS_JOB_TRANSFER_POLICY** enumeration defines ID values corresponding to DO properties.<br/>                                                                                                                                                        |
| [**DownloadMode**](downloadmode.md)<br/>                             | Defines the different download modes that Delivery Optimization uses. <br/>                                                                                                                                                                                   |
| [**SwarmStatus**](swarmstatus.md)<br/>                               | Defines the status of a file within the delivery optimization client.<br/>                                                                                                                                                                                    |
