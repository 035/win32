# Offline Files API Functions

The following functions are used with Offline Files.

## In this section

<dl> <dt>

[**OfflineFilesEnable**](offlinefilesenable.md)
</dt> <dd>

Enables or disables the Offline Files feature.

</dd> <dt>

[**OfflineFilesQueryStatus**](offlinefilesquerystatus.md)
</dt> <dd>

Determines whether the Offline Files feature is enabled and, if so, whether it is active.

</dd> <dt>

[**OfflineFilesQueryStatusEx**](offlinefilesquerystatusex.md)
</dt> <dd>

Determines whether the Offline Files feature is enabled and, if so, whether it is active and available. This function is identical to the [**OfflineFilesQueryStatus**](offlinefilesquerystatus.md) function, except that it has an additional output parameter.

</dd> <dt>

[**OfflineFilesStart**](offlinefilesstart.md)
</dt> <dd>

Starts the Offline Files service.

</dd> </dl>
