# Offline Files API Functions

The following functions are used with Offline Files.

## In this section

<dl> <dt>

[**OfflineFilesEnable**](/previous-versions/windows/desktop/api/CscApi/nf-cscapi-offlinefilesenable)
</dt> <dd>

Enables or disables the Offline Files feature.

</dd> <dt>

[**OfflineFilesQueryStatus**](/previous-versions/windows/desktop/api/CscApi/nf-cscapi-offlinefilesquerystatus)
</dt> <dd>

Determines whether the Offline Files feature is enabled and, if so, whether it is active.

</dd> <dt>

[**OfflineFilesQueryStatusEx**](/previous-versions/windows/desktop/api/CscApi/nf-cscapi-offlinefilesquerystatusex)
</dt> <dd>

Determines whether the Offline Files feature is enabled and, if so, whether it is active and available. This function is identical to the [**OfflineFilesQueryStatus**](/previous-versions/windows/desktop/api/CscApi/nf-cscapi-offlinefilesquerystatus) function, except that it has an additional output parameter.

</dd> <dt>

[**OfflineFilesStart**](/previous-versions/windows/desktop/api/CscApi/nf-cscapi-offlinefilesstart)
</dt> <dd>

Starts the Offline Files service.

</dd> </dl>
