# Offline Files API Functions

The following functions are used with Offline Files.

## In this section

<dl> <dt>

[**OfflineFilesEnable**](/windows/previous-versions/CscApi/nf-cscapi-offlinefilesenable?branch=master)
</dt> <dd>

Enables or disables the Offline Files feature.

</dd> <dt>

[**OfflineFilesQueryStatus**](/windows/previous-versions/CscApi/nf-cscapi-offlinefilesquerystatus?branch=master)
</dt> <dd>

Determines whether the Offline Files feature is enabled and, if so, whether it is active.

</dd> <dt>

[**OfflineFilesQueryStatusEx**](/windows/previous-versions/CscApi/nf-cscapi-offlinefilesquerystatusex?branch=master)
</dt> <dd>

Determines whether the Offline Files feature is enabled and, if so, whether it is active and available. This function is identical to the [**OfflineFilesQueryStatus**](/windows/previous-versions/CscApi/nf-cscapi-offlinefilesquerystatus?branch=master) function, except that it has an additional output parameter.

</dd> <dt>

[**OfflineFilesStart**](/windows/previous-versions/CscApi/nf-cscapi-offlinefilesstart?branch=master)
</dt> <dd>

Starts the Offline Files service.

</dd> </dl>
