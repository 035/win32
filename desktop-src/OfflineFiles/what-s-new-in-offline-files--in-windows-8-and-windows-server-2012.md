# What's New in Offline Files, in Windows 8 and Windows Server 2012

This topic identifies what is new or changed in Offline Files in Windows 8 and Windows Server 2012.

## Windows 8 and Windows Server 2012

Windows 8 and Windows Server 2012 added the following functions:

-   [**OfflineFilesQueryStatusEx**](/windows/previous-versions/CscApi/nf-cscapi-offlinefilesquerystatusex?branch=master)
-   [**OfflineFilesStart**](/windows/previous-versions/CscApi/nf-cscapi-offlinefilesstart?branch=master)

Windows 8 and Windows Server 2012 changed the following WMI classes:

-   Added the [**RenameItemEx**](win32-offlinefilescache-renameitem.md) method to the [**Win32\_OfflineFilesCache**](win32-offlinefilescache.md) WMI class.
-   Changed a property of the [**Win32\_OfflineFilesConnectionInfo**](win32-offlinefilesconnectioninfo.md) WMI class.
