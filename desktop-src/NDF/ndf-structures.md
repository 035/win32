# NDF Structures

The following data structures support the Network Diagnostics Framework (NDF) API, which allows Microsoft's extensible Helper Classes to be extended by other developers.

-   [**DIAG\_SOCKADDR**](/windows/desktop/api/ndattrib/ns-ndattrib-tagsock_addr)
-   [**DiagnosticsInfo**](/windows/desktop/api/ndhelper/ns-ndhelper-tagdiagnosticsinfo)
-   [**HELPER\_ATTRIBUTE**](/windows/desktop/api/ndattrib/ns-ndattrib-taghelper_attribute)
-   [**HelperAttributeInfo**](/windows/desktop/api/ndhelper/ns-ndhelper-taghelperattributeinfo)
-   [**HYPOTHESIS**](/windows/desktop/api/ndhelper/ns-ndhelper-taghypothesis)
-   [**HypothesisResult**](/windows/desktop/api/ndhelper/ns-ndhelper-taghypothesisresult)
-   [**LIFE\_TIME**](/windows/desktop/api/ndattrib/ns-ndattrib-taglife_time)
-   [**OCTET\_STRING**](/windows/desktop/api/ndattrib/ns-ndattrib-tagoctet_string)
-   [**RepairInfo**](/windows/desktop/api/ndattrib/ns-ndattrib-tagrepairinfo)
-   [**RepairInfoEx**](/windows/desktop/api/ndattrib/ns-ndattrib-tagrepairinfoex)
-   [**RootCauseInfo**](/windows/desktop/api/ndattrib/ns-ndattrib-tagrootcauseinfo)
-   [**ShellCommandInfo**](/windows/desktop/api/ndattrib/ns-ndattrib-tagshellcommandinfo)
-   [**UiInfo**](/windows/desktop/api/ndattrib/ns-ndattrib-taguiinfo)
