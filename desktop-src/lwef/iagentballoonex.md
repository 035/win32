# IAgentBalloonEx

\[Microsoft Agent is deprecated as of Windows 7, and may be unavailable in subsequent versions of Windows.\]

[**IAgentBalloonEx**](https://msdn.microsoft.com/library/windows/desktop/ms697893) is derived from the **IAgentBalloon** interface. It includes all the **IAgentBalloon** methods and provides access to additional functions.

**Methods in Vtable Order**



| IAgentBalloonEx Methods                                  | Description                                                        |
|----------------------------------------------------------|--------------------------------------------------------------------|
| [**GetStyle**](lwef-getstyle_method)                     | Returns the word balloon's output style.                           |
| [**SetStyle**](lwef-setstyle_method)                     | Sets the word balloon's output style.                              |
| [**SetNumLines**](lwef-setnumlines_method)               | Sets the number lines output in the word balloon.                  |
| [**SetNumCharsPerLine**](lwef-setnumcharsperline_method) | Sets the number of characters per line output in the word balloon. |
