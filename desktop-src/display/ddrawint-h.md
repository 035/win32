# Ddrawint.h

This section contains reference topics for the Ddrawint.h header.

## In this section



| Topic                                                                                           | Description                                                                                                                                                                                                                                                                                                       |
|-------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [*PDD\_CANCREATESURFACE*](cancreated3dbuffer.md)<br/>                                    | The [**CanCreateD3DBuffer**](cancreated3dbuffer.md) callback function determines whether the driver can create a driver-level command or vertex buffer of the specified description.<br/>                                                                                                                  |

| [*PDD\_GETDRIVERSTATE*](d3dgetdriverstate.md)<br/>                                       | The [*D3dGetDriverState*](d3dgetdriverstate.md) function is used by both the Microsoft DirectDraw and Microsoft Direct3D runtimes to obtain information from the driver about its current state.<br/>                                                                                                      |

| [**DD\_CANCREATEVPORTDATA**](dd-cancreatevportdata.md)<br/>                              | The DD\_CANCREATEVPORTDATA structure contains the information required for the driver to determine whether a [video port extensions (VPE)](wdkgloss.v#wdkgloss-video-port-extensions--vpe-) object can be created.<br/>                                                |
| [**DD\_CLIPPER\_GLOBAL**](dd-clipper-global.md)<br/>                                     | The DD\_CLIPPER\_GLOBAL structure contains the global DirectDrawClipper data that can be shared between object instances. <br/>                                                                                                                                                                             |

| [**DD\_COLORCONTROLDATA**](dd-colorcontroldata.md)<br/>                                  | The DD\_COLORCONTROLDATA structure contains the color control information for the specified overlay.<br/>                                                                                                                                                                                                   |

| [**DD\_CREATESURFACEDATA**](dd-createsurfacedata.md)<br/>                                | The DD\_CREATESURFACEDATA structure contains information necessary to create a surface--in the case of [*CreateD3DBuffer*](created3dbuffer.md), a command or vertex buffer.<br/>                                                                                                                           |
| [**DD\_CREATEVPORTDATA**](dd-createvportdata.md)<br/>                                    | The DD\_CREATEVPORTDATA structure contains the information necessary to describe the [video port extensions (VPE)](wdkgloss.v#wdkgloss-video-port-extensions--vpe-) object being created.<br/>                                                                         |
| [**DD\_D3DBUFCALLBACKS**](dd-d3dbufcallbacks.md)<br/>                                    | The DD\_D3DBUFCALLBACKS structure is used only by drivers that implement driver level allocation of command and vertex buffers.<br/>                                                                                                                                                                        |
| [**DD\_DESTROYMOCOMPDATA**](dd-destroymocompdata.md)<br/>                                | The DD\_DESTROYMOCOMPDATA structure contains the information required to finish performing motion compensation. <br/>                                                                                                                                                                                       |
| [**DD\_DESTROYPALETTEDATA**](dd-destroypalettedata.md)<br/>                              | The DD\_DESTROYPALETTEDATA structure contains information necessary to destroy the specified palette.<br/>                                                                                                                                                                                                  |
| [**DD\_DESTROYSURFACEDATA**](dd-destroysurfacedata.md)<br/>                              | The DD\_DESTROYSURFACEDATA structure contains information necessary to destroy the specified surface--in the case of [*DestroyD3DBuffer*](destroyd3dbuffer.md), a command or vertex buffer.<br/>                                                                                                           |
| [**DD\_DESTROYVPORTDATA**](dd-destroyvportdata.md)<br/>                                  | The DD\_DESTROYVPORTDATA structure contains the information necessary for the driver to clean up.<br/>                                                                                                                                                                                                      |
