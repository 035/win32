# Dmemmgr.h

This section contains reference topics for the Dmemmgr.h header.

## In this section



| Topic                                                                  | Description                                                                                                                                                                                                                              |
|------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DD\_GETHEAPALIGNMENTDATA**](/windows/win32/dmemmgr/ns-dmemmgr-_dd_getheapalignmentdata?branch=master)<br/> | The DD\_GETHEAPALIGNMENTDATA structure contains data on required alignments from a particular heap.<br/>                                                                                                                           |
| [**HEAPALIGNMENT**](/windows/win32/dmemmgr/ns-dmemmgr-_heapalignment?branch=master)<br/>                      | The HEAPALIGNMENT structure contains data specifying the alignment requirements for a given display memory heap. <br/>                                                                                                             |
| [**HeapVidMemAllocAligned**](/windows/win32/dmemmgr/nf-dmemmgr-heapvidmemallocaligned?branch=master)<br/>    | The [**HeapVidMemAllocAligned**](/windows/win32/dmemmgr/nf-dmemmgr-heapvidmemallocaligned?branch=master) function allocates [off\_screen\_memory](wdkgloss.o#off-screen-memory) for a display driver by using the DirectDraw video memory heap manager.<br/> |
| [**VidMemFree**](/windows/win32/dmemmgr/nf-dmemmgr-vidmemfree?branch=master)<br/>                            | The **VidMemFree** function frees [off-screen memory](wdkgloss.o#wdkgloss-off-screen-memory) allocated for a display driver by [**HeapVidMemAllocAligned**](/windows/win32/dmemmgr/nf-dmemmgr-heapvidmemallocaligned?branch=master).<br/>           |
| [**VMEMHEAP**](/windows/win32/dmemmgr/ns-dmemmgr-_vmemheap?branch=master)<br/>                                | The VMEMHEAP structure contains information about the heap.<br/>                                                                                                                                                                   |
