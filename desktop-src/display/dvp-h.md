# Dvp.h

This section contains reference topics for the Dvp.h header.

## In this section



| Topic                                                           | Description                                                                                                                                                                                                                                                                            |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DDVIDEOPORTBANDWIDTH**](/windows/win32/dvp/ns-dvp-_ddvideoportbandwidth?branch=master)<br/> | The DDVIDEOPORTBANDWIDTH structure describes the bandwidth characteristics of an overlay when used with a particular [video port extensions (VPE)](wdkgloss.v#wdkgloss-video-port-extensions--vpe-) object/pixel format configuration.<br/> |
| [**DDVIDEOPORTCAPS**](/windows/win32/dvp/ns-dvp-_ddvideoportcaps?branch=master)<br/>           | The DDVIDEOPORTCAPS structure describes the capabilities and alignment restrictions of a hardware video port.<br/>                                                                                                                                                               |
| [**DDVIDEOPORTCONNECT**](/windows-hardware/drivers/ddi/content/ksmedia/ns-ksmedia-_ddvideoportconnect?branch=master)<br/>     | The DDVIDEOPORTCONNECT structure describes a hardware video port connection.<br/>                                                                                                                                                                                                |
| [**DDVIDEOPORTDESC**](/windows/win32/dvp/ns-dvp-_ddvideoportdesc?branch=master)<br/>           | The DDVIDEOPORTDESC structure describes the [video port extensions (VPE)](wdkgloss.v#wdkgloss-video-port-extensions--vpe-) object being created.<br/>                                                                                       |
| [**DDVIDEOPORTINFO**](/windows/win32/dvp/ns-dvp-_ddvideoportinfo?branch=master)<br/>           | The DDVIDEOPORTINFO structure describes how the driver should transfer video data to a surface (or to surfaces); DDVIDEOPORTINFO is a member of the [**DD\_VIDEOPORT\_LOCAL**](/windows/win32/ddrawint/ns-ddrawint-_dd_videoport_local?branch=master) structure.<br/>                                                           |
