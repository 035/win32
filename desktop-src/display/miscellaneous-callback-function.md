# Miscellaneous Callback Function

## 

This section provides a reference page for the miscellaneous callback function. The DirectDraw driver supplies this callback function when the GUID\_MiscellaneousCallbacks GUID is passed to its [**DdGetDriverInfo**](ddgetdriverinfo.md) function.
