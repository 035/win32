# Wingdi.h

This section contains reference topics for the Wingdi.h header.

## In this section



| Topic                                                                                                      | Description                                                                                                                                                                                                                                                                              |
|------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DEVMODEW**](/windows/win32/wingdi/ns-wingdi-_devicemodew?branch=master)<br/>                                                                    | The DEVMODEW structure is used for specifying characteristics of display and print devices in the Unicode (wide) character set.<br/>                                                                                                                                               |
| [**DISPLAYCONFIG\_SUPPORT\_VIRTUAL\_RESOLUTION**](/windows/win32/wingdi/ns-wingdi-displayconfig_support_virtual_resolution?branch=master)<br/> | The DISPLAYCONFIG\_SUPPORT\_VIRTUAL\_RESOLUTION structure contains information on the state of virtual resolution support for the monitor.<br/>                                                                                                                                    |
| [**DISPLAYCONFIG\_DESKTOP\_IMAGE\_INFO**](/windows/win32/wingdi/ns-wingdi-displayconfig_desktop_image_info?branch=master)<br/>                 | The DISPLAYCONFIG\_DESKTOP\_IMAGE\_INFO structure contains information about the image displayed on the desktop.<br/>                                                                                                                                                              |
| [**DISPLAYCONFIG\_2DREGION**](/windows/win32/wingdi/ns-wingdi-displayconfig_2dregion?branch=master)<br/>                                       | The DISPLAYCONFIG\_2DREGION structure represents a point or an offset in a two-dimensional space.<br/>                                                                                                                                                                             |
| [**DISPLAYCONFIG\_ADAPTER\_NAME**](/windows/win32/wingdi/ns-wingdi-displayconfig_adapter_name?branch=master)<br/>                              | The DISPLAYCONFIG\_ADAPTER\_NAME structure contains information about the display adapter.<br/>                                                                                                                                                                                    |
| [**DISPLAYCONFIG\_DEVICE\_INFO\_HEADER**](/windows/win32/wingdi/ns-wingdi-displayconfig_device_info_header?branch=master)<br/>                 | The DISPLAYCONFIG\_DEVICE\_INFO\_HEADER structure contains display information about the device.<br/>                                                                                                                                                                              |
| [**DISPLAYCONFIG\_MODE\_INFO**](/windows/win32/wingdi/ns-wingdi-displayconfig_mode_info?branch=master)<br/>                                    | The DISPLAYCONFIG\_MODE\_INFO structure contains either source mode or target mode information.<br/>                                                                                                                                                                               |
| [**DISPLAYCONFIG\_PATH\_INFO**](/windows/win32/wingdi/ns-wingdi-displayconfig_path_info?branch=master)<br/>                                    | The DISPLAYCONFIG\_PATH\_INFO structure is used to describe a single path from a target to a source.<br/>                                                                                                                                                                          |
| [**DISPLAYCONFIG\_PATH\_SOURCE\_INFO**](/windows/win32/wingdi/ns-wingdi-displayconfig_path_source_info?branch=master)<br/>                     | The DISPLAYCONFIG\_PATH\_SOURCE\_INFO structure contains source information for a single path.<br/>                                                                                                                                                                                |
| [**DISPLAYCONFIG\_PATH\_TARGET\_INFO**](/windows/win32/wingdi/ns-wingdi-displayconfig_path_target_info?branch=master)<br/>                     | The DISPLAYCONFIG\_PATH\_TARGET\_INFO structure contains target information for a single path.<br/>                                                                                                                                                                                |
| [**DISPLAYCONFIG\_RATIONAL**](/windows/win32/wingdi/ns-wingdi-displayconfig_rational?branch=master)<br/>                                       | The DISPLAYCONFIG\_RATIONAL structure describes a fractional value that represents vertical and horizontal frequencies of a video mode (that is, vertical sync and horizontal sync).<br/>                                                                                          |
| [**DISPLAYCONFIG\_SET\_TARGET\_PERSISTENCE**](/windows/win32/wingdi/ns-wingdi-displayconfig_set_target_persistence?branch=master)<br/>         | The DISPLAYCONFIG\_SET\_TARGET\_PERSISTENCE structure contains information about setting the display.<br/>                                                                                                                                                                         |
| [**DISPLAYCONFIG\_SOURCE\_DEVICE\_NAME**](/windows/win32/wingdi/ns-wingdi-displayconfig_source_device_name?branch=master)<br/>                 | The [**DISPLAYCONFIG\_SOURCE\_DEVICE\_NAME**](/windows/win32/wingdi/ns-wingdi-displayconfig_source_device_name?branch=master) structure contains the GDI device name for the source or view.<br/>                                                                                                                            |
| [**DISPLAYCONFIG\_SOURCE\_MODE**](/windows/win32/wingdi/ns-wingdi-displayconfig_source_mode?branch=master)<br/>                                | The [**DISPLAYCONFIG\_SOURCE\_MODE**](/windows/win32/wingdi/ns-wingdi-displayconfig_source_mode?branch=master) structure represents a point or an offset in a two-dimensional space.<br/>                                                                                                                                    |
| [**DISPLAYCONFIG\_TARGET\_BASE\_TYPE**](/windows/win32/wingdi/ns-wingdi-displayconfig_target_base_type?branch=master)<br/>                     | Specifies base output technology info for a given target ID.<br/>                                                                                                                                                                                                                  |
| [**DISPLAYCONFIG\_TARGET\_DEVICE\_NAME**](/windows/win32/wingdi/ns-wingdi-displayconfig_target_device_name?branch=master)<br/>                 | The DISPLAYCONFIG\_TARGET\_DEVICE\_NAME structure contains information about the target.<br/>                                                                                                                                                                                      |
| [**DISPLAYCONFIG\_TARGET\_DEVICE\_NAME\_FLAGS**](/windows/win32/wingdi/ns-wingdi-displayconfig_target_device_name_flags?branch=master)<br/>    | The [**DISPLAYCONFIG\_TARGET\_DEVICE\_NAME\_FLAGS**](/windows/win32/wingdi/ns-wingdi-displayconfig_target_device_name_flags?branch=master) structure contains information about a target device.<br/>                                                                                                                        |
| [**DISPLAYCONFIG\_TARGET\_MODE**](/windows/win32/wingdi/ns-wingdi-displayconfig_target_mode?branch=master)<br/>                                | The DISPLAYCONFIG\_TARGET\_MODE structure describes a display path target mode.<br/>                                                                                                                                                                                               |
| [**DISPLAYCONFIG\_TARGET\_PREFERRED\_MODE**](/windows/win32/wingdi/ns-wingdi-displayconfig_target_preferred_mode?branch=master)<br/>           | The [**DISPLAYCONFIG\_TARGET\_PREFERRED\_MODE**](/windows/win32/wingdi/ns-wingdi-displayconfig_target_preferred_mode?branch=master) structure contains information about the preferred mode of a display.<br/>                                                                                                               |
| [**DISPLAYCONFIG\_VIDEO\_SIGNAL\_INFO**](/windows/win32/wingdi/ns-wingdi-displayconfig_video_signal_info?branch=master)<br/>                   | The DISPLAYCONFIG\_VIDEO\_SIGNAL\_INFO structure contains information about the video signal for a display.<br/>                                                                                                                                                                   |
| [**DISPLAYCONFIG\_DEVICE\_INFO\_TYPE**](/windows/win32/wingdi/ne-wingdi-displayconfig_device_info_type?branch=master)<br/>                     | The DISPLAYCONFIG\_DEVICE\_INFO\_TYPE enumeration specifies the type of display device info to configure or obtain through the [**DisplayConfigSetDeviceInfo**](/windows/win32/winuser/nf-winuser-displayconfigsetdeviceinfo?branch=master) or [**DisplayConfigGetDeviceInfo**](/windows/win32/winuser/nf-winuser-displayconfiggetdeviceinfo?branch=master) function.<br/> |
| [**DISPLAYCONFIG\_MODE\_INFO\_TYPE**](/windows/win32/wingdi/ne-wingdi-displayconfig_mode_info_type?branch=master)<br/>                         | The DISPLAYCONFIG\_MODE\_INFO\_TYPE enumeration specifies that the information that is contained within the [**DISPLAYCONFIG\_MODE\_INFO**](/windows/win32/wingdi/ns-wingdi-displayconfig_mode_info?branch=master) structure is either source or target mode.<br/>                                                           |
| [**DISPLAYCONFIG\_PIXELFORMAT**](/windows/win32/wingdi/ne-wingdi-displayconfig_pixelformat?branch=master)<br/>                                 | The DISPLAYCONFIG\_PIXELFORMAT enumeration specifies pixel format in various bits per pixel (BPP) values.<br/>                                                                                                                                                                     |
| [**DISPLAYCONFIG\_ROTATION**](/windows/win32/wingdi/ne-wingdi-displayconfig_rotation?branch=master)<br/>                                       | The DISPLAYCONFIG\_ROTATION enumeration specifies the clockwise rotation of the display.<br/>                                                                                                                                                                                      |
| [**DISPLAYCONFIG\_SCALING**](/windows/win32/wingdi/ne-wingdi-displayconfig_scaling?branch=master)<br/>                                         | The DISPLAYCONFIG\_SCALING enumeration specifies the scaling transformation applied to content displayed on a video present network (VidPN) present path.<br/>                                                                                                                     |
| [**DISPLAYCONFIG\_SCANLINE\_ORDERING**](/windows/win32/wingdi/ne-wingdi-displayconfig_scanline_ordering?branch=master)<br/>                    | The DISPLAYCONFIG\_SCANLINE\_ORDERING enumeration specifies the method that the display uses to create an image on a screen.<br/>                                                                                                                                                  |
| [**DISPLAYCONFIG\_TOPOLOGY\_ID**](/windows/win32/wingdi/ne-wingdi-displayconfig_topology_id?branch=master)<br/>                                | The DISPLAYCONFIG\_TOPOLOGY\_ID enumeration specifies the type of display topology.<br/>                                                                                                                                                                                           |
| [**DISPLAYCONFIG\_VIDEO\_OUTPUT\_TECHNOLOGY**](/windows/win32/wingdi/ne-wingdi-displayconfig_video_output_technology?branch=master)<br/>       | The DISPLAYCONFIG\_VIDEO\_OUTPUT\_TECHNOLOGY enumeration specifies the target's connector type.<br/>                                                                                                                                                                               |
