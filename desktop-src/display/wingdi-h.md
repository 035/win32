# Wingdi.h

This section contains reference topics for the Wingdi.h header.

## In this section



| Topic                                                                                                      | Description                                                                                                                                                                                                                                                                              |

| [**DISPLAYCONFIG\_DEVICE\_INFO\_HEADER**](displayconfig-device-info-header.md)<br/>                 | The DISPLAYCONFIG\_DEVICE\_INFO\_HEADER structure contains display information about the device.<br/>                                                                                                                                                                              |
| [**DISPLAYCONFIG\_MODE\_INFO**](displayconfig-mode-info.md)<br/>                                    | The DISPLAYCONFIG\_MODE\_INFO structure contains either source mode or target mode information.<br/>                                                                                                                                                                               |
| [**DISPLAYCONFIG\_PATH\_INFO**](displayconfig-path-info.md)<br/>                                    | The DISPLAYCONFIG\_PATH\_INFO structure is used to describe a single path from a target to a source.<br/>                                                                                                                                                                          |

| [**DISPLAYCONFIG\_PATH\_TARGET\_INFO**](displayconfig-path-target-info.md)<br/>                     | The DISPLAYCONFIG\_PATH\_TARGET\_INFO structure contains target information for a single path.<br/>                                                                                                                                                                                |
| [**DISPLAYCONFIG\_RATIONAL**](displayconfig-rational.md)<br/>                                       | The DISPLAYCONFIG\_RATIONAL structure describes a fractional value that represents vertical and horizontal frequencies of a video mode (that is, vertical sync and horizontal sync).<br/>                                                                                          |
| [**DISPLAYCONFIG\_SET\_TARGET\_PERSISTENCE**](displayconfig-set-target-persistence.md)<br/>         | The DISPLAYCONFIG\_SET\_TARGET\_PERSISTENCE structure contains information about setting the display.<br/>                                                                                                                                                                         |
| [**DISPLAYCONFIG\_SOURCE\_DEVICE\_NAME**](displayconfig-source-device-name.md)<br/>                 | The [**DISPLAYCONFIG\_SOURCE\_DEVICE\_NAME**](displayconfig-source-device-name.md) structure contains the GDI device name for the source or view.<br/>                                                                                                                            |
| [**DISPLAYCONFIG\_SOURCE\_MODE**](displayconfig-source-mode.md)<br/>                                | The [**DISPLAYCONFIG\_SOURCE\_MODE**](displayconfig-source-mode.md) structure represents a point or an offset in a two-dimensional space.<br/>                                                                                                                                    |
| [**DISPLAYCONFIG\_TARGET\_BASE\_TYPE**](displayconfig-target-base-type.md)<br/>                     | Specifies base output technology info for a given target ID.<br/>                                                                                                                                                                                                                  |
| [**DISPLAYCONFIG\_TARGET\_DEVICE\_NAME**](displayconfig-target-device-name.md)<br/>                 | The DISPLAYCONFIG\_TARGET\_DEVICE\_NAME structure contains information about the target.<br/>                                                                                                                                                                                      |
| [**DISPLAYCONFIG\_TARGET\_DEVICE\_NAME\_FLAGS**](displayconfig-target-device-name-flags.md)<br/>    | The [**DISPLAYCONFIG\_TARGET\_DEVICE\_NAME\_FLAGS**](displayconfig-target-device-name-flags.md) structure contains information about a target device.<br/>                                                                                                                        |
| [**DISPLAYCONFIG\_TARGET\_MODE**](displayconfig-target-mode.md)<br/>                                | The DISPLAYCONFIG\_TARGET\_MODE structure describes a display path target mode.<br/>                                                                                                                                                                                               |
| [**DISPLAYCONFIG\_TARGET\_PREFERRED\_MODE**](displayconfig-target-preferred-mode.md)<br/>           | The [**DISPLAYCONFIG\_TARGET\_PREFERRED\_MODE**](displayconfig-target-preferred-mode.md) structure contains information about the preferred mode of a display.<br/>                                                                                                               |
| [**DISPLAYCONFIG\_VIDEO\_SIGNAL\_INFO**](displayconfig-video-signal-info.md)<br/>                   | The DISPLAYCONFIG\_VIDEO\_SIGNAL\_INFO structure contains information about the video signal for a display.<br/>                                                                                                                                                                   |
| [**DISPLAYCONFIG\_DEVICE\_INFO\_TYPE**](displayconfig-device-info-type.md)<br/>                     | The DISPLAYCONFIG\_DEVICE\_INFO\_TYPE enumeration specifies the type of display device info to configure or obtain through the [**DisplayConfigSetDeviceInfo**](displayconfigsetdeviceinfo.md) or [**DisplayConfigGetDeviceInfo**](displayconfiggetdeviceinfo.md) function.<br/> |
| [**DISPLAYCONFIG\_MODE\_INFO\_TYPE**](displayconfig-mode-info-type.md)<br/>                         | The DISPLAYCONFIG\_MODE\_INFO\_TYPE enumeration specifies that the information that is contained within the [**DISPLAYCONFIG\_MODE\_INFO**](displayconfig-mode-info.md) structure is either source or target mode.<br/>                                                           |
| [**DISPLAYCONFIG\_PIXELFORMAT**](displayconfig-pixelformat.md)<br/>                                 | The DISPLAYCONFIG\_PIXELFORMAT enumeration specifies pixel format in various bits per pixel (BPP) values.<br/>                                                                                                                                                                     |
| [**DISPLAYCONFIG\_ROTATION**](displayconfig-rotation.md)<br/>                                       | The DISPLAYCONFIG\_ROTATION enumeration specifies the clockwise rotation of the display.<br/>                                                                                                                                                                                      |
| [**DISPLAYCONFIG\_SCALING**](displayconfig-scaling.md)<br/>                                         | The DISPLAYCONFIG\_SCALING enumeration specifies the scaling transformation applied to content displayed on a video present network (VidPN) present path.<br/>                                                                                                                     |
| [**DISPLAYCONFIG\_SCANLINE\_ORDERING**](displayconfig-scanline-ordering.md)<br/>                    | The DISPLAYCONFIG\_SCANLINE\_ORDERING enumeration specifies the method that the display uses to create an image on a screen.<br/>                                                                                                                                                  |
| [**DISPLAYCONFIG\_TOPOLOGY\_ID**](displayconfig-topology-id.md)<br/>                                | The DISPLAYCONFIG\_TOPOLOGY\_ID enumeration specifies the type of display topology.<br/>                                                                                                                                                                                           |
| [**DISPLAYCONFIG\_VIDEO\_OUTPUT\_TECHNOLOGY**](displayconfig-video-output-technology.md)<br/>       | The DISPLAYCONFIG\_VIDEO\_OUTPUT\_TECHNOLOGY enumeration specifies the target's connector type.<br/>                                                                                                                                                                               |
