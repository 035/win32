# Ddraw.h

This section contains reference topics for the Ddraw.h header.

## In this section



| Topic                                               | Description                                                                                                 |
|-----------------------------------------------------|-------------------------------------------------------------------------------------------------------------|

| [**DDSCAPS**](ddscaps.md)<br/>               | The DDSCAPS structure defines the capabilities of a Microsoft DirectDraw surface object.<br/>         |
| [**DDSCAPS2**](ddscaps2.md)<br/>             | The DDSCAPS2 structure defines additional capabilities of a Microsoft DirectDraw surface object.<br/> |
