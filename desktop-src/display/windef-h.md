# Windef.h

This section contains reference topics for the Windef.h header.

## In this section



| Topic                               | Description                                                                                                      |
|-------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**POINT**](/windows/win32/windef/ns-windef-tagpoint?branch=master)<br/>   | The POINT structure defines the x- and y-coordinates of a point.<br/>                                      |
| [**POINTL**](/windows/win32/windef/ns-windef-_pointl?branch=master)<br/> | The POINTL structure defines the x- and y-coordinates of a point.<br/>                                     |
| [**POINTS**](/windows/win32/windef/ns-windef-tagpoints?branch=master)<br/> | The POINTS structure defines the x- and y-coordinates of a point.<br/>                                     |
| [**RECT**](/windows/win32/windef/ns-windef-tagrect?branch=master)<br/>     | The RECT structure defines a rectangle by the coordinates of its upper-left and lower-right corners.<br/>  |
| [**RECTL**](/windows/win32/windef/ns-windef-_rectl?branch=master)<br/>   | The RECTL structure defines a rectangle by the coordinates of its upper-left and lower-right corners.<br/> |
| [**SIZE**](/windows/win32/windef/ns-windef-tagsize?branch=master)<br/>     | The SIZE structure defines the width and height of a rectangle.<br/>                                       |
