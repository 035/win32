# Winuser.h

This section contains reference topics for the Winuser.h header.

## In this section



| Topic                                                                         | Description                                                                                                                                                                                                           |
|-------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DisplayConfigGetDeviceInfo**](/windows/win32/winuser/nf-winuser-displayconfiggetdeviceinfo?branch=master)<br/>   | The [**DisplayConfigGetDeviceInfo**](/windows/win32/winuser/nf-winuser-displayconfiggetdeviceinfo?branch=master) function retrieves display configuration information about the device.<br/>                                                                |
| [**DisplayConfigSetDeviceInfo**](/windows/win32/winuser/nf-winuser-displayconfigsetdeviceinfo?branch=master)<br/>   | The [**DisplayConfigSetDeviceInfo**](/windows/win32/winuser/nf-winuser-displayconfigsetdeviceinfo?branch=master) function sets the properties of a target.<br/>                                                                                             |
| [**GetDisplayConfigBufferSizes**](/windows/win32/winuser/nf-winuser-getdisplayconfigbuffersizes?branch=master)<br/> | The [**GetDisplayConfigBufferSizes**](/windows/win32/winuser/nf-winuser-getdisplayconfigbuffersizes?branch=master) function retrieves the size of the buffers that are required to call the [**QueryDisplayConfig**](/windows/win32/winuser/nf-winuser-querydisplayconfig?branch=master) function.<br/> |
| [**QueryDisplayConfig**](/windows/win32/winuser/nf-winuser-querydisplayconfig?branch=master)<br/>                   | The [**QueryDisplayConfig**](/windows/win32/winuser/nf-winuser-querydisplayconfig?branch=master) function retrieves information about all possible display paths for all display devices, or views, in the current setting.<br/>                            |
| [**SetDisplayConfig**](/windows/win32/winuser/nf-winuser-setdisplayconfig?branch=master)<br/>                       | The [**SetDisplayConfig**](/windows/win32/winuser/nf-winuser-setdisplayconfig?branch=master) function modifies the display topology, source, and target modes by exclusively enabling the specified paths in the current session.<br/>                      |
