# Dinputd.h

This section contains reference topics for the Dinputd.h header.

## In this section



| Topic                                                                   | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DIDEVICESTATE**](didevicestate.md)<br/>                       | The [**DIDEVICESTATE**](didevicestate.md) structure returns information about the state of a force feedback device. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**DIDRIVERVERSIONS**](didriverversions.md)<br/>                 | The [**DIDRIVERVERSIONS**](didriverversions.md) structure is used by the DirectInput effect driver to report version information back to DirectInput. The semantics of the version numbers are left to the discretion of the device driver. The only requirement is that later versions have higher version numbers than earlier versions. <br/>                                                                                                                                                                                                                                                                                                                                                                                    |
| [**DIEFFECTATTRIBUTES**](dieffectattributes.md)<br/>             | The [**DIEFFECTATTRIBUTES**](dieffectattributes.md) structure describes the information contained in the "Attributes" value of the registry key for each effect that is supported by a force-feedback device. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**DIFFDEVICEATTRIBUTES**](diffdeviceattributes.md)<br/>         | The DIFFDEVICEATTRIBUTES structure describes the information contained in the "Attributes" value of the **OEMForceFeedback** registry key. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**DIFFOBJECTATTRIBUTES**](diffobjectattributes.md)<br/>         | The DIFFOBJECTATTRIBUTES structure describes the information contained in the "FFAttributes" value of the registry key for each "object" on a force-feedback device. If the "FFAttributes" value is absent, then the object is assumed not to support force feedback. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**DIHIDFFINITINFO**](dihidffinitinfo.md)<br/>                   | The [**DIHIDFFINITINFO**](dihidffinitinfo.md) structure is used by DirectInput to provide information to a HID force-feedback driver about the device it is being asked to control.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**DIJOYCONFIG**](dijoyconfig.md)<br/>                           | The DIJOYCONFIG structure contains information about a joystick's configuration. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**DIJOYTYPEINFO**](dijoytypeinfo.md)<br/>                       | The [**DIJOYTYPEINFO**](dijoytypeinfo.md) structure contains information about a joystick type. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**DIJOYUSERVALUES**](dijoyuservalues.md)<br/>                   | The [**DIJOYUSERVALUES**](dijoyuservalues.md) structure contains information about the user's joystick settings. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**DIOBJECTATTRIBUTES**](diobjectattributes.md)<br/>             | The DIOBJECTATTRIBUTES structure describes the information contained in the "Attributes" value of the registry key for each "object" on a device. If the "Attributes" value is absent, then default attributes are used. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**DIOBJECTCALIBRATION**](diobjectcalibration.md)<br/>           | The DIOBJECTCALIBRATION structure describes the information contained in the "Calibration" value of the registry key for each axis on a device.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IDirectInputEffectDriver**](idirectinputeffectdriver.md)<br/> | These three methods allow additional interfaces to be added to the DirectInputEffectDriver object without affecting the functionality of the original interface. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**IDirectInputJoyConfig8**](idirectinputjoyconfig8.md)<br/>     | **IDirectInputJoyConfig8** interface contains methods that allow hardware developers who are writing property sheets to write and read information to and from the registry. If you need to open registry keys, you should use the [**IDirectInputJoyConfig8::OpenConfigKey**](idirectinputjoyconfig8-openconfigkey.md) and [**IDirectInputJoyConfig8::OpenTypeKey**](idirectinputjoyconfig8-opentypekey.md) methods rather than opening registry keys directly. Using either of these methods ensures that the correct registry branch is opened. In addition, the **IDirectInputJoyConfig8** interface will be supported in future versions of DirectInput when the underlying registry data may be structured differently.<br/> |
