# WCM Enumerations

The following table lists enumerations for the settings management infrastructure.



| Enumeration                                                          | Purpose                                                                                                                                                                                                                             |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WcmDataType**](/windows/previous-versions/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0003?branch=master)                                   | Enumerates the data types that may be returned from the [**ISettingsItem::GetDataType**](/windows/previous-versions/WcmConfig/nf-wcmconfig-isettingsitem-getdatatype?branch=master) method.                                                                                                     |
| [**WcmNamespaceAccess**](/windows/previous-versions/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0007?branch=master)                     | Describes the options that are passed to [**ISettingsEngine::GetNamespace**](/windows/previous-versions/WcmConfig/nf-wcmconfig-isettingsengine-getnamespace?branch=master) to choose how the namespace is accessed.                                                                             |
| [**WcmNamespaceEnumerationFlags**](/windows/previous-versions/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0002?branch=master) | Describes the various types of enumeration flags.                                                                                                                                                                                   |
| [**WcmRestrictionFacets**](/windows/previous-versions/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0005?branch=master)                 | Enumerates the facet values that may be returned by the [**ISettingsItem::GetRestrictionFacets**](/windows/previous-versions/WcmConfig/nf-wcmconfig-isettingsitem-getrestrictionfacets?branch=master) method.                                                                                   |
| [**WcmSettingType**](/windows/previous-versions/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0004?branch=master)                             | Describes setting types that are returned from the [**ISettingsItem::GetSettingType**](/windows/previous-versions/WcmConfig/nf-wcmconfig-isettingsitem-getsettingtype?branch=master) method and defines the object model type for the calling [**ISettingsItem**](/windows/previous-versions/WcmConfig/nn-wcmconfig-isettingsitem?branch=master) interface. |
| [**WcmTargetMode**](/windows/previous-versions/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0001?branch=master)                               | Enumerates the various target modes.                                                                                                                                                                                                |
| [**WcmUserStatus**](/windows/previous-versions/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0006?branch=master)                               | Describes the status of the user.                                                                                                                                                                                                   |
