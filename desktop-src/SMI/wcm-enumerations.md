# WCM Enumerations

The following table lists enumerations for the settings management infrastructure.



| Enumeration                                                          | Purpose                                                                                                                                                                                                                             |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WcmDataType**](wcmdatatype.md)                                   | Enumerates the data types that may be returned from the [**ISettingsItem::GetDataType**](isettingsitem-getdatatype.md) method.                                                                                                     |
| [**WcmNamespaceAccess**](wcmnamespaceaccess.md)                     | Describes the options that are passed to [**ISettingsEngine::GetNamespace**](isettingsengine-getnamespace.md) to choose how the namespace is accessed.                                                                             |
| [**WcmNamespaceEnumerationFlags**](wcmnamespaceenumerationflags.md) | Describes the various types of enumeration flags.                                                                                                                                                                                   |
| [**WcmRestrictionFacets**](wcmrestrictionfacets.md)                 | Enumerates the facet values that may be returned by the [**ISettingsItem::GetRestrictionFacets**](isettingsitem-getrestrictionfacets.md) method.                                                                                   |
| [**WcmSettingType**](wcmsettingtype.md)                             | Describes setting types that are returned from the [**ISettingsItem::GetSettingType**](isettingsitem-getsettingtype.md) method and defines the object model type for the calling [**ISettingsItem**](isettingsitem.md) interface. |
| [**WcmTargetMode**](wcmtargetmode.md)                               | Enumerates the various target modes.                                                                                                                                                                                                |
| [**WcmUserStatus**](wcmuserstatus.md)                               | Describes the status of the user.                                                                                                                                                                                                   |
