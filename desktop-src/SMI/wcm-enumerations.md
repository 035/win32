# WCM Enumerations

The following table lists enumerations for the settings management infrastructure.



| Enumeration                                                          | Purpose                                                                                                                                                                                                                             |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WcmDataType**](/previous-versions/windows/desktop/api/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0003)                                   | Enumerates the data types that may be returned from the [**ISettingsItem::GetDataType**](/previous-versions/windows/desktop/api/WcmConfig/nf-wcmconfig-isettingsitem-getdatatype) method.                                                                                                     |
| [**WcmNamespaceAccess**](/previous-versions/windows/desktop/api/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0007)                     | Describes the options that are passed to [**ISettingsEngine::GetNamespace**](/previous-versions/windows/desktop/api/WcmConfig/nf-wcmconfig-isettingsengine-getnamespace) to choose how the namespace is accessed.                                                                             |
| [**WcmNamespaceEnumerationFlags**](/previous-versions/windows/desktop/api/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0002) | Describes the various types of enumeration flags.                                                                                                                                                                                   |
| [**WcmRestrictionFacets**](/previous-versions/windows/desktop/api/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0005)                 | Enumerates the facet values that may be returned by the [**ISettingsItem::GetRestrictionFacets**](/previous-versions/windows/desktop/api/WcmConfig/nf-wcmconfig-isettingsitem-getrestrictionfacets) method.                                                                                   |
| [**WcmSettingType**](/previous-versions/windows/desktop/api/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0004)                             | Describes setting types that are returned from the [**ISettingsItem::GetSettingType**](/previous-versions/windows/desktop/api/WcmConfig/nf-wcmconfig-isettingsitem-getsettingtype) method and defines the object model type for the calling [**ISettingsItem**](/previous-versions/windows/desktop/api/WcmConfig/nn-wcmconfig-isettingsitem) interface. |
| [**WcmTargetMode**](/previous-versions/windows/desktop/api/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0001)                               | Enumerates the various target modes.                                                                                                                                                                                                |
| [**WcmUserStatus**](/previous-versions/windows/desktop/api/WcmConfig/ne-wcmconfig-__midl___midl_itf_wcmconfig_0000_0000_0006)                               | Describes the status of the user.                                                                                                                                                                                                   |
