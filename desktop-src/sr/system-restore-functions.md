# System Restore Functions

The following functions are used with System Restore.



| Function                                             | Description                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**SRRemoveRestorePoint**](/windows/win32/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint?branch=master) | Deletes the specified restore point.                                                                          |
| [**SRSetRestorePoint**](/windows/win32/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa?branch=master)       | Specifies the beginning and the ending of a set of changes so that System Restore can create a restore point. |
