# System Restore Functions

The following functions are used with System Restore.



| Function                                             | Description                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**SRRemoveRestorePoint**](srremoverestorepoint.md) | Deletes the specified restore point.                                                                          |
| [**SRSetRestorePoint**](srsetrestorepoint.md)       | Specifies the beginning and the ending of a set of changes so that System Restore can create a restore point. |
