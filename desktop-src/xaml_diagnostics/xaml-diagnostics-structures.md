# XAML Diagnostics Structures

This section describes the structures that are used by the XAML Diagnostics APIs.



| Structure                                                | Description                                                                           |
|----------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**BitmapDescription**](bitmapdescription.md)           | Represents information about the bitmap stored in [**IBitmapData**](ibitmapdata.md). |
| [**CollectionElementValue**](collectionelementvalue.md) | Represents an element in a collection.                                                |
| [**EnumType**](enumtype.md)                             | Represents a XAML Runtime enumeration.                                                |
| [**ParentChildRelation**](parentchildrelation.md)       | Associates a parent object with a child object.                                       |
| [**PropertyChainSource**](propertychainsource.md)       | Represents the source object (a Style) of a target type.                              |
| [**PropertyChainValue**](propertychainvalue.md)         | Represents a property defined on an element.                                          |
| [**SourceInfo**](sourceinfo.md)                         | Represents information about an object’s XAML source document.                        |
| [**VisualElement**](visualelement.md)                   | Represents a XAML element in the Live Visual Tree in Microsoft Visual Studio.         |
