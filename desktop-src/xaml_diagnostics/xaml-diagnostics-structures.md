# XAML Diagnostics Structures

This section describes the structures that are used by the XAML Diagnostics APIs.



| Structure                                                | Description                                                                           |
|----------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**BitmapDescription**](/previous-versions/windows/desktop/api/xamlom/ns-xamlom-bitmapdescription)           | Represents information about the bitmap stored in [**IBitmapData**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ibitmapdata). |
| [**CollectionElementValue**](/previous-versions/windows/desktop/api/xamlom/ns-xamlom-collectionelementvalue) | Represents an element in a collection.                                                |
| [**EnumType**](/previous-versions/windows/desktop/api/xamlom/ns-xamlom-enumtype)                             | Represents a XAML Runtime enumeration.                                                |
| [**ParentChildRelation**](/previous-versions/windows/desktop/api/xamlom/ns-xamlom-parentchildrelation)       | Associates a parent object with a child object.                                       |
| [**PropertyChainSource**](/previous-versions/windows/desktop/api/xamlom/ns-xamlom-propertychainsource)       | Represents the source object (a Style) of a target type.                              |
| [**PropertyChainValue**](/previous-versions/windows/desktop/api/xamlom/ns-xamlom-propertychainvalue)         | Represents a property defined on an element.                                          |
| [**SourceInfo**](/previous-versions/windows/desktop/api/xamlom/ns-xamlom-sourceinfo)                         | Represents information about an object’s XAML source document.                        |
| [**VisualElement**](/previous-versions/windows/desktop/api/xamlom/ns-xamlom-visualelement)                   | Represents a XAML element in the Live Visual Tree in Microsoft Visual Studio.         |
