# XAML Diagnostics Structures

This section describes the structures that are used by the XAML Diagnostics APIs.



| Structure                                                | Description                                                                           |
|----------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**BitmapDescription**](/windows/previous-versions/xamlom/ns-xamlom-bitmapdescription?branch=master)           | Represents information about the bitmap stored in [**IBitmapData**](/windows/previous-versions/xamlom/nn-xamlom-ibitmapdata?branch=master). |
| [**CollectionElementValue**](/windows/previous-versions/xamlom/ns-xamlom-collectionelementvalue?branch=master) | Represents an element in a collection.                                                |
| [**EnumType**](/windows/previous-versions/xamlom/ns-xamlom-enumtype?branch=master)                             | Represents a XAML Runtime enumeration.                                                |
| [**ParentChildRelation**](/windows/previous-versions/xamlom/ns-xamlom-parentchildrelation?branch=master)       | Associates a parent object with a child object.                                       |
| [**PropertyChainSource**](/windows/previous-versions/xamlom/ns-xamlom-propertychainsource?branch=master)       | Represents the source object (a Style) of a target type.                              |
| [**PropertyChainValue**](/windows/previous-versions/xamlom/ns-xamlom-propertychainvalue?branch=master)         | Represents a property defined on an element.                                          |
| [**SourceInfo**](/windows/previous-versions/xamlom/ns-xamlom-sourceinfo?branch=master)                         | Represents information about an object’s XAML source document.                        |
| [**VisualElement**](/windows/previous-versions/xamlom/ns-xamlom-visualelement?branch=master)                   | Represents a XAML element in the Live Visual Tree in Microsoft Visual Studio.         |
