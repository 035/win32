# XAML Diagnostics Interfaces

This section describes the interfaces that are used by the XAML Diagnostics APIs.



| Interface                                                          | Description                                                                            |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**IBitmapData**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ibitmapdata)                                 | Represents an image associated with a node in the visual tree.                         |
| [**IVisualTreeService**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ivisualtreeservice)                   | Provides methods to manage a XAML visual tree.                                         |
| [**IVisualTreeService2**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ivisualtreeservice2)                 | Adds capabilities to [**IVisualTreeService**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ivisualtreeservice).                 |
| [**IVisualTreeService3**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ivisualtreeservice3)                 | Adds capabilities to [**IVisualTreeService2**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ivisualtreeservice2).               |
| [**IVisualTreeServiceCallback**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ivisualtreeservicecallback)   | Communicates the state of the visual tree.                                             |
| [**IVisualTreeServiceCallback2**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ivisualtreeservicecallback2) | Adds capabilities to [**IVisualTreeServiceCallback**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ivisualtreeservicecallback). |
| [**IXamlDiagnostics**](/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ixamldiagnostics)                       | Represents a XAML Diagnostics session.                                                 |
