# XAML Diagnostics Interfaces

This section describes the interfaces that are used by the XAML Diagnostics APIs.



| Interface                                                          | Description                                                                            |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**IBitmapData**](/windows/previous-versions/xamlom/nn-xamlom-ibitmapdata?branch=master)                                 | Represents an image associated with a node in the visual tree.                         |
| [**IVisualTreeService**](/windows/previous-versions/xamlom/nn-xamlom-ivisualtreeservice?branch=master)                   | Provides methods to manage a XAML visual tree.                                         |
| [**IVisualTreeService2**](/windows/previous-versions/xamlom/nn-xamlom-ivisualtreeservice2?branch=master)                 | Adds capabilities to [**IVisualTreeService**](/windows/previous-versions/xamlom/nn-xamlom-ivisualtreeservice?branch=master).                 |
| [**IVisualTreeService3**](/windows/previous-versions/xamlom/nn-xamlom-ivisualtreeservice3?branch=master)                 | Adds capabilities to [**IVisualTreeService2**](/windows/previous-versions/xamlom/nn-xamlom-ivisualtreeservice2?branch=master).               |
| [**IVisualTreeServiceCallback**](/windows/previous-versions/xamlom/nn-xamlom-ivisualtreeservicecallback?branch=master)   | Communicates the state of the visual tree.                                             |
| [**IVisualTreeServiceCallback2**](/windows/previous-versions/xamlom/nn-xamlom-ivisualtreeservicecallback2?branch=master) | Adds capabilities to [**IVisualTreeServiceCallback**](/windows/previous-versions/xamlom/nn-xamlom-ivisualtreeservicecallback?branch=master). |
| [**IXamlDiagnostics**](/windows/previous-versions/xamlom/nn-xamlom-ixamldiagnostics?branch=master)                       | Represents a XAML Diagnostics session.                                                 |
