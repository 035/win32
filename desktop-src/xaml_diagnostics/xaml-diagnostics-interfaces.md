# XAML Diagnostics Interfaces

This section describes the interfaces that are used by the XAML Diagnostics APIs.



| Interface                                                          | Description                                                                            |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**IBitmapData**](ibitmapdata.md)                                 | Represents an image associated with a node in the visual tree.                         |
| [**IVisualTreeService**](ivisualtreeservice.md)                   | Provides methods to manage a XAML visual tree.                                         |
| [**IVisualTreeService2**](ivisualtreeservice2.md)                 | Adds capabilities to [**IVisualTreeService**](ivisualtreeservice.md).                 |
| [**IVisualTreeService3**](ivisualtreeservice3.md)                 | Adds capabilities to [**IVisualTreeService2**](ivisualtreeservice2.md).               |
| [**IVisualTreeServiceCallback**](ivisualtreeservicecallback.md)   | Communicates the state of the visual tree.                                             |
| [**IVisualTreeServiceCallback2**](ivisualtreeservicecallback2.md) | Adds capabilities to [**IVisualTreeServiceCallback**](ivisualtreeservicecallback.md). |
| [**IXamlDiagnostics**](ixamldiagnostics.md)                       | Represents a XAML Diagnostics session.                                                 |
