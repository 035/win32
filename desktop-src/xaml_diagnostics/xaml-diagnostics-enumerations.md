# XAML Diagnostics Enumerations

This section describes the enumerations that are used by the XAML Diagnostics APIs.



| Enumeration                                                    | Description                                                                                           |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [**BaseValueSource**](basevaluesource.md)                     | Defines constants that specify where the effective value of a property was set.                       |
| [**MetadataBit**](metadatabit.md)                             | Defines constants that are used to define the PropertyChainValue returned from XAML Diagnostics.      |
| [**RenderTargetBitmapOptions**](rendertargetbitmapoptions.md) | Defines constants that specify what parts of the visual tree should be rendered.                      |
| [**ResourceType**](resourcetype.md)                           | Defines constants that specify the type of a resource in a resource dictionary.                       |
| [**VisualElementState**](visualelementstate.md)               | Defines constants that specify the state of an element in the visual tree.                            |
| [**VisualMutationType**](visualmutationtype.md)               | Defines constants that specify whether the element was added to or removed from the live visual tree. |
