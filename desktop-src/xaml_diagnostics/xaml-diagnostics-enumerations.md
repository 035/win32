# XAML Diagnostics Enumerations

This section describes the enumerations that are used by the XAML Diagnostics APIs.



| Enumeration                                                    | Description                                                                                           |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [**BaseValueSource**](/windows/previous-versions/xamlom/ne-xamlom-basevaluesource?branch=master)                     | Defines constants that specify where the effective value of a property was set.                       |
| [**MetadataBit**](/windows/previous-versions/xamlom/ne-xamlom-metadatabit?branch=master)                             | Defines constants that are used to define the PropertyChainValue returned from XAML Diagnostics.      |
| [**RenderTargetBitmapOptions**](/windows/previous-versions/xamlom/ne-xamlom-rendertargetbitmapoptions?branch=master) | Defines constants that specify what parts of the visual tree should be rendered.                      |
| [**ResourceType**](/windows/previous-versions/xamlom/ne-xamlom-resourcetype?branch=master)                           | Defines constants that specify the type of a resource in a resource dictionary.                       |
| [**VisualElementState**](/windows/previous-versions/xamlom/ne-xamlom-visualelementstate?branch=master)               | Defines constants that specify the state of an element in the visual tree.                            |
| [**VisualMutationType**](/windows/previous-versions/xamlom/ne-xamlom-visualmutationtype?branch=master)               | Defines constants that specify whether the element was added to or removed from the live visual tree. |
