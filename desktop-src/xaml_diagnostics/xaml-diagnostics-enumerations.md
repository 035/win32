# XAML Diagnostics Enumerations

This section describes the enumerations that are used by the XAML Diagnostics APIs.



| Enumeration                                                    | Description                                                                                           |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [**BaseValueSource**](/previous-versions/windows/desktop/api/xamlom/ne-xamlom-basevaluesource)                     | Defines constants that specify where the effective value of a property was set.                       |
| [**MetadataBit**](/previous-versions/windows/desktop/api/xamlom/ne-xamlom-metadatabit)                             | Defines constants that are used to define the PropertyChainValue returned from XAML Diagnostics.      |
| [**RenderTargetBitmapOptions**](/previous-versions/windows/desktop/api/xamlom/ne-xamlom-rendertargetbitmapoptions) | Defines constants that specify what parts of the visual tree should be rendered.                      |
| [**ResourceType**](/previous-versions/windows/desktop/api/xamlom/ne-xamlom-resourcetype)                           | Defines constants that specify the type of a resource in a resource dictionary.                       |
| [**VisualElementState**](/previous-versions/windows/desktop/api/xamlom/ne-xamlom-visualelementstate)               | Defines constants that specify the state of an element in the visual tree.                            |
| [**VisualMutationType**](/previous-versions/windows/desktop/api/xamlom/ne-xamlom-visualmutationtype)               | Defines constants that specify whether the element was added to or removed from the live visual tree. |
