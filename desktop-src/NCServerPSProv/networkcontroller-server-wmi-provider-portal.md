# Network Controller Server WMI Provider

## Purpose

The Network Controller Server WMI Provider enables management of a network controller server. The classes are defined in NCServerPSProvider.mof and implemented in NCServerPSProvider.dll.

## In this section

<dl> <dt>

[**NetworkControllerApplicationConfiguration**](networkcontrollerapplicationconfiguration.md)
</dt> <dd>

Contains the network controller application configuration data.

</dd> <dt>

[**NetworkControllerClusterConfiguration**](networkcontrollerclusterconfiguration.md)
</dt> <dd>

Describes the configuration information for a network controller cluster.

</dd> <dt>

[**NetworkControllerCredential**](networkcontrollercredential.md)
</dt> <dd>

Stores network controller credentials.

</dd> <dt>

[**NetworkControllerDiagnosticInfo**](networkcontrollerdiagnosticinfo.md)
</dt> <dd>

Contains network controller diagnostic information.

</dd> <dt>

[**NetworkControllerManifestVersion**](networkcontrollermanifestversion.md)
</dt> <dd>

Describes the network Controller cluster and application manifest version.

</dd> <dt>

[**NetworkControllerNodeApplicationInfo**](networkcontrollernodeapplicationinfo.md)
</dt> <dd>

Contains information on a network controller node application.

</dd> <dt>

[**NetworkControllerNodeClusterInfo**](networkcontrollernodeclusterinfo.md)
</dt> <dd>

Contains information on a network controller node cluster.

</dd> <dt>

[**ServiceFabricVersion**](servicefabricversion.md)
</dt> <dd>

Describes the service fabric installed version.

</dd> <dt>

[**PS\_NetworkController**](ps-networkcontroller.md)

</dd> <dt>

[**PS\_NetworkControllerNode**](ps-networkcontrollernode.md)
</dt> <dd>

[**NetworkControllerClusterUpgradeProgressInfo**](networkcontrollerclusterupgradeprogressinfo.md)
