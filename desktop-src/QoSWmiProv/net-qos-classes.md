# NetQosCim Provider

The NetQosCim provider supplies data for network quality of service (QoS) and QoS setting data. The classes are declared in QoSWmi.mof and implemented in QoSWmi.dll.

## In this section



| Topic                                                                            | Description                                                                                                                                                                |
|----------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSFT\_NetQosPolicySettingData**](msft-netqospolicysettingdata.md)<br/> | A QoS policy.<br/>                                                                                                                                                   |
| [**MSFT\_NetSettingData**](msft-netsettingdata-netqoscim.md)<br/>         | Serves as a base class for classes that provide access to network settings data, such as [**MSFT\_NetQosPolicySettingData**](msft-netqospolicysettingdata.md).<br/> |
