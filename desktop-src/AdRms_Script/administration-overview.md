# Administration Overview

Beginning with Windows Server 2008, AD RMS implements an easy to use scripting API so that you can automate management of AD RMS resources and services. The following illustration shows the infrastructure levels at which you can perform management tasks.

![ad rms administration overview](images/scripting-api-administration-overview-2-9632.png)

You can use the AD RMS Scripting API to create scripts that manage specific servers, configure clusters, and set content protection policies for the enterprise.
