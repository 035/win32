# WinSAT Interfaces

\[WinSAT interfaces may be altered or unavailable for releases after Windows 8.1.\]

The Windows System Assessment Tool (WinSAT) defines the following interfaces that you can use to initiate assessments on the computer and to query formal assessments in the WinSAT data store:

-   [**IInitiateWinSATAssessment**](iinitiatewinsatassessment.md)
-   [**IProvideWinSATAssessmentInfo**](iprovidewinsatassessmentinfo.md)
-   [**IProvideWinSATResultsInfo**](iprovidewinsatresultsinfo.md)
-   [**IProvideWinSATVisuals**](iprovidewinsatvisuals.md)
-   [**IQueryAllWinSATAssessments**](iqueryallwinsatassessments.md)
-   [**IQueryRecentWinSATAssessment**](iqueryrecentwinsatassessment.md)
-   [**IWinSATInitiateEvents**](iwinsatinitiateevents.md)

WinSAT also defines a Windows Management Instrumentation (WMI) Managed Object Format (MOF) [class](winsat-mof-classes.md) that provides similar functionality.
