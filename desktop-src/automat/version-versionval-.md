# version(versionval)

Specifies a version number.

## Parameters

<dl> <dt>

<span id="versionval"></span><span id="VERSIONVAL"></span>*versionval*
</dt> <dd>

A real number in the format *n*.*m*, where *n* is a major version number and *m* is a minor version number.

</dd> </dl>

## Allowed on

Library, struct, module, dispinterface, interface, coclass, enum, union.
