# helpfile(filename)

Sets the name of the Help file.

## Parameters

<dl> <dt>

<span id="filename"></span><span id="FILENAME"></span>*filename*
</dt> <dd>

The name for the help file.

</dd> </dl>

## Allowed on

Library.

## Remarks

Retrieved through the [GetDocumentation](AA65E143-47DB-4241-9C66-FE3A1DCF1F0A) functions in the [**ITypeLib**](itypelib.md) and [**ITypeInfo**](itypeinfo.md) interfaces.

All types in a library share the same Help file.
