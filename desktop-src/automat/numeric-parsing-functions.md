# Numeric Parsing Functions

Automation supports string parsing through two low-level API functions. String information is passed using the [**NUMPARSE**](/previous-versions/windows/desktop/api/OleAuto/ns-oleauto-numparse) structure.

## In this section



| Topic                                                       | Description                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**VarNumFromParseNum**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varnumfromparsenum)<br/> | Converts parsed results to a variant.<br/>                                                    |
| [**VarParseNumFromStr**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varparsenumfromstr)<br/> | Parses a string, and creates a type-independent description of the number it represents.<br/> |
| [**NUMPARSE**](/previous-versions/windows/desktop/api/OleAuto/ns-oleauto-numparse)<br/>                     | Specifies numeric parsing information.<br/>                                                   |
