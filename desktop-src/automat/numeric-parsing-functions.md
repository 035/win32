# Numeric Parsing Functions

Automation supports string parsing through two low-level API functions. String information is passed using the [**NUMPARSE**](/windows/previous-versions/OleAuto/ns-oleauto-numparse?branch=master) structure.

## In this section



| Topic                                                       | Description                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**VarNumFromParseNum**](/windows/previous-versions/OleAuto/nf-oleauto-varnumfromparsenum?branch=master)<br/> | Converts parsed results to a variant.<br/>                                                    |
| [**VarParseNumFromStr**](/windows/previous-versions/OleAuto/nf-oleauto-varparsenumfromstr?branch=master)<br/> | Parses a string, and creates a type-independent description of the number it represents.<br/> |
| [**NUMPARSE**](/windows/previous-versions/OleAuto/ns-oleauto-numparse?branch=master)<br/>                     | Specifies numeric parsing information.<br/>                                                   |
