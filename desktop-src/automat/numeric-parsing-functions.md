# Numeric Parsing Functions

Automation supports string parsing through two low-level API functions. String information is passed using the [**NUMPARSE**](numparse.md) structure.

## In this section



| Topic                                                       | Description                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**VarNumFromParseNum**](varnumfromparsenum.md)<br/> | Converts parsed results to a variant.<br/>                                                    |
| [**VarParseNumFromStr**](varparsenumfromstr.md)<br/> | Parses a string, and creates a type-independent description of the number it represents.<br/> |
| [**NUMPARSE**](numparse.md)<br/>                     | Specifies numeric parsing information.<br/>                                                   |
