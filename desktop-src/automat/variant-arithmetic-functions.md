# Variant Arithmetic Functions

The variant arithmetic functions are described in the following table.

## In this section



| Topic                                       | Description                                                                                          |
|---------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**VarAbs**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varabs)<br/>         | Returns the absolute value of a variant.<br/>                                                  |
| [**VarAdd**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varadd)<br/>         | Returns the sum of two variants.<br/>                                                          |
| [**VarAnd**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varand)<br/>         | Performs a bitwise And operation between two variants of any integral type.<br/>               |
| [**VarCat**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcat)<br/>         | Concatenates two variants and returns the result.<br/>                                         |
| [**VarCmp**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcmp)<br/>         | Compares two variants.<br/>                                                                    |
| [**VarDiv**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardiv)<br/>         | Returns the result from dividing two variants. <br/>                                           |
| [**VarEqv**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vareqv)<br/>         | Performs a bitwise equivalence on two variants.<br/>                                           |
| [**VarFix**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varfix)<br/>         | Returns the integer portion of a variant.<br/>                                                 |
| [**VarIdiv**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varidiv)<br/>       | Converts two variants of any type to integers then returns the result from dividing them.<br/> |
| [**VarImp**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varimp)<br/>         | Performs a bitwise implication on two variants.<br/>                                           |
| [**VarInt**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varint)<br/>         | Returns the integer portion of a variant.<br/>                                                 |
| [**VarMod**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varmod)<br/>         | Divides two variants and returns only the remainder.<br/>                                      |
| [**VarMul**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varmul)<br/>         | Returns the result from multiplying two variants.<br/>                                         |
| [**VarNeg**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varneg)<br/>         | Performs logical negation on a variant.<br/>                                                   |
| [**VarNot**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varnot)<br/>         | Performs the bitwise not negation operation on a variant.<br/>                                 |
| [**VarOr**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varor)<br/>           | Performs a logical disjunction on two variants. <br/>                                          |
| [**VarPow**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varpow)<br/>         | Returns the result of performing the power function with two variants.<br/>                    |
| [**VarR4CmpR8**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varr4cmpr8)<br/> | Compares two variants of types float and double.<br/>                                          |
| [**VarR8Pow**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varr8pow)<br/>     | Performs the power function for variants of type double.<br/>                                  |
| [**VarR8Round**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varr8round)<br/> | Rounds a variant of type double to the specified number of decimal places.<br/>                |
| [**VarRound**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varround)<br/>     | Rounds a variant to the specified number of decimal places.<br/>                               |
| [**VarSub**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varsub)<br/>         | Subtracts two variants.<br/>                                                                   |
| [**VarXor**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varxor)<br/>         | Performs a logical exclusion on two variants.<br/>                                             |
