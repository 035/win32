# Variant Arithmetic Functions

The variant arithmetic functions are described in the following table.

## In this section



| Topic                                       | Description                                                                                          |
|---------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**VarAbs**](/windows/previous-versions/OleAuto/nf-oleauto-varabs?branch=master)<br/>         | Returns the absolute value of a variant.<br/>                                                  |
| [**VarAdd**](/windows/previous-versions/OleAuto/nf-oleauto-varadd?branch=master)<br/>         | Returns the sum of two variants.<br/>                                                          |
| [**VarAnd**](/windows/previous-versions/OleAuto/nf-oleauto-varand?branch=master)<br/>         | Performs a bitwise And operation between two variants of any integral type.<br/>               |
| [**VarCat**](/windows/previous-versions/OleAuto/nf-oleauto-varcat?branch=master)<br/>         | Concatenates two variants and returns the result.<br/>                                         |
| [**VarCmp**](/windows/previous-versions/OleAuto/nf-oleauto-varcmp?branch=master)<br/>         | Compares two variants.<br/>                                                                    |
| [**VarDiv**](/windows/previous-versions/OleAuto/nf-oleauto-vardiv?branch=master)<br/>         | Returns the result from dividing two variants. <br/>                                           |
| [**VarEqv**](/windows/previous-versions/OleAuto/nf-oleauto-vareqv?branch=master)<br/>         | Performs a bitwise equivalence on two variants.<br/>                                           |
| [**VarFix**](/windows/previous-versions/OleAuto/nf-oleauto-varfix?branch=master)<br/>         | Returns the integer portion of a variant.<br/>                                                 |
| [**VarIdiv**](/windows/previous-versions/OleAuto/nf-oleauto-varidiv?branch=master)<br/>       | Converts two variants of any type to integers then returns the result from dividing them.<br/> |
| [**VarImp**](/windows/previous-versions/OleAuto/nf-oleauto-varimp?branch=master)<br/>         | Performs a bitwise implication on two variants.<br/>                                           |
| [**VarInt**](/windows/previous-versions/OleAuto/nf-oleauto-varint?branch=master)<br/>         | Returns the integer portion of a variant.<br/>                                                 |
| [**VarMod**](/windows/previous-versions/OleAuto/nf-oleauto-varmod?branch=master)<br/>         | Divides two variants and returns only the remainder.<br/>                                      |
| [**VarMul**](/windows/previous-versions/OleAuto/nf-oleauto-varmul?branch=master)<br/>         | Returns the result from multiplying two variants.<br/>                                         |
| [**VarNeg**](/windows/previous-versions/OleAuto/nf-oleauto-varneg?branch=master)<br/>         | Performs logical negation on a variant.<br/>                                                   |
| [**VarNot**](/windows/previous-versions/OleAuto/nf-oleauto-varnot?branch=master)<br/>         | Performs the bitwise not negation operation on a variant.<br/>                                 |
| [**VarOr**](/windows/previous-versions/OleAuto/nf-oleauto-varor?branch=master)<br/>           | Performs a logical disjunction on two variants. <br/>                                          |
| [**VarPow**](/windows/previous-versions/OleAuto/nf-oleauto-varpow?branch=master)<br/>         | Returns the result of performing the power function with two variants.<br/>                    |
| [**VarR4CmpR8**](/windows/previous-versions/OleAuto/nf-oleauto-varr4cmpr8?branch=master)<br/> | Compares two variants of types float and double.<br/>                                          |
| [**VarR8Pow**](/windows/previous-versions/OleAuto/nf-oleauto-varr8pow?branch=master)<br/>     | Performs the power function for variants of type double.<br/>                                  |
| [**VarR8Round**](/windows/previous-versions/OleAuto/nf-oleauto-varr8round?branch=master)<br/> | Rounds a variant of type double to the specified number of decimal places.<br/>                |
| [**VarRound**](/windows/previous-versions/OleAuto/nf-oleauto-varround?branch=master)<br/>     | Rounds a variant to the specified number of decimal places.<br/>                               |
| [**VarSub**](/windows/previous-versions/OleAuto/nf-oleauto-varsub?branch=master)<br/>         | Subtracts two variants.<br/>                                                                   |
| [**VarXor**](/windows/previous-versions/OleAuto/nf-oleauto-varxor?branch=master)<br/>         | Performs a logical exclusion on two variants.<br/>                                             |
