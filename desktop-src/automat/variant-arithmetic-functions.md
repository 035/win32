# Variant Arithmetic Functions

The variant arithmetic functions are described in the following table.

## In this section



| Topic                                       | Description                                                                                          |
|---------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**VarAbs**](varabs.md)<br/>         | Returns the absolute value of a variant.<br/>                                                  |
| [**VarAdd**](varadd.md)<br/>         | Returns the sum of two variants.<br/>                                                          |
| [**VarAnd**](varand.md)<br/>         | Performs a bitwise And operation between two variants of any integral type.<br/>               |
| [**VarCat**](varcat.md)<br/>         | Concatenates two variants and returns the result.<br/>                                         |
| [**VarCmp**](varcmp.md)<br/>         | Compares two variants.<br/>                                                                    |
| [**VarDiv**](vardiv.md)<br/>         | Returns the result from dividing two variants. <br/>                                           |
| [**VarEqv**](vareqv.md)<br/>         | Performs a bitwise equivalence on two variants.<br/>                                           |
| [**VarFix**](varfix.md)<br/>         | Returns the integer portion of a variant.<br/>                                                 |
| [**VarIdiv**](varidiv.md)<br/>       | Converts two variants of any type to integers then returns the result from dividing them.<br/> |
| [**VarImp**](varimp.md)<br/>         | Performs a bitwise implication on two variants.<br/>                                           |
| [**VarInt**](varint.md)<br/>         | Returns the integer portion of a variant.<br/>                                                 |
| [**VarMod**](varmod.md)<br/>         | Divides two variants and returns only the remainder.<br/>                                      |
| [**VarMul**](varmul.md)<br/>         | Returns the result from multiplying two variants.<br/>                                         |
| [**VarNeg**](varneg.md)<br/>         | Performs logical negation on a variant.<br/>                                                   |
| [**VarNot**](varnot.md)<br/>         | Performs the bitwise not negation operation on a variant.<br/>                                 |
| [**VarOr**](varor.md)<br/>           | Performs a logical disjunction on two variants. <br/>                                          |
| [**VarPow**](varpow.md)<br/>         | Returns the result of performing the power function with two variants.<br/>                    |
| [**VarR4CmpR8**](varr4cmpr8.md)<br/> | Compares two variants of types float and double.<br/>                                          |
| [**VarR8Pow**](varr8pow.md)<br/>     | Performs the power function for variants of type double.<br/>                                  |
| [**VarR8Round**](varr8round.md)<br/> | Rounds a variant of type double to the specified number of decimal places.<br/>                |
| [**VarRound**](varround.md)<br/>     | Rounds a variant to the specified number of decimal places.<br/>                               |
| [**VarSub**](varsub.md)<br/>         | Subtracts two variants.<br/>                                                                   |
| [**VarXor**](varxor.md)<br/>         | Performs a logical exclusion on two variants.<br/>                                             |
