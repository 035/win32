# helpstringdll(dllname)

Sets the name of the DLL to use to perform the document string lookup (localization).

## Parameters

<dl> <dt>

<span id="dllname"></span><span id="DLLNAME"></span>*dllname*
</dt> <dd>

The name of the dll.

</dd> </dl>

## Allowed on

Type library.

## Remarks

Retrieved through the **GetDocumentation2** functions in the [**ITypeLib2**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypelib2) and [**ITypeInfo2**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypeinfo2) interfaces.
