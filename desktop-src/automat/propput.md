# propput

Specifies a property-setting function.

## Allowed on

Functions, methods in interfaces, dispinterfaces.

## Flags

<dl> INVOKE\_PROPERTYPUT  
</dl>

## Remarks

The property must have the same name as the function. Only one [propget](propget.md), propput, and [propputref](propputref.md) can be specified.
