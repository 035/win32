# helpstring(string)

Sets the Help string.

## Parameters

<dl> <dt>

<span id="string"></span><span id="STRING"></span>*string*
</dt> <dd>

The name of the help string.

</dd> </dl>

## Allowed on

Library, interface, dispinterface, struct, enum, union, module, typedef, method, struct member, enum value, property, coclass, const.

## Remarks

Retrieved through the [GetDocumentation](https://msdn.microsoft.com/windows/desktop/AA65E143-47DB-4241-9C66-FE3A1DCF1F0A) functions in the [**ITypeLib**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypelib) and [**ITypeInfo**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypeinfo) interfaces.
