# Error-Handling Functions

The error handling functions are used to create, set and retrieve error objects.

## In this section



| Topic                                                 | Description                                                                                                                                      |
|-------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateErrorInfo**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-createerrorinfo)<br/> | Creates an instance of a generic error object.<br/>                                                                                        |
| [**GetErrorInfo**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-geterrorinfo)<br/>       | Obtains the error information pointer set by the previous call to [**SetErrorInfo**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-seterrorinfo) in the current logical thread.<br/> |
| [**SetErrorInfo**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-seterrorinfo)<br/>       | Sets the error information object for the current logical thread of execution.<br/>                                                        |
