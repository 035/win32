# Type Description Functions

The following functions are provided for loading, registering, and querying type libraries.

## In this section



| Topic                                                                                 | Description                                                                                                                                                                                                          |
|---------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**LHashValOfName**](lhashvalofname.md)<br/>                                   | Computes a hash value for a name.<br/>                                                                                                                                                                         |
| [**LHashValOfNameSys**](lhashvalofnamesys.md)<br/>                             | Computes a hash value for a name.<br/>                                                                                                                                                                         |
| [**LHashValOfNameSysA**](lhashvalofnamesysa.md)<br/>                           | Computes a hash value for the specified name.<br/>                                                                                                                                                             |
| [**LoadRegTypeLib**](loadregtypelib.md)<br/>                                   | Uses registry information to load a type library.<br/>                                                                                                                                                         |
| [**LoadTypeLib**](loadtypelib.md)<br/>                                         | Loads and registers a type library.<br/>                                                                                                                                                                       |
| [**LoadTypeLibEx**](loadtypelibex.md)<br/>                                     | Loads a type library and (optionally) registers it in the system registry.   <br/>                                                                                                                             |
| [**OaEnablePerUserTLibRegistration**](oaenableperusertlibregistration.md)<br/> | Enables the [**RegisterTypeLib**](registertypelib.md) function to override default registry mappings under Windows Vista Service Pack 1 (SP1), Windows Server 2008, and later operating system versions.<br/> |
| [**QueryPathOfRegTypeLib**](querypathofregtypelib.md)<br/>                     | Retrieves the path of a registered type library.<br/>                                                                                                                                                          |
| [**RegisterTypeLib**](registertypelib.md)<br/>                                 | Adds information about a type library to the system registry.<br/>                                                                                                                                             |
| [**RegisterTypeLibForUser**](registertypelibforuser.md)<br/>                   | Registers a type library for use by the calling user.<br/>                                                                                                                                                     |
| [**UnRegisterTypeLib**](unregistertypelib.md)<br/>                             | Removes type library information from the system registry. Use this API to allow applications to properly uninstall themselves.<br/>                                                                           |
| [**UnRegisterTypeLibForUser**](unregistertypelibforuser.md)<br/>               | Removes type library information that was registered by using [**RegisterTypeLibForUser**](registertypelibforuser.md).<br/>                                                                                   |
