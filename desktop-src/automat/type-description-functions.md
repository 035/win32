# Type Description Functions

The following functions are provided for loading, registering, and querying type libraries.

## In this section



| Topic                                                                                 | Description                                                                                                                                                                                                          |
|---------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**LHashValOfName**](/windows/previous-versions/OleAuto/nf-oleauto-lhashvalofname?branch=master)<br/>                                   | Computes a hash value for a name.<br/>                                                                                                                                                                         |
| [**LHashValOfNameSys**](/windows/previous-versions/OleAuto/nf-oleauto-lhashvalofnamesys?branch=master)<br/>                             | Computes a hash value for a name.<br/>                                                                                                                                                                         |
| [**LHashValOfNameSysA**](/windows/previous-versions/OleAuto/nf-oleauto-lhashvalofnamesysa?branch=master)<br/>                           | Computes a hash value for the specified name.<br/>                                                                                                                                                             |
| [**LoadRegTypeLib**](/windows/previous-versions/OleAuto/nf-oleauto-loadregtypelib?branch=master)<br/>                                   | Uses registry information to load a type library.<br/>                                                                                                                                                         |
| [**LoadTypeLib**](/windows/previous-versions/OleAuto/nf-oleauto-loadtypelib?branch=master)<br/>                                         | Loads and registers a type library.<br/>                                                                                                                                                                       |
| [**LoadTypeLibEx**](/windows/previous-versions/OleAuto/nf-oleauto-loadtypelibex?branch=master)<br/>                                     | Loads a type library and (optionally) registers it in the system registry.   <br/>                                                                                                                             |
| [**OaEnablePerUserTLibRegistration**](/windows/previous-versions/OleAuto/nf-oleauto-oaenableperusertlibregistration?branch=master)<br/> | Enables the [**RegisterTypeLib**](/windows/previous-versions/OleAuto/nf-oleauto-registertypelib?branch=master) function to override default registry mappings under Windows Vista Service Pack 1 (SP1), Windows Server 2008, and later operating system versions.<br/> |
| [**QueryPathOfRegTypeLib**](/windows/previous-versions/OleAuto/nf-oleauto-querypathofregtypelib?branch=master)<br/>                     | Retrieves the path of a registered type library.<br/>                                                                                                                                                          |
| [**RegisterTypeLib**](/windows/previous-versions/OleAuto/nf-oleauto-registertypelib?branch=master)<br/>                                 | Adds information about a type library to the system registry.<br/>                                                                                                                                             |
| [**RegisterTypeLibForUser**](/windows/previous-versions/OleAuto/nf-oleauto-registertypelibforuser?branch=master)<br/>                   | Registers a type library for use by the calling user.<br/>                                                                                                                                                     |
| [**UnRegisterTypeLib**](/windows/previous-versions/OleAuto/nf-oleauto-unregistertypelib?branch=master)<br/>                             | Removes type library information from the system registry. Use this API to allow applications to properly uninstall themselves.<br/>                                                                           |
| [**UnRegisterTypeLibForUser**](/windows/previous-versions/OleAuto/nf-oleauto-unregistertypelibforuser?branch=master)<br/>               | Removes type library information that was registered by using [**RegisterTypeLibForUser**](/windows/previous-versions/OleAuto/nf-oleauto-registertypelibforuser?branch=master).<br/>                                                                                   |
