# ITypeInfo Data Types

This section contains information on data types used with the [**ITypeInfo**](/windows/previous-versions/oaidl/nn-oaidl-itypeinfo?branch=master) interface.

## In this section



| Topic                                   | Description                                                                                                                         |
|-----------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**HREFTYPE**](hreftype.md)<br/> | Represents a handle to a type description.<br/>                                                                               |
| [**MEMBERID**](memberid.md)<br/> | Identifies the member in a type description. For [**IDispatch**](/windows/previous-versions/oaidl/nn-oaidl-idispatch?branch=master) interfaces, this is the same as DISPID. <br/> |
