# ITypeInfo Data Types

This section contains information on data types used with the [**ITypeInfo**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypeinfo) interface.

## In this section



| Topic                                   | Description                                                                                                                         |
|-----------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**HREFTYPE**](hreftype.md)<br/> | Represents a handle to a type description.<br/>                                                                               |
| [**MEMBERID**](memberid.md)<br/> | Identifies the member in a type description. For [**IDispatch**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-idispatch) interfaces, this is the same as DISPID. <br/> |
