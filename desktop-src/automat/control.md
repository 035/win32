# control

Indicates that the item represents a control from which a container site will derive additional type libraries or coclasses.

## Allowed on

Type libraries, coclasses.

## Flags

<dl> TYPEFLAG\_FCONTROL  
LIBFLAG\_FCONTROL  
</dl>

## Remarks

This attribute allows type libraries that describe controls to be marked so that they are not displayed in type browsers intended for nonvisual objects.
