# lcid(numid)

Identifies the locale for a type library.

## Parameters

<dl> <dt>

<span id="numid"></span><span id="NUMID"></span>*numid*
</dt> <dd>

A 32-bit LCID. The LCID is typically entered in hexadecimal format.

</dd> </dl>

## Allowed on
