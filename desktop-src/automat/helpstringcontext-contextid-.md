# helpstringcontext(contextid)

Sets the string context in the Help file.

## Parameters

<dl> <dt>

<span id="contextid"></span><span id="CONTEXTID"></span>*contextid*
</dt> <dd>

The ID for the context string.

</dd> </dl>

## Allowed on

Type library, type information (TypeInfo), function, and variable level.

## Remarks

Retrieved by the **GetDocumentation2** functions in the [**ITypeLib**](itypelib.md) and [**ITypeInfo2**](itypeinfo2.md) interfaces. The *contextid* is typically a 32-bit Help context identifier in the Help file.
