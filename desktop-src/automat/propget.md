# propget

Specifies a property-accessor function.

## Allowed on

Functions, methods in interfaces, dispinterfaces.

## Flags

<dl> INVOKE\_PROPERTYGET  
</dl>

## Remarks

The property must have the same name as the function. At most, one of propget, [propput](propput.md), and [propputref](propputref.md) can be specified for a function.
