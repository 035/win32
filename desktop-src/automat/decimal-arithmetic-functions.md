# Decimal Arithmetic Functions

The decimal arithmetic functions are described in the following table.

## In this section



| Topic                                         | Description                                                                             |
|-----------------------------------------------|-----------------------------------------------------------------------------------------|
| [**VarDecAbs**](/windows/previous-versions/OleAuto/nf-oleauto-vardecabs?branch=master)<br/>     | Retrieves the absolute value of a variant of type decimal.<br/>                   |
| [**VarDecAdd**](/windows/previous-versions/OleAuto/nf-oleauto-vardecadd?branch=master)<br/>     | Adds two variants of type decimal.<br/>                                           |
| [**VarDecCmp**](/windows/previous-versions/OleAuto/nf-oleauto-vardeccmp?branch=master)<br/>     | Compares two variants of type decimal.<br/>                                       |
| [**VarDecCmpR8**](/windows/previous-versions/OleAuto/nf-oleauto-vardeccmpr8?branch=master)<br/> | Compares a variant of type decimal with the a value of type double.<br/>          |
| [**VarDecDiv**](/windows/previous-versions/OleAuto/nf-oleauto-vardecdiv?branch=master)<br/>     | Divides two variants of type decimal.<br/>                                        |
| [**VarDecFix**](/windows/previous-versions/OleAuto/nf-oleauto-vardecfix?branch=master)<br/>     | Retrieves the integer portion of a variant of type decimal.<br/>                  |
| [**VarDecInt**](/windows/previous-versions/OleAuto/nf-oleauto-vardecint?branch=master)<br/>     | Retrieves the integer portion of a variant of type decimal.<br/>                  |
| [**VarDecMul**](/windows/previous-versions/OleAuto/nf-oleauto-vardecmul?branch=master)<br/>     | Multiplies two variants of type decimal.<br/>                                     |
| [**VarDecNeg**](/windows/previous-versions/OleAuto/nf-oleauto-vardecneg?branch=master)<br/>     | Performs logical negation on a variant of type decimal. <br/>                     |
| [**VarDecRound**](/windows/previous-versions/OleAuto/nf-oleauto-vardecround?branch=master)<br/> | Rounds a variant of type decimal to the specified number of decimal places. <br/> |
| [**VarDecSub**](/windows/previous-versions/OleAuto/nf-oleauto-vardecsub?branch=master)<br/>     | Subtracts two variants of type decimal.<br/>                                      |
