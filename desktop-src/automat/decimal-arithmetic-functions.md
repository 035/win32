# Decimal Arithmetic Functions

The decimal arithmetic functions are described in the following table.

## In this section



| Topic                                         | Description                                                                             |
|-----------------------------------------------|-----------------------------------------------------------------------------------------|
| [**VarDecAbs**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardecabs)<br/>     | Retrieves the absolute value of a variant of type decimal.<br/>                   |
| [**VarDecAdd**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardecadd)<br/>     | Adds two variants of type decimal.<br/>                                           |
| [**VarDecCmp**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardeccmp)<br/>     | Compares two variants of type decimal.<br/>                                       |
| [**VarDecCmpR8**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardeccmpr8)<br/> | Compares a variant of type decimal with the a value of type double.<br/>          |
| [**VarDecDiv**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardecdiv)<br/>     | Divides two variants of type decimal.<br/>                                        |
| [**VarDecFix**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardecfix)<br/>     | Retrieves the integer portion of a variant of type decimal.<br/>                  |
| [**VarDecInt**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardecint)<br/>     | Retrieves the integer portion of a variant of type decimal.<br/>                  |
| [**VarDecMul**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardecmul)<br/>     | Multiplies two variants of type decimal.<br/>                                     |
| [**VarDecNeg**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardecneg)<br/>     | Performs logical negation on a variant of type decimal. <br/>                     |
| [**VarDecRound**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardecround)<br/> | Rounds a variant of type decimal to the specified number of decimal places. <br/> |
| [**VarDecSub**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-vardecsub)<br/>     | Subtracts two variants of type decimal.<br/>                                      |
