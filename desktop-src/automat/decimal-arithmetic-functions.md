# Decimal Arithmetic Functions

The decimal arithmetic functions are described in the following table.

## In this section



| Topic                                         | Description                                                                             |
|-----------------------------------------------|-----------------------------------------------------------------------------------------|
| [**VarDecAbs**](vardecabs.md)<br/>     | Retrieves the absolute value of a variant of type decimal.<br/>                   |
| [**VarDecAdd**](vardecadd.md)<br/>     | Adds two variants of type decimal.<br/>                                           |
| [**VarDecCmp**](vardeccmp.md)<br/>     | Compares two variants of type decimal.<br/>                                       |
| [**VarDecCmpR8**](vardeccmpr8.md)<br/> | Compares a variant of type decimal with the a value of type double.<br/>          |
| [**VarDecDiv**](vardecdiv.md)<br/>     | Divides two variants of type decimal.<br/>                                        |
| [**VarDecFix**](vardecfix.md)<br/>     | Retrieves the integer portion of a variant of type decimal.<br/>                  |
| [**VarDecInt**](vardecint.md)<br/>     | Retrieves the integer portion of a variant of type decimal.<br/>                  |
| [**VarDecMul**](vardecmul.md)<br/>     | Multiplies two variants of type decimal.<br/>                                     |
| [**VarDecNeg**](vardecneg.md)<br/>     | Performs logical negation on a variant of type decimal. <br/>                     |
| [**VarDecRound**](vardecround.md)<br/> | Rounds a variant of type decimal to the specified number of decimal places. <br/> |
| [**VarDecSub**](vardecsub.md)<br/>     | Subtracts two variants of type decimal.<br/>                                      |
