# The Property Frame Object

Property frame objects are the user interface mechanism that displays one or more property pages for a control. Automation provides a standard implementation of a property frame that can be accessed by using the [**OleCreatePropertyFrame**](https://msdn.microsoft.com/library/windows/desktop/ms678437) and [**OleCreatePropertyFrameIndirect**](https://msdn.microsoft.com/library/windows/desktop/ms693353) helper function.

For more information on these functions, see [**OleCreatePropertyFrame**](https://msdn.microsoft.com/library/windows/desktop/ms678437) and [**OleCreatePropertyFrameIndirect**](https://msdn.microsoft.com/library/windows/desktop/ms693353).
