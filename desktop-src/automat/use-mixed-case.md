# Use Mixed Case

All identifiers should use mixed case, rather than underscores, to separate words.



| Use           | Don't use                                                      |
|---------------|----------------------------------------------------------------|
| ShortcutMenus | Shortcut\_Menus, Shortcutmenus, SHORTCUTMENUS, SHORTCUT\_MENUS |
| BasedOn       | basedOn                                                        |
