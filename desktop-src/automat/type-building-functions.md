# Type Building Functions

This section contains reference information for the functions you can use to build type libraries.

## In this section



| Topic                                               | Description                                                                                                               |
|-----------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**CreateTypeLib**](/windows/previous-versions/OleAuto/nf-oleauto-createtypelib?branch=master)<br/>   | Provides access to a new object instance that supports the [**ICreateTypeLib**](/windows/previous-versions/oaidl/nn-oaidl-icreatetypelib?branch=master) interface.<br/> |
| [**CreateTypeLib2**](/windows/previous-versions/OleAuto/nf-oleauto-createtypelib2?branch=master)<br/> | Creates a type library in the current file format.<br/>                                                             |
| [**OaBuildVersion**](/windows/previous-versions/OleAuto/nf-oleauto-oabuildversion?branch=master)<br/> | Retrieves the build version of OLE Automation.<br/>                                                                 |
