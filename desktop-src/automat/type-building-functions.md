# Type Building Functions

This section contains reference information for the functions you can use to build type libraries.

## In this section



| Topic                                               | Description                                                                                                               |
|-----------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**CreateTypeLib**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-createtypelib)<br/>   | Provides access to a new object instance that supports the [**ICreateTypeLib**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-icreatetypelib) interface.<br/> |
| [**CreateTypeLib2**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-createtypelib2)<br/> | Creates a type library in the current file format.<br/>                                                             |
| [**OaBuildVersion**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-oabuildversion)<br/> | Retrieves the build version of OLE Automation.<br/>                                                                 |
