# Picture Functions

The following table describes the picture functions.

## In this section



| Topic                                                           | Description                                                                |
|-----------------------------------------------------------------|----------------------------------------------------------------------------|
| [**OleLoadPictureFile**](/windows/desktop/api/OleCtl/nf-olectl-oleloadpicturefile)<br/>     | Creates an **IPictureDisp** object from a picture file on disk.<br/> |
| [**OleLoadPictureFileEx**](/windows/desktop/api/OleCtl/nf-olectl-oleloadpicturefileex)<br/> | Loads a picture from a file.<br/>                                    |
| [**OleSavePictureFile**](/windows/desktop/api/OleCtl/nf-olectl-olesavepicturefile)<br/>     | Saves a picture to a file.<br/>                                      |
