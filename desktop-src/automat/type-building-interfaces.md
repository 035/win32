# Type Building Interfaces

This section contains reference information for the interfaces you can use to build type libraries.

## In this section



| Topic                                                     | Description                                                                                                                                                                                                                                                                   |
|-----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ICreateTypeInfo**](icreatetypeinfo.md)<br/>     | Provides the tools for creating and administering the type information defined through the type description.<br/>                                                                                                                                                       |
| [**ICreateTypeInfo2**](icreatetypeinfo2.md)<br/>   | Provides the tools for creating and administering the type information defined through the type description.<br/>                                                                                                                                                       |
| [**ICreateTypeLib**](icreatetypelib.md)<br/>       | Provides the methods for creating and managing the component or file that contains type information.<br/>                                                                                                                                                               |
| [**ICreateTypeLib2**](icreatetypelib2.md)<br/>     | Provides the methods for creating and managing the component or file that contains type information.<br/>                                                                                                                                                               |
| [**ITypeChangeEvents**](itypechangeevents.md)<br/> | Enables clients to subscribe to type change notifications on objects that implement the [**ITypeInfo**](itypeinfo.md), [**ITypeInfo2**](itypeinfo2.md), [**ICreateTypeInfo**](icreatetypeinfo.md), and [**ICreateTypeInfo2**](icreatetypeinfo2.md) interfaces.<br/> |
