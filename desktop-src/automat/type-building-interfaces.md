# Type Building Interfaces

This section contains reference information for the interfaces you can use to build type libraries.

## In this section



| Topic                                                     | Description                                                                                                                                                                                                                                                                   |
|-----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ICreateTypeInfo**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-icreatetypeinfo)<br/>     | Provides the tools for creating and administering the type information defined through the type description.<br/>                                                                                                                                                       |
| [**ICreateTypeInfo2**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-icreatetypeinfo2)<br/>   | Provides the tools for creating and administering the type information defined through the type description.<br/>                                                                                                                                                       |
| [**ICreateTypeLib**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-icreatetypelib)<br/>       | Provides the methods for creating and managing the component or file that contains type information.<br/>                                                                                                                                                               |
| [**ICreateTypeLib2**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-icreatetypelib2)<br/>     | Provides the methods for creating and managing the component or file that contains type information.<br/>                                                                                                                                                               |
| [**ITypeChangeEvents**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypechangeevents)<br/> | Enables clients to subscribe to type change notifications on objects that implement the [**ITypeInfo**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypeinfo), [**ITypeInfo2**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypeinfo2), [**ICreateTypeInfo**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-icreatetypeinfo), and [**ICreateTypeInfo2**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-icreatetypeinfo2) interfaces.<br/> |
