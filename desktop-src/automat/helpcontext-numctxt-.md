# helpcontext(numctxt)

Sets the context in the Help file.

## Parameters

<dl> <dt>

<span id="numctxt"></span><span id="NUMCTXT"></span>*numctxt*
</dt> <dd>

A 32-bit Help context identifier in the Help file.

</dd> </dl>

## Allowed on

Library, interface, dispinterface, struct, enum, union, module, typedef, method, struct member, enum value, property, coclass, const.

## Remarks

Retrieved by the [GetDocumentation](https://msdn.microsoft.com/windows/desktop/AA65E143-47DB-4241-9C66-FE3A1DCF1F0A) functions in the [**ITypeLib**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypelib) and [**ITypeInfo**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypeinfo) interfaces.
