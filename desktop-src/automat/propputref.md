# propputref

Specifies a property-setting function that uses a reference instead of a value.

## Allowed on

Functions, methods in interfaces, dispinterfaces.

## Flags

<dl> INVOKE\_PROPERTYPUTREF  
</dl>

## Remarks

The property must have the same name as the function. Only one [propget](propget.md), [propput](propput.md), and propputref can be specified.
