# Currency Arithmetic Functions

The currency arithmetic functions are described in the following table.

## In this section



| Topic                                       | Description                                                                             |
|---------------------------------------------|-----------------------------------------------------------------------------------------|
| [**VarCyAbs**](/windows/previous-versions/OleAuto/nf-oleauto-varcyabs?branch=master)<br/>     | Retrieves the absolute value of a variant of type currency.<br/>                  |
| [**VarCyAdd**](/windows/previous-versions/OleAuto/nf-oleauto-varcyadd?branch=master)<br/>     | Adds two variants of type currency.<br/>                                          |
| [**VarCyCmp**](/windows/previous-versions/OleAuto/nf-oleauto-varcycmp?branch=master)<br/>     | Compares two variants of type currency.<br/>                                      |
| [**VarCyCmpR8**](/windows/previous-versions/OleAuto/nf-oleauto-varcycmpr8?branch=master)<br/> | Compares a variant of type currency with a value of type double.<br/>             |
| [**VarCyFix**](/windows/previous-versions/OleAuto/nf-oleauto-varcyfix?branch=master)<br/>     | Retrieves the integer portion of a variant of type currency.<br/>                 |
| [**VarCyInt**](/windows/previous-versions/OleAuto/nf-oleauto-varcyint?branch=master)<br/>     | Retrieves the integer portion of a variant of type currency.<br/>                 |
| [**VarCyMul**](/windows/previous-versions/OleAuto/nf-oleauto-varcymul?branch=master)<br/>     | Multiplies two variants of type currency.<br/>                                    |
| [**VarCyMulI4**](/windows/previous-versions/OleAuto/nf-oleauto-varcymuli4?branch=master)<br/> | Multiplies a currency value by a 32-bit integer.<br/>                             |
| [**VarCyMulI8**](/windows/previous-versions/OleAuto/nf-oleauto-varcymuli8?branch=master)<br/> | Multiplies a currency value by a 64-bit integer.<br/>                             |
| [**VarCyNeg**](/windows/previous-versions/OleAuto/nf-oleauto-varcyneg?branch=master)<br/>     | Performs a logical negation on a variant of type currency.<br/>                   |
| [**VarCyRound**](/windows/previous-versions/OleAuto/nf-oleauto-varcyround?branch=master)<br/> | Rounds a variant of type currency to the specified number of decimal places.<br/> |
| [**VarCySub**](/windows/previous-versions/OleAuto/nf-oleauto-varcysub?branch=master)<br/>     | Subtracts two variants of type currency.<br/>                                     |
