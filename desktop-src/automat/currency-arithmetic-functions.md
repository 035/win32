# Currency Arithmetic Functions

The currency arithmetic functions are described in the following table.

## In this section



| Topic                                       | Description                                                                             |
|---------------------------------------------|-----------------------------------------------------------------------------------------|
| [**VarCyAbs**](varcyabs.md)<br/>     | Retrieves the absolute value of a variant of type currency.<br/>                  |
| [**VarCyAdd**](varcyadd.md)<br/>     | Adds two variants of type currency.<br/>                                          |
| [**VarCyCmp**](varcycmp.md)<br/>     | Compares two variants of type currency.<br/>                                      |
| [**VarCyCmpR8**](varcycmpr8.md)<br/> | Compares a variant of type currency with a value of type double.<br/>             |
| [**VarCyFix**](varcyfix.md)<br/>     | Retrieves the integer portion of a variant of type currency.<br/>                 |
| [**VarCyInt**](varcyint.md)<br/>     | Retrieves the integer portion of a variant of type currency.<br/>                 |
| [**VarCyMul**](varcymul.md)<br/>     | Multiplies two variants of type currency.<br/>                                    |
| [**VarCyMulI4**](varcymuli4.md)<br/> | Multiplies a currency value by a 32-bit integer.<br/>                             |
| [**VarCyMulI8**](varcymuli8.md)<br/> | Multiplies a currency value by a 64-bit integer.<br/>                             |
| [**VarCyNeg**](varcyneg.md)<br/>     | Performs a logical negation on a variant of type currency.<br/>                   |
| [**VarCyRound**](varcyround.md)<br/> | Rounds a variant of type currency to the specified number of decimal places.<br/> |
| [**VarCySub**](varcysub.md)<br/>     | Subtracts two variants of type currency.<br/>                                     |
