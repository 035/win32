# Currency Arithmetic Functions

The currency arithmetic functions are described in the following table.

## In this section



| Topic                                       | Description                                                                             |
|---------------------------------------------|-----------------------------------------------------------------------------------------|
| [**VarCyAbs**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcyabs)<br/>     | Retrieves the absolute value of a variant of type currency.<br/>                  |
| [**VarCyAdd**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcyadd)<br/>     | Adds two variants of type currency.<br/>                                          |
| [**VarCyCmp**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcycmp)<br/>     | Compares two variants of type currency.<br/>                                      |
| [**VarCyCmpR8**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcycmpr8)<br/> | Compares a variant of type currency with a value of type double.<br/>             |
| [**VarCyFix**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcyfix)<br/>     | Retrieves the integer portion of a variant of type currency.<br/>                 |
| [**VarCyInt**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcyint)<br/>     | Retrieves the integer portion of a variant of type currency.<br/>                 |
| [**VarCyMul**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcymul)<br/>     | Multiplies two variants of type currency.<br/>                                    |
| [**VarCyMulI4**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcymuli4)<br/> | Multiplies a currency value by a 32-bit integer.<br/>                             |
| [**VarCyMulI8**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcymuli8)<br/> | Multiplies a currency value by a 64-bit integer.<br/>                             |
| [**VarCyNeg**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcyneg)<br/>     | Performs a logical negation on a variant of type currency.<br/>                   |
| [**VarCyRound**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcyround)<br/> | Rounds a variant of type currency to the specified number of decimal places.<br/> |
| [**VarCySub**](/previous-versions/windows/desktop/api/OleAuto/nf-oleauto-varcysub)<br/>     | Subtracts two variants of type currency.<br/>                                     |
