# displaybind

Indicates that a property should be displayed as bindable to the user.

## Allowed on

Property.

## Flags

<dl> FUNCFLAG\_FDISPLAYBIND  
VARFLAG\_FDISPLAYBIND  
</dl>

## Remarks

Properties that have the displaybindattribute must also have the [bindable](https://msdn.microsoft.com/windows/desktop/D5A4DABD-5271-4AD4-81A5-739EFE062C03) attribute. An object can support data binding and not have this attribute.
