# MEMBERID

Identifies the member in a type description. For [**IDispatch**](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-idispatch) interfaces, this is the same as DISPID.


```C++
typedef DISPID MEMBERID;
```
