# Region Functions

The following functions are used with regions.



| Function                                                       | Description                                                                                  |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**CombineRgn**](combinergn.md)                               | Combines two regions and stores the result in a third region.                                |
| [**CreateEllipticRgn**](createellipticrgn.md)                 | Creates an elliptical region.                                                                |
| [**CreateEllipticRgnIndirect**](createellipticrgnindirect.md) | Creates an elliptical region.                                                                |
| [**CreatePolygonRgn**](createpolygonrgn.md)                   | Creates a polygonal region.                                                                  |
| [**CreatePolyPolygonRgn**](createpolypolygonrgn.md)           | Creates a region consisting of a series of polygons.                                         |
| [**CreateRectRgn**](createrectrgn.md)                         | Creates a rectangular region.                                                                |
| [**CreateRectRgnIndirect**](createrectrgnindirect.md)         | Creates a rectangular region.                                                                |
| [**CreateRoundRectRgn**](createroundrectrgn.md)               | Creates a rectangular region with rounded corners.                                           |
| [**EqualRgn**](equalrgn.md)                                   | Checks the two specified regions to determine whether they are identical.                    |
| [**ExtCreateRegion**](extcreateregion.md)                     | Creates a region from the specified region and transformation data.                          |
| [**FillRgn**](fillrgn.md)                                     | Fills a region by using the specified brush.                                                 |
| [**FrameRgn**](framergn.md)                                   | Draws a border around the specified region by using the specified brush.                     |
| [**GetPolyFillMode**](getpolyfillmode.md)                     | Retrieves the current polygon fill mode.                                                     |
| [**GetRegionData**](getregiondata.md)                         | Fills the specified buffer with data describing a region.                                    |
| [**GetRgnBox**](getrgnbox.md)                                 | Retrieves the bounding rectangle of the specified region.                                    |
| [**InvertRgn**](invertrgn.md)                                 | Inverts the colors in the specified region.                                                  |
| [**OffsetRgn**](offsetrgn.md)                                 | Moves a region by the specified offsets.                                                     |
| [**PaintRgn**](paintrgn.md)                                   | Paints the specified region by using the brush currently selected into the device context.   |
| [**PtInRegion**](ptinregion.md)                               | Determines whether the specified point is inside the specified region.                       |
| [**RectInRegion**](rectinregion.md)                           | Determines whether any part of the specified rectangle is within the boundaries of a region. |
| [**SetPolyFillMode**](setpolyfillmode.md)                     | Sets the polygon fill mode for functions that fill polygons.                                 |
| [**SetRectRgn**](setrectrgn.md)                               | Converts a region into a rectangular region with the specified coordinates.                  |
