# Region Functions

The following functions are used with regions.



| Function                                                       | Description                                                                                  |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**CombineRgn**](/windows/win32/Wingdi/nf-wingdi-combinergn?branch=master)                               | Combines two regions and stores the result in a third region.                                |
| [**CreateEllipticRgn**](/windows/win32/Wingdi/nf-wingdi-createellipticrgn?branch=master)                 | Creates an elliptical region.                                                                |
| [**CreateEllipticRgnIndirect**](/windows/win32/Wingdi/nf-wingdi-createellipticrgnindirect?branch=master) | Creates an elliptical region.                                                                |
| [**CreatePolygonRgn**](/windows/win32/Wingdi/nf-wingdi-createpolygonrgn?branch=master)                   | Creates a polygonal region.                                                                  |
| [**CreatePolyPolygonRgn**](/windows/win32/Wingdi/nf-wingdi-createpolypolygonrgn?branch=master)           | Creates a region consisting of a series of polygons.                                         |
| [**CreateRectRgn**](/windows/win32/Wingdi/nf-wingdi-createrectrgn?branch=master)                         | Creates a rectangular region.                                                                |
| [**CreateRectRgnIndirect**](/windows/win32/Wingdi/nf-wingdi-createrectrgnindirect?branch=master)         | Creates a rectangular region.                                                                |
| [**CreateRoundRectRgn**](/windows/win32/Wingdi/nf-wingdi-createroundrectrgn?branch=master)               | Creates a rectangular region with rounded corners.                                           |
| [**EqualRgn**](/windows/win32/Wingdi/nf-wingdi-equalrgn?branch=master)                                   | Checks the two specified regions to determine whether they are identical.                    |
| [**ExtCreateRegion**](/windows/win32/Wingdi/nf-wingdi-extcreateregion?branch=master)                     | Creates a region from the specified region and transformation data.                          |
| [**FillRgn**](/windows/win32/Wingdi/nf-wingdi-fillrgn?branch=master)                                     | Fills a region by using the specified brush.                                                 |
| [**FrameRgn**](/windows/win32/Wingdi/nf-wingdi-framergn?branch=master)                                   | Draws a border around the specified region by using the specified brush.                     |
| [**GetPolyFillMode**](/windows/win32/Wingdi/nf-wingdi-getpolyfillmode?branch=master)                     | Retrieves the current polygon fill mode.                                                     |
| [**GetRegionData**](/windows/win32/Wingdi/nf-wingdi-getregiondata?branch=master)                         | Fills the specified buffer with data describing a region.                                    |
| [**GetRgnBox**](/windows/win32/Wingdi/nf-wingdi-getrgnbox?branch=master)                                 | Retrieves the bounding rectangle of the specified region.                                    |
| [**InvertRgn**](/windows/win32/Wingdi/nf-wingdi-invertrgn?branch=master)                                 | Inverts the colors in the specified region.                                                  |
| [**OffsetRgn**](/windows/win32/Wingdi/nf-wingdi-offsetrgn?branch=master)                                 | Moves a region by the specified offsets.                                                     |
| [**PaintRgn**](/windows/win32/Wingdi/nf-wingdi-paintrgn?branch=master)                                   | Paints the specified region by using the brush currently selected into the device context.   |
| [**PtInRegion**](/windows/win32/Wingdi/nf-wingdi-ptinregion?branch=master)                               | Determines whether the specified point is inside the specified region.                       |
| [**RectInRegion**](/windows/win32/Wingdi/nf-wingdi-rectinregion?branch=master)                           | Determines whether any part of the specified rectangle is within the boundaries of a region. |
| [**SetPolyFillMode**](/windows/win32/Wingdi/nf-wingdi-setpolyfillmode?branch=master)                     | Sets the polygon fill mode for functions that fill polygons.                                 |
| [**SetRectRgn**](/windows/win32/Wingdi/nf-wingdi-setrectrgn?branch=master)                               | Converts a region into a rectangular region with the specified coordinates.                  |
