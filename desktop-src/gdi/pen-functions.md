# Pen Functions

The following functions are used with pens.



| Function                                       | Description                                                                                            |
|------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**CreatePen**](createpen.md)                 | Creates a logical pen that has the specified style, width, and color.                                  |
| [**CreatePenIndirect**](createpenindirect.md) | Creates a logical cosmetic pen that has the style, width, and color specified in a structure.          |
| [**ExtCreatePen**](extcreatepen.md)           | Creates a logical cosmetic or geometric pen that has the specified style, width, and brush attributes. |
| [**SetDCPenColor**](setdcpencolor.md)         | Sets the current device context pen color.                                                             |
