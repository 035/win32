# Pen Functions

The following functions are used with pens.



| Function                                       | Description                                                                                            |
|------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**CreatePen**](/windows/win32/Wingdi/nf-wingdi-createpen?branch=master)                 | Creates a logical pen that has the specified style, width, and color.                                  |
| [**CreatePenIndirect**](/windows/win32/Wingdi/nf-wingdi-createpenindirect?branch=master) | Creates a logical cosmetic pen that has the style, width, and color specified in a structure.          |
| [**ExtCreatePen**](/windows/win32/Wingdi/nf-wingdi-extcreatepen?branch=master)           | Creates a logical cosmetic or geometric pen that has the specified style, width, and brush attributes. |
| [**SetDCPenColor**](/windows/win32/Wingdi/nf-wingdi-setdcpencolor?branch=master)         | Sets the current device context pen color.                                                             |
