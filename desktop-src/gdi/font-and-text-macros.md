# Font and Text Macros

The following macros are used with fonts and text.



| Macro                            | Description                                            |
|----------------------------------|--------------------------------------------------------|
| [**DeleteFont**](/windows/win32/Windowsx/nf-windowsx-deletefont?branch=master) | Deletes a specified font.                              |
| [**SelectFont**](/windows/win32/Windowsx/nf-windowsx-selectfont?branch=master) | Selects a font into the specified device context (DC). |
