# Filled Shape Functions

The following functions are used with filled shapes.



| Function                           | Description                                                     |
|------------------------------------|-----------------------------------------------------------------|
| [**Chord**](chord.md)             | Draws an area bounded by an ellipse and a line segment.         |
| [**Ellipse**](ellipse.md)         | Draws an ellipse.                                               |
| [**FillRect**](fillrect.md)       | Fills a rectangle using a brush.                                |
| [**FrameRect**](framerect.md)     | Draws a border around a rectangle using a brush.                |
| [**InvertRect**](invertrect.md)   | Inverts the color values of the pixels in a rectangle.          |
| [**Pie**](pie.md)                 | Draws a pie-shaped wedge bounded by an ellipse and two radials. |
| [**Polygon**](polygon.md)         | Draws a polygon.                                                |
| [**PolyPolygon**](polypolygon.md) | draws a series of closed polygons.                              |
| [**Rectangle**](rectangle.md)     | Draws a rectangle.                                              |
| [**RoundRect**](roundrect.md)     | Draws a rectangle with rounded corners.                         |
