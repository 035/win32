# Filled Shape Functions

The following functions are used with filled shapes.



| Function                           | Description                                                     |
|------------------------------------|-----------------------------------------------------------------|
| [**Chord**](/windows/win32/Wingdi/nf-wingdi-chord?branch=master)             | Draws an area bounded by an ellipse and a line segment.         |
| [**Ellipse**](/windows/win32/Wingdi/nf-wingdi-ellipse?branch=master)         | Draws an ellipse.                                               |
| [**FillRect**](/windows/win32/Winuser/nf-winuser-fillrect?branch=master)       | Fills a rectangle using a brush.                                |
| [**FrameRect**](/windows/win32/Winuser/nf-winuser-framerect?branch=master)     | Draws a border around a rectangle using a brush.                |
| [**InvertRect**](/windows/win32/Winuser/nf-winuser-invertrect?branch=master)   | Inverts the color values of the pixels in a rectangle.          |
| [**Pie**](/windows/win32/Wingdi/nf-wingdi-pie?branch=master)                 | Draws a pie-shaped wedge bounded by an ellipse and two radials. |
| [**Polygon**](/windows/win32/Wingdi/nf-wingdi-polygon?branch=master)         | Draws a polygon.                                                |
| [**PolyPolygon**](/windows/win32/Wingdi/nf-wingdi-polypolygon?branch=master) | draws a series of closed polygons.                              |
| [**Rectangle**](/windows/win32/Wingdi/nf-wingdi-rectangle?branch=master)     | Draws a rectangle.                                              |
| [**RoundRect**](/windows/win32/Wingdi/nf-wingdi-roundrect?branch=master)     | Draws a rectangle with rounded corners.                         |
