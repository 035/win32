# Brush Functions

The following functions are used with brushes.



| Function                                                   | Description                                                |
|------------------------------------------------------------|------------------------------------------------------------|
| [**CreateBrushIndirect**](/windows/win32/Wingdi/nf-wingdi-createbrushindirect?branch=master)         | Creates a brush with a specified style, color, and pattern |
| [**CreateDIBPatternBrushPt**](/windows/win32/Wingdi/nf-wingdi-createdibpatternbrushpt?branch=master) | Creates a brush with the pattern from a DIB                |
| [**CreateHatchBrush**](/windows/win32/Wingdi/nf-wingdi-createhatchbrush?branch=master)               | Creates a brush with a hatch pattern and color             |
| [**CreatePatternBrush**](/windows/win32/Wingdi/nf-wingdi-createpatternbrush?branch=master)           | Creates a brush with a bitmap pattern                      |
| [**CreateSolidBrush**](/windows/win32/Wingdi/nf-wingdi-createsolidbrush?branch=master)               | Creates a brush with a solid color                         |
| [**GetBrushOrgEx**](/windows/win32/Wingdi/nf-wingdi-getbrushorgex?branch=master)                     | Gets the brush origin for a device context                 |
| [**GetSysColorBrush**](/windows/win32/Winuser/nf-winuser-getsyscolorbrush?branch=master)               | Gets a handle to a brush that corresponds to a color index |
| [**PatBlt**](/windows/win32/Wingdi/nf-wingdi-patblt?branch=master)                                   | Paints a rectangle                                         |
| [**SetBrushOrgEx**](/windows/win32/Wingdi/nf-wingdi-setbrushorgex?branch=master)                     | Sets the brush origin for a device context                 |
| [**SetDCBrushColor**](/windows/win32/Wingdi/nf-wingdi-setdcbrushcolor?branch=master)                 | Sets the current device context brush color.               |
