# Brush Functions

The following functions are used with brushes.



| Function                                                   | Description                                                |
|------------------------------------------------------------|------------------------------------------------------------|
| [**CreateBrushIndirect**](createbrushindirect.md)         | Creates a brush with a specified style, color, and pattern |
| [**CreateDIBPatternBrushPt**](createdibpatternbrushpt.md) | Creates a brush with the pattern from a DIB                |
| [**CreateHatchBrush**](createhatchbrush.md)               | Creates a brush with a hatch pattern and color             |
| [**CreatePatternBrush**](createpatternbrush.md)           | Creates a brush with a bitmap pattern                      |
| [**CreateSolidBrush**](createsolidbrush.md)               | Creates a brush with a solid color                         |
| [**GetBrushOrgEx**](getbrushorgex.md)                     | Gets the brush origin for a device context                 |
| [**GetSysColorBrush**](getsyscolorbrush.md)               | Gets a handle to a brush that corresponds to a color index |
| [**PatBlt**](patblt.md)                                   | Paints a rectangle                                         |
| [**SetBrushOrgEx**](setbrushorgex.md)                     | Sets the brush origin for a device context                 |
| [**SetDCBrushColor**](setdcbrushcolor.md)                 | Sets the current device context brush color.               |
