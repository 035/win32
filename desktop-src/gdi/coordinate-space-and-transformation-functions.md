# Coordinate Space and Transformation Functions

The following functions are used with coordinate spaces and transformations.



| Function                                                                       | Description                                                                                                                      |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**ClientToScreen**](clienttoscreen.md)                                       | Converts the client-area coordinates of a specified point to screen coordinates.                                                 |
| [**CombineTransform**](combinetransform.md)                                   | Concatenates two world-space to page-space transformations.                                                                      |
| [**DPtoLP**](dptolp.md)                                                       | Converts device coordinates into logical coordinates.                                                                            |
| [**GetCurrentPositionEx**](getcurrentpositionex.md)                           | Retrieves the current position in logical coordinates.                                                                           |
| [**GetDisplayAutoRotationPreferences**](getdisplayautorotationpreferences.md) | Gets the orientation preferences of the display.                                                                                 |
| [**GetGraphicsMode**](getgraphicsmode.md)                                     | Retrieves the current graphics mode for the specified device context.                                                            |
| [**GetMapMode**](getmapmode.md)                                               | Retrieves the current mapping mode.                                                                                              |
| [**GetViewportExtEx**](getviewportextex.md)                                   | Retrieves the x-extent and y-extent of the current viewport for the specified device context.                                    |
| [**GetViewportOrgEx**](getviewportorgex.md)                                   | Retrieves the x-coordinates and y-coordinates of the viewport origin for the specified device context.                           |
| [**GetWindowExtEx**](getwindowextex.md)                                       | Retrieves the x-extent and y-extent of the window for the specified device context.                                              |
| [**GetWindowOrgEx**](getwindoworgex.md)                                       | Retrieves the x-coordinates and y-coordinates of the window origin for the specified device context.                             |
| [**GetWorldTransform**](getworldtransform.md)                                 | Retrieves the current world-space to page-space transformation.                                                                  |
| [**LPtoDP**](lptodp.md)                                                       | Converts logical coordinates into device coordinates.                                                                            |
| [**MapWindowPoints**](mapwindowpoints.md)                                     | Converts (maps) a set of points from a coordinate space relative to one window to a coordinate space relative to another window. |
| [**ModifyWorldTransform**](modifyworldtransform.md)                           | Changes the world transformation for a device context using the specified mode.                                                  |
| [**OffsetViewportOrgEx**](offsetviewportorgex.md)                             | Modifies the viewport origin for a device context using the specified horizontal and vertical offsets.                           |
| [**OffsetWindowOrgEx**](offsetwindoworgex.md)                                 | Modifies the window origin for a device context using the specified horizontal and vertical offsets.                             |
| [**ScaleViewportExtEx**](scaleviewportextex.md)                               | Modifies the viewport for a device context using the ratios formed by the specified multiplicands and divisors.                  |
| [**ScaleWindowExtEx**](scalewindowextex.md)                                   | Modifies the window for a device context using the ratios formed by the specified multiplicands and divisors.                    |
| [**ScreenToClient**](screentoclient.md)                                       | Converts the screen coordinates of a specified point on the screen to client coordinates.                                        |
| [**SetDisplayAutoRotationPreferences**](setdisplayautorotationpreferences.md) | Sets the orientation preferences of the display.                                                                                 |
| [**SetGraphicsMode**](setgraphicsmode.md)                                     | Sets the graphics mode for the specified device context.                                                                         |
| [**SetMapMode**](setmapmode.md)                                               | Sets the mapping mode of the specified device context.                                                                           |
| [**SetViewportExtEx**](setviewportextex.md)                                   | Sets the horizontal and vertical extents of the viewport for a device context by using the specified values.                     |
| [**SetViewportOrgEx**](setviewportorgex.md)                                   | Specifies which device point maps to the window origin (0,0).                                                                    |
| [**SetWindowExtEx**](setwindowextex.md)                                       | Sets the horizontal and vertical extents of the window for a device context by using the specified values.                       |
| [**SetWindowOrgEx**](setwindoworgex.md)                                       | Specifies which window point maps to the viewport origin (0,0).                                                                  |
| [**SetWorldTransform**](setworldtransform.md)                                 | Sets a two-dimensional linear transformation between world space and page space for the specified device context.                |
