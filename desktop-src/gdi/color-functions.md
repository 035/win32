# Color Functions

The following functions are used with color.



| Function                                                   | Description                                                                                                                                           |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AnimatePalette**](animatepalette.md)                   | Replaces entries in the specified logical palette.                                                                                                    |
| [**CreateHalftonePalette**](createhalftonepalette.md)     | Creates a halftone palette for the specified device context (DC).                                                                                     |
| [**CreatePalette**](createpalette.md)                     | Creates a logical palette.                                                                                                                            |
| [**GetColorAdjustment**](getcoloradjustment.md)           | Retrieves the color adjustment values for the specified DC.                                                                                           |
| [**GetNearestColor**](getnearestcolor.md)                 | Retrieves a color value identifying a color from the system palette that will be displayed when the specified color value is used.                    |
| [**GetNearestPaletteIndex**](getnearestpaletteindex.md)   | Retrieves the index for the entry in the specified logical palette most closely matching a specified color value.                                     |
| [**GetPaletteEntries**](getpaletteentries.md)             | Retrieves a specified range of palette entries from the given logical palette.                                                                        |
| [**GetSystemPaletteEntries**](getsystempaletteentries.md) | Retrieves a range of palette entries from the system palette that is associated with the specified DC.                                                |
| [**GetSystemPaletteUse**](getsystempaletteuse.md)         | Retrieves the current state of the system (physical) palette for the specified DC.                                                                    |
| [**RealizePalette**](realizepalette.md)                   | Maps palette entries from the current logical palette to the system palette.                                                                          |
| [**ResizePalette**](resizepalette.md)                     | Increases or decreases the size of a logical palette based on the specified value.                                                                    |
| [**SelectPalette**](selectpalette.md)                     | Selects the specified logical palette into a device context.                                                                                          |
| [**SetColorAdjustment**](setcoloradjustment.md)           | Sets the color adjustment values for a DC using the specified values.                                                                                 |
| [**SetPaletteEntries**](setpaletteentries.md)             | Sets RGB (red, green, blue) color values and flags in a range of entries in a logical palette.                                                        |
| [**SetSystemPaletteUse**](setsystempaletteuse.md)         | Allows an application to specify whether the system palette contains 2 or 20 static colors.                                                           |
| [**UnrealizeObject**](unrealizeobject.md)                 | Resets the origin of a brush or resets a logical palette.                                                                                             |
| [**UpdateColors**](updatecolors.md)                       | Updates the client area of the specified device context by remapping the current colors in the client area to the currently realized logical palette. |
