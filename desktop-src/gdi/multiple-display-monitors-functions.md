# Multiple Display Monitors Functions

The following functions provide support for multiple monitors.



| Function                                           | Description                                                                                                                                                  |
|----------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EnumDisplayMonitors**](enumdisplaymonitors.md) | Enumerates display monitors that intersect a region formed by the intersection of a specified clipping rectangle and the visible region of a device context. |
| [**GetMonitorInfo**](getmonitorinfo.md)           | Retrieves information about a display monitor.                                                                                                               |
| [**MonitorEnumProc**](monitorenumproc.md)         | An application-defined callback function that is called by the [**EnumDisplayMonitors**](enumdisplaymonitors.md) function.                                  |
| [**MonitorFromPoint**](monitorfrompoint.md)       | Retrieves a handle to the display monitor that contains a specified point.                                                                                   |
| [**MonitorFromRect**](monitorfromrect.md)         | Retrieves a handle to the display monitor that has the largest area of intersection with a specified rectangle.                                              |
| [**MonitorFromWindow**](monitorfromwindow.md)     | Retrieves a handle to the display monitor that has the largest area of intersection with the bounding rectangle of a specified window.                       |
