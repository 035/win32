# Rectangle Functions

The following functions are used with rectangles.



| Function                               | Description                                                                                                                                   |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**CopyRect**](/windows/win32/Winuser/nf-winuser-copyrect?branch=master)           | Copies the coordinates of one rectangle to another.                                                                                           |
| [**EqualRect**](/windows/win32/Winuser/nf-winuser-equalrect?branch=master)         | Determines whether the two specified rectangles are equal by comparing the coordinates of their upper-left and lower-right corners.           |
| [**InflateRect**](/windows/win32/Winuser/nf-winuser-inflaterect?branch=master)     | Increases or decreases the width and height of the specified rectangle.                                                                       |
| [**IntersectRect**](/windows/win32/Winuser/nf-winuser-intersectrect?branch=master) | Calculates the intersection of two source rectangles and places the coordinates of the intersection rectangle into the destination rectangle. |
| [**IsRectEmpty**](/windows/win32/Winuser/nf-winuser-isrectempty?branch=master)     | Determines whether the specified rectangle is empty.                                                                                          |
| [**OffsetRect**](/windows/win32/Winuser/nf-winuser-offsetrect?branch=master)       | Moves the specified rectangle by the specified offsets.                                                                                       |
| [**PtInRect**](/windows/win32/Winuser/nf-winuser-ptinrect?branch=master)           | Determines whether the specified point lies within the specified rectangle.                                                                   |
| [**SetRect**](/windows/win32/Winuser/nf-winuser-setrect?branch=master)             | Sets the coordinates of the specified rectangle.                                                                                              |
| [**SetRectEmpty**](/windows/win32/Winuser/nf-winuser-setrectempty?branch=master)   | Creates an empty rectangle in which all coordinates are set to zero.                                                                          |
| [**SubtractRect**](/windows/win32/Winuser/nf-winuser-subtractrect?branch=master)   | Determines the coordinates of a rectangle formed by subtracting one rectangle from another.                                                   |
| [**UnionRect**](/windows/win32/Winuser/nf-winuser-unionrect?branch=master)         | Creates the union of two rectangles.                                                                                                          |
