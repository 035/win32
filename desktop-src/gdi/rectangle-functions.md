# Rectangle Functions

The following functions are used with rectangles.



| Function                               | Description                                                                                                                                   |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**CopyRect**](copyrect.md)           | Copies the coordinates of one rectangle to another.                                                                                           |
| [**EqualRect**](equalrect.md)         | Determines whether the two specified rectangles are equal by comparing the coordinates of their upper-left and lower-right corners.           |
| [**InflateRect**](inflaterect.md)     | Increases or decreases the width and height of the specified rectangle.                                                                       |
| [**IntersectRect**](intersectrect.md) | Calculates the intersection of two source rectangles and places the coordinates of the intersection rectangle into the destination rectangle. |
| [**IsRectEmpty**](isrectempty.md)     | Determines whether the specified rectangle is empty.                                                                                          |
| [**OffsetRect**](offsetrect.md)       | Moves the specified rectangle by the specified offsets.                                                                                       |
| [**PtInRect**](ptinrect.md)           | Determines whether the specified point lies within the specified rectangle.                                                                   |
| [**SetRect**](setrect.md)             | Sets the coordinates of the specified rectangle.                                                                                              |
| [**SetRectEmpty**](setrectempty.md)   | Creates an empty rectangle in which all coordinates are set to zero.                                                                          |
| [**SubtractRect**](subtractrect.md)   | Determines the coordinates of a rectangle formed by subtracting one rectangle from another.                                                   |
| [**UnionRect**](unionrect.md)         | Creates the union of two rectangles.                                                                                                          |
