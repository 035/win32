# Bitmap Functions

The following functions are used with bitmaps.



| Function                                                 | Description                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [**AlphaBlend**](alphablend.md)                         | Displays a bitmap with transparent or semitransparent pixels.  |
| [**BitBlt**](bitblt.md)                                 | Performs a bit-block transfer.                                 |
| [**CreateBitmap**](createbitmap.md)                     | Creates a bitmap.                                              |
| [**CreateBitmapIndirect**](createbitmapindirect.md)     | Creates a bitmap.                                              |
| [**CreateCompatibleBitmap**](createcompatiblebitmap.md) | Creates a bitmap compatible with a device.                     |
| [**CreateDIBitmap**](createdibitmap.md)                 | Creates a device-dependent bitmap (DDB) from a DIB.            |
| [**CreateDIBSection**](createdibsection.md)             | Creates a DIB that applications can write to directly.         |
| [**ExtFloodFill**](extfloodfill.md)                     | Fills an area of the display surface with the current brush.   |
| [**GetBitmapDimensionEx**](getbitmapdimensionex.md)     | Gets the dimensions of a bitmap.                               |
| [**GetDIBColorTable**](getdibcolortable.md)             | Retrieves RGB color values from a DIB section bitmap.          |
| [**GetDIBits**](getdibits.md)                           | Copies a bitmap into a buffer.                                 |
| [**GetPixel**](getpixel.md)                             | Gets the RGB color value of the pixel at a given coordinate.   |
| [**GetStretchBltMode**](getstretchbltmode.md)           | Gets the current stretching mode.                              |
| [**GradientFill**](gradientfill.md)                     | Fills rectangle and triangle structures.                       |
| [**LoadBitmap**](loadbitmap.md)                         | Loads a bitmap from a module's executable file.                |
| [**MaskBlt**](maskblt.md)                               | Combines the color data in the source and destination bitmaps. |
| [**PlgBlt**](plgblt.md)                                 | Performs a bit-block transfer.                                 |
| [**SetBitmapDimensionEx**](setbitmapdimensionex.md)     | Sets the preferred dimensions to a bitmap.                     |
| [**SetDIBColorTable**](setdibcolortable.md)             | Sets RGB values in a DIB.                                      |
| [**SetDIBits**](setdibits.md)                           | Sets the pixels in a bitmap using color data from a DIB.       |
| [**SetDIBitsToDevice**](setdibitstodevice.md)           | Sets the pixels in a rectangle using color data from a DIB.    |
| [**SetPixel**](setpixel.md)                             | Sets the color for a pixel.                                    |
| [**SetPixelV**](setpixelv.md)                           | Sets a pixel to the best approximation of a color.             |
| [**SetStretchBltMode**](setstretchbltmode.md)           | Sets the bitmap stretching mode.                               |
| [**StretchBlt**](stretchblt.md)                         | Copies a bitmap and stretches or compresses it.                |
| [**StretchDIBits**](stretchdibits.md)                   | Copies the color data in a DIB.                                |
| [**TransparentBlt**](transparentblt.md)                 | Performs a bit-block transfer of color data.                   |
