# Line and Curve Functions

The following functions are used with lines and curves.



| Function                                   | Description                                                                                                   |
|--------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**AngleArc**](/windows/win32/Wingdi/nf-wingdi-anglearc?branch=master)               | Draws a line segment and an arc.                                                                              |
| [**Arc**](/windows/win32/Wingdi/nf-wingdi-arc?branch=master)                         | Draws an elliptical arc.                                                                                      |
| [**ArcTo**](/windows/win32/Wingdi/nf-wingdi-arcto?branch=master)                     | Draws an elliptical arc.                                                                                      |
| [**GetArcDirection**](/windows/win32/Wingdi/nf-wingdi-getarcdirection?branch=master) | Retrieves the current arc direction for the specified device context.                                         |
| [**LineDDA**](/windows/win32/Wingdi/nf-wingdi-linedda?branch=master)                 | Determines which pixels should be highlighted for a line defined by the specified starting and ending points. |
| [**LineDDAProc**](/windows/win32/Wingdi/nc-wingdi-lineddaproc?branch=master)         | An application-defined callback function used with the LineDDA function.                                      |
| [**LineTo**](/windows/win32/Wingdi/nf-wingdi-lineto?branch=master)                   | Draws a line from the current position up to, but not including, the specified point.                         |
| [**MoveToEx**](/windows/win32/Wingdi/nf-wingdi-movetoex?branch=master)               | Updates the current position to the specified point and optionally returns the previous position.             |
| [**PolyBezier**](/windows/win32/Wingdi/nf-wingdi-polybezier?branch=master)           | Draws one or more Bézier curves.                                                                              |
| [**PolyBezierTo**](/windows/win32/Wingdi/nf-wingdi-polybezierto?branch=master)       | Draws one or more Bézier curves.                                                                              |
| [**PolyDraw**](/windows/win32/Wingdi/nf-wingdi-polydraw?branch=master)               | Draws a set of line segments and Bézier curves.                                                               |
| [**Polyline**](/windows/win32/Wingdi/nf-wingdi-polyline?branch=master)               | Draws a series of line segments by connecting the points in the specified array.                              |
| [**PolylineTo**](/windows/win32/Wingdi/nf-wingdi-polylineto?branch=master)           | Draws one or more straight lines.                                                                             |
| [**PolyPolyline**](/windows/win32/Wingdi/nf-wingdi-polypolyline?branch=master)       | Draws multiple series of connected line segments.                                                             |
| [**SetArcDirection**](/windows/win32/Wingdi/nf-wingdi-setarcdirection?branch=master) | Sets the drawing direction to be used for arc and rectangle functions.                                        |
