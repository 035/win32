# Line and Curve Functions

The following functions are used with lines and curves.



| Function                                   | Description                                                                                                   |
|--------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**AngleArc**](anglearc.md)               | Draws a line segment and an arc.                                                                              |
| [**Arc**](arc.md)                         | Draws an elliptical arc.                                                                                      |
| [**ArcTo**](arcto.md)                     | Draws an elliptical arc.                                                                                      |
| [**GetArcDirection**](getarcdirection.md) | Retrieves the current arc direction for the specified device context.                                         |
| [**LineDDA**](linedda.md)                 | Determines which pixels should be highlighted for a line defined by the specified starting and ending points. |
| [**LineDDAProc**](lineddaproc.md)         | An application-defined callback function used with the LineDDA function.                                      |
| [**LineTo**](lineto.md)                   | Draws a line from the current position up to, but not including, the specified point.                         |
| [**MoveToEx**](movetoex.md)               | Updates the current position to the specified point and optionally returns the previous position.             |
| [**PolyBezier**](polybezier.md)           | Draws one or more Bézier curves.                                                                              |
| [**PolyBezierTo**](polybezierto.md)       | Draws one or more Bézier curves.                                                                              |
| [**PolyDraw**](polydraw.md)               | Draws a set of line segments and Bézier curves.                                                               |
| [**Polyline**](polyline.md)               | Draws a series of line segments by connecting the points in the specified array.                              |
| [**PolylineTo**](polylineto.md)           | Draws one or more straight lines.                                                                             |
| [**PolyPolyline**](polypolyline.md)       | Draws multiple series of connected line segments.                                                             |
| [**SetArcDirection**](setarcdirection.md) | Sets the drawing direction to be used for arc and rectangle functions.                                        |
