# Querying State Variables

The [**IUPnPService**](/windows/win32/Upnp/nn-upnp-iupnpservice?branch=master) interface provides the [**QueryStateVariable**](/windows/win32/Upnp/nf-upnp-iupnpservice-querystatevariable?branch=master) method, that returns the value of a specified state variable.

The UPnP Forum discourages the use of [**QueryStateVariable**](/windows/win32/Upnp/nf-upnp-iupnpservice-querystatevariable?branch=master). Device writers have been encouraged to write appropriate actions to provide this information. Device writers are under no obligation to implement **QueryStateVariable**.

See the section [Invoking Actions](invoking-actions.md).
