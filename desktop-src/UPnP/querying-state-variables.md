# Querying State Variables

The [**IUPnPService**](iupnpservice.md) interface provides the [**QueryStateVariable**](iupnpservice-querystatevariable.md) method, that returns the value of a specified state variable.

The UPnP Forum discourages the use of [**QueryStateVariable**](iupnpservice-querystatevariable.md). Device writers have been encouraged to write appropriate actions to provide this information. Device writers are under no obligation to implement **QueryStateVariable**.

See the section [Invoking Actions](invoking-actions.md).
