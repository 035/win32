# Device Host API Reference

The following interfaces are part of the Device Host API with UPnP technology. The Device Host with UPnP technology supports Microsoft Visual Basic and C++.

This API does not provide support for Microsoft Visual Basic Scripting Edition (VBScript).



| Interface                                                  | Description                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**IUPnPDeviceControl**](/windows/win32/Upnphost/nn-upnphost-iupnpdevicecontrol?branch=master)           | Used by the device host to manage devices and related services.                                     |
| [**IUPnPDeviceProvider**](/windows/win32/Upnphost/nn-upnphost-iupnpdeviceprovider?branch=master)         | Used by the device host to start and stop device providers.                                         |
| [**IUPnPEventSink**](/windows/win32/Upnphost/nn-upnphost-iupnpeventsink?branch=master)                   | Used by devices to send event notifications to the device host.                                     |
| [**IUPnPEventSource**](/windows/win32/Upnphost/nn-upnphost-iupnpeventsource?branch=master)               | Used by the device host to subscribe or unsubscribe to events provided by the hosted service.       |
| [**IUPnPRegistrar**](/windows/win32/Upnphost/nn-upnphost-iupnpregistrar?branch=master)                   | Used by devices to register with the device host.                                                   |
| [**IUPnPRemoteEndpointInfo**](/windows/win32/Upnphost/nn-upnphost-iupnpremoteendpointinfo?branch=master) | Used by devices to obtain information about a requester (that is, a control point) and the request. |
| [**IUPnPReregistrar**](/windows/win32/Upnphost/nn-upnphost-iupnpreregistrar?branch=master)               | Used by devices to re-register with the device host.                                                |
