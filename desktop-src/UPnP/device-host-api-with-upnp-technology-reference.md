# Device Host API Reference

The following interfaces are part of the Device Host API with UPnP technology. The Device Host with UPnP technology supports Microsoft Visual Basic and C++.

This API does not provide support for Microsoft Visual Basic Scripting Edition (VBScript).



| Interface                                                  | Description                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**IUPnPDeviceControl**](iupnpdevicecontrol.md)           | Used by the device host to manage devices and related services.                                     |
| [**IUPnPDeviceProvider**](iupnpdeviceprovider.md)         | Used by the device host to start and stop device providers.                                         |
| [**IUPnPEventSink**](iupnpeventsink.md)                   | Used by devices to send event notifications to the device host.                                     |
| [**IUPnPEventSource**](iupnpeventsource.md)               | Used by the device host to subscribe or unsubscribe to events provided by the hosted service.       |
| [**IUPnPRegistrar**](iupnpregistrar.md)                   | Used by devices to register with the device host.                                                   |
| [**IUPnPRemoteEndpointInfo**](iupnpremoteendpointinfo.md) | Used by devices to obtain information about a requester (that is, a control point) and the request. |
| [**IUPnPReregistrar**](iupnpreregistrar.md)               | Used by devices to re-register with the device host.                                                |
