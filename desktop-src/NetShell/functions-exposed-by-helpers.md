# Functions Exposed by Helpers

The following functions are implemented by Helpers, and used by NetShell for various tasks such as initializing helpers, registering helpers with Windows, and so on:

-   [**InitHelperDll**](/windows/previous-versions/Netsh/nc-netsh-ns_dll_init_fn?branch=master)
-   [**FN\_HANDLE\_CMD**](/windows/previous-versions/Netsh/nc-netsh-fn_handle_cmd?branch=master) (Command function)
-   [**NS\_CONTEXT\_COMMIT\_FN**](/windows/previous-versions/Netsh/nc-netsh-ns_context_commit_fn?branch=master) (Commit function)
-   [**NS\_CONTEXT\_CONNECT\_FN**](/windows/previous-versions/Netsh/nc-netsh-ns_context_connect_fn?branch=master) (Connect function)
-   [**NS\_CONTEXT\_DUMP\_FN**](/windows/previous-versions/Netsh/nc-netsh-ns_context_dump_fn?branch=master) (Dump function)
-   [**NS\_DLL\_STOP\_FN**](/windows/previous-versions/Netsh/nc-netsh-ns_dll_stop_fn?branch=master) (DLL Stop function)
-   [**NS\_HELPER\_START\_FN**](/windows/previous-versions/Netsh/nc-netsh-ns_helper_start_fn?branch=master) (Start function)
-   [**NS\_HELPER\_STOP\_FN**](/windows/previous-versions/Netsh/nc-netsh-ns_helper_stop_fn?branch=master) (Stop function)
-   [**NS\_OSVERSIONCHECK**](/windows/previous-versions/Netsh/nc-netsh-ns_osversioncheck?branch=master) (Operating System Check function)
