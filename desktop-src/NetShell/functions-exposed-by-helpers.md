# Functions Exposed by Helpers

The following functions are implemented by Helpers, and used by NetShell for various tasks such as initializing helpers, registering helpers with Windows, and so on:

-   [**InitHelperDll**](/previous-versions/windows/desktop/api/Netsh/nc-netsh-ns_dll_init_fn)
-   [**FN\_HANDLE\_CMD**](/previous-versions/windows/desktop/api/Netsh/nc-netsh-fn_handle_cmd) (Command function)
-   [**NS\_CONTEXT\_COMMIT\_FN**](/previous-versions/windows/desktop/api/Netsh/nc-netsh-ns_context_commit_fn) (Commit function)
-   [**NS\_CONTEXT\_CONNECT\_FN**](/previous-versions/windows/desktop/api/Netsh/nc-netsh-ns_context_connect_fn) (Connect function)
-   [**NS\_CONTEXT\_DUMP\_FN**](/previous-versions/windows/desktop/api/Netsh/nc-netsh-ns_context_dump_fn) (Dump function)
-   [**NS\_DLL\_STOP\_FN**](/previous-versions/windows/desktop/api/Netsh/nc-netsh-ns_dll_stop_fn) (DLL Stop function)
-   [**NS\_HELPER\_START\_FN**](/previous-versions/windows/desktop/api/Netsh/nc-netsh-ns_helper_start_fn) (Start function)
-   [**NS\_HELPER\_STOP\_FN**](/previous-versions/windows/desktop/api/Netsh/nc-netsh-ns_helper_stop_fn) (Stop function)
-   [**NS\_OSVERSIONCHECK**](/previous-versions/windows/desktop/api/Netsh/nc-netsh-ns_osversioncheck) (Operating System Check function)
