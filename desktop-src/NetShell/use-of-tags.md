# Use of Tags

All command arguments should have optional tags. For example, "add name &lt;string&gt;" is incorrect. The correct form would be "add name \[name=\]&lt;string&gt;". All tags should be shown in lower case.

When a command is generated by a dump script, all of the optional tags for the arguments should be included.
