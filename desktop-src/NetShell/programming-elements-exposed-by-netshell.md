# Programming Elements Exposed by NetShell

The following functions are exported by NetShell, through the Netsh.exe and Netsh.lib files:

-   [**MatchEnumTag**](/previous-versions/windows/desktop/api/Netsh/nf-netsh-matchenumtag)
-   [**MatchToken**](/previous-versions/windows/desktop/api/Netsh/nf-netsh-matchtoken)
-   [**PreprocessCommand**](/previous-versions/windows/desktop/api/Netsh/nf-netsh-preprocesscommand)
-   [**PrintError**](/previous-versions/windows/desktop/api/Netsh/nf-netsh-printerror)
-   [**PrintMessage**](/previous-versions/windows/desktop/api/Netsh/nf-netsh-printmessage)
-   [**PrintMessageFromModule**](/previous-versions/windows/desktop/api/Netsh/nf-netsh-printmessagefrommodule)
-   [**RegisterContext**](/previous-versions/windows/desktop/api/Netsh/nf-netsh-registercontext)
-   [**RegisterHelper**](/previous-versions/windows/desktop/api/Netsh/nf-netsh-registerhelper)

The following structures are also available:

-   [**CMD\_ENTRY**](/previous-versions/windows/desktop/api/Netsh/ns-netsh-_cmd_entry)
-   [**CMD\_GROUP\_ENTRY**](/previous-versions/windows/desktop/api/Netsh/ns-netsh-_cmd_group_entry)
-   [**NS\_CONTEXT\_ATTRIBUTES**](/previous-versions/windows/desktop/api/Netsh/ns-netsh-_ns_context_attributes)
-   [**NS\_HELPER\_ATTRIBUTES**](/previous-versions/windows/desktop/api/Netsh/ns-netsh-_ns_helper_attributes)
-   [**TAG\_TYPE**](/previous-versions/windows/desktop/api/Netsh/ns-netsh-_tag_type)
