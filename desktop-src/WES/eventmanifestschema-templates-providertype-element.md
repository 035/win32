# templates (ProviderType) Element

Defines a list of templates that specify the data to include with the events.

``` syntax
<xs:element name="templates"
    type="TemplateListType"
 />
```

The **templates** element is defined by the [**ProviderType**](eventmanifestschema-providertype-complextype.md) complex type.
