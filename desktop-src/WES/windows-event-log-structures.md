# Windows Event Log Structures

Windows Event Log defines the following structures.



| Structure                                | Description                                                    |
|------------------------------------------|----------------------------------------------------------------|
| [**EVT\_RPC\_LOGIN**](/windows/win32/WinEvt/ns-winevt-_evt_rpc_login?branch=master) | Contains the information used to connect to a remote computer. |
| [**EVT\_VARIANT**](/windows/win32/WinEvt/ns-winevt-_evt_variant?branch=master)      | Contains event data or property values.                        |
