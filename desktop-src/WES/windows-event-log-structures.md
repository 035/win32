# Windows Event Log Structures

Windows Event Log defines the following structures.



| Structure                                | Description                                                    |
|------------------------------------------|----------------------------------------------------------------|
| [**EVT\_RPC\_LOGIN**](evt-rpc-login.md) | Contains the information used to connect to a remote computer. |
| [**EVT\_VARIANT**](evt-variant.md)      | Contains event data or property values.                        |
