# Windows Event Log Enumerations

Windows Event Log defines the following enumerations.



| Enumeration                                                                          | Description                                                                                                                        |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**EVT\_CHANNEL\_CLOCK\_TYPE**](/windows/win32/WinEvt/ne-winevt-_evt_channel_clock_type?branch=master)                          | Defines the values that specify the type of time stamp to use when logging events channel.                                         |
| [**EVT\_CHANNEL\_CONFIG\_PROPERTY\_ID**](/windows/win32/WinEvt/ne-winevt-_evt_channel_config_property_id?branch=master)         | Defines the identifiers that identify the configuration properties of a channel.                                                   |
| [**EVT\_CHANNEL\_ISOLATION\_TYPE**](/windows/win32/WinEvt/ne-winevt-_evt_channel_isolation_type?branch=master)                  | Defines the default access permissions to apply to the channel.                                                                    |
| [**EVT\_CHANNEL\_REFERENCE\_FLAGS**](/windows/win32/WinEvt/ne-winevt-_evt_channel_reference_flags?branch=master)                | Defines the values that specify how a channel is referenced.                                                                       |
| [**EVT\_CHANNEL\_SID\_TYPE**](/windows/win32/WinEvt/ne-winevt-_evt_channel_sid_type?branch=master)                              | Defines the values that determine whether the event includes the security identifier (SID) of the principal that logged the event. |
| [**EVT\_CHANNEL\_TYPE**](/windows/win32/WinEvt/ne-winevt-_evt_channel_type?branch=master)                                       | Defines the type of a channel.                                                                                                     |
| [**EVT\_EVENT\_METADATA\_PROPERTY\_ID**](/windows/win32/WinEvt/ne-winevt-_evt_event_metadata_property_id?branch=master)         | Defines the identifiers that identify the metadata properties of an event definition.                                              |
| [**EVT\_EVENT\_PROPERTY\_ID**](/windows/win32/WinEvt/ne-winevt-_evt_event_property_id?branch=master)               | Defines the values that determine the query information to retrieve.                                                               |
| [**EVT\_EXPORTLOG\_FLAGS**](/windows/win32/WinEvt/ne-winevt-_evt_exportlog_flags?branch=master)                                 | Defines values that indicate whether the events come from a channel or log file.                                                   |
| [**EVT\_FORMAT\_MESSAGE\_FLAGS**](/windows/win32/WinEvt/ne-winevt-_evt_format_message_flags?branch=master)                      | Defines the values that specify the message string from the event to format.                                                       |
| [**EVT\_LOG\_PROPERTY\_ID**](/windows/win32/WinEvt/ne-winevt-_evt_log_property_id?branch=master)                                | Defines the identifiers that identify the log file metadata properties of a channel or log file.                                   |
| [**EVT\_LOGIN\_CLASS**](/windows/win32/WinEvt/ne-winevt-_evt_login_class?branch=master)                                         | Defines the types of connection methods you can use to connect to the remote computer.                                             |
| [**EVT\_OPEN\_LOG\_FLAGS**](/windows/win32/WinEvt/ne-winevt-_evt_open_log_flags?branch=master)                                  | Defines the values that specify whether to open a channel or exported log file.                                                    |
| [**EVT\_PUBLISHER\_METADATA\_PROPERTY\_ID**](/windows/win32/WinEvt/ne-winevt-_evt_publisher_metadata_property_id?branch=master) | Defines the identifiers that identify the metadata properties of a provider.                                                       |
| [**EVT\_QUERY\_FLAGS**](/windows/win32/WinEvt/ne-winevt-_evt_query_flags?branch=master)                                         | Defines the values that specify how to return the query results and whether you are query against a channel or log file.           |
| [**EVT\_QUERY\_PROPERTY\_ID**](/windows/win32/WinEvt/ne-winevt-_evt_query_property_id?branch=master)                            | Defines the identifiers that identify the query information that you can retrieve.                                                 |
| [**EVT\_RENDER\_CONTEXT\_FLAGS**](/windows/win32/WinEvt/ne-winevt-_evt_render_context_flags?branch=master)                      | Defines the values that specify the type of information to access from the event.                                                  |
| [**EVT\_RENDER\_FLAGS**](/windows/win32/WinEvt/ne-winevt-_evt_render_flags?branch=master)                                       | Defines the values that specify what to render.                                                                                    |
| [**EVT\_RPC\_LOGIN\_FLAGS**](/windows/win32/WinEvt/ne-winevt-_evt_rpc_login_flags?branch=master)                                | Defines the types of authentication that you can use to authenticate the user when connecting to a remote computer.                |
| [**EVT\_SEEK\_FLAGS**](/windows/win32/WinEvt/ne-winevt-_evt_seek_flags?branch=master)                                           | Defines the relative position in the result set from which to seek.                                                                |
| [**EVT\_SUBSCRIBE\_FLAGS**](/windows/win32/WinEvt/ne-winevt-_evt_subscribe_flags?branch=master)                                 | Defines the possible values that specify when to start subscribing to events.                                                      |
| [**EVT\_SUBSCRIBE\_NOTIFY\_ACTION**](/windows/win32/WinEvt/ne-winevt-_evt_subscribe_notify_action?branch=master)                | Defines the possible types of data that the subscription service can deliver to your callback.                                     |
| [**EVT\_SYSTEM\_PROPERTY\_ID**](/windows/win32/WinEvt/ne-winevt-_evt_system_property_id?branch=master)                          | Defines the identifiers that identify the system-defined properties of an event.                                                   |
| [**EVT\_VARIANT\_TYPE**](/windows/win32/WinEvt/ne-winevt-_evt_variant_type?branch=master)                                       | Defines the possible data types of a variant data item.                                                                            |
