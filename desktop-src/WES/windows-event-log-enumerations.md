# Windows Event Log Enumerations

Windows Event Log defines the following enumerations.



| Enumeration                                                                          | Description                                                                                                                        |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**EVT\_CHANNEL\_CLOCK\_TYPE**](evt-channel-clock-type.md)                          | Defines the values that specify the type of time stamp to use when logging events channel.                                         |
| [**EVT\_CHANNEL\_CONFIG\_PROPERTY\_ID**](evt-channel-config-property-id.md)         | Defines the identifiers that identify the configuration properties of a channel.                                                   |
| [**EVT\_CHANNEL\_ISOLATION\_TYPE**](evt-channel-isolation-type.md)                  | Defines the default access permissions to apply to the channel.                                                                    |
| [**EVT\_CHANNEL\_REFERENCE\_FLAGS**](evt-channel-reference-flags.md)                | Defines the values that specify how a channel is referenced.                                                                       |
| [**EVT\_CHANNEL\_SID\_TYPE**](evt-channel-sid-type.md)                              | Defines the values that determine whether the event includes the security identifier (SID) of the principal that logged the event. |
| [**EVT\_CHANNEL\_TYPE**](evt-channel-type.md)                                       | Defines the type of a channel.                                                                                                     |
| [**EVT\_EVENT\_METADATA\_PROPERTY\_ID**](evt-event-metadata-property-id.md)         | Defines the identifiers that identify the metadata properties of an event definition.                                              |
| [**EVT\_EVENT\_PROPERTY\_ID**](evt-subscription-event-property-id.md)               | Defines the values that determine the query information to retrieve.                                                               |
| [**EVT\_EXPORTLOG\_FLAGS**](evt-exportlog-flags.md)                                 | Defines values that indicate whether the events come from a channel or log file.                                                   |
| [**EVT\_FORMAT\_MESSAGE\_FLAGS**](evt-format-message-flags.md)                      | Defines the values that specify the message string from the event to format.                                                       |
| [**EVT\_LOG\_PROPERTY\_ID**](evt-log-property-id.md)                                | Defines the identifiers that identify the log file metadata properties of a channel or log file.                                   |
| [**EVT\_LOGIN\_CLASS**](evt-login-class.md)                                         | Defines the types of connection methods you can use to connect to the remote computer.                                             |
| [**EVT\_OPEN\_LOG\_FLAGS**](evt-open-log-flags.md)                                  | Defines the values that specify whether to open a channel or exported log file.                                                    |
| [**EVT\_PUBLISHER\_METADATA\_PROPERTY\_ID**](evt-publisher-metadata-property-id.md) | Defines the identifiers that identify the metadata properties of a provider.                                                       |
| [**EVT\_QUERY\_FLAGS**](evt-query-flags.md)                                         | Defines the values that specify how to return the query results and whether you are query against a channel or log file.           |
| [**EVT\_QUERY\_PROPERTY\_ID**](evt-query-property-id.md)                            | Defines the identifiers that identify the query information that you can retrieve.                                                 |
| [**EVT\_RENDER\_CONTEXT\_FLAGS**](evt-render-context-flags.md)                      | Defines the values that specify the type of information to access from the event.                                                  |
| [**EVT\_RENDER\_FLAGS**](evt-render-flags.md)                                       | Defines the values that specify what to render.                                                                                    |
| [**EVT\_RPC\_LOGIN\_FLAGS**](evt-rpc-login-flags.md)                                | Defines the types of authentication that you can use to authenticate the user when connecting to a remote computer.                |
| [**EVT\_SEEK\_FLAGS**](evt-seek-flags.md)                                           | Defines the relative position in the result set from which to seek.                                                                |
| [**EVT\_SUBSCRIBE\_FLAGS**](evt-subscribe-flags.md)                                 | Defines the possible values that specify when to start subscribing to events.                                                      |
| [**EVT\_SUBSCRIBE\_NOTIFY\_ACTION**](evt-subscribe-notify-action.md)                | Defines the possible types of data that the subscription service can deliver to your callback.                                     |
| [**EVT\_SYSTEM\_PROPERTY\_ID**](evt-system-property-id.md)                          | Defines the identifiers that identify the system-defined properties of an event.                                                   |
| [**EVT\_VARIANT\_TYPE**](evt-variant-type.md)                                       | Defines the possible data types of a variant data item.                                                                            |
