# inType (InputTypeListType) Element

Defines an input data type.

``` syntax
<xs:element name="inType"
    type="InputType"
 />
```

The **inType** element is defined by the [**InputTypeListType**](eventmanifestschema-inputtypelisttype-complextype.md) complex type.
