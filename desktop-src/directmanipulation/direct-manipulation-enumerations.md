# Direct Manipulation Enumerations

The topics contained in this section provide the reference specifications for [Direct Manipulation](direct-manipulation-portal.md) enumerations.

## In this section



| Topic                                                                                                           | Description                                                                                                                                                                                                                                            |
|-----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DIRECTMANIPULATION\_AUTOSCROLL\_CONFIGURATION**](directmanipulation-autoscroll-configuration.md)<br/> | Determines the type and direction of automatic scrolling animation to apply. <br/>                                                                                                                                                               |
| [**DIRECTMANIPULATION\_CONFIGURATION**](directmanipulation-configuration.md)<br/>                        | Defines the interaction configuration states available in [Direct Manipulation](direct-manipulation-portal.md).<br/>                                                                                                                            |
| [**DIRECTMANIPULATION\_DRAG\_DROP\_CONFIGURATION**](directmanipulation-drag-drop-configuration.md)<br/>  | Defines behaviors for the drag-drop interaction.<br/>                                                                                                                                                                                            |
| [**DIRECTMANIPULATION\_DRAG\_DROP\_STATUS**](directmanipulation-drag-drop-status.md)<br/>                | Defines the drag-and-drop interaction states for the viewport.<br/>                                                                                                                                                                              |
| [**DIRECTMANIPULATION\_GESTURE\_CONFIGURATION**](directmanipulation-gesture-configuration.md)<br/>       | Defines the gestures that can be passed to [**SetManualGesture**](idirectmanipulationviewport-setmanualgesture.md).<br/>                                                                                                                        |
| [**DIRECTMANIPULATION\_HITTEST\_TYPE**](directmanipulation-hittest-type.md)<br/>                         | Defines how hit testing is handled by [Direct Manipulation](direct-manipulation-portal.md) when using a dedicated hit-test thread registered through [**RegisterHitTestTarget**](idirectmanipulationmanager-registerhittesttarget.md).<br/>    |
| [**DIRECTMANIPULATION\_HORIZONTALALIGNMENT**](directmanipulation-horizontalalignment.md)<br/>            | Defines the horizontal alignment options for content within a viewport.<br/>                                                                                                                                                                     |
| [**DIRECTMANIPULATION\_INPUT\_MODE**](directmanipulation-input-mode.md)<br/>                             | Defines the threading behavior for [**SetInputMode**](idirectmanipulationviewport-setinputmode.md) or [**SetUpdateMode**](idirectmanipulationviewport-setupdatemode.md). The exact meaning of each constant depends on the method called.<br/> |
| [**DIRECTMANIPULATION\_INTERACTION\_TYPE**](directmanipulation-interaction-type.md)<br/>                 | Defines gestures recognized by [Direct Manipulation](direct-manipulation-portal.md).<br/>                                                                                                                                                       |
| [**DIRECTMANIPULATION\_MOTION\_TYPES**](directmanipulation-motion-types.md)<br/>                         | Defines the [Direct Manipulation](direct-manipulation-portal.md) motion type.<br/>                                                                                                                                                              |
| [**DIRECTMANIPULATION\_SNAPPOINT\_COORDINATE**](directmanipulation-snappoint-coordinate.md)<br/>         | Defines the coordinate system for a collection of snap points.<br/>                                                                                                                                                                              |
| [**DIRECTMANIPULATION\_SNAPPOINT\_TYPE**](directmanipulation-snappoint-type.md)<br/>                     | Modifies how the final inertia end position is calculated.<br/>                                                                                                                                                                                  |
| [**DIRECTMANIPULATION\_STATUS**](directmanipulation-status.md)<br/>                                      | Defines the possible states of [Direct Manipulation](direct-manipulation-portal.md).<br/>                                                                                                                                                       |
| [**DIRECTMANIPULATION\_VERTICALALIGNMENT**](directmanipulation-verticalalignment.md)<br/>                | Defines the vertical alignment settings for content within the viewport.<br/>                                                                                                                                                                    |
| [**DIRECTMANIPULATION\_VIEWPORT\_OPTIONS**](directmanipulation-viewport-options.md)<br/>                 | Defines the input behavior options for the viewport.<br/>                                                                                                                                                                                        |
