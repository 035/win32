# Direct Manipulation Enumerations

The topics contained in this section provide the reference specifications for [Direct Manipulation](direct-manipulation-portal.md) enumerations.

## In this section



| Topic                                                                                                           | Description                                                                                                                                                                                                                                            |
|-----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DIRECTMANIPULATION\_AUTOSCROLL\_CONFIGURATION**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_autoscroll_configuration?branch=master)<br/> | Determines the type and direction of automatic scrolling animation to apply. <br/>                                                                                                                                                               |
| [**DIRECTMANIPULATION\_CONFIGURATION**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_configuration?branch=master)<br/>                        | Defines the interaction configuration states available in [Direct Manipulation](direct-manipulation-portal.md).<br/>                                                                                                                            |
| [**DIRECTMANIPULATION\_DRAG\_DROP\_CONFIGURATION**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_drag_drop_configuration?branch=master)<br/>  | Defines behaviors for the drag-drop interaction.<br/>                                                                                                                                                                                            |
| [**DIRECTMANIPULATION\_DRAG\_DROP\_STATUS**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_drag_drop_status?branch=master)<br/>                | Defines the drag-and-drop interaction states for the viewport.<br/>                                                                                                                                                                              |
| [**DIRECTMANIPULATION\_GESTURE\_CONFIGURATION**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_gesture_configuration?branch=master)<br/>       | Defines the gestures that can be passed to [**SetManualGesture**](/windows/previous-versions/DirectManipulation/nf-directmanipulation-idirectmanipulationviewport-setmanualgesture?branch=master).<br/>                                                                                                                        |
| [**DIRECTMANIPULATION\_HITTEST\_TYPE**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_hittest_type?branch=master)<br/>                         | Defines how hit testing is handled by [Direct Manipulation](direct-manipulation-portal.md) when using a dedicated hit-test thread registered through [**RegisterHitTestTarget**](/windows/previous-versions/DirectManipulation/nf-directmanipulation-idirectmanipulationmanager-registerhittesttarget?branch=master).<br/>    |
| [**DIRECTMANIPULATION\_HORIZONTALALIGNMENT**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_horizontalalignment?branch=master)<br/>            | Defines the horizontal alignment options for content within a viewport.<br/>                                                                                                                                                                     |
| [**DIRECTMANIPULATION\_INPUT\_MODE**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_input_mode?branch=master)<br/>                             | Defines the threading behavior for [**SetInputMode**](/windows/previous-versions/DirectManipulation/nf-directmanipulation-idirectmanipulationviewport-setinputmode?branch=master) or [**SetUpdateMode**](/windows/previous-versions/DirectManipulation/nf-directmanipulation-idirectmanipulationviewport-setupdatemode?branch=master). The exact meaning of each constant depends on the method called.<br/> |
| [**DIRECTMANIPULATION\_INTERACTION\_TYPE**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_interaction_type?branch=master)<br/>                 | Defines gestures recognized by [Direct Manipulation](direct-manipulation-portal.md).<br/>                                                                                                                                                       |
| [**DIRECTMANIPULATION\_MOTION\_TYPES**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_motion_types?branch=master)<br/>                         | Defines the [Direct Manipulation](direct-manipulation-portal.md) motion type.<br/>                                                                                                                                                              |
| [**DIRECTMANIPULATION\_SNAPPOINT\_COORDINATE**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_snappoint_coordinate?branch=master)<br/>         | Defines the coordinate system for a collection of snap points.<br/>                                                                                                                                                                              |
| [**DIRECTMANIPULATION\_SNAPPOINT\_TYPE**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_snappoint_type?branch=master)<br/>                     | Modifies how the final inertia end position is calculated.<br/>                                                                                                                                                                                  |
| [**DIRECTMANIPULATION\_STATUS**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_status?branch=master)<br/>                                      | Defines the possible states of [Direct Manipulation](direct-manipulation-portal.md).<br/>                                                                                                                                                       |
| [**DIRECTMANIPULATION\_VERTICALALIGNMENT**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_verticalalignment?branch=master)<br/>                | Defines the vertical alignment settings for content within the viewport.<br/>                                                                                                                                                                    |
| [**DIRECTMANIPULATION\_VIEWPORT\_OPTIONS**](/windows/previous-versions/directmanipulation/ne-directmanipulation-directmanipulation_viewport_options?branch=master)<br/>                 | Defines the input behavior options for the viewport.<br/>                                                                                                                                                                                        |
