# BasicDevice.Type property

Gets an enumeration value indicating the device type of the DLNA device.

This property is read-only.

## Syntax


```C++
HRESULT get_Type(
  [out] DeviceTypes *value
);
```



## Property value

A pointer to a [**DeviceTypes**](devicetypes.md) value.
