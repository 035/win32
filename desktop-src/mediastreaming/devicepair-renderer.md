# DevicePair::Renderer property

Gets the renderer for the active basic device pair.

This property is read-only.

## Syntax


```C++
HRESULT get_Renderer(
  [out] ActiveBasicDevice **value
);
```



## Property value

Receives a [**ActiveBasicDevice**](https://www.bing.com/search?q=**ActiveBasicDevice**) object that represents the renderer device.
