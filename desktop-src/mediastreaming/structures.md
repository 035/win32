# Structures

The [Media Streaming API](media-streaming-api-portal.md) provides the following structures.

## In this section



| Topic                                                           | Description                                                                                                                                        |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PlaySpeed**](playspeed.md)<br/>                       | Represents a playback speed as a rational number. <br/>                                                                                      |
| [**PositionInformation**](positioninformation.md)<br/>   | Contains the current values of media playback position information obtained from the DMR.<br/>                                               |
| [**RenderingParameters**](renderingparameters.md)<br/>   | Contains the current values of rendering parameters on the DMR.<br/>                                                                         |
| [**TrackInformation**](trackinformation.md)<br/>         | Contains the current track number and duration as part of the [**PositionInformation**](positioninformation.md) obtained from the DMR.<br/> |
| [**TransportInformation**](transportinformation.md)<br/> | Contains the current values of media playback transport-related information obtained from the DMR.<br/>                                      |
