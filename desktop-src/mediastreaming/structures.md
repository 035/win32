# Structures

The [Media Streaming API](media-streaming-api-portal.md) provides the following structures.

## In this section



| Topic                                                           | Description                                                                                                                                        |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PlaySpeed**](https://www.bing.com/search?q=**PlaySpeed**)<br/>                       | Represents a playback speed as a rational number. <br/>                                                                                      |
| [**PositionInformation**](https://www.bing.com/search?q=**PositionInformation**)<br/>   | Contains the current values of media playback position information obtained from the DMR.<br/>                                               |
| [**RenderingParameters**](https://www.bing.com/search?q=**RenderingParameters**)<br/>   | Contains the current values of rendering parameters on the DMR.<br/>                                                                         |
| [**TrackInformation**](https://www.bing.com/search?q=**TrackInformation**)<br/>         | Contains the current track number and duration as part of the [**PositionInformation**](https://www.bing.com/search?q=**PositionInformation**) obtained from the DMR.<br/> |
| [**TransportInformation**](https://www.bing.com/search?q=**TransportInformation**)<br/> | Contains the current values of media playback transport-related information obtained from the DMR.<br/>                                      |
