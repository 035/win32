# Reset method of the CIM\_LogicalDevice class

Resets the sensor.

This method is inherited from [**CIM\_LogicalDevice**](cim-logicaldevice.md).

## Syntax


```mof
uint32 Reset();
```



## Parameters

This method has no parameters.
