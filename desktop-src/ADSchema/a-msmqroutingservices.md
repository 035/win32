# MSMQ-Routing-Services attribute

Indicates whether the MSMQ installed on this computer provides MSMQ routing services.

|-------------------|--------------------------------------|
| CN                | MSMQ-Routing-Services                |
| Ldap-Display-Name | mSMQRoutingServices                  |
| Size              | \-                                   |
| Update Privilege  | \-                                   |
