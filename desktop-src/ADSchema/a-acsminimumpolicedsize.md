# ACS-Minimum-Policed-Size attribute

The **ACS-Minimum-Policed-Size** attribute is for internal use only. Based on RFC2210.



|                   |                                      |
|-------------------|--------------------------------------|
| CN                | ACS-Minimum-Policed-Size             |
| Ldap-Display-Name | aCSMinimumPolicedSize                |
| Size              | 8 bytes                              |
| Update Privilege  | \-                                   |
| Update Frequency  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1315              |
| System-Id-Guid    | 8d0e7195-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Interval**](s-interval.md)       |
