# Enhanced Storage Interfaces

The Enhanced Storage client programming interface defines and utilizes the following interfaces.



| Interface                                                        | Description                                                                                                   |
|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**IEnhancedStorageACT**](ienhancedstorageact.md)               | Used to obtain information and perform operations for an Enhanced Storage Addressable Command Target (ACT).   |
| [**IEnhancedStorageACT2**](ienhancedstorageact2.md)             | Used to obtain information for a 1667 Addressable Contact Target (ACT).                                       |
| [**IEnumEnhancedStorageACT**](ienumenhancedstorageact.md)       | Used to enumerate Enhanced Storage ACTs.                                                                      |
| [**IEnhancedStorageSilo**](ienhancedstoragesilo.md)             | Used to obtain information and perform operations for a Silo.                                                 |
| [**IEnhancedStorageSiloAction**](ienhancedstoragesiloaction.md) | Used to enumerate Silos and invoke actions specified by [**IEnhancedStorageSilo**](ienhancedstoragesilo.md). |
