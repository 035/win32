# Enhanced Storage Interfaces

The Enhanced Storage client programming interface defines and utilizes the following interfaces.



| Interface                                                        | Description                                                                                                   |
|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**IEnhancedStorageACT**](/previous-versions/windows/desktop/api/EhStorAPI/nn-ehstorapi-ienhancedstorageact)               | Used to obtain information and perform operations for an Enhanced Storage Addressable Command Target (ACT).   |
| [**IEnhancedStorageACT2**](/previous-versions/windows/desktop/api/EhStorAPI/nn-ehstorapi-ienhancedstorageact2)             | Used to obtain information for a 1667 Addressable Contact Target (ACT).                                       |
| [**IEnumEnhancedStorageACT**](/previous-versions/windows/desktop/api/EhStorAPI/nn-ehstorapi-ienumenhancedstorageact)       | Used to enumerate Enhanced Storage ACTs.                                                                      |
| [**IEnhancedStorageSilo**](/previous-versions/windows/desktop/api/EhStorAPI/nn-ehstorapi-ienhancedstoragesilo)             | Used to obtain information and perform operations for a Silo.                                                 |
| [**IEnhancedStorageSiloAction**](/previous-versions/windows/desktop/api/EhStorAPI/nn-ehstorapi-ienhancedstoragesiloaction) | Used to enumerate Silos and invoke actions specified by [**IEnhancedStorageSilo**](/previous-versions/windows/desktop/api/EhStorAPI/nn-ehstorapi-ienhancedstoragesilo). |
