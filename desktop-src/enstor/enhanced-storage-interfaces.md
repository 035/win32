# Enhanced Storage Interfaces

The Enhanced Storage client programming interface defines and utilizes the following interfaces.



| Interface                                                        | Description                                                                                                   |
|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**IEnhancedStorageACT**](/windows/previous-versions/EhStorAPI/nn-ehstorapi-ienhancedstorageact?branch=master)               | Used to obtain information and perform operations for an Enhanced Storage Addressable Command Target (ACT).   |
| [**IEnhancedStorageACT2**](/windows/previous-versions/EhStorAPI/nn-ehstorapi-ienhancedstorageact2?branch=master)             | Used to obtain information for a 1667 Addressable Contact Target (ACT).                                       |
| [**IEnumEnhancedStorageACT**](/windows/previous-versions/EhStorAPI/nn-ehstorapi-ienumenhancedstorageact?branch=master)       | Used to enumerate Enhanced Storage ACTs.                                                                      |
| [**IEnhancedStorageSilo**](/windows/previous-versions/EhStorAPI/nn-ehstorapi-ienhancedstoragesilo?branch=master)             | Used to obtain information and perform operations for a Silo.                                                 |
| [**IEnhancedStorageSiloAction**](/windows/previous-versions/EhStorAPI/nn-ehstorapi-ienhancedstoragesiloaction?branch=master) | Used to enumerate Silos and invoke actions specified by [**IEnhancedStorageSilo**](/windows/previous-versions/EhStorAPI/nn-ehstorapi-ienhancedstoragesilo?branch=master). |
