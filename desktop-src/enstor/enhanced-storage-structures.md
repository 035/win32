# Enhanced Storage Structures

The Enhanced Storage client programming interface defines and uses the following structures.



| Structure                                                                                            | Description                                                                                                    |
|------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**ACT\_AUTHORIZATION\_STATE**](act-authorization-state.md)                                         | Contains data that describes the current authorization state of a Addressable Command Target (ACT).<br/> |
| [**ENHANCED\_STORAGE\_PASSWORD\_SILO\_INFORMATION**](enhanced-storage-password-silo-information.md) | Contains data that defines the capabilities and requirements of a password silo. <br/>                   |
| [**SILO\_INFO**](silo-info.md)                                                                      | Contains information that identifies and describes the silo.<br/>                                        |
