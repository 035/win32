# Enhanced Storage Structures

The Enhanced Storage client programming interface defines and uses the following structures.



| Structure                                                                                            | Description                                                                                                    |
|------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**ACT\_AUTHORIZATION\_STATE**](/windows/previous-versions/EhStorAPI/ns-ehstorapi-_act_authorization_state?branch=master)                                         | Contains data that describes the current authorization state of a Addressable Command Target (ACT).<br/> |
| [**ENHANCED\_STORAGE\_PASSWORD\_SILO\_INFORMATION**](/windows/previous-versions/EhStorExtensions/ns-ehstorextensions-_enhanced_storage_password_silo_information?branch=master) | Contains data that defines the capabilities and requirements of a password silo. <br/>                   |
| [**SILO\_INFO**](/windows/previous-versions/EhStorAPI/ns-ehstorapi-_silo_info?branch=master)                                                                      | Contains information that identifies and describes the silo.<br/>                                        |
