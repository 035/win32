# Enhanced Storage Constants

This section provides details about the Enhanced Storage API constants.



| Constant Type                                              | Description                                                                     |
|------------------------------------------------------------|---------------------------------------------------------------------------------|
| [**CERT\_CAPABILITY**](cert-capability.md)                | Indicate the capabilities supported by the certificate silo.                    |
| [**CERT\_TYPE**](cert-type.md)                            | Indicates the type of certificate being used for validation.                    |
| [**CERT\_VALIDATION\_POLICY**](cert-validation-policy.md) | Indicates the certificate validation policy used during certificate validation. |
