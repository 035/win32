# Enhanced Storage

## Purpose

Enhanced Storage provides an extensible platform for accessing additional functions made available by enhanced storage devices; above and beyond the primary function of data storage.

## Developer audience

The Enhanced Storage API is designed for use with C/C++.

## Run-time requirements

Enhanced Storage is supported natively in Windows Vista with Service Pack 2 (SP2) and Windows 7.
