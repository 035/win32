# Constants

This section provides the reference specifications for [Touch Injection](touch-injection-portal.md) constants.



| Constant/value                                                                                                                                                                                                                                           | Description                                                       |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------|
| <span id="MAX_TOUCH_COUNT"></span><span id="max_touch_count"></span><dl> <dt>**MAX\_TOUCH\_COUNT**</dt> <dt>256</dt> </dl>                            | Specifies the maximum number of simultaneous contacts.<br/> |
| <span id="TOUCH_FEEDBACK_DEFAULT_"></span><span id="touch_feedback_default_"></span><dl> <dt>**TOUCH\_FEEDBACK\_DEFAULT** </dt> <dt>0x1</dt> </dl>    | Specifies default touch visualizations.<br/>                |
| <span id="TOUCH_FEEDBACK_INDIRECT_"></span><span id="touch_feedback_indirect_"></span><dl> <dt>**TOUCH\_FEEDBACK\_INDIRECT** </dt> <dt>0x2</dt> </dl> | Specifies indirect touch visualizations.<br/>               |
| <span id="TOUCH_FEEDBACK_NONE_"></span><span id="touch_feedback_none_"></span><dl> <dt>**TOUCH\_FEEDBACK\_NONE** </dt> <dt>0x3</dt> </dl>             | Specifies no touch visualizations.<br/>                     |
