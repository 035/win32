# Functions

The topics in this section provide the reference specifications for [Touch Injection](touch-injection-portal.md) functions.

## In this section



| Topic                                                                   | Description                                                                                                                                                        |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InitializeTouchInjection**](/windows/desktop/api/Winuser/nf-winuser-initializetouchinjection)<br/> | Configures the touch injection context for the calling application and initializes the maximum number of simultaneous contacts that the app can inject.<br/> |
| [**InjectTouchInput**](/windows/desktop/api/Winuser/nf-winuser-injecttouchinput)<br/>                 | Simulates touch input.<br/>                                                                                                                                  |
