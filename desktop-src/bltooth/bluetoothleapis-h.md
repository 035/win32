# Bluetoothleapis.h

This section contains reference topics for the Bluetoothleapis.h header.

## In this section



| Topic                                                                                   | Description                                                                                                                                                      |
|-----------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BluetoothGATTGetCharacteristics**](/windows/desktop/api/bluetoothleapis/nf-bluetoothleapis-bluetoothgattgetcharacteristics)<br/>   | Gets all the characteristics available for the specified service.<br/>                                                                                     |
| [**BluetoothGATTGetDescriptors**](/windows/desktop/api/bluetoothleapis/nf-bluetoothleapis-bluetoothgattgetdescriptors)<br/>           | Gets all the descriptors available for the specified characteristic.<br/>                                                                                  |
| [**BluetoothGATTGetIncludedServices**](/windows/desktop/api/bluetoothleapis/nf-bluetoothleapis-bluetoothgattgetincludedservices)<br/> | Gets all the included services available for a given service.<br/>                                                                                         |
| [**BluetoothGATTRegisterEvent**](/windows/desktop/api/bluetoothleapis/nf-bluetoothleapis-bluetoothgattregisterevent)<br/>             | Registers a routine to be called back during a characteristic value change event on the given characteristic identified by its characteristic handle.<br/> |
| [**BluetoothGATTSetDescriptorValue**](/windows/desktop/api/BluetoothLEApis/nf-bluetoothleapis-bluetoothgattsetdescriptorvalue)<br/>   | Writes the specified descriptor value to the Bluetooth device.<br/>                                                                                        |
