# Bluetoothapis.h

This section contains reference topics for the Bluetoothapis.h header.

## In this section



| Topic                                                                                         | Description                                                                                  |
|-----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**BluetoothGATTAbortReliableWrite**](/windows/win32/BluetoothLEApis/nf-bluetoothleapis-bluetoothgattabortreliablewrite?branch=master)<br/>         | Specifies the end of reliable write procedures, and the writes should be aborted.<br/> |
| [**BluetoothGATTBeginReliableWrite**](/windows/win32/BluetoothLEApis/nf-bluetoothleapis-bluetoothgattbeginreliablewrite?branch=master)<br/>         | Specifies that reliable writes are about to begin.<br/>                                |
| [**BluetoothGATTEndReliableWrite**](/windows/win32/BluetoothLEApis/nf-bluetoothleapis-bluetoothgattendreliablewrite?branch=master)<br/>             | Specifies the end of reliable writes, and the writes should be committed.<br/>         |
| [**BluetoothGATTGetCharacteristicValue**](/windows/win32/BluetoothLEApis/nf-bluetoothleapis-bluetoothgattgetcharacteristicvalue?branch=master)<br/> | Gets the value of the specified characteristic<br/>                                    |
| [**BluetoothGATTGetDescriptorValue**](/windows/win32/BluetoothLEApis/nf-bluetoothleapis-bluetoothgattgetdescriptorvalue?branch=master)<br/>         | Gets the value of the specified descriptor.<br/>                                       |
| [**BluetoothGATTGetServices**](/windows/win32/BluetoothLEApis/nf-bluetoothleapis-bluetoothgattgetservices?branch=master)<br/>                       | Gets all the primary services available for a server.<br/>                             |
| [**BluetoothGATTSetCharacteristicValue**](/windows/win32/BluetoothLEApis/nf-bluetoothleapis-bluetoothgattsetcharacteristicvalue?branch=master)<br/> | Writes the specified characteristic value to the Bluetooth device.<br/>                |
| [**BluetoothGATTUnregisterEvent**](/windows/win32/BluetoothLEApis/nf-bluetoothleapis-bluetoothgattunregisterevent?branch=master)<br/>               | Unregisters the given characteristic value change event.<br/>                          |
| [**BluetoothSetLocalServiceInfo**](/windows/win32/bluetoothapis/nf-bluetoothapis-bluetoothsetlocalserviceinfo?branch=master)<br/>               | Sets local service information for a specific Bluetooth radio.<br/>                    |
