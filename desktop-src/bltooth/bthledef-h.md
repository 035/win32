# Bthledef.h

This section contains reference topics for the Bthledef.h header.

## In this section



| Topic                                                                                                                      | Description                                                                                                                                                |
|----------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [*PFNBLUETOOTH\_GATT\_EVENT\_CALLBACK*](bluetooth-gatt-notification-callback-function.md)<br/>                      | Profile drivers implement a Bluetooth GATT event callback to be called whenever the value of a specific characteristic changes.<br/>                 |
| [**BLUETOOTH\_GATT\_VALUE\_CHANGED\_EVENT**](bluetooth-gatt-value-changed-event.md)<br/>                            | The BLUETOOTH\_GATT\_VALUE\_CHANGED\_EVENT structure describes a changed attribute value.<br/>                                                       |
| [**BLUETOOTH\_GATT\_VALUE\_CHANGED\_EVENT\_REGISTRATION**](bluetooth-gatt-value-changed-event-registration.md)<br/> | The BLUETOOTH\_GATT\_VALUE\_CHANGED\_EVENT\_REGISTRATION structure describes one or more characteristics that have changed.<br/>                     |
| [**BTH\_LE\_GATT\_CHARACTERISTIC**](bth-le-gatt-characteristic.md)<br/>                                             | The BTH\_LE\_GATT\_CHARACTERISTIC structure describes a Bluetooth Low Energy (LE) generic attribute (GATT) profile characteristic.<br/>              |
| [**BTH\_LE\_GATT\_CHARACTERISTIC\_VALUE**](bth-le-gatt-characteristic-value.md)<br/>                                | The BTH\_LE\_GATT\_CHARACTERISTIC\_VALUE structure describes a Bluetooth Low Energy (LE) generic attribute (GATT) profile characteristic value.<br/> |
| [**BTH\_LE\_GATT\_DESCRIPTOR**](bth-le-gatt-descriptor.md)<br/>                                                     | The BTH\_LE\_GATT\_DESCRIPTOR structure describes a Bluetooth Low Energy (LE) generic attribute (GATT) profile descriptor.<br/>                      |
| [**BTH\_LE\_GATT\_DESCRIPTOR\_TYPE**](bth-le-gatt-descriptor-type.md)<br/>                                          | The BTH\_LE\_GATT\_DESCRIPTOR\_TYPE enumeration describes the different types of Bluetooth LE generic attributes (GATT).<br/>                        |
| [**BTH\_LE\_GATT\_DESCRIPTOR\_VALUE**](bth-le-gatt-descriptor-value.md)<br/>                                        | The BTH\_LE\_GATT\_DESCRIPTOR\_VALUE structure describes a parent characteristic.<br/>                                                               |
| [**BTH\_LE\_GATT\_EVENT\_TYPE**](bth-le-gatt-event-type.md)<br/>                                                    | The BTH\_LE\_GATT\_EVENT\_TYPE enumeration describes the different types of Bluetooth Low Energy (LE) generic attribute (GATT) profile events.<br/>  |
| [**BTH\_LE\_GATT\_SERVICE**](bth-le-gatt-service.md)<br/>                                                           | The BTH\_LE\_GATT\_SERVICE structure describes a Bluetooth Low Energy (LE) generic attribute (GATT) profile service.<br/>                            |
| [**BTH\_LE\_UUID**](bth-le-uuid.md)<br/>                                                                            | The BTH\_LE\_UUID structure contains information about a Bluetooth Low Energy (LE) Universally Unique Identifier (UUID).<br/>                        |
