# DWM Structures

This section contains information about the Desktop Window Manager (DWM) structures.

## In this section



| Topic                                                                     | Description                                                                                                                                               |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DWM\_BLURBEHIND**](/windows/win32/Dwmapi/ns-dwmapi-_dwm_blurbehind?branch=master)<br/>                      | Specifies DWM blur-behind properties. Used by the [**DwmEnableBlurBehindWindow**](/windows/win32/Dwmapi/nf-dwmapi-dwmenableblurbehindwindow?branch=master) function.<br/>                     |
| [**DWM\_PRESENT\_PARAMETERS**](/windows/win32/Dwmapi/ns-dwmapi-_dwm_present_parameters?branch=master)<br/>     | Specifies DWM video frame parameters for frame composition. Used by the [**DwmSetPresentParameters**](/windows/win32/Dwmapi/nf-dwmapi-dwmsetpresentparameters?branch=master) function.<br/>   |
| [**DWM\_THUMBNAIL\_PROPERTIES**](/windows/win32/Dwmapi/ns-dwmapi-_dwm_thumbnail_properties?branch=master)<br/> | Specifies DWM thumbnail properties. Used by the [**DwmUpdateThumbnailProperties**](/windows/win32/Dwmapi/nf-dwmapi-dwmupdatethumbnailproperties?branch=master) function.<br/>                 |
| [**DWM\_TIMING\_INFO**](/windows/win32/Dwmapi/ns-dwmapi-_dwm_timing_info?branch=master)<br/>                   | Specifies DWM composition timing information. Used by the [**DwmGetCompositionTimingInfo**](/windows/win32/Dwmapi/nf-dwmapi-dwmgetcompositiontiminginfo?branch=master) function.<br/>         |
| [**MilMatrix3x2D**](/windows/win32/Dwmapi/ns-dwmapi-_milmatrix3x2d?branch=master)<br/>                         | Specifies a 3x2 matrix that describes a transform. <br/>                                                                                            |
| [**UNSIGNED\_RATIO**](/windows/win32/Dwmapi/ns-dwmapi-_unsigned_ratio?branch=master)<br/>                      | Defines a data type used by the DWM APIs. It represents a generic ratio and is used for different purposes and units even within a single API.<br/> |
