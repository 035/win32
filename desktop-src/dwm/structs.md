# DWM Structures

This section contains information about the Desktop Window Manager (DWM) structures.

## In this section



| Topic                                                                     | Description                                                                                                                                               |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DWM\_BLURBEHIND**](dwm-blurbehind.md)<br/>                      | Specifies DWM blur-behind properties. Used by the [**DwmEnableBlurBehindWindow**](dwmenableblurbehindwindow.md) function.<br/>                     |
| [**DWM\_PRESENT\_PARAMETERS**](dwm-present-parameters.md)<br/>     | Specifies DWM video frame parameters for frame composition. Used by the [**DwmSetPresentParameters**](dwmsetpresentparameters.md) function.<br/>   |
| [**DWM\_THUMBNAIL\_PROPERTIES**](dwm-thumbnail-properties.md)<br/> | Specifies DWM thumbnail properties. Used by the [**DwmUpdateThumbnailProperties**](dwmupdatethumbnailproperties.md) function.<br/>                 |
| [**DWM\_TIMING\_INFO**](dwm-timing-info.md)<br/>                   | Specifies DWM composition timing information. Used by the [**DwmGetCompositionTimingInfo**](dwmgetcompositiontiminginfo.md) function.<br/>         |
| [**MilMatrix3x2D**](milmatrix3x2d.md)<br/>                         | Specifies a 3x2 matrix that describes a transform. <br/>                                                                                            |
| [**UNSIGNED\_RATIO**](unsigned-ratio.md)<br/>                      | Defines a data type used by the DWM APIs. It represents a generic ratio and is used for different purposes and units even within a single API.<br/> |
