# DWM Constants and Enumerations

This section contains information about the constants and enumerations used in the Desktop Window Manager (DWM) APIs.

## In this section



| Topic                                                                                     | Description                                                                                                                                                                                                     |
|-------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DWM Blur Behind Constants**](dwm-bb-constants.md)<br/>                          | Flags used by the [**DWM\_BLURBEHIND**](dwm-blurbehind.md) structure to indicate which of its members contain valid information.<br/>                                                                    |
| [**DWM Enable Composition Constants**](dwm-ec-constants.md)<br/>                   | Flags used by the [**DwmEnableComposition**](dwmenablecomposition.md)  function to change the state of DWM composition.<br/>                                                                             |
| [**DWM\_SOURCE\_FRAME\_SAMPLING**](dwm-source-frame-sampling.md)<br/>              | Flags used by the [**DwmSetPresentParameters**](dwmsetpresentparameters.md) function to specify the frame sampling type.<br/>                                                                            |
| [**DWM\_TAB\_WINDOW\_REQUIREMENTS**](dwm-tab-window-requirements.md)<br/>          | Returned by DwmGetUnmetTabRequirements to indicate the requirements needed for a window to put tabs in the application title bar.<br/>                                                                    |
| [**DWM\_TNP Constants**](dwm-tnp-constants.md)<br/>                                | Flags used by the [**DWM\_THUMBNAIL\_PROPERTIES**](dwm-thumbnail-properties.md) structure to indicate which of its members contain valid information.<br/>                                               |
| [**DWMFLIP3DWINDOWPOLICY**](dwmflip3dwindowpolicy.md)<br/>                         | Flags used by the [**DwmSetWindowAttribute**](dwmsetwindowattribute.md) function to specify the Flip3D window policy.<br/>                                                                               |
| [**DWMNCRENDERINGPOLICY**](dwmncrenderingpolicy.md)<br/>                           | Flags used by the [**DwmSetWindowAttribute**](dwmsetwindowattribute.md) function to specify the non-client area rendering policy.<br/>                                                                   |
| [**DWMTRANSITION\_OWNEDWINDOW\_TARGET**](dwmtransition-ownedwindow-target.md)<br/> | Identifies the target.<br/>                                                                                                                                                                               |
| [**DWMWINDOWATTRIBUTE**](dwmwindowattribute.md)<br/>                               | Flags used by the [**DwmGetWindowAttribute**](dwmgetwindowattribute.md) and [**DwmSetWindowAttribute**](dwmsetwindowattribute.md) functions to specify window attributes for non-client rendering.<br/> |
| [**GESTURE\_TYPE**](gesture-type.md)<br/>                                          | Identifies the gesture type specified in [**DwmRenderGesture**](dwmrendergesture.md).<br/>                                                                                                               |
