# DWM Constants and Enumerations

This section contains information about the constants and enumerations used in the Desktop Window Manager (DWM) APIs.

## In this section



| Topic                                                                                     | Description                                                                                                                                                                                                     |
|-------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DWM Blur Behind Constants**](dwm-bb-constants.md)<br/>                          | Flags used by the [**DWM\_BLURBEHIND**](/windows/win32/Dwmapi/ns-dwmapi-_dwm_blurbehind?branch=master) structure to indicate which of its members contain valid information.<br/>                                                                    |
| [**DWM Enable Composition Constants**](dwm-ec-constants.md)<br/>                   | Flags used by the [**DwmEnableComposition**](/windows/win32/Dwmapi/nf-dwmapi-dwmenablecomposition?branch=master)  function to change the state of DWM composition.<br/>                                                                             |
| [**DWM\_SOURCE\_FRAME\_SAMPLING**](/windows/win32/Dwmapi/ne-dwmapi-dwm_source_frame_sampling?branch=master)<br/>              | Flags used by the [**DwmSetPresentParameters**](/windows/win32/Dwmapi/nf-dwmapi-dwmsetpresentparameters?branch=master) function to specify the frame sampling type.<br/>                                                                            |
| [**DWM\_TAB\_WINDOW\_REQUIREMENTS**](/windows/win32/dwmapi/ne-dwmapi-dwm_tab_window_requirements?branch=master)<br/>          | Returned by DwmGetUnmetTabRequirements to indicate the requirements needed for a window to put tabs in the application title bar.<br/>                                                                    |
| [**DWM\_TNP Constants**](dwm-tnp-constants.md)<br/>                                | Flags used by the [**DWM\_THUMBNAIL\_PROPERTIES**](/windows/win32/Dwmapi/ns-dwmapi-_dwm_thumbnail_properties?branch=master) structure to indicate which of its members contain valid information.<br/>                                               |
| [**DWMFLIP3DWINDOWPOLICY**](/windows/win32/Dwmapi/ne-dwmapi-dwmflip3dwindowpolicy?branch=master)<br/>                         | Flags used by the [**DwmSetWindowAttribute**](/windows/win32/Dwmapi/nf-dwmapi-dwmsetwindowattribute?branch=master) function to specify the Flip3D window policy.<br/>                                                                               |
| [**DWMNCRENDERINGPOLICY**](/windows/win32/Dwmapi/ne-dwmapi-dwmncrenderingpolicy?branch=master)<br/>                           | Flags used by the [**DwmSetWindowAttribute**](/windows/win32/Dwmapi/nf-dwmapi-dwmsetwindowattribute?branch=master) function to specify the non-client area rendering policy.<br/>                                                                   |
| [**DWMTRANSITION\_OWNEDWINDOW\_TARGET**](/windows/win32/Dwmapi/ne-dwmapi-dwmtransition_ownedwindow_target?branch=master)<br/> | Identifies the target.<br/>                                                                                                                                                                               |
| [**DWMWINDOWATTRIBUTE**](/windows/win32/Dwmapi/ne-dwmapi-dwmwindowattribute?branch=master)<br/>                               | Flags used by the [**DwmGetWindowAttribute**](/windows/win32/Dwmapi/nf-dwmapi-dwmgetwindowattribute?branch=master) and [**DwmSetWindowAttribute**](/windows/win32/Dwmapi/nf-dwmapi-dwmsetwindowattribute?branch=master) functions to specify window attributes for non-client rendering.<br/> |
| [**GESTURE\_TYPE**](/windows/win32/Dwmapi/ne-dwmapi-gesture_type?branch=master)<br/>                                          | Identifies the gesture type specified in [**DwmRenderGesture**](/windows/win32/Dwmapi/nf-dwmapi-dwmrendergesture?branch=master).<br/>                                                                                                               |
