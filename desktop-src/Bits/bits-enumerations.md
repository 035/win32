# BITS Enumerations

The [Background Intelligent Transfer Service (BITS) interfaces](bits-interfaces.md) use the following enumerations.

## In this section



| Topic                                                                      | Description                                                                                                                                                                                                      |
|----------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BG\_AUTH\_SCHEME**](bg-auth-scheme.md)<br/>                      | The [**BG\_AUTH\_SCHEME**](bg-auth-scheme.md) enumeration defines the constant values that specify the authentication scheme to use when a proxy or server requests user authentication.<br/>             |
| [**BG\_AUTH\_TARGET**](bg-auth-target.md)<br/>                      | The [**BG\_AUTH\_TARGET**](bg-auth-target.md) enumeration defines the constant values that specify whether the credentials are used for proxy or server user authentication requests.<br/>                |
| [**BG\_CERT\_STORE\_LOCATION**](bg-cert-store-location.md)<br/>     | The [**BG\_CERT\_STORE\_LOCATION**](bg-cert-store-location.md) enumeration defines the location of the certificate store.<br/>                                                                            |
| [**BG\_ERROR\_CONTEXT**](bg-error-context.md)<br/>                  | The [**BG\_ERROR\_CONTEXT**](bg-error-context.md) enumeration defines the constant values that specify the context in which the error occurred.<br/>                                                      |
| [**BG\_JOB\_PRIORITY**](bg-job-priority.md)<br/>                    | The [**BG\_JOB\_PRIORITY**](bg-job-priority.md) enumeration defines the constant values that specify the priority level of a job. <br/>                                                                   |
| [**BG\_JOB\_PROXY\_USAGE**](bg-job-proxy-usage.md)<br/>             | The [**BG\_JOB\_PROXY\_USAGE**](bg-job-proxy-usage.md) enumeration defines constant values that specify which proxy to use for file transfers. You can define different proxy settings for each job.<br/> |
| [**BG\_JOB\_STATE**](bg-job-state.md)<br/>                          | The [**BG\_JOB\_STATE**](bg-job-state.md) enumeration defines constant values for the different states of a job.<br/>                                                                                     |
| [**BG\_JOB\_TYPE**](bg-job-type.md)<br/>                            | The [**BG\_JOB\_TYPE**](bg-job-type.md) enumeration defines constant values that specify the type of transfer job, such as download.<br/>                                                                 |
| [**BITS\_COST\_STATE**](bits-cost-state.md)<br/>                    | The BITS\_COST\_STATE enumeration defines the constant values that specify the BITS cost state.<br/>                                                                                                       |
| [**BITS\_FILE\_PROPERTY\_ID**](bits-file-property-id.md)<br/>       | Enumerates values that define ID values corresponding to BackgroundCopyFile properties.<br/>                                                                                                               |
| [**BITS\_JOB\_PROPERTY\_ID**](bits-job-property-id.md)<br/>         | Specifies the ID of the property for the BITS job.<br/>                                                                                                                                                    |
| [**BITS\_JOB\_TRANSFER\_POLICY**](bits-job-transfer-policy.md)<br/> | Enumeration that defines ID values corresponding to BITS properties.<br/>                                                                                                                                  |
