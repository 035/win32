# IPrintOemUni2 interface

This section describes the methods defined for the IPrintOemUni2 COM interface.

In addition to the methods that belong to the IPrintOemUni2 COM interface, this interface includes all of the methods that belong to the [**IPrintOemUni**](iprintoemuni-interface.md) COM interface.

## Members

The **IPrintOemUni2** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/windows/desktop/33f1d79a-33fc-4ce5-a372-e08bda378332) interface. **IPrintOemUni2** also has these types of members:

-   [Methods](#methods)

### Methods

The **IPrintOemUni2** interface has these methods.



| Method                                                             | Description                                                                                                                                                                                                                                                                                                                                         |
|:-------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetImplementedMethod**](iprintoemuni2-getimplementedmethod.md) | The `IPrintOemUni2::GetImplementedMethod` method is used by Unidrv to determine which **IPrintOemUni2** interface methods a rendering plug-in has implemented.<br/>                                                                                                                                                                           |
| [**WritePrinter**](iprintoemuni2-writeprinter.md)                 | The `IPrintOemUni2::WritePrinter` method, if supported, enables a rendering plug-in to capture all output data generated by a Unidrv driver. If this method is not supported, the output data would otherwise be sent to the spooler in a call to the spooler's WritePrinter API (described in the Microsoft Windows SDK documentation).<br/> |
