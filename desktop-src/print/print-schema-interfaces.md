# Print Schema Interfaces

This section provides information about the interfaces that enable the new print driver in Windows 8 (the V4 print driver) to query and manipulate print schema elements.

Here is a list of the print schema interfaces:

-   [**IPrintSchemaAsyncOperation**](iprintschemaasyncoperation-interface.md)
-   [**IPrintSchemaAsyncOperationEvent**](iprintschemaasyncoperationevent-interface.md)
-   [**IPrintSchemaCapabilities**](iprintschemacapabilities-interface.md)
-   [**IPrintSchemaCapabilities2**](iprintschemacapabilities2.md)
-   [**IPrintSchemaDisplayableElement**](iprintschemadisplayableelement-interface.md)
-   [**IPrintSchemaElement**](iprintschemaelement-interface.md)
-   [**IPrintSchemaFeature**](iprintschemafeature-interface.md)
-   [**IPrintSchemaNUpOption**](iprintschemanupoption-interface.md)
-   [**IPrintSchemaOption**](iprintschemaoption-interface.md)
-   [**IPrintSchemaOptionCollection**](iprintschemaoptioncollection.md)
-   [**IPrintSchemaPageImageableSize**](iprintschemapageimageablesize-interface.md)
-   [**IPrintSchemaPageMediaSizeOption**](iprintschemapagemediasizeoption-interface.md)
-   [**IPrintSchemaParameterDefinition**](iprintschemaparameterdefinition.md)
-   [**IPrintSchemaParameterInitializer**](iprintschemaparameterinitializer.md)
-   [**IPrintSchemaTicket**](iprintschematicket-interface.md)
-   [**IPrintSchemaTicket2**](iprintschematicket2.md)
