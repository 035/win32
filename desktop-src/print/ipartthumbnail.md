# IPartThumbnail interface

The **IPartThumbnail** interface is an abstraction for thumbnails in an XPS document.

## Members

The **IPartThumbnail** interface inherits from [IPartBase](ipartbase.md). **IPartThumbnail** also has these types of members:

-   [Methods](#methods)

### Methods

The **IPartThumbnail** interface has these methods.



| Method                                                                                  | Description                                                                            |
|:----------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------|
| [**IPartThumbnail::GetThumbnailProperties**](ipartthumbnail-getthumbnailproperties.md) | The **GetThumbnailProperties** method gets the thumbnail properties.<br/>        |
| [**IPartThumbnail::SetThumbnailContent**](ipartthumbnail-setthumbnailcontent.md)       | The **SetThumbnailContent** method sets the thumbnail content for the part.<br/> |
