# Methods for Rendering Plug-Ins

## 

This section describes the COM interfaces used for communication between Microsoft's printer drivers ([*Unidrv*](wdkgloss.u#wdkgloss-unidrv) and [*Pscript*](wdkgloss.p#wdkgloss-pscript)) and vendor-supplied [rendering plug-ins](print.rendering_plug_ins). The following topics are provided:

<dl>

[IPrintOemPS Interface](iprintoemps-interface.md)  
[IPrintOemPS2 Interface](iprintoemps2-interface.md)  
[IPrintOemDriverPS Interface](iprintoemdriverps-interface.md)  
[IPrintOemUni Interface](iprintoemuni-interface.md)  
[IPrintOemUni2 Interface](iprintoemuni2-interface.md)  
[IPrintOemUni3 Interface](iprintoemuni3-interface.md)  
[IPrintOemDriverUni Interface](iprintoemdriveruni-interface.md)  
[IPrintCorePS2 Interface](iprintcoreps2-interface.md)
