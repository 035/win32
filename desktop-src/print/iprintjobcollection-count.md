# IPrintJobCollection::Count property

Gets the number of jobs in the print queue.

This property is read-only.

## Syntax


```C++
HRESULT get_Count(
  [out, retval] ULONG *pulCount
);
```



## Property value

The number of jobs in the print queue.

## Error codes

Returns an **HRESULT** value, if the property call was successful. Otherwise it returns the appropriate error code.
