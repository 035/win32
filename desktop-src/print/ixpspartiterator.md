# IXpsPartIterator interface

The `IXpsPartIterator` interface is an iterator for XPS parts.

## Members

The **IXpsPartIterator** interface inherits from the [**IUnknown**](com.iunknown) interface. **IXpsPartIterator** also has these types of members:

-   [Methods](#methods)

### Methods

The **IXpsPartIterator** interface has these methods.



| Method                                                        | Description                                                                                |
|:--------------------------------------------------------------|:-------------------------------------------------------------------------------------------|
| [**IXpsPartIterator::Current**](ixpspartiterator-current.md) | The `Current` method provides the current URI and part.<br/>                         |
| [**IXpsPartIterator::IsDone**](ixpspartiterator-isdone.md)   | The `IsDone` method determines whether the iterator has finished the iteration.<br/> |
| [**IXpsPartIterator::Next**](ixpspartiterator-next.md)       | The `Next` method advances the iterator to the next part.<br/>                       |
| [**IXpsPartIterator::Reset**](ixpspartiterator-reset.md)     | The `Reset` method sets the iterator to the first element.<br/>                      |
