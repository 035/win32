# IFixedPage::SetPagePart method

The **SetPagePart** method associates a new part with the page.

## Syntax


```C++
HRESULT SetPagePart(
  [in] IUnknown *pUnk
);
```



## Parameters

<dl> <dt>

*pUnk* \[in\]
</dt> <dd>

A pointer to the new part.

</dd> </dl>

## Return value

**SetPagePart** returns an **HRESULT** value.
