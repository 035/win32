# IPrintCoreHelperUni2 interface

The IPrintCoreHelperUni2 interface derives from the [IPrintCoreHelperUni](iprintcorehelperuni-interface.md) interface and adds a method.

## Members

The **IPrintCoreHelperUni2** interface inherits from the [**IUnknown**](com.iunknown) interface. **IPrintCoreHelperUni2** also has these types of members:

-   [Methods](#methods)

### Methods

The **IPrintCoreHelperUni2** interface has these methods.



| Method                                                          | Description                                                                                                          |
|:----------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------|
| [**GetNamedCommand**](iprintcorehelperuni2-getnamedcommand.md) | The [**GetNamedCommand**](iprintcorehelperuni2-getnamedcommand.md) method returns the specified command.<br/> |
