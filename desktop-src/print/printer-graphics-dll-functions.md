# Printer Graphics DLL Functions

## 

The graphics DDI functions that must be defined by a [printer graphics DLL](print.printer_graphics_dll) are described in [GDI Functions Implemented by Printer and Display Drivers](display.gdi_functions_implemented_by_printer_and_display_drivers). For a list of the subset of graphics driver functions that are relevant to printer graphics DLLs, see [Functions Defined by Printer Graphics DLLs](print.functions_defined_by_printer_graphics_dlls).
