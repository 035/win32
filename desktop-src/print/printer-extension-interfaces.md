# Printer Extension Interfaces

This section provides information about the interfaces and methods that make it possible to create custom print experiences for the new version of the print driver (the V4 print driver). The V4 print driver is available in Windows 8 and later versions of Windows.

Here is a list of the interfaces:

-   [**IPrinterBidiSetRequestCallback**](iprinterbidisetrequestcallback.md)
-   [**IPrinterExtensionAsyncOperation**](iprinterextensionasyncoperation.md)
-   [**IPrinterExtensionContext**](iprinterextensioncontext-interface.md)
-   [**IPrinterExtensionContextCollection**](iprinterextensioncontextcollection.md)
-   [**IPrinterExtensionEvent**](iprinterextensionevent-interface.md)
-   [**IPrinterExtensionEventArgs**](iprinterextensioneventargs.md)
-   [**IPrinterExtensionManager**](iprinterextensionmanager-interface.md)
-   [**IPrinterExtensionRequest**](iprinterextensionrequest-interface.md)
-   [**IPrinterQueue**](iprinterqueue-interface.md)
-   [**IPrinterQueue2**](iprinterqueue2.md)
-   [**IPrinterQueueView**](iprinterqueueview.md)
-   [**IPrinterQueueViewEvent**](iprinterqueueviewevent.md)
-   [**IPrinterQueueEvent**](iprinterqueueevent-interface.md)
-   [**IPrinterScriptContext**](iprinterscriptcontext.md)
-   [**IPrintJob**](iprintjob.md)
-   [**IPrintJobCollection**](iprintjobcollection.md)
