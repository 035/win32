# IPrintOemPS2 interface

This section describes the methods defined for the **IPrintOemPS2** COM interface. In addition to these methods, this interface includes all of the methods defined in the [**IPrintOemPS**](iprintoemps-interface.md) COM interface.

## Members

The **IPrintOemPS2** interface inherits from the [**IUnknown**](com.iunknown) interface. **IPrintOemPS2** also has these types of members:

-   [Methods](#methods)

### Methods

The **IPrintOemPS2** interface has these methods.



| Method                                                                    | Description                                                                                                                                                                                                                                                                                                                                                |
|:--------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IPrintOemPS2::GetPDEVAdjustment**](iprintoemps2-getpdevadjustment.md) | The `IPrintOemPS2::GetPDEVAdjustment` method enables a plug-in to override specific [*PDEV*](wdkgloss.p#wdkgloss-pdev) settings.<br/>                                                                                                                                                                                                  |
| [**IPrintOemPS2::WritePrinter**](iprintoemps2-writeprinter.md)           | The `IPrintOemPS2::WritePrinter` method, if supported, enables a rendering plug-in to capture all output data generated by a Postscript driver. If this method is not supported, the output data would otherwise be sent to the spooler in a call to the spooler's **WritePrinter** API (described in the Microsoft Windows SDK documentation).<br/> |
