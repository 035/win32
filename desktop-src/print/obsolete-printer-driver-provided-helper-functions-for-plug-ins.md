# Obsolete Printer Driver-Provided Helper Functions for Plug-ins

## 

The functions listed in this section, which are declared in printoem.h, are obsolete printer driver helper functions. These helper functions have been superseded by the core driver helper methods exposed by one or more of the [IPrintOemDriverUI](print.iprintoemdriverui_com_interface), [IPrintCoreUI2](print.iprintcoreui2_com_interface), [IPrintOemDriverUni](print.iprintoemdriveruni_com_interface), [IPrintOemDriverPS](print.iprintoemdriverps_com_interface), and [IPrintCorePS2](print.iprintcoreps2_com_interface) COM interfaces.
