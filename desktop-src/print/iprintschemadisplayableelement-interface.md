# IPrintSchemaDisplayableElement interface

Provides the displayable string for a PrintCapabilites PrintSchema element.

## Members

The **IPrintSchemaDisplayableElement** interface inherits from [**IPrintSchemaElement**](iprintschemaelement-interface.md). **IPrintSchemaDisplayableElement** also has these types of members:

-   [Properties](#properties)

### Properties

The **IPrintSchemaDisplayableElement** interface has these properties.



| Property                                                                     | Access type          | Description                                         |
|:-----------------------------------------------------------------------------|:---------------------|:----------------------------------------------------|
| [**DisplayName**](iprintschemadisplayableelement-displayname.md)<br/> | Read-only<br/> | Gets a displayable string for this item.<br/> |
