# IPrintSchemaPageMediaSizeOption interface

Exposes a Print Schema PageMediaSize Option element.

## Members

The **IPrintSchemaPageMediaSizeOption** interface inherits from [**IPrintSchemaOption**](iprintschemaoption-interface.md). **IPrintSchemaPageMediaSizeOption** also has these types of members:

-   [Properties](#properties)

### Properties

The **IPrintSchemaPageMediaSizeOption** interface has these properties.



| Property                                                                              | Access type          | Description                                        |
|:--------------------------------------------------------------------------------------|:---------------------|:---------------------------------------------------|
| [**HeightInMicrons**](iprintschemapagemediasizeoption-heightinmicrons.md)<br/> | Read-only<br/> | Gets the height of the page in microns.<br/> |
| [**WidthInMicrons**](iprintschemapagemediasizeoption-widthinmicrons.md)<br/>   | Read-only<br/> | Gets the width of the page in microns.<br/>  |
