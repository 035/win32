# IXpsRasterizationFactory1 interface

In Windows 8, the improvement of XPSRas to handle high precision colors has led to the development of a new interface, **IXPSRasterizationFactory1**.

## Members

The **IXpsRasterizationFactory1** interface inherits from the [**IUnknown**](https://msdn.microsoft.com/33f1d79a-33fc-4ce5-a372-e08bda378332) interface. **IXpsRasterizationFactory1** also has these types of members:

-   [Methods](#methods)

### Methods

The **IXpsRasterizationFactory1** interface has these methods.



| Method                                                                 | Description                                                                                                     |
|:-----------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**CreateRasterizer**](xpsrasterizationfactory1-createrasterizer1.md) | The `CreateRasterize` method creates an XPS rasterizer object that can handle high precision colors.<br/> |
