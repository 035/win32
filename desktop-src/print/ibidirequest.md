# IBidiRequest interface

The **IBidiRequest** interface allows an application or other objects to compose a bidi request.

## Members

The **IBidiRequest** interface inherits from the [**IUnknown**](com.iunknown) interface. **IBidiRequest** also has these types of members:

-   [Methods](#methods)

### Methods

The **IBidiRequest** interface has these methods.



| Method                                                            | Description                                                  |
|:------------------------------------------------------------------|:-------------------------------------------------------------|
| [**GetEnumCount**](ibidirequest-ibidirequest--getenumcount.md)   | Gets the number of output items.<br/>                  |
| [**GetOutputData**](ibidirequest-ibidirequest--getoutputdata.md) | Gets the output data coming back from the device.<br/> |
| [**GetResult**](ibidirequest-ibidirequest--getresult.md)         | Gets the result code.<br/>                             |
| [**SetInputData**](ibidirequest-ibidirequest--setinputdata.md)   | Sets the data to send to the device.<br/>              |
| [**SetSchema**](ibidirequest-ibidirequest--setschema.md)         | Sets the bidi schema string.<br/>                      |
