# IPrintSchemaOptionCollection interface

Exposes a collection of [**IPrintSchemaOption**](iprintschemaoption-interface.md) objects.

## Members

The **IPrintSchemaOptionCollection** interface inherits from the [**IDispatch**](ebbff4bc-36b2-4861-9efa-ffa45e013eb5) interface. **IPrintSchemaOptionCollection** also has these types of members:

-   [Methods](#methods)
-   [Properties](#properties)

### Methods

The **IPrintSchemaOptionCollection** interface has these methods.



| Method                                              | Description                                                                                        |
|:----------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**GetAt**](iprintschemaoptioncollection-getat.md) | Gets a pointer to an [**IPrintSchemaOption**](iprintschemaoption-interface.md) object.<br/> |



 

### Properties

The **IPrintSchemaOptionCollection** interface has these properties.



| Property                                                           | Access type          | Description                                                                                                                    |
|:-------------------------------------------------------------------|:---------------------|:-------------------------------------------------------------------------------------------------------------------------------|
| [**Count**](iprintschemaoptioncollection-count.md)<br/>     | Read-only<br/> | Gets a count of the number of [**IPrintSchemaOption**](iprintschemaoption-interface.md) objects in the collection.<br/> |
| [**NewEnum**](iprintschemaoptioncollection-newenum.md)<br/> | Read-only<br/> | Gets a pointer to the enumerants of **IPrintSchemaOptionCollection** objects.<br/>                                       |
