# Methods for User Interface Plug-Ins

## 

This section describes the COM interfaces used for communication between Microsoft's printer drivers ([*Unidrv*](wdkgloss.u#wdkgloss-unidrv) and [*Pscript*](wdkgloss.p#wdkgloss-pscript)) and vendor-supplied [user interface plug-ins](print.user_interface_plug_ins). The following topics are provided:

<dl>

[IPrintOemUI Interface](iprintoemui-interface.md)  
[IPrintOemUI2 Interface](iprintoemui2-interface.md)  
[IPrintOemDriverUI Interface](iprintoemdriverui-interface.md)  
[IPrintCoreUI2 Interface](iprintcoreui2-interface.md)  
</dl>
