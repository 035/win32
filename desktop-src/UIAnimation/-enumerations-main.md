# Enumerations

Reference documentation for enumerations defined by the Windows Animation Manager.

## In this section



| Topic                                                                                       | Description                                                                                                                                          |
|---------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**UI\_ANIMATION\_DEPENDENCIES**](ui-animation-dependencies.md)<br/>                 | Defines which aspects of an interpolator depend on a given input.<br/>                                                                         |
| [**UI\_ANIMATION\_IDLE\_BEHAVIOR**](ui-animation-idle-behavior.md)<br/>              | Defines the behavior of a timer when the animation manager is idle.<br/>                                                                       |
| [**UI\_ANIMATION\_MANAGER\_STATUS**](ui-animation-manager-status.md)<br/>            | Defines the activity status of an animation manager.<br/>                                                                                      |
| [**UI\_ANIMATION\_MODE**](ui-animation-mode.md)<br/>                                 | Defines animation modes.<br/>                                                                                                                  |
| [**UI\_ANIMATION\_PRIORITY\_EFFECT**](ui-animation-priority-effect.md)<br/>          | Defines potential effects on a storyboard if a priority comparison returns false.<br/>                                                         |
| [**UI\_ANIMATION\_REPEAT\_MODE**](ui-animation-repeat-mode.md)<br/>                  | Defines the pattern for a loop iteration.<br/>                                                                                                 |
| [**UI\_ANIMATION\_ROUNDING\_MODE**](ui-animation-rounding-mode.md)<br/>              | Defines the rounding modes to be used when the value of an animation variable is converted from a floating-point type to an integer type.<br/> |
| [**UI\_ANIMATION\_SCHEDULING\_RESULT**](ui-animation-scheduling-result.md)<br/>      | Defines results for storyboard scheduling.<br/>                                                                                                |
| [**UI\_ANIMATION\_SLOPE**](ui-animation-slope.md)<br/>                               | Defines animation slope characteristics.<br/>                                                                                                  |
| [**UI\_ANIMATION\_STORYBOARD\_STATUS**](ui-animation-storyboard-status.md)<br/>      | Defines the status for a storyboard.<br/>                                                                                                      |
| [**UI\_ANIMATION\_TIMER\_CLIENT\_STATUS**](ui-animation-timer-client-status.md)<br/> | Defines activity status for a timer's client.<br/>                                                                                             |
| [**UI\_ANIMATION\_UPDATE\_RESULT**](ui-animation-update-result.md)<br/>              | Defines results for animation updates.<br/>                                                                                                    |
