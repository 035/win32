# Animation Management Interfaces

This section contains the reference specifications for the Windows Animation Manager interfaces that support animation management.

## In this section



| Topic                                                                                   | Description                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAnimationManager**](iuianimationmanager.md)<br/>                           | Defines the animation manager, which provides a central interface for creating and managing animations.<br/>                                                       |
| [**IUIAnimationManager2**](iuianimationmanager2.md)<br/>                         | Defines an [**animation manager**](iuianimationmanager2.md), which provides a central interface for creating and managing animations in multiple dimensions.<br/> |
| [**IUIAnimationManagerEventHandler**](iuianimationmanagereventhandler.md)<br/>   | Defines a method for handling status updates to an animation manager.<br/>                                                                                         |
| [**IUIAnimationManagerEventHandler2**](iuianimationmanagereventhandler2.md)<br/> | Defines a method for handling updates to an [**animation manager**](iuianimationmanager2.md).<br/>                                                                |
