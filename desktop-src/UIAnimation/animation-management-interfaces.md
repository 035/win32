# Animation Management Interfaces

This section contains the reference specifications for the Windows Animation Manager interfaces that support animation management.

## In this section



| Topic                                                                                   | Description                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAnimationManager**](/windows/win32/UIAnimation/nn-uianimation-iuianimationmanager?branch=master)<br/>                           | Defines the animation manager, which provides a central interface for creating and managing animations.<br/>                                                       |
| [**IUIAnimationManager2**](/windows/win32/UIAnimation/nn-uianimation-iuianimationmanager2?branch=master)<br/>                         | Defines an [**animation manager**](/windows/win32/UIAnimation/nn-uianimation-iuianimationmanager2?branch=master), which provides a central interface for creating and managing animations in multiple dimensions.<br/> |
| [**IUIAnimationManagerEventHandler**](/windows/win32/UIAnimation/nn-uianimation-iuianimationmanagereventhandler?branch=master)<br/>   | Defines a method for handling status updates to an animation manager.<br/>                                                                                         |
| [**IUIAnimationManagerEventHandler2**](/windows/win32/UIAnimation/nn-uianimation-iuianimationmanagereventhandler2?branch=master)<br/> | Defines a method for handling updates to an [**animation manager**](/windows/win32/UIAnimation/nn-uianimation-iuianimationmanager2?branch=master).<br/>                                                                |
