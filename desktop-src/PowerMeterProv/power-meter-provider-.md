# Power Meter Provider

This section describes the Windows Management Instrumentation (WMI) provider that supports the Power Metering and Budgeting (PMB) infrastructure. These WMI classes are the primary interface for the query of Power Meter Interface (PMI) information from underlying power meters on the system.

The PMB WMI class objects conform to version 1.1.0 of the Distributed Management Task Force (DMTF) Power Supply Profile. For more information, refer to the DMTF Power Supply Profile documentation ([http://www.dmtf.org/standards/published\_documents/DSP1015\_1.1.0.pdf](http://go.microsoft.com/fwlink/p/?linkid=147809)).
