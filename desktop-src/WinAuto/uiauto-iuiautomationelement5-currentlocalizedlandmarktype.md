# IUIAutomationElement5::CurrentLocalizedLandmarkType property

Gets a string containing the current localized landmark type for the automation element.

This property is read-only.

## Syntax


```C++
HRESULT get_CurrentLocalizedLandmarkType(
  [out] BSTR *retVal
);
```



## Property value

A string containing the current localized landmark type for the automation element.
