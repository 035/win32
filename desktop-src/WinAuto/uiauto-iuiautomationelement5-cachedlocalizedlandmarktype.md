# IUIAutomationElement5::CachedLocalizedLandmarkType property

Gets a string containing the cached localized landmark type for the automation element.

This property is read-only.

## Syntax


```C++
HRESULT get_CachedLocalizedLandmarkType(
  [out, retval] BSTR *retVal
);
```



## Property value

A string containing the cached localized landmark type for the automation element.
