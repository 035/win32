# UI Automation Structures

This section describes the structures used by UI Automation client applications and UI Automation provider for Microsoft Win32.

## In this section



| Structure                                                                            | Description                                                                                   |
|--------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](uiauto-extendedproperty.md)<br/>                       | Contains information about an extended property. <br/>                                  |
| [**UIAutomationEventInfo**](uiauto-uiautomationeventinfostruct.md)<br/>       | Contains information about a custom event.<br/>                                         |
| [**UIAutomationMethodInfo**](uiauto-uiautomationmethodinfostruct.md)<br/>     | Contains information about a method that is supported by a custom control pattern.<br/> |
| [**UIAutomationParameter**](uiauto-uiautomationparameterstruct.md)<br/>       | Contains information about a parameter of a custom control pattern.<br/>                |
| [**UIAutomationPatternInfo**](uiauto-uiautomationpatterninfostruct.md)<br/>   | Contains information about a custom control pattern.<br/>                               |
| [**UIAutomationPropertyInfo**](uiauto-uiautomationpropertyinfostruct.md)<br/> | Contains information about a custom property.<br/>                                      |
| [**UiaChangeInfo**](uiachangeinfo.md)<br/>                                    | Contains data about a UI Automation change that occurred.<br/>                          |
| [**UiaPoint**](uiauto-uiapointstruct.md)<br/>                                 | Contains the coordinates of a point. <br/>                                              |
| [**UiaRect**](uiauto-uiarectstruct.md)<br/>                                   | Contains the position and size of a rectangle, in screen coordinates.<br/>              |
