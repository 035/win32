# UI Automation Structures

This section describes the structures used by UI Automation client applications and UI Automation provider for Microsoft Win32.

## In this section



| Structure                                                                            | Description                                                                                   |
|--------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](/windows/win32/UIAutomationClient/ns-uiautomationclient-extendedproperty?branch=master)<br/>                       | Contains information about an extended property. <br/>                                  |
| [**UIAutomationEventInfo**](/windows/win32/UIAutomationCore/ns-uiautomationcore-uiautomationeventinfo?branch=master)<br/>       | Contains information about a custom event.<br/>                                         |
| [**UIAutomationMethodInfo**](/windows/win32/UIAutomationCore/ns-uiautomationcore-uiautomationmethodinfo?branch=master)<br/>     | Contains information about a method that is supported by a custom control pattern.<br/> |
| [**UIAutomationParameter**](/windows/win32/UIAutomationCore/ns-uiautomationcore-uiautomationparameter?branch=master)<br/>       | Contains information about a parameter of a custom control pattern.<br/>                |
| [**UIAutomationPatternInfo**](/windows/win32/UIAutomationCore/ns-uiautomationcore-uiautomationpatterninfo?branch=master)<br/>   | Contains information about a custom control pattern.<br/>                               |
| [**UIAutomationPropertyInfo**](/windows/win32/UIAutomationCore/ns-uiautomationcore-uiautomationpropertyinfo?branch=master)<br/> | Contains information about a custom property.<br/>                                      |
| [**UiaChangeInfo**](/windows/win32/UIAutomationCore/ns-uiautomationcore-uiachangeinfo?branch=master)<br/>                                    | Contains data about a UI Automation change that occurred.<br/>                          |
| [**UiaPoint**](/windows/win32/UIAutomationCore/ns-uiautomationcore-uiapoint?branch=master)<br/>                                 | Contains the coordinates of a point. <br/>                                              |
| [**UiaRect**](/windows/win32/UIAutomationCore/ns-uiautomationcore-uiarect?branch=master)<br/>                                   | Contains the position and size of a rectangle, in screen coordinates.<br/>              |
