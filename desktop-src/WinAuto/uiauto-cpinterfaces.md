# Control Pattern Interfaces for Providers

This section describes *control pattern* interfaces implemented by UI Automation providers to expose information about controls in unmanaged Windows applications.

## In this section



| Interface                                                                          | Description                                                                                                                                                                                                                                                                                                         |
|------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAnnotationProvider**](https://msdn.microsoft.com/library/windows/desktop/hh448757)<br/>           | Exposes the properties of an annotation in a document.<br/>                                                                                                                                                                                                                                                   |
| [**IDockProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-idockprovider?branch=master)<br/>                           | Provides access to an element in a docking container. <br/>                                                                                                                                                                                                                                                   |
| [**IDragProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-idragprovider?branch=master)<br/>                           | Enables a Microsoft UI Automation element to describe itself as an element that can be dragged as part of a drag-and-drop operation.<br/>                                                                                                                                                                     |
| [**IDropTargetProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-idroptargetprovider?branch=master)<br/>               | Enables a UI Automation element to describe itself as an element that can receive a drop of a dragged element as part of a UI Automation drag-and-drop operation.<br/>                                                                                                                                        |
| [**IExpandCollapseProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iexpandcollapseprovider?branch=master)<br/>       | Provides access to a control that visually expands to display content, and collapses to hide content. <br/>                                                                                                                                                                                                   |
| [**IGridItemProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-igriditemprovider?branch=master)<br/>                   | Provides access to individual child controls of containers that implement [**IGridProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-igridprovider?branch=master).<br/>                                                                                                                                                                                 |
| [**IGridProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-igridprovider?branch=master)<br/>                           | Provides access to controls that act as containers for a collection of child elements organized in a two-dimensional logical coordinate system that can be traversed (that is, a UI Automation client can move to adjacent controls) by using the keyboard.<br/>                                              |
| [**IInvokeProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iinvokeprovider?branch=master)<br/>                       | Provides access to controls that initiate or perform a single, unambiguous action and do not maintain state when activated.<br/>                                                                                                                                                                              |
| [**IItemContainerProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iitemcontainerprovider?branch=master)<br/>         | Provides access to controls that act as containers of other controls, such as a virtual list-view. <br/>                                                                                                                                                                                                      |
| [**ILegacyIAccessibleProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-ilegacyiaccessibleprovider?branch=master)<br/> | Enables UI Automation clients to access the underlying [**IAccessible**](/windows/win32/oleacc/nn-oleacc-iaccessible?branch=master) implementation of Microsoft Active Accessibility elements.<br/>                                                                                                                                                     |
| [**IMultipleViewProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-imultipleviewprovider?branch=master)<br/>           | Provides access to controls that provide, and are able to switch between, multiple representations of the same set of information or child controls.<br/>                                                                                                                                                     |
| [**IObjectModelProvider**](https://msdn.microsoft.com/library/windows/desktop/hh448776)<br/>         | Provides access to the underlying object model implemented by a control or application. <br/>                                                                                                                                                                                                                 |
| [**IRangeValueProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-irangevalueprovider?branch=master)<br/>               | Provides access to controls that can be set to a value within a range. <br/>                                                                                                                                                                                                                                  |
| [**IScrollItemProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iscrollitemprovider?branch=master)<br/>               | Provides access to individual child controls of containers that implement [**IScrollProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iscrollprovider?branch=master). <br/>                                                                                                                                                                            |
| [**IScrollProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iscrollprovider?branch=master)<br/>                       | Provides access to controls that act as scrollable containers for a collection of child objects.<br/>                                                                                                                                                                                                         |
| [**ISelectionItemProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider?branch=master)<br/>         | Provides access to individual, selectable child controls of containers that implement [**ISelectionProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iselectionprovider?branch=master). <br/>                                                                                                                                                          |
| [**ISelectionProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iselectionprovider?branch=master)<br/>                 | Provides access to controls that act as containers for a collection of individual, selectable child items.<br/>                                                                                                                                                                                               |
| [**ISelectionProvider2**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iselectionprovider2?branch=master)<br/>               | Extends the [**ISelectionItemProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider?branch=master) interface to provide information about selected items.<br/>                                                                                                                                                                       |
| [**ISpreadsheetItemProvider**](https://msdn.microsoft.com/library/windows/desktop/hh448790)<br/> | Provides access to information about an item (cell) in a spreadsheet. <br/>                                                                                                                                                                                                                                   |
| [**ISpreadsheetProvider**](https://msdn.microsoft.com/library/windows/desktop/hh448796)<br/>         | Provides access to items (cells) in a spreadsheet.<br/>                                                                                                                                                                                                                                                       |
| [**IStylesProvider**](https://msdn.microsoft.com/library/windows/desktop/hh448800)<br/>                   | Provides access to the visual styles associated with the content of a document.<br/>                                                                                                                                                                                                                          |
| [**ISynchronizedInputProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-isynchronizedinputprovider?branch=master)<br/> | Enables UI Automation client applications to direct the mouse or keyboard input to a specific UI element. <br/>                                                                                                                                                                                               |
| [**ITableItemProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itableitemprovider?branch=master)<br/>                 | Provides access to child controls of containers that implement [**ITableProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itableprovider?branch=master). <br/>                                                                                                                                                                                         |
| [**ITableProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itableprovider?branch=master)<br/>                         | Provides access to controls that act as containers for a collection of child elements. The children of this element must implement [**ITableItemProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itableitemprovider?branch=master) and be organized in a two-dimensional logical coordinate system that can be traversed by using the keyboard. <br/> |
| [**ITextChildProvider**](https://msdn.microsoft.com/library/windows/desktop/hh437317)<br/>            | Provides access to a text-based control (or an object embedded in text) that is a child or descendant of another text-based control. <br/>                                                                                                                                                                    |
| [**ITextEditProvider**](https://msdn.microsoft.com/library/windows/desktop/dn302166)<br/>               | Extends the [**ITextProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itextprovider?branch=master) interface to enable UI Automation providers to expose programmatic text-edit actions.<br/>                                                                                                                                                          |
| [**ITextProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itextprovider?branch=master)<br/>                           | Provides access to controls that contain text. <br/>                                                                                                                                                                                                                                                          |
| [**ITextProvider2**](https://msdn.microsoft.com/library/windows/desktop/hh448818)<br/>                     | Extends the [**ITextProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itextprovider?branch=master) interface to enable UI Automation providers to expose textual content that is the target of an annotation, and information about a caret that belongs to the provider.<br/>                                                                         |
| [**ITextRangeProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itextrangeprovider?branch=master)<br/>                 | Provides access to a span of continuous text in a text container that implements [**ITextProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itextprovider?branch=master) or [**ITextProvider2**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itextprovider2?branch=master).<br/>                                                                                                                       |
| [**ITextRangeProvider2**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itextrangeprovider2?branch=master)<br/>               | Extends the [**ITextRangeProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itextrangeprovider?branch=master) interface to enable UI Automation providers to invoke context menus.<br/>                                                                                                                                                                 |
| [**IToggleProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itoggleprovider?branch=master)<br/>                       | Provides access to controls that can cycle through a set of states and maintain a state after it is set.<br/>                                                                                                                                                                                                 |
| [**ITransformProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itransformprovider?branch=master)<br/>                 | Provides access to controls that can be moved, resized, and/or rotated within a two-dimensional space. <br/>                                                                                                                                                                                                  |
| [**ITransformProvider2**](https://msdn.microsoft.com/library/windows/desktop/hh448824)<br/>           | Extends the [**ITransformProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-itransformprovider?branch=master) interface to enable UI Automation providers to expose properties to support the viewport zooming functionality of a control. <br/>                                                                                                        |
| [**IValueProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-ivalueprovider?branch=master)<br/>                         | Provides access to controls that have an intrinsic value that does not span a range, and that can be represented as a string. <br/>                                                                                                                                                                           |
| [**IVirtualizedItemProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-ivirtualizeditemprovider?branch=master)<br/>     | Provides access to virtualized items, which are items that are represented by placeholder automation elements in the UI Automation tree.<br/>                                                                                                                                                                 |
| [**IWindowProvider**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iwindowprovider?branch=master)<br/>                       | Provides access to the fundamental window-based functionality of a control.<br/>                                                                                                                                                                                                                              |
