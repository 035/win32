# IUIAutomationElement5::CachedLandmarkType property

Gets the cached landmark type ID for the automation element.

This property is read-only.

## Syntax


```C++
HRESULT get_CachedLandmarkType();
```



## Property value

The cached landmark type ID for the automation element.
