# In-Context and Out-of-Context Hook Functions

When registering a hook function with [**SetWinEventHook**](/windows/win32/Winuser/nf-winuser-setwineventhook?branch=master), clients specify whether the hook function is *in-context* or *out-of-context*. These terms describe the memory location of the hook function relative to the server's address space.
