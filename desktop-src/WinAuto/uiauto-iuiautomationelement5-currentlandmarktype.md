# IUIAutomationElement5::CurrentLandmarkType property

Gets the current landmark type ID for the automation element.

This property is read-only.

## Syntax


```C++
HRESULT get_CurrentLandmarkType(
  [out, retval] LANDMARKTYPEID *retVal
);
```



## Property value

The current landmark type ID for the automation element.
