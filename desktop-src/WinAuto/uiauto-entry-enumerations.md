# UI Automation Enumerations

This section describes the enumerated types used by UI Automation client applications and UI Automation provider for Microsoft Win32.

## In this section



| Enumeration                                                                        | Description                                                                                                                                                                                                                                                          |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActiveEnd**](https://msdn.microsoft.com/library/windows/desktop/hh448742)<br/>                               | Contains possible values for the [**SelectionActiveEnd**](uiauto-textattribute-ids.md#uia-selectionactiveendattributeid) text attribute, which indicates the location of the caret relative to a text range that represents the currently selected text.<br/> |
| [**AnimationStyle**](/windows/win32/UIAutomationCore/ne-uiautomationcore-animationstyle?branch=master)<br/>                     | Contains values for the AnimationStyle text attribute.<br/>                                                                                                                                                                                                    |
| [**AsyncContentLoadedState**](/windows/win32/UIAutomationCoreApi/ne-uiautomationcoreapi-asynccontentloadedstate?branch=master)<br/>   | Contains values that describe the progress of asynchronous loading of content.<br/>                                                                                                                                                                            |
| [**AutomationElementMode**](/windows/win32/UIAutomationClient/ne-uiautomationclient-automationelementmode?branch=master)<br/>       | Contains values that specify the type of reference to use when returning UI Automation elements.<br/>                                                                                                                                                          |
| [**AutomationIdentifierType**](/windows/win32/UIAutomationCoreApi/ne-uiautomationcoreapi-automationidentifiertype?branch=master)<br/>         | Contains values used in the [**UiaLookupId**](/windows/win32/UIAutomationCoreApi/nf-uiautomationcoreapi-uialookupid?branch=master) function.<br/>                                                                                                                                                                   |
| [**BulletStyle**](/windows/win32/UIAutomationCore/ne-uiautomationcore-bulletstyle?branch=master)<br/>                           | Contains values for the BulletStyle text attribute.<br/>                                                                                                                                                                                                       |
| [**CapStyle**](/windows/win32/UIAutomationCore/ne-uiautomationcore-capstyle?branch=master)<br/>                                 | Contains values that specify the value of the CapStyle text attribute.<br/>                                                                                                                                                                                    |
| [**CaretBidiMode**](https://msdn.microsoft.com/library/windows/desktop/hh448744)<br/>                       | Contains possible values for the [**CaretBidiMode**](uiauto-textattribute-ids.md#uia-caretbidimodeattributeid) text attribute, which indicates whether the caret is in text that flows from left to right, or from right to left.<br/>                        |
| [**CaretPosition**](https://msdn.microsoft.com/library/windows/desktop/hh448745)<br/>                       | Contains possible values for the [**CaretPosition**](uiauto-textattribute-ids.md#uia-caretpositionattributeid) text attribute, which indicates the location of the caret relative to a line of text in a text range.<br/>                                     |
| [**ConditionType**](/windows/win32/UIAutomationCoreApi/ne-uiautomationcoreapi-conditiontype?branch=master)<br/>                       | Contains values that specify a type of [**UiaCondition**](/windows/win32/UIAutomationCoreApi/ns-uiautomationcoreapi-uiacondition?branch=master).<br/>                                                                                                                                                                 |
| [**DockPosition**](/windows/win32/UIAutomationCore/ne-uiautomationcore-dockposition?branch=master)<br/>                            | Contains values that specify the location of a docking window represented by the [Dock](uiauto-implementingdock.md) *control pattern*.<br/>                                                                                                                   |
| [**EventArgsType**](/windows/win32/UIAutomationCoreApi/ne-uiautomationcoreapi-eventargstype?branch=master)<br/>                       | Contains values that specify the event type described by a [**UiaEventArgs**](/windows/win32/UIAutomationCoreApi/ns-uiautomationcoreapi-uiaeventargs?branch=master) structure.<br/>                                                                                                                                   |
| [**ExpandCollapseState**](/windows/win32/UIAutomationCore/ne-uiautomationcore-expandcollapsestate?branch=master)<br/>           | Contains values that specify the state of a UI element that can be expanded and collapsed.<br/>                                                                                                                                                                |
| [**FlowDirections**](/windows/win32/UIAutomationCore/ne-uiautomationcore-flowdirections?branch=master)<br/>                     | Contains values for the TextFlowDirections text attribute. <br/>                                                                                                                                                                                               |
| [**FillType**](/windows/win32/UIAutomationCore/ne-uiautomationcore-filltype?branch=master)<br/>                                 | Contains values for the FillType attribute.<br/>                                                                                                                                                                                                               |
| [**HorizontalTextAlignmentEnum**](/windows/win32/UIAutomationCore/?branch=master)<br/>   | Contains values for the HorizontalTextAlignment text attribute.<br/>                                                                                                                                                                                           |
| [**LiveSetting**](https://msdn.microsoft.com/library/windows/desktop/hh437308)<br/>                           | Contains possible values for the LiveSetting property. This property is implemented by provider elements that are part of a live region.<br/>                                                                                                                  |
| [**NavigateDirection**](/windows/win32/UIAutomationCore/ne-uiautomationcore-navigatedirection?branch=master)<br/>                          | Contains values used to specify the direction of navigation within the Microsoft UI Automation tree.<br/>                                                                                                                                                      |
| [**NormalizeState**](/windows/win32/UIAutomationCoreApi/ne-uiautomationcoreapi-normalizestate?branch=master)<br/>                     | Contains values that specify the behavior of [**UiaGetUpdatedCache**](/windows/win32/UIAutomationCoreApi/nf-uiautomationcoreapi-uiagetupdatedcache?branch=master).<br/>                                                                                                                                             |
| [**NotificationKind**](uiauto-notificationkind.md)<br/>                     | Defines values that indicate the type of a notification event, and a hint to the listener about the processing of the event.<br/>                                                                                                                              |
| [**NotificationProcessing**](uiauto-notificationprocessing.md)<br/>         | Defines values that indicate how a notification should be processed.<br/>                                                                                                                                                                                      |
| [**OrientationType**](/windows/win32/UIAutomationCore/ne-uiautomationcore-orientationtype?branch=master)<br/>                       | Contains values that specify the orientation of a control.<br/>                                                                                                                                                                                                |
| [**OutlineStyles**](/windows/win32/UIAutomationCore/ne-uiautomationcore-outlinestyles?branch=master)<br/>                       | Contains values for the OutlineStyle text attribute.<br/>                                                                                                                                                                                                      |
| [**PropertyConditionFlags**](/windows/win32/UIAutomationClient/ne-uiautomationclient-propertyconditionflags?branch=master)<br/>     | Contains values used in creating property conditions.<br/>                                                                                                                                                                                                     |
| [**ProviderOptions**](/windows/win32/UIAutomationCore/ne-uiautomationcore-provideroptions?branch=master)<br/>                       | Contains values that specify the type of UI Automation provider. The [**IRawElementProviderSimple::ProviderOptions**](/windows/win32/UIAutomationCore/nf-uiautomationcore-irawelementprovidersimple-get_provideroptions?branch=master) property uses this enumeration.<br/>                                               |
| [**ProviderType**](/windows/win32/UIAutomationCoreApi/ne-uiautomationcoreapi-providertype?branch=master)<br/>                         | Contains values that specify the type of a client-side (proxy) UI Automation provider.<br/>                                                                                                                                                                    |
| [**RowOrColumnMajor**](/windows/win32/UIAutomationCore/ne-uiautomationcore-roworcolumnmajor?branch=master)<br/>                 | Contains values that specify whether data in a table should be read primarily by row or by column.<br/>                                                                                                                                                        |
| [**SayAsInterpretAs**](/windows/win32/UIAutomationCore/ne-uiautomationcore-sayasinterpretas?branch=master)<br/>                     | Defines the values that indicate how a text-to-speech engine should interpret specific data.<br/>                                                                                                                                                              |
| [**ScrollAmount**](/windows/win32/UIAutomationCore/ne-uiautomationcore-scrollamount?branch=master)<br/>                             | Contains values that specify the direction and distance to scroll.<br/>                                                                                                                                                                                        |
| [**StructureChangeType**](/windows/win32/UIAutomationCore/ne-uiautomationcore-structurechangetype?branch=master)<br/>           | Contains values that specify the type of change in the UI Automation tree structure.<br/>                                                                                                                                                                      |
| [**SupportedTextSelection**](/windows/win32/UIAutomationCore/ne-uiautomationcore-supportedtextselection?branch=master)<br/>     | Contains values that specify the supported text selection attribute.<br/>                                                                                                                                                                                      |
| [**SynchronizedInputType**](/windows/win32/UIAutomationCore/ne-uiautomationcore-synchronizedinputtype?branch=master)<br/>       | Contains values that specify the type of synchronized input.<br/>                                                                                                                                                                                              |
| [**TextDecorationLineStyle**](/windows/win32/UIAutomationCore/ne-uiautomationcore-textdecorationlinestyle?branch=master)<br/>              | Contains values that specify the OverlineStyle, StrikethroughStyle, and UnderlineStyle text attributes.<br/>                                                                                                                                                   |
| [**TextEditChangeType**](/windows/win32/UIAutomationCore/ne-uiautomationcore-texteditchangetype?branch=master)<br/>             | Describes the text editing change being performed by controls when text-edit events are raised or handled.<br/>                                                                                                                                                |
| [**TextPatternRangeEndpoint**](/windows/win32/UIAutomationCore/ne-uiautomationcore-textpatternrangeendpoint?branch=master)<br/> | Contains values that specify the endpoints of a text range.<br/>                                                                                                                                                                                               |
| [**TextUnit**](/windows/win32/UIAutomationCore/ne-uiautomationcore-textunit?branch=master)<br/>                                 | Contains values that specify units of text for the purposes of navigation.<br/>                                                                                                                                                                                |
| [**ToggleState**](/windows/win32/UIAutomationCore/ne-uiautomationcore-togglestate?branch=master)<br/>                               | Contains values that specify the toggle state of a UI Automation element that implements the [Toggle](uiauto-implementingtoggle.md) *control pattern*.<br/>                                                                                                   |
| [**TreeScope**](/windows/win32/UIAutomationClient/ne-uiautomationclient-treescope?branch=master)<br/>                               | Contains values that specify the scope of various operations in the UI Automation tree.<br/>                                                                                                                                                                   |
| [**TreeTraversalOptions**](uiauto-treetraversaloptions.md)<br/>             | Defines values that can be used to customize tree navigation order.<br/>                                                                                                                                                                                       |
| [**UIAutomationType**](/windows/win32/UIAutomationCore/ne-uiautomationcore-uiautomationtype?branch=master)<br/>                 | Contains values used to indicate UI Automation data types.<br/>                                                                                                                                                                                                |
| [**VisualEffects**](/windows/win32/UIAutomationCore/ne-uiautomationcore-visualeffects?branch=master)<br/>                       | Contains values for the VisualEffects attribute.<br/>                                                                                                                                                                                                          |
| [**WindowInteractionState**](/windows/win32/UIAutomationCore/ne-uiautomationcore-windowinteractionstate?branch=master)<br/>     | Contains values that specify the current state of the window for purposes of user interaction.<br/>                                                                                                                                                            |
| [**WindowVisualState**](/windows/win32/UIAutomationCore/ne-uiautomationcore-windowvisualstate?branch=master)<br/>               | Contains values that specify the visual state of a window.<br/>                                                                                                                                                                                                |
| [**ZoomUnit**](/windows/win32/UIAutomationCore/ne-uiautomationcore-zoomunit?branch=master)<br/>                                     | Contains possible values for the [**IUIAutomationTransformPattern2::ZoomByUnit**](/windows/win32/UIAutomationClient/nf-uiautomationclient-iuiautomationtransformpattern2-zoombyunit?branch=master) method, which zooms the viewport of a control by the specified unit.<br/>                                              |
