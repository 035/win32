# Control Pattern Interfaces for Clients

This section describes control pattern interfaces for unmanaged UI Automation client applications.

## In this section



| Interface                                                                                                | Description                                                                                                                                                                                                                                                                                                                                             |
|----------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAutomationAnnotationPattern**](https://msdn.microsoft.com/library/windows/desktop/hh437209)<br/>           | Provides access to the properties of an annotation in a document.<br/>                                                                                                                                                                                                                                                                            |
| [**IUIAutomationCustomNavigationPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationcustomnavigationpattern?branch=master)<br/>   | Exposes a method to support access by a Microsoft UI Automation client to controls that support a custom navigation order.<br/>                                                                                                                                                                                                                   |
| [**IUIAutomationDockPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationdockpattern?branch=master)<br/>                           | Provides access to a control that enables child elements to be arranged horizontally and vertically, relative to each other.<br/>                                                                                                                                                                                                                 |
| [**IUIAutomationDragPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationdragpattern?branch=master)<br/>                           | Provides access to information exposed by a UI Automation provider for an element that can be dragged as part of a drag-and-drop operation.<br/>                                                                                                                                                                                                  |
| [**IUIAutomationDropTargetPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationdroptargetpattern?branch=master)<br/>               | Provides access to drag-and-drop information exposed by a UI Automation provider for an element that can be the drop target of a drag-and-drop operation.<br/>                                                                                                                                                                                    |
| [**IUIAutomationExpandCollapsePattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationexpandcollapsepattern?branch=master)<br/>       | Provides access to a control that can visually expand to display content, and collapse to hide content.<br/>                                                                                                                                                                                                                                      |
| [**IUIAutomationGridItemPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationgriditempattern?branch=master)<br/>                   | Provides access to a child control in a grid-style container that supports the [**IUIAutomationGridPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationgridpattern?branch=master) interface.<br/>                                                                                                                                                                                |
| [**IUIAutomationGridPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationgridpattern?branch=master)<br/>                           | Provides access to a control that acts as a container for a collection of child controls that are organized in a two-dimensional logical coordinate system that can be traversed by row and column.<br/>                                                                                                                                          |
| [**IUIAutomationInvokePattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationinvokepattern?branch=master)<br/>                       | Exposes a method that enables a client application to invoke the action of a control (typically a button).<br/>                                                                                                                                                                                                                                   |
| [**IUIAutomationItemContainerPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationitemcontainerpattern?branch=master)<br/>         | Exposes a method that retrieves an item from a container, such as a virtual list.<br/>                                                                                                                                                                                                                                                            |
| [**IUIAutomationLegacyIAccessiblePattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationlegacyiaccessiblepattern?branch=master)<br/> | Exposes methods and properties that enable UI Automation clients to retrieve UI information from Microsoft Active Accessibility (MSAA) servers.<br/>                                                                                                                                                                                              |
| [**IUIAutomationMultipleViewPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationmultipleviewpattern?branch=master)<br/>           | Provides access to a control that can switch between multiple representations of the same information or set of child controls.<br/>                                                                                                                                                                                                              |
| [**IUIAutomationObjectModelPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationobjectmodelpattern?branch=master)<br/>             | Provides access to the underlying object model implemented by a control or application. <br/>                                                                                                                                                                                                                                                     |
| [**IUIAutomationPatternHandler**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iuiautomationpatternhandler?branch=master)<br/>                     | Returns a client API wrapper object and to unmarshal property and method requests to an actual provider instance.<br/>                                                                                                                                                                                                                            |
| [**IUIAutomationPatternInstance**](/windows/win32/UIAutomationCore/nn-uiautomationcore-iuiautomationpatterninstance?branch=master)<br/>                   | Represents a control pattern object. The client API wrapper uses this interface to implement all property and method calls in terms of the [**GetProperty**](/windows/win32/UIAutomationCore/nf-uiautomationcore-iuiautomationpatterninstance-getproperty?branch=master) and [**CallMethod**](/windows/win32/UIAutomationCore/nf-uiautomationcore-iuiautomationpatterninstance-callmethod?branch=master) methods.<br/>                                          |
| [**IUIAutomationRangeValuePattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationrangevaluepattern?branch=master)<br/>               | Provides access to a control that presents a range of values.<br/>                                                                                                                                                                                                                                                                                |
| [**IUIAutomationScrollItemPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationscrollitempattern?branch=master)<br/>               | Exposes a method that enables an item in a scrollable view to be placed in a visible portion of the view.<br/>                                                                                                                                                                                                                                    |
| [**IUIAutomationScrollPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationscrollpattern?branch=master)<br/>                       | Provides access to a control that acts as a scrollable container for a collection of child elements.<br/>                                                                                                                                                                                                                                         |
| [**IUIAutomationSelectionItemPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionitempattern?branch=master)<br/>         | Provides access to the selectable child items of a container control that supports [**IUIAutomationSelectionPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern?branch=master).<br/>                                                                                                                                                                            |
| [**IUIAutomationSelectionPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern?branch=master)<br/>                 | Provides access to a control that contains selectable child items. The children of this element support [**IUIAutomationSelectionItemPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionitempattern?branch=master).<br/>                                                                                                                                               |
| [**IUIAutomationSelectionPattern2**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern2?branch=master)<br/>               | Extends the [**IUIAutomationSelectionPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern?branch=master) interface to provide information about selected items.<br/>                                                                                                                                                                                             |
| [**IUIAutomationSpreadsheetItemPattern**](https://msdn.microsoft.com/library/windows/desktop/hh437267)<br/> | Enables a client application to retrieve information about an item (cell) in a spreadsheet.<br/>                                                                                                                                                                                                                                                  |
| [**IUIAutomationSpreadsheetPattern**](https://msdn.microsoft.com/library/windows/desktop/hh437280)<br/>         | Enables a client application to access the items (cells) in a spreadsheet.<br/>                                                                                                                                                                                                                                                                   |
| [**IUIAutomationStylesPattern**](https://msdn.microsoft.com/library/windows/desktop/hh437282)<br/>                   | Enables UI Automation clients to retrieve the visual styles associated with an element in a document.<br/>                                                                                                                                                                                                                                        |
| [**IUIAutomationSynchronizedInputPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationsynchronizedinputpattern?branch=master)<br/> | Provides access to the keyboard or mouse input of a control.<br/>                                                                                                                                                                                                                                                                                 |
| [**IUIAutomationTableItemPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtableitempattern?branch=master)<br/>                 | Provides access to a child element in a container that supports [**IUIAutomationTablePattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtablepattern?branch=master). <br/>                                                                                                                                                                                                      |
| [**IUIAutomationTablePattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtablepattern?branch=master)<br/>                         | Provides access to a control that acts as a container for a collection of child elements. The children of this element support [**IUIAutomationTableItemPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtableitempattern?branch=master) and are organized in a two-dimensional logical coordinate system that can be traversed by row and column.<br/>                       |
| [**IUIAutomationTextChildPattern**](https://msdn.microsoft.com/library/windows/desktop/hh437296)<br/>             | Provides access a text-based control (or an object embedded in text) that is a child or descendant of another text-based control. <br/>                                                                                                                                                                                                           |
| [**IUIAutomationTextEditPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtexteditpattern?branch=master)<br/>                   | Provides access to a control that modifies text, for example a control that performs auto-correction or enables input composition through an Input Method Editor (IME).<br/>                                                                                                                                                                      |
| [**IUIAutomationTextPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern?branch=master)<br/>                           | Provides access to a control that contains text.<br/>                                                                                                                                                                                                                                                                                             |
| [**IUIAutomationTextPattern2**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern2?branch=master)<br/>                         | Extends the [**IUIAutomationTextPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern?branch=master) interface. <br/>                                                                                                                                                                                                                                                  |
| [**IUIAutomationTextRange**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange?branch=master)<br/>                               | Provides access to a span of continuous text in a container that supports the [**IUIAutomationTextPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern?branch=master) interface. Client applications can use the [**IUIAutomationTextRange**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange?branch=master) interface to select, compare, and retrieve embedded objects from the text span.<br/> |
| [**IUIAutomationTextRange2**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange2?branch=master)<br/>                             | Extends the [**IUIAutomationTextRange**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange?branch=master) interface to enable UI Automation clients to programmatically invoke context menus.<br/>                                                                                                                                                                              |
| [**IUIAutomationTextRange3**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange3?branch=master)<br/>                             | Extends the [**IUIAutomationTextRange2**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange2?branch=master) interface to support faster access to the underlying rich text data on a text range.<br/>                                                                                                                                                                           |
| [**IUIAutomationTextRangeArray**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrangearray?branch=master)<br/>                     | Represents a collection of [**IUIAutomationTextRange**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange?branch=master) objects.<br/>                                                                                                                                                                                                                                          |
| [**IUIAutomationTogglePattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtogglepattern?branch=master)<br/>                       | Provides access to a control that can cycle through a set of states, and maintain a state after it is set.<br/>                                                                                                                                                                                                                                   |
| [**IUIAutomationTransformPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtransformpattern?branch=master)<br/>                 | Provides access to a control that can be moved, resized, or rotated.<br/>                                                                                                                                                                                                                                                                         |
| [**IUIAutomationTransformPattern2**](https://msdn.microsoft.com/library/windows/desktop/hh437304)<br/>           | Extends the [**IUIAutomationTransformPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationtransformpattern?branch=master) interface to enable UI Automation clients to programmatically access the viewport zooming functionality of a control.<br/>                                                                                                                              |
| [**IUIAutomationValuePattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationvaluepattern?branch=master)<br/>                         | Provides access to a control that contains a value that does not span a range and that can be represented as a string.<br/>                                                                                                                                                                                                                       |
| [**IUIAutomationVirtualizedItemPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationvirtualizeditempattern?branch=master)<br/>     | Represents an virtualized item, which is an item that is represented by a placeholder automation element in the UI Automation tree.<br/>                                                                                                                                                                                                          |
| [**IUIAutomationWindowPattern**](/windows/win32/UIAutomationClient/nn-uiautomationclient-iuiautomationwindowpattern?branch=master)<br/>                       | Provides access to the fundamental functionality of a window.<br/>                                                                                                                                                                                                                                                                                |
