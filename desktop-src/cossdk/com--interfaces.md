# COM+ Interfaces

The following are the COM+ interfaces.



| Interface                                                                | Description                                                                                                                                                                                                                                                                      |
|--------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ContextInfo**](/windows/win32/ComSvcs/nn-comsvcs-contextinfo?branch=master)                                       | Retrieves transaction, activity, and context information on the current context object.                                                                                                                                                                                          |
| [**ContextInfo2**](/windows/win32/ComSvcs/nn-comsvcs-contextinfo2?branch=master)                                     | Provides additional information about an object's context, supplementing the information that is available through the [**ContextInfo**](/windows/win32/ComSvcs/nn-comsvcs-contextinfo?branch=master) interface.                                                                                                             |
| [**IAppDomainHelper**](/windows/win32/ComSvcs/nn-comsvcs-iappdomainhelper?branch=master)                             | Binds a managed object to an application domain, which is an isolated environment where applications execute.                                                                                                                                                                    |
| [**IAssemblyLocator**](/windows/win32/ComSvcs/nn-comsvcs-iassemblylocator?branch=master)                             | Retrieves information about an assembly when using managed code in the .NET Framework common language runtime.                                                                                                                                                                   |
| [**IAsyncErrorNotify**](/windows/win32/ComSvcs/nn-comsvcs-iasyncerrornotify?branch=master)                           | Used to implement error trapping on the asynchronous batch work that is submitted through the activity created by [**CoCreateActivity**](/windows/win32/ComSvcs/nf-comsvcs-cocreateactivity?branch=master).                                                                                                                  |
| [**ICheckSxSConfig**](/windows/win32/ComSvcs/nn-comsvcs-ichecksxsconfig?branch=master)                               | Used to check the configuration of the current side-by-side assembly.                                                                                                                                                                                                            |
| [**IComActivityEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomactivityevents?branch=master)                         | Notifies the subscriber if an activity is created, destroyed, or timed out.                                                                                                                                                                                                      |
| [**IComAppEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomappevents?branch=master)                                   | Notifies the subscriber if a COM+ server application is started, shut down, or forced to shut down.                                                                                                                                                                              |
| [**IComApp2Events**](/windows/win32/ComSvcs/nn-comsvcs-icomapp2events?branch=master)                                 | Notifies the subscriber if a COM+ server application is loaded, shut down, or paused.                                                                                                                                                                                            |
| [**IComCRMEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomcrmevents?branch=master)                                   | Notifies the subscriber about activities of the Compensating Resource Manager (CRM) feature of Component Services.                                                                                                                                                               |
| [**IComExceptionEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomexceptionevents?branch=master)                       | Notifies the subscriber when an unhandled exception occurs in the user's code.                                                                                                                                                                                                   |
| [**IComIdentityEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomidentityevents?branch=master)                         | Notifies the subscriber about an activity that is part of an Internet Information Services (IIS) Active Server Pages (ASP) page.                                                                                                                                                 |
| [**IComInstanceEvents**](/windows/win32/ComSvcs/nn-comsvcs-icominstanceevents?branch=master)                         | Notifies the subscriber of an object's creation or release.                                                                                                                                                                                                                      |
| [**IComInstance2Events**](/windows/win32/ComSvcs/nn-comsvcs-icominstance2events?branch=master)                       | Notifies the subscriber if an object is created or released by a client.                                                                                                                                                                                                         |
| [**ICOMLBArguments**](/windows/win32/ComSvcs/nn-comsvcs-icomlbarguments?branch=master)                               | Used to activate the COM+ component load balancing service.                                                                                                                                                                                                                      |
| [**IComLTxEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomltxevents?branch=master)                                   | Notifies the subscriber of events that relate to COM+ transactions.                                                                                                                                                                                                              |
| [**IComMethodEvents**](/windows/win32/ComSvcs/nn-comsvcs-icommethodevents?branch=master)                             | Notifies the subscriber if an object's method has been called, returned, or generated an exception.                                                                                                                                                                              |
| [**IComMethod2Events**](/windows/win32/ComSvcs/nn-comsvcs-icommethod2events?branch=master)                           | Notifies the subscriber if an object's method has been called, returned, or generated an exception.                                                                                                                                                                              |
| [**IComObjectConstructionEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomobjectconstructionevents?branch=master)     | Notifies the subscriber if a constructed object is created in an object pool.                                                                                                                                                                                                    |
| [**IComObjectConstruction2Events**](/windows/win32/ComSvcs/nn-comsvcs-icomobjectconstruction2events?branch=master)   | Notifies the subscriber if a constructed object is created.                                                                                                                                                                                                                      |
| [**IComObjectEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomobjectevents?branch=master)                             | Notifies the subscriber if an instance of a just-in-time (JIT) activated object has been created or freed.                                                                                                                                                                       |
| [**IComObjectPoolEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomobjectpoolevents?branch=master)                     | Notifies the subscriber when a new object is added to the pool.                                                                                                                                                                                                                  |
| [**IComObjectPool2Events**](/windows/win32/ComSvcs/nn-comsvcs-icomobjectpool2events?branch=master)                   | Notifies the subscriber if a transactional or non-transactional object is added to or obtained from the object pool.                                                                                                                                                             |
| [**IComObjectPoolEvents2**](/windows/win32/ComSvcs/nn-comsvcs-icomobjectpoolevents2?branch=master)                   | Notifies the subscriber when a new object is created for or removed from the pool.                                                                                                                                                                                               |
| [**IComQCEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomqcevents?branch=master)                                     | Notifies the subscriber if a queued message is created, de-queued, or moved to a retry or dead letter queue.                                                                                                                                                                     |
| [**IComResourceEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomresourceevents?branch=master)                         | Notifies the subscriber if a resource is created, allocated, tracked, or destroyed.                                                                                                                                                                                              |
| [**IComSecurityEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomsecurityevents?branch=master)                         | Notifies the subscriber if the authentication of a method call succeeded or failed.                                                                                                                                                                                              |
| [**IComThreadEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomthreadevents?branch=master)                             | Notifies the subscriber if a single-threaded apartment (STA) is created or terminated, and when an apartment thread is allocated.                                                                                                                                                |
| [**IComTrackingInfoCollection**](/windows/win32/ComSvcs/nn-comsvcs-icomtrackinginfocollection?branch=master)         | Retrieves the type of a tracking information collection and the number of objects it contains.                                                                                                                                                                                   |
| [**IComTrackingInfoEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomtrackinginfoevents?branch=master)                 | Notifies the subscriber when the tracking information for a collection changes.                                                                                                                                                                                                  |
| [**IComTrackingInfoObject**](/windows/win32/ComSvcs/nn-comsvcs-icomtrackinginfoobject?branch=master)                 | Retrieves the properties of a tracking information object.                                                                                                                                                                                                                       |
| [**IComTrackingInfoProperties**](/windows/win32/ComSvcs/nn-comsvcs-icomtrackinginfoproperties?branch=master)         | Retrieves the total number of properties associated with a tracking information object and their names.                                                                                                                                                                          |
| [**IComTransactionEvents**](/windows/win32/ComSvcs/nn-comsvcs-icomtransactionevents?branch=master)                   | Notifies the subscriber if the Microsoft Distributed Transaction Coordinator (DTC) transaction starts, commits, or aborts.                                                                                                                                                       |
| [**IComTransaction2Events**](/windows/win32/ComSvcs/nn-comsvcs-icomtransaction2events?branch=master)                 | Notifies the subscriber if a Microsoft Distributed Transaction Coordinator (DTC) transaction starts, commits, or aborts. The subscriber is also notified when the transaction is in the prepare phase of the two-phase commit protocol.                                          |
| [**IComUserEvent**](/windows/win32/ComSvcs/nn-comsvcs-icomuserevent?branch=master)                                   | Notifies the subscriber of the specified user-defined metrics.                                                                                                                                                                                                                   |
| [**IContextProperties**](/windows/win32/ComSvcs/nn-comsvcs-icontextproperties?branch=master)                         | Provides access to context object properties.                                                                                                                                                                                                                                    |
| [**IContextState**](/windows/win32/ComSvcs/nn-comsvcs-icontextstate?branch=master)                                   | Controls object deactivation and transaction voting by manipulating context state flags.                                                                                                                                                                                         |
| [**IContextTransactionInfo**](icontexttransactioninfo.md)               | Provides access to context object properties that relate to transactions.                                                                                                                                                                                                        |
| [**ICreateWithLocalTransaction**](/windows/win32/ComSvcs/nn-comsvcs-icreatewithlocaltransaction?branch=master)       | Creates a COM+ object that executes within the scope of the specified local transaction.                                                                                                                                                                                         |
| [**ICreateWithTipTransactionEx**](/windows/win32/ComSvcs/nn-comsvcs-icreatewithtiptransactionex?branch=master)       | Creates an object that is enlisted within a manual transaction using the Transaction Internet Protocol (TIP).                                                                                                                                                                    |
| [**ICreateWithTransactionEx**](/windows/win32/ComSvcs/nn-comsvcs-icreatewithtransactionex?branch=master)             | Creates an object that is enlisted within a manual transaction.                                                                                                                                                                                                                  |
| [**ICrmCompensator**](/windows/win32/ComSvcs/nn-comsvcs-icrmcompensator?branch=master)                               | Delivers unstructured log records to the CRM Compensator when using Microsoft Visual C++.                                                                                                                                                                                        |
| [**ICrmCompensatorVariants**](/windows/win32/ComSvcs/nn-comsvcs-icrmcompensatorvariants?branch=master)               | Delivers structured log records to the CRM Compensator when using Microsoft Visual Basic.                                                                                                                                                                                        |
| [**ICrmFormatLogRecords**](/windows/win32/ComSvcs/nn-comsvcs-icrmformatlogrecords?branch=master)                     | Converts the log records to viewable format so that they can be presented using a generic monitoring tool.                                                                                                                                                                       |
| [**ICrmLogControl**](/windows/win32/ComSvcs/nn-comsvcs-icrmlogcontrol?branch=master)                                 | Is the means by which the CRM Worker and CRM Compensator write records to the log and make them durable.                                                                                                                                                                         |
| [**ICrmMonitor**](/windows/win32/ComSvcs/nn-comsvcs-icrmmonitor?branch=master)                                       | Captures a snapshot of the current state of the CRM and holds a specific CRM clerk.                                                                                                                                                                                              |
| [**ICrmMonitorClerks**](/windows/win32/ComSvcs/nn-comsvcs-icrmmonitorclerks?branch=master)                           | Retrieves information about the state of clerks.                                                                                                                                                                                                                                 |
| [**ICrmMonitorLogRecords**](/windows/win32/ComSvcs/nn-comsvcs-icrmmonitorlogrecords?branch=master)                   | Monitors the individual log records maintained by a specific CRM clerk for a given transaction.                                                                                                                                                                                  |
| [**IDispenserDriver**](/windows/win32/ComSvcs/nn-comsvcs-idispenserdriver?branch=master)                             | Is called by the holder of the COM+ Resource Dispenser to create, enlist, evaluate, and destroy a resource.                                                                                                                                                                      |
| [**IDispenserManager**](/windows/win32/ComSvcs/nn-comsvcs-idispensermanager?branch=master)                           | Connects to the Dispenser Manager.                                                                                                                                                                                                                                               |

| [**IEventControl**](/windows/win32/Eventsys/nn-eventsys-ieventcontrol?branch=master)                                   | Controls the behavior of an event object, the object that fires an event to its subscribers.                                                                                                                                                                                     |
| [**IEventObjectChange**](/windows/win32/Eventsys/nn-eventsys-ieventobjectchange?branch=master)                         | Notifies subscribers of changes to the event store.                                                                                                                                                                                                                              |

| [**IEventObjectCollection**](/windows/win32/Eventsys/nn-eventsys-ieventobjectcollection?branch=master)                 | Manages objects in an event objects collection.                                                                                                                                                                                                                                  |
| [**IEventProperty**](/windows/win32/Eventsys/?branch=master)                                 | Stores event properties.                                                                                                                                                                                                                                                         |
| [**IEventSubscription**](/windows/win32/EventSys/nn-eventsys-ieventsubscription?branch=master)                         | Specifies information about the relationship between an event subscriber and an event to which it is subscribing.                                                                                                                                                                |
