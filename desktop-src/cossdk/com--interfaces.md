# COM+ Interfaces

The following are the COM+ interfaces.



| Interface                                                                | Description                                                                                                                                                                                                                                                                      |
|--------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ContextInfo**](contextinfo.md)                                       | Retrieves transaction, activity, and context information on the current context object.                                                                                                                                                                                          |
| [**ContextInfo2**](contextinfo2.md)                                     | Provides additional information about an object's context, supplementing the information that is available through the [**ContextInfo**](contextinfo.md) interface.                                                                                                             |
| [**IAppDomainHelper**](iappdomainhelper.md)                             | Binds a managed object to an application domain, which is an isolated environment where applications execute.                                                                                                                                                                    |
| [**IAssemblyLocator**](iassemblylocator.md)                             | Retrieves information about an assembly when using managed code in the .NET Framework common language runtime.                                                                                                                                                                   |
| [**IAsyncErrorNotify**](iasyncerrornotify.md)                           | Used to implement error trapping on the asynchronous batch work that is submitted through the activity created by [**CoCreateActivity**](cocreateactivity.md).                                                                                                                  |
| [**ICheckSxSConfig**](ichecksxsconfig.md)                               | Used to check the configuration of the current side-by-side assembly.                                                                                                                                                                                                            |
| [**IComActivityEvents**](icomactivityevents.md)                         | Notifies the subscriber if an activity is created, destroyed, or timed out.                                                                                                                                                                                                      |
| [**IComAppEvents**](icomappevents.md)                                   | Notifies the subscriber if a COM+ server application is started, shut down, or forced to shut down.                                                                                                                                                                              |
| [**IComApp2Events**](icomapp2events.md)                                 | Notifies the subscriber if a COM+ server application is loaded, shut down, or paused.                                                                                                                                                                                            |
| [**IComCRMEvents**](icomcrmevents.md)                                   | Notifies the subscriber about activities of the Compensating Resource Manager (CRM) feature of Component Services.                                                                                                                                                               |
| [**IComExceptionEvents**](icomexceptionevents.md)                       | Notifies the subscriber when an unhandled exception occurs in the user's code.                                                                                                                                                                                                   |
| [**IComIdentityEvents**](icomidentityevents.md)                         | Notifies the subscriber about an activity that is part of an Internet Information Services (IIS) Active Server Pages (ASP) page.                                                                                                                                                 |
| [**IComInstanceEvents**](icominstanceevents.md)                         | Notifies the subscriber of an object's creation or release.                                                                                                                                                                                                                      |
| [**IComInstance2Events**](icominstance2events.md)                       | Notifies the subscriber if an object is created or released by a client.                                                                                                                                                                                                         |
| [**ICOMLBArguments**](icomlbarguments.md)                               | Used to activate the COM+ component load balancing service.                                                                                                                                                                                                                      |
| [**IComLTxEvents**](icomltxevents.md)                                   | Notifies the subscriber of events that relate to COM+ transactions.                                                                                                                                                                                                              |
| [**IComMethodEvents**](icommethodevents.md)                             | Notifies the subscriber if an object's method has been called, returned, or generated an exception.                                                                                                                                                                              |
| [**IComMethod2Events**](icommethod2events.md)                           | Notifies the subscriber if an object's method has been called, returned, or generated an exception.                                                                                                                                                                              |
| [**IComObjectConstructionEvents**](icomobjectconstructionevents.md)     | Notifies the subscriber if a constructed object is created in an object pool.                                                                                                                                                                                                    |
| [**IComObjectConstruction2Events**](icomobjectconstruction2events.md)   | Notifies the subscriber if a constructed object is created.                                                                                                                                                                                                                      |
| [**IComObjectEvents**](icomobjectevents.md)                             | Notifies the subscriber if an instance of a just-in-time (JIT) activated object has been created or freed.                                                                                                                                                                       |
| [**IComObjectPoolEvents**](icomobjectpoolevents.md)                     | Notifies the subscriber when a new object is added to the pool.                                                                                                                                                                                                                  |
| [**IComObjectPool2Events**](icomobjectpool2events.md)                   | Notifies the subscriber if a transactional or non-transactional object is added to or obtained from the object pool.                                                                                                                                                             |
| [**IComObjectPoolEvents2**](icomobjectpoolevents2.md)                   | Notifies the subscriber when a new object is created for or removed from the pool.                                                                                                                                                                                               |
| [**IComQCEvents**](icomqcevents.md)                                     | Notifies the subscriber if a queued message is created, de-queued, or moved to a retry or dead letter queue.                                                                                                                                                                     |
| [**IComResourceEvents**](icomresourceevents.md)                         | Notifies the subscriber if a resource is created, allocated, tracked, or destroyed.                                                                                                                                                                                              |
| [**IComSecurityEvents**](icomsecurityevents.md)                         | Notifies the subscriber if the authentication of a method call succeeded or failed.                                                                                                                                                                                              |
| [**IComThreadEvents**](icomthreadevents.md)                             | Notifies the subscriber if a single-threaded apartment (STA) is created or terminated, and when an apartment thread is allocated.                                                                                                                                                |
| [**IComTrackingInfoCollection**](icomtrackinginfocollection.md)         | Retrieves the type of a tracking information collection and the number of objects it contains.                                                                                                                                                                                   |
| [**IComTrackingInfoEvents**](icomtrackinginfoevents.md)                 | Notifies the subscriber when the tracking information for a collection changes.                                                                                                                                                                                                  |
| [**IComTrackingInfoObject**](icomtrackinginfoobject.md)                 | Retrieves the properties of a tracking information object.                                                                                                                                                                                                                       |
| [**IComTrackingInfoProperties**](icomtrackinginfoproperties.md)         | Retrieves the total number of properties associated with a tracking information object and their names.                                                                                                                                                                          |
| [**IComTransactionEvents**](icomtransactionevents.md)                   | Notifies the subscriber if the Microsoft Distributed Transaction Coordinator (DTC) transaction starts, commits, or aborts.                                                                                                                                                       |
| [**IComTransaction2Events**](icomtransaction2events.md)                 | Notifies the subscriber if a Microsoft Distributed Transaction Coordinator (DTC) transaction starts, commits, or aborts. The subscriber is also notified when the transaction is in the prepare phase of the two-phase commit protocol.                                          |
| [**IComUserEvent**](icomuserevent.md)                                   | Notifies the subscriber of the specified user-defined metrics.                                                                                                                                                                                                                   |
| [**IContextProperties**](icontextproperties.md)                         | Provides access to context object properties.                                                                                                                                                                                                                                    |
| [**IContextState**](icontextstate.md)                                   | Controls object deactivation and transaction voting by manipulating context state flags.                                                                                                                                                                                         |
| [**IContextTransactionInfo**](icontexttransactioninfo.md)               | Provides access to context object properties that relate to transactions.                                                                                                                                                                                                        |
| [**ICreateWithLocalTransaction**](icreatewithlocaltransaction.md)       | Creates a COM+ object that executes within the scope of the specified local transaction.                                                                                                                                                                                         |
| [**ICreateWithTipTransactionEx**](icreatewithtiptransactionex.md)       | Creates an object that is enlisted within a manual transaction using the Transaction Internet Protocol (TIP).                                                                                                                                                                    |
| [**ICreateWithTransactionEx**](icreatewithtransactionex.md)             | Creates an object that is enlisted within a manual transaction.                                                                                                                                                                                                                  |
| [**ICrmCompensator**](icrmcompensator.md)                               | Delivers unstructured log records to the CRM Compensator when using Microsoft Visual C++.                                                                                                                                                                                        |
| [**ICrmCompensatorVariants**](icrmcompensatorvariants.md)               | Delivers structured log records to the CRM Compensator when using Microsoft Visual Basic.                                                                                                                                                                                        |
| [**ICrmFormatLogRecords**](icrmformatlogrecords.md)                     | Converts the log records to viewable format so that they can be presented using a generic monitoring tool.                                                                                                                                                                       |
| [**ICrmLogControl**](icrmlogcontrol.md)                                 | Is the means by which the CRM Worker and CRM Compensator write records to the log and make them durable.                                                                                                                                                                         |
| [**ICrmMonitor**](icrmmonitor.md)                                       | Captures a snapshot of the current state of the CRM and holds a specific CRM clerk.                                                                                                                                                                                              |
| [**ICrmMonitorClerks**](icrmmonitorclerks.md)                           | Retrieves information about the state of clerks.                                                                                                                                                                                                                                 |
| [**ICrmMonitorLogRecords**](icrmmonitorlogrecords.md)                   | Monitors the individual log records maintained by a specific CRM clerk for a given transaction.                                                                                                                                                                                  |
| [**IDispenserDriver**](idispenserdriver.md)                             | Is called by the holder of the COM+ Resource Dispenser to create, enlist, evaluate, and destroy a resource.                                                                                                                                                                      |
| [**IDispenserManager**](idispensermanager.md)                           | Connects to the Dispenser Manager.                                                                                                                                                                                                                                               |
| [**IEnumEventObject**](ienumeventobject.md)                             | Enumerates the event objects that are registered in the COM+ events store.                                                                                                                                                                                                       |
| [**IEnumNames**](ienumnames.md)                                         | Enumerates names.                                                                                                                                                                                                                                                                |
| [**IEventClass**](ieventclass.md)                                       | Associates a class of event objects with the event interface those objects implement.                                                                                                                                                                                            |
| [**IEventClass2**](ieventclass2.md)                                     | Used to set and obtain data on event class objects.                                                                                                                                                                                                                              |
| [**IEventControl**](ieventcontrol.md)                                   | Controls the behavior of an event object, the object that fires an event to its subscribers.                                                                                                                                                                                     |
| [**IEventObjectChange**](ieventobjectchange.md)                         | Notifies subscribers of changes to the event store.                                                                                                                                                                                                                              |
| [**IEventObjectChange2**](ieventobjectchange2.md)                       | Notifies subscribers of changes to the event store while including partition and application ID information.                                                                                                                                                                     |
| [**IEventObjectCollection**](ieventobjectcollection.md)                 | Manages objects in an event objects collection.                                                                                                                                                                                                                                  |
| [**IEventProperty**](ieventproperty.md)                                 | Stores event properties.                                                                                                                                                                                                                                                         |
| [**IEventSubscription**](ieventsubscription.md)                         | Specifies information about the relationship between an event subscriber and an event to which it is subscribing.                                                                                                                                                                |
| [**IEventSubscription2**](ieventsubscription2.md)                       | Extends the [**IEventSubscription**](ieventsubscription.md) interface.                                                                                                                                                                                                          |
| [**IEventSubscription3**](ieventsubscription3.md)                       | Extends the [**IEventSubscription2**](ieventsubscription2.md) interface.                                                                                                                                                                                                        |
| [**IEventSystem**](ieventsystem.md)                                     | Provides access to the event data store.                                                                                                                                                                                                                                         |
| [**IEventSystem2**](ieventsystem2.md)                                   | Extends the [**IEventSystem**](ieventsystem.md) interface.                                                                                                                                                                                                                      |
| [**IFiringControl**](ifiringcontrol.md)                                 | Fires an event to a single subscription.                                                                                                                                                                                                                                         |
| [**IGetAppTrackerData**](igetapptrackerdata.md)                         | Enables administrative applications to retrieve statistical information about running COM+ applications.                                                                                                                                                                         |
| [**IGetContextProperties**](igetcontextproperties.md)                   | Enables the caller to obtain the properties associated with the current object's context.                                                                                                                                                                                        |
| [**IGetSecurityCallContext**](igetsecuritycallcontext.md)               | Retrieves a reference to an object created from the [**SecurityCallContext**](securitycallcontext.md) class that is associated with the current call.                                                                                                                           |
| [**IHolder**](iholder.md)                                               | Allocates or frees resources for an installed Resource Dispenser.                                                                                                                                                                                                                |
| [**IManagedActivationEvents**](imanagedactivationevents.md)             | Used to create and destroy stubs for managed objects within the current COM+ context.                                                                                                                                                                                            |
| [**IManagedObjectInfo**](imanagedobjectinfo.md)                         | Describes the stub for a managed object.                                                                                                                                                                                                                                         |
| [**IManagedPoolAction**](imanagedpoolaction.md)                         | Enables an object to be notified before it is released from a COM+ object pool.                                                                                                                                                                                                  |
| [**IManagedPooledObj**](imanagedpooledobj.md)                           | Describes how a managed object is used in the COM+ object pool.                                                                                                                                                                                                                  |
| [**IMessageMover**](imessagemover.md)                                   | Moves messages from one queue to another queue.                                                                                                                                                                                                                                  |
| [**IMTSActivity**](imtsactivity.md)                                     | Submits batch work through the activity created by the [**MTSCreateActivity**](mtscreateactivity.md) function.                                                                                                                                                                  |
| [**IMTSCall**](imtscall.md)                                             | Implements the batch work that is submitted through the activity created by the [**MTSCreateActivity**](mtscreateactivity.md) function.                                                                                                                                         |
| [**IMtsEventInfo**](imtseventinfo.md)                                   | Describes user-defined events.                                                                                                                                                                                                                                                   |
| [**IMtsEvents**](imtsevents.md)                                         | Provides methods for obtaining information about the running package and establishing event sinks.                                                                                                                                                                               |
| [**IMtsGrp**](imtsgrp.md)                                               | Provides methods for enumerating through running packages.                                                                                                                                                                                                                       |
| [**IMTSLocator**](imtslocator.md)                                       | Describes a single event that provides access to the [**IMtsEvents**](imtsevents.md) interface of the event dispatcher for the current process.                                                                                                                                 |
| [**IMTxAS**](https://msdn.microsoft.com/library/windows/desktop/ms681597)                                                 | Equivalent to the following C++ functions: [**GetObjectContext**](getobjectcontext.md), [**RecycleSurrogate**](recyclesurrogate.md), and [**SafeRef**](saferef.md).                                                                                                           |
| [**IMultiInterfaceEventControl**](imultiinterfaceeventcontrol.md)       | Controls the behavior of an event object, the object that fires an event to its subscribers.                                                                                                                                                                                     |
| [**IMultiInterfacePublisherFilter**](imultiinterfacepublisherfilter.md) | Manages a filtered subscription cache for an event method.                                                                                                                                                                                                                       |
| [**IObjectConstruct**](iobjectconstruct.md)                             | Controls the object construction process by passing in parameters from other methods or objects.                                                                                                                                                                                 |
| [**IObjectConstructString**](iobjectconstructstring.md)                 | Provides access to a constructor string. Use it when you want to specify the parameters during the construction of your object.                                                                                                                                                  |
| [**IObjectContext**](iobjectcontext.md)                                 | Provides access to the current object's context. An object's context is primarily used when working with transactions or dealing with the security of an object.                                                                                                                 |
| [**IObjectContextActivity**](iobjectcontextactivity.md)                 | Retrieves the activity identifier associated with the current object context.                                                                                                                                                                                                    |
| [**IObjectContextInfo**](iobjectcontextinfo.md)                         | Retrieves transaction, activity, and context information on the current context object.                                                                                                                                                                                          |
| [**IObjectContextInfo2**](iobjectcontextinfo2.md)                       | Extends the [**IObjectContextInfo**](iobjectcontextinfo.md) interface.                                                                                                                                                                                                          |
| [**IObjectContextTip**](iobjectcontexttip.md)                           | Retrieves properties describing the Transaction Internet Protocol (TIP) transaction context.                                                                                                                                                                                     |
| [**IObjectControl**](iobjectcontrol.md)                                 | Defines context-specific initialization and cleanup procedures for your COM+ objects, and specifies whether the objects can be recycled.                                                                                                                                         |
| [**IObjPool**](iobjpool.md)                                             | Represents the key to each object in the transaction resource pool.                                                                                                                                                                                                              |
| [**IPlaybackControl**](iplaybackcontrol.md)                             | Enables participation in the abnormal handling of server-side playback errors and client-side failures of the Message Queuing delivery mechanism.                                                                                                                                |
| [**IPoolManager**](ipoolmanager.md)                                     | Enables the caller to control an object pool.                                                                                                                                                                                                                                    |
| [**IProcessInitializer**](iprocessinitializer.md)                       | Provides methods that can be called whenever Dllhost.exe starts up or shuts down.                                                                                                                                                                                                |
| [**IPublisherFilter**](ipublisherfilter.md)                             | Acts as a callback interface so that event publishers can control which subscribers receive event notifications or the order in which subscribers are notified.                                                                                                                  |
| [**ISecurityCallContext**](isecuritycallcontext.md)                     | Provides access to security methods and information about the security call context of the current call.                                                                                                                                                                         |
| [**ISecurityCallersColl**](isecuritycallerscoll.md)                     | Provides access to information about individual callers in a collection of callers.                                                                                                                                                                                              |
| [**ISecurityIdentityColl**](isecurityidentitycoll.md)                   | Provides access to a collection of security information representing a caller's identity. The items available in this collection are the SID, the account name, the authentication service, the authentication level, and the impersonation level.                               |
| [**ISecurityProperty**](isecurityproperty.md)                           | Determines the security ID of the current object's original caller or direct caller. However, the preferred way to get information about an object's callers is to use the [**ISecurityCallContext**](isecuritycallcontext.md) interface.                                       |
| [**ISelectCOMLBServer**](iselectcomlbserver.md)                         | Activates the COM+ component load balancing service.                                                                                                                                                                                                                             |
| [**ISendMethodEvents**](isendmethodevents.md)                           | Describes an event class that notifies subscribers whenever a method on the object that implements it either is called or returns from a call.                                                                                                                                   |
| [**IServiceActivity**](iserviceactivity.md)                             | Used to call the batch work that is submitted through the activity created by [**CoCreateActivity**](cocreateactivity.md).                                                                                                                                                      |
| [**IServiceCall**](iservicecall.md)                                     | Used to implement the batch work that is submitted through the activity created by [**CoCreateActivity**](cocreateactivity.md).                                                                                                                                                 |
| [**IServiceComTIIntrinsicsConfig**](iservicecomtiintrinsicsconfig.md)   | Configures the COM Transaction Integrator (COMTI) intrinsics for the work that is done when calling the [**CoCreateActivity**](cocreateactivity.md) or [**CoEnterServiceDomain**](coenterservicedomain.md) function.                                                           |
| [**IServiceIISIntrinsicsConfig**](iserviceiisintrinsicsconfig.md)       | Configures the IIS intrinsics for the work that is done when calling the [**CoCreateActivity**](cocreateactivity.md) or [**CoEnterServiceDomain**](coenterservicedomain.md) function.                                                                                          |
| [**IServiceInheritanceConfig**](iserviceinheritanceconfig.md)           | Determines whether to construct a new context based on the current context or to create a new context based solely on the information in **CServiceConfig**.                                                                                                                     |
| [**IServicePartitionConfig**](iservicepartitionconfig.md)               | Configures how partitions are used for the work that is done when calling either [**CoCreateActivity**](cocreateactivity.md) or [**CoEnterServiceDomain**](coenterservicedomain.md).                                                                                           |
| [**IServicePool**](iservicepool.md)                                     | Used to manage a COM+ object pool.                                                                                                                                                                                                                                               |
| [**IServicePoolConfig**](iservicepoolconfig.md)                         | Used to configure an object pool.                                                                                                                                                                                                                                                |
| [**IServiceSxSConfig**](iservicesxsconfig.md)                           | Configures side-by-side assemblies for the work that is done when calling either [**CoCreateActivity**](cocreateactivity.md) or [**CoEnterServiceDomain**](coenterservicedomain.md).                                                                                           |
| [**IServiceSynchronizationConfig**](iservicesynchronizationconfig.md)   | Configures the synchronization for the work that is done when calling either [**CoCreateActivity**](cocreateactivity.md) or [**CoEnterServiceDomain**](coenterservicedomain.md).                                                                                               |
| [**IServiceSysTxnConfig**](iservicesystxnconfig.md)                     | Enables you to run a set of code in the scope of an existing transaction that you specify with a transaction proxy.                                                                                                                                                              |
| [**IServiceThreadPoolConfig**](iservicethreadpoolconfig.md)             | Configures the thread pool of the activity object that is returned by calling [**CoCreateActivity**](cocreateactivity.md).                                                                                                                                                      |
| [**IServiceTrackerConfig**](iservicetrackerconfig.md)                   | Configures the tracker property for the work that is done when calling either [**CoCreateActivity**](cocreateactivity.md) or [**CoEnterServiceDomain**](coenterservicedomain.md).                                                                                              |
| [**IServiceTransactionConfig**](iservicetransactionconfig.md)           | Extends the [**IServiceTransactionConfigBase**](iservicetransactionconfigbase.md) interface.                                                                                                                                                                                    |
| [**IServiceTransactionConfigBase**](iservicetransactionconfigbase.md)   | Configures the transaction services for the work that is done when calling either [**CoCreateActivity**](cocreateactivity.md) or [**CoEnterServiceDomain**](coenterservicedomain.md).                                                                                          |
| [**ISharedProperty**](isharedproperty.md)                               | Exposes property methods that you can use to set or retrieve the value of a shared property.                                                                                                                                                                                     |
| [**ISharedPropertyGroup**](isharedpropertygroup.md)                     | Used to create and access the shared properties in a shared property group.                                                                                                                                                                                                      |
| [**ISharedPropertyGroupManager**](isharedpropertygroupmanager.md)       | Used to create shared property groups and to obtain access to existing shared property groups.                                                                                                                                                                                   |
| [**ISystemAppEventData**](isystemappeventdata.md)                       | Notifies the subscriber when a COM+ application instance is created or reconfigured.                                                                                                                                                                                             |
| [**IThreadPoolKnobs**](ithreadpoolknobs.md)                             | Used to control the behavior of thread pools.                                                                                                                                                                                                                                    |
| [**ITransactionContext**](itransactioncontext.md)                       | Enables you to compose the work of multiple COM+ objects in a single transaction and explicitly commit or abort the transaction.                                                                                                                                                 |
| [**ITransactionContextEx**](itransactioncontextex.md)                   | Extends the [**ITransactionContext**](itransactioncontext.md) interface to be compatible with Automation.                                                                                                                                                                       |
| [**ITransactionProperty**](itransactionproperty.md)                     | Used to get the transaction resource pool.                                                                                                                                                                                                                                       |
| [**ITransactionProxy**](itransactionproxy.md)                           | Provides a way for a COM+ transaction context to work with a non-DTC transaction.                                                                                                                                                                                                |
| [**ITransactionResourcePool**](itransactionresourcepool.md)             | Maintains a list of pooled objects, keyed by [**IObjPool**](iobjpool.md), that are used until the transaction completes.                                                                                                                                                        |
| [**ITransactionStatus**](itransactionstatus.md)                         | Used to discover the status of the transaction that is completed by the call to [**CoLeaveServiceDomain**](coleaveservicedomain.md) when **CServiceConfig** is configured to use transactions in the call to [**CoEnterServiceDomain**](coenterservicedomain.md).              |
| [**ObjectContext**](objectcontext.md)                                   | Provides access to the current object's context. An object's context is primarily used when working with transactions or dealing with the security of an object. Provides the same functionality as [**IObjectContext**](iobjectcontext.md), but is compatible with Automation. |
| [**ObjectControl**](iobjectcontrol.md)                                  | Defines context-specific initialization and cleanup procedures for your COM+ objects, and to specify whether the objects can be recycled. Provides the same functionality as [**IObjectControl**](iobjectcontrol.md), but is compatible with Automation.                        |
| [**SecurityProperty**](isecurityproperty.md)                            | Enables you to obtain information about the current object's original caller and direct caller.                                                                                                                                                                                  |
