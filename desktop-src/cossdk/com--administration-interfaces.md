# COM+ Administration Interfaces

The following are the COM+ administration interfaces.



| Interface                                        | Description                                                                                                                                                                                      |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ICatalogCollection**](icatalogcollection.md) | Enables you to enumerate, add, remove, and retrieve items in any collection in the COM+ catalog and to access related collections.                                                               |
| [**ICatalogObject**](icatalogobject.md)         | Enables you to get and put properties exposed by objects in the COM+ catalog.                                                                                                                    |
| [**ICOMAdminCatalog**](icomadmincatalog.md)     | Initiates a session to do programmatic COM+ administration, access collections in the catalog, install COM+ applications and components, start and stop services, and connect to remote servers. |
| [**ICOMAdminCatalog2**](icomadmincatalog2.md)   | Controls the interactions of applications, components, and partitions.                                                                                                                           |
