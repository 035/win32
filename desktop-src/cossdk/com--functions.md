# COM+ Functions

The following are the COM+ functions.



| Function                                                                 | Description                                                                                                                             |
|--------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**CoCreateActivity**](/windows/win32/ComSvcs/nf-comsvcs-cocreateactivity?branch=master)                             | Creates an activity to do synchronous or asynchronous batch work that can use COM+ services without needing to create a COM+ component. |
| [**CoEnterServiceDomain**](/windows/win32/ComSvcs/nf-comsvcs-coenterservicedomain?branch=master)                     | Used to enter code that can then use COM+ services.                                                                                     |
| [**CoGetDefaultContext**](/windows/win32/combaseapi/nf-combaseapi-cogetdefaultcontext?branch=master)                       | Retrieves a reference to the default context of the specified apartment.                                                                |
| [**CoLeaveServiceDomain**](/windows/win32/ComSvcs/nf-comsvcs-coleaveservicedomain?branch=master)                     | Used to leave code that uses COM+ services.                                                                                             |
| [**ComPlusCompleteCbbSetup**](compluscompletecbbsetup.md)               | Completes a catalog migration on the destination computer.                                                                              |
| [**GetComPlusPackageInstallStatus**](getcompluspackageinstallstatus.md) | Indicates whether the 64-bit Common Language Runtime (CLR) is installed.                                                                |
| [**GetDispenserManager**](/windows/win32/MtxDM/nf-mtxdm-getdispensermanager?branch=master)                       | Retrieves the dispenser manager's [**IDispenserManager**](/windows/win32/ComSvcs/nn-comsvcs-idispensermanager?branch=master) interface.                                             |
| [**GetManagedExtensions**](/windows/win32/ComSvcs/nf-comsvcs-getmanagedextensions?branch=master)                     | Determines whether the installed version of COM+ supports special features provided to manage serviced components (managed objects).    |
| [**GetObjectContext**](/windows/win32/ComSvcs/nf-comsvcs-getobjectcontext?branch=master)                             | Retrieves a reference to the context that is associated with the current COM+ object.                                                   |
| [**MTSCreateActivity**](/windows/win32/ComSvcs/nf-comsvcs-mtscreateactivity?branch=master)                           | Creates an activity in a single-threaded apartment to do synchronous or asynchronous batch work.                                        |
| [**RecycleSurrogate**](/windows/win32/ComSvcs/nf-comsvcs-recyclesurrogate?branch=master)                             | Recycles the calling process.                                                                                                           |
| [**SafeRef**](/windows/win32/ComSvcs/nf-comsvcs-saferef?branch=master)                                               | Obsolete; do not use.                                                                                                                   |
