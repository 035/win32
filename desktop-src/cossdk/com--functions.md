# COM+ Functions

The following are the COM+ functions.



| Function                                                                 | Description                                                                                                                             |
|--------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**CoCreateActivity**](cocreateactivity.md)                             | Creates an activity to do synchronous or asynchronous batch work that can use COM+ services without needing to create a COM+ component. |
| [**CoEnterServiceDomain**](coenterservicedomain.md)                     | Used to enter code that can then use COM+ services.                                                                                     |
| [**CoGetDefaultContext**](cogetdefaultcontext.md)                       | Retrieves a reference to the default context of the specified apartment.                                                                |
| [**CoLeaveServiceDomain**](coleaveservicedomain.md)                     | Used to leave code that uses COM+ services.                                                                                             |
| [**ComPlusCompleteCbbSetup**](compluscompletecbbsetup.md)               | Completes a catalog migration on the destination computer.                                                                              |
| [**GetComPlusPackageInstallStatus**](getcompluspackageinstallstatus.md) | Indicates whether the 64-bit Common Language Runtime (CLR) is installed.                                                                |
| [**GetDispenserManager**](getdispensermanager.md)                       | Retrieves the dispenser manager's [**IDispenserManager**](idispensermanager.md) interface.                                             |
| [**GetManagedExtensions**](getmanagedextensions.md)                     | Determines whether the installed version of COM+ supports special features provided to manage serviced components (managed objects).    |
| [**GetObjectContext**](getobjectcontext.md)                             | Retrieves a reference to the context that is associated with the current COM+ object.                                                   |
| [**MTSCreateActivity**](mtscreateactivity.md)                           | Creates an activity in a single-threaded apartment to do synchronous or asynchronous batch work.                                        |
| [**RecycleSurrogate**](recyclesurrogate.md)                             | Recycles the calling process.                                                                                                           |
| [**SafeRef**](saferef.md)                                               | Obsolete; do not use.                                                                                                                   |
