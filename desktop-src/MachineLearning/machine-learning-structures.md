# Windows Machine Learning Structures

The following topics describe the COM structures for the Windows Machine Learning SDK.

## In this section



| Topic                                                                                          | Description                                                          |
|------------------------------------------------------------------------------------------------|----------------------------------------------------------------------|
| [**WINML\_BINDING\_DESC**](https://www.bing.com/search?q=**WINML\_BINDING\_DESC**)<br/>                      | Contains a description of the WinML binding.<br/>              |
| [**WINML\_IMAGE\_BINDING\_DESC**](https://www.bing.com/search?q=**WINML\_IMAGE\_BINDING\_DESC**)<br/>         | Contains properties for the binding of the type image.<br/>    |
| [**WINML\_IMAGE\_VARIABLE\_DESC**](https://www.bing.com/search?q=**WINML\_IMAGE\_VARIABLE\_DESC**)<br/>       | Contains properties for the image variable description.<br/>   |
| [**WINML\_MAP\_BINDING\_DESC**](https://www.bing.com/search?q=**WINML\_MAP\_BINDING\_DESC**)<br/>             | Contains properties for the binding of type map.<br/>          |
| [**WINML\_MAP\_VARIABLE\_DESC**](https://www.bing.com/search?q=**WINML\_MAP\_VARIABLE\_DESC**)<br/>           | Contains description properties of the map variable.<br/>      |
| [**WINML\_MODEL\_DESC**](https://www.bing.com/search?q=**WINML\_MODEL\_DESC**)<br/>                          | Contains description properties of the model.<br/>             |
| [**WINML\_RESOURCE\_BINDING\_DESC**](https://www.bing.com/search?q=**WINML\_RESOURCE\_BINDING\_DESC**)<br/>   | Contains description properties of the resource binding.<br/>  |
| [**WINML\_SEQUENCE\_BINDING\_DESC**](https://www.bing.com/search?q=**WINML\_SEQUENCE\_BINDING\_DESC**)<br/>   | Contains description properties of the sequence binding.<br/>  |
| [**WINML\_SEQUENCE\_VARIABLE\_DESC**](https://www.bing.com/search?q=**WINML\_SEQUENCE\_VARIABLE\_DESC**)<br/> | Contains description properties of the sequence variable.<br/> |
| [**WINML\_TENSOR\_BINDING\_DESC**](https://www.bing.com/search?q=**WINML\_TENSOR\_BINDING\_DESC**)<br/>       | Contains description properties of the tensor binding.<br/>    |
| [**WINML\_TENSOR\_VARIABLE\_DESC**](https://www.bing.com/search?q=**WINML\_TENSOR\_VARIABLE\_DESC**)<br/>     | Contains description properties of the tensor variable.<br/>   |
| [**WINML\_VARIABLE\_DESC**](https://www.bing.com/search?q=**WINML\_VARIABLE\_DESC**)<br/>                    | Contains description properties of the variable.<br/>          |
