# Windows Machine Learning Structures

The following topics describe the COM structures for the Windows Machine Learning SDK.

## In this section



| Topic                                                                                          | Description                                                          |
|------------------------------------------------------------------------------------------------|----------------------------------------------------------------------|
| [**WINML\_BINDING\_DESC**](machinelearning.winml_binding_desc)<br/>                      | Contains a description of the WinML binding.<br/>              |
| [**WINML\_IMAGE\_BINDING\_DESC**](machinelearning.winml_image_binding_desc)<br/>         | Contains properties for the binding of the type image.<br/>    |
| [**WINML\_IMAGE\_VARIABLE\_DESC**](machinelearning.winml_image_variable_desc)<br/>       | Contains properties for the image variable description.<br/>   |
| [**WINML\_MAP\_BINDING\_DESC**](machinelearning.winml_map_binding_desc)<br/>             | Contains properties for the binding of type map.<br/>          |
| [**WINML\_MAP\_VARIABLE\_DESC**](machinelearning.winml_map_variable_desc)<br/>           | Contains description properties of the map variable.<br/>      |
| [**WINML\_MODEL\_DESC**](machinelearning.winml_model_desc)<br/>                          | Contains description properties of the model.<br/>             |
| [**WINML\_RESOURCE\_BINDING\_DESC**](machinelearning.winml_resource_binding_desc)<br/>   | Contains description properties of the resource binding.<br/>  |
| [**WINML\_SEQUENCE\_BINDING\_DESC**](machinelearning.winml_sequence_binding_desc)<br/>   | Contains description properties of the sequence binding.<br/>  |
| [**WINML\_SEQUENCE\_VARIABLE\_DESC**](machinelearning.winml_sequence_variable_desc)<br/> | Contains description properties of the sequence variable.<br/> |
| [**WINML\_TENSOR\_BINDING\_DESC**](machinelearning.winml_tensor_binding_desc)<br/>       | Contains description properties of the tensor binding.<br/>    |
| [**WINML\_TENSOR\_VARIABLE\_DESC**](machinelearning.winml_tensor_variable_desc)<br/>     | Contains description properties of the tensor variable.<br/>   |
| [**WINML\_VARIABLE\_DESC**](machinelearning.winml_variable_desc)<br/>                    | Contains description properties of the variable.<br/>          |
