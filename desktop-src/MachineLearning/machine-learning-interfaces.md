# Windows Machine Learning Interfaces

The following topics describe the COM interfaces for the Windows Machine Learning SDK.

## In this section



| Topic                                                                             | Description                                                                                                                                                                                |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IWinMLEvaluationContext**](https://www.bing.com/search?q=**IWinMLEvaluationContext**)<br/> | Represents the context to bind inputs and outputs to a WinML model.<br/>                                                                                                             |
| [**IWinMLModel**](https://www.bing.com/search?q=**IWinMLModel**)<br/>                         | Represents a Windows Machine Learning model with corresponding metadata; includes model descriptions (name, author, versioning, etc.), as well as expected inputs and outputs. <br/> |
| [**IWinMLRuntime**](https://www.bing.com/search?q=**IWinMLRuntime**)<br/>                     | Represents the runtime to load and evaluate a WinML model.<br/>                                                                                                                      |
| [**IWinMLRuntimeFactory**](https://www.bing.com/search?q=**IWinMLRuntimeFactory**)<br/>       | Represents the factory that creates the WinML runtime for model loading and evaluation.<br/>                                                                                         |
