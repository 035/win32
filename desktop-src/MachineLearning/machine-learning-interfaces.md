# Windows Machine Learning Interfaces

The following topics describe the COM interfaces for the Windows Machine Learning SDK.

## In this section



| Topic                                                                             | Description                                                                                                                                                                                |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IWinMLEvaluationContext**](https://msdn.microsoft.com/D4C9B16A-B351-41E4-AD42-20C25F3CC404)<br/> | Represents the context to bind inputs and outputs to a WinML model.<br/>                                                                                                             |
| [**IWinMLModel**](https://msdn.microsoft.com/604ABFCC-9CA0-409D-A3FF-D5C59758462E)<br/>                         | Represents a Windows Machine Learning model with corresponding metadata; includes model descriptions (name, author, versioning, etc.), as well as expected inputs and outputs. <br/> |
| [**IWinMLRuntime**](https://msdn.microsoft.com/C2FD74A1-EE38-46B1-98A8-43557485F92E)<br/>                     | Represents the runtime to load and evaluate a WinML model.<br/>                                                                                                                      |
| [**IWinMLRuntimeFactory**](https://msdn.microsoft.com/7817A028-031C-49AA-A17A-4364DC0E78D0)<br/>       | Represents the factory that creates the WinML runtime for model loading and evaluation.<br/>                                                                                         |
