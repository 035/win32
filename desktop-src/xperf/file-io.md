# File IO

**Overview:** Increased file I/O operations can dramatically affect system and application performance, as shown in the following screen shot.

![screen shot of a window showing a file i/o graph](images/gr-img029.png)

**Graph Type:** Histogram

**Y-axis Units:** Counts of file I/O operations

**Required Flags:** FILE\_IO+FILE\_IO\_INIT

**Events Captured:** File I/O

**Legend Description:** Lists all file I/O types.

**Graph Description:** Displays all file I/O operations as a histogram over time.
