# On/Off Transition Trace Capture tool

This paper provides information about how to use the On/Off Transition Trace Capture tool on Windows Vista and later versions of the operating system. It provides guidelines for original equipment manufacturers (OEMs) to measure performance during on/off transitions.

This document accompanies the Microsoft Windows 2008 Software Development Kit release of the Xperf tools.
