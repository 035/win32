# The Driver Delay Action

The driver delay action produces a text report that summarizes the various metrics regarding drivers. The usage for this action is:


```
-a drvdelay [-min [n] -name driver -range T1 T2] 
```





| Option                  | Description                                       |
|-------------------------|---------------------------------------------------|
| min n <br/>       | Show only delays longer than *n* use <br/>  |
| range T1 T2 <br/> | Show delays between T1 and T2, in use <br/> |
| name driver <br/> | Show only delays by driver<br/>             |
