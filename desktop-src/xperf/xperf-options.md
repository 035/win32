# Xperf Options

The following are the command-line help options:



| Command                                                 | Meaning                                                                             |
|---------------------------------------------------------|-------------------------------------------------------------------------------------|
| **xperf -help**<br/>                              | Displays this help page<br/>                                                  |
| [**xperf -help view**](view.md)<br/>             | Displays the help page for **Xperfview**, the graphical user interface<br/>   |
| [**xperf -help start**](start.md)<br/>           | Displays the help page for the logger start options<br/>                      |
| [**xperf -help profile**](profile.md)<br/>       | Displays the help page for action invocation and action help options<br/>     |
| [**xperf -help providers**](providers.md)<br/>   | Displays the help page for the known tracing flags<br/>                       |
| [**xperf -help stackwalk**](stackwalk.md)<br/>   | Displays the help page for stack walking options<br/>                         |
| [**xperf -help stop**](stop.md)<br/>             | Displays the help page for the logger stop options<br/>                       |
| [**xperf -help merge**](merge.md)<br/>           | Displays the help page for merging multiple trace files<br/>                  |
| [**xperf -help processing**](processing.md)<br/> | Displays the help page for trace processing options<br/>                      |
| [**xperf -help symbols**](symbols.md)<br/>       | Displays the help page for symbol decoding configuration<br/>                 |
| [**xperf -help query**](query.md)<br/>           | Displays the help page for query options<br/>                                 |
| [**xperf -help mark**](mark.md)<br/>             | Displays the help page for mark and mark-flush<br/>                           |
| [**xperf -help format**](format.md)<br/>         | Displays the help page for time and timespan formats on the command line<br/> |
| [**xperf -help advanced**](advanced.md)<br/>     | Displays the help page for the advanced options.<br/>                         |
