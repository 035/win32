# ReadyBoot Quick Start Guide

The following is a brief example of using Windows Performance Analyzer (WPA) to identify ReadyBoot activity through the use of the ReadyBoot I/O graph and a ReadyBoot summary table.

This example is not a complete review of WPA function or ReadyBoot analysis. It is meant to give the first time user of WPA a high level overview of a small number of the functions provided for analyzing ReadyBoot performance. Please see [MSDN](http://go.microsoft.com/fwlink/p/?linkid=141497) for a complete discussion of WPA. For more complete information on ReadyBoot analysis, see the [ReadyBoot Analysis](readyboot-analysis.md) section of this document.

For installation support please see the [System Requirements](system-requirements.md) and [Installation](installation.md) sections of this document.
