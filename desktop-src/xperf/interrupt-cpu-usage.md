# Interrupt CPU Usage

**Overview:** Displays CPU resources consumed by servicing interrupts, as shown in the following screen shot.

![screen shot of a graph showing time as a percentage spent servicing interrupts](images/gr-img022.png)

**Graph Type:** Usage graph

**Y-axis Units:** Percentage of CPU usage

**Required Flags:** INTERRUPT

**Events Captured:** Service interrupt events

**Legend Description:** Active CPUs on the system

**Graph Description:** This graph displays the percentage of the total CPU resource each processor spends servicing device interrupts.
