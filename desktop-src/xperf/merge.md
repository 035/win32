# merge

Displays the trace merge options.

``` syntax
xperf -merge F1 F2 ... MEtl
```

### Example

The following example merges F1 F2 ... into MEtl and adds image identification information that is required for safe symbol decoding.

**-merge  F1 F2 ... MEtl**
