# mark

Displays mark and mark-flush options.

``` syntax
xperf -m | -mf <mark text>
```

## Parameters

<dl> <dt>

<span id="_______________-m__________string"></span><span id="_______________-M__________STRING"></span> **-m** string
</dt> <dd>

Log a mark in the current trace.

</dd> <dt>

<span id="_______________-mf_________string___________"></span><span id="_______________-MF_________STRING___________"></span> **-mf** string 
</dt> <dd>

Log a mark in the current trace and, if FOOTPRINT is enabled, flush working sets.

</dd> </dl>
