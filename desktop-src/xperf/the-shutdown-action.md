# The Shutdown Action

The shutdown action produces a text file that summarizes the various metrics regarding shutdown. The usage for this action is:


```
-a shutdown [options] 
```





| Option                     | Description                                                |
|----------------------------|------------------------------------------------------------|
| minCPUToShow <br/>   | Minimum percentage (0-100) of CPU usage to show<br/> |
| maxFilesToShow <br/> | Maximum number of files to show<br/>                 |
