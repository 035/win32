# The Filename Action

The filename action produces a text report that summarizes the various filenames in the trace. The usage for this action is:


```
-a filename [-range T1 T2] 
```





| Option                  | Description                                                    |
|-------------------------|----------------------------------------------------------------|
| range T1 T2 <br/> | Show file names recorded between T1 and T2, in use <br/> |
