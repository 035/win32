# Built-in Profiles

To display all the currently known profiles, run the following command:


```
xperf -profiles
```



The command displays the entire **InSequentialFile** and **InBuffer** profiles in the profile file (perf.xpp) as shown in the following table.



| Profile Name                                          | Description                                                                                              |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| perf!DWMProfiles.InSequentialFile <br/>         | DWM tracing profile in sequential file<br/>                                                        |
| perf!DWMProfiles.InBuffer<br/>                  | DWM tracing profile in buffer<br/>                                                                 |
| perf!FileIOProfiles.InSequentialFile<br/>       | File I/O tracing profile in sequential file<br/>                                                   |
| perf!FileIOProfiles.InBuffer<br/>               | File I/O tracing profile in buffer<br/>                                                            |
| perf!GamingProfiles.InSequentialFile <br/>      | Games tracing profile in sequential file<br/>                                                      |
| perf!GamingProfiles.InBuffer <br/>              | Games tracing profile in buffer<br/>                                                               |
| perf!GeneralProfiles.InSequentialFile<br/>      | Common system metrics tracing profile in sequential file<br/>                                      |
| perf!GeneralProfiles.InBuffer<br/>              | Common system metrics tracing profile in buffer<br/>                                               |
| perf!HibernateProfiles.InSequentialFile <br/>   | Resume from hibernation tracing profile in sequential file<br/>                                    |
| perf!HibernateProfiles.InBuffer <br/>           | Resume from hibernation tracing profile in buffer<br/>                                             |
| perf!IE7Profiles.InSequentialFile <br/>         | IE7 tracing profile in sequential file<br/>                                                        |
| perf!IE7Profiles.InBuffer <br/>                 | IE7 tracing profile in buffer<br/>                                                                 |
| perf!MediaCenterProfiles.InSequentialFile <br/> | MediaCenter tracing profile in sequential file<br/>                                                |
| perf!MediaCenterProfiles.InBuffer <br/>         | MediaCenter tracing profile in buffer<br/>                                                         |
| perf!MemoryProfiles.InSequentialFile<br/>       | Memory usage tracing profile in sequential file<br/>                                               |
| perf!MemoryProfiles.InBuffer <br/>              | Memory usage tracing profile in buffer<br/>                                                        |
| perf!MultimediaProfiles.InSequentialFile <br/>  | Multimedia tracing profile in sequential file<br/>                                                 |
| perf!MultimediaProfiles.InBuffer <br/>          | Multimedia tracing profile in buffer<br/>                                                          |
| perf!PerfCoreProfiles.InSequentialFile<br/>     | Basic system metrics tracing profile (included with all built-in profiles) in sequential file<br/> |
| perf!PerfCoreProfiles.InBuffer<br/>             | Basic system metrics tracing profile (included with all built-in profiles) in buffer<br/>          |
| perf!RegistryProfiles.InSequentialFile<br/>     | Registry tracing profile in sequential file<br/>                                                   |
| perf!RegistryProfiles.InBuffer<br/>             | Registry tracing profile in buffer<br/>                                                            |
| perf!SleepProfiles.InSequentialFile <br/>       | Resume from sleep tracing profile in sequential file<br/>                                          |
| perf!SleepProfiles.InBuffer <br/>               | Resume from sleep tracing profile in buffer<br/>                                                   |
| perf!StdProfile<br/>                            | Common definitions used in built-in profiles (not startable)<br/>                                  |
