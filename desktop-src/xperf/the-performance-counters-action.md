# The Performance Counters Action

The performance counters action produces a text file that summarizes the various metrics regarding performance counters. The usage for this action is:


```
-a perfctrs [-range T1 T2] 
```





| Option                  | Description                                                   |
|-------------------------|---------------------------------------------------------------|
| range T1 T2 <br/> | Show file names recorded between T1 and T2, in use<br/> |
