# The Win Logon Action

The Win Logon action produces a text file that summarizes the various metrics logon. The usage for this action is:


```
-a winlogon [-range T1 T2] 
```





| Option                  | Description                                      |
|-------------------------|--------------------------------------------------|
| range T1 T2 <br/> | Show events between T1 and T2, in use<br/> |
