# Windows Performance Analyzer (WPA)

Windows Performance Analyzer (WPA) is a set of performance monitoring tools used to produce in-depth performance profiles of Microsoft Windows operating systems and applications.

This documentation discusses the tools in the Windows Performance Analyzer as shown in the following table.



| Section                                                                                               | Description                                                                                                                                  |
|-------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| [Introduction: Intro to WPA](introduction--intro-to-wpa.md)<br/>                               | Contains an overview of WPA components, architecture, installation and a Quick Start manual.<br/>                                      |
| [Quick Start Guide: WPA Basics](quick-start-guide--wpa-basics.md)<br/>                         | Covers information and examples you will need to effectively use WPA.<br/>                                                             |
| [Feature Level Documentation: WPA Features](feature-level-documentation--wpa-features.md)<br/> | Describes specific features including Graphs, Summary Tables, and a Command Line Reference<br/>                                        |
| [Performance Analysis Whitepapers](performance-analysis-whitepapers.md)<br/>                   | Illustrates how to use special tools and procedures to analyze specific facets of performance such as Processor Power Management.<br/> |
| [Application Programming Interfaces](application-programming-interfaces.md)<br/>               | Introduces an API for programmatically capturing kernel traces.<br/>                                                                   |
