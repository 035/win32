# The Marks Action

The marks action produces a text file that summarizes the various marks placed in a trace. The usage for this action is:


```
 -a marks [-summary] [-range T1 T2] 
```





| Option                  | Description                                                        |
|-------------------------|--------------------------------------------------------------------|
| summary <br/>     | Display the marks in this trace <br/>                        |
| range T1 T2 <br/> | Only display the marks in the time range from T1 to T2 <br/> |
