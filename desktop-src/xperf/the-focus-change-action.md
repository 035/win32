# The Focus Change Action

The focus change action produces a text report that summarizes the various focus changes regarding the Z-ordering of windows. The usage for this action is:


```
-a focuschange [-range T1 T2] 
```





| Option                  | Description                                                                            |
|-------------------------|----------------------------------------------------------------------------------------|
| range T1 T2 <br/> | Show Windows thread focus change events recorded between T1 and T2, in use <br/> |
