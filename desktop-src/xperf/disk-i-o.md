# Disk I/O

**Overview:** Displays disk reads, writes, and flushes in a histogram, as shown in the following screen shot.

![screen shot of a graph showing disk reads, writes, and flushes in a histogram](images/gr-img016.png)

**Graph Type:** Histogram graph

**Y-axis Units:** Count of disk read, write and flush operations

**Required Flags:** DISK\_IO

**Events Captured:** Disk I/O events

**Legend Description:** The legend allows the user to identify disk reads, writes and flushes based on less than or equal to low priority and greater than or equal to normal priority.

**Graph Description:** A histogram graph that displays disk activity over time. This graph can provide the analyst with a high level overview of system disk I/O. Please see the [Types of Graphs](types-of-graphs.md) section of this document for the explanation of Histogram graphs.
