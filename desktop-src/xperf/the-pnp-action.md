# The PnP Action

The PnP action produces a text file that summarizes the various metrics regarding context switches. The usage for this action is:


```
-a pnp [-range T1 T2] 
```





| Option                  | Description                                      |
|-------------------------|--------------------------------------------------|
| range T1 T2 <br/> | Show events between T1 and T2, in use<br/> |
