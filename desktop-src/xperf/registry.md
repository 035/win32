# Registry

**Overview:** Displays registry access in a histogram graph, as shown in the following screen shot.

![registry access graph](images/gr-img032.png)

**Graph Type:** Histogram

**Y-axis Units:** Count of registry accesses

**Required Flags:** REGISTRY

**Events Captured:** Registry

**Legend Description:** Type of registry access

**Graph Description:** Displays all registry activity on a histogram over time.
