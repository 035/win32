# D1198: Copy To Read-Only Bitmap

An attempt has been made to copy data to a read only or immutable bitmap.



|             |       |
|-------------|-------|
| Error Level | Error |



 

## Possible Causes

An attempt has been made to copy data to a read-only or immutable bitmap.

## Possible Fixes

Verify that any bitmaps that are written to are not created as immutable.
