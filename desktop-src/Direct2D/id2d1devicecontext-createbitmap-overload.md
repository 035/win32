# ID2D1DeviceContext::CreateBitmap methods

Creates a bitmap that can be used as a target surface, for reading back to the CPU, or as a source for the [**DrawBitmap**](/windows/desktop/api/D2d1_1/) and [**ID2D1BitmapBrush**](/windows/desktop/api/d2d1/) APIs. In addition, color context information can be passed to the bitmap.[**CreateBitmap**](/windows/desktop/api/D2d1_1/)[**CreateBitmap**](/windows/desktop/api/D2d1_1/)
