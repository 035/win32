# ID2D1DeviceContext::CreateBitmap methods

Creates a bitmap that can be used as a target surface, for reading back to the CPU, or as a source for the [**DrawBitmap**](id2d1devicecontext-drawbitmap.md) and [**ID2D1BitmapBrush**](id2d1bitmapbrush.md) APIs. In addition, color context information can be passed to the bitmap.[**CreateBitmap**](id2d1devicecontext-createbitmap.md)[**CreateBitmap**](id2d1devicecontext-createbitmap2.md)
