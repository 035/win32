# D1161: CopyFromMemory Requires CPU Readable Bitmap

This method is not allowed on a bitmap created with [**D2D1\_BITMAP\_OPTIONS\_CPU\_READ**](/windows/win32/D2d1_1/ne-d2d1_1-d2d1_bitmap_options?branch=master).



|             |       |
|-------------|-------|
| Error Level | Error |



 

## Possible Causes

This message may occur during a call to [**ID2D1Bitmap::CopyFromMemory**](/windows/win32/d2d1/?branch=master). If the bitmap was created using D2D1\_BITMAP\_OPTIONS\_CPU\_READ, Direct2D will report an error and the debug layer will output this message.

## Possible Fixes

If it is necessary for an application to call [**ID2D1Bitmap::CopyFromMemory**](/windows/win32/d2d1/?branch=master), ensure that the bitmap is not created as CPU-readable. If it is not possible to change the surface attributes of the bitmap, consider whether there is an equivalent rendering operation for the desired result.
