# ID2D1Geometry::Simplify methods

Creates a simplified version of the geometry that contains only lines and (optionally) cubic Bezier curves and writes the result to an [**ID2D1SimplifiedGeometrySink**](/windows/win32/d2d1/?branch=master).

| [**Simplify(D2D1\_GEOMETRY\_SIMPLIFICATION\_OPTION,D2D1\_MATRIX\_3X2\_F&,ID2D1SimplifiedGeometrySink\*)**](/windows/win32/d2d1/?branch=master)              | Creates a simplified version of the geometry that contains only lines and (optionally) cubic Bezier curves and writes the result to an [**ID2D1SimplifiedGeometrySink**](/windows/win32/d2d1/?branch=master).<br/> |
