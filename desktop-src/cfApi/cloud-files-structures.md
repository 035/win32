# Cloud Filter Structures

The following structures are used in creating and maintaining placeholder files and directories.

## In this section



| Topic                                                                                   | Description                                                                                                                               |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**CF\_CALLBACK\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827517)<br/>                          | Contains common callback information.<br/>                                                                                          |
| [**CF\_CALLBACK\_PARAMETERS**](https://msdn.microsoft.com/library/windows/desktop/mt827519)<br/>              | Contains callback specific parameters such as file offset, length, flags, etc.<br/>                                                 |
| [**CF\_CALLBACK\_REGISTRATION**](https://msdn.microsoft.com/library/windows/desktop/mt827520)<br/>          | The callbacks to be registered by the sync provider.<br/>                                                                           |
| [**CF\_FILE\_RANGE**](https://msdn.microsoft.com/library/windows/desktop/mt827529)<br/>                                | Specifies a range of data in a placeholder file.<br/>                                                                               |
| [**CF\_FILE\_RANGE\_BUFFER**](https://msdn.microsoft.com/library/windows/desktop/mt844616)<br/>                | A structure to describe the location and range of data in a file.<br/>                                                              |
| [**CF\_FS\_METADATA**](https://msdn.microsoft.com/library/windows/desktop/mt827530)<br/>                              | Placeholder file or directory metadata.<br/>                                                                                        |
| [**CF\_HYDRATION\_POLICY**](https://msdn.microsoft.com/library/windows/desktop/mt827533)<br/>                    | Specifies the primary hydration policy and its modifier.<br/>                                                                       |
| [**CF\_OPERATION\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827543)<br/>                        | Information about an operation on a placeholder file or folder.<br/>                                                                |
| [**CF\_OPERATION\_PARAMETERS**](https://msdn.microsoft.com/library/windows/desktop/mt827544)<br/>            | Parameters of an operation on a placeholder file or folder.<br/>                                                                    |
| [**CF\_PLACEHOLDER\_BASIC\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827551)<br/>       | Basic placeholder information.<br/>                                                                                                 |
| [**CF\_PLACEHOLDER\_CREATE\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827553)<br/>     | Contains placeholder information for creating new placeholder files or directories. <br/>                                           |
| [**CF\_PLACEHOLDER\_STANDARD\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827556)<br/> | Standard placeholder information.<br/>                                                                                              |
| [**CF\_PLATFORM\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827558)<br/>                          | Returns information for the cloud files platform. This is intended for sync providers running on multiple versions of Windows.<br/> |
| [**CF\_POPULATION\_POLICY**](https://msdn.microsoft.com/library/windows/desktop/mt827559)<br/>                  | Specifies the primary population policy and its modifier.<br/>                                                                      |
| [**CF\_PROCESS\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827562)<br/>                            | Contains information about a user process.<br/>                                                                                     |
| [**CF\_SYNC\_POLICIES**](https://msdn.microsoft.com/library/windows/desktop/mt827567)<br/>                          | Defines the sync policies used by a sync root.<br/>                                                                                 |
| [**CF\_SYNC\_REGISTRATION**](https://msdn.microsoft.com/library/windows/desktop/mt827569)<br/>                  | The details of the sync provider and sync root to be registered.<br/>                                                               |
| [**CF\_SYNC\_ROOT\_BASIC\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827570)<br/>          | Basic sync root information.<br/>                                                                                                   |
| [**CF\_SYNC\_ROOT\_PROVIDER\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827572)<br/>    | Sync root provider information.<br/>                                                                                                |
| [**CF\_SYNC\_ROOT\_STANDARD\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827573)<br/>    | Standard sync root information.<br/>                                                                                                |
| [**CF\_SYNC\_STATUS**](https://msdn.microsoft.com/library/windows/desktop/mt845792)<br/>                              | Used in a [**CF\_OPERATION\_INFO**](https://msdn.microsoft.com/library/windows/desktop/mt827543) structure to describe the status of a specified sync root.<br/>     |
