# WDS Transport Server Functions

Functions provided by the multicast server that interface with content providers.



| Function                                                                         | Description                                              |
|----------------------------------------------------------------------------------|----------------------------------------------------------|
| [**WdsTransportServerAllocateBuffer**](/windows/win32/Wdstpdi/nf-wdstpdi-wdstransportserverallocatebuffer?branch=master)     | Allocates a buffer in memory.                            |
| [**WdsTransportServerCompleteRead**](/windows/win32/Wdstpdi/nf-wdstpdi-wdstransportservercompleteread?branch=master)         | Provides status of read operation.                       |
| [**WdsTransportServerFreeBuffer**](/windows/win32/Wdstpdi/nf-wdstpdi-wdstransportserverfreebuffer?branch=master)             | Releases memory used by a buffer.                        |
| [**WdsTransportServerRegisterCallback**](/windows/win32/Wdstpdi/nf-wdstpdi-wdstransportserverregistercallback?branch=master) | Registers a provider callback with the multicast server. |
| [**WdsTransportServerTrace**](/windows/win32/Wdstpdi/nf-wdstpdi-wdstransportservertrace?branch=master)                       | Sends a debugging message.                               |
| [**WdsTransportServerTraceV**](/windows/win32/Wdstpdi/nf-wdstpdi-wdstransportservertracev?branch=master)                     | Sends a debugging message.                               |
