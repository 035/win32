# WDS Transport Server Functions

Functions provided by the multicast server that interface with content providers.



| Function                                                                         | Description                                              |
|----------------------------------------------------------------------------------|----------------------------------------------------------|
| [**WdsTransportServerAllocateBuffer**](wdstransportserverallocatebuffer.md)     | Allocates a buffer in memory.                            |
| [**WdsTransportServerCompleteRead**](wdstransportservercompleteread.md)         | Provides status of read operation.                       |
| [**WdsTransportServerFreeBuffer**](wdstransportserverfreebuffer.md)             | Releases memory used by a buffer.                        |
| [**WdsTransportServerRegisterCallback**](wdstransportserverregistercallback.md) | Registers a provider callback with the multicast server. |
| [**WdsTransportServerTrace**](wdstransportservertrace.md)                       | Sends a debugging message.                               |
| [**WdsTransportServerTraceV**](wdstransportservertracev.md)                     | Sends a debugging message.                               |
