# WDS Transport Management Interfaces

Windows Deployment Services (WDS) Transport Management uses the following interfaces:

-   [**IWdsTransportCacheable**](iwdstransportcacheable.md)
-   [**IWdsTransportClient**](iwdstransportclient.md)
-   [**IWdsTransportCollection**](iwdstransportcollection.md)
-   [**IWdsTransportConfigurationManager**](iwdstransportconfigurationmanager.md)
-   [**IWdsTransportConfigurationManager2**](iwdstransportconfigurationmanager2.md)
-   [**IWdsTransportContent**](iwdstransportcontent.md)
-   [**IWdsTransportDiagnosticsPolicy**](iwdstransportdiagnosticspolicy.md)
-   [**IWdsTransportManager**](iwdstransportmanager.md)
-   [**IWdsTransportMulticastSessionPolicy**](iwdstransportmulticastsessionpolicy.md)
-   [**IWdsTransportNamespace**](iwdstransportnamespace.md)
-   [**IWdsTransportNamespaceAutoCast**](iwdstransportnamespaceautocast.md)
-   [**IWdsTransportNamespaceManager**](iwdstransportnamespacemanager.md)
-   [**IWdsTransportNamespaceScheduledCast**](iwdstransportnamespacescheduledcast.md)
-   [**IWdsTransportNamespaceScheduledCastAutoStart**](iwdstransportnamespacescheduledcastautostart.md)
-   [**IWdsTransportNamespaceScheduledCastManualStart**](iwdstransportnamespacescheduledcastmanualstart.md)
-   [**IWdsTransportServer**](iwdstransportserver.md)
-   [**IWdsTransportServicePolicy**](iwdstransportservicepolicy.md)
-   [**IWdsTransportSession**](iwdstransportsession.md)
-   [**IWdsTransportSetupManager**](iwdstransportsetupmanager.md)

The following are available beginning with Windows 8 and Windows Server 2012.

-   [**IWdsTransportContentProvider**](iwdstransportcontentprovider.md)
-   [**IWdsTransportServer2**](iwdstransportserver2.md)
-   [**IWdsTransportServicePolicy2**](iwdstransportservicepolicy2.md)
-   [**IWdsTransportSetupManager2**](iwdstransportsetupmanager2.md)
-   [**IWdsTransportTftpClient**](iwdstransporttftpclient.md)
-   [**IWdsTransportTftpManager**](iwdstransporttftpmanager.md)
