# WDS Transport Management Interfaces

Windows Deployment Services (WDS) Transport Management uses the following interfaces:

-   [**IWdsTransportCacheable**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportcacheable?branch=master)
-   [**IWdsTransportClient**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportclient?branch=master)
-   [**IWdsTransportCollection**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportcollection?branch=master)
-   [**IWdsTransportConfigurationManager**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportconfigurationmanager?branch=master)
-   [**IWdsTransportConfigurationManager2**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportconfigurationmanager2?branch=master)
-   [**IWdsTransportContent**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportcontent?branch=master)
-   [**IWdsTransportDiagnosticsPolicy**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportdiagnosticspolicy?branch=master)
-   [**IWdsTransportManager**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportmanager?branch=master)
-   [**IWdsTransportMulticastSessionPolicy**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportmulticastsessionpolicy?branch=master)
-   [**IWdsTransportNamespace**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportnamespace?branch=master)
-   [**IWdsTransportNamespaceAutoCast**](/windows/win32/Wdstptmgmt/?branch=master)
-   [**IWdsTransportNamespaceManager**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportnamespacemanager?branch=master)
-   [**IWdsTransportNamespaceScheduledCast**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportnamespacescheduledcast?branch=master)
-   [**IWdsTransportNamespaceScheduledCastAutoStart**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportnamespacescheduledcastautostart?branch=master)
-   [**IWdsTransportNamespaceScheduledCastManualStart**](/windows/win32/Wdstptmgmt/?branch=master)
-   [**IWdsTransportServer**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportserver?branch=master)
-   [**IWdsTransportServicePolicy**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportservicepolicy?branch=master)
-   [**IWdsTransportSession**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportsession?branch=master)
-   [**IWdsTransportSetupManager**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportsetupmanager?branch=master)

The following are available beginning with Windows 8 and Windows Server 2012.

-   [**IWdsTransportContentProvider**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportcontentprovider?branch=master)
-   [**IWdsTransportServer2**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportserver2?branch=master)
-   [**IWdsTransportServicePolicy2**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportservicepolicy2?branch=master)
-   [**IWdsTransportSetupManager2**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransportsetupmanager2?branch=master)
-   [**IWdsTransportTftpClient**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransporttftpclient?branch=master)
-   [**IWdsTransportTftpManager**](/windows/win32/Wdstptmgmt/nn-wdstptmgmt-iwdstransporttftpmanager?branch=master)
