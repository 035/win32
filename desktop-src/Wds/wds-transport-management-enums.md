# WDS Transport Management Enumerations

Windows Deployment Services (WDS) Transport Management API uses the following enumerations:

-   [**WDSTRANSPORT\_DIAGNOSTICS\_COMPONENT\_FLAGS**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0009)
-   [**WDSTRANSPORT\_DISCONNECT\_TYPE**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0004)
-   [**WDSTRANSPORT\_FEATURE\_FLAGS**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0001)
-   [**WDSTRANSPORT\_IP\_ADDRESS\_SOURCE\_TYPE**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0007)
-   [**WDSTRANSPORT\_IP\_ADDRESS\_TYPE**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0006)
-   [**WDSTRANSPORT\_NAMESPACE\_TYPE**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0003)
-   [**WDSTRANSPORT\_NETWORK\_PROFILE\_TYPE**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0008)
-   [**WDSTRANSPORT\_PROTOCOL\_FLAGS**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0002)
-   [**WDSTRANSPORT\_SERVICE\_NOTIFICATION**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0005)
-   [**WDSTRANSPORT\_SLOW\_CLIENT\_HANDLING\_TYPE**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0010)

The following is available beginning with Windows 8 and Windows Server 2012.

-   [**WDSTRANSPORT\_TFTP\_CAPABILITY**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0012)
-   [**WDSTRANSPORT\_UDP\_PORT\_POLICY**](/windows/desktop/api/Wdstptmgmt/ne-wdstptmgmt-__midl___midl_itf_wdstptmgmt_0000_0000_0011)
