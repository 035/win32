# WDS Transport Management Enumerations

Windows Deployment Services (WDS) Transport Management API uses the following enumerations:

-   [**WDSTRANSPORT\_DIAGNOSTICS\_COMPONENT\_FLAGS**](wdstransport-diagnostics-component-flags.md)
-   [**WDSTRANSPORT\_DISCONNECT\_TYPE**](wdstransport-disconnect-type.md)
-   [**WDSTRANSPORT\_FEATURE\_FLAGS**](wdstransport-feature-flags.md)
-   [**WDSTRANSPORT\_IP\_ADDRESS\_SOURCE\_TYPE**](wdstransport-ip-address-source-type.md)
-   [**WDSTRANSPORT\_IP\_ADDRESS\_TYPE**](wdstransport-ip-address-type.md)
-   [**WDSTRANSPORT\_NAMESPACE\_TYPE**](wdstransport-namespace-type.md)
-   [**WDSTRANSPORT\_NETWORK\_PROFILE\_TYPE**](wdstransport-network-profile-type.md)
-   [**WDSTRANSPORT\_PROTOCOL\_FLAGS**](wdstransport-protocol-flags.md)
-   [**WDSTRANSPORT\_SERVICE\_NOTIFICATION**](wdstransport-service-notification.md)
-   [**WDSTRANSPORT\_SLOW\_CLIENT\_HANDLING\_TYPE**](wdstransport-slow-client-handling-type.md)

The following is available beginning with Windows 8 and Windows Server 2012.

-   [**WDSTRANSPORT\_TFTP\_CAPABILITY**](wdstransport-tftp-capability.md)
-   [**WDSTRANSPORT\_UDP\_PORT\_POLICY**](wdstransport-udp-port-policy.md)
