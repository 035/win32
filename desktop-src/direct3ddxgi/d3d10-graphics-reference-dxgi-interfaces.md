# DXGI Interfaces

This section contains info about the interfaces provided by DXGI.

## In this section



| Topic                                                               | Description                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDXGIAdapter**](/windows/win32/DXGI/nn-dxgi-idxgiadapter?branch=master)<br/>                     | The [**IDXGIAdapter**](/windows/win32/DXGI/nn-dxgi-idxgiadapter?branch=master) interface represents a display subsystem (including one or more GPUs, DACs and video memory). <br/>                                                                                                                            |
| [**IDXGIAdapter1**](/windows/win32/DXGI/nn-dxgi-idxgiadapter1?branch=master)<br/>                   | The [**IDXGIAdapter1**](/windows/win32/DXGI/nn-dxgi-idxgiadapter1?branch=master) interface represents a display sub-system (including one or more GPU's, DACs and video memory). <br/>                                                                                                                        |
| [**IDXGIAdapter2**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgiadapter2?branch=master)<br/>                   | The [**IDXGIAdapter2**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgiadapter2?branch=master) interface represents a display subsystem, which includes one or more GPUs, DACs, and video memory. <br/>                                                                                                                     |
| [**IDXGIAdapter3**](/windows/win32/dxgi1_4/nn-dxgi1_4-idxgiadapter3?branch=master)<br/>                   | This interface adds some memory residency methods, for budgeting and reserving physical memory.<br/>                                                                                                                                                                    |
| [**IDXGIAdapter4**](/windows/win32/DXGI1_6/nn-dxgi1_6-idxgiadapter4?branch=master)<br/>                   | This interface represents a display subsystem, and extends this family of interfaces to expose a method to check for an adapter's compatibility with Arbitrary Code Guard (ACG).<br/>                                                                                   |
| [**IDXGIDebug**](/windows/win32/DXGIDebug/nn-dxgidebug-idxgidebug?branch=master)<br/>                         | This interface controls debug settings, and can only be used if the debug layer is turned on.<br/>                                                                                                                                                                      |
| [**IDXGIDebug1**](/windows/win32/DXGIDebug/nn-dxgidebug-idxgidebug1?branch=master)<br/>                       | Controls debug settings for Microsoft DirectX Graphics Infrastructure (DXGI). You can use the [**IDXGIDebug1**](/windows/win32/DXGIDebug/nn-dxgidebug-idxgidebug1?branch=master) interface in Windows Store apps. <br/>                                                                                                 |
| [**IDXGIDecodeSwapChain**](/windows/win32/dxgi1_3/nn-dxgi1_3-idxgidecodeswapchain?branch=master)<br/>     | Represents a swap chain that is used by desktop media apps to decode video data and show it on a [DirectComposition](https://msdn.microsoft.com/library/windows/desktop/hh449188) surface. <br/>                                                                                                               |
| [**IDXGIDevice**](/windows/win32/DXGI/nn-dxgi-idxgidevice?branch=master)<br/>                       | An [**IDXGIDevice**](/windows/win32/DXGI/nn-dxgi-idxgidevice?branch=master) interface implements a derived class for DXGI objects that produce image data. <br/>                                                                                                                                              |
| [**IDXGIDevice1**](/windows/win32/DXGI/nn-dxgi-idxgidevice1?branch=master)<br/>                     | An [**IDXGIDevice1**](/windows/win32/DXGI/nn-dxgi-idxgidevice1?branch=master) interface implements a derived class for DXGI objects that produce image data. <br/>                                                                                                                                            |
| [**IDXGIDevice2**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgidevice2?branch=master)<br/>                     | The [**IDXGIDevice2**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgidevice2?branch=master) interface implements a derived class for DXGI objects that produce image data. The interface exposes methods to block CPU processing until the GPU completes processing, and to offer resources to the operating system. <br/> |
| [**IDXGIDevice3**](/windows/win32/DXGI1_3/nn-dxgi1_3-idxgidevice3?branch=master)<br/>                     | The [**IDXGIDevice3**](/windows/win32/DXGI1_3/nn-dxgi1_3-idxgidevice3?branch=master) interface implements a derived class for DXGI objects that produce image data. The interface exposes a method to trim graphics memory usage by the DXGI device. <br/>                                                          |
| [**IDXGIDevice4**](/windows/win32/dxgi1_5/nn-dxgi1_5-idxgidevice4?branch=master)<br/>                     | This interface provides updated methods to offer and reclaim resources.<br/>                                                                                                                                                                                            |
| [**IDXGIDeviceSubObject**](/windows/win32/DXGI/nn-dxgi-idxgidevicesubobject?branch=master)<br/>     | Inherited from objects that are tied to the device so that they can retrieve a pointer to it.<br/>                                                                                                                                                                      |
| [**IDXGIDisplayControl**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgidisplaycontrol?branch=master)<br/>       | The [**IDXGIDisplayControl**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgidisplaycontrol?branch=master) interface exposes methods to indicate user preference for the operating system's stereoscopic 3D display behavior and to set stereoscopic 3D display status to enable or disable. <br/>                          |
| [**IDXGIFactory**](/windows/win32/DXGI/nn-dxgi-idxgifactory?branch=master)<br/>                     | An [**IDXGIFactory**](/windows/win32/DXGI/nn-dxgi-idxgifactory?branch=master) interface implements methods for generating DXGI objects (which handle full screen transitions). <br/>                                                                                                                          |
| [**IDXGIFactory1**](/windows/win32/DXGI/nn-dxgi-idxgifactory1?branch=master)<br/>                   | The [**IDXGIFactory1**](/windows/win32/DXGI/nn-dxgi-idxgifactory1?branch=master) interface implements methods for generating DXGI objects.<br/>                                                                                                                                                               |
| [**IDXGIFactory2**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgifactory2?branch=master)<br/>                   | The [**IDXGIFactory2**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgifactory2?branch=master) interface includes methods to create a newer version swap chain with more features than [**IDXGISwapChain**](/windows/win32/DXGI/nn-dxgi-idxgiswapchain?branch=master) and to monitor stereoscopic 3D capabilities. <br/>                                          |
| [**IDXGIFactory3**](/windows/win32/DXGI1_3/nn-dxgi1_3-idxgifactory3?branch=master)<br/>                   | Enables creating DXGI objects. <br/>                                                                                                                                                                                                                                    |
| [**IDXGIFactory4**](/windows/win32/DXGI1_4/nn-dxgi1_4-idxgifactory4?branch=master)<br/>                   | Enables creating DXGI objects. <br/>                                                                                                                                                                                                                                    |
| [**IDXGIFactory5**](/windows/win32/DXGI1_5/nn-dxgi1_5-idxgifactory5?branch=master)<br/>                   | This interface enables a single method to support variable refresh rate displays.<br/>                                                                                                                                                                                  |
| [**IDXGIFactory6**](/windows/win32/dxgi1_6/nn-dxgi1_6-idxgifactory6?branch=master)<br/>                   | This interface enables a single method that enumerates graphics adapters based on a given GPU preference.<br/>                                                                                                                                                          |
| [**IDXGIFactoryMedia**](/windows/win32/dxgi1_3/nn-dxgi1_3-idxgifactorymedia?branch=master)<br/>           | Creates swap chains for desktop media apps that use [DirectComposition](https://msdn.microsoft.com/library/windows/desktop/hh449188) surfaces to decode and display video. <br/>                                                                                                                               |
| [**IDXGIInfoQueue**](/windows/win32/DXGIDebug/nn-dxgidebug-idxgiinfoqueue?branch=master)<br/>                 | This interface controls the debug information queue, and can only be used if the debug layer is turned on.<br/>                                                                                                                                                         |
| [**IDXGIKeyedMutex**](/windows/win32/DXGI/nn-dxgi-idxgikeyedmutex?branch=master)<br/>               | Represents a keyed mutex, which allows exclusive access to a shared resource that is used by multiple devices.<br/>                                                                                                                                                     |
| [**IDXGIObject**](/windows/win32/DXGI/nn-dxgi-idxgiobject?branch=master)<br/>                       | An [**IDXGIObject**](/windows/win32/DXGI/nn-dxgi-idxgiobject?branch=master) interface is a base interface for all DXGI objects; **IDXGIObject** supports associating caller-defined (private data) with an object and retrieval of an interface to the parent object. <br/>                                   |
| [**IDXGIOutput**](/windows/win32/DXGI/nn-dxgi-idxgioutput?branch=master)<br/>                       | An [**IDXGIOutput**](/windows/win32/DXGI/nn-dxgi-idxgioutput?branch=master) interface represents an adapter output (such as a monitor).<br/>                                                                                                                                                                  |
| [**IDXGIOutput1**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgioutput1?branch=master)<br/>                     | An [**IDXGIOutput1**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgioutput1?branch=master) interface represents an adapter output (such as a monitor).<br/>                                                                                                                                                                |
| [**IDXGIOutput2**](/windows/win32/DXGI1_3/nn-dxgi1_3-idxgioutput2?branch=master)<br/>                     | Represents an adapter output (such as a monitor). The [**IDXGIOutput2**](/windows/win32/DXGI1_3/nn-dxgi1_3-idxgioutput2?branch=master) interface exposes a method to check for multiplane overlay support on the primary output adapter.<br/>                                                                       |
| [**IDXGIOutput3**](/windows/win32/DXGI1_3/nn-dxgi1_3-idxgioutput3?branch=master)<br/>                     | Represents an adapter output (such as a monitor). The [**IDXGIOutput3**](/windows/win32/DXGI1_3/nn-dxgi1_3-idxgioutput3?branch=master) interface exposes a method to check for overlay support.<br/>                                                                                                                |
| [**IDXGIOutput4**](/windows/win32/DXGI1_4/nn-dxgi1_4-idxgioutput4?branch=master)<br/>                     | Represents an adapter output (such as a monitor). The [**IDXGIOutput4**](/windows/win32/DXGI1_4/nn-dxgi1_4-idxgioutput4?branch=master) interface exposes a method to check for overlay color space support.<br/>                                                                                                    |
| [**IDXGIOutput5**](/windows/win32/DXGI1_5/nn-dxgi1_5-idxgioutput5?branch=master)<br/>                     | Represents an adapter output (such as a monitor). The [**IDXGIOutput5**](/windows/win32/DXGI1_5/nn-dxgi1_5-idxgioutput5?branch=master) interface exposes a single method to specify a list of supported formats for fullscreen surfaces.<br/>                                                                       |
| [**IDXGIOutput6**](/windows/win32/DXGI1_6/nn-dxgi1_6-idxgioutput6?branch=master)<br/>                     | Represents an adapter output (such as a monitor). The [**IDXGIOutput6**](/windows/win32/DXGI1_6/nn-dxgi1_6-idxgioutput6?branch=master) interface exposes methods to provide specific monitor capabilities.<br/>                                                                                                     |
| [**IDXGIOutputDuplication**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgioutputduplication?branch=master)<br/> | The [**IDXGIOutputDuplication**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgioutputduplication?branch=master) interface accesses and manipulates the duplicated desktop image. <br/>                                                                                                                                     |
| [**IDXGIResource**](/windows/win32/DXGI/nn-dxgi-idxgiresource?branch=master)<br/>                   | An [**IDXGIResource**](/windows/win32/DXGI/nn-dxgi-idxgiresource?branch=master) interface allows resource sharing and identifies the memory that a resource resides in. <br/>                                                                                                                                 |
| [**IDXGIResource1**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgiresource1?branch=master)<br/>                 | An [**IDXGIResource1**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgiresource1?branch=master) interface extends the [**IDXGIResource**](/windows/win32/DXGI/nn-dxgi-idxgiresource?branch=master) interface by adding support for creating a subresource surface object and for creating a handle to a shared resource. <br/>                                    |
| [**IDXGISurface**](/windows/win32/DXGI/nn-dxgi-idxgisurface?branch=master)<br/>                     | The [**IDXGISurface**](/windows/win32/DXGI/nn-dxgi-idxgisurface?branch=master) interface implements methods for image-data objects.<br/>                                                                                                                                                                      |
| [**IDXGISurface1**](/windows/win32/DXGI/nn-dxgi-idxgisurface1?branch=master)<br/>                   | The [**IDXGISurface1**](/windows/win32/DXGI/nn-dxgi-idxgisurface1?branch=master) interface extends the [**IDXGISurface**](/windows/win32/DXGI/nn-dxgi-idxgisurface?branch=master) by adding support for using Windows Graphics Device Interface (GDI) to render to a DXGI surface.<br/>                                                             |
| [**IDXGISurface2**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgisurface2?branch=master)<br/>                   | The [**IDXGISurface2**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgisurface2?branch=master) interface extends the [**IDXGISurface1**](/windows/win32/DXGI/nn-dxgi-idxgisurface1?branch=master) interface by adding support for subresource surfaces and getting a handle to a shared resource.<br/>                                                            |
| [**IDXGISwapChain**](/windows/win32/DXGI/nn-dxgi-idxgiswapchain?branch=master)<br/>                 | An [**IDXGISwapChain**](/windows/win32/DXGI/nn-dxgi-idxgiswapchain?branch=master) interface implements one or more [**surfaces**](/windows/win32/DXGI/nn-dxgi-idxgisurface?branch=master) for storing rendered data before presenting it to an output.<br/>                                                                                         |
| [**IDXGISwapChain1**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgiswapchain1?branch=master)<br/>               | Provides presentation capabilities that are enhanced from [**IDXGISwapChain**](/windows/win32/DXGI/nn-dxgi-idxgiswapchain?branch=master). These presentation capabilities consist of specifying dirty rectangles and scroll rectangle to optimize the presentation.<br/>                                      |
| [**IDXGISwapChain2**](/windows/win32/dxgi1_3/nn-dxgi1_3-idxgiswapchain2?branch=master)<br/>               | Extends [**IDXGISwapChain1**](/windows/win32/DXGI1_2/nn-dxgi1_2-idxgiswapchain1?branch=master) with methods to support swap back buffer scaling and lower-latency swap chains.<br/>                                                                                                                                 |
| [**IDXGISwapChain3**](/windows/win32/DXGI1_4/nn-dxgi1_4-idxgiswapchain3?branch=master)<br/>               | Extends [**IDXGISwapChain2**](/windows/win32/dxgi1_3/nn-dxgi1_3-idxgiswapchain2?branch=master) with methods to support getting the index of the swap chain's current back buffer and support for color space.<br/>                                                                                                  |
| [**IDXGISwapChain4**](/windows/win32/dxgi1_5/nn-dxgi1_5-idxgiswapchain4?branch=master)<br/>               | This interface exposes a single method for setting video metadata.<br/>                                                                                                                                                                                                 |
| [**IDXGISwapChainMedia**](/windows/win32/dxgi1_3/nn-dxgi1_3-idxgiswapchainmedia?branch=master)<br/>       | This swap chain interface allows desktop media applications to request a seamless change to a specific refresh rate.<br/>                                                                                                                                               |
