# DXGI Structures

This section contains info about the structures provided by DXGI.

## In this section



| Topic                                                                                     | Description                                                                                                                                                                                                                                                                                                |
|-------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**D3DCOLORVALUE**](d3dcolorvalue.md)<br/>                                         | Represents a color value with alpha, which is used for transparency. <br/>                                                                                                                                                                                                                           |
| [**DXGI\_ADAPTER\_DESC**](dxgi-adapter-desc.md)<br/>                               | Describes an adapter (or video card) by using DXGI 1.0.<br/>                                                                                                                                                                                                                                         |
| [**DXGI\_ADAPTER\_DESC1**](dxgi-adapter-desc1.md)<br/>                             | Describes an adapter (or video card) using DXGI 1.1.<br/>                                                                                                                                                                                                                                            |
| [**DXGI\_ADAPTER\_DESC2**](dxgi-adapter-desc2.md)<br/>                             | Describes an adapter (or video card) that uses Microsoft DirectX Graphics Infrastructure (DXGI) 1.2.<br/>                                                                                                                                                                                            |
| [**DXGI\_ADAPTER\_DESC3**](dxgi-adapter-desc3.md)<br/>                             | Describes an adapter (or video card) that uses DXGI 1.6.<br/>                                                                                                                                                                                                                                        |
| [**DXGI\_DECODE\_SWAP\_CHAIN\_DESC**](dxgi-decode-swap-chain-desc.md)<br/>         | Used with [**IDXGIFactoryMedia**](idxgifactorymedia.md)::[**CreateDecodeSwapChainForCompositionSurfaceHandle**](idxgifactorymedia-createdecodeswapchainforcompositionsurfacehandle.md) to describe a decode swap chain.<br/>                                                                       |
| [**DXGI\_DISPLAY\_COLOR\_SPACE**](dxgi-display-color-space.md)<br/>                | Don't use this structure; it is not supported and it will be removed from the header in a future release. <br/>                                                                                                                                                                                      |
| [**DXGI\_FRAME\_STATISTICS**](dxgi-frame-statistics.md)<br/>                       | Describes timing and presentation statistics for a frame.<br/>                                                                                                                                                                                                                                       |
| [**DXGI\_FRAME\_STATISTICS\_MEDIA**](dxgi-frame-statistics-media.md)<br/>          | Used to verify system approval for the app's custom present duration (custom refresh rate).<br/>                                                                                                                                                                                                     |
| [**DXGI\_GAMMA\_CONTROL**](dxgi-gamma-control.md)<br/>                             | Controls the settings of a gamma curve.<br/>                                                                                                                                                                                                                                                         |
| [**DXGI\_GAMMA\_CONTROL\_CAPABILITIES**](dxgi-gamma-control-capabilities.md)<br/>  | Controls the gamma capabilities of an adapter.<br/>                                                                                                                                                                                                                                                  |
| [**DXGI\_HDR\_METADATA\_HDR10**](dxgi-hdr-metadata-hdr10.md)<br/>                  | Describes the metadata for HDR10, used when video is compressed using High Efficiency Video Coding (HEVC).<br/>                                                                                                                                                                                      |
| [**DXGI\_INFO\_QUEUE\_FILTER**](dxgi-info-queue-filter.md)<br/>                    | Describes a debug message filter, which contains lists of message types to allow and deny.<br/>                                                                                                                                                                                                      |
| [**DXGI\_INFO\_QUEUE\_FILTER\_DESC**](dxgi-info-queue-filter-desc.md)<br/>         | Describes the types of messages to allow or deny to pass through a filter.<br/>                                                                                                                                                                                                                      |
| [**DXGI\_INFO\_QUEUE\_MESSAGE**](dxgi-info-queue-message.md)<br/>                  | Describes a debug message in the information queue.<br/>                                                                                                                                                                                                                                             |
| [**DXGI\_JPEG\_AC\_HUFFMAN\_TABLE**](dxgi-jpeg-ac-huffman-table.md)<br/>           | Describes a JPEG AC huffman table.<br/>                                                                                                                                                                                                                                                              |
| [**DXGI\_JPEG\_DC\_HUFFMAN\_TABLE**](dxgi-jpeg-dc-huffman-table.md)<br/>           | Describes a JPEG DC huffman table.<br/>                                                                                                                                                                                                                                                              |
| [**DXGI\_JPEG\_QUANTIZATION\_TABLE**](dxgi-jpeg-quantization-table.md)<br/>        | Describes a JPEG quantization table.<br/>                                                                                                                                                                                                                                                            |
| [**DXGI\_MATRIX\_3X2\_F**](dxgi-matrix-3x2-f.md)<br/>                              | Represents a 3x2 matrix. Used with [**GetMatrixTransform**](idxgiswapchain2-getmatrixtransform.md) and [**SetMatrixTransform**](idxgiswapchain2-setmatrixtransform.md) to indicate the scaling and translation transform for [**SwapChainPanel**](w_ui_xaml_ctrl.swapchainpanel) swap chains.<br/> |
| [**DXGI\_MAPPED\_RECT**](dxgi-mapped-rect.md)<br/>                                 | Describes a mapped rectangle that is used to access a surface.<br/>                                                                                                                                                                                                                                  |
| [**DXGI\_MODE\_DESC**](dxgi-mode-desc.md)<br/>                                     | Describes a display mode.<br/>                                                                                                                                                                                                                                                                       |
| [**DXGI\_MODE\_DESC1**](dxgi-mode-desc1.md)<br/>                                   | Describes a display mode and whether the display mode supports stereo.<br/>                                                                                                                                                                                                                          |
| [**DXGI\_OUTPUT\_DESC**](dxgi-output-desc.md)<br/>                                 | Describes an output or physical connection between the adapter (video card) and a device.<br/>                                                                                                                                                                                                       |
| [**DXGI\_OUTPUT\_DESC1**](dxgi-output-desc1.md)<br/>                               | Describes an output or physical connection between the adapter (video card) and a device, including additional information about color capabilities and connection type.<br/>                                                                                                                        |
| [**DXGI\_OUTDUPL\_DESC**](dxgi-outdupl-desc.md)<br/>                               | The DXGI\_OUTDUPL\_DESC structure describes the dimension of the output and the surface that contains the desktop image. The format of the desktop image is always [**DXGI\_FORMAT\_B8G8R8A8\_UNORM**](direct3ddxgi.dxgi_format#dxgi-format-b8g8r8a8-unorm).<br/>         |
| [**DXGI\_OUTDUPL\_FRAME\_INFO**](dxgi-outdupl-frame-info.md)<br/>                  | The [**DXGI\_OUTDUPL\_FRAME\_INFO**](dxgi-outdupl-frame-info.md) structure describes the current desktop image.<br/>                                                                                                                                                                                |
| [**DXGI\_OUTDUPL\_MOVE\_RECT**](dxgi-outdupl-move-rect.md)<br/>                    | The [**DXGI\_OUTDUPL\_MOVE\_RECT**](dxgi-outdupl-move-rect.md) structure describes the movement of a rectangle.<br/>                                                                                                                                                                                |
| [**DXGI\_OUTDUPL\_POINTER\_POSITION**](dxgi-outdupl-pointer-position.md)<br/>      | The [**DXGI\_OUTDUPL\_POINTER\_POSITION**](dxgi-outdupl-pointer-position.md) structure describes the position of the hardware cursor.<br/>                                                                                                                                                          |
| [**DXGI\_OUTDUPL\_POINTER\_SHAPE\_INFO**](dxgi-outdupl-pointer-shape-info.md)<br/> | The [**DXGI\_OUTDUPL\_POINTER\_SHAPE\_INFO**](dxgi-outdupl-pointer-shape-info.md) structure describes information about the cursor shape.<br/>                                                                                                                                                      |
| [**DXGI\_PRESENT\_PARAMETERS**](dxgi-present-parameters.md)<br/>                   | Describes information about present that helps the operating system optimize presentation.<br/>                                                                                                                                                                                                      |
| [**DXGI\_QUERY\_VIDEO\_MEMORY\_INFO**](dxgi-query-video-memory-info.md)<br/>       | Describes the current video memory budgeting parameters.<br/>                                                                                                                                                                                                                                        |
| [**DXGI\_RATIONAL**](dxgi-rational.md)<br/>                                        | Represents a rational number.<br/>                                                                                                                                                                                                                                                                   |
| [**DXGI\_RGB**](dxgi-rgb.md)<br/>                                                  | Represents an RGB color.<br/>                                                                                                                                                                                                                                                                        |
| [**DXGI\_RGBA**](dxgi-rgba.md)<br/>                                                | Represents a color value with alpha, which is used for transparency. <br/>                                                                                                                                                                                                                           |
| [**DXGI\_SAMPLE\_DESC**](dxgi-sample-desc.md)<br/>                                 | Describes multi-sampling parameters for a resource.<br/>                                                                                                                                                                                                                                             |
| [**DXGI\_SHARED\_RESOURCE**](dxgi-shared-resource.md)<br/>                         | Represents a handle to a shared resource.<br/>                                                                                                                                                                                                                                                       |
| [**DXGI\_SURFACE\_DESC**](dxgi-surface-desc.md)<br/>                               | Describes a surface.<br/>                                                                                                                                                                                                                                                                            |
| [**DXGI\_SWAP\_CHAIN\_DESC**](dxgi-swap-chain-desc.md)<br/>                        | Describes a swap chain.<br/>                                                                                                                                                                                                                                                                         |
| [**DXGI\_SWAP\_CHAIN\_DESC1**](dxgi-swap-chain-desc1.md)<br/>                      | Describes a swap chain.<br/>                                                                                                                                                                                                                                                                         |
| [**DXGI\_SWAP\_CHAIN\_FULLSCREEN\_DESC**](dxgi-swap-chain-fullscreen-desc.md)<br/> | Describes full-screen mode for a swap chain.<br/>                                                                                                                                                                                                                                                    |
| [**\_LUID**](-luid.md)<br/>                                                        | Describes a local identifier for an adapter.<br/>                                                                                                                                                                                                                                                    |
