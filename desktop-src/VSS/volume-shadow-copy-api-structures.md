# Volume Shadow Copy API Structures

The following structures are defined in the Volume Shadow Copy API.



| Structure                                                           | Description                                                                                                                                                                                                                                                         |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**VSS\_COMPONENTINFO**](vss-componentinfo.md)                     | Contains information about a given component.                                                                                                                                                                                                                       |
| [**VSS\_DIFF\_AREA\_PROP**](vss-diff-area-prop.md)                 | Describes associations between source volumes containing the original file data and volumes containing the shadow copy storage area.                                                                                                                                |
| [**VSS\_DIFF\_VOLUME\_PROP**](vss-diff-volume-prop.md)             | Describes a shadow copy storage area volume.                                                                                                                                                                                                                        |
| [**VSS\_MGMT\_OBJECT\_PROP**](vss-mgmt-object-prop.md)             | Describes the properties of a volume, shadow copy storage volume, or a shadow copy storage area.                                                                                                                                                                    |
| [**VSS\_MGMT\_OBJECT\_UNION**](vss-mgmt-object-union.md)           | A union of [**VSS\_VOLUME\_PROP**](vss-volume-prop.md), [**VSS\_DIFF\_VOLUME\_PROP**](vss-diff-volume-prop.md), and [**VSS\_DIFF\_AREA\_PROP**](vss-diff-area-prop.md) structures used by the [**VSS\_MGMT\_OBJECT\_PROP**](vss-mgmt-object-prop.md) structure. |
| [**VSS\_OBJECT\_PROP**](vss-object-prop.md)                        | Describes the properties of a provider, volume, shadow copy, or shadow copy set.                                                                                                                                                                                    |
| [**VSS\_OBJECT\_UNION**](vss-object-union.md)                      | A union of [**VSS\_SNAPSHOT\_PROP**](vss-snapshot-prop.md) and [**VSS\_PROVIDER\_PROP**](vss-provider-prop.md) structures, used by the [**VSS\_OBJECT\_PROP**](vss-object-prop.md) structure.                                                                    |
| [**VSS\_PROVIDER\_PROP**](vss-provider-prop.md)                    | Specifies shadow copy provider properties.                                                                                                                                                                                                                          |
| [**VSS\_SNAPSHOT\_PROP**](vss-snapshot-prop.md)                    | Contains the properties of a shadow copy or shadow copy set.                                                                                                                                                                                                        |
| [**VSS\_VOLUME\_PROP**](vss-volume-prop.md)                        | Describes the properties of a shadow copy source volume.                                                                                                                                                                                                            |
| [**VSS\_VOLUME\_PROTECTION\_INFO**](vss-volume-protection-info.md) | Contains information about a volume's shadow copy protection level.                                                                                                                                                                                                 |
