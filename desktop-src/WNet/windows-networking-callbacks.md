# Windows Networking Callbacks

The following function prototypes are used in Windows Networking.



| Function Prototype                                 | Description                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](pfngetprofilepath.md)     | Prototype for a function that obtains a user profile path.                                                            |
| [**PFNPROCESSPOLICIES**](pfnprocesspolicies.md)   | Prototype for a function that returns the user account policy information associated with a specific Windows process. |
| [**PFNRECONCILEPROFILE**](pfnreconcileprofile.md) | Prototype of a function that reconciles a local profile with its corresponding global profile.                        |
