# Windows Networking Callbacks

The following function prototypes are used in Windows Networking.



| Function Prototype                                 | Description                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](https://www.bing.com/search?q=**PFNGETPROFILEPATH**)     | Prototype for a function that obtains a user profile path.                                                            |
| [**PFNPROCESSPOLICIES**](https://www.bing.com/search?q=**PFNPROCESSPOLICIES**)   | Prototype for a function that returns the user account policy information associated with a specific Windows process. |
| [**PFNRECONCILEPROFILE**](https://www.bing.com/search?q=**PFNRECONCILEPROFILE**) | Prototype of a function that reconciles a local profile with its corresponding global profile.                        |
