# XML HTTP Extended Request enumerations

The following enumerations are used with the XML HTTP Extended Request interfaces.

## In this section



| Topic                                                                 | Description                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XHR\_AUTH enum**](/previous-versions/windows/desktop/api/msxml6/ne-msxml6-_xhr_auth)<br/>                         | Specifies whether to allow authentication to be used to connect to a proxy or to connect to the HTTP server.<br/>                                                                                                                                                                                   |
| [**XHR\_CERT\_ERROR\_FLAG enum**](/previous-versions/windows/desktop/api/msxml6/ne-msxml6-_xhr_cert_error_flag)<br/> | Defines flags that indicate server certificate errors during SSL negotiation with the server by handling the [**OnServerCertificateReceived**](/previous-versions/windows/desktop/api/msxml6/nf-msxml6-ixmlhttprequest3callback-onservercertificatereceived) method on the [**IXMLHTTPRequest3Callback**](/previous-versions/windows/desktop/api/msxml6/nn-msxml6-ixmlhttprequest3callback) interface.<br/> |
| [**XHR\_CERT\_IGNORE\_FLAG**](/previous-versions/windows/desktop/api/msxml6/ne-msxml6-_xhr_cert_ignore_flag)<br/>    | Defines flags that you can assign to an outgoing HTTP request to ignore certain certificate errors by calling the [**SetProperty**](/previous-versions/windows/desktop/api/msxml6/nf-msxml6-ixmlhttprequest2-setproperty) method on the [**IXMLHTTPRequest3**](/previous-versions/windows/desktop/api/msxml6/nn-msxml6-ixmlhttprequest3) interface.<br/>                                                    |
| [**XHR\_COOKIE\_FLAG enum**](/previous-versions/windows/desktop/api/msxml6/ne-msxml6-_xhr_cookie_flag)<br/>          | Defines a set of flags that you can assign to a cookie in the HTTP cookie jar by calling the [**SetCookie**](/previous-versions/windows/desktop/api/msxml6/nf-msxml6-ixmlhttprequest2-setcookie) method or query from the HTTP cookie jar by calling the [**GetCookie**](/previous-versions/windows/desktop/api/msxml6/nf-msxml6-ixmlhttprequest2-getcookie) method.<br/>                                   |
| [**XHR\_COOKIE\_STATE enum**](/previous-versions/windows/desktop/api/msxml6/ne-msxml6-_xhr_cookie_state)<br/>        | Specifies the state of the cookie.<br/>                                                                                                                                                                                                                                                             |
| [**XHR\_CRED\_PROMPT enum**](/previous-versions/windows/desktop/api/msxml6/ne-msxml6-_xhr_cred_prompt)<br/>          | Specifies whether to allow credential prompts to the user for authentication. <br/>                                                                                                                                                                                                                 |
| [**XHR\_PROPERTY enum**](/previous-versions/windows/desktop/api/msxml6/ne-msxml6-_xhr_property)<br/>                 | Defines properties that you can assign to an outgoing HTTP request by calling the [**SetProperty**](/previous-versions/windows/desktop/api/msxml6/nf-msxml6-ixmlhttprequest2-setproperty) method.<br/>                                                                                                                                                  |
