# XML HTTP Extended Request enumerations

The following enumerations are used with the XML HTTP Extended Request interfaces.

## In this section



| Topic                                                                 | Description                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XHR\_AUTH enum**](/windows/previous-versions/msxml6/ne-msxml6-_xhr_auth?branch=master)<br/>                         | Specifies whether to allow authentication to be used to connect to a proxy or to connect to the HTTP server.<br/>                                                                                                                                                                                   |
| [**XHR\_CERT\_ERROR\_FLAG enum**](/windows/previous-versions/msxml6/ne-msxml6-_xhr_cert_error_flag?branch=master)<br/> | Defines flags that indicate server certificate errors during SSL negotiation with the server by handling the [**OnServerCertificateReceived**](/windows/previous-versions/msxml6/nf-msxml6-ixmlhttprequest3callback-onservercertificatereceived?branch=master) method on the [**IXMLHTTPRequest3Callback**](/windows/previous-versions/msxml6/nn-msxml6-ixmlhttprequest3callback?branch=master) interface.<br/> |
| [**XHR\_CERT\_IGNORE\_FLAG**](/windows/previous-versions/msxml6/ne-msxml6-_xhr_cert_ignore_flag?branch=master)<br/>    | Defines flags that you can assign to an outgoing HTTP request to ignore certain certificate errors by calling the [**SetProperty**](/windows/previous-versions/msxml6/nf-msxml6-ixmlhttprequest2-setproperty?branch=master) method on the [**IXMLHTTPRequest3**](/windows/previous-versions/msxml6/nn-msxml6-ixmlhttprequest3?branch=master) interface.<br/>                                                    |
| [**XHR\_COOKIE\_FLAG enum**](/windows/previous-versions/msxml6/ne-msxml6-_xhr_cookie_flag?branch=master)<br/>          | Defines a set of flags that you can assign to a cookie in the HTTP cookie jar by calling the [**SetCookie**](/windows/previous-versions/msxml6/nf-msxml6-ixmlhttprequest2-setcookie?branch=master) method or query from the HTTP cookie jar by calling the [**GetCookie**](/windows/previous-versions/msxml6/nf-msxml6-ixmlhttprequest2-getcookie?branch=master) method.<br/>                                   |
| [**XHR\_COOKIE\_STATE enum**](/windows/previous-versions/msxml6/ne-msxml6-_xhr_cookie_state?branch=master)<br/>        | Specifies the state of the cookie.<br/>                                                                                                                                                                                                                                                             |
| [**XHR\_CRED\_PROMPT enum**](/windows/previous-versions/msxml6/ne-msxml6-_xhr_cred_prompt?branch=master)<br/>          | Specifies whether to allow credential prompts to the user for authentication. <br/>                                                                                                                                                                                                                 |
| [**XHR\_PROPERTY enum**](/windows/previous-versions/msxml6/ne-msxml6-_xhr_property?branch=master)<br/>                 | Defines properties that you can assign to an outgoing HTTP request by calling the [**SetProperty**](/windows/previous-versions/msxml6/nf-msxml6-ixmlhttprequest2-setproperty?branch=master) method.<br/>                                                                                                                                                  |
