# XML HTTP Extended Request enumerations

The following enumerations are used with the XML HTTP Extended Request interfaces.

## In this section



| Topic                                                                 | Description                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XHR\_AUTH enum**](xhr-auth.md)<br/>                         | Specifies whether to allow authentication to be used to connect to a proxy or to connect to the HTTP server.<br/>                                                                                                                                                                                   |
| [**XHR\_CERT\_ERROR\_FLAG enum**](xhr-cert-error-flag.md)<br/> | Defines flags that indicate server certificate errors during SSL negotiation with the server by handling the [**OnServerCertificateReceived**](ixmlhttprequest3callback-onservercertificatereceived.md) method on the [**IXMLHTTPRequest3Callback**](ixmlhttprequest3callback.md) interface.<br/> |
| [**XHR\_CERT\_IGNORE\_FLAG**](xhr-cert-ignore-flag.md)<br/>    | Defines flags that you can assign to an outgoing HTTP request to ignore certain certificate errors by calling the [**SetProperty**](ixmlhttprequest2-setproperty.md) method on the [**IXMLHTTPRequest3**](ixmlhttprequest3.md) interface.<br/>                                                    |
| [**XHR\_COOKIE\_FLAG enum**](xhr-cookie-flag.md)<br/>          | Defines a set of flags that you can assign to a cookie in the HTTP cookie jar by calling the [**SetCookie**](ixmlhttprequest2-setcookie.md) method or query from the HTTP cookie jar by calling the [**GetCookie**](ixmlhttprequest2-getcookie.md) method.<br/>                                   |
| [**XHR\_COOKIE\_STATE enum**](xhr-cookie-state.md)<br/>        | Specifies the state of the cookie.<br/>                                                                                                                                                                                                                                                             |
| [**XHR\_CRED\_PROMPT enum**](xhr-cred-prompt.md)<br/>          | Specifies whether to allow credential prompts to the user for authentication. <br/>                                                                                                                                                                                                                 |
| [**XHR\_PROPERTY enum**](xhr-property.md)<br/>                 | Defines properties that you can assign to an outgoing HTTP request by calling the [**SetProperty**](ixmlhttprequest2-setproperty.md) method.<br/>                                                                                                                                                  |
