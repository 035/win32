# XML HTTP Extended Request structures

The following enumerations are used with the XML HTTP Extended Request interfaces.

## In this section



| Topic                                                  | Description                                                                                                                                                                                                                                           |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XHR\_CERT structure**](/windows/previous-versions/msxml6/ns-msxml6-tagxhr_cert?branch=master)<br/>     | Defines a buffer that points to an encoded certificate.<br/>                                                                                                                                                                                    |
| [**XHR\_COOKIE structure**](/windows/previous-versions/msxml6/ns-msxml6-tagxhr_cookie?branch=master)<br/> | Defines a cookie that you can add to the HTTP cookie jar by calling the [**SetCookie**](/windows/previous-versions/msxml6/nf-msxml6-ixmlhttprequest2-setcookie?branch=master) method or retrieve from the HTTP cookie jar by calling the [**GetCookie**](/windows/previous-versions/msxml6/nf-msxml6-ixmlhttprequest2-getcookie?branch=master) method.<br/> |
