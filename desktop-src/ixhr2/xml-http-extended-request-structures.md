# XML HTTP Extended Request structures

The following enumerations are used with the XML HTTP Extended Request interfaces.

## In this section



| Topic                                                  | Description                                                                                                                                                                                                                                           |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XHR\_CERT structure**](/previous-versions/windows/desktop/api/msxml6/ns-msxml6-tagxhr_cert)<br/>     | Defines a buffer that points to an encoded certificate.<br/>                                                                                                                                                                                    |
| [**XHR\_COOKIE structure**](/previous-versions/windows/desktop/api/msxml6/ns-msxml6-tagxhr_cookie)<br/> | Defines a cookie that you can add to the HTTP cookie jar by calling the [**SetCookie**](/previous-versions/windows/desktop/api/msxml6/nf-msxml6-ixmlhttprequest2-setcookie) method or retrieve from the HTTP cookie jar by calling the [**GetCookie**](/previous-versions/windows/desktop/api/msxml6/nf-msxml6-ixmlhttprequest2-getcookie) method.<br/> |
