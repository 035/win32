# XML HTTP Extended Request structures

The following enumerations are used with the XML HTTP Extended Request interfaces.

## In this section



| Topic                                                  | Description                                                                                                                                                                                                                                           |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XHR\_CERT structure**](xhr-cert.md)<br/>     | Defines a buffer that points to an encoded certificate.<br/>                                                                                                                                                                                    |
| [**XHR\_COOKIE structure**](xhr-cookie.md)<br/> | Defines a cookie that you can add to the HTTP cookie jar by calling the [**SetCookie**](ixmlhttprequest2-setcookie.md) method or retrieve from the HTTP cookie jar by calling the [**GetCookie**](ixmlhttprequest2-getcookie.md) method.<br/> |
