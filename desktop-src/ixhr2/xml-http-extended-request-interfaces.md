# XML HTTP Extended Request interfaces

XML HTTP Extended Request supports the following interfaces, which are grouped according to the inheritance hierarchy.

## In this section



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Topic</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td>[<strong>IXMLHTTPRequest2</strong>](/windows/previous-versions/msxml6/nn-msxml6-ixmlhttprequest2?branch=master)<br/></td>
<td>Provides the methods and properties needed to configure and send HTTP requests and use callbacks to receive notifications during HTTP response processing. <br/>
<blockquote>
[!Note]<br />
This interface is supported on Windows Phone 8.1.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td>[<strong>IXMLHTTPRequest2Callback</strong>](/windows/previous-versions/msxml6/nn-msxml6-ixmlhttprequest2callback?branch=master)<br/></td>
<td>Defines callbacks that notify an application with an outstanding [<strong>IXMLHTTPRequest2</strong>](/windows/previous-versions/msxml6/nn-msxml6-ixmlhttprequest2?branch=master) request of events that affect HTTP request and response processing. <br/>
<blockquote>
[!Note]<br />
This interface is supported on Windows Phone 8.1.
</blockquote>
<br/></td>
</tr>
<tr class="odd">
<td>[<strong>IXMLHTTPRequest3</strong>](/windows/previous-versions/msxml6/nn-msxml6-ixmlhttprequest3?branch=master)<br/></td>
<td>Provides the methods and properties needed to configure and send HTTP requests and use callbacks to receive notifications during HTTP response processing. Derives from the [<strong>IXMLHTTPRequest2</strong>](/windows/previous-versions/msxml6/nn-msxml6-ixmlhttprequest2?branch=master) interface. <br/>
<blockquote>
[!Note]<br />
This interface is supported on Windows Phone 8.1.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td>[<strong>IXMLHTTPRequest3Callback</strong>](/windows/previous-versions/msxml6/nn-msxml6-ixmlhttprequest3callback?branch=master)<br/></td>
<td>Defines callbacks that notify an application with an outstanding [<strong>IXMLHTTPRequest3</strong>](/windows/previous-versions/msxml6/nn-msxml6-ixmlhttprequest3?branch=master) request of events that affect HTTP request and response processing. Derives from the [<strong>IXMLHTTPRequest2Callback</strong>](/windows/previous-versions/msxml6/nn-msxml6-ixmlhttprequest2callback?branch=master) interface.
<blockquote>
[!Note]<br />
This interface is supported on Windows Phone 8.1.
</blockquote>
<br/> <br/></td>
</tr>
</tbody>
</table>
