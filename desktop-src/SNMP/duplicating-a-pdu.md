# Duplicating a PDU

The [**SnmpDuplicatePdu**](/windows/win32/Winsnmp/nf-winsnmp-snmpduplicatepdu?branch=master) function duplicates a PDU, allocating any necessary memory. To release resources allocated by **SnmpDuplicatePdu** for a new PDU, a WinSNMP application must call the [**SnmpFreePdu**](/windows/win32/Winsnmp/nf-winsnmp-snmpfreepdu?branch=master) function.
