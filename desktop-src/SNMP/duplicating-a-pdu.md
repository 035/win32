# Duplicating a PDU

The [**SnmpDuplicatePdu**](snmpduplicatepdu.md) function duplicates a PDU, allocating any necessary memory. To release resources allocated by **SnmpDuplicatePdu** for a new PDU, a WinSNMP application must call the [**SnmpFreePdu**](snmpfreepdu.md) function.
