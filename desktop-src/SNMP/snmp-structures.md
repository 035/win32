# SNMP Structures

\[SNMP is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. Instead, use [Windows Remote Management](https://msdn.microsoft.com/library/aa384426), which is the Microsoft implementation of WS-Man.\]

The following table lists structures that are used with SNMP.



| SNMP Structure                                         | Description                                                                                             |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**AsnAny**](/windows/win32/Snmp/ns-snmp-asnany?branch=master)                           | Describes the type and value of a specified SNMP variable.                                              |
| [**AsnCounter64**](/windows/win32/Snmp/?branch=master)               | Contains a 64-bit counter that is specified by two values describing its low-order and high-order bits. |
| [**AsnObjectIdentifier**](/windows/win32/Snmp/ns-snmp-asnobjectidentifier?branch=master) | Describes an object identifier (OID).                                                                   |
| [**AsnOctetString**](/windows/win32/Snmp/ns-snmp-asnoctetstring?branch=master)           | Represents a string of octets, usually in byte values.                                                  |
| [**SnmpVarBind**](/windows/win32/Snmp/ns-snmp-snmpvarbind?branch=master)                 | Describes an SNMP variable binding.                                                                     |
| [**SnmpVarBindList**](/windows/win32/Snmp/ns-snmp-snmpvarbindlist?branch=master)         | Contains a list of SNMP variable bindings.                                                              |
