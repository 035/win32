# SNMP Structures

\[SNMP is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. Instead, use [Windows Remote Management](https://msdn.microsoft.com/library/aa384426), which is the Microsoft implementation of WS-Man.\]

The following table lists structures that are used with SNMP.



| SNMP Structure                                         | Description                                                                                             |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**AsnAny**](asnany-str.md)                           | Describes the type and value of a specified SNMP variable.                                              |
| [**AsnCounter64**](asncounter64-str.md)               | Contains a 64-bit counter that is specified by two values describing its low-order and high-order bits. |
| [**AsnObjectIdentifier**](asnobjectidentifier-str.md) | Describes an object identifier (OID).                                                                   |
| [**AsnOctetString**](asnoctetstring-str.md)           | Represents a string of octets, usually in byte values.                                                  |
| [**SnmpVarBind**](snmpvarbind-str.md)                 | Describes an SNMP variable binding.                                                                     |
| [**SnmpVarBindList**](snmpvarbindlist-str.md)         | Contains a list of SNMP variable bindings.                                                              |
