# Collections

The Authorization Manager API provides interfaces that represent collections of other objects. For example, the [**IAzRoles**](iazroles.md) interface represents a collection of [**IAzRole**](iazrole.md) objects.

The other interfaces that represent collections are [**IAzApplications**](iazapplications.md), [**IAzApplicationGroups**](iazapplicationgroups.md), [**IAzOperations**](iazoperations.md), [**IAzScopes**](iazscopes.md), and [**IAzTasks**](iaztasks.md). Each of these interfaces provides properties that facilitate enumerating objects in a collection.
