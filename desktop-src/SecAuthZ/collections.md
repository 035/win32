# Collections

The Authorization Manager API provides interfaces that represent collections of other objects. For example, the [**IAzRoles**](/windows/win32/Azroles/nn-azroles-iazroles?branch=master) interface represents a collection of [**IAzRole**](/windows/win32/Azroles/nn-azroles-iazrole?branch=master) objects.

The other interfaces that represent collections are [**IAzApplications**](/windows/win32/Azroles/nn-azroles-iazapplications?branch=master), [**IAzApplicationGroups**](/windows/win32/Azroles/nn-azroles-iazapplicationgroups?branch=master), [**IAzOperations**](/windows/win32/Azroles/nn-azroles-iazoperations?branch=master), [**IAzScopes**](/windows/win32/Azroles/nn-azroles-iazscopes?branch=master), and [**IAzTasks**](/windows/win32/Azroles/nn-azroles-iaztasks?branch=master). Each of these interfaces provides properties that facilitate enumerating objects in a collection.
