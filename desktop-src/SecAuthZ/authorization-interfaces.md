# Authorization Interfaces

The following interfaces are used with authorization applications.

## In this section



| Topic                                                                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAzApplication**](/windows/win32/Azroles/nn-azroles-iazapplication?branch=master)<br/>                   | Defines an installed instance of an application. An [**IAzApplication**](/windows/win32/Azroles/nn-azroles-iazapplication?branch=master) object is created when an application is installed.<br/>                                                                                                                                                                                                                                                                                                                                           |
| [**IAzApplication2**](/windows/win32/Azroles/nn-azroles-iazapplication2?branch=master)<br/>                 | Inherits from the [**IAzApplication**](/windows/win32/Azroles/nn-azroles-iazapplication?branch=master) interface and implements additional methods to initialize [**IAzClientContext2**](/windows/win32/Azroles/nn-azroles-iazclientcontext2?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                               |
| [**IAzApplication3**](/windows/win32/Azroles/nn-azroles-iazapplication3?branch=master)<br/>                 | Provides methods to manage [**IAzRoleAssignment**](/windows/win32/Azroles/nn-azroles-iazroleassignment?branch=master), [**IAzRoleDefinition**](/windows/win32/Azroles/nn-azroles-iazroledefinition?branch=master), and [**IAzScope2**](/windows/win32/Azroles/nn-azroles-iazscope2?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                     |
| [**IAzApplications**](/windows/win32/Azroles/nn-azroles-iazapplications?branch=master)<br/>                 | Represents a collection of [**IAzApplication**](/windows/win32/Azroles/nn-azroles-iazapplication?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**IAzApplicationGroup**](/windows/win32/Azroles/nn-azroles-iazapplicationgroup?branch=master)<br/>         | Defines a collection of principals.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IAzApplicationGroup2**](/windows/win32/Azroles/nn-azroles-iazapplicationgroup2?branch=master)<br/>       | Extends the [**IAzApplicationGroup**](/windows/win32/Azroles/nn-azroles-iazapplicationgroup?branch=master) interface by adding support for the **BizRule** group type.<br/>                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzApplicationGroups**](/windows/win32/Azroles/nn-azroles-iazapplicationgroups?branch=master)<br/>       | Represents a collection of [**IAzApplicationGroup**](/windows/win32/Azroles/nn-azroles-iazapplicationgroup?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IAzAuthorizationStore**](/windows/win32/Azroles/nn-azroles-iazauthorizationstore?branch=master)<br/>      | Defines the container that is the root of the authorization policy store.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**IAzAuthorizationStore2**](/windows/win32/Azroles/nn-azroles-iazauthorizationstore2?branch=master)<br/>   | Inherits from the [**AzAuthorizationStore**](/windows/win32/Azroles/nn-azroles-iazauthorizationstore?branch=master) object and implements methods to create and open [**IAzApplication2**](/windows/win32/Azroles/nn-azroles-iazapplication2?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                |
| [**IAzAuthorizationStore3**](/windows/win32/Azroles/nn-azroles-iazauthorizationstore3?branch=master)<br/>   | Extends the [**IAzAuthorizationStore2**](/windows/win32/Azroles/nn-azroles-iazauthorizationstore2?branch=master) interface with methods that manage business rule (BizRule) support and caching.<br/>                                                                                                                                                                                                                                                                                                                                       |
| [**IAzBizRuleContext**](/windows/win32/Azroles/nn-azroles-iazbizrulecontext?branch=master)<br/>              | Contains information about a Business Rule (BizRule) operation.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**IAzBizRuleInterfaces**](/windows/win32/Azroles/nn-azroles-iazbizruleinterfaces?branch=master)<br/>       | Provides methods and properties used to manage a list of [**IDispatch**](ebbff4bc-36b2-4861-9efa-ffa45e013eb5) interfaces that can be called by business rule (BizRule) scripts.<br/>                                                                                                                                                                                                                                                                                                           |
| [**IAzBizRuleParameters**](/windows/win32/Azroles/nn-azroles-iazbizruleparameters?branch=master)<br/>       | Provides methods and properties used to manage a list of parameters that can be passed to business rule (BizRule) scripts.<br/>                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzClientContext**](/windows/win32/Azroles/nn-azroles-iazclientcontext?branch=master)<br/>               | Maintains the state that describes a particular client.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**IAzClientContext2**](/windows/win32/Azroles/nn-azroles-iazclientcontext2?branch=master)<br/>             | Inherits from the [**IAzClientContext**](/windows/win32/Azroles/nn-azroles-iazclientcontext?branch=master) interface and implements new methods that manipulate the client context.<br/>                                                                                                                                                                                                                                                                                                                                                    |
| [**IAzClientContext3**](/windows/win32/Azroles/nn-azroles-iazclientcontext3?branch=master)<br/>             | Extends the [**IAzClientContext2**](/windows/win32/Azroles/nn-azroles-iazclientcontext2?branch=master) interface.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IAzNameResolver**](/windows/win32/Azroles/nn-azroles-iaznameresolver?branch=master)<br/>                 | Translates [security identifiers](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-security-identifier-gly) (SIDs) into principal display names.<br/>                                                                                                                                                                                                                                                                                                                                 |
| [**IAzObjectPicker**](/windows/win32/Azroles/nn-azroles-iazobjectpicker?branch=master)<br/>                 | Displays a dialog box that allows users to select one or more principals from a list.<br/>                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IAzOperation**](/windows/win32/Azroles/nn-azroles-iazoperation?branch=master)<br/>                       | Defines a low-level operation supported by an application.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzOperation2**](/windows/win32/Azroles/nn-azroles-iazoperation2?branch=master)<br/>                     | Extends the [**IAzOperation**](/windows/win32/Azroles/nn-azroles-iazoperation?branch=master) with a method that returns the role assignments associated with the operation.<br/>                                                                                                                                                                                                                                                                                                                                                            |
| [**IAzOperations**](/windows/win32/Azroles/nn-azroles-iazoperations?branch=master)<br/>                     | Represents a collection of [**IAzOperation**](/windows/win32/Azroles/nn-azroles-iazoperation?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**IAzPrincipalLocator**](/windows/win32/Azroles/nn-azroles-iazprincipallocator?branch=master)<br/>         | Locates and chooses Active Directory Application Mode (ADAM) principals in Authorization Manager.<br/>                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IAzRole**](/windows/win32/Azroles/nn-azroles-iazrole?branch=master)<br/>                                 | Defines the set of operations that can be performed by a set of users within a scope.<br/>                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IAzRoleAssignment**](/windows/win32/Azroles/nn-azroles-iazroleassignment?branch=master)<br/>             | Represents a role to which users and groups can be assigned.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**IAzRoleAssignments**](/windows/win32/Azroles/nn-azroles-iazroleassignments?branch=master)<br/>           | Represents a collection of [**IAzRoleAssignment**](/windows/win32/Azroles/nn-azroles-iazroleassignment?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**IAzRoleDefinition**](/windows/win32/Azroles/nn-azroles-iazroledefinition?branch=master)<br/>             | Represents one or more [**IAzRoleDefinition**](/windows/win32/Azroles/nn-azroles-iazroledefinition?branch=master), [**IAzTask**](/windows/win32/Azroles/nn-azroles-iaztask?branch=master), and [**IAzOperation**](/windows/win32/Azroles/nn-azroles-iazoperation?branch=master) objects that specify a set of operations.<br/>                                                                                                                                                                                                                                                                                                      |
| [**IAzRoleDefinitions**](/windows/win32/Azroles/nn-azroles-iazroledefinitions?branch=master)<br/>           | Represents a collection of [**IAzRoleDefinition**](/windows/win32/Azroles/nn-azroles-iazroledefinition?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**IAzRoles**](/windows/win32/Azroles/nn-azroles-iazroles?branch=master)<br/>                               | Represents a collection of [**IAzRole**](/windows/win32/Azroles/nn-azroles-iazrole?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IAzScope**](/windows/win32/Azroles/nn-azroles-iazscope?branch=master)<br/>                               | Defines a logical container of resources to which the application manages access.<br/>                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IAzScope2**](/windows/win32/Azroles/nn-azroles-iazscope2?branch=master)<br/>                             | Extends the [**IAzScope**](/windows/win32/Azroles/nn-azroles-iazscope?branch=master) interface to manage [**IAzRoleAssignment**](/windows/win32/Azroles/nn-azroles-iazroleassignment?branch=master) and [**IAzRoleDefinition**](/windows/win32/Azroles/nn-azroles-iazroledefinition?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                    |
| [**IAzScopes**](/windows/win32/Azroles/nn-azroles-iazscopes?branch=master)<br/>                             | Represents a collection of [**IAzScope**](/windows/win32/Azroles/nn-azroles-iazscope?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IAzTask**](/windows/win32/Azroles/nn-azroles-iaztask?branch=master)<br/>                                 | Describes a set of operations.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IAzTask2**](/windows/win32/Azroles/nn-azroles-iaztask2?branch=master)<br/>                               | Extends the [**IAzTask**](/windows/win32/Azroles/nn-azroles-iaztask?branch=master) interface with a method that returns the role assignments associated with the task.<br/>                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzTasks**](/windows/win32/Azroles/nn-azroles-iaztasks?branch=master)<br/>                               | Represents a collection of [**IAzTask**](/windows/win32/Azroles/nn-azroles-iaztask?branch=master) objects.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IeAxiService**](ieaxiservice.md)<br/>                       | Initializes a system service object to install an ActiveX object when the current user does not have permission to install the object.<br/>                                                                                                                                                                                                                                                                                                                                                     |
| [**IeAxiServiceCallback**](ieaxiservicecallback.md)<br/>       | Called by the [**IeAxiSystemInstaller**](ieaxisysteminstaller.md) interface to verify that an ActiveX object can be installed.<br/>                                                                                                                                                                                                                                                                                                                                                            |
| [**IeAxiSystemInstaller**](ieaxisysteminstaller.md)<br/>       | Installs an ActiveX object.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**IEffectivePermission**](/windows/win32/Aclui/?branch=master)<br/>       | Provides a means to determine effective permission for a security principal on an object.<br/>                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**IEffectivePermission2**](/windows/win32/Aclui/?branch=master)<br/>     | Provides a way to determine effective permission for a security principal on an object.<br/>                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISecurityInformation**](/windows/win32/Aclui/?branch=master)<br/>       | Enables the access control editor to communicate with the caller of the [**CreateSecurityPage**](/windows/win32/Aclui/nf-aclui-createsecuritypage?branch=master) and [**EditSecurity**](/windows/win32/Aclui/nf-aclui-editsecurity?branch=master) functions.<br/>                                                                                                                                                                                                                                                                                                               |
| [**ISecurityInformation2**](/windows/win32/Aclui/?branch=master)<br/>     | Enables the access control editor to obtain information from the client that is not provided by the [**ISecurityInformation**](/windows/win32/Aclui/?branch=master) interface.<br/>                                                                                                                                                                                                                                                                                                                        |
| [**ISecurityInformation3**](/windows/win32/Aclui/?branch=master)<br/>     | Provides methods necessary for displaying an elevated access control editor when a user clicks the **Edit** button on an access control editor page that displays an image of a shield on that **Edit** button.<br/>                                                                                                                                                                                                                                                                            |
| [**ISecurityInformation4**](/windows/win32/Aclui/?branch=master)<br/>     | Enables the access control editor (ACE) to obtain the share's security descriptor to initialize the share page.<br/>                                                                                                                                                                                                                                                                                                                                                                            |
| [**ISecurityObjectTypeInfo**](/windows/win32/Aclui/?branch=master)<br/> | Provides a means of determining the source of inherited [access control entries](https://msdn.microsoft.com/library/windows/desktop/ms721532#-security-access-control-entry-gly) (ACEs) in [discretionary access control lists](https://msdn.microsoft.com/library/windows/desktop/ms721573#-security-discretionary-access-control-list-gly) (DACLs) and [system access control lists](https://msdn.microsoft.com/library/windows/desktop/ms721625#-security-system-access-control-list-gly) (SACLs).<br/> |



 

Authorization interfaces are categorized according to usage as follows:
