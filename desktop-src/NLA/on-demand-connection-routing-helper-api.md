# On Demand Connection Routing Helper API reference

The following topics provide details for the On Demand Connection Routing Helper.



| Reference                                                                          | Description                                                                                                                                                                 |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OnDemandGetRoutingHint**](ondemandgetroutinghint.md)                           | Look up a destination in the Route Request cache and, if a match is found, returns the corresponding Interface ID.<br/>                                               |
| [**OnDemandRegisterNotification**](ondemandregisternotification.md)               | Register to be notified when the Route Requests cache is modified.<br/>                                                                                               |
| [**OnDemandUnregisterNotification**](ondemandunregisternotification.md)           | Unregister for notifications and clean up resources.<br/>                                                                                                             |
| [**NET\_INTERFACE\_CONTEXT**](net-interface-context.md)                           | The interface context that is part of the [**NET\_INTERFACE\_CONTEXT\_TABLE**](net-interface-context-table.md) structure.<br/>                                       |
| [**NET\_INTERFACE\_CONTEXT\_TABLE**](net-interface-context-table.md)              | The table of [**NET\_INTERFACE\_CONTEXT**](net-interface-context.md) structures.<br/>                                                                                |
| [**GetInterfaceContextTableForHostName**](getinterfacecontexttableforhostname.md) | This function retrieves an interface context table for the given hostname and connection profile filter.<br/>                                                         |
| [**FreeInterfaceContextTable**](freeinterfacecontexttable.md)                     | This function frees the interface context table retrieved using the [**GetInterfaceContextTableForHostName**](getinterfacecontexttableforhostname.md) function.<br/> |
