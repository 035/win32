# NLM Structures

The following structures support the Network List Manager API:



| Structure                                                           | Description                                                          |
|---------------------------------------------------------------------|----------------------------------------------------------------------|
| [**NLM\_SIMULATED\_PROFILE\_INFO**](nlm-simulated-profile-info.md) | Represents profile information to be used during cost simulation.    |
| [**NLM\_SOCKADDR**](nlm-sockaddr.md)                               | Represents the IPv4/IPv6 destination address.                        |
| [**NLM\_USAGE\_DATA**](nlm-usage-data.md)                          | Represents information that indicates the data usage of a data plan. |
| [**NLM\_DATA\_PLAN\_STATUS**](nla-nlm_data_plan_status)             | Represents the current status of a data plan.                        |
