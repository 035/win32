# NLM Structures

The following structures support the Network List Manager API:



| Structure                                                           | Description                                                          |
|---------------------------------------------------------------------|----------------------------------------------------------------------|
| [**NLM\_SIMULATED\_PROFILE\_INFO**](/windows/win32/Netlistmgr/ns-netlistmgr-nlm_simulated_profile_info?branch=master) | Represents profile information to be used during cost simulation.    |
| [**NLM\_SOCKADDR**](/windows/win32/Netlistmgr/ns-netlistmgr-nlm_sockaddr?branch=master)                               | Represents the IPv4/IPv6 destination address.                        |
| [**NLM\_USAGE\_DATA**](/windows/win32/Netlistmgr/ns-netlistmgr-nlm_usage_data?branch=master)                          | Represents information that indicates the data usage of a data plan. |
| [**NLM\_DATA\_PLAN\_STATUS**](nla-nlm_data_plan_status)             | Represents the current status of a data plan.                        |
