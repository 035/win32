# NLM Enumerations

The following are the NLM enumerations.

| Enumeration                                                                 | Description                                                                                |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**NLM\_CONNECTION\_COST**](nlm-connection-cost.md)                        | Specifies a set of cost levels and cost flags supported in Windows 8 Cost APIs.            |
| [**NLM\_CONNECTION\_PROPERTY\_CHANGE**](nlm-connection-property-change.md) | Defines changes made to the properties of a network connection.                            |
| [**NLM\_CONNECTIVITY**](nlm-connectivity.md)                               | Provides notification whenever connectivity related parameters have changed.               |
| [**NLM\_DOMAIN\_TYPE**](nlm-domain-type.md)                                | Specifies the domain type of a network.                                                    |
| [**NLM\_ENUM\_NETWORK**](nlm-enum-network.md)                              | Specifies what types of networks are enumerated.                                           |
| [**NLM\_INTERNET\_CONNECTIVITY**](nlm-internet-connectivity.md)            | Defines a set of flags that provide additional data for IPv4 or IPv6 network connectivity. |
| [**NLM\_NETWORK\_CLASS**](nlm-network-class.md)                            | Defines a set of flags that specify if a network has been identified.                      |
| [**NLM\_NETWORK\_CATEGORY**](nlm-network-category.md)                      | Specifies the category type of a network.                                                  |
| [**NLM\_NETWORK\_PROPERTY\_CHANGE**](nlm-network-property-change.md)       | Defines changes made to the properties of a network.                                       |
