# NLM Enumerations

The following are the NLM enumerations.

| Enumeration                                                                 | Description                                                                                |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**NLM\_CONNECTION\_COST**](/windows/win32/Netlistmgr/ne-netlistmgr-nlm_connection_cost?branch=master)                        | Specifies a set of cost levels and cost flags supported in Windows 8 Cost APIs.            |
| [**NLM\_CONNECTION\_PROPERTY\_CHANGE**](/windows/win32/Netlistmgr/ne-netlistmgr-nlm_connection_property_change?branch=master) | Defines changes made to the properties of a network connection.                            |
| [**NLM\_CONNECTIVITY**](/windows/win32/Netlistmgr/ne-netlistmgr-nlm_connectivity?branch=master)                               | Provides notification whenever connectivity related parameters have changed.               |
| [**NLM\_DOMAIN\_TYPE**](/windows/win32/Netlistmgr/ne-netlistmgr-nlm_domain_type?branch=master)                                | Specifies the domain type of a network.                                                    |
| [**NLM\_ENUM\_NETWORK**](/windows/win32/Netlistmgr/ne-netlistmgr-nlm_enum_network?branch=master)                              | Specifies what types of networks are enumerated.                                           |
| [**NLM\_INTERNET\_CONNECTIVITY**](/windows/win32/Netlistmgr/ne-netlistmgr-nlm_internet_connectivity?branch=master)            | Defines a set of flags that provide additional data for IPv4 or IPv6 network connectivity. |
| [**NLM\_NETWORK\_CLASS**](/windows/win32/Netlistmgr/ne-netlistmgr-nlm_network_class?branch=master)                            | Defines a set of flags that specify if a network has been identified.                      |
| [**NLM\_NETWORK\_CATEGORY**](/windows/win32/Netlistmgr/ne-netlistmgr-nlm_network_category?branch=master)                      | Specifies the category type of a network.                                                  |
| [**NLM\_NETWORK\_PROPERTY\_CHANGE**](/windows/win32/Netlistmgr/ne-netlistmgr-nlm_network_property_change?branch=master)       | Defines changes made to the properties of a network.                                       |
