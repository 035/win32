# Data Deduplication API

## Purpose

The Data Deduplication API enables backup applications to implement their own optimized backup and restore methods that take advantage of the Data Deduplication feature. It also enables the creation of WMI scripts for managing the Data Deduplication feature.

## Developer audience

The Data Deduplication API is designed for C and C++ developers. Familiarity with COM and client/server architecture is required.

## Run-time requirements

The Data Deduplication API is included in Windows Server 2012 and later.
