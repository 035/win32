# Data Deduplication Management WMI API Reference

## In this section

<dl> <dt>

[**MSFT\_DedupFileMetadata**](msft-dedupfilemetadata.md)
</dt> <dd>

Represents a file that has data deduplication metadata.

</dd> <dt>

[**MSFT\_DedupJob**](msft-dedupjob.md)
</dt> <dd>

Represents a currently active data deduplication job.

</dd> <dt>

[**MSFT\_DedupJobSchedule**](msft-dedupjobschedule.md)
</dt> <dd>

Represents a data deduplication schedule.

</dd> <dt>

[**MSFT\_DedupVolume**](msft-dedupvolume.md)
</dt> <dd>

Represents a volume that has data deduplication metadata.

</dd> <dt>

[**MSFT\_DedupVolumeMetadata**](msft-dedupvolumemetadata.md)
</dt> <dd>

Represents the data deduplication metadata for a volume.

</dd> <dt>

[**MSFT\_DedupVolumeStatus**](msft-dedupvolumestatus.md)
</dt> <dd>

Represents the data deduplication status for a volume.

</dd> </dl>
