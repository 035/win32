# CertEnroll Enumerations

The following enumerations are supported by the Certificate Enrollment API.

| Enumeration                                                                              | Description                                                                                                                                                                                                                                                                                                              |
|------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AlgorithmFlags**](/windows/win32/CertEnroll/ne-certenroll-algorithmflags?branch=master)                                            | Contains flags that can be used to refine the search for a cryptographic algorithm.                                                                                                                                                                                                                                      |
| [**AlgorithmOperationFlags**](/windows/win32/CertEnroll/ne-certenroll-algorithmoperationflags?branch=master)                          | Specifies the operations that an algorithm can perform. This is used to retrieve the operational capabilities of a CSP algorithm or CSP status information based on those capabilities.                                                                                                                                  |
| [**AlgorithmType**](/windows/win32/CertEnroll/ne-certenroll-algorithmtype?branch=master)                                              | Specifies the intended purpose of a cryptographic algorithm.                                                                                                                                                                                                                                                             |
| [**AlternativeNameType**](/windows/win32/CertEnroll/ne-certenroll-alternativenametype?branch=master)                                  | Specifies the alternative name types that can be specified when creating an **AlternativeName** extension.                                                                                                                                                                                                               |
| [**CERTENROLL\_OBJECTID**](/windows/win32/CertEnroll/ne-certenroll-certenroll_objectid?branch=master)                                 | Contains the predefined [*object identifiers*](https://msdn.microsoft.com/library/windows/desktop/ms721599#-security-object-identifier-gly) (OIDs) supported the Certificate Enrollment API.                                                                                                                                          |
| [**CERTENROLL\_PROPERTYID**](/windows/win32/CertEnroll/ne-certenroll-certenroll_propertyid?branch=master)                             | Contains predefined OIDs for external properties supported by the Certificate Enrollment API.                                                                                                                                                                                                                            |
| [**CommitTemplateFlags**](/windows/win32/Certenroll/ne-certenroll-committemplateflags?branch=master)                                       | Specifies options for saving and deleting templates.                                                                                                                                                                                                                                                                     |
| [**EncodingType**](/windows/win32/CertEnroll/ne-certenroll-encodingtype?branch=master)                                                | Specifies the type of encoding applied to a byte array for display purposes.                                                                                                                                                                                                                                             |
| [**EnrollmentCAProperty**](/windows/win32/Certenroll/ne-certenroll-enrollmentcaproperty?branch=master)                                     | Specifies certification authority property values.                                                                                                                                                                                                                                                                       |
| [**EnrollmentDisplayStatus**](/windows/win32/CertEnroll/ne-certenroll-enrollmentdisplaystatus?branch=master)                          | Specifies whether to display enrollment status information in a user interface.                                                                                                                                                                                                                                          |
| [**EnrollmentEnrollStatus**](/windows/win32/CertEnroll/ne-certenroll-enrollmentenrollstatus?branch=master)                            | Specifies the enrollment status of a certificate request.                                                                                                                                                                                                                                                                |
| [**EnrollmentPolicyFlags**](/windows/win32/Certenroll/ne-certenroll-enrollmentpolicyflags?branch=master)                                   | Specifies group policy flags.                                                                                                                                                                                                                                                                                            |
| [**EnrollmentPolicyServerPropertyFlags**](/windows/win32/Certenroll/ne-certenroll-enrollmentpolicyserverpropertyflags?branch=master)       | Specifies the default policy server.                                                                                                                                                                                                                                                                                     |
| [**EnrollmentSelectionStatus**](/windows/win32/CertEnroll/ne-certenroll-enrollmentselectionstatus?branch=master)                      | Specifies whether the enrollment status of an object will be monitored during the enrollment process.                                                                                                                                                                                                                    |
| [**EnrollmentTemplateProperty**](/windows/win32/Certenroll/ne-certenroll-enrollmenttemplateproperty?branch=master)                         | Contains property values for a given template.                                                                                                                                                                                                                                                                           |
| [**InnerRequestLevel**](/windows/win32/CertEnroll/ne-certenroll-innerrequestlevel?branch=master)                                      | Specifies the containment level of a certificate request within a PKCS \#7 or CMC request.                                                                                                                                                                                                                               |
| [**InstallResponseRestrictionFlags**](/windows/win32/CertEnroll/ne-certenroll-installresponserestrictionflags?branch=master)          | Contains flags that identify the restrictions placed on the local installation of a certificate chain.                                                                                                                                                                                                                   |
| [**KeyIdentifierHashAlgorithm**](/windows/win32/CertEnroll/ne-certenroll-keyidentifierhashalgorithm?branch=master)                    | Specifies the algorithm used to hash the public key in a certificate request.                                                                                                                                                                                                                                            |
| [**ObjectIdGroupId**](/windows/win32/CertEnroll/ne-certenroll-objectidgroupid?branch=master)                                          | Specifies the category or group to which an OID belongs.                                                                                                                                                                                                                                                                 |
| [**ObjectIdPublicKeyFlags**](/windows/win32/CertEnroll/ne-certenroll-objectidpublickeyflags?branch=master)                            | Specifies whether a public key algorithm is used for signing or for encryption.                                                                                                                                                                                                                                          |
| [**PFXExportOptions**](/windows/win32/CertEnroll/ne-certenroll-pfxexportoptions?branch=master)                                        | Specifies how much of a certificate chain is included when creating a Personal Information Exchange (PFX) message.                                                                                                                                                                                                       |
| [**Pkcs10AllowedSignatureTypes**](/windows/win32/CertEnroll/ne-certenroll-pkcs10allowedsignaturetypes?branch=master)                  | Specifies the type of signature permitted when signing a certificate request.                                                                                                                                                                                                                                            |
| [**PolicyQualifierType**](/windows/win32/CertEnroll/ne-certenroll-policyqualifiertype?branch=master)                                  | Specifies the type of qualifier applied to a certificate policy.                                                                                                                                                                                                                                                         |
| [**PolicyServerUrlFlags**](/windows/win32/Certenroll/ne-certenroll-policyserverurlflags?branch=master)                                     | Contains certificate enrollment policy (CEP) server flags.                                                                                                                                                                                                                                                               |
| [**PolicyServerUrlPropertyID**](/windows/win32/Certenroll/ne-certenroll-policyserverurlpropertyid?branch=master)                           | Contains values that specify the type of property value to be returned by the [**GetStringProperty**](/windows/win32/Certenroll/nf-certenroll-ix509policyserverurl-getstringproperty?branch=master) method or set by the [**SetStringProperty**](/windows/win32/Certenroll/nf-certenroll-ix509policyserverurl-setstringproperty?branch=master) method on the [**IX509PolicyServerUrl**](/windows/win32/Certenroll/nn-certenroll-ix509policyserverurl?branch=master) interface. |
| [**RequestClientInfoClientId**](/windows/win32/CertEnroll/ne-certenroll-requestclientinfoclientid?branch=master)                      | Specifies the type of application that created a certificate request.                                                                                                                                                                                                                                                    |
| [**WebEnrollmentFlags**](/windows/win32/Certenroll/ne-certenroll-webenrollmentflags?branch=master)                                         | Specifies web enrollment behavior.                                                                                                                                                                                                                                                                                       |
| [**WebSecurityLevel**](/windows/win32/CertEnroll/ne-certenroll-websecuritylevel?branch=master)                                        | Specifies whether a web-enabled method or property is safe for scripting.                                                                                                                                                                                                                                                |
| [**X500NameFlags**](/windows/win32/CertEnroll/ne-certenroll-x500nameflags?branch=master)                                              | Specifies the display and encoding characteristics of a distinguished name or relative distinguished name.                                                                                                                                                                                                               |
| [**X509CertificateEnrollmentContext**](/windows/win32/CertEnroll/ne-certenroll-x509certificateenrollmentcontext?branch=master)        | Specifies the nature of the end entity for which the certificate is intended.                                                                                                                                                                                                                                            |
| [**X509CertificateTemplateEnrollmentFlag**](/windows/win32/CertEnroll/ne-certenroll-x509certificatetemplateenrollmentflag?branch=master)   | Contains values that specify server and client actions during enrollment.                                                                                                                                                                                                                                                |
| [**X509CertificateTemplateGeneralFlag**](/windows/win32/CertEnroll/ne-certenroll-x509certificatetemplategeneralflag?branch=master)         | Contains use and modification information about templates and associated certificates.                                                                                                                                                                                                                                   |
| [**X509CertificateTemplatePrivateKeyFlag**](/windows/win32/CertEnroll/ne-certenroll-x509certificatetemplateprivatekeyflag?branch=master)   | Contains values that specify client actions regarding a private key.                                                                                                                                                                                                                                                     |
| [**X509CertificateTemplateSubjectNameFlag**](/windows/win32/CertEnroll/ne-certenroll-x509certificatetemplatesubjectnameflag?branch=master) | Contains values that specify server and client actions concerning subject names.                                                                                                                                                                                                                                         |
| [**X509EnrollmentPolicyExportFlags**](/windows/win32/Certenroll/ne-certenroll-x509enrollmentpolicyexportflags?branch=master)               | Specifies what items to export from the policy server.                                                                                                                                                                                                                                                                   |
| [**X509EnrollmentPolicyLoadOption**](/windows/win32/Certenroll/ne-certenroll-x509enrollmentpolicyloadoption?branch=master)                 | Specifies how to retrieve policy from the policy server.                                                                                                                                                                                                                                                                 |
| [**X509KeySpec**](/windows/win32/CertEnroll/ne-certenroll-x509keyspec?branch=master)                                                  | Specifies the intended use of a key for a legacy CSP.                                                                                                                                                                                                                                                                    |
| [**X509KeyUsageFlags**](/windows/win32/CertEnroll/ne-certenroll-x509keyusageflags?branch=master)                                      | Specifies the purpose of a key contained in a certificate.                                                                                                                                                                                                                                                               |
| [**X509PrivateKeyExportFlags**](/windows/win32/CertEnroll/ne-certenroll-x509privatekeyexportflags?branch=master)                           | Specifies the export policy for a private key.                                                                                                                                                                                                                                                                           |
| [**X509PrivateKeyProtection**](/windows/win32/CertEnroll/ne-certenroll-x509privatekeyprotection?branch=master)                             | Specifies the level of private key protection supported by a KSP or CSP.                                                                                                                                                                                                                                                 |
| [**X509PrivateKeyUsageFlags**](/windows/win32/CertEnroll/ne-certenroll-x509privatekeyusageflags?branch=master)                             | Specifies the permitted uses of a private key.                                                                                                                                                                                                                                                                           |
| [**X509PrivateKeyVerify**](/windows/win32/CertEnroll/ne-certenroll-x509privatekeyverify?branch=master)                                     | Specifies whether a user interface is displayed during private key verification and whether verification can proceed if the CSP or KSP represent a smart card.                                                                                                                                                           |
| [**X509ProviderType**](/windows/win32/CertEnroll/ne-certenroll-x509providertype?branch=master)                                        | Specifies the type of a cryptographic service provider or key service provider.                                                                                                                                                                                                                                          |
| [**X509RequestInheritOptions**](/windows/win32/CertEnroll/ne-certenroll-x509requestinheritoptions?branch=master)                      | Specifies how keys, extension values, and external properties are inherited when a new request is created from an existing certificate.                                                                                                                                                                                  |
| [**X509RequestType**](/windows/win32/CertEnroll/ne-certenroll-x509requesttype?branch=master)                                          | Specifies the certificate request type.                                                                                                                                                                                                                                                                                  |
| [**X509SCEPDisposition**](/windows/win32/certpol/ne-certpol-x509scepdisposition?branch=master)                                       | Describes the resulting disposition of a request to process a response message.                                                                                                                                                                                                                                          |
| [**X509SCEPFailInfo**](/windows/win32/certpol/ne-certpol-x509scepfailinfo?branch=master)                                             | Describes the nature of an SCEP certificate enrollment failure.                                                                                                                                                                                                                                                          |
