# Cryptographic Provider Interfaces

The following interfaces can be used to retrieve information about cryptographic providers.



| Interface                                    | Description                                                                  |
|----------------------------------------------|------------------------------------------------------------------------------|
| [**ICspAlgorithm**](icspalgorithm.md)       | Represents an algorithm implemented by a cryptographic provider.             |
| [**ICspAlgorithms**](icspalgorithms.md)     | Manages a collection of [**ICspAlgorithm**](ix509namevaluepair.md) objects. |
| [**ICspInformation**](icspinformation.md)   | Provides access to general information about a cryptographic provider.       |
| [**ICspInformations**](icspinformations.md) | Manages a collection of [**ICspInformation**](icspinformation.md) objects.  |
| [**ICspStatus**](icspstatus.md)             | Contains information about a cryptographic provider/algorithm pair.          |
| [**ICspStatuses**](icspstatuses.md)         | Manages a collection of [**ICspStatus**](icspstatus.md) objects.            |
