# IObjectIds Methods

The [**IObjectIds**](iobjectids.md) interface exposes the following methods.

## In this section



| Topic                                                            | Description                                                                                     |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**Add Method**](iobjectids-add-method.md)<br/>           | adds an [**IObjectId**](iobjectid.md) object to the collection.<br/>                     |
| [**AddRange Method**](iobjectids-addrange-method.md)<br/> | adds a range of [**IObjectId**](iobjectid.md) objects to the collection.<br/>            |
| [**Clear Method**](iobjectids-clear-method.md)<br/>       | removes all [**IObjectId**](iobjectid.md) objects from the collection.<br/>              |
| [**Remove Method**](iobjectids-remove-method.md)<br/>     | removes an [**IObjectId**](iobjectid.md) object from the collection by index value.<br/> |
