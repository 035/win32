# IObjectIds Methods

The [**IObjectIds**](/windows/win32/CertEnroll/nn-certenroll-iobjectids?branch=master) interface exposes the following methods.

## In this section



| Topic                                                            | Description                                                                                     |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**Add Method**](/windows/win32/CertEnroll/nf-certenroll-iobjectids-add?branch=master)<br/>           | adds an [**IObjectId**](/windows/win32/CertEnroll/nn-certenroll-iobjectid?branch=master) object to the collection.<br/>                     |
| [**AddRange Method**](/windows/win32/CertEnroll/nf-certenroll-iobjectids-addrange?branch=master)<br/> | adds a range of [**IObjectId**](/windows/win32/CertEnroll/nn-certenroll-iobjectid?branch=master) objects to the collection.<br/>            |
| [**Clear Method**](/windows/win32/CertEnroll/nf-certenroll-iobjectids-clear?branch=master)<br/>       | removes all [**IObjectId**](/windows/win32/CertEnroll/nn-certenroll-iobjectid?branch=master) objects from the collection.<br/>              |
| [**Remove Method**](/windows/win32/CertEnroll/nf-certenroll-iobjectids-remove?branch=master)<br/>     | removes an [**IObjectId**](/windows/win32/CertEnroll/nn-certenroll-iobjectid?branch=master) object from the collection by index value.<br/> |
