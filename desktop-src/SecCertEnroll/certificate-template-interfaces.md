# Certificate Template Interfaces

The following interfaces can be used to programmatically access, modify, and save certificate request templates.



| Interface                                                                    | Description                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**IX509CertificateTemplate**](/windows/win32/Certenroll/nn-certenroll-ix509certificatetemplate?branch=master)                 | Represents a certificate request template.                                                    |
| [**IX509CertificateTemplates**](/windows/win32/Certenroll/nn-certenroll-ix509certificatetemplates?branch=master)               | Manages a collection of [**IX509CertificateTemplate**](/windows/win32/Certenroll/nn-certenroll-ix509certificatetemplate?branch=master) objects. |
| [**IX509CertificateTemplateWritable**](/windows/win32/Certenroll/nn-certenroll-ix509certificatetemplatewritable?branch=master) | Adds a template to or deletes it from a template store.                                       |
