# Certificate Template Interfaces

The following interfaces can be used to programmatically access, modify, and save certificate request templates.



| Interface                                                                    | Description                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**IX509CertificateTemplate**](ix509certificatetemplate.md)                 | Represents a certificate request template.                                                    |
| [**IX509CertificateTemplates**](ix509certificatetemplates.md)               | Manages a collection of [**IX509CertificateTemplate**](ix509certificatetemplate.md) objects. |
| [**IX509CertificateTemplateWritable**](ix509certificatetemplatewritable.md) | Adds a template to or deletes it from a template store.                                       |
