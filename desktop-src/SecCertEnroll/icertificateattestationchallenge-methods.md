# ICertificateAttestationChallenge Methods

The [**ICertificateAttestationChallenge**](icertificateattestationchallenge.md) interface supports the following methods.

## In this section



| Topic                                                                                           | Description                                                                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DecryptChallenge method**](icertificateattestationchallenge-decryptchallenge.md)<br/> | Decrypts the challenge from the [Certificate Management over CMS](https://msdn.microsoft.com/library/windows/desktop/ms721572#-security-certificate-management-over-cms-gly) (CMC) response and creates a re-encrypted response to send to the CA.<br/> |
| [**Initialize method**](icertificateattestationchallenge-initialize.md)<br/>             | Initialize using the full [Certificate Management over CMS](https://msdn.microsoft.com/library/windows/desktop/ms721572#-security-certificate-management-over-cms-gly) (CMC) response returned from the CA.<br/>                                        |
