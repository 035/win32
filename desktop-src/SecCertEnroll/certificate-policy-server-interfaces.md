# Certificate Policy Server Interfaces

The following interfaces can be used to programmatically configure a certificate enrollment policy (CEP) server.



| Interface                                                            | Description                                                                                                   |
|----------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**IX509EnrollmentPolicyServer**](/windows/win32/Certenroll/nn-certenroll-ix509enrollmentpolicyserver?branch=master)   | Represents a CEP server.                                                                                      |
| [**IX509PolicyServerListManager**](/windows/win32/CertEnroll/nn-certenroll-ix509policyserverlistmanager?branch=master) | Manages a collection of [**IX509PolicyServerUrl**](/windows/win32/Certenroll/nn-certenroll-ix509policyserverurl?branch=master) objects.                         |
| [**IX509PolicyServerUrl**](/windows/win32/Certenroll/nn-certenroll-ix509policyserverurl?branch=master)                 | Specifies or retrieves property values associated with the CEP server and updates associated registry values. |
