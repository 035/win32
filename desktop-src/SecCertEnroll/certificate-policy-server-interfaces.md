# Certificate Policy Server Interfaces

The following interfaces can be used to programmatically configure a certificate enrollment policy (CEP) server.



| Interface                                                            | Description                                                                                                   |
|----------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**IX509EnrollmentPolicyServer**](ix509enrollmentpolicyserver.md)   | Represents a CEP server.                                                                                      |
| [**IX509PolicyServerListManager**](ix509policyserverlistmanager.md) | Manages a collection of [**IX509PolicyServerUrl**](ix509policyserverurl.md) objects.                         |
| [**IX509PolicyServerUrl**](ix509policyserverurl.md)                 | Specifies or retrieves property values associated with the CEP server and updates associated registry values. |
