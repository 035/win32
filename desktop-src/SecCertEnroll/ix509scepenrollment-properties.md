# IX509SCEPEnrollment Properties

The [**IX509SCEPEnrollment**](/windows/win32/Certenroll/nn-certenroll-ix509scepenrollment?branch=master) interface exposes the following properties.

## In this section



| Topic                                                                                              | Description                                                                                                                                            |
|----------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Certificate property**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-get_certificate?branch=master)<br/>                         | Gets the certificate for the request.<br/>                                                                                                       |
| [**CertificateFriendlyName property**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-get_certificatefriendlyname?branch=master)<br/> | Gets or sets the friendly name for the certificate.<br/>                                                                                         |
| [**FailInfo property**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-get_failinfo?branch=master)<br/>                               | Gets information when the [**ProcessResponseMessage**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-processresponsemessage?branch=master) method detects a failed environment.<br/> |
| [**OldCertificate property**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-get_oldcertificate?branch=master)<br/>                   | Gets or sets an old certificate that a request is intended to replace.<br/>                                                                      |
| [**Request property**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-get_request?branch=master)<br/>                                 | Gets the inner PKCS10 request.<br/>                                                                                                              |
| [**ServerCapabilities property**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-put_servercapabilities?branch=master)<br/>           | Sets the preferred hash and encryption algorithms for the request.<br/>                                                                          |
| [**SignerCertificate property**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-get_signercertificate?branch=master)<br/>             | Gets or sets the signer certificate for the request.<br/>                                                                                        |
| [**Silent property**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-put_silent?branch=master)<br/>                                   | Gets or sets whether to allow UI during the request.<br/>                                                                                        |
| [**Status property**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-get_status?branch=master)<br/>                                   | Gets the status of the request.<br/>                                                                                                             |
| [**TransactionId property**](/windows/win32/Certenroll/nf-certenroll-ix509scepenrollment-get_transactionid?branch=master)<br/>                     | Gets or sets the transaction id for the request.<br/>                                                                                            |
