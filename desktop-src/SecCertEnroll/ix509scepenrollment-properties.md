# IX509SCEPEnrollment Properties

The [**IX509SCEPEnrollment**](ix509scepenrollment.md) interface exposes the following properties.

## In this section



| Topic                                                                                              | Description                                                                                                                                            |
|----------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Certificate property**](ix509scepenrollment-certificate.md)<br/>                         | Gets the certificate for the request.<br/>                                                                                                       |
| [**CertificateFriendlyName property**](ix509scepenrollment-certificatefriendlyname.md)<br/> | Gets or sets the friendly name for the certificate.<br/>                                                                                         |
| [**FailInfo property**](ix509scepenrollment-failinfo.md)<br/>                               | Gets information when the [**ProcessResponseMessage**](ix509scepenrollment-processresponsemessage.md) method detects a failed environment.<br/> |
| [**OldCertificate property**](ix509scepenrollment-oldcertificate.md)<br/>                   | Gets or sets an old certificate that a request is intended to replace.<br/>                                                                      |
| [**Request property**](ix509scepenrollment-request.md)<br/>                                 | Gets the inner PKCS10 request.<br/>                                                                                                              |
| [**ServerCapabilities property**](ix509scepenrollment-servercapabilities.md)<br/>           | Sets the preferred hash and encryption algorithms for the request.<br/>                                                                          |
| [**SignerCertificate property**](ix509scepenrollment-signercertificate.md)<br/>             | Gets or sets the signer certificate for the request.<br/>                                                                                        |
| [**Silent property**](ix509scepenrollment-silent.md)<br/>                                   | Gets or sets whether to allow UI during the request.<br/>                                                                                        |
| [**Status property**](ix509scepenrollment-status.md)<br/>                                   | Gets the status of the request.<br/>                                                                                                             |
| [**TransactionId property**](ix509scepenrollment-transactionid.md)<br/>                     | Gets or sets the transaction id for the request.<br/>                                                                                            |
