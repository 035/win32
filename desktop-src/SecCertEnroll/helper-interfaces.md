# Helper Interfaces

The following general use interfaces are supported by the Certificate Enrollment API.



| Interface                                                                    | Description                                                                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IBinaryConverter**](/windows/win32/CertEnroll/nn-certenroll-ibinaryconverter?branch=master)                                 | Creates a Unicode-encoded string from a byte array, creates a byte array from a Unicode-encoded string, and modifies the type of Unicode encoding applied to a string. |
| [**ICertificateAttestationChallenge**](/windows/win32/Certenroll/nn-certenroll-icertificateattestationchallenge?branch=master) | Supports certificate attestation challenges.                                                                                                                           |
| [**IObjectId**](/windows/win32/CertEnroll/nn-certenroll-iobjectid?branch=master)                                               | Represents an object identifier.                                                                                                                                       |
| [**IObjectIds**](/windows/win32/CertEnroll/nn-certenroll-iobjectids?branch=master)                                             | Manages a collection of [**IObjectId**](/windows/win32/CertEnroll/nn-certenroll-iobjectid?branch=master) objects.                                                                                                        |
| [**IX500DistinguishedName**](/windows/win32/CertEnroll/nn-certenroll-ix500distinguishedname?branch=master)                     | Represents an X.500 distinguished name.                                                                                                                                |
| [**IX509EndorsementKey**](/windows/win32/Certenroll/nn-certenroll-ix509endorsementkey?branch=master)                           | X.509 Endorsement Key Interface                                                                                                                                        |
