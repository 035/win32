# Helper Interfaces

The following general use interfaces are supported by the Certificate Enrollment API.



| Interface                                                                    | Description                                                                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IBinaryConverter**](ibinaryconverter.md)                                 | Creates a Unicode-encoded string from a byte array, creates a byte array from a Unicode-encoded string, and modifies the type of Unicode encoding applied to a string. |
| [**ICertificateAttestationChallenge**](icertificateattestationchallenge.md) | Supports certificate attestation challenges.                                                                                                                           |
| [**IObjectId**](iobjectid.md)                                               | Represents an object identifier.                                                                                                                                       |
| [**IObjectIds**](iobjectids.md)                                             | Manages a collection of [**IObjectId**](iobjectid.md) objects.                                                                                                        |
| [**IX500DistinguishedName**](ix500distinguishedname.md)                     | Represents an X.500 distinguished name.                                                                                                                                |
| [**IX509EndorsementKey**](ix509endorsementkey.md)                           | X.509 Endorsement Key Interface                                                                                                                                        |
