# Windows Gadget Platform

\[ The Windows Gadget Platform/Sidebar is available for use in the following versions of Windows: Windows 7, Windows Vista, and Windows Server 2008. It may be altered or unavailable in subsequent versions. \]

## Purpose

The Windows Gadget Platform is the framework for creating and hosting mini-applications called gadgets on the Windows desktop.

## Developer audience

This platform is designed for use by gadget developers who want to create compelling and personalized desktop experiences. It is also for Windows Vista Sidebar gadget developers who want to transition to the Windows Gadget Platform.

Familiarity with HTML and Web-based scripting languages, such as JScript and VBScript, is required.
