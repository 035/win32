# AVX Functions

The following are the Intel AVX functions.

## In this section



| Topic                                                                   | Description                                                                                                                    |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [**CopyContext**](copycontext.md)<br/>                           | Copies a source context structure (including any XState) onto an initialized destination context structure.<br/>         |
| [**GetEnabledXStateFeatures**](getenabledxstatefeatures.md)<br/> | Gets a mask of enabled XState features on x86 or x64 processors.<br/>                                                    |
| [**GetXStateFeaturesMask**](getxstatefeaturesmask.md)<br/>       | Returns the mask of XState features set within a [**CONTEXT**](wow64-context.md) structure.<br/>                        |
| [**InitializeContext**](initializecontext.md)<br/>               | Initializes a [**CONTEXT**](context-str.md) structure inside a buffer with the necessary size and alignment.<br/>       |
| [**LocateXStateFeature**](locatexstatefeature.md)<br/>           | Retrieves a pointer to the processor state for an XState feature within a [**CONTEXT**](context-str.md) structure.<br/> |
| [**SetXStateFeaturesMask**](setxstatefeaturesmask.md)<br/>       | Sets the mask of XState features set within a [**CONTEXT**](context-str.md) structure.<br/>                             |
