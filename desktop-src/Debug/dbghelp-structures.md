# DbgHelp Structures

The following are the DbgHelp structures:

## In this section



| Topic                                                                                                 | Description                                                                                                                                                                                        |
|-------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_IMAGE\_RUNTIME\_FUNCTION\_ENTRY**](-image-runtime-function-entry.md)<br/>                 | Represents an entry in the function table on 64-bit Windows.<br/>                                                                                                                            |
| [**ADDRESS64**](address64-str.md)<br/>                                                         | Represents an address. It is used in the [**STACKFRAME64**](stackframe64-str.md) structure.<br/>                                                                                            |
| [**API\_VERSION**](api-version-str.md)<br/>                                                    | Contains the library version.<br/>                                                                                                                                                           |
| [**FPO\_DATA**](fpo-data-str.md)<br/>                                                          | Represents the stack frame layout for a function on an x86 computer when frame pointer omission (FPO) optimization is used. The structure is used to locate the base of the call frame.<br/> |
| [**IMAGE\_DEBUG\_INFORMATION**](image-debug-information-str.md)<br/>                           | Contains debugging information.<br/>                                                                                                                                                         |
| [**IMAGEHLP\_CBA\_EVENT**](imagehlp-cba-event-str.md)<br/>                                     | Contains information about a debugging event.<br/>                                                                                                                                           |
| [**IMAGEHLP\_CBA\_READ\_MEMORY**](imagehlp-cba-read-memory-str.md)<br/>                        | Contains information about a memory read operation.<br/>                                                                                                                                     |
| [**IMAGEHLP\_DEFERRED\_SYMBOL\_LOAD64**](imagehlp-deferred-symbol-load64-str.md)<br/>          | Contains information about a deferred symbol load.<br/>                                                                                                                                      |
| [**IMAGEHLP\_DUPLICATE\_SYMBOL64**](imagehlp-duplicate-symbol64-str.md)<br/>                   | Contains duplicate symbol information.<br/>                                                                                                                                                  |
| [**IMAGEHLP\_GET\_TYPE\_INFO\_PARAMS**](imagehlp-get-type-info-params-str.md)<br/>             | Contains type information for a module.<br/>                                                                                                                                                 |
| [**IMAGEHLP\_LINE64**](imagehlp-line64-str.md)<br/>                                            | Represents a source file line.<br/>                                                                                                                                                          |
| [**IMAGEHLP\_MODULE64**](imagehlp-module64-str.md)<br/>                                        | Contains module information.<br/>                                                                                                                                                            |
| [**IMAGEHLP\_STACK\_FRAME**](imagehlp-stack-frame-str.md)<br/>                                 | Contains the stack frame information.<br/>                                                                                                                                                   |
| [**IMAGEHLP\_SYMBOL64**](imagehlp-symbol64-str.md)<br/>                                        | Contains symbol information.<br/>                                                                                                                                                            |
| [**KDHELP64**](kdhelp64-str.md)<br/>                                                           | Information that is used by kernel debuggers to trace through user-mode callbacks in a thread's kernel stack.<br/>                                                                           |
| [**LOADED\_IMAGE**](loaded-image-str.md)<br/>                                                  | Contains information about the loaded image.<br/>                                                                                                                                            |
| [**MINIDUMP\_CALLBACK\_INFORMATION**](minidump-callback-information-str.md)<br/>               | Contains a pointer to an optional callback function that can be used by the [**MiniDumpWriteDump**](minidumpwritedump.md) function.<br/>                                                    |
| [**MINIDUMP\_CALLBACK\_INPUT**](minidump-callback-input-str.md)<br/>                           | Contains information used by the [**MiniDumpCallback**](minidumpcallback.md) function.<br/>                                                                                                 |
| [**MINIDUMP\_CALLBACK\_OUTPUT**](minidump-callback-output-str.md)<br/>                         | Contains information returned by the [**MiniDumpCallback**](minidumpcallback.md) function.<br/>                                                                                             |
| [**MINIDUMP\_DIRECTORY**](minidump-directory-str.md)<br/>                                      | Contains the information needed to access a specific data stream in a minidump file.<br/>                                                                                                    |
| [**MINIDUMP\_EXCEPTION**](minidump-exception-str.md)<br/>                                      | Contains exception information.<br/>                                                                                                                                                         |
| [**MINIDUMP\_EXCEPTION\_INFORMATION**](minidump-exception-information-str.md)<br/>             | Contains the exception information written to the minidump file by the [**MiniDumpWriteDump**](minidumpwritedump.md) function.<br/>                                                         |
| [**MINIDUMP\_EXCEPTION\_STREAM**](minidump-exception-stream-str.md)<br/>                       | Represents an exception information stream.<br/>                                                                                                                                             |
| [**MINIDUMP\_FUNCTION\_TABLE\_DESCRIPTOR**](minidump-function-table-descriptor-str.md)<br/>    | Represents a function table stream.<br/>                                                                                                                                                     |
| [**MINIDUMP\_FUNCTION\_TABLE\_STREAM**](minidump-function-table-stream-str.md)<br/>            | Represents the header for the function table stream.<br/>                                                                                                                                    |
| [**MINIDUMP\_HANDLE\_DATA\_STREAM**](minidump-handle-data-stream-str.md)<br/>                  | Represents the header for a handle data stream.<br/>                                                                                                                                         |
| [**MINIDUMP\_HANDLE\_DESCRIPTOR**](minidump-handle-descriptor-str.md)<br/>                     | Contains the state of an individual system handle at the time the minidump was written.<br/>                                                                                                 |
| [**MINIDUMP\_HANDLE\_DESCRIPTOR\_2**](minidump-handle-descriptor-2.md)<br/>                    | Describes the state of an individual system handle at the time the minidump was written.<br/>                                                                                                |
| [**MINIDUMP\_HANDLE\_OBJECT\_INFORMATION**](minidump-handle-object-information.md)<br/>        | Contains object-specific information for a handle.<br/>                                                                                                                                      |
| [**MINIDUMP\_HANDLE\_OPERATION\_LIST**](minidump-handle-operation-list.md)<br/>                | Contains a list of handle operations.<br/>                                                                                                                                                   |
| [**MINIDUMP\_HEADER**](minidump-header-str.md)<br/>                                            | Contains header information for the minidump file.<br/>                                                                                                                                      |
| [**MINIDUMP\_INCLUDE\_MODULE\_CALLBACK**](minidump-include-module-callback-str.md)<br/>        | Contains information for the [**MiniDumpCallback**](minidumpcallback.md) function when the callback type is **IncludeModuleCallback**.<br/>                                                 |
| [**MINIDUMP\_INCLUDE\_THREAD\_CALLBACK**](minidump-include-thread-callback-str.md)<br/>        | Contains information for the [**MiniDumpCallback**](minidumpcallback.md) function when the callback type is **IncludeThreadCallback**.<br/>                                                 |
| [**MINIDUMP\_IO\_CALLBACK**](minidump-io-callback.md)<br/>                                     | Contains I/O callback information.<br/>                                                                                                                                                      |
| [**MINIDUMP\_LOCATION\_DESCRIPTOR**](minidump-location-descriptor-str.md)<br/>                 | Contains information describing the location of a data stream within a minidump file.<br/>                                                                                                   |
| [**MINIDUMP\_MEMORY\_DESCRIPTOR**](minidump-memory-descriptor-str.md)<br/>                     | Describes a range of memory.<br/>                                                                                                                                                            |
| [**MINIDUMP\_MEMORY\_INFO**](minidump-memory-info-str.md)<br/>                                 | Describes a region of memory.<br/>                                                                                                                                                           |
| [**MINIDUMP\_MEMORY\_INFO\_LIST**](minidump-memory-info-list-str.md)<br/>                      | Contains a list of memory regions.<br/>                                                                                                                                                      |
| [**MINIDUMP\_MEMORY\_LIST**](minidump-memory-list-str.md)<br/>                                 | Contains a list of memory ranges.<br/>                                                                                                                                                       |
| [**MINIDUMP\_MISC\_INFO**](minidump-misc-info-str.md)<br/>                                     | Contains a variety of information.<br/>                                                                                                                                                      |
| [**MINIDUMP\_MISC\_INFO\_2**](minidump-misc-info-2.md)<br/>                                    | Represents information in the miscellaneous information stream.<br/>                                                                                                                         |
| [**MINIDUMP\_MODULE**](minidump-module-str.md)<br/>                                            | Contains information for a specific module.<br/>                                                                                                                                             |
| [**MINIDUMP\_MODULE\_CALLBACK**](minidump-module-callback-str.md)<br/>                         | Contains module information for the [**MiniDumpCallback**](minidumpcallback.md) function when the callback type is [**ModuleCallback**](minidump-callback-type.md).<br/>                   |
| [**MINIDUMP\_MODULE\_LIST**](minidump-module-list-str.md)<br/>                                 | Contains a list of modules.<br/>                                                                                                                                                             |
| [**MINIDUMP\_READ\_MEMORY\_FAILURE\_CALLBACK**](minidump-read-memory-failure-callback.md)<br/> | Contains information about a failed memory read operation.<br/>                                                                                                                              |
| [**MINIDUMP\_STRING**](minidump-string-str.md)<br/>                                            | Describes a string.<br/>                                                                                                                                                                     |
| [**MINIDUMP\_SYSTEM\_INFO**](minidump-system-info-str.md)<br/>                                 | Contains processor and operating system information.<br/>                                                                                                                                    |
| [**MINIDUMP\_THREAD**](minidump-thread-str.md)<br/>                                            | Contains information for a specific thread.<br/>                                                                                                                                             |
| [**MINIDUMP\_THREAD\_CALLBACK**](minidump-thread-callback-str.md)<br/>                         | Contains thread information for the [**MiniDumpCallback**](minidumpcallback.md) function when the callback type is [**ThreadCallback**](minidump-callback-type.md).<br/>                   |
| [**MINIDUMP\_THREAD\_EX**](minidump-thread-ex-str.md)<br/>                                     | Contains extended information for a specific thread.<br/>                                                                                                                                    |
| [**MINIDUMP\_THREAD\_EX\_CALLBACK**](minidump-thread-ex-callback-str.md)<br/>                  | Contains extended thread information for the [**MiniDumpCallback**](minidumpcallback.md) function when the callback type is [**ThreadExCallback**](minidump-callback-type.md).<br/>        |
| [**MINIDUMP\_THREAD\_EX\_LIST**](minidump-thread-ex-list-str.md)<br/>                          | Contains a list of threads.<br/>                                                                                                                                                             |
| [**MINIDUMP\_THREAD\_INFO**](minidump-thread-info-str.md)<br/>                                 | Contains thread state information.<br/>                                                                                                                                                      |
| [**MINIDUMP\_THREAD\_INFO\_LIST**](minidump-thread-info-list-str.md)<br/>                      | Contains a list of threads.<br/>                                                                                                                                                             |
| [**MINIDUMP\_THREAD\_LIST**](minidump-thread-list-str.md)<br/>                                 | Contains a list of threads.<br/>                                                                                                                                                             |
| [**MINIDUMP\_UNLOADED\_MODULE**](minidump-unloaded-module-str.md)<br/>                         | Contains information about a module that has been unloaded. This information can help diagnose problems calling code that is no longer loaded.<br/>                                          |
| [**MINIDUMP\_UNLOADED\_MODULE\_LIST**](minidump-unloaded-module-list-str.md)<br/>              | Contains a list of unloaded modules.<br/>                                                                                                                                                    |
| [**MINIDUMP\_USER\_STREAM**](minidump-user-stream-str.md)<br/>                                 | Contains user-defined information stored in a data stream.<br/>                                                                                                                              |
| [**MINIDUMP\_USER\_STREAM\_INFORMATION**](minidump-user-stream-information-str.md)<br/>        | Contains a list of user data streams used by the [**MiniDumpWriteDump**](minidumpwritedump.md) function.<br/>                                                                               |
| [**MODLOAD\_CVMISC**](modload-cvmisc.md)<br/>                                                  | Contains CodeView and Misc records.<br/>                                                                                                                                                     |
| [**MODLOAD\_DATA**](modload-data-str.md)<br/>                                                  | Contains module data.<br/>                                                                                                                                                                   |
| [**OMAP**](omap.md)<br/>                                                                       | Describes an entry in an address map.<br/>                                                                                                                                                   |
| [**SOURCEFILE**](sourcefile-str.md)<br/>                                                       | Contains source file information.<br/>                                                                                                                                                       |
| [**SRCCODEINFO**](srccodeinfo-str.md)<br/>                                                     | Contains line information.<br/>                                                                                                                                                              |
| [**STACKFRAME64**](stackframe64-str.md)<br/>                                                   | Represents a stack frame.<br/>                                                                                                                                                               |
| [**STACKFRAME\_EX**](stackframe-ex.md)<br/>                                                    | Represents an extended stack frame.<br/>                                                                                                                                                     |
| [**SYMBOL\_INFO**](symbol-info-str.md)<br/>                                                    | Contains symbol information.<br/>                                                                                                                                                            |
| [**SYMSRV\_INDEX\_INFO**](symsrv-index-info.md)<br/>                                           | Contains symbol server index information.<br/>                                                                                                                                               |
| [**TI\_FINDCHILDREN\_PARAMS**](ti-findchildren-params-str.md)<br/>                             | Contains type index information. It is used by the [**SymGetTypeInfo**](symgettypeinfo.md) function.<br/>                                                                                   |
