# DbgHelp Structures

The following are the DbgHelp structures:

## In this section



| Topic                                                                                                 | Description                                                                                                                                                                                        |
|-------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_IMAGE\_RUNTIME\_FUNCTION\_ENTRY**](/windows/win32/WinNT/ns-winnt-_image_runtime_function_entry?branch=master)<br/>                 | Represents an entry in the function table on 64-bit Windows.<br/>                                                                                                                            |
| [**ADDRESS64**](/windows/win32/DbgHelp/ns-dbghelp-_tagaddress?branch=master)<br/>                                                         | Represents an address. It is used in the [**STACKFRAME64**](/windows/win32/DbgHelp/ns-dbghelp-_tagstackframe?branch=master) structure.<br/>                                                                                            |
| [**API\_VERSION**](/windows/win32/DbgHelp/ns-dbghelp-api_version?branch=master)<br/>                                                    | Contains the library version.<br/>                                                                                                                                                           |
| [**FPO\_DATA**](/windows/win32/WinNT/ns-winnt-_fpo_data?branch=master)<br/>                                                          | Represents the stack frame layout for a function on an x86 computer when frame pointer omission (FPO) optimization is used. The structure is used to locate the base of the call frame.<br/> |
| [**IMAGE\_DEBUG\_INFORMATION**](/windows/win32/DbgHelp/ns-dbghelp-_image_debug_information?branch=master)<br/>                           | Contains debugging information.<br/>                                                                                                                                                         |
| [**IMAGEHLP\_CBA\_EVENT**](/windows/win32/DbgHelp/ns-dbghelp-_imagehlp_cba_event?branch=master)<br/>                                     | Contains information about a debugging event.<br/>                                                                                                                                           |
| [**IMAGEHLP\_CBA\_READ\_MEMORY**](/windows/win32/DbgHelp/ns-dbghelp-_imagehlp_cba_read_memory?branch=master)<br/>                        | Contains information about a memory read operation.<br/>                                                                                                                                     |
| [**IMAGEHLP\_DEFERRED\_SYMBOL\_LOAD64**](/windows/win32/DbgHelp/ns-dbghelp-_imagehlp_deferred_symbol_load?branch=master)<br/>          | Contains information about a deferred symbol load.<br/>                                                                                                                                      |
| [**IMAGEHLP\_DUPLICATE\_SYMBOL64**](/windows/win32/DbgHelp/ns-dbghelp-_imagehlp_duplicate_symbol?branch=master)<br/>                   | Contains duplicate symbol information.<br/>                                                                                                                                                  |
| [**IMAGEHLP\_GET\_TYPE\_INFO\_PARAMS**](/windows/win32/DbgHelp/ns-dbghelp-_imagehlp_get_type_info_params?branch=master)<br/>             | Contains type information for a module.<br/>                                                                                                                                                 |
| [**IMAGEHLP\_LINE64**](/windows/win32/DbgHelp/ns-dbghelp-_imagehlp_line?branch=master)<br/>                                            | Represents a source file line.<br/>                                                                                                                                                          |
| [**IMAGEHLP\_MODULE64**](/windows/win32/DbgHelp/ns-dbghelp-_imagehlp_module?branch=master)<br/>                                        | Contains module information.<br/>                                                                                                                                                            |
| [**IMAGEHLP\_STACK\_FRAME**](/windows/win32/DbgHelp/ns-dbghelp-_imagehlp_stack_frame?branch=master)<br/>                                 | Contains the stack frame information.<br/>                                                                                                                                                   |
| [**IMAGEHLP\_SYMBOL64**](/windows/win32/DbgHelp/ns-dbghelp-_imagehlp_symbol?branch=master)<br/>                                        | Contains symbol information.<br/>                                                                                                                                                            |
| [**KDHELP64**](/windows/win32/DbgHelp/ns-dbghelp-_kdhelp?branch=master)<br/>                                                           | Information that is used by kernel debuggers to trace through user-mode callbacks in a thread's kernel stack.<br/>                                                                           |
| [**LOADED\_IMAGE**](/windows/win32/DbgHelp/ns-dbghelp-_loaded_image?branch=master)<br/>                                                  | Contains information about the loaded image.<br/>                                                                                                                                            |
| [**MINIDUMP\_CALLBACK\_INFORMATION**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_callback_information?branch=master)<br/>               | Contains a pointer to an optional callback function that can be used by the [**MiniDumpWriteDump**](/windows/win32/minidumpapiset/nf-minidumpapiset-minidumpwritedump?branch=master) function.<br/>                                                    |
| [**MINIDUMP\_CALLBACK\_INPUT**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_callback_input?branch=master)<br/>                           | Contains information used by the [**MiniDumpCallback**](/windows/win32/minidumpapiset/nc-minidumpapiset-minidump_callback_routine?branch=master) function.<br/>                                                                                                 |
| [**MINIDUMP\_CALLBACK\_OUTPUT**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_callback_output?branch=master)<br/>                         | Contains information returned by the [**MiniDumpCallback**](/windows/win32/minidumpapiset/nc-minidumpapiset-minidump_callback_routine?branch=master) function.<br/>                                                                                             |
| [**MINIDUMP\_DIRECTORY**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_directory?branch=master)<br/>                                      | Contains the information needed to access a specific data stream in a minidump file.<br/>                                                                                                    |
| [**MINIDUMP\_EXCEPTION**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_exception?branch=master)<br/>                                      | Contains exception information.<br/>                                                                                                                                                         |
| [**MINIDUMP\_EXCEPTION\_INFORMATION**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_exception_information?branch=master)<br/>             | Contains the exception information written to the minidump file by the [**MiniDumpWriteDump**](/windows/win32/minidumpapiset/nf-minidumpapiset-minidumpwritedump?branch=master) function.<br/>                                                         |
| [**MINIDUMP\_EXCEPTION\_STREAM**](/windows/win32/minidumpapiset/ns-minidumpapiset-minidump_exception_stream?branch=master)<br/>                       | Represents an exception information stream.<br/>                                                                                                                                             |
| [**MINIDUMP\_FUNCTION\_TABLE\_DESCRIPTOR**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_function_table_descriptor?branch=master)<br/>    | Represents a function table stream.<br/>                                                                                                                                                     |
| [**MINIDUMP\_FUNCTION\_TABLE\_STREAM**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_function_table_stream?branch=master)<br/>            | Represents the header for the function table stream.<br/>                                                                                                                                    |
| [**MINIDUMP\_HANDLE\_DATA\_STREAM**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_handle_data_stream?branch=master)<br/>                  | Represents the header for a handle data stream.<br/>                                                                                                                                         |
| [**MINIDUMP\_HANDLE\_DESCRIPTOR**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_handle_descriptor?branch=master)<br/>                     | Contains the state of an individual system handle at the time the minidump was written.<br/>                                                                                                 |
| [**MINIDUMP\_HANDLE\_DESCRIPTOR\_2**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_handle_descriptor_2?branch=master)<br/>                    | Describes the state of an individual system handle at the time the minidump was written.<br/>                                                                                                |
| [**MINIDUMP\_HANDLE\_OBJECT\_INFORMATION**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_handle_object_information?branch=master)<br/>        | Contains object-specific information for a handle.<br/>                                                                                                                                      |
| [**MINIDUMP\_HANDLE\_OPERATION\_LIST**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_handle_operation_list?branch=master)<br/>                | Contains a list of handle operations.<br/>                                                                                                                                                   |
| [**MINIDUMP\_HEADER**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_header?branch=master)<br/>                                            | Contains header information for the minidump file.<br/>                                                                                                                                      |
| [**MINIDUMP\_INCLUDE\_MODULE\_CALLBACK**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_include_module_callback?branch=master)<br/>        | Contains information for the [**MiniDumpCallback**](/windows/win32/minidumpapiset/nc-minidumpapiset-minidump_callback_routine?branch=master) function when the callback type is **IncludeModuleCallback**.<br/>                                                 |
| [**MINIDUMP\_INCLUDE\_THREAD\_CALLBACK**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_include_thread_callback?branch=master)<br/>        | Contains information for the [**MiniDumpCallback**](/windows/win32/minidumpapiset/nc-minidumpapiset-minidump_callback_routine?branch=master) function when the callback type is **IncludeThreadCallback**.<br/>                                                 |
| [**MINIDUMP\_IO\_CALLBACK**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_io_callback?branch=master)<br/>                                     | Contains I/O callback information.<br/>                                                                                                                                                      |
| [**MINIDUMP\_LOCATION\_DESCRIPTOR**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_location_descriptor?branch=master)<br/>                 | Contains information describing the location of a data stream within a minidump file.<br/>                                                                                                   |
| [**MINIDUMP\_MEMORY\_DESCRIPTOR**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_memory_descriptor?branch=master)<br/>                     | Describes a range of memory.<br/>                                                                                                                                                            |
| [**MINIDUMP\_MEMORY\_INFO**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_memory_info?branch=master)<br/>                                 | Describes a region of memory.<br/>                                                                                                                                                           |
| [**MINIDUMP\_MEMORY\_INFO\_LIST**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_memory_info_list?branch=master)<br/>                      | Contains a list of memory regions.<br/>                                                                                                                                                      |
| [**MINIDUMP\_MEMORY\_LIST**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_memory64_list?branch=master)<br/>                                 | Contains a list of memory ranges.<br/>                                                                                                                                                       |
| [**MINIDUMP\_MISC\_INFO**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_misc_info?branch=master)<br/>                                     | Contains a variety of information.<br/>                                                                                                                                                      |
| [**MINIDUMP\_MISC\_INFO\_2**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_misc_info_2?branch=master)<br/>                                    | Represents information in the miscellaneous information stream.<br/>                                                                                                                         |
| [**MINIDUMP\_MODULE**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_module?branch=master)<br/>                                            | Contains information for a specific module.<br/>                                                                                                                                             |
| [**MINIDUMP\_MODULE\_CALLBACK**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_module_callback?branch=master)<br/>                         | Contains module information for the [**MiniDumpCallback**](/windows/win32/minidumpapiset/nc-minidumpapiset-minidump_callback_routine?branch=master) function when the callback type is [**ModuleCallback**](/windows/win32/minidumpapiset/ne-minidumpapiset-_minidump_callback_type?branch=master).<br/>                   |
| [**MINIDUMP\_MODULE\_LIST**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_module_list?branch=master)<br/>                                 | Contains a list of modules.<br/>                                                                                                                                                             |
| [**MINIDUMP\_READ\_MEMORY\_FAILURE\_CALLBACK**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_read_memory_failure_callback?branch=master)<br/> | Contains information about a failed memory read operation.<br/>                                                                                                                              |
| [**MINIDUMP\_STRING**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_string?branch=master)<br/>                                            | Describes a string.<br/>                                                                                                                                                                     |
| [**MINIDUMP\_SYSTEM\_INFO**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_system_info?branch=master)<br/>                                 | Contains processor and operating system information.<br/>                                                                                                                                    |
| [**MINIDUMP\_THREAD**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_thread?branch=master)<br/>                                            | Contains information for a specific thread.<br/>                                                                                                                                             |
| [**MINIDUMP\_THREAD\_CALLBACK**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_thread_callback?branch=master)<br/>                         | Contains thread information for the [**MiniDumpCallback**](/windows/win32/minidumpapiset/nc-minidumpapiset-minidump_callback_routine?branch=master) function when the callback type is [**ThreadCallback**](/windows/win32/minidumpapiset/ne-minidumpapiset-_minidump_callback_type?branch=master).<br/>                   |
| [**MINIDUMP\_THREAD\_EX**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_thread_ex?branch=master)<br/>                                     | Contains extended information for a specific thread.<br/>                                                                                                                                    |
| [**MINIDUMP\_THREAD\_EX\_CALLBACK**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_thread_ex_callback?branch=master)<br/>                  | Contains extended thread information for the [**MiniDumpCallback**](/windows/win32/minidumpapiset/nc-minidumpapiset-minidump_callback_routine?branch=master) function when the callback type is [**ThreadExCallback**](/windows/win32/minidumpapiset/ne-minidumpapiset-_minidump_callback_type?branch=master).<br/>        |
| [**MINIDUMP\_THREAD\_EX\_LIST**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_thread_ex_list?branch=master)<br/>                          | Contains a list of threads.<br/>                                                                                                                                                             |
| [**MINIDUMP\_THREAD\_INFO**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_thread_info?branch=master)<br/>                                 | Contains thread state information.<br/>                                                                                                                                                      |
| [**MINIDUMP\_THREAD\_INFO\_LIST**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_thread_info_list?branch=master)<br/>                      | Contains a list of threads.<br/>                                                                                                                                                             |
| [**MINIDUMP\_THREAD\_LIST**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_thread_list?branch=master)<br/>                                 | Contains a list of threads.<br/>                                                                                                                                                             |
| [**MINIDUMP\_UNLOADED\_MODULE**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_unloaded_module?branch=master)<br/>                         | Contains information about a module that has been unloaded. This information can help diagnose problems calling code that is no longer loaded.<br/>                                          |
| [**MINIDUMP\_UNLOADED\_MODULE\_LIST**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_unloaded_module_list?branch=master)<br/>              | Contains a list of unloaded modules.<br/>                                                                                                                                                    |
| [**MINIDUMP\_USER\_STREAM**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_user_stream?branch=master)<br/>                                 | Contains user-defined information stored in a data stream.<br/>                                                                                                                              |
| [**MINIDUMP\_USER\_STREAM\_INFORMATION**](/windows/win32/minidumpapiset/ns-minidumpapiset-_minidump_user_stream_information?branch=master)<br/>        | Contains a list of user data streams used by the [**MiniDumpWriteDump**](/windows/win32/minidumpapiset/nf-minidumpapiset-minidumpwritedump?branch=master) function.<br/>                                                                               |
| [**MODLOAD\_CVMISC**](/windows/win32/DbgHelp/ns-dbghelp-_modload_cvmisc?branch=master)<br/>                                                  | Contains CodeView and Misc records.<br/>                                                                                                                                                     |
| [**MODLOAD\_DATA**](/windows/win32/DbgHelp/ns-dbghelp-_modload_data?branch=master)<br/>                                                  | Contains module data.<br/>                                                                                                                                                                   |

| [**SOURCEFILE**](/windows/win32/DbgHelp/ns-dbghelp-_sourcefile?branch=master)<br/>                                                       | Contains source file information.<br/>                                                                                                                                                       |
| [**SRCCODEINFO**](/windows/win32/DbgHelp/ns-dbghelp-_srccodeinfo?branch=master)<br/>                                                     | Contains line information.<br/>                                                                                                                                                              |
| [**STACKFRAME64**](/windows/win32/DbgHelp/ns-dbghelp-_tagstackframe?branch=master)<br/>                                                   | Represents a stack frame.<br/>                                                                                                                                                               |

| [**SYMSRV\_INDEX\_INFO**](/windows/win32/Dbghelp/ns-dbghelp-symsrv_index_info?branch=master)<br/>                                           | Contains symbol server index information.<br/>                                                                                                                                               |
| [**TI\_FINDCHILDREN\_PARAMS**](/windows/win32/DbgHelp/ns-dbghelp-_ti_findchildren_params?branch=master)<br/>                             | Contains type index information. It is used by the [**SymGetTypeInfo**](/windows/win32/Dbghelp/nf-dbghelp-symgettypeinfo?branch=master) function.<br/>                                                                                   |
