# View::DisplaySelectionPropertySheet method

The **DisplaySelectionPropertySheet** method displays the property sheet for the current selection.

## Syntax


```VB
View.DisplaySelectionPropertySheet()
```



## Parameters

This method has no parameters.

## Return value

This method does not return a value.
