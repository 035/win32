# MMCListPadInfo.HasButton property

The HasButton property returns a value to indicate whether the listpad's button should be displayed.

This property is read-only.

## Syntax


```VB
MMCListPadInfo.HasButton
```



## Property value

BOOL value of **TRUE** if the listpad's button should be displayed, or **FALSE** otherwise.
