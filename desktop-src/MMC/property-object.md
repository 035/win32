# Property Object object

The **Property** object encapsulates a single property and contains no methods.

## Members

The **Property Object** object has these types of members:

-   [Properties](#properties)

### Properties

The **Property Object** object has these properties.



| Property                                   | Access type           | Description                                                                    |
|:-------------------------------------------|:----------------------|:-------------------------------------------------------------------------------|
| [**Name**](property-name.md)<br/>   | Read-only<br/>  | Name of the property.<br/>                                               |
| [**Value**](property-value.md)<br/> | Read/write<br/> | Value of the property. This is the default property of this object.<br/> |
