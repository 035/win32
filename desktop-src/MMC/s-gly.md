</dd> <dt>

<span id="mmc_scope_item_gly"></span><span id="MMC_SCOPE_ITEM_GLY"></span>**scope item**
</dt> <dd>

A [*static node*](#mmc-static-node-gly) or [*enumerated node*](e-gly.md#mmc-enumerated-node-gly) in the snap-in's scope pane.

</dd> <dt>

<span id="mmc_scope_node_gly"></span><span id="MMC_SCOPE_NODE_GLY"></span>**scope node**
</dt> <dd>

A node in the Visual Basic snap-in's [*scope pane*](#mmc-scope-pane-gly).

</dd> <dt>

<span id="mmc_scope_pane_gly"></span><span id="MMC_SCOPE_PANE_GLY"></span>**scope pane**
</dt> <dd>

The left pane in a [*view*](v-gly.md#mmc-view-gly) that contains the [*console tree*](c-gly.md#mmc-console-tree-gly). The user can hide the scope pane in a view. The term "scope pane" is used interchangeably with "console tree" in the MMC SDK documentation.

</dd> <dt>

<span id="mmc_scope_pane_item_gly"></span><span id="MMC_SCOPE_PANE_ITEM_GLY"></span>**scope pane item**
</dt> <dd>

An instance of a Visual Basic snap-in's [*scope item*](#mmc-scope-item-gly) in the [*scope pane*](#mmc-scope-pane-gly) of a specific [*view*](v-gly.md#mmc-view-gly).

</dd> <dt>

<span id="mmc_snap_in_gly"></span><span id="MMC_SNAP_IN_GLY"></span>**snap-in**
</dt> <dd>

An OLE InProc server that performs a server management task as part of an MMC console. MMC serves as a host for snap-in-defined user interfaces, but does not limit what the snap-ins can do or how they communicate with the managed services.

</dd> <dt>

<span id="mmc_snap_in_designer_for_visual_basic_gly"></span><span id="MMC_SNAP_IN_DESIGNER_FOR_VISUAL_BASIC_GLY"></span>**Snap-in Designer for Visual Basic**
</dt> <dd>

An [*ActiveX designer*](a-gly.md#mmc-activex-designer-gly) that enables a snap-in to be written entirely in Visual Basic.

</dd> <dt>

<span id="mmc_snap_in_manager_gly"></span><span id="MMC_SNAP_IN_MANAGER_GLY"></span>**Snap-in Manager**
</dt> <dd>

A reference to the Add/Remove Snap-in dialog box.

</dd> <dt>

<span id="mmc_snap_in_menu_gly"></span><span id="MMC_SNAP_IN_MENU_GLY"></span>**snap-in menu**
</dt> <dd>

A [*Menu button*](m-gly.md#mmc-menu-button-gly) that a stand-alone or extension snap-in adds to the [*menu bar*](m-gly.md#mmc-menu-bar-gly).

</dd> <dt>

<span id="mmc_snap_in_taskpad_gly"></span><span id="MMC_SNAP_IN_TASKPAD_GLY"></span>**snap-in taskpad**
</dt> <dd>

A [*Taskpad*](t-gly.md#mmc-taskpad-gly) implemented by a snap-in, as opposed to a [*console taskpad*](c-gly.md#mmc-console-taskpad-gly).

</dd> <dt>

<span id="mmc_snap_in_toolbar_gly"></span><span id="MMC_SNAP_IN_TOOLBAR_GLY"></span>**snap-in toolbar**
</dt> <dd>

A section of the [*toolbar*](t-gly.md#mmc-toolbar-gly) that contains the [*toolbar buttons*](t-gly.md#mmc-toolbar-button-gly) added by stand-alone and extension snap-ins.

</dd> <dt>

<span id="mmc_special_cookie_gly"></span><span id="MMC_SPECIAL_COOKIE_GLY"></span>**special cookie**
</dt> <dd>

A non-standard [*cookie*](c-gly.md#mmc-cookie-gly) that MMC sends to the snap-in's [**IComponent::QueryDataObject**](icomponent-querydataobject.md) method. Special cookies are not associated with scope and result items, and their values are determined by MMC.

</dd> <dt>

<span id="mmc_special_data_object_gly"></span><span id="MMC_SPECIAL_DATA_OBJECT_GLY"></span>**special data object**
</dt> <dd>

A non-standard [*data object*](d-gly.md#mmc-data-object-gly) that MMC sends to the snap-in's [**IComponent::Notify**](icomponent-notify.md) or [**IExtendControlbar::ControlbarNotify**](iextendcontrolbar-controlbarnotify.md) methods.

</dd> <dt>

<span id="mmc_stand_alone_snap_in_gly"></span><span id="MMC_STAND_ALONE_SNAP_IN_GLY"></span>**stand-alone snap-in**
</dt> <dd>

A snap-in that, when loaded in a console, can perform its designated management task as the only snap-in loaded in the console.

</dd> <dt>

<span id="mmc_standard_list_view_gly"></span><span id="MMC_STANDARD_LIST_VIEW_GLY"></span>**standard list view**
</dt> <dd>

A default result pane view type that displays a collection of items, each consisting of an icon and a label, and provides several ways to display and arrange the items.

</dd> <dt>

<span id="mmc_standard_list_view_taskpad_gly"></span><span id="MMC_STANDARD_LIST_VIEW_TASKPAD_GLY"></span>**standard list view taskpad**
</dt> <dd>

A taskpad view style that displays the taskpad title, a list control that displays items that can be selected, and tasks that can perform actions on the selected items.

</dd> <dt>

<span id="mmc_standard_taskpad_gly"></span><span id="MMC_STANDARD_TASKPAD_GLY"></span>**standard taskpad**
</dt> <dd>

Default taskpad view style that displays the taskpad title and tasks.

</dd> <dt>

<span id="mmc_standard_verb_gly"></span><span id="MMC_STANDARD_VERB_GLY"></span>**standard verb**
</dt> <dd>

Also see [*verb*](v-gly.md#mmc-verb-gly).

</dd> <dt>

<span id="mmc_static_node_gly"></span><span id="MMC_STATIC_NODE_GLY"></span>**static node**
</dt> <dd>

A node in the MMC namespace that is present as long as the snap-in providing it is loaded. Only [*stand-alone snap-ins*](#mmc-stand-alone-snap-in-gly) can have static nodes. Stand-alone snap-ins use their static nodes to store setup configuration and relative placement in the console.

</dd> <dt>

<span id="mmc_system_modal_gly"></span><span id="MMC_SYSTEM_MODAL_GLY"></span>**system modal**
</dt> <dd>

A type of user interface element (for example, dialog box, message box, oe property sheet) for which all applications are suspended until the user responds. System-modal messages notify the user of serious, potentially damaging errors that require immediate attention.

</dd> </dl>
