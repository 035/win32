# MMCTask.DisplayObject property

The **DisplayObject** property returns an **MMCDisplayObject** object.

This property is read-only.

## Syntax


```VB
MMCTask.DisplayObject
```



## Property value

The returned property is an **MMCDisplayObject** object.

## Remarks

The returned **MMCDisplayObject** object contains information about the appearance of the task's button.
