<span id="mmc_image_list_gly"></span><span id="MMC_IMAGE_LIST_GLY"></span>**image list**
</dt> <dd>

A collection of bitmaps that a snap-in can use as icons for items in the [*console tree*](c-gly.md#mmc-console-tree-gly) in the scope pane, or in a list view in the result pane.

In a Visual Basic snap-in, an object that contains a collection of Image objects, where each object represents an image in the console tree or in a list view.

</dd> <dt>

<span id="mmc_imported_data_gly"></span><span id="MMC_IMPORTED_DATA_GLY"></span>**imported data**

The data from a C++ snap-in that is imported to a Visual Basic snap-in.

</dd> <dt>

<span id="mmc_in_process_server_gly"></span><span id="MMC_IN_PROCESS_SERVER_GLY"></span>**in-process server**
