<dl> <dt>

<span id="mmc_handle_gly"></span><span id="MMC_HANDLE_GLY"></span>**handle**
</dt> <dd>

Created by MMC to uniquely identify an inserted scope or result item. Scope item handles are of type HSCOPEITEM, while result item handles are of type HRESULTITEM.

</dd> <dt>

<span id="mmc_hresultitem_gly"></span><span id="MMC_HRESULTITEM_GLY"></span>**HRESULTITEM**
</dt> <dd>

[*Handle*](#mmc-handle-gly) to a result item.

</dd> <dt>

<span id="mmc_hscopeitem_gly"></span><span id="MMC_HSCOPEITEM_GLY"></span>**HSCOPEITEM**
</dt> <dd>

[*Handle*](#mmc-handle-gly) to a scope item.

</dd> <dt>

<span id="mmc_html_help_collection_file_gly"></span><span id="MMC_HTML_HELP_COLLECTION_FILE_GLY"></span>**HTML Help collection file**
</dt> <dd>

Merged HTML Help file (with a .col extension) that combines the information from the index and full-text search of multiple compiled Help (.chm) files and displays the merged Help files in a single table of contents. For each console file, MMC maintains a collection file that merges the Help files of all snap-ins loaded in the console, as well as the Help files of enabled extension snap-ins and dynamic snap-ins.

</dd> <dt>

<span id="mmc_html_view_gly"></span><span id="MMC_HTML_VIEW_GLY"></span>**HTML view**
</dt> <dd>

Result pane view of the selected scope item in which the snap-in can display a custom webpage.

</dd> </dl>
