# T

<dl> <dt>

<span id="mmc_task_gly"></span><span id="MMC_TASK_GLY"></span>**task**
</dt> <dd>

Action on a [*taskpad*](#mmc-taskpad-gly) that consists of an image, a label, a description, and a mechanism for telling the snap-in to perform that action.

</dd> <dt>

<span id="mmc_taskpad_gly"></span><span id="MMC_TASKPAD_GLY"></span>**taskpad**
</dt> <dd>

Result pane view on the selected scope item that is represented as a dynamic HTML (DHTML) page. The taskpad DHTML page lists actions, called [*tasks*](#mmc-task-gly), that can be performed on the selected node.

</dd> <dt>

<span id="mmc_taskpad_extension_gly"></span><span id="MMC_TASKPAD_EXTENSION_GLY"></span>**taskpad extension**
</dt> <dd>

[*Extension snap-in*](https://www.bing.com/search?q=*Extension snap-in*) that extends the functionality of a [*primary snap-in*](https://www.bing.com/search?q=*primary snap-in*) by adding [*tasks*](#mmc-task-gly) to the [*taskpad view*](#mmc-taskpad-view-gly).

</dd> <dt>

<span id="mmc_taskpad_view_gly"></span><span id="MMC_TASKPAD_VIEW_GLY"></span>**taskpad view**
</dt> <dd>

Result pane view that displays a [*taskpad*](#mmc-taskpad-gly).

</dd> <dt>

<span id="mmc_toolbar_gly"></span><span id="MMC_TOOLBAR_GLY"></span>**toolbar**
</dt> <dd>

Area of the main MMC window that contains [*toolbar buttons*](#mmc-toolbar-button-gly).

</dd> <dt>

<span id="mmc_toolbar_button_gly"></span><span id="MMC_TOOLBAR_BUTTON_GLY"></span>**toolbar button**
</dt> <dd>

Button on the [*toolbar*](#mmc-toolbar-gly) that consists of an icon and an associated action.

</dd> <dt>

<span id="mmc_toolbar_control_gly"></span><span id="MMC_TOOLBAR_CONTROL_GLY"></span>**toolbar control**
</dt> <dd>

MMC control for managing the [*toolbar*](#mmc-toolbar-gly) in the main MMC window.

</dd> <dt>

<span id="mmc_toolbar_extension_gly"></span><span id="MMC_TOOLBAR_EXTENSION_GLY"></span>**toolbar extension**
</dt> <dd>

[*Extension snap-in*](https://www.bing.com/search?q=*Extension snap-in*) that extends the functionality of a [*primary snap-in*](https://www.bing.com/search?q=*primary snap-in*). A toolbar extension adds [*toolbar buttons*](#mmc-toolbar-button-gly) or [*menu buttons*](https://www.bing.com/search?q=*menu buttons*) for scope or result items of the extended [*node type*](https://www.bing.com/search?q=*node type*).

</dd> <dt>

<span id="mmc_trappable_error_gly"></span><span id="MMC_TRAPPABLE_ERROR_GLY"></span>**trappable error**
</dt> <dd>

Runtime error that occurs in a Visual Basic snap-in that can be detected and handled by the snap-in using the **On Error** statement.

</dd> </dl>
