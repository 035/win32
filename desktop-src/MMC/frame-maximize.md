# Frame::Maximize method

The **Maximize** method maximizes the frame's size.

## Syntax


```VB
Frame.Maximize()
```



## Parameters

This method has no parameters.

## Return value

This method does not return a value.

## Examples


```VB
' Maximize the Frame.
objFrame.Maximize
```
