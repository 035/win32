# Frame Object object

The **Frame** object encapsulates the frame (for the MMC main window) for the application.

## Members

The **Frame Object** object has these types of members:

-   [Methods](#methods)
-   [Properties](#properties)

### Methods

The **Frame Object** object has these methods.



| Method                             | Description                                       |
|:-----------------------------------|:--------------------------------------------------|
| [**Maximize**](frame-maximize.md) | Maximizes the frame.<br/>                   |
| [**Minimize**](frame-minimize.md) | Minimizes the frame.<br/>                   |
| [**Restore**](frame-restore.md)   | Restores the frame to its normal size.<br/> |



 

### Properties

The **Frame Object** object has these properties.



| Property                                  | Access type           | Description                                    |
|:------------------------------------------|:----------------------|:-----------------------------------------------|
| [**Bottom**](frame-bottom.md)<br/> | Read/write<br/> | The bottom coordinate of the frame.<br/> |
| [**Left**](frame-left.md)<br/>     | Read/write<br/> | The left coordinate of the frame.<br/>   |
| [**Right**](frame-right.md)<br/>   | Read/write<br/> | The right coordinate of the frame.<br/>  |
| [**Top**](frame-top.md)<br/>       | Read/write<br/> | The top coordinate of the frame.<br/>    |
