# W

<dl> <dt>

<span id="mmc_watermark_gly"></span><span id="MMC_WATERMARK_GLY"></span>**watermark**
</dt> <dd>

Bitmap image used in a [*Wizard97-style wizard*](#mmc-wizard97-style-wizard-gly) that conveys a message to the user.

</dd> <dt>

<span id="mmc_welcome_page_gly"></span><span id="MMC_WELCOME_PAGE_GLY"></span>**Welcome page**
</dt> <dd>

The first page of a [*Wizard97-style wizard*](#mmc-wizard97-style-wizard-gly). The Welcome page typically has a bitmap image, ([*watermark*](#mmc-watermark-gly)), that conveys a message to the user.

</dd> <dt>

<span id="mmc_windows_management_instrumentation_wmi__gly"></span><span id="MMC_WINDOWS_MANAGEMENT_INSTRUMENTATION_WMI__GLY"></span>**Windows Management Instrumentation (WMI)**
</dt> <dd>

the Microsoft implementation of Web-Based Enterprise Management (WBEM), which is an industry initiative to develop a standard technology for accessing management information in an enterprise environment. WMI uses the industry-standard Common Information Model (CIM) to represent systems, applications, networks, devices, and other managed objects in an enterprise environment. MMC snap-ins can be used to display any information that is stored in WMI.

</dd> <dt>

<span id="mmc_wizard_gly"></span><span id="MMC_WIZARD_GLY"></span>**wizard**
</dt> <dd>

Type of [*property sheet*](https://www.bing.com/search?q=*property sheet*). Wizards present pages one at a time in sequential order controlled by the application. MMC enables a snap-in to add either standard wizard pages or [*Wizard97 pages*](#mmc-wizard97-page-gly).

</dd> <dt>

<span id="mmc_wizard97_page_gly"></span><span id="MMC_WIZARD97_PAGE_GLY"></span>**Wizard97 page**
</dt> <dd>

Wizard pages used with [*Wizard97-style wizard*](#mmc-wizard97-style-wizard-gly). Snap-ins are encouraged to use the Wizard97 style for consistent style throughout MMC.

</dd> <dt>

<span id="mmc_wizard97_style_wizard_gly"></span><span id="MMC_WIZARD97_STYLE_WIZARD_GLY"></span>**Wizard97-style wizard**
</dt> <dd>

Wizard style that improves the usability of wizards by implementing an introductory ([*Welcome*](#mmc-welcome-page-gly)) and finish ([*Completion*](https://www.bing.com/search?q=*Completion*)) pages, interior page titles, and a consistent presentation of information.

</dd> </dl>
