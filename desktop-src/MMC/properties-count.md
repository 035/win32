# Properties::Count property

The **Count** property returns the number of [**Property**](property-object.md) objects that are in the [**Properties collection**](properties-collection.md). This property is read-only.

## Syntax


```VB
Property Count As Long
```



## Property value

The number of [**Property objects**](property-object.md) contained in this collection.
