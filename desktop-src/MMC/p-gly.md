# P

<dl> <dt>

<span id="mmc_policy_gly"></span><span id="MMC_POLICY_GLY"></span>**Policy**
</dt> <dd>

A feature that administrators can use to determine a group's access to tools and functionality for computers or users. MMC administrators can use this feature to prevent individual users and groups of users from accessing specified snap-in functionality.

</dd> <dt>

<span id="mmc_primary_object_gly"></span><span id="MMC_PRIMARY_OBJECT_GLY"></span>**primary object**
</dt> <dd>

A snap-in's object. For snap-ins that expose the **IComponentData** interface and the **ISnapinAbout** interface by means of two different objects, the primary object is the snap-in object that exposes the **IComponentData** interface.

</dd> <dt>

<span id="mmc_primary_snap_in_gly"></span><span id="MMC_PRIMARY_SNAP_IN_GLY"></span>**primary snap-in**
</dt> <dd>

A snap-in (either a [*stand-alone snap-in*](s-gly.md#mmc-stand-alone-snap-in-gly) or a [*namespace extension*](n-gly.md#mmc-namespace-extension-gly)) that inserts scope or result items of node types that can be extended by one or more extension snap-ins. The primary snap-in is responsible for registering the [*node type GUIDs*](n-gly.md#mmc-node-type-guid-gly) of its extendible [*node types*](n-gly.md#mmc-node-type-gly).

</dd> <dt>

<span id="mmc_property_page_gly"></span><span id="MMC_PROPERTY_PAGE_GLY"></span>**property page**
</dt> <dd>

A child window of a [*property sheet*](#mmc-property-sheet-gly) that contains control windows for setting a group of related properties.

</dd> <dt>

<span id="mmc_property_page_extension_gly"></span><span id="MMC_PROPERTY_PAGE_EXTENSION_GLY"></span>**property page extension**
</dt> <dd>

An [*Extension snap-in*](e-gly.md#mmc-extension-snap-in-gly) that extends the functionality of a [*primary snap-in*](#mmc-primary-snap-in-gly). The extension snap-in adds its own property pages to the property sheet of a particular [*node type's*](n-gly.md#mmc-node-type-gly) scope or result items.

</dd> <dt>

<span id="mmc_property_sheet_gly"></span><span id="MMC_PROPERTY_SHEET_GLY"></span>**property sheet**
</dt> <dd>

A Window in which the user can view and edit item properties. A property sheet contains one or more overlapping child windows ([*property pages*](#mmc-property-page-gly)).

</dd> </dl>
