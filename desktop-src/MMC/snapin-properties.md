# SnapIn::Properties property

The **Properties** property returns the [**Properties collection**](properties-collection.md) for the snap-in. This property is read-only.

## Syntax


```VB
Property Properties As Properties
```



## Property value

The [**Properties**](properties-collection.md) collection for the snap-in.
