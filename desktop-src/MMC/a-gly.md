<dl> <dt>

<span id="mmc_activex_gly"></span><span id="MMC_ACTIVEX_GLY"></span>**ActiveX**
</dt> <dd>

the Microsoft brand name for the technologies that enable interoperability using the Component Object Model (COM). ActiveX technology includes, but is not limited to, OLE.

</dd> <dt>

<span id="mmc_activex_control_gly"></span><span id="MMC_ACTIVEX_CONTROL_GLY"></span>**ActiveX control**
</dt> <dd>

User interface element created using ActiveX technology.

</dd> <dt>

<span id="mmc_activex_designer_gly"></span><span id="MMC_ACTIVEX_DESIGNER_GLY"></span>**ActiveX designer**
</dt> <dd>

In-process ActiveX object that extends the functionality of the Visual Basic IDE. Visual Basic 6.0 ships with numerous ActiveX designers. The [*Snap-in Designer for Visual Basic*](s-gly.md#mmc-snap-in-designer-for-visual-basic-gly) is an ActiveX designer.

</dd> <dt>

<span id="mmc_application_modal_gly"></span><span id="MMC_APPLICATION_MODAL_GLY"></span>**application modal**
</dt> <dd>

User interface element (for example, dialog box, message box, property sheet) for which the user must respond before continuing work in the current application.

</dd> <dt>

<span id="mmc_author_mode_gly"></span><span id="MMC_AUTHOR_MODE_GLY"></span>**author mode**
</dt> <dd>

Access mode to a custom console that grants the user of the console full access to all MMC functionality, including the ability to add or remove snap-ins, create new windows, create taskpad views and tasks, add items to the Favorites list, and view all portions of the console tree.

</dd> <dt>

<span id="mmc_auto_create_node_gly"></span><span id="MMC_AUTO_CREATE_NODE_GLY"></span>**auto-create node**
</dt> <dd>

Node in a Visual Basic snap-in that is defined at design time and that is automatically created by the designer runtime when the parent of the node is expanded.

</dd> <dt>

<span id="mmc_automation_object_model_gly"></span><span id="MMC_AUTOMATION_OBJECT_MODEL_GLY"></span>**automation object model**
</dt> <dd>

For MMC 2.0. Feature that allows scripts to interact with MMC. Also used by [*view extensions*](v-gly.md#mmc-view-extension-gly).

</dd> </dl>
