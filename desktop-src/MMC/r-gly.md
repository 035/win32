<dl> <dt>

<span id="mmc_registry_gly"></span><span id="MMC_REGISTRY_GLY"></span>**registry**
</dt> <dd>

System-defined database in which applications and system components store and retrieve configuration data. MMC uses the registry to retrieve the location of snap-in DLLs and to determine whether a [*node type*](n-gly.md#mmc-node-type-gly) is extendible, and if so, which [*extension snap-ins*](e-gly.md#mmc-extension-snap-in-gly) extend that node type's functionality.

</dd> <dt>

<span id="mmc_required_extension_gly"></span><span id="MMC_REQUIRED_EXTENSION_GLY"></span>**required extension**
</dt> <dd>

[*Extension snap-in*](e-gly.md#mmc-extension-snap-in-gly) that is automatically added to a [*primary snap-in*](p-gly.md#mmc-primary-snap-in-gly) that is loaded in an MMC console.

</dd> <dt>

<span id="mmc_result_item_gly"></span><span id="MMC_RESULT_ITEM_GLY"></span>**result item**
</dt> <dd>

Item inserted by a snap-in in a [*list view*](l-gly.md#mmc-list-view-gly).

</dd> <dt>

<span id="mmc_result_pane_gly"></span><span id="MMC_RESULT_PANE_GLY"></span>**result pane**
</dt> <dd>

Right pane in a [*view*](v-gly.md#mmc-view-gly). The result pane has one of four possible view types: [*standard list view*](s-gly.md#mmc-standard-list-view-gly), [*taskpad view*](t-gly.md#mmc-taskpad-view-gly), [*custom OCX view*](c-gly.md#mmc-custom-ocx-view-gly), and [*custom web view*](c-gly.md#mmc-custom-web-view-gly).

</dd> <dt>

<span id="mmc_result_view_gly"></span><span id="MMC_RESULT_VIEW_GLY"></span>**result view**
</dt> <dd>

Contents of the result pane.

</dd> </dl>
