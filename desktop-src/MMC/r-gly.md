<dl> <dt>

<span id="mmc_registry_gly"></span><span id="MMC_REGISTRY_GLY"></span>**registry**
</dt> <dd>

System-defined database in which applications and system components store and retrieve configuration data. MMC uses the registry to retrieve the location of snap-in DLLs and to determine whether a [*node type*](https://www.bing.com/search?q=*node type*) is extendible, and if so, which [*extension snap-ins*](https://www.bing.com/search?q=*extension snap-ins*) extend that node type's functionality.

</dd> <dt>

<span id="mmc_required_extension_gly"></span><span id="MMC_REQUIRED_EXTENSION_GLY"></span>**required extension**
</dt> <dd>

[*Extension snap-in*](https://www.bing.com/search?q=*Extension snap-in*) that is automatically added to a [*primary snap-in*](https://www.bing.com/search?q=*primary snap-in*) that is loaded in an MMC console.

</dd> <dt>

<span id="mmc_result_item_gly"></span><span id="MMC_RESULT_ITEM_GLY"></span>**result item**
</dt> <dd>

Item inserted by a snap-in in a [*list view*](https://www.bing.com/search?q=*list view*).

</dd> <dt>

<span id="mmc_result_pane_gly"></span><span id="MMC_RESULT_PANE_GLY"></span>**result pane**
</dt> <dd>

Right pane in a [*view*](https://www.bing.com/search?q=*view*). The result pane has one of four possible view types: [*standard list view*](https://www.bing.com/search?q=*standard list view*), [*taskpad view*](https://www.bing.com/search?q=*taskpad view*), [*custom OCX view*](https://www.bing.com/search?q=*custom OCX view*), and [*custom web view*](https://www.bing.com/search?q=*custom web view*).

</dd> <dt>

<span id="mmc_result_view_gly"></span><span id="MMC_RESULT_VIEW_GLY"></span>**result view**
</dt> <dd>

Contents of the result pane.

</dd> </dl>
