# Windows Server 2008 Technologies

The following table describes new and updated technologies for Windows Server 2008.



| Feature                                                          | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Active Directory Rights Management Services<br/>           | Rights Management provides the ability for client computers to run applications based on Rights Management technologies.<br/> For a list of API additions, see [RMS in Windows Server 2008](https://msdn.microsoft.com/library/cc542533).<br/>                                                                                                                                                                                                                       |
| Active Directory Rights Management Services Scripting<br/> | The Microsoft Windows Active Directory Rights Management Services (AD RMS) Scripting API can be used to create scripts to administer an AD RMS installation.<br/> For more information, see [Active Directory Rights Management Services Scripting API](https://msdn.microsoft.com/library/bb968797).<br/>                                                                                                                                                                 |
| Core Windows<br/>                                          | For a list of API additions, see [Windows Server 2008](winprog-windows_server_2008).<br/>                                                                                                                                                                                                                                                                                                                                                                    |
| Distributed File System<br/>                               | Distributed File System (DFS) allows administrators to group shared folders located on different servers and present them to users as a virtual tree of folders known as a namespace. A namespace provides numerous benefits, including increased availability of data, load sharing, and simplified data migration.<br/> For more information, see the [What's New in DFS in Windows Server 2008](https://msdn.microsoft.com/library/bb736254).<br/> |
| Failover Cluster API<br/>                                  | For a list of API additions, see [What's New in the Failover Cluster API](https://msdn.microsoft.com/library/aa373124).<br/>                                                                                                                                                                                                                                                                                                                                                             |
| File Server Resource Manager<br/>                          | The File Server Resource Manager (FSRM) API is used to limit the size of a given directory using directory quotas, restrict the type of data that can be stored under a given directory using file screens, and generate storage reports that administrators can use to analyze storage utilization.<br/> For more information, see [File Server Resource Manager](https://msdn.microsoft.com/library/bb972746).<br/>                                     |
| Server Core for Windows Server 2008 <br/>                  | Server Core is a minimal installation of Windows Server 2008 that requires users to complete the initial configuration from the command line.<br/> For more information, see [Server Core](https://msdn.microsoft.com/library/ms723891).<br/>                                                                                                                                                                                                                                       |
| Terminal Services<br/>                                     | For a list of updates, see [What's New in Windows Server 2008](https://msdn.microsoft.com/library/bb892076).<br/>                                                                                                                                                                                                                                                                                                                                             |
| Volume Shadow Copy Service<br/>                            | The Volume Shadow Copy Service (VSS) is a set of COM APIs that implements a framework to allow volume backups to be performed while applications on a system continue to write to the volumes.<br/> For more information, see the [What's New in VSS in Windows Server 2008](https://msdn.microsoft.com/library/windows/desktop/bb972758).<br/>                                                                                                |
| Windows Error Reporting<br/>                               | For a list of new features, see [What's New in WER](https://msdn.microsoft.com/library/windows/desktop/bb513640).<br/>                                                                                                                                                                                                                                                                                                                                                                             |
