# Windows 8 and 8.1 Technologies

These are the new and updated technologies for use in desktop apps for Windows 8 and Windows 8.1:

-   [New Technologies](#new-technologies)
-   [Updated Technologies](#updated-technologies)
-   [Windows 8.1 Update API Notes](#windows-81-update-api-notes)

For an alphabetical list of functions and interfaces that are new for desktop apps, see [Windows 8 and 8.1 API List](https://msdn.microsoft.com/library/hh920511).

For info about Windows Store apps, see [Learn to build Windows Store apps](http://msdn.microsoft.com/library/windows/apps/).

## New Technologies



| Feature                                      | Description                                                                                                                                                                                                                                                                                         |
|----------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| App Packaging and Deployment<br/>      | Enables developers to produce, consume, or interact with app packages.<br/> For more info, see [App Packaging and Deployment](https://msdn.microsoft.com/library/windows/desktop/hh446593).<br/>                                                                                                                                |
| Assessment Execution Engine<br/>       | Enables the management and execution of Windows system assessments. Assessments can help a person understand the state of a system and remedy problems with performance, reliability, or functionality.<br/> For more info, see [Assessment Execution Engine](https://msdn.microsoft.com/library/windows/desktop/hh437709).<br/> |
| Compression API<br/>                   | Enables developers to manage versions, service, and extend the Windows MSZIP, XPRESS, XPRESS\_HUFF, and LZMS compression algorithms.<br/> For more info, see [Compression API](https://msdn.microsoft.com/library/windows/desktop/hh437596).<br/>                                                                        |
| Data Deduplication<br/>                | Enables backup applications to implement their own customized backup and restore methods that take advantage of the Data Deduplication feature.<br/> For more info, see [Data Deduplication API](https://msdn.microsoft.com/library/hh449204).<br/>                                             |

| Web Socket Protocol Component API<br/> | Enables asynchronous, bi-directional communication channels over HTTP that work across existing network intermediaries.<br/> For more info, see [Web Socket Protocol Component API](https://msdn.microsoft.com/library/windows/desktop/hh437448).<br/>                                             |

| Windows Storage Management API<br/>    | Provides a WMI interface for managing a wide range of storage configurations, from single-disk desktops to external storage arrays.<br/> For more info, see [Windows Storage Management API](https://msdn.microsoft.com/library/windows/desktop/hh830613).<br/>                                      |
| XAudio2<br/>                           | Provides a signal processing and mixing foundation for games.<br/> For more info, see [XAudio2](87557bb3-9d92-c85e-4e7a-d21af03db624).<br/>                                                                                                                                             |
| XInput<br/>                            | Enables interact with the Xbox 360 Controller when it is connected to a Windows computer.<br/> For more info, see [XInput](fbbc651b-9264-9b5d-01c6-efc58f50b19d).<br/>                                                                                                                  |

## Windows 8.1 Update API Notes

The following API can only be used in Windows 8.1 Update and later. In order to use these API in your apps, you must \#define the following constant in your project.

`#define WINBLUE_KBSPRING14`

This macro must be defined before including the *windows.h* header file for the first time.
