# Client Application Functions

The following functions are supported for client application development by the Windows Biometric Framework API.

## In this section



| Topic                                                                                       | Description                                                                                                                                                                                                                              |
|---------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WinBioAcquireFocus**](/windows/win32/Winbio/nf-winbio-winbioacquirefocus?branch=master)<br/>                                 | Acquires window focus.<br/>                                                                                                                                                                                                        |
| [**WinBioAsyncEnumBiometricUnits**](/windows/win32/Winbio/nf-winbio-winbioasyncenumbiometricunits?branch=master)<br/>           | Asynchronously enumerates all attached biometric units that match the input factor type.<br/>                                                                                                                                      |
| [**WinBioAsyncEnumDatabases**](/windows/win32/Winbio/nf-winbio-winbioasyncenumdatabases?branch=master)<br/>                     | Asynchronously enumerates all registered databases that match a specified type.<br/>                                                                                                                                               |
| [**WinBioAsyncEnumServiceProviders**](/windows/win32/Winbio/nf-winbio-winbioasyncenumserviceproviders?branch=master)<br/>       | Asynchronously returns information about installed biometric service providers. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                  |
| [**WinBioAsyncMonitorFrameworkChanges**](/windows/win32/Winbio/nf-winbio-winbioasyncmonitorframeworkchanges?branch=master)<br/> | Starts an asynchronous monitor of changes to the biometric framework.<br/>                                                                                                                                                         |
| [**WinBioAsyncOpenFramework**](/windows/win32/Winbio/nf-winbio-winbioasyncopenframework?branch=master)<br/>                     | Opens a handle to the biometric framework. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                                                       |
| [**WinBioAsyncOpenSession**](/windows/win32/Winbio/nf-winbio-winbioasyncopensession?branch=master)<br/>                         | Asynchronously connects to a biometric service provider and one or more biometric units. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                         |
| [**WinBioCancel**](/windows/win32/Winbio/nf-winbio-winbiocancel?branch=master)<br/>                                             | Cancels all pending biometric operations for a specified session. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                                |
| [**WinBioCaptureSample**](/windows/win32/Winbio/nf-winbio-winbiocapturesample?branch=master)<br/>                               | Captures a biometric sample and fills a biometric information record (BIR) with the raw or processed data.<br/>                                                                                                                    |
| [**WinBioCaptureSampleWithCallback**](/windows/win32/Winbio/nf-winbio-winbiocapturesamplewithcallback?branch=master)<br/>       | Captures a biometric sample asynchronously and returns the raw or processed data in a biometric information record (BIR).<br/>                                                                                                     |
| [**WinBioCloseFramework**](/windows/win32/Winbio/nf-winbio-winbiocloseframework?branch=master)<br/>                             | Closes a framework handle previously opened with [**WinBioAsyncOpenFramework**](/windows/win32/Winbio/nf-winbio-winbioasyncopenframework?branch=master). Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                   |
| [**WinBioCloseSession**](/windows/win32/Winbio/nf-winbio-winbioclosesession?branch=master)<br/>                                 | Closes a biometric session and releases associated resources. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                                    |
| [**WinBioControlUnit**](/windows/win32/Winbio/nf-winbio-winbiocontrolunit?branch=master)<br/>                                   | Allows the caller to perform vendor-defined control operations on a biometric unit. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                              |
| [**WinBioControlUnitPrivileged**](/windows/win32/Winbio/nf-winbio-winbiocontrolunitprivileged?branch=master)<br/>               | Allows the caller to perform privileged vendor-defined control operations on a biometric unit. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                   |
| [**WinBioDeleteTemplate**](/windows/win32/Winbio/nf-winbio-winbiodeletetemplate?branch=master)<br/>                             | Deletes a biometric template from the template store. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                                            |
| [**WinBioEnrollBegin**](/windows/win32/Winbio/nf-winbio-winbioenrollbegin?branch=master)<br/>                                   | Initiates a biometric enrollment sequence and creates an empty biometric template. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                               |
| [**WinBioEnrollCapture**](/windows/win32/Winbio/nf-winbio-winbioenrollcapture?branch=master)<br/>                               | Captures a biometric sample and adds it to a template. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                                           |
| [**WinBioEnrollCaptureWithCallback**](/windows/win32/Winbio/nf-winbio-winbioenrollcapturewithcallback?branch=master)<br/>       | Asynchronously captures a biometric sample and adds it to a template.<br/>                                                                                                                                                         |
| [**WinBioEnrollCommit**](/windows/win32/Winbio/nf-winbio-winbioenrollcommit?branch=master)<br/>                                 | Finalizes a pending biometric template and saves it to the database associated with the biometric unit used for enrollment. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>      |
| [**WinBioEnrollDiscard**](/windows/win32/Winbio/nf-winbio-winbioenrolldiscard?branch=master)<br/>                               | Ends the enrollment sequence and discards a pending biometric template. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                          |
| [**WinBioEnrollSelect**](/windows/win32/winbio/nf-winbio-winbioenrollselect?branch=master)<br/>                                 | Specifies the individual that you want to enroll when data that represents multiple individuals is present in the sample buffer. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/> |
| [**WinBioEnumBiometricUnits**](/windows/win32/Winbio/nf-winbio-winbioenumbiometricunits?branch=master)<br/>                     | Enumerates all attached biometric units that match the input type.<br/>                                                                                                                                                            |
| [**WinBioEnumDatabases**](/windows/win32/Winbio/nf-winbio-winbioenumdatabases?branch=master)<br/>                               | Enumerates all registered databases that match a specified type.<br/>                                                                                                                                                              |
| [**WinBioEnumEnrollments**](/windows/win32/Winbio/nf-winbio-winbioenumenrollments?branch=master)<br/>                           | Retrieves the biometric sub-factors enrolled for a specified identity and biometric unit. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                        |
| [**WinBioEnumServiceProviders**](/windows/win32/Winbio/nf-winbio-winbioenumserviceproviders?branch=master)<br/>                 | Retrieves information about installed biometric service providers. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                               |
| [**WinBioFree**](/windows/win32/Winbio/nf-winbio-winbiofree?branch=master)<br/>                                                 | Releases memory allocated for the client application by an earlier call to a Windows Biometric Framework API function. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>           |
| [**WinBioGetCredentialState**](/windows/win32/Winbio/nf-winbio-winbiogetcredentialstate?branch=master)<br/>                     | Retrieves a value that specifies whether credentials have been set for the specified user. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                       |
| [**WinBioGetDomainLogonSetting**](/windows/win32/Winbio/nf-winbio-winbiogetdomainlogonsetting?branch=master)<br/>               | Retrieves a value that specifies whether users can log on to a domain by using biometric information.<br/>                                                                                                                         |
| [**WinBioGetEnabledSetting**](/windows/win32/Winbio/nf-winbio-winbiogetenabledsetting?branch=master)<br/>                       | Retrieves a value that specifies whether the Windows Biometric Framework is currently enabled.<br/>                                                                                                                                |
| [**WinBioGetEnrolledFactors**](/windows/win32/winbio/nf-winbio-winbiogetenrolledfactors?branch=master)<br/>                     | Gets information about the biometric enrollments that the specified user has on the computer.<br/>                                                                                                                                 |
| [**WinBioGetLogonSetting**](/windows/win32/Winbio/nf-winbio-winbiogetlogonsetting?branch=master)<br/>                           | Retrieves a value that indicates whether users can log on by using biometric information.<br/>                                                                                                                                     |
| [**WinBioGetProperty**](/windows/win32/Winbio/nf-winbio-winbiogetproperty?branch=master)<br/>                                   | Retrieves a session, unit, or template property. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                                                 |
| [**WinBioIdentify**](/windows/win32/Winbio/nf-winbio-winbioidentify?branch=master)<br/>                                         | Captures a biometric sample and determines whether it matches an existing biometric template. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                    |
| [**WinBioIdentifyWithCallback**](/windows/win32/Winbio/nf-winbio-winbioidentifywithcallback?branch=master)<br/>                 | Asynchronously captures a biometric sample and determines whether it matches an existing biometric template.<br/>                                                                                                                  |
| [**WinBioLocateSensor**](/windows/win32/Winbio/nf-winbio-winbiolocatesensor?branch=master)<br/>                                 | Retrieves the ID number of a biometric unit selected interactively by a user.<br/>                                                                                                                                                 |
| [**WinBioLocateSensorWithCallback**](/windows/win32/Winbio/nf-winbio-winbiolocatesensorwithcallback?branch=master)<br/>         | Asynchronously retrieves the ID number of the biometric unit selected interactively by a user.<br/>                                                                                                                                |
| [**WinBioLockUnit**](/windows/win32/Winbio/nf-winbio-winbiolockunit?branch=master)<br/>                                         | Locks a biometric unit for exclusive use by a single session. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                                    |
| [**WinBioLogonIdentifiedUser**](/windows/win32/Winbio/nf-winbio-winbiologonidentifieduser?branch=master)<br/>                   | causes a fast user switch to the account associated with the last successful identification operation performed by the biometric session.<br/>                                                                                     |
| [**WinBioMonitorPresence**](/windows/win32/winbio/nf-winbio-winbiomonitorpresence?branch=master)<br/>                           | Turns on the face-recognition or iris-monitoring mechanism for the specified biometric unit. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                     |
| [**WinBioOpenSession**](/windows/win32/Winbio/nf-winbio-winbioopensession?branch=master)<br/>                                   | Connects to a biometric service provider and one or more biometric units.<br/>                                                                                                                                                     |
| [**WinBioRegisterEventMonitor**](/windows/win32/Winbio/nf-winbio-winbioregistereventmonitor?branch=master)<br/>                 | Registers a callback function to receive event notifications from the service provider associated with an open session.<br/>                                                                                                       |
| [**WinBioReleaseFocus**](/windows/win32/Winbio/nf-winbio-winbioreleasefocus?branch=master)<br/>                                 | Releases window focus.<br/>                                                                                                                                                                                                        |
| [**WinBioRemoveAllCredentials**](/windows/win32/Winbio/nf-winbio-winbioremoveallcredentials?branch=master)<br/>                 | Removes all credentials from the store. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                                                          |
| [**WinBioRemoveAllDomainCredentials**](/windows/win32/Winbio/nf-winbio-winbioremovealldomaincredentials?branch=master)<br/>     | Removes all user credentials for the current domain from the store. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                              |
| [**WinBioRemoveCredential**](/windows/win32/Winbio/nf-winbio-winbioremovecredential?branch=master)<br/>                         | Deletes a biometric logon credential for a specified user. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                                       |
| [**WinBioSetCredential**](/windows/win32/Winbio/nf-winbio-winbiosetcredential?branch=master)<br/>                               | Saves a biometric logon credential for the current user. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                                                                         |
| [**WinBioSetProperty**](/windows/win32/winbio/nf-winbio-winbiosetproperty?branch=master)<br/>                                   | Sets the value of a standard property associated with a biometric session, unit, template, or account. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                           |
| [**WinBioUnlockUnit**](/windows/win32/Winbio/nf-winbio-winbiounlockunit?branch=master)<br/>                                     | Releases the session lock on the specified biometric unit.<br/>                                                                                                                                                                    |
| [**WinBioUnregisterEventMonitor**](/windows/win32/Winbio/nf-winbio-winbiounregistereventmonitor?branch=master)<br/>             | cancels event notifications from the service provider associated with an open biometric session.<br/>                                                                                                                              |
| [**WinBioVerify**](/windows/win32/Winbio/nf-winbio-winbioverify?branch=master)<br/>                                             | Captures a biometric sample and determines whether the sample corresponds to the specified user identity. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                        |
| [**WinBioVerifyWithCallback**](/windows/win32/Winbio/nf-winbio-winbioverifywithcallback?branch=master)<br/>                     | Asynchronously captures a biometric sample and determines whether the sample corresponds to the specified user identity.<br/>                                                                                                      |
| [**WinBioWait**](/windows/win32/Winbio/nf-winbio-winbiowait?branch=master)<br/>                                                 | Blocks caller execution until all pending biometric operations for a session have been completed or canceled. Starting with Windows 10, build 1607, this function is available to use with a mobile image.<br/>                    |
